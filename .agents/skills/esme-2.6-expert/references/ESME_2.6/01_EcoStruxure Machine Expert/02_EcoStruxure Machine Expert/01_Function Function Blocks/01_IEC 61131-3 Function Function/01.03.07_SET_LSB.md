# SET\_LSB

This standard [bit manipulation function](BitManipulationFunctions_Overview.html#BitManipulationFunctions_Overview) contained in the <BIT\_UTIL> library writes the value applied at input LSB to the least significant Byte of the bit string applied at input DATA.

| Parameter | Data type | Description |
| --- | --- | --- |
| LSB | BYTE | Value to be written to the LSB of DATA |
| DATA | WORD | Input bit string to be modified |
| OUT | WORD | Output bit string with modified written LSB |
