# AxisStatus Output

Bitwise status output is DWORD data type. Every Boolean function block output is mapped to one bit of this status DWORD which can be further processed and evaluated in the application.

Data type: DWORD

**NOTE:**

Bits that are not listed in the table below are not for use and are reserved.

| Bit | Meaning | Corresponding FB input/output |
| --- | --- | --- |
| 0 | [HW STO enabled](function_MotionFB.html#function_MotionFB__STOhardwired) | S\_HW\_STO\_Active |
| 1 | STO function is active | [S\_STO\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 3 | SOS function is active | [S\_SOS\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 5 | SS1 function is active | [S\_SS1\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 7 | SS2 function is active | [S\_SS2\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 9 | SLS1 function is active | [S\_SLS1\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 10 | SLS2 function is active | [S\_SLS2\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 11 | SLS3 function is active | [S\_SLS3\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 12 | SLS4 function is active | [S\_SLS4\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 13 | SDIpos function is active | [S\_SDIneg\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 14 | SDIneg function is active | [S\_SDIpos\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 15 | SMS function is active | [S\_SMS\_SafetyActive](SafetyActiveOut_MotionFB.html#SafetyActiveOut_MotionFB) |
| 18 | Ramp monitoring is active | S\_RampMonitoringActive |
| 19 | Safety-related function requested | A SAFEFALSE signal is applied to at least one of the [S\_\*\_Request](motionFB_SRequestInputs.html#motionFB_SRequestInputs) inputs |
| 20 | Requested functions are active | [S\_AllReqFuncActive](AllReqFuncActive_MotionFB.html#AllReqFuncActive_MotionFB) |
| 22 | Operational (safety logic state) | – |
| 23 | No error in the safety logic | [S\_NotErrFUNC](NotErrFunc_MotionFB.html#NotErrFunc_MotionFB) |
