# NE / NE\_S - Comparison: <>

This comparison function compares the operands connected to the input parameters in regard to inequality. The function is available as standard function NE and safety-related function NE\_S.

## NE

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | ELEMENTARY | First input value |
| IN2 | ELEMENTARY | Second input value |
| OUT | BOOL | TRUE if IN1 and IN2 are not equal.  FALSE if IN1 and IN2 are equal. |

## NE\_S

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | SAFEELEMENTARY | First input value |
| IN2 | SAFEELEMENTARY | Second input value |
| OUT | SAFEBOOL | TRUE if IN1 and IN2 are not equal.  FALSE if IN1 and IN2 are equal. |

**NOTE:**

All input parameters must have the same data type.
