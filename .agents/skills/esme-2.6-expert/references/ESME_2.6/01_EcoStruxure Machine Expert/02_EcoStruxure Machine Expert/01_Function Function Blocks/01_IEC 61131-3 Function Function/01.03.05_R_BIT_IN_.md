# R\_BIT\_IN\_\*

This standard [bit manipulation function](BitManipulationFunctions_Overview.html#BitManipulationFunctions_Overview) contained in the <BIT\_UTIL> library resets a particular bit in the bit string applied at input IN.

**NOTE:**

Depending on the data type applied at input IN, the function R\_BIT\_IN\_BYTE, R\_BIT\_IN\_DWORD or R\_BIT\_IN\_WORD has to be used.

| Parameter | Data type | Description |
| --- | --- | --- |
| IN | BYTE, WORD, DWORD (see note above) | Input bit string |
| BIT\_NO | BYTE | Number (position) of the bit to be reset to FALSE |
| OUT | Same as data type at 'IN' (see note above) | Value with FALSE at bit BIT\_NO |

EIO0000002267.00
