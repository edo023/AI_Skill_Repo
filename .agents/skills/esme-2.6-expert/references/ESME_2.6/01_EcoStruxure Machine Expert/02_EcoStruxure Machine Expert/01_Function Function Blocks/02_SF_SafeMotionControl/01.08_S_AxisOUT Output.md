# S\_AxisOUT Output

Output for connecting the SafeAxisOut data item provided by the safety logic in the 'Devices' window. This output must not remain unconnected as the safety logic needs the information written to this output stored to the SafeAxisOut data item.

Data type: SAFEDWORD

**Connection:** This output **must** be connected to the SafeAxisOut data item provided by the safety logic which is controlling the respective drive or axis.

Connect the output as follows:

1. Click the Safety Module in the devices tree (in the 'Devices' window).
2. Drag the SafeAxisOut data item into the respective code worksheet.
3. In the appearing 'Variable' dialog box, declare a related global output variable. Either accept the variable name proposed by EcoStruxure Machine Expert - Safety or define/select another one.
4. After confirming the variable dialog box, the shape of the variable is added to the cursor.

   Drop it on the S\_AxisOUT function block output by a click to connect it directly on insertion.

Note that connecting the incorrect data item can cause hazards because the incorrect axis is controlled/monitored.
