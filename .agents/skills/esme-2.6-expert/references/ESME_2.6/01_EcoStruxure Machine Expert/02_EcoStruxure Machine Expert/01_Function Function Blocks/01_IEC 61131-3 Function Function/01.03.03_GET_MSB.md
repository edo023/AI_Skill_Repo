# GET\_MSB

This standard [bit manipulation function](BitManipulationFunctions_Overview.html#BitManipulationFunctions_Overview) contained in the <BIT\_UTIL> library reads the value of the most significant Byte (MSB) of the bit string applied at input IN.

| Parameter | Data type | Description |
| --- | --- | --- |
| IN | WORD | Input bit string |
| OUT | BYTE | Value of the MSB |
