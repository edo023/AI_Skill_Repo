# DiagCode Output

Output for diagnostic message of the function block.

Data type: WORD

**Connection:** If required the signal can be connected to a standard variable of the WORD data type.

Function block-specific diagnostic and status messages are output via this output parameter and can be made available to higher-level diagnostic tools.

The diagnostic code is specified as WORD data type. The respective hexadecimal value is specified in the "[Diagnostic codes](codes_MotionFB.html#codes_MotionFB)" help topic.

EIO0000002271.03
