# Implementation of Safety Requirements from Applicable Standards

The safety-related function block has been developed according to the safety requirements (from applicable standards) listed in this section. Other requirements than these standards must be observed when implementing the safety-related function.

This section describes either how the function block meets the requirements of the standards or what measures need to be taken to meet the requirements of the standards.

| Standards | Context / Requirement | Implementation |
| --- | --- | --- |
| EN ISO 12100-2 | Start-up after interruption of supply voltage / spontaneous restart | The function block supports a start-up inhibit and restart inhibit of the function block after   * start-up of the safety logic or activation of the function block * removing an STO or SS1 request for the safety-related function.  Plan and implement the start-up behavior according to your risk analysis. To help prevent an unintended start-up, you may need to perform an additional function start once the safety-related function has been reset. This will depend on both the results of the risk analysis and the signal path of the reset signal. |
| EN ISO 13849-1 | Manual reset device | The Reset input supports the function of a manual reset device.  **NOTE:**  Resetting does not occur with a falling edge (TRUE > FALSE), as specified by standard ISO 13849-1, but with a rising edge (FALSE > TRUE). |
| ISO 13850 | Positive opening operation | Only use approved control devices, which conform to standard ISO 13850. |
| ISO 13850 | Using wires and cables as manual controls | Meet the requirements of this standard, according to the risk analysis you carried out. |
| EN 60204 | Safety of machinery | The function block executes the stop categories 0, 1 and 2. |
| IEC 61800-5-2 | Adjustable speed electrical power drive systems - Part 5-2: Safety requirements - Functional Safety | Monitoring of safety-related functions according to the following sections:  * 4.2.2.2: Safe torque off (STO) * 4.2.2.3: Safe stop 1 (SS1) * 4.2.2.4: Safe stop 2 (SS2) * 4.2.3.1: Safe operating stop (SOS) * 4.2.3.4: Safely-limited speed (SLS) * 4.2.3.10: Safe direction (SDI) |
