# Grid in FBD/LD Worksheets

The code editor provides a grid which can be activated by selecting 'Layout > Grid'. The grid is displayed as black dots in the worksheet.

Activation of the grid is only applied to the present worksheet.

If the grid is activated, a set insertion mark and each newly inserted object is automatically aligned to the grid.

The grid has no effect when moving existing objects.
