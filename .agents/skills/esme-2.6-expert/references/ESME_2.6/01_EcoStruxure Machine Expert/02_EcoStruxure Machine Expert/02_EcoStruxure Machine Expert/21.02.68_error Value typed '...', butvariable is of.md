# Error: Value typed '...', but variable is of type '...'!

The value of a literal connected to a formal parameter of a FU/FB has an incorrect data type (the formal parameter expects a different data type).

Error examples: The Boolean literal 'FALSE' cannot be connected to an input of an ADD function which expects an Integer value. The Integer literal '120' cannot be connected to an input of an AND function which expects a Boolean value.

* In the message window, double-click the error message.

  The worksheet opens and the cursor marks the error.
* Double-click the literal concerned to open the 'Variable' dialog.
* Enter a literal with the data type expected by the formal parameter.

**NOTE:**

Double-clicking the formal parameter inside the block icon opens the ['Formal Parameter'dialog](dialog_formalparameter.html#dialog_formalparameter) where the expected data type is specified.
