# Error: Input format incorrect!

Literal with incorrect data type used in a code worksheet.

This literal has been inserted directly using the 'Variable' dialog (while editing the code) or it has first been defined in the variables worksheet by entering an initial value for a variable and was later inserted using the 'Variable' dialog.

* In the message window, double-click the error message.

  Depending on the way the literal has been defined, the code worksheet or the variables grid opens and the error is marked.
* Enter a literal with the expected data type or enter the expected initial value.
