# 'Online Layout' Dialog

The 'Online Layout' dialog is used to modify the display of online values in code worksheets, i.e., which values are displayed and how they are represented. It is called by selecting 'Online > Online Layout...'. The dialog can also be opened by selecting 'Online Layout...' from the context menu of a variable.

**NOTE:**

The 'Online Layout' dialog is only available while Machine Expert – Safety is online, i.e., if worksheets are displayed in [variable status](DisplayVariableStatus.html#DisplayVariableStatus).

|  |  |
| --- | --- |
| **Dialog area 'Show values'** | |
| Only values | Hides the names of variables and displays the values. |
| Value beside symbol | Displays the value beside the name. |
| Value below symbol | Displays the value below the name. |
| **Dialog area 'Boolean values'** | |
| explicit Boolean values | Activates the representation of explicit Boolean values. |
| textual Boolean values | Displays the words 'TRUE' and 'FALSE' in addition to the colors red and blue. |
| numeric Boolean values | Displays the numbers '0' and '1' in addition to the colors red and blue. |
| show variable values | Displays the values for all variables. |
| show parameter values | Displays the values for all formal parameters. |

**NOTE:**

The ['Debug' dialog](dialog_debug_resource_name.html#dialog_debug_resource_name) also provides options to customize the online display.
