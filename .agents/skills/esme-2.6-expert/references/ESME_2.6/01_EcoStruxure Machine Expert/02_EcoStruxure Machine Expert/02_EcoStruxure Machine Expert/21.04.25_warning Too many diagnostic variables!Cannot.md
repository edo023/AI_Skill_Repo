# Warning: Too many diagnostic variables! Cannot create diagnostic information for variable '...'.

This message is only relevant if the Safety Logic Controller supports implicit function block diagnostic.

The available memory for function block diagnostic is limited. Thus, the diagnostic flag can only be set for a particular number of variables. If the 'DIAG' flag is set for too many variables, this message is generated.

Reduce the number of diagnostic variables:

* Examine the local variables worksheets for variables that are currently not needed as diagnostic variables
* and clear the 'DIAG' flag of these variables.
