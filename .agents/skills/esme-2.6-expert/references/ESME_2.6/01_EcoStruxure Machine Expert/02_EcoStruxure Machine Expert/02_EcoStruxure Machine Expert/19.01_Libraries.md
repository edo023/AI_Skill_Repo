# Libraries

According to the IEC 61131 standard, libraries with reusable functions and function blocks are supported. Due to specific safety requirements, Machine Expert – Safety only allows the use of specially designed and safety-related firmware libraries (file extension is \*.swl).

Libraries management (i.e., adding and deleting) is done in the ['Libraries' folder](subtree_library_intheprojecttree.html#subtree_library_intheprojecttree) of the project tree. After adding a library, the contained functions and function blocks can be inserted into the code using the Edit Wizard.
