# Error: Missing data type declaration or function block code body! '...' undefined!

This error occurs if a code worksheet of an [instantiated](instantiation.html#instantiation) function block is missing, or if a data type is used which has not been declared. Possibly, there is a typing error in an FB instance name or data type name.

* Edit the code of the function block or
* correct the typing error.
