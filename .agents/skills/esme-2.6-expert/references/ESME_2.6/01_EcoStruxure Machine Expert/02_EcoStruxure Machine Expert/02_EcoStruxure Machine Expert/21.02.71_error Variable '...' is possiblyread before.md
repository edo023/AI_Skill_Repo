# Error: Variable '...' is possibly read before written! This could result in storing behaviour of the application

Basically, variables must be written before they can be read. Otherwise, the application may show a storing behavior because the result of writing a value would not be effective until a later cycle.

Double-click the error in the message window to jump to the suspected worksheet (where the variable concerned is marked) and correct the code. The execution order displayed inside of FU/FB icons and above left power rails are helpful to eliminate this error.
