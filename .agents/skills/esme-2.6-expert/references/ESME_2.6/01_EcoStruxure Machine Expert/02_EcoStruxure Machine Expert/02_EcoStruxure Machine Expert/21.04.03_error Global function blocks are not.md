# Error: Global function blocks are not supported. (Specific Code Generator)

The global variables worksheet contains an FB instance declaration. Function block instances, however, can only be declared in a local variables worksheet.

Double-click the error in the message window to jump to the global worksheet (where the invalid declaration is marked). Delete the global instance declaration and declare it in the local variables worksheet of the POU where the FB instance is called.
