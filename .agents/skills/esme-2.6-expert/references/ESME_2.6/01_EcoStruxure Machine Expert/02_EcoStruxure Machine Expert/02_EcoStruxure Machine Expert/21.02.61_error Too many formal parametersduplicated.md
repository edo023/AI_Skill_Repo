# Error: Too many formal parameters duplicated in function! Change your layout!

Inputs of AND/AND\_S/OR/OR\_S functions can be duplicated to prevent unnecessary function cascading. The code contains a function at which too many formal parameters have been duplicated.

Double-click the error in the message window to jump to the suspected worksheet (where the error is marked). Delete as many duplicated inputs as necessary and use additional functions instead. If a formal parameter is already connected to a variable, the connection must be deleted prior to deleting the input. Refer to the topic ["Function Inputs:Adding/Deleting"](DuplicatingAND_OR_Inputs.html#DuplicatingAND_OR_Inputs)" for details.
