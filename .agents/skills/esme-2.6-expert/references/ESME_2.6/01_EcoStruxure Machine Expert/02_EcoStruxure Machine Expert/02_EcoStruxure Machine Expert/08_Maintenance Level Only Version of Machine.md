# Maintenance Level Only Version of Machine Expert – Safety

Depending on your Machine Expert – Safety version, you might only have access to the Maintenance level.

If you are working with such a Machine Expert – Safety version, the following operations are possible:

* Opening all worksheet types and properties dialogs of objects in read-only mode.
* Monitoring the Safety Logic Controller by [displaying the variable status](DisplayVariableStatus.html#DisplayVariableStatus).
* [Printing the project](printingandpreview.html#printingandpreview).

After entering the Safety Logic Controller password, the following operations are also possible:

* [Downloading](downloadingaproject.html#downloadingaproject) the project to the Safety Logic Controller.
* Controlling the Safety Logic Controller (start, stop, reset, ...).
* [Switching between safe and debug mode](SafePLC_OperatingModes.html#SafePLC_OperatingModes).
* Executing debug commands for [overwriting and forcing variables](debuggingtheproject.html#debuggingtheproject).
* Generating the application image file using the 'Project > Create Application Download' command if the Safety Logic Controller device parameter 'ApplicationDownloadAllowed' = 'Yes'.

  Refer to the topic ["Application Download"](SE_ApplicationDownload.html#SE_ApplicationDownload) for further information.

**Further Information:**

For further information on the password handling refer to the topic ["Password Protection for Projects and Safety Logic Controller"](PasswordProtection.html#PasswordProtection).
