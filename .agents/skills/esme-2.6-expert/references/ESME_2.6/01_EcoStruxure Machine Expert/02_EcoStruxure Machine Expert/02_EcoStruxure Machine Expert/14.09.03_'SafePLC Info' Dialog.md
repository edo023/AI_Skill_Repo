# 'SafePLC Info' Dialog

The 'SafePLC Info' dialog appears if you click the 'Info' button in the ['SafePLC' dialog](dialogSafePLC.html#dialogSafePLC) or in the ['Debug' dialog](dialog_debug_resource_name.html#dialog_debug_resource_name).

The dialog shows the following information:

| **Dialog element** | **Meaning** |
| --- | --- |
| Name | Safety-related project loaded in Machine Expert – Safety/on the Safety Logic Controller. |
| Date | Date and time of the last project modification. |
| CRC | CRC checksum of the project. |
| User | User who has downloaded the project as well as the currently logged-in user. |
| State | Operational [state of the Safety Logic Controller](possibleSafePLCstates.html#possibleSafePLCstates). |
| Variables forced | Indicates whether variables are forced. |
| Errors | Indicates whether device error messages can be uploaded from the Safety Logic Controller and displayed in the 'SafePLC Errors' tab of the [message window](messagewindow.html#messagewindow). |
| Cycle time  Execution time (max.) | Time period needed from the beginning of one control cycle to the beginning of the next cycle and pure execution time needed to process the program code.  Using these values and further information contained in the Safety Logic Controller manual, you are able to calculate the overall response time of the safety function. |
| Data | Free memory for application data. |
| Program | Free memory for application code. |
| Firmware version | Firmware version number of the Safety Logic Controller. |
