# Error: Not enough memory for force list! Reduce the amount of forced variables.

Too many variables are forced in debug mode.

Unforce all variables which are not needed to be forced for the present debug operation. Use the ['Debug' dialog](dialog_debug_resource_name.html#dialog_debug_resource_name) for that purpose.

**NOTE:**

Forced values are shown on a pink background in online variables worksheets.
