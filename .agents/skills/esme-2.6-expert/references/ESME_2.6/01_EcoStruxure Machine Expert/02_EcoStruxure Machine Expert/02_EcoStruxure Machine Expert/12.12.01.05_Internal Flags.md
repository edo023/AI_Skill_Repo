# Internal Flags

If functions are connected to each other via a direct line (from connection point to connection point) Machine Expert – Safety may need an internal temporary flag for compiling the project.

If the data type of an internal flag is not determined by the structure of the network, you have do this manually using the ['Flag Type' dialog](changingtheflagtype.html#changingtheflagtype). This dialog contains a drop-down combo box for specifying the data type of the flag. The flag type dialog is called by double-clicking on the particular connection line.

**NOTE:**

Depending on the code context, the manual selection of a flag type is possible but not necessary. In such a case, the manually set flag type is not verified by the compiler.
