# Error: Instance signature diverges from type signature!

A signature of an FB instance in the code differs from the signature of the FB type in the variables worksheet.

Error example: An instance of the FB type x is used in a code worksheet. Then, type x is modified in the appropriate local variables worksheet of the FB, for example, by deleting an input. After this modification, the instance in the code worksheet has not been updated.

Double-click the error in the message window to jump to the suspected worksheet (where the error is marked) and correct.
