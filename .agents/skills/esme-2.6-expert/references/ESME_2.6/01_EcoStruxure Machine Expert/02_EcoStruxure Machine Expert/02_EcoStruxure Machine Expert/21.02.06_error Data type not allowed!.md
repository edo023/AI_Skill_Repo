# Error: Data type not allowed!

In the 'Flag Type' dialog, a data type has been specified which is not allowed according to the IEC 61131 standard.

1. In the message window, double-click the error message.

   The worksheet opens and the cursor marks the function output connected to the flag.
2. Select 'Object Properties...' from the context menu of the flag (connection line).
3. In the ['Flag type' dialog](changingtheflagtype.html#changingtheflagtype), open the drop-down combo box and select a data type.
