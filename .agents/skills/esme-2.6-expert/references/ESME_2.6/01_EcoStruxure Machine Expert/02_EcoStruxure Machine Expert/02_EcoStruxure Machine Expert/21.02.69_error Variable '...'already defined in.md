# Error: Variable '...' already defined in global variable worksheet!

A local and a global variable with the same name are declared. According to the IEC 61131 standard, the names of global variables must be unique within the project.

Double-click the error in the message window to jump to the local variables worksheet (where the declaration is marked) and rename the local variable. Afterwards, you have to modify the related code accordingly.
