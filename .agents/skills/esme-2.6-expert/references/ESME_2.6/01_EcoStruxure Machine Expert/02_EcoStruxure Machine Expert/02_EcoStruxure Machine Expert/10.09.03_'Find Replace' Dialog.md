## General information on finding and replacing text elements

The system provides two possibilities of searching and replacing text elements in code and variables worksheets: Local and global text search/replacement.

The find function is applied to comments, variables, function/function block names, FB instance names, formal parameter names, contacts, and coils. For text replacement some rules/restrictions apply. Refer to the section ["Rules for replacing text elements"](findingtextelementsdialogfind.html#findingtextelementsdialogfind__Rules4Replace) below.

Finding/replacing text elements **locally** only affects the active worksheet. The **global** find/replace operation is applied to several or all worksheets, depending on your selection in the 'Global Find/Global Replace' dialog.

**NOTE:**

No "replace all matches" functionality is available. Instead, each replacement must be performed individually.

**NOTE:**

In ST POUs, the following restrictions apply to both the global and local text search:

* No text replacements using the 'Find and Replace' function can be done. You can only find text and overwrite it manually.
* Only the search direction 'Down' is supported.
* The search function is not available while the ST editor runs online mode.

For both local and global operation, the ['Find/Replace' dialog](findingtextelementsdialogfind.html#findingtextelementsdialogfind__DialogFind) is used. The dialog provides different search/replace options for local and global operation as listed in the [dialog description](findingtextelementsdialogfind.html#findingtextelementsdialogfind__DialogFind).

## Local text search & replacement

1. Left click into the code or variables worksheet where you want to find or replace text.
2. Select 'Edit > Find...' to open the 'Find/Replace' dialog.
3. In the 'Find What' dialog field, enter the text string to be found/replaced or select an entry from the drop-down list.
4. Only for text replacement: In the 'Replace With' field, enter the new string or select an entry from the drop-down list.

   Observe the restrictions mentioned in the note above.
5. Mark one or both search options (['Whole word' or 'Match Case'](findingtextelementsdialogfind.html#findingtextelementsdialogfind__DialogFind)).
6. Modify the search direction if desired.
7. Click 'Find Next' to start the search operation within the worksheet. The first match of this text element is marked.
8. Click 'Replace' to replace this match and find the next one.

## Global text search & replacement

1. To find/replace text only in particular worksheets or POUs, mark the related icons in the project tree. Multi-selection is possible by holding the <Ctrl> or the <Shift> key pressed while left clicking the desired tree icons.
2. Select 'Edit > Global Find...' (even if you want to replace text). The 'Global Find/Global Replace' dialog appears.
3. In the 'Find What' dialog field, enter the text string to be found/replaced or select an entry from the drop-down list.
4. Only for text replacement: In the 'Replace With' field, enter the new string or select an entry from the drop-down list.

   Observe the restrictions mentioned in the note above.
5. Mark one or several of the options (see [dialog description](findingtextelementsdialogfind.html#findingtextelementsdialogfind__DialogFind)).
6. Using the checkboxes in the 'Selection' dialog area, you can include/exclude worksheet types.
7. Modify the search direction if desired.
8. To apply the search/replacement only to the worksheets selected in step 1, activate the 'Marked' radio button.

   If 'All' is activated, any selection in the project tree is ignored.
9. Click 'Find Next' to start the search operation. The first match of this text element is marked.
10. Click 'Replace' to replace this match and find the next one.

## Finding the next/previous element

Using the commands 'Edit > Find Previous' or 'Edit > Find Next' the same text element can be searched again without using the 'Find' dialog (search direction 'Up' or 'Down' as set in the dialog). Both commands are available for local and global search operations.

## 'Find/Replace' dialog (local and global)

| **Dialog element** | **Meaning** |
| --- | --- |
| Find What: | String to be found/replaced. |
| Replace With: | Replacement string (only available on the '(Global) Replace' tab). |
| Whole word | The found text has to match with the whole search text. |
| Match Case | If activated, the search is done case-sensitive: only text matching upper and lower cases of the entered word is found. |
| 'Selection' area | Worksheet types to which the search/replace operation is to be applied (only available for global text operation). |
| Direction:'Up' or 'Down' | Search direction in the worksheet.  **NOTE:**  In ST code, only the search direction 'Down' is supported. |
| Range'All' or 'Marked' | Range of worksheets included in the search/replace operation (only available for global text operation).   * All - all worksheets that match the settings in the 'Selection' dialog area are processed. Any icon selection in the project tree is ignored. * Marked - only worksheets which are marked in the project tree are processed. Additionally, the 'Selection' settings are applied. |
| Find Next | Searches for the next text element. |
| Replace | Replaces the found and marked text match by the text specified in the 'Replace With' field.  **NOTE:**  No "replace all matches" functionality is available. Instead, each replacement must be performed individually.  Observe the restrictions for ST POUs mentioned in the note in section "[General Information...](findingtextelementsdialogfind.html#findingtextelementsdialogfind__FindReplace_General)" above. |
