# Error: Too many connection points in network! Reduce size of network!

An LD network contains too many junctions of lines and objects. Reduce the number of connection points by restructuring the suspected LD code.

Double-click the error in the message window to jump to the suspected worksheet (where the error is marked) and split the affected LD network into several networks. Insert additional networks and declare/use new variables to substitute connection points/lines.
