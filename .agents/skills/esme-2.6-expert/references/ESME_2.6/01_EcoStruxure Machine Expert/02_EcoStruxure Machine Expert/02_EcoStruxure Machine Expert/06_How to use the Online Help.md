# How to use the Online Help

**NOTE:**

Knowledge of MS-Windows is presumed for understanding the descriptions contained in this online help.

## How to call the online help

To call the online help

* select the 'Contents' item in the '?' menu.

  The help contents appears. Search for a help topic as described below.

  Or
* press <F1> in an active dialog or window.

  The context-sensitive help topic for the active control/window appears.

  Or
* press <Shift> + <F1> while an element/object in a worksheet is selected (e.g., a variable or any object or block), to call an element/object-related help topic.

## Finding a topic in the help system

After opening the help contents via the '?' menu, you have several possibilities for finding the required information.

* Use the 'Contents' tab and browse through the help system.
* Click the 'Index' tab and enter the keyword you are looking for.
* Click on 'Search' to use the full text search functionality. Type your request into the text field. Enclose the expression in quotation marks to find only topics which literally contain the search text. Observe the find options at the bottom of the 'Search' tab.

  After clicking 'List Topics' or pressing <Enter>, a list of topics is displayed which contain the search expression.

  To display the desired help topic, double-click the related entry in the result list.
* Navigate via the breadcrumbs at the top of each help page. This way, you can also access the navigation pages.
* The navigation pages of the HTML5 help system also provide a search function in the upper right corner.

## Toggled help text

Some help information is contained in so-called help toggles. Toggles are represented by a triangle at the beginning of the line.

![](images/arrowright.gif) If a toggle is closed, only its heading is visible but the contained text is hidden.

![](images/arrowdown.gif) An expanded toggle shows its content on a colored background.

In the top right corner of each page that contains at least one toggle, the 'Expand All/Collapse All' buttons can be used to expand/close all toggles in the current help topic.

**NOTE:**

Toggles are printed as currently shown. Expand all desired toggles before printing a page.

## Calling help on functions and function blocks

Machine Expert – Safety provides full help on firmware functions/function blocks. There are three possibilities to call function/function block help:

* Select '? > Help on FB/FU' to open the table of contents.
* Call block-specific help information on each function and function block in the [Edit Wizard](editwizard_generaldescription.html#editwizard_generaldescription) as follows: Right-click on the specific function or function block in the Edit Wizard selection area and select 'Help on FB/FU' from the context menu.
* Right-click on a function/FB inserted in an FBD/LD code worksheet and select 'Help on FB/FU' from the context menu.
