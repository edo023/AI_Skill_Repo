# Error: Value out of range!

The value of a literal exceeds the valid range determined by the data type of this literal.

Error example: negative Integer literal or Integer literal > 32767.

Double-click the error in the message window to jump to the suspected worksheet (where the error is marked) and correct the constant value according to its [data type](datatypesinSafeFOX.html#datatypesinSafeFOX).
