# Error (ST parser): Identifier expected

The ST parser detected a missing identifier in an ST expression. This could be, for example, a variable, an FB instance name, or a function name. The code position where the element is expected is underlined red in the code editor.

**NOTE:**

The Edit Wizard prevents entering syntactical errors. Using the Edit Wizard, syntax templates prepared with place holders can be inserted.

**Further Information:**

Refer to the topic ["ST Code Elements"](elementsintheSTeditor.html#elementsintheSTeditor) for information on the ST syntax.
