# Error: Invalid connection!

A variable is connected in an invalid way because the appropriate device is not available.

Possibly, a device has been removed from the bus structure thus deleting the terminal to which the variable was connected.

Double-click the error in the message window to jump to the global variables worksheet where the 'Terminal' field of the affected variable is marked red. Connect the variable to a valid terminal or delete the declaration.
