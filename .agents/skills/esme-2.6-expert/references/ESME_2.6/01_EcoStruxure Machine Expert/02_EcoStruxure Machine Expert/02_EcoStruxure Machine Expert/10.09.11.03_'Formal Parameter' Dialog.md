# 'Formal Parameter' Dialog

The read-only 'Formal Parameter' dialog displays the properties of [formal parameters](formalparameter.html#formalparameter) of [functions and function blocks](functionandfunctionblock.html#functionandfunctionblock).

The dialog is called by double-clicking on the formal parameter name (inside the FU/FB symbol), or by right-clicking on the parameter name and selecting the context menu item 'Object Properties...'.

The dialog displays the following information:

* 'Name' indicates the name of the formal parameter.
* 'Type' indicates the parameter type (input or output).
* 'Data type' indicates the IEC 61131 data type of the formal parameter.
