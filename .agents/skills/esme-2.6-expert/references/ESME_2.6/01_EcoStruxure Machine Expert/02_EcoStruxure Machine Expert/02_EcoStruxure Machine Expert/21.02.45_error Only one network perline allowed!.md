# Error: Only one network per line allowed!

More than one LD network are at the same vertical position in the code. This is not allowed.

The code editor prevents the insertion of a new network on the right of an existing network. This layout error, however, may result from moving existing networks.

1. In the message window, double-click the error message.

   The code worksheet opens and the network concerned is marked.
2. Move the network to a valid vertical position at the left worksheet border.
