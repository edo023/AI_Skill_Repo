# New Project

Projects cannot be created in Machine Expert – Safety. Safety-related projects are automatically opened/created when launching Machine Expert – Safety from Machine Expert.

Machine Expert – Safety is coupled with Machine Expert. After adding a Safety Logic Controller in the 'Devices' tree in Machine Expert, the command 'Machine Expert - Safety > Edit Project' is available in the 'Project' menu and in the context menu of the SLC tree node of Machine Expert.

When starting Machine Expert – Safety this way, the related project is automatically loaded into Machine Expert – Safety. If no safety-related project yet exists, a new project is automatically created.

Due to this coupling, the safety-related devices as well as their process data items (device terminals) are automatically handed over from Machine Expert to Machine Expert – Safety.

In Machine Expert – Safety, the devices are available in the tree view of the 'Devices' window (Bus Navigator). From here, process data items can be [inserted into the code](SE_AssignProcessDataItems.html#SE_AssignProcessDataItems) via drag & drop, thus creating the related global I/O variable.
