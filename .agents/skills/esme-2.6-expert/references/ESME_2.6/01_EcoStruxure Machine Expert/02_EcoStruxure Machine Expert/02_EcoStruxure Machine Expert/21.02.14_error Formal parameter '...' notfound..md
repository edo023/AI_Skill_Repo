# Error: Formal parameter '...' not found.

The error message specifies a user FB that has been modified by removing a formal parameter: The VAR\_INPUT or VAR\_OUTPUT declaration of the input or output variable has been deleted in the variables worksheet of this user FB POU.

In the project code, an instance of this user FB, however, has not yet been updated to match the new definition. The deleted input or output formal parameter is still used in this FB call although it is no longer available in the FB definition.

Update the affected instance by replacing the user FB by itself.

Proceed as follows:

1. In the FBD/LD code, right-click the FB icon to be replaced and select 'Object Properties...' from the context menu.
2. In the 'Function/Function Block' dialog, activate the 'Replace FB/FU' checkbox.

   The 'Group' and 'Name' combo boxes are now available.
3. Select the same FB type using the 'Group' and the 'Name' combo box.

   Leave the instance name unmodified.
4. Confirm the dialog.

   The old FB instance is now replaced according to the new FB definition, i.e., the respective formal parameter is no longer visible at the updated FB icon.
5. Complete the modified safety logic, for example, by reconnecting the objects that became unconnected due to the deleted formal parameter.
