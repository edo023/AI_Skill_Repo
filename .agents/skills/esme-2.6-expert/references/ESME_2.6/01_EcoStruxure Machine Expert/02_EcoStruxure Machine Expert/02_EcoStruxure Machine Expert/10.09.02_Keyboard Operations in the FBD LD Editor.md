# Keyboard Operations in the FBD/LD Editor

The following table shows the defined shortcuts and the resulting cursor operations:

| Key combination | Operation |
| --- | --- |
| only <cursor key> | The adjacent object in the respective direction is marked. By pressing only the cursor keys, all objects types are considered, i.e., lines, variables, formal parameters and function/FB symbols. |
| <cursor key> + <Ctrl> | The adjacent function/FB symbol is marked. Lines, variables and formal parameters are ignored. |
| <cursor key> + <Shift>  and  <cursor key> + <Shift> + <Ctrl> | The mouse pointer moves to the desired direction without marking an object.  Pressing <Ctrl> accelerates the movement of the pointer.  <SPACE> sets the cursor at the current position. If an object is under the mouse pointer, the object is marked. |

**NOTE:**

Using keyboard shortcuts, it is not possible to mark several objects. [Use the mouse](handlingobjectsinthegraphiceditor.html#handlingobjectsinthegraphiceditor) for this purpose.
