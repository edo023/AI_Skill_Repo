# Error: Debug watchdog (10 min) elapsed! The debug mode is active but the client is logged out. (SafePLC time: '...')

While the Safety Logic Controller runs in debug mode, a debug watchdog timer is active. If you log out without switching back to safe mode, the debug watchdog will elapse after 10 minutes.

The purpose of this watchdog is to ensure that no variables remain forced or overwritten if the communication between Safety Logic Controller and Machine Expert – Safety (designated as client in the error message) is interrupted.

If the debug watchdog timer elapses, the Safety Logic Controller sets the state to STOP [Debug] and writes an error into the error stack. In this case, you cannot switch to safe mode again.

Reset the Safety Logic Controller or download the application program to the Safety Logic Controller.
