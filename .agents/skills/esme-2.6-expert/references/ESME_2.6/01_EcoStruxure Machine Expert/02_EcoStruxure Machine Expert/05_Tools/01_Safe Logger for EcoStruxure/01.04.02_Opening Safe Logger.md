# Opening Safe Logger

## Opening Safe Logger

There are various ways to open Safe Logger in your EcoStruxure Machine Expert project:

Procedure 1:

| Step | Action |
| --- | --- |
| 1 | Double-click the logic controller in the Devices tree. This opens the device editor. |
| 2 | Click the Safe Logger tab. |

Procedure 2:

| Step | Action |
| --- | --- |
| 1 | Right-click the logic controller in the Devices tree. |
| 2 | Select Show Safe Logger from the contextual menu. This opens the Safe Logger tab. |

Procedure 3:

| Step | Action |
| --- | --- |
| 1 | Select Tools > External Tools > Open Diagnostics. |
| 2 | In Diagnostics, select Collect Data. |
| 3 | Select the required controller and click the Connect button. |
| 4 | Select Safe Logger. |
