# DiscrepancyTime input

Specification of the permissible discrepancy time for monitoring the switching operation at the S\_ChannelNC and S\_ChannelNO inputs.

Data type: TIME

Initial value: #0ms

**Connection**: Connect this input to a time literal, i.e., to a constant of the data type TIME.

**Background:** As a result of mechanical influences, signals may have a tolerance when switching contacts. By specifying the DiscrepancyTime value, the time period is defined within which the switching operations at the inputs S\_ChannelNC and S\_ChannelNO must occur to be recognized as simultaneous. This means both inputs can switch to SAFETRUE or SAFEFALSE within the set discrepancy time.

**NOTE:**

The switchover from SAFEFALSE to SAFETRUE is monitored for both inputs, as well as the switchover from SAFETRUE to SAFEFALSE.
