# Ready output

Output for signaling "Function block activated/not activated".

Data type: BOOL

**Connection:** If required, the signal can be connected to a standard variable of the BOOL data type or to other standard functions/function blocks.

| Value | Description |
| --- | --- |
| TRUE | Function block is activated (Activate = TRUE) and the output parameters represent the present state of the safety-related function. |
| FALSE | Function block is not activated (Activate = FALSE) and all outputs of the function block are switched to FALSE/SAFEFALSE. |
