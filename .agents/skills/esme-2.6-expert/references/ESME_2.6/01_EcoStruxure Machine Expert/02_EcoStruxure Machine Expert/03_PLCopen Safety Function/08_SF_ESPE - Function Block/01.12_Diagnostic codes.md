# Diagnostic codes

**NOTE:**

This documentation refers to the monitored electro-sensitive protective equipment as ESPE for short.

The hexadecimal diagnostic codes listed here are output as WORD data type at the DiagCode output of the function block.

| Code (hex) | Description | Options for remedy |
| --- | --- | --- |
| 0000 | The function block is not activated. The Activate input is FALSE.  Depending on the connection of Activate, the inactive function block can also indicate an error in the connected periphery. | Activate the function block by switching Activate to TRUE.  Correct the error in the periphery according to the device description. |
| 8000 | The ESPE has not triggered (S\_ESPE\_In = SAFETRUE). | No measures required. |
| 8001 | The function block is activated. This state of the function block is completed after one cycle of the Safety Logic Controller. | No measures required. |
| 8002 | The start-up inhibit is specified with S\_StartReset = SAFEFALSE.  * ESPE has triggered (S\_ESPE\_In = SAFEFALSE) * **or** the wiring is damaged or disconnected * **or** the safety-related input device, to which the ESPE is connected, is switched off or inoperable. | * Remove the ESPE safety-related request. * In case of ESPE that has not triggered, verify the wiring and the safety-related input device to which the ESPE is connected. |
| 8003 | The ESPE is no longer triggered (S\_ESPE\_In = SAFETRUE).  The start-up inhibit of the function block is active. | Perform a reset on the function block. |
| 8004 | ESPE has triggered (S\_ESPE\_In = SAFEFALSE).  * ESPE has triggered (S\_ESPE\_In = SAFEFALSE) * **or** the wiring is damaged or disconnected * **or** the safety-related input device, to which the ESPE is connected, is switched off or inoperable. | * Remove the request for the safety-related function of the ESPE and then perform a reset if S\_AutoReset = SAFEFALSE. * In case of ESPE that has not triggered, verify the wiring and the safety-related input device to which the ESPE is connected. |
| 8005 | The ESPE is no longer triggered. The restart inhibit of the function block is active (S\_AutoReset = SAFEFALSE). | Perform a reset on the function block. |
| C001 | Once the request for the safety-related function of the ESPE has been removed, a permanent TRUE is detected at Reset. | * Switch Reset = FALSE. * Verify the reset control device and its wiring. |
| C002 | Once the request for the safety-related function of the ESPE has been removed, a permanent TRUE is detected at Reset. | * Switch Reset = FALSE. * Verify the reset control device and its wiring. |
