# MaxMutingTime input

Specification of the maximum permissible time for the entire muting operation. If the muting operation is not completed within this time, the S\_AOPD\_Out output is switched to the defined safe state SAFEFALSE and the function block signals an error (Error switches to TRUE).

Data type: TIME

Initial value: #0ms

**Connection**: Connect this input to a time literal, i.e., to a constant of the data type TIME.

The following is valid for a material flow direction from left to right:

* This timer starts when the first muting sensor located **before** the safety equipment (at input MutingSwitch11) provides a TRUE signal (i.e., there is an object inside the detection area of the sensor).
* The muting operation is complete when the first muting sensor located **behind** the safety equipment (at the MutingSwitch21 input) provides FALSE again.

The following is valid for a material flow direction from right to left:

* This timer starts when the first muting sensor located **before** the safety equipment (at input MutingSwitch22) provides a TRUE signal (i.e., there is an object inside the detection area of the sensor).
* The muting operation is complete when the first muting sensor located **behind** the safety equipment (at the MutingSwitch12 input) provides FALSE again.

**Further Information:**

Also refer to [the example muting operation procedure shown in the functional description](function_MutingSeq.html#function_MutingSeq__function_mutingPar_Process) for this function block.
