# MonitoringTime input

Specification of the maximum permissible response time between the request and confirmation of a safety-related function.

Data type: TIME

Initial value: #0ms

**Connection**: Connect this input to a time literal, i.e., to a constant of the data type TIME.

Once the request of a safety-related function is signaled by S\_OpMode = SAFEFALSE (the S\_SafetyRequest output forwards this to the connected safety-related periphery), the feedback signal at the S\_Acknowledge input must become SAFETRUE within the response time specified here. Otherwise, the function block will output an error message (output Error = TRUE) and the S\_SafetyActive output will be switched to the defined safe state (SAFEFALSE).

The time value to be configured depends on the safety response time of the functional safety system. The safety response time is the time between arrival of the signal at the input channel and output of the switch-off signal at the device output.

In order to calculate the safety response time of your functional safety system, select the menu item 'Project > Response time calculator' in EcoStruxure Machine Expert - Safety. More information can be found in the chapter "Safety response time" of the EcoStruxure Machine Expert - Safety Online Help.
