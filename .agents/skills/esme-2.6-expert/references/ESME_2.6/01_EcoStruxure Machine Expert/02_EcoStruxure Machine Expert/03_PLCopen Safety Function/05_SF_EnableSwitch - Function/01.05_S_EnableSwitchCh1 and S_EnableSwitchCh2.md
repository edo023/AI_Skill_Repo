# S\_EnableSwitchCh1 and S\_EnableSwitchCh2 inputs

S\_EnableSwitchCh1 is the state-controlled input for the signal resulting from contacts E1 and E2 of the connected manually actuated, three-stage enable switch.

S\_EnableSwitchCh2 is the input for the signal resulting from contacts E3 and E4 of the connected enable switch.

Data type: SAFEBOOL

Initial value: SAFEFALSE

**Connection:** Connect these inputs to an enable switch connected to the Safety Logic Controller, either directly or via other safety-related functions/function blocks. Constant SAFETRUE or SAFEFALSE values are not permitted.

If the function block is active (input Activate = TRUE), it evaluates the following:

* The signal level of contacts E1 and E2 at input S\_EnableSwitchCh1 and
* The signal level of contacts E3 and E4 at input S\_EnableSwitchCh2

The function block performs these evaluations for switching stages 1, 2, and 3, as shown in the diagram below:

![](images/PROG_Requirements_EnableSwitch.png)

**NOTE:**

The error state of the function block can only be exited if the cause of the error no longer exists. To leave the error state, release the enable switch to move it to switch position 1. If a restart inhibit has been set with S\_AutoReset = SAFEFALSE, it must then be removed by pressing the reset button.
