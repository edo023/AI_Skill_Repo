# DiscTime21\_22 input

Specification of the permissible discrepancy time for the switching operation of the muting sensors at the MutingSwitch21 and MutingSwitch22 inputs.

Data type: TIME

Initial value: #0ms

**Connection**: Connect this input to a time literal, i.e., to a constant of the data type TIME.

Once the first input has switched to TRUE, the second input must follow within the time set at DiscTime21\_22. If both inputs switch to TRUE within this time, active muting remains active (S\_MutingActive = SAFETRUE).

**NOTE:**

If muting is active (S\_MutingActive = SAFETRUE), the light beam of the safety equipment may be interrupted. The S\_AOPD\_Out output remains SAFETRUE.

The time set at DiscTime21\_22 is exceeded if the second muting sensor input does not switch to TRUE within the set duration. The S\_AOPD\_Out enable output is then switched to the defined safe state SAFEFALSE and the function block signals an error (Error = TRUE).

**Further Information:**

Also refer to [the example muting operation procedure shown in the functional description](function_mutingPar.html#function_mutingPar__function_mutingPar_Process) for this function block.

**Maximum value:** 4 s
