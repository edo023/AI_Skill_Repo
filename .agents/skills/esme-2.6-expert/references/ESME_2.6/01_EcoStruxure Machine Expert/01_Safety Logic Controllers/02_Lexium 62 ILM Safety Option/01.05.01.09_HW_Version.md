# HW\_Version

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module, DIO8 module,  Lexium LXM62 Power Supply,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  Logic Motion Controller Device |
| Traceable | No |

## Functional Description

Displays the hardware version of the device.

If the device in the  Devices tree is virtual (non-real), then the parameter value is DRIVE\_VIRTUAL.

EIO0000002293.01
