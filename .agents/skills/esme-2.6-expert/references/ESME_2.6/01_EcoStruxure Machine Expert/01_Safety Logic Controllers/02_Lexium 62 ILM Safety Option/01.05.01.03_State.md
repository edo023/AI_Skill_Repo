# State

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Displays the status of the Safety Logic.

| Value | Meaning |
| --- | --- |
| disabled / 0 | Safety Logic is deactivated. |
| error / 1 | An error was detected during conversion. |
| not ready / 2 | Safety Logic has been activated; Sercos phase start-up (Phase 1…3). |
| virtual / 3 | Safety Logic has been activated; Sercos bus in phase 4, but the associated motor is virtual. The Safety Logic is not ready for operation. |
| active / 4 | Safety Logic has been activated; Sercos bus in phase 4. |
