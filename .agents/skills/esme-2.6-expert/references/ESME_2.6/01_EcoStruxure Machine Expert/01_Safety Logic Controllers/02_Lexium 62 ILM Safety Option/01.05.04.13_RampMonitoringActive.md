# RampMonitoringActive

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status of ramp monitoring.

| Value | Meaning |
| --- | --- |
| on / 1 | Ramp monitoring is active. |
| off / 0 | Ramp monitoring is inactive. |
