# SLS\_4\_Speed

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the Safely Limited Speed (SLS) 4 (velocity to be monitored after t1+t2).

| Value | Data type | Meaning |
| --- | --- | --- |
| 0...65535 | LREAL | Safely Limited Speed (SLS) 4 (units/s) |

For more information, refer to [*Safely Limited Speed (SLS)*](D-SE-0062422.html#D-SE-0062422).
