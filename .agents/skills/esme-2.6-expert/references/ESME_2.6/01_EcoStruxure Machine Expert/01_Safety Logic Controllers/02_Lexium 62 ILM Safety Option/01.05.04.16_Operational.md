# Operational

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status of the state machine in the openSAFETY drive (SafeMC).

| Value | Meaning |
| --- | --- |
| on / 1 | State machine is in operational state. |
| off / 0 | State machine is not in operational state. |
