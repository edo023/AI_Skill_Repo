# EncoderDirection

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Displays the encoder direction.

| Value | Meaning |
| --- | --- |
| inverted / 0 | The encoder direction is inverted. |
| not inverted / 1 | The encoder direction is not inverted. |

EIO0000002293.01
