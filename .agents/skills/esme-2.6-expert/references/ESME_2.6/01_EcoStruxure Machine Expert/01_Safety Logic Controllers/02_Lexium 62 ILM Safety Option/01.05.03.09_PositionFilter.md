# PositionFilter

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

This parameter defines the filter time for the position value which is calculated from the encoder inputs. The filter time affects the velocity as well because the calculation of the velocity depends on the encoder position. The filtered position is used for the calculation of the velocity and as input for the safety-related functions SOS, SS2, and SDI. The velocity is used for the monitoring of the deceleration ramp in the safety-related functions and for monitoring the speed limits of SLS and SMS.
