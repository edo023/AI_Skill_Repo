# ParameterValid

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Indicates that the parameters listed below are read from device.

NOTE: The configuration of the parameters is done by using EcoStruxure Machine Expert - Safety.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | Parameter is invalid. |
| yes / 1 | BOOL | Parameter is valid. |
