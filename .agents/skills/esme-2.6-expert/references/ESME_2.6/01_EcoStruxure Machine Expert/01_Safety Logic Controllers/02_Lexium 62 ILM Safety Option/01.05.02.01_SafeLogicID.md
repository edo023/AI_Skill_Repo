# SafeLogicID

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Displays the unique number of the corresponding Safety Logic Controller.

| Value | Data type | Meaning |
| --- | --- | --- |
| 1…1000 (default = 1) | UINT | Unique number of a Safety Logic Controller. |
