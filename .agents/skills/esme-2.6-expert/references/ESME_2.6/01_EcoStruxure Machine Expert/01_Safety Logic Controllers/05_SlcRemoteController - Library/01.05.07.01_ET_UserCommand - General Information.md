# ET\_UserCommand - General Information

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

## Description

This enumeration lists the possible commands the function block FB\_RemoteController can execute.

## Enumeration Elements

| Name | Value | Description | RemoteControlAllowed required(1) |
| --- | --- | --- | --- |
| ReadStatus | 0 | Request status and, if supported, the extended status of the SLC. | No |
| ConfirmMoreThan4ModuleExchange | 1 | Confirm replacement of more than four modules. | Yes |
| Confirm4ModuleExchange | 2 | Confirm replacement of four modules. | Yes |
| Confirm3ModuleExchange | 3 | Confirm replacement of three modules. | Yes |
| Confirm2ModuleExchange | 4 | Confirm replacement of two modules. | Yes |
| Confirm1ModuleExchange | 5 | Confirm replacement of one module. | Yes |
| ConfirmFirmwareExchange | 6 | Confirm new firmware version (FW-ACKN). | Yes |
| ConfirmSafeKeyExchange | 7 | Confirm memory key replacement (SK-XCHN). | Yes |
| StartLEDTest | 8 | Start LED Test (TEST). | Yes |
| StartSystemScan | 9 | Start system scan (SCAN). | Yes |
| CopySafeKeyStart | 10 | Start memory key copy (SK-COPY). | Yes |
| CopySafeKeyContinue | 11 | Continue memory key copy after replacing memory key. | Yes |
| FormatSafeKey | 12 | Format memory key. | No |
| RestartDevice | 13 | Restart device. | No |
| SetSafeKeyPassword | 14 | Modify the memory key password. | No: Set or modify the password on an empty memory key  Yes: Modify the password after download of project |
| ActivateSetupMode | 16 | Activate the setup mode for TM5CSLC300FS and TM5CSLC400FS. | Yes |
| DeActivateSetupMode | 17 | Deactivate the setup mode for TM5CSLC300FS and TM5CSLC400FS. | Yes |

(1) The commands marked “No” can be executed remotely even if the SLC parameter RemoteControlAllowed is set to “NO”. Commands marked “Yes” can only be executed remotely if the SLC parameter RemoteControlAllowed is set to “Yes-ATTENTION”.
