# ET\_State - General Information

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

## Description

This enumeration lists the possible states of the last "Enter" command.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Idle | 0 | Idle.  Waiting for next command. |
| EnterCommandReceived | 1 | Entered command received. |
| EnterCommandExecuting | 2 | Executing entered command executing. |
| InitializingEnterCommand | 3 | Initializing entered command. |
| WaitingForRestart | 4 | Entered command completed. SLC will shortly restart automatically. |
