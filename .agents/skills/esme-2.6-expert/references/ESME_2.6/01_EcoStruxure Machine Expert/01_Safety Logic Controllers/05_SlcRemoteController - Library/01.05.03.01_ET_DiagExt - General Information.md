# ET\_DiagExt - General Information

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_DiagExt specifies the POU-specific diagnostic and status messages.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Ok | 0 | Ok |
| Disabled | 1 | The POU is disabled. |
| Initializing | 2 | The POU is being initialized. |
| WaitForExecution | 3 | The POU is ready and waiting for execution or for a user input. |
| Running | 4 | The POU is busy processing a request or internal procedures. |
| UnknownCommandData | 10 | The provided command data is incorrect. |
| UnknownCommand | 11 | The provided command is incorrect. |
| UnknownDiagExt | 12 | The provided DiagExt is incorrect. |
| UnknownSafeOSState | 13 | The provided SafeOSState is indeterminable. |
| UnknownState | 14 | The provided state is incorrect. |
| UnknownStatus | 15 | The provided status is incorrect. |
| UnknownUserCommand | 16 | The provided user command is incorrect. |
| InvalidIfSafeLogic | 20 | The ifSafeLogic interface is invalid. |
| InvalidUserCommand | 21 | The requested user command is invalid. |
| UnknownInternalState | 22 | The POU detected an indeterminable internal state. |
| GlobalSercosInterfaceInvalid | 23 | The global Sercos interface is invalid. |
| InvalidFilePath | 24 | The path specified by the input variable i\_sFilepath is invalid. |
| InvalidPassword | 25 | The password specified by the input variable i\_sPassword is invalid. |
| InvalidFile | 26 | The file with the path specified at the input i\_sFilepath is invalid. |
| CommunicationFailure | 30 | An error was detected while attempting to communicate with the SLC. |
| CommandFailedToExecute | 31 | A command was successfully sent to the SLC but terminated unsuccessfully. |
| WaitForCRCConfirm | 32 | Waiting for confirmation of the CRC which has been returned by q\_udiCRC via the input parameter i\_xConfirmCRC. |
