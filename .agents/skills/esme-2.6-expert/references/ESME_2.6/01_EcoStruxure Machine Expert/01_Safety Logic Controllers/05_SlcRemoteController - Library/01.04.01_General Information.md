# General Information

## Description

The library SlcRemoteController provides the functionality to read the status of **Safety Logic Controller (SLC)** devices remotely and executes maintenance operations on these **SLC** devices via the Sercos interface. Additionally, the library provides the function block FB\_DownloadApplication for downloading the application in EcoStruxure Machine Expert by using the Sercos interface.

Maintenance operations include:

* Confirmation of replaced memory key, safety-related modules, and firmware.
* Enforcement of an LED test, a system scan, or a restart of the device.

These operations can also be performed by setting the rotary switch on the SLC to the corresponding position and pressing the confirmation button Enter longer than 0.5 s. This library allows you to execute the function while the program is running on the SLC and therefore be triggered via HMI terminals.

The manual and remote executions of commands are interlocked. This means while a command on the hardware is being executed triggered by the Enter button, no other command can be requested using the library and the other way around. Only one command can be processed by the SLC at a time.

However, if copying of a memory key is requested via a remote command (CopySafeKeyStart), a continuation (CopySafeKeyContinue) can be triggered via hardware access after the memory key has been replaced.

While a command is being executed, further command requests are denied. To confirm replaced components, follow the specified confirmation sequence: Confirm the memory key replacement first, then the replaced safety-related modules, and finally the firmware update.

A command automatically modifies the status at the output q\_stStatus and, if supported, the extended status at the output q\_stStatusExtended of the function block FB\_RemoteController. If the processing status or result is needed, the command "ReadStatus" has to be sent and the status determined afterwards. The returned value of the command "ReadStatus" is only valid if the value of the SLC parameter BootState is greater than 2.

Every command request except "ReadStatus" is logged in the SafeLog, independent of the result (executed or denied because of the authorization denial).

The remote control functionality has to be activated in the Machine Expert - Safety application by setting the SLC parameter RemoteControlAllowed to "Yes-ATTENTION".

NOTE: To download a safety-related application using the function block FB\_DownloadApplication, it is not necessary to set the SLC parameter RemoteControlAllowed to "Yes-ATTENTION" in Machine Expert - Safety.

Certain remote commands can be triggered even if the parameter RemoteControlAllowed of the SLC is set to NO. For details, refer to [ET\_UserCommand](D-SE-0075078.html).
