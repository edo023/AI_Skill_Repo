# IO Configuration

## General

The parameters of the I/O configuration determine which data of the I/O image are exchanged between Safety Logic Controller (SLC) and PacDrive LMC (Logic Motion Controller (LMC)).

The following table shows the connection between values of the I/O configuration and the I/O images:

| I/O configuration | Corresponding I/O image |
| --- | --- |
| [SLC2LMC\_NumberOfBOOLs](D-SE-0065088.html#D-SE-0065088) | [SLC2LMC\_BOOL](D-SE-0065596.html#D-SE-0065596) |
| [SLC2LMC\_NumberOfBOOLsExt](D-SE-0065089.html#D-SE-0065089) | [SLC2LMC\_BOOLExt](D-SE-0065600.html#D-SE-0065600) |
| [SLC2LMC\_NumberOfINTs](D-SE-0065550.html#D-SE-0065550) | [SLC2LMC\_INT](D-SE-0065604.html#D-SE-0065604) |
| [SLC2LMC\_NumberOfUINTs](D-SE-0065551.html#D-SE-0065551) | [SLC2LMC\_UINT](D-SE-0065606.html#D-SE-0065606) |
| [SLC2LMC\_NumberOfUDINTs](D-SE-0065552.html#D-SE-0065552) | [SLC2LMC\_UDINT](D-SE-0065607.html#D-SE-0065607) |
| [LMC2SLC\_NumberOfBOOLs](D-SE-0065553.html#D-SE-0065553) | [LMC2SLC\_BOOL](D-SE-0065608.html#D-SE-0065608) |
| [LMC2SLC\_NumberOfBOOLsExt](D-SE-0065556.html#D-SE-0065556) | [LMC2SLC\_BOOLExt](D-SE-0065609.html#D-SE-0065609) |
| [LMC2SLC\_NumberOfINTs](D-SE-0065557.html#D-SE-0065557) | [LMC2SLC\_INT](D-SE-0065610.html#D-SE-0065610) |
| [LMC2SLC\_NumberOfUINTs](D-SE-0065558.html#D-SE-0065558) | [LMC2SLC\_UINT](D-SE-0065611.html#D-SE-0065611) |
| [LMC2SLC\_NumberOfUDINTs](D-SE-0065559.html#D-SE-0065559) | [LMC2SLC\_UDINT](D-SE-0065612.html#D-SE-0065612) |

NOTE: The data configured in EcoStruxure Machine Expert - Safety must be used; otherwise Sercos cannot run-up correctly. For values of the data type BOOL or ExtBOOL, it is sufficient to use one boolean value per byte.

NOTE: The IO data is only configured once during project download. It can not be modified at runtime.
