# ConfiguredTopologyAddress

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos Drive,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  Logic Motion Controller Device |
| Traceable | Yes |

## Functional Description

Configures the topological address where the device is expected inside the Sercos network. This parameter is only taken into account if the identification mode TopologyAddress / 2 has been selected.
