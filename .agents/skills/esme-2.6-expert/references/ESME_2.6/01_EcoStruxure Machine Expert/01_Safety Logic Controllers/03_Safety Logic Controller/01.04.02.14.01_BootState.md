# BootState

## General

|  |  |
| --- | --- |
| Type | AS |
| Devices supporting the parameter | TM5CSLCx00FS |
| Traceable | No |

## Functional Description

Displays the boot state of the TM5CSLCx00FS.

| Value | Meaning |
| --- | --- |
| * Startup not initiated / 0 * Initializing SafeOS / 1 * Cyclic HW test running / 2 * OpenSAFETY-Stack running / 3 * SafeOS running / 4 | Boot state of the TM5CSLCx00FS |
