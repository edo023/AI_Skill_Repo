# WorkingMode

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos Drive,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  Logic Motion Controller Device |
| Traceable | Yes |

## Functional Description

With this command, you can set the working mode of the device.

| Value | Data type | Meaning |
| --- | --- | --- |
| virtual / 0 | DINT | The device is operated virtually. |
| real / 1 | DINT | The device is being searched for on in the Sercos network:   * Device found: -> device is used. * Device not found: -> diagnostic message; Sercos phase up not possible. |
| deacativated / 2 | DINT | Deactivated drives can remain in the controller configuration like virtual drives, although they do not physically exist. The difference from the virtual drives is that these deactivated drives are not simulated in cyclical real-time processing. This can reduce processing time, allowing you to enter a larger total of unused drives in the controller configuration. |
| optional / 3 | DINT | The device is being searched for on in the Sercos network:   * Device found: -> device is used. * Device not found: -> device is operated in "virtual / 0" working mode. |
