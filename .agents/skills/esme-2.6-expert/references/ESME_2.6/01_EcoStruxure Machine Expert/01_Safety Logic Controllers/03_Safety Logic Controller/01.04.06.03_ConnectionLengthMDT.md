# ConnectionLengthMDT

## General

|  |  |
| --- | --- |
| Type | – |
| Devices supporting the parameter | Safety Logic Controller TM5CSLCx00FS |
| Traceable | – |

## Functional Description

Displays the Sercos III connection length (MDT) that is used by this device in the MDT (access: AD).

EIO0000003541.02
