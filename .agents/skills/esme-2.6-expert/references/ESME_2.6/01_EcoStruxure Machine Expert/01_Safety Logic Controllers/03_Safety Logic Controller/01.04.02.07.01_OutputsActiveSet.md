# OutputsActiveSet

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Safety Logic Controller TM5CSLCx00FS,  Bus Interface TM5NS31,  Sercos I/O |
| Traceable | – |

## Functional Description

Enables the outputs of the I/O modules.

The parameter additionally starts the cyclic safety communication at the Safety Logic Controller and the bus interface.

NOTE: This parameter has to set again after each Sercos run-up (comparable to parameter ControllerEnableSet for drives).

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The outputs of the I/O modules are disabled. |
| on / 1 | BOOL | The outputs of the I/O modules are enabled. |
