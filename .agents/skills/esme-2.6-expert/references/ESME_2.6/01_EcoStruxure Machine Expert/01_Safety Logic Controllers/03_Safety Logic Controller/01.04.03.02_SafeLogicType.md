## General

|  |  |

| Devices supporting the parameter | TM5CSLCx00FS |

## Functional Description

Selects a type for the Safety Logic Controller.

The configuration has to take place before the start-up of EcoStruxure Machine Expert - Safety, as this also modifies the type of the Safety Logic Controller in the safety-related configuration.
