# SafeLogicId

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | TM5CSLCx00FS |
| Traceable | Yes |

## Functional Description

Used to set the SafeLogicId of the Safety Logic Controller.

The Safety Logic Controller is a unique number which must be the same for the Safety Logic Controller and the corresponding SafeIO.
