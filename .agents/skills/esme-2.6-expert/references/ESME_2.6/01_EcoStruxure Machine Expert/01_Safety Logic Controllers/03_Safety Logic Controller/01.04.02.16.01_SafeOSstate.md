# SafeOSstate

## General

|  |  |
| --- | --- |
| Type | AS |
| Devices supporting the parameter | TM5CSLCx00FS |
| Traceable | No |

## Functional Description

Displays the state of the TM5CSLCx00FS.

| Value | Meaning |
| --- | --- |
| * Invalid / 0 * ON / 1 * Loading / 2 * SafeSTOP / 3 * SafeRUN / 4 * DebugHALT / 5 * DebugSTOP / 6 * DebugRUN / 7 * No execution / 8 | State of the TM5CSLCx00FS |

EIO0000003541.02
