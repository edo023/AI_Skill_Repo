# LMC2SLC\_NumberOfUINTs

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Safety Logic Controller TM5CSLCx00FS |

## Functional Description

Number of UINT values that are transmitted from the PacDrive LMC (Logic Motion Controller (LMC)) to the Safety Logic Controller (SLC).

EIO0000003541.02
