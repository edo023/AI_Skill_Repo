# LMC2SLC\_NumberOfUINTs

## General

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can configure the number of UINT values that are transmitted from the logic/motion controller (LMC) to the Safety Logic Controller (SLC).
