# LMC2SLC\_INT

## General

|  |  |
| --- | --- |
| Type | INT |
| Channels | LMC2SLC\_INT0  LMC2SLC\_INT1  LMC2SLC\_INT2  LMC2SLC\_INT3  ..........................................  LMC2SLC\_INT29 |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

In order to use one or more variables you must first enable them. You can enable the variables by specifying the number of variables in the corresponding parameter LMC2SLC\_NumberOfINTs.

By default, one variable is assigned to each INT channel.
