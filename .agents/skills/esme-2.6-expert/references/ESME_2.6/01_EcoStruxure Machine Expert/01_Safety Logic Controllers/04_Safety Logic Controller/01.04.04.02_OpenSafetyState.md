| --- | --- |

## Functional Description

With this parameter, you can read the state of the OpenSAFETY communication of the device.

* Value Pre OP:

  The OpenSAFETY communication is in pre-operational state.
* Value Operational:

  The OpenSAFETY communication is in operational state.

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if the device is simulated or if Sercos is not in phase CP2 or higher, the default value (Pre OP) is indicated here.

Use the function block SLCD.FB\_ReadOpenSafetyState to read the parameter.
