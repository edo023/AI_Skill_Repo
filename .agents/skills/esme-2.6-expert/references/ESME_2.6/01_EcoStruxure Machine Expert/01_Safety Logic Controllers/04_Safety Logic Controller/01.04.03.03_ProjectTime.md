## Functional Description

With this parameter, you can read the time stamp of the EcoStruxure Machine Expert - Safety project.

NOTE: In EcoStruxure Machine Expert the time stamp of the project is stored in UTC, however, in EcoStruxure Machine Expert - Safety the time stamp is adjusted according to the setting of the time zone of the local PC.

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if the device is simulated or if Sercos is not in phase CP2 or higher, the default time (DT#1970-01-01-0:0:0) is indicated here.

EIO0000003877.02
