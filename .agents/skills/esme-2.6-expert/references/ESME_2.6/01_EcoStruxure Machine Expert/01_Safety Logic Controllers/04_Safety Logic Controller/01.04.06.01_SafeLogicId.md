# SafeLogicId

## General

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can read the SafeLogicId of the Safety Logic Controller.

The value is a unique number which must be the same for the Safety Logic Controller and the corresponding safety-related devices.

EIO0000003877.02
