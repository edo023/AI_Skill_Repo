# OutputsActive

## General

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can read the status of the output data of the cyclic safety communication.

* Value FALSE:

  The output data is inactive.
* Value TRUE:

  The output data is active.
