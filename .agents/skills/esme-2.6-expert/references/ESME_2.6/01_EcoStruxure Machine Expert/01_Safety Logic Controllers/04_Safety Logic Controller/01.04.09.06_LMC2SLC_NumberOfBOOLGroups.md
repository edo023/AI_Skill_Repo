# LMC2SLC\_NumberOfBOOLGroups

## General

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can configure the number of BOOLGroup values (1 BOOLGroup correspond to 8 BOOL values) that are transmitted from the logic/motion controller (LMC) to the Safety Logic Controller (SLC).

EIO0000003877.02
