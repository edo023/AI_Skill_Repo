# InputsValid

## General

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can read the status of the input data of the cyclic safety communication.

* Value FALSE:

  The input data is invalid.
* Value TRUE:

  The input data is valid.
