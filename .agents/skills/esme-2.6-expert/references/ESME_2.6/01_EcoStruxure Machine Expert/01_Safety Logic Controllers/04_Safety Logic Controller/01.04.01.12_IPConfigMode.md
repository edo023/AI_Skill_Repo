# IPConfigMode

## General

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

With this parameter, you can configure the method of how the IP address, subnet mask, and gateway of the device is set.

* Value Automatic IP address assignment / 0 (default):

  The IP address, subnet mask, and gateway of the device is set automatically.
* Value Manual IP address assignment (hardware) / 1:

  The IP address, subnet mask, and gateway of the device can be set via the device assistant tool.
