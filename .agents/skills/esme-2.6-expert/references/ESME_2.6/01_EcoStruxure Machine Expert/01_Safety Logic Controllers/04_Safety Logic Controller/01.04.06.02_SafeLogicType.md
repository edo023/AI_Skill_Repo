# SafeLogicType

## General

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can configure the type for the Safety Logic Controller.

Possible values:

* Value SLC100 / 0
* Value SLC200 / 1
* Value SLC300 / 2
* Value SLC400 / 3 (default)

The configuration has to take place before the first start-up of EcoStruxure Machine Expert - Safety, as this also modifies the type of the Safety Logic Controller in the safety-related configuration.
