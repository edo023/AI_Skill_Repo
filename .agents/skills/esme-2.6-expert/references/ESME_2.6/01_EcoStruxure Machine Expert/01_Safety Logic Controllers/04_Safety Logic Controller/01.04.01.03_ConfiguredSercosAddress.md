# ConfiguredSercosAddress

## General

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

With this parameter, you can configure the Sercos address of the device.

The configured Sercos address is only taken into account if the parameter IdentificationMode is set to Sercos mode.

The hardware switches of the device for the Sercos address have to be set to the same value than the value in this parameter, for details see the hardware guide of the device.
