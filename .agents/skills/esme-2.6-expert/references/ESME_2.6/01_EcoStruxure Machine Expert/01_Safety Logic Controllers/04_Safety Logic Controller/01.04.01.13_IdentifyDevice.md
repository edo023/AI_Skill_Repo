# IdentifyDevice

|  |  |
| --- | --- |
| Offline editable | Yes |

## Functional Description

With this parameter, you can enable and disable the device identification in order to facilitate identifying Sercos devices in the Sercos network.

The precondition is an operational Sercos communication (Sercos phases 2, 3 and 4).

* Value FALSE (default):

  Device identification is disabled.
* Value TRUE:

  Device identification is enabled (orange flashing of the S3 LED of the device).

NOTE: The parameter value is transferred from the master to the slave via the Sercos service channel. Typically, this takes about 10 ms, but the transmission can take up to 1 second depending on the load on the Sercos service channel.
