# ApplicationType

## General

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | No |

## Functional Description

With this parameter, you can read the application type of the device.

This parameter can be modified by writing the Sercos parameter IDN S-0-1302.0.3. The function block FB\_WriteIDN can be used to modify this parameter.

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if the device is simulated or if Sercos is not in phase CP2 or higher, an empty string is indicated here.

EIO0000003877.02
