# LMC2SLC\_UINT

## General

|  |  |
| --- | --- |
| Type | UINT |
| Channels | LMC2SLC\_UINT0  LMC2SLC\_UINT1  LMC2SLC\_UINT2  LMC2SLC\_UINT3  ..........................................  LMC2SLC\_UINT29 |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

In order to use one or more variables you must first enable them. You can enable the variables by specifying the number of variables in the corresponding parameter LMC2SLC\_NumberOfUINTs.

By default, one variable is assigned to each UINT channel.
