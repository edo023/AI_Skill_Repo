# VendorCode

## General

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can read the vendor code of the device.

The vendor code is a unique number assigned to each vendor and helps identifying a Sercos device in the Sercos network.

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if the device is simulated or if Sercos is not in phase CP2 or higher, the value 0 is indicated here.

EIO0000003877.02
