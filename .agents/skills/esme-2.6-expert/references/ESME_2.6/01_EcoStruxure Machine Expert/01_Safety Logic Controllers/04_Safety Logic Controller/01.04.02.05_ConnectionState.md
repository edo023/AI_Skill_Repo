# ConnectionState

## General

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can read the connection state of the device.

* Value Operational:

  Sercos state of the master is CP4, slave is present on Sercos network, real-time communication is running.
* Value PhasingUp:

  Sercos state of the master is CP0, CP1, CP2, or CP3.
* Value NoCommunication:

  Sercos state of the master is CP4, slave is not present on Sercos network.
* Value NoData:

  Sercos state of the master is CP4, slave is present on Sercos network, real-time communication is not running.
* Value NotRunning:

  Sercos state of the master is Phase Error or NRT.
