# EncoderResolutionLinear

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Displays the encoder resolution for linear encoders. The shown value is only valid if parameter EncoderType = LinearEncoder.

| Value | Data type | Meaning |
| --- | --- | --- |
| 100…10000 (default = 1000) | UDINT | Encoder resolution.  Unit = period length in µm. |
