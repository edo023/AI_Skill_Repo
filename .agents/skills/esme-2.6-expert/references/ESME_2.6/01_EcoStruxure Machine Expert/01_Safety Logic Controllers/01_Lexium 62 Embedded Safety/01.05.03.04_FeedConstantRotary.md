## Functional Description

Is used to enter the feed constant. The feed constant defines in which units the position information is calculated. As units, for example, millimeters, degrees, and inches can be defined. The defined unit is also used for velocity (Velocity in unit/s) and acceleration (Acceleration in units/s2).

The feed constant is the path (in units) which was covered by one rotation of the drive shaft.

In the case of position encoders without the gear parameters GearIn and GearOut, the reference point is the encoder shaft, not the output shaft.

NOTE: Modifications to the parameter are only applied during the Sercos phase up (communication phase 0 => communication phase 4).

## Examples

* If a toothed belt covers, for example, 120 mm per output shaft revolution, then 120 is entered in the FeedConstant. One unit thus is one mm.
* If a disc covers, for example, 360 degrees per output shaft revolution, then 360 is entered in the FeedConstant. One unit is one degree.

The following example for an installation shows how the parameters FeedConstant, GearIn, and GearOut are to be defined.

![](images/T001150555.gif)

Example installation for the parameters FeedConstant, GearIn, GearOut

The ball screw spindle has a thread pitch of 0.25 inch per shaft revolution. The FeedConstant is therefore 0.25 and defines the units as inches.

The units for velocity and acceleration are thus inches/s and inches/s2.

The units for position, velocity, and acceleration are defined by the following parameters:

* FeedConstant = 0.25
* GearIn = 1
* GearOut = 2
