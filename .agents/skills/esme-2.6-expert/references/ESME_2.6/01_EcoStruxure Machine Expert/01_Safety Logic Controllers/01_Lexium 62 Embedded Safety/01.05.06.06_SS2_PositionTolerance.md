# SS2\_PositionTolerance

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the position tolerance for downtime monitoring.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0...65535 | LREAL | Position tolerance (in units) for downtime monitoring. |

For more information, refer to [*Safe Stop 2 (SS2)*](D-SE-0062423.html#D-SE-0062423).
