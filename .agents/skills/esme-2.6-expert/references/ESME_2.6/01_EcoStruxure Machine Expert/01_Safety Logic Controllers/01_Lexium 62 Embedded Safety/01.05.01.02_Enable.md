# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Enables or disables the processing of the Safety Logic.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The Safety Logic is not processed by the real-time process. |
| on / 1 | BOOL | The Safety Logic is activated and processed by the real-time process. |

NOTE: Modifications to the parameter are only applied during the Sercos phase up (communication phase 0 => communication phase 4).
