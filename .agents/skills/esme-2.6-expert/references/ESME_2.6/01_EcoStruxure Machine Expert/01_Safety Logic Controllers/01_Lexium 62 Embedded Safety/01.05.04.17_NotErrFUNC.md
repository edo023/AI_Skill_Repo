# NotErrFUNC

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status of the safety-related functions.

| Value | Meaning |
| --- | --- |
| true / 1 | No error detected. |
| false / 0 | A limit is exceeded or a general error is detected. |
