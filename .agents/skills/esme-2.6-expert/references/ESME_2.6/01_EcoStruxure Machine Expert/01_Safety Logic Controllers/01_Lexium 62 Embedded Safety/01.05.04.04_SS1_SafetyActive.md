# SS1\_SafetyActive

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status of the safety-related function SS1 (Safe Stop 1).

| Value | Meaning |
| --- | --- |
| on / 1 | Safety-related function is active. The motor is without torque. |
| off / 0 | Safety-related function is not active. |

For more information, refer to [*Safe Stop 1 (SS1)*](D-SE-0062421.html#D-SE-0062421).
