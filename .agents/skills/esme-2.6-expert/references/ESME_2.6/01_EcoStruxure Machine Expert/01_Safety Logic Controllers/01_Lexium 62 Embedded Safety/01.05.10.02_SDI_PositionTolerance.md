# SDI\_PositionTolerance

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the position tolerance for downtime monitoring.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0...65535 | LREAL | Position tolerance (in units) for downtime monitoring. |

For more information, refer to [*Safe Direction (SDI)*](D-SE-0062429.html#D-SE-0062429).
