# Sercos III Bus Interface Installation

## Introduction

The installation procedure of the Sercos III Bus Interface is to assemble the 4 components and then install it directly on the DIN rail.

NOTE: If the Sercos III Bus Interface is already installed and wired, or the connectors are pre-wired, be sure to remove all power before attempting these procedures.

## Sercos III Bus Interface Installation

The following procedure gives step by step instructions to assemble and install a Sercos III Bus Interface on a DIN rail:

| Step | Action | |
| --- | --- | --- |
| 1 | Remove bus base, the electronic modules and terminal block from protective packaging. Verify for visible mechanical damage. | |
| 2 | Insert interface electronic module in the [slot of the bus base](D-SE-0015378.html#D-SE-0015378__D-SE-0015378.5). |  |
| 3 | Push the interface electronic module straight into the bus base until it clicks. | |
| 4 | Insert the IPDM electronic module in the [guides of the bus base](D-SE-0015378.html#D-SE-0015378__D-SE-0015378.5). |  |
| 5 | Push the electronic module straight into the bus base until it clicks. | |
| 6 | Hang the bottom edge of the terminal block in its [hinge on the bus base](D-SE-0015378.html#D-SE-0015378__D-SE-0015378.5). |  |
| 7 | Rotate upward. |  |
| 8 | Press the terminal block until it clicks in place. |  |
| 9 | Push the locking lever all the way up on all of the bus base. This opens the locking mechanism. | |
| 10 | Hang the Sercos III Bus Interface in the desired position on the mounting rail. |  |
| 11 | Close the locking mechanism by pushing the lever down. | |
