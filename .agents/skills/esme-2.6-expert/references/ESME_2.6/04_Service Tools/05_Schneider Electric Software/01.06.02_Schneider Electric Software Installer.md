# Schneider Electric Software Installer Settings

## Overview

To edit the Schneider Electric Software Installer settings, click the tool icon in the upper right corner, and select Settings from the contextual menu.

## Settings

| Setting | Description |
| --- | --- |
| Select application language | Select a language for the Schneider Electric Software Installer from the list. |
| Select Temporary Download Folder | Select a folder to store temporary files during installation. |
