# Install Add-Ons and Patches

## Install Add-Ons and Patches

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. |
| 2 | Select the option Modify installed software. |
| 3 | In the Select a product area, select the following options:   * Product: Select your installed product. * Version: Select the version of your installed product. |
| 4 | In the Choose an action area, select Add-Ons/Patches. |
| 5 | Select Source: Online and click Next.  NOTE: You can also use one of the other [two options](D-SE-0088006.html#D-SE-0088006__D-SE-0088006.4), but if there is an Internet connection available, using the Source: Online downloads the software directly from the Schneider Electric server. |
| 6 | Click Next and browse for the Add-Ons or Patches. |
| 7 | From the list of Components, select an Add-On or a Patch to install. |
| 8 | Click the Start Installation/Uninstallation button. |
