# EcoStruxure Machine Expert Twin Licenses

## Overview

The following types of EcoStruxure Machine Expert Twin license are available:

* Builder License

  The Builder License allows you to do emulation of a machine if a controller is connected. Using predefined components as well as customized objects, you can create a complete machine flow. You can modify the properties of the components and write limited code to control the model.
* Developer License

  The Developer License allows you to do emulation, but also to create customized 3-D models and components you can use for building your machine flow by using C# or importing CAD files. Furthermore, you can use this license type to add customized windows to the software to create your own application.
* RemoteViewer License

  The RemoteViewer License allows you to connect to a running instance of EcoStruxure Machine Expert Twin and visualize the machine. The Web API supports different external clients, such as PCs, mobile devices, and tablets. The remote access to the 3-D digital twin model can be used for remote maintenance, remote assistance, and remote training.

NOTE: Software installation is independent of the activation of software product licenses.

## Categories of Licenses

To meet different customer application requirements, each license type is available in three categories:
