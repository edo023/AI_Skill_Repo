# Special License Cases

## Vijeo-Designer

Vijeo-Designer can be used in two modes:

* started from outside EcoStruxure Machine Expert for example via Start menu,
* started from EcoStruxure Machine Expert.

Vijeo-Designer can be used in both modes in parallel.

Depending on how the Vijeo-Designer is started, registration/licensing is handled in a different way.

* When started from outside EcoStruxure Machine Expert, you need a dedicated Vijeo-Designer license.
* Started from EcoStruxure Machine Expert licensing of Vijeo-Designer is covered by EcoStruxure Machine Expert licensing as described in the previous sections.

## Seats (Single, Team, Entity)

If you want to run EcoStruxure Machine Expert on several PCs, you have to order a license for several seats.

For example, with an order for 10 seats, you can activate a license for EcoStruxure Machine Expert on 10 different PCs.

Current offers:

* Single (1 seat)
* Team (10 seats)
* Entity (100 seats)

## Update

There are two kinds of updates for EcoStruxure Machine Expert:

* Minor update (for example from 4.0 to 4.1)

  No need to activate a new license.
* Major update (for example from 3.1 to 4.0)

  It is required to activate a new license.

## Virtual Machines

There is no difference in licensing whether you run EcoStruxure Machine Expert on a physical PC or on a virtual machine.

When you switch between different snapshots on a virtual machine, the activated license for EcoStruxure Machine Expert can become untrusted. In this case, you have to reactivate the license.
