# Buy and Install CODESYS Add-Ons and Packages

## Buy and Install CODESYS Add-Ons and Packages

To buy and install add-ons and packages provided by CODESYS, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. |
| 2 | Select the option Modify installed software. |
| 3 | In the Select a product area, select the following options:   * Product: Select your installed product. * Version: Select the version of your installed product. |
| 4 | In the Choose an action area, select Add-Ons/Patches. |
| 5 | In the Modifying an existing software screen, click the CoDeSys Store button.  Result: Your Internet browser opens the *https://store.codesys.com* website. |
| 6 | Proceed as indicated on the CODESYS website to buy and install CODESYS packages.  NOTE: Make sure that the package you select is compatible with your installation. To display a list of compatible packages, click the CoDeSys Compatibility button in the Modifying an existing software screen. |
| 7 | After you have downloaded the CODESYS package installation file to your local PC, click the From File button in the Modifying an existing software screen, browse to the installation file, and click the Open button.  **Result**: The CODESYS package is listed in the Components list of the Modifying an existing software screen. |
| 8 | Select the component and click the Start Installation or Uninstallation button. |
