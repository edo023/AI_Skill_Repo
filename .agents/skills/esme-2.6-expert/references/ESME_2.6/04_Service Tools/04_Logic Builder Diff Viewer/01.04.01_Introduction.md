# Introduction

## Overview

The Logic Builder Diff Viewer allows you to compare two Logic Builder objects.

Logic Builder Diff Viewer supports POUs in ST, CFC, SFC, IL, FBD, actions, or devices.

You can compare the following objects:

* An object within a Logic Builder project.
* A single file of the file system.
* An SVN item.

Compared objects are shown in a comparison view of the Logic Builder Diff Viewer [dialog](../../../../../api/crossBook?lang=en-US&virtualBookName=D-SE-0066765.html#D-SE-0066765) that provides a compare view similar to the *Project Compare - Differences* [view](../../SoMMenu&topicID=D_SE_0083939).

NOTE: For information on the meaning of colors, the symbols and icons of the compare view, refer to the descriptions under *Project Compare - Differences* [view](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0083939).

## How to Install the Logic Builder Diff Viewer

The installation of the Logic Builder Diff Viewer requires an installation of SoMachine Motion V4.4 or greater.

To install the Logic Builder Diff Viewer, open the Configuration Manager  and activate the check box Logic Builder Diff Viewer in the tab Auxiliary Tools .

## How to Open the Logic Builder Diff Viewer Dialog

There are different ways to open the Logic Builder Diff Viewer dialog:

* In EcoStruxure Machine Expert Logic Builder, click Tools→External Tools →Open Logic Builder Diff Viewer.
* In the Windows Explorer, right-click a project and select LogicBuilder Diff Viewer... .
* In TortoiseSVN:

  + Right-click an svn-file and click  LogicBuilder Diff Viewer....
  + Double-click a Logic Builder object.
  + Right-click a Logic Builder object and select Show changes .
