# About the Book

## Document Scope

This document describes the Logic Builder Diff Viewer. The Logic Builder Diff Viewer is a software tool that allows you to compare two individual objects.

An object can be any object inside a Logic Builder project, a single file of the file system or an SVN item.

## Validity Note

This document has been updated for the release of Logic Builder Diff Viewer V21 delivered with EcoStruxureTM Machine Expert V2.1.1.
