# Compare an Object within SVN with a File Object

## Overview

You can compare an object in TortoiseSVN to a file of the file system.

## How to Compare an Object within SVN with an Object of the File System

| Step | Action |
| --- | --- |
| 1 | In the Log Messages – TortoiseSVN  dialog, right-click an object and click  Open with...: |
| 2 | In the Open with dialog, select the Logic Builder Diff Viewer program and click OK .  **Result:** The Select files... dialog opens. |
| 3 | In the Select files... dialog, the  Left field contains the path of the file saved with the *.tmp* extension to be compared.  Enter the path of the second file into the Right field or browse to the respective file and click OK.  **Result:**   * If the selected files contain only one object, the Logic Builder Diff Viewer opens and presents the [comparison result](D-SE-0066751.html#D-SE-0066751__D-SE-0066751.2). * If one of the files contains more than one object, the  Select objects dialog opens first. |
| 4 | In the Select objects [dialog](D-SE-0066771.html#D-SE-0066771), select an object and click  OK.  **Result**: The compare view of the Logic Builder Diff Viewer presents the [comparison results](D-SE-0066751.html#D-SE-0066751__D-SE-0066751.2). |
