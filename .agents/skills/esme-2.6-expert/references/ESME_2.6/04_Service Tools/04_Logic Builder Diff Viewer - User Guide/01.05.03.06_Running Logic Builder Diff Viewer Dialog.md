# Running Logic Builder Diff Viewer Dialog

## Description

When you open a project, the Logic Builder Diff Viewer tries to connect to a running server instance of your Logic Builder installation. If the Logic Builder Diff Viewer cannot find a running server instance, it starts a new server instance.

For projects that are marked as [editable](D-SE-0066770.html#D-SE-0066770), every time a new server instance is started before and closed after usage.

The  Running Logic Builder Diff Viewer dialog lists the running server instances and their connected clients.

| Element | Description |
| --- | --- |
| PID | Process ID: Each server is listed with its PID (Process ID as represented inside the Windows Task Manager) and its  Start Time |
| Connected Client | The connected clients with their application titles are listed.  You can select one or more entries. |
| Refresh | Click this button to update the list of connected clients. |
| Shut down | Click this button to shut down a selected server.  NOTE: Shutting down the selected server(s) also closes the windows of connected clients. |
| Help | Click the Help button to display the online help. |
| Close | Click this button to close the dialog without any changes. |
