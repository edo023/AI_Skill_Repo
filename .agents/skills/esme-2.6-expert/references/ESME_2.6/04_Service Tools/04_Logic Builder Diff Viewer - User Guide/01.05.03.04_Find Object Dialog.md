# Find Object Dialog

## Overview

Click the magnifier symbol in the toolbar of the Select Objects  dialog or enter a string inside the  Select Objects dialog to open the Find Object dialog:

* You can enter a text in the search field. The found objects are listed with the full Path  in the devices tree.
* Select a node and click Open  to select the node in the tree.
* Click Close to close the dialog.
