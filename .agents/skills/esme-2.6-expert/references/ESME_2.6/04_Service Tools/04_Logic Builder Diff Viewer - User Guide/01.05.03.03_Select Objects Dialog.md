# Select Objects Dialog

## Overview

The  Select Objects dialog is displayed if more than one object has been detected in a file.

This can occur, for example, if you have selected a Logic Builder project or a PLCOpenXML file with several objects.

The Select Objects  dialog has a left and right pane that corresponds to the previously selected left and right file:

* At least one file or object must be selected on both sides.
* If you click OK, the Select Objects dialog is closed, and the objects are added in the [compare view](D-SE-0066773.html#D-SE-0066773).
* Click Cancel to close the dialog without changes.
* Objects that could not be imported correctly are marked with the symbol ![](images/T001804987.gif) on the right of the object, providing detailed Project information  in a tooltip.

## Using the Toolbar in the Select Objects Dialog

The toolbar is displayed in the right and in the left pane.

![](images/T001804961.gif)

Move the cursor over a button in the toolbar to view the associated command as a tooltip:

| Button | Description |
| --- | --- |
| Open | Opens a new file or project inside the view.  This is only available on the right side. |
| Synchronize | Synchronizes the view with the view in the other pane.  If selected, the other tree view items are synchronized by the selected one on this side.  A search starts to find the item:   1. Search using the path. 2. If the search using the path did not return the item, search using the name only. |
| Collapse all | Collapses all items. |
| Expand all | Expands all items. |
| Find object | Opens the Find Object [dialog](D-SE-0066772.html#D-SE-0066772) with the name of the selected item. |
| Compare object | Compares the content of each object of one side with the content of the corresponding object on the other side.   * Different objects are marked with red bold letters. * Unavailable objects are marked with blue bold letters. |

## Using Keys to Navigate in the Select Objects Dialog

| Key | Action |
| --- | --- |
| **\*** | Expand all nodes of the selected item. |
| **/** | Collapse all nodes of the selected item. |
| **+** | Expand the selected node. |
| **-** | Collapse the selected node. |
| Page Up | Select top node inside visible range. |
| Page Down | Select bottom node inside visible range. |
| Home | Select top root node. |
| End | Select bottom node. |
| Left Arrow | Collapse the selected node. |
| Right Arrow | Expand the selected node. |
| Up Arrow | Select previous node. |
| Down Arrow | Select next node. |
