# Compare Files Using Windows Explorer

## Overview

You can call the Logic Builder Diff Viewer from Windows Explorer.

## How to Compare Files from the Windows Explorer

You can also open the Logic Builder Diff Viewer using the context menu in Windows Explorer.

| Step | Action |
| --- | --- |
| 1 | Right-click a project file in Windows Explorer and click Logic Builder Diff Viewer.  **Result:** The Select files... [dialog](D-SE-0066770.html#D-SE-0066770) opens, where the  Left field contains the selected file. |
| 2 | In the Select files... dialog, enter a file for the Right field.  **Result:**   * If the selected files contain only one object, the Logic Builder Diff Viewer dialog opens presenting the compare view. * If one file contains more than one object, the  Select objects [dialog](D-SE-0066771.html#D-SE-0066771) opens first. |
