# Compare Two Objects Between Different Trees

## Overview

TortoiseSVN can compare two trees inside a repository. As a result a list of objects is displayed in a dialog.

Each of these objects can be compared.

For information on how to compare these objects, refer also to [*How to Compare two Revisions of one Object*](D-SE-0066759.html#D-SE-0066759__D-SE-0066759.3).

## How to Compare Two Objects Between Trees

| Step | Action |
| --- | --- |
| 1 | Right-click the tree node you want to be compared and click Mark for comparison .  **Result**:The selected tree node is displayed in bold. |
| 2 | Right-click the other tree node to be compared to the marked one and click Compare URLs.  **Result:** The  Changed Files dialog opens. |
| 3 | Double click or right-click a file and select Compare revisions .  **Result:** The dialog Logic Builder Diff Viewer opens presenting the comparison results. |
