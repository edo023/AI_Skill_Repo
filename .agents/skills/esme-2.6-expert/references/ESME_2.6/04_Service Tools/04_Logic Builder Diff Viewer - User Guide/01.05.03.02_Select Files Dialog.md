# Select Files Dialog

## Overview

Click  File→ Open to open the  Select Files... dialog. In the fields Left and  Right, select the files to compare either by browsing to the respective folder or by dragging a file to the fields.

| Element | Description |
| --- | --- |
| Left | If you have selected a file from inside SVNTortoise or from [Windows Explorer context menu](D-SE-0066763.html#D-SE-0066763) in a previous step, the file is entered automatically in the field  Left.  The  Down Arrow key on the right of the text fields provides a list of the previously selected files. |
| Right | Select the file to be compared.  The Down Arrow key on the right of the text fields provides a list of the previously selected files. |
| Browse... | Browse to the file to compare.  You can also drag a file from the file system to the fields  Left and Right. |
| Make left project editable | If you select this checkbox, you can modify the project selected in the Left  field within the compare view.  This field is not enabled for SVN-version controlled projects. SVN-version controlled projects need to be unlocked before. |
| Compare Options | With the checkboxes you can select various comparison options.  Select the checkbox Ignore Whitespace  if you want to ignore whitespaces. Semantically relevant whitespaces, for example in string literals, are not ignored.  Select the checkbox Ignore Comments  if you want to ignore comments.  Select the checkbox  Ignore Properties if you want to ignore properties, for example folders, access rights, exclusion from build, POU bitmaps. |
| OK | Click the OK button to open the compare view. |
| Cancel | Click the Cancel button to close the dialog and return to the main menu of the Logic Builder Diff Viewer. |
| Help | Click the Help button to display the online help. |
