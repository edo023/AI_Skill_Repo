# Message MS1402: The cycle time of the selected motion and the cycle time of the selected load must be equal.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1402 | The cycle time of the selected motion and the cycle time of the selected load must be equal. | Error |

## Cause of the Message

The cycle time you defined in the Motion  [diagram](D-SE-0058871.html#D-SE-0058871) does not match the cycle time you defined in the Load Profile .

## Effect

Calculations cannot be performed.

## Solution

Verify the cycle time values in the Motion  [diagram](D-SE-0058871.html#D-SE-0058871) and in the  Load Profile and adapt them.
