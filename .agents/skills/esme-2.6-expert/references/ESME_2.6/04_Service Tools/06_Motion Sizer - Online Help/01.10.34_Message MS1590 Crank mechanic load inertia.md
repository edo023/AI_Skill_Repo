# Message MS1590: Crank mechanic load inertia in result is shown for the maximum transmission ratio possible.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1590 | Crank mechanic load inertia in result is shown for the maximum transmission ratio possible. | Information |

## Cause of the Message

For the load case Crank , a gearbox is selected.

## Effect

For the calculations, Motion Sizer uses the maximum transmission ratio that is available for the selected gearbox.

## Solution

This message is a reminder that there are other transmission ratios available.
