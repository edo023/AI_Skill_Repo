# Message MS2101: Capacity of DC bus exceeds the permissible limit.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS2101 | Capacity of DC bus exceeds the permissible limit. | Error |

## Cause of the Message

Motion Sizer calculations have shown that the sum of the capacities in a shared DC bus is exceeded.

## Effect

Calculations cannot be performed.

## Solution

Verify the total capacity of the drives that have been assigned to the same power circuit. Make sure that the total capacity that is allowed in the power circuit is not exceeded.
