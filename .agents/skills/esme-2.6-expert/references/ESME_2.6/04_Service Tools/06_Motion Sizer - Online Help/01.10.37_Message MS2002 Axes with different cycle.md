# Message MS2002: Axes with different cycle times are in the same power circuit.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS2002 | Axes with different cycle times are in the same power circuit. | Error |

## Cause of the Message

Axes with different cycle times have been assigned to the same power circuit.

## Effect

Motion Sizer cannot perform power circuit calculations for several axes if the cycle times differ.

## Solution

In order to perform power circuit calculations with Motion Sizer, use the same cycle time for those axes that are grouped in one power circuit.

If this is not possible, calculate the power circuit manually.
