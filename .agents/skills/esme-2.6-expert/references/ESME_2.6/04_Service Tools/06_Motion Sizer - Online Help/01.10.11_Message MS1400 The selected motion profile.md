# Message MS1400: The selected motion profile does not match the selected load profile. The load profile must have the same x offset, cycle time, x min and x max as the motion profile.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1400 | The selected motion profile does not match the selected load profile. The load profile must have the same x offset, cycle time, x min and x max as the motion profile. | Information |

## Cause of the Message

The motion profile and the load diagram selected for the mechanics of your machine application do not match. The values for the parameters  x offset, Cycle time as well as the  X value for start and end points of a segment must be identical in both profiles.

## Effect

Motion Sizer calculations do not lead to usable results if the values differ between load and motion.

## Solution

Verify the values for x offset, Cycle time as well as the X value for start and end points you defined for the Cam and the load [diagrams](D-SE-0058871.html#D-SE-0058871) and adapt them.

Select a load case that corresponds to the motion profile when you define the [mechanic](D-SE-0057930.html#D-SE-0057930) of your machine application.
