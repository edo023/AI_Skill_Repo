# Message MS1201: No environment information has been provided for the axis. Possible derating is not applied in calculations.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1201 | No environment information has been provided for the axis. Possible derating is not applied in calculations. | Information |

## Cause of the Message

In your Motion Sizer project, you did not enter values for temperature and/or operating altitude of your motor, drive, or gearbox in the  Environment editor.

## Effect

For the calculations, Motion Sizer does not apply derating for the motor, drive, or gearbox because no values are provided in the Environment  editor.

## Solution

If your motor, drive, or gearbox is operated under specific environmental conditions that cause derating, enter the respective values in the  Environment editor. They will then be considered during calculations.
