# Message MS1451: Gap between segments: X end of segment #{0} < X start of segment #{1}

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1451 | Gap between segments: X end of segment #{0} < X start of segment #{1} | Error |

## Cause of the Message

The value you defined in the Motion  [diagram](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.4) for the  X parameter of the end point of a cam or a load segment is less than the X parameter of the start point of the next cam or load segment.

## Effect

Calculations cannot be performed.

## Solution

Verify the  X parameter values for the subsequent cam or load segments. Make sure that the X  parameter of the end point of a preceding segment is identical to the X parameter of the start point of the following segment.
