# Message MS1507: The notification level 2 for motor inertia ratio is exceeded.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1507 | The notification level 2 for motor inertia ratio is exceeded. | Advisory |

## Cause of the Message

The total inertia value calculated for the load in the inertia calculation dialog exceeds the second threshold that is specific for the selected motor. It corresponds to the parameter WJJ2 in the motor configuration.

## Effect

The selected motor does not meet the requirements of the application because it cannot overcome the inertia of the load.

## Solution

Select a motor with greater torque and speed values from the Device selection editor.

Reduce the load of the application.
