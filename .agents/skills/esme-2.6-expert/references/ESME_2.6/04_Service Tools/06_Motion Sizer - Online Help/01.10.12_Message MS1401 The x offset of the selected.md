# Message MS1401: The x offset of the selected motion does not match the x offset of the selected load.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1401 | The x offset of the selected motion does not match the x offset of the selected load. | Information |

## Cause of the Message

The x offset values you defined in the Motion  [diagram](D-SE-0058871.html#D-SE-0058871) do not match the  x offset values you defined in the Load Profile.

## Effect

Motion Sizer calculations do not lead to usable results if the  x offset values differ between load and motion.

## Solution

Verify the  x offset values in the  Motion [diagrams](D-SE-0058871.html#D-SE-0058871) and in the  Load Profile and adapt them.
