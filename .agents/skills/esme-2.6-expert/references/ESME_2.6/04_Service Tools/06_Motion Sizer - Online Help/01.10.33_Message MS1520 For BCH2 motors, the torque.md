# Message MS1520: For BCH2 motors, the torque by speed curve does not display the real motor performance.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1520 | For BCH2 motors, the torque by speed curve does not display the real motor performance. For further information, refer to the [BCH2 user guides](D-SE-0057929.3.html#D-SE-0057929.3__RelatedDocuments-DA1AB868). | Advisory |

## Cause of the Message

A BCH2 motor has been selected for an axis.

## Effect

Motion Sizer does not display the accurate torque by speed curve.

## Solution

For further information, refer to the [BCH2 user guides](D-SE-0057929.3.html#D-SE-0057929.3__RelatedDocuments-DA1AB868).
