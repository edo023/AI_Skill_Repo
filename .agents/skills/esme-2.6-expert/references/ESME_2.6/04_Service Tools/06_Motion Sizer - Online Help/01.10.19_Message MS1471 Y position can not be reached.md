# Message MS1471: Y position can not be reached with given segment definition.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1471 | Y position can not be reached with given segment definition. | Error |

## Cause of the Message

The Motion profile > Sine and linear combined has been selected in the Motion of a [cam diagram](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.4). With the setting  Input parameters > Curvature Start, Curvature End, the values you entered for the left and right Y parameters do not comply with the value you entered for the  Curvature. The  Y coordinate at the end point of the segment cannot be reached with the given start  Y coordinate and  Curvature values.

## Effect

Calculations cannot be performed.

## Solution

Verify the values for the Y parameters at the Left segment edge - start  and at the Right segment edge - end and the value for the parameter  Curvature. The impacts that the different values have are illustrated in no time in the position curve.
