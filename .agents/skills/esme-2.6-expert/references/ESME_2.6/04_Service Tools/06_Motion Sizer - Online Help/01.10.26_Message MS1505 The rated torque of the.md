# Message MS1505: The rated torque of the gearbox is exceeded.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1505 | The rated torque of the gearbox is exceeded. | Advisory |

## Cause of the Message

Motion Sizer calculations have shown that the selected gearbox cannot take up the average value of the torques that are available in the application.

## Effect

If you designed your motion process according to the parameters you entered, the gearbox would not support the average value of the torques that are available in the application.

## Solution

Select a gearbox with a greater Effective torque at effective speed value from the Device selection editor.

Adapt the requirements of the application:

* Reduce the load.
* Reduce the inertia.
* Reduce the acceleration and deceleration requirements.
