# Message MS3105: Unknown gearbox selected for axis: device does not exist in product database.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS3105 | Unknown gearbox selected for axis: device does not exist in product database. | Error |

## Cause of the Message

Your project contains a gearbox that is not available in the product database.

## Effect

Calculations for this gearbox cannot be performed.

## Solution

Make the required product database available on the PC that opens the Motion Sizer project.
