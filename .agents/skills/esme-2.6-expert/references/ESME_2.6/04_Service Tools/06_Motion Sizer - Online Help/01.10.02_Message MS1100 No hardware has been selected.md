# Message MS1100: No hardware has been selected for the axis

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1100 | No hardware has been selected for the axis | Information |

## Cause of the Message

In your Motion Sizer project, an axis exists, but you have not yet selected the hardware (motor, drive, gearbox) for this axis.

## Effect

For this axis, calculations cannot be performed.

## Solution

Select a motor, drive, or gearbox from the Device selection  editor.
