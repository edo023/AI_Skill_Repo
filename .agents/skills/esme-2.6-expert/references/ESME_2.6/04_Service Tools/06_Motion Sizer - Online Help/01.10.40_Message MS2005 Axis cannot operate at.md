# Message MS2005: Axis cannot operate at specified position in power circuit.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS2005 | Axis cannot operate at specified position in power circuit. | Error |

## Cause of the Message

Drives with different requirements have been assigned to the same power circuit. Specific measures, such as an additional connection module, are required to operate the selected types of drives in the same power circuit.

## Effect

Calculations cannot be performed.

## Solution

Refer to the documentation of the drives to verify the operation requirements.
