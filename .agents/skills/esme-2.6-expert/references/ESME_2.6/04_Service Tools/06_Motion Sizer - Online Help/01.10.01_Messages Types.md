# Messages Types

## Overview

The following types of messages are displayed in Motion Sizer:

| Message type | Icon | Description | Solution |
| --- | --- | --- | --- |
| Error |  | Motion Sizer cannot process the entered parameters correctly because invalid input data have been detected. This has the effect that no calculation results are available or that some of the inputs are not considered in the calculation.  An error indicates that calculations have not been performed correctly and that subsequent calculations could not be performed. | In order to get meaningful and complete calculation results, perform the actions required to remove the errors listed. |
| Advisory |  | During the calculation process Motion Sizer has detected that the selection is not valid. This means that the configuration does not meet the application requirements or that equipment is used outside of the specified range.  An advisory indicates that the machine (if built as modeled in the project) would not perform as expected. | In order to find the hardware for the machine that meets the requirements of your application, perform the actions required to remove the advisory messages listed. |
| Information |  | Motion Sizer provides additional information. For example, further information that is worth to be considered or hints on possible optimizations. | – |
