# Message MS3100: Unknown motor-drive-combination selected for axis: device does not exist in product database.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS3100 | Unknown motor-drive-combination selected for axis: device does not exist in product database. | Error |

## Cause of the Message

Your project contains a motor or drive that is not available in the product database.

## Effect

Calculations for this motor or drive cannot be performed.

## Solution

Make the required product database available on the PC that opens the Motion Sizer project.
