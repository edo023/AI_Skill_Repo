# Message MS1452: Segment is inverted: X start > X end

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1452 | Segment is inverted: X start > X end | Error |

## Cause of the Message

The value you defined in the Motion  [diagram](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.4) for the  X parameter of the start point of a cam or a load segment is greater than the X parameter of the end point of the segment.

## Effect

Calculations cannot be performed.

## Solution

Make sure that the value for the X parameter of the start point is less than the X parameter of the end point of the segment.
