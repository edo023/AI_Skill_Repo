# Message MS1303: The velocity exceeds its defined limits.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1303 | The velocity exceeds its defined limits. | Advisory |

## Cause of the Message

The velocity values you defined in the Motion  [diagram](D-SE-0058871.html#D-SE-0058871) are beyond the limits you defined in the Motion limits  editor for the Velocity range .

## Effect

Calculations cannot be performed.

## Solution

Adapt the values defined in the Motion diagram.

Adapt the limits defined for the  Velocity range in the  Motion limits editor.
