# Message MS1304: The acceleration exceeds its defined limits.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1304 | The acceleration exceeds its defined limits. | Advisory |

## Cause of the Message

The acceleration values you defined in the Motion [diagram](D-SE-0058871.html#D-SE-0058871) are beyond the limits you defined in the Motion limits  editor for the Acceleration range.

## Effect

Calculations cannot be performed.

## Solution

Adapt the values defined in the Motion diagram.

Adapt the limits defined for the  Acceleration range in the  Motion limits editor.
