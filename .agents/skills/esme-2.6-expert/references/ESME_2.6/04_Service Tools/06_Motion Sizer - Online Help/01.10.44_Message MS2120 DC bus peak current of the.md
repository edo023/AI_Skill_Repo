# Message MS2120: DC bus peak current of the application exceeds the DC bus peak current of the selected hardware.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS2120 | DC bus peak current of the application exceeds the DC bus peak current of the selected hardware. | Advisory |

## Cause of the Message

Motion Sizer calculations have shown that the drive you selected does not provide sufficient peak current for the application.

## Effect

Calculations cannot be performed.

## Solution

Select a drive with a greater DC bus peak current from the Device selection editor.

Adapt the requirements of the application.

* Reduce the load.
* Reduce the acceleration and deceleration requirements.
