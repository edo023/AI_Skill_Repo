# Message MS1495: Unit type of load profile doesn't match the load case.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1495 | Unit type of load profile doesn't match the load case. | Error |

## Cause of the Message

The unit selected in the Load Profile  does not comply with the type of motion that is described in the selected load case of the Mechanic .

## Effect

Calculations cannot be performed.

## Solution

Verify the type of motion that is described in the selected load case of the  Mechanic. In the  Load Profile, select the suitable unit for rotary or for linear motion.
