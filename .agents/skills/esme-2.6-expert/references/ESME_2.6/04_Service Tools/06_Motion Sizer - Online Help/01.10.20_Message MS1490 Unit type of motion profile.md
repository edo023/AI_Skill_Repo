# Message MS1490: Unit type of motion profile doesn't match the unit type expected by the load case.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1490 | Unit type of motion profile doesn't match the unit type expected by the load case. | Error |

## Cause of the Message

The unit selected for the parameter Feed constant in the Mechanic of the [general load case](D-SE-0057932.html#D-SE-0057932__D-SE-0057932.3) or the [load case rotary table](D-SE-0059090.html#D-SE-0059090__D-SE-0059090.3) does not comply with the unit selected in the Motion  of a [cam diagram](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.3).

## Effect

Calculations cannot be performed.

## Solution

Verify the unit selected for the parameter Feed constant  in the Mechanic  of the [general load case](D-SE-0057932.html#D-SE-0057932__D-SE-0057932.3) or the [load case rotary table](D-SE-0059090.html#D-SE-0059090__D-SE-0059090.3) as well as the unit selected in the Motion  of a [cam diagram](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.3). Select the same unit of measure for both parameters.
