# Message MS1504: The rated speed of the gearbox is exceeded.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1504 | The rated speed of the gearbox is exceeded. | Advisory |

## Cause of the Message

Motion Sizer calculations have shown that the selected gearbox cannot support the rated speed at the input that is required by the application.

## Effect

If you designed your motion process according to the parameters you entered, the gearbox would not support the rated speed at the input that is required by the application.

## Solution

Select another gearbox in the Device selection  editor.

Reduce the speed in the motion profile.
