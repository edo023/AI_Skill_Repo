# Message MS1111: Another transmission ratio is available for this gearbox that could result in a lower peak torque.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1111 | Another transmission ratio is available for this gearbox that could result in a lower peak torque. | Information |

## Cause of the Message

An alternative transmission ratio is available for the selected gearbox. This transmission ratio could result in a lower peak torque.

## Effect

You can reduce the peak torque if you select the alternative transmission ratio. For further information, refer to the Torque over gearbox factor diagram.

## Solution

Select the alternative transmission ratio for the gearbox from the **Device selection**.
