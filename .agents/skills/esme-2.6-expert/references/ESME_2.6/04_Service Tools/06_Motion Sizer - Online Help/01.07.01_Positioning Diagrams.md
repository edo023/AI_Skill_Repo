# Positioning Diagrams

## Overview

The positioning diagrams in Motion Sizer allow you to design the motion of your axis graphically over time.

You can define segments for each positioning diagram that correspond to the different sequences of motion that your axis performs. The properties of a segment allow you to enter specific values for the motion parameters. These parameters are equivalent to the input parameters of the function block FB\_VarioPosJerk in the PD\_PacDriveLib library that is provided by EcoStruxure Machine Expert.

## Properties of a Positioning Diagram

The properties of a positioning diagram provide the following parameters:

| Parameter | Description | Value range |
| --- | --- | --- |
| Name | A name for the selected positioning diagram that is unique within the axis. | Text |
| Description | Additional information related to the selected positioning diagram. | Text |
| Units | The unit of measure for the parameters. | * mm * m * in * ° (degrees) * Unit (user defined) |

## Properties of a Segment

The properties of a segment provide the following parameters:

| Parameter | Description | Value range |
| --- | --- | --- |
| Name | A name of your choice for the selected segment. | Text |
| Motion profile section | | |
| Distance | The distance that the axis travels within this segment.  Use negative values to describe movements in reverse direction. | Positive and negative values, including 0. |
| Dwell time | A dwell time is inserted at the end of the segment. If the value is different from 0, then the parameters  End speed and  End acceleration are automatically set to 0 and cannot be edited. | Positive values including 0. |
| Maximal speed | The maximum speed that is allowed for the segment.  The values are positive, but the velocity is in negative direction if the Distance  is negative. | Positive values, 0 is not allowed. |
| Maximal acceleration | The maximum acceleration that is allowed for the segment.  The values are positive, but the acceleration can be in positive or negative direction, depending on the direction of the distance.  Maximal acceleration  can be a deceleration if the parameter  Start speed >  Maximal speed. In this case, the velocity during the acceleration phase is > Maximal speed. | Positive values, 0 is not allowed. |
| Maximal deceleration | The maximum deceleration that is allowed for the segment.  Enter a positive value, but for this parameter the acceleration is executed in reverse direction. The deceleration can be in positive or negative direction, depending on the direction indicated by the distance parameter.  Maximal deceleration can be an acceleration if the parameter End speed  > Maximal speed . In this case, the velocity during the acceleration phase is > Maximal speed . | Positive values, 0 is not allowed. |
| Maximal jerk | The maximum jerk that is allowed during acceleration and deceleration. | Positive values, 0 is not allowed. |
| Start point section | | |
| Start speed | The speed at the start of the segment.  This parameter can be defined for the first segment of a diagram only. For other diagrams, it is automatically determined by the parameter  End speed of the previous segment.  Use negative values to describe movements in reverse direction. | Absolute values, positive, and negative, including 0. |
| Start acceleration | The acceleration at the start of the segment.  This parameter can be defined for the first segment of a diagram only. For other diagrams, it is automatically determined by the parameter End acceleration  of the previous segment.  Use negative values to describe accelerations in reverse direction. | Absolute values, positive, and negative, including 0. |
| End point section | | |
| End speed | The velocity for the end of the segment.  If the parameter Dwell time  has a value >0, then this parameter is fixed to 0.  Use negative values to describe movements in reverse direction. | Absolute values. |
| End acceleration | The acceleration for the end of the segment.  If the parameter Dwell time has a value >0, then this parameter is fixed to 0.  Use negative values to describe accelerations in reverse direction. | Absolute values. |
