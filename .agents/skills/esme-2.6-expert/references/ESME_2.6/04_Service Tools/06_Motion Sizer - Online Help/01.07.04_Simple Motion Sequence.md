# Simple Motion Sequence

## Overview

The simple motion sequence in Motion Sizer allows you to design the motion of your axis in a simplified way, compared to cam diagrams. You can select more general input parameters defining time and end position or velocity that are key to your application. In addition to these input parameters, you can specify a higher or lower acceleration for your application via the Smooth parameter. Motion Sizer then calculates the remaining parameters to model the movement of your servo axis.

You can define segments for each simple motion sequence that correspond to the different sequences of motion that your axis performs.

## Properties of a Simple Motion Sequence

The properties of a simple motion sequence provide the following parameters:

| Parameter | Description | Value range |
| --- | --- | --- |
| Name | A name for the selected simple motion sequence that is unique within the axis. | Text |
| Description | Additional information related to the selected simple motion sequence. | Text |
| Units | The unit of measure for the parameters. | * mm * m * in * ° (degrees) * Unit (user defined) |

## Properties of a Segment

The properties of a segment provide several parameters. The parameters and their general descriptions are listed in this section. The  Input parameters list allows you to enable two parameters of the Right segment edge - end section that are given by your application for input. The remaining parameters will be calculated by Motion Sizer based on the input values.

The table describes the parameters of a simple motion segment:

| Parameter | Description | Value range |
| --- | --- | --- |
| Input parameters | Defines the parameters that are entered as input parameters. They are used by Motion Sizer as a basis to calculate the remaining parameters required to model the movement of your servo axis. | * time end, position end * position end, total time * time end, distance * distance, total time * time end, velocity end * velocity end, total time |
| Smooth | The type of acceleration that is allowed.  The lower the value, the higher the acceleration.  The higher the value, the lower the acceleration. | Decimal number greater than 0, less than or equal to 1. |
| Left segment edge - start section | | |
| Time | The time the segment starts. | Positive values, including 0. Default = s. |
| Position | The position of the axis at the segment start. | Decimal number |
| Velocity | The velocity of the axis at the segment start. | Decimal number |
| Right segment edge - end section | | |
| Time | The time the segment ends. | User input(1) or calculated by Motion Sizer.  Positive values, including 0. Default = s.  Must be greater than start Time . |
| Total time | The elapsed time of the segment. | User input(1) or calculated by Motion Sizer.  Positive values, 0 is not allowed. Default = s. |
| Position | The position of the axis at the segment end. | User input(1) or calculated by Motion Sizer.  Decimal number |
| Distance | The distance that the axis travels within this segment. | User input(1) or calculated by Motion Sizer.  Positive and negative values, including 0. |
| Velocity | The velocity of the axis at the segment end. | User input(1) or calculated by Motion Sizer.  Positive and negative values, including 0. |
