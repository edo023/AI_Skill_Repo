# Message MS2116: Selected drive is not compatible to selected braking resistor.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS2116 | Selected drive is not compatible to selected braking resistor. | Error |

## Cause of the Message

The external braking resistor is not compatible with the selected drive.

## Effect

Calculations cannot be performed.

## Solution

Select a braking resistor that is compatible with the selected drive.
