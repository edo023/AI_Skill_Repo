# Message MS1470: Orientation of motion (delta Y) does not fit to defined curvatures at segment border(s).

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1470 | Orientation of motion (delta Y) does not fit to defined curvatures at segment border(s). | Error |

## Cause of the Message

The Motion profile > Sine and linear combined has been selected in the Motion of a [cam diagram](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.4). With the setting  Input parameters > Curvature Start, Curvature End, the values entered for the left and right Y parameters result in an orientation of motion that does not comply with the value entered for the Curvature .

## Effect

Calculations cannot be performed.

## Solution

Verify the values for the Y parameters at the Left segment edge - start  and at the Right segment edge - end and the value for the parameter  Curvature. The impacts that the different values have are illustrated in no time in the position curve.
