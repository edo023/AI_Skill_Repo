# Load Diagrams

## Overview

The load diagrams in Motion Sizer are used to define a load that is exerted on a servo axis during specific sequences of motion. The motion of the axis is defined in a corresponding motion diagram.

Use the same segment parameter settings for a load diagram as for the respective cam diagram. Define the additional load in the load diagram with the parameter  Y of the Left segment edge - start section.

## Plus / Minus Sign of the Load

In the final torque calculation, the individual torques resulting from either dynamics or friction as well as additional torques are added, taking into account the direction of the torque.

Consequently, the plus and minus sign (+/-) of the load in load diagrams is defined as follows:

* The load sign is **equal** to the direction of the motion if the **load opposes** the motion of the axis and therefore requires more torque.
* The load sign is **opposite** to the direction of the motion if the **load supports** the motion of the axis.

| Sign of direction of the motion (sign of velocity) | Sign of supporting load | Sign of opposing load |
| --- | --- | --- |
| + | - | + |
| - | + | - |

## Properties of a Load Diagram

The properties of a load diagram are identical to the [properties of a cam diagram](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.3). The only exception is the parameter Units : For load diagrams, only the units  lb, N , and N•m , lb•ft  are available.

## Properties of a Segment

The properties of a segment of the load diagram are identical to the [properties of a segment of the cam diagram](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.4). The only exception is the parameter Y  of the Left segment edge - start section. For load diagrams, use this parameter to define the additional load that is exerted on the servo axis within the specific segment.

## Motion Profiles Available for Combinations of Start Conditions and End Conditions

For load diagrams, the combinations of Start condition  and End condition  result in the same Motion profile  values as for [cam profiles](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.5).

## Parameters Available for the Different Motion Law Combinations

For load diagrams, the same dependencies apply between the selected motion laws and the available parameters as for [cam profiles](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.6).
