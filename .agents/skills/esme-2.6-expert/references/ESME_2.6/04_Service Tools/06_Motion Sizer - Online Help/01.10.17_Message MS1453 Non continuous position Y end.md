# Message MS1453: Non continuous position: Y end of segment #{0} != Y start of segment #{1}

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1453 | Non continuous position: Y end of segment #{0} != Y start of segment #{1} | Error |

## Cause of the Message

The value you defined in the Motion  [diagram](D-SE-0058886.html#D-SE-0058886__D-SE-0058886.4) for the  Y parameter of the end point of a cam or a load segment is greater than the Y parameter of the start point of the next cam or load segment.

## Effect

Calculations cannot be performed.

## Solution

Verify the  Y parameter values for the subsequent cam or load segments. Make sure that the value for the  Y parameter for the end point of a preceding segment is identical to the Y parameter of the start point of the following segment.
