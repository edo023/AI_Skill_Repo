# Message MS2004: Axes with different supply voltage or different supply phase count are in the same power circuit.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS2004 | Axes with different supply voltage or different supply phase count are in the same power circuit. | Error |

## Cause of the Message

Axes with drives that have different power supply requirements have been assigned to the same power circuit.

## Effect

Motion Sizer cannot perform power circuit calculations because it is not allowed to have drives with different power supply requirements (for example, different supply voltage or different supply phase) in the same power circuit.

## Solution

Only group drives of the same family with the same power supply requirements into one power circuit.
