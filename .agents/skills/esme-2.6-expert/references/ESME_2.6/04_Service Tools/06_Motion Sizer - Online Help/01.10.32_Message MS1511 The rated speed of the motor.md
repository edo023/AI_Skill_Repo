# Message MS1511: The rated speed of the motor is exceeded.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1511 | The rated speed of the motor is exceeded. | Advisory |

## Cause of the Message

Motion Sizer calculations have shown that the selected motor does not provide the rated speed that is required in the application.

## Effect

If you designed your motion process according to the parameters you entered, the motor would not provide the rated speed that is required in the application.

## Solution

Select a motor with a greater Nominal speed  value from the Device selection  editor.

Use a gearbox.
