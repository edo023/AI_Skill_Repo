# Message MS1506: The notification level 1 for motor inertia ratio is exceeded.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS1506 | The notification level 1 for motor inertia ratio is exceeded. | Advisory |

## Cause of the Message

The total inertia value calculated for the load in the inertia calculation dialog exceeds the first threshold that is specific for the selected motor. It corresponds to the parameter WJJ1 in the motor configuration.

## Effect

The selected motor is not optimized to meet the requirements of the application because it can hardly overcome the inertia of the load.

## Solution

Select a motor with greater torque and speed values from the Device selection editor.

Reduce the load of the application.
