# Message MS2003: Axes with incompatible drive families in the same power circuit.

## Overview

| Message ID | Message description | Message type |
| --- | --- | --- |
| MS2003 | Axes with incompatible drive families in the same power circuit. | Error |

## Cause of the Message

Axes with drives belonging to different families (for example, Lexium 62 and Lexium 32) have been assigned to the same power circuit.

## Effect

Motion Sizer cannot perform power circuit calculations because it is not allowed to have drives belonging to different families (for example, Lexium 62 and Lexium 32) in the same power circuit.

## Solution

Only group drives of the same family into one power circuit.
