# Install New Software

## Overview

After [you have started the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. Select the option Install new software.

The Select a product area provides three lists to select the options for:

* Source (source to install the software from)

  + Online (Schneider Electric server)
  + Browse from local or network folder
  + Select Directly a Removable Medium (such as USB mass storage device).

    NOTE: If you insert a removable medium after you have started the software, it is displayed in the Source list. For this, the root folder of the medium must provide a suitable product to be installed.
* Product (a list of suitable products found on the source)
* Version (list of available versions of the selected product found on the source)

## Install New Software (Online)

To install new software from the Schneider Electric server, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. |
| 2 | Select the option Install new software. |
| 3 | In the Select a product area, select the following options:   * Source: Online * Product: Select the product you want to install. * Version: Select the version of the product you want to install. |
| 4 | Click Next. |
| 5 | Choose the components you want to install by executing one of the following steps:   * From the Features tab, select one or more options in order to install the components related to the selected feature(s). You can also use the Select all/Reset selection button at the top of the list.  NOTE: The numbers in brackets behind each feature indicate the total number of components of each feature compared to the components you have installed via the Individual components tab or via a previous installation. * From the Licenses tab, select either the Standard or Professional option to install the components that are included in the selected license.  The components included in the selected license type are displayed on the right-hand side of the dialog box. For a list of functions and add-ons that are supported by the different licenses, refer to the [Overview of Licenses and Supported Features](D-SE-0075111.html#D-SE-0075111__D-SE-0075111.29) * Select the Individual components tab to display Components listed with the available versions for selection. * Select the language or languages you want to install for your product.  NOTE: By default, English and the language of your operating system are installed. You can select additional languages (Select languages to install) to be installed. To modify installed languages after the installation has been completed, refer to [Modify the Product Language](D-SE-0088009.html#D-SE-0088009__ModifyTheProductLanguage-9498BE0C). |
| 6 | As an optional step, click the Installation paths tab to configure individual folders for the installation files of   * online help * DTMs * Vijeo-Designer |
| 7 | As an optional step, click the Help tab to select online help language versions for download and local installation.  To customize the help installation, open the Schneider Electric [Help Manager](D-SE-0095358.html#D-SE-0095358__D-SE-0095358.2) by clicking the Open help management tool button. |
| 8 | Click Next to continue with the installation. |
| 9 | Read and accept the License Agreement to continue with the installation of the product and click Start Installation.  NOTE: If Windows Security messages are displayed during driver installation of the gateway, confirm with Install.  **Result**: After the installation has finished successfully, the Installation Completed message is displayed. |
| 10 | Click Finish to close the Schneider Electric Software Installer or click the Open Release Notes button to view the Release Notes document of the present installation. |

## Install New Software (From Local/Network Folder)

To install new software from a local/network folder, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. |
| 2 | Select the option Install new software. |
| 3 | In the Select a product area, select the following options:   * Source: Browse from local or network folder  Select the folder and browse for the installation file.  NOTE: The Source must provide a suitable product. * Product: Select the product you want to install. * Version: Select the version of the product you want to install. |
| 4 | Click Next. |
| 5 | Proceed as described under [Install New Software (Online)](#D-SE-0088006__D-SE-0088006.2). |

## Install New Software (From a Removable Medium)

To install new software from a removable medium (such as USB mass storage device, external hard disk, DVD), proceed as follows:

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. |
| 2 | Select the option Install new software. |
| 3 | In the Select a product area, select the following options:   * Source: Select Directly a Removable Medium  Select the drive/folder and browse for the installation file.  NOTE: The Source must provide a suitable product. * Product: Select the product you want to install. * Version: Select the version of the product you want to install. |
| 4 | Click Next. |
| 5 | Proceed as described under [Install New Software (Online)](#D-SE-0088006__D-SE-0088006.2). |
