# License Verification

## Overview

The license for EcoStruxure Machine Expert covers the following software parts:

* EcoStruxure Machine Expert Logic Builder
* Vijeo-Designer

The usage of the following software parts may require separate licenses:

* Add-ons, for example SVN (Subversion client)
* Libraries
* Device descriptions

## During Launch Of EcoStruxure Machine Expert

During launch of EcoStruxure Machine Expert, the following licenses are verified:

* License for EcoStruxure Machine Expert

  If no permanent license for EcoStruxure Machine Expert is activated and the trial period has expired, a dialog box reminds you to activate the license. You can start the Schneider Electric License Manager from the dialog box.
* Licenses for installed add-ons

  For add-ons different license verifications might be performed. For details, refer to the online help of the respective add-on.

## During Build of Project

During Build (compile) of your project, the following licenses are verified (if needed):

* Libraries
* Device descriptions

The license for a library/device is verified, as soon as the library/device is part of your project.

If a license for a library or a device (device description) is required, one of the following messages is displayed in the Messages view:

* Trial period is running. The trial license expires in {n} days.

  Open the Schneider Electric License Manager and activate a permanent license. Without activation, you can use the software until the trial period is expired.
* Trial license expired.

  Open the Schneider Electric License Manager and activate the license.
* License not available.

  Open the Schneider Electric License Manager and activate the license.
* A free registration is required to use this item.

  Open the Schneider Electric License Manager and start the free registration.
* Not supported on your platform.

  A library or device cannot be used with your EcoStruxure Machine Expert installation. Delete the library or device from your project.

NOTE: You can open the Schneider Electric License Manager by clicking the button ... right beside the message.

NOTE: After activation, restart your EcoStruxure Machine Expert software and run the Build (compile) process again.
