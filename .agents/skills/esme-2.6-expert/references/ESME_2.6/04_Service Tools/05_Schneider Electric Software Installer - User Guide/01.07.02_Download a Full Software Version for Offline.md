# Download a Full Software Version for Offline Installation

## Overview

Using Download full software version you can download a software version to your PC for offline installation.

## Download a Full Software Version for Offline Installation

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. |
| 2 | Select Download full software version by clicking the tool icon in the upper right corner.  **Result**: The installer searches for suitable products to download. |
| 3 | In the Select a product area, select the following options:   * Product: Select the product to be downloaded to your PC. * Version: Select the version of product to be downloaded to your PC. |
| 4 | Click on Change destination folder, and select the folder on your PC to which the product version will be downloaded. |
| 5 | Click Download full software.  **Result**: The files necessary to complete an installation of the product are downloaded to the selected destination folder. |
