# Export/Import Installation Configuration

## Overview

You can use export and import of an installation configuration, for example, to apply the same configuration to different software instances.

## Export

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. |
| 2 | Select the option Modify installed Software or the option Install new software. |
| 3 | In the Select a product area, select the following options:   * Product: Select your product. * Version: Select the version of your product. |
| 4 | In the Choose an action area, select Customize Version. |
| 5 | Select Source: Online and click Next.  NOTE: You can also use one of the other [two options](D-SE-0088006.html#D-SE-0088006__D-SE-0088006.4), but if there is an Internet connection available, using the Source: Online downloads the software directly from the Schneider Electric server. |
| 6 | Click the tool icon in the upper right corner, and select Export installation configuration from the menu.  **Result**: A dialog box to select the export folder is displayed. |
| 7 | Select the export folder and click OK.  **Result**: The installation configuration is exported as a *\*.json* file for each installed product and version. Example: *ComponentList\_ESME\_1.3.0.0\_20.04.08.02.json* and *ComponentList\_ESME\_1.2.0.0\_20.02.07.01.json* if EcoStruxure Machine Expert V1.2 and V1.3 are installed. |

## Import

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. |
| 2 | Select the option Modify installed Software or the option Install new software. |
| 3 | In the Select a product area, select the following options:   * Product: Select your product. * Version: Select the version of your product. |
| 4 | In the Choose an action area, select Customize Version. |
| 5 | Select Source: Online and click Next.  NOTE: You can also use one of the other [two options](D-SE-0088006.html#D-SE-0088006__D-SE-0088006.4), but if there is an Internet connection available, using the Source: Online downloads the software directly from the Schneider Electric server. |
| 6 | Click the tool icon in the upper right corner, and select Import installation configuration from the menu.  **Result**: A dialog box to select the file to be imported is displayed. |
| 7 | Select the file to be imported and click OK.  **Result**: The installation configuration is imported to your software instance. |
