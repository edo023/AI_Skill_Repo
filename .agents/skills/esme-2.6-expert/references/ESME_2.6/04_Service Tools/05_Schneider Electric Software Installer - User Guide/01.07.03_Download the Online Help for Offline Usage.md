# Download the Online Help for Offline Usage

## Download the Online Help for Offline Usage

To download the online help to your PC for offline usage, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the What would you like to do? - Select a task view is displayed. |
| 2 | Select the option Modify installed software. |
| 3 | In the Modifying an existing software - Select a product view, select the suitable software version from the Version list. |
| 4 | From the Choose an action section, select the option Customize version and click the Next button. |
| 5 | In the next view, select the Help tab. |
| 6 | From the Help tab, click the Open help management tool button.  **Result**: The Schneider Electric Help Manager opens. |

The Schneider Electric Help Manager allows you to select specific software documents or a bundle of documents that is related to a topic (such as service tools or controllers) and the language or languages of your choice for download. After successful download, the downloaded sections of the online help are available for your reference even if no Internet connection is available.
