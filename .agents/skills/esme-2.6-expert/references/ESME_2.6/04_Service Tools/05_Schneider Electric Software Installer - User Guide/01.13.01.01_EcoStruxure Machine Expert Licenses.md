# EcoStruxure Machine Expert Licenses

## Overview

Various license types of EcoStruxure Machine Expert are available to meet different customer application requirements. They provide a different set of functions and differ in the number of users that can work simultaneously (seats).

* Free License

  The Free License is available for free on a single seat after registration. It provides limited access to EcoStruxure Machine Expert functions to allow you to evaluate the software as indicated in the table in section [*Overview of Licenses and Supported Features*](#D-SE-0075111__D-SE-0075111.29) for the Free License.
* Standard License

  The Standard License is available after registration and activation for up to 10 seats. It provides access to various EcoStruxure Machine Expert functions. Some Add-ons are available separately to provide a high degree of flexibility.
* Trial License

  The Trial License provides the functions listed in the table in section [*Overview of Licenses and Supported Features*](#D-SE-0075111__D-SE-0075111.29) for 42 days. After this trial period has expired, you have to upgrade or activate the software. Otherwise, limited functions are available as indicated in the table in section [*Overview of Licenses and Supported Features*](#D-SE-0075111__D-SE-0075111.29) for the Free License.
* Professional License

  The Professional License is available after registration and activation as a site license for up to 100 seats. It provides access to all EcoStruxure Machine Expert functions, and to all Add-ons.
* University License

  The University License is especially dedicated to universities. After registration and activation, all EcoStruxure Machine Expert functions, and some Add-ons are available for free to students.

NOTE: Software installation is independent of the activation of software product licenses.

## Information on License Status

You can find information about your license status (temporary or permanent license)

* At the top right of the Get started dialog box.
* Under Logic Builder > Help > About > License.

## Activating the License

You can activate the license of your software product to get a permanent license.

License activation can be performed using the following software:

| Software | Location |
| --- | --- |
| Schneider Electric License Manager | Start > Programs > Schneider Electric > License Manager > License Manager |
| EcoStruxure Machine Expert Logic Builder | Menu Help > Manage Licenses... |

If the trial period of your software product expires before you have activated a license, a free license is offered when you launch the Logic Builder. A dialog box is displayed, allowing you to perform a registration to enable the free license.

After you have performed the registration procedure, restart the Logic Builder to use EcoStruxure Machine Expert with the free license that supports the functions as indicated in the table [*Overview of Licenses and Supported Features*](#D-SE-0075111__D-SE-0075111.29). If you intend to use an unsupported feature, a dialog box is displayed offering to open the Manage licenses dialog box for you to activate the corresponding license upon clicking the Yes button.

## Schneider Electric License Manager

You can start the Schneider Electric License Manager

* From the dialog box that reminds you to activate the license.
* From the PC via Start > Programs > Schneider Electric > License Manager > License Manager.

You can use the Schneider Electric License Manager to activate (update, return, repair) your Schneider Electric software products licenses.

Further, you can display and manage the licenses activated with the Schneider Electric License Manager.

For detailed information, refer to the online help of the Schneider Electric License Manager.

## Activation Principle and Methods

After ordering and paying for a product, you receive an activation ID. With this activation ID, you can send an activation request to Schneider Electric. Upon confirmation, an activation response from Schneider Electric is sent (see the table below).

The data in the activation response is validated by the Schneider Electric License Manager and added to the trusted storage.

Then, you can use the software product within the scope of your license.

There are different ways to exchange the activation request and activation response:

| Activation method | Internet access required | Format of activation request / response |
| --- | --- | --- |
| By web | Yes | The Schneider Electric License Manager generates the activation request and transmits the data through an Internet connection to the Schneider Electric License Server. The activation response is automatically received and processed by the Schneider Electric License Manager. |
| By web portal  By e-mail | These options are only available if no Internet connection is detected on the PC. Use these options if an alternate device having an Internet connection can be used. | The Schneider Electric License Manager generates an XML request file. The XML request file needs to be submitted to Schneider Electric through the Software Licensing Web Portal or by email. In response, an XML response file is provided for download in the Software Licensing Web Portal or sent by email. To complete the activation, the XML response file needs to be loaded in the Schneider Electric License Manager. |
| By phone | This option is only available if no Internet connection is detected. Use this option if an alternate device having an Internet connection is not available. | The Schneider Electric License Manager generates a Short code request. The Short code request is a string of characters and digits which needs to be given to the operator during a phone call with the Software Registration Center. During the call, another Short code is generated, this time by the operator, and given to you to complete the registration. To complete the activation, the Short code response needs to be entered in the Schneider Electric License Manager. |

For detailed information, refer to the online help of the Schneider Electric License Manager.
