# Modify Installed Software

## Overview

Using Modify Installed Software you can adapt individual components of your installed software.

## Modify Installed Software

| Step | Action |
| --- | --- |
| 1 | After [starting the installer](D-SE-0087998.html#D-SE-0087998__D-SE-0087998.3), the Select a Task dialog box is displayed. |
| 2 | Select the option Modify installed software. |
| 3 | In the Select a product area, select the following options:   * Product: Select your installed product. * Version: Select the version of your installed product. |
| 4 | Select Customize Version. |
| 5 | Select Source: Online and click Next.  NOTE: You can also use one of the other [two options](D-SE-0088006.html#D-SE-0088006__D-SE-0088006.4), but if there is an Internet connection available, using the Source: Online downloads the software directly from the Schneider Electric server. |
| 6 | Proceed as described under [Install New Software (Online)](D-SE-0088006.html#D-SE-0088006__D-SE-0088006.2). |

## Modify the Product Language

To modify the installed languages, proceed as described in the *Modify Installed Software* paragraph and then select the Languages tab.

The Languages tab displays the languages available for the Schneider Electric Software Installer. As English is the default language it is not available for selection. The languages that are already installed are displayed with the respective check box selected.

To add one or more languages, select the corresponding check boxes and click Next.

To remove one or more languages, clear the corresponding check boxes and click Next.

For the remaining steps, refer to [Install New Software (Online)](D-SE-0088006.html#D-SE-0088006__D-SE-0088006.2).

## Modify Installed Components

To modify the installed components, proceed as described in the *Modify Installed Software* paragraph and then select the Features tab to see how many components you have installed per feature. The numbers in brackets behind each feature indicate the total number of components of each feature compared to the components you have installed.

To modify the installed components, select the Individual components tab and select or clear the check boxes of the individual components you want to add to your installation or you want to remove. Click Next.

For the remaining steps, refer to [Install New Software (Online)](D-SE-0088006.html#D-SE-0088006__D-SE-0088006.2).
