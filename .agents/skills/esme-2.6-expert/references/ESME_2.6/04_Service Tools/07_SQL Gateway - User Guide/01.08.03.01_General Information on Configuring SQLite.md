# General Information on Configuring SQLite Connections

## Overview

This chapter provides information on how to configure a connection to a file-based SQLite database.

## SQLite Software Components

The following SQL components are required to establish a connection to an SQLite database:

* SQL client for SQLite (not supplied by Schneider Electric)
* SQLite database created with the SQL client

## Supported Data Types

The following data types are supported for SQLite:

* INTEGER
* TEXT
* REAL
* NUMERIC

## General Considerations for SQLite Connections

It is a best practice to save the SQLite database file locally on the PC on which the SQL Gateway is installed.
