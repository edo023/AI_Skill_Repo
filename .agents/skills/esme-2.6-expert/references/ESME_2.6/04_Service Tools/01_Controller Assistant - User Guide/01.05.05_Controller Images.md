# Controller Images

## Definition

The sum of the controller files managed in Controller Assistant are designated as image. The Controller Assistant can read in an image from different sources such as a controller, a flash disk, or a file. The last image read in from Controller Assistant is designated as active image.

NOTE: The Controller Assistant can only manage one active image at a time.

After the Controller Assistant has read in an image, it can write the same to different destinations such as a controller, a flash disk, or a file.

The image itself comprises the files that are typically located on the flash disk of a controller. It is comprised in the firmware, the EcoStruxure Machine Expert/EcoStruxure Automation Expert - Motion project and the controller data. The files contained in the image can be displayed and edited. An image is stored as a compressed file with the file extension *bpd*.
