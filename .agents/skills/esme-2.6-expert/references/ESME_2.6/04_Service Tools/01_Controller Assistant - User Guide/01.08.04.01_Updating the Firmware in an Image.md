# Updating the Firmware in an Image

## Overview

Clicking the Update firmware ... button in the Process image / Create image new dialog starts the firmware update procedure for an image.

## Step for Selecting the Update Mode and Compatibility Options

In this step, you specify the update mode and compatibility options.

Items in the dialog:

| Item | Description |
| --- | --- |
| SERCOS device family only | NOTE: This option is only available if you have previously selected a PacDrive LMC controller.  Select this option to update the firmware of a Sercos device family in the image.  Next steps with this selection:  * Select the firmware versions (as described in [Step for Selecting Devices (Only for PacDrive LMC Controllers)](D-SE-0037693.html#D-SE-0037693__Step2LMCControllers-9831418E) of the procedure for updating the firmware outside of an image). * Select the communication settings for the controller (as described in [Step for Communication Settings](D-SE-0037693.html#D-SE-0037693__D-SE-0037693.6) of the procedure for updating the firmware outside of an image).  Then click the Execute button to complete the firmware update in the image. |
| Compatible | If you select this option, only compatible firmware versions for the controller are provided in the next step.  Next steps with this selection:  * Select the controller firmware version (as described in [Step for Selecting the Controller and Firmware](D-SE-0037693.html#D-SE-0037693__D-SE-0037693.5) of the procedure for updating the firmware outside of an image). * Select the communication settings for the controller (as described in [Step for Communication Settings](D-SE-0037693.html#D-SE-0037693__D-SE-0037693.6) of the procedure for updating the firmware outside of an image).  Then click the Execute button to complete the firmware update in the image. |
| General | If you select this option, multiple controller types and firmware versions are provided for selection in the next step.  Next steps with this selection:  * Select the controller firmware version (as described in [Step for Selecting the Controller and Firmware](D-SE-0037693.html#D-SE-0037693__D-SE-0037693.5) of the procedure for updating the firmware outside of an image). * Select the communication settings for the controller (as described in [Step for Communication Settings](D-SE-0037693.html#D-SE-0037693__D-SE-0037693.6) of the procedure for updating the firmware outside of an image).  Then click the Execute button to complete the firmware update in the image. |

Carefully manage the IP addresses because each device on the network requires a unique address. Having multiple devices with the same IP address can cause unintended operation of your network and associated equipment.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  * Verify that there is only one main controller configured on the network or remote link. * Verify that all devices have unique addresses. * Obtain your IP address from your system administrator. * Confirm that the IP address of the device is unique before placing the system into service. * Do not assign the same IP address to any other equipment on the network. * Update the IP address after cloning any application that includes Ethernet communications to a unique address.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |
