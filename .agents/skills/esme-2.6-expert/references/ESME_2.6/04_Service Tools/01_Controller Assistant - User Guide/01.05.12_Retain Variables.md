# Retain Variables

## General

With Controller Assistant you can store Retain Variables in an image.

Restrictions:

* Retain Variables can only be stored in an image in online mode.
* An image containing Retain Variables cannot be written to a flash card.

## Reading an Image

When an image is read, the Retain Variables can only be stored to an image if the application on the controller is stopped.

If the application is running, Controller Assistant displays a message that allows you to confirm to stop the application.

After the image has been created, and if the application was stopped during reading, Controller Assistant displays a message that allows you to confirm to restart the application.

## Writing an Image

If the image you want to write contains Retain Variables, Controller Assistant displays a message that allows you to confirm to overwrite the Retain Variables in the controller.

## Activation/Deactivation

Reading and writing of Retain Variables can be activated and deactivated in the configuration file ControllerAssistant.exe.config. The configuration file is stored in the same folder as the file ControllerAssistant.exe.

Setting: `<add key="SupportRetainVariables" value="true" />`

If the attribute `value` is set to `true`, reading and writing of Retain Variables is activated.

If the attribute `value` is set to `false`, reading and writing of Retain Variables is deactivated.

## Error Messages

If an error is detected during reading or writing a Retain Variable, an error messages is generated. The error code of the error message refers to the IEC functions RetainImageLoad or RetainImageSave.
