# Signal Off

## Overview

The identification of a controller through an optical signal or via a series of acoustic signals (refer to the [*Signal On* chapter](D-SE-0032049.html#D-SE-0032049)) is stopped.

## Call

```
<ToolPath> -NetmanageCommand SignalOff <MacAddress>
```

## Parameter Description

| Call Parameters | Content |
| --- | --- |
| `<ToolPath>` | Path of the tool to be executed that supports the Network Device Identification command-line call, see example underneath this table. |
| `<MacAddress>` | MAC address of the desired controller |

## Example

Entry

```
"c:\Programs\Schneider Electric\ControllerAssistant\ControllerAssistant.exe" -NetmanageCommand SignalOff 00:04:17:05:00:78
```

## Result

By a successful processing of the call 0, otherwise 1.

EIO0000001671.07
