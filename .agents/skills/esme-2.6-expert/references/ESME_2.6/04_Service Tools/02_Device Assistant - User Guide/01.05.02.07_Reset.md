# Reset

## Overview

The  Reset feature reboots the device. It is used in the Device Assistant, primarily, to apply a newly downloaded firmware to a device.

Right-click the device list and select Reset to restart the selected device.

NOTE: The restart is executed without confirmation prompt.

By selecting several devices in the device list, you can reset several devices at once. Select more than one device by holding  Ctrl or Shift when clicking.
