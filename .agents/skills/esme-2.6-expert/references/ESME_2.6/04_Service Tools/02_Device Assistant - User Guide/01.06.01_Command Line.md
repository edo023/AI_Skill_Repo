# Command Line

## Functions of the Command Line

Click Start > Run  > Command Line to retrieve additional functions by using the command line of the computer.

The format of the command line is:

`DeviceAssistant.exe <TFTP-Command> <IP-Adress> [<Additional Parameter 1>... [<Additional Parameter n>]]`

The following functions are supported:

| Function | Description |
| --- | --- |
| Update | Executes a firmware update from a file (extension \*.DAT, \*.fw, \*.seco, \*sefirmware; from V1.6.0.0 or later).  **Example:** `DeviceAssistant.exe ElauTftpCommand_Update 10.128.129.65 "c:\FIRM_V001200.dat` |
| Reset | Unlike the Reset command in the graphical user interface, this function is only supported by some devices like Lexium 62.  Resets the device (Sercos slave) (restart). The following example calls the reset after 5000 milliseconds.  **Example:** `DeviceAssistant.exe ElauTftpCommand_Reset 10.128.129.65 5000` |

## Error Levels

If the execution was successful, the error level 0 is returned during batch operation. If an error occurred, the returned value is greater than 0.

Possible error levels are:

| Error Level | Description |
| --- | --- |
| 0 | Command execution successful |
| 1 | Command execution unsuccessful |
| 2 | Invalid command |
| 3 | Invalid argument number |
