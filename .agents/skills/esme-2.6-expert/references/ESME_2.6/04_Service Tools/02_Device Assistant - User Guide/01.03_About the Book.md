# About the Book

## Document Scope

This user guide describes the Device Assistant. The Device Assistant is a software tool that allows you to scan Sercos devices and get information about device type, communication settings, and firmware version.

The Device Assistant performs the following main functions:

* Scan Sercos devices and possible submodules. This way you get information about the device type, communication settings, and the firmware version.
* Provide a signal function to identify the correct device.
* Set communication settings.
* Prepare a device for the Sercos bus if it was exchanged in a service case, for example set the application type or Sercos address.
* Update firmware of supported devices.
