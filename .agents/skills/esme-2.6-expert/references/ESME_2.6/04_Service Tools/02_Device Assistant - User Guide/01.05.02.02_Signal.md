# Signal

## Overview

Right-click the desired device (Sercos slaves) and select Signal .

Use the Signal  function to verify, whether the correct device has been selected.

The status LEDs of the selected device light up green for a certain period. The exact duration depends on the individual Sercos slaves.

NOTE: By selecting several devices in the device list, the signalization can be executed for several devices at a time. Select more than one device by holding  Ctrl or Shift when clicking.

For further information, refer to [Brief instruction](D-SE-0059201.html#D-SE-0059201) and [Command line](D-SE-0059202.html#D-SE-0059202).
