# Panel Mounting Procedure: : (Except HMIGTO2315/HMIGTO5315/HMIGTO6315)

Panel Mounting Procedure: : (Except HMIGTO2315/HMIGTO5315/HMIGTO6315)

|  |
| --- |
| NOTICE |
| PANEL UNSTEADY WHEN UNSECURED |
| Keep HMIGTO stabilized in the panel-cut while you are installing or removing the screw fasteners. |
| Failure to follow these instructions can result in equipment damage. |

| Step | Action |
| --- | --- |
| 1 | Place the panel on a clean and level surface with the display facing downward. |
| 2 | Check that the panel's gasket is seated securely into the gasket's groove, which runs around the perimeter of the panel frame. |
| 3 | Cut a hole in the installation panel as defined by the HMIGTO's panel cutout dimensions.  [HMIGTO1300/HMIGTO1310](../Kanpai_-_CH_Specifications/Kanpai_-_CH_Specifications-11.htm#XREF_D_SE_0059305_5)  [HMIGTO2300/HMIGTO2310](../Kanpai_-_CH_Specifications/Kanpai_-_CH_Specifications-21.htm#XREF_D_SE_0059296_5)  [HMIGTO3510/HMIGTO4310](../Kanpai_-_CH_Specifications/Kanpai_-_CH_Specifications-31.htm#XREF_D_SE_0059314_5)  [HMIGTO5310](../Kanpai_-_CH_Specifications/Kanpai_-_CH_Specifications-41.htm#XREF_D_SE_0059323_5)  [HMIGTO6310](../Kanpai_-_CH_Specifications/Kanpai_-_CH_Specifications-51.htm#XREF_D_SE_0059332_5) |
| 4 | Insert the HMIGTO into the panel-cut. |
| 5 | Insert the installation fasteners into the HMIGTO's insertion slots on the top and bottom sides (left and right sides for the HMIGTO1300/1310). Slide the fasteners to the back. If the fasteners are not correctly attached, the HMIGTO may shift or fall out.  G-SE-0017237.1.gif-high.gif      HMIGTO2310  G-SE-0013878.2.gif-high.gif      HMIGTO1310  G-SE-0013877.2.gif-high.gif |
| 6 | Insert each of the fasteners shown below. Make sure that you pull the fastener back until it is flush with the rear of the attachment hole insert.  G-SE-0017235.1.gif-high.gif |
| 7 | Use a Phillips screwdriver to tighten each fastener screw and secure the panel in place. The necessary torque is 0.5 Nm (4.4 lb•in).  G-SE-0017238.1.gif-high.gif |

|  |
| --- |
| NOTICE |
| BROKEN ENCLOSURE |
| oDo not exert more than 0.5 Nm (4.4 lb•in) of torque when tightening the fastener’s screws.  oUse on flat surface of a Type 1, Type 4X (Indoor Use Only) or Type 13 Enclosure. |
| Failure to follow these instructions can result in equipment damage. |
