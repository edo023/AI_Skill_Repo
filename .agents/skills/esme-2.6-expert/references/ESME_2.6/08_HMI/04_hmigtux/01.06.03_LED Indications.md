LED Indications

LED Indications

Status LED

| Color | Indicator | HMIG3X |
| --- | --- | --- |
| Green | ON | In operation |
| Orange | Flashing | Software starting up |
| Red | ON | Power is ON. |
| Red/Green | Alternating | Display Module connection error. |
| Orange/Red | Alternating | SD Card boot error. |
| - | OFF | Power is OFF. |

Card Access LED

| Color | Indicator | HMIG3X |
| --- | --- | --- |
| Green | ON | Storage card is inserted. |
| - | OFF | Storage card is not inserted or is not detected. |

COM1 LED

| Color | Indicator | Description |
| --- | --- | --- |
| Yellow | ON | Data transmission is in progress. |
| - | OFF | No data transmission. |
