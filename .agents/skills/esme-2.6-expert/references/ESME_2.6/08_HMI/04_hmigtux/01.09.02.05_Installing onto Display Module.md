| Step | Action |
| --- | --- |
| 1 | Pull the LOCK forward on the top of the Box Module to release the lock as illustrated.  G-SE-0070102.1.gif-high.gif  1 LOCK |
| 2 | Insert the protruding points on the bottom left and right of the Box Module into the two holes on the back of the Display Module to attach the Box Module.  G-SE-0070103.1.gif-high.gif  2 Protruding points  3 Box Module  4 Holes for insertion  5 Display Module |
| 3 | Fully push on the LOCK on the top of the Box Module to mount the Box Module on the Display Module  G-SE-0070104.1.gif-high.gif  1 LOCK |
