Installation with Spring Clips

|  |
| --- |
| Caution_Color.gifCAUTION |

|  |
| --- |
| Caution_Color.gifCAUTION |
| TERMINAL UNSTEADY WHEN UNSECURED |
| Keep terminal stabilized in the panel cut-out while you are installing or removing the spring clips. |
| Failure to follow these instructions can result in injury or equipment damage. |

NOTE: Use the installation gasket to absorb vibration and to repel water.

| Step | Action |
| --- | --- |
| 1 | Place the unit on a clean and level surface with the display panel facing downward. |
| 2 | Check that the unit’s [installation gasket](../Lynx_Maintenance_Intro/Lynx_Maintenance_Intro-2.htm#XREF_D_SA_0021434_1) is seated securely into the gasket’s groove, which runs around the perimeter of the panel’s frame. |
| 3 | Create the correct sized opening required to install the unit, using the [installation dimensions](../Lynx_Specifications_Intro/Lynx_Specifications_Intro-29.htm#XREF_D_SA_0021500_1). |
| 4 | Insert the unit into the panel cut-out. |
| 5 | Adjust the spring clips for the panel thickness:  o1.5 mm (0.06 in.) ≤ panel thickness ≤ 4 mm (0.16 in.) (position 1),  o4 mm (0.16 in.) ≤ panel thickness ≤ 6 (0.23 in.) mm (position 2).  G-SA-0033874.1.gif-high.gif |
| 6 | Lock the spring clips by pressing simultaneously on the top and the bottom with two fingers.  G-SA-0033875.1.gif-high.gif |
| 7 | To remove the spring clips:  oUnlock the spring clips by pressing on the back:  G-SA-0033876.1.gif-high.gif |

NOTE: Spring clips fasteners are required for NEMA Type 4 protection for XBT GK series.
