# Serial Interface COM2

Serial Interface COM2

The following table lists the serial interface COM2 of XBT GT2000/4000/5000/6000/7000 and XBT GK series units.

| Interface | Description |
| --- | --- |
| Serial interface COM2 RJ45 | |
| Asynchronous Transmission | RS-485 |
| Data Length | 7 or 8 bits |
| Stop Bit | 1 or 2 bits |
| Parity | None, odd or even |
| Data Transmission Speed | 2,400 bps to 187.5Kbps |
