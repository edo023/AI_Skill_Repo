# Changing Network Settings

Changing Network Settings

| Step | Action |
| --- | --- |
| 1 | In the Settings menu, touch the Offline tab. |
| 2 | Touch the Network icon. |
| 3 | Touch any of the three fields (IP Address, Subnet Mask, or Default Gateway) and a keypad will appear. |
| 4 | Enter the desired network addresses. |

NOTE: To apply changes to the network settings restart the unit by touching To Run Mode and return to the application.
