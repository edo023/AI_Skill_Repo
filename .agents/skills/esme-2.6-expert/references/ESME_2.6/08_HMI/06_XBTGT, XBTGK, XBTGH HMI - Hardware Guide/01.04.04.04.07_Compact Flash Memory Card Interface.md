# Compact Flash Memory Card Interface

Compact Flash Memory Card Interface

The Compact Flash memory card is used for XBT GT 2000 series and higher except XBT GT2110 units and all XBT GK and XBT GH models.

This slot accepts any of the following Compact Flash (CF) memory cards:

oXBT ZGM128 (128 Mb)

oXBT ZGM256 (256 Mb)

oMPC YN0 0CFE 00N (512 Mb)

oMPC YN0 0CF10 0N (1 Gb)
