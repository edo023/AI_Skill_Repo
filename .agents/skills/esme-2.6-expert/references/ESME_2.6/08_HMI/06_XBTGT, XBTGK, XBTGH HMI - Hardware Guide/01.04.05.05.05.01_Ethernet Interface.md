# Ethernet Interface

Ethernet Interface

The Ethernet Interface is used for the following units:

oXBT GT1135

oXBT GT1335

oXBT GT2130

oXBT GT2330

oXBT GT2430

oXBT GT2930

oXBT GT4000 series

oXBT GT5000 series

oXBT GT6000 series

oXBT GT7000 series

oXBT GK2330

oXBT GK5330

oXBT GH2460

This interface complies with the IEEE 802.3 standard for Ethernet:

o10Base-T/100Base-TX (XBT GT1005/2000/4000/5000/6000/7000, XBT GK, and XBT GH series) connections.

The following table describes the LED colors and status for all target machines except the XBT GT1135, XBT GT1335 and XBT GT6330:

| LED | Contents |
| --- | --- |
| Orange | oWhen the power supply is ON: LED lights up.  oWhen sending or receiving: LED blinks. |
| Green | When linking: LED lights up. |
