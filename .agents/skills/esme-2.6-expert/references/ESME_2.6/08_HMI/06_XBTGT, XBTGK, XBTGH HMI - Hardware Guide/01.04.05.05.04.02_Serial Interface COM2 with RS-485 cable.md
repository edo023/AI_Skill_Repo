# Serial Interface COM2 with RS-485 cable

Serial Interface COM2 with RS-485 cable

This interface is used to connect a RS-485 serial cable to a XBT GT2000/4000/5000/6000/7000 or XBT GK series unit. RJ45 8-pin plug connector is used.

| Pin Connection | Pin | Signal Name | Direction | Meaning |
| --- | --- | --- | --- | --- |
| G-SA-0033762.1.gif-high.gif | 1 | Not connected | - | - |
| 2 | Not connected | - | - |
| 3 | Not connected | - | - |
| 4 | D1 | Output/Input | Transfer Data (RS-485) |
| 5 | D0 | Output/Input | Transfer Data (RS-485) |
| 6 | RTS | Output | Request To Send |
| 7 | Not connected | - | - |
| 8 | SG | - | Signal Ground |
