# Inserting the SD Card

![](images/logo.png)

Inserting the SD Card

NOTE: As shown in the image below (example on the left-hand side), you can set the Write-Control Tab to prevent write operations to the SD Card. Push the tab up, as shown in the example on the right-hand side, to release the lock and enable writing to the SD Card. Before using a commercial-type SD Card, read the manufacturer's instructions.

![G-SE-0015007.1.gif-high.gif](images/G-SE-0015007.1.gif-high.gif)

| Step | Action |
| --- | --- |
| 1 | Push down the tab on the SD Card/battery cover, then pull open the cover.  G-SE-0050400.1.gif-high.gif    1   SD Card/battery cover |
| 2 | Insert the SD Card into the SD Card Slot with the front face of the SD Card facing up and push until you hear it “click”.  G-SE-0050401.1.gif-high.gif    1   SD Card |
| 3 | Close the SD Card/battery cover. |
