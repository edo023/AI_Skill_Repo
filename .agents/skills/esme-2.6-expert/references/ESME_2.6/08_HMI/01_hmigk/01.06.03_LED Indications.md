# LED Indications

![](images/logo.png)

LED Indications

Status LED

| Color | Indicator | Description |
| --- | --- | --- |
| Green | ON | In operation. |
| Orange | Flashing | Software starting up. |
| Red | ON | Power is ON. |
| — | OFF | Power is OFF. |

SD Card Access LED

| Color | Indicator | Description |
| --- | --- | --- |
| Green | ON | SD Card is inserted and available to use. |
| Flashing | SD Card is inserted and being accessed. |
| — | OFF | SD Card is not inserted or accessed. |

COM2 LED

| Color | Indicator | Description |
| --- | --- | --- |
| Yellow | ON | Data transmission is in progress. |
| OFF | No data transmission. |
