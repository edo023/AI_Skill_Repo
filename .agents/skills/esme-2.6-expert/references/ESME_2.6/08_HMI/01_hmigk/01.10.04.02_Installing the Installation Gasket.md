# Installing the Installation Gasket

Installing the Installation Gasket

| Stage | Description |
| --- | --- |
| 1 | Place the panel on a flat, level surface, with the display face pointing down. |
| 2 | Remove the gasket from the panel. |
| 3 | Attach the new gasket to the panel. Position the gasket in the installation groove so that the gasket seam is at the bottom of the panel. First, insert the gasket into the 4 corners, in the order shown in the image below. Then, insert the rest of the gasket into the installation groove.  NOTE: When using a tool to insert the gasket, make sure the tool does not catch the rubber gasket and cause a tear.  G-SE-0050403.1.gif-high.gif    A   Gasket seam |
| 4 | The upper surface of the gasket should protrude approximately 2.0 mm (0.08 in) from the groove. Check the gasket is inserted correctly before installing this product into a panel.  G-SE-0050404.1.gif-high.gif |

The gasket must be inserted correctly into the groove for moisture resistance for this product.
