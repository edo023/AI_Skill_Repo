# USB Type Mini B

![](images/logo.png)

USB Type Mini B

Introduction

When using a USB device, you can attach a USB holder to the USB interface on the side of the unit to prevent the USB cable from being disconnected.

Attaching the USB Holder

| Step | Action |
| --- | --- |
| 1 | Insert the USB cable into the USB host interface.  G-SE-0011783.1.gif-high.gif    1   USB cable |
| 2 | Attach the USB holder to fix the USB cable in place.  G-SE-0011784.1.gif-high.gif    1   USB holder  2   USB cable |
| 3 | Insert the USB holder into the tab.  G-SE-0013065.1.gif-high.gif    1   USB holder  2   USB cable |

Removing the USB Holder

Squeeze the tab of the USB holder and then remove the USB holder.
