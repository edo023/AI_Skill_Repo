Interface Specifications

Introduction

All HMIGXO panels are provided with serial and USB Interfaces.

Serial Interface COM1

The following table describes the COM1 serial interface:

| Interface | Description |
| --- | --- |
| Serial interface D-Sub9 | |
| Asynchronous transmission | RS-232C |
| Data length | 7 or 8 bits |
| Stop bit | 1 or 2 bits |
| Parity | none, odd or even |
| Data transmission speed | 2,400...115,200 bps |

Serial Interface COM2

The following table describes the COM2 serial interface:

| Interface | Description |
| --- | --- |
| Serial interface D-Sub9 | |
| Asynchronous transmission | RS-422/485 |
| Data length | 7 or 8 bits |
| Stop bit | 1 or 2 bits |
| Parity | none, odd or even |
| Data transmission speed | 2,400 bps to 187.5 Kbps |

USB Interface (USB Peripherals)

The following table describes the USB type A interface:

| Interface | | Description |
| --- | --- | --- |
| Host interface | | |
| Transmission speed | high speed | 480 Mbps |
| full speed | 12 Mbps |
| low speed | 1.5 Mbps |
| Maximum current supplied | | 500 mA |

USB Interface (Application Download)

USB mini B V2.0 type connector is used for application download.
