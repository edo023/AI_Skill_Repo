# USB Holder Type A

![](images/logo.png)

USB Holder Type A

Introduction

When using a USB device, attaching a USB holder to the USB interface on the side of the panel helps prevent the USB cable from being disconnected.

Attaching the USB Holder

| Step | Action |
| --- | --- |
| 1 | Attach the USB holder to the USB Host Interface on the main panel. Hook the upper pick of the USB holder to the attachment hole of the main panel, and insert the lower pick as shown below to fix the USB holder.  G-SE-0012174.1.gif-high.gif    1   USB holder |
| 2 | Insert the USB cable into the USB host interface.  G-SE-0012175.1.gif-high.gif    1   USB holder  2   USB cable |
| 3 | Attach the USB cover to fix the USB cable in place. Insert the USB cover into the tab of the USB holder.  G-SE-0012176.1.gif-high.gif    1   USB holder  2   USB cover |

Removing the USB Holder

Lift up the tab of the USB holder and then remove the USB cover.
