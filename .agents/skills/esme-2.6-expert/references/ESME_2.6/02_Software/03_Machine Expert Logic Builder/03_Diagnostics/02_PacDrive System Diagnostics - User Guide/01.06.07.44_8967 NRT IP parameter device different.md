# 8967 NRT IP parameter device different

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| Cause | Solution |
| --- | --- |
| * The logical subdevices (for example, the two drive objects of the double drive) have different parameter values for IP addressing. * The message logger entry is triggered by one of the two logical devices in the PLC configuration. * In the field Ext. Diagnosis, the name of the second device of the double drive is specified. | * Verify whether the two drive objects in the PLC configuration contain the same values for IPConfigMode and correct if necessary. * If IPConfigMode of the two drive objects equals `Manual IP addressing (software) / 2`, set the parameters IPAddress, Subnetmask and Gateway likewise. |

NOTE: The unsuccessful configuration of a double drive slave may be the reason why the slave is not being configured.
