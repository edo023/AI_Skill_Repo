# 8507 Sercos write cycle overflow

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The transfer of the target position to the Sercos bus proceeds too late. The last target positions may have been transferred twice. This results in a velocity jump in the drives (refer to parameter RTPWriteRes).

| Cause | Solution |
| --- | --- |
| The real-time process is overloaded. | Reduce the number of objects with real-time tasks (for example, virtual axes, virtual encoders, logical encoders, sum encoders, etc.). |
