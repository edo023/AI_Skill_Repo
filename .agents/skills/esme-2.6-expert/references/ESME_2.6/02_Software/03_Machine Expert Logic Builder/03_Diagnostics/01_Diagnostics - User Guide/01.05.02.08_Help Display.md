# Help Display

## Overview

A help for the usage of the call parameters is displayed.

## Call

```
<ToolPath> -NetmanageCommand <Help>
```

## Parameter Description

| Call Parameters | Content |
| --- | --- |
| `<ToolPath>` | Path of the tool to be executed that supports the Network Device Identification command-line call, see example underneath this table. |
| `<Help>` | "?" (enter a question mark to get the usage information) |

## Example

Entry

```
"c:\Program Files (x86)\Schneider Electric\Tools\Diagnostics\Diagnostics.exe" -NetmanageCommand ?
```

## Result

By a successful processing of the call 0, otherwise 1.
