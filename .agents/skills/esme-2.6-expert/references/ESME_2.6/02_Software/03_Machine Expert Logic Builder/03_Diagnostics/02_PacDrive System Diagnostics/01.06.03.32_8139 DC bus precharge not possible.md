# 8139 DC bus precharge not possible

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | - | - |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | BP1 | BP1 |

In the Lexium 62 Power Supply or Lexium 52, the detected error message is triggered if the pre-charge of the DC bus could not be completed after a predefined time, that is internally calculated with data from the device type plate. If this error is triggered, then the device has to be reset or turned off /on again.

| Cause | Solution |
| --- | --- |
| An external load is connected to the DC bus. | Remove external load. |
| The parameter MainsVoltageMode is configured in an incorrect way. | Verify the parameter MainsVoltageMode and correct it, if necessary. |
| During the pre-charge, the mains phases were disconnected temporarily. | Verify if the cable for the mains connection is connected to the device correctly. |
