# 8792 SoSafe configuration mismatch

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

This diagnostic message is reported if a mismatch of parameters of the DSM objects is detected between EcoStruxure Machine Expert and SoSafe.

For rotary drives (for example, Lexium 62 ILM, Lexium 62DxS) the following parameters are verified:

* EncoderType = `Rotary Encoder / 0`
* EncoderResolutionRotary
* FeedConstantRotary
* GearIn
* GearOut

For linear drives (for example, Lexium 62DxSLinear) the following parameters are verified:

* EncoderType = `Linear Encoder / 1`
* EncoderResolutionLinear
* FeedConstantLinear

DiagExtMsg:

* Shows the parameter name of the configuration mismatch.
