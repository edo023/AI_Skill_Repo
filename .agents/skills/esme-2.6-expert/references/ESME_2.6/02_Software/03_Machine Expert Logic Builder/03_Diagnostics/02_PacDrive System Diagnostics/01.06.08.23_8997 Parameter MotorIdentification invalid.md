# 8997 Parameter MotorIdentification invalid

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

The parameter MotorIdentification has an invalid value.

| Cause | Solution |
| --- | --- |
| The parameter is set to `Motor with type plate / 0`, but the electronic motor nameplate is saved in the servo amplifier. | * Verify and correct if necessary the axis assignment in the PLC configuration. * Set parameter to `Motor without type plate / 1`. |
| The parameter is set to `Motor without type plate / 1` but the electronic motor nameplate is saved in the encoder EEPROM of the motor. | * Verify whether the Encoder is connected and correct it if necessary. * Verify and correct if necessary the axis assignment in the PLC configuration. * Set parameter to `Motor with type plate / 0`. |
