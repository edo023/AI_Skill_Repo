# 8964 NRT IPAddressRangeDynamic used up

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| Cause | Solution |
| --- | --- |
| The automatic IP address allocation system cannot allocate any more IP addresses from IPAddressRangeDynamic. | Verify whether the calculated address range IPAddressRangeDynamic is large enough to provide the slaves with IP addresses using IPConfigMode = `Automatic IP addressing / 0` and correct if necessary. |
