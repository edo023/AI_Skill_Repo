# 8186 DC bus voltage high

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | D | E |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | D | E |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | D | – |

* The DC bus voltage is too high. A further increase can lead to the diagnostic message [8108 DC bus overvoltage](D-SE-0063850.html#D-SE-0063850).

| Device | Thresholds |
| --- | --- |
| Lexium 62 Power Supply | * 410 Vdc if BrakingResistorMode is set to `230 V mode / 0`. * 850 Vdc if BrakingResistorMode is set to `400 V mode / 1` (default value). |

The voltage threshold for this diagnostic message is only dependent on the parameter BrakingResistorMode. The parameter MainsVoltageMode has no influence on this threshold.

For information on causes and measures, see [8108 DC bus overvoltage](D-SE-0063850.html#D-SE-0063850).

* The DC bus voltage of the multi carrier segment is too high (Udc > 60 Vdc).

| For the Multi carrier transport system: | |
| --- | --- |
| Cause | Solution |
| The settings of the rotary switches on the multi carrier connection module are incorrect. | Set the rotary switches (P1, P2) of the connection module according to the information in the [Lexium™ MC12 multi carrier Hardware Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=MLS-HWG&topicID=ConnectionModule_74374974). |
| Detected multi carrier hardware error: The segment is inoperative. | Replace the multi carrier segment. |
