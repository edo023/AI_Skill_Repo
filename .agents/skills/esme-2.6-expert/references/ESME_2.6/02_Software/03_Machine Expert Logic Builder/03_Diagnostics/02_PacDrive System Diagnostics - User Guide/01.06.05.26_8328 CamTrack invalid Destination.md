# 8328 CamTrack invalid Destination

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

The CamTrack in the Destination parameter contains an invalid value.

| Cause | Solution |
| --- | --- |
| The Destination parameter was specified either without the symbolic name of an output group or with none. If an IEC task detects an error (for example, access error using pointer), it triggers a diagnostic message. | Verify and correct if necessary the invalid parameter value. |
