# 8122 Shutdown due to velocity limit

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | BD2 | CD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

* The active speed of rotation is greater than the maximum speed of rotation of the motor that means 111.1% of MaxVel is exceeded.
* An MH3 motor indicating a diagnostic message [8122 Shutdown due to velocity limit](#D-SE-0063862) can also provoke the diagnostic messages [8107 Overcurrent](D-SE-0063849.html#D-SE-0063849) or [8119 Power stage short-circuit / ground fault](D-SE-0063858.html#D-SE-0063858).

| Cause | Solution |
| --- | --- |
| Error occurs on a CAM movement function (for example, MultiCam()) due to incorrect curve or profile data. | Verify and correct if necessary the curve data and program. |
| Jumps occurred in the master encoder. | Verify and correct if necessary the master encoder. |
| An error occurred on a CAM movement function due to an incorrect position manipulation, for example, FC\_SetposSingle() function. | Verify and correct if necessary the program. |
| Detected hardware error: Commutation error, that is, the motor encoder (SinCos) is not calibrated or leveled. | Contact customer service. |
