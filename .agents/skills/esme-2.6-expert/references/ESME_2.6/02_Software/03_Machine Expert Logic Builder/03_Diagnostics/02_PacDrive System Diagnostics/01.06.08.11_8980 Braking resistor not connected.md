# 8980 Braking resistor not connected

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | BD2 | BD2 |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | BP | BP |

This diagnostic message is triggered if an external braking resistor is configured but not connected.

| Cause | Solution |
| --- | --- |
| The external braking resistor is not connected to the device. | Connect the braking resistor. |
| The internal braking resistor shall be used but the parameter ExternalBrakingResistorEnableSet is set to `on / 1`. | Set the parameter ExternalBrakingResistorEnableSet to `off / 0`. |
| The external braking resistor is inoperable. | Replace the braking resistor. |
