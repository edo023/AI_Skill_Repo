# 8829 Velocity too high for ZeroTrack

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

The TM5SE1MISC20005 encoder output module detects when the ZeroTrack can no longer be generated correctly.

| Cause | Solution |
| --- | --- |
| The TM5SE1MISC20005 encoder output module has a limitation on ZeroTrack generation with high velocities.  The ZeroTrack cannot be generated if the speed is greater than 1 revolution per 5 cycles. | This leads to a maximum speed of:   * 12,000 revolutions/minute (1 ms cycle time) * 6,000 revolutions/minute (2 ms cycle time) * 3,000 revolutions/minute (4 ms cycle time) |
