# 8137 Motorless

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

When switching the servo amplifier from motorless to normal mode, a motor error is reported that has occurred in the meantime.

Motor errors that result in the diagnostic message:

* [8102 Motor overtemperature](D-SE-0063844.html#D-SE-0063844)
* [8105 Encoder position invalid](D-SE-0063846.html#D-SE-0063846)
* [8111 Shutdown due to tracking deviation](D-SE-0063854.html#D-SE-0063854)
* [8116 Commutation error detected](D-SE-0063856.html#D-SE-0063856)
* [8117 Motor type plate not readable](D-SE-0063857.html#D-SE-0063857)
* [8122 Shutdown due to velocity limit](D-SE-0063862.html#D-SE-0063862)

The above diagnostic messages are disabled in the motorless mode (value of parameter MotorIdentification = motorless / 1) and result in the diagnostic message when reverting to normal mode (value of parameter MotorIdentification = Motor with type plate / 0 or Motor without type plate /2).

NOTE: The [8105](D-SE-0063846.html#D-SE-0063846) diagnostic message can also result in the 8137 Motorless diagnostic message if the messages occurred in normal mode and the mode is switched to motorless and then back to normal mode afterwards.

| Cause | Solution |
| --- | --- |
| A motor error has occurred. | Perform a hardware reset or switch back to motorless mode. |
| The servo amplifier was not switched off when coupling a motor to a servo amplifier that was set to motorless. Then it was reset to normal mode. | Perform a hardware reset on the servo amplifier or switch back to motorless mode. |
