# 8104 Control voltage out of range

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | BD2 | CD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | BP1 | BP1 |

This diagnostic message is triggered if the control voltage (24 Vdc) is not within the permitted range anymore.

The drive was stopped because an error-free function of the device could no longer be affirmed.

| Cause | Solution |
| --- | --- |
| The control voltage is too high or too low. | Verify and correct if necessary the control voltage (see technical data of the device). |

NOTE: This diagnostic message is only available for Lexium 52 drives.
