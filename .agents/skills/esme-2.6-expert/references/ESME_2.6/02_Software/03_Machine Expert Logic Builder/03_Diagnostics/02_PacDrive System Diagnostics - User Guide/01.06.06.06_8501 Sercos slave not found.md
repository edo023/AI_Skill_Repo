# 8501 Sercos slave not found

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The device that was expected at the position (x), was not found.

| Cause | Solution |
| --- | --- |
| Detected error when assigning the identification features. | Verify and correct if necessary the settings of the identification features (for example: ConfiguredTopologyAddress) |

A Sercos slave listed in the PLC configuration cannot be found in the Sercos bus.

| Cause | Solution |
| --- | --- |
| Sercos slave not present in the Sercos bus. | Connect the Sercos slave to the Sercos bus or remove the Sercos slave from the PLC configuration or parameterize it as a virtual device |
