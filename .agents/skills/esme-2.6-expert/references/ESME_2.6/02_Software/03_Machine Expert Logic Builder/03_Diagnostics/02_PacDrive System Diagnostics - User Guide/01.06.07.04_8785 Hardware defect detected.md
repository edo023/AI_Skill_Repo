# 8785 Hardware defect detected

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

A hardware error is detected.

| Ext. diagnosis | Meaning |
| --- | --- |
| `Spi` | `Instance= OutputGroup`  The status of the digital output remains unchanged until the supply voltage (connector CN1) has fallen to 0 volts. The diagnostic message can be entered in the message log when the controller is switched off, as the voltage drop of the supply voltage causes a Spi error before the CPU detects a power outage.  `Instance= InputGroup`  The digital inputs cannot be evaluated. The diagnostic message can be entered in the message log when the controller is switched off, as the voltage drop of the supply voltage causes a Spi error before the CPU detects a power outage.  Verify and correct if necessary the supply voltage. |
| `Ref PowerFail` | Evaluation of the digital inputs in accordance with the en61131-2 type 1 standard is no longer supported. The diagnostic message can be entered in the message log when the controller is switched off, as the voltage drop of the supply voltage causes a Ref PowerFail error before the CPU detects a power outage. |

| Cause | Solution |
| --- | --- |
| Detected software error: Module overload. | Contact your local Schneider Electric service representative. |
