# 8255 Firewall applied default rules

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

The default set of firewall rules was applied as the firewall configuration file was not found or an error was detected in the firewall configuration file.

Refer to diagnostic messages [8253](D-SE-0104260.html#D-SE-0104260) or [8254](D-SE-0104261.html#D-SE-0104261) to find a possible cause and solution.
