# 8105 Encoder position invalid

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | – | – |

The encoder position could not be determined.

| Cause | Solution |
| --- | --- |
| Encoder voltage is incorrect or unavailable. | Verify correct encoder voltage. |
| Incorrect encoder cable or incorrect wiring. | Verify correct wiring of the encoder with the correct encoder cable. |
| The 24 Vdc control voltage is too low. | Verify correct control voltage. |
| Encoder inoperative. | Replace the motor or the encoder. |

The following diagnostics messages provide additional information on the cause of this condition:

* [8171 Encoder comm. disturbance detected](D-SE-0063890.html#D-SE-0063890)
* [8958 Encoder communication not possible](D-SE-0064088.html#D-SE-0064088)
* [8176 Encoder connection break](D-SE-0104889.html#D-SE-0104889)
* [8175 Extended message for encoder disturbances](D-SE-0104890.html#D-SE-0104890) (from drive logger)
