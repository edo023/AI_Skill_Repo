# 8562 Carrier negotiation communication error detected

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | AD | – |

An error was detected during carrier data transfer between neighboring segments.

| Cause | Solution |
| --- | --- |
| EMC interferences during communication between segments. | * Verify the mounting and fastening of the communication interconnects of the segments. * Verify that the correct communication interconnect is inserted at the end of an open track. |
| The communication between neighboring segments is unsuccessful. |
| Sercos phase-up has been started before the segments are initialized. | Wait until all segments are initialized before starting Sercos phase-up. |
