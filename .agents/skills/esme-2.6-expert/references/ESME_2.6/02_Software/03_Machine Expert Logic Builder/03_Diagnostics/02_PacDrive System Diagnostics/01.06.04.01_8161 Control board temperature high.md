# 8161 Control board temperature high

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | D | E |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | D | E |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | D | – |

* The temperature of the power board inside the housing is too high. If the temperature continues to increase, the diagnostic message [8142 Control board overtemperature](D-SE-0063877.html#D-SE-0063877) is triggered.
* The temperature of the internal multi carrier segment controller is high.

| For the Multi carrier transport system: | |
| --- | --- |
| Cause | Solution |
| Insufficient ventilation of the multi carrier segment or ambient temperature too high. | * Improve the thermal transition between the multi carrier segment and the mounting plate (for example, by using thermal paste). * Improve the surface ventilation of the multi carrier segment. |
