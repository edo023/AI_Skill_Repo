# 8566 LMC connected to the incorrect port of the track

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | AD | – |

The Sercos wiring is not correct. The P1 port (CN12) of the LMC (Logic Motion Controller) has to be connected to the P1 port of the track.

Refer to the [wiring example for a closed track](../../../../../api/crossBook?lang=en-US&virtualBookName=MLS-HWG&topicID=WiringExample_942160D1) and the [wiring example for an open track](../../../../../api/crossBook?lang=en-US&virtualBookName=MLS-HWG&topicID=AdditionalWiringExample_AAB02800) in the Lexium™ MC12 multi carrier Hardware Guide.

| Cause | Solution |
| --- | --- |
| The Sercos wiring is not correct. The P1 port of the LMC (Logic Motion Controller) is not connected to the P1 port of the track. | Connect the P1 port of the LMC (Logic Motion Controller) to the P1 port of the track. |
