# 8330 Program master job not executable

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The diagnostic message is triggered if the system is unable to process the master positioning job of an axis. Even though no errors were detected by the job parameter area checks (for example, detected error in the pre-calculation of generator data).

The detected error is shown in the extended diagnostic message as GT=XXX,MJId=YYY. XXX indicates the generator type while YYY indicates the Master-Job-Id of the job that caused the issue.

Generator type:

|  |  |  |
| --- | --- | --- |
| 0 | → | CAM |
| 1 | → | POS |
