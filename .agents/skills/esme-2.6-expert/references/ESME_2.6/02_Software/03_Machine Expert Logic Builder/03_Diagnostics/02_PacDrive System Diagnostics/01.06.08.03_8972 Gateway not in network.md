# 8972 Gateway not in network

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| Cause | Solution |
| --- | --- |
| The manually entered gateway (IPConfigMode = `Manual IP addressing (hardware) / 1` or `Manual IP addressing (software) / 2`) is not within the network that was defined with NetID. | Verify the manually entered gateway and adjust it so that it is within the network that was defined with NetID, if necessary. |
| If this message occurs during boot time of the controller, the permanently saved gateway is not within the network. | Verify the permanently saved gateway and adjust it so that it is within the configured network during boot. |
