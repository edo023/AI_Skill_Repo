# 8126 Power stage temperature high

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | D | E |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | D | - |

* The cooling element of the PacDrive servo amplifier is overheated. An [8101 Power stage overtemperature](D-SE-0063843.html#D-SE-0063843) error may result.
* The temperature of the power stage of the multi carrier segment is high.

| For the Multi carrier transport system: | |
| --- | --- |
| Cause | Solution |
| The cooling of the multi carrier segments is not sufficient. | Improve the thermal transition between the multi carrier segments and the mounting plate (for example, by using thermal paste). |
