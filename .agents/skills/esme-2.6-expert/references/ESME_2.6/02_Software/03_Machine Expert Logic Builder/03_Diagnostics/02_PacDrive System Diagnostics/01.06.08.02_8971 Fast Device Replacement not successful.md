# 8971 Fast Device Replacement not successful

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

The diagnostic message shows that the firmware of a device could not be exchanged. In the ext. diagnosis, the detected error is described more precisely.

Meaning of **tftp err/06**: IPTFTP ERR TRANSFER TIMEOUT

## Restrictions

The **Fast Device Replacement** can be performed on the PacDrive devices and power supplies. If **Fast Device Replacement** is performed, then carry out the addressing of the devices via the topological address. Otherwise, an exact allocation of the devices cannot be affirmed after the exchange. For this, the value `TopologyAddress/2` has to be allocated to the parameter IdentificationMode of the devices.
