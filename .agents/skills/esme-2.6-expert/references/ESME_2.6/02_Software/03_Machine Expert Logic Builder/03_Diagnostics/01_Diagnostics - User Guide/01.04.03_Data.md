# Data

## Overview

In the context of this document, data means the collection of the data of the controller and the peripherals.

The following components can be available:

* Data (objects and their parameters) of the  PLC configuration.
* One or several message loggers from the controller.
* One or several message loggers from the file system.
* Additional information from the dialog box  Machine info.
* Data view settings.

The data is stored in xml format.

When saving, you can designate whether the file is to be saved directly as an .xml file or [compressed](D-SE-0041428.html#D-SE-0041428).

The compressed form is the default setting and has the file extension .pdi.

NOTE: The .pdi file extension can be replaced by the .zip extension to be unpacked using a standard data compression program.
