# 8572 Physical Carrier not unique

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

During phase-up of the Sercos bus, the logical carrier objects from the controller configuration are linked to the physical carriers found on the track in phase 2.

This message is displayed if more than one physical carrier was found for a logical carrier from the controller configuration based on its identification parameters.
