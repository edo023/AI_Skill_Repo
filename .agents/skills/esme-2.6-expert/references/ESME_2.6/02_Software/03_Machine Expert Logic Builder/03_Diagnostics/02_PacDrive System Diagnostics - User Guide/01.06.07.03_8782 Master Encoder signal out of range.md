# 8782 Master Encoder signal out of range

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

A hardware encoder error is detected at the incremental encoder input (**BT-4/ENC1**, incremental encoder module **INC-4**), or incremental encoder module **TM5SE1IC20005**.

NOTE: Hardware encoder error. Reset the PacDrive controller. Only acknowledge the diagnostic message after reinitialization or homing.

| CAUTION | |
| --- | --- |
|  | POSITION LOSS DUE TO SERCOS BUS ERROR  Only acknowledge the diagnostic message after reinitialization or homing.  Failure to follow these instructions can result in injury or equipment damage. |

If necessary, use the function FC\_SysReset() or FC\_PrgResetAndStart().

| Cause | Solution |
| --- | --- |
| Detected wiring error: Encoder cable at the encoder or the encoder input has been removed or is inoperable. | * Verify the encoder cable and replace if necessary. * Verify and correct or replace if necessary the ground connection (shield). |
| The encoder/encoder supply or encoder simulation is not ready. | * Disable the monitoring with the parameter CheckOff by standard. * If the encoder or encoder simulation is ready, activate the monitoring in the program. |
| Encoder inoperable. | Replace the encoder. |

Additional possible cause by PacDrive LMC Eco:

| Cause | Solution |
| --- | --- |
| The connected encoder does not match the configured encoder. (SinCos encoder connected, incremental encoder configured). | Adjust the configuration of the encoder or exchange the connected encoder. |
