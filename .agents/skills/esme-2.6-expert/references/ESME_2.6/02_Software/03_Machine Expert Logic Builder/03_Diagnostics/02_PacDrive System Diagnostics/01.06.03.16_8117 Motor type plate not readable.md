# 8117 Motor type plate not readable

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

The electronic motor type plate cannot be read from the encoder or contains invalid data.

| Cause | Solution |
| --- | --- |
| Detected wiring error: The encoder cable is not correct. | Verify and correct if necessary the wiring. |
| The control voltage of the device (24 Vdc) is too low. | Verify and correct if necessary control voltage. |
| Detected hardware error: The encoder is damaged. | Exchange the motor or the encoder. |
| The setting of the parameter MotorIdentification is incorrect (in the logger of the drive the Parameter 1 has the value `407`). | Set MotorIdentification to the correct value. |
