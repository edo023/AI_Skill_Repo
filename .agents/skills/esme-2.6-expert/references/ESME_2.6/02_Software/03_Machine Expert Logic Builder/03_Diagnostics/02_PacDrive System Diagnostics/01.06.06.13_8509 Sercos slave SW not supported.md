# 8509 Sercos slave SW not supported

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The firmware version of the Sercos slave is incorrect.

| Cause | Solution |
| --- | --- |
| The firmware version of the Sercos slaves is greater than the firmware version of the PacDrive controller in the first four positions. | Update the controller version. |
| Ext. diagnosis = `Vxx.yy.zz BLD`  The Sercos slave is in bootloader mode. The version of the Bootloader is shown in the message logger under Ext. Diagnosis. The Sercos slave can also be found using the information on the object and instance provided in the message logger. | * Reduce the number of objects with real-time tasks (for example, virtual axes, virtual encoders, logical encoders, sum encoders, etc.). * Reset the Sercos slaves. * Acknowledge the diagnostic message.   The Sercos bus is running up in phase 4. |
