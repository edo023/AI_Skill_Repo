# 8780 Encoder output frequence > 1MHz

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The incremental speed given out was too high.

NOTE: Incremental speed too high. Position loss at the incremental encoder output. Carry out the referencing of the amplified incremental encoder inputs or systems again.

Causes and troubleshooting measures are listed below.

| Cause | Solution |
| --- | --- |
| The speed of the encoder to be simulated is too high. | Reduce the speed of the encoder. |
| The resolution and/or FeedConstant parameters have not been entered correctly. | Verify and correct if necessary the parameters. |
| The connector of the incremental encoder was connected or removed while the device was energized (control voltage on). | * Switch off the device. * Connect or remove the connector. * Switch on the device. |

| WARNING | |
| --- | --- |
|  | INCORRECT ENCODER OUTPUT  Remove control voltage power before connecting or removing the incremental encoder connector.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |
