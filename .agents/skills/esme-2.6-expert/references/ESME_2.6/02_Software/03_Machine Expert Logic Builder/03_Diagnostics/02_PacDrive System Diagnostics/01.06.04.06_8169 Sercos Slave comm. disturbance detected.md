A temporary signal error on the Sercos bus has been detected. The MDT data may be incorrect due to this error detection. In this case, the data of the last cycle is reused.

Temporary detected errors can have the following causes:

* Detected error of the master synchronous telegram (MST)
* Detected error of the master data telegram (MDT)
* Detected error of the frame check sequence (FCS; checksum of the Ethernet frame)

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| EMC issues. | Verify and correct if necessary the wiring. |

* Single detected error in the Sercos data telegram of the Multi carrier transport system

| For the Multi carrier transport system: | |
| --- | --- |
| Cause | Solution |
| The communication interconnects of the segments are not inserted correctly. | Verify the wiring and the mounting/fastening of the communication interconnects of the segments. |
