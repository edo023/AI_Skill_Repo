# 8511 CPU time overflow

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The real-time process is overloaded (CycleLoad >95 %). The DiagExtMsg parameter displays the measured load as Load = `xxx %` with `xxx` indicating the load in %.

NOTE: Having detected the error, the actual and target values for the encoders, generators, and axes are no longer processed in the real-time process. As a result, the target and actual values parameter of the axes show invalid values.

| Cause | Solution |
| --- | --- |
| Too many objects have been entered into the PLC configuration. | Increase the CycleTime parameter of the Sercos interface in the PLC configuration. |
