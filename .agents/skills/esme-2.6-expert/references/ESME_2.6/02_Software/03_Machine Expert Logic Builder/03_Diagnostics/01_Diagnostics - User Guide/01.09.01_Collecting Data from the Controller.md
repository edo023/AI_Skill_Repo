# Collecting Data from the Controller

## Overview

This chapter provides a step by step description for the most common type of data connection to a controller via Ethernet .

![](images/T001804614.gif)

Proceed as follows:

| Step | Action |
| --- | --- |
| 1 | On the [**Home** window](D-SE-0041424.html#D-SE-0041424), click the  Collect data ... button.  **Result:** The [**Connect to controller via...** window](D-SE-0042143.html#D-SE-0042143) opens. If the data was previously collected, you will see one dialog box to save it and then another one to select the controller. |
| 2 | Verify that the PC and the desired controller are connected to the Ethernet network linked directly via a crossed network cable (RJ45). |
| 3 | Verify the network settings of the PC.  NOTE: You can also use the tool Network Device Identification for detecting the correct address of the controller. This will display the IP address and subnetmask of that network, even if the network settings of your computer do not match. (Nevertheless, a cable must still be connected to an active network card of the PC). To configure your PC accordingly, the subnet mask must match the subnet mask of the controller. The IP addresses must differ in one of the last digits. |
| 4 | Select Connection Mode > IP Address  and enter the IP address of the controller into the field IP Address: of the Connect to controller via...  window, or  select another [connection mode](D-SE-0077990.html#D-SE-0077990), enter the required data, or double-click the controller you want to connect to. |
| 5 | Click the Connect to... button.  For PacDrive controllers (except for PacDrive M controllers), you might be prompted for [secured access to the controller](D-SE-0104615.html#D-SE-0104615).  **Result:** After the data has been successfully collected, you are automatically returned to the [**Home** window](D-SE-0041424.html#D-SE-0041424). From there, you can [view](D-SE-0041410.html#D-SE-0041410), [save](D-SE-0041428.html#D-SE-0041428), or [send the data directly by email](D-SE-0041435.html#D-SE-0041435). |
