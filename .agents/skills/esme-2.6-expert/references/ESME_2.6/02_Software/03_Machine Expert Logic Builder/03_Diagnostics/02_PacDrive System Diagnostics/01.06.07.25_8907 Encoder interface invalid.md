# 8907 Encoder interface invalid

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

The encoder interface of the motor name plate is not according to the encoder interface identified by the device.

| Cause | Solution |
| --- | --- |
| The variable uiEncoderType in the data structure ST\_UserMotorData is set to an incorrect value. | Set the variable to the correct value. |
| The wrong encoder is connected to this device. | Verify and correct if necessary the wiring of the encoder. |
| Encoder is inoperable. | Replace the motor, or the encoder if possible. |
