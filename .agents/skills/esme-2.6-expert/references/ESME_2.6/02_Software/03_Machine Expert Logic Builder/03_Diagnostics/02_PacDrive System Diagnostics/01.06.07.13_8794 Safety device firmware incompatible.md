# 8794 Safety device firmware incompatible with SLC type

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

During Sercos phase-up, the firmware versions of the safety-related modules are verified for compatibility with the openSafety version used.

Sercos phase-up is interrupted if one of the safety-related modules is incompatible and if the version verification is enabled.

| Cause | Solution |
| --- | --- |
| Incompatible firmware version of a safety-related module. | Update the firmware version of the safety-related module using FDR or the service tool Device Assistant. |
