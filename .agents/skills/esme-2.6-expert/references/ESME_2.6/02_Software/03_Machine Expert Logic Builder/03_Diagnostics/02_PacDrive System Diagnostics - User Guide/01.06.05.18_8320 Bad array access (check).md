# 8320 Bad array access (check)

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

An array access error in the user program is detected by the Check.lib. The message logger call hierarchy function can be used to backtrace the function block call or nesting.

| Cause | Solution |
| --- | --- |
| While accessing an array, the area limits of that array were violated in the user program. | Verify and correct if necessary the program. |
