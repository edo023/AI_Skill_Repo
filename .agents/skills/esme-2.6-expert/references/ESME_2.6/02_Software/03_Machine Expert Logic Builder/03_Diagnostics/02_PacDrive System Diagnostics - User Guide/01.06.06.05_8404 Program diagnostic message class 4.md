# 8404 Program diagnostic message class 4

## Description

[Refer to Diagnostic class (standard): 4](D-SE-0063169.html#D-SE-0063169)

A diagnostic message from the program has been triggered with the FC\_DiagMsgWrite() function.

| Cause | Solution |
| --- | --- |
| The diagnostic message was triggered by the program. | See the description of your program.  It is possible to display an additional diagnostic text with the FC\_DiagMsgWrite() function. |

NOTE: You cannot acknowledge this diagnostic message because this diagnostic message is assigned to [diagnostic class 4](D-SE-0063169.html#D-SE-0063169__D-SE-0063169.2). You must restart the controller.
