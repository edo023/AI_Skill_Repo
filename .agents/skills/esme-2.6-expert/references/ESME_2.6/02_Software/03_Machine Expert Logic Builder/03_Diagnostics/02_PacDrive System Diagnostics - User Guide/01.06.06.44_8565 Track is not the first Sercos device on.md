# 8565 Track is not the first Sercos device on the LMC

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | AD | – |

The first segment of the track has detected that it is not the first Sercos device after the LMC (Logic Motion Controller). No other Sercos device is allowed between the P1 port of the LMC and the first segment of the track.

| Cause | Solution |
| --- | --- |
| Incorrect Sercos wiring. | Verify the Sercos wiring and adapt if necessary. |
