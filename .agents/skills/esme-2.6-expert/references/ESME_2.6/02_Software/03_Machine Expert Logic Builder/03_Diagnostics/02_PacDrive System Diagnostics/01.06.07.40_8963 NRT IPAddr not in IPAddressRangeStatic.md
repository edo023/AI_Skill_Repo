# 8963 NRT IPAddr not in IPAddressRangeStatic

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| Cause | Solution |
| --- | --- |
| The manually allocated IP address is not within IPAddressRangeStatic. | * Verify whether a slave with IPConfigMode unequal `Automatic IP addressing / 0` is configured and correct if necessary. * Verify whether there is an IP address that is not within the calculated IPAddressRangeStatic and correct if necessary. * If IPConfigMode = `Manual IP addressing (hardware) / 1`, verify the IP address that was set on the hardware (for example with a switch) or via the Drives Assistant. * If IPConfigMode = `Manual IP addressing (software) / 2`, verify the IP address that was set in the parameter. |
