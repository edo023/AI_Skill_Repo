# 8726 Firmware of the module is incorrect

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

This message shows that the firmware of a fieldbus module is incorrect.

| Cause | Solution |
| --- | --- |
| Firmware download for a fieldbus module via the controller was not successful. | Repeat the firmware download. |
