# 8730 Bad master parameter data

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The slave and/or master data of the configuration file are incorrect. The error code specified after `Err=` corresponds to the error code of the ErrorCode parameter.

| Cause | Solution |
| --- | --- |
| Communication errors when transferring the PLC configuration from Logic Builder to controller. | Transfer the PLC configuration again. |
| Detected hardware error: Controller or flash disk inoperable. | Replace the controller. |
