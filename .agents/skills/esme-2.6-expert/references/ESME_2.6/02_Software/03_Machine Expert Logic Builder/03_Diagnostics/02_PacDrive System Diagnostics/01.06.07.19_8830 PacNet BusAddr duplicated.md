# 8830 PacNet BusAddr duplicated

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

Two PacNet modules are configured with the same bus address (BusAddr) in your project.

This leads to a misconfiguration because the bus address must be unique.

| Cause | Solution |
| --- | --- |
| The same PacNet bus address (BusAddr) has been assigned to two PacNet modules. | Verify that the PacNet bus address is unique for each module and correct if necessary. |
