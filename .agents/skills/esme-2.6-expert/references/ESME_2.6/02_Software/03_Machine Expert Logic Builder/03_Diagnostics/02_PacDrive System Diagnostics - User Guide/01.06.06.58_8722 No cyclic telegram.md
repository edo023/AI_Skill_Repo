# 8722 No cyclic telegram

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The exchange of cyclical data was interrupted.

NOTE: No error is displayed when the connection between the PROFIBUS DP Slave and PROFIBUS DP Master is interrupted if the response monitoring in the slave was deactivated by the master. In this case, the diagnostic code 8722 is not triggered.

| Cause | Solution |
| --- | --- |
| The master interrupts the sending of cyclical data. | Verify and correct if necessary the master. |
| Cable error. | Verify and correct if necessary the wiring. |
| The slave address has been assigned twice in the network. | Verify and correct if necessary the configuration. |
