# 8965 NRT IP parameter read not possible

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| Cause | Solution |
| --- | --- |
| * The Sercos III parameter reading process cannot be executed. * The parameters IPAddress, Subnet mask, and Gateway are read from the slaves via Sercos III if IPConfigMode = `Manual IP addressing (hardware) / 1` is set. | Verify whether the device supports the parameters and correct if necessary. |
