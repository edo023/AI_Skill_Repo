# 8129 Power stage load high

## Description

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |

The power stage is overloaded. The overload level has been increased up to 80 %. This may cause an error [8120 Power stage overload](D-SE-0063860.html#D-SE-0063860) if the load of the power stage is not reduced.
