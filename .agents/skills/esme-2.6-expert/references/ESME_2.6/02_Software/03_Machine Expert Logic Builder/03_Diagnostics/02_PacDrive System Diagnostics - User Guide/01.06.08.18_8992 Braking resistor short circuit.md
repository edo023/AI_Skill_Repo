# 8992 Braking resistor short circuit

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | AP | AP |

This diagnostic message is triggered if a short circuit was detected on the input for the external braking resistor.

| Cause | Solution |
| --- | --- |
| The external braking resistor is short-circuited. | Verify and correct if necessary the wiring of the external braking resistor. |
| The external braking resistor is inoperable. | Replace braking resistor. |
