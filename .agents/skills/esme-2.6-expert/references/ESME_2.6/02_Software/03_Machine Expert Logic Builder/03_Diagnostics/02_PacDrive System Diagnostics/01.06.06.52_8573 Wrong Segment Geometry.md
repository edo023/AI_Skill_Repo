# 8573 Wrong Segment Geometry

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

When starting up the Sercos bus, in phase 2 the geometry parameters (length and angle) of the detected segments are validated with the expected parameters that are entered in the controller configuration.

Refer to [Lexium™ MC multi carrier, Device Objects and Parameters Guide\SegmentGeometry](../../../../../api/crossBook?lang=en-US&virtualBookName=MCRDOaPG&topicID=SegmGeo_9D6B90FC).

This message is displayed if the scanned segment does not match the configured dimensions.
