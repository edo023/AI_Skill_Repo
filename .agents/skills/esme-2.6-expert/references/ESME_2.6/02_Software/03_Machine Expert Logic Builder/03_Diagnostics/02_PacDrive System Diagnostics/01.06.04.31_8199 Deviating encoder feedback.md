# 8199 Deviating encoder feedback

## Description

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |

| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

There is a significant dynamic deviation between the position of the motor encoder and the position of the machine encoder. A static position offset is not relevant. A slip of up to 50 % is acceptable; otherwise this diagnostic message is triggered.
