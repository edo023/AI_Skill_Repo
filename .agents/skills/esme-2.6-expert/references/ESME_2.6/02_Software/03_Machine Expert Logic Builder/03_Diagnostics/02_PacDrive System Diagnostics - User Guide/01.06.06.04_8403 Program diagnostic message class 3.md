# 8403 Program diagnostic message class 3

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

A diagnostic message from the program has been triggered with the FC\_DiagMsgWrite() function.

| Cause | Solution |
| --- | --- |
| The diagnostic message was triggered by the program. | See the description of your program.  It is possible to display an additional diagnostic text with the FC\_DiagMsgWrite() function. |
