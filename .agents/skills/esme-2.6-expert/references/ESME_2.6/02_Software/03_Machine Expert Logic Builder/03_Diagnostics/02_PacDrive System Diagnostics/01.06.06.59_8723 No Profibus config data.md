# 8723 No Profibus config data

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

This diagnostic message only occurs in the boot phase of the PacDrive controller or with bus enable. No configuration file was found on the flash disk of the PacDrive controller.

A DiagExtCode may be generated.

| Ext. diagnosis | Meaning |
| --- | --- |
| `Database` | Unable to delete the SyCon configuration database on the PROFIBUS card software. |
| `bad load` | No configuration file was found on the flash disk of the controller. |
| `bad sldata` | The slave data of the configuration file is incorrect.  This should occur together with the 8730 Bad master parameter data diagnostic message. If not, the incorrect configuration results in the PROFIBUS card resetting. |
| `bad madata` | The master bus data of the configuration file is incorrect.  This should occur together with the 8730 Bad master parameter data diagnostic message. If not, the incorrect configuration results in the PROFIBUS card resetting. |

| Cause | Solution |
| --- | --- |
| Communication errors. | Retransmit the PLC configuration. |
| The slave address has been assigned twice in the network. | Verify and correct if necessary the configuration. |
| Detected hardware error: Flash disk inoperable. | Replace the flash disk. |
| Detected hardware error: Controller is inoperable. | Replace the controller. |
