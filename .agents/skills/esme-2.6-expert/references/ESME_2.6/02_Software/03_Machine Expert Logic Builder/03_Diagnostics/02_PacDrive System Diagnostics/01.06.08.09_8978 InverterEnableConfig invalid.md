# 8978 InverterEnableConfig invalid

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

## Ext. Diagnosis = `x(HW)!=y(Cfg)`

The hardware configuration (HW) of the real device (Lexium 62 ILM Drive Module) does not match the set value of the parameter InverterEnableConfig (Cfg). `x` and `y` each are the corresponding ENUM value.

**Example**:

Ext. diagnosis = `1(HW)!=0(Cfg)`

The optional module DIS1 is not connected to the Lexium 62 ILM, even though the parameter value of the InverterEnableConfig is `off / 0`. This means that the optional module should be connected.

| Cause | Solution |
| --- | --- |
| - | * Verify the parameter InverterEnableConfig. Adjust the parameter if necessary. * Verify the hardware configuration of the Lexium 62 ILM Drive Module. If necessary, connect or remove the DIS1. |
