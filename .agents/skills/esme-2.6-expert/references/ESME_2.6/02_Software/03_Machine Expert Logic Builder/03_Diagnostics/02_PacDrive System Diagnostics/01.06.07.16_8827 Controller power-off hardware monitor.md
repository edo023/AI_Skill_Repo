# 8827 Controller power-off/hardware monitor

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

MsgFilter: Bit 1 Diagnosis Messages

The hardware provides an internal message logger that logs error-dependent controller deactivations. The next time the controller is started, the error logger is analyzed, and the cause of the error can be given in the message logger. In addition, a more specific message is indicated ([8055 Controller message HW monitor](D-SE-0063837.html#D-SE-0063837)) that can be enabled by the MsgFilter parameter and activated via the Logic Builder message logger.

| Cause | Solution |
| --- | --- |
| DiagExtMsg defines the detected error in further detail (high or low voltage, high or low temperature, fan error, temperature sensor error):   * Voltage<Min * Voltage>Max * Temp<Min * Temp>Max * Fan error * LM75 error | Contact your local Schneider Electric service representative. |
