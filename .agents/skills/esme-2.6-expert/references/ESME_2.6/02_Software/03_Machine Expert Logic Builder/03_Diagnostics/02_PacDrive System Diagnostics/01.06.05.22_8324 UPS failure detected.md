# 8324 UPS failure detected

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

An internal error in the UPS is detected.

| Cause | Solution |
| --- | --- |
| The battery of the UPS is depleted. | Charge the battery. |
| The power supply of the UPS is working, but the **ON/OFF** input of the UPS is set to **OFF**. | If the voltage supply is correct, then the **ON/OFF** input must be **ON** in order to use the UPS. |
| An UPS object is specified in the PLC configuration, but no UPS is connected. | Verify and correct if necessary the program. |
| An internal UPS error has occurred. | Verify and correct if necessary the program. |

## Ext. diagnosis: `PIC fail` or `State = 5`

| Cause | Solution |
| --- | --- |
| An internal PIC controller error is detected. | Contact your local Schneider Electric service representative. |
