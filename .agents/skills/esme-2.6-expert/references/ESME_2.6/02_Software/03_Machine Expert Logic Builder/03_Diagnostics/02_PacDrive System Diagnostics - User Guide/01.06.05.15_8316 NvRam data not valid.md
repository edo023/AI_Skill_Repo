# 8316 NvRam data not valid

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

The NvRam lost the data. This means that the data in the message logger and the variable values are invalid. In this case, the message logger is automatically deleted by the system.

| Cause | Solution |
| --- | --- |
| An internal system error is detected. | Replace the PacDrive controller. |
