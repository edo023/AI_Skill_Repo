# 8990 Firmware update not possible

## Description

[Refer to Diagnostic class (standard): 1](D-SE-0063169.html#D-SE-0063169)

The firmware of the device could not be updated.

| Cause | Solution |
| --- | --- |
| Sercos is in phase 4. A firmware update is only possible in phase 0. | Switch Sercos to phase 0. |
| The selected firmware is not compatible with the device (for example, the Lexium 52 firmware shall be installed in a Lexium 62 drive). | Verify whether the selected firmware is compatible with the device. |
