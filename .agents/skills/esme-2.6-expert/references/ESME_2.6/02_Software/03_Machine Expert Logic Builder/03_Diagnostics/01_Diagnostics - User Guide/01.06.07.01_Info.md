# Info

## Overview

Click the  Diagnostics icon to open the [system menu](D-SE-0041404.html#D-SE-0041404). Select the  About command to open the  About Diagnostics dialog box. It provides general information and technical information on Diagnostics.

The dialog box consists of the following tabs:

| Tab | Description |
| --- | --- |
| General | General information on Diagnostics. |
| Technical information | System information. |

Close the dialog box with OK .
