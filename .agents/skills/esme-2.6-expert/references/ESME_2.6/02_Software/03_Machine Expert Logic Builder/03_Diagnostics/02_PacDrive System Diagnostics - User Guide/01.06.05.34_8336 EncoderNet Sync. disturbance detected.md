# 8336 EncoderNet Sync. disturbance detected

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

An error is detected in the encoder network, when synchronizing the controller system clocks.

## Ext. diagnosis: No Sync: xxx

| Cause | Solution |
| --- | --- |
| More than half of the synchronization telegrams (xxx) specified in SlaveCycleErrorLimit have not been received sequentially. If more synchronization telegrams are not received in a row than specified in SlaveCycleErrorLimit, then the diagnostic message [8335 EncoderNet Sync. disturbance detected](D-SE-0063938.html#D-SE-0063938) is triggered. | This can appear for a number of external reasons related to the network such as wiring, network traffic, performance or other related equipment. Verify and correct if necessary any external equipment or wiring disturbing communications. |
