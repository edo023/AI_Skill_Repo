# 8142 Control board overtemperature

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | BD2 | CD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | BP1 | BP1 |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | BD2 | – |

* The temperature of the power board inside the housing is too high.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| Insufficient ventilation or ambient temperature too high. | * Verify and correct if necessary the ventilation in the control cabinet. * If installed, ensure that the air conditioning unit is functioning properly. |
| Detected hardware error: The temperature sensor is inoperable. | Replace the device. |

* The temperature of the internal multi carrier segment controller is too high.

| For the Multi carrier transport system: | |
| --- | --- |
| Cause | Solution |
| Insufficient ventilation of the multi carrier segment or ambient temperature too high. | * Improve the thermal transition between the multi carrier segment and the mounting plate (for example, by using thermal paste). * Improve the surface ventilation of the multi carrier segment. |
