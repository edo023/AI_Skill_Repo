# 8001 Diagnosis acknowledgement

## Description

[Refer to Diagnostic class (standard): 1](D-SE-0063169.html#D-SE-0063169)

A diagnostic message was acknowledged.

This message is always entered in the message logger even if the diagnostic class (see [Diagnostic Classes](D-SE-0063169.html#D-SE-0063169)) of the message was set to 0 using the `FC_DiagConfigSet2()` function.
