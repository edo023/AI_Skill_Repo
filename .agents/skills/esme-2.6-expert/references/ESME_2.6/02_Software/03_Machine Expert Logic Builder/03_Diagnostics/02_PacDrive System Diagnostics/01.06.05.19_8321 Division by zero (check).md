# 8321 Division by zero (check)

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

In the user program, a division by zero was found by the Check.lib. The message logger call hierarchy function can be used to backtrace the function block call or nesting.

| Cause | Solution |
| --- | --- |
| A division by zero is to be executed in the program. | Verify divisors and avoid division by zero in the program. |
