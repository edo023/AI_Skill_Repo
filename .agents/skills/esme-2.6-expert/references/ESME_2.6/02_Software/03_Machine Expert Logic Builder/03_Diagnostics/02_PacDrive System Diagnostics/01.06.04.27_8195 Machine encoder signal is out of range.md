# 8195 Machine encoder signal is out of range

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | – | – |

A hardware encoder error has been detected. There is measured a voltage in the analog sinus or cosinus encoder tracks that is not matching the specified voltage range of the encoder.

| Cause | Solution |
| --- | --- |
| Detected wiring error: Encoder cable at the encoder or the servo amplifier / PacDrive controller has been removed or is inoperable. | * Verify the encoder cable and replace if necessary. * Verify and correct if necessary the ground connection (shield). |
| The encoder voltage is unavailable or incorrect. | Verify and correct if necessary the encoder voltage. |
| The control voltage of the device (24 Vdc) is too low. | Verify and correct if necessary the control voltage. |
| Detected hardware error: Encoder is inoperable. | Replace the encoder. |
