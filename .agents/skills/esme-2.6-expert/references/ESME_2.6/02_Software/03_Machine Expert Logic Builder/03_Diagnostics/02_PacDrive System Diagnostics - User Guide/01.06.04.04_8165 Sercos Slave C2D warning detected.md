# 8165 Sercos Slave C2D warning detected

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

Reaction: Manufacturer-specific

The following diagnostic message is triggered by the following objects:

* Sercos drive
* TM5NS31 (S3IO)
* Safe Logic Controller

The extended diagnostic message (ExtDiagMsg) can be displayed in two different formats:

* Diag = 16#aaaa with aaaa is the error code from Sercos IDN S-0-0012.

  Refer to the [Sercos C2D Error Messages](#D-SE-0063886__D-SE-0063886.5) for details.
* Diag = 0xaaaaaaaa with aaaaaaaa corresponds to the diagnostic message from the parameter *[DiagnosticNumber](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.BusInterfaceTM5NS31&topicID=D_SE_0086074)* (Sercos IDN S-0-0390).

## Sercos C2D Error Messages

Format of the ExtDiagMsg: `Diag 0 16#aaaa` with `aaaa` is the error code from Sercos IDN S-0-0012.

A Sercos alert of the diagnostic class 2 (C2D) has been detected. The extended diagnostic contains a data word that provides information on the cause.

The individual bits of this data word have the following meanings:(Sercos C2D alert (Assignment according to the Sercos specification IDN S-0-0012)):

| Bit | Meaning |
| --- | --- |
| Bit 0 | Overload alert |
| Bit 1 | Amplifier overtemperature alert |
| Bit 2 | Motor overtemperature alert |
| Bit 3 | Cooling error alert |
| Bit 4 | Reserved |
| Bit 5 | Feedback error |
| Bit 6 | Reserved |
| Bit 7 | Reserved |
| Bit 8 | Reserved |
| Bit 9 | Undervoltage error |
| Bit 10 | Reserved |
| Bit 11 | Excessive velocity deviation |
| Bit 12 | Reserved |
| Bit 13 | Target position outside of travel range |
| Bit 14 | Reserved |
| Bit 15 | Manufacturer-specific error (refer to diagnostic message [8166](D-SE-0063885.html#D-SE-0063885)) |
