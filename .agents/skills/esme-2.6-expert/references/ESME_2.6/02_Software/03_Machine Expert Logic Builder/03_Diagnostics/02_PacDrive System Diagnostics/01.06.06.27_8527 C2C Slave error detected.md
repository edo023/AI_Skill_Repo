# 8527 C2C Slave error detected

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

A C2C Slave error is detected.

The extended diagnostic provides information on the cause:

| Ext. diagnosis | Meaning |
| --- | --- |
| `Pkt handling` | C2C Slave packet handling error is detected. |
| `No write IRQ` | C2C Slave is missing an IRQ. |
| `W 0x` | C2C Slave I/O writing error is detected. |
| `R 0x` | C2C Slave I/O reading error is detected. |
