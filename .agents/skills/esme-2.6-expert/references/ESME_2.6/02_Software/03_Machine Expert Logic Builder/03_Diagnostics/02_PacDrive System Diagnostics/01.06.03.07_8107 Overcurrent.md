# 8107 Overcurrent

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | AP | AP |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | AD | – |

* The current through the power stage of the servo amplifier or in the DC bus of the power supply is too high. In contrast to the diagnostic messages [8120 Power stage overload](D-SE-0063860.html#D-SE-0063860) and [8146 DC bus overload](D-SE-0063880.html#D-SE-0063880), the device is switched off immediately if the maximum current is exceeded.
* An MH3 motor indicating a diagnostic message [8122 Shutdown due to velocity limit](D-SE-0063862.html#D-SE-0063862) can also provoke the diagnostic message 8107 Overcurrent.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| Detected wiring error: The shielding of the motor cable is not attached properly. | Verify and correct if necessary the wiring. |
| Servo amplifier: The current controller parameters are incorrect. | Verify the power control parameters Curr\_P\_Gain, Curr\_I\_Gain, and CurrFilter and set them to default if necessary. |
| Detected motor hardware error: The motor has a short-circuit. | Replace the motor. |
| Servo amplifier: The PowerStageFrequency is parameterized with 4 kHz. | * The parameter VoltageFeedForwardMode must be set to the value `Dynamic behavior of the control loop / 3`. * Verify the parameters Vel\_P\_Gain, Vel\_I\_Gain, and Pos\_P\_Gain and adjust them if necessary, to avoid an instable behavior of the controller. |

* The current through the coil of the multi carrier segment is too high (> 1.3 \* Ic\_peak).

| For the Multi carrier transport system: | |
| --- | --- |
| Cause | Solution |
| The current control loop parameters are incorrect. | Verify the parameters, and set them to default if necessary. |
| Detected multi carrier hardware error: The segment has a coil short-circuit. | Replace the multi carrier segment. |
