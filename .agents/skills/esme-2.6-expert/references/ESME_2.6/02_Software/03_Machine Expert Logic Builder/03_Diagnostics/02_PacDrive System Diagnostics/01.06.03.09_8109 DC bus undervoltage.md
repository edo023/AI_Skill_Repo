# 8109 DC bus undervoltage

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | BD2 | E (CD for Lexium 52) |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | BP2 | BP2 |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | BD2 | – |

The DC bus voltage is too low.

| Device | Threshold |
| --- | --- |
| Lexium 52 | 380 Vdc in 400 Vac operation |
| Lexium 62 Drive | * 380 Vdc in 400 Vac operation * 180 Vdc in 230 Vac operation |
| Lexium 62 Power Supply | * 400 Vdc in 400 Vac operation * 200 Vdc in 230 Vac operation |
| Lexium 62 ILM | * 380 Vdc in 400 Vac operation * 180 Vdc in 230 Vac operation |
| Multi carrier transport system | 40.8 Vdc |

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| The mains voltage is too low. | Verify and correct if necessary the mains voltage (see technical data). |
| The power supply has detected an error and the Ready signal is switched off. The motor has run on until the DC bus voltage has decreased too far. | Remove error in the power supply. |
| The DC bus was overloaded. | * Run motors with reduced acceleration. * Adapt motion profiles. |

NOTE: The braking resistor is controlled also after an undervoltage error detected in the power supply unit.

This can be necessary if too much energy is supplied in the DC bus because the motors slow down on a mains power outage.

NOTE: After this diagnostic message has been triggered, the braking resistor (bleeder) is switched to the precharge function.

This is why it is not possible to extract feed in energy from the drive via the braking resistor. The braking resistor can only be used to decrease feed back energy after a successful precharge

Refer to parameter InternalDeviceState, 16#89 | 137: Wait until bleeder resistor is ready for the precharge.

| For the Multi carrier transport system: | |
| --- | --- |
| Cause | Solution |
| The power supply of the Multi carrier transport system is overloaded. | Verify the dimensioning of the power supplies. Connect additional power supplies if necessary. |
| The voltage drop between the Multi carrier transport system and the connection module is too high. | Verify power cable for correct length and cross-section. |
