# 8727 Session Timeout

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

The connection between a browser and the visualization (WebVisu) has timed out.

Potential causes include:

* The browser did not close the session.
* The browser closed the session, but the controller did not receive this information.
* The browser window stopped working.
* The connection is lost.
* A connection was terminated without a logout.
