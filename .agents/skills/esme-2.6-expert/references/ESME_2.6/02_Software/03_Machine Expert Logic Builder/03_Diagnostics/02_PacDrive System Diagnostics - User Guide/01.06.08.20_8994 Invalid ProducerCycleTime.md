# 8994 Invalid ProducerCycleTime

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

## Ext. diagnosis:

`xxxxxxx ns`

with `xxxxxxx` = Value of the incorrect configured parameter ProducerCycleTime.

This message is triggered by the Sercos phase-up if the parameter ProducerCycleTime of a Sercos I/O device is not an integer multiple of the Sercos cycle time.
