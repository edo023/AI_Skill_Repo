# 8110 Phase missing

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | BD | CD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | BP2 | BP2 |

This diagnostic message is triggered if a phase of the mains connection is interrupted for more than 1 second.

The drive (Lexium 52) is stopped to protect the device from overload.

| Cause | Solution |
| --- | --- |
| A phase of the mains connection is not connected to the device. | Verify and correct if necessary the wiring of the mains connection. |
