# 8146 DC bus overload

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | - | - |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | BP1 | BP1 |

The DC bus is overloaded. In this case, unlike diagnostic message [8107 Overcurrent](D-SE-0063849.html#D-SE-0063849), switch off does not occur as soon as the current threshold has been exceeded. The thermal overload of the device is observed for a specific time.

| Cause | Solution |
| --- | --- |
| Too many loads are connected to the DC bus. | * It may be that the existing device constellation is sufficient by adapting the application (for example, less acceleration). * Use another power supply and distribute supply of loads. |
