# 8189 Safety module communication not possible

## Description

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |

The drive could not communicate with the safety module.

| Cause | Solution |
| --- | --- |
| The safety module detected an error and changed to the defined safe state. | * Power cycle the device. Restart your system. * Verify the Safe Logger for diagnostic messages triggered by the safety module. |
| Device is inoperable. | Replace device. |
