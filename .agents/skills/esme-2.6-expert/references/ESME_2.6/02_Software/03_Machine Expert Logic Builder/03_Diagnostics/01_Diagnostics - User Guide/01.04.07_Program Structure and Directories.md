# Program Structure and Directories

## Overview

The directory of the Diagnostics contains further subdirectories. You can update them manually, if required.

| Directory | Description |
| --- | --- |
| ExternalResource | This folder and its subfolders contain files which come from external sources, such as Logic Builder or external configuration files for file views. |
| ExternalResource\Epas | Contains files from Logic Builder. |
| ExternalResource\Epas\ConfigFiles | Content of the subdirectory ConfigFiles from Logic Builder. If errors are detected when [collecting data from the controller](D-SE-0041405.html#D-SE-0041405), you can manually add the complete content of the subfolder ConfigFiles of your latest Logic Builder version to this subdirectory. This overwrites any pre-existing files with identical names. |
| Help | Among other things, this subfolder contains the Diagnostics online help file. |
| Print | This subfolder contains template files for printing. |
