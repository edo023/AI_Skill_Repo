# 8341 CamTrack invalid Position type

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

The CamTrack in the parameter PositionType contains an invalid value.

| Cause | Solution |
| --- | --- |
| No valid position type (parameter PositionType) of the position source Position-Source ([8327 CamTrack invalid PositionSource](D-SE-0063930.html#D-SE-0063930)) has been entered. | Verify and correct if necessary the parameters PositionType and PositionSource. |
