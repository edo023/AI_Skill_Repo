# 8720 No module found

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

Module identification could not be found.

A `DIO8` (optional module of Lexium 62 ILM) was configured on the Lexium 62 ILM Drive Module and activated (Enable = TRUE), but not detected during the Sercos phase boot.

| Cause | Solution |
| --- | --- |
| The firmware file on the flash disk could not be found or is incorrect. | Verify if a valid firmware file is present and replace it if necessary. |
| Optional module not found. | Install module or send in the controller/drive for retrofitting. |
| An optional module was entered in the PLC configuration that is not available. | Verify and correct if necessary the PLC configuration. |
| A **phys. master encoder** (SinCos-Geber) is entered in the PLC configuration on the **BT-4/ENC1**, but no phys. master encoder is detected on the connector **BT-4/ENC1**. | * Plug the incremental encoder cable in at the BT-4/ENC1 connection. * Verify and replace if necessary the encoder cable. |
| The I/O address is incorrectly set. | Verify and correct if necessary the I/O address. |
| Detected hardware error: Module inoperable. | Replace the module. |
| Detected hardware error: Controller is inoperable. | Replace the controller. |
