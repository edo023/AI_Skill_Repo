# 8337 Parameter DynIECData value too high

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

Filter type 1 **general system messages**

The dynamic memory area, which was specified with the DynIECDataSize parameter, could not be reserved due to insufficient memory (RAM). During program start, a check is performed to determine if there is at least 4 Mb of available memory after the dynamic memory area is reserved (also see the parameter Memoryfree).

| Cause | Solution |
| --- | --- |
| The dynamic memory area to be reserved is too high. | Reduce the value of the parameter DynIECDataSize. |
