# 8781 Master Encoder no connection

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The jumper in the encoder cable connector to the controller was not detected.

| Cause | Solution |
| --- | --- |
| Detected wiring error: Encoder cable to the controller removed. | Insert in the encoder cable. |
| Detected wiring error: Encoder cable is inoperable. | Verify the encoder cable and replace if necessary. |

Additional possible cause by PacDrive LMC Eco:

| Cause | Solution |
| --- | --- |
| The connected encoder does not match the configured encoder. (Incremental encoder connected, SinCos encoder configured). | Adjust the configuration of the encoder or exchange the connected encoder. |
