# 8047 PIC firmware download

## Description

[Refer to Diagnostic class (standard): 1](D-SE-0063169.html#D-SE-0063169)

The firmware of the PIC controller is replaced. In DiagExtMsg, the file name of the firmware file can be read.

If an error occurs during the firmware exchange, the [8826 PCI update error](D-SE-0064072.html#D-SE-0064072) diagnostic message is triggered.
