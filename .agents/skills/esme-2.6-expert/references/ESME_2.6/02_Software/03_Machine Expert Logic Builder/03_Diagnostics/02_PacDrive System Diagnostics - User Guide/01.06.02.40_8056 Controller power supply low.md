# 8056 Controller power supply low

## Description

[Refer to Diagnostic class (standard): 1](D-SE-0063169.html#D-SE-0063169)

`MsgFilter: Bit 1 Diagnosis Messages`

The hardware provides an internal error logger that logs the causes for error-dependent controller deactivations. The next time the controller is started up the error logger is analyzed and the cause of the error can be indicated in the message logger. If the reason for the shutdown was a supply voltage outage, the message 8056 Controller power supply low is provided.

In addition, a more specific message is provided ([8055 Controller message HW monitor](D-SE-0063837.html#D-SE-0063837)) that can be enabled by the MsgFilter parameter and activated via the Logic Builder message logger.

This message is intended to be a debug message and provide information on the cause of the deactivation.
