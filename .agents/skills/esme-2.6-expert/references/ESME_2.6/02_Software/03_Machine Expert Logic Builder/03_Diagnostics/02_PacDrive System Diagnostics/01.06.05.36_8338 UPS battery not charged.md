# 8338 UPS battery not charged

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The battery is not sufficiently charged.

## Ext. diagnosis: State = 2

| Cause | Solution |
| --- | --- |
| The PacDrive controller has been switched off for too long. | Switch on the PacDrive controller.   * The battery is charged automatically when the PacDrive controller is on. |
| The UPS has been operating in the battery mode (State = 3) for an extended period. | Verify and correct if necessary the supply voltage. |
| The battery capacity is insufficient due to aging. | Contact your local Schneider Electric service representative. |
