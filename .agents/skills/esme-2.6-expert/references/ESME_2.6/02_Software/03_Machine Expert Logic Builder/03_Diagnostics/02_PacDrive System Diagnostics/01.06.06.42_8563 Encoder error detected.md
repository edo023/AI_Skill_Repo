# 8563 Encoder error detected

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | AD | – |

An error was detected in the position measuring system (encoder).

| Cause | Solution |
| --- | --- |
| The encoder system is disturbed by external magnetic fields. | Remove the external sources of interference (magnetic fields) and restart the Multi carrier transport system. |
| Detected multi carrier hardware error: The encoder magnet of a carrier is inoperable. | Verify the encoder magnet of the carrier. Replace the encoder magnet or the carrier if necessary. |
| Detected multi carrier hardware error: The encoder system of a segment is inoperable. | Replace the multi carrier segment. |
