# 8574 Unconfigured physical carrier found

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

When the Sercos bus is started up, the configured carriers from the Devices tree are linked to the physical carriers on the multi carrier track during switch to phase 2.

This diagnostic message is displayed in the track object if one or more physical carriers could not be linked to a logical carrier from the project.

In this case, physical carriers would not be controlled by the project.
