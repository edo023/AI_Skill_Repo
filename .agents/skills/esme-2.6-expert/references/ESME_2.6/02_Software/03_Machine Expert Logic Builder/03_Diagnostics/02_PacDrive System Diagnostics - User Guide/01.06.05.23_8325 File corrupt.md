# 8325 File corrupt

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

An issue (file non-existent or CRC error) is detected while reading the retain file. The retain file is read by the flash disk while the PacDrive controller restarts if the Enable parameter of the UPS object is set to External / 1.

| Cause | Solution |
| --- | --- |
| An UPS object is specified in the PLC configuration, but no UPS is connected. | Verify and correct if necessary the program. |
| The UPS is not functioning properly. | Verify and replace if necessary the UPS. |
