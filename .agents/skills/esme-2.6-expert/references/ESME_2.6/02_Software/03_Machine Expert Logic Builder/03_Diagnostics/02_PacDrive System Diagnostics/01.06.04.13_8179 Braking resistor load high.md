# 8179 Braking resistor load high

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | - | - |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | D | E |

Thermal overload of the braking resistor is greater than 90%.

| Cause | Solution |
| --- | --- |
| The drive sizing is incorrect. | Verify and correct if necessary the drive sizing. |
| Hardware error detected: The braking resistor or triggering is not operating properly. | Contact customer service. |

NOTE: You can use a parallel connection of power supplies.

* When the first power supply exceeds thermal overload of the braking resistor (> 90%), the diagnostic message 8179 Braking resistor load high is triggered.
* Before the first power supply reaches 100%, the second power supply joins.

Refer to [Parallel Connection of Several Lexium 62 Power Supplies (LXM62PD84A11000)](../../../../../api/crossBook?lang=en-US&virtualBookName=LXM62HW&topicID=D_SE_0052483).
