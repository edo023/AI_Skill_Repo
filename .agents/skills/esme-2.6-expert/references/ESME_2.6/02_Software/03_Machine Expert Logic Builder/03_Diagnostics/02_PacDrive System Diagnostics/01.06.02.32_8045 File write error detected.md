# 8045 File write error detected

## Description

[Refer to Diagnostic class (standard): 1](D-SE-0063169.html#D-SE-0063169)

An error was detected when writing in a file.

| Cause | Solution |
| --- | --- |
| Insufficient memory space. | * Verify, if the memory capacity is sufficient (parameter RamDiskFree, Diskfree). * Increase the RamDiskSize  Default: 1 Mbyte |
| An attempt was made to write a trace file to the RAM disk (ram0). | Increase the RamDiskSize.  Default: 1 Mbyte |
