# Remove Controller

## Overview

The RemoveController command removes a single controller from the controller list.

## Call

```
<ToolPath> -NetmanageCommand RemoveController <ID>
```

## Parameter Description

| Call parameters | Content |
| --- | --- |
| `<ToolPath>` | Path of the tool to be executed that supports the Network Device Identification command-line call, see example underneath this table. |
| `<ID>` | Unique ID of the controller (MAC address). |

## Example

Entry

```
"c:\Programs\Schneider Electric\Diagnostics\Diagnostics.exe" -NetmanageCommand RemoveController 00:04:17:05:00:78
```

## Result

By a successful processing of the call 0, otherwise 1.

EIO0000002005.05
