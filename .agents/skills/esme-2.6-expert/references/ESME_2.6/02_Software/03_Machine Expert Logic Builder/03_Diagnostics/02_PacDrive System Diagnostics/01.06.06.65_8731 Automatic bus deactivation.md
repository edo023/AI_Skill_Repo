# 8731 Automatic bus deactivation

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The Auto Clear Bit in the master is set and at least one slave is not exchanging cyclical I/O data. The error code specified after `Err=` corresponds to the error code of the ErrorCode parameter.

| Cause | Solution |
| --- | --- |
| Slave has been disabled. | Reactivate slave. |
| At least one slave is not configured correctly. | Verify and correct if necessary the configuration. |
| Detected wiring error: Cable error in bus. | Verify and correct or replace if necessary the cable. |
| Detected wiring error: Incorrect bus topology. | Verify and correct or replace if necessary the terminals, transmission lines, and so on. |
