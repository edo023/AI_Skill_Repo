# 8138 Motor/Drive combination not supported

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | BD2 | CD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

An impermissible motor is connected to the servo amplifier.

| Cause | Solution |
| --- | --- |
| Encoder interface not supported by the firmware. | * Use another encoder interface. * Replace firmware |
| Rated current of the holding brake to operate on this servo drive is too high. | Use a compatible servo drive.  Refer to the technical data in the operating manual. |

NOTE: Motors of the type SH3205xxxxFxxxx (SH3 205 series motors with brake) can only be operated with the following devices:

* LXM62D D45
* LXM62D C13
* The Lexium 52 devices

If an SH3205xxxxFxxxx motor is connected to any other device (LXM62D U60, LXM62D D15, or LXM62D D27), this diagnostic message is triggered.
