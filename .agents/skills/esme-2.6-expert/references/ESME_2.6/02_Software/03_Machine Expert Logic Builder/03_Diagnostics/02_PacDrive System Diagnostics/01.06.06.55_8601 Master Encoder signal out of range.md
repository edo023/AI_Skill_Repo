# 8601 Master Encoder signal out of range

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

Unable to receive analog data from the SinCos master encoder.

NOTE: Hardware encoder error. Reset the PacDrive controller.

| CAUTION | |
| --- | --- |
|  | POSITION LOSS DUE TO SERCOS BUS ERROR  Only acknowledge the diagnostic message after reinitialization or homing.  Failure to follow these instructions can result in injury or equipment damage. |

If necessary, use the functions FC\_SysReset() or FC\_PrgResetAndStart().

| Cause | Solution |
| --- | --- |
| Detected wiring error: The encoder cable is not plugged in correctly or is inoperable. | Verify the encoder cable and replace if necessary. |
| Detected hardware error: The SinCos encoder is inoperable. | Replace the encoder. |
| Detected hardware error: The PacDrive LMC is inoperable. | Replace the controller. |
