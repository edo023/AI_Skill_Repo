# 8190 Current control parameter reduced

For Lexium 62 Standard, Lexium 62 ILM and Lexium 52 used with the motors of the types SH3040 and BSH040 it is necessary to reduce the internal current control loop parameters to get a stable control behavior. This reduction is indicated by this diagnostic message.

EIO0000003533.07
