# 8789 PacNet comm. disturbance detected

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

A single bus error is detected.

| Ext. diagnosis | Meaning |
| --- | --- |
| `M=xx n.res` | The module with bus address xx is not responding. The transfer is disrupted.  Verify and correct if necessary the wiring of the module. |
| PacNet alert. An RX clock is missing. | Verify and correct if necessary the wiring of the module. |
