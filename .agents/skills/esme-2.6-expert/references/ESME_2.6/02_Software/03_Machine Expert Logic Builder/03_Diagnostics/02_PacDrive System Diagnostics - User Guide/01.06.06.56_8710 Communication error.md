# 8710 Communication error

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The exchange of cyclical data was interrupted.

NOTE: If response monitoring is deactivated, the diagnostic code 8710 is not triggered.

| Cause | | Solution |
| --- | --- | --- |
| The controlling device 1) interrupts the sending of cyclical data. | | Verify and correct if necessary the controlling device. |
| Cable error. | | Verify and correct if necessary the wiring. |
| The address of the controlled device 2) has been assigned twice in the network. | | Verify and correct if necessary the configuration. |
| 1) controlling device:   * Ethernet/IP: Scanner * PROFIBUS: Master * PROFINET: Controller * Sercos: Master   2) controlled device:   * Ethernet/IP: Adapter * PROFIBUS: Slave * PROFINET: Device * Sercos: Slave | | |
