# 8101 Power stage overtemperature

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | BD2 | CD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | – | – |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | BD2 | – |

* The cooling element of the PacDrive servo amplifier is too hot.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| Insufficient ventilation or ambient temperature too high. | * Verify and correct if necessary the device fans and ventilation slots (if available). * Verify and correct if necessary the ventilation in switching cabinet. * If installed, ensure that the air conditioning unit is functioning properly. |
| The power stage or the power rectifier is overloaded. | * Verify and correct if necessary the combination motor / drive. * Verify and correct if necessary the calculation of motor and drive. |
| Detected hardware error: The temperature sensor is inoperable. | Replace the device. |

* The temperature of the power stage of the multi carrier segment is too high.

| For the Multi carrier transport system: | |
| --- | --- |
| Cause | Solution |
| Insufficient ventilation or ambient temperature too high. | Lower the ambient temperature or improve the surface ventilation of the multi carrier segment. |
| Thermal transition between the multi carrier segment and the mounting plate is not sufficient. | Improve the thermal transition between the multi carrier segment and the mounting plate (for example, by using thermal paste). |
