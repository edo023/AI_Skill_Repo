# 8333 EncoderNet receiving data not possible

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

In the encoder network, an error is detected when distributing the encoder position.

## Ext. diagnosis: Dupl. Obj.

| Cause | Solution |
| --- | --- |
| Multiple initialization of object (SYN\_DIN, SYN\_DOUT). | Verify and correct if necessary the initialization of objects. |

## Ext. diagnosis: Int.Err=-xxx

| Cause | Solution |
| --- | --- |
| Internal error code. | Contact your local Schneider Electric service representative. |

## Ext. diagnosis: SyncLost

| Cause | Solution |
| --- | --- |
| Synchronization was lost while the object (SYN\_DIN, SYN\_DOUT) was active. | This can appear for a number of external reasons related to the network such as wiring, network traffic, performance or other related equipment. Verify and correct if necessary any external equipment or wiring disturbing communications. |

## Ext. diagnosis: No Data: xxx

| Cause | Solution |
| --- | --- |
| The number of data sets (xxx) that was not received sequentially exceeds the limit specified in DataCycleErrorLimit. | This can appear for a number of external reasons related to the network such as wiring, network traffic, performance or other related equipment. Verify and correct if necessary any external equipment or wiring disturbing communications. |

## Ext. diagnosis: Send error

| Cause | Solution |
| --- | --- |
