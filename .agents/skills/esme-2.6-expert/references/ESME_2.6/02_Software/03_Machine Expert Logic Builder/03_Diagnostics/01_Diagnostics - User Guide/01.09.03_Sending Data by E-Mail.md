# Sending Data by E-Mail

## Overview

These instructions describe step by step how to send data from Diagnostics via e-mail.

The following description refers to the Microsoft Outlook provided by the operating system:

| Step | Action |
| --- | --- |
| 1 | On the [**Home** window](D-SE-0041404.html#D-SE-0041404), click the  Send to... button. |
| 2 | Enter any missing information in the following dialog box [**Machine info**](D-SE-0041429.html#D-SE-0041429).  **Result:** A progress bar shows how the data is processed and transmitted to the default e-mail program of your computer.  **Result:** The Send dialog box of Microsoft Outlook opens. |
| 3 | Now you can use the  New Message dialog box to send the e-mail with the attached controller data. Usually, the subject line and text body contain automatically generated entries, which you can modify according to your own needs. |
| 4 | Click the Send button. |

NOTE: You must be connected to the Internet or a network which is connected to the Internet in order to send data. Normally, you need to open your e-mail program and explicitly send your message (also refer to the description of your e-mail program).
