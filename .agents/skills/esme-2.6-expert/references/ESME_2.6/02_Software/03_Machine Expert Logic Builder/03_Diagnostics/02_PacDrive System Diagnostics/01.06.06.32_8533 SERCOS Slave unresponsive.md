# 8533 SERCOS Slave unresponsive

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

A Sercos device does not respond. This message is only triggered if the parameter SystemReaction (Sercos Robustness Function) of the Sercos device is set to systemReaction 1.

| Cause | Solution |
| --- | --- |
| The Sercos device no longer updates data | * Verify the Sercos cable and replace if necessary * Verify the Sercos device and replace if necessary |
