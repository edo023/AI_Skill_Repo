# 8517 Sercos addressing not unique

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

It is impossible to affect a clear allocation of the slave to a real Sercos slave.

| Cause | Solution |
| --- | --- |
| Several real slaves with matching identification features were found. | Verify and correct if necessary the parameter IdentificationMode and the corresponding identification feature. |
