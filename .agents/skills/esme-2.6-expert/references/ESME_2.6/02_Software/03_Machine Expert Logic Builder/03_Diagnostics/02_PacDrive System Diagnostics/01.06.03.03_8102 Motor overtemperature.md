# 8102 Motor overtemperature

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | BD2 | CD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | – | – |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | BD2 | – |

* Before the diagnostic message 8102 Motor overtemperature, the diagnostic message [8127 Motor temperature high](D-SE-0063866.html#D-SE-0063866) is triggered.

  The triggering threshold is provided by the electronic nameplate of the connected motor.

  The motor temperature is measured either by a temperature sensor or by a thermal model.

  The MotorTemperatureMonitoring parameter is used to determine how the motor temperature is measured.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| Detected wiring error: The temperature switch in the motor winding is not connected properly. | Verify and correct if necessary the wiring. |
| Detected wiring error: The shielding of the motor cable is not attached properly. | Verify and correct if necessary the shielded terminal on the bottom of the servo amplifier and the terminal boxes of the motor. |
| Holding brake not vented. | Release the brake. |
| The motor is overloaded. | * Drive the system at a slower speed. * Verify and correct if necessary the drive sizing. * Use a larger motor for this application. |
| Detected commutation error: The encoder is not aligned / leveled. | Align / level the encoder. |
| Detected hardware error: The temperature switch in the motor winding is inoperable. | Replace the motor. |

* The temperature of the multi carrier segment coils is too high.

|  |  |
| --- | --- |
| **For the Multi carrier transport system:** | |
| **Cause** | **Solution** |
| The multi carrier segment coils are overloaded. | Verify the loading conditions on the carrier (friction, load, external forces). |
| The cooling of the multi carrier segment is not sufficient. | Improve the thermal transition between the multi carrier segments and the mounting plate (for example, by using thermal paste). |
