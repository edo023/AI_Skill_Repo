# 8740 Device cannot be disabled

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

An attempt was made in online mode to disable the device via the parameter Enable. The device can only be disabled via this parameter in offline mode.

| Cause | Solution |
| --- | --- |
| An attempt was made to disable the device in online mode by setting the parameter Enable to "off / 0". | Set the parameter to "off / 0" in offline mode. |
