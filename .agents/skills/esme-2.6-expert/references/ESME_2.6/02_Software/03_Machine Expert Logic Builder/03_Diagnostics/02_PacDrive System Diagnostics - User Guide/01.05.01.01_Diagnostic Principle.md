# Diagnostic Principle

## Description

The PacDrive LMC system supports different diagnostic messages. A diagnostic class is assigned to each diagnostic message. The diagnostic class indicates the criticality of a detected error.

The error reaction performed on the system level depends on the diagnostic class. Additional reactions may be performed independently in the connected devices.

NOTE: In the PacDrive system, the term "actual" refers to real time, real world measured values. The term "reference" refers to values that are derived or otherwise calculated.
