# 8974 Brake voltage too low

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| Lexium 62, Lexium 62 ILD , Lexium 52 | D | E |
| Lexium 62 ILM | BD1 | D |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

The detected error is set if the control voltage of the device (24 V direct voltage) is insufficient to release the brake. It can be set before or after the release of the brake. The brake voltage is constantly monitored. If the value of the brake voltage is lower than specified in the technical data sheet of the device, then the detected error is set.

| Cause | Solution |
| --- | --- |
| - | Verify whether the control voltage of the device (24 V direct voltage) is within the specified limit values (see technical data sheet of the device) when the brake is released and correct if necessary. |
