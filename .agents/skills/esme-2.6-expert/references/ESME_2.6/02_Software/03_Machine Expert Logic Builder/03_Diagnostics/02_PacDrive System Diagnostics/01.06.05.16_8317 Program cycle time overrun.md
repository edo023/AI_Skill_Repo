# 8317 Program cycle time overrun

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

A cycle time error is detected. This detected error is triggered when the cycle time is exceeded by more than one system clock (default value for a system clock is 250 µs). The DiagExtMsg shows the current cycle time / cycle time from the task configuration in ms.

NOTE: If the source of a diagnostic message is an IEC task, then it can only be displayed, if the task configuration of the application was not changed at the time of the diagnostic outlet. After the download of a changed application, no task name is displayed in the field Instance.
