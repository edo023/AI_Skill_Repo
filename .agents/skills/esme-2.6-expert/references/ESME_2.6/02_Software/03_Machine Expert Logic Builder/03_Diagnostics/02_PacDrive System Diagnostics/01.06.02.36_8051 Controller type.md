# 8051 Controller type

## Description

[Refer to Diagnostic class (standard): 1](D-SE-0063169.html#D-SE-0063169)

The PacDrive controller was restarted. Shortly before the start procedure ends, the controller type is saved as a diagnostic message in the message logger.

NOTE: The controller type may not be displayed in full because the message text may not have sufficient character space.
