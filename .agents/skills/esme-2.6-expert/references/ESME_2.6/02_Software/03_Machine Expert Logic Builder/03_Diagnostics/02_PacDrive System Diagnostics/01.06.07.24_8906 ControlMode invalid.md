# 8906 ControlMode invalid

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

| Cause | Solution |
| --- | --- |
| * The parameter ControlMode has an invalid value. * ControlMode is set to `open-loop control / 1` and a synchronous motor is used (MotorClass is `Synchronous motor / 0`). * ControlMode is set to `Encoderless velocity closed-loop control / 2` and an asynchronous motor is used. * ControlMode is set to `open-loop control / 1` or `Encoderless velocity closed-loop control / 2` and a linear motor is used. | Synchronous motors can only be operated in closed-loop control, change ControlMode to `closed-loop control / 0` or `Encoderless velocity closed-loop control / 2`.  Asynchronous motors can not be operated with encoderless velocity closed loop control, change ControlMode to `closed-loop control / 0`or `open-loop control / 1`.  Linear motors can only be used with closed-loop control, change ControlMode to `closed-loop control / 0`. |
