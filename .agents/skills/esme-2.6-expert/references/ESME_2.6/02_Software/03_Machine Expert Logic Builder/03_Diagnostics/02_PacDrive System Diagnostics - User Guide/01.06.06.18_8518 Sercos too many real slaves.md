# 8518 Sercos too many real slaves

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The number of real Sercos drives is too great. (Only drives are taken into account as Sercos slaves).

Ext. diagnosis: **HW <maximum number of Sercos slaves on the controller>/<Number of configured Sercos slaves>**

| Cause | Solution |
| --- | --- |
| Limited by the controller type:  The configured number of servo drives in closed-loop control exceeds the maximum number for the PacDrive controller type, and the Sercos phase-up is canceled. | Reduce the number of configured servo drives. |
