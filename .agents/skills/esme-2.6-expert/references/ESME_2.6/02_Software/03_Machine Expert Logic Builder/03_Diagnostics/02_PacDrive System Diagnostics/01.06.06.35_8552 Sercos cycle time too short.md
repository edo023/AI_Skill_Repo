# 8552 Sercos cycle time too short

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | AD | – |

The configured Sercos cycle time is too short for the system. Therefore a correct function of the multi carrier transport system is not possible.

| Cause | Solution |
| --- | --- |
| Excessive number of Sercos devices in the system. Excessive ring delay for the cycle time. | Increase the Sercos cycle time to the next higher value. |
