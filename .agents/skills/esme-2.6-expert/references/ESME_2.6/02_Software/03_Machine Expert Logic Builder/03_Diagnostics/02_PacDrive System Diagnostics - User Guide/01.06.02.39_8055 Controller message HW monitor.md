# 8055 Controller message HW monitor

## Description

[Refer to Diagnostic class (standard): 1](D-SE-0063169.html#D-SE-0063169)

`MsgFilter`: Bit 14 `Filter Type 15`

Refer to the diagnostic messages [8056 Supply voltage low](D-SE-0063838.html#D-SE-0063838) and [8827 Controller power-off/hardware monitor](D-SE-0064074.html#D-SE-0064074).
