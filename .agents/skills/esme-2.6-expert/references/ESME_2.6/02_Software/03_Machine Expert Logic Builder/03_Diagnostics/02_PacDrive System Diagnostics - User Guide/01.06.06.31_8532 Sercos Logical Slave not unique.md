# 8532 Sercos Logical Slave not unique

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

If a physical slave cannot be uniquely assigned to a logical slave, all suitable logical slaves send such a message.

| Ext. Diagnosis | Meaning |
| --- | --- |
| `Topol.Adr=<address>` | Topological address of the physical slave. |
