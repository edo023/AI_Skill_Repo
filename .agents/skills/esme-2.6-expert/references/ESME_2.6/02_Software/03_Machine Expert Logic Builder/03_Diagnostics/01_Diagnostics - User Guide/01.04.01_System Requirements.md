# System Requirements

## Hardware Requirements

The PC for Diagnostics installation must meet the following hardware requirements:

| Equipment | Minimum | Recommended |
| --- | --- | --- |
| Processor | Intel® CoreTM 2 Duo (or equivalent) | Intel® CoreTM i7 (or equivalent) |
| RAM | 2 GB | 4 GB |
| Free hard drive space | 6 GB | 12 GB |
| Monitor | Resolution: 1024 x 768 pixel | Resolution: 1280 x 1024 pixel |
| Drive | DVD drive | |
| Peripherals | Mouse | |
| Keyboard | |
| USB interface | |
| Network interface | Ethernet | |

## Software Requirements

The Diagnostics tool supports the following operating systems:

* Microsoft Windows 7 32 bit
* Microsoft Windows 7 64 bit
* Microsoft Windows 8.1 32 bit
* Microsoft Windows 8.1 64 bit (with minor restrictions)
* Microsoft Windows 10 32 bit
* Microsoft Windows 10 64 bit
* Microsoft .NET Framework 4.7
