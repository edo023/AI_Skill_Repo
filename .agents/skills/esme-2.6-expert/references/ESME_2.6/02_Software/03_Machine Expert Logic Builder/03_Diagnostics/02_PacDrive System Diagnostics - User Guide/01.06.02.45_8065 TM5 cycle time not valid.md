# 8065 TM5 cycle time not valid

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

DiagExtMsg: `ModuleNbr.x`

x = number of the module under the Sercos III bus interface (TM5NS31)

| If... | Then... |
| --- | --- |
| The module is used with one of the following configuration profiles:   * TM5SDM8DTS\_TSITSO * TM5SDM8DTS\_OITSO * TM5SDM8DTS\_HFOIDO * TM5SDM8DTS\_DIHFOO | This diagnostic message is triggered by the TM5SDM8DTS module during Sercos phase change to phase 2. |
| The value of the TM5CycleTime parameter (parameter of the Sercos III bus interface TM5NS31) is less than 500 µs. |

The Sercos phase up is stopped and the diagnostic message is triggered.

For example, the following configuration triggers the diagnostic message [8065 TM5 cycle time not valid](#D-SE-0063842) with DiagExtMsg: `ModuleNbr.2` as the TM5SDM8DTS\_HFOIDO does not work with the configured TM5CycleTime.

* Sercos III bus interface (TM5NS31) with TM5CycleTime = 250 µs

  + TM5SPS3
  + TM5SDM8DTS\_HFOIDO
