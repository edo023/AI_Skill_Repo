# Remove Inactive Controllers

## Overview

The RemoveAllInactiveController command removes the controllers that do not respond to a network scan, which are indicated by a red cross in the controller list.

## Call

```
<ToolPath> -NetmanageCommand RemoveAllInactiveController
```

## Parameter Description

| Call parameters | Content |
| --- | --- |
| `<ToolPath>` | Path of the tool to be executed that supports the Network Device Identification command-line call, see example underneath this table. |

## Example

Entry

```
"c:\Programs\Schneider Electric\Diagnostics\Diagnostics.exe" -NetmanageCommand RemoveAllInactiveController
```

## Result

By a successful processing of the call 0, otherwise 1.
