| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| The brake voltage measured by the servo amplifier is lower than the necessary voltage to maintain or release the brake of the motor. | Verify and replace if necessary the 24 V power supply. |
| A short-circuit has occurred in the DC bus (drive side). | Replace the device. |
| Internal error in the device is detected. | Replace the device. |

| --- | --- |
| Cause | Solution |
