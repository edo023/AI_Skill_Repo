# 8119 Power stage short-circuit / ground fault

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

* The motor and motor lines are verified for short circuits (phase short circuit).
* An MH3 motor indicating a diagnostic message [8122 Shutdown due to velocity limit](D-SE-0063862.html#D-SE-0063862) can also provoke this diagnostic message (8119 Power stage short-circuit / ground fault).

| Cause | Solution |
| --- | --- |
| Detected wiring error: Short circuit in motor cable. | Replace the motor cable. |
| Detected hardware error: Short circuit in the motor (phase to phase or phase to housing). | Replace the motor. |
| Detected hardware error: Power stage in servo drive inoperable. | Replace the servo drive. |
