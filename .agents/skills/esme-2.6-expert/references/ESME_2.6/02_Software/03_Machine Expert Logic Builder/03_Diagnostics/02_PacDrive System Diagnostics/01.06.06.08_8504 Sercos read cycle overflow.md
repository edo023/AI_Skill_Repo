# 8504 Sercos read cycle overflow

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The transfer of the actual position of the axes (servo amplifiers) to the PacDrive controller proceeds too late. The actual values of the next Sercos cycle have already been read. The actual velocity for the program may not be correct.

| Cause | Solution |
| --- | --- |
| Sercos bus is overloaded. | Increase CycleTime parameter in the PLC configuration. |
| A device in the Sercos loop is switched off or was reset. | Switch on Sercos slave. |
