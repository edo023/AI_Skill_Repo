# 8133 Speed-dependent current reduction

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | D | E |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

This is a protective measure to help protect the power stage of the servo amplifier from high thermal load when moving to the stop position.

When moving to stop, the reference position of the drive may not be reached. Through the integral part of the speed controller, this static tracking deviation is integrated so that the reference current increases up to the current limitation. The diagnostic message can be triggered if the current exceeds 70% of the peak current of the power stage.

The following conditions must be met to trigger the message:

* The frequency of the output voltage has to be less than 5 Hz.

  The frequency corresponds to a velocity limit VelocityLimit in units per second, and is calculated as follows:

  VelocityLimit = `5 Hz * FeedConstant / PolePairs`.

  Refer to the operating instructions of the motor.
* The current must be greater than 70% of the peak current of the power stage.

  Refer to the operating instructions of the drive system.
* For peak current of the power stage, the message is triggered after 100 ms.

  For currents that are below the peak current it takes correspondingly longer.

| Cause | Solution |
| --- | --- |
| The axis is moved to a block. | * Verify and correct if necessary the traversing range of the axis. * Reduce peak currents with UserDrivePeakCurrent or UserCurrentLimit. |
| The brake was not opened. | Verify and correct if necessary the wiring of the brake. |
