# 8163 Sercos Slave C1D error detected

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

Reaction: manufacturer-specific

The diagnostic message is triggered by the following objects:

* Sercos drive
* TM5NS31 (S3IO)
* Safe Logic Controller

The extended diagnostic message (ExtDiagMsg) can be displayed in two different formats:

* Diag = 16#aaaa with aaaa is the error code from Sercos IDN S-0-0011.

  Refer to the [Sercos C1D Error Messages](#D-SE-0063884__D-SE-0063884.5) for details.
* Diag = 0xaaaaaaaa with aaaaaaaa corresponds to the diagnostic message from the parameter *[DiagnosticNumber](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.BusInterfaceTM5NS31&topicID=D_SE_0086074)* (Sercos IDN S-0-0390).

## Sercos C1D Error Messages

Format of the ExtDiagMsg:

`Diag = 16#aaaa` with `aaaa` is the error code from Sercos IDN S-0-0011.

A Sercos error of the diagnostic class 1 (C1D) has been detected. The extended diagnostic contains a data word that provides information on the cause.

The table shows the individual bits of this data word and their meanings (Sercos C1D error (assignment according to the Sercos specification IDN S-0-0011)):

| Bit | Meaning |
| --- | --- |
| Bit 0 | Overload shut-down |
| Bit 1 | Amplifier overtemperature shut-down |
| Bit 2 | Motor overtemperature shut-down |
| Bit 3 | Cooling error shut-down |
| Bit 4 | Control voltage error |
| Bit 5 | Feedback error |
| Bit 6 | Error in the commutation system |
| Bit 7 | Overcurrent error |
| Bit 8 | Overvoltage error |
| Bit 9 | Undervoltage error |
| Bit 10 | Power supply phase error |
| Bit 11 | Excessive position deviation |
| Bit 12 | Communication error |
| Bit 13 | Overtravel limit is exceeded (shut-down) |
| Bit 14 | Reserved |
| Bit 15 | Manufacturer-specific error (refer to diagnostic message [8164](D-SE-0063885.html#D-SE-0063885).) |
