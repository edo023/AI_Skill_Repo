# Diagnostic Messages Overview

## Description

This table provides an overview of the diagnostic messages of the following devices:

* Lexium 52
* Lexium 62 Power Supply
* Lexium 62 Single Drive
* Lexium 62 Double Drive
* Lexium 62 Advanced Plus
* Lexium 62 ILD
* Lexium 62 ILM
* Lexium™ MC multi carrier (multi carrier transport system)

For information on the device reactions AD, BD1, and so on, refer to [Device Reactions - Drive](D-SE-0063389.html#D-SE-0063389) and [Device Reactions - Power Supply](D-SE-0063391.html#D-SE-0063391).
