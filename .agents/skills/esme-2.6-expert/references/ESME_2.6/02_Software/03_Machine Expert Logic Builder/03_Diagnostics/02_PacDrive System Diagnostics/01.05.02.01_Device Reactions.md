# Device Reactions

## General Information on Device Reactions

Device reactions are caused by an event taking place in a connected drive (Lexium 52, Lexium 62 Single Drive / Lexium 62 Double Drive, Lexium 62 ILM, Lexium 62 Advanced Plus, Lexium 62 ILD), power supply unit (Lexium 52, Lexium 62 Power Supply) or Multi carrier transport system.

The device reaction takes place in the device which triggered the reaction. When a device reaction has been triggered, a diagnostic message, which can be evaluated on the superordinate controller, is transmitted.

## Minimum Reaction

A configured reaction must have the same or greater priority than the defined minimum reaction.
