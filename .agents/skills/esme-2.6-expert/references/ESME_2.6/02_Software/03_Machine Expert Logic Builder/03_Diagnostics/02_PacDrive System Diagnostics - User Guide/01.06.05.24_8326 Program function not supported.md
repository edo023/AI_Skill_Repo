# 8326 Program function not supported

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

A function has been called that is not supported by the PacDrive system. Further information on **Object type** and **Instance** are displayed in the message logger. The DiagExtMsg also provides information on the cause of the detected error.

| Cause | Solution |
| --- | --- |
| A function is not supported by the hardware used. | Verify the hardware version with the help of the nameplate. |
