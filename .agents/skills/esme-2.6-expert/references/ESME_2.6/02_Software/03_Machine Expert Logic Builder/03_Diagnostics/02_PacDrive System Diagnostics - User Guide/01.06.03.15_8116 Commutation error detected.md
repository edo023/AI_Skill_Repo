# 8116 Commutation error detected

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | AD | AD |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | - | - |

Unable to determine the commutation of the motor. The commutation is the information where the magnets of a synchronous motor are physically placed relative to the encoder position.

For motor encoders that are coupled to the motor non-inverted and without a gearbox in between, this information is stored in the motor type plate. In the other cases the commutation has to be detected. This detection is controlled by the parameter MotorCommutationControl.

| Cause | Solution |
| --- | --- |
| Motor turns when starting (boot procedure) the servo amplifier. | Ensure that the motor is still when starting (boot procedure) the servo amplifier. |
| Detected wiring error: encoder cable is not plugged in or inoperable. | Verify and correct if necessary the wiring. |
| Detected error when determining the motor commutation (MotorCommutationControl). | The cause is displayed via the parameter MotorCommutationState. |
