# 8111 Shutdown due to tracking deviation

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Version | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |
| --- | --- | --- | --- |
| [Drive](D-SE-0063389.html#D-SE-0063389) | PacDrive 3 V 3.1 | BD1 | BD2 |
| [Drive](D-SE-0063389.html#D-SE-0063389) | PacDrive 3 V 3.1 SP1 and greater | BD2 | BD2 |
| Power supply | - | - | - |
| [Multi carrier transport system](DeviceReactions-MultiCarrier-36D130D7.html#DeviceReactions-MultiCarrier-36D130D7) | - | BD2 | - |

* A major tracking deviation has occurred which is greater than 8 times the value of the parameter TrackingDeviationLimit.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| This error normally occurs in connection with the CAM movement functions (for example, `MultiCam()` ). | Verify and correct if necessary curve data. |
| The axis should be operated at regulated speeds. | Set parameter Pos\_P\_Gain to `0`. |

* The tracking deviation of the Multi carrier transport system is excessive.

| For the Multi carrier transport system: | |
| --- | --- |
| Cause | Solution |
| Excessive acceleration value of the carrier. | * Verify the controller parameters. * Reduce the acceleration value. |
| The carrier is mechanically blocked. | Verify that the carrier can move freely. |
