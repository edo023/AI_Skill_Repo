# 8064 TM5 I/O data length exceeded

## Description

[Refer to Diagnostic class (standard): 3](D-SE-0063169.html#D-SE-0063169)

The I/O data length exceeded the maximum possible data length. The maximum possible data length is based on the TM5 cycle time.

| Cause | Solution |
| --- | --- |
| Excessive synchronous I/O data. | Increase the TM5 cycle time. |
