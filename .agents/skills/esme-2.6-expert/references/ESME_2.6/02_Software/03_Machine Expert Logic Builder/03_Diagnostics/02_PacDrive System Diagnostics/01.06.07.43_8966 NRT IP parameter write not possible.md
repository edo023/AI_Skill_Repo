# 8966 NRT IP parameter write not possible

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| Cause | Solution |
| --- | --- |
| * The Sercos III parameter writing process cannot be executed. * The parameters IPAddress, Subnet mask, and Gateway are transferred from the slaves via Sercos III. (By IPConfigMode = `Automatic IP addressing / 0` or `manual IP addressing (software) / 2`) | Verify whether the device supports the parameters and correct if necessary. |
