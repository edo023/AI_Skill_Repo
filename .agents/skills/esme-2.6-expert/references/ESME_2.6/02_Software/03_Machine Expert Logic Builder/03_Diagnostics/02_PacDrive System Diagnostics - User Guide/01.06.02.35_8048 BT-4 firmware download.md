# 8048 BT-4 firmware download

## Description

[Refer to Diagnostic class (standard): 1](D-SE-0063169.html#D-SE-0063169)

The firmware of the bus terminal BT-4/DIO1 or BT-4/ENC1 is replaced.

Other requirements:

* Firmware file for BT-4/DIO1: BD1\_xxxx.ESV
* Firmware file for BT-4/ENC1: BE1\_xxxx.ESV

In order to execute the update, the firmware file for the BT-4/DIO1 or BT-4/ENC1 bus terminal must be copied to the root directory of the controller. The update will be automatically executed at the next startup.

NOTE: After the update, the BT-4/DIO1 or BT-4/ENC1 bus terminals must be switched off and on. A standard reset on the controller is sufficient. The update is only executed if a bus terminal has a differing firmware version.
