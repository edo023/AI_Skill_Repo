# 8327 CamTrack invalid PositionSource

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

The CamTrack in the PositionSource parameter contains an invalid value.

| Cause | Solution |
| --- | --- |
| The symbolic name of a logical encoder or an axis was not specified in the PositionSource parameter. | Verify and correct if necessary the value in the PositionSource parameter. |
