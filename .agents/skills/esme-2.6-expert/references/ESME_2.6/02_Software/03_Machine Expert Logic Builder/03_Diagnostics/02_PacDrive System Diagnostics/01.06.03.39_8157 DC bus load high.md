# 8157 DC bus load high

## Description

[Refer to Diagnostic class (standard): 2](D-SE-0063169.html#D-SE-0063169)

| [Device reactions](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.2) | Default reaction | [Minimum reaction](D-SE-0063393.html#D-SE-0063393__D-SE-0063393.4) |

| [Drive](D-SE-0063389.html#D-SE-0063389) | - | - |
| [Power supply](D-SE-0063391.html#D-SE-0063391) | D | E |

The overload of the DC bus is at 80 %. This is indicated by the parameter DC\_BusOverload. If the overload increases to 100 %, then the diagnostic message [8146 DC bus overload](D-SE-0063880.html#D-SE-0063880) is triggered.
