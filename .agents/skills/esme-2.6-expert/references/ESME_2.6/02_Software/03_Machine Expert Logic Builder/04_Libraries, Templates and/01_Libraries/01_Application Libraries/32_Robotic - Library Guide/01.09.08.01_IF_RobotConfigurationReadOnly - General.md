## Task

The interface reads the configuration data from the robot via the provided methods and properties.

## Description

The methods and properties in the interface are identical to those in the interface [IF\_RobotConfiguration](D-SE-0100887.html#D-SE-0100887).

For the properties only the Get-Accessor is available.

## Methods

| Name | Description |
| --- | --- |
| [GetConfiguredDrive](D-SE-0075529.html#D-SE-0075529) | Reading the configured axes of the robot. |
| [GetEmergencyParameter2](D-SE-0075514.html#D-SE-0075514) | Reading the emergency parameters. |
| [GetTool](IF_RobotConfiguration-GetToolMethod-6D6A93A4.html) | Reading the parameters of a configured tool. |

## Properties

| Name | Data type | Accessing | Description |
