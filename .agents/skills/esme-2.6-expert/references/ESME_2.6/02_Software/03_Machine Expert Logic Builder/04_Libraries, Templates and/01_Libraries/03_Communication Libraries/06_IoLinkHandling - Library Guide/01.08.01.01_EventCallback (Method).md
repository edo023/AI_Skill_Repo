# EventCallback (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

The method is called by the system in case the registered event is thrown. The method provides detailed information about the event and the possibility to react to it.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_pEventParam | Pointer to [ST\_EventParameters](ST_EventParameters-0C90CD2A.html#ST_EventParameters-0C90CD2A) | Pointer to structure containing event parameters. |
