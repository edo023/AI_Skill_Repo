## Task

Interface for homing a robot.

## Methods

| Name | Description |
| --- | --- |
| LimitSwitch | Homing to a hardware limit switch. |
| MoveOnPos | Homing by moving to an absolute position. |
| RestorePosFromAxisEncoder | Homing by restoring the position value after reading the EncoderPosition. |
| RestorePosFromRetain | Homing by restoring the position value after reading the remanent data. |
| Sensor | Homing to a digital signal. |
| SetPosAxisPosition | Homing by setting the RefPosition. |
| Torque | Homing by moving to a limit stop. |
| TouchProbe | Homing to a TouchProbe signal. |
| WritePos | Homing by writing the position. |
