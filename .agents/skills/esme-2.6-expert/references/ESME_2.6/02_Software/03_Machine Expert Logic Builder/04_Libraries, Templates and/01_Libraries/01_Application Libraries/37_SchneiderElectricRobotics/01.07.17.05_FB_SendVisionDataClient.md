## Task

Register the function block to the Application Logger.

## Description

With the method RegisterLoggerPoint, the function block FB\_SendVisionDataClient can be registered to the Application Logger.

Refer to ApplicationLogger Library Guide.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifParent | APL.IF\_LoggerPoint | Parent logger point under which the FB\_SendVisionDataClient is registered. |
| i\_sName | STRING[80] | The name of the FB\_SendVisionDataClient logger point that is shown in the Application Logger is declared here. |
