# FB_AxisModule - Behavior in the Operating Modes

![](images/logo.png)

FB\_AxisModule - Behavior in the Operating Modes

General

Command Processing

A successful executed command is displayed as follows:

q\_xCMDActive = FALSE

q\_diCMDActive = last command

q\_xCMDDone = TRUE

iq\_diCMD = 0

 Operating mode commands are not possible if an error reaction [TPL.ET\_Reaction.AsyncStop](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1), [TPL.ET\_Reaction.SyncStopEL](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1) or [TPL.ET\_Reaction.SyncStopEH](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1) is pending.

You cannot enable the position control via the [DriveEnableOn](../../../../../../api/crossBook?lang=en-US&virtualBookName=Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command if an error reaction [TPL.ET\_Reaction.AsyncStop](../../../PD.Lib.Template&topicID=D_SE_0078354_1), [TPL.ET\_Reaction.SyncStopEL](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1) is pending.

NoOp

No operating mode choosen. The module is in wait state.

The technology POU [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) is being activated.

Basic monitoring systems are enabled:

oMonitoring the diagnostic messages of the axis

oMonitoring of the hardware limit switches

oMonitoring of the reduced velocity

Possible state commands

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The position control of the axis is activated.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The position control of the axis is being disabled.

MultiCam

The operating mode MultiCam is selected. The axis is appended as slave to a master via cams.

The operating mode is parameterized in the structures [ST\_MultiCam](../Structures/Structures-10.htm#XREF_D_SE_0077229_1) and [ST\_Main](../Structures/Structures-7.htm#XREF_D_SE_0077223_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [PDL.FB\_MultiCam](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087313_1) are being activated.

If the axis shall be activated in an ILS network, the pstIlsData parameter of the ST\_MultiCam structure must be linked to the (global) stIlsData structure, to which also the pstIlsData parameter of the ST\_EndlessFeed structure of the master module was linked. Subsequently, the technology POU [PDL.FB\_PreCalcMultiCam](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087330_1) is called up first.

oMonitoring of the reduced velocity

 If the operating mode was entered via the [CamWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command, a warm start is preselected. Subsequently, a WsStart command is waited for.

o[WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> After the warm start has been prepared via the commands [CamWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) - [Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1), the warm start motion is started via [WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1).

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The position control of the axis is activated.

Positioning

The operating mode Positioning is selected. The axis is moved relatively by a certain distance or absolutely to a position.

The operating mode is parameterized in the structures [ST\_Positioning](../Structures/Structures-11.htm#XREF_D_SE_0077231_1) and [ST\_Main](../Structures/Structures-7.htm#XREF_D_SE_0077223_1).

In the operating mode Positioning the [Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command may be overwritten by the [Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command. The internal start bit may also be reset by the i\_xPosStop input of the [ST\_Main](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) structure, so that the axis starts to move again with the next start signal without having received a [Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command.
