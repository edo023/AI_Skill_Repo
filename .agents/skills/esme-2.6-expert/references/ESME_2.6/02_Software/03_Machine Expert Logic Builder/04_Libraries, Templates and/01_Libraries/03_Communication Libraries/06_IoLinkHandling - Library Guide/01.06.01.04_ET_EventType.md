# ET\_EventType

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_EventType contains the types of the event.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Information | 0 | The event reports an information. |
| Warning | 1 | The event reports an alert. |
| Error | 2 | The event reports a detected error. |
