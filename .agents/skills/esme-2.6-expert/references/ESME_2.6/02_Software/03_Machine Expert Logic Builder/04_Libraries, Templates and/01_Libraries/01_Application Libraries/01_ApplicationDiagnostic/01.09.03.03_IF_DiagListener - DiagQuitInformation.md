## Task

Method to process a diagnostic quit command for diagnostic messages of type information inside the listener.

## Description

The interface method DiagQuitInformation  is called by the instance of the FB\_DiagnosticHandler in the event a diagnostic quit command for messages of type information is triggered.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_diDiagCode | DINT | Indicates the diagnostic code of the diagnostic message that should be reset.  If the input value is 0, no dedicated diagnostic message is targeted. |
| i\_diSourceId | DINT | Indicates the source ID of the diagnostic message that should be reset.  If the input value is 0, no dedicated source ID is targeted. |
| i\_ifRootSource | IF\_DiagHandlerIdentification | Indicates the root source of the diagnostic message that should be reset.  If the input value is 0, no dedicated root source is targeted. |
