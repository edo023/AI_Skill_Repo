Description

This structure contains the parameters and feedback data for the control movement of the FB\_Unwinder based on a torque controller.

NOTE: If the foil between the bobbin and the feedback sensor is cracked, then the bobbin can move with a very high velocity, depending on the friction of the mechanic.

NOTE: For this control mode the parameters of the bobbin drive have to be set a special way. See "Using the unwinder TorqueMode".

The bobbin axis is moved in TorqueMode. The position of the precontroller has no direct effect on the axis. Only the current, due to an acceleration of the precontroller has effect on the axis. The torque of the axis is determined with a control loop with the feedback of the tension measurement system of the foil.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_lrPGain | LREAL | The KP factor for the torque controller. |
| i\_timIGain | TIME | The TI factor for the torque controller. |
| i\_lrMaxOffsetTorque | LREAL | The maximum torque for the axis. |
| i\_xSuperimposeTorque | BOOL | TRUE: The TorqueMode works parallel to the normal calculated current in the drive.  FALSE: The torque of the drive is only determined by the torque controller. |
