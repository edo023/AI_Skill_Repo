## Task

Reading the emergency parameters of a specific component.

## Description

With the method GetEmergencyParameter2(…), the configured data for an emergency stop of a specific robotic component can be read.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etComponent | [ET\_RobotComponent](D-SE-0075489.html#D-SE-0075489) | Specifies which robot component emergency parameter to read. AuxAx10   For further information, refer to ET\_RobotComponent. |
