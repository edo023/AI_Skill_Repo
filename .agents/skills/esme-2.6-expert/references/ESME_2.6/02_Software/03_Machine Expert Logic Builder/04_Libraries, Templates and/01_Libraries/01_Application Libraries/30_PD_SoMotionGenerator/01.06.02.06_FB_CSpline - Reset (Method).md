Task

Release of dynamic memory

Description

This method releases the dynamic memory used for the last generated CSpline. When generating a new CSpline, this method does not have to be called up. In this case, the dynamic memory that is already in use is released automatically. This method should be called up if no further CSplines are to be generated in the progress of the program.
