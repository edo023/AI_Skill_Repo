# ST\_ConnectionSettings

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

The structure ST\_ConnectionSettings contains the parameter used for the connection to the MQTT server.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sClientId | STRING[255] | Unique identifier for the client. |
| uiKeepAlive | UINT | Keep-alive parameter, interval in seconds the MQTT server can expect control messages. After a duration equal to one and a half times the keep-alive value, the server disconnects the client. If the value is set to 0, the keep-alive mechanism is disabled. |
| stServerParameter | [ST\_MqttServerParameter](D-SE-0086511.html) | Structure to specify the parameter to establish a TCP connection to the MQTT server. |
| stCredentials | [ST\_Credentials](D-SE-0086512.html) | Structure to specify the user credentials. |
| stWillMessage | [ST\_WillMessage](D-SE-0086513.html) | Structure to specify the Will message. |
| stTlsSettings | [ST\_TlsSettings](D-SE-0095473.html) | Structure to specify the TLS (Transport Layer Security) configuration. |
| xCleanStart | BOOL | Specifies whether the connection starts a new session (TRUE) or is a continuation of an existing session for the client ID (FALSE).  NOTE: With version V3.1.1 of the MQTT standard, the value of xCleanStart is used to set the clean session flag according to the MQTT V3 specification. |
| etMqttVersion | [ET\_MqttVersion](ET_MqttVersion-D8850F84.html) | Specifies the version of the MQTT protocol to use.  Default setting: ET\_MqttVersion.V3\_1\_1 |
