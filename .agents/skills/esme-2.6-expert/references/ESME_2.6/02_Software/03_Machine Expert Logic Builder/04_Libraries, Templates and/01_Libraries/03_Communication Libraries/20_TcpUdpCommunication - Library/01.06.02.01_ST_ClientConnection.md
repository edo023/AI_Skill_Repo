# ST\_ClientConnection

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.4.0 |
| Inherits from: | - |

## Description

The structure ST\_ClientConnection contains information about the clients connected to the server.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sClientIP | STRING(15) | The IP address of the connected client |
| uiClientPort | UINT | The port number of the connected client |
| udiBytesAvailableToRead | UDINT | Number of bytes that are available to be read from that client. |
| uliTotalBytesReceived | ULINT | Total number of bytes received from this client. |
| uliTotalBytesSent | ULINT | Total number of bytes sent to this client. |
