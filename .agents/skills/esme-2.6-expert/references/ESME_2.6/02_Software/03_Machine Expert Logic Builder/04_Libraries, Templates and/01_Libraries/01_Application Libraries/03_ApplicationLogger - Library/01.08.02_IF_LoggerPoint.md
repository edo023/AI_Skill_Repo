# IF\_LoggerPoint

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.4.0 |
| Inherits from: | * SYSTEM.IQueryInterface |

## Description

This interface provides the connection to an FB\_LoggerPoint. Using this interface, the data of the logger point can be read out.

NOTE: As of version V2.0.0.0, this data type is an alias of the data type APL\_Base.IF\_LoggerPoint defined in the ApplicationLoggerBase library. The ApplicationLoggerBase library contains the basic implementations of the logger functions.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| etLogLevel | [ET\_LogLevel](D-SE-0077662.html#D-SE-0077662) | Read | Logger level that you selected for this logger point. |
| sName | STRING[80] | Read | Name that was given during the registration of the logger point. |
| sSource | STRING[40] | Read | Source that was given during the registration of the logger point. |
| sType | STRING[80] | Read | Type that was given during the registration of the logger point. |
| udiId | UDINT | Read | Identification number of the logger point. |
| udiParentId | UDINT | Read | Identification number of the logger point below with which this logger point is registered. |
