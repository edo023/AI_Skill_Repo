# ET\_DiagExt

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.4.0 |

## Description

The enumeration ET\_DiagExt specifies the POU-specific diagnostic and status messages.

NOTE: As of version V2.0.0.0, this data type is an alias of the data type APL\_Base.ET\_Result defined in the ApplicationLoggerBase library. The ApplicationLoggerBase library contains the basic implementations of the logger functions.
