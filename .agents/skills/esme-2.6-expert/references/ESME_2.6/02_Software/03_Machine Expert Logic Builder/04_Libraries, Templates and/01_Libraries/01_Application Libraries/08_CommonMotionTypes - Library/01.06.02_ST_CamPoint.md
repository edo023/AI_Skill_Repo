# ST_CamPoint

![](images/logo.png)

ST\_CamPoint

ST\_CamPoint - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

Description

Data structure for describing a point of a motion cam.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| lrX | LREAL | X position of the starting point of the cam section. |
| lrY | LREAL | Y position of the starting point of the cam section. |
| etCamType | [ET\_CamType](../Enumerations/Enumerations-2.htm#XREF_D_SE_0096118_1) | Cam type, see [ET\_CamType](../Enumerations/Enumerations-2.htm#XREF_D_SE_0096118_1). |
| lrM | LREAL | Gradient of the cam in the start point of the cam section. |
| lrK | LREAL | Curvature of the cam in the start point of the cam section.  (only required for ET\_CamType.Poly5com) |
| lrYUser(1) | LREAL | Scaling factor of the user profile.  Only required when using a user profile. |
| diProfileId(1) | DINT | Profile Id  Only required when using a user profile. Only user profiles must be loaded outside of the cam and transferred here. |
| lrYUserOffset(1) | LREAL | YOffset  Only required when using a user profile. |
| lrLambda | LREAL | Turning point |
| lrC | LREAL | Straight line or curve component of the profile (parable2, simplsin, poly5, inclisin, modacctr, modisin). |
| (1) This parameter is only supported by PacDrive controllers. | | |
