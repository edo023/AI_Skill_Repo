Task

Operating hours counter

Description

Seconds, minutes and hours are counted as long as i\_xStart = TRUE. The values are saved in the external variables connected to the in/outputs iq\_uliOperatingHours, iq\_uliOperat­ingMin and iq\_uliOperatingSec. If these variables were defined as PERSISTENT RETAIN the variables keeps their value after a power switch off and on of the controller. If the task interval of the calling task is greater than 1s, or if the calling task is delayed by another task, then the function block continues to count correctly, but the steps in iq\_uliOperatingSec are larger than 1s.
