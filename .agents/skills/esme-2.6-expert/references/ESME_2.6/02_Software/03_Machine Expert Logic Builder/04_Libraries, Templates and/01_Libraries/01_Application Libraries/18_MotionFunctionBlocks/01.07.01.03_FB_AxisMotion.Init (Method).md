# FB\_AxisMotion.Init (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.1.0 |
| Versions: | Installed version |

## Task

Assign the axis used for the Move commands.

## Description

The method FB\_AxisMotion.Init is used to assign an axis for FB\_AxisMotion and its Move commands. The assigned axis is verified, and an error message is generated when the assigned axis is not valid.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifAxis | CMT.IF\_AxisIdentification | The axis to be executed. |

| Output | Data type | Description |
| --- | --- | --- |

| q\_etResult | ET\_Result | POU-specific output on the diagnostic; q\_xError = FALSE -> Status message; q\_xError = TRUE -> Diagnostic message. |
| q\_sResultMsg | STRING[80] | Event-triggered message that gives additional information on the diagnostic state. |
