## Task

Interface that contains methods related to events that occur during operation of the targets handler function block.

## Description

The interface contains methods related to events that occur during operation of the targets handler function block.

You can implement the interface depending on your specific requirements.

## Methods

| Name | Description |
| --- | --- |
| OnAddTarget | Provides user-specific diagnostics when a target is added to the list. |
| OnAddTargetWithSlots | Provides user-specific diagnostics when a target with slots is added to the list. |
| OnRemoveTarget | Provides user-specific diagnostics when a target is removed from the list. |
| OnSetSlotOwnerInTarget | Provides user-specific diagnostics when the owner of a slot inside a target is set. |
| OnSetSlotProcessedInTarget | Provides user-specific diagnostics when the processed state of a slot inside a target is set. |
| OnSetTargetOwner | Provides user-specific diagnostics when the owner of a target is set. |
| OnSetTargetProcessed | Provides user-specific diagnostics when the processed state of a target is set. |
