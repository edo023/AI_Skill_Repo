Task

Function for changing cam points.

Description

 This function will retrieve the status of the instant new cam signal of the axis specified by the iq\_stAxisModuleItf input. The instant new cam signal will cause the axis to reload the cam points specified by the currently selected MultiCam table and run the changes in the current cycle as shown below.

![G-SE-0064627.1.gif-high.gif](images/G-SE-0064627.1.gif-high.gif)

The new cam data takes effect at the InstantXLimMax blend point. The blend point can be specified with the [FC\_SetInstantNewCamPosition](Functions-28.htm#XREF_D_SE_0077203_1) function.

The instant new cam signal also provides feedback. The axis will reset the signal to false at the blend point if the new cam points are accepted.

The instant new cam signal can be set TRUE with the [FC\_SetInstantNewCam](Functions-27.htm#XREF_D_SE_0077201_1) function.
