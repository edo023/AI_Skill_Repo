# ET\_CarrierCategory

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.4.7.0 |

## Description

The enumeration ET\_CarrierCategory contains the carrier category for the carrier emulation data. The values of the enumeration can be converted to STRING that can be used in a project.

Example for the Lexium™ MC12 carrier:

```
GVL_OpcUaMulticarrier.G_stMCR.astCarriers[1].sCategory := TO_STRING(DTC.ET_CarrierCategory.LXMMC12Carrier);
```

Example for the Lexium™ MC12 Smart Carrier:

```
GVL_OpcUaMulticarrier.G_stMCR.astCarriers[1].sCategory := TO_STRING(DTC.ET_CarrierCategory.LXMMC12SmartCarrier);
```

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| LXMMC12Carrier | 0 | Lexium™ MC12 carrier |
| LXMMC12SmartCarrier | 1 | Lexium™ MC12 Smart Carrier |
