# ST\_NameplateResponse - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.2.0 |
| Inherits from: | - |

## Description

Structure for the data of the electronic nameplate of a Sercos device received with [FC\_NameplateResponse](FC_NameplateResponse-GeneralInforma-091A8983.html) after a request with FC\_NameplateRequest.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| sVendorName | STRING(80) | Vendor name of the device (parameter S-0-1300.x.02) |
| wVendorCode | WORD | Vendor code of the device (parameter S-0-1300.x.03) . The vendor code is a unique number assigned to each vendor. |
| sDeviceName | STRING[80] | Device name, manufacturer-specific (parameter S-0-1300.x.04) |
| sVendorDeviceID | STRING[80] | Vendor device ID (parameter S-0-1300.x.05). The vendor device ID is a unique ID managed by the vendor. |
| wFunctionRevision | WORD | Function revision of the device (parameter S-0-1300.x.07) |
| sHWRevision | STRING[80] | Hardware revision of the device (parameter S-0-1300.x.08) |
| sFWRevision | STRING[80] | Firmware version of the device (parameter S-0-1300.x.09) |
| sFirmwareLoaderRevision | STRING[80] | Firmware loader or boot loader revision (parameter S-0-1300.x.10) |
| sSerialNumber | STRING[80] | Serial number of the device (parameter S-0-1300.x.12) |
| sManufacturingDate | STRING[80] | Date and time of manufacture of the device (parameter S-0-1300.x.13) |
| dwOerationalHours | DWORD | Hours of operation of the device (parameter S-0-1300.x.20) |
| sHostname | STRING[80] | Network hostname of the device (parameter S-0-1039) |
| wNumberOfSlaves | WORD | Number of slaves connected to the device |
