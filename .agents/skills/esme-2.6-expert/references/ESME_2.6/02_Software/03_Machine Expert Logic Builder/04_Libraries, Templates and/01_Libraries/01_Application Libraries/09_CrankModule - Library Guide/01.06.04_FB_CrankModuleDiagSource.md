Task

Output of the additional interface of the [FB\_CrankModule](Function_Blocks-3.htm#XREF_D_SE_0088740_1) in a diagnostics prompt.

Description

To diagnose the template project with PacDrive Diagnostics, this POU provides the variables in the [ST\_ModuleInterface](../Structures/Structures-8.htm#XREF_D_SE_0088764_1) of a crank module as a diagnostic source. It lists modules, starting with the MainMachine. For each CrankModule the entire additional interface is output.
