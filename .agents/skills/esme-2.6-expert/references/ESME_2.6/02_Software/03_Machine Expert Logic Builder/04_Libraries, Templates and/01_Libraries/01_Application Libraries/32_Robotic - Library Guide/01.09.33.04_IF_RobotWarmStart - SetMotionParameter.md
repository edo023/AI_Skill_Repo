## Task

Setting the warm start motion parameters.

## Description

With the method SetMotionParameter(...), the warm start motion parameters can be determined.

The parameters are:

* Maximum velocity (change of the position per time unit)
* Maximum acceleration (change of the velocity per time unit)
* Maximum deceleration (change of the velocity per time unit)
* Ramp (change of the acceleration/deceleration) with which the warm start motion of each component of the robot has to be executed.

The specification of the maximum values of the jogging motion parameters:

* Can be executed again at any time.
* Applies to the warm start motion.
