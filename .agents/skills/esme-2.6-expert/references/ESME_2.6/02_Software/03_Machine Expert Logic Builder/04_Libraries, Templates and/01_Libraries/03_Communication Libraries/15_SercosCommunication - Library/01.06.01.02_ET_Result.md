# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result contains the possible values that indicate the result of operations executed by a function or a function block.

NOTE: As of version V1.4.0.0, this data type is an alias of the data type SERC\_CT.ET\_Result  defined in the controller independent common library SercosCommunicationCommonTypes.
