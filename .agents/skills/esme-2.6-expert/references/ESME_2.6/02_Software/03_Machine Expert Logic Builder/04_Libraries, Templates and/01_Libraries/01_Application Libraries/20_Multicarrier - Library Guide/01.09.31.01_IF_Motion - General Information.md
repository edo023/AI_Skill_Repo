## Task

Interface for setting the motion parameters, velocity limitations and stop commands for a carrier within a Lexium™ MC multi carrier transport system.

## Description

The interface provides several methods and properties for defining the motions of a carrier.

The move commands [StopCarrier](IF_MotionStopCarrier-587B0D50.html#IF_MotionStopCarrier-587B0D50) and [StopCarrierWithEmergencyParameter](StopCarrierEmergPara-AF81B4BA.html#StopCarrierEmergPara-AF81B4BA) are directly related to the interface.

Through the properties, the interface is related to the move commands Jogging, MoveDirectly, MoveGapControl, MovePureSmg, MoveSyncFromStandstill, and MovePosAndSync.

The various move commands provide different properties and methods for moving carriers on a track.
