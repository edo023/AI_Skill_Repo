# ET_PosType

![](images/logo.png)

ET\_PosType

ET\_PosType - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Description

Enumeration type for describing the various position values of an axis.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Position | 0 | Position of the axis |
| RefPosition | 1 | Reference position of the axis |
| MechPosition | 2 | Mechanical position of the axis |
| MechRefPosition | 3 | Mechanical reference position of the axis |
| ShaftMechPosition | 4 | Mechanical position on the motor shaft |
| ShaftMechRefPosition | 5 | Mechanical reference position on the motor shaft |
