## Description

Given a 2D vector and a scalar value, the function returns a new 2D vector resulting from multiplying the scalar value by each element of the input vector.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stVectorA | SE\_MATH.ST\_Vector2D | A 2D vector. |
| i\_lrScalar | LREAL | A scalar value to multiply to the elements of i\_stVector. |
