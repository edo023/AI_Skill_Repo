## New Features

Using the method IF\_RobotPSeries.InitializeOrientationZ(), the fourth axis can be configured to use the OrientationZ component.

## Modifications

**none**

## Enhancements

* The methods SetLRealParameter, OptionalEquipment, InitializeRobot, InitializeRotationalAxis and the property raetOptionalEquipment were added to the interface IF\_RobotPSeries.
* The parameter UserCurrentLimit can be modified with the method IF\_RobotPSeries.SetControlLoopParameter().
* The parameter MassTcpPlate can be set to the maximum of 6 kg with the method IF\_RobotPSeries.SetLrealParameter().
