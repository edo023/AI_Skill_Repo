# FC\_ReadDataStatusRequest - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.2.0 |

## Task

This function sends the request for the ReadDataStatus service to a Sercos device to read the data status of a Sercos parameter (IDN).

## Functional Description

This function sends the request for the ReadDataStatus service to a Sercos device to read the data status of a Sercos parameter (IDN). The Sercos device is specified by its node identifier (MAC address). The Sercos parameter is specified by its IDN. The response to the request is obtained with the function FC\_ReadDataStatusResponse.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sIPAddress | STRING[15] | IP address of the Sercos device |
| i\_uiSlaveIndex | UDINT | Sercos slave index of the Sercos device (allows you to address Sercos devices A and B of a Lexium 62 Double Drive) |
| i\_udiIDN | UDINT | Identification number (IDN) of the parameter to be verified |

| Output | Data type | Description |
| --- | --- | --- |
| q\_udiTransactionID | UDINT | Returns the transaction ID to be used for receiving the response |
