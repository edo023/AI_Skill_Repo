# Input Pin Description

![](images/logo.png)

Input Pin Description

Input Pin Description

| Input | Data type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block. |
| i\_xDrvRun | BOOL | TRUE: Hoist is running.  FALSE: Hoist is not running. |
| i\_rLoadNom | REAL | Nominal load  Range: 0...3.438  Unit: kg |
| i\_rLoadAct | REAL | Actual load  Range: 0...3.438  Unit: kg |
| i\_xStandard | BOOL | Used standard  TRUE: ISO  FALSE: FEM |

i\_xEn

When TRUE, this input enables the function block.

When FALSE, the function block enters a fallback state in which it does not register usage of the hoist.

i\_xDrvRun

Informs the function block that the hoist is running. The information can be obtained from bit 2 of the Altivar drive status word.

i\_rLoadNom

Nominal load of the hoist [kg]. The value must be greater than 0.0.

i\_rLoadAct

Actual load [kg]

If weight of the lifting accessories exceeds 5% of the nominal load of the hoist, their weight must be included in the input value. The value must not be negative.

i\_xStandard

Switches the output between FEM (FALSE) and ISO (TRUE) standard form.

It is possible to switch between standards because the calculations are identical. The outputs get fully updated after next power cycle or at midnight.
