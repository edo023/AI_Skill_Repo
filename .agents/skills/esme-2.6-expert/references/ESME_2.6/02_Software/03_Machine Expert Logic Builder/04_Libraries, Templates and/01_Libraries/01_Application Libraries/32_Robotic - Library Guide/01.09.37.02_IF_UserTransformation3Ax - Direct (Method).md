## Task

Forward transformation of the kinematics.

## Description

The positions of the axes are transferred at the i\_PositionA, i\_PositionB, and i\_PositionC inputs.

They must be converted to Cartesian coordinates within this method and returned as return values at the q\_stCoordinate output.

Those calculated values are further processed by the robot.
