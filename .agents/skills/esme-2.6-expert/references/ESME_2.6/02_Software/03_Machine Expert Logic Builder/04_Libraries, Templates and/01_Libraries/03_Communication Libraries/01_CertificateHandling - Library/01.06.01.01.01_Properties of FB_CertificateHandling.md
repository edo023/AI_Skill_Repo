# Properties of FB\_CertificateHandling

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| CommonName | STRING[64] | Read | The string containing the common name attribute of the certificate. |
| Country | STRING[2] | Read | The string containing the country code attribute of the certificate consisting of two characters. |
| DnsName | STRING[255] | Read | The string containing the Domain Name System (DNS) used as alternative name. |
| EndDate | DT | Read | The end date of the validity of the certificate. |
| Handle | SysTypes.RTS\_IEC\_HANDLE | Read | The handle of the certificate. |
| IpAddress | STRING[255] | Read | The string containing the IP address used as alternative name. |
| IsCertificateSelected | BOOL | Read | If TRUE, a certificate is selected. |
| IsClientAuthentication | BOOL | Read | If TRUE, the extended use of the certificate is set to client authentication. |
| IsDateValid | BOOL | Read | If TRUE, the controller time is within the validity period attribute of the certificate. |
| IsSelfSigned | BOOL | Read | If TRUE, the certificate is self-signed. |
| IsServerAuthentication | BOOL | Read | If TRUE, the extended use of the certificate is set to server authentication. |
| Locality | STRING[128] | Read | The string containing the locality attribute of the certificate. |
| Organization | STRING[64] | Read | The string containing the organization attribute of the certificate. |
| OrganizationUnit | STRING[64] | Read | The string containing the organization unit attribute of the certificate. |
| Result | ET\_Result | Read | Indicates the result of the last method call.  The result is reset if another method or property is called. |
| StartDate | DT | Read | The start date of the validity of the certificate. |
| State | ET\_State | Read | Indicates the state of the function block. |
| StateProvince | STRING[128] | Read | The string containing the state or province attribute of the certificate. |

NOTE: Verify the value of the property Result after each property access.
