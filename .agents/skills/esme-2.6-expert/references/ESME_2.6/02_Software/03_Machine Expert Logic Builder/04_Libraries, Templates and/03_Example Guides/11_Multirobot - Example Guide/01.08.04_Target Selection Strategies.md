# Target Selection Strategies

## Overview

The target selection strategies are used to select a valid target for a robot. The demo project contains a global variable named RoboticCellMonitoring.G\_astRoboticCellTargetSelection. This variable can be edited at runtime to affect the behavior of a RobotCell.

The target selection strategies are called in the SR\_SupervisionModule.Logic method to implement a pick and place logic. Each robot and tracking system pair have a specific target selection strategy configuration.

In the demo, the target selection strategies are initialized in the Smart Template module RobotCell by the method Init\_Supervisor.

## Example Code

The following example shows how to configure a FIFO target selection strategy for Robot1 operating on the tracking system Tracking1:

```
//select a robot
etRobotId := SERT.ET_SystemEntity.Robot1;
//select a tracking system
etTrackingId := ROB.ET_CoordinateSystem.Tracking1;
//select a library target selection strategy
G_astRoboticCellTargetSelection[etRobotId].astTracking[etTrackingId].etStrategy := ET_RobotTargetSelectionStrategyType.FIFO;
//configure the FIFO strategy
G_astRoboticCellTargetSelection[etRobotId].astTracking[etTrackingId].xAccurateDistanceCalculation := TRUE;
G_astRoboticCellTargetSelection[etRobotId].astTracking[etTrackingId].xSelectTargetsWithAnyOwner := FALSE;
G_astRoboticCellTargetSelection[etRobotId].astTracking[etTrackingId].xSelectTargetsWithNoOwner := FALSE;
```

For further information, refer to [SERT.IF\_TargetSelectionStrategy](../../../../../api/crossBook?lang=en-US&virtualBookName=SERToolb&topicID=D_SE_0098015).
