# Example

## Creating a Motion Task

Procedure for creating a motion task:

| Step | Action |
| --- | --- |
| 1 | Display the Devices tree by clicking View > Navigators > Devices tree. |
| 2 | In the tree of your controller below the nodes PLC Logic > Application > Task Configuration, select Motion. |
| 3 | Set the value of Priority to 1. |
| 4 | Set the value of Type to External. |
| 5 | Set the value of External Event to AFTER\_RTP. |
| 6 | Add the following libraries via the Library Manager:   * CNCExtension (Schneider Electric) * SM3\_CNC (CODESYS GmbH, formerly Smart Software Solutions GmbH) |

## Sample Project

A sample project is provided along with the CNCExtension library. To open the sample project, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | In the EcoStruxure Machine Expert Logic Builder, execute the command New Project. |
| 2 | In the New Project dialog box, select From Example from the Project type list. |
| 3 | On the right-hand side of the New Project dialog box, select the controller **M262** from the list of the controllers.  Result: Available examples are listed in the Matching Examples text box. |
| 4 | Select your example from the Matching Examples list. |
| 5 | Enter a name for the new project, and select the file location. |
| 6 | Click the OK button.  Result: A new project is created based on the selected example. |
