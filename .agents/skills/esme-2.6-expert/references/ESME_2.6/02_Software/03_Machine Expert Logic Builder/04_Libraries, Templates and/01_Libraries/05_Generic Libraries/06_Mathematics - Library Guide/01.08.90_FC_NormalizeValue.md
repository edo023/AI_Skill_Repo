# FC\_NormalizeValue

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function performs a modulo division of a real number. The lower interval limit is specified as an input value. The function is an extension of the arithmetic operator MOD:

* Modulo division of a real value.
* Lower interval limit can be specified (the lower interval limit of MOD is 0).

The function returns the remainder of the division that is within the interval [i\_lrIntervalStart, (i\_lrIntervalStart + i\_lrIntervalLength)].

Calculation:

Result = ((i\_lrValue - i\_lrIntervalStart) MOD i\_lrIntervalLength) + i\_lrIntervalStart
