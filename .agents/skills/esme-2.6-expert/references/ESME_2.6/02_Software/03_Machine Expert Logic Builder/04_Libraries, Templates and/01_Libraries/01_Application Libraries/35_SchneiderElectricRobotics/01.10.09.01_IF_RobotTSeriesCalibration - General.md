## Task

Interface to execute the calibration for a Lexium T Robot.

## Description

The interface contains methods and properties to execute the calibration for a Lexium T Robot.

Methods must not be called cyclically.

## Methods

| Name | Description |
| --- | --- |
| BrakeRelease | Method to select which brakes of the robot axes shall be opened. |
| MoveToMountPosition | Axes of the robot move to a specific position to mount the lower arms and the parallel plate. |
| StandardProcedure | Execute the procedure OnTorque and MoveToCheckPosition. |
| Stop | Stops the ongoing calibration mode. |
