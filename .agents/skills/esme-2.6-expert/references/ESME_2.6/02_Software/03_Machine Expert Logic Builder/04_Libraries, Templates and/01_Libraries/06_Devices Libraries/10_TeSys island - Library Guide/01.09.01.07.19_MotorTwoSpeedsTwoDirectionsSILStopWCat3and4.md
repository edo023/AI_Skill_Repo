# MotorTwoSpeedsTwoDirectionsSILStopWCat3and4 - Functional Description

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V2.0.3.0 |

## Functional Description

The function block MotorTwoSpeedsTwoDirectionsSILStopWCat3and4 is used to manage a two-speed motor in two directions (forward and reverse), with stop category 0 or stop category 1, Wiring category 3 and category 4.

NOTE: Stop category 0 or stop category 1 are according to IEC 60204-1.
