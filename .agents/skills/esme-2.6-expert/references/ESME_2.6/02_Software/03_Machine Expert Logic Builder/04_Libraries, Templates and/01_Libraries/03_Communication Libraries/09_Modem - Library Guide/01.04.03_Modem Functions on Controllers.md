# Modem Functions on Controllers

![](images/logo.png)

Modem Functions on Controllers

Introduction

This topic describes management and operations of the controllers’ modem communication functions.

NOTE: Communication functions are processed asynchronously with regard to the application task that called the function.

Available Function Blocks

This table describes the modem function blocks available to controllers:

| Function | Description |
| --- | --- |
| [Dial](../Modem_function_blocks/Modem_function_blocks-3.htm#XREF_D_SE_0003119_1) | Dial establishes communications through a modem to a remote device. |
| [HangUp](../Modem_function_blocks/Modem_function_blocks-4.htm#XREF_D_SE_0003122_1) | HangUp closes a previously opened connection. |
| [SendSMS](../Modem_function_blocks/Modem_function_blocks-6.htm#XREF_D_SE_0003123_1) | SendSMS sends SMS. |
| [ReceiveSMS](../Modem_function_blocks/Modem_function_blocks-6.htm#XREF_D_SE_0003123_1) | ReceiveSMS allows the controller to receive SMS. |
| [ConfigSim](../Modem_function_blocks/Modem_function_blocks-8.htm#XREF_D_SE_0004492_1) | Use ConfigSim when your SIM card requires either a PIN code, an SMS center phone number, or an initialization command. |
