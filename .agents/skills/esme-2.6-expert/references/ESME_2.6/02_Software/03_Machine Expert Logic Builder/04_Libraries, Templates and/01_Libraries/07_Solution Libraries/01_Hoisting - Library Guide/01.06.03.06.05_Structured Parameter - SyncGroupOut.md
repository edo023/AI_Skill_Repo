# Structured Parameter - SyncGroupOut

![](images/logo.png)

Structured Parameter - SyncGroupOut

Structure: SyncGroupOut

Structure of parameters related to the synchronization of groups

| Parameter | Data type | Description |
| --- | --- | --- |
| xSyncState | BOOL | Synchronization state  TRUE: Synchronization is active.  FALSE: Synchronization is inactive. |
| diActPosSync | DINT | Actual position of the synchronization group  Range: 0...65535  Unit: in. |
| iActSpdSync | INT | Actual speed of the synchronization group  Range: -32768...32767  Unit: RPM |
| iAxisCount | INT | Number of axes within the synchronization group  Range: 0...32767 |
| wAlrmId | WORD | Alarm identification  Range: 0...65535 |

xSyncState

Synchronization state

TRUE: Synchronization is active.

FALSE: Synchronization is inactive.

There must be at least one axis in a synchronization group in order to enter the synchronized state.

diActPosSync

Synchronous position of the synchronization group. The value is a mean value of positions of all axes within the synchronization group. This output can be used for nested synchronization.

iActSpdSync

Synchronous speed of the synchronization group. The value is a mean value of speeds of all axes within the synchronization group. This output can be used for nested synchronization.

iAxisCount

Number of axes within the synchronization group. A virtual axis consisting of multiple physical axes is considered as a single axis. This output can be used for nested synchronization.

wAlrmId

This output identifies the detected alarm of the synchronization group.

| Alarm bit | Description |
| --- | --- |
| 0 | Position deviation of at least one of the axes within the synchronization group exceeded the maximum allowed deviation. |
| 1 | Proportional gain of the position controller is lower than 0.  i\_astSyncGroup[i].rKpPosSync < 0.0 |
| 2 | Acceleration ramp time parameter of the synchronization group is outside the allowed range (1…999). |
| 3 | Deceleration ramp time parameter of the synchronization group is outside the allowed range (1…999). |
| 4 | i\_astSyncGroup[i].wDrvSpdNomSync is outside the allowed range (1...6000). |
| 5 | At least one of the drives in the synchronization group not ready. |
| 6 | At least one of the encoder interfaces in the synchronization group not ready. |
| 7 | At least one of the axis interfaces in this synchronization group is configured incorrectly. |
