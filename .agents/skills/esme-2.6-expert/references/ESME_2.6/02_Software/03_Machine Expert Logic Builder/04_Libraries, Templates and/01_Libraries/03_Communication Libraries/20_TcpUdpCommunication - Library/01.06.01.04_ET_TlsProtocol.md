# ET\_TlsProtocol

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.4.0 |

## Description

The enumeration ET\_TlsProtocol is used to define the protocol version for the for TCP/IP connections using TLS (Transport Layer Security).

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| TLSv11 | 0 | The protocol version is discontinued. |
| TLSv12 | 1 | TLS V1.2 client or server |
| TLSv13 | 2 | TLS V1.3 client or server |
