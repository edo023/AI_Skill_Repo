# ST\_XYStruct

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |

## Description

The structure ST\_XYStruct represents the relative extreme values of a fifth degree polynomial.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| diCount | DINT | Number of existing relative extreme values. |
| alrX | ARRAY [0..3] OF LREAL | X value of the i-th relative extreme value. |
| alrY | ARRAY [0..3] OF LREAL | Y value of the i-th relative extreme value. |
