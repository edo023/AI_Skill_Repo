## Task

Registers a listener that receives the diagnostic messages.

## Description

The method RegisterListener is used to register an interface of type IF\_DiagListener. Depending on the events, the methods of the interface are called from the instance of the diagnostic handler. The instance implementing the interface can be used to implement ,for example a logger or a list of active diagnostic messages. An example of an active diagnostic list is provided by the FB\_ActiveDiagnostics function block.

NOTE: The listener is represented by a function block which must implement the interface [IF\_DiagListener](IF_DiagnosticListener-44A99293.html#IF_DiagnosticListener-44A99293).

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Indicates TRUE if the listener has been registered successfully.  If FALSE is returned, an invalid interface is assigned to the input i\_ifDiagnosticListener. |
