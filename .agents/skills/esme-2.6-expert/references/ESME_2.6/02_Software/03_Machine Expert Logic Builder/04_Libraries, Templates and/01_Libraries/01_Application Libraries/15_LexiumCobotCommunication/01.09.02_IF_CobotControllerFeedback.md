## Task

The function block implementing this interface is used to retrieve data from the Lexium Cobot.

## Description

The interface provides several methods and properties to establish a TCP connection to a Lexium Cobot controller and to retrieve feedback data. The interface is implemented by the function block FB\_CobotControllerFeedback.

## Methods

| Name | Description |
| --- | --- |
| [Connect](FB_CobotControllerFeedback-ConnectM-27E70005.html) | Establishing a connection to the TCP server of a Lexium Cobot controller without using TLS encryption. |
| [ConnectTLS](FB_CobotControllerFeedback-ConnectT-27E797CE.html) | Establishing a connection to the TCP server of a Lexium Cobot controller using TLS encryption. |
| [Disconnect](FB_CobotControllerFeedback-Disconne-27E7FC7A.html) | Disconnecting an established connection from the TCP server of a Lexium Cobot controller. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| etConnectionState | [ET\_ConnectionState](ET_ConnectionState-1E1DE583.html) | Read | Specifies the connection state of the function block to the Lexium Cobot controller. |
| rstData | REFERENCE TO [ST\_FeedbackData](ST_FeedbackData-2C7FE476.html) | Read | Specifies the data structure in which the robot feedback data is provided.  The robot feedback data is updated when new data is available from the Lexium Cobot controller during each call of the function block. |
| xTLSUsed | BOOL | Read | TRUE: A connection to the TCP server of a Lexium Cobot controller using TLS encryption is established. |
