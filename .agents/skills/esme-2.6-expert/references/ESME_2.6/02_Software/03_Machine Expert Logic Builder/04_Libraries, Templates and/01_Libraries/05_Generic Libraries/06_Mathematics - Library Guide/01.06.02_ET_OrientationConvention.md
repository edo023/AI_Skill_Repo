# ET\_OrientationConvention

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.1.0.0 |

## Description

Convention for the rotation angles of an orientation.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No orientation convention selected. |
| XYZ | 1 | First rotation about X, then rotation about Y and then rotation about Z. |
| ZYX | 2 | First rotation about Z, then rotation about Y and then rotation about X. |
| ZXY | 3 | First rotation about Z, then rotation about X and then rotation about Y. |
