NOTE: The elements FB\_WriteDriveEncoder und FC\_ReadDriveEncoder are obsolete.

Use instead the elements [FB\_HomeAbs](../Function_Blocks_G_to_H/Function_Blocks_G_to_H-6.htm#XREF_D_SE_0087278_1) and [FB\_HomeSetPos](../Function_Blocks_G_to_H/Function_Blocks_G_to_H-18.htm#XREF_D_SE_0087290_1) with the mode PDL.[ET\_HomeSetPosMode](../Enumerations/Enumerations-10.htm#XREF_D_SE_0087223_1).RestoreFromAxisEncoder.

Task

The position of the motor sensor should be read.

Description

The function reads the encoder position of the axis and reconstructs the position dependent on i\_lrHomePos, i\_lrAbsEncOffset, i\_lrUserPeriod and the rotating direction of the axis. In doing so, the same parameter has to be used as when setting the position with the function block [FB\_WriteDriveEncoder](../Function_Blocks_R_to_Z/Function_Blocks_R_to_Z-38.htm#XREF_D_SE_0087383_1). A SetPos is called up in the function.

The function must be called up once in the initialization part op the program for referencing the axes!
