# FB\_MultiAxisCustomJob

## Description

The function block FB\_MultiAxisCustomJob is the central function block for the coordination and management of the axes in a multi-axis custom job. It orchestrates the execution of a user-defined motion algorithm across multiple axes within a single RTP cycle.

The function block manages the instances of FB\_SlaveChannel for subordinate axes and FB\_MasterChannel for master axes connected to the group.

The function block uses the interface IF\_MultiAxisCustomJobCallback to access the user-defined logic by orchestrating the calls to the Prepare and Calculate methods provided by the interface.

Upon activation via the Start method, the function block triggers the initialization phase by calling the Prepare method of the interface. Subsequently, it cyclically calls the Calculate method of the interface. This provides the environment in which the user-defined algorithm calculates motion profiles and applies target values to the subordinate axes using the method WriteMovementValues of the respective instances of the function block FB\_SlaveChannel.

Throughout execution, the function block monitors the state of the multi-axis group and indicates the current operating state via the etState property.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
