## New Features

* The commands [SetCoordinateSystem](FB_CobotController-SetCoordinateSys-3A5C3D53.html) and [SetTCP](FB_CobotController-SetTCPMethod-3A63DA08.html) are added to the function block [FB\_CobotController](FB_CobotController-GeneralInformati-1E9FD546.html) and the corresponding interface [IF\_CobotController](IF_CobotController-2C7B91A8.html)
* The robot feedback data stToolEndButtons and sSafetyChecksum are added to the structure [ST\_FeedbackData](ST_FeedbackData-2C7FE476.html) and can be retrieved using the function block [FB\_CobotControllerFeedback](FB_CobotControllerFeedback-GeneralI-27E4BC7A.html).

## Enhancements

* Values greater than 65535 transferred at the input i\_udiCommandId of the methods [FB\_CobotController.MoveC](FB_CobotController-MoveCMethod-7FEABFCC.html), [FB\_CobotController.MoveL](FB_CobotController-MoveLMethod-7FE8CB4B.html), [FB\_CobotController.MoveJ](FB_CobotController-MoveJMethod-7FEB6D55.html) and [FB\_CobotController.MoveTCP](FB_CobotController-MoveTCPMethod-7FEBF718.html) are properly retrieved through the robot feedback data ST\_FeedbackData.udiCommandId using the function block [FB\_RobotControllerFeedback](FB_CobotControllerFeedback-GeneralI-27E4BC7A.html). The maximum value which can be transferred and properly retrieved is 2,147,483,647.
* The diagnostic message NameInvalid is added to the method [FB\_CobotController.SetSystemVariable](FB_CobotController-ModifySystemVari-27E0C52F.html).
* The diagnostic message OrientationConventionInvalid is added to the methods [FB\_CobotController.MoveC](FB_CobotController-MoveCMethod-7FEABFCC.html), [FB\_CobotController.MoveL](FB_CobotController-MoveLMethod-7FE8CB4B.html), [FB\_CobotController.MoveTCP](FB_CobotController-MoveTCPMethod-7FEBF718.html) and [FB\_CobotController.CalcInverseKinematics](FB_CobotController-CalcInverseKinem-7F6E83A2.html).
* The diagnostic message IdentifierRange is added to the method [FB\_CobotController.SetSystemVariable](FB_CobotController-ModifySystemVari-27E0C52F.html).
* The diagnostic message ValueRange is added to the method [FB\_CobotController.SetSystemVariable](FB_CobotController-ModifySystemVari-27E0C52F.html).
* The diagnostic message TaskInvalid is added to the function block [FB\_RobotControllerFeedback](FB_CobotControllerFeedback-GeneralI-27E4BC7A.html).
* The diagnostic message CommandIdRange is added to the methods [FB\_CobotController.MoveC](FB_CobotController-MoveCMethod-7FEABFCC.html), [FB\_CobotController.MoveL](FB_CobotController-MoveLMethod-7FE8CB4B.html) and [FB\_CobotController.MoveTCP](FB_CobotController-MoveTCPMethod-7FEBF718.html)

  .
* Setting a brakepoint in a task in which the function block [FB\_CobotController](FB_CobotController-GeneralInformati-1E9FD546.html) or [FB\_RobotControllerFeedback](FB_CobotControllerFeedback-GeneralI-27E4BC7A.html) is called, works properly.
