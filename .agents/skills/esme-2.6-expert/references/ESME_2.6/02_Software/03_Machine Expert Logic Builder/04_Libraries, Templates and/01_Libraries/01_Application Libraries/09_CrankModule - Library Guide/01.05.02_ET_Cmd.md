# ET_Cmd

![](images/logo.png)

ET\_Cmd

ET\_Cmd - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.2.0 |

Description

The enumeration describes all commands of the [FB\_CrankModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0088740_1).

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Homing | 610 | Change to operation mode Homing. |
| Manual | 620 | Change to operation mode Manual. |
| CamCs | 630 | Change to operation mode Cam and prepare a cold start. |
| CamWs | 631 | Change to operation mode Cam and prepare a warm start. |
| Endless | 640 | Change to operation mode Endless. |
| Positioning | 650 | Change to operation mode Positioning. |
| BrakeRelease | 670 | Change to operation mode Brake. |
| Start | 600 | Start the currently selected operation mode. |
| StartTrig | 612 | Trigger a start for the current operation mode and delete the start flag. |
| StartTrigWaitInPos | 613 | Trigger a start in the operation mode Positioning. The command is only signaled as successful after the position has been reached. |
| Stop | 601 | Cancel the command and return to the operation mode NoOp. |
| Hold | 602 | Cancel a start command. |
| WsStart | 605 | Trigger a warm start in the operation mode Cam or Additional. |
| DriveEnableOn | 603 | Enable the position control of the axis. |
| DriveEnableOff | 604 | Disable the position control of the axis. |
