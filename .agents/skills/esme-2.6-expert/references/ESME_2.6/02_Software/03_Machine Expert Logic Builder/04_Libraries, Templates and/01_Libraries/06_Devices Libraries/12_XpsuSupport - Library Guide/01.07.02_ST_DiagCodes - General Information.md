## Description

The structure ST\_DiagCodes contains the [diagnostics codes](D-SE-0094593.html#D-SE-0094593__D-SE-0094593.10) of the XPSU safety module, each represented as a BOOLEAN structure member. The structure is used at the output q\_stDiagCode of the function block FB\_XpsuDiag. Depending on the XPSU safety module used and on the bit sequence provided to the input i\_xDiagSignal of the function block, the corresponding structure element is set to TRUE.
