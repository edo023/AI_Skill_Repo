# Function Templates

## What are Function Templates

Function templates allow you to make dedicated control and visualization functionality that has been developed in one project available to other projects. They help to standardize the usage of field devices and application functions throughout different projects. For additional information, refer to [Templates](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083783).

## Adding a Function Template

To add an application function from a function template, right-click a suitable node (see the list hereafter) and execute the command Add Function From Template from the contextual menu.

Suitable nodes are:

* An Application node in the Applications tree
* A folder node under an Application node in the Applications tree
* A controller node in the Functional tree
* A functional module node in the Functional tree

As a result, the Add Function From Template dialog box opens. In the dialog box, select and configure the function template to add. For further information, refer to [Adding Functions from Template](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083799).
