# General Information

![](images/logo.png)

General Information

Description

The library "CrankModule" contains the crank module for the template as well as the required auxiliary functions.

The crank module controls a real axis for a crank and a virtual axis for a slide with general motion tasks such as EndlessFeed, Positioning, MultiCam, etc.

Namespace of the Library

The default namespace of this library is CRKM.

The POUs, data structures, enumerations and constants have to be addressed via this namespace.
