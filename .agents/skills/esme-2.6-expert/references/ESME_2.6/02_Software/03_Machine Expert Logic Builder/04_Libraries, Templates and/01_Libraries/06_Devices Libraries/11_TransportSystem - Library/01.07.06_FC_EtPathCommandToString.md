Task

Converting an enumeration element of the type [ET\_PathCommand](../Enumerations/Enumerations-6.htm#XREF_D_SE_0090682_1) into a string.

Description

Every enumeration element has a name or value. The return value of the function is the name of the enumeration element.

| i\_etPathCommand | [ET\_PathCommand](../Enumerations/Enumerations-6.htm#XREF_D_SE_0090682_1) | The enumeration value that has to be converted. |
