# ST\_RobotModifiedCoordinateSystem

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.4.1.0 |
| Inherits from: | - |

## Description

Structure to store internal information on the modified coordinate system of a robot.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| xEnable | BOOL | If TRUE, the function block will translate and rotate the original coordinate system accordingly with the value of stCartesianPose. |
| stCartesianPose | ST\_CartesianPose | This structure contains the translation, the orientation and the orientation convention used to describe the modified coordinate system. |
