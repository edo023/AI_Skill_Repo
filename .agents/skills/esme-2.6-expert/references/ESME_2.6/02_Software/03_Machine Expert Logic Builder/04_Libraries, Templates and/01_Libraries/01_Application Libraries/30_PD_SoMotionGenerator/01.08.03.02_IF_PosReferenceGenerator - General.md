Task

Interface for implementing external positioning jobs

Description

It is possible to transfer external positioning jobs to the SoMotionGenerator. For this purpose, the user has to transfer a function block to the structure [ST\_ExternalPos](../Structures/Structures-5.htm#XREF_D_SE_0089486_1) via the parameter ifExternalPosGenerator which implements the interface IF\_PosReferenceGenerator.

The calculation of reference values is executed in the method Calculate() and is called up by the SoMotionGenerator in the real-time cycle.
