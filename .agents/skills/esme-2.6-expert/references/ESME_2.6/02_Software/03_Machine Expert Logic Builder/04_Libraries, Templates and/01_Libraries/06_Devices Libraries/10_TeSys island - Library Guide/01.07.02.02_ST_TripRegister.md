## Description

The structure ST\_TripRegister contains date, time, and an identifier for the most recent detected trip event.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| RTS\_SYSTIMEDATE | SysTimeRtc.RTS\_SYSTIMEDATE | Structure of date and time from the [SysTimeRtc library](../../../../../api/crossBook?lang=en-US&virtualBookName=RTCmanag&topicID=D_SE_0067239). |
| uiTripId | UINT | Identifier of the detected trip event.  Description of the detected trip events:   * 0000: No error detected * 0001: Thermal overload * 0002: Motor overheat * 0003: Jam * 0004: Undercurrent * 0005: Long start * 0006: Overcurrent * 0007: Stall * 0008: Ground current * 0009: Current phase reversal * 0010: Phase configuration * 0011: Current phase unbalance * 0012: Current phase loss |
