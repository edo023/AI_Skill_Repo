## New Functions

* [**IF\_ConfigurationAdvanced.ExternalPositionSource(…)**](D-SE-0096933.html#D-SE-0096933)

  The feature provides the necessary elements to configure and implement an external position source of the robot and write the reference values for the robot components Cartesian, orientation, and auxiliary axes.
* [**ST\_ModuleInterface.iq\_ifWarmStart**](D-SE-0076969.html#D-SE-0076969__D-SE-0076969.4)

  The interface of type *[ROB.IF\_RobotWarmStart](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=D_SE_0096624)* provides methods and properties to perform a sequenced warm start of the configured robot components.

## Modifications

**None**

## Enhancements

* **FB\_RoboticModuleVisController - FR\_RoboticInterface1015x650 - SercDrive**

  Reading the RefPosition of a SerDrive object for visualization has been improved.
* **FB\_RoboticModuleVisController - FR\_RoboticInterface1015x650**

  The visualization on switching between robots with or without auxiliary axes and/or orientations has been improved.
* **Diagnostic messages of methods in IF\_Manual**

  Forwarding the diagnostic messages from the library Robotic to the library RoboticModule of the following methods has been improved:

  + IF\_Manual.SetMaxAcceleration(…)
  + IF\_Manual.SetMaxDeceleration(…)
  + IF\_Manual.SetMaxVelocity(…)
  + IF\_Manual.SetParameter(…)
  + IF\_Manual.SetRamp(…)
