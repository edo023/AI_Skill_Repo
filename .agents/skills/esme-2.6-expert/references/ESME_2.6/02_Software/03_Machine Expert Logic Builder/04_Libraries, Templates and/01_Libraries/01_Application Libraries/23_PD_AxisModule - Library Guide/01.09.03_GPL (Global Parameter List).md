Description

The global parameter list contains the global constants of the PD\_AxisModule library which can be overwritten project-specifically in the library manager.

Global Parameters

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_udiMaxNumberOfCamTables | UDINT | 4 | The highest index of the cam table which is to be switched in the axis module using the TabSwitch function. Counting starts at 0. |
