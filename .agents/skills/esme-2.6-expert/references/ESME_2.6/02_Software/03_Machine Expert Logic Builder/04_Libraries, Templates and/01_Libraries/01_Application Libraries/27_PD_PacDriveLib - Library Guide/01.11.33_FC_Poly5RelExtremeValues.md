Task

Calculate all relative extreme of the 5th polynomial order.

Description

Calculates all relative extreme values of the 5th order polynomial given by the coefficient array i\_alrCoefficients (for its entire definition range, that is, the entire real number range). The relative extreme values are returned as a structure of the type [ST\_XYStruct](../Structures/Structures-49.htm#XREF_D_SE_0087804_1).

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_alrCoefficients | ARRAY[0 .. 5] OF LREAL | List of the coefficients, with the element 5 being the coefficient of the highest order and the element 0 being the coefficient of the lowest order. |
