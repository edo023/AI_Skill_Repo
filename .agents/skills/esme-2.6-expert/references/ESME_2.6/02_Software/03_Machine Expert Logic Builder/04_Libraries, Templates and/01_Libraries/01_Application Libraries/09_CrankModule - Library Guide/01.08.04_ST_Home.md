# ST_Home

ST\_Home

ST\_Home - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.2.0 |
| Inherits from: | - |

Description

In this structure the parameters and the return values from the operation mode Homing of the function block [FB\_CrankModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0088740_1) are summarized.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_xEnableInitAbsReadPos | BOOL | Activate the absolute encoder homing when enabling the module. See "Absolute encoder homing".  TRUE: = When initializing the module the absolute position of the axis encoder is read out and the axis is referenced to this position. Hereby the i\_etMode parameter of this structure must be allocated with [RestorePosFromAxisEncoder](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1) or with [RestorePosFromRetain](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).  FALSE: = No absolute value encoder homing takes place. |
| i\_xResetHomeOk | BOOL | TRUE: The q\_xHomeOk marker of the AxisModule is reset. |
| i\_etMode | [PDL.ET\_HomeMode](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1) | The mode with which the homing shall be performed. |
| stTouchProbe | [PDL.ST\_HomeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087740_1) | Parameter for the homing with Touch Probe. |
| stSensor | [PDL.ST\_HomeIn](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087730_1) | Parameter for the homing to the sensor. |
| stLimitSwitch | [PDL.ST\_HomeLimitSwitch](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087732_1) | Parameter for the homing to the limit switch. |
| stTorque | [PDL.ST\_HomeTorque](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087738_1) | Parameter for the homing to the limit stop. |
| stMoveOnPos | [PDL.ST\_HomeMoveOnPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087734_1) | Parameter for the homing with motion to the position. |
| stSetPos | [ST\_HomeSetPos](Structures-5.htm#XREF_D_SE_0088758_1) | Parameter for the homing with setting the position. |
| stWritePos | [PDL.ST\_HomeWritePos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087742_1) | Parameter for writing the axis encoder. |
