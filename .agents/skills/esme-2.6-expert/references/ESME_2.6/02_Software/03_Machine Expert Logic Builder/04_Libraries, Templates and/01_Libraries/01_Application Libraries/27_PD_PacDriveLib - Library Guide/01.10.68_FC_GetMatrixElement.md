# FC_GetMatrixElement

![](images/logo.png)

FC\_GetMatrixElement

FC\_GetMatrixElement - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Read the element of a matrix.

Description

Returns the matrix element that is located in line i\_diRow and column i\_diCol of a matrix. The pointer i\_plrMatrixStart must point to the start of the memory area in which the matrix is stored. i\_diNumCols must contain the number of the columns of the matrix. i\_diRow and i\_diCol both start at 0.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_plrMatrixStart | POINTER TO LREAL | Pointer to the start address of the matrix to be processed |
| i\_diNumCols | DINT | Number of columns contained in the matrix |
| i\_diRow | DINT | Line containing the matrix element to be read |
| i\_diCol | DINT | Column containing the matrix element to be read |
