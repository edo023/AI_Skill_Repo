Description

This structure contains all the module specific parameters and return values for the FB\_UnwinderModule.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| stMain | ST\_Main | Cross operating mode parameters and return values. |
| stAuto | ST\_Automatic | Parameters and return values of the operating mode Automatic. |
| stManual | ST\_Manual | Parameters and return values of the operating mode Manual. |
| stBrake | ST\_BrakeRelease | Parameters and return values of the operating mode Brake. |
