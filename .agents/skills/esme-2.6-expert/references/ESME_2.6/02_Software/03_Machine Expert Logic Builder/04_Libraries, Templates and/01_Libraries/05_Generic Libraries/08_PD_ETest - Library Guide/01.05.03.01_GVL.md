## Description

The global variable list contains the global variables of the PD\_ETest library.

## Global Variables

| Variable | Data type | Description |
| --- | --- | --- |
| G\_ifMeasurandList | IF\_MeasurandList | The variable:   * Is valid and initialized only during test execution. * Requires EcoStruxure Machine Expert version V2.0 or later. If you attempt to execute tests in earlier versions, make sure to verify the validity of the variable and call the obsolete variant of programming code for recording measured values `TestManager.fbMeasurandList.AddMeasurand(...)` as described in the documentation applicable to the EcoStruxure Machine Expert version used. |
