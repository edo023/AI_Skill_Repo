# ST\_Element

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.4.0 |
| Inherits from: | – |

## Description

The structure ST\_Element is part of ST\_Content. It contains specific information for each element in the selected external FTP server directory after ET\_FtpCommand.List has been executed.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sName | STRING [255] | Name of the element. |
| xIsDirectory | BOOL | TRUE: Element is a directory.  FALSE: Element is a file. |
| udiSize | UDINT | File size in bytes. |
| sDate | STRING [255] | Last modification date, displayed in one of the following formats:   * <month day time>, for example, Jul 05 14:30 * <month day year>, for example, Jul 05 2018 |
