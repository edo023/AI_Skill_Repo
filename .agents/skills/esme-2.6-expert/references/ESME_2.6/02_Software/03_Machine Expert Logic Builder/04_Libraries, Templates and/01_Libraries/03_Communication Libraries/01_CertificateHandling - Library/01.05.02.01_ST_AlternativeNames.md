# ST\_AlternativeNames

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

The structure ST\_AlternativeNames is used to provide the alternative names as part of the attributes of a certificate.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sIpAddress | STRING[15] | The IP address used as alternative name. |
| sUri | STRING[255] | The Uniform Resource Identifier (URI) used as alternative name. |
| sDns | STRING[255] | The Domain Name System (DNS) used as alternative name. |
| sEMail | STRING[255] | The email address used as alternative name. |
