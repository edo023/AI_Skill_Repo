# ET_PreControllerMode

![](images/logo.png)

ET\_PreControllerMode

ET\_PreControllerMode - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

Description

This enumeration defines the different modes for the precontrol movement of the FB\_Unwinder.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Master | 1 | There should be a precontrol according to the movement of the feeder. |
| Velocity | 2 | There should be a precontrol according to a specified (middle) velocity of the feeder. |
