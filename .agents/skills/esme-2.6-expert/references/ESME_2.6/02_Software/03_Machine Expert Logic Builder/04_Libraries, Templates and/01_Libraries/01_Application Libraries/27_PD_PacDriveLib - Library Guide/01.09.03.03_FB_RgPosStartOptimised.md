Task

Trigger the calculation of the required motion.

Description

This method triggers the calculation of the required motion.

In case of success the job result ([GetJobResult](Function_Blocks_R_to_Z-12.htm#XREF_D_SE_0087353_1)) is set to Ok, otherwise to UnableToCreateMotionProfiles.

As this method has no return value, the job result has to be checked afterwards (see method [GetJobResult](Function_Blocks_R_to_Z-12.htm#XREF_D_SE_0087353_1)).

The diagnostic messages have to be checked after execution (see method [GetJobResult](Function_Blocks_R_to_Z-12.htm#XREF_D_SE_0087353_1)).
