# ET\_State

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.1.2.0 |

## Description

The enumeration ET\_State values indicate the state of the function block.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Idle | 0 | The function block is in idle state. |
| RetrievingCertificate | 1 | The function block is retrieving the certificate. |
| CertificateSelected | 2 | The certificate is selected. |
| RetrievingAttribute | 3 | The function block is retrieving the attribute. |
| DeletingCertificate | 4 | The function block is deleting the certificate. |
| CopyingCertificate | 5 | The function block is copying the certificate. |
