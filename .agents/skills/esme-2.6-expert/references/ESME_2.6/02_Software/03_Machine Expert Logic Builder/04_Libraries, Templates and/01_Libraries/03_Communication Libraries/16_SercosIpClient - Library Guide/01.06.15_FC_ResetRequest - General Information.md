# FC\_ResetRequest - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.2.0 |

## Task

This function sends the request for the Reset service. This service resets one or several Sercos devices on the network.

## Functional Description

This function sends the request for the Reset service. Because the Reset service interrupts the communication, the reset has to be delayed by the server (Sercos device). Sercos devices do not send a response to this request.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiTimeout | UDINT | Waiting time in milliseconds until the reset is performed by the Sercos device |
| i\_stNodeIdentifiers | ST\_NodeIdentifiers | Node identifiers of the Sercos devices to be reset. See [ST\_NodeIdentifiers](ST_NodeIdentifiers-GeneralInformati-089B770B.html) for additional information. |
