## Task

Interface with parameters from the move command [MoveDirectly](IF_MoveDirectly-5B82D7F8.html#IF_MoveDirectly-5B82D7F8).

## Description

The feedback interface provides parameters from the movement of the carrier with the move command MoveDirectly.
