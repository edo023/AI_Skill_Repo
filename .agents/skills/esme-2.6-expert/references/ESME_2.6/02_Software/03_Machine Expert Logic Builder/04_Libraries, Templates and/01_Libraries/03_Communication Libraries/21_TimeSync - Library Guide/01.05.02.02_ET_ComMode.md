# ET_ComMode

![](images/logo.png)

ET\_ComMode

Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.7.0 |

Description

The enumeration ET\_ComMode defines the communication mode of the [SNTP](../glossary/glossary.htm#XREF_D_SE_0024697_818) client. This version of the library only supports unicast mode. Therefore, the value is fixed to 0.

Enumeration Elements

| Name | Data type | Value | Description |
| --- | --- | --- | --- |
| Unicast | UDINT | 0 | A point-to-point connection is established: A [unicast](../glossary/glossary.htm#XREF_D_SE_0055638_22) client sends a request to a designated server at its unicast address and expects a reply from which it can determine the time. |
