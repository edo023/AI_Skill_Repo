Description

In this structure the parameters for the homing of the [FB\_AxisModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0077136_1) are summarized at which the position is set.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_lrHomePosition | LREAL | The position to which the axis is to be set. See [PDL.ST\_HomeSetPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087736_1) |
| i\_lrEncoderPosition | LREAL | The position to which the logical encoder is to be set. See [PDL.ST\_HomeSetPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087736_1) |
| i\_lrUserPeriod | LREAL | The period in which the position is to be set. See [PDL.ST\_HomeSetPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087736_1) |
| i\_lrAbsEncOffset | LREAL | Offset of the encoder jump. See [PDL.ST\_HomeSetPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087736_1) |
| i\_lrRestoreWindow | LREAL | Maximum permissible motion of the axis in switched-off state. See [PDL.ST\_HomeSetPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087736_1) |
| iq\_pstRestorePosRetain | POINTER TO [PDL.ST\_AxisRestorePosition](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087712_1) | Pointer onto the variable to which the axis position was saved. See [PDL.ST\_HomeSetPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087736_1) |
