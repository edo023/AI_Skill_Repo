## Description

The method SetPosition of the function block [FB\_MasterChannel](FB_MasterChannel-4CFA1D24.html) is used to adjust the internal position reference of the master channel. This allows the user-defined algorithm to apply an absolute or relative offset to the tracked master axis position.

Calling this method only affects the position value provided by the master channel; it has no effect on the physical position or movement of the axis connected to the master channel. Following the call, the cyclic tracking of the axis movement continues from the newly defined position.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrPosition | LREAL | The new position value to be applied to the master channel reference. |
| i\_xRelative | BOOL | If set to TRUE, the value of i\_lrPosition is added to the current master channel position (relative). If set to FALSE, the position is set as an absolute value. |
