# FB\_WriteFile Rules and Constraints

## Rules and Constraints

Comply with the following constraints for opening or creating files on the file system of the controller, or on the extended memory:

* Do not change the data from the buffer provided via i\_pbBuffer until the output q\_xBusy of function block indicates TRUE.
* Do not access the same file with different function blocks at the same time. Use the output q\_xBusy to lock the execution of different function blocks.
