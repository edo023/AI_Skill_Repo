Task

Initialization of the random generator [FB\_RandomGenerator](Function_Blocks_R_to_Z-3.htm#XREF_D_SE_0087344_1)

Description

As the generation of the random numbers is effected in accordance with the linear congruence method, a start value i\_diNstart is required which is set by means of the Init method. This can be any integer > 0 which is located within the value range of data type DINT.

This method must have been called up once before calling up the [GetDint](Function_Blocks_R_to_Z-4.htm#XREF_D_SE_0087345_1) or [GetLreal](Function_Blocks_R_to_Z-5.htm#XREF_D_SE_0087346_1) methods.
