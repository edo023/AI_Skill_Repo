## Functional Description

SR\_<Camera Name> provides a vision server to receive data from a camera.

If a camera is connected to this server (q\_xClientConnected = TRUE), SR\_<Camera Name> can receive data.

SR\_<Camera Name>.xVisionDataReceived signals that data has been received. The data can be read with the methods GetHeader, GetProducts, and GetAdditionalString.

If data is already received, the method *[ClearVisionData](../../../../../api/crossBook?lang=en-US&virtualBookName=SERToolb&topicID=D_SE_0071257)* must be called before new data can be received.
