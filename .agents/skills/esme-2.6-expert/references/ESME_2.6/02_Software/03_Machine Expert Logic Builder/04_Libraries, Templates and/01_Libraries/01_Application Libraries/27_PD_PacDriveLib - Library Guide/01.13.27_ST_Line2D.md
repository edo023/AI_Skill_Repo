Description

This data structure defines a straight line in the 2-dimensional space which passes through the points stPoint1 and stPoint2. The points stPoint1 and stPoint2 must be different.

For the straight lines, the following default parameter representation is always used:

Line(Parameter) = Line.stPoint1 + Parameter \* (Line.stPoint2 - Line.stPoint1)
