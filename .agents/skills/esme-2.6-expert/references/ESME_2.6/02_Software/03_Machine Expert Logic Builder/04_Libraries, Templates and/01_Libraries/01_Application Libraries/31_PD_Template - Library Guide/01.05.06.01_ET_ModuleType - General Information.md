# ET\_ModuleType - General Information

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.4.0 |

## Description

This enumeration type contains numbers for the various module types offered by Schneider Electric.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No equipment module |
| NodeModule | 1 | Module for structuring the machine |
| MachineModule | 2 | Module for the machine |
| AxisModule | 3 | Module for general motion of an axis |
| MultiBeltModule | 4 | Module for a MultiBelt mechanism |
| SmartInfeedModule | 5 | Module for a smart infeed mechanism |
| RoboticModule | 6 | Module for a robot mechanism |
| UnwinderModule | 7 | Module for a winder mechanism |
| SealingModule | 8 | Module for a sealing mechanism |
| AtvModule | 9 | Module for a frequency converter of type Avitar |
| IlxModule | 10 | Module for an integrated drive of type ILA, ILE or ILS |
| CrankModule | 11 | Module for a crank mechanism |
| ValveModule | 12 | Module for a valve |
| CapperModule | 13 | Module for a locking mechanism |
| RoboticModuleForTemplate | 13 | Module for a robot mechanism for Template |
