# FC_CommonPointsOfTwoParallelograms

![](images/logo.png)

FC\_CommonPointsOfTwoParallelograms

FC\_CommonPointsOfTwoParallelograms - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Calculates the intersections of two parallelograms in the 2-dimensional space.

Description

Calculates the intersections of the parallelograms i\_stParallelogram1 and i\_stParallel­ogram2. If there are no intersections, an additional verification is performed to determine whether one of the objects is contained in the other.
