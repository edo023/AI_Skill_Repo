# ET\_SecurityLevel

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.3.4.0 |

## Description

The enumeration ET\_SecurityLevel specifies the security level for the SNMPv3 request sent by the SNMP manager.

## Enumeration Elements

| Name | Data type | Value | Description |
| --- | --- | --- | --- |
| NotDefined | BYTE | 0 | A security level is not defined. |
| NoAuthNoPriv | BYTE | 1 | Without authentication and without privacy. |
| AuthNoPriv | BYTE | 2 | With authentication but without privacy. |
| AuthPriv | BYTE | 3 | With authentication and with privacy. |
