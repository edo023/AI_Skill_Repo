## New Features

The interface IF\_TrackingSource with the following methods is added:

* ConfigDone - Verify and complete the configuration of the tracking source.
* EnableComponent - Additional tracking components can be enabled.
* EnablePositionMonitoring - Enable the monitoring of the tracking position to be contiguous.
* GetCoordinateSystem - Read the geometrical configuration of the tracking coordinate system.
* GetMotionParameters - Read the motion parameters of the tracking source.
* GetOptionalParameters - Read the optional tracking parameters of the tracking source.
* ModifyCoordinateSystem - Modify the geometrical configuration of the tracking coordinate system.
* SetCoordinateSystem - Geometrical configuration of the tracking coordinate system.
* SetMotionParameters - Set the motion parameters for the tracking source.
* SetOptionalParameters - Sets optional parameters for the tracking source.
* UpdateTrackingDataCartesian - Called by the FB\_Robot every Sercos cycle.

To the interface IF\_RobotConfiguration, the following methods are added:

* AddTrackingSource - Adding a configured tracking source to the robot.
* GetTrackingSource - Getting the interface to a configured tracking source of the robot.

The interface IF\_UserRobotFeedbackKinematic with the following property is added:

ralrRefPositionJoint - User-defined joint values for user-defined kinematics.

## Modifications

Configuring a robot as a four-axis Delta robot is no longer supported. The method Delta4Ax() of the interface IF\_RobotConfiguration was removed.

## Enhancements

**Modulo axis on the M262 platform**

In case an axis configured to the robot is configured as a modulo axis, the diagnostic message GD.ET\_Diag.DriveConditionInvalid / ET\_DiagExt.ModuloAxisNotSupported is returned by the robot on the M262 platform.

**Calculating a Motion Profile**

Under certain conditions, the diagnostic message ET\_Diag.UnexpectedProgramBehavior was returned by a move command (MoveL, MoveS, MoveC) in connection with calculating the motion profile of the connected path. Calculating the motion profile is improved.

**Getting the space position on path**

Under certain conditions, the diagnostic message ET\_Diag.UnexpectedProgramBehavior was returned by FB\_Robot.etDiag in connection with getting the space position on path. Getting the space position on path is improved.
