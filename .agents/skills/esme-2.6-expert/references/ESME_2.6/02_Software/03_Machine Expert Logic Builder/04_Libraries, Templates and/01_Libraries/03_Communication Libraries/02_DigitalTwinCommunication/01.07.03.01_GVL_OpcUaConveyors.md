# GVL\_OpcUaConveyors

## Overview

|  |  |
| --- | --- |
| Type: | Global variables |
| Available as of: | V1.1.0.0 |

## Description

The global variable list GVL\_OpcUaConveyors  contains variable declarations for the build and emulation of conveyors through OPC UA communication by using the Symbol Configuration.

Verify that in the [Symbol Configuration](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083586) the emulation data are selected and the access rights are in line with the application needs.
