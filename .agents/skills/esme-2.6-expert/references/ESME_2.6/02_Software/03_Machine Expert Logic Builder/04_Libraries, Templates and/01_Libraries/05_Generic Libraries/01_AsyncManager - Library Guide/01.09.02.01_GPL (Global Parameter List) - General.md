## Description

Global parameters of this library.

## Global Parameters

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_udiMaxNumOfReusableObjects | UDINT | 100 | Determines the number of jobs which can be managed by the AsyncManager at the same time. The value corresponds to the length of the queue containing the jobs to be executed asynchronously. The jobs are distributed across the available asynchronous tasks. |
| Gc\_udiNumOfAsyncTasks | UDINT(1..10) | 1 | Determines the number of additional tasks to be created during the initialization of the AsyncManager. |
