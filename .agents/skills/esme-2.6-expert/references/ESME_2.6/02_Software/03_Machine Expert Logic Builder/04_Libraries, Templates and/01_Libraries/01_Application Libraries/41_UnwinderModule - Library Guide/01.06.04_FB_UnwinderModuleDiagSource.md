# FB_UnwinderModuleDiagSource

![](images/logo.png)

FB\_UnwinderModuleDiagSource

FB\_UnwinderModuleDiagSource - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.0.0.0 |

Task

Output of the additional interface of the FB\_UnwinderModule in a Diagnostics prompt.

Description

This POU provides the variables in the additional interface of all unwinder modules as a diagnostics source to diagnose the template project with Diagnostics. It lists all the modules, starting with the main machine. For every unwinder module, the entire additional interface of this module is output.

Interface

| Input | Data type | Description |
