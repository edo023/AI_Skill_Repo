Manual Commissioning

Description

The manual commissioning described in this chapter can be performed alternatively to the usual [Commissioning](../Commissioning/Commissioning-1.htm#XREF_D_SE_0091179_1). The expenditure of time for the manual commissioning is higher. This is why you should use the [Commissioning](../Commissioning/Commissioning-1.htm#XREF_D_SE_0091179_1).

![G-SE-0071530.1.gif-high.gif](images/G-SE-0071530.1.gif-high.gif)

| Step | Action |
| --- | --- |
| 1 | Double-click Library Manager in the PLC Configuration.  Result: The library manager with the libraries already connected is displayed. |
| 2 | Insert the AutoTune.lib library.  Click Add library....  Result: The window "Add Library" opens.  Select the AutoTune library from the Application category and add by clicking OK. |

Integrating the AutoTune Visualization

| Step | Action |
| --- | --- |
| 1 | Add a new visualization object to the project. Right-click Application and subsequently Add object ... > Visualization.  Result: The window "Add Visualization" opens. |
| 2 | Enter the name of the new visualization (e.g. AutoTuneVis) and confirm by clicking Open. |
| 3 | Open the new visualization and add a "Frame" object. Select the "Frame" symbol in the "ToolBox" window.  G-SE-0071531.1.gif-high.gif |
| 4 | Place the "Frame" object in the visualization.  G-SE-0071532.1.gif-high.gif |
| 5 | Right-click on the "Frame" object and select "Frame selection".  Result: The "Select Visualization" window appears. |
| 6 | Select the "AUT.VIS\_AutoTune" visualization and click the right arrow. Confirm your selection by clicking OK. |
| 7 | Select the "Frame" object and change the following properties:  oX = 0  oY = 0  oWidth = 1024  oHeight = 768 |
| 8 | In the properties, click the button under Reference > AUT.VIS\_AutoTune > iq\_stAutoTuneVisData.  G-SE-0071533.1.gif-high.gif |
| 9 | In the input assistant, select the variable AUT > AutoTune > GVL > G\_stAutoTuneVisData and click OK.  G-SE-0071534.1.gif-high.gif |

Integrating the SR\_AutoTune Program

| Step | Action |
| --- | --- |
| 1 | Right-click the Task Configuration and select Add object > Task.  Result: The window "Add Task" opens. |
| 2 | Enter the name of the new task (e.g. TASK\_SR\_AutoTune) and confirm by clicking Open. |
| 3 | Open the new task object. |
| 4 | Set the cycle time to 100 ms and disable the watchdog.  G-SE-0071535.1.gif-high.gif |
| 5 | Click Add Call-up..., select AUT > AutoTune > SR\_AutoTune and confirm by clicking OK in the input assistant.  G-SE-0071536.1.gif-high.gif |
