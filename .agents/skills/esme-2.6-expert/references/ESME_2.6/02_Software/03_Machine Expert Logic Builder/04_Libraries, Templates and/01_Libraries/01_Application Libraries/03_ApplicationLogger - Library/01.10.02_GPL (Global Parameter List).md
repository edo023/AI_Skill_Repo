# GPL (Global Parameter List)

## Overview

|  |  |
| --- | --- |
| Type: | Global parameters |
| Available as of: | V1.0.4.0 |

## Description

GPL (**G**lobal **P**arameter **L**ist) of the **ApplicationLogger** library.

NOTE: As of version V2.0.0.0, the variables are read-only. If you want to adjust the value of a variable, you need to adjust the corresponding value in the **ApplicationLoggerBase** library. If a default value of a global parameter has been modified in a version prior to V2.0.0.0, verify the value of the global parameter after an upgrade to the newest version.

## Global Parameters

As of version V2.0.0.0:

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_etDefaultLogLevel | ET\_LogLevel | APL\_Base.GPL.Gc\_etDefaultLogLevel (Default value: Nothing) | The default logger level of the Application Logger. |
| Gc\_udiMaxNumberOfLogEntries | UDINT | APL\_Base.GPL.Gc\_udiMaxNumberOfLogEntries (Default value: 500) | The maximum number of messages that are stored in the ring buffer of the Application Logger on the controller. |
| Gc\_udiMaxNumberOfLoggerPoints | UDINT | APL\_Base.Gc\_udiMaxNumberOfLoggerPoints (Default value: 100) | The maximum number of logger points that can be registered to the Application Logger. |

Version prior to V2.0.0.0:

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_etDefaultLogLevel | ET\_LogLevel | ET\_LogLevel.Nothing | The default logger level of the Application Logger. |
| Gc\_udiMaxNumberOfLogEntries | UDINT | 500 | The maximum number of messages that are stored in the ring buffer of the Application Logger on the controller. |
| Gc\_udiMaxNumberOfLoggerPoints | UDINT | 100 | The maximum number of logger points that can be registered to the Application Logger. |

NOTE: If the parameter Gc\_udiMaxNumberOfLogEntries is set too low, messages in the ring buffer can be overwritten and get lost. For the Application Logger to automatically load new messages, the setting in Gc\_udiMaxNumberOfLogEntries must be greater than 5s/interval of log entries.
