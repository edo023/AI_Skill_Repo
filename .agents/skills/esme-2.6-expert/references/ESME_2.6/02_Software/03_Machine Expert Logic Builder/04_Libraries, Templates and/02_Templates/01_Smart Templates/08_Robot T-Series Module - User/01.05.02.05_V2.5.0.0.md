* ModifyCoordinateSystem can compensate orientation set in RobotCell automatically.

* IN/OUT variable iq\_stRobotModuleInterface is added.

* The building of Active and Ready feedback value is improved
* Start Sercos scan button is renamed to Scan configuration.
* Communication variables for DigitalTwin are improved.
