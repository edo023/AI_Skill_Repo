# ET\_SearchTypeFilter

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_SearchTypeFilter specifies which structure element of ST\_PeerDeviceInfo is used for the search.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| IPv6Address | 0 | Search filter is set to ST\_PeerDeviceInfo.i\_sIPv6Address. |
| IPv4Address | 1 | Search filter is set to ST\_PeerDeviceInfo.i\_sIPv4Address. |
| FriendlyName | 2 | Search filter is set to ST\_PeerDeviceInfo.i\_sFriendlyName. |
| ModelName | 3 | Search filter is set to ST\_PeerDeviceInfo.i\_sModelName. |
| NoFilter | 4 | If this filter type is set, the search mechanism is providing data of the first peer with a unique ID greater than the one provided in i\_udiSearchUniqueID (FC\_GetPeerScanData). |

## Used By

* FC\_GetPeerScanData
