# FC\_Matrix3DScalarMultiplication

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

Given a 3D matrix and a scalar value, the function returns a new 3D matrix resulting from multiplying the scalar value by each element of the input matrix.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stMatrix | [ST\_Matrix3D](ST_Matrix3D-GeneralInformation-0C04E9BF.html#ST_Matrix3D-GeneralInformation-0C04E9BF) | A 3D matrix. |
| i\_lrScalar | LREAL | A scalar value to add to the elements of i\_stMatrix. |

| Output | Data type | Description |
| --- | --- | --- |

## Return Value
