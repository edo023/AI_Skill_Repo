# IF_ProductManager - GetFirstProductInterfaceOnBelt (Method)

![](images/logo.png)

IF\_ProductManager - GetFirstProductInterfaceOnBelt (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Returns the interface to the first product of a belt

Description

The product that is closest to the end of the belt is determined and its interface returned.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiBeltId | UDINT | The number of the belt the first product of which is to be determined |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) | General, library-independent statement on the diagnosis.  A value unequal GD.ET\_Diag.Ok equates an error message. |
| q\_etDiagExt | [ET\_DiagExt](../Enumerations/Enumerations-4.htm#XREF_D_SE_0080267_1) | POU-specific output on the diagnosis.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> error message |
| q\_ifFirstProduct | [IF\_Product](Interfaces-14.htm#XREF_D_SE_0080325_1) | The interface with the product that is closest to the end of the belt |

Diagnostic Messages

| q\_etDiag | q\_etDiagExt | Enumeration value | Description |

| InputParameterInvalid | [BeltIdRange](#XREF_D_SE_0080331_7) | 11 | BeltId is outside the valid range. |

BeltIdRange

|  |  |
| --- | --- |
| Enumeration name: | BeltIdRange |
| Enumeration value: | 11 |
| Description: | BeltId is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid value has been transferred at the input i\_udiBeltId. | At the i\_udiBeltId input, the number of the belt has to be specified the interface of the first product of which is to be returned. The number must be between one and the number of parameterized belts. |

NoProductExisting

|  |  |
| --- | --- |
| Enumeration name: | NoProductExisting |
| Enumeration value: | 71 |
| Description: | No product could be found. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The product does not exist. | It must be made sure that a product is located on the belt selected at the i\_udiBeltId input. |

Ok

|  |  |
