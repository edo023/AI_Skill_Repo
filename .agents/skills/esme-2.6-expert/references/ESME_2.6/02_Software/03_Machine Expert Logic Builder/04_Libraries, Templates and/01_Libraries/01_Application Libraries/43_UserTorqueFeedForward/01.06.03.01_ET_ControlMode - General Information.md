# ET\_ControlMode - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.5.1.0 |
| Versions: | Installed version |

## Description

The enumeration ET\_ControlMode contains the possible values regarding the torque feed forward control modes.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Disabled | 0 | The feed forward is disabled. Neither the drive nor the POU perform any calculations. The torque feed forward value set by the POUs in this library has no effect. |
| Default | 1 | Default feed forward in the drive.  The feed forward is activated. The feed forward values are automatically calculated in the drive. The feed forward is influenced by the drive parameters   * J\_Gear * GearIn * GearOut * J\_Load * StaticFriction * ViscousFriction   The POU does not perform any calculations. The torque feed forward value set by the POUs in this library has no effect. |
| UserLoad | 2 | Feed forward by the user.  The feed forward is activated. You must calculate the feed forward values and transfer them to the POU with the corresponding parameter ([FB\_TorqueFeedForward.i\_lrTorque](D-SE-0090563.html) or [FB\_TorqueFeedForward2.SetRefTorque.i\_lrValue](FB_TorqueFeedForward2-SetRefTorqueM-46B17177.html)). In doing so, torques must be calculated that occur due to the load (mechanics without gear box and product). The torque for acceleration of the motor (including gear box) is automatically calculated in the drive and added to the feed forward value. This calculation is influenced by the drive parameter J\_Gear. |
| UserAdditional | 3 | Feed forward by the user and default feed forward.  The feed forward is activated. You must calculate the feed forward values that deviate from the default feed forward and transfer them to the POU with the corresponding parameter ([FB\_TorqueFeedForward.i\_lrTorque](D-SE-0090563.html) or [FB\_TorqueFeedForward2.SetRefTorque.i\_lrValue](FB_TorqueFeedForward2-SetRefTorqueM-46B17177.html)). The default feed forward is automatically calculated in the drive (refer to ET\_ControlMode.Default).  The user-specific feed forward value is added to the automatically calculated feed forward value. |
| UserFull | 4 | Feed forward by the user only.  The feed forward is activated. You must calculate the feed forward values and transfer them to the POU with the corresponding parameter ([FB\_TorqueFeedForward.i\_lrTorque](D-SE-0090563.html) or [FB\_TorqueFeedForward2.SetRefTorque.i\_lrValue](FB_TorqueFeedForward2-SetRefTorqueM-46B17177.html)). In doing so, torques must be calculated that occur due to the load, gearbox and motor.  No calculations are performed within the drive. |
