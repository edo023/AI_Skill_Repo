## Functional Description

The `Fahrenheit_TO_Celsius` function converts temperature in Fahrenheit to Celsius.

Use `Celsius_TO_Fahrenheit` for the reverse process.

Formula: T\_Celcius = [(T\_Fahrenheit - 32) / 1.8]
