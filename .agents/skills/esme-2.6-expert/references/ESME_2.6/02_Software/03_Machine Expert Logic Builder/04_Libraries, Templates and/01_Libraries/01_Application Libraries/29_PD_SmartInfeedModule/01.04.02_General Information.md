# General Information

![](images/logo.png)

General Information

Description

The library "PD\_SmartInfeedModule" contains the equipment module for the template as well as all required auxiliary functions.

The SmartInfeed module activates an infeed distance with serial belts.

NOTE: The equipment module includes no application-specific algorithms, i.e. the method used to implement the infeed must be selected by the user. This must be implemented as a component of the application program and determines how the technology module of the [PD\_SmartInfeed](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SmartInfeed&topicID=D_SE_0090098_1) library is activated.

NOTE: Only Controller OnBoard I/O touch probes and drive touch probes are supported by the functionalities of this library. TM5 module touch probes are not supported.

Namespace of the Library

The default namespace of this library is SIM.

The POUs, data structures, enumerations and constants have to be addressed via this namespace.
