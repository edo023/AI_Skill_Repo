| Inherits from: | - |

Description

Data structure for the limits to be observed by the slave axes of an ILS axes system.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| lrPosVel | LREAL | Limit value for the positive velocity of the slave axis. |
| lrNegVel | LREAL | Limit value for the negative velocity of the slave axis. |
| lrPosAcc | LREAL | Limit value for the positive acceleration of the slave axis. |
| lrNegAcc | LREAL | Limit value for the negative acceleration of the slave axis. |
