# General Information

## Overview

The SchneiderElectricRobotics Toolbox-Migration library provides structures, functions and function blocks:

* To generate random Cartesian poses.
* To parameterize and generate a list of targets.
* To handle targets in a moving system.
* To select a target from a moving system, and to balance the load of a robot in a multi-robot application.

## Namespace of the Library

The default namespace of the SchneiderElectricRobotics Toolbox-Migration library is SERT\_M.

The POUs, data structures, enumerations, and constants must be addressed via this namespace.
