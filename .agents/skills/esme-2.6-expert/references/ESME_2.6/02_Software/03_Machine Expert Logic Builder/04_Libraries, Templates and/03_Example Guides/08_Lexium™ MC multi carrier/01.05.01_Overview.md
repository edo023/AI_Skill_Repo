# Overview

## Short Description

The example shows the usage of the Lexium™ MC multi carrier functionalities in the operation modes Prepare and Automatic (Auto) to control two Lexium™ MC multi carrier tracks and their carriers.

The exception handling and the command handling of the template PacDrive 3 Template Full with the function block FB\_Multicarrier are also explained. For more information on the function block FB\_Multicarrier, refer to the [Multicarrier library](../../../../../api/crossBook?lang=en-US&virtualBookName=MLSLib&topicID=FB_Multicarrier_5B874FA7).

Additionally, the project provides several application examples in the separate operation mode [TestMulticarrier](TestMulticarrier-2D7F45E5.html).
