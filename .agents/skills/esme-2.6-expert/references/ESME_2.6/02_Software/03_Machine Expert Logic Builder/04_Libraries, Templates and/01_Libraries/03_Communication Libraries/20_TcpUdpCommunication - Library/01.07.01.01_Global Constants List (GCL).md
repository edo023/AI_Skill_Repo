# Global Constants List (GCL)

## Overview

|  |  |
| --- | --- |
| Type: | Global constants |
| Available as of: | V1.0.0.0 |

## Description

The global constants list contains the global constants of the TcpUdpCommunication library.

## Global Constants

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_sLibraryVersion | STRING[80] | Vx.x.x.0(1) | Library version |
| Gc\_uiIPStringSize | UINT | 15 | Maximum length for a STRING used for specifying an IP address. |
| Gc\_udiBufferSize | UDINT | 10000 | Default value for setting the socket options “SendBuffersize” and “ReceiveBuffersize”. |
| Gc\_siMulticastDefaultTimeToLive | SINT | 16 | Default value for setting the socket option “MulticastDefaultTimeToLive” (FB\_UDPPeer). |
| **(1)** This value varies to indicate the version of the library. | | | |
