# FC_EtStateToString

![](images/logo.png)

FC\_EtStateToString

FC\_EtStateToString - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Convert the status messages of the [FB\_SmartInfeedModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080825_1) to a string.

Description

The status message i\_etState of the SmartInfeed module from the enumeration list [ET\_State](../Enumerations/Enumerations-6.htm#XREF_D_SE_0080823_1) is converted to a string.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etState | [ET\_State](../Enumerations/Enumerations-6.htm#XREF_D_SE_0080823_1) | The status to be converted |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) | General, library-independent statement on the diagnostic  A value unequal GD.ET\_Diag.Ok equates an error message. |
| q\_etDiagExt | [ET\_DiagExt](../Enumerations/Enumerations-3.htm#XREF_D_SE_0080817_1) | POU-specific output on the diagnostic  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> error message |

Return Value

| Data type | Description |
| --- | --- |
| STRING[80] | The status converted to a string |

| InputParameterInvalid | [StateUnknown](#XREF_D_SE_0080843_9) | 31 | The status is indeterminable. |
