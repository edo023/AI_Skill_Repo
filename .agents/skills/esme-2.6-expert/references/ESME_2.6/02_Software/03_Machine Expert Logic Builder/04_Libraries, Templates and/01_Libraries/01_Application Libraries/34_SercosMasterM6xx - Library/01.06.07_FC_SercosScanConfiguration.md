# FC\_SercosScanConfiguration

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

The FC\_SercosScanConfiguration function of the library SercosMasterM6xx scans the Sercos bus for the Sercos devices connected to the controller, including devices not yet configured.

If the Sercos communication phase is less than CP2, the function transitions to communication phase CP2 and scans the Sercos bus. After the scan, the function resets the communication phase to the initial communication phase.

If the Sercos communication phase is greater than or equal to CP2, the function scans the Sercos bus without changing the communication phase.

## Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stSercosConfiguration | [SMI.ST\_SercosConfiguration](../../../../../api/crossBook?lang=en-US&virtualBookName=SercosMasterInterface&topicID=ST_SercosConfiguration_53A948C8) | Structure containing the data on the scanned devices. |

NOTE: The structure SMI.ST\_SercosConfiguration is only filled with correct values if the return value is 0.

If the function is not successfully executed, the values in the structure are not modified.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: Error detected  -2: Not possible to transition to Sercos communication phase CP2 |
