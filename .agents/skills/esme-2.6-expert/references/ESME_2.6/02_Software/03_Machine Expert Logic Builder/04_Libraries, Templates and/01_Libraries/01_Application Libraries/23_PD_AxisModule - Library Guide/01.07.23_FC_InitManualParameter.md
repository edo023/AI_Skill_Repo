Task

Function for initialization of an axis that is controlled by the [FB\_AxisModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0077136_1) function block.

Description

This function is used to assign values for the manual operating mode (jogging) for the axis specified by iq\_stAxisModuleItf.

The manual operating mode allows an axis to be jogged either continuously or up to a specified distance. Continuous jogging is specified by setting the parameter i\_xEndless to TRUE. The axis will stop after the i\_lrMaxDistance distance if i\_xEndless is FALSE. The jog signal has to be removed then reapplied to continue jogging.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrVelocity | LREAL | Specifies the velocity for jogging in units/s. |
| i\_lrAcceleration | LREAL | Specifies the acceleration in units/s². |
| i\_lrDeceleration | LREAL | Specifies the deceleration in units/s2. |
| i\_lrJerk | LREAL | Specifies the jerk in units/s3. |
| i\_lrPeriod | LREAL | The position of the axis is kept between zero and period. The axis will not move if period = 0 |
| i\_lrMaxDistance | LREAL | Specifies the maximum distance the axis will move when xEndless = false |
| i\_xEndless | BOOL | xEndless = true; the axis will continue to move as long as one of the jog inputs is true. xEndless = false; the axis will continue to move as long as one of the jog inputs is true or until the distance specified by lrStep is reached. Need to remove the jog input then reapply it to continue jogging. |

| Input/Output | Data type | Description |
