## Description

Structure to store information on the kinematic parameters of a Delta robot with two axes. Refer to *[ROB.IF\_RobotConfiguration.Delta2Ax](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=D_SE_0075522)* for further details on the parameters contained in the structure.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| etWorkingPlane | *[ROB.ET\_WorkingPlane](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=D_SE_0075495)* | Specification in which working plane the robot is operating. |
| lrLengthAB0 | LREAL | Length of the distance between the axes A and B. |
| lrLengthA1 | LREAL | Length of the upper arm mounted on axis A. |
| lrLengthA2 | LREAL | Length of the lower arm mounted on axis A. |
| lrLengthB1 | LREAL | Length of the upper arm mounted on axis B. |
| lrLengthB2 | LREAL | Length of the lower arm mounted on axis B. |
| lrLengthAB3 | LREAL | Length of the distance between the two suspension points of the lower arms A and B. |
| lrLengthA4 | LREAL | Offset of the TCP in extension of the lower arm of axis A. |
| lrLengthA5 | LREAL | Offset of the TCP perpendicular to the offset `i_lrOffsetA4`. |
