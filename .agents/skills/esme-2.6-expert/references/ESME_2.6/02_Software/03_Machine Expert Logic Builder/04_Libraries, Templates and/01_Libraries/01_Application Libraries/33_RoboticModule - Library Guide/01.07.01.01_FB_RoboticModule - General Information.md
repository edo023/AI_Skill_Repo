## Task

The RoboticModule is an Equipment Module for robot functionalities in the PacDrive Template.

An Equipment Module provides the following functions:

* Encapsulation of basic functions.
* Implementation of operation modes.
* Conformity to the Standard Module Interface of the template.

## Description

The FB\_RoboticModule function block is used to manage various operation modes and to control the position of the axes. It also controls the transformation for the kinematics of the robot, if available. The POU reacts to entries on its *[TPL.ST\_StandardModuleInterface](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078570)* that are defined by the template. The iq\_diCMD input supports the commands listed in ET\_Cmd.
