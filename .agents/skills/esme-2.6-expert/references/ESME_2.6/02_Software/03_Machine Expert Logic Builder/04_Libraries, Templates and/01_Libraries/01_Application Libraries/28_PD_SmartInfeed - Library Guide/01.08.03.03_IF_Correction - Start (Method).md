# IF_Correction - Start (Method)

![](images/logo.png)

IF\_Correction - Start (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

A correction motion on a belt of the [IF\_Infeed](Interfaces-11.htm#XREF_D_SE_0080320_1) is to be started.

Description

A correction motion overlays the motion of the belt. The correction motion moves the belt by the distance i\_lrCorrection. For the correction, the motion parameters of the [IF\_Correction](Interfaces-4.htm#XREF_D_SE_0080313_1) interface are taken into account.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrCorrection | LREAL | Path of the correction motion |
| i\_udiBeltId | UDINT | Number of the belt that is to perform the correction. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) | General, library-independent statement on the diagnosis.  A value not equal to ET\_Diag.Ok corresponds to an error message. |
| q\_etDiagExt | [ET\_DiagExt](../Enumerations/Enumerations-4.htm#XREF_D_SE_0080267_1) | POU-specific output on the diagnosis.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> error message |
