ST\_SystemCam - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

Description

Data structure for parameterization of cams that have been generated using the system function [SystemInterface.FC\_ProfileLoad()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085263_1)

NOTE: Used user profiles must be available on the flash disk. When replacing the flash disk, make sure that all profile data are copied.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| diProfileID | DINT | Profile ID of active cam. Return value of [SystemInterface.FC\_ProfileLoad()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085263_1). |
| lrYFactor | LREAL | Y scaling of the system cam. The effective reference value is the result of the multiplication of the system cam by this factor. |
