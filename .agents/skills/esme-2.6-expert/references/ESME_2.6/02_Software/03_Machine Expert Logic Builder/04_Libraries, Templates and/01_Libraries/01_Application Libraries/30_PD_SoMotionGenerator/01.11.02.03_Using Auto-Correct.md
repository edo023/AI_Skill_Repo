# Using Auto-Correct

![](images/logo.png)

Using Auto-Correct

Description

Normally a polynomial is specified by its initial and final values of the position, velocity, acceleration and jerk. The polynomials cams have an option “xAutoConnect”. With this flag set to TRUE not the values specified in the structure ST\_Poly7Cam  are used but the current master and slave conditions in the active channel.

Thus the used values are

lrXstart  <-- lrLogicalMasterPosition

lrYstart <-- f(lrRefPosition\_A)

lrMstart  <-- g(lrRefVelocity\_A, lrLogicalMasterVelocity\_A)

lrKstart  <-- h(lrRefAcceleration\_A, lrLogicalMasterAcceleration\_A)

lrRstart  <-- 0

with f, g, h some functions to calculate the next pos - , vel - and acc - values out of the last known master and slave values. The same is true for the  ST\_Poly5Cam.
