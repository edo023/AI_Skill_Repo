# UABrowseResultMask

## Overview

|  |  |
| --- | --- |
| Type: | Alias |
| Available as of: | V2.0.0.0 |

## Description

The alias UABrowseResultMask specifies which fields of the UAReferenceDescription are requested from the server.

UABrowseResultMask is a bitmask and every bit of the byte represents a field.

## Supported Bitmasks

| Bitmask | Description |
| --- | --- |
| 16#1 | ReferenceType |
| 16#2 | IsForward |
| 16#4 | NodeClass |
| 16#8 | BrowseName |
| 16#10 | DisplayName |
| 16#20 | TypeDefinition |

## Used By

* UABrowseDescription
