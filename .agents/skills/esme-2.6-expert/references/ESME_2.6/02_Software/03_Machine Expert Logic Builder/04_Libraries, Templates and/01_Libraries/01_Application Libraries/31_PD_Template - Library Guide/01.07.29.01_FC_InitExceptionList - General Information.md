# FC\_InitExceptionList - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.4.0 |

## Task

Initialization of the ST\_ExceptionList data structure.

## Description

All of the functions provided in the PD\_Template library that access the ST\_ExceptionList data structure use this function. Each user that accesses the ST\_ExceptionList data structure directly (by avoiding the functions provided by the template) must ensure that the xInit flag in the structure is TRUE. If not, this function must be executed.
