## Task

Quits diagnostic messages of type equal to [ET\_Category.Information](ET_Category-GeneralInformation-36C5F24D.html).

## Description

The method DiagQuitInformation  is used to quit diagnostic messages of type equal to [ET\_Category.Information](ET_Category-GeneralInformation-36C5F24D.html). The command is forwarded to all child diagnostic handlers. The optional inputs provide the possibility to filter the information to quit.

NOTE: In case the property stDiagMessage indicates a subsequent occurred diagnostic message which cannot be deleted with the call of the method DiagQuitInformation, also the information in the property stDiagMessageFirst is kept.

## Interface

| Input | Data type | Description |
| --- | --- | --- |

| i\_diSourceId | DINT | Optional input to specify the source ID of the diagnostic messages to quit. |
| i\_ifRootSource | IF\_DiagHandlerIdentification | Optional input to specify the root source of the diagnostic messages to quit. |
