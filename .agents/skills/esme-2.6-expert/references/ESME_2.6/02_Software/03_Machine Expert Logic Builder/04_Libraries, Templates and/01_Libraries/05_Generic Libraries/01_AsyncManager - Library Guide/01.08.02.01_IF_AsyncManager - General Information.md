## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

## Task

Transfer and execute user-specific code to a task running asynchronously to the program.

## Description

The interface IF\_AsyncManager is implemented by this library and is used for the following purposes:

* Initialize the AsyncManager
* Assign the program part which has to be executed asynchronously to the calling task

NOTE: It is not required to implement the interface IF\_AsyncManager in your application.

## Methods

| Name | Description |
| --- | --- |
| Init | Initialization of the asynchronous mechanism and generation of the asynchronous tasks |
| Start | Transfer job to asynchronous task |
