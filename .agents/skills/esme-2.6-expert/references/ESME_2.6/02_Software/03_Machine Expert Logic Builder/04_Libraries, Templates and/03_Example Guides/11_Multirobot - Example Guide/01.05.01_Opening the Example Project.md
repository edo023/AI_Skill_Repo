# Opening the Example Project

The example project is installed on your PC along with the programming software. To open the project example, proceed as follows:

| Step | Action | Comment |
| --- | --- | --- |
| 1 | In the EcoStruxure Machine Expert Logic Builder, execute the command New Project. | – |
| 2 | In the New Project dialog box, select From Example from the Project type list. | – |
| 3 | Select your controller range from the Controller list. | NOTE: The controller range PacDrive LMC 300/400/402/600/802/902C must be selected. |
| 4 | Select SmartTemplate MultiRobot from the drop down menu. | – |
| 5 | Enter a name for the new project, and choose the file location. | – |
| 6 | Click on OK.  **Result**: A new project is created based on the selected example. |  |
