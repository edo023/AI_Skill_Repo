# ET\_Category - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Category contains the categories for diagnostic messages.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No diagnostic message. |
| Information | 1 | Diagnostic message of type information. |
| Advisory | 2 | Diagnostic message of type advisory. |
| Error | 3 | Diagnostic message of type error. |
