Task

Get time for total positioning.

Description

This method returns the time for the total positioning (T11+T12+T13+T2+T31+T32+T33).

Refer to figure in [FB\_RgPosStartOptimised](Function_Blocks_R_to_Z-8.htm#XREF_D_SE_0087349_1) for meaning of indices 11, 12, 13, 2, 31, 32, 33.
