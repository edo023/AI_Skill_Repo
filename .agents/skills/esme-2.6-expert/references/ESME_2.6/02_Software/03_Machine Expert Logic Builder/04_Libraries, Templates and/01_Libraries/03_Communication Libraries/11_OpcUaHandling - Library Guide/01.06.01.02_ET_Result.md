# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result specifies the possible values that indicate the result of operations executed by the POUs of this library.

NOTE: In case the returned error ID is not part of the enumeration, refer to the PLCopen specification for more details (<https://opcfoundation.org>).
