# ST\_PropertiesPublish

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_PropertiesPublish indicates the properties of the PUBLISH message sent by the MQTT client or received from the MQTT broker.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| etPayloadFormatIndicator | [ET\_PayloadFormatIndicator](ET_PayloadForInd-D886A098.html) | Indicates the format of the payload of the published message. |
| diMessageExpiryInterval | DINT | Indicates the lifetime of the message in seconds. It is sent as the publication expiry interval when the MQTT broker publishes the message. |
| uiTopicAlias | UINT | Indicates the topic alias for the topic of this message.  NOTE: Send the topic name only once in combination with a topic alias. To reduce the message size when publishing further messages for this topic, it is sufficient to specify only the previously specified topic alias and to leave the topic name empty. |
| sResponseTopic | STRING[255] | Indicates the topic name for a response message to the published message.  NOTE: The length of the value is limited to 255 characters. If it exceeds the maximum length, it is truncated and the output q\_xTruncatedResponseData is set to TRUE. |
| abyCorrelationData | ARRAY[0..GPL.Gc\_uiMax LengthOfCorrelationData-1]OF BYTE | Specifies the correlation data used to identify the response message. NOTE: If the maximum length is exceeded, the array of bytes is truncated and the output q\_xTruncatedResponseData is set to TRUE. |
| uiCorrelationDataLength | UINT | Specifies the length of the correlation data in bytes. |
| sContentType | STRING[255] | Indicates the content of the published message.  NOTE: The length of the value is limited to 255 characters. If it exceeds the maximum length, it is truncated and the output q\_xTruncatedResponseData is set to TRUE. |
| astUserProperties | ARRAY[1..GPL.Gc\_uiMaxNumberOfUserProperties] OF ST\_UserProperty | User properties of the PUBLISH message.  NOTE: A user property with an empty name represents the end of the list. |
