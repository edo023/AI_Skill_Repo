# FC\_Vector3DRotationAboutLine

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function rotates a point around a straight line in the three-dimensional space.

The point i\_stPoint is rotated in its orthogonal plane to the straight line i\_stLine by angle i\_lrAngle. The angle of rotation is mathematically positive (right-hand rule) around a vector along the straight line with orientation from point 2 to point 1 of the straight line.
