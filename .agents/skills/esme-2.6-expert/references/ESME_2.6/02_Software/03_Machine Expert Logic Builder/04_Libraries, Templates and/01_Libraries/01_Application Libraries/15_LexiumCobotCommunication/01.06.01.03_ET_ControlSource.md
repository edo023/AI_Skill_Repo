# ET\_ControlSource

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

This enumeration type is used to address a specific control source of the Lexium Cobot.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No control source of the Lexium Cobot is addressed. |
| Stick | 1 | The Stick is addressed as control source of the Lexium Cobot. The control source can be handed over to Stick by requesting the control on the control stick of the Lexium Cobot. |
| App | 2 | The App is addressed as control source of the Lexium Cobot.  The control source can be handed over to App by requesting the control on the EcoStruxure Cobot Expert. |
| Remote | 3 | Remote is addressed as control source of the Lexium Cobot. The control source can be handed over to Remote by using EcoStruxure Cobot Expert. |
