Description

The structure ST\_EtherNetIpConfiguration provides information on the EtherNet/IP  targets.

Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| uiConfiguredSlaves | UINT | Number of configured EtherNet/IP targets detected. |
| uiConfiguredConnections | UINT | Number of configured connections detected. |
| uiOperationalConnections | UINT | Number of connections in operational state. |
