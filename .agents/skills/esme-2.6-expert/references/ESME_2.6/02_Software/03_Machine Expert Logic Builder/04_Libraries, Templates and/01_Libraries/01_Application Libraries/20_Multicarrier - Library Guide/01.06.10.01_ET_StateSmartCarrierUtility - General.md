# ET\_StateSmartCarrierUtility - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.8.4.0 |

## Description

The enumeration ET\_StateSmartCarrierUtility specifies the status of the function block [FB\_SmartCarrierUtility](SmartCarrierUtility-B8A36EDB.html#SmartCarrierUtility-B8A36EDB) for reading the physical IDs of Smart Carriers and pairing the carriers.

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| NotActive | 0 | The function block FB\_SmartCarrierUtility is not enabled. |
| WaitForStart | 1 | Waiting for the start (i\_xStart) of the function block FB\_SmartCarrierUtility. |
| WaitForStartMethod | 2 | The function block FB\_SmartCarrierUtility is waiting for the call of a Start method. |
| Disabling | 8 | The function block FB\_SmartCarrierUtility is being disabled. |
| ReadCarriersPhysicalIdsActive | 50 | The reading process of the physical IDs of the carriers is active. |
| PairAllCarriersActive | 51 | The pairing process of the carriers is active. |
| ReadCarriersPhysicalIdsSuccessful | 100 | The reading process of the physical IDs of the carriers is successful. |
| PairAllCarriersSuccessful | 101 | The pairing process of the carriers is successful. |
| Error | 1000 | An error has been detected. For more information, refer to the diagnostic outputs of the function block FB\_SmartCarrierUtility. |
