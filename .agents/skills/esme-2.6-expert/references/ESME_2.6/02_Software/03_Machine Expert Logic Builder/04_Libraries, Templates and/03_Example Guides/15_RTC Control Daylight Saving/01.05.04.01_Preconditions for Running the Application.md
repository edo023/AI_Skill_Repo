# Preconditions for Running the Application Example

## Overview

To run the application example, the following preconditions must be fulfilled:

* The application is downloaded and is running on the associated controller
* An NTP or SNTP time server is available to the controller via the Ethernet network. This can be also your PC, if it is configured accordingly.
