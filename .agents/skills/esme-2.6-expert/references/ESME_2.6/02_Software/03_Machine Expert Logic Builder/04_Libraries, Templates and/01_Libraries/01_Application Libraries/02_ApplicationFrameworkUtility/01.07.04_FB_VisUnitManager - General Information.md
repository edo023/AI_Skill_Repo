# FB\_VisUnitManager - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.1.0.0 |

## Task

The instances of the function block FB\_VisUnitManager register to the higher-level visualization manager function block.

## Description

The function blocks FB\_VisUnitManager and FB\_VisApplicationManager are used to navigate through a framework structure in a visualization and to provide the data from the units. The instances of the FB\_VisUnitManager register to the higher-level visualization manager function block and thus create the framework tree for the visualization.

For more information on the use of the function block FB\_VisUnitManager, refer to [Visualization Managers](VisualizationManagers-E0420982.html).
