# ST\_PropertiesConnect

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_PropertiesConnect specifies the properties sent by the MQTT client inside the CONNECT message.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| udiSessionExpiryInterval | UDINT | Specifies the session expiry interval requested by the MQTT client. |
| uiReceiveMaximum | UINT | Specifies the number of QoS 1 publications that the MQTT client is willing to process concurrently.  Default value: 1 |
| udiMaximumPacketSize | UDINT | Specifies the maximum packet size that the MQTT client can accept.  Default value: Gc\_uiMaxNumber OfBytesToExchange |
| uiTopicAliasMaximum | UINT | Specifies the maximum value that the MQTT client accepts as a topic alias sent by the MQTT server. |
| xRequestResponseInformation | BOOL | When TRUE, the MQTT client requests the MQTT server to return response information in the CONNACK message. |
| xRequestProblemInformation | BOOL | When TRUE, the MQTT client requests the MQTT server to return a reason string or user properties on packets where it is allowed. |
| astUserProperties | ARRAY[1..GPL.Gc\_uiMaxNumberOfUserProperties] OF ST\_UserProperty | Specifies the user properties to send to the MQTT broker.  NOTE: A user property with an empty name represents the end of the list. |
| sAuthenticationMethod | STRING[255] | Specifies the authentication method used and how to interpret the authentication data.  NOTE: If enhanced authentication is requested, a callback must be registered to the MQTT client using the method RegisterCallbackEnhancedAuthentication. |
| uiAuthenticationDataLength | UINT | Specifies the length of the authentication data. |
| abyAuthenticationData | ARRAY[0...254] OF BYTE | Specifies the authentication data. |
| stWillMessageProperties | [ST\_PropertiesWillMessage](ST_PropertiesWillM-D8DCC1AB.html) | Specifies the properties associated with the Will message. |
