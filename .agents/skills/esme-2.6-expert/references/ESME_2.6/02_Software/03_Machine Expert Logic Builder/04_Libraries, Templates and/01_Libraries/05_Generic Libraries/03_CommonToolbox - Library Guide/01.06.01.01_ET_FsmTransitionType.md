# ET\_FsmTransitionType

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.2.9.0 |

## Description

The enumeration ET\_FsmTransitionType specifies the transitions types.

## Enumeration Elements

| Name | Data type | Value | Description |
| --- | --- | --- | --- |
| NotSet | UDINT | 0 | No transition type set. |
| Perform\_Initial | UDINT | 1 | Indicates that the initial state transition was performed. |
| Perform\_StateChange | UDINT | 2 | Indicates that a state change was performed. |
| Perform\_SelfTransition | UDINT | 3 | Indicates that a self-transition was performed. |
| Request\_StateChange | UDINT | 11 | Indicates a state change request. |
| Request\_SelfTransition | UDINT | 12 | Indicates a self-transition request. |
