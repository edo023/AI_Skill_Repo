## Description

With the method SetFeedbackState(…) the state of a robot feedback can be defined.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etName | [ET\_RobotFeedback](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=ET_RobFeedb_C7947528) | Selects the robot feedback option to set the state on. |
| i\_etValue | [ET\_FeedbackState](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=ET_RobotFeedbackState_GeneralInform_C7957C98) | Selects the state for the robot feedback option. |

## Operating Modes

| Operating Mode | Description |
| --- | --- |
| Disable Robot Feedback | Robot feedback can be disabled by calling the method SetFeedbackState(…).  At the input i\_etName the robot feedback to be disabled must be selected.  At the input i\_etValue the value ET\_FeedbackState.Disabled must be provided.  In case the output q\_etDiag reports ET\_Diag.Ok the robot feedback has been successfully disabled. |
| Enable Robot Feedback | Robot feedback can be enabled by calling the method SetFeedbackState(…).  At the input i\_etName the robot feedback to be enabled must be selected.  At the input i\_etValue the value ET\_FeedbackState.Enabled must be provided.  In case the output q\_etDiag reports ET\_Diag.Ok the robot feedback has been successfully enabled. |
