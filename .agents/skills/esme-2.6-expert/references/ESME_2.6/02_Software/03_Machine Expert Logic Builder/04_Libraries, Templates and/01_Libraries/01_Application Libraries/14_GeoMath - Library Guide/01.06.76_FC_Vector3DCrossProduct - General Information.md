## Description

Given two 3D input vectors, the function returns their cross product that results in a new 3D vector being normal to the plane containing the two input vectors.

| Input | Data type | Description |
| --- | --- | --- |

| i\_stVectorB | SE\_MATH.ST\_Vector3D | Second 3D vector. |
