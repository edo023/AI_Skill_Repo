# IF\_Trigger2Access

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The access interface IF\_Trigger2Access allows you to access the user function interface [IF\_Trigger](IF_Trigger-4F901ECA.html#IF_Trigger-4F901ECA) for position capture with a touchprobe.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| triggerCap2 | [IF\_Trigger](IF_Trigger-4F901ECA.html#IF_Trigger-4F901ECA) | Get | Accesses the user function interface IF\_Trigger. |
