## Task

Method to process and/or modify a diagnostic message that is forwarded to the parent of the diagnostic handler.

## Description

The interface method ConfigDiagMsgForwarding is called by the instance of the FB\_DiagnosticHandler in the event a diagnostic message is forwarded to the parent.

All information about the incoming diagnostic message is provided with the method parameters and can be processed within the method. In addition, it is possible to modify the forwarded severity.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stIncomingDiagMsg | [ST\_DiagMessageHandler](ST_DiagMessageHandler-37410F11.html) | Indicates the incoming diagnostic message. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_uiForwardedSeverity | UINT | Indicates the severity of the diagnostic message that is forwarded to the parent. It indicates the proposed value according to the mapping table. Modify this value to meet your requirements. |
