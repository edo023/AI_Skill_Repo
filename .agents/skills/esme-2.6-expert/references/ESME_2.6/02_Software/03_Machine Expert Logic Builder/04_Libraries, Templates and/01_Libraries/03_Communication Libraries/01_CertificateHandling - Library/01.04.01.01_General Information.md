# General Information

## Library Overview

The CertificateHandling library implements function blocks which provide access to the certificate store of the controller. It allows you to create self-signed certificates and CSR (Certificate Signing Requests) and to install certificates from within your application. The certificates can be used for secured TCP communication, for example, implemented by the use of the TCP client and server function blocks of the TcpUdpCommunication library.

The CertificateHandling library is supported by the following controller platforms:

* Modicon M262 Logic/Motion Controllers
* PacDrive LMC controllers

## General Considerations

NOTE: Schneider Electric adheres to industry best practices in the development and implementation of control systems. This includes a "Defense-in-Depth" approach to secure an Industrial Control System. This approach places the controllers behind one or more firewalls to restrict access to authorized personnel and protocols only.
