## Description

The method EventCallback of the function block [FB\_MultiAxisCustomJob](FB_MultiAxisCustomJob-4CFA57F9.html) is used by the system to process events. It allows the function block to respond to specific runtime events or task triggers that are necessary for the synchronized execution of the multi-axis custom job.

Under normal operating conditions, this method is called automatically by the controller runtime and does not need to be called explicitly by the user application.
