# ST\_LogEntry

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | — |

## Description

Structure for the information of one log entry of the Application Logger.

NOTE: The timestamp of a created message is in UTC. Use the function [FC\_ConvertLogEntryTimeStampUtcToLocal](FC_ConvertLogEntryTimeStampUtcToLoc-37E8D9CC.html#FC_ConvertLogEntryTimeStampUtcToLoc-37E8D9CC) to convert the timestamp from UTC to the local time zone of the controller.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| udiId | UDINT | Identification number of the logger message. |
| udiLoggerPointId | UDINT | Identification of the logger point that sent the message. |
| etLoglevel | [ET\_LogLevel](ET_LogLevel-37599385.html#ET_LogLevel-37599385) | Logger level for which the message has been sent. |
| udiDiagCode | UDINT | ET\_Result number that was sent for this logger message. |
| dtDate | DATE | Date when the message was created. |
| todTime | TIME\_OF\_DAY | Time when the message was created. |
| sMessage | STRING[255] | The logger message. |
