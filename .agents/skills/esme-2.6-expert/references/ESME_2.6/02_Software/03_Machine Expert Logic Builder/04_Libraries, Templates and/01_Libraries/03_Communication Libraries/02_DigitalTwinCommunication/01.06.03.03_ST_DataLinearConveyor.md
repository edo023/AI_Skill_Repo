## Description

The structure ST\_DataLinearConveyor contains the configuration data and the dynamic data to configure and control a conveyor in EcoStruxure Machine Expert Twin.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stConfigData | [ST\_ConfigDataLinearConveyor](ST_ConfigDataLinearConv-22380707.html#ST_ConfigDataLinearConv-22380707) | The structure that contains the configuration data for a linear conveyor. |
| lrPosition | LREAL | The position value of the conveyor, used when the control mode PositioningAbsolute or PositioningModuloRollover is selected in [ET\_ConveyorControlMode](ET_ConvCtrlMode-212A6260.html#ET_ConvCtrlMode-212A6260). |
| lrVelocity | LREAL | The velocity value of the conveyor, used when Velocity is selected in [ET\_ConveyorControlMode](ET_ConvCtrlMode-212A6260.html#ET_ConvCtrlMode-212A6260). |
