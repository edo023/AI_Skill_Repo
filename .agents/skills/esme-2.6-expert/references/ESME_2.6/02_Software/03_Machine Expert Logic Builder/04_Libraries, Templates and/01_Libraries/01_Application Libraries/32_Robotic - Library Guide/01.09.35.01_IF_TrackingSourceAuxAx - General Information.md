## Task

The interface is used to exchange tracking data for auxiliary axes between the tracking source and the FB\_Robot function block.

## Description

In case the robot is tracking one or more auxiliary axes, the tracking source must implement the interface IF\_TrackingSourceAuxAx.

The method UpdateTrackingDataAuxAx must be implemented according to the needs of the tracking.

The interface cannot be used as standalone, it is an addition to IF\_TrackingSource. For further information, refer to [Implement a User Specific Tracking Source](ImplementAUserSpecificTrackingSourc-CEB3C47B.html).

## Methods

| Name | Description |
| --- | --- |
| UpdateTrackingDataAuxAx | Method is called by the FB\_Robot every Sercos cycle. It must be implemented to update the positions and velocities of the auxiliary axis tracking components on every call. |
