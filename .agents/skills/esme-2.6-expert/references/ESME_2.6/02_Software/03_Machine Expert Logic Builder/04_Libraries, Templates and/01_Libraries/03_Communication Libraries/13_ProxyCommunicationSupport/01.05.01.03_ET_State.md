# ET\_State

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.2.0 |

## Description

The enumeration ET\_State values indicate the state of operations executed by the function blocks of this library.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Idle | 0 | The function block is ready for executing a connection method. |
| ConnectingToProxy | 1 | The connection to the proxy server is being established.  NOTE: Continue calling the ConnectToProxy method until it indicates TRUE at q\_xDone or q\_xError. |
| ConnectingToRemoteServer | 2 | The connection to the remote server is being established.  NOTE: Continue calling the ConnectToRemoteServer method until it indicates TRUE at q\_xDone or q\_xError. |
| UpgradingSocket | 3 | The connection to the remote server has been established and is now being upgraded to TLS.  NOTE: Continue calling the ConnectToRemoteServer method until it indicates TRUE at q\_xDone or q\_xError. |

## Used By

* FB\_ProxyHandlerBase
* FB\_ProxyHandlerHttpConnect
