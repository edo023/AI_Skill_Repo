# IF_TargetManager - AddTargetPos (Method)

![](images/logo.png)

IF\_TargetManager - AddTargetPos (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Create a target at a position on the target belt.

Description

Adds a new target to the list of targets that is located at the i\_lrPosition position of the target belt.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrPosition | LREAL | Position of the target belt at which the target is to be entered |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) | General, library-independent statement on the diagnosis.  A value unequal GD.ET\_Diag.Ok equates an error message. |
| q\_etDiagExt | [ET\_DiagExt](../Enumerations/Enumerations-4.htm#XREF_D_SE_0080267_1) | POU-specific output on the diagnosis.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> error message |

| Description: | The maximum number of manageable targets has been reached. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | There was an attempt to add more than the maximum number of targets to the infeed distance. | At no time more than [Gc\_udiMaxNumberOfTargets](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0080374_1) targets may be defined for the infeed distance.  The global parameter [Gc\_udiMaxNumberOfTargets](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0080374_1) must be set to a value that is greater than the maximum number of targets for the infeed distance. |
