# ReleaseCertificate Method

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.1.2.0 |

![](images/0000807856.png)

## Functional Description

This method is used to release the selected certificate and the bound resources.

## Interface

| Return value | Data type | Description |
| --- | --- | --- |
| ReleaseCertificate | BOOL | If this output is set to TRUE, the certificate and the resources have been released. |
