# IF\_ApplicationLogger2

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_ApplicationLogger](IF_ApplicationLogger-37D0982A.html#IF_ApplicationLogger-37D0982A) |

## Description

This interface extends the interface [IF\_ApplicationLogger](IF_ApplicationLogger-37D0982A.html#IF_ApplicationLogger-37D0982A) and provides the methods and properties of the function block FB\_ApplicationLoggerHandler.

You can control the Application Logger with those methods and properties.

## Methods

| Name | Description |
| --- | --- |
| [DeleteAllLogEntries](IF_ApplicationLogger2-DeleteAllLogE-37D86476.html#IF_ApplicationLogger2-DeleteAllLogE-37D86476) | Deletes the log entries from the Application Logger. |
| [DisableLogger](IF_ApplicationLogger2-DisableLogger-37D9EA9A.html#IF_ApplicationLogger2-DisableLogger-37D9EA9A) | Disables the logging of messages. |
| [EnableLogger](IF_ApplicationLogger2-EnableLoggerM-37D9F7D3.html#IF_ApplicationLogger2-EnableLoggerM-37D9F7D3) | Re-enables the logging of messages. |
| [GetLogEntry](IF_ApplicationLogger2-GetLogEntryMe-37DA003D.html#IF_ApplicationLogger2-GetLogEntryMe-37DA003D) | Returns a stored logger message. |
| [GetLoggerPoint](IF_ApplicationLogger2-GetLoggerPoin-37DA07C8.html#IF_ApplicationLogger2-GetLoggerPoin-37DA07C8) | Returns a registered logger point. |
| [SetLogLevel](IF_ApplicationLogger2-SetLogLevelMe-386951AE.html#IF_ApplicationLogger2-SetLogLevelMe-386951AE) | Sets the log level of a logger point. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| udiFirstLogEntryId | UDINT | Read | The identification number of the oldest logger message that is stored in the buffer of the Application Logger. |
| udiLastLogEntryId | UDINT | Read | The identification number of the newest logger message that is stored in the buffer of the Application Logger. |
| udiNumberOfLogEntries | UDINT | Read | The number of logger messages that are stored in the buffer of the Application Logger. |
| udiNumberOfLoggerPoints | UDINT | Read | The number of logger points that are registered in the Application Logger. |
