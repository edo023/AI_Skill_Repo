# FC_SetMatrixElement

| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Describe the element of a matrix.

Description

This describes the matrix element that is located in line i\_diRow and column i\_diCol of a matrix with the value i\_lrValue. The pointer i\_plrMatrixStart must point to the start of the memory area where the matrix is stored, i\_diNumCols must contain the number of columns in the matrix. i\_diRow and i\_diCol both start at 0.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_plrMatrixStart | POINTEr TO LREAL | Pointer to the start address of the matrix to be processed |
| i\_diNumCols | DINT | Number of columns contained in the matrix |
| i\_diRow | DINT | Line containing the matrix element to be written |
| i\_diCol | DINT | Column containing the matrix element to be written |
| i\_lrValue | LREAL | The value which is to be written into the addressed matrix element. |
