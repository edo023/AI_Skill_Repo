# ET\_Result - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.1.0.0 |

## Description

The enumeration ET\_Result contains the possible values that indicate the result of operations executed by the POUs of this library.

## Enumeration Elements

| Name | Value (INT) | Description |
| --- | --- | --- |
| Ok | 0 | The object assigned on the input i\_ifParent is valid. |
| InterfaceInvalid | 1 | The object assigned on the input i\_ifParent is invalid. |
