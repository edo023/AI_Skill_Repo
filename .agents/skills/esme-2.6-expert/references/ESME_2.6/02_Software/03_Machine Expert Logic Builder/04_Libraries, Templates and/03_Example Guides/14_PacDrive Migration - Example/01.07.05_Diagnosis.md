# Diagnosis

## Replace MainSystemFaults

| Step | Action |
| --- | --- |
| 1 | Copy and paste the action MainSystemFaults from the migration project template. |
| 2 | Add the interfaces ifController, ifControllerDiagnosticAccess and ifControllerDiagnosticin to your project. They can be copied from the migration project. |

## Modify Diagnosis Functions and Variables

| Step | Action |
| --- | --- |
| 1 | In SR\_MainMachine.InitModuleAction, comment out the function FC\_DiagNoQuitSet, as it is not available in the M660 libraries. |
| 2 | Comment out the variables stDiagSource and ifDiagnosis in variable declaration of SR\_MainMachine. |

## Modify the Task Configuration

The task Task\_SR\_Diagnostic is not supported, so it must be deleted in the Task Configuration.
