## Task

Write encoder of the rotational axis.

## Description

By calling up the method the encoder of the rotational axis is written to the value of i\_lrValue.

For additional information, refer to the chapter [Calibration](D-SE-0075364.html#D-SE-0075364).

## Interface

| Input | Data type | Description |
