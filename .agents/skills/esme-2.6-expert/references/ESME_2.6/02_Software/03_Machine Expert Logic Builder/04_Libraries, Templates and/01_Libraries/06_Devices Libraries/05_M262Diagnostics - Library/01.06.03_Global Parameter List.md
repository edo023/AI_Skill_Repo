Description

The global parameter list (GPL) contains global constants which are used by certain components of this library. The parameters can be edited individually for each application where the library is used. The modification must be done within the Library Manager of the project where the library is referenced. Values of variables of type CONSTANT cannot be modified during runtime. They must be set to the desired value before downloading the application to the controller.

Global Parameters

| Variable | Data type | Default value | Range | Description |
| --- | --- | --- | --- | --- |
| Gc\_uiMaxNumOfDiagEntries | UINT | 200 | 50... 500 | The value indicates the number of messages which can be stored in an array of type DiagTable. |
| Gc\_uiNumOfDiagEntriesVisuDiagTable | UINT | 15 | 15 | The value indicates the number of messages which can be displayed in the visualization DiagTable.  NOTE: The value for this variable cannot be modified. |
