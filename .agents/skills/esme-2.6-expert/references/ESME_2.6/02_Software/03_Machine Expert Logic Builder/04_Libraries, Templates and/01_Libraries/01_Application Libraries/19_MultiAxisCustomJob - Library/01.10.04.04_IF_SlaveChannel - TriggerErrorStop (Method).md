## Description

The method TriggerErrorStop of the interface [IF\_SlaveChannel](IF_SlaveChannel-4D6988AD.html) is used to trigger an error stop of the subordinate axis connected to the channel. If this method is called, the affected axis transitions to the operating state ErrorStop.

The multi-axis custom job continues to execute and continues to call the Calculate method of the user-defined algorithm for the remaining axes in the group. After an axis has left the group via this method, it cannot be re-integrated during the current execution of the multi-axis custom job.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
