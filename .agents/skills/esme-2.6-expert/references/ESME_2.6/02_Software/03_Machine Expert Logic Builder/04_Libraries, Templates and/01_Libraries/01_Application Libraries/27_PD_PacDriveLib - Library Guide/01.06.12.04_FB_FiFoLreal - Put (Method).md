Task

Entering a value into the FIFO

Description

The value from i\_lrValue is entered into the FIFO and the number of entered values is incremented by one.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrValue | LREAL | The value to be entered |
