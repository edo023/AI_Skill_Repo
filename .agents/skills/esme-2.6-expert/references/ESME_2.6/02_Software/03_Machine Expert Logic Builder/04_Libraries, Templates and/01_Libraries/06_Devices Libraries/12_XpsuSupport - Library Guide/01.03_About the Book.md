# About the Book

## Document Scope

The present document describes the function blocks FB\_XpsuDiag and FB\_XpsuMain provided by the XpsuSupport library. The library is used in conjunction with safety modules of the XPSU range. It provides diagnostics functionality and performs calculations concerning the service life and due dates for specific types of maintenance of XPSU safety modules as well as connected equipment.
