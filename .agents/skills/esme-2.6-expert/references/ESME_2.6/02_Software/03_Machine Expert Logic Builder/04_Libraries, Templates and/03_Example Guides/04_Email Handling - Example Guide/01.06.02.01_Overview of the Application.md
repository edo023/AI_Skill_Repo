# Overview of the Application

## Overview

The controller application implements the email client for sending and receiving emails. By using the visualization, the email client can be controlled and monitored with the Logic Builder. The following features are supported:

Manual control:

* Create and send an email
* Trigger two different events for the automatic creation and sending of an email
* Verify whether new emails are available
* Download new emails including their attachments
* Delete emails

Auto processing of email communication:

* Cyclic verification for new emails
* Downloading and processing new emails one by one
* Deletion of emails from undefined senders or non-supported content
* Creating and sending a response depending on the content of the email
