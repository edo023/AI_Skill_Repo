Task

Determining the sign value of a LREAL value

Description

The sign value is defined as follows:

oi\_lrValue > 0.0 -> +1

oi\_lrValue < 0.0 -> -1

oi\_lrValue = 0.0 -> 0

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrValue | LREAL | Value of the number of which the sign value is to be determined |
