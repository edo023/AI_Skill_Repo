# ET_AccelerationCharacteristics

![](images/logo.png)

ET\_AccelerationCharacteristics

ET\_AccelerationCharacteristics - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.5.8.0 |
| Versions: | Current version |

Description

Enumeration type for value ranges of the jerk value during the acceleration phases of a positioning movement.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Undefined | 0 | Undefine profile type |
| PZN | 1 | Acceleration profile has 3 segments: positive, zero, negative jerk |
| PN | 2 | Acceleration profile has 2 segments: positive, negative jerk |
| NZP | 3 | Acceleration profile has 3 segments: negative, zero, positive jerk |
| NP | 4 | Acceleration profile has 2 segments: negative, positive jerk |
| PZP | 5 | Acceleration profile has 3 segments: positive, zero, positive jerk |
| NZN | 6 | Acceleration profile has 3 segments: negative, zero, negative jerk |
