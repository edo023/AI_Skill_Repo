Task

T FlipFlop

Description

T flip flop (toggle) with Reset input

If i\_xReset is set, output q\_xQ is forced to FALSE independently of the input i\_xD.

In the case of a rising edge of the input i\_xD, the T flip flop shows the following behavior:

q\_xQ = NOT q\_xQ
