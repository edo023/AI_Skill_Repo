# ET\_PrivacyProtocol

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.3.4.0 |

## Description

The enumeration ET\_PrivacyProtocol contains the supported methods for encrypting Protocol Data Units (PDU) inside SNMPv3 requests.

## Enumeration Elements

| Name | Data type | Value | Description |
| --- | --- | --- | --- |
| None | BYTE | 0 | No encryption. |
| DES | BYTE | 1 | Encryption with the protocol DES. |
| AES128 | BYTE | 2 | Encryption with the protocol AES128. |
