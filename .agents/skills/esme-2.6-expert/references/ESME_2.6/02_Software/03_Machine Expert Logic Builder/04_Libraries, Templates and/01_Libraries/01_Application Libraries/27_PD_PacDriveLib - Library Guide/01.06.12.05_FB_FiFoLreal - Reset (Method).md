# FB_FiFoLreal - Reset (Method)

![](images/logo.png)

FB\_FiFoLreal - Reset (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Reset the values entered in the FIFO as well as the number of values entered to zero.

Description

All values entered in the FIFO will be reset to zero. The number of values entered will be zero after having called up this method.
