# Overview of the Application Example

## Overview

The application example implements several programs and a visualization.

The programs implement the logic for the XML file handling and for the control of the display on the visualization. Using the visualization the program execution can be controlled and monitored.

## Main Features

The main features of this application example include:

* Creating an XML file with specific content
* Reading the content of an XML file
* Measuring the duration of the read and write operations
* Processing, modification, or creation of data in XML format
