Task

Flasher with differing frequencies

Description

The function implements a multiplex flashing light. A positive signal at i\_xStart enables the flashing sequence. All bits start with 0. The least significant bit alternates between 0 and 1 with the period i\_liTimeBase, the next higher bit with 2 \* i\_liTimeBase, the next higher with 4 \* i\_liTimeBase, etc.

This function block must be called up cyclically with the period i\_liTimeBase (or more frequently).
