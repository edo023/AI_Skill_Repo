# UAMonitoredVariables

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.0.0.0 |
| Inherits from: | - |
| Attribute: | ``` 'pack_mode' := '1' ``` |

## Description

The structure UAMonitoredVariables is used to provide information about monitored variables.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| Values | ARRAY [1..GPL.MAX\_ELEMENTS\_QUEUELIST] OF [ST\_Variable](D-SE-0099975.html) | Array containing information about the variables and the corresponding memory areas. |
| TimeStamps | ARRAY [1..GPL.MAX\_ELEMENTS\_QUEUELIST] OF DT | Source time stamp returned by the server. |
| NodeQualityIDs | ARRAY [1..GPL.MAX\_ELEMENTS\_QUEUELIST] OF DWORD | Contains an error code for each valid element of the Variable array. |
| NewValuesCount | UINT | Counts the values which where updated starting from the lowest element of the values. |
