# ET\_JogMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

This enumeration type is used to address a specific jog mode of the Lexium Cobot.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Continuous | 1 | The jog mode Continuous of the Lexium Cobot is addressed. |
| Relative | 2 | The jog mode Relative of the Lexium Cobot is addressed. |
| Absolute | 3 | The jog mode Absolute of the Lexium Cobot is addressed. |
