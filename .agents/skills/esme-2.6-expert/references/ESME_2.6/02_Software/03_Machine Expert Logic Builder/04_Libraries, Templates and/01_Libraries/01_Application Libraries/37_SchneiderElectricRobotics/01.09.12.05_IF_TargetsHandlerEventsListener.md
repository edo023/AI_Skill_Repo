## Task

Provides user-specific diagnostics when the owner of a slot inside a target is set.

## Description

The OnSetSlotOwnerInTarget method is called by the targets handler where the implementation of IF\_TargetsHandlerEventsListener has been subscribed every time the owner of a slot inside a target is set.

The following method triggers the event:

* SetSlotOwnerInTarget

The diagnostics provided as output depend on your specific implementation. They are ignored by the library.
