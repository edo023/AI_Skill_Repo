# FC\_GetConnectionDataAddress

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

The FC\_GetConnectionDataAddress function of the library SercosMasterM6xx gets the address and length of the connection data.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_wTopoAddr | WORD | Topology address of the Sercos device. |
| i\_etConnectionId | [SMI.ET\_ConnectionID](../../../../../api/crossBook?lang=en-US&virtualBookName=SercosMasterInterface&topicID=ET_ConnectionID_5343DAEE) | Type of Sercos telegram (AT, MDT) determining the direction of communication. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_puiDataPointer | POINTER TO UINT | Address of the connection data. |
| q\_udiDataLength | UDINT | Length of the connection data |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -461: Error detected: Incorrect Sercos communication phase. |
