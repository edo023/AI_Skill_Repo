# ST\_DataAxisTSeries – General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V2.14.0.0 |
| Inherits from: | - |

## Description

Contains a list of parameters to read out axis related data from a Lexium T Robot.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| diGearIn | DINT | GearIn for axes A and B |
| diGearOut | DINT | GearOut for axes A and B |
| lrUserDrivePeakCurrentMin | LREAL | Minimum value for UserDrivePeakCurrent for axes A and B |
| lrUserDrivePeakCurrentDefault | LREAL | Default value for UserDrivePeakCurrent for axes A and B |
| lrUserDrivePeakCurrentMax | LREAL | Maximum value for UserDrivePeakCurrent for axes A and B |
| xMotorDirectionMotorA | BOOL | Motor direction for axis A |
| xMotorDirectionMotorB | BOOL | Motor direction for axis B |
| lrFeedConstant | LREAL | FeedConstant for axes A and B |
| diMotorTemperatureMonitoring | DINT | Indicates the way the motor temperature is monitored for axes A and B. |
