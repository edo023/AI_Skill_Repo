# FB_PathForJerkPositioning - SetThreeJerkPhases (Method)

![](images/logo.png)

FB\_PathForJerkPositioning - SetThreeJerkPhases (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.5.3.0 |
| Versions: | Current version |

Task

Set a path consisting of three jerk phases.

Description

This method sets a path consisting of three jerk phases. The duration and the jerk value of each phase is specified.

The path starts the position that was set with SetInitialPositions.

This method provides no diagnostic information and no direct feedback value of the method.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrT0 | LREAL | Duration of jerk phase 1 in s |
| i\_lrJ0 | LREAL | Jerk value for jerk phase 1 in units/s3 |
| i\_lrT1 | LREAL | Duration of jerk phase 2 in s |
| i\_lrJ1 | LREAL | Jerk value for jerk phase 2 in units/s3 |
| i\_lrT2 | LREAL | Duration of jerk phase 3 in s |
| i\_lrJ3 | LREAL | Jerk value for jerk phase 3 in units/s3 |
