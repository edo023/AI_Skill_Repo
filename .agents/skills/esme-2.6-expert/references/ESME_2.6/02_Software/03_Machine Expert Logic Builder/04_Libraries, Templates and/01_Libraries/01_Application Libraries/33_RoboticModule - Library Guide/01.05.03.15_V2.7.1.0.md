## Enhancements

* **Sending module commands**

  The module command sequence ET\_Cmd.Homing -> ET\_Cmd.Start -> ET\_Cmd.Homing -> ET\_Cmd.Start is fully supported.
* **Feedback about HomeOk**

  The feedback q\_xHomeOk stays TRUE after the module is already homed and the module command ET\_Cmd.Homing is sent without sending the command ET\_Cmd.Start afterwards.
* **Diagnostic messages**

  Diagnostic messages regarding axis D and orientations as well as for software limits are improved.
* **Support for a configured reaction offset**

  A configured ST\_StandardModuleInterface.iqstExt.udiReactionOffset is fully supported.
* **Usage of configured emergency parameters to stop an active robot movement**

  Sending the module command ET\_Cmd.Stop in parallel to an incoming reaction TPL.ET\_Reaction.SyncStopEL / TPL.ET\_Reaction.SyncStopEH is improved. An active robot movement is stopped with configured emergency parameters.
