# ET\_EntityOffset

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.2.0.0 |

## Description

The enumeration ET\_EntityOffset defines the entity ID ranges for each mechatronic component. You can use the values of the ET\_EntityOffset to match these ranges. Set the unique UnivocalId variable of the mechatronic component within the range described by ET\_EntityOffset in accordance with the category of the component.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No entity ID range is defined. |
| Robot | 200 | The entity ID range (udiUnivocalId) from 200...299 is exclusive to robots. |
| Conveyor | 300 | The entity ID range (udiUnivocalId) from 300...399 is exclusive to conveyors. |
| TargetGenerator | 500 | The entity ID range (udiUnivocalId) from 500...599 is exclusive to target generators. |
| Track | 800 | The entity ID range (udiUnivocalId) from 800...899 is exclusive to tracks. |
| Carrier | 10000 | The entity ID range (udiUnivocalId) from 10000...19999 is exclusive to Lexium™ MC12 carriers. |
