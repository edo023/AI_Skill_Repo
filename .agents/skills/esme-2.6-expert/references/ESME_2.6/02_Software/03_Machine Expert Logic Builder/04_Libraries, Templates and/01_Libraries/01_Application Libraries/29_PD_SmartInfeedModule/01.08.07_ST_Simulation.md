Description

In this structure, the parameters and the return values of the simulation of the [FB\_SmartInfeed­Module](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080825_1) are summarized.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_xEnableSimulation | BOOL | TRUE: The simulation is being activated. The simulation only works if the module is in the operation mode Automatic. |
| i\_stSimParameter | SI.ST\_SimParameter | Parameters of the simulation |
| q\_stSimFeedback | SI.ST\_SimFeedBack | Return values of the simulation |
