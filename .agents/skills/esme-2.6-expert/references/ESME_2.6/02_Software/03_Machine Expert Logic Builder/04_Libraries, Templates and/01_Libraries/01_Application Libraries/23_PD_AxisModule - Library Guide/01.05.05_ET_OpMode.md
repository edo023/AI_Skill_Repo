# ET_OpMode

![](images/logo.png)

ET\_OpMode

ET\_OpMode - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.2.0 |
| Versions: | Current version |

Description

The enumeration describes all operating modes of the [FB\_AxisModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0077136_1).

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| NoOp | 0 | No operating mode selected |
| Homing | 10 | Homing of the axis |
| Manual | 20 | Touch control mode |
| MultiCam | 30 | Run master slave cams |
| Endless | 40 | Endless feed with (constant) velocity |
| EndlessIls | 41 | Endless feed as master the slave axes of which run cams with limit monitoring. (Intelligent lineshaft) |
| Positioning | 50 | Positioning mode |
| BrakeRelease | 70 | Release the brake of the axis |
| Additional | 80 | Activate an external POU |
