About the Book

Document Scope

The function blocks described here are used to control LXM28A drives on CANopen fieldbuses under the SoMachine software environment. The function blocks are compliant with the IEC 61131-3 standard.

NOTE: Read and understand this document and all related documents before installing, operating, or maintaining your system.

Validity Note

This document has been updated for the release of the Lexium 28 AddOn V4.1.0.5.

Remote Devices

Remote control operating devices may lead to unintended equipment operation by:

oincorrect operation

oinsufficient view on the machine during operation

ounintentional manipulation

The manufacturer or the operating company of the machine must take precautions to avoid unintentional equipment operation that may be caused by remote control.

Care must be taken and provisions made for use of this product as a control device to avoid inadvertent consequences of commanded machine operation, state changes, or alteration of data memory or machine operating parameters.

|  |
| --- |
| Warning_Color.gifWARNING |
