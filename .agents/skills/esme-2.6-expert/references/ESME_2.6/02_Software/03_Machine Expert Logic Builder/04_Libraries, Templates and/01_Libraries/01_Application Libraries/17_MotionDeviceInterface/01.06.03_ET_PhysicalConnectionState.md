# ET\_PhysicalConnectionState

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | INT |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_PhysicalConnectionState of the library MotionDeviceInterface contains the connection states of a physical Sercos device. The values identify whether or not the firmware was able match a configured device to a device connected to the Sercos bus.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| ConnectionToPhysicalDeviceExistent | 0 | A matching physical device for the configured device was found during phase-up. |
| ConnectionToPhysicalDeviceNotExistent | 1 | No matching physical device for the configured device was found during phase-up. |
