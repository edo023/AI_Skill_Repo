# FR\_DynStateModel\_2022

## Overview

|  |  |
| --- | --- |
| Type: | Visualization frame |
| Available as of: | V1.4.2.0 |

## Task

Dynamically display the visualization of the state model of the present unit mode.

## Functional Description

The FR\_DynStateModel\_2022 dynamically displays the visualization of the state model during online operation. The state model is compliant to the PackML base state model as defined in ANSI/ISA TR88.00.02-2022.

## Interface

| Input / output | Data type | Description |
| --- | --- | --- |
| i\_stVisInterface | ST\_VisInterface | Interface to the FB\_VisController |

## Example

The displayed state model is dynamically adapted based on the state model defined.
