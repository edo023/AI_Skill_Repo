# ET_CamType

![](images/logo.png)

ET\_CamType

ET\_CamType - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Description

Enumeration type for mathematically describing the shape of a motion.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Straight | 0 | Straight |
| Poly5Com | 1 | General polynomial of the fifth degree |
| SimplSin | 2 | Simple sinus |
| IncliSin(1) | 3 | Inclined sine |
| ModiSin(1) | 4 | Modified sinus |
| ModAccTr(1) | 5 | Modified acceleration trapezoid |
| Poly5 | 6 | Standard polynomial of the fifth degree |
| Parable2(1) | 7 | Quadratic parabola |
| UserCam(1) | 8 | User curve |
| ModiSinCom(1) | 9 | General modified sinusoidal line |
| ModAccTrCom(1) | 10 | General mod. acceleration trapezoid |
| HarmoComb(1) | 11 | Harmonic combination |
| SinStraightComb(1) | 12 | Sinus-straight-combination  Segment consisting of a sine line combination. If it is used as part of [ST\_MultiCam](../Structures/Structures-3.htm#XREF_D_SE_0096123_1), it is defined by the start curvature and the end curvature (parameter lrK of both cam points) and by Y, i.e. equivalent to i\_diParSelect = 367 (refer to SystemInterface library guide, FC\_ProfileSetParam).  Use simple sine (SimplSin) if you need a sine profile which is defined by lambda, C, and Y, i.e. equivalent to i\_diParSelect = 123 (refer to SystemInterface library guide, FC\_ProfileSetParam).  Other input permutations offered by FC\_ProfileSetParam are not supported if you use [ST\_MultiCam](../Structures/Structures-3.htm#XREF_D_SE_0096123_1). |
| (1) This parameter is only supported by PacDrive controllers. | | |
