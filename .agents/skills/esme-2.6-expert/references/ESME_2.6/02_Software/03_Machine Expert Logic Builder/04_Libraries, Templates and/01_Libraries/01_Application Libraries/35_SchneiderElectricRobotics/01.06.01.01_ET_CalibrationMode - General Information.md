# ET\_CalibrationMode - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.0.0 |

## Description

Parameters to select the calibration mode.

For detailed information, refer to calibration of [Lexium P Robot](D-SE-0075364.html) or [Lexium T Robot](D-SE-0075368.html).

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No calibration mode. |
| StandardProcedure | 10 | The standard procedure consists of several calibration modes executed successively. |
| HomeOnTorque | 20 | Home on torque calibration mode. |
| MoveToCheckPosition | 30 | Moves to a check position. |
| MoveToMountPosition | 40 | Moves to a mount position. |
| BrakeRelease | 50 | Releases the brakes. |
| WriteEncoderRotationalAxis | 60 | Writes the encoder position of the rotational axis. |
| WriteAxisEncoder | 70 | Writes the encoder of an axis. |
