Task

Calculation of a unit vector that is orthogonal to a given 2-dimensional vector.

Description

This calculates a default vector which is orthogonal relative to the given vector i\_stV.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stV | [ST\_Vector2D](../Structures/Structures-47.htm#XREF_D_SE_0087800_1) | Vector in relation to which an orthogonal unit vector is to be calculated. |
