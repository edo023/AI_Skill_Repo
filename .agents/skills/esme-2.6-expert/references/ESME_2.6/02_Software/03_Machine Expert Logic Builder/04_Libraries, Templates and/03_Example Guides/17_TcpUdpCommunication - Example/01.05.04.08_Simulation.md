# Simulation

![](images/logo.png)

Simulation

Overview

The application provides a simulation mode. You can activate the simulation mode for each function separately from the respective visualization.

The table lists the processes in simulation mode for each function:

| Function | Process in simulation mode |
| --- | --- |
| [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) [client](../glossary/glossary.htm#XREF_D_SE_0055638_5) | Cyclically initiates a data exchange with the TCP [server](../glossary/glossary.htm#XREF_D_SE_0055638_18) of the other controller. |
| TCP [server](../glossary/glossary.htm#XREF_D_SE_0055638_18) | The selected [port](../glossary/glossary.htm#XREF_D_SE_0055638_16) opens automatically and remains open. |
| [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) [peer](../glossary/glossary.htm#XREF_D_SE_0055638_15) sender | Cyclically sends data to the [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) [peer](../glossary/glossary.htm#XREF_D_SE_0055638_15) of the other controller. |
| [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) [peer](../glossary/glossary.htm#XREF_D_SE_0055638_15) receiver | The selected [port](../glossary/glossary.htm#XREF_D_SE_0055638_16) opens automatically and remains open. |

You can set the time interval for sending data in the respective visualization screen.
