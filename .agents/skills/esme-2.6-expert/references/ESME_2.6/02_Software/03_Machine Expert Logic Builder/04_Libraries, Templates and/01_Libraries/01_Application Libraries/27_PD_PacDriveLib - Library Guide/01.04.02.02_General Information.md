# General Information

General Information

Description

The library "PD\_PacDriveLib" is a collection of auxiliary functions from various areas of application. It contains technology function blocks for simple as well as more complex motion tasks.

Namespace of the Library

The default namespace of this library is PDL.

The POUs, data structures, enumerations and constants have to be addressed via this namespace.
