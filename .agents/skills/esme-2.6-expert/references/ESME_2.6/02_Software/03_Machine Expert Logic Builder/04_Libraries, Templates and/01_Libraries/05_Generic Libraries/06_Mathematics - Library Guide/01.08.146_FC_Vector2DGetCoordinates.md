# FC\_Vector2DGetCoordinates

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the coordinates of a point in a two-dimensional coordinate system that is spanned by two linearly independent vectors:

i\_stPoint = q\_lrC1 \* i\_stVectorA + q\_lrC2 \* i\_stVectorB

The coordinates of the point i\_stPoint are converted to the coordinates of the coordinate system which is spanned by the vectors i\_stVectorA and i\_stVectorB. The vectors i\_stVectorA and i\_stVectorB must be linearly independent and must not be parallel.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stPoint | [ST\_Vector2D](ST_Vector2D-GeneralInformation-0BFF6B0C.html#ST_Vector2D-GeneralInformation-0BFF6B0C) | Point for which the coordinates are to be converted. |
| i\_stVectorA | [ST\_Vector2D](ST_Vector2D-GeneralInformation-0BFF6B0C.html#ST_Vector2D-GeneralInformation-0BFF6B0C) | Vector A which spans the new coordinate system. |
| i\_stVectorB | [ST\_Vector2D](ST_Vector2D-GeneralInformation-0BFF6B0C.html#ST_Vector2D-GeneralInformation-0BFF6B0C) | Vector B which spans the new coordinate system. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
| q\_lrC1 | LREAL | Coordinate of the point i\_stPoint on the axis i\_stVectorA. |
| q\_lrC2 | LREAL | Coordinate of the point i\_stPoint on the axis i\_stVectorB. |
