# ET\_Result - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.5.1.0 |
| Versions: | Installed version |

## Description

The enumeration ET\_Result contains the possible values that indicate the result of operations executed by calling a function block, function, method or property.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Ok | 0 | The operation is completed successfully. |
| InvalidSercosState | 1 | The execution is not permitted in the present Sercos state. |
| DriveDoesNotSupportThisFunction | 2 | The assigned drive does not support the required function. |
| RequiredIdnMappingNotDone | 3 | The required Sercos parameter IDN is not configured to the cyclic data channel. |
| ConfigurationNotDone | 4 | The required Sercos IDNs are not successfully added to the cyclic data channel. |
| InvalidInterface | 5 | The given drive interface is invalid. |
| InvalidMode | 6 | The input i\_etValue of the method SetControlMode must be within a valid range. |
| SercosReadRequestIssue | 7 | The Sercos read request was not processed successfully. |
| SercosWriteRequestIssue | 8 | The Sercos write request was not processed successfully. |
| CreatingAsyncTaskFailed | 9 | The creation of at least one of the additional tasks was unsuccessful. |
| UnexpectedFeedback | 10 | An internal system function returned an indeterminable feedback. |
| InvalidWorkingMode | 11 | The working mode of the assigned drive must be physical. |
| LoggerPointNotRegistered | 12 | The logger point is not registered. |
| FeatureNotSupported | 14 | The execution of this function is not supported on this controller type. |
| DivisonByZero | 15 | One or more parameters for calculating the required reference torque is zero. |
| AsyncTaskTimedOut | 16 | The method IF\_AsyncManager.Start, timed out. |
