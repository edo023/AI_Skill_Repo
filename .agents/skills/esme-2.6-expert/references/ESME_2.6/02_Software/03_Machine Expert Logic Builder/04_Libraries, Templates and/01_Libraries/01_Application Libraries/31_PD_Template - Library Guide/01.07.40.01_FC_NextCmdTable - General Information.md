# FC\_NextCmdTable - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.4.0 |

## Task

The function determines the next table in a list of command tables.

## Description

The function helps to easily initialize a list of command tables.

The function writes the iq\_udiTableIndex value into the iq\_udiTableIndex variable and adds a 1. This is the designation for the next command table for the FB\_ModuleController.
