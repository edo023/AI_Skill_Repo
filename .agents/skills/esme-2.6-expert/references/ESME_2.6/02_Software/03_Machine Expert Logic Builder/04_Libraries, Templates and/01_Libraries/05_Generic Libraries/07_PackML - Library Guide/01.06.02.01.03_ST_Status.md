# ST\_Status

## Overview

|  |  |
| --- | --- |
| Type | Data structure |
| Available as of | V1.0.1.0 |

## Description

The structure ST\_Status represents the list of PackTags of type Status as defined in ANSI/ISA TR88.00.02-2022.

Status tags are used to describe the operation of the unit machine.
