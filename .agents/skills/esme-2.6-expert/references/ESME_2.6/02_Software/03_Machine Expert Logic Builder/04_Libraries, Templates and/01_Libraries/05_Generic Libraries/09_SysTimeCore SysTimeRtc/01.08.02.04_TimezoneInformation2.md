# TimezoneInformation2

## Data Type Description

This structure is used to provide the information about the present timezone.

| Name | Type | Description |
| --- | --- | --- |
| utcOffsetSeconds | DINT | Offset to the UTC in seconds including adjustments for daylight saving time. It is the number of seconds you need to add to the UTC to get the local time. |
| isDST | DINT | Daylight saving time information:   * positive value = Daylight saving time is enabled. * 0 = Daylight saving time is disabled. * negative value = The information is not available. |
| szTimezone | STRING[127] | Displays the local timezone in two different ways:   * The name, such as Europe/Paris. * The (POSIX (Portable Operating System Interface)) abbreviation, such as CET of the local time zone of the IANA tz database (also known as Olson tz database). |
