## Task

Configure a drive mapping for OrientationZ.

## Description

By calling up the method, the drive handed over at input i\_ifDrive is used to configure a drive mapping for OrientationZ.

The drive handed over at input i\_ifDrive is configured as ROB.ET\_RobotComponent.AxisD for the robot, and the OrientationZ (ROB.ET\_RobotComponent.OrientationZ) is then mapped to AxisD.

NOTE: Use ET\_RobotPSeriesComponent.RotationalAxis if you want to call the methods SetControlLoopParameter, GetControlLoopParameter or Calibration.

For further information, refer to the methods [ROB.IF\_RobotConfiguration.AddDrive](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=6B565F46), [ROB.IF\_RobotConfiguration.MapComponentToDrive](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=6C61B078), [IF\_RobotPSeries.SetControlLoopParameter](D-SE-0096805.html), [IF\_RobotPSeries.GetControlLoopParameter](D-SE-0096802.html), and [IF\_RobotPSeriesCalibration](D-SE-0075220.html).
