# Scaling the Velocity Override of a Robot

## Overview

This example shows how to configure and use FB\_LinearTrackingVelOverrideScaling to make an online adaptation of the velocity override of a robot.

| Step | Description |
| --- | --- |
| 1 | Configure an instance of either FB\_Robot or FB\_RoboticModule.  For example with an instance of FB\_RoboticModule named fbRoboticModule and a structure RM.ST\_ModuleInterface named stRobotInterface. |
| 2 | Limit the value of the minimum variation allowed for the velocity override parameter. This would prevent internal calculation on each small variation of the velocity override parameter. stRobotInterface.iq\_ifConfiguration.ifAdvanced.lrMinVelOverrideVariation := 0.1  For example, setting lrMinVelOverrideVariation to a value of 0.1 means that the value of the velocity override parameter will only be updated if set to a value different from the current one by an amount greater than or equal to 0.1. |
| 3 | Declare an instance of FB\_LinearTrackingVelOverrideScaling named fbVelOverrideScaling and configure it as follows:   ``` fbVelOverrideScaling.Configuration(     i_ifRobotMotion := stRobotInterface.iq_ifMotion,     i_ifRobotFeedback := stRobotInterface.iq_ifFeedback,     i_ifRobotConfigurationReadOnly := stRobotInterface.iq_ifConfigurationReadOnly,     q_xError => xError,     q_etResult => etResult,     q_sResultMsg => sResultMsg, );  IF xError THEN      //error handling here END_IF ```   After a successful configuration the property fbVelOverrideScaling.xConfigDone is set to TRUE. |
| 4 | Call the method EvaluateScaledVelOverride to scale the velocity override accordingly to the target, the tracking system state and the robot state:   ``` lrScaledVelOverride := fbVelOverrideScaling.EvaluateScaledVelOverride(     i_stTargetPosition := stTargetPosition, //position of the target in the          tracking coordinate system identified by i_etTrackingCoordinateSystem     i_etTrackingCoordinateSystem := ROB.ET_CoordinateSystem.Tracking1,     i_lrTrackingContactPosition := 200.0, //position along the tracking          direction where the robot will reach the target     i_lrMaxVelOverride := 90.0, //maximum vel override value that the robot is          allowed to use for the motion toward the target  ); ```   The variable lrScaledVelOverride is of type LREAL. |
| 5 | Now it is possible to provide lrScaledVelOverride as the new velocity override value stRobotInterface.iq\_ifMotion.lrVelOverride := lrScaledVelOverride. |
