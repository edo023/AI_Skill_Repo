# IF\_Cmd – MoveSuperImposed (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.1.0 |
| Versions: | Installed version |

## Task

Performs a superimposed movement with a specified position offset with reference to the position of an ongoing movement.

## Description

The functionality of this method corresponds to the functionality of MC\_MoveSuperimposed for PLCopen. Refer to [M262 Synchronized Motion Control Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PLCO&topicID=D_SE_0086571).

## Inputs

The inputs of this method are described in MC\_MoveSuperimposed. Refer to [M262 Synchronized Motion Control Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PLCO&topicID=D_SE_0086571).

NOTE: In comparison to the respective descriptions for PLCopen standard, as in the M262 Synchronized Motion Control Library Guide, the inputs of the function block FB\_AxisMotion include a syntax with a prefix, for example i\_ifMaster, compared to Master in the M262 Synchronized Motion Control Library Guide. However the functionalities are the same. For more details refer also to [IF\_Cmd – General Information](IF_CmdGeneralInformation-44AB82A7.html).
