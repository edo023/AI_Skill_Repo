# Operation without Tuning

## Overview

* When tuning is disabled, the FastMove functionality calculates a maximum acceleration, deceleration and velocity for each movement sent with IF\_RobotMotion.FastMoveS  or IF\_RobotMotion.FastMoveSExtended.
* If one or more of these parameters exceed the user values (for example, using the method IF\_RobotMotion.SetMotionParameter), the calculated values are limited to the user values.
* If the acceleration or deceleration is too high or too low, it can be adapted using the method IF\_FastMove.SetParameter. A change of the acceleration or deceleration factor is applied to all movements. An individual adaptation of movements is not possible.
* It is not possible to use the movements in combination with other move commands. The move commands FastMoveS and FastMoveSExtended always end at the given target and the blending zone is always zero.
