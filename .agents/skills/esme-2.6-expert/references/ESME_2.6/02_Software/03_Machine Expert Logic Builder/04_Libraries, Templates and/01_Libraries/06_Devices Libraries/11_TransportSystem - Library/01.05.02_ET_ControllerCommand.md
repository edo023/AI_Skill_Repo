# ET_ControllerCommand

![](images/logo.png)

ET\_ControllerCommand

ET\_ControllerCommand - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.4.0 |

Description

This enumeration defines the commands that can be processed by the 3rd party linear transport system and are issued directly at the function block [FB\_Connection](../Function_Blocks/Function_Blocks-2.htm#XREF_D_SE_0090686_1). These commands affect the entire transport system.

Type: UINT

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| RequestControllerStatus | 0 | Request status from the transport system controller.  In case system is operational, FB\_Connection only indicates q\_xDone = True. Otherwise error message is generated using q\_etDiag and q\_etDiagExt. |
