# V2.0.0.0

## Description

The Smart Template Camera Module provides:

* Configuration of a camera object
* Calibration of a Cognex camera
* Management of Cognex camera jobs
* A vision server to receive data from a camera
* Configuration of a target generation, for example, to simulate products
