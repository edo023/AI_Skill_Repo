# ET\_ParameterSetPID

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

The enumeration ET\_ParameterSetPID specifies the different types of the PID parameter set.

Also refer to chapters [Auto-Tuning (Heating) - FB\_HeatingControl](D-SE-0106416.html) and [Auto-Tuning (Heating) - FB\_HeatingControl2](AutoTunFB_HC2-77237AB4.html).

## Enumeration Elements

| Name | Data type | Value | Description |
| --- | --- | --- | --- |
| Default | INT | 0 | Default PID parameter set is used. |
| Custom | INT | 1 | Customized PID parameter set is used.  The default values are provided which you can customize. |
| Slow | INT | 2 | Conservative PID parameter set is used. |
| Medium | INT | 3 | Medium PID parameter set is used. |
| Offensive | INT | 4 | Aggressive PID parameter set is used. |
