# FC_CommonPointsOfParallelogramAndLine

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Calculates the intersections of a straight line and a parallelogram in the 2-dimensional space.

Description

Calculates the intersections of the straight line i\_stLine and the parallelogram i\_stParal­lelogram. If a straight line coincides with one of the parallelogram's edges, the number of common points is 99. If the straight line does not intersect with the parallelogram at all, the number of common points is 0.
