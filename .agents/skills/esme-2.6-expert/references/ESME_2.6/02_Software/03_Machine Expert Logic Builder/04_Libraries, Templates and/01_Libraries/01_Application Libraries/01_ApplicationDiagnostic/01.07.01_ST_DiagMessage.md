# ST\_DiagMessage

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

The structure ST\_DiagMessage contains information to be provided when setting a diagnostic message.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sSource | STRING | Identifies the name of the source of the diagnostic message. |
| udiSourceId | UDINT | Numerical identifier of the source of the diagnostic message. |
| diDiagCode | DINT | Diagnostic code of the diagnostic message. |
| sMessage | STRING | Text message of the diagnostic message. |
