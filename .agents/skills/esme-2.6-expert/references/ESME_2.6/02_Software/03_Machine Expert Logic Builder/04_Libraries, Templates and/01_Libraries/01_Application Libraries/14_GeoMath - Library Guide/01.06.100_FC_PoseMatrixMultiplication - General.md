## Description

Given two pose matrices, the function evaluates the result of their multiplication. This is not equivalent to a generic multiplication between two 4D matrices since this implementation is specifically for the case of matrices representing Cartesian poses.

NOTE: There is no internal verification about i\_stPoseMatrixA and i\_stPoseMatrixB being valid pose matrices.

NOTE: The multiplication of two pose matrices is not a commutative operation.
