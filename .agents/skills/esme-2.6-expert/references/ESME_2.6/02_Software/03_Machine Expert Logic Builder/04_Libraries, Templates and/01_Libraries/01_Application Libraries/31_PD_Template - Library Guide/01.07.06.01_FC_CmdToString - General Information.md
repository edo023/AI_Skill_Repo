# FC\_CmdToString - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function Block |
| Available as of: | V1.1.0.0 |
| Support for: | PacDrive pilot template architecture |

## Task

Function for processing with axis commands for axes controlled by the function block *[AXM.FB\_AxisModule](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077136)*.

## Description

The value of this function equates the name of the axis command that was selected via the i\_diCmd input. The axis commands have the following titles:

| Command Table | Title |
| --- | --- |
| AXM.ET\_Cmd.Homing | Homing |
| AXM.ET\_Cmd.Manual | Manual |
| AXM.ET\_Cmd.CamCs | Cam cold start |
| AXM.ET\_Cmd.CamWs | Cam warm start |
| AXM.ET\_Cmd.Endless | Endless |
| AXM.ET\_Cmd.EndlessIls | EndlessIls |
| AXM.ET\_Cmd.Positioning | Positioning |
| AXM.ET\_Cmd.BrakeRelease | Release holding brake |
| AXM.ET\_Cmd.AdditionalCs | Coldstart of mode: Additional |
| AXM.ET\_Cmd.AdditionalWs | Warmstart of mode: Additional |
| AXM.ET\_Cmd.Start | Start |
| AXM.ET\_Cmd.StartTrig | Starttrig |
| AXM.ET\_Cmd.StartTrigWaitInPos | Starttrig wait in pos |
| AXM.ET\_Cmd.Stop | Stop |
| AXM.ET\_Cmd.Hold | Hold |
| AXM.ET\_Cmd.DriveEnableOn | Position Control on |
| AXM.ET\_Cmd.DriveEnableOff | Position control off |
| AXM.ET\_Cmd.WsStart | WarmStart start |
