# ET\_Result - General Information

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.3.0 |

## Description

The enumeration ET\_Result is used by the POUs of this library. It contains the literal constants describing the result or the status of the execution of a POU.

The values 10, 20, 30, 40, and 50 represent status information (the value of the output bError of the function block FB\_ControlAxisByPosCnc is FALSE). The other values represent diagnostics information (the output bError of the function block FB\_ControlAxisByPosCnc is TRUE).

| Halt | 40 | Operating state Halt: The axis is stopping (falling edge at the input bEnable) |
