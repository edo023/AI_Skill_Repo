## Description

The structure ST\_RequestInformation contains two structures: one for the input data necessary to send a request and one for the output data to present the response to the user.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| i\_stRequest | [ST\_Request](D-SE-0081037.html#D-SE-0081037) | Structure containing required user input data |
| q\_stResponse | [ST\_Response](D-SE-0081038.html#D-SE-0081038) | Structure containing response to user request |
