# Homing Crank via Slide Axis

![](images/logo.png)

Homing Crank via Slide Axis

Overview

Homing is made via the slide axis for all homing modes except the following modes:

oMoveOnPosAbs

oWriteAxisEncoder

oRestorePosFromAxisEncoder

oRestorePosFromRetain

NOTE: Make sure, that the slide/linear axis position is used for calculating the parameter i\_lrHomePosition (not the crank/rotative axis position in degree).

Parameters Referring to the Linear Slide Axis
