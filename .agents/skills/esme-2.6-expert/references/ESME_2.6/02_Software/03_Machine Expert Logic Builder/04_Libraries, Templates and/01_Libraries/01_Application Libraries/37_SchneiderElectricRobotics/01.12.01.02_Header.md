# Header

## Overview

| Designation | Content | Description |
| --- | --- | --- |
| Identification string | `Vision` | Indicates that a vision system is sending data. |
| String length | `9999` | Size/length of the entire data protocol (BYTE).  (Minimum length: `23` = size of the data header (`22`) + carriage return  Maximum length: `9999`)  Range: `0023...9999` |
| Protocol index | `9999999999` | Each data protocol contains its index, which has to be incremented by the vision system. That allows to identify cases of unsuccessful communications. |
| Vision Id | `99` | Each vision system is identified by its vision Id, which must be set in the vision program/job.  Range: `01...99` |
