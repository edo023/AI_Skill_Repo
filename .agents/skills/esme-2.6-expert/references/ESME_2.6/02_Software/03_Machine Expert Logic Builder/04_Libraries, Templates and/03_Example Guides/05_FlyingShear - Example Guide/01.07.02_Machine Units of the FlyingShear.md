# Machine Units of the FlyingShear

The FlyingShear machine consists of two separate machine units.

Each machine unit is represented in the FlyingShear example application and controlled by a separate software unit. For further information, refer to [Project Structure](Project-E77015E5.html).

| Machine unit => software unit | Description |
| --- | --- |
| ProductBelt | Transport belt that transports a continuous product that needs to be cut. |
| FlyingShear | Moves the shears above the belt, synchronizes the shears with the product speed, and cuts the product at defined positions. |
