# TimezoneInformation

## Data Type Description

This information describes a local timezone with standard time and daylight saving time (also known as summer and winter time).

| Name | Type | Description | Unit |
| --- | --- | --- | --- |
| ulStandardDate(1) | UDINT | Date to switch to standard time (from summer to winter time). | Local time in seconds since January 1st, 1970 00:00:00. |
| ulDaylightDate(1) | UDINT | Date to switch to daylight saving time (from winter to summer time). |
| szStandardName | STRING[32] | Standard name of the timezone (winter time). | String with maximum 32 characters |
| szDaylightName | STRING[32] | Daylight saving time name (summer time). | String with maximum 32 characters |
| iBias | INT | Offset of the UTC to the local standard time. | minutes |
| iStandardBias | INT | This parameter is obsolete. It is ignored while using the function SysTimeRtcSetTimezone. | – |
| iDaylightBias | INT | Additional offset to daylight saving time (summer time). | minutes |
| **(1)** The dates for switching between standard and daylight saving time must be updated each year. An example for updating the switching dates can be found within the application example *RTC Control / Daylight Saving Time / SNTP Client Example* which is provided with the EcoStruxure Machine Expert software. For more information about the example, refer to the [RTC Control / Daylight Saving Time / SNTP Client Example Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=exRTC&topicID=D_SE_0077642). | | | |

The calculations performed during conversion from local time to UTC and vice versa are as follows:

* UTC = local time + (iBias) + (iDaylightBias)
* local time = UTC - (iBias) - (iDaylightBias)
