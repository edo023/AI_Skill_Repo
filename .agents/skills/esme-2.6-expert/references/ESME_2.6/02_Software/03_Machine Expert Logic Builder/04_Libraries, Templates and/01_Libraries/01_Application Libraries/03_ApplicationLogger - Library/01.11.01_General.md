# General

## General

The usage of the Application Logger is slightly different to the usage of other loggers. This section provides information on how to use the Application Logger.

## Connection to the ApplicationLoggerBase Library

As of version V2.0.0.0, the **ApplicationLogger** library uses the **ApplicationLoggerBase** library as a basis for some POUs.

The following POUs are affected:

* [ET\_DiagExt](D-SE-0077660.html#D-SE-0077660)
* [ET\_LogLevel](D-SE-0077662.html#D-SE-0077662)
* [IF\_LoggerPoint](D-SE-0077681.html#D-SE-0077681)
* [ST\_LogEntry](D-SE-0077683.html#D-SE-0077683)
* [GPL (Global Parameter List)](D-SE-0077687.html#D-SE-0077687)
