## Description

Given two 4D input matrices, the function returns the element-wise subtraction of their elements.

![](images/0000611088.png)

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stMatrixA | SE\_MATH.ST\_Matrix4D | First 4D matrix of the subtraction. |
| i\_stMatrixB | SE\_MATH.ST\_Matrix4D | Second 4D matrix of the subtraction. |
