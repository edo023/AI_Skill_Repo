ST\_PosStop

ST\_PosStop - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.1.0.0 |
| Inherits from: | - |

Description

Data structure for parameterization of a PosStop type positioning job

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| lrMaxAcceleration | LREAL | Maximum acceleration of the stop motion (positive value)  Unit: length units/s2 |
| lrMaxDeceleration | LREAL | Maximum deceleration of the stop motion (positive value)  Unit: length units/s2 |
| lrAbsJerk | LREAL | Maximum jerk of the stop motion (positive value) |
| xChannelBundling | BOOL | TRUE := Cancels the active motion jobs of all the channels, adds up their reference values and starts a stop motion on channel A with these values as start condition. May only be used for channel A jobs. |
