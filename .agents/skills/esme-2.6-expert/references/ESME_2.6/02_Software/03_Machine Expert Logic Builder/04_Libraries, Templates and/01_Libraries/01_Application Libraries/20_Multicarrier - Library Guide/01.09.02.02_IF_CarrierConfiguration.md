# IF\_CarrierConfiguration - ConfigureApplicationLoggerEntries (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

![](images/0000757643.png)

## Task

Activating or deactivating the sending of Application Logger entries.

## Description

With the method ConfigureApplicationLoggerEntries, you can activate or deactivate the sending of messages from the carrier to the Application Logger.

## Inputs

| Input | Data type | Description |
| --- | --- | --- |
| i\_xApplicationLoggerEntriesActive | BOOL | Activating/Deactivating the sending of messages to the Application Logger. |

## Outputs

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | Indicates TRUE if an error has been detected. For details, refer to q\_etResult and q\_sResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-509D6EF3.html#ET_Result-509D6EF3) | Provides diagnostic and status information as a numeric value. If q\_xError = FALSE, q\_etResult provides status information. If q\_xError = TRUE, q\_etResult provides diagnostic/error information. |
| q\_sResultMsg | STRING [255] | Provides additional diagnostic and status information as a text message. |
