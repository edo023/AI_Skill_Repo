# ST\_DiagMessageListener

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | ST\_DiagMessageHandler |

## Description

The structure ST\_DiagMessageListener extends the structure ST\_DiagMessageHandler and contains extended information indicating the time when the diagnostic message was set.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| ldtTimeStamp | LDATE\_AND\_TIME | Indicates the time when the diagnostic message was set. |
