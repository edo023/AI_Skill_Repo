Task

Rotation of a point in the 3-dimensional space about the X axis.

Description

Carries out a rotation of the point represented by the position vector i\_stV about the X axis using the angle i\_lrAngle in an anti-clockwise direction. For this purpose, the angle i\_lrAngle must be specified in degrees.

| i\_lrAngle | LREAL | Angle by which the point is to be rotated |
