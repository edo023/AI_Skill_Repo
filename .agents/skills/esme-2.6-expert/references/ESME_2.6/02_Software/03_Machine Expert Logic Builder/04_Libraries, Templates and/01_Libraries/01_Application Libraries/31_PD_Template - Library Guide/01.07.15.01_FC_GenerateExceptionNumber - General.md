## Task

The Function generates the udiExceptionNumber for a ST\_Exception structure.

## Description

The udiExceptionNumber of a ST\_Exception structure has to be a unique number to identify the exception which is used on e.g. a HMI to translate the exception into local language. The structure of udiExceptionNumber is defined as XXXYYYY where YYYY is the value of ET\_DiagExt which describes the exception, and XXX is the value of ET\_ExceptionSource which specifies the ET\_DiagExt table where the ET\_DiagExt comes from.

The input i\_udiDiagExt is devided modulo by 10000. If values are given to the input greater than 9999, the result is not a unique number anymore, because there can be another ET\_DiagExt with a value below 10000 which leads to the same number.
