Task

Function for initialization of an axis that is controlled by the [FB\_AxisModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0077136_1) function block.

Description

This function is used to assign values for the positioning operating mode for the axis specified by iq\_stAxisModuleItf.

The type of motion is specified by i\_etPosMode and there are three types as follows:

oRelative: final position = current position + target distance

oAbsolute: final position = target position

oEndless: Endposition = 0 + target position The current position of the axis is reset to zero before the move starts, typically used with rotary axis with a machine period.
