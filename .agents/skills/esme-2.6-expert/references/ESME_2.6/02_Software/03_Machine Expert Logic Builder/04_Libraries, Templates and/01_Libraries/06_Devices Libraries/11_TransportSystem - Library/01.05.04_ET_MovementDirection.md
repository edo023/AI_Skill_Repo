# ET_MovementDirection

![](images/logo.png)

ET\_MovementDirection

ET\_MovementDirection - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.4.0 |

Description

This enumeration type contains the movement directions for a [ST\_VehicleDestination](../Structures/Structures-5.htm#XREF_D_SE_0090706_1).

Type: UINT

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Bidirectional | 0 | Use shortest distance to reach destination.  If the destination is on another path as the vehicle and if system is a closed loop, use forward movement. |
| Downstream | 1 | Forward movement, starting at upstream end of a motor. |
| Upstream | 2 | Backward movement, starting at downstream end of a motor. |
