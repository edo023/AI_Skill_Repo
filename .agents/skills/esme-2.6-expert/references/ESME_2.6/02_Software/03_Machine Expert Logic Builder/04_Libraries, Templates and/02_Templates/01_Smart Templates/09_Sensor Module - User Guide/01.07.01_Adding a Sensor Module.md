# Adding a Sensor Module

## Adding a Sensor Module

| Step | Action |
| --- | --- |
| 1 | Open the Add Module dialog box:   * Click Project > Add Module... , or * Right-click the Application tree and select Add Module...   **Result:** The Add Module dialog box opens. |
| 2 | Select the module type Sensor and click Next.  **Result:** The Add Sensor dialog box opens. |
| 3 | Enter the Name of your sensor object and click Add Module.  **Result:** The Sensor Module is inserted. |
