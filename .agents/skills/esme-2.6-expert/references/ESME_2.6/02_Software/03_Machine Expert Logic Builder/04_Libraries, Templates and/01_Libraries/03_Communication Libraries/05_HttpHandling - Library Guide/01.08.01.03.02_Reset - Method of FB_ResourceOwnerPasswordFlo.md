# Reset - Method of FB\_ResourceOwnerPasswordFlow

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.3.1.0 |

## Functional Description

The method Reset of the FB\_ResourceOwnerPasswordFlow is used to reset the following internally saved information:

* Access token
* Refresh token
* Authentication data specified by the properties of the function block

The reset is executed only if the authorization process is not in progress.
