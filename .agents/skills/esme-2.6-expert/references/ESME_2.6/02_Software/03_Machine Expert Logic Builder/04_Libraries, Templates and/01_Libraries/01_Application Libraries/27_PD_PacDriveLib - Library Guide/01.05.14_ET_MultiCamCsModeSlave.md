Overview

|  |  |
| --- | --- |

| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Description

Enumeration type for the cold start mode of the [FB\_MultiCam](../Function_Blocks_I_to_Q/Function_Blocks_I_to_Q-14.htm#XREF_D_SE_0087313_1) slave.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| NoChangeOfSlavePosition | 0 | Do not manipulate the slave position. |
| SetSlavePositionToFirstCamPosition | 1 | Set slave position to 1st cam point |
| MoveSlaveToFirstCamPosition | 2 | Axis (slave position) moves to the 1st cam point |
| SetSlavePositionToCamPosition | 3 | Set slave position to the corresponding cam position.  The cold start of the master ([ET\_MultiCamCsModeMaster](Enumerations-13.htm#XREF_D_SE_0087229_1)) is executed first. |
