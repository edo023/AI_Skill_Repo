# ET\_ModeTrackCalibration - General Information

|  |  |
| --- | --- |

| Available as of: | V1.2.5.0 |

## Description

The enumeration ET\_ModeTrackCalibration specifies the track calibration mode, depending on the working direction of the track in automatic operation mode.

For more information on the working direction, refer to [Working Direction](IntroMC_CoordSys-0FC9FA31.html#IntroMC_CoordSys-0FC9FA31__WorkingDirection-0FC9F3F6). For more information on the moving direction, refer to [Moving Directions](IntroMC_MovDir-10BB46E9.html#IntroMC_MovDir-10BB46E9).

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| Forward | 0 | The track calibration is performed in forward direction. |
| Backward | 1 | The track calibration is performed in backward direction. |
| BothDirections | 2 | The track calibration is performed in forward and backward direction. NOTE: This calibration mode is only applicable to tracks that are operated both in clockwise (Not inverted) and counterclockwise (Inverted) working direction. |
