M660 controller are supported.

## Modifications

* A tool configured by the method ConfigureTool is now taken into account for monitoring the work envelope.
* A call of the parameter SetTrackingDeviationLimit with component All sets, in addition, the value for orientation.
* The function block FB\_EllipticSpline.CalcHalfSpline in combination with the parameter SetWorkingPlane = ET\_WorkingPlane.XY uses the Z element instead of the 2nd element of the working plane. The improved calculation is based on the 2nd coordinate.
* The retaining of the connected path is enforced after a warm start, if the robot was already in target.
* The verification of the position passed by the ExternalPositionSource in case a component is not configured is improved. The value of any not configured component must be zero.

## Enhancements

The MoveJ motion of the Lexium SCARA Robot is improved.
