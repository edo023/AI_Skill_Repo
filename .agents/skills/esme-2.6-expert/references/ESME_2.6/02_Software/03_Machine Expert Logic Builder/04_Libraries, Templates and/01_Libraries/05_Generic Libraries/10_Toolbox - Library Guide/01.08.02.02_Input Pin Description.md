# Input Pin Description

## Input Pin Description

This table describes the input pins of the `Filter_AnalogInput` function block:

| Input | Data Type | Description |
| --- | --- | --- |
| `i_xEn` | `BOOL` | TRUE: Enabled  FALSE: Disabled |
| `i_rIput` | `REAL` | Analog input variable  Range: 1.17e-38...3.4e+38 |
| `i_rMaxIputDiff` | `REAL` | Maximum difference between current and previous inputs for plausibility check  Range: 0...3.4e+38  `i_rMaxIputDiff` < 0 generates detected error |
| `i_xErrRst` | `BOOL` | TRUE: Reset the detected error (on rising edge).  (Optional) |
