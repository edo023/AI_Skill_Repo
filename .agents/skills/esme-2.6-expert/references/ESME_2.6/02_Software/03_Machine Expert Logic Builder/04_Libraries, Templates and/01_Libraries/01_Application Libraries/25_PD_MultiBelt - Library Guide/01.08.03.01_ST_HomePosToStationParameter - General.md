# ST\_HomePosToStationParameter - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |
| Versions: | Current version |

## Description

This structure contains the parameters that are necessary for the homing with the help of the function block FB\_HomePosToStation.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| stMove | [ST\_Movement](D-SE-0077763.html#D-SE-0077763) | Motion parameter for the homing on Touchprobe. |
| lrMinDistance | LREAL | Indicates the minimum distance that train 1 has to move. Otherwise all trains move a complete round.  This can be used to empty a station.  If a value > 0 is entered, all trains drive a whole pass, even if they were in the right position at the beginning of the homing. |
| uiStation | UINT | Indicates to which station the trains are to be moved. |
