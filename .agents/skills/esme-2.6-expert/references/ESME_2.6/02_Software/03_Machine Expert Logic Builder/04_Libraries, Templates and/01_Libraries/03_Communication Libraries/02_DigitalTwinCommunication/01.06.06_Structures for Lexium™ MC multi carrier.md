# Structures for Lexium™ MC multi carrier

## Good Practices for Starting the Emulation

To build and emulate the Lexium™ MC multi carrier device in EcoStruxure Machine Expert Twin, use the Multicarrier Configuration editor implemented in the Logic Builder. Select the option Create Emulation Data in the Code Generation tab of the Project > Project Settings > Multicarrier Configuration Editor [dialog box](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=TPC_MLS_Config_Tools_Options_D995A499) to generate the appropriate code using the DigitalTwinCommunication library.

To exchange emulation data with EcoStruxure Machine Expert Twin, start an OPC UA server on your controller. Refer to [Preconditions for Emulation with Multicarrier Configuration Editor in the Lexium™ MC multi carrier Example Guide for further information](../../../../../api/crossBook?lang=en-US&virtualBookName=exMulCar&topicID=EmulatPrecon_37601F4C).
