# ET_Vector3DTo2D

ET\_Vector3DTo2D

ET\_Vector3DTo2D - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Description

Enumeration type for the mapping relation of the conversion of a 3-dimensional vector into a 2-dimensional vector.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| XyToXy | 1 | X component to X component, Y component to Y component |
| YxToXy | 2 | Y component to X component, X component to Y component |
| XzToXy | 3 | X component to X component, Z component to Y component |
| ZxToXy | 4 | Z component to X component, X component to Y component |
| YzToXy | 5 | Y component to X component, Z component to Y component |
| ZyToXy | 6 | Z component to X component, Y component to Y component |
