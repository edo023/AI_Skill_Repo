# ST_RadiusDetermination

![](images/logo.png)

ST\_RadiusDetermination

ST\_RadiusDetermination - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

Description

This structure contains the parameters and feedback data for the calculation of the radius and mass moment of inertia of the bobbin.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_xSetRadiusNow | BOOL | TRUE: Instead of the internal calculated radius of the bobbin the specified radius at the input i\_lrSetRadiusValue is used. A new radius calculation starts when this input is set back to FALSE. |
| i\_lrSetRadiusValue | LREAL | The set radius (for example, of an external measurement) which should be used as the radius of the bobbin. |
| i\_lrMasterMeasurementPeriod | LREAL | The distance of the master movement over which a new radius of the bobbin is measured. |
| i\_lrMaxRadiusChange | LREAL | To help prevent incorrect calculations, a new calculated radius of the bobbin can only be i\_lrMaxRadiusChange larger or smaller than the radius. A radius set via i\_lrSetRadiusValue or measured during the start movement is not limited. |
| i\_xConsiderControlValue | BOOL | FALSE: During the radius calculation the feedback value of the sensor (for example, for the dancer) is not considered. |
| i\_lrFoilWidth | LREAL | The width (mm) of the winded foil to calculate the J-Load of the bobbin. |
| i\_lrFoilDensity | LREAL | The density (kg/m3) of the winded foil to calculate the J-Load of the bobbin. |
| i\_lrJLoadGain | LREAL | The calculated J-Load of the bobbin is multiplied with this factor before the J-Load of the axis is written. |
| i\_lrCoilRadius | LREAL | The radius (mm) of the bobbin coil to calculate the J-Load of the bobbin. |
| iq\_plrRadius | POINTER TO LREAL | The radius of the bobbin. Start movements are calculated according to this radius. The radius is overwritten every time a new radius is calculated or set via i\_xSetRadiusNow. |
