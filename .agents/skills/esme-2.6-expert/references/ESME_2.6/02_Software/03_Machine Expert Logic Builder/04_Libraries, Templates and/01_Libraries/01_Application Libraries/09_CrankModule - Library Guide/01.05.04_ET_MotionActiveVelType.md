# ET_MotionActiveVelType

![](images/logo.png)

ET\_MotionActiveVelType

ET\_MotionActiveVelType - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.2.0 |

Description

The enumeration describes all the velocity sources on the basis of which the q\_xMotionActive output of the [FB\_CrankModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0088740_1) is set if the velocity exceeds lrMotionActiveVelLimit.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| RefVelocity | 1 | Reference velocity of the axis. |
| Velocity | 2 | Actual velocity of the axis. |
| MechRefVelocity | 3 | Dead time compensated reference velocity of the axis. |
| MechVelocity | 4 | Dead time compensated velocity of the axis. |
