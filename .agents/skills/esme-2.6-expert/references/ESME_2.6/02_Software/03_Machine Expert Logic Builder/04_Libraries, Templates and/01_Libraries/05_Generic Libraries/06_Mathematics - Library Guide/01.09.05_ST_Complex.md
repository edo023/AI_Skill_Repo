# ST\_Complex

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |

## Description

The structure ST\_Complex represents complex numbers (also referred to as imaginary numbers), represented as lrRe + lr I m \* i with i2 = -1.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| lrRe | LREAL | Real part of the complex number. |
| lrIm | LREAL | Imaginary part of the complex number. |
