# Input and Output Pin Description

![](images/logo.png)

Input and Output Pin Description

Input Pin Description

This table describes the input pins of the TeSysU\_CtrlCmdCyc\_CANopen function block:

| Input | Data Type | Description | Control Unit |
| --- | --- | --- | --- |
| i\_wReg455 | WORD | This input must be linked to the status register that is transported in the CANopen TxPDO1 parameter. | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| i\_xFwdRun | BOOL | Motor [run](../glossary/glossary.htm#XREF_D_SE_0024697_591) forward command | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| i\_xRevRun | BOOL | Motor run reverse command | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| i\_xRstFlt | BOOL | Reset detected error | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| i\_xRstWarn | BOOL | Reset communication loss alert | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| i\_xTherOvldTest | BOOL | Automatic thermal overload test | oLUCM |
| i\_xTripTest | BOOL | Overcurrent trip test | oLUCM |

Output Pin Description

This table describes the output pins of the TeSysU\_CtrlCmdCyc\_CANopen function block:

| Output | Data Type | Description | Control Unit |
| --- | --- | --- | --- |
| q\_wReg704 | WORD | This output must be linked to control of the system register that is transported in the CANopen RxPDO1 parameter. | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| q\_wReg703 | WORD | This output must be linked to the control of communication module register that is transported in the CANopen RxPDO1 parameter. | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| q\_xRdy | BOOL | TeSysU rotary handle is turned to On position and there is no detected error. | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| q\_xCls | BOOL | Pole status: closed | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| q\_xTrip | BOOL | TeSysU rotary handle is turned to Trip position. | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| q\_xFlt | BOOL | All detected errors | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| q\_xAlrm | BOOL | All alerts | oLUCA/LUCL  oLUCB/C/D  oLUCM |
| q\_xRstAuth | BOOL | Detected error reset authorized status | oLUCB/C/D  oLUCM |
| q\_xStrt | BOOL | Start in progress  0 = Descending current is lower than 150[%](../glossary/glossary.htm#XREF_D_SE_0024697_611) [FLA](../glossary/glossary.htm#XREF_D_SE_0024697_708)  1 = Ascending current is greater than 10% FLA | oLUCB/C/D  oLUCM |
| q\_xRun | BOOL | Motor runs with detection of current, if greater than 10% FLA. | oLUCB/C/D  oLUCM |
| q\_wCurrAvge | WORD | Average motor current (% FLA)  32 = 100% FLA  63 = 200% FLA  Range: 0...63 | oLUCB/C/D  oLUCM |
