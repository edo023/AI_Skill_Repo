# General Information - FB\_FsmTransitionLogger

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.2.9.0 |

## Task

Logging of transitions processed by the FB\_FiniteStateMachine.

## Description

The function block FB\_FsmTransitionLogger implements a logger that records the requested and executed state transitions of the FB\_FiniteStateMachine function block.

The FB\_FsmTransitionLogger functions as a ring buffer. If the buffer is full, the earliest log entry is deleted when a new log entry is made.

NOTE: The FB\_FsmTransitionLogger operates independently from the application logger.

## Interface

The function block provides the generic constant uiNumberOfLogEntries. The value for this constant must be defined during build time. It cannot be modified during runtime.

The value specifies the size of the array located inside the function block which contains the logger messages.

The generic constant must be defined in the declaration of the instance of this function block.

For example:

```
VAR
       fbFsmTransitionLogger : SE_CTBX.FB_FsmTransitionLogger<100>;
END_VAR
```

The function block must not be called in the application.

## Methods

| Name | Description |
| --- | --- |
| Clear | Clears the logger entries. |
