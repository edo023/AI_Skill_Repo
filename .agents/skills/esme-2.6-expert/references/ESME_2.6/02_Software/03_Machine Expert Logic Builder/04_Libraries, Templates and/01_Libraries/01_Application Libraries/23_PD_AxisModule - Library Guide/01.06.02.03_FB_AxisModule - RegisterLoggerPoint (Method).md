# FB_AxisModule - RegisterLoggerPoint (Method)

![](images/logo.png)

FB\_AxisModule - RegisterLoggerPoint (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | 1.4.0.0 |

Task

The method registers the internal logger point in the [Application Logger](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.ApplicationLogger&topicID=D_SE_0090069_1).

Description

The function block contains a logger point implemented for the [Application Logger](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.ApplicationLogger&topicID=D_SE_0090069_1). After successfully calling-up this method, the function block sends messages to the Application Logger via this logger point. This method is identical to the call-up of the [RegisterLoggerPoint](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.ApplicationLogger&topicID=D_SE_0077672_1) method at [APL.FB\_LoggerPoint](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.ApplicationLogger&topicID=D_SE_0077670_1).

After registration:

oThe function block automatically sends messages to the Application Logger.

o The [APL.ET\_LogLevel](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.ApplicationLogger&topicID=D_SE_0077662_1) enumeration (logger level) describes the type and importance of a message.

oDepending on the logger level (ET\_LogLevel), various information is send to the Application Logger.

| [APL.ET\_LogLevel](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.ApplicationLogger&topicID=D_SE_0077662_1) | Information sent to the Application Logger |
| --- | --- |
| 10 (EmergencyMessage)...40 (Warning) | Messages, exceptions and warnings, reported by the function block |
| 50 (UserAction)...60 (ExternalEvent) | Commands, sent to the function block |
| 70 (StatusMessage) | Status changes at the outputs of the function block |
| 80 (DebugMessage) | Internal messages (for expert debugging only) |
