## Task

Implements the collision handler of a T-Series robot.

## Description

The function block is an extension of FB\_CollisionHandlerDelta2Ax and allows a simplified configuration of the collision handler of a Delta2Ax starting from a configured instance of FB\_RobotTSeries.

For Modicon M262/M660 Motion Controllers, the data type is CMI.IF\_AxisIdentification.

For more information, refer to [FB\_CollisionHandlerDelta2Ax](../../../../../api/crossBook?lang=en-US&virtualBookName=CollisionDetection&topicID=BCE042CB).
