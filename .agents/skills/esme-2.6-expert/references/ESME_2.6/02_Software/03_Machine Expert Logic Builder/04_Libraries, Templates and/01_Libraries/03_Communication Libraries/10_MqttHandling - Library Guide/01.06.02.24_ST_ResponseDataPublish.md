# ST\_ResponseDataPublish

## Overview

|  |  |
| --- | --- |

| Available as of: | V2.4.4.0 |

## Description

The structure ST\_ResponseDataPublish contains information extracted from the response sent by the MQTT broker after publishing a message with QoS greater than 0.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| etReasonCode | [ET\_ReasonCode](ET_ReasonCode-D887BF12.html) | Indicates the reason code in the PUBACK message sent by the MQTT broker. |
| stProperties | [ST\_PropertiesPuback](ST_PropertiesPuback-D8F264DF.html) | Indicates the properties of the PUBACK message sent by the MQTT broker. |
