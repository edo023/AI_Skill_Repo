# ET\_AxisMotionCommand

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

The enumeration ET\_AxisMotionCommand is used for the enumeration of the active command.

## Enumeration Elements

| Name | Description |
| --- | --- |
| None | No command has been selected. |
| PowerOn | Enables the power stage. |
| PowerOff | Disables the power stage. |
| Stop | Stops the drive without the possibility to interrupt it. |
| Halt | Stops the drive with the possibility to interrupt it. |
| MoveVelocity | Performs movement with specified target velocity. |
| CamIn | Performs movement by following a master. |
| MoveAbsolute | Performs a movement to a specified absolute target position. |
| MoveRelative | Performs a movement with a specified distance with reference to the position. |
| MoveAdditive | Performs a movement with a specified distance with reference to the previous target position. |
| CustomJob | Controls an axis by a custom algorithm. |
| MoveSuperimposed | Performs a superimposed movement with a specified position offset with reference to the position of an ongoing movement. |
| GearIn | Coupling of a master axis and a subordinate axis. |
| PhasingAbsolute | Creates a position offset between the position of a master axis and the position of this master axis as it is seen by the subordinate axis. |
| Jog | In the operating mode Jog, a movement is started using the properties xJogForward and xJogBackward. |
