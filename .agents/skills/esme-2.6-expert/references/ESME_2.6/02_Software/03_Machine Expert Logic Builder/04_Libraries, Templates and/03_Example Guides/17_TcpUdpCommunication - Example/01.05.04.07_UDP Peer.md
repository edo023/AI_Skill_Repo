# UDP Peer

![](images/logo.png)

UDP Peer

Overview

The program SR\_UdpPeer implements the program code for the [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) [peer](../glossary/glossary.htm#XREF_D_SE_0055638_15) functionality. For control and monitoring of this functionality, the visualization screen VisuUdpPeer is provided in the example project.

The variables which link the visualization with the program code are defined in the global variable list GVL\_VisuUdpPeer.

Opening and Binding

In the visualization, enter the number of the [port](../glossary/glossary.htm#XREF_D_SE_0055638_16) you want to open and the [IP address](../glossary/glossary.htm#XREF_D_SE_0055638_11) of the corresponding [Ethernet](../glossary/glossary.htm#XREF_D_SE_0055638_7) interface. To open this UDP [port](../glossary/glossary.htm#XREF_D_SE_0055638_16), click the Open & Bind button in the visualization.

You can close an open port by clicking the Close button.

Sending Data

When the [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) port is open and bound, a [data packet](Project_Information-6.htm#XREF_D_SE_0088484_3) can be sent to a remote [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) [peer](../glossary/glossary.htm#XREF_D_SE_0055638_15). The remote [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) [peer](../glossary/glossary.htm#XREF_D_SE_0055638_15) is selected by its [IP address](../glossary/glossary.htm#XREF_D_SE_0055638_11) and the UDP [port](../glossary/glossary.htm#XREF_D_SE_0055638_16). You can modify the text message in the visualization. To send data to the remote [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) [peer](../glossary/glossary.htm#XREF_D_SE_0055638_15), click the Send button in the visualization.

Receiving Data

If the [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) [peer](../glossary/glossary.htm#XREF_D_SE_0055638_15) detects that data has been received on the bound [port](../glossary/glossary.htm#XREF_D_SE_0055638_16), the data is read from the receive buffer. The received data is converted into the user-defined [data format](Project_Information-6.htm#XREF_D_SE_0088484_3). In this example application, the data is not processed, but displayed on the visualization for information purposes.
