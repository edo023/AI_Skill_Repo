# ET_Diag

![](images/logo.png)

ET\_Diag

ET\_Diag - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.1.0 |
| Versions: | Current version |

Description

Enumeration type describing the general, library-independent diagnostic.

A value not equal to ET\_Diag.Ok corresponds to an diagnostic message.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| [OK](#XREF_D_SE_0076228_5) | 0 | OK |
| [UnexpectedProgramBehavior](#XREF_D_SE_0076228_6) | 1 | Unintended behavior of the program. |
| [DriveConditionInvalid](#XREF_D_SE_0076228_7) | 2 | The necessary requirements for the intended use of the drive are not fulfilled. |
| [InputParameterInvalid](#XREF_D_SE_0076228_8) | 3 | An input parameter has an invalid value |
| [PowerSupplyConditionInvalid](#XREF_D_SE_0076228_9) | 4 | The requirements for enabling the drive power supply are nor fulfilled. |
| [FileHandlingInvalid](#XREF_D_SE_0076228_10) | 5 | The file cannot be accessed. |
| [ControllerConditionInvalid](#XREF_D_SE_0076228_11) | 6 | The controller does not provide the required performance characteristics. |
| [WarmStartConditionInvalid](#XREF_D_SE_0076228_12) | 7 | The warm start could not be executed. |
| [LimitReached](#XREF_D_SE_0076228_13) | 8 | A limit has been reached |
| [HomingFailed](#XREF_D_SE_0076228_14) | 9 | Referencing the drive was not successful |
| [SercosConditionInvalid](#XREF_D_SE_0076228_15) | 10 | The Sercos bus is not in the required state. |
| [ExecutionAborted](#XREF_D_SE_0076228_16) | 12 | The further execution of a program section aborted. |
