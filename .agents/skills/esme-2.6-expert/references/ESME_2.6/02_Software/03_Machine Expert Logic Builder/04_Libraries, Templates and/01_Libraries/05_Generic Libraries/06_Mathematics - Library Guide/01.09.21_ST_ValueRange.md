# ST\_ValueRange

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.1.12.0 |

## Description

The structure ST\_ValueRange contains a minimum and a maximum value defining a range.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| lrMin | LREAL | Minimum value of the range. |
| lrMax | LREAL | Maximum value of the range. |
