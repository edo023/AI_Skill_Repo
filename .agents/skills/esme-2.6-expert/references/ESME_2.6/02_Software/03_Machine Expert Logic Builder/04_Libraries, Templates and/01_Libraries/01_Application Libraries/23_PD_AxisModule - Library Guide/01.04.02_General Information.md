# General Information

![](images/logo.png)

General Information

Description

The "PD\_AxisModule" IEC library contains the equipment module for the template as well as all required auxiliary functions.

The axis module controls one individual servo axis with general motion tasks such as EndlessFeed, Positioning, MultiCam, etc

Namespace of the Library

The default namespace of this library is AXM.

The POUs, data structures, enumerations and constants have to be addressed via this namespace.
