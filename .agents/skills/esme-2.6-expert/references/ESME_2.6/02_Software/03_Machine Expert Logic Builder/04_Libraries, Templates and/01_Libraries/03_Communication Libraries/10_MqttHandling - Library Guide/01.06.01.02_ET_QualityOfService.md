# ET\_QualityOfService

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_QualityOfService specifies the Quality of Service (QoS) with which application messages are sent or topics subscribed.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| QoS0 | 0 | QoS level 0: At most one delivery.  No response is sent by the receiver and no retry is performed by the sender. |
| QoS1 | 1 | QoS level 1: At least one delivery.  Each application message is assigned with a unique packet identifier and the reception needs to be acknowledged by the receiver. |
| QoS2 | 2 | QoS level 2: Exactly one delivery.  NOTE: QoS level 2 is not supported by the MQTT client. It can be received after a connection has been established as a value of the [property uiMaximumQoS](ST_PropertiesConnack-D8E1CD7A.html) in the CONNACK message sent by the MQTT broker. |

## Used By

* ST\_WillMessage
* ST\_PublishMessage
* ST\_SubscribeTopic
