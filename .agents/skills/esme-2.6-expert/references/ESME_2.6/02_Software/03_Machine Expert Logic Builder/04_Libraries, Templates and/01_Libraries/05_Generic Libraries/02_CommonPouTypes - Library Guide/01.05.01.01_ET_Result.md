# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.4.0 |

## Description

The enumeration ET\_Result contains the possible values returned by the [control methods](ControlMethods-D1D3BB4F.html).

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| Busy | 0 | The method is being executed. |
| Done | 1 | The method has been executed successfully. |
| Error | 2 | An error has been detected while executing the method. |
