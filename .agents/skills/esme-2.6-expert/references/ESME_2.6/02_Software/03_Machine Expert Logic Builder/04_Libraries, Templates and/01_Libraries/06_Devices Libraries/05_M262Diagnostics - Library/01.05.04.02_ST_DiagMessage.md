Description

The structure ST\_DiagMessage provides information on the diagnostic messages.

Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| etSeverity | [ET\_Severity](../Enumerations/Enumerations-5.htm#XREF_D_SE_0097652_1) | Severity of the diagnostic message. |
| uliTimestamp | ULINT | Time stamp of the diagnostic message, provided as high-resolution time stamp which indicates the number of milliseconds since January 1st, 1970 00:00:00:000. |
| udiId | UDINT | Unique identifier of the diagnostic message. |
| sMessage | STRING(255) | Diagnostic message. |
| etSource | [ET\_DiagSource](../Enumerations/Enumerations-3.htm#XREF_D_SE_0097651_1) | Source of the diagnostic message. |
