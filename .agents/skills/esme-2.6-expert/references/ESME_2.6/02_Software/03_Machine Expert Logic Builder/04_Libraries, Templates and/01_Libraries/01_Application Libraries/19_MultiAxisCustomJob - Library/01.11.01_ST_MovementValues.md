# ST\_MovementValues

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | 1.0.0.0 |

## Description

The structure ST\_MovementValues contains the movement values (position, velocity, and acceleration) for both master and subordinate axes of a multi-axis custom job.

When used with a master channel, the structure receives the movement data of the master axis. When used with a subordinate channel, the structure provides the target values calculated by the user-defined algorithm to the axis.

ST\_MovementValues is an alias of MOIN:ST\_MovementValues of the MotionInterface library.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
