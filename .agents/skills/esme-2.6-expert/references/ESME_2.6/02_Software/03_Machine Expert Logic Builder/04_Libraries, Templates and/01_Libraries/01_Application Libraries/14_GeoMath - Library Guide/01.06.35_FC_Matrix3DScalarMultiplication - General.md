## Description

Given a 3D matrix and a scalar value, the function returns a new 3D matrix resulting from multiplying the scalar value by each element of the input matrix.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stMatrix | SE\_MATH.ST\_Matrix3D | A 3D matrix. |
| i\_lrScalar | LREAL | A scalar value to add to the elements of i\_stMatrix. |
