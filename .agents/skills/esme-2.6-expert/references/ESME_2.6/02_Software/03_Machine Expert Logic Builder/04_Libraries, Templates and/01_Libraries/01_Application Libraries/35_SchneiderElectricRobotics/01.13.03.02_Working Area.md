# Working Area

## Overview

There is a monitoring of the working area in the following situations:

* If a move command is sent and xDisableWorkEnvelope = FALSE:

  + TCP position is verified
  + Axis position is verified
  + Orientation is not verified
* If a move command is sent and xDisableWorkEnvelope = TRUE:

  + TCP position is not verified
  + Axis position is not verified
  + Orientation is not verified

In addition, the monitoring includes a verification of the following actions when the robot is active:

* The present TCP position
* The present axis position
* The present Z orientation
