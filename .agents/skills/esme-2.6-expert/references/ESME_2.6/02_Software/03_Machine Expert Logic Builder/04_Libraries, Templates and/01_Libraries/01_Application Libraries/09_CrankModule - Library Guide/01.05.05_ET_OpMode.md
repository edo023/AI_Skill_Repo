# ET_OpMode

![](images/logo.png)

ET\_OpMode

ET\_OpMode - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.2.0 |

Description

The enumeration describes all the operating modes of the [FB\_CrankModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0088740_1).

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| NoOp | 0 | No operation mode selected. |
| Homing | 10 | Homing of the axis. |
| Manual | 20 | Touch control mode. |
| MultiCam | 30 | Run master slave cams. |
| Endless | 40 | Endless feed with (constant) velocity. |
| Positioning | 50 | Positioning mode. |
| BrakeRelease | 70 | Release the brake of the axis. |
