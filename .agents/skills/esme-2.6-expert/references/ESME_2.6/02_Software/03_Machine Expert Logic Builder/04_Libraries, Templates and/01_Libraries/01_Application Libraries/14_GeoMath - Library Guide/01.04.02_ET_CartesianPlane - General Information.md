## Description

The enumeration ET\_CartesianPlane describes the possible Cartesian plane combinations.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No plane selected. |
| XY | 1 | The XY Cartesian plane. |
| XZ | 2 | The XZ Cartesian plane. |
| YZ | 3 | The YZ Cartesian plane. |
