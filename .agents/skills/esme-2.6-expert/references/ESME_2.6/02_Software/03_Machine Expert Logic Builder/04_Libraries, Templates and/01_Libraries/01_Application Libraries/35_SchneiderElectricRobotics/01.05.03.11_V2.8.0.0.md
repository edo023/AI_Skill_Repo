## New Functions

* **Lexium T Robot Feedback**

  The property ifFeedback of type IF\_RobotTSeriesFeedback has been added to FB\_RobotTSeries.

  It contains for example indicators for tracking deviation, motor overload, and load of the axes of a Lexium T Robot.

  It can be used, for example, to simulate a robotics application to get feedback about its feasibility.
* **Lexium P Robot / Lexium S Robot / Lexium T Robot Interface**

  Interfaces IF\_RobotPSeries, IF\_RobotSSeries, and IF\_RobotTSeries have been added to access the methods and properties implemented by the function blocks FB\_RobotPSeries, FB\_RobotSSeries, and FB\_RobotTSeries.
