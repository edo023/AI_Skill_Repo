# ST\_Command

## Overview

|  |  |
| --- | --- |
| Type | Data structure |
| Available as of | V1.0.1.0 |

## Description

The structure ST\_Command represents the list of PackTags of type Command as defined in ANSI/ISA TR88.00.02-2022.

Command tags are used to control the operation of the unit machine.

## Structure Elements

For the GPL data types, refer to [Global Parameters](D-SE-0077960.html#D-SE-0077960).
