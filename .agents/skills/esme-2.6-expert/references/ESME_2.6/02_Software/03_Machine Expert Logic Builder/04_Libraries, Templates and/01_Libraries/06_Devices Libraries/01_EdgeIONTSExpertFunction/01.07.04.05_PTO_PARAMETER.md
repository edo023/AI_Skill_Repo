# PTO\_PARAMETER

## Description

The PTO\_PARAMETER enumeration describes the types of PTO parameters.

## Enumeration Elements

The enumeration data type ENUM contains the following values:

| Parameter name | Data type | Parameter number | R/W | Description |
| --- | --- | --- | --- | --- |
| CommandedPosition | DINT | 1 | R | Provides the distance traveled during the previous motion command. |
| SWPosLimit | DINT | 2 | R/W | The value of the SW Limit Positive Enabled parameter. |
| SWNegLimit | DINT | 3 | R/W | The value of the SW Positive Limit parameter. |
| EnableSWLimitPos | BOOL | 4 | R/W | The value of the SW Limit Negative Enabled parameter. |
| EnableSWLimitNeg | BOOL | 5 | R/W | The value of the SW Negative Limit parameter. |
| EnableDirPos | BOOL | 6 | R/W | The value of the Direction Positive Enabled parameter. |
| EnableDirNeg | BOOL | 7 | R/W | The value of the Direction Negative Enabled parameter. |
| MaxVelocityAppl | DINT | 9 | R/W | The value of the Maximum Velocity parameter. |
| CommandedVelocity | DINT | 11 | R | Provides the target velocity value of the previous motion command. |
| MaxAccelerationAppl | DINT | 13 | R/W | The value of the Maximum Acceleration parameter. |
| MaxDecelerationAppl | DINT | 15 | R/W | The value of the Maximum Deceleration parameter. |
| Delay | DINT | 1000 | R/W | The value of the Delay parameter. |
