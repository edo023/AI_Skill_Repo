## Interface in Case of Node Type 'Non Template'

| Input | Data type | Description |
| --- | --- | --- |
| i\_xEnable | BOOL | A rising edge FALSE -> TRUE activates the POU, a falling edge TRUE -> FALSE deactivates the POU.  A deactivated POU does not execute any action. |
| i\_xAsyncStop | BOOL | Initiates an immediate asynchronous stop. The process cannot be accessed while the exception is pending. No commands can be executed while the reaction is pending. You can still execute certain special operating modes and commands required for exception elimination. |
| i\_xSyncStopEL | BOOL | Initiates a synchronous stop (monitored by a configurable timeout to wait until the robot path movement stops). Deactivates the process after the synchronous stop has been completed. The process cannot be accessed while the exception is pending. If the timeout is reached, the function transitions to an asynchronous stop. |
| i\_xSyncStopEH | BOOL | Initiates a synchronous stop (monitored by a configurable timeout to wait until the robot path movement stops). Does not disable the process after completion of the synchronous stop. |
| i\_xStopEndOfCycle | BOOL | Initiates a stop at the end of the cycle. Does not deactivate the process. |
| i\_xDiagQuit | BOOL | A rising edge FALSE -> TRUE cancels an active exception of the POU. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_etCmd | [RM.ET\_Cmd](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.RoboticModule&topicID=D_SE_0076886). | Transfers a module command to the module. |
| iq\_stRobotModuleInterface | [RM.ST\_ModuleInterface](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.RoboticModule&topicID=D_SE_0076969). | Robotic Module specific parameters. |

| Output | Data type | Description |

| q\_xActive | BOOL | TRUE: The POU is active. If the output is TRUE while the i\_xEnable is deactivated, the POU must first terminate its ongoing processing before transitioning this output to FALSE.  FALSE: The POU is inactive |
| q\_xReady | BOOL | TRUE: The POU is ready to operate and can accept user commands.  FALSE: The function block is not ready to accept user commands. |
| q\_etDiag | *[GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228)* | General library-independent statement on the diagnostic. A value not equal to GD.ET\_Diag.Ok corresponds to a diagnostic message. |

| q\_sDiagExt | STRING[80] | The value of the respective enumeration of q\_udiDiagExt. |
| q\_sMsg | STRING[80] | Event-triggered message that gives more information on the diagnostic state. |
| q\_xException | BOOL | An error was detected and an exception is active. |

| q\_xSyncStopEL | BOOL | A synchronous stop (monitored by a configurable timeout to wait until the robot path movement stops) is active. Deactivates the process after the synchronous stop has been completed. The process cannot be accessed while the exception is pending. If the timeout is reached, the function transitions to an asynchronous stop. |
| q\_xSyncStopEH | BOOL | A synchronous stop (monitored by a configurable timeout to wait until the robot path movement stops) is active. Does not disable the process after completion of the synchronous stop. |
