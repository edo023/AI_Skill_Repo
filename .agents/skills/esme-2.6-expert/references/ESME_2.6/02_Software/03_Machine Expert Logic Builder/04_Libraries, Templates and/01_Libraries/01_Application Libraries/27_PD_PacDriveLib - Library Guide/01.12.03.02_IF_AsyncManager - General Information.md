Task

Transfer and execute code, which the user has written, to a task running asynchronously to the program.

Description

Using this interface, additional tasks can be generated that run asynchronously to the task in which the interface is allocated. Function blocks, which must implement [IF\_Async](Interfaces-3.htm#XREF_D_SE_0087703_1), can be transferred to these asynchronous tasks. The Job method of the function block then is executed in an asynchro­nously running task.
