| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |
| Versions: | Current version |

Task

Access to values and functions of a belt of type Series

Description

A belt of type Series is driven by an axis, which is controlled by [FB\_Infeed](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080279_1).

Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ifDrive | SystemConfigurationItf.IF\_Drive | Read/Write | The axis that drives the belt |
| udiMasterBeltId | UDINT | Read/Write | The number of the belt that serves as CAM master for this belt |
| lrActVelRatio | LREAL | Read | The velocity ratio of the feed forward motion with which the belt follows the master |
| xReadyForMotionCommand | BOOL | Read | TRUE: The belt is ready to perform changes of the feed forward velocity or correction motions. |
| sDriveName | STRING[40] | Read | The name of the axis that drives the belt |
