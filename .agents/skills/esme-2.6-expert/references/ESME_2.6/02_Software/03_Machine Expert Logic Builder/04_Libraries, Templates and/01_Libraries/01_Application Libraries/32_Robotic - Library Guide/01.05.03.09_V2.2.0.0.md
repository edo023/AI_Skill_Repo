## Supported Hardware

* PacDrive LMC Eco
* PacDrive LMC Pro
* PacDrive LMC Pro2

## Software Requirements

* SoMachine Motion V4.3 SP1

## Firmware Requirements

PacDrive 3 V4.3 SP1

* PacDrive LMC Eco V1.54.20.3 or greater
* PacDrive LMC Pro V1.54.20.3 or greater
* PacDrive LMC Pro2 V1.54.20.3 or greater

## New Functions

* IF\_RobotFeedback.ifSpace.rstEstimatedStopPosition

  The position represents the estimated Cartesian stop position on the connected path including a potential tracking.
* IF\_RobotFeedback.ifTrajectoryStorage.ifSpace.rstEstimatedStopPosition

  The position represents the estimated Cartesian stop position on the connected path.
* IF\_RobotFeedback.ifTracking.rstEstimatedStopPosition

  The position represents the estimated Cartesian stop position where an active tracking is stopped.

## Modifications

* None

## Enhancements

* In several situations the diagnostic message ET\_Diag.UnexpectedProgramBehavior is returned by the robot in connection with calculating the motion profile. Calculating the motion profile of the robot has been improved in order to reduce the occurrence of errors.
* A watchdog exception does not occur in connection with calling the method IF\_RobotMotion.ClearSegmentsFromId().
* The application logger entries generated by the call of IF\_RobotMotion.MoveSync are more detailed.
