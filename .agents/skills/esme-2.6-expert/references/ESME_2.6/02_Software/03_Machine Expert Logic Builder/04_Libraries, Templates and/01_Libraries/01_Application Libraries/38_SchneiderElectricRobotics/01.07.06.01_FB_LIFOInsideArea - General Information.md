## Task

Search for a valid target with the smallest position along the direction of the tracking.

## Description

The LIFO (Last In First Out) algorithm searches for a valid target with the smallest position along the direction of the tracking.

![](images/DD01426926.gif)

Other than the position, the owner of each target is also accounted in applying the algorithm, depending on the values of i\_xSelectTargetsWithNoOwner and i\_xSelectTargetsWithAnyOwner provided on the last successful call of the method [**SetData**](D-SE-0098052.html#D-SE-0098052).
