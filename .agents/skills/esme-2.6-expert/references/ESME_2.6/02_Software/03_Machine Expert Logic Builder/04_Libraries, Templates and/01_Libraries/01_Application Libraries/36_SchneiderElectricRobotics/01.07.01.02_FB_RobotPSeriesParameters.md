## Task

Configure a drive mapping for OrientationZ.

## Description

By calling up the method, the drive handed over at input i\_ifDrive is used to configure a drive mapping for OrientationZ.

The drive handed over at input i\_ifDrive is configured as ROB.ET\_RobotComponent.AxisD for the robot and the OrientationZ (ROB.ET\_RobotComponent.OrientationZ) is then mapped to AxisD.

NOTE: Use [ET\_RobotPSeriesComponent.RotationalAxis](D-SE-0074967.html) if you want to call the methods SetControlLoopParameter, GetControlLoopParameter or Calibration.

For further information, refer to the methods *[ROB.IF\_RobotConfiguration.AddDrive](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=6B565F46)* (see Robotic, Library Guide), *[ROB.IF\_RobotConfiguration.MapComponentToDrive](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=6C61B078)* (see Robotic, Library Guide), *[SER.IF\_RobotPSeries.SetControlLoopParameter](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRobotics&topicID=D_SE_0096805)* (see SchneiderElectricRobotics, Library Guide)*, [SER.IF\_RobotPSeries.GetControlLoopParameter](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRobotics&topicID=D_SE_0096802)* (see SchneiderElectricRobotics, Library Guide), and *[SER.IF\_RobotPSeriesCalibration](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRobotics&topicID=D_SE_0075220)* (see SchneiderElectricRobotics, Library Guide)
