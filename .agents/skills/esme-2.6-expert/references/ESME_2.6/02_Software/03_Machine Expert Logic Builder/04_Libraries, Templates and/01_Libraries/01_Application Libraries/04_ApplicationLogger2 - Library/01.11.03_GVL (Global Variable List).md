# GVL (Global Variable List)

## Overview

|  |  |
| --- | --- |
| Type: | Global variables |
| Available as of: | V1.0.0.0 |

## Description

GVL (**G**lobal **V**ariable **L**ist) of the **ApplicationLogger2** library.

## Global Variables

| Variable | Data type | Description |
| --- | --- | --- |
| G\_ifApplicationLogger | IF\_ApplicationLogger2 | With this global interface it is possible to enable the Application Logger (method RegisterCommunicationService) and to control the Application Logger (see [IF\_ApplicationLogger2](IF_ApplicationLogger2-37D6A572.html#IF_ApplicationLogger2-37D6A572)).  This interface serves as a root point of the logger points. The name of this logger point is set to Application Logger.  NOTE: Do not modify the value of this variable. |
