Task

Convert planar polar coordinates into 2 dimensional Cartesian coordinates.

Description

The polar coordinates are converted to cartesian coordinates. In doing so, it is assumed that the pole of the polar coordinate system corresponds to the origin of the cartesian coordinate system and that the horizontal axis of the polar coordinate system corresponds to the X axis of the cartesian coordinate system.
