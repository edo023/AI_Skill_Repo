|  |  |
| --- | --- |

## Description

The structure ST\_Recipe is a collection of tags providing recipe information

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| ID | DINT | ID of ST\_Recipe |
| Name | STRING | Name of ST\_Recipe |
| Unit | STRING[6] | Unit of ST\_Recipe |
| PrimaryQty | REAL | Primary quantity value of the recipe |
| ProcessVariables | ST\_ProcessVariables | Process variables for the recipe |
| Ingredients | ST\_Ingredients | Ingredient information for the recipe |
