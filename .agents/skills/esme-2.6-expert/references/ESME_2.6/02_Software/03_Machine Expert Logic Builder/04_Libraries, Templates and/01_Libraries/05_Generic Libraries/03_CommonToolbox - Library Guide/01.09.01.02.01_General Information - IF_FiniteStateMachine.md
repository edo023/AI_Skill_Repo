# General Information - IF\_FiniteStateMachine

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.3.0.0 |

## Description

The interface IF\_FiniteStateMachine is implemented by the function block FB\_FiniteStateMachine.

It provides the methods and properties to control the function block.

## Methods

| Name | Description |
| --- | --- |
| Reset | Resets the finite state machine.  Refer to [Reset (Method)](FB_FiniteStateMachineReset-4CEAE574.html). |
| SetNextState | Requests the transition to a different state.  Refer to [SetNextState (Method)](FB_FiniteStateMachineSetNextState-4CEC64C4.html). |
| Update | Performs the requested transition from the present state to the next state.  Refer to [Update (Method)](FB_FiniteStateMachineUpdate-4CEF552C.html). |
| AplRegisterLoggerPoint | Registers a logger point into the Application Logger.  Refer to [AplRegisterLoggerPoint (Method)](FB_FiniteStateMachineAplRegisterLoggerPoint-4CF37F2D.html). |
| AddStateConversion | Adds a custom string conversion for the state values to the finite state machine.  Refer to [AddStateConversion (Method)](FB_FiniteStateMachineAddStateConversion-4CF5D372.html). |
| AddTransitionLogger | Adds a transition logger to the finite state machine.  Refer to [AddTransitionLogger (Method)](FB_FiniteStateMachineAddTransitionLogger-4CF7AD91.html). |

## Properties

Refer to the properties described under [Properties](FB_FiniteStateMachineGenInfo-4CE32C56.html#FB_FiniteStateMachineGenInfo-4CE32C56__Properties-4CE47E5C).
