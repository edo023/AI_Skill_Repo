ET\_MultiCamCsModeMaster - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Description

Enumeration type for the cold start mode of the [FB\_MultiCam](../Function_Blocks_I_to_Q/Function_Blocks_I_to_Q-14.htm#XREF_D_SE_0087313_1) master.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| NoChangeOfMasterPosition | 0 | Do not manipulate the master position. |
| SetMasterPositionToFirstCamPosition | 1 | Set master position to 1st cam point |
| SetMasterPositionToMinusInfinite | 2 | Set master position to a very high negative value. |
| SetMasterPositionToExtCsMasterPosition | 3 | Set master position to lrCsMasterPosition from [ST\_MultiCamExt](../Structures/Structures-33.htm#XREF_D_SE_0087772_1). |
