# FB\_MultiCombine - StartChannelBundling (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | 1.0.0.0 |

## Description

The method StartChannelBundling of the function block [FB\_MultiCombine](FB_MultiCombine-58411E70.html) is used to consolidate movement into a single axis.

When this method is called, the function block captures the velocity and acceleration values of the subordinate axis and applies them to a new motion job on the specified channel.

Motion jobs on the other channels in the group are aborted, and those axes transition to the operating state Standstill.

A new motion job can be initiated on any channel in the next RTP cycle.

If MC\_MoveSuperimposed  is active on the subordinate axis or any channel at the moment of bundling, the superimposed movement is aborted.
