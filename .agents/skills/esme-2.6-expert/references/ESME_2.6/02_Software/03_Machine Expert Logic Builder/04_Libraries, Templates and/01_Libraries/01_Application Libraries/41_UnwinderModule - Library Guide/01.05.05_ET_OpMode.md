# ET_OpMode

![](images/logo.png)

ET\_OpMode

ET\_OpMode - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

Description

The enumeration describes all the operating modes of the FB\_UnwinderModule.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| NoOp | 0 | No operating mode is selected. |
| Manual | 20 | Touch control mode. |
| Automatic | 30 | Unwinding operating mode. |
| BrakeRelease | 70 | Release the brake of the axis. |
