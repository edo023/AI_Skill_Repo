## Task

Setting the ramp.

## Description

With the method SetRamp(...), the ramp (change of the acceleration/deceleration) with which the path motion of the robot has to be executed can be determined.

[Detailed description of the motion parameter Ramp](D-SE-0075796.html#D-SE-0075796).

A modified value of the ramp is active for the motion commands sent after calling this method.
