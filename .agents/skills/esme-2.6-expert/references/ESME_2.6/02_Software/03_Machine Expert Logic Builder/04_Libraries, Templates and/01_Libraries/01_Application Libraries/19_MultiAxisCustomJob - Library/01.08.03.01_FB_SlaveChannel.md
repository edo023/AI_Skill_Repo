# FB\_SlaveChannel

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Implements: | IF\_SlaveChannel, IF\_SlaveChannelInternal |
| Available as of: | 1.0.0.0 |

## Description

The function block FB\_SlaveChannel serves as the link to a subordinate axis within a multi-axis group. It provides the mechanism to monitor the operating state of the axis and to apply the target values (position, velocity, and acceleration) calculated by the user-defined motion algorithm.

Each subordinate axis to be controlled by FB\_MultiAxisCustomJob must be associated with an instance of FB\_SlaveChannel. During execution, the user-defined algorithm uses this function block to write movement values to the axis in each RTP cycle.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
