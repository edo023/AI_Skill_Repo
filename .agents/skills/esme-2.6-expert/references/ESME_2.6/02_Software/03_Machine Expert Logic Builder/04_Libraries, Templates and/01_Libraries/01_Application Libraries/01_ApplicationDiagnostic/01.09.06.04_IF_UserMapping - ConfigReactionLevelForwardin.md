## Task

Method to process and/or modify the reaction that is forwarded to a child of the diagnostic handler.

## Description

The interface method ConfigReactionLevelForwarding is called by the instance of the FB\_DiagnosticHandler in the event a reaction is forwarded to a child of the diagnostic handler.

All information about the diagnostic message that triggered the reaction is provided with the method parameters and can be processed within the method. In addition, it is possible to modify the forwarded reaction level.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_uiOwnReactionLevel | UINT | Reaction level on the local diagnostic handler. |
| i\_stIncomingDiagMsg | [ST\_DiagMessageHandler](ST_DiagMessageHandler-37410F11.html) | Diagnostic message triggering the reaction. |
| i\_ifTargetChild | IF\_DiagHandlerIdentification | Target child to which the reaction is forwarded. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_uiForwardedReactionLevel | UINT | Indicates the reaction level that is forwarded to the target child. It indicates the proposed value according to the mapping table.  You can modify this value to meet your requirements. |
