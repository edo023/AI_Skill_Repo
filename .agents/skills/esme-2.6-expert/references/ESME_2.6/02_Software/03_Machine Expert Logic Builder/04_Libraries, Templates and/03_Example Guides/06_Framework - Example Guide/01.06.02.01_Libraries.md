# Libraries

## Overview

The following Schneider Electric libraries are added to the Library Manager with the Framework example project:

| Library | Function |
| --- | --- |
| ApplicationDiagnostic | The library provides the necessary functions to implement end-to-end diagnostics across multiple units or modules, including reaction-management, in your application.  For more information, refer to the [ApplicationDiagnostic Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=AppDiag&topicID=ApplicationDiagnostic_36B56E5B). |
| ApplicationFrameworkUtility | The library offers a comprehensive set of tools and functions to streamline application development within a framework environment.  For more information, refer to the [ApplicationFrameworkUtility Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=AFULib&topicID=ApplicationFrameworkUtility_3846E203). |
| ApplicationLogger2 | The library supports the functionalities to establish a logging for the application program.  For more information, refer to the [ApplicationLogger2 Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=APL2LG&topicID=). |
| CommonToolbox | The library has been designed:   * To be used on the controller platforms M••• and LMC•••. * To support functions and function blocks such as mathematical functions or function blocks.   For more information, refer to the [CommonToolbox Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=comtb&topicID=D_SE_0105154). |
| MotionFunctionBlocks | This library provides functionalities to facilitate the implementation of motion control in your application.  For more information, refer to the [MotionFunctionBlocks Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=MoFBLib&topicID=MotionFunctionBlocks_Library_3F055507). |
| PackML | This library provides components to facilitate the implementation of an application based on the PackML standard.  For more information, refer to the [PackML Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PackMLli&topicID=D_SE_0077921). |
| SercosDeviceInterface | User function interface for Sercos device parameters, such as SercosIdentification (reserved for internal use only). |
| SercosMaster (exclusive to M262) | This library provides elements to connect to the Sercos. It is exclusive to M262 Logic/Motion Controllers.  For more information, refer to the [SercosMaster Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=SerMaLG&topicID=D_SE_0096983). |
