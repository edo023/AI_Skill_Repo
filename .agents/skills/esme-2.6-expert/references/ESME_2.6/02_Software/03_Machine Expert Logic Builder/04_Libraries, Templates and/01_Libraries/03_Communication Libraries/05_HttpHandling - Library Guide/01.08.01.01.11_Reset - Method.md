# Reset - Method

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Functional Description

The method Reset is used to leave the state Error. Depending on the state where the error was detected, the succeeding state after calling Reset is Connected or Disconnecting and finally Idle.

## State Transition of the Client

| Stage | Description |
| --- | --- |
| 1 | Initial state: Error |
| 2 | Function call |
| 3 | Depending on the state where the error was detected, either:   * State = Connected * State = Disconnecting -> Final state: Idle. It stays in Error if an error is detected again. |
