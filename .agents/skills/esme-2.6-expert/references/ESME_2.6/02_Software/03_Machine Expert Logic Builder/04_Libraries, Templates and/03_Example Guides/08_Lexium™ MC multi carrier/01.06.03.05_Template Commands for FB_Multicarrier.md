# Template Commands for FB\_Multicarrier

## Overview

The function block FB\_Multicarrier is not a standard template module. Therefore, the subroutines SR\_MulticarrierModule1 and SR\_MulticarrierModule2 each provide a state machine to provide the standard template command handling for this function block. Since the SR\_MulticarrierModule1 and SR\_MulticarrierModule2 architectures are identical with the SR\_MulticarrierModule of the Lexium™ MC multi carrier, refer to the [Template Commands for FB\_Multicarrierchapter of the Lexium™ MC multi carrier Example Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=exMulCar&topicID=TemplCMDs_2C148C1B) for further information.
