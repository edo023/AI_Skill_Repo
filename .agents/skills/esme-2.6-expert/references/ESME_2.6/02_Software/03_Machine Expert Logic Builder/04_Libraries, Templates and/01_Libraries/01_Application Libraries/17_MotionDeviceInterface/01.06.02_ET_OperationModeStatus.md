# ET\_OperationModeStatus

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | UDINT |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_OperationModeStatus of the library MotionDeviceInterface contains the active operating mode of a drive.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Position | 0 | The active operating mode of the drive is Cyclic Synchronous Position. |
| Velocity | 1 | The active operating mode of the drive is Cyclic Synchronous Velocity. |
| Torque | 2 | The active operating mode of the drive is Cyclic Synchronous Torque. |
| NotAvailable | 3 | The active operating mode of the drive cannot be determined. Possible reasons:   * The active operating mode is not a cyclic synchronous operating mode (for example, Homing). * The drive has detected an error. * The drive is disabled. * The drive is not in Sercos communication phase CP 4. |
