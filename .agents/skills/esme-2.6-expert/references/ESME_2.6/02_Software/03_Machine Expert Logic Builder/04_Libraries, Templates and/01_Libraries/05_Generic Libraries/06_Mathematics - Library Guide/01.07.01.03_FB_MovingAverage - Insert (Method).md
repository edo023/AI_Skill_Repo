# FB\_MovingAverage - Insert (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.1.0.0 |

## Description

This method inserts a value for the calculation of the moving average value into the buffer of the function block FB\_MovingAverage.
