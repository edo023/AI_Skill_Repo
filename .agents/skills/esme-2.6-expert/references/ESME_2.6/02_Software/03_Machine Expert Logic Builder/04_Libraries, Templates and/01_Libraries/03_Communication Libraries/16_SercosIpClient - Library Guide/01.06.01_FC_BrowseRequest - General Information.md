# FC\_BrowseRequest - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.2.0 |

## Task

This function sends the request for the Browse service. The Browse service is used to detect Sercos devices within the same subnet.

## Functional Description

This function sends the request for the Browse service to detect Sercos devices within the same subnet. Only devices with a Sercos address between the lower and the upper address range are to respond. If both values are 0, all devices are to respond.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_uiLowerSercosAddress | UINT | Lower value of address range of Sercos devices to respond. If both the lower and the upper value are 0, all devices are to respond. |
| i\_uiUpperSercosAddress | UINT | Upper value of address range of Sercos devices to respond. If both the lower and the upper value are 0, all devices are to respond. |
| i\_bMasterOnly | BOOL | TRUE: Only devices which have a Sercos master interface are to respond. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_udiTransactionID | UDINT | Returns the transaction ID to be used for receiving the response |

## Return Value

| Data type | Description |
| --- | --- |
| ET\_Result | Refer to [ET\_Result](ET_Result-GeneralInformation-08184BB1.html). |
