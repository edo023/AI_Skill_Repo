# FC\_Vector3DNormalize

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

Given a 3D input vector, the function returns the normalized vector that is a vector with the same direction of the input vector but with Euclidean norm equal to 1.0.

| Input | Data type | Description |
| --- | --- | --- |
