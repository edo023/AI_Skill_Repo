# IF\_MotionValues – General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.1.0 |
| Inherits from: | — |

## Description

The interface is used to get the motion values of an axis.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| lrPosition | lreal | Get | Position of the axis in user-defined units. |
| lrVelocity | lreal | Get | Velocity of the axis in units/s. |
| lrAcceleration | lreal | Get | Acceleration of the axis in units/s2. |
