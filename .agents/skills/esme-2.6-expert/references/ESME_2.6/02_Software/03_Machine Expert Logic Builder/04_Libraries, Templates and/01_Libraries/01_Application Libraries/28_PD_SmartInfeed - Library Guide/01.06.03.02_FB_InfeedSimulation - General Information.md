# FB_InfeedSimulation - General Information

![](images/logo.png)

FB\_InfeedSimulation - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Simulation of a product flow for the [FB\_Infeed](Function_Blocks-3.htm#XREF_D_SE_0080279_1) POU

Description

The POU generates virtual products for the [FB\_Infeed](Function_Blocks-3.htm#XREF_D_SE_0080279_1) and calculates the motion of the products on the basis of the belt motions. The virtual product flow is output in a visualization and the Touchprobe signals for the [FB\_Infeed](Function_Blocks-3.htm#XREF_D_SE_0080279_1) POU are simulated according to this virtual product flow.

The simulation of the Touchprobe signals is updated in the pattern of the Sercos task.
