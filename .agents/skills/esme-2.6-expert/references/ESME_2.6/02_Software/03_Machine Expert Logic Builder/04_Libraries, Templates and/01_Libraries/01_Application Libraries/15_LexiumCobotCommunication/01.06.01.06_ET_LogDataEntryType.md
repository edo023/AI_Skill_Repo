# ET\_LogDataEntryType

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

This enumeration type is used to address the type of a log data entry provided by the Lexium Cobot controller.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No log data entry type is addressed. |
| Error | 1 | The log data entry type Error is addressed. |
| Status | 2 | The log data entry type Status is addressed. |
