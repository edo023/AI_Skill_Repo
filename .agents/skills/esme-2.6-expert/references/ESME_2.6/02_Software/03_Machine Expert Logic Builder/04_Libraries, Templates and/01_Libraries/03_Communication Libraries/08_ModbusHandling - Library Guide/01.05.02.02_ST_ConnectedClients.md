# ST\_ConnectedClients

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.1.0.0 |
| Extends: | - |

## Description

The structure ST\_ConnectedClients provides information about a client connection on the Modbus TCP server.

## Structure Elements

| Name | Data type | Default value | Description |
| --- | --- | --- | --- |
| sIpAddress | STRING[15] | NULL | Specifies the IP address of the client connection. |
| uiTcpPort | UINT | 0 | Specifies the TCP port of the client connection. |
