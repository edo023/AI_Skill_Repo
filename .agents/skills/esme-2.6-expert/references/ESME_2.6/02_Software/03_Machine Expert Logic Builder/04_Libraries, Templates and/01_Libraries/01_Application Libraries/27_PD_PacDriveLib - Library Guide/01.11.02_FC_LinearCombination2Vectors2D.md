| Available as of: | V1.0.3.0 |

Task

Calculates the linear combination of two vectors in the 2-dimensional space.

Description

Calculates the following linear combination: i\_lrK1 \* i\_stV1 + i\_lrK2 \* i\_stV2

Interface

| Input | Data type | Description |
| --- | --- | --- |

| i\_stV1 | [ST\_Vector2D](../Structures/Structures-47.htm#XREF_D_SE_0087800_1) | Vector 1 |
| i\_lrK2 | LREAL | Scalar 2 |
| i\_stV2 | [ST\_Vector2D](../Structures/Structures-47.htm#XREF_D_SE_0087800_1) | Vector 2 |
