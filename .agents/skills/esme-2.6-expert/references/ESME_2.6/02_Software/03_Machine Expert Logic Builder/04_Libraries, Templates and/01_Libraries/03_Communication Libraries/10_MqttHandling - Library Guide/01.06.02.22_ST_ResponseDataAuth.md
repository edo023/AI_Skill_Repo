## Description

The structure ST\_ResponseDataAuth contains information extracted from the response sent by the MQTT broker in an AUTH message.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| etReasonCode | [ET\_ReasonCode](ET_ReasonCode-D887BF12.html) | Indicates the reason code in the AUTH message sent by the MQTT broker. |
| stProperties | [ST\_PropertiesPuback](ST_PropertiesPuback-D8F264DF.html) | Indicates the properties of the AUTH message sent by the MQTT broker. |
