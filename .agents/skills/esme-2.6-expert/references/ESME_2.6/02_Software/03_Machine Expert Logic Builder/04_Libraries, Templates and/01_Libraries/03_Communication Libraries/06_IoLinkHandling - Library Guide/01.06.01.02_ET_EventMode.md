# ET\_EventMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_EventMode contains the modes of the event.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| ReservedModeBit | 0 | Reserved |
| OneTimeEvent | 1 | One-Time event |
| EventNoLongerReported | 2 | Event no longer reported (for example, voltage OK again) |
| EventReported | 3 | Event reported (for example, insufficient voltage) |
