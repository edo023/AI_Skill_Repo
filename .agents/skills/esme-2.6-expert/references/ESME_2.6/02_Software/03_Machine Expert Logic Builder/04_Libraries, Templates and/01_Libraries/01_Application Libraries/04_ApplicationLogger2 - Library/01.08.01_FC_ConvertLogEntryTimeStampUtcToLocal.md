# FC\_ConvertLogEntryTimeStampUtcToLocal

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

This function converts the UTC (Universal Coordinated Time) timestamp of a log entry into a timestamp with the local time zone of the controller.

The timestamp of the ST\_LogEntry that is retrieved using the method GetLogEntry of the GVL.F\_ifApplicationLogger or FB\_ApplicationLoggerHandler is provided as UTC.

## Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stLogEntryUTC | [ST\_LogEntry](ST_LogEntry-37629944.html#ST_LogEntry-37629944) | Log entry with UTC timestamp (read out using the "GetLogEntry" method, for example) |
| iq\_stLogEntryLocal | [ST\_LogEntry](ST_LogEntry-37629944.html#ST_LogEntry-37629944) | Log entry with timestamp based on the local time zone of the controller. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-375A58EE.html#ET_Result-375A58EE) | Provides diagnostic and status information as a numerical value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
| q\_diOffset | DINT | Time difference between UTC and the local time of the controller, in seconds.  Local time of the controller = UTC + value of q\_diOffset |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: Conversion successful. |
