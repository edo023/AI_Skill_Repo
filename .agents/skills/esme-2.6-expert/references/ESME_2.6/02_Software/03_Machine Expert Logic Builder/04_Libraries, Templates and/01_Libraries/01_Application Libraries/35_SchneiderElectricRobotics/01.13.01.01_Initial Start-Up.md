# Initial Start-Up

## Overview

This chapter explains the steps to operate a Lexium P Robot in the program.

The following libraries are required:

* SchneiderElectricRobotics
* SchneiderElectricRobotics Parameters
* Robotic or RoboticModule

## Instances

Every Lexium P Robot requires its own instance of SER.FB\_RobotPSeries. (for example fbP4S0R : SER.FB\_RobotPSeries) and its instance of ROB.FB\_Robot (for example fbRobot : ROB.FB\_Robot) or an instance of the RM.FB\_RoboticModule.

## Initialize

Initialize the Lexium P Robot by calling the method [fbP4S0R.InitializeRobot()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRoboticsParameters&topicID=D_SE_0074974).

If the output [q\_xRotationalAxisRequired](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRoboticsParameters&topicID=D_SE_0074974) is TRUE, then the robot you selected has a rotational axis and you have to call [fbP4S0R.InitializeRotationalAxis()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRoboticsParameters&topicID=D_SE_0074975).

Connect the FB\_RoboticPSeries to the FB\_Robot by configuring the FB\_Robot. Call [fbRobot.ifConfiguration.SchneiderElectricRobot()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=D_SE_0075519) for the RoboticModule and accordingly [stRobotInterface.iq\_ifConfiguration.SchneiderElectricRobot()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.RoboticModule&topicID=D_SE_0076928). At the input i\_ifRobot, assign fbP4S0R.ifSchneiderElectricRobot.

The configuration is completed by calling the methods SetEmergencyParameter() and ConfigDone().

## Additional Configuration

The Lexium P Robot can be adapted with several methods:

* [OptionalEquipment()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRoboticsParameters&topicID=D_SE_0074976)
* SetKinematicParameter()
* SetControlLoopParameter()

## Call SER.SR\_SlowCyclicExecution

Call the program SER.SR\_SlowCyclicExecution in a task with task interval 10 ms.

The program has to be called only once, even if several robots of SchneiderElectricRobotics are used.

## Code

```
// Get interface instance
ifConfiguration := fbRobot.ifConfiguration;
// ifConfiguration := stRobotInterface.iq_ifConfiguration;
// Initialize P-Series
fbPSeries.InitializeRobot(i_etRobotType := SERP.ET_RobotPSeries.VR,
                              i_ifDriveA := DRV_RobotDriveA,
                              i_ifDriveB := DRV_RobotDriveB,
                              i_ifDriveC := DRV_RobotDriveC,
                              q_etDiag => etDiag,
                              q_etDiagExt => etDiagExt,
                              q_sMsg => sMsg,
                              q_xRotationalAxisRequired => xRotationalAxisRequired);
IF xRotationalAxisRequired THEN
   fbPSeries.InitializeRotationalAxis(i_ifDrive := DRV_RobotDriveRotAx,
                              i_lrPeriodStart := lrPeriodStart,
                              i_lrPeriodLength := lrPeriodLength,
                              q_etDiag => etDiag,
                              q_etDiagExt => etDiagExt,
                              q_sMsg => sMsg);
(*
//Alternative instead of InitializeRotationalAxis use mapping to OrientationZ 
   fbPSeries.InitializeOrientationZ(i_ifDrive := DRV_RobotDriveRotAx,
                              q_etDiag => etDiag,
                              q_etDiagExt => etDiagExt,
                              q_sMsg => sMsg); 
*)
```

```
END_IF

// Configure P-Series
ifConfiguration.SchneiderElectricRobot( i_ifRobot := fbPSeries.ifSchneiderElectricRobot,
                                       q_etDiag => etDiag,
                                       q_etDiagExt => etDiagExtSer,
                                       q_sMsg => sMsg);

// Call configuration method for the emergency parameter
ifConfiguration.SetEmergencyParameter( i_lrMaxDeceleration := 10000.0,
                                       i_lrRamp := 25.0,
                                       q_etDiag => etDiag,
                                       q_etDiagExt => etDiagExt,
                                       q_sMsg => sMsg);

// Sign config done
ifConfiguration.ConfigDone(q_etDiag => etDiag,
                           q_etDiagExt => etDiagExt,
                           q_sMsg => sMsg);
```
