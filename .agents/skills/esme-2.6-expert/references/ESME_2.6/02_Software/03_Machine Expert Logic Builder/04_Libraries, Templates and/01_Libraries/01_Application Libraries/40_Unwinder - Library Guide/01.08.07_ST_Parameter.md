# ST_Parameter

![](images/logo.png)

ST\_Parameter

ST\_Parameter - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

Description

This structure contains the parameters and feedback data for the FB\_Unwinder function block.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| stRadiusDetermination | ST\_RadiusDetermination | Parameter and feedback data for the radius measurement that takes place during the job. |
| stFeedbackSignal | ST\_FeedBackSignal | Parameter and feedback data for the feedback signal of the control loop. |
| stPreController | ST\_Precontrol | Parameter and feedback data for the precontrol movement of the bobbin, according to the movement of the feeder. |
| stController | ST\_Controller | Parameter and feedback data for the controller movement of the bobbin, according to the feedback signal of the sensor. |
| stStartMovement | ST\_StartMovement | Parameter and feedback data for the bobbin movement at the beginning of the job, until the precontroller and controller take over the control of the bobbin. |
| i\_lrStopDeceleration | LREAL | Deceleration of a stop movement if i\_xEnable goes FALSE during the movement of the bobbin. |
| i\_lrStopJerk | LREAL | Jerk of a stop movement if i\_xEnable goes FALSE during the movement of the bobbin. |
| q\_lrBobbinPosition | LREAL | Actual position of the bobbin. |
| q\_lrFeederPosition | LREAL | Actual position of the master. |
