## Description

The structure ST\_Dimensions contains parameters used for specifying the dimensions of a product or a tool.

## Structure Elements

| Input | Data type | Unit | Description |
| --- | --- | --- | --- |
| lrLength | LREAL | mm | The specified length of the product or tool. |
| lrWidth | LREAL | mm | The specified width of the product or tool. |
| lrHeight | LREAL | mm | The specified height of the product or tool. |
