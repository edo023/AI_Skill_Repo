# ET\_WarmStartProductHandling - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

## Description

This enumeration contains the ways to handle a recognized product during the warm start.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Remove | 1 | If a warm start is executed, then the products that are between the train and the sensor have to be removed. The MultiBelt discards the last start signal. |
| PutIn | 2 | If a warm start is executed, then the products that are between the train and the sensor have to be placed in the train manually. The MultiBelt performs this step during the warm start. |
