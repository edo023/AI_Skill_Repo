# ET_Cmd

![](images/logo.png)

ET\_Cmd

ET\_Cmd - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.2.0 |
| Versions: | Current version |

Description

The enumeration describes all commands of the [FB\_AxisModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0077136_1).

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Homing | 10 | Change to operating mode Homing. |
| Manual | 20 | Change to operating mode Manual. |
| CamCs | 30 | Change to operating mode Cam and prepare a cold start. |
| CamWs | 31 | Change to operating mode Cam and prepare a warm start. |
| Endless | 40 | Change to operating mode Endless. |
| EndlessIls | 41 | Change to operating mode Endlesslls. |
| Positioning | 50 | Change to operating mode Positioning. |
| BrakeRelease | 70 | Change to operating mode Brake. |
| AdditionalCs | 80 | Change to operating mode Additional and prepare a cold start. |
| AdditionalWs | 81 | Change to operating mode Additional and prepare a warm start. |
| Start | 100 | Start the currently selected operating mode. |
| StartTrig | 101 | Trigger a start for the current operating mode and delete the start flag. |
| StartTrigWaitInPos | 102 | Trigger a start in the operating mode Pos. The command is only signaled as successful after the position has been reached. |
| Stop | 110 | Cancel the current command and return to the operating mode NoOp. |
| Hold | 120 | Cancel a current start command. |
| WsStart | 150 | Trigger a warm start in the operating mode Cam or Additional. |
| DriveEnableOn | 130 | Enable the position control of the axis. |
| DriveEnableOff | 140 | Disable the position control of the axis. |
