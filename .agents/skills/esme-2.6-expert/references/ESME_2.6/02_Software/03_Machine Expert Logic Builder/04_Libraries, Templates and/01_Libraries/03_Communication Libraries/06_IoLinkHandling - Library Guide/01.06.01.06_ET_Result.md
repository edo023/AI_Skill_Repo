# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result values indicate the result of operations executed by the function blocks of this library.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Ok | 0 | Ok |
| ChannelNumberOutOfRange | 2 | The channel number is out of range. The channel number must be between 1 and 4. |
| InvalidInput | 3 | The specified input is invalid. Refer to result message for detailed information. |
| RequestFailed | 4 | Sending a request is not successful. |
| ReadOrWriteNotPossible | 5 | The requested operation is not possible. |
| Timeout | 6 | A timeout occurred. Verify the communication connections, and if necessary, increase the timeout value. |
| InternalError | 7 | An internal error is detected. Refer to result message for detailed information. |
| BufferSizeTooSmall | 8 | The received data are bigger than the provided application buffer. |
| BufferSizeTooBig | 9 | The data to send cannot be bigger than 232 bytes. |
| ReadInformationCorrupted | 10 | The information received is invalid. |
| NotSupported | 11 | The feature is not supported. |
| Disabled | 100 | The function block is disabled. |
| Running | 101 | The function block is running. |
| EventCallbackNotRegistered | 400 | The event callback could not be registered. |
