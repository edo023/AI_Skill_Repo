# ST\_DeclampingStation

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.3.0.0 |
| Inherits from: | - |

## Description

The structure ST\_DeclampingStation is used as input/output of the visualization frame [FR\_DeclampingStation](FRDeclampStation-E1CBAD12.html).

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| diTimer | DINT | The variable diTimer is used for auto-triggering the station. It contains information about the time that has elapsed since the station is in a state for enabling the triggering of a process. |
| diTimeState | DINT | The variable diTimeState is used for auto-triggering the station. It contains information about the trigger state and determines which action is executed, based on the timing defined in diTimer. |
| xAutoTrigger | BOOL | If xAutoTrigger is set to TRUE, the station automatically triggers the next process when the specified process time has elapsed. If xAutoTrigger is set to FALSE, you must activate the station manually. |
| xDriveStationEmpty | BOOL | If xDriveStationEmpty is set to TRUE, the carriers are moved out of the station. |
| xHoldProcess | BOOL | If xHoldProcess is set to TRUE, the station is put on hold. |
| xRestart | BOOL | If xRestart is set to TRUE, the station is restarted. |
| xStartDeclamping | BOOL | If xStartDeclamping is set to TRUE, the declamping process is triggered. When the carriers are moved out of the station, xStartDeclamping is set to FALSE. |
| xStartMoveOutFromDeclamping | BOOL | If xStartMoveOutFromDeclamping is set to TRUE, the carriers are moved out of the station. When the carriers have left the station, xStartMoveOutFromDeclamping is set to FALSE. |
| xStopProcess | BOOL | If xStopProcess is set to TRUE, the station is stopped. |
