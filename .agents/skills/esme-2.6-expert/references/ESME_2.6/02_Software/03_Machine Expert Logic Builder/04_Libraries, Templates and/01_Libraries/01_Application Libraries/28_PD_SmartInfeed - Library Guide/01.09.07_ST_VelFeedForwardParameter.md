| Available as of: | V1.0.0.0 |
| Inherits from: | - |
| Versions: | Current version |

Description

The structure contains the parameters of the [FB\_MasterVelFeedForward](../Function_Blocks/Function_Blocks-10.htm#XREF_D_SE_0080286_1) POU.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| udiBeltId | UDINT | The number of the feed belt from which the accepted products per minute are to be determined |
| timAverageTime | TIME | The period of time over which the measured values for [lrPartsPerMinuteInfeed](Structures-6.htm#XREF_D_SE_0080364_1) and [lrTargetsPerMinuteOutfeed](Structures-6.htm#XREF_D_SE_0080364_1) are to be averaged |
| timRefreshTime | TIME | The interval at which the values for [lrPartsPerMinuteInfeed](Structures-6.htm#XREF_D_SE_0080364_1) and [lrTargetsPerMinuteOutfeed](Structures-6.htm#XREF_D_SE_0080364_1) are to be updated |
| lrAverageProductDistAtStart | LREAL | The mean product distance that is to be assumed as long as no measured values are available after the start |
| lrAverageTargetDistAtStart | LREAL | The mean target difference that is to be assumed as long as no measured values are available after the start |
