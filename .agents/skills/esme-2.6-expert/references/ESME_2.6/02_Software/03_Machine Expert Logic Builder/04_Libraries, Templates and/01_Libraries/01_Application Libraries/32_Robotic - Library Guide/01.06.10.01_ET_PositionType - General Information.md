# ET\_PositionType - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V3.3.0.0 |

## Description

Position types of the robot.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No position type |
| Start | 1 | Value specifies the start position of a robot movement. |
| Stop | 2 | Value specifies the stop position of a robot movement. |
| Target | 3 | Value specifies the target position of a robot movement. |
| SplinePoint | 4 | Value specifies a spline point of a robot spline movement. |
| Circular | 5 | Value specifies the circular point of a robot circular movement. |
| RefPositionSpace | 6 | Value specifies the reference position of a robot movement in space. |
| SpaceTarget | 7 | Value specifies the target position of a robot movement in space. |
