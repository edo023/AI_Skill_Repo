# SendData (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method is used to initiate the transmission of the specified data to the connected server. The return value of type ET\_Result is used to indicate the execution result of the method.

## Inputs

| Parameter | Data Type | Description |
| --- | --- | --- |
| i\_pbyData | POINTER TO BYTE | Reference to the data to be transmitted |
| i\_udiDataSize | UDINT | Size of the data to be sent in bytes. |
| i\_ifStatus | [IF\_CallbackStatusSend](IF_CallbackStatusSend-GeneralInform-840E6DC2.html#IF_CallbackStatusSend-GeneralInform-840E6DC2) | Interface to retrieve the status of the data transmission. |
