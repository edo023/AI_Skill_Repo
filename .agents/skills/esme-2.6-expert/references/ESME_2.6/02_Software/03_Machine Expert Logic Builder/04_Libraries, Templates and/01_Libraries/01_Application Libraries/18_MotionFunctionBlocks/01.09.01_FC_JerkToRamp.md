# FC\_JerkToRamp

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.1.0 |
| Version: | Installed version |

## Task

Conversion of a value from jerk to ramp.

## Description

The jerk, that is the change in acceleration over time, is converted to ramp.

The unit ramp is defined as follows:

* Ramp = 100: The acceleration changes by i\_lrAccBase within 1.0 second.
* Ramp = 50: The acceleration changes by i\_lrAccBase within 0.5 second.
* Ramp = 150: The acceleration changes by i\_lrAccBase within 1.5 seconds.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_IrJerk | LREAL | Jerk (change in acceleration over time) in units/s3 |
| i\_IrAccBase | LREAL | Change in acceleration used as basis for calculation.  NOTE: If i\_lrAccBase = 0, ten times the gravitational acceleration is used as basis for calculation. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | The output is set to TRUE if an error has been detected during the execution. |
| q\_etResult | ET\_Result | POU-specific output on the diagnostic; q\_xError = FALSE -> Status message; q\_xError = TRUE -> Diagnostic message. |
| q\_sResultMsg | STRING[80] | Event-triggered message that gives additional information on the diagnostic state. |

## Return Value

| Data type | Description |
| --- | --- |
| LREAL | Value of the input i\_lrJerk in ramp. |
