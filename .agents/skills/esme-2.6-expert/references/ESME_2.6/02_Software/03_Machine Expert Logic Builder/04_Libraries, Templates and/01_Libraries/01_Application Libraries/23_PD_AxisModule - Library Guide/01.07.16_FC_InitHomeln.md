Task

Function for initialization of an axis that is controlled by the [FB\_AxisModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0077136_1) function block.

Description

This function is used to specify that an axis, when homed, will be homed by seeking an input or sensor signal. The i\_etMode input specifies the homing direction:

| i\_etHomingMode | Home Action |
| --- | --- |
| PDL.ET\_HomeMode.PosDirectionPosEdgeSensor | Move in the positive direction until the rising edge is detected. |
| PDL.ET\_HomeMode.NegDirectionPosEdgeSensor | Move in the negative direction until the rising edge is detected. |
| PDL.ET\_HomeMode.NegDirectionNegEdgeSensor | Move in the negative direction until the falling edge is detected. |
| PDL.ET\_HomeMode.PosDirectionNegEdgeSensor | Move in the positive direction until the falling edge is detected. |

All the possible values of i\_etMode are enumerated in PDL.ET\_HomeMode; however, the function returns FALSE if none of the above four values is used.

The i\_xRotativeSystem input specifies how the axis will search for the boolean signal if the sensor is activated when the homing starts. An example for the PosDirectionPosEdgeSensor mode: If i\_xRotativeSystem = FALSE, the axis will move in the negative direction until the input turns off (falling edge). The axis then reverses direction and begins a new. If i\_xRotativeSystem = TRUE, the axis will move in the positive direction until the rising edge of the input is found.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etMode | PDL.ET\_HomeMode | Specifies the specific type of homing to perform |
| i\_lrPosition | LREAL | Specifies the axis’s position after the homing process is complete in units |
| i\_lrVelocity | LREAL | Specifies the homing velocity unless reduced velocity is enabled in units/sec |
| i\_lrAcceleration | LREAL | Specifies the homing acceleration in units/sec^2 |
| i\_lrDeceleration | LREAL | Specifies the homing deceleration in units/sec^2 |
| i\_lrJerk | LREAL | Specifies the jerk for homing in units/sec^3 |
| i\_lrOffset | LREAL | Specifies the distance in units the axis will travel after sensing the input. The position of the axis after this move is set to the value of i\_lrPosition |
| i\_lrMaxTravel | LREAL | Specifies the maximum distance in units the axis will move when searching for the input. An error is generated if the axis can’t find the input. The direction is determined by the specific homing mode |
| i\_xRotativeSystem | BOOL | Specifies how the axis will seek the input if the input is on when homing starts |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stAxisModuleItf |  | The axis interface structure of the associated axis |
