# IF\_CarrierFeedbackMotionValuesChannelB

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.1.9.0 |
| Inherits from | – |

## Task

Interface providing reference motion parameters from superimposed carrier movements (channel B).

## Description

The feedback interface provides status values from the reference motion parameters for the superimposed part of a carrier movement on channel B.

## Properties

| Name | Data type | Unit | Value range | Accessing | Description |
| --- | --- | --- | --- | --- | --- |
| lrRefAcceleration | LREAL | mm/s2 | – | Read | Indicates the reference acceleration for the superimposed part of a carrier movement on channel B. |
| lrRefPosition | LREAL | mm | – | Read | Indicates the reference position for the superimposed part of a carrier movement on channel B. |
| lrRefTrackposition | LREAL | mm | 0.0 ≤ lrRefTrackposition ≤ lrTrackLength(1) | Read | Indicates the reference track position for the superimposed part of a carrier movement on channel B. |
| lrRefVelocity | LREAL | mm/s | – | Read | Indicates the reference velocity for the superimposed part of a carrier movement on channel B. |
| **(1)** For more information on the track length, refer to [lrTrackLength](FeedbConfig-D619B88F.html#FeedbConfig-D619B88F). | | | | | |
