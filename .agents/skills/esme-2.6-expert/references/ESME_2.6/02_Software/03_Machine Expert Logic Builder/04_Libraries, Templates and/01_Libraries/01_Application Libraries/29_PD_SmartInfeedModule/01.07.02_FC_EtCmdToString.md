# FC_EtCmdToString

![](images/logo.png)

FC\_EtCmdToString

FC\_EtCmdToString - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Convert the commands of the [FB\_SmartInfeedModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080825_1) to a string.

Description

The command of the SmartInfeed module at the i\_etCmd input from the [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) enumeration list is returned as a string.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etCmd | [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) | The command to be converted to a string |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) | General, library-independent statement on the diagnostic.  A value unequal GD.ET\_Diag.Ok equates an error message. |
| q\_etDiagExt | [ET\_DiagExt](../Enumerations/Enumerations-3.htm#XREF_D_SE_0080817_1) | POU-specific output on the diagnostic  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> error message |

Return Value

| Data type | Description |
| --- | --- |
| STRING[20] | The command as a string |

Diagnostic Messages

| q\_etDiag | q\_etDiagExt | Enumeration value | Description |
| --- | --- | --- | --- |
| OK | [Ok](#XREF_D_SE_0080835_9) | 0 | Ok |
| InputParameterInvalid | [CmdUnknown](#XREF_D_SE_0080835_8) | 23 | Indeterminable command. |

Ok
