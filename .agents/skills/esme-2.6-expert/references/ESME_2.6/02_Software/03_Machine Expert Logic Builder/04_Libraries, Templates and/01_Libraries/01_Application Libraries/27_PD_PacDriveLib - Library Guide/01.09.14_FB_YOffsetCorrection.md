Task

A position correction (e.g. print marker) should be carried out on one axis.

Description

The function block carries out a correction using the YOffset Generator (positioning) which has the beginning XStart and the duration XFactor, in conjunction with a master encoder Encoder.

![G-SE-0068654.1.gif-high.gif](images/G-SE-0068654.1.gif-high.gif)

NOTE: The positioning's velocity, acceleration and deceleration are determined in such a way that the correction is completed within the master encoder distance XFactor. This is only possible when the encoder velocity does not change during the position correction. Otherwise, the encoder distance within which the correction occurs changes.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xEnable | BOOL | A rising edge FALSE -> TRUE activates the POU, a falling edge TRUE -> FALSE deactivates the POU.  A deactivated POU does not execute any actions. |
| i\_ifDrive | IF\_Drive | Input for the axis that shall be controlled. |
| i\_lencMaster | L\_ENC | Input for the logical encoder (master) |
| i\_xStart | BOOL | TRUE: The correction is started as soon as the position of the master is >= i\_lrXStartPos.  FALSE: If a correction has been started before, it will be stopped. |
| i\_lrCorrection | LREAL | Value of the correction |
| i\_lrXStartPos | LREAL | Position of the master from which the correction is to be started. |
| i\_lrXFactor | LREAL | Distance covered by the master within which the correction is to be executed. |
| i\_lrMaxCorrVel | LREAL | Maximum velocity at which the correction is to be executed. |
| i\_lrMaxCorrAcc | LREAL | Maximum acceleration at which the correction is to be executed. |
