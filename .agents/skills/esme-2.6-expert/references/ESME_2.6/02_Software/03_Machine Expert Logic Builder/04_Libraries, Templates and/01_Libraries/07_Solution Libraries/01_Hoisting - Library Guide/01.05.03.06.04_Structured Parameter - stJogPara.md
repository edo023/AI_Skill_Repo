# Structured Parameter - stJogPara

![](images/logo.png)

Structured Parameter - stJogPara

Structure: stJogPara

Structure of parameters related to jog movement.

| Parameter | Data Type | Description |
| --- | --- | --- |
| diJogDist | DINT | Defines the distance of a movement after rising edge of Jog command. This parameter is used to perform jog movement by a certain distance.  Setting i\_stJogPara.diJogDist to 0 disables this option. The axis in this case enters immediately the continuous movement mode.  Range: 0...2147483647  Default value: 0  Scaling/Unit: user units |
| wJogDelay | WORD | Configures delay between finishing the defined distance jog movement and start of continuous jog movement.  Setting i\_stJogPara.wJogDelay to 0 disables this option. The axis in this case enters immediately the continuous movement mode.  Range: 0...65535  Default value: 0  Scaling/Unit: ms |
