# General Information - FB\_RuntimeMeasurement2

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.2.9.0 |

## Task

The function block FB\_RuntimeMeasurement2 is used for the runtime measurement of the program code.

## Description

The function block is used for time measurement. In addition to the actual measured time value, it provides the statistical values for minimum, maximum, and mean values.

Compared to FB\_RuntimeMeasurement, FB\_RuntimeMeasurement2 provides greater data types of the properties for retrieving the time values. This allows for measuring a long duration with a smaller time resolution (for example, measuring in nanoseconds for a time greater than 4294967295 ns).

## Interface

The function block does not provide any input or output parameters. An instance of the function block must not be called in your application program. To control and monitor the time measurement, the function block provides methods and properties.

## Methods

The methods are equal to the methods of the function block FB\_RuntimeMeasurement. For details, refer to the descriptions in the chapter [FB\_RuntimeMeasurement](D-SE-0105432.html).

| Name | Description |
| --- | --- |
| [Reset](D-SE-0105434.html) | Resets the function block to its initial state. |
| [Start](D-SE-0105453.html) | Starts a new measurement. |
| [End](D-SE-0105454.html) | Completes a presently running time measurement. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| uliActTime | ULINT | Read | Duration of the last measurement. Unit: ns |
| uliAverageTime | ULINT | Read | Mean duration of the measurements performed since the last reset. Unit: ns |
| udiCycleCount | UDINT | Read | Number of measurements performed since the last reset |
| uliMaxTime | ULINT | Read | Longest measured period since the last reset. Unit: ns |
| uliMinTime | ULINT | Read | Shortest measured period since the last reset. Unit: ns |
| etResolution | [ET\_TimeResolution](ET_TimeRes-443D3F41.html#ET_TimeRes-443D3F41) | Write | Resolution that is used for indicating the measured values. |
