## Task

Setting the parameters for a stop and an emergency stop of the tracking system.

## Description

The method SetTrackingStopParameters(...) must be used to set the stop and emergency stop parameters for all configured tracking systems when at least one rotative tracking system is configured or when the stop parameters for at least one tracking system are set.

The value for the stop deceleration must be less or equal to the emergency deceleration while the value for the stop ramp must be greater or equal to the emergency stop ramp.

The stop and the emergency stop of the tracking cannot be triggered by you. The stops are executed automatically by the system when the tracking is aborting its present synchronization or when an emergency stop for the robot is triggered.

For further information, refer to ROB\_Tracking Type SetTrackingStopParameters.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etTrackingSystem | [ROB.ET\_CoordinateSystem](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=D_SE_0075477) | ID of the tracking system.  Valid values are: ROB.ET\_CoordinateSystem.Tracking1…ROB.ET\_CoordinateSystem.Tracking30.  A tracking system with the ID must be configured. |
| i\_lrStopDeceleration | LREAL | Deceleration (modification of velocity) in case a stop is issued.  Valid values are:  Stop deceleration > 0.0 and stop deceleration ≤ emergency deceleration.  Unit: [mm/sec2] |
| i\_lrStopRamp | LREAL | Ramp (modification of deceleration) in case a stop is issued.  Valid values are:  Stop ramp > 0.0 and stop ramp ≥ emergency ramp. |
| i\_lrEmergencyDeceleration | LREAL | Deceleration (modification of velocity) in case an emergency stop is issued.  Valid values are:  Emergency deceleration > 0.0 and emergency deceleration ≥ stop deceleration.  Unit: [mm/sec2] |
| i\_lrEmergencyRamp | LREAL | Ramp (modification of deceleration) in case an emergency stop is issued.  Valid values are:  Emergency ramp > 0.0 and emergency ramp ≤ stop ramp. |
