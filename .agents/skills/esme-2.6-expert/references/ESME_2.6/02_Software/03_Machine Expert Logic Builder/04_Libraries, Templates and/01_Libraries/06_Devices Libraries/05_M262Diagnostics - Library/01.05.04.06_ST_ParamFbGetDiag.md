Description

The structure ST\_ParamFbGetDiag specifies parameters of the function block FB\_GetDiagnosticData.

Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| timTimeoutGetDiagData | TIME | Specifies the timeout for the execution of the command GetPlcDiag with the function block FB\_GetDiagnosticData. |
| timTimeoutCreateFile | TIME | Specifies the timeout for the execution of the command CreateFileFromDiagTable with the function block FB\_GetDiagnosticData. |
| byPlcDiagOptEthItfToCheck | BYTE | Specifies the Ethernet interfaces which are analyzed by the function block FB\_GetDiagnosticData. Each bit represents an interface, bit0 = ETH1, bit1 = ETH2, and so on. |
