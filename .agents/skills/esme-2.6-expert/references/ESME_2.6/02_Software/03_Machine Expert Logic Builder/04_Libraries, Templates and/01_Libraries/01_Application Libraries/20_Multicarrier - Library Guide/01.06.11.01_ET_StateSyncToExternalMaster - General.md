# ET\_StateSyncToExternalMaster - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.5.10.0 |

## Description

The enumeration ET\_StateSyncToExternalMaster specifies the status of the MovePosAndSync command for the synchronized movement of a carrier (refer to [IF\_MovePosAndSync](IF_MovePosAndSync-4526BC87.html#IF_MovePosAndSync-4526BC87)).

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| NotActive | 0 | The move command MovePosAndSync is not active. |
| WaitForPhasingZoneReached | 1 | The carrier is waiting for the master to reach the phasing zone.  The phasing zone is defined by the parameters i\_lrXStartPhasingToMaster and i\_lrXEndPhasingToMaster that are specified in the method [IF\_MovePosAndSync - StartSyncFromStandstill (Method)](MovePosAndSync-StartSyncFrStand-4544E151.html). |
| Phasing | 2 | The carrier is in the phasing zone and is synchronizing to a master. |
| InPhaseToMaster | 3 | The carrier is synchronized to a master by a one-to-one cam. |
