Function Block Description

Horizontal movement of a crane trolley or bridge induces sway on a suspended load. The AntiSwayOpenLoop\_2 function block helps to reduce this induced sway caused by horizontal movement.

The AntiSwayOpenLoop\_2 function block is designed for the following:

oCalculate a speed reference profile to correct for sway induced by horizontal movement of the crane.

oIncrease the comfort of crane operation.

oProvide an innovative highly efficient Anti-sway control without additional sensors.

oProvide information about the stop distance for use in automatic positioning cranes.
