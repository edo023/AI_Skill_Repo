Start with a New Project

To start with a new project, use File > New Project > Default Project.

Add a Smart Template Module to a Project

| Step | Action |
| --- | --- |
| 1 | To start with a new project, use File > New Project > Default Project. |
| 2 | Click View > Modules to open the Modules view. |
| 3 | Right-click Application (root node). |
| 4 | Select Add Module… from the contextual menu.  Result: The Add Module dialog is displayed. The Smart Template modules installed in your project are listed.  You can filter the list using the Full text search field or the Vendor list. |
| 5 | Select the Smart Template module to be added and click Next.  NOTE: One or more clicks on the Next button display one or more additional plugin-specific dialogs that are part of the plugins referenced by the selected module.  Result: After clicking the Next button (once or several times), the Add <Module name> dialog is displayed. |
| 6 | Enter a Name. |
| 7 | Click Add Module.  Result:  oThe [Module Manager](SmrtTplt_Using_Smart_Template_Modules-3.htm#XREF_D_SE_0089676_1) is inserted beneath the Application (root node). This is done automatically once (by adding the first Smart Template module).  oThe module is inserted in parallel to the Module Manager node.  oThe Configuration data node is displayed beneath the module.  Also refer to [Import / Export of Configuration Data](SmrtTplt_Using_Smart_Template_Modules-5.htm#XREF_D_SE_0092643_1).  oThe [Module Interface](SmrtTplt_Using_Smart_Template_Modules-4.htm#XREF_D_SE_0092551_1) node is displayed beneath the module.  oThe Configuration data tab of the inserted module is displayed. |

Depending on the Smart Template module, you can add a Smart Template submodule.
