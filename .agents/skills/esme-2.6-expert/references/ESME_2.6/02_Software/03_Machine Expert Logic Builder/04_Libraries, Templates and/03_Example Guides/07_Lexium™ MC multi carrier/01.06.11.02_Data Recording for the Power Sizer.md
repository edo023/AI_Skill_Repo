# Data Recording for the Power Sizer

## Overview

To record the data for the MC12 PowerSizer, execute the following steps:

| Step | Action |
| --- | --- |
| **1** | Download the project to your controller. |
|  |
| **2** | Start your application and let it run. For more information, refer to the operation mode [Auto](AutoMode-2D258DF0.html). |
| **3** | In the folder MC12PowerSizingDataRecord, open the visualization VIS\_MC12PowerSizingDataRecord. |
| **4** | Click the button Enable/Disable Recording to enable the recording process. |
| **5** | In the Time field, specify the duration of the recording in seconds. The default value is 10 seconds. |
| **6** | In the max. file size field, specify the maximum size of your file in MB. The default value is 8 MB. |
| **7** | Press the button Start Recording to start the recording.  **Result:** The recording progress is displayed in the progress bar.  When the Recording progress bar is complete and the Done indicator light is green, the recording is terminated and the process is disabled automatically. |
| **8** | Open your controller device object and go to the Files tab, refresh the view, select the file MC12PowerSizingDataRecord.csv and download it to your computer. |
| **9** | Open EcoStruxure Machine Expert Twin and load the file MC12PowerSizingDataRecord.csv. For more information, refer to the [EcoStruxure Machine Expert Twin Getting Started User Guide](https://product-help.se.com/Machine%20Expert%20Twin/latest/en-US/Tw_GetSt?t=Tw_GetSt%2FFMIMenu-E01D9C39.html). |
