# ET\_BrakeState - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.0.0 |

## Description

The enumeration describes the states of a motor brake.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Engaged | 0 | Brake is active. |
| Released | 1 | Brake is released. |
