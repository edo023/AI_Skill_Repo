# FB\_VisUnitManager - UpdateVisConnectorReference (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.1.0.0 |

## Description

The method UpdateVisConnectorReference is used for updating the data of the function block specified in the inputs of the method.

For more information, refer to [Providing Data for Visualization](ProvidingDataForVis-DCA4DB61.html).

## Inputs

| Name | Data type | Description |
| --- | --- | --- |
| i\_uiVisConnectorIndex | UINT | Specifies the array index of the visualization function block which is assigned on the input i\_pbyVisConnectorAddress. |
| i\_pbyVisConnectorAddress | POINTER TO BYTE | Specifies the address of the instanced visualization function block. |
