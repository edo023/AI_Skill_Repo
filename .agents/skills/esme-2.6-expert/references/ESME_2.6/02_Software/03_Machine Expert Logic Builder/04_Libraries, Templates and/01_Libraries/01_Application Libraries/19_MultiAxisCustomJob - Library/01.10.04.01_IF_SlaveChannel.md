# IF\_SlaveChannel

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The interface IF\_SlaveChannel serves as the link to a subordinate axis within a multi-axis group. It provides the mechanism to monitor the operating state of the axis and to apply the target values (position, velocity, and acceleration) calculated by the user-defined motion algorithm.

Each subordinate axis to be controlled by IF\_MultiAxisCustomJob must be associated with an instance of IF\_SlaveChannel. During execution, the user-defined algorithm uses this interface to write movement values to the axis in each RTP cycle.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
