FB\_RunTimeMeasurement - General Information

Overview

| --- | --- |
| Type: | Function block |
| Available as of: | V1.0.3.0 |
| Inherits from: | - |
| Implements: | - |
| Versions: | Current version |

Task

Runtime measurement of program code.

Description

The function block is used for diagnostic. Interruptions by higher priority tasks or system tasks are not taken into account.

Interface

NOTE: POU is not addressable!

The POU must not be called up.

The function block can be used by calling it up over methods and properties.

Methods

| Name | Description |
| --- | --- |
| [End](Function_Blocks_R_to_Z-19.htm#XREF_D_SE_0087361_1) | Completes a currently running time measurement |

| [Start](Function_Blocks_R_to_Z-21.htm#XREF_D_SE_0087363_1) | Starts a new measurement |

Properties

| udiMinTime | UDINT | Read | Shortest measured period since the last reset. Unit: ns |
