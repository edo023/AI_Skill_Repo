# Close Method

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.4.0 |

![](images/T002930715.gif)

## Task

Disconnect the clients and close the server socket.

## Functional Description

Disconnects the clients and closes the server socket, possibly discarding data in the receive buffers of the client. The server is not monitoring for new connections afterwards.

The BOOL return value is TRUE if the function was executed successfully. Evaluate the property Result, in case the return value is FALSE.

## State Transition of the Client

| Stage | Description |
| --- | --- |
| 1 | Initial state: `Listening` |
| 2 | Functional call |
| 3 | State: `Idle` |
