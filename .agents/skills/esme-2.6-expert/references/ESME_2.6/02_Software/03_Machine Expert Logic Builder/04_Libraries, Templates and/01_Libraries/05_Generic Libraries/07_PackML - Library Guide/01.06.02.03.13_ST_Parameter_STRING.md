# ST\_Parameter\_STRING

## Overview

|  |  |
| --- | --- |
| Type | Data structure |
| Available as of | V1.4.2.0 |

## Description

The structure ST\_Parameter\_STRING provides information for parameters or values with STRING data type.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| ID | DINT | ID of the parameter |
| Name | STRING(GPL.Gc\_uiMaxLengthOfParameterName) | Name of the parameter |
| Unit | STRING[6] | Unit of the parameter |
| Value | STRING | Value of the parameter |

## Used By

* ST\_Administration
* ST\_Command
* ST\_Status
* ST\_Ingredients
* ST\_ProcessVariables
