# Adding Device Module to the Project

## Instantiation of the Device Module

The instantiation of this Device Module requires that a Modbus IOScanner be added to the serial interface of your controller. Using [**Add Function from Template**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083799), you can:

* select the Modbus IOScanner which manages the device.
* assign the Modbus slave address for the device.
