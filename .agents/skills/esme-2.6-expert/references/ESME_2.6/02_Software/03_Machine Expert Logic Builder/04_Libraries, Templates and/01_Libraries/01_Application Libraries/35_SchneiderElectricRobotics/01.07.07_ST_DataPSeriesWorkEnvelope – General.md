## Overview

## Description

Contains a list of parameters to read out work envelope related data from a Lexium P Robot.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| lrDiameter | LREAL | Work envelope diameter |
| lrHeight | LREAL | Work envelope height |
| lrZOffset | LREAL | Z offset to shift the coordinate origin of the robot inside the work envelope. |
