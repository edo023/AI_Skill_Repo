## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.1.0 |

## Description

The interface is used to get the status information on the ongoing movement.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| xAccelerating | BOOL | Get | TRUE: The motor accelerates. |
| xConstantVelocity | BOOL | Get | TRUE: A movement at a constant velocity is performed. |
| xDecelerating | BOOL | Get | TRUE: The motor decelerates. |
| xDirectionNegative | BOOL | Get | TRUE: The motor shaft rotates in negative direction. |
| xDirectionPositive | BOOL | Get | TRUE: The motor shaft rotates in positive direction. |
