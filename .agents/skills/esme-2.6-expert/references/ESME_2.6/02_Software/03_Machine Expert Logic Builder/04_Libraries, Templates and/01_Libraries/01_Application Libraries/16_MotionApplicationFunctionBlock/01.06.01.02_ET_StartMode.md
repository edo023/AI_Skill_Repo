# ET\_StartMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_StartMode is used to define the [warm start mode](WarmStart-45EBC5A3.html).

## Enumeration Elements

| Name | Value (USINT) | Description |
| --- | --- | --- |
| MoveToRestPosition | 1 | The slave moves to the rest position and stores the FIFO values. | MoveToCurvePosition | 2 | The slave moves to the position on the curve. For further information, refer to:   * [Warm start mode MoveToCurvePosition for FlyingShear](WarmStart-45EBC5A3.html#WarmStart-45EBC5A3__MoveToCurvePositionWarmStartModeMov-46E4DD81) * [Warm Start Mode MoveToCurvePosition for RotaryKnife](WarmStartRK-E4828741.html#WarmStartRK-E4828741__MoveToCurvePositionWarmStartModeMov-E482BB59) |
| ContinueWithOffsetCurve | 3 | The slave continues with an offset curve.
