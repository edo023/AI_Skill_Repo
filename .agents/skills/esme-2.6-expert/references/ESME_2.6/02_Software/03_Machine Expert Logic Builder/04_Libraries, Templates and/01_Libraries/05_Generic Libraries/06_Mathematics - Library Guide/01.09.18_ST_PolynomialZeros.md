# ST\_PolynomialZeros

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |

## Description

The structure ST\_PolynomialZeros represents the zeros of equations of the second or higher degree as complex numbers.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| diCount | DINT | Number of zeros |
| astZeros | ARRAY [1..5] OF [ST\_Complex](ST_Complex-GeneralInformation-0C0358C0.html#ST_Complex-GeneralInformation-0C0358C0) | The zeros represented as complex numbers. Only the first diCount values contain valid values. |
