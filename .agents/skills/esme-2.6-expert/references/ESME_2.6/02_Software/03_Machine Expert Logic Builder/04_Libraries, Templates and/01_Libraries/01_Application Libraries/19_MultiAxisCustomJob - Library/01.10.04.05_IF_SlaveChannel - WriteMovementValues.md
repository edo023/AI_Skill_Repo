## Description

The method WriteMovementValues of the interface [IF\_SlaveChannel](IF_SlaveChannel-4D6988AD.html) is used to write the calculated target values for position, velocity, and acceleration to the subordinate axis. These values must be written in every RTP cycle during the execution of the multi-axis custom job.

The values are written to the axis for the current cycle without further verification. If the method is called while the subordinate axis is in the operating state ErrorStop, the values are not written and an error is detected.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
