# FC\_EtLogLevelToString

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

This functions converts the enumeration element [ET\_LogLevel](ET_LogLevel-37599385.html#ET_LogLevel-37599385) into a string. The return value of the function is the name of the enumeration element.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etLogLevel | [ET\_LogLevel](ET_LogLevel-37599385.html#ET_LogLevel-37599385) | The [ET\_LogLevel](ET_LogLevel-37599385.html#ET_LogLevel-37599385) value that has to be converted. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-375A58EE.html#ET_Result-375A58EE) | Provides diagnostic and status information as a numerical value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |

## Return Value

| Data type | Description |
| --- | --- |
| STRING[40] | Name of the enumeration element. |
