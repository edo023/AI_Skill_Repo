# Global Constants List (GCL)

## Overview

|  |  |
| --- | --- |
| Type: | Global constants |
| Available as of: | V1.0.2.0 |

## Description

The global constants list contains the global constants of the SercosIpClient library.

## Global Constants

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_diNoWait | DINT | 0 | No waiting time for response, that is, immediate response required. This constant can be used for the input i\_diTimeoutMs. |
| Gc\_diWaitForEver | DINT | -1 | No timeout for response, that is, response can be transmitted at any time. This constant can be used for the input i\_diTimeoutMs. |
