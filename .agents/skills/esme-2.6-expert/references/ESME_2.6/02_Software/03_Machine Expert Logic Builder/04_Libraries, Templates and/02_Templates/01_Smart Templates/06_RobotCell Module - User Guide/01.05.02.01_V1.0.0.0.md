# V1.0.0.0

## Description

The Smart Template RobotCell Module provides the functionalities to add a robot cell to an application and to add other Smart Template modules to this robot cell (for example, the Smart Template modules for robots, conveyors, cameras, or sensors).

## System Requirements

Using the Smart Template module with other versions of software or firmware may have results other than those described in the present documentation.
