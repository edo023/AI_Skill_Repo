# Initialize a T-Series Robot Collision Handler

## Overview

The following variables are used in the example:

  

```
etRobotType : SERP.ET_RobotTSeries;
fbRobotTSeries : SER.FB_RobotTSeries;
fbCollisionHandlerTSeries : SER.FB_CollisionHandlerTSeries;
stOptionalParameters : SER.ST_CollisionHandlerTSeriesOptionalParameter;
etConfigurationA : SERP.ET_RobotTSeriesConfiguration;
etConfigurationB : SERP.ET_RobotTSeriesConfiguration;
etPlane : ROB.ET_WorkingPlane;
//diagnostics
etDiag : GD.ET_Diag;
etDiagExtSERP : SERP.ET_DiagExt;
etResultSER : SER.ET_Result;
sResultMsg : STRING(80);
```

## How to Initialize a T-Series Robot Collision Handler

| Step | Action |
| --- | --- |
| 1 | Initialize an object of type FB\_RobotTSeries:   ```       fbRobotTSeries.InitializeRobot(             i_etRobotType := etRobotType             i_ifDriveA := DRV_RobotA,             i_ifDriveB := DRV_RobotB,             i_etConfigurationA := etConfigurationA,             i_etConfigurationB := etConfigurationB,             i_etPlane := etPlane, //<- this must be ROB.ET_WorkingPlane.XZ             q_etDiag => etDiag,             q_sMsg => sResultMsg,             q_xRotationalAxisRequired =>        ); ```   In this case, DRV\_RobotA, DRV\_RobotB are the drives linked to the joints of the robot.  NOTE: The method SetParametersFromRobotTSeries only supports i\_etPlane equal to ROB.ET\_WorkingPlane.XZ. |
| Optional:  Before calling SetParametersFromRobotTSeries, it is possible to set the optional parameters:    ```       stOptionalParameters.lrCollisionMargin := 20.0; //[mm]       stOptionalParameters.xModelParallelLinkage := TRUE;        fbCollisionHandlerTSeries.SetOptionalParameters(             i_stData := stOptionalParameters,             q_xError => xError,             q_etResult => etResultSER,             q_sResultMsg => sResultMsg       ); ``` |
| 2 | After a successful initialization, you can initialize the collision handler as follows:   ```       fbCollisionHandlerTSeries.SetParametersFromRobotTSeries(             i_ifRobotTSeries:= fbRobotTSeries,             q_xError => xError,             q_etResult => etResultSER,             q_sResultMsg => sResultMsg       ); ``` |
