## Functional Description

The method GetRandomTargetsNoRotation reads out the parameter for random targets without rotation.

## Interface

The interface is identical to FB\_RandomTargetsGenTar.GetTargetsInPlane(), except the two inputs i\_lrMinRotation, i\_lrMaxRotation. These two inputs are set to 0.0.

For more information, refer to FB\_RandomTargetsGenerator in the SchneiderElectricRobotics Toolbox Library Guide Online Help.

## Diagnostic Messages

The diagnostic messages are identical to FB\_RandomTargetsGenTar.GetTargetsInPlane().
