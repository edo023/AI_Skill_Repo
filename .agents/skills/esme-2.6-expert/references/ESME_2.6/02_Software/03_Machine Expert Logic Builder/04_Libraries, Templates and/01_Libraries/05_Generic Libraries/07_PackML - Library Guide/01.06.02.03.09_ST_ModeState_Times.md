# ST\_ModeState\_Times

## Overview

|  |  |
| --- | --- |
| Type | Data structure |
| Available as of | V1.4.2.0 |

## Description

The structure ST\_ModeState\_Times is a collection of tags providing information for the cumulative time tracker.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| Mode | DINT | Represents the cumulative amount of time (in seconds) spent in each mode since the last timer and counter reset was executed |
| State | ARRAY [0..GCL.Gc\_uiMaxNumberOfStates] OF DINT | Represents the cumulative amount of time (in seconds) spent in each state of a particular mode since the last timer and counter reset was executed |
