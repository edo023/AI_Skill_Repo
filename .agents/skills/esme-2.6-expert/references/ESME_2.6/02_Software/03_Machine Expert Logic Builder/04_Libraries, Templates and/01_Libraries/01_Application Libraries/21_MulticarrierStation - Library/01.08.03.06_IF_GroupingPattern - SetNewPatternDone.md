## Task

Finishing the creation of a new carrier group pattern.

## Description

The method SetNewPatternDone is a mandatory method for creating a group pattern. When you finish the creation of a new group pattern, you must call the method SetNewPatternDone in a last step.

The return value SetNewPatternDone of type BOOL indicates TRUE if a pattern has been set successfully.

## Inputs

The method has no inputs.

## Outputs

The method has no outputs.
