# FC\_Matrix2DAddition

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

Given two 2D input matrices, the function returns the element-by-element addition of the elements of the matrices.

![](images/DD00823952.png)

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stMatrixA | [ST\_Matrix2D](ST_Matrix2D-GeneralInformation-0C04D259.html#ST_Matrix2D-GeneralInformation-0C04D259) | First 2D matrix to be added. |
| i\_stMatrixB | [ST\_Matrix2D](ST_Matrix2D-GeneralInformation-0C04D259.html#ST_Matrix2D-GeneralInformation-0C04D259) | Second 2D matrix to be added. |

| Output | Data type | Description |
| --- | --- | --- |

| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |

## Return Value

| Data type | Description |
| --- | --- |
| [ST\_Matrix2D](ST_Matrix2D-GeneralInformation-0C04D259.html#ST_Matrix2D-GeneralInformation-0C04D259) | The function returns a matrix containing the element-wise addition of the elements of the two input matrices. |
