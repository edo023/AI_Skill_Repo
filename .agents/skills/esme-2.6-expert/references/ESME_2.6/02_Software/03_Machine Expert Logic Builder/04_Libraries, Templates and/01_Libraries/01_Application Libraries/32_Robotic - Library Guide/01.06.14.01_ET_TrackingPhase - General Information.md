# ET\_TrackingPhase - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.6.0.0 |

## Description

Enumeration type to specify the phase of a tracking movement.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Inactive | 0 | Tracking is inactive. |
| Starting | 10 | A tracking movement is starting to a configured linear tracking system. |
| Synchronous | 20 | A tracking movement is synchronous to a configured linear tracking system. |
| Stopping | 30 | A tracking movement is stopping from a configured linear tracking system. |
