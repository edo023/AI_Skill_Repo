# Error Management of Network Variable List (NVL)

![](images/logo.png)

Error Management of Network Variable List (NVL)

Function Block Description

The function block NETVARRESETERROR restarts the communication exchange of variables after an NVL error is detected.

This function block resets detected duplicate list identifiers which are indicated by the output parameters dwDuplicateListIdIp1 or dwDuplicateListIdIp2 of the function block NETVARGETDIAGINFO.

![G-SE-0016809.1.gif-high.gif](images/G-SE-0016809.1.gif-high.gif)

Input Parameters

| Parameter | Type | Comment |
| --- | --- | --- |
| xExecute | BOOL | rising edge: The detected NVL errors are reset and the communication is restarted.  falling edge: The outputs of the function block are reset. |
| dwListIdentity | DWORD | List identifier used by the GVLs on which the errors need to be reset. |

Output Parameters

| Parameter | Type | Comment |
| --- | --- | --- |
| xDone | BOOL | The reset after detected error(s) has been completed successfully. |
| xBusy | BOOL | The function block is active. |
| xError | BOOL | TRUE: NETVARRESETERROR function block execution error has been detected.  FALSE: No NETVARRESETERROR function block execution error has been detected. |
