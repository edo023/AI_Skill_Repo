# ET\_FastMoveParameter - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.11.0.0 |

## Description

The enumeration ET\_FastMoveParameter specifies the parameters for the feature FastMove.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| AccelerationFactor | 1 | Acceleration factor for FastMove, when the tuning is disabled. |
| DecelerationFactor | 2 | Deceleration factor for FastMove, when the tuning is disabled. |
| TrackingDeviationOnPath | 3 | Tracking deviation in 3D space for FastMove, when the tuning is enabled. |
| TrackingDeviationInTarget | 4 | Tracking deviation in the target position for FastMove when the tuning is enabled. |
