## Description

The structure ST\_MqttServerParameter contains the parameter to establish a TCP connection to the MQTT server.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sIpAddress | STRING[15] | IP address of the MQTT server to connect to. |
| uiPort | UINT | Port address of the MQTT server. The default port for the MQTT protocol is 1883. |
| sHostname | STRING[255] | Specifies the fully qualified host name by which the remote server can be reached.  The specification of the server by the host name is exclusive to connections established through a proxy server which is configured by the interface IF\_ProxyHandler. |
