# UAReferenceDescription

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.0.0.0 |
| Inherits from: | - |
| Attribute: | ``` 'pack_mode' := '1' ``` |

## Description

The structure UAReferenceDescription provides information about references and target nodes.

## Structure Element

| Name | Data type | Description |
| --- | --- | --- |
| ReferenceTypeID | [UANodeID](D-SE-0099967.html#D-SE-0099967__D-SE-0099967.4) | Identifier for the reference type from the starting node to the target node. |
| IsForward | BOOL | Indicates whether a forward reference is followed. |
| NodeID | [UAExpandedNodeID](D-SE-0099971.html) | Indicates the node ID of the target node. This can also be a node in another server. |
| BrowseName | STRING[255] | Indicates the qualified name of the target node. |
| DisplayName | [UALocalizedText](D-SE-0099974.html) | Indicates the localized name of the target node. |
| NodeClass | [UANodeClassMask](D-SE-0100000.html#D-SE-0100000__D-SE-0100000.5) | Node class of a returned node such as object, variable, or method. |
| TypeDefinition | [UAExpandedNodeID](D-SE-0099971.html) | Indicates the node ID of the object or variable type of the target node. |
