| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Calculates the distance of a point from a straight line in a 3-dimensional space.

Description

This calculates the distance of the point i\_stPoint from the straight line i\_stLine. In addition, the perpendicular of the point onto the straight line is calculated.
