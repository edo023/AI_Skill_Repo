# MC_ReadDigitalInput

![](images/logo.png)

MC\_ReadDigitalInput

MC\_ReadDigitalInput - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function Block |
| Available as of: | 1.0.5.0 |
| Versions: | current version |

Task

A digital input is to be read out.

Description

The Value parameter of the digital input applied at the Input input is returned.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| Input | SystemConfiguration.G\_IN | The digital input that is to be read out |
| Enable | BOOL | TRUE: The value of the digital input is read out. |

| Value | BOOL | Value of the selected input signal |
