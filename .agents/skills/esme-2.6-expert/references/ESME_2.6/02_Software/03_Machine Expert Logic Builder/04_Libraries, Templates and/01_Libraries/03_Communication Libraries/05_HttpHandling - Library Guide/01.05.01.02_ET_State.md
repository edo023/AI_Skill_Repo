# ET\_State

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_State contains the possible values that indicate the state of the client.

## Enumeration Elements

| Name | Value  INT | Description |
| --- | --- | --- |
| Idle | 0 | The client is in idle state and ready to be connected to the server. |
| Connecting | 10 | The client is connecting to the server. |
| Connected | 20 | The client is connected to the server and ready to send a request. |
| SendingRequest | 30 | The client is sending a request to the server. |
| Disconnecting | 40 | The client is disconnecting from the server. |
| ResponseAvailable | 50 | The client has received a response to the sent request which must be acknowledged. |
| Error | -10 | The client is in error state. |
