# FB\_TorqueFeedForward2 - SetControlMode (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.5.1.0 |
| Versions: | Installed version |

## Description

The method is used to set the requested control mode for the torque feed forward control on the drive. It defines how the drive interprets and applies the externally provided torque feed forward value within its internal control loops.

NOTE: If the execution of this method was not successful, the former active torque control mode is still active.

The following conditions must be fulfilled:

* The function block must be successfully configured.
* The Sercos ActualState must be phase 4 (operational).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etValue | [ET\_ControlMode](ET_ControlMode-GeneralInformation-45AA9ADE.html) | Control mode for the torque feed forward control on the drive. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error is detected.  For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-45A2DE35.html) | Provides diagnostic and status information as a numeric value. |
| q\_etResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |

## Troubleshooting

Possible diagnostic information (q\_xError = TRUE)

| Outputs of the method indicate the values | Cause | Solution |
| --- | --- | --- |
| q\_etResult = ConfigurationNotDone | The function block is not configured. | Configure the function block with the Configure() method. |
| q\_etResult = InvalidSercosState | The Sercos ActualState is not as required. | Set the Sercos ActualState to phase 4. |
| q\_etResult = InvalidWorkingMode | The working mode of the assigned drive is invalid. | Set the working mode of the drive to Physical. |
| q\_etResult = DriveDoesNotSupportThisFunction | The selected axis does not support this function. | Select an axis that supports this function (for example, Lexium LXM62 Drive, Lexium LXM52 Drive or Lexium ILM62 Drive module). |
| q\_etResult = InvalidMode | The value for the control mode is out of range. | Set  i\_etValue to a value defined in [ET\_ControlMode](ET_ControlMode-45A4677D.html). |
