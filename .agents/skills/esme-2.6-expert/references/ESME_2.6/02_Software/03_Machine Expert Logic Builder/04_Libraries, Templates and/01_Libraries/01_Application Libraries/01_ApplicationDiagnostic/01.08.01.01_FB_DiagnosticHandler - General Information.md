## Task

The function block is used to receive diagnostic messages, process them and forward them.

## Description

The function block provides several methods and properties to receive diagnostic messages, process them by triggering appropriate reactions and forward them to other diagnostic handlers. Therefore, an instance of the FB\_DiagnosticHandler must be implemented in each logical unit and referenced to others according to the logical structure of your application.
