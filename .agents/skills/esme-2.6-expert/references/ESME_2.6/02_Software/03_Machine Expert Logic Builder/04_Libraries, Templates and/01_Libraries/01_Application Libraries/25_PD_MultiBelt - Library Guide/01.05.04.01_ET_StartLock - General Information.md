# ET\_StartLock - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

## Description

Using this enumeration, you can determine which type of start signals are permitted.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| AllSignals | 1 | All start signals are permitted. The stations react to Touchprobe and BOOL signals. |
| BoolOnly | 2 | Only bool signals are permitted. The station only reacts to the bit xStart. |
| TpOnly | 3 | Only Touchprobe signals are permitted. The station only reacts to ifTpStart signals. |
| Disabled | 4 | No start signals are permitted. The station does not react to external signals. |
