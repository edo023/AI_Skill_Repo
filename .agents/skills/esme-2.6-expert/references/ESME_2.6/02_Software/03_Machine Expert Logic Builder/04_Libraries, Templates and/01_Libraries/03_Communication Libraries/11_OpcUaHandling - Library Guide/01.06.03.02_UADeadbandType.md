# UADeadbandType

## Overview

|  |  |
| --- | --- |
| Type: | Alias |
| Available as of: | V2.0.0.0 |

## Description

The alias UADeadbandType specifies which type of deadband is applied, if any.

UADeadbandType is a bitmask and every bit of the byte represents a deadband.

## Supported Bitmasks

| Bitmask | Description |
| --- | --- |
| 16#0 | No deadband calculation is applied. |
| 16#1 | Absolute deadband calculation is applied. |
| 16#2 | Percent deadband calculation is applied. |
