# RegisterLoggerPoint Method

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.5.2.0 |

## Task

Registering a logger point for the function block.

## Description

The RegisterLoggerPoint method registers the function blocks as a logger point to the global application logger that is used to store log messages.

The application logger itself must be initialized by calling the [RegisterCommunicationService method](../../../../../api/crossBook?lang=en-US&virtualBookName=APL2LG&topicID=IF_ApplicationLogger_RegisterCommun_37D13059) of the application logger before the registration of the function block logger point.

The function block supports one logger point. A modification of the logger level is ignored if the logger point is not registered. An unsuccessful registration of the logger point has no impact on the function block.

The name of the logger point in the application logger is defined by the input i\_sName. The input i\_ifParent specifies the parent logger point under which the logger point for the function block is registered in the logger point tree.

## Input Pin Description

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifParent | APL2.IF\_LoggerPoint | Interface of the application logger.  APL2 is the default namespace of the ApplicationLogger2 library. For further information, refer to the [ApplicationLogger2 Library Guide.](../../../../../api/crossBook?lang=en-US&virtualBookName=APL2LG&topicID=IF_LoggerPoint_37BB64E2)  Parent logger point under which the logger point of the function block is registered. If the input is set to 0, the global logger point APL.GVL.G\_ifApplicationLogger is defined as parent. |
| i\_sName | STRING [80] | The name of the logger point that is displayed in the application logger. |

## Output Pin Description

| Output | Data Type | Description |
| --- | --- | --- |

| q\_etResult | [ET\_Result](ET_Result-412078C8.html) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING [80] | Provides additional diagnostic and status information as a text message. |
