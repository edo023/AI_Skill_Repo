Task

Converting from HEX to binary code for BYTE.

Description

The function converts values in the decimal range 0..255 from HEX format into decimal format. Each input BYTE is to be interpreted as an ASCII text character. The low Byte from the input forms the low bit of the output.

This function is well-suited to receive hex-coded numbers from the serial interface or to read them from a file.
