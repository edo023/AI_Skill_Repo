# IF\_ScalingAccess

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The access interface IF\_ScalingAccess allows you to access the user function interface [IF\_Scaling](IF_Scaling-4F8E6EE1.html#IF_Scaling-4F8E6EE1) in order to read and write scaling parameters for a drive.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| Scaling | [IF\_Scaling](IF_Scaling-4F8E6EE1.html#IF_Scaling-4F8E6EE1) | Get | Accesses the user function interface IF\_Scaling. |
