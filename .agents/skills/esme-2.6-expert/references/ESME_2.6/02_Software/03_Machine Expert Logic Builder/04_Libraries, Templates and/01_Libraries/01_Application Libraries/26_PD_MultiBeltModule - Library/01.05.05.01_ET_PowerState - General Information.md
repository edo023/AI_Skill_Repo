# ET\_PowerState - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.4.0 |
| Versions: | Current version |

## Description

This structure shows the state of the position control of the MultiBelt axes.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Disabled | 1 | The module is deactivated.  i\_xEnable := FALSE |
| ReadAbsEncoder | 2 | The position of the multiturn encoder of the axes is being read. |
| ClearEncoderReading | 3 | Encoder reading canceled due to a detected error. |
| PowerOn | 4 | Position control of the axes is enabled. |
| PowerOff | 5 | Position control of the axes disabled. The module is waiting for a signal to enable the position control. |
| WaitForDisabled | 6 | The module is waiting for the axes to be disabled. |
