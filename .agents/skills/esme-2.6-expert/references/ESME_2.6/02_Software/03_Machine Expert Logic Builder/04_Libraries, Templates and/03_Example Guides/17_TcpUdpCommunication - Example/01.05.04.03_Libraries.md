Libraries

Overview

The following libraries have been added to the Library Manager of each application:

| Library | Company | Function |
| --- | --- | --- |
| TcpUdpCommunication | Schneider Electric | Provides the components for communication over the [Ethernet](../glossary/glossary.htm#XREF_D_SE_0055638_7) network with the use of the transport protocols [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) and/or [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21). |
| SysTimeRtc | System | Provides the components to manage the real time clock ([RTC](../glossary/glossary.htm#XREF_D_SE_0024697_729)) of the controllers. |
| SysMem | System | Provides components to access the dynamic memory management of the runtime system. |

The TcpUdpCommunication library provides parameters to configure the supported functions. You can access these parameters in the global parameter list (GPL) available in the Library Manager of the application.

The table lists the parameters that have been adjusted in the example application:

| Parameter | Initial value | New value |
| --- | --- | --- |
| Gc\_uiTCPServerMaxConnections) | 30 | 5 |
