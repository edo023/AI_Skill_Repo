# ET\_Parameter - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.4.0.0 |

## Description

Parameter that can be set or overwritten by you.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No parameter. |
| TcpPlateSize | 1 | Distance between the suspension points of the lower arms and the center of the flange plate.  Range: 50 ≤ TcpPlateSize ≤ 75  Setting ET\_Parameter.TcpPlateSize successfully resets the internal value of the TCP plate mass to 0 kg.  The mass of the user-specific TCP plate can be considered by calling the method SetKinematikParameter(ET\_KinematicParameter.GripperMass). |
| MassTcpPlate | 2 | Mass of the TCP plate.  Range: 0 ≤ MassTcpPlate ≤ 1.0. |
