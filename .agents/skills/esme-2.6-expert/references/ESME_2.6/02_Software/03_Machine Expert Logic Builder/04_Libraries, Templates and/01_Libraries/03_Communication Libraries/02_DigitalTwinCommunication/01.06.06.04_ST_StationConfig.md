# ST\_StationConfig

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

The structure ST\_StationConfig contains parameters to define a station in EcoStruxure Machine Expert Twin.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sCategory | STRING[80] | The predefined string which identifies the 3-D model to use in EcoStruxure Machine Expert Twin.  Example:  `'StandardStation'` |
| udiStationId | UDINT | The unique station ID.  Example:  `1` |
| udiColor | UDINT | The color of the text of the station (sTextLabel) selected with [ET\_EmulationColors](ET_EmulationColors-66CA3B7A.html).  Example:  ET\_EmulationColors.Black |
| sTextLabel | STRING[80] | The name that is displayed for the station on the track. If the string is empty, the station is not displayed in the emulation.  Example:  `'Station 1'` |
| udiTrackId | UDINT | The track on which the station is located.  Example:  `1` |
| lrWaitPosition | LREAL | The waiting position of the station that is represented by a green line on the track.  If lrWaitPosition is equal to lrProcessPosition, only the blue marker is displayed on the track (also see the examples below the table).  Example:  `800` |
| lrProcessPosition | LREAL | The process position of the station that is represented by a blue line on the track.  Example:  `1100` |

Examples:

| If lrWaitPosition | Then... |
| --- | --- |
| **<** lrProcessPosition | The green and the blue lines are both displayed. |
| **=** lrProcessPosition = 1100 | Only the blue line is displayed. |
