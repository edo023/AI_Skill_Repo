## Description

This alias allows you to create an instance of the function block FB\_ProxyHandlerHttpConnect directly from the MqttHandling library. Thus, avoiding the need to add the ProxyCommunicationSupport library to your application. For further information, refer to the [ProxyCommunicationSupport Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PrxComSu&topicID=FB_ProxyHandlerHttpConnect_03D94129).
