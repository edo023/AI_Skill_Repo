## Description

The function template HttpClient provides a ready-to-use coding template as a pattern to implement an http client in your application. It implements the following features:

* http client with the use of the HttpHandling library.
* DNS (Domain Name System) client with the use of the TcpUdpCommunication library.
* Function block implementing the interface TCPUDP.IF\_VerifyCertCallback, which is assigned to the http client to evaluate the result of the verification of the X509 certificate on secured connections.

## Compatibility

The described function template can be used in applications of the controller families supported by EcoStruxure Machine Expert and supporting Ethernet communications.
