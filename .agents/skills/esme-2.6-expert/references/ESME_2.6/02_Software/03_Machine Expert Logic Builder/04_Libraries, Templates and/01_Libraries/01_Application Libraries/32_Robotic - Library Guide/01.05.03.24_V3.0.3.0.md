## New Features

The Modicon M262 Motion Controllers TM262M25MESS8T and TM262M35MESS8T are supported now with the following stipulations:

* Sercos cycle time must be set to 4 ms.
* One M262 controller can control one robot.
* The method IF\_RobotConfiguration.SchneiderElectricRobot is not supported.
* The tracking methods IF\_RobotConfiguration.AddLinearTrackingSystem, IF\_RobotConfiguration.AddLinearTrackingSystem2, IF\_RobotConfiguration.AddLinearTrackingSystem3, and IF\_RobotConfiguration.AddRotativeTrackingSystem are not supported.
* For Modicon M262 and M660 Motion Controllers: If FB\_Robot and/or IF\_Robot report a diagnostic message and the axis is in MOIN.ET\_AxisState.ErrorStop state, FB\_Robot and/or IF\_Robot must be disabled and PLCO.MC\_Reset must be executed to reset the axis.
* IF\_RobotConfigurationAdvanced.xShiftMoveSyncStartPosition

  The start position on the connected path of a MoveSync command is shifted to the earliest possible position.

## Known Operational Anomalies

For Modicon M262 Motion Controllers with an axis configured as modulo, the property xEnable of FB\_Robot does not operate as intended.

If you set xEnable to TRUE a diagnostic message is reported.

Workaround:

1. Disable the robot.
2. Modify the axis from modulo to linear.
3. Enable the robot. **Result**: A diagnostic message is reported.
4. Disable the robot again.
5. Enable the robot.
