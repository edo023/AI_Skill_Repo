# IF_ProductManager - GetProductInterfaceById (Method)

![](images/logo.png)

IF\_ProductManager - GetProductInterfaceById (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Returns the interface of the product with the specified identification number

Description

The list of all products is searched and the product with the specified identification number is determined. The interface with this product is returned.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiProductId | UDINT | The identification number of the searched product |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) | General, library-independent statement on the diagnosis.  A value unequal GD.ET\_Diag.Ok equates an error message. |
| q\_etDiagExt | [ET\_DiagExt](../Enumerations/Enumerations-4.htm#XREF_D_SE_0080267_1) | POU-specific output on the diagnosis.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> error message |
| q\_ifProduct | [IF\_Product](Interfaces-14.htm#XREF_D_SE_0080325_1) | The interface with the product with the identification number |
