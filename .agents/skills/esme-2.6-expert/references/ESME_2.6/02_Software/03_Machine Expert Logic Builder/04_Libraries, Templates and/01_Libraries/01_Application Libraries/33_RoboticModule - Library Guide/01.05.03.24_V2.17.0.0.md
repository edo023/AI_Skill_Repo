## New Features

* **IF\_Configuration - AddTrackingSource()**

  Adding a configured tracking source to the robot.
* **IF\_Configuration - GetTrackingSource()**

  Getting the interface to a configured tracking source of the robot.

## Modifications

**Configuring a robot as a four-axis Delta robot is no longer supported.**

* IF\_Configuration - Delta4Ax() has been removed.
* IF\_Manual - xEnableAxisJoggingDelta4Ax has been removed.

**Functionalities related to Lexium S Robots are no longer supported.**

* IF\_Configuraton - BrakeReleaseButton() has been removed.
* ET\_Cmd - BrakeReleaseButton and Service have been removed.
* ET\_State - BrakeWaitUntilContactorOpen, BrakeWaitUntilContactorClosed, ServiceRunning, ServiceInitializing and ServiceExit have been removed.
* ET\_OpMode - BrakeReleaseButton and Service have been removed.
