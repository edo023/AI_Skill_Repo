# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | INT |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result contains the values that indicate the result of operations executed by the POUs of this library.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Ok | 0 | No error detected. |
| UnexpectedReturnValue | 1 | Indeterminable return value from system.  Contact your Schneider Electric representative. |
| InvalidHandle | 2 | No device or object connected to the specified handle. |
| SlaveChannelNotAddedToMultiAxisCustomJob | 3 | No subordinate channel was added to FB\_SlaveChannel or subordinate channel has been reset. |
| SlaveChannelInvalid | 4 | The specified subordinate channel is invalid. |
| NoSlaveChannelAddedToMultiAxisJob | 5 | No subordinate channel has been added to the multi-axis custom job. You must create at least one instance of FB\_SlaveChannel for FB\_MultiAxisCustomJob to work. |
| ASlaveChannelOfMultiAxisCustomJobWasUnableToStart | 6 | Reserved |
| CannotWriteMovementValuesToSlaveChannel | 7 | Movement values could not be written to the subordinate channel. |
| MaxNumberOfSlaveChannelsExceeded | 8 | An attempt has been made to add another subordinate channel. The additional subordinate channel exceeds the maximum number of subordinate channels. For details, refer to [Limitations](Multi-AxisCustomJobs-572EC46B.html#Multi-AxisCustomJobs-572EC46B__Limitations-57B6BFFF). |
| MasterChannelNotAddedToMultiAxisCustomJob | 9 | No master channel has been added to the multi-axis custom job. |
| MasterChannelInvalid | 10 | The master channel is invalid. |
| MasterMovementDataNotValid | 11 | The movement values of the master channel are invalid. |
| MasterSlaveCascadeFormsLoop | 12 | Master axis/subordinate axis function blocks create a loop (the master itself as subordinate axis, or a following subordinate axis with the initial master as subordinate axis).  Resolve the loop. |
| MaxNumberOfMasterChannelsExceeded | 13 | The number of master channels connected exceeds the maximum number of master channels [Limitations](Multi-AxisCustomJobs-572EC46B.html#Multi-AxisCustomJobs-572EC46B__Limitations-57B6BFFF).  Reduce the number of master channels. |
| MultiAxisCustomJobIsExecuting | 14 | The state of the multi-axis custom job is Executing. |
| MultiAxisCustomJobNotStarted | 15 | An attempt has been made to stop a multi-axis custom job that has not started. |
| MultiAxisCustomJobChannelLock | 16 | An attempt has been made to lock a channel of a multi-axis custom job that was not successful. |
| InvalidMultiAxisCustomJobCallback | 17 | The multi-axis custom job callback provided is invalid. |
| InvalidOperationForActiveMultiAxisCustomJob | 18 | The operation provided is invalid for the active multi-axis custom job. |
| NoBusCommunication | 19 | Communication via the motion bus is interrupted. |
| NotSupportedWithLogicalEncoder | 20 | An attempt has been made to execute a multi-axis custom job with a logical encoder axis.  A multi-axis custom jobs do not support logical encoder axes. |
| NotSupportedWithFeedbackAxis | 21 | An attempt has been made to execute a multi-axis custom job with an encoder axis.  A multi-axis custom jobs do not support encoder axes. |
| ModuloAxisNotSupported | 22 | An attempt has been made to execute a multi-axis custom job with a modulo axis.  A multi-axis custom jobs do not support encoder axes. |
| JobAborted | 23 | The running job was aborted by another job. |
| JobStartedWhileAxisIsHoming | 24 | The command cannot be executed while the axis is in the operating state Homing.Refer to [Executing a Multi-Axis Custom Job](Multi-AxisCustomJobs-572EC46B.html#Multi-AxisCustomJobs-572EC46B__ExecutingAMulti-AxisCustomJob-5731A411). |
| AxisIsDisabled | 25 | Function block cannot be executed because the axis is in the operating state Disabled.  Verify that the axis is not in the operating state Disabled when attempting to start a new function block. |
| AxisInErrorStop | 26 | Function block cannot be executed because an axis error has been detected and the axis is in operating state ErrorStop.  Verify that the axis is not in the operating state ErrorStop when attempting to start a new function block. |
| AxisIsStopping | 27 | Function block cannot be executed because an MC\_Stop function block is active and the axis is in the operating state Stopping.  Verify that the axis is not in the operating state Stopping when attempting to start a new function block. |
| AxisInvalid | 28 | No axis is specified or specified axis does not support the required function.  Verify that an axis has been specified and that the axis supports multi-axis custom jobs. |
| AxisAlreadyUsed | 29 | The axis is already used by another function block. |
| AxisIsHoming | 30 | The command cannot be executed while the axis is being homed. |
| DriveInError | 31 | The drive is in the operating state Error.  Use the function block MC\_Reset to reset the detected error. |
