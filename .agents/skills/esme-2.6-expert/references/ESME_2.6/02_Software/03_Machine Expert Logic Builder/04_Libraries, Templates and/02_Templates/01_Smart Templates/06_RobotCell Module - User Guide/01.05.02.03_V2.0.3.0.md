## New Features

* A 3D-view of the assigned modules is added.

## Modifications

* Orientation convention, surface plane and direction are set to fix values and cannot be changed.
* The order of the tabs is changed to improve the step by step configuration.
* The assigning of a targets handler for a target generator is changed. It no longer needs to be first enabled and can be used immediately. Targets handler are updated automatically.
* The structure for communication with Digital Twin is set automatically for all configured modules and target types of the RobotCell.

## Enhancements

* You can decide if you want to use an internal FB\_EntitiesHandler, or a global FB\_EntitiesHandler.
* For assigning a robot or conveyor you can select the ET\_SystemEntity value.
* New input/output variables iq\_astRobotModuleInterface and iq\_astConveyorModuleInterface are added.
* You can refresh the calculation for the relative transformation between robot and conveyor linked to a tracking system.
