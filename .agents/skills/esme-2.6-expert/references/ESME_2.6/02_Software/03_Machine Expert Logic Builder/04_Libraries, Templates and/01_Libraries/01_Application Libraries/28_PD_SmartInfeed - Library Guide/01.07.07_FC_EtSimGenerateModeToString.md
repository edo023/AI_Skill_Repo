# FC_EtSimGenerateModeToString

FC\_EtSimGenerateModeToString

FC\_EtSimGenerateModeToString - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Converting an enumeration element of type [ET\_SimGenerateMode](../Enumerations/Enumerations-7.htm#XREF_D_SE_0080273_1) of the PD\_SmartInfeed library to a string

Description

Every enumeration element has a name or value. The return value of the function is the name of the enumeration element.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etMode | [ET\_SimGenerateMode](../Enumerations/Enumerations-7.htm#XREF_D_SE_0080273_1) | The ET\_SimGenerateMode value to be converted |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) | General library-independent statement on the diagnostic.  A value unequal to GD.ET\_Diag.Ok corresponds to a diagnostic message. |
| q\_etDiagExt | [ET\_DiagExt](../Enumerations/Enumerations-4.htm#XREF_D_SE_0080267_1) | POU-specific output on the diagnostic.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> diagnostic message |

Return Value

| Data type | Description |
| --- | --- |
| STRING[80] | Name of the enumeration element that has been transferred at the i\_etType input |
