# General Information

## Library Overview

The SysTimeRtc library implements functions for the real time clock (RTC) management of the runtime system.

The SysTimeCore library provides access to high resolution time routines for time calculations.

Depending on the runtime system, the real time clock (RTC) of the controller is provided in standard or high resolution time stamp. That is, the time stamp represents the seconds (standard resolution) or milliseconds (high resolution) since January 1st, 1970 00:00:00. It is also known as UNIX format.

## Controller Platforms

The RTC in standard resolution is supported by all Schneider Electric controllers. Refer to the [table](D-SE-0084196.1.html#D-SE-0084196.1__D-SE-0084196.2) in the Document Scope chapter to verify for which controller supports the high resolution RTC.

## Function Template

In conjunction with the libraries, the function template RealTimeClock Control is provided. It provides a ready-to-use coding template to control the real time clock (RTC) of the controller in your application. For further information, refer to the [Function Templates User Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=DevTempl&topicID=RealTimeFuncTemp_8EC83A3F).
