ET\_Result

Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.2.0 |

Description

The enumeration ET\_Result contains the possible values that indicate the result of operations executed by the [POUs](../glossary/glossary.htm#XREF_D_SE_0076433_158) of this library.

Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Status information, indicated by q\_etResult if q\_xError = FALSE | | |
| Idle | 0 | The function block is ready to be executed. |
| OK | 1 | The function block has been executed successfully. |
| ReadIndexFile | 10 | The function block is reading the content of the file referenced by the input i\_sIndexFilePath. |
| ReadDriveParameter | 11 | The parameters of the drive that you defined in the [Index File](../../../../../../api/crossBook?lang=en-US&virtualBookName=Specifications/Specifications-2.htm#XREF_D_SE_0081722_1) are read. These identification numbers ([IDN](../../../SerComLi&topicID=D_SE_0055484_1)) are read via the Sercos III channel with an asynchronous task and are saved to the internal memory of the controller. |
| SaveConfiguration | 12 | The parameters are saved in a [Parameter File](../Specifications/Specifications-4.htm#XREF_D_SE_0081727_1) in the controller. The file and the path are referenced by the input i\_sParameterFilePath. |
| ReadParameterFile | 13 | The saved parameters (IDN) referenced by the input i\_sParameterFilePath are read into the internal memory. |
| WriteDriveParameter | 14 | The saved parameters (IDN) referenced by the input i\_sParameterFilePath are written via the Sercos III to the drive with an asynchronous task. |
| SaveToEEPROM | 15 | The parameters are stored in the nonvolatile memory area of the drive. |
| CreatingAsyncTask | 16 | An asynchronous task is being created for reading and writing Sercos service data. |
| Error information, indicated by q\_etResult if q\_xError = TRUE | | |
| CreatingAsyncTaskFailed | 30 | An error has been detected while creating an asynchronous task. |
| Timeout | 31 | The execution time exceeds the limit defined by GPL.Gc\_uiMaxExecutionTime.  Also refer to, [GPL (Global Parameter List)](../GPL/GPL.htm#XREF_D_SE_0081616_1). |
| InputParameterInvalid | 32 | The input i\_ifDrive is invalid or 0. |
| IdnNotationInvalid | 33 | A string defining a Sercos Parameter Identification Number (IDN) (included in the parameters sIndexFilePath or sParameterFilePath) does not meet the Sercos III specification.  Refer to the [Sercos III Identification Number](../../../../../../api/crossBook?lang=en-US&virtualBookName=SerComLi&topicID=D_SE_0055484_1) chapter of the SercosCommunication Library Guide. |
| AxisStateInvalid | 34 | Sercosdrive AxisState must be 1. |
| VendorIdMismatch | 35 | The VendorID defined in the CSV file does not correspond to the VendorDeviceID read out from the connected drive. |
| FileInconsistent | 37 | The [Index File](../Specifications/Specifications-2.htm#XREF_D_SE_0081722_1) or [Parameter File](../Specifications/Specifications-4.htm#XREF_D_SE_0081727_1) does not meet the specifications. |
| FilePathInvalid | 38 | The specified file path has invalid syntax. |
| FileInvalid | 39 | The content of the specified file to be read is not supported. |
| FileOpenFailed | 40 | An error has been detected while opening the file. |
| FileWriteFailed | 41 | An error has been detected while writing the file. |
| FileCloseFailed | 42 | An error has been detected while closing the file. |
| FileReadFailed | 43 | While reading the file, an internal error has been detected. |
| FilePathTooLong | 44 | The specified file path without file extension characters is not within the valid range. It is not possible to add the default file extension.  The file path without file extension is limited to 255 characters minus length of the default file. |
| FileNameTooLong | 45 | The specified file name including the file extension exceeds the length of 126 characters. |
| FileNameInvalid | 46 | The specified file name is invalid. |
| TableTooSmall | 47 | The internal table does not provide enough cells. Verify the value of GPL.Gc\_uiMaxNumOfRowsRead. |
| SercosDriveIssue | 50 | An error has been detected by the Sercos drive while reading or writing a parameter (For example, an invalid IDN has been detected).  Refer to the output q\_sResultMsg for further information on the invalid parameter.  Refer to the Error codes of the parameter channel (service channel) chapter of the EcoStruxure Machine Expert – Basic Online Help. |
| SercosPhaseIssue | 51 | The Sercos is not in communication phase 4. |
| SercosAddressInvalid | 52 | The logical address is invalid. |
| SercosServiceRequestIssue | 53 | A service request error has been detected. |
| SercosServiceTimeout | 54 | The Sercos service timeout. |
| SercosServiceNotSupported | 55 | The addressed drive does not support the ServiceData function. |
| SercosServiceInvalid | 56 | The service transfer is invalid. |
| UnexpectedFeedbackValue | 999 | An internal error has been detected.  Contact your Schneider Electric support. |
