## Enhancements

* The robot reports an exception if cartesian jogging is requested and the manual parameters are not configured for a cartesian component.
* The output q\_xException of the default module interface is reset when no exception of the module is active anymore.
* The behavior of IF\_Homing.xResetHomeOk is fully supported in the present version.
