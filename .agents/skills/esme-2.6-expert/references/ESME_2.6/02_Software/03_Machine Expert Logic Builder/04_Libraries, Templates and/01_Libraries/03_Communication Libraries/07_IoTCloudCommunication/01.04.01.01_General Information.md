# General Information

## Library Overview

The library IoTCloudCommunication provides the functionality to establish a connection from the machine controller to the EcoStruxure Machine Advisor cloud service and the transmission of machine data through this connection.

Preconditions for establishing a connection to the EcoStruxure Machine Advisor and to transmit data are:

* Your machine must be [registered](https://ecostruxure-machine-advisor.se.app/faq/track-manage-your-machine-fleet/) in your organization in EcoStruxure Machine Advisor.
* The connectivity configuration between the Machine Advisor and the controller (gateway) is done inside the MONITOR > Configuration Panel for your registered machine.

NOTE: During the configuration you will obtain the connection parameters which are required for the communication client in your application.

## Example Project

In conjunction with the library, an example project is provided. The example project demonstrates how to implement the components from the IoTCloudCommunication library.

The example project is installed on your PC along with the programming software. To open the project example, proceed as follows:

| Step | Action | Comment |
| --- | --- | --- |
| 1 | In the EcoStruxure Machine Expert Logic Builder, execute the command New Project. | – |
| 2 | In the New Project dialog box, select From Example from the Project type list. | – |
| 3 | On the right-hand side of the New Project dialog box, click the button Toggle Filter. | **Result**: Available examples are listed in the drop down menu. |
| 4 | Select your example from the drop down menu. | – |
| 5 | Select your controller from the Controllers list. | – |
| 6 | Enter a name for the new project, and select the file location. | – |
| 7 | Click the OK button. | **Result**: A new project is created based on the selected example. |

This approach places the controllers behind one or more firewalls to restrict access to authorized personnel and protocols only.
