# Machine Units of the TopLoader

The TopLoader machine consists of four separated machine units.

Each machine unit is represented in the TopLoader example application and controlled by a separated software unit. For further information, refer to [Software Units Representing Mechanical Units](ProjectStructure-7AD3B398.html#ProjectStructure-7AD3B398__SoftwareUnitsRepresentingMechanicUn-7B34E3C0).

| Machine unit => software unit | Graphical representation | Description |
| --- | --- | --- |
| PickPlace |  | Linear system that picks the products from the pickup station of the product belt and places the products into the carton on the carton belt. |
| ProductBelt |  | Transport belt that transports the products to the pickup station. |
| CartonBelt |  | Transport belt that transports the cartons to the station where the products are placed into the carton. From there, the transport belt moves the cartons filled with products to the sealing station, where the carton is closed and sealed. Afterwards, the transport belt moves the sealed cartons out of the machine. |
| Sealing |  | Workstation that closes and seals the cartons. |
