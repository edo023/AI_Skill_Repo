Description

The structure ST\_ModbusTCPConfiguration provides information on the Modbus TCP slaves.

Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| uiConfiguredSlaves | UINT | Number of configured Modbus TCP slaves detected. |
| uiConfiguredChannels | UINT | Number of configured channels detected. |
| uiOperationalChannels | UINT | Number of channels in operational state. |
