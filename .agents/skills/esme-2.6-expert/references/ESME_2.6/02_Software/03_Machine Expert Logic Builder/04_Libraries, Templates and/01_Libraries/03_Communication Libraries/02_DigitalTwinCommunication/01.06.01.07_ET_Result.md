# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.1.0.0 |

## Description

The enumeration ET\_Result specifies the possible values that indicate the result of operations executed by the POUs of this library.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Ok | 0 | Ok |
| EntityIdInvalid | 1 | The ID of the entity is invalid. |
| TargetIdInvalid | 2 | The ID of the target is invalid. |
| TargetTypeInvalid | 3 | The target type is invalid. |
| OrientationConventionInvalid | 4 | The orientation convention is invalid. |
| ColorInvalid | 5 | The color identifier is invalid. |
| BufferFull | 6 | The internal events buffer is full. |
| NoTwinConnected | 7 | No EcoStruxure Machine Expert Twin is connected. |
| ConnectionTimeOut | 8 | The connection to the EcoStruxure Machine Expert Twin was interrupted or terminated. Within the time defined with [udiTimeoutTwin](FB_EventWriter-25622A1A.html#FB_EventWriter-25622A1A__Properties-25625691), no connection to EcoStruxure Machine Expert Twin was detected. |
| ReferencePositionIgnored | 9 | The given reference position was ignored for the event. |
