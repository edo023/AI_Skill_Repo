# Behavior in the Operating Modes

![](images/logo.png)

Behavior in the Operating Modes

Command Processing

A successful executed command is displayed as follows:

q\_xCMDActive = FALSE

q\_diCMDActive = last command

q\_xCMDDone = TRUE

iq\_diCMD = 0

 Operation mode commands are not possible if an error reaction [TPL.ET\_Reaction](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1).AsyncStop, [TPL.ET\_Reaction](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1).SyncStopEL or [TPL.ET\_Reaction](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1).SyncStopEH is pending.

You cannot enable the position control via the [DriveEnableOn](../../../../../../api/crossBook?lang=en-US&virtualBookName=Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) command if an error reaction [TPL.ET\_Reaction](../../../PD.Lib.Template&topicID=D_SE_0078354_1).AsyncStop, [TPL.ET\_Reaction](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1).SyncStopEL is pending.

NoOp

No operation mode choosen. The module is in wait state.

The technology POU [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) is activated for each slave axis managed by the module.

The diagnostic messages of all managed slave axes are monitored.

Possible state commands

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is enabled.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is disabled.

Possible operation mode commands

o[Manual](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Manual](#XREF_D_SE_0080863_5) takes place.

o[Auto](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Automatic](#XREF_D_SE_0080863_6).

o[BrakeRelease](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [BrakeRelease](#XREF_D_SE_0080863_7) takes place.

Manual

The operation mode Manual is selected. The slave axes of the infeed distance managed by the module can be activated in jogging mode.

The operation mode is parameterized in the structures [ST\_Manual](../Structures/Structures-5.htm#XREF_D_SE_0080853_1) and [ST\_Main](../Structures/Structures-4.htm#XREF_D_SE_0080851_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [PDL.FB\_Jogging](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087305_1) are being activated.

Via the array [ST\_Manual](../Structures/Structures-5.htm#XREF_D_SE_0080853_1).i\_axBeltSelect, the axes are selected that are to be moved in jogging mode. The first element selects belt one, the second element belt two, etc. Only the elements of those belts are checked that are driven by an axis managed by the module.

The diagnostic messages of all managed slave axes are monitored.

Possible state commands

oi\_xJogForw and i\_xJogBack of the [ST\_Manual](../Structures/Structures-5.htm#XREF_D_SE_0080853_1) structure move the selected axes in positive and in negative direction.

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is enabled.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is disabled.

Possible operation mode commands

o[Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [NoOp](#XREF_D_SE_0080863_4) takes place.

o[Auto](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Automatic](#XREF_D_SE_0080863_6).

o[BrakeRelease](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [BrakeRelease](#XREF_D_SE_0080863_7) takes place.

Automatic

The operation mode Automatic is selected. The axes are moved in the infeed mode according to the algorithm.

The operation mode is parameterized in the structures [ST\_Automatic](../Structures/Structures-2.htm#XREF_D_SE_0080847_1) and [ST\_Main](../Structures/Structures-4.htm#XREF_D_SE_0080851_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [SI.FB\_Infeed](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SmartInfeed&topicID=D_SE_0080279_1) are being activated.

If [ST\_Automatic](../../../../../../api/crossBook?lang=en-US&virtualBookName=Structures/Structures-2.htm#XREF_D_SE_0080847_1).i\_xEnableFeedForw is set to TRUE, the technology POU [SI.FB\_MasterVel­FeedForward](../../../PD.Lib.SmartInfeed&topicID=D_SE_0080286_1) is additionally activated.

If [ST\_Simulation](../../../../../../api/crossBook?lang=en-US&virtualBookName=Structures/Structures-7.htm#XREF_D_SE_0080857_1).i\_xEnableSimulaiton is set to TRUE, the technology POU [SI.FB\_InfeedSim­ulation](../../../PD.Lib.SmartInfeed&topicID=D_SE_0080283_1) is additionally activated.

The diagnostic messages of all managed slave axes are monitored.

Possible state commands

o[Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Infeed is started.

o[Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The infeed is stopped.

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is enabled.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is disabled.

Possible operation mode commands

o[Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [NoOp](#XREF_D_SE_0080863_4) takes place.

o[Manual](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Manual](#XREF_D_SE_0080863_5) takes place.

o[BrakeRelease](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [BrakeRelease](#XREF_D_SE_0080863_7) takes place.

BrakeRelease

The operation mode BrakeRelease is selected. The slave axes of the infeed distance managed by the module are not moved. Only the holding brakes of the axes are released and engaged.

The operation mode is parameterized in the structures [ST\_BrakeRelease](../Structures/Structures-3.htm#XREF_D_SE_0080849_1) and [ST\_Main](../Structures/Structures-4.htm#XREF_D_SE_0080851_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [PDL.FB\_Brake](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087255_1) are being activated.

Via the array [ST\_BrakeRelease](../Structures/Structures-3.htm#XREF_D_SE_0080849_1).i\_axDriveSelect, the axes are selected at which the brake is to be released. The first element selects the drive of the first belt, the second element the drive of the second belt, etc. Only the elements of those belts are checked that are driven by an axis managed by the module.

The diagnostic messages of all managed slave axes are monitored.

Possible state commands

o[Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The selected holding brakes are released.

o[Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> All holding brakes are engaged.

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is enabled.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is disabled.

Possible operation mode commands

o[Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [NoOp](#XREF_D_SE_0080863_4) takes place.

o[Manual](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Manual](#XREF_D_SE_0080863_5) takes place.

o[Auto](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Automatic](#XREF_D_SE_0080863_6).
