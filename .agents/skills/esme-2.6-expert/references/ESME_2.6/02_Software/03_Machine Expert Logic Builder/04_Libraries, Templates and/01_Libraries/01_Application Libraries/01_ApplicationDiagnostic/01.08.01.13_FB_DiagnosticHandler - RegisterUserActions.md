## Task

Allows to register a callback for user-defined source code in case of a reaction.

## Description

The method RegisterUserActions is used to register an interface of type IF\_UserActions. In the event a reaction is triggered by a diagnostic message the method TriggerReaction is called by the instance of the diagnostic handler. The instance implementing the interface can be used to implement user defined logic that is executed in the event a reaction is triggered.

NOTE: The user-defined source code to be called must be added in the method TriggerReaction of a function block defined in the application which must implement the interface IF\_UserActions.

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Indicates TRUE if the callback for user actions has been registered successfully.  If FALSE is returned, an invalid interface is assigned to the input i\_ifDiagnosticListener. |

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifUserActions | IF\_UserAction | Specifies the interface referencing to the function block implementing the callback. |
