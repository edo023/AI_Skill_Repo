# FC\_Polynomial5AbsExtremeValuesDerivatives

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the extreme values of a fifth order equation. It determines the absolute extreme values of the fifth degree polynomial and its first and second derivative to the interval [i\_lrX0, i\_lrX1] given by the coefficients array i\_stCoefficients ("curve tracing").

If the absolute extreme values of a polynomial, which meets specific boundary conditions, are to be calculated, the coefficient array of the polynomial can first be calculated by using the function FC\_Polynomial5Coefficients.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrX0 | LREAL | Lower limit of the considered X interval. |
| i\_lrX1 | LREAL | Upper limit of the considered X interval. |
| i\_alrCoefficients | ARRAY[0..5] OF LREAL | Coefficient array of the polynomial for which the absolute extreme values are to be calculated, with the element 5 being the coefficient of the highest degree and the element 0 being the coefficient of the lowest degree. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
| q\_lrXAbsMin | LREAL | X value of the absolute minimum of the polynomial for the considered interval. |
| q\_lrYAbsMin | LREAL | Y value of the absolute minimum of the polynomial for the considered interval. |
| q\_lrXAbsMax | LREAL | X value of the absolute maximum of the polynomial for the considered interval. |
| q\_lrYAbsMax | LREAL | Y value of the absolute maximum of the polynomial for the considered interval. |

| q\_lrDeriv1YAbsMax | LREAL | Y value of the absolute maximum of the first derivative of the polynomial |
| q\_lrDeriv2XAbsMin | LREAL | X value of the absolute minimum of the second derivative of the polynomial |
| q\_lrDeriv2YAbsMin | LREAL | Y value of the absolute minimum of the second derivative of the polynomial |
| q\_lrDeriv2XAbsMax | LREAL | X value of the absolute maximum of the second derivative of the polynomial |
| q\_lrDeriv2YAbsMax | LREAL | Y value of the absolute maximum of the second derivative of the polynomial |
