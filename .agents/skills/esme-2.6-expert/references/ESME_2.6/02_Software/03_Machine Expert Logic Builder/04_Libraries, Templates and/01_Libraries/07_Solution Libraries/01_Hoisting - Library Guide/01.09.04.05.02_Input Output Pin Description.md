Input/Output Pin Description

Input/Output Pin Description

| Input/Output | Data Type | Description |
| --- | --- | --- |
| iq\_wExeCntActl | WORD | This pin is used for testing the number of executions of POUs within one cycle of a main cyclic task. Connect a global variable to this input. The global variable must be incremented by one within each called POU. The number of POUs must correspond to the value of i\_wExeCntCfg. The DiagnosticCoverage\_2 FB may be executed at any position within the main cycle task.  Range: 0...65535 |
