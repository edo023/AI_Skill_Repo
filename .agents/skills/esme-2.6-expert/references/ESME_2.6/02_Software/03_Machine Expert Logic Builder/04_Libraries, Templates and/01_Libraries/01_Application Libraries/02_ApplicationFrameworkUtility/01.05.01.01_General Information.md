# General Information

## Library Overview

The ApplicationFrameworkUtility library provides functions for the framework for the Modicon M660 motion controller.

The framework and, with it, the ApplicationFrameworkUtility library have been developed for the M660 platform as the successor of the PacDrive system.

The framework and the ApplicationFrameworkUtility library are also compatible with the M262 platform.
