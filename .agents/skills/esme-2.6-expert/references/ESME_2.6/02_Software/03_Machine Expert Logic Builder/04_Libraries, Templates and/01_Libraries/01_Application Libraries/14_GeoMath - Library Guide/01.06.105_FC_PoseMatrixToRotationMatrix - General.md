## Description

Given a pose matrix, the function returns a 3D matrix representing the rotational part of the input pose.

![](images/0000627868.png)

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stPoseMatrix | SE\_MATH.ST\_Matrix4D | A 4D matrix describing a Cartesian pose. |
