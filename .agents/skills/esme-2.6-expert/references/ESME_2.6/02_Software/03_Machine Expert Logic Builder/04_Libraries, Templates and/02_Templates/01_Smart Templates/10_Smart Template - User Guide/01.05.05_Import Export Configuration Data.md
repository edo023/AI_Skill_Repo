# Import / Export Configuration Data

![](images/logo.png)

Import / Export Configuration Data

Overview

The configuration data provides the state of the module. This includes the settings made in the Smart Template editors; for example, the selected robot type, camera settings, and so on.

The configuration data can be exported or imported using the contextual menu of the Configuration data node.

NOTE: The import command is not provided by all Smart Template modules.

Export

| Step | Action |
| --- | --- |
| 1 | Right-click the Configuration data node and select Export configuration from the contextual menu.  Result: A browse dialog is displayed. |
| 2 | Browse to the folder where you want to save the configuration data and confirm with Save. |

Import

| Step | Action |
| --- | --- |
| 1 | Right-click the Configuration data node and select Import configuration from the contextual menu.  Result: A browse dialog is displayed. |
| 2 | Browse to the folder that provides the configuration data you want to import and select the respective XML file. |
| 3 | Click Open.  Result: The configuration data is imported to your project. |
