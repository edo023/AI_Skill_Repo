# AbortDataTransmission (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method is used to abort a requested data transmission. The return value of type BOOL indicates TRUE if the data transmission was aborted.

## Inputs

| Parameter | Data Type | Description |
| --- | --- | --- |
| i\_ifStatus | [IF\_CallbackStatusSend](IF_CallbackStatusSend-GeneralInform-840E6DC2.html#IF_CallbackStatusSend-GeneralInform-840E6DC2) | The interface is used to identify the request for data transmission which has to be aborted. |
