# ET\_AuthenticationProtocol

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.3.4.0 |

## Description

The enumeration ET\_AuthenticationProtocol contains the supported methods for user authentication inside SNMPv3 requests.

## Enumeration Elements

| Name | Data type | Value | Description |
| --- | --- | --- | --- |
| None | BYTE | 0 | No authentication. |
| SHA1 | BYTE | 1 | Authentication with the protocol SHA1. |
| MD5 | BYTE | 2 | Authentication with the protocol MD5. |
