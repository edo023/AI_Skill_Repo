Overview

|  |  |

Description

Selection of edge detection during measured value recording. Determines for which edge [ST\_Touchprobe](../Structures/Structures-17.htm#XREF_D_SE_0089510_1).xCaptureOk: reports: = TRUE.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Rising | 1 | [ST\_Touchprobe](../Structures/Structures-17.htm#XREF_D_SE_0089510_1).xCaptureOk: = TRUE when a rising edge is detected by [ST\_Touchprobe](../Structures/Structures-17.htm#XREF_D_SE_0089510_1).ginTP |
| Falling | 2 | [ST\_Touchprobe](../Structures/Structures-17.htm#XREF_D_SE_0089510_1).xCaptureOk: = TRUE when a falling edge is detected by [ST\_Touchprobe](../Structures/Structures-17.htm#XREF_D_SE_0089510_1).ginTP |
