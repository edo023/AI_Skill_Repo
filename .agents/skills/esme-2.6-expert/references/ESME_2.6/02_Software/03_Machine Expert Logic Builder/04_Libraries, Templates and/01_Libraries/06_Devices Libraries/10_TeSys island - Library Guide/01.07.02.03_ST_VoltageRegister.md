# ST\_VoltageRegister

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.0.3.0 |

## Description

The structure ST\_VoltageRegister contains the maximum voltage magnitude for a voltage dip or swell and the start and stop time stamp of the voltage dip or swell.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| uiMagnitude | UINT | Maximum voltage magnitude for a voltage dip or swell.  (Unit: V) |
| stStartDate | SysTimeRtc.RTS\_SYSTIMEDATE | Start time stamp of the voltage dip or swell.  Refer to [SysTimeRtc library](../../../../../api/crossBook?lang=en-US&virtualBookName=RTCmanag&topicID=D_SE_0067239). |
| stStopDate | SysTimeRtc.RTS\_SYSTIMEDATE | Stop time stamp of the voltage dip or swell.  Refer to [SysTimeRtc library](../../../../../api/crossBook?lang=en-US&virtualBookName=RTCmanag&topicID=D_SE_0067239). |
