## Overview

NOTE: This library is for projects exclusively.

The library Collision Detection V1.0.0.0 is released for projects exclusively.

## Description

This library implements a set of functions for the collision detection in a 3D environment with the following purposes:

* Define bounding volumes in a 3D space.
* Perform a collision query between two or more bounding volumes.
* Perform a minimum distance calculation between two or more bounding volumes
