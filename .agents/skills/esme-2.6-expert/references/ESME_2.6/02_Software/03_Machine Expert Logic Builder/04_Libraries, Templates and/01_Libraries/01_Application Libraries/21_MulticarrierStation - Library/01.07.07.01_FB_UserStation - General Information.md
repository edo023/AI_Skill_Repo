## Task

Implementing a user-defined station.

## Description

For application-specific requirements, you can create user-defined stations and adapt them to your needs.

If you use the Update > To Code command in the Multicarrier Configuration editor, a function block is generated for each user station. The function blocks are instantiated by the program [SR\_CallStationsGenerated](SRCallStationsGenerated-E3E2C410.html#SRCallStationsGenerated-E3E2C410).

Since the user station functions blocks can be modified, they are not overwritten when you re-generate the code by using the Update > To Code command.

For helping to demonstrate the creation of user-defined stations, refer to the Example project with application examples within the EcoStruxure Machine Expert software. For more information, refer to the [Lexium™ MC multi carrier Example Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=exMulCar&topicID=).
