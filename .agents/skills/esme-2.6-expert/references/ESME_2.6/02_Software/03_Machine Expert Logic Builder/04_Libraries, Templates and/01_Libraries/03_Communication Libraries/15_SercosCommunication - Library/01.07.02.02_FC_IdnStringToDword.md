# FC\_IdnStringToDword

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.2.1.0 |

## Functional Description

The function FC\_IdnStringToDword builds the structured IDN with the symbolic addressing input given as STRING.

Accepted input formats: `<S/P>-<PS>-<DBN> or <S/P>-<PS>-<DBN>.<SI>.<SE>`

Examples:

| Input i\_sIdnAddress | IDN as DWORD |
| --- | --- |
| S-0-0023 | 23 |
| S-0-1040.0.128 | 8389648 |
| P-0-3006.0.10 | 691134 |

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sIdnAddress | STRING [20]] | Symbolic parameter address as STRING |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etResult | ET\_Result | Provides diagnostic and status information as an enumeration value. |

| Data type | Description |
| --- | --- |
