Task

Calculation of a unit vector which is orthogonal relative to a given vector in the 3-dimensional space.

Description

In the 3-dimensional space, there is an infinite number of unit vectors which are orthogonal relative to a given vector different from zero. This function calculates such a vector. Using the function [FC\_CrossProduct](Functions_A_to_J-37.htm#XREF_D_SE_0087457_1), an orthogonal tripod can then be constructed from these two vectors.
