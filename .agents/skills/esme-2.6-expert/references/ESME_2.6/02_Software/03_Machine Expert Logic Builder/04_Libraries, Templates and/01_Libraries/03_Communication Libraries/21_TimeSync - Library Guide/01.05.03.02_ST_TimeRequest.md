Description

The structure ST\_TimeRequest contains specific information for sending an [SNTP](../glossary/glossary.htm#XREF_D_SE_0024697_818) request to an (S)NTP time server.

Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| i\_sServerIp | STRING[15] | IP address of the (S)NTP time server. It must be unique address that consists of four numbers in the range of 0...255 separated by periods. The IP address has to be unique in the network. |
| i\_uiServerPort | UINT | [SNTP](../glossary/glossary.htm#XREF_D_SE_0024697_818) port of the time server.  If the input is empty, the default port 123 is used. |
| i\_timServerTimeOut | TIME | Maximum time to wait for a response from the server. The value must be greater than zero. |
| i\_etComMode | ET\_ComMode | The communication mode defined with the ET\_ComMode [enumeration](../Enumerations/Enumerations-2.htm#XREF_D_SE_0081065_3). |
