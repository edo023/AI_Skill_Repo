# GVL (Global Variable List) - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Global variables |
| Available as of: | V1.0.0.0 |

## Description

GVL (Global Variable List) of the AsyncManager library.

## Global Variables

| Variable | Data type | Description |
| --- | --- | --- |
| G\_ifAsyncMgr | IF\_AsyncManager2 | This instance is the interface to the AsyncManager. This interface has to be used to initialize the AsyncManager and to provide the program part for asynchronous execution. |
| G\_xOnlineChangeRequested | BOOL | This variable is set to TRUE when an online change is requested and is about to be executed. When the variable is FALSE, it can be used as a condition to allow the start of asynchronous jobs. As long as the variable is TRUE, asynchronous jobs should not be started.  NOTE: A high number of pending asynchronous jobs in the queue, as well as long execution times of individual jobs, may delay or even block the online change process. |
