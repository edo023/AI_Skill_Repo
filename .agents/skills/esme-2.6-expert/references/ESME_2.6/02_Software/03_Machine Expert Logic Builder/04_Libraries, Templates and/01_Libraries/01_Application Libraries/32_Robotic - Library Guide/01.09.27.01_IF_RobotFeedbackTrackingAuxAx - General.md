## Task

Interface with return values of an auxiliary axis tracking movement of the robot.

## Description

In case of an active auxiliary axis tracking movement in a configured tracking source configured by IF\_RobotConfiguration.AddTrackingSource, the feedback values are provided within this feedback interface.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| lrEstimatedStopPosition | LREAL | Read | Estimated stop position for the auxiliary axis where an active tracking will stop. |
| lrOffsetPosition | LREAL | Read | Offset position of the tracking movement for the auxiliary axis. |
