## Description

The method CleanGroup of the function block [FB\_MultiAxisCustomJob](FB_MultiAxisCustomJob-4CFA57F9.html) removes all master channels and subordinate channels that were previously added to the multi-axis group. This resets the group configuration to its initial state.

This method can only be called when the function block is in the Idle state. If the method is called while the multi-axis custom job is active (for example, in the Executing state), the system detects an error and the group configuration remains unmodified.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
