# Brake Wear Detection

![](images/logo.png)

Brake Wear Detection

Calculation of the Pulse Frequency

When the drive is not in RUN state and the motor moves, the function block collects the pulses and converts them to the revolutions of the motor shaft. If the calculated rotations are greater than the defined revolution threshold value i\_rRevoBrakWearMax input, the break wear alarm is signaled.
