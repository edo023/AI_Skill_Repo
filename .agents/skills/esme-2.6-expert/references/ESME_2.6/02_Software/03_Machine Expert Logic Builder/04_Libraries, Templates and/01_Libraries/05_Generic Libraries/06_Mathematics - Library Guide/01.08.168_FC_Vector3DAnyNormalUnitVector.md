# FC\_Vector3DAnyNormalUnitVector

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

In the three-dimensional space, there is an infinite number of unit vectors which are orthogonal relative to a given vector different from zero. This function calculates such a vector. Using the function FC\_Vector3DCrossProduct, an orthogonal tripod can then be constructed from these two vectors.
