# ET_Cmd

![](images/logo.png)

ET\_Cmd

ET\_Cmd - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

Description

The enumeration describes all the commands of the FB\_UnwinderModule.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Manual | 520 | Change to operating mode Manual. |
| Automatic | 530 | Change to operating mode Automatic. |
| BrakeRelease | 540 | Change to operating mode Brake. |
| Start | 570 | Start the selected operating mode. |
| Stop | 571 | Cancel the command and return to the operating mode NoOp. |
| Hold | 572 | Cancel start command. |
| DriveEnableOn | 573 | Enable the position control of the axis. |
| DriveEnableOff | 574 | Disable the position control of the axis. |
