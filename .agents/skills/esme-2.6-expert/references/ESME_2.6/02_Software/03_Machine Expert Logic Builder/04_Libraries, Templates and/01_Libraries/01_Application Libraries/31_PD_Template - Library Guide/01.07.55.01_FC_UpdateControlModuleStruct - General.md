# FC\_UpdateControlModuleStruct - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.4.0 |

## Task

Function for initialization and update of values in an array of type ST\_ControlModuleInterface.

## Description

The function initializes and updates values of type ST\_ControlModuleInterface in an array. The values that must be updated are addresses of interfaces which are designed Control Module. A control module is a normal function block, which is used to control specific hardware components. Such a function block has no typical characteristics of an equipment module. In the beginning and at the end of every online change, an update of the addresses is necessary.
