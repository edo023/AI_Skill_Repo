# ET_MotionActiveVelType

![](images/logo.png)

ET\_MotionActiveVelType

ET\_MotionActiveVelType - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Description

The enumeration describes all velocity sources on the basis of which the outputs [q\_xMotionActive](../Structures/Structures-4.htm#XREF_D_SE_0080851_1) and [q\_axMotionActiveBelt](../Structures/Structures-4.htm#XREF_D_SE_0080851_1) of the [FB\_SmartInfeedModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080825_1) are set if the velocity exceeds [i\_lrMotionActiveVelLimit](../Structures/Structures-4.htm#XREF_D_SE_0080851_1).

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| RefVelocity | 1 | Reference velocity of the axis |
| Velocity | 2 | Velocity of the axis |
| MechRefVelocity | 3 | Dead time compensated reference velocity of the axis |
| MechVelocity | 4 | Dead time compensated velocity of the axis |
