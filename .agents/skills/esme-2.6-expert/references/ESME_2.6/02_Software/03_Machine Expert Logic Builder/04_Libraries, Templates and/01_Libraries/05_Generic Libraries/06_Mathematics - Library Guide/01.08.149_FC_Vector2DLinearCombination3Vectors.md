# FC\_Vector2DLinearCombination3Vectors

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the linear combination of three vectors in the two-dimensional space: i\_lrScalarA \* i\_stVectorA + i\_lrScalarB \* i\_stVectorB + i\_lrScalarC \* i\_stVectorC.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrScalarA | LREAL | Scalar A |
| i\_stVectorA | [ST\_Vector2D](ST_Vector2D-GeneralInformation-0BFF6B0C.html#ST_Vector2D-GeneralInformation-0BFF6B0C) | Vector A |
| i\_lrScalarB | LREAL | Scalar B |
| i\_stVectorB | [ST\_Vector2D](ST_Vector2D-GeneralInformation-0BFF6B0C.html#ST_Vector2D-GeneralInformation-0BFF6B0C) | Vector B |
| i\_lrScalarC | LREAL | Scalar C |
| i\_stVectorC | [ST\_Vector2D](ST_Vector2D-GeneralInformation-0BFF6B0C.html#ST_Vector2D-GeneralInformation-0BFF6B0C) | Vector C |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |

| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |

## Return Value

| --- | --- |
