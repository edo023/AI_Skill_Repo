Status Description

Status Description

Description

The status indicates the current status of AutoTune and of the drive to be optimized.

| Status text and meaning |
| --- |
| Off: Controller optimization is not enabled. The AUT.G\_xAutoTuneEnable input is false. |
| Wait for sercos phase 4: Controller optimization is enabled, but the sercos is not in phase 4. This is why the drive cannot be optimized. |
| Inactive: The controller optimization is enabled but the selected axis is not ready for operation. Therefore the drive cannot be optimized. |
| Aborted and inactive: Optimization has been canceled and the selected axis is not ready for operation. Therefore the drive cannot be optimized. |
| Internal error: AutoTune has detected an error. Therefore the drive cannot be optimized. |
| External error: The drive has detected an error. Therefore the drive cannot be optimized. |
| Jogging: The drive is in jogging mode. |
| Positioning: The drive is being positioned. |
| Ready to optimize, waiting for configuration ok: The selected drive would be ready for optimization, but the configuration of the axis limits has not yet been acknowledged. |
| Ready to optimize: The selected drive is ready for optimization. |
| Aborted and ready to optimize: Optimization was canceled and the selected drive is ready for optimization. |
| Start: Optimization has been started. |
| SelectNextOptimizationPart: AutoTune decides which parameters will be optimized in the next step. |
| Optimizing velocity controller, pause: Optimization of the velocity controller has been stopped. By pressing the pause button optimization is continued. |
| Optimizing velocity controller, release brake: The speed controller is being optimized. The motor brake is being released. |
| Optimizing velocity controller, start noise: The speed controller is being optimized. The noise generator has been started. |
| Optimizing velocity controller, measurement: The speed controller is being optimized. The measured values for calculating the Bode plot are traced. |
| Optimizing velocity controller, best possible stop: The speed controller is being optimized. The drive is being stopped. |
| Optimizing velocity controller, wait brake: The speed controller is being optimized. The drive is waiting for the motor brake to engage. |
| Optimizing velocity controller, calculate Bode: The speed controller is being optimized. The Bode plot is calculated. |
| Optimizing velocity controller, calculate parameters: The speed controller is being optimized. New values are calculated for the controller parameters Vel\_P\_Gain, Vel\_I\_Gain, VelFilter and CurrRefFilter and the J\_Load parameter. |
| Optimizing velocity controller, done: Optimization of the velocity controller has been completed. |
| Optimizing velocity controller, aborted: Optimization of the velocity controller was canceled. |
| Optimizing position controller, pause: Optimization of the position controller has been stopped. By pressing the pause button optimization is continued. |
| Optimizing position controller, release brake: The position controller is being optimized. The motor brake is being released. |
| Optimizing position controller, start noise: The position controller is being optimized. The noise generator has been started. |
| Optimizing position controller, measurement: The position controller is being optimized. The measured values for calculating the Bode plot are traced. |
| Optimizing position controller, best possible stop: The position controller is being optimized. The drive is being stopped. |
| Optimizing position controller, wait brake: The position controller is being optimized. The drive is waiting for the motor brake to engage. |
| Optimizing position controller, calculate Bode: The position controller is being optimized. The Bode plot is calculated. |
| Optimizing position controller, calculate parameters: The position controller is being optimized. A new value is calculated for the controller parameter Pos\_P\_Gain. |
| Optimizing position controller, done: Optimization of the position controller has been completed. |
| Optimizing position controller, aborted: Optimization of the position controller has been canceled. |
| Optimization done, no parameters found: Optimization was completed without an error. No parameters were found for at least one optimization area. |
| Optimization done, parameters have to be checked: The optimization was completed without errors. For at least one optimization area parameters were found that have to be checked. |
| Optimization done, parameters were found: The optimization was completed without errors. For all optimization areas parameter were found. |
| No parameters found, inactive: For at least one optimization area no parameters were found. Another optimization is currently not possible because the selected axis is not ready for operation. |
| Parameters that have to be checked, inactive: For at least one optimization area parameters were found that have to be checked. Another optimization is currently not possible because the selected axis is not ready for operation. |
| Parameters found, inactive: For all the optimization areas parameters were found. Another optimization is currently not possible because the selected axis is not ready for operation. |
| No parameters found, ready to optimize: For at least one optimization area no parameters were found. The selected drive is ready for another optimization. |
| Parameters that have to be checked, ready to optimize: For at least one optimization area parameters were found that have to be checked. The selected drive is ready for another optimization. |
| Parameters found, ready to optimize: For all the optimization areas parameters were found. The selected drive is ready for another optimization. |
