# FB_PathForJerkPositioning - SetSingleJerkPhase (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.5.3.0 |
| Versions: | Current version |

Task

Set a path consisting of a single jerk.

Description

This method sets a path consisting of a single jerk. The duration and the jerk value are specified.

The path starts the position that was set with SetInitialPositions.

This method provides no diagnostic information and no direct feedback value of the method.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrT0 | LREAL | Duration of jerk phase in s |
| i\_lrJ0 | LREAL | Jerk value as vectors of the single phase in units/s3 |
