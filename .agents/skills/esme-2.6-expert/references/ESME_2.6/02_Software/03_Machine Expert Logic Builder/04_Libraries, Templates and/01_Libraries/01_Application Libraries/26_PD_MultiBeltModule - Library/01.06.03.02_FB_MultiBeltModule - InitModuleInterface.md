# FB\_MultiBeltModule - InitModuleInterface (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: |  |
| Versions: | Current version |

## Task

ST\_ModuleStation and ST\_ModuleTrain contain references on MultiBelt data. The InitModuleInterface method establishes these references.

## Description

As input the method contains the ST\_ModuleInterface and thereby establishes the reference on the MultiBelt data. It is required to call up this method once when initializing the MultiBeltModule. See [Example of an Init\_MultiBeltModule action](D-SE-0077917.html#D-SE-0077917).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| iq\_stMultiBeltModuleItf | [ST\_ModuleInterface](D-SE-0079141.html#D-SE-0079141) | The MultiBeltModuleInterface of the node in which the FB\_MultiBeltModule is integrated. |
