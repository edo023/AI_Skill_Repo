# ST_Main

ST\_Main

ST\_Main - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.2.0 |
| Inherits from: | - |
| Versions: | Current version |

Description

The structure contains cross-operating mode parameters and return values of the [FB\_AxisModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0077136_1).

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_ifDrive | SystemConfigurationItf.IF\_Drive | The axis to be activated by the axis module. |
| i\_ifMaster | SystemConfigurationItf.IF\_Master | The master for the axis module. |
| i\_timMasterStop | TIME | The maximum period within which the i\_ifMaster master must have stopped in the case of an error reaction [SynchronStopEh](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1) or [SynchronStopEl](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1). |
| i\_xHwLimitPos | BOOL | Here, the signal of a hardware limit switch for the positive direction of motion can be transferred. The parameter must be written cyclically. |
| i\_xHwLimitNeg | BOOL | Here, the signal of a hardware limit switch for the negative direction of motion can be transferred. The parameter must be written cyclically. |
| i\_xStart | BOOL | Start signal for all operating modes. Can be used instead of the command Start. |
| i\_lrMotionActiveVelLimit | LREAL | Velocity as of which a motion is reported to q\_xMotionActive. |
| i\_etMotionAciveVelType | [ET\_MotionActiveVelType](../Enumerations/Enumerations-4.htm#XREF_D_SE_0077128_1) | Velocity mode for determining the motion for q\_xMotionActive. |
| stReducedVelocity | [PDL.ST\_ReducedVelocity](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087790_1) | Parameter for the functionality "Reduced velocity". The [Operating modes](../Function_Blocks/Function_Blocks-5.htm#XREF_D_SE_0077326_1) NoOp, Homing, Manual, Multicam, Endless, EndlessIls, Positioning, BrakeRelease and Additional support the feature "Reduced velocity". |
| iq\_lencEncoder | SystemConfiguration.L\_ENC | The logical encoder assigned to the axis module. |
| q\_xDriveActive | BOOL | TRUE: The position control of the axis is activated. |
| q\_etOpMode | [ET\_AxisModuleOpModes](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) | The active operating mode of the axis module. |
| q\_etState | [ET\_AxisModuleState](../Enumerations/Enumerations-7.htm#XREF_D_SE_0077134_1) | The active state of the axis module. |
| q\_xHomeOk | BOOL | TRUE: Homing has been performed successfully. |
| q\_xSynchron | BOOL | TRUE: The axis synchronously follows the master. |
| q\_lrPosition | LREAL | The axis position. |
| q\_lrPositionX | LREAL | The master position. |
| q\_sDriveName | STRING[40] | The name of the slave axis. |
| q\_xMotionActive | BOOL | TRUE: The axis is in motion according to the definition of i\_lrMotionActiveVelLimit and i\_etModtionActiveVelType. |
