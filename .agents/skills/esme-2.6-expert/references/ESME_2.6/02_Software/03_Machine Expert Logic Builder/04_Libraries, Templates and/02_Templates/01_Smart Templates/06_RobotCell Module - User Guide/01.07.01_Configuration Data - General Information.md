# Configuration Data - General Information

## Overview

| Step | Action |
| --- | --- |
| 1 | Go to the editor Modules (View > Modules). |
| 2 | Double-click the module node.  Alternatively, you can double-click Configuration data. |

## Configuration Data Tabs

The RobotCell Module Configuration Data window provides several tabs to configure the corresponding objects of the RobotCell Module .

| Tab | Description |
| --- | --- |
| [General Tab](GeneralTab-66C10EAB.html) | To display and to configure RobotCell Module default parameters. |
| [Conveyors Tab](ConveyorsTab-674BAE11.html) | To display and to configure the conveyors, which are available in the RobotCell Module. |
| [Target Generators Tab](TargetGeneratorsTab-68CB6017.html) | To display the modules available in the RobotCell Module that can create targets. |
| [Target Handlers Tab](TargetHandlersTab-68D9ADC1.html) | To create target handlers, for example, to handle targets on a conveyor (for example, to add/remove targets and track targets). |
| [Tracking Systems Tab](LinearTrackingSystemsTab-68D2C395.html) | To display and to configure the tracking systems of the RobotCell Module. |
| [Target Types Tab](TargetTypesTab-68E02DDC.html) | To create target types, for example, to display different target types with regard to shape, size, and color in a visualization. |
| [Robots Tab](RobotsTab-66D3DF61.html) | To display and to configure the robots, which are available in the RobotCell Module. |
| [3D Layout Tab](DLayoutTab-8AB2EEAE.html) | To display the conveyors, target generators and robots shown in the 3D Scene of the Robotcell. |
| [Explorer Tab](ExplorerTab-68E33570.html) | To display the user interface of the RobotCell Module. |
