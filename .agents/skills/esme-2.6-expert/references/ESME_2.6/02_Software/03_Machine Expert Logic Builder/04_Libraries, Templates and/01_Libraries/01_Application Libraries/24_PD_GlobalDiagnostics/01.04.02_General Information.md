# General Information

![](images/logo.png)

General Information

Description

The library "PD\_GlobalDiagnostics" contains the enumeration ET\_Diag. This is an enumeration by means of which a general statement on the diagnostic of POUs is made, independent of any library. It also contains converting functions for enumerations.

A value unequal to ET\_Diag.Ok corresponds to an exception message.

Namespace of the Library

The default namespace of this library is GD.

The POUs, data structures, enumerations and constants have to be addressed via this namespace.
