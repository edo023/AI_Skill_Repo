## Task

Interface with return values for the active path segment.

## Description

This interface is part of IF\_RobotFeedback and contains data on the active motion job.

A motion job can be issued with the method IF\_RobotMotion.MoveL, IF\_RobotMotion.MoveS or IF\_RobotMotion.MoveC.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| lrMaxAcceleration | LREAL | Read | Parameterized maximum acceleration of the motion job or the motion segment.  Unit: [Units/s²] |
| lrMaxDeceleration | LREAL | Read | Parameterized maximum deceleration of the motion job.  Unit: [Units/s²] |
| lrMaxEndZone | LREAL | Read | Configured maximum size of the zone at the end of the motion job.  This zone was defined by using the motion command.  Unit: [Units] |
| lrMaxStartZone | LREAL | Read | Configured maximum size of the zone at the start of the motion job.  This zone was defined by using the previous motion command.  Unit: [Units] |
| lrMaxVelocity | LREAL | Read | Parameterized maximum velocity of the motion job.  Unit: [Units/s] |
| lrRamp | LREAL | Read | Parameterized ramp of the motion job.  Unit: [-] |
| lrStartZone | LREAL | Read | Size of the zone at the start of the motion command.  This zone was defined by using the previous motion command and shortened if needed.  Unit: [Units] |
| lrEndzone | LREAL | Read | Size of the zone at the end of the motion command.  This zone was defined by using the motion command and shortened if needed.  Unit: [Units] |
