# ST\_ResAccLimitParameters - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V2.6.1.0 |
| Inherits from: | - |

## Description

Structure to describe additional parameters of the functionality Resultant Acceleration Limitation Space.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| `xEnablePreAnalysis` | `BOOL` | TRUE: Enables the analysis of the calculated default motion profile in case the parametrized limit for the resulting acceleration space would be exceeded. In case the limit would be exceeded, a motion profile is calculated so that the resulting acceleration is limited to the parametrized value. |
| `xDisableLimitationOnLinear` | `BOOL` | TRUE: Disables the limitation of the resulting acceleration for linear movements. |
