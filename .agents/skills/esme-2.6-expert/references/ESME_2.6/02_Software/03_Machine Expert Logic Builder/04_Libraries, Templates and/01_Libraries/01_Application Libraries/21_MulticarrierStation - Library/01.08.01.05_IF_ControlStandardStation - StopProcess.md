# IF\_ControlStandardStation - StopProcess (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

![](images/0000760266.png)

## Task

Stopping the carriers in the process.

## Description

With the method StopProcess, you can stop the carriers in the process and also the carriers that are on the way into the process.

For reactivating the process, you must call the method [Restart](Restart-EED689B5.html#Restart-EED689B5).

## Inputs

The method has no inputs.

## Outputs

The method has no outputs.
