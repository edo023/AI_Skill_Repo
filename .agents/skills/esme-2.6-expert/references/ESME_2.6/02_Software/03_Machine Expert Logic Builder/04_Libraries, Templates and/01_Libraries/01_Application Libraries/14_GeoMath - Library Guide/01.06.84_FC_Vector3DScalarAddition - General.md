## Description

Given a 3D vector and a scalar value, the function returns a new 3D vector resulting from adding the scalar value to each element of the input vector.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stVector | SE\_MATH.ST\_Vector3D | A 3D vector. |
| i\_lrScalar | LREAL | A scalar value to add to the elements of i\_stVector. |

| Output | Data type | Description |
