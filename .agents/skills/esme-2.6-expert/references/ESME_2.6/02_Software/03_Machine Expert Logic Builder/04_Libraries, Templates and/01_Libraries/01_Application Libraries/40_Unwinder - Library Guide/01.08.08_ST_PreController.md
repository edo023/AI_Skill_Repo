# ST_PreController

![](images/logo.png)

ST\_PreController

ST\_PreController - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

Description

This structure contains the parameters and feedback data for the precontol movement of the FB\_Unwinder.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_etMode | ET\_PreControllerMode | To select which PreController mode shall be used. |
| stMaster | ST\_PreControllerMaster | Parameters and feedback data for the PreController with a master input. |
| stVelocity | ST\_PreControllerVelocity | Parameters and feedback data for the PreController with a set velocity input. |
| q\_lrPreControlPosition | LREAL | Actual position of the bobbin based on the PreController. This output can be used to analyze and optimize the control behavior of the unwinder. |
