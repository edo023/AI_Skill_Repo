Description

In this structure, the parameters and the return values of the operation mode Automatic of the [FB\_SmartInfeedModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080825_1) are summarized.

NOTE: The iq\_ifInfeed element may only be accessed after the [InitModuleInterface](../Function_Blocks/Function_Blocks-4.htm#XREF_D_SE_0080826_1) method of the [FB\_SmartInfeedModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080825_1) has been called up. Otherwise, an exception error is generated that causes an immediate program stop.
