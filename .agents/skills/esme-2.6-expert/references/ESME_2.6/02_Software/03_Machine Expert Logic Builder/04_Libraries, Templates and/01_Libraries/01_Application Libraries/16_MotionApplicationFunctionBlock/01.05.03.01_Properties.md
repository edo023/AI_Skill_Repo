# Properties

| Property | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ifLoggerPoint | APL2.IF\_LoggerPointConfiguration | Read | Returns an interface to configure a logger point.  APL2 is the default namespace of the ApplicationLogger2 library. For further information, refer to the [ApplicationLogger2 Library Guide.](../../../../../api/crossBook?lang=en-US&virtualBookName=APL2LG&topicID=ET_LogLevel_37599385) |

The function blocks of the MotionApplicationFunctionBlocks library support the following logger levels:

| Logger level | Description |
| --- | --- |
| Exception | A logger entry is created when an error is detected. |
| ExternalEvent | A logger entry is created when an external event occurs (for example, a Touchprobe is triggered). |
| StatusMessage | A logger entry is created when the status of the function block changes (for example, the function block is enabled). |
| DebugMessage | Additional logger entries for debugging (for example, internal stop command). |
