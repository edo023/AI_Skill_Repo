# ET_PathCommand

![](images/logo.png)

ET\_PathCommand

ET\_PathCommand - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.4.0 |

Description

This enumeration defines the commands which can be executed by a path ([ST\_Path](../Structures/Structures-3.htm#XREF_D_SE_0090702_1)).

Type: UINT

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Resume | 0 | Resume movement of the vehicles on this path. |
| Suspend | 1 | Suspend movement of the vehicles on this path. |
| RequestStatus | 2 | Read the data of the path from the transport system and update the outputs of the function block. |
