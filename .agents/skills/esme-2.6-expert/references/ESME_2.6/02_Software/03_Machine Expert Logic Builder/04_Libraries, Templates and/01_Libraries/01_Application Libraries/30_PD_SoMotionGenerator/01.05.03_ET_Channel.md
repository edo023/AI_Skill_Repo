# ET_Channel

![](images/logo.png)

ET\_Channel

ET\_Channel - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.0.0 |

Description

List type for selecting the channel when assigning new motion jobs. The enumeration value None must not be used for assigning new motion jobs.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No channel selected |
| A | 1 | Channel A |
| B | 2 | Channel B |
| C | 3 | Channel C |
