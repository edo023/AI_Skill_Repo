Task

Initializing the memory area of [FB\_MovingAverage](Function_Blocks_I_to_Q-9.htm#XREF_D_SE_0087308_1)

Description

The memory area of [FB\_MovingAverage](Function_Blocks_I_to_Q-9.htm#XREF_D_SE_0087308_1) is being initialized. After calling up this method the properties of [FB\_MovingAverage](Function_Blocks_I_to_Q-9.htm#XREF_D_SE_0087308_1) provide the following values:

olrAverage: 0.0

ouiLevel: 0

ouiOrder: Value of i\_uiOrder, if valid

This method has to be called up at least once before calling up the Insert method.
