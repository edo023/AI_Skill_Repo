# IF\_ConnectionManager.GetNumberOfOpenConnections (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

The method GetNumberOfOpenConnections indicates the number of open connections that are held by the global Modbus TCP connection manager.

## Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| GetNumberOfOpenConnections | USINT | The return value indicates the number of open connections that are held by the global Modbus TCP connection manager. |
