# ET\_PositionTrend

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_PositionTrend specifies whether set positions of the axis are considered for sampling.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| WithSetPos | 0 | The set positions of the axis are considered for sampling. This results in a trace graph with setpos positions. |
| Continuous | 1 | The axis position value that is first sampled is calculated on actual mechanical position. This results in a continuous trace graph. |
