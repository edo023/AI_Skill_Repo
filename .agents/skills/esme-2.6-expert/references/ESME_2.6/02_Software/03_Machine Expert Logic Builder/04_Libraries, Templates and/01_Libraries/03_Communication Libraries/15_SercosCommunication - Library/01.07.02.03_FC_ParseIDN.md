# FC\_ParseIDN

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Functional Description

The function FC\_ParseIDN parses an IDN given as DWORD to the single elements. It has no return value.

NOTE: The validity of the input is indeterminable by the function, therefore, ensure that the input represents a valid Sercos IDN; otherwise, the outputs must be considered undefined.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_dwIDN | DWORD | IDN to be parsed |

| Output | Data type | Description |
| --- | --- | --- |
| q\_udiSI | UDINT | Structure instance of the IDN |
| q\_udiSE | UDINT | Structure element of the IDN |
| q\_xP | BOOL | S/P parameter of the IDN:  S are Sercos specific parameters.  P are manufacturer specific parameters.  0 = S  1 = P |
| q\_udiPS | UDINT | Parameter set of the IDN |
| q\_udiIDN | UDINT | Data block number (DBN) of the IDN |
