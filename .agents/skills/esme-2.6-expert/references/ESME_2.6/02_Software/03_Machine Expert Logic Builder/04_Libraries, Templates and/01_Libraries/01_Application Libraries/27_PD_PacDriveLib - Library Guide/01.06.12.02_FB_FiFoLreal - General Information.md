Task

FIFO buffer

Description

The function block is a FIFO buffer for LREAL values with a size of [Gc\_diMaxNumberOfElemen­tsInFiFo](../Global_Elements/Global_Elements-2.htm#XREF_D_SE_0087806_1) (16) elements.

Interface

NOTE: POU is not addressable!

The POU must not be called up.

The function block can be used by calling it up over methods and properties.

Methods

| Name | Description |
| --- | --- |
| [Get](Function_Blocks_A_to_F-20.htm#XREF_D_SE_0087269_1) | Reading a value from the FIFO |
| [Put](Function_Blocks_A_to_F-21.htm#XREF_D_SE_0087270_1) | Entering a value into the FIFO |
| [Reset](Function_Blocks_A_to_F-22.htm#XREF_D_SE_0087271_1) | Reset the values entered in the FIFO as well as the number of values entered to zero. |

Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| diCapacity | DINT | Read | Maximum filling level of the FIFO |
| diCount | DINT | Read | Filling level of the FIFO |
