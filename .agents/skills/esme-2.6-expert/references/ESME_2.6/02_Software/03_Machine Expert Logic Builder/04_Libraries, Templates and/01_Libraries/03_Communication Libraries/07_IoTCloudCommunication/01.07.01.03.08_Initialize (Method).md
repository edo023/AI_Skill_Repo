# Initialize (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method is used to initialize the function block. Independent from the state of the function block the internal process is aborted, the connection to the server is closed, send requests in queue are discarded, and counters are set to 0. After this method is executed the state is changed to Initialize. The initialization is processed by executing the method Cyclic. The state after the initialization is completed is Idle.

This method has no inputs or outputs.
