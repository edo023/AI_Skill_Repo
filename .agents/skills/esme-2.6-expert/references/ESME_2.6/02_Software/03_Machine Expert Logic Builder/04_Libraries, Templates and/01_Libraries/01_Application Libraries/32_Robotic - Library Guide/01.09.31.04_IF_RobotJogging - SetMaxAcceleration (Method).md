## Task

Setting the maximum jogging acceleration.

## Description

With the method SetMaxAcceleration(...), you can determine the maximum acceleration (change of the velocity) with which the jogging motion of the robot has to be executed.

NOTE: If MaxDeceleration has the value 0.0 or if the call-up of the method SetMaxDeceleration(...) sets it to the value 0.0, then MaxDeceleration is set to MaxAcceleration for jogging motion commands.
