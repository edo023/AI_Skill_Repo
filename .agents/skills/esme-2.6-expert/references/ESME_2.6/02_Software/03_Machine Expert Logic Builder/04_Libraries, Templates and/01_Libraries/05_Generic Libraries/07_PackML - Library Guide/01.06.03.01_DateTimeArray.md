## Description

## Structure Elements

| Variable | Data type | Data type |
| --- | --- | --- |
| DateTimeArray | ARRAY [0..6] OF DINT | The data elements associated with date and time shall be in the format per ISO 8601. [0] = year, [1] = month, [2] = day, [3] = hour, [4] = min, [5] = sec, [6] = µs (fraction of seconds) |
