# FC\_FirstLastStepEmptyTable - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.4.0 |

## Task

Function for initializing an empty command table

## Description

The function helps to initialize tables easily.

The function writes a 1 into iq\_udiStepIndex. The first field index of the table is assigned with this variable.

0 is entered into iq\_diStep. This is the identification of a step for the FB\_ModuleController. Signals that it has reached the end of a command table.
