# FC\_Distance3DTwoLineSegments

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the distance between a line segment i\_stLineSegment1 and a line segment i\_stLineSegment2 in the three-dimensional space.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stLineSegment1 | [ST\_LineSegment3D](ST_LineSegment3D-GeneralInformation-0C04BAB2.html#ST_LineSegment3D-GeneralInformation-0C04BAB2) | Line segment 1 |
| i\_stLineSegment2 | [ST\_LineSegment3D](ST_LineSegment3D-GeneralInformation-0C04BAB2.html#ST_LineSegment3D-GeneralInformation-0C04BAB2) | Line segment 2 |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
| q\_lrDistance | LREAL | Calculated distance between i\_stLineSegment1 and i\_stLineSegment2. |

## Diagnostic Messages
