# FB\_ApplicationLoggerHandler - GetLogEntry (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method searches in the ring buffer for logger messages inside the Application Logger for a message with the given ID. If this message is found, a copy of the logger entry including the message and the source information is returned.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiId | UDINT | The identification number of the logger message that has to be read. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_stLogEntry | [ST\_LogEntry](ST_LogEntry-37629944.html#ST_LogEntry-37629944) | The logger entry with message and source information. |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-375A58EE.html#ET_Result-375A58EE) | Provides diagnostic and status information as a numerical value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
