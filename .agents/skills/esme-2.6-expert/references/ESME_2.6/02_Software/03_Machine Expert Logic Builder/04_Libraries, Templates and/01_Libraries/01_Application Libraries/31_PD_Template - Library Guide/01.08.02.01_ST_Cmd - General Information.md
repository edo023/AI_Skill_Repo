# ST\_Cmd - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.4.0 |
| Inherits from: | - |

## Description

This data type reflects a command within a command table. Command sequences can be created with a table.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| sTitle | STRING[40] | Title of this command. |
| diStep | DINT | In which step in a table (command sequence) should this command be executed. |
| udiModuleName | UDINT | Which module the command shall be sent to. |
| diCmd | DINT | Command that shall be sent to the module. |
| timTimeOut | TIME := t#60s | If feedback is requested, 0 s is no time limit. |
| xEndCmd | BOOL | Shows the end of a table. |
