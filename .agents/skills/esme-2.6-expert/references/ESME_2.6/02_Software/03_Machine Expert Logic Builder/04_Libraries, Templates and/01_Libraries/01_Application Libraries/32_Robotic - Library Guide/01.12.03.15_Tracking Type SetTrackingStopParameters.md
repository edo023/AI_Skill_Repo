# Tracking Type SetTrackingStopParameters

## Overview

In case of AddLinearTrackingSystem, the method SetMaxAccResultant defines the StopParemters, and there are no separate parameters for tracking.

In case of rotative tracking or call of SetTrackingStopParameters for any TrackingSystem, the method SetTrackingStopParameters defines for the tracking the same parameters.

In case of TrackingSource, the method SetMotionParameters defines for every tracking its own parameter.
