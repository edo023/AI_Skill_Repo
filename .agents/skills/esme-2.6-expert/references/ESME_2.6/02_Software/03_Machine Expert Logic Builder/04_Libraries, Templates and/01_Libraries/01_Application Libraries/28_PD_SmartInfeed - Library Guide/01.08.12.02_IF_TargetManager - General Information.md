Task

Access to values and functions of the TargetManager

Description

The TargetManager manages the targets on the infeed distance of the [FB\_Infeed](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080279_1) POU.

Methods

| Name | Description |
| --- | --- |
| [AddTargetDistance](Interfaces-31.htm#XREF_D_SE_0080345_1) | Create a target at a distance from the last target. |
| [AddTargetPos](Interfaces-32.htm#XREF_D_SE_0080346_1) | Create a target at a position on the target belt. |
| [RemoveFirstTarget](Interfaces-33.htm#XREF_D_SE_0080347_1) | Delete the first target. |

Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| xEnableTargetGenerator | BOOL | Read/Write | TRUE: The TargetGenerator creates the targets. |

| raifTargets | REFERENCE TO ARRAY[1..[Gc\_udiMaxNumberOfTargets](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0080374_1)] OF [IF\_Target](Interfaces-27.htm#XREF_D_SE_0080340_1) | Read | Access to values and functions of the targets |
| lrOffset | LREAL | Read/Write | The offset of the targets from the zero position of the target belt |
| udiNextTargetId | UDINT | Read/Write | The identification number of the subsequent target |
| lrNextFeeTarget | LREAL | Read | The distance between beginning of the target belt and the next target that is pushed onto the target belt |
| udiNumberOfTargets | UDINT | Read | The number of all entered targets |
