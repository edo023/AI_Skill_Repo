# ET\_EmulationColors

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_EmulationColors contains the colors supported by EcoStruxure Machine Expert Twin. This representation is based on the X11 color names.
