## Task

Register the function block to the Application Logger.

## Description

With the method RegisterLoggerPoint, the function block FB\_CognexTelnet can be registered to the Application Logger.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifParent | APL.IF\_LoggerPoint | Parent logger point under which the FB\_TelnetCommunication is registered. |
| i\_sName | STRING[80] | The name of the FB\_TelnetCommunication logger point that is shown in the Application Logger has to be declared here. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | General library-independent statement on the diagnostic.  A value equal to TRUE corresponds to a diagnostic message. |
| q\_etDiagExt | ET\_RESULT | POU-specific output on the diagnostic:   * q\_xError = FALSE -> Status message * q\_xError = TRUE -> Diagnostic message |
