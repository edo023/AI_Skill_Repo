# FC_UpdateSubModuleInterfaceEight

![](images/logo.png)

FC\_UpdateSubModuleInterfaceEight

FC\_UpdateSubModuleInterfaceEight - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |
| Support for: | PacDrive3 pilot template architecture |
| Versions: | Current version |

Task

Function for updating the sub module add interface structure.

Description

Mainly, this function provides the same functionality as the [FC\_UpdateSingleSubModuleInterface](Functions-31.htm#XREF_D_SE_0077209_1) function. The only difference is that you can update 8 addresses of different AxisModules by this function.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xOnlineChange | BOOL | The sub module add interface is updated when true |
| i\_pdwSubModuleAddInterfaces | POINTER TO DWORD | Specifies a pointer to the sub module add interface |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stAxisModule1Itf to iq\_stAxisModule8Itf | ST\_ModuleInterface | Specifies an axis’s interface structure. Up to 8 axes can be added/updated to the sub module add interface |
