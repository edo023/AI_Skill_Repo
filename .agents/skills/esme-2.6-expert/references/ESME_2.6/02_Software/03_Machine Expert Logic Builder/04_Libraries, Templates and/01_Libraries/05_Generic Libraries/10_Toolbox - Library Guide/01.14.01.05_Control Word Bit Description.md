# Control Word Bit Description

## Functionality

This table describes the control word bits:

| Bit Position | Description |
| --- | --- |
| 0...7 | Not used |
| 8 | Rising edge from 0 to 1 manually opens the valve (output `q_xOpen` set to TRUE) in manual mode. |
| 9 | Rising edge from 0 to 1 manually closes the valve (output `q_xCls` set to TRUE) in manual mode. |
| 10...15 | Not used |
| 16 | Acknowledges internal and external detected errors indicated at the output `q_xErr`. This is rising edge triggered. |
| 17...31 | Not used |
