## Functional Description

By calling the method, the configuration data for the OPC/UA data structure of the Lexium P Robot can be read.

## Interface

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | General, library-independent statement on the diagnostic.  A value not equal to ET\_Diag.Ok corresponds to a diagnostic message. |

| q\_stData | DTC.ST\_ConfigDataRobotPSeries | Read out data that have been configured to emulate this robot in EcoStruxure Machine Expert Twin. |
