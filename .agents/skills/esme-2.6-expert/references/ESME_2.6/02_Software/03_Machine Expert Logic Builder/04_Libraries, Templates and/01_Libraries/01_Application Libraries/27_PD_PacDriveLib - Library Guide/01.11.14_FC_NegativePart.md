Task

Determine the negative part of a value.

Description

The negative part is defined as follows:

oi\_lrValue >= 0.0 -> 0.0

oi\_lrValue < 0.0 -> ABS(i\_lrValue)

Together with the function [FC\_PositivePart](Functions_K_to_Z-36.htm#XREF_D_SE_0087607_1) the following identity applies to all real lrValues:

lrValue = FC\_PositivePart(lrValue) - FC\_NegativePart(lrValue)
