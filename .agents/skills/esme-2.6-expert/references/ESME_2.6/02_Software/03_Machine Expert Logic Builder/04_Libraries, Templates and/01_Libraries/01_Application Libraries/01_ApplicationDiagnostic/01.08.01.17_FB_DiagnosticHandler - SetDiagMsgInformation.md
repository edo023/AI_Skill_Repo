## Task

Sets a diagnostic message of category [ET\_Category.Information](ET_Category-GeneralInformation-36C5F24D.html).

## Description

The method SetDiagMsgInformation is used to set a diagnostic message of type category [ET\_Category.Information](ET_Category-GeneralInformation-36C5F24D.html) at a diagnostic handler. The diagnostic message can be quit with the method DiagQuitInformation without the need to confirm process completion.

NOTE: Messages set after the diagnostic quit command was triggered are handled as a new diagnostic message. If resetting the function block reporting the diagnostic message takes more than one cycle and its error output remains TRUE, the diagnostic message should be set on a rising edge of the error output to avoid a new reaction while the diagnostic quit is still in progress.

NOTE: To help avoid unnecessary processor load, call this method only once when the cause of the diagnostic message occurs.
