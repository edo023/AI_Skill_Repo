# ET_SimGenerateMode

![](images/logo.png)

ET\_SimGenerateMode

ET\_SimGenerateMode - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Description

The enumeration describes the methods for generating signals upon which the [FB\_InfeedSimu­lation](../Function_Blocks/Function_Blocks-7.htm#XREF_D_SE_0080283_1) POU generates products or targets.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No signals are generated. |
| Equidistant | 1 | Signals with the same distance of [lrDistance](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) are generated. |
| EquidistantGroups | 2 | Groups of signals are generated with [udiSignalsInGroup](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) signals per group at a distance of [lrDistance](../Structures/Structures-4.htm#XREF_D_SE_0080360_1). A distance of [lrDistanceGroup](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) is generated between the groups. |
| Random | 3 | Signals are generated with a random distance between [lrMaxDistance](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) and [lrMinDistance](../Structures/Structures-4.htm#XREF_D_SE_0080360_1). |
| RandomGroups | 4 | Groups of signals are generated with a random number of signals between [udiMinSignalsInGroup](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) and [udiMaxSignalsInGroup](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) per group with a random distance of between [IrMinDistance](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) and [lrMaxDistance](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) within a group. Between the groups, a random distance between [lrMinDistanceGroup](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) and [lrMaxDistanceGroup](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) is created. |
| Pattern | 5 | Signals are generated following a pattern specified in [alrPattern](../Structures/Structures-4.htm#XREF_D_SE_0080360_1). In [alrPattern](../Structures/Structures-4.htm#XREF_D_SE_0080360_1) the distance to each subsequent signal is specified. If the distance to the next signal is negative or all signals of the ARRAY have been sent, the pattern starts from the beginning. |
