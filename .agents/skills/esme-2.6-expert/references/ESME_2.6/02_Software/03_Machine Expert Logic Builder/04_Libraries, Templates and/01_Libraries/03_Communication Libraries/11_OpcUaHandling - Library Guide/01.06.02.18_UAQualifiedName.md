## Description

The structure UAQualifiedName is used to provide the information about the name and the corresponding namespace index.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| NamespaceIndex | UINT | The namespace index where Name is provided. |
| Name | STRING [255] | Name of qualified name. |
