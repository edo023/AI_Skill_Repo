# ST\_RobotEntityData

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.4.1.0 |
| Inherits from: | - |

## Description

Structure to store information on a robot entity. This information is used to identify a specific robot entity to describe the position and orientation of such entity into the Cartesian space and to define its kinematic structure.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sName | STRING[80] | A string used to identify the robot. |
| etSystemId | ET\_SystemEntity | An enumeration used to unequivocally identify the robot in the system. |
| etType | ET\_RobotType | An enumeration used to identify a robot type. |
| stBasePose | ST\_CartesianPose | A Cartesian pose that describes how the robot is displaced in the Cartesian space with reference to a global coordinate system. |
| stKinematicParameters | ST\_RobotKinematicParameters | A structure containing information on the robot kinematics. |
| stModifiedCoordinateSystem | ST\_RobotModifiedCoordinateSystem | A structure that describes if a transformation is applied or not to the original coordinate system of the robot. |
