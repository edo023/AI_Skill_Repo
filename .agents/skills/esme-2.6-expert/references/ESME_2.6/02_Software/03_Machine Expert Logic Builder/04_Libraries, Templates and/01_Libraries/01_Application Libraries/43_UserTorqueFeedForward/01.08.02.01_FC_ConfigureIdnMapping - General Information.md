# FC\_ConfigureIdnMapping - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.5.1.0 |
| Versions: | Installed version |

## Task

Configures the required IDN in the IDN mapping on an M660 Motion Controller.

## Description

This function is used to add the required IDN P-0-1069.0.0 to the IDN mapping of the drive object provided at the input i\_ifDrive.

The function is supported only on the M660 Motion Controller.

The following precondition must be fulfilled:

* The Sercos ActualState must be in phase 2.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifDrive | CMI.If\_AxisIdentification | Drive for which additional IDN is to be configured. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error is detected.  For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-45A2DE35.html) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |

## Troubleshooting

Possible diagnostic information (q\_xError = TRUE)

| Outputs of the method indicate the values | Cause | Solution |
| --- | --- | --- |
| q\_etResult = InvalidInterface | The given interface is invalid. | Assign a valid interface to the input i\_ifDrive. |
| q\_etResult = InvalidSercosState | The Sercos ActualState is not as required. | Set the Sercos ActualState to phase 2. |
| q\_etResult = InvalidWorkingMode | The working mode of the assigned drive is invalid. | Set the state of the desired working mode of the drive to Physical. |
| q\_etResult = DriveDoesNotSupportThisFunction | The selected axis does not support this function. | Select an axis that supports this function (for example, Lexium LXM62 Drive, Lexium LXM52 Drive or Lexium ILM62 Drive module). |
| q\_etResult = RequiredIdnMappingNotDone | The required parameter IDN P-0-1069.0.0 is not configured as additional IDN for cyclic data exchange. | Execute the method again. If the issue persists, contact your Schneider Electric support representative. |
| q\_etResult = FeatureNotSupported | The function is executed on a not supported controller. | Execute this function on an M660 Motion Controller. |
