# ET_DiagSource

![](images/logo.png)

ET\_DiagSource

Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

Description

The enumeration ET\_DiagSource contains the possible values that indicate the source of a diagnostic message.

Enumeration Elements

| Name | Value  (UDINT) | Description |
| --- | --- | --- |
| NotSet | 0 | No source for diagnostic message is set. |
| FbGetDiag | 1 | The diagnostic message is created executing the function block FB\_GetDiagnosticData with command GetPlcDiag. |
| MotionKernel | 2 | The diagnostic message was received from the controller [MotionKernel](../glossary/glossary.htm#XREF_D_SE_0097798_4) component. |
