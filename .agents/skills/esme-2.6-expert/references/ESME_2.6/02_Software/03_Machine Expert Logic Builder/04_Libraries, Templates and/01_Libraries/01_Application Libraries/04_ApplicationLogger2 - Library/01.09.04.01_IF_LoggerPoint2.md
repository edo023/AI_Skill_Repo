|  |  |
| --- | --- |

## Description

This interface extends the interface APL\_Base.IF\_LoggerPoint and provides the methods of the function block [FB\_LoggerPoint](FB_LoggerPoint-37E4C005.html#FB_LoggerPoint-37E4C005). Via this interface it is possible to register logger points and send messages to the Application Logger. In addition, the data of a logger point can be read out.

NOTE: This data type is an alias of the data type APL\_Base.IF\_LoggerPoint2 defined in the **ApplicationLoggerBase** library. The **ApplicationLoggerBase** library contains the basic implementations of the logger functions.
