# ET_StartMovementMode

![](images/logo.png)

ET\_StartMovementMode

ET\_StartMovementMode - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

Description

This enumeration defines the movement behavior of the FB\_Unwinder during the start phase.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No start movement. The unwinder immediately starts with a precontrol- and control movement. |
| MoveToRefPosition | 1 | At first the dancer is moved to its reference position. Afterwards the unwinder moves according to the precontrol- and control movement. |
| SpliceFromStandstill | 2 | The unwinder waits until the machine feeds the foil and then it accelerates the bobbin so that it moves the dancer to its reference position. Afterwards the unwinder moves according to the precontrol- and control movement. |
| Torque | 3 | The bobbin is moved with a fixed torque until i\_lrStartThreshold is reached. Afterwards the unwinder moves according to precontrol and control movement. |
