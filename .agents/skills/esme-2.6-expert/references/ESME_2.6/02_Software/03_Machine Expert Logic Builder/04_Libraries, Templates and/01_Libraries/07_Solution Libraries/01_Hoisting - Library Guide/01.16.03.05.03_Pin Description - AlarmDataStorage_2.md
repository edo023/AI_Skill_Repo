# Pin Description - AlarmDataStorage_2

![](images/logo.png)

Pin Description - AlarmDataStorage\_2

Input Pin Description

| Input | Data Type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block.  Refer to [detailed description](#XREF_D_SE_0021124_4) below this table. |
| i\_xDevAlrm | BOOL | Alarm event input monitored by the FB. On a rising edge the monitoring is started until a falling edge is detected.  TRUE: Recording active  FALSE: Recording inactive |
| i\_wAlrmNb | WORD | Number of incident to be displayed at the output.  Range: 0...20  Refer to [detailed description](#XREF_D_SE_0021124_5) below this table. |
| i\_xRst | BOOL | Resets all retained values.  A rising edge on this input resets the following retained values:  oq\_wAlrmNb is set to zero  oq\_dtAlrm [1...20] will be set to 1977-07-07-07:07:07  oq\_wAlrmDur [1...20] will be set to zero  TRUE: Resets function block  FALSE: No action |

i\_xEn

By setting the i\_xEn input to FALSE the output states will be overwritten by a defined fallback state.

As soon as the function block gets enabled again the output states will be updated corresponding to the function block behavior as described above. There is no additional action required from you to re-establish the enabled state.

The fallback states of AlarmDataStorage\_2 are given in the table below.

| Output | Data Type | Fallback State |
| --- | --- | --- |
| q\_xEn | BOOL | FALSE |
| q\_wAlrmFreeHrs | WORD | 0 |
| q\_dtAlrm | DT | 1977-07-07-07:07:07 |
| q\_wAlrmDur | WORD | 0 |
| q\_xAlrm | BOOL | FALSE |
| q\_wAlrmNb | WORD | not altered - retained value |

i\_wAlrmNb

The input is the selector for the alarm event to be displayed at the corresponding outputs q\_dtAlrm and q\_wAlrmDur. The FB is able to store 20 alarm incidents with time stamp and duration. The most recent alarm is always stored in position [1] and therefore the oldest recorded event at position [20].

Output Description

| Output | Data Type | Description |
| --- | --- | --- |
| q\_xEn | BOOL | TRUE: Function block enabled.  FALSE: Function block disabled. |
| q\_wAlrmFreeHrs | WORD | Total hours since the last detected alarm event.  Range: 0...65535  Scaling/Unit: 1 hour |
| q\_dtAlrm | DT | Displays the date and start time of the alarm event, which was selected by the number entered at i\_wAlrmNb out of register of the last 20 alarms.  Range: 1970...21xx  Scaling/Unit: YYYY MMDD HHMMSS |
| q\_wAlrmDur | WORD | Displays the duration of the detected alarm event, which was selected by the number entered at i\_wAlrmNb out of the register of the last 20 alarms.  Range: 0...65535  Scaling/Unit: 1 second |
| q\_xAlrm | BOOL | Displays the status of the alarm input i\_xDevAlrm. If i\_xDevAlrm is TRUE, q\_xAlrm gets TRUE and vice versa.  TRUE: Alarm detected  FALSE: No alarm detected |
| q\_wAlrmNb | WORD | Displays the total number of alarm events detected since the last reset or initial start.  Range: 0...65535  Scaling/Unit: 1 alarm |
