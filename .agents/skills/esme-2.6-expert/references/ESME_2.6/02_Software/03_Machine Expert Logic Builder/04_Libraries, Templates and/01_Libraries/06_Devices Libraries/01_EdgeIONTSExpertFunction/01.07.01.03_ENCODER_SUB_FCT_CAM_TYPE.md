# ENCODER\_SUB\_FCT\_CAM\_TYPE

## Description

The ENCODER\_SUB\_FCT\_CAM\_TYPE enumeration defines the three types of CAMs that are available. For further information, refer to [CAM Mode Principle Description](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Exp_UG&topicID=CAMModePrincipleDescription_5EE78856).

## Enumeration Elements

The enumeration data type ENUM contains the following values:

| Name | Value | Description |
| --- | --- | --- |
| POSITION | 0 | A Position type CAM is available. |
| MONOSTABLE | 1 | A Monostable type CAM is available. |
| BRAKING | 2 | A Braking type CAM is available. |
