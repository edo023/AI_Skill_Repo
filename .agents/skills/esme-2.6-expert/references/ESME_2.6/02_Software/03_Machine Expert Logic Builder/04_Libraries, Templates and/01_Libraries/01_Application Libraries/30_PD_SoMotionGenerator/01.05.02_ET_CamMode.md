# ET_CamMode

![](images/logo.png)

ET\_CamMode

ET\_CamMode - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.0.0 |

Description

Behavior of a cam when reaching the minimum or maximum value of the logical master position

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Endless | 0 | The logical master position and the position of the slave axis are automatically reset by one period upon reaching the end position [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrXEnd. The cam is not canceled.  When reaching the start position [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrXStart, the logical master position and the slave axis position are set forward by one period. The cam is not canceled. |
| NoLimit | 1 | This has no influence on the logical master position and the slave position. |
| HighLimit | 2 | The cam is canceled upon reaching [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrXEnd. |
| LowLimit | 3 | The cam is canceled upon reaching [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrXStart. |
| BothLimits | 4 | The cam is canceled upon reaching [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrXEnd or [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrXStart. |
