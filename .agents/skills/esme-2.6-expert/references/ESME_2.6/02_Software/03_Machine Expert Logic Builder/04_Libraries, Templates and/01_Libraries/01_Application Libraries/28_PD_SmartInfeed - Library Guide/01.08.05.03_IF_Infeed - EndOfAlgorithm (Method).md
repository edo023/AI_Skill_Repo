Task

Resetting all signals for the algorithm

Functional Description

The [FB\_Infeed](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080279_1) automatically sends signals, which can be evaluated in the algorithm. These signals must be reset by the algorithm. Using this method, all signals can be reset at once.

The following signals are reset for all belts:

o[xProductIntake](Interfaces-2.htm#XREF_D_SE_0080311_1)

o[xProductLeftBelt](Interfaces-2.htm#XREF_D_SE_0080311_1)
