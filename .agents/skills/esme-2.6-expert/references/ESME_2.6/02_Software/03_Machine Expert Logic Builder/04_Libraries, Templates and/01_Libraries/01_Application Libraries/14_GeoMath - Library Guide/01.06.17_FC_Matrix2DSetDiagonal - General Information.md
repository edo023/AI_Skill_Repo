## Description

Given two scalar values, the function returns a new 2D diagonal matrix that has such elements on its main diagonal.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrE11 | LREAL | First value to insert in the main diagonal. |
| i\_lrE22 | LREAL | Second value to insert in the main diagonal. |

| Output | Data type | Description |
