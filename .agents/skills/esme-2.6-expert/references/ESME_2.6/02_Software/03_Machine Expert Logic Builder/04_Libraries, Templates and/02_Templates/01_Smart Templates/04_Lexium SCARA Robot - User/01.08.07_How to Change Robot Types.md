# How to Change Robot Types

## General

This chapter provides an example of changing the robot type.

## Restrictions

Within the Robot Type Change dialog box, it is not possible to change the following configured parameters / data:

* Robot name
* Drive type
* Node type
* Generate POU instance (option available if the node type is Non Template

## Starting Robot Type Change

| Step | Action |
| --- | --- |
| 1 | Open MyRobot.ConfigurationData >  Basic configurations and click Robot Type Change. |
| 2 | Select the desired robot by entering the reference number of the robot or by using the filters.  **Result:** The robot is displayed in the Change Robot Type dialog box. |
| 3 | Click Start Robot Type Change and then click Yes. |
| 4 | Click OK in the Robot Type Change dialog box.  **Result:** The Basic configuration window is updated with the new robot type. |
