## Task

Interface with return values for the velocity limitation of individual robot axes.

## Description

This interface is part of IF\_RobotFeedbackVelocityLimitation and contains data on velocity limitation of individual robot axes.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| etMode | ET\_VelocityLimitationMode | Read | Current mode of the velocity limitation. |
| lrMaxRefVelocity | LREAL | Read | Current limit for the axis. |
| lrRefPositionDeviation | LREAL | Read | Deviation of RefPosition of the axis caused by the limitation |
| lrRefPosition | LREAL | Read | Reference position of the axis without limitation. |
| lrRefVelocity | LREAL | Read | Reference velocity of the axis without limitation. |
| lrRefAcceleration | LREAL | Read | Reference acceleration of the axis without limitation. |
