## Functional Description

The method GetTargets can be called when the corresponding property xNewTargetsAvailable is set to TRUE.

The sensor target generation uses a method GetTargets and the property xNewTargetAvailable.

If new targets are available, the property xNewTargetAvailable is set.

The method and the property are:

* MethodSR\_<MySensor>.GetTargets
* PropertySR\_<MySensor>.xNewTargetAvailable

The property xNewTargetsAvailable is set when new targets are available.

The method GetTargets provides the corresponding targets of the type SERT.ST\_GeneratedTargetsData.

## Interface

The GetTargets method has only output parameters.

| Output | Data type | Description |
| --- | --- | --- |
| GetTargets | SERT.ST\_GeneratedTargetsData | * The targets are displayed on the sensor in the 3D-view. * The targets must correspond to the current plane and offset of the sensor. |
