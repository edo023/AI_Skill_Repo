# ResetError (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method is used to acknowledge the error state of the function block. Once this method is executed, the error information is reset. Depending on the error, the subsequent state is Idle or Connected.

This method has no inputs and outputs.
