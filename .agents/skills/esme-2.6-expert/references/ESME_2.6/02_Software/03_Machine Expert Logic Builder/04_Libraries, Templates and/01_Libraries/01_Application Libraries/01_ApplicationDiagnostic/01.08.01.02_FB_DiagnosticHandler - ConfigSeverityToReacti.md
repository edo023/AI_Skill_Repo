## Task

Configures the mapping of the severity of a diagnostic message to a reaction level.

## Description

The method ConfigSeverityToReactionLevel is used to configure the mapping of the severity of a diagnostic message of type equal to [ET\_Category.Error](ET_Category-GeneralInformation-36C5F24D.html) to a reaction level. The mapping can be configured individually for each source of a diagnostic message.

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Indicates TRUE if the configuration is valid and applied. |

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xSetConfigForAllSources | BOOL | If set to TRUE, the mapping table is applied for all sources.  If set to FALSE, the input i\_ifSourceChild specifies the source for which the mapping table is applied. |
| i\_ifSourceChild | IF\_DiagHandlerAcces | This input specifies the source diagnostic handler for which the mapping table is applied.  If this input is set to 0, the mapping table is applied for diagnostic messages set through the method SetDiagMsgError.  This input is relevant only if i\_xSetConfigForAllSources is FALSE. |
| i\_auiReactionLevel | ArraySeverityToReactionLevelForwarding | Table specifying the mapping of the severity, represented by the index of the array of a diagnostic message to a reaction level represented by the value of the corresponding array element. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | Value is TRUE if an error was detected. For details, refer to q\_etResult and q\_sResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-36C542C6.html) | Provides diagnostic and status information as a numeric value.  If q\_xError  = FALSE, q\_etResult provides status information. If q\_xError  = TRUE, q\_etResult provides diagnostic/error information. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text value. |
