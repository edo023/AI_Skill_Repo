# ET\_ConveyorControlMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.1.0.0 |

## Description

The enumeration ET\_ConveyorControlMode defines the control mode of a conveyor.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No valid control mode is selected. |
| Velocity | 1 | The conveyor is driven in velocity mode. |
| PositioningAbsolute | 2 | The conveyor is driven in absolute positioning mode.  The position must be consistent. Jumps in position value are not allowed. |
| PositioningModuloRollover | 3 | The conveyor is driven in positioning mode with a modulo or rollover value.  The position is kept in the range from 0 to stConfigDataLinearConveyor.lrModuloRolloverValue. For more information, refer to [ST\_ConfigDataLinearConveyor](ST_ConfigDataLinearConv-22380707.html#ST_ConfigDataLinearConv-22380707). |
