ET\_ParId

ET\_ParId - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.2.0 |

Description

The enumeration describes the default indexes of the cam switchover of the [FB\_CrankModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0088740_1).

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Idle | 0 | Index for wait cam. |
| Start | 1 | Index for start cam. |
| Cont | 2 | Index for work cam. |
| Stop | 3 | Index for stop cam. |
| User | 4 | Index for user cam. |
