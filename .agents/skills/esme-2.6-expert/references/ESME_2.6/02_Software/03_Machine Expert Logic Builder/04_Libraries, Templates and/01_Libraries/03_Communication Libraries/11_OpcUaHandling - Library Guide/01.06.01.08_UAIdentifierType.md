# UAIdentifierType

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration UAIdentifierType specifies the types of the OPC UA identifier.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| UAIT\_Numeric | 0 | Identifier specified with a numeric value. |
| UAIT\_String | 1 | Identifier specified with a string value. |
| UAIT\_GUID | 2 | Identifier specified with a GUID (Globally Unique Identifier) value. |
| UAIT\_Opaque | 3 | Not supported. |
