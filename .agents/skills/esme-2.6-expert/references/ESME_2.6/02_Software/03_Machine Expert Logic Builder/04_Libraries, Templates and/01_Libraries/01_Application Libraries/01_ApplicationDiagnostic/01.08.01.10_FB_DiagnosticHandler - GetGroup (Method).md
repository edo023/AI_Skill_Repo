## Task

Returns an array of interfaces of a group of diagnostic handlers.

## Description

The method GetGroupStatus is used to obtain an array of interfaces of a group of diagnostic handlers. The first entry in the array contains the interface to the diagnostic handler on which the method is executed. The other entries contain the interfaces to the recursively subordinate diagnostic handlers.

## Return Value

| Data type | Description |
| --- | --- |
| UINT | Indicates the number of diagnostic handlers in the group. |
