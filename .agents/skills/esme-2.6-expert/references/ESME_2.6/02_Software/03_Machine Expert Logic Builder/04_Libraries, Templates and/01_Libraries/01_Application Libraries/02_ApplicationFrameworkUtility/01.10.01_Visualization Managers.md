# Visualization Managers

## General Information

For displaying data in the visualization, you must create frames for the different types of data such as diagnostics data, axis data, or the status data of a unit. You can display the different data in one frame, in separate frames or in a combined form, depending on your requirements. For each frame, you must create a corresponding function block that provides the data for the frame. Each unit instantiates these function blocks and provides the data for the visualization.

For more information on the use of frames, refer to the [Framework example guide](../../../../../api/crossBook?lang=en-US&virtualBookName=exFramwo&topicID=).

For selecting a unit in a visualization and displaying the corresponding data in the frames, the library provides visualization managers: [FB\_VisApplicationManager](FB_VisAppManager-Gen-DA85FB15.html) and [FB\_VisUnitManager](FB_VisUnitManager-Gen-E0421E74.html).
