# ET\_TrackingStopParameter - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.16.0.0 |

## Description

Enumeration type to specify which stop parameter is addressed.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| StopDeceleration | 1 | Deceleration used for a stop. |
| StopRamp | 2 | Ramp used for a stop. |
| EmergencyDeceleration | 3 | Emergency deceleration. |
| EmergencyRamp | 4 | Emergency ramp. |
