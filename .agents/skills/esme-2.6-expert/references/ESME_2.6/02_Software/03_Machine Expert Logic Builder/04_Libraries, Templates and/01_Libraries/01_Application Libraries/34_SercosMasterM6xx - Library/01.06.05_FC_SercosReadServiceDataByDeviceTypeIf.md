# FC\_SercosReadServiceDataByDeviceTypeIf

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

The FC\_SercosReadServiceDataByDeviceTypeIf function of the library SercosMasterM6xx reads data through the Sercos service channel from the Sercos device addressed with the specified device type interface.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifDeviceType | [DeviceTypeInterfaces.IF\_Device\_Type](../../../../../api/crossBook?lang=en-US&virtualBookName=DeviceTypeInterfaces&topicID=IF_Device_Type_56CAF679) | Device type interface of the Sercos device |
| i\_dwIDN | DWORD | IDN to be read |
| i\_wType | WORD | Type of action through service channel:   * 0: Read name * 1: Read attribute * 2: Read unit * 3: Read minimum value * 4: Read maximum value * 5: Read user data * 6: Read user data with variable length |
| i\_lwPointerToData | LWORD | Pointer to memory buffer. |
| i\_wMaxDataLen | WORD | Length of bytes to read. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_uiReadDataLen | UINT | Specification of the data length of the Sercos IDN if i\_wType is 6. |
| iq\_uiMaxReadDataLen | UINT | Specification of the maximum data length of the Sercos IDN if i\_wType is 6. |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: Error detected: The topology address is invalid.  -431: Error detected: Error detected during service request (for example, timeout).  -445: Error detected: Service timeout  -461: Error detected: In the active Sercos communication phase, reading parameters through the service channel is not supported.  -464: Error detected: Invalid service transfer |
