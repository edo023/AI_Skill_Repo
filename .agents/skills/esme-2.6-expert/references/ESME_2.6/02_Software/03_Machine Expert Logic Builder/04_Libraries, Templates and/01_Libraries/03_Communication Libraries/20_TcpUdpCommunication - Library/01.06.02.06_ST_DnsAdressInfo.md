|  |  |
| --- | --- |

| Available as of: | V1.0.4.0 |

## Description

The structure ST\_DnsAddressInfo contains information about the resolved domain name received from the DNS server.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sIpAddress | STRING(15) | The IP address of the resolved domain name. |
| dwTTL | DWORD | The validity time (TTL) of the IP address. (Time in seconds indicating how long the IP address can be cached.) |
