Description

This structure contains the parameters and feedback data if the FB\_Unwinder is not connected to the master.

The position of the feeder is assumed to be moving with the specified velocity. Changes to the feeder velocity give the assumption as if the feeder is performing a new positioning command.

The bobbin performs the positioning movement parallel to the assumed positioning movements of the feeder.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_lrVelocity | LREAL | The (middle) velocity of the feeder. |
| i\_lrAcceleration | LREAL | The value to accelerate to the master velocity. |
| i\_lrDeceleration | LREAL | The value to decelerate to the master velocity. |
| i\_lrJerk | LREAL | The jerk to accelerate and decelerate the master velocity. |
