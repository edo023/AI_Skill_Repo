# FC\_Distance3DCylinderSphere

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the distance between a cylinder i\_stCylinder and a sphere i\_stSphere.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stCylinder | [ST\_Cylinder](ST_Cylinder-GeneralInformation-0C03A0F8.html#ST_Cylinder-GeneralInformation-0C03A0F8) | Cylinder |
| i\_stSphere | [ST\_Sphere](ST_Sphere-GeneralInformation-0C085EA6.html#ST_Sphere-GeneralInformation-0C085EA6) | Sphere |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
| q\_lrDistance | LREAL | This calculates the distance between i\_stCylinder and i\_stSphere. |

## Diagnostic Messages

| q\_xError | q\_etResult | Enumeration value | Description |
| --- | --- | --- | --- |
| FALSE | Ok | 0 | Success |
| TRUE | DefinitionCylinder | 42 | The definition of Cylinder is not correct. |
| TRUE | RadiusRangeSphere | 22 | The radius of the sphere is outside the valid range. |

## DefinitionCylinder

|  |  |
| --- | --- |
| Enumeration name: | DefinitionCylinder |
| Enumeration value: | 42 |
| Description: | The definition of Cylinder is not correct. |

| Cause | Solution |
| --- | --- |
| The value a the input i\_stCylinder.lrRadius is less than or equal to 0. | The radius of the cylinder must be greater than zero. |
| The inputs i\_stCylinder.stAxisStart and i\_stCylinder.stAxisEnd have been set to the same point. | The distance between i\_stCylinder.stAxisStart and i\_stCylinder.stAxisEnd must be at least Gc\_lrZeroTolerance. |
