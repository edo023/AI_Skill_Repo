# CANopen Function Blocks

![](images/logo.png)

CANopen Function Blocks

Function Block Offer Description

This table describes the [function](../glossary/glossary.htm#XREF_D_SE_0024697_714) block offer when TeSys are connected through a [CANopen](../glossary/glossary.htm#XREF_D_SE_0024697_650) [network](../glossary/glossary.htm#XREF_D_SE_0024697_152):

| Function Block | Description | For more information |
| --- | --- | --- |
| TeSysU\_CtrlCmdCyc\_CANopen | This function block is dedicated to the control and command of a single TeSysU starter [controller](../glossary/glossary.htm#XREF_D_SE_0024697_661) (up to 32 A/15 kW or 20 hp) with any control unit and a LULC08 CANopen communication module. | Refer TeSysU\_CtrlCmdCyc\_CANopen chapter |
| TeSysT\_CtrlCmdCyc\_CANopen | This function block is dedicated to the control and command of a single TeSys T LTMR••C•• CANopen controller with or without the LTM E expansion module. | Refer TeSysT\_CtrlCmdCyc\_CANopen chapter |
