## Functional Description

This function block implements the interface IF\_CallbackStatusSend and the implementation to receive the status information about the requested data transmission from the instance of the function block FB\_ESMAClientMonitoringApi.

An instance of this function block can be assigned to the method SendData() of the function block FB\_ESMAClientMonitoringApi.
