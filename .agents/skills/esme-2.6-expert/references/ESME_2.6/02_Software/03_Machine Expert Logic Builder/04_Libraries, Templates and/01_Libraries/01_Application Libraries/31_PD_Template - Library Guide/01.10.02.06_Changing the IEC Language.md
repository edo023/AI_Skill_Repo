# Changing the IEC Language

## General

Normally when you start the QuickMotionProgramming project, the function block view (FUP) is used for displaying your code. If you want to switch to the ladder logic (LD) or instruction list view (IL), proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Open the action whose IEC language should be changed, e.g. from FUP to LD. |
| 2 | In the FBD/LD/IL menu select View > View as ladder logic or View > View as instruction list. |
| 3 | The IEC language of the action has changed, e.g. to the IEC language ladder logic: |
