# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result specifies the possible values that indicate the result of operations executed by the POUs of this library.

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| Disabled | 0 | The function block is disabled. |
| Ok | 1 | The function block is being executed successfully. |
| SercosReadError | 2 | Unable to read the parameter of the drive. |
| SercosWriteError | 3 | Unable to write the parameter to the drive. |
| TimeOut | 4 | A timeout period has elapsed during execution. |
| InputParameterInvalid | 5 | A setting of an input variable is not valid. For further information, refer to the q\_sResultMsg output. |
| AsyncManagerError | 6 | An exception has been detected by the internal asynchronous manager. |
| NotReadyForSampling | 7 | The drive configured for sampling data cannot execute the sampling process. |
| BuildIDNError | 8 | The Sercos parameter is not valid. |
| DriveConditionInvalid | 9 | The drive is not ready for sampling data. |
| SercosNotInPhaseFour | 10 | Sercos is not in phase 4. Thus, sampling data is not possible. |
| SamplingIsBlocked | 11 | Sampling cannot start because the drive is still busy with another sampling process. |
| SamplingInDriveInError | 12 | An error has been detected in the drive that is in sampling mode. |
| RtpError | 13 | An internal error has been detected. |
| ControllerConditionInvalid | 14 | The controller settings for fast sampling are not valid. |
| SampledDataAvailable | 15 | The data sampling process has been completed successfully. Collected data are available. |
| SamplingData | 16 | The data sampling process is being executed. |
| IecTraceError | 17 | An internal error has been detected. |
