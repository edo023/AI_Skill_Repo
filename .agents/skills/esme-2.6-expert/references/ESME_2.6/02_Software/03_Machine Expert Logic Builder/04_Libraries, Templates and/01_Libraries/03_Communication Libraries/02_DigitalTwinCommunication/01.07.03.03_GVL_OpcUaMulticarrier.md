# GVL\_OpcUaMulticarrier

## Overview

|  |  |
| --- | --- |
| Type: | Global variables |
| Available as of: | V1.0.0.0 |

## Description

The global variable list GVL\_OpcUaMulticarrier contains variable declarations for the build and emulation of the Lexium™ MC multi carrier via OPC UA communication by using the Symbol Configuration.

Verify that in the [Symbol Configuration](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083586) the emulation data are selected and the access rights are in line with the application needs.

NOTE: Use this DTC.GVL\_OpcUaMulticarrier only if your installation of EcoStruxure Machine Expert includes the Multicarrier library. If the DTC.GVL\_OpcUaMulticarrier is called without the Multicarrier library being installed, compile errors will be detected.
