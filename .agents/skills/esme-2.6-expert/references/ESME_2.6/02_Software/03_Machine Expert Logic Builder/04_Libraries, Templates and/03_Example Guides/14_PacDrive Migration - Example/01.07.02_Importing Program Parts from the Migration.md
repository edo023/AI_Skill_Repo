# Importing Program Parts from the Migration Example

To migrate your PacDrive controller to a Modicon controller M660 proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Open your Example Project as described in [Opening the Example Project](OpenExample-7AA4774D.html). |
| 2 | Convert your controller as described in [Converting Devices](ConvertingDevices-FD794B09.html). |
| 3 | Open another Logic Builder window and open the PacDriveMigration example project as template. |
| 4 | Arrange the windows so that they are adjacent to each other. This allows you to drag and drop modules from the example project into your own project. |
| 5 | Import the folders AxisModuleLib Migration, Migration Helper and PacDriveLib Migration from the migration example project. |
| 6 | Import the folder Sercos\_Template.  NOTE: The parameter PhaseCheckMode is not implemented in the parameter set of the LXM62P power supply. To change to the 1-phase mode you must import the folder Sercos\_Template and run the program SR\_Sercos. |
| 7 | Modify the Start\_Action program in SR\_MainMachine by copying the Start\_Action program from the example project and paste it into your project.   ``` SR_Sercos () ;  IF SR_Sercos.q_xSercosRun THEN     APL.G_ifApplicationLogger. RegisterCommunicationService (q_etDiag =>, q_etDiagExt => ) ;     stMachineInterface.i_xEnable := TRUE; 	     pTaskInfo	:=__CURRENTTASK. pTaskInfo;     G_diTaskIntervalInMs := TO_DINT( pTaskInfo^.dwInterval / 1_000);     ifController := DED.GetRoot ();     xSuccess  :=__QUERYINTERFACE (ifController, ifControllerDiagnosticAccess); 	IF xSuccess THEN         ifControllerDiagnostic := ifControllerDiagnosticAccess.Diagnostic; END IF ``` |
| 8 | Adapt the object names of the imported POUs, if necessary. |
| 9 | Adapt SR\_HwCopyIO.  Adaption of the safety relevant hardware variables. |
