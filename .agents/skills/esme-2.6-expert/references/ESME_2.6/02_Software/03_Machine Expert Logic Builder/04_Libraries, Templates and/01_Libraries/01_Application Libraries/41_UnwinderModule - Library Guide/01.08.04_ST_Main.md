# ST_Main

![](images/logo.png)

ST\_Main

ST\_Main - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

Description

This structure contains overall operating mode parameters and return values of the FB\_UnwinderModule.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_ifDrive | SystemConfigurationITf. IF\_Drive | The axis that controls the bobbin. |
| i\_timMasterStop | TIME | The maximum time period in which the master must stop in case of an error reaction SynchronStopEH or SynchronStopEL. |
| i\_lrMotionActiveVelLimit | LREAL | Velocity where a motion is reported to q\_xMotionActive. |
| i\_etMotionActiveVelType | ET\_MotionActiveVelType | Velocity type to determine the motion for q\_xMotionActive. |
| q\_xDriveActive | BOOL | TRUE: The position control of the axis is active. |
| q\_etOpMode | ET\_OpMode | The active operating mode of the UnwinderModule. |
| q\_etState | ET\_State | The active state of the UnwinderModule. |
| q\_xMotionActive | BOOL | TRUE: The axis is in motion according to the definition of i\_lrMotionActiveVelLimit and i\_etMotionActiveVelType. |
