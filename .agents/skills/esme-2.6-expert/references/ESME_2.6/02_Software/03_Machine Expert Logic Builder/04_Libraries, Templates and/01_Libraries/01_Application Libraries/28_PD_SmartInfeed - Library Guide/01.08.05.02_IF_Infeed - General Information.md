Task

Interface for accessing the values and functions of the [FB\_Infeed](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080279_1) POU

Description

The algorithm for the [FB\_Infeed](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080279_1) requires extensive access to the values and functions of the POU. Thus, these are collected and provided in this interface.
