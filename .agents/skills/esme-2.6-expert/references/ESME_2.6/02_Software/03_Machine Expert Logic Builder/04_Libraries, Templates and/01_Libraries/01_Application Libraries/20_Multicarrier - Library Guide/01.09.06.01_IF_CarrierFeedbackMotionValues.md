# IF\_CarrierFeedbackMotionValues

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from | – |

## Task

Interface providing status values for the movement of a carrier.

## Description

The feedback interface provides status values for the reference motion parameters and the actual motion values of a carrier.
