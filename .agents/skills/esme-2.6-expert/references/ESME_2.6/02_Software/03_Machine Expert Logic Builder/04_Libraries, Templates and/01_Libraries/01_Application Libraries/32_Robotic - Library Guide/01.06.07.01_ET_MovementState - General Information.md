# ET\_MovementState - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.0.0 |

## Description

Enumeration type to describe a motion status.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Idle | 0 | The robot is at standstill. |
| Accelerating | 1 | The robot accelerates along the path. |
| InVelocity | 2 | The robot moves along the path with constant velocity. |
| Decelerating | 3 | The robot decelerates along the path. |
| External | 4 | The robot movement is executed by an external position source. |
