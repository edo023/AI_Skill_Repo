# ST\_Credentials

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.1.6.0 |
| Inherits from: | - |
| Attribute: | ``` 'pack_mode' := '1' ``` |

## Description

The structure ST\_Credentials contains the credentials required to access variables inside the symbol configuration in case symbol sets are enabled.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sUsername | STRING [255] | The user name for accessing variables if symbol sets are enabled. |
| sPassword | STRING [255] | The password for accessing variables if symbol sets are enabled. |
