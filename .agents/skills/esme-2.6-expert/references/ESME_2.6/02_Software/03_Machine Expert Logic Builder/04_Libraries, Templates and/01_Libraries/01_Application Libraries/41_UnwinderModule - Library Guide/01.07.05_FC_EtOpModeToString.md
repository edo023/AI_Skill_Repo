# FC_EtOpModeToString

![](images/logo.png)

FC\_EtOpModeToString

FC\_EtOpModeToString - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

Task

Convert the operating modes of the FB\_UnwinderModule to a string.

Description

Convert the operating mode of the UnwinderModule by the i\_etOpMode input from the ET\_OpMode enumeration list to a string.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etOpMode | ET\_OpMode | The operating mode that has to be converted to a string. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) | General, library-independent statement on the diagnostic.  A value not equal to ET\_Diag.Ok corresponds to a diagnostic message. |
| q\_etDiagExt | ET\_DiagExt | POU-specific output for the diagnostic.  q\_etDiag = ET\_Diag.Ok -> Status message  q\_etDiag <> ET\_Diag.Ok -> Diagnostic message |

Return Value

| Data type | Description |
| --- | --- |
| STRING[20] | The operating mode that was converted to a string. |

Diagnostic Messages

| q\_etDiag | q\_etDiagExt | Enumeration value | Description |
| --- | --- | --- | --- |
| OK | [Ok](#XREF_D_SE_0077490_8) | 0 | Proceeding normally, satisfactory or under control, or other affirmative acknowledgement. |
| InputParameterInvalid | [OpModeRange](#XREF_D_SE_0077490_9) | 55 | OpMode is outside of the valid range. |
