# ST\_MqttClientConfiguration

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

This structure is used to provide the configuration parameters specific for the MQTT client.

## Structure Elements

| Name | Data type | Default value | Description |
| --- | --- | --- | --- |
| uiKeepAlive | UINT | 0 | Maximum time interval in seconds the server can expect between two messages. |
| timAutoReconnectDelay | TIME | T#10S | Delay time before next attempt to reconnect after an interruption of the connection to the server. |
| usiAutoReconnectRetries | USINT | 0 | Number of attempts of automatic reconnection after an interruption of the connection to the server. |
| timConnectionTimeout | TIME | T#10S | Timeout for establishing a connection to the server. |
