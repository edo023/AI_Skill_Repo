# FC\_RotationMatrixInverse

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

Given a 3D rotation matrix, the function evaluates its inverse transformation.

The input matrix must be a rotation matrix. This function does not return a proper result if called on a generic 3D matrix.

When called on a valid matrix, the algorithm implemented by this function returns the same result of the generic FC\_Matrix3DInverse but with a faster execution.

NOTE: There is no internal verification about i\_stRotationMatrix being a valid rotation matrix.
