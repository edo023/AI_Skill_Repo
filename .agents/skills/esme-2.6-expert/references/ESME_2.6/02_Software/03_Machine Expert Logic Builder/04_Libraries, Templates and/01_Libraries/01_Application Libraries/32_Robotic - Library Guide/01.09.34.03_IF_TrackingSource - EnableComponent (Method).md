## Task

Enable one or more additional tracking components.

## Description

By default, the tracking is only enabled for cartesian components. Additional individual components or groups of components can be enabled with this method.

For each enabled component, the tracking motion parameters must be set with the method SetMotionParameters. In case a component is not available, the method IF\_RoboConfiguration.AddTrackingSource will report an exception.
