# Functional Description

## Program - SR\_<name function template>

The function template provides a program that has been created in programming language ST (Structured Text). This program is added in a new subfolder of the Application node. The name of the subfolder is assigned in the Add Function From Template dialog box. The program has the same name as the folder with the prefix SR\_.

A new task calling this program is added to the project. Therefore, the program must not be called explicitly inside your application. The program provides methods and properties that can be called inside your application to read the real time clock, the local time, and the time zone information out of the controller. The method SetRtc allows you to set the real time clock of the controller.

To facilitate a diagnostic process, the program provides local variables that contain the last detected errors.

## Visualization - Vis\_<name function template>

The function template implements a visualization to control and monitor functions. The name of the visualization corresponds to the name assigned within the Add Function From Template dialog box with the prefix Vis\_. It is available in the same folder as the program.

The visualization references the inputs and outputs of the program to control the real time clock.
