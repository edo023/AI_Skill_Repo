# SR\_<Conveyor Name> - Logic (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V2.1.1.0 |

## Functional Description

The method Logic is called cyclically and can be used to program any kind of logic for the conveyor.

For more detailed information, refer to [Logic Method and ModuleInterface](D-SE-0097893.html#D-SE-0097893).

## Interface

* In case of node type 'PacDrive 3 Template' and node type 'Non Template':

  | Input/Output | Data type | Description |
  | --- | --- | --- |
  | iq\_stConveyorModuleItf | [AXM. ST\_ModuleInterface](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077227) | Adapt the parameter for the different operation modes. |
* In case of node type 'Physical encoder' the interface has no inputs and outputs.
