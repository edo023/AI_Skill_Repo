# FB\_CertificateHandling

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.1.2.0 |

## Functional Description

The function block FB\_CertificateHandling is used to access the X509 certificates installed on the controller. It allows you to read the information from a certificate.

The function block has no inputs or outputs. The functionalities are provided via the methods and properties. A cyclic call of the function block is not permitted.

Use the following command sequence to access a certificate:

* Select an installed certificate using the methods SelectCertificateByCommonName or SelectFirstCertificate.
* Verify the value of the property State if a certificate is selected.
* Use the properties and methods of the function block to access the information of the selected certificate or to perform the desired tasks based on the properties or methods.
* Use the method ReleaseCertificate to release the certificate and the bound resources.

The properties State and Result are updated on each executed property or method of the function block. They must be used to verify the result of the last performed action.
