# ExecuteCurrentState (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.4.2.0 |
| Inherits from: | – |

## Description

If an instance of the FB\_StateModelHandlerBase is assigned to the present unit control mode, this method is used to invoke the state method of the present state. For assigning an instance of the FB\_StateModelHandlerBase to a unit control mode, the unit mode must be defined using the DefineUnitModeWithHandler() method.
