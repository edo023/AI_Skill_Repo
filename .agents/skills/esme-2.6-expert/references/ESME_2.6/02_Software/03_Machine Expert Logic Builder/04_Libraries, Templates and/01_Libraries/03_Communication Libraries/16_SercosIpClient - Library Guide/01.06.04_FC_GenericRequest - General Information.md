# FC\_GenericRequest - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.2.0 |

## Task

This function sends an S/IP request to a Sercos device.

## Functional Description

This function sends an S/IP request to a Sercos device. The Sercos device is specified by its IP address (the IP address can also be the broadcast address). The function can transmit any data passed to it.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sIPAddress | STRING[15] | IP address of Sercos device |
| i\_udiMessageType | UDINT | Message type to identify request |
| i\_stData | ST\_Data | Data to be transmitted by request |

| Output | Data type | Description |
| --- | --- | --- |
| q\_udiTransactionID | UDINT | Returns the transaction ID to be used for receiving the response |

## Return Value

| Data type | Description |
| --- | --- |
| ET\_Result | Refer to [ET\_Result](ET_Result-GeneralInformation-08184BB1.html). |
