# ST\_XmlUserDefinedHeader

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.8.0 |
| Inherits from: | - |

## Description

The structure ST\_XmlUserDefinedHeader allows you to define a header that is written to the XML file.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sUserComment | STRING[255] | Enter a text that is converted to an XML comment. It is written at the beginning of the XML file. |
| pbyAdditionalContent | POINTER TO BYTE | Pointer to the buffer provided by the application. It contains the content that is to be written at the beginning of the XML file in addition to the comment. |
| udiNumBytesToWrite | UDINT | Specifies the size in bytes for the additional content. |

NOTE: When no additional content is to be written, udiNumBytesToWrite and pbyAdditionalContent can be set to zero.
