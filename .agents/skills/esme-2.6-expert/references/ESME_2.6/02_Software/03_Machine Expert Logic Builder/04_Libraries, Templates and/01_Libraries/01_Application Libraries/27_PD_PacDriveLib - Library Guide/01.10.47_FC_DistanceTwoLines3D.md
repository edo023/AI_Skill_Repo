|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Calculates the distance between two straight lines in the 3-dimensional space.

Description

Calculates the distance between the straight lines i\_stLine1 and i\_stLine2 in the 3-dimensional space. In addition, the base point of the common perpendicular will be calculated for both straight lines.
