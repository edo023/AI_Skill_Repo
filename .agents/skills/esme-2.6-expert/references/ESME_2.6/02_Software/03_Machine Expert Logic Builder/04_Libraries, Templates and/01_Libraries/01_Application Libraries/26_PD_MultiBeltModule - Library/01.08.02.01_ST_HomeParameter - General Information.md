# ST\_HomeParameter - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.4.0 |
| Versions: | Current version |

## Description

This structure contains parameters and data that become effective in the operation mode Homing.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_etMode | [ET\_HomeMode](D-SE-0079096.html#D-SE-0079096) | Selection of the homing modes module. |
| i\_xEnableInitAbsReadPos | BOOL | Enables reading of position values of the axes from the retain area or the motor encoder. |
| i\_xUseTeachedStations | BOOL | Enables the use of the teached station positions. See [Teaching of station positions](D-SE-0077919.html#D-SE-0077919). |
| i\_xResetHomeOk | BOOL | Resetting the HomeOk output. This enforces homing of the trains. |
| i\_stPosToStationParameter | [MTB.ST\_HomePosToStationParameter](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.MultiBelt&topicID=D_SE_0077757) | Parameter for the ET\_HomeMode.PosToStation homing mode |
| i\_stOnTpParameter | [MTB.ST\_HomeOnTpParameter](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.MultiBelt&topicID=D_SE_0077755) | Parameter for the ET\_HomeMode.HomeOnTp homing mode |
| astSetPosParameter | ARRAY[1..[MTB.Gc\_uiMaxNumberOfBelts](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.MultiBelt&topicID=D_SE_0077781)] OF [PDL.ST\_HomeSetPos](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087736) | Parameter for the ET\_HomeMode.SetPos homing mode |
