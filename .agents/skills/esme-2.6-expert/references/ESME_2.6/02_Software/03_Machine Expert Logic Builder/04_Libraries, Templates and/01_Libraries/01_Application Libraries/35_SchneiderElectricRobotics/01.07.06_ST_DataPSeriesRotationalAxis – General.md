# ST\_DataPSeriesRotationalAxis – General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V2.14.0.0 |
| Inherits from: | - |

## Description

Contains a list of parameters to read out rotational axis related data from a Lexium P Robot.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| xRotationalAxisConfigured | BOOL | TRUE: Rotational axis available on this robot  FALSE: Rotational axis not available on this robot |
| diGearIn | DINT | GearIn for rotational axis |
| diGearOut | DINT | GearOut for rotational axis |
| lrUserDrivePeakCurrentMin | LREAL | Minimum value for UserDrivePeakCurrent for rotational axis |
| lrUserDrivePeakCurrentDefault | LREAL | Default value for UserDrivePeakCurrent for rotational axis |
| lrUserDrivePeakCurrentMax | LREAL | Maximum value for UserDrivePeakCurrent for rotational axis |
| lrFeedConstant | LREAL | FeedConstant for rotational axis |
| diMotorTemperatureMonitoring | DINT | The way the motor temperature is monitored for rotational axis. |
