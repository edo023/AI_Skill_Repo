# FC_InitEndlessParameter

![](images/logo.png)

FC\_InitEndlessParameter

FC\_InitEndlessParameter - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |
| Support for: | PacDrive3 pilot template architecture |
| Versions: | Current version |

Task

Function for initialization of an axis that is controlled by the [FB\_AxisModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0077136_1) function block.

Description

This function is used to assign values for the endless feed operating mode (run at constant velocity) for the axis specified by iq\_stAxisModuleItf.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrVelocity | LREAL | Specifies the run velocity unless reduced velocity is enabled, the velocity can be changed while the axis is moving. The axis will stop if the velocity is set to zero (i\_lrVelocity > -0.1 AND i\_lrVelocity < 0.1) and will wait until it becomes > 0.1 or < -0.1 in units/sec |
| i\_lrAcceleration | LREAL | Specifies the acceleration in units/sec^2 |
| i\_lrDeceleration | LREAL | Specifies the deceleration in units/sec^2 |
| i\_lrJerk | LREAL | Specifies the jerk in units/sec^3 |
| i\_lrPeriod | LREAL | The position of the axis is kept between zero and period. The axis will not move if period = 0 |
| i\_lrStopPosition | LREAL | Specifies the position the axis will go to when stopping in units |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stAxisModuleItf | ST\_ModuleInterface | The axis interface structure of the associated axis |
