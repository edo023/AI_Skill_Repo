# ET\_StartSimulationMode - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.4.0 |
| Versions: | Current version |

## Description

This enumeration selects the type of signal generation of the function block FB\_MulitBeltSimulation. The start signals only serve for testing and optimizing the application.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Equidistant | 1 | Signals are generated in fixed distances. |
| EquidistantGroups | 2 | Groups of signals are generated in fixed distances. |
| Random | 3 | Random signals are generated. |
| RandomGroups | 4 | Random groups of signals are generated. |
