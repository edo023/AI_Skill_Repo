# ST\_Matrix4D

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |

## Description

The structure ST\_Matrix4D represents a quadratic matrix with four lines and four columns.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| alrElement | ARRAY [1..4, 1..4] OF LREAL | Two-dimensional field of the matrix with the primary field representing the lines and the secondary field representing the columns. |
