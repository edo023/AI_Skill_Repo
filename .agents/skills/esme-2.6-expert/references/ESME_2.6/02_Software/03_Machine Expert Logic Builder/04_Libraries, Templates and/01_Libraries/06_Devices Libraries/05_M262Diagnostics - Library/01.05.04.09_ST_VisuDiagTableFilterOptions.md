Description

The structure ST\_VisuDiagTableFilterOptions specifies which diagnostic messages are displayed in the visualization [DiagTable](../Visualization/Visualization-3.htm#XREF_D_SE_0097660_1).

Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| xInfos | BOOL | Display messages with severity Information. |
| xWarnings | BOOL | Display messages with severity Warning. |
| xErrors | BOOL | Display messages with severity Error. |
| xSourceFbGetDiag | BOOL | Display messages with source FbGetDiag. |
| xSourceMotionKernel | BOOL | Display messages with source MotionKernel. |
