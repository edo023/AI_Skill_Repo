General Information

Description

The library "UnwinderModule" encapsulates the functionalities of the Unwinder Library and supports the interfaces to include the functionalities into a Template project.

Therefore, the following template functionalities can be used together with the functionality of the Unwinder Library:

oCommand handling

oError handling

oJogging management

Namespace of the Library

The default namespace of this library is UWM.

The POUs, data structures, enumerations and constants have to be addressed via this namespace.
