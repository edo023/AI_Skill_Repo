Description

Structure for parameterization of the homing modes for homing "Writing of position".

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_lrHomePosition | LREAL | Reference position at the end of the referencing.  The i\_lrHomePosition position value does not have to be within the range (0 ... EncoderPeriod or i\_lrUserPeriod. |
| i\_lrUserPeriod | LREAL | Machine period  > 0: periodic application: after homing, the axis position is in the range (i\_lrHomePosition MOD i\_lrUserPeriod); i\_lrAbsEncOffset is not considered.    = 0: not periodic application: after homing, the axis position is in the range - i\_lrAbsEncOffset ... (EncoderPeriod - i\_lrAbsEncOffset) |
| i\_lrAbsEncOffset | LREAL | Absolute value by which the axis position can be reconstructed in the negative range in the case of non-periodic applications.  This value must be greater than 0 for 'i\_lrUserPeriod = 0'. |
