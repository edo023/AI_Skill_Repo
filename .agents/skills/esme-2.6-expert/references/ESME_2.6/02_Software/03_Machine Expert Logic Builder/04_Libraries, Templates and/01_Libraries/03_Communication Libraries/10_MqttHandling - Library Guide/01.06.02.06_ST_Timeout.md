| --- | --- |

## Description

The structure ST\_Timeout contains the parameter to configure the timeout parameters.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| timTimeout | TIME | After this time is elapsed, the execution is canceled. If the value is T#0s, the default value T#10s is applied. |
| timRetransmissionTimeout | TIME | After this time is elapsed, a retransmission of the unacknowledged message is triggered. If the value is T#0s, the default value T#6s is applied. |
