# ST\_DiagMessageError

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | ST\_DiagMessage |

## Description

The structure ST\_DiagMessageError extends the structure ST\_DiagMessage and contains additional information indicating the severity of the diagnostic message of type error.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| uiSeverity | UINT | Severity of the diagnostic message. |
