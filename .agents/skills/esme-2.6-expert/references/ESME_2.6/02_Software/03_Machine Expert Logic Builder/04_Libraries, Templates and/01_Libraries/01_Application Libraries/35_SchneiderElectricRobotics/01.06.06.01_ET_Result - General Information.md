# ET\_Result - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.12.1.0 |

## Description

The enumeration ET\_Result contains the library diagnostic results.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Ok | 0 | Successful |
| RobotNotInitialized | 1 | The robot is not initialized. |
| InterfaceInvalid | 2 | The Interface is invalid. |
| CollisionMarginRange | 3 | The collision margin value is outside the admissible range. |
| ConfigurationNotSupported | 4 | The configuration of the robot function block is not supported. |
| RadiusRange | 5 | The provided value for the radius is outside the admissible range. |
