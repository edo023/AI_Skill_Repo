Description

The structure ST\_InfoEthernetDevices provides information on the Sercos slaves, EtherNet/IP  targets, and the Modbus TCP slaves.

Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stSercos | [ST\_SercosConfiguration](Structures-7.htm#XREF_D_SE_0097653_1) | Information on the configured Sercos slaves. |
| stEtherNetIP | [ST\_EtherNetIpConfiguration](Structures-3.htm#XREF_D_SE_0097655_1) | Information on the configured EtherNet/IP  targets. |
| stModbusTCP | [ST\_ModbusTCPConfiguration](Structures-5.htm#XREF_D_SE_0097654_1) | Information on the configured Modbus TCP slaves. |
