# FB\_MotorDataFileRead2 - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.7.4.0 |

## Task

This function block converts a binary file with motor data into a data structure.

## Description

This function block reads the specified binary file and writes the motor data from the file to an [ST\_UserMotorData](D-SE-0090544.html) structure.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xExecute | BOOL | Upon a rising edge the reading process is started. |
| i\_sFilename | STRING[255] | File name of the binary file with the motor data to be read. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xBusy | BOOL | Indicates that the process of reading the motor data file is in progress. |
| q\_xDone | BOOL | Indicates that the motor data file was read successfully. |
| q\_xError | BOOL | Indicates that an error is detected during the process of reading the motor data file. |
| q\_etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | General library-independent statement on the diagnostic.  A value unequal to GD.ET\_Diag.Ok corresponds to a diagnostic message. |
| q\_etDiagExt | [ET\_DiagExt](D-SE-0090520.html#D-SE-0090520) | POU-specific output on the diagnostic.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> diagnostic message |
| q\_sMsg | STRING[80] | Event-triggered message which gives more detailed information on the diagnostic state. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stUserMotorData | [ST\_UserMotorData](D-SE-0090544.html) | Structure that contains the application data read from the file. |

## Diagnostic Messages

| q\_etDiag | q\_etDiagExt | Enumeration value | Description |
| --- | --- | --- | --- |
| FileHandlingInvalid | FileAccessNotPossible | 14 | The file cannot be accessed. |
| FileHandlingInvalid | FileCorrupted | 65 | The format of the file is not supported. |
| FileHandlingInvalid | FileReadFailed | 64 | An error was detected while reading the file. |
| InputParameterInvalid | InvalidFilename | 67 | The given filename is invalid. |
| UnexpectedProgramBehavior | UnknownState | 63 | An unknown state is detected for the internal state machine. |

## FileAccessNotPossible

| Cause | Solution |
| --- | --- |
| The motor data is not readable. | Motor data of the defined drive cannot be read. |

## FileCorrupted

| Cause | Solution |
| --- | --- |
| The value for segment length in the file is invalid. | Only files created by the POU from this library are supported. |

## FileReadFailed

| Cause | Solution |
| --- | --- |
| An error was detected while reading the file. | Refer to the result msg from FB\_ReadFile for more details. |

## InvalidFilename

| Cause | Solution |
| --- | --- |
| The parameteri\_sFilename is a null string. | Assign the correct filename to i\_sFilename. |

## UnknownState

| Cause | Solution |
| --- | --- |
| An error occurred during internal processing. | Contact your Schneider Electric service representative. |
