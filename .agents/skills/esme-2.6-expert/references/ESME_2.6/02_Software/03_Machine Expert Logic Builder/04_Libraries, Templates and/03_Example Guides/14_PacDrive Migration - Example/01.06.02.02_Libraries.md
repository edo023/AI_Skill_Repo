# Libraries

## Overview

The following Schneider Electric libraries have been added to the Library Manager with the PacDrive Migration example project:

| Library | Function |
| --- | --- |
| ApplicationLogger | Supports the functionalities to store and manage a set of logger points to establish a logging for the application program.  For more information, refer to the [*ApplicationLogger Library Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=APL2LG&topicID=LibraryGuideEIO000000_00_2C85D7FD). |
| AsyncManager | The library offers the functionality to call time-intensive jobs asynchronously to avoid cycle time overruns without the need to separately create additional tasks.  For more information, refer to [*AsyncManager Library Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=AsyManLG&topicID=D_SE_0094746). |
| CAA Mathematics | This library provides elementary functions for handling mathematical operations.  For more information, refer to the [*CAA Mathematics Library Documentation*](../../../../../api/crossBook?lang=en-US&virtualBookName=CAA_Mathematics&topicID=). |
| CmpIecTask | System management library.  For more information, refer to the [*CmpIecTask Library Documentation*](../../../../../api/crossBook?lang=en-US&virtualBookName=CmpIecTask&topicID=) |
| CommonMotionTypes | Support of common motion data types independent from the controller platform.  For more information, refer to the [*CommonMotionTypes Library Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=ComMoTy&topicID=D_SE_0096115_13). |
| CommonToolbox | Support for functions and function blocks for various tasks (for example, diagnostics, conversion) independent from the controller platform.  For more information, refer to the [*CommonToolbox Library Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=comtb&topicID=D_SE_0105154). |
| IecRtsServices | This library contains the IEC implementations for particular services of the runtime system (reserved for internal use only). |
| MotionDeviceInterface | The library offers the functionality to access device handles. Internal access to device creation, data and configuration. |
| PD\_GlobalDiagnostics | Contains the enumeration with the general information about the exception for diagnostic output `q_etDiag` of the PacDrive 3 three-layer diagnostic concept for libraries.  For more information, refer to the [*PD\_GlobalDiagnostics Library Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=). |
| PD\_Template | Contains recurring functionalities for machine programming such as functions and structures for design, modular and scalable architecture and offers extensive functions for diagnostics and exception handling.  For more information, refer to the [*PD\_Template Library Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078585). |
| PLCopen MC part 1 | Motion control according to PLCopen Motion Control Part 1 v2.0 (formerly parts 1 and 2).  For more information, refer to the [*M262 Synchronized Motion Control Library Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=PLCO&topicID=D_SE_0088323). |
| SysFile | System management library.  For more information, refer to the [*SysFile Library Documentation*](../../../../../api/crossBook?lang=en-US&virtualBookName=SysFile&topicID=). |
| SysSem | System management library.  For more information, refer to the [*SysSem Library Documentation*](../../../../../api/crossBook?lang=en-US&virtualBookName=SysSem&topicID=). |
| SysTime••• | Controller real time clock management.  For more information, refer to the [*SysTimeRtc and SysTimeCore Library Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=RTCmanag&topicID=D_SE_0084196). |
