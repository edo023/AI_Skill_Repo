# IF\_ConnectionManager.CloseAllConnections (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

The method CloseAllConnections closes the connections that are held by the global Modbus TCP connection manager.

NOTE: All connections are closed, pending requests are discarded.

## Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| CloseAllConnections | BOOL | The return value indicates TRUE if the method was executed successfully. |

EIO0000004401.03
