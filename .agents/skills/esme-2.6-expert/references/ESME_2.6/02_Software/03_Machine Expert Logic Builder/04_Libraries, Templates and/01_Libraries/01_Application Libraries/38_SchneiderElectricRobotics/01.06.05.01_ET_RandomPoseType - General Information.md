# ET\_RandomPoseType - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.1.0.0 |

## Description

Adds a new enumerator to the library containing the following values.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No random pose type selected. |
| Pose | 1 | Random position and orientation in a range. |
| PoseOrientationList | 2 | Random position in a range. Orientation randomly picked from a list. |
| PoseInPlane | 3 | Random position in a range contained in a plane. Rotation about the axis normal to the plane. |
| PoseInPlaneRotationList | 4 | Random position in a range contained in a plane. Rotation about the axis normal to the plane randomly selected from a list. |
