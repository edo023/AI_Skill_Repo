## Functional Description

The FC\_StateSelectionAsDintFromBool function is a utility function for definition of the PackTags StatesDisabled and StatesModeChangeAllowed that allows you to assign the corresponding parameter to the method [DefineUnitMode of the FB\_UnitModeManager2](TPC_PackMLli_FB_UniMdMng2_DefineUnitMode.html).

The Boolean inputs representing the PackML states are combined using the OR expression to the return value of type DINT.
