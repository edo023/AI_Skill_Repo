# ST\_VariableToMonitor

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

This structure is used to specify a variable which is monitored by the FB\_ESMADataAcquisition.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| pbyVar | POINTER TO BYTE | Pointer to the variable. |
| etDatatype | ET\_DataType | Data type of the variable. |
| wsName | WSTRING(GCL.Gc\_uiLengthVarNameUtf16) | Name of the variable.  The name is used to identify the variable in the data telegram sent to the cloud application. |
| timTimeInterval | TIME | Time interval for the monitoring option.  The monitoring option determines when the value of the variable is added to the data telegram. If the value is set to T#0s, the value of the variable is sampled when its value changes, otherwise the value is sampled in the given interval. |
