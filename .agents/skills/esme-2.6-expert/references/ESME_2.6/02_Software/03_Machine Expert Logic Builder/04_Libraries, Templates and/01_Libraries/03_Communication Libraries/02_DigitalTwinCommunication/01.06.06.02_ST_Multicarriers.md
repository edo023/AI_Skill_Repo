## Description

The structure ST\_Multicarriers contains arrays of the multi carrier objects that can be built and emulated in EcoStruxure Machine Expert Twin.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| astTracks | ARRAY [1..MCR.GPL.Gc\_udiMaxNumberOfTracks] OF ST\_TrackConfig | The tracks that can be built and emulated.  MCR is the default namespace of the Multicarrier library. For further information, refer to the [Multicarrier Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=MLSLib&topicID=MulticarrierLG_5A6C118D). |
| astCarriers | ARRAY [1..MCR.GPL.Gc\_udiMaxNumberOfCarriers] OF ST\_CarrierConfig | The carriers that can be built and emulated.  MCR is the default namespace of the Multicarrier library. For further information, refer to the [Multicarrier Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=MLSLib&topicID=MulticarrierLG_5A6C118D). |
| astStations | ARRAY [1..GPL.Gc\_udiMaxNumberOfStations] OF ST\_StationConfig | The stations that can be built and emulated. |
