# Module Interface

![](images/logo.png)

Module Interface

Overview

The Module Interface provides access to variables of the module.

Double-click the Module Interface beneath the module to open the editor.

The editor provides two vertical sections:

oThe STANDARD\_INTERFACE section provides read-only access to the internal variables of the module.

oWith the USER\_INTERFACE section, you can define variables for your module.

Refer to [Tabular Declaration Editor](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083519_1).
