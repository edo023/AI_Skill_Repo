# Structured Parameter - AxisIntfOut

![](images/logo.png)

Structured Parameter - AxisIntfOut

Structure: AxisIntfOut

Structure of parameters related to the axis interface

| Parameter | Data type | Description |
| --- | --- | --- |
| xSync | BOOL | TRUE: Synchronization is active  FALSE: Synchronization is inactive |
| wActSyncGroup | WORD | Number of the synchronization group the axis is assigned to  Range: 1...65535 |
| xDrvFwd | BOOL | Forward direction command  TRUE / FALSE |
| xDrvRev | BOOL | Reverse direction command  TRUE / FALSE |
| wDrvSpdTarg | WORD | Speed reference  Range: 0...6000  Unit: RPM |
| wDrvAcc | WORD | Acceleration ramp time  Range: 1...999  Unit: 0.1 s |
| wDrvDec | WORD | Deceleration ramp time  Range: 1...999  Unit: 0.1 s |
| rEncPosDev | REAL | Position deviation  Range: 1.4 -45...3.4 +38  Unit: m or in. |
| wAlrmId | WORD | Alarm identification  Range: 0...32 |

xSync

Synchronization state of an axis. The value is TRUE if the axis is synchronized within a synchroni­zation group.

wActSyncGroup

Number of the synchronization group the axis is assigned to. When the axis is not assigned to any synchronization group, the value of this output is 0.

xDrvFwd, xDrvRev

Direction commands for the axis. The commands are intended to be connected to a device control function block.

wDrvSpdTarg

Target speed for the axis [RPM].

wDrvAcc, wDrvDec

Acceleration and deceleration ramp times for the axis, respectively. These parameters are used for correction of the position deviation and must be written to the variable speed drives continuously.

rEncPosDev

Actual value of deviation of the axis from the synchronous position.

The parameter is in meters if the axes within the synchronization group have defined linear and nominal speeds, or in pulses if they do not have defined linear and nominal speeds.

wAlrmId

This output identifies the detected alarm of the axis.

| Alarm bit | Description |
| --- | --- |
| 0 | Position deviation of the axis exceeded the maximum allowed deviation. |
| 1 | Selected synchronization group is undefined. |
| 2 | High-speed value out of range (1…6000). |
| 3 | Low speed value out of range (0…high-speed – 1). |
| 4 | Low speed value is greater than 0 and the axis is part of a synchronization group with flexible directions. This combination is not supported. |
| 5 | Acceleration ramp time parameter of the axis is outside the allowed range (1…999). |
| 6 | Deceleration ramp time parameter of the axis is outside the allowed range (1…999). |
| 7 | Nominal speed parameter of the axis is 0 and at least one of the other axes has this parameter or the linear speed parameter defined. |
| 8 | Linear speed parameter of the axis is less than or equal to 0 and at least one of the other axes has this parameter or the nominal speed parameter defined. |
| 9 | Parameter of encoder pulses per motor revolution is 0. |
| 10 | Resolution of encoder interface is not in the supported range (8...32 bits). |
| 11 | Drive is not ready. |
| 12 | Encoder interface is not ready |
