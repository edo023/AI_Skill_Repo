# FC_Vector2DTo3D

![](images/logo.png)

FC\_Vector2DTo3D

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Conversion of a 2-dimensional vector into a 3-dimensional vector.

Description

The components of the 2-dimensional vector are transferred into the 3-dimensional vector. The mapping regulation defining which components of the 2-dimensional vector are transferred into which components of the 3-dimensional vector is specified by means of i\_etMode.
