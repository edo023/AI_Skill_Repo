Task

Access to values and functions of the generator for the targets of an [FB\_Infeed](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080279_1) POU

Description

The targets of the [FB\_Infeed](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080279_1) can be created automatically or entered manually via the [IF\_Target­Manager](Interfaces-30.htm#XREF_D_SE_0080344_1) interface.
