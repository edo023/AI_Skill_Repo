## Supported Hardware

* PacDrive LMC Eco
* PacDrive LMC Pro
* PacDrive LMC Pro2

## Software Requirements

* SoMachine Motion V4.3

## Firmware Requirements

PacDrive 3 V4.3

* PacDrive LMC Eco V1.54.2.1 or greater
* PacDrive LMC Pro V1.54.2.1 or greater
* PacDrive LMC Pro2 V1.54.2.1 or greater

## New Functions

* None

## Modifications

* FB\_EllipticSpline.SetWorkingPlane

  A configured working plane can be reset by using the element ET\_WorkingPlane.None.

## Enhancements

* During the disable phase of the robot, a peak was visible in a trace of the LMC parameter CycleLoad. This could lead into a SERCOS cycle time overrun. This peak does not occur anymore and the disable behavior is fully supported in the present version.
* IF\_RobotJogging.raxJogCartesianPossible

  In case of a two-dimensional robot, the robot indicates that jogging into the third dimension is not possible.
