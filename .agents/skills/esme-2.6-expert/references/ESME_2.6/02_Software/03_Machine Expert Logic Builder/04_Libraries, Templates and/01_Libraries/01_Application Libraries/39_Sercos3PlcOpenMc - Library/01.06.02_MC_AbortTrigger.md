# MC_AbortTrigger

![](images/logo.png)

MC\_AbortTrigger

MC\_AbortTrigger - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function Block |
| Available as of: | 1.0.5.0 |
| Versions: | current version |

Task

The trigger started via an [MC\_TouchProbe](Function_Blocks-28.htm#XREF_D_SE_0090945_1) is to be cancelled.

Description

This function block calls up [SystemInterface.FC\_TPReset](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085379_1) to cancel the active Touchprobe tracing.
