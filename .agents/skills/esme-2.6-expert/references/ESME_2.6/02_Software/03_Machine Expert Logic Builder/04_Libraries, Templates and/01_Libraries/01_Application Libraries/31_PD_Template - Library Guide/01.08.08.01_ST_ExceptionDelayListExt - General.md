# ST\_ExceptionDelayListExt - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.4.0 |

## Description

This is a file type in which additional information for the respective exceptions can be saved.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| udiNext | UDINT | Via this variable the entries in the list of exceptions are interlinked with delayed reactions. |
