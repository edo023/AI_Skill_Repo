# ET\_WarmStartMode - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.12.1.0 |

## Description

Enumeration type to describe the warm start mode.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Default | 0 | Warm start mode Default can be used to move all robot components simultaneously back on path. |
| Sequenced | 1 | Warm start mode Sequenced can be used to move each robot component one after another back on path. |
