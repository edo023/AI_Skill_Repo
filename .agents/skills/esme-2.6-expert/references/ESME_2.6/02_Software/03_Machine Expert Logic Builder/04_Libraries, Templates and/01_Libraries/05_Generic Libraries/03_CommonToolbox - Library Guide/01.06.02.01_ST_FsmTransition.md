# ST\_FsmTransition

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.2.9.0 |

## Description

The structure ST\_FsmTransition provides information about a transition processed by the FB\_FiniteStateMachine. The structure is used as data type for a log entry at the FB\_FsmTransitionLogger.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| etType | ET\_TransitionType | Indicates the type of the logged transition |
| diStartState | DINT | Indicates the start state of the logged transition. |
| diTargetState | DINT | Indicates the target state of the logged transition. |
| sEvent | STRING | Indicates the event causing the transition. |
| sStartState | STRING | Indicates the start state of the logged transition as a string. |
| sTargetState | STRING | Indicates the target state of the logged transition as a string. |
| ldtTimestamp | LDT | Indicates the timestamp of the logged transition. |
