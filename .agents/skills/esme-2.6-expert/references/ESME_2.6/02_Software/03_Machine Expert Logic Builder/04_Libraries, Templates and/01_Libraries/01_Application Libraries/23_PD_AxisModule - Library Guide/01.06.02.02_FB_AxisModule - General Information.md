![](images/logo.png)

Task

The function block is used to move an axis. Various operating modes and motion functions are supported.

Description

Via the module, the most important module functionalities of the [PD\_PacDriveLib](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0090101_1) can be used in template projects.

The function block is addressed by the [Standard Module Interface](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078570_1) and AxisModule Interface.

Further details on how to operate the module can be found in the section [Behavior in the operating modes](Function_Blocks-5.htm#XREF_D_SE_0077326_1).
