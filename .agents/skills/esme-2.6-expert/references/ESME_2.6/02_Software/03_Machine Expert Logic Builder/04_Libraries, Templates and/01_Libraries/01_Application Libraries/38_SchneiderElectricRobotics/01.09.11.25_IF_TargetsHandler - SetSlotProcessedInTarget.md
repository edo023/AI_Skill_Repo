## Task

Sets the processed state of a slot inside a container target.

## Description

The method SetSlotProcessedInTarget allows you to set the processed state of a slot inside a container target.

If the processed state of the slot was set to FALSE and, as an effect of the call of this method, is now set to TRUE, the value of uiNumberOfProcessedSlots inside the structure of type ST\_RobotTargetWithSlots linked to the container target, is automatically increased.

If the processed state of the slot was set to TRUE and, as an effect of the call of this method, is now set to FALSE, the value of uiNumberOfProcessedSlots inside the structure of type ST\_RobotTargetWithSlots linked to the container target, is automatically decreased.
