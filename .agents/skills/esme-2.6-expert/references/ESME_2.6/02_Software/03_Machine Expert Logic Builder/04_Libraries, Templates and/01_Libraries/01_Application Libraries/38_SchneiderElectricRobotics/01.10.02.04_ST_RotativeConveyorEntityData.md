# ST\_RotativeConveyorEntityData

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.6.0.0 |
| Inherits from: | - |

## Description

Structure to store information for a rotative conveyor.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| etSystemId | [ET\_SystemEntity](D-SE-0098381.html#D-SE-0098381) | An enumeration used to unequivocally identify the target generator in the system.  Valid values are in the range ET\_SystemEntity.Conveyor1...ET\_SystemEntity.Conveyor30. |
| sName | STRING[80] | Name for the rotative conveyor. |
| stBasePose | [ST\_CartesianPose](D-SE-0080396.html#D-SE-0080396) | A cartesian pose used to describe the position and orientation of the rotative conveyor. |
| etSurfacePlane | SE\_MATH.ET\_CartesianPlane | An enumeration to define surface plane of the conveyor. |
| lrInnerRadius | LREAL | Inner radius of the rotative conveyor. |
| lrOuterRadius | LREAL | Outer radius of the rotative conveyor. |
| lrStartAngle | LREAL | Start angle for the rotative conveyor |
| lrDistanceAngle | LREAL | Distance angle from start angle to end of rotative conveyor. |
| xClockWiseRotation | BOOL | Moves the conveyor clockwise. |
