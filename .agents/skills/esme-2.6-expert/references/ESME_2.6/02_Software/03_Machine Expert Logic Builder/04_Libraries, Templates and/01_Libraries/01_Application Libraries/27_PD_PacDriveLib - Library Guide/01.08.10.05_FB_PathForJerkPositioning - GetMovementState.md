# FB_PathForJerkPositioning - GetMovementState (Method)

FB\_PathForJerkPositioning - GetMovementState (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.5.3.0 |
| Versions: | Current version |

Task

Get the position, velocity, acceleration, and jerk for a specific time.

Description

This method returns the movement state (position, velocity, acceleration, jerk) at its outputs for a time specified by i\_lrTime.

If the time is outside the domain of the path, the initial or the final movement state is returned.

This method provides no diagnostic information and no direct feedback value of the method. The result is presented via the declared outputs of the method.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrTime | LREAL | Time specified for the movement state |

| Output | Data type | Description |
| --- | --- | --- |
| q\_lrPosition | LREAL | Position at the time specified by i\_lrTime |
| q\_lrVelocity | LREAL | Velocity at the time specified by i\_lrTime |
| q\_lrAcceleration | LREAL | Acceleration at the time specified by i\_lrTime |
| q\_lrJerk | LREAL | Jerk at the time specified by i\_lrTime |
