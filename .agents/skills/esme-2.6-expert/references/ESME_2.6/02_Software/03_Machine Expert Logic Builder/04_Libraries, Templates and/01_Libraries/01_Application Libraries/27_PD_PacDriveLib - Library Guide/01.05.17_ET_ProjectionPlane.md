Description

Enumeration type for defining a coordinate system plane.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| XY | 1 | Plane which is spanned by the X and the Y axis |
| XZ | 2 | Plane which is spanned by the X and the Z axis |
| YZ | 3 | Plane which is spanned by the Y and the Z axis |
