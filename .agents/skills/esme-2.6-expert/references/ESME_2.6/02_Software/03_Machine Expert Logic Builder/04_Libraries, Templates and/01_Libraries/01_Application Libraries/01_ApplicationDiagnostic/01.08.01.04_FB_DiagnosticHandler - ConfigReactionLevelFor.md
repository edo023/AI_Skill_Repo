## Task

Configures the reaction level of the reaction to forward to the children.

## Description

The method ConfigReactionLevelForwarding is used to configure the reaction level that is forwarded to the child diagnostic handlers. The mapping can be configured individually for each child diagnostic handler.

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Indicates TRUE if the configuration is valid and applied. |

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xSetConfigForAllTargets | BOOL | If set to TRUE, the mapping table is applied for all sources.  If set to FALSE, the input i\_ifSourceChild specifies the source for which the mapping table is applied. |
| i\_ifTargetChild | IF\_DiagHandlerAcces | This input specifies the source diagnostic handler for which the mapping table is applied.  If this input is set to 0, the mapping table is applied for diagnostic messages set through the method SetDiagMsgError.  This input is relevant only if i\_xSetConfigForAllSources is FALSE. |
| i\_auiReactionLevel | ArrayReactionLevelForwarding | Table specifying the mapping of the local reaction level and the reaction level that is forwarded to the child diagnostic handler. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | Value is TRUE if an error was detected. For details, refer to q\_etResult and q\_sResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-36C542C6.html) | Provides diagnostic and status information as a numeric value.  If q\_xError  = FALSE, q\_etResult provides status information. If q\_xError  = TRUE, q\_etResult provides diagnostic/error information. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text value. |
