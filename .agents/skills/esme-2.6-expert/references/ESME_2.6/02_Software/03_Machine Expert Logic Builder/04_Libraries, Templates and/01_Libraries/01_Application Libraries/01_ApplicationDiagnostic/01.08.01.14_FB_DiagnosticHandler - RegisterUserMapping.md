## Task

Registers a callback for user-defined source code in case a diagnostic message is set or forwarded by the child.

## Description

The method RegisterUserMapping is used to register a callback for user-defined source code in case a diagnostic message is set or forwarded by the child.

It allows to overwrite:

* The incoming diagnostic message and the mapped local reaction level.
* The forwarded severity of the diagnostic message forwarded to the parent.
* The forwarded reaction level to the children.

NOTE: The user-defined source code to be called must be added in the appropriate methods of a function block defined in the application which must implement the interface [IF\_UserMapping](IF_UserMapping-GeneralInformation-45241C44.html).

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Indicates TRUE if the callback for user mapping has been registered successfully.  If FALSE is returned, an invalid interface is assigned to the input i\_ifDiagnosticListener. |

## Interface

| Input | Data type | Description |
