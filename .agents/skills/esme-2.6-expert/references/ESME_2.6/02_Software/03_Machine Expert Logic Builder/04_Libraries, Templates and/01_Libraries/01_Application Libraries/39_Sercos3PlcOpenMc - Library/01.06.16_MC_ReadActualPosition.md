# MC_ReadActualPosition

![](images/logo.png)

MC\_ReadActualPosition

MC\_ReadActualPosition - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function Block |
| Available as of: | 1.0.5.0 |
| Versions: | current version |

Task

The position of the axis is to be read out.

Description

The RefPosition parameter of the axis is returned.
