# ET\_ModeFileOpen

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.8.0 |

## Description

The enumeration ET\_ModeFileOpen specifies the mode for opening a file.

## Enumeration Elements

| Name | Value (INT) | Description |
| --- | --- | --- |
| NotSet | 0 | No mode has been selected. |
| Append | 1 | An existing file is opened and the specified content is appended.  If the file does not exist, the function block indicates an error. |
| AppendPlus | 2 | As for Append, an existing file is opened and the specified content is appended but with AppendPlus, if the file does not exist, a new file is created. |
| Create | 3 | A file is created and the specified content is written.  If the file already exists, the function block indicates an error. |
| CreatePlus | 4 | As for Create, a file is created and the specified content is written but with CreatePlus, if the file already exists, the content is overwritten. |
