Task

Get the time of a specific positioning phase.

Description

This method returns the time of a specific positioning phase (T11, T12, T13, T2, T31, T32, T33). Refer to figure in [FB\_RgPosStartOptimised](Function_Blocks_R_to_Z-8.htm#XREF_D_SE_0087349_1) for meaning of indices 11, 12, 13, 2, 31, 32, 33.

For example GetDeltaT32 returns the time of phase 32, the phase of constant acceleration while decomposing the cruising velocity.

If a phase is omitted, zero is returned for this phase.

Return Value

| Data type | Description |
| --- | --- |
| LREAL | Returning the time of the specific positioning phase. |
