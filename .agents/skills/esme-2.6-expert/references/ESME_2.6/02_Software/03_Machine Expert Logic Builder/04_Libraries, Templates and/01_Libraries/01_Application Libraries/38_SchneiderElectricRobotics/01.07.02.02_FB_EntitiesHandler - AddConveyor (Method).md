## Task

Store a new conveyor entity inside the function block.

## Description

The method AddConveyor is used to store a new conveyor entity inside the function block.

Once a conveyor has been stored with a specific ID, it is not possible to use such ID to store a new conveyor entity, since every entity ID must be unique in the system.

NOTE: It is possible to reconfigure an already added conveyor by calling the method again for the same conveyor ID.
