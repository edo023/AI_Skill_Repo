# FC_EtDiagExtToString

![](images/logo.png)

FC\_EtDiagExtToString

FC\_EtDiagExtToString - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Converting an enumeration element of type [ET\_DiagExt](../Enumerations/Enumerations-4.htm#XREF_D_SE_0080267_1) of the PD\_SmartInfeed library to a string

Description

Every enumeration element has a name or value. The return value of the function is the name of the enumeration element.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etDiagExt | [ET\_DiagExt](../Enumerations/Enumerations-4.htm#XREF_D_SE_0080267_1) | The ET\_DiagExt value to be converted |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) | General, library-independent statement on the diagnosis.  A value unequal GD.ET\_Diag.Ok equates an error message. |
| q\_etDiagExt | [ET\_DiagExt](../Enumerations/Enumerations-4.htm#XREF_D_SE_0080267_1) | POU-specific output on the diagnosis.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> error message |

Return Value

| Data type | Description |
| --- | --- |
| STRING[80] | Name of the enumeration element that was transferred at the input i\_etDiagExt. |

Diagnostic Messages

| q\_etDiag | q\_etDiagExt | Enumeration value | Description |
| --- | --- | --- | --- |
| OK | [Ok](#XREF_D_SE_0080299_8) | 0 | Ok |
| InputParameterInvalid | [UnknownDiagExt](#XREF_D_SE_0080299_10) | 10 | DiagExt is unknown. |
| UnexpectedProgramBehavior | [UnexpectedDiagExt](#XREF_D_SE_0080299_9) | 9 | DiagExt is unexpected. |
