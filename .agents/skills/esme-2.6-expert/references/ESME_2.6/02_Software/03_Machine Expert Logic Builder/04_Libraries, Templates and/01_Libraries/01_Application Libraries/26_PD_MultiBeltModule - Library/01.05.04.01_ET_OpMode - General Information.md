# ET\_OpMode - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.4.0 |
| Versions: | Current version |

## Description

This enumeration contains the operation modes of the MultiBeltModule.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| NoOp | 1 | No operation mode selected. |
| Homing | 2 | Home the axes according to the homing mode defined in ST\_HomeParameter. |
| Manual | 3 | Manual jogging of all axes. |
| Automatic | 4 | Automatic mode. The Multibelt technology POUs are being executed. |
| BrakeRelease | 5 | Operation mode to release the brakes of selected axes. |
| Service | 6 | Operation mode for commissioning. Single axis homing, release brakes, jogging and teach station can be controlled using bit signals.  **Safety-related functions are deactivated in this operation mode.** |
