# FC_MatrixProduct

![](images/logo.png)

FC\_MatrixProduct

FC\_MatrixProduct - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Calculates the product of two matrixes (not necessarily quadratic).

Description

Calculates the product of two (not necessarily quadratic) matrices. To be able calculate the product of two matrices, the following condition has to be fulfilled: i\_diCols1 = i\_diRows2

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_plrMatrix1 | POINTER TO LREAL | Pointer on the beginning of the memory area in which matrix 1 is stored. |
| i\_diRows1 | DINT | Number of lines of matrix 1. Must be >= 1. |
| i\_diCols1 | DINT | Number of columns of matrix 1. Must be >= 1. |
| i\_plrMatrix2 | POINTER TO LREAL | Pointer on the beginning of the memory area in which matrix 2 is stored. |
| i\_diRows2 | DINT | Number of lines of matrix 2. Must be >= 1. |
| i\_diCols2 | DINT | Number of columns of matrix 2. Must be >= 1. |
| i\_plrMatrixResult | POINTER TO LREAL | Pointer on the beginning of the memory space in which the product matrix1 \* matrix2 has to be copied. |
