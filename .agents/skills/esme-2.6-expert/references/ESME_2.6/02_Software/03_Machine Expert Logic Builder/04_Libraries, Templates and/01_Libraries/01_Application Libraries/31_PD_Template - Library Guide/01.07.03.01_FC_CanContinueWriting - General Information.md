## Task

The approval to process is queried in the routines using FC\_CanContinueWriting to ensure that Diagnostic does nor require system resources in critical routines.

## Description

Processing blockage of FC\_CanContinueWriting is kept until FB\_DiagnosticsManager resets the state (to the next cycle start).

The method checks the system state for cycle timeouts and overfilled ramdisk. Further configuration details are described in the ST\_CanContinueWriting structure.
