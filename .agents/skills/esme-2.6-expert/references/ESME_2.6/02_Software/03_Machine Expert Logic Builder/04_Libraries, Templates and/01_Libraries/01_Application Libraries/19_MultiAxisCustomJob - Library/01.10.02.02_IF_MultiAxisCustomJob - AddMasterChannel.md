## Description

The method AddMasterChannel of the interface [IF\_MultiAxisCustomJob](IF_MultiAxisCustomJob-4D7A4899.html) is used to assign a master axis to the axis group of the multi-axis custom job. Connecting a master axis allows the user-defined algorithm to reference the movement values of the master axis as inputs for calculating target values for the subordinate axes.

This method must be called before the multi-axis custom job is started. Master channels cannot be added or modified while the job is in the state Executing.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
