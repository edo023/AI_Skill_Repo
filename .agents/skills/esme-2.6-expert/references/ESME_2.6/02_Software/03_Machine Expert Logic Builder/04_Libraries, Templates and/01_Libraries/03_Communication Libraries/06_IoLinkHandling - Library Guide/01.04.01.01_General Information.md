# General Information

## Library Overview

The library IoLinkHandling provides the functionality to configure the IO-Link master and the IO-Link devices from within the application.

Therefore, dedicated function blocks to read and write device parameters, configure IO-Link master user parameters, and react on events generated by the IO-Link master and devices are implemented.

NOTE: For this library, qualified-access-only is set. The POUs, data structures, enumerations, and constants must be accessed using the namespace of the library. The default namespace of the library is SE\_IoLink.

## Controller Platforms

The IoLinkHandling library is supported by the following controller platforms:

* Modicon M262 Logic/Motion Controllers
* PacDrive LMC controllers

This approach places the controllers behind one or more firewalls to restrict access to authorized personnel and protocols only.
