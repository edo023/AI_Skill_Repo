# FC\_Polynomial5NormalizedCoefficients

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the coefficients of a standardized polynomial from the boundary conditions. The function fulfills the same task as the function FC\_Polynomial5Coefficients. The only difference is that a standardized X space is used before the polynomial coefficients are calculated. This is done by the following transformation:

P(X) = Pnorm((X - i\_lrX0) / (i\_lrX1 - i\_lrX0))

Pnorm is a polynomial with an X space standardized to the interval [0, 1] to which the following applies:

P(i\_lrX0) = Pnorm(0)

P(i\_lrX1) = Pnorm(1)

Compared to FC\_Polynomial5Coefficients, this way of proceeding has the **advantage** of a reduced sensitivity with regard to rounding errors (in particular, if i\_lrX0 is large). A **disadvantage**, however, is that, in addition to the polynomial coefficients, the values i\_lrX0 and i\_lrX1 must be stored. This is done in the extended polynomial coefficient structure [ST\_Poly5NormalizedCoefficients](ST_Poly5NormalizedCoefficients-Gene-0C05B3FB.html#ST_Poly5NormalizedCoefficients-Gene-0C05B3FB).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrX0 | LREAl | Left boundary point of the X interval |
| i\_lrY0 | LREAl | Required function value of the polynomial at the point i\_lrX0 |
| i\_lrM0 | LREAl | Required gradient of the polynomial at the point i\_lrX0 |
| i\_lrK0 | LREAl | Required curvature of the polynomial at the point i\_lrX0 |
| i\_lrX1 | LREAl | Right boundary point of the X interval |
| i\_lrY1 | LREAl | Required function value of the polynomial at the point i\_lrX1 |
| i\_lrM1 | LREAl | Required gradient of the polynomial at the point i\_lrX1 |
| i\_lrK1 | LREAl | Required curvature of the polynomial at the point i\_lrX1 |
