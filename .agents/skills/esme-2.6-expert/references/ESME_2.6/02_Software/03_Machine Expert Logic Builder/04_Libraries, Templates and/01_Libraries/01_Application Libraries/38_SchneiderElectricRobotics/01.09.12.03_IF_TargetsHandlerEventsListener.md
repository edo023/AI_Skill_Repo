## Task

Provides user-specific diagnostics when a target with slots is added to the list.

## Description

The OnAddTargetWithSlots method is called by the targets handler where the implementation of IF\_TargetsHandlerEventsListener has been subscribed every time a target and its slots are added to the list.

The following methods trigger the event:

* AddTarget
* AddTargetFromTargetWithSlots
* AddTargetWithSlots

The diagnostics provided as output depend on your specific implementation. They are ignored by the library.
