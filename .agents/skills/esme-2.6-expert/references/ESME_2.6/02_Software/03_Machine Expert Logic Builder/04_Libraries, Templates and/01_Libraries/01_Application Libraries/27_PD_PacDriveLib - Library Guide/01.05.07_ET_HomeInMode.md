ET\_HomeInMode - General Information

Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Description

Enumeration type to describe the homing modes for homing to a sensor signal.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| PosDirectionPosEdgeSensor | 0 | oMotion into positive axial direction  oWaiting for a positive edge of the sensor (normally open contact) |
| NegDirectionPosEdgeSensor | 1 | oMotion into negative axial direction  oWaiting for a positive edge of the sensor (normally open contact) |
| NegDirectionNegEdgeSensor | 2 | oMotion into negative axial direction  oWaiting for a negative edge of the sensor (normally closed contact) |
| PosDirectionNegEdgeSensor | 3 | oMotion into positive axial direction  oWaiting for a negative edge of the sensor (normally closed contact) |
