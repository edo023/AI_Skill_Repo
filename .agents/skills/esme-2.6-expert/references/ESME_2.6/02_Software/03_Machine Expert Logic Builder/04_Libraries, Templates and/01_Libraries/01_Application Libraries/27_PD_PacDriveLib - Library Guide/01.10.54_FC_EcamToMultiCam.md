# FC_EcamToMultiCam

![](images/logo.png)

FC\_EcamToMultiCam

FC\_EcamToMultiCam - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |

Task

Reading a file created using Motion Builder

Description

NOTE:

The function FB\_EcamToMultiCam only works correctly if the used data structures are saved with Pack\_Mode = 1.

The function reads files generated by the Motion Builder Toolkit (<Name>.dat) and saves the data to a structure of type [ST\_MultiCam](../Structures/Structures-32.htm#XREF_D_SE_0087770_1).

| Input | Data type | Description |
| --- | --- | --- |
