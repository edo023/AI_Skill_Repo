# ET\_Result - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.7.0 |
| Versions: | Current version |

## Description

The enumeration ET\_Result contains the possible values that indicate the result of operations executed by the POUs of this library.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| If q\_xError of a function block is set to FALSE, one of the following status messages is presented. | | |
| Ok | 0 | Ok |
| InitAlreadySuccessfullyCalled | 1 | The method GVL.G\_ifAsyncMgr.Init has already been executed successfully. |
| CreatingAsyncTaskNames | 2 | The creation of the names for the additional tasks is in progress. |
| CreatingAsyncTasks | 3 | The creation of the additional tasks is in progress. |
| WaitUntilAsyncTasksCreated | 4 | Wait until asynchronous tasks have been generated. |
| If q\_xError of a function block is set to TRUE, one of the following error messages is presented. | | |
| UnexpectedFeedback | 100 | An internal system function returned an indeterminable feedback. |
| AsyncAddWasNotSuccessful | 101 | The function for passing of the method Job to the additional task detected an error. |
| InterfaceCastFailed | 102 | The system function \_\_QUERYINTERFACE was unsuccessful. |
| MaxNumOfReusableObjectsExceeded | 103 | Maximum number of simultaneous assigned asynchronous jobs has been reached.  You can either restructure your application program in order to limit the number of outsourced functions at the same time or you can increase the value in the parameter GPL.Gc\_MaxNumOfReusableObjects. |
| SemaphoreCreateFailed | 104 | The system function used for creating a semaphore was unsuccessful. |
| AsyncInvalid | 105 | The object assigned to the input i\_ifAsync is invalid. |
| NoAsyncTasksCreated | 106 | The method GVL.G\_ifAsyncMgr.Init was not yet successfully executed. |
| ColdResetOfControllerNecessary | 108 | The parameter GPL.Gc\_udiNumOfAsyncTasks has been modified. Perform a Cold Reset. |
| AsyncInitNotFinished | 109 | The initialization of the AsyncManager is not finished. |
| CreatingAsyncTasksFailed | 110 | The creation of at least one of the additional tasks was unsuccessful. |
