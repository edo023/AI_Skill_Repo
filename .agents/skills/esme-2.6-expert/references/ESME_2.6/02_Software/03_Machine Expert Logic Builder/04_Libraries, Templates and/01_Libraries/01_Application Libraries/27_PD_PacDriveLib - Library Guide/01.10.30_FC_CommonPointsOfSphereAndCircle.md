FC\_CommonPointsOfSphereAndCircle

FC\_CommonPointsOfSphereAndCircle - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Calculates the intersections of a circle and a sphere in the 3-dimensional space.

Description

Calculates the intersections of the sphere i\_stSphere and the circle i\_stCircle. If there are no intersections, the q\_diNumberOfCommonPoints is 0.
