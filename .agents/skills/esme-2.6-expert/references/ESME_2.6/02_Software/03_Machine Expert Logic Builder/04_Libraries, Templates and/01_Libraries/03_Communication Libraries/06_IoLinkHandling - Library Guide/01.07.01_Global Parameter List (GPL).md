# Global Parameter List (GPL)

## Overview

|  |  |
| --- | --- |
| Type: | Global parameters |
| Available as of: | V1.2.5.0 |

## Description

The global parameter list (GPL) contains global constants that are used by certain components of this library. The parameters can be edited individually for each application where the library is used and are automatically used by the respective POU.

## Global Variables

| Variable | Data Type | Default value | Value range | Description |
| --- | --- | --- | --- | --- |
| `Gc_udiEventLogListSize` | UDINT | 100 | 10...1000 | Maximum number of events to be logged in the list. |
