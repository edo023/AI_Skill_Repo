# FB\_MovingAverage - Init (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.1.0.0 |

## Description

This method initializes the memory area of the function block FB\_MovingAverage. The properties of the function block FB\_MovingAverage provide the following values after you have called this method:

* lrAverage: 0.0
* uiLevel: 0
* uiOrder: Value of i\_uiOrder, if valid

This method has to be called at least once before you can call the method Insert.
