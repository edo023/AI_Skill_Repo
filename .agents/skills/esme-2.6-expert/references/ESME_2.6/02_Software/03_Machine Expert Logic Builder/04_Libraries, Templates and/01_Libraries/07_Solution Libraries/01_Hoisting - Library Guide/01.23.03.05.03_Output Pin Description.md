# Output Pin Description

![](images/logo.png)

Output Pin Description

Output Pin Description

| Output | Data Type | Description |
| --- | --- | --- |
| q\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block. |
| q\_xLampYell | BOOL | Output for the alert light.  TRUE: Alert indicator active and inactive for 6 s continuously.  FALSE: No alert |
| q\_xLampRed | BOOL | Output for the alarm light.  TRUE: Alarm indicator is permanently ON.  FALSE: No alarm |
| q\_xHorn | BOOL | Output for the horn during an alarm or alert.  TRUE: Alarm  PULSE: Alert  FALSE: OK |
| q\_xAlrm | BOOL | Alarm detection indicator.  TRUE: Alarm detected  FALSE: No alarm detected |
| q\_wAlrmld | WORD | Detected Alarm identification.  Range: 0...7  For more information [refer to Notifications](#XREF_D_SE_0003452_3). |
| q\_adtDttmAlrm | Array of DT | Alarm event register output to access from the HMI. It stores the last 11 alarm events which are time stamped (0...10). The 0 position is the most recent event.  Range: 0...10 |

Notifications

| Bit Number | Description |
| --- | --- |
| 0 | No signal has been received from anemometer for more than 8 hours. |
| 1 | i\_wAlrtThsh was too high and has been set to the maximum allowed value. |
| 2 | i\_wAlrmThsh was too high and has been set to the maximum allowed value. |
