FC\_Rotation2D

Overview

|  |  |

| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Rotation of a point in the 2-dimensional space by an angle.

Description

The point with the position vector i\_stV is rotated about the coordinate origin by an angle of i\_lrAngle.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stV | [ST\_Vector2D](../Structures/Structures-47.htm#XREF_D_SE_0087800_1) | Point to be rotated |
| i\_lrAngle | LREAL | Angle by which the point is to be rotated (in degrees) |
