# ET_MovementStatus

![](images/logo.png)

ET\_MovementStatus

ET\_MovementStatus - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.4.0 |

Description

This enumeration type defines whether a movement on the related path is possible or not.

Type: UINT.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Enabled | 0 | Movement is possible/allowed. |
| SuspendedByCommand | 1 | No movement because of previously issued SupendMovement command. |
| EmergencyStopActive | 2 | Movement suspended after detecting an emergency stop. |
| InterlockActive | 3 | Movement suspended after detecting a related I/O signal from an interlock circuit. |
| Unknown | 4 | Invalid movement status |
