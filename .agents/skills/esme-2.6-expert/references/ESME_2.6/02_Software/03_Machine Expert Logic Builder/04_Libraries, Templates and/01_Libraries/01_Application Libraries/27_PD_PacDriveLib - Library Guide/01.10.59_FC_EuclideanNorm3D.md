Task

Euclidean norm of a 3-dimensional vector.

Description

This calculates the Euclidean default of the vector i\_stV:

![G-SE-0068649.1.gif-high.gif](images/G-SE-0068649.1.gif-high.gif)

The Euclidean Norm defines the length of a vector.
