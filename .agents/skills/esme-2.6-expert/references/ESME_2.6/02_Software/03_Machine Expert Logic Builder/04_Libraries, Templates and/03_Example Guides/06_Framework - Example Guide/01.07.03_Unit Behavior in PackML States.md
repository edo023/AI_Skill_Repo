# Unit Behavior in PackML States

For connecting units, it is a good practice to create a guideline defining the behavior of a unit in specific PackML states. The following guidelines are used in the framework project. Adjust them to meet your specific guidelines.

| PackML state | Unit behavior |
| --- | --- |
| Aborted | Initial state after enabling the machine.  Servo motors are torque-free. Safety-related doors are unlocked |
| Aborting | Stops the machine.  Servo motors are stopped and disabled. To perform a synchronous stop, the parent unit controlling the master axis stops the master axis first. Afterward, it sends the Abort command to the subunits. |
| Clearing | The machine is under power. The control loop of servo drives is activated. |
| Stopping | The machine is stopped in a controlled way.  The servo motors are stopped. To perform a stop at the end of the machine cycle, the parent unit controlling the master axis moves to the end of the cycle and then sends the Stop command to the subunits. |
| Stopped | The machine is stopped. Servo motors stand still with the activated control loop. |
| Resetting | A diagnostic reset / error acknowledgement is executed if necessary. Recipe data is loaded. The safety-related door locks are engaged. |
| Idle | The machine is ready to start. |
| Starting | Homing is executed if needed. A warmstart is executed if needed. The horn is engaged if needed. Cams for operation mode Producing are loaded and activated. |
| Execute | The machine is producing. |
| Completing | The machine finishes the batch of products. |
| Completed | The machine stands still and waits for a start with the next batch of products. |
| Held | A minor error was detected in the unit (or at least one of its subunits). Production is stopped until the error is resolved and external actions (filling the magazine, filling the glue, etc.) are performed. |
| Suspended | The unit cannot produce because it is input-starved or output-blocked. The unit changes back to Execute automatically once the starved/blocked situation is resolved. |
