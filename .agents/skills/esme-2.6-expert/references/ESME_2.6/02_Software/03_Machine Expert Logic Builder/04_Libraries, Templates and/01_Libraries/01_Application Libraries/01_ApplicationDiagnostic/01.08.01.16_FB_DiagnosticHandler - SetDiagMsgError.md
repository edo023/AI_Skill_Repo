## Task

Sets a diagnostic message of category [ET\_Category.Error](ET_Category-GeneralInformation-36C5F24D.html).

## Description

The method SetDiagMsgError is used to set a diagnostic message of type error at a diagnostic handler equal to [ET\_Category.Error](ET_Category-GeneralInformation-36C5F24D.html) .

NOTE: Messages set after the diagnostic quit command was triggered are handled as a new diagnostic message. If resetting the function block reporting the diagnostic message takes more than one cycle and its error output remains TRUE, the diagnostic message should be set on a rising edge of the error output to avoid a new reaction while the diagnostic quit is in progress.

NOTE: To help avoid unnecessary processor load, call this method only once when the cause of the diagnostic message occurs.
