## Description

The interface FB\_MasterChannel serves as the interface for a master axis within a multi-axis group. It tracks the movement of a connected master axis to provide a continuous position reference for the user-defined motion algorithm.

The interface calculates the distance covered by the master axis in each RTP cycle and maintains an internal position reference based on these increments. Discontinuities in the absolute position of the connected axis (such as modulo rollovers) do not affect this reference.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
