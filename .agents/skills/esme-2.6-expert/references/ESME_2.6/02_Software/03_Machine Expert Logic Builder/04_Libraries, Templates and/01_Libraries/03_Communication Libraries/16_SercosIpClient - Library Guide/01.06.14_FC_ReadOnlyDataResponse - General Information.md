# FC\_ReadOnlyDataResponse - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.2.0 |

## Task

This function returns the response to the ReadOnlyData service from a Sercos device. This service is used to read a parameter.

## Functional Description

This function returns the response to the ReadOnlyData service from a Sercos device to which a request had been sent via FC\_ReadOnlyDataRequest. The response to the request is saved in the parameter iq\_stData.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiTransactionID | UDINT | Transaction ID of the corresponding request |
| i\_diTimeoutMs | DINT | Timeout in milliseconds for waiting for the response |

| Output | Data type | Description |
| --- | --- | --- |
| q\_uiReadDataLength | UINT | Length in bytes of the data that was read |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stData | ST\_Data | Structure to store the data to be read |

## Return Value

| Data type | Description |
| --- | --- |
| ET\_Result | Refer to [ET\_Result](ET_Result-GeneralInformation-08184BB1.html). |
