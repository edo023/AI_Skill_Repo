# ET\_FeedbackState - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V3.2.0.0 |

## Description

Enumeration type to address the robot feedback state.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No feedback state has been selected. |
| Disabled | 1 | Feedback state Disabled has been selected. |
| Enabled | 2 | Feedback state Enabled has been selected. |
