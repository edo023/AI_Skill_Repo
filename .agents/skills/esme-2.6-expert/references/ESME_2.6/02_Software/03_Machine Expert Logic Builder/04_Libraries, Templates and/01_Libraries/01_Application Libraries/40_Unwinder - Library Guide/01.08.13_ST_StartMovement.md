# ST_StartMovement

![](images/logo.png)

ST\_StartMovement

ST\_StartMovement - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

Description

This structure contains the parameters and feedback data for the start movement of the FB\_Unwinder.

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_etMode | ET\_StartMovementMode | Selection of the start behavior until the continuous work starts. |
| stMoveToRefPosition | ST\_StartMovementToRefPosition | Parameters and feedback data for the start behavior to move a dancer to the middle position. |
| stSpliceFromStandstill | ST\_StartMovementSpliceFromStandstill | Parameters and feedback data for the start behavior to wait until the spliced foil moved the dancer. |
| stTorque | ST\_StartMovementTorque | Parameters and feedback data for the start behavior to move with torque until a feedback value is detected. |
