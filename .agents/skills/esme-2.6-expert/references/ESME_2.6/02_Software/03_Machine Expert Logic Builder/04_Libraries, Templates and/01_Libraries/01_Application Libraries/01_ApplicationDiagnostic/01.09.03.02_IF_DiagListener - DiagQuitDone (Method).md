## Task

Confirms that the diagnostic quit process is completed.

## Description

The method DiagQuitDone is used to confirm that the diagnostic quit process is completed. If the listener is registered to a diagnostic handler, the method will be called by the diagnostic handler as soon as the completion of the diagnostic quit process at the diagnostic handler is confirmed.
