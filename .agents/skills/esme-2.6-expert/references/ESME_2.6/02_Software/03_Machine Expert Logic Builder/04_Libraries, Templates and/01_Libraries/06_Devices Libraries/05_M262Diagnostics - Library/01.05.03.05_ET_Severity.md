# ET_Severity

![](images/logo.png)

ET\_Severity

Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

Description

The enumeration ET\_Severity contains the possible values that indicate the source of a diagnostic message.

Enumeration Elements

| Name | Value  (UDINT) | Severity of the diagnostic message: |
| --- | --- | --- |
| NotSet | 0 | Not set |
| Info | 1 | Information |
| Warning | 2 | Advisory |
| Error | 3 | Error |

Used By
