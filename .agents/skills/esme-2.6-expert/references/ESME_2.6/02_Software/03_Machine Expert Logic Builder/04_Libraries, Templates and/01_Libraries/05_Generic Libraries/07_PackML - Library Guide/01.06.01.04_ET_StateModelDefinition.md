# ET\_StateModelDefinition

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.4.2.0 |

## Description

This enumeration is used to define the state model of a unit mode. The states 1...17 of the PackML status model are represented by bits 1...17. State 0 is an undefined state, therefore bit 0 is not used.

## Enumeration Elements

| Name | Data type | Value | Description |
| --- | --- | --- | --- |
| AbortedAndStopped | DINT | 000000001000000100 bin | The bits for the states Aborted and Stopped are TRUE. Can be used to define the states where a mode change is allowed. |
| NoStateDisabled | DINT | 0 bin | None of the bits are TRUE, all states are enabled. Can be used to define the states which are disabled. |
| HoldDisabled | DINT | 1110000000000 bin | The bits for the states Holding, Held and UnHolding are TRUE. Can be used to define the states which are disabled. |
| SuspendDisabled | DINT | 110000000100000 bin | The bits for the states Suspending, Suspended and UnSuspending are TRUE. Can be used to define the states which are disabled. |
| CompleteDisabled | DINT | 110000000000000000 bin | The bits for the states Completing and Complete are TRUE. Can be used to define the states which are disabled. |
| Clearing | DINT | 2 hex | Bit 1 represents the state Clearing. |
| Stopped | DINT | 4 hex | Bit 2 represents the state Stopped. |
| Starting | DINT | 8 hex | Bit 3 represents the state Starting. |
| Idle | DINT | 10 hex | Bit 4represents the state Idle. |
| Suspended | DINT | 20 hex | Bit 5 represents the state Suspended. |
| Execute | DINT | 40 hex | Bit 6 represents the state Execute. |
| Stopping | DINT | 50 hex | Bit 7 represents the state Stopping. |
| Aborting | DINT | 100 hex | Bit 8 represents the state Aborting. |
| Aborted | DINT | 200 hex | Bit 9 represents the state Aborted. |
| Holding | DINT | 400 hex | Bit 10 represents the state Holding. |
| Held | DINT | 800 hex | Bit 11 represents the state Held. |
| UnHolding | DINT | 1000 hex | Bit 12 represents the state UnHolding. |
| Suspending | DINT | 2000 hex | Bit 13 represents the state Suspending. |
| Unsuspending | DINT | 4000 hex | Bit 14 represents the state Unsuspending. |
| Resetting | DINT | 8000 hex | Bit 15 represents the state Resetting. |
| Completing | DINT | 10000 hex | Bit 16 represents the state Completing. |
| Complete | DINT | 20000 hex | Bit 17 represents the state Complete. |
