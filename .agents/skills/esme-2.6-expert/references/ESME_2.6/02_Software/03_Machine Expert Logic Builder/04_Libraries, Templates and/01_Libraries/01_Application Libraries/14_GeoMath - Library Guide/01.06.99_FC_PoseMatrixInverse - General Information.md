## Description

Given a 4D matrix representing a Cartesian pose, the function evaluates its inverse transformation.

The input matrix must be a homogeneous transformation matrix representing a Cartesian pose. This function will not return a proper result if called on a generic 4D matrix.

When called on a valid matrix, the algorithm implemented by this function will always return the same result of the generic FC\_Matrix4DInverse but with a faster execution.

NOTE: There is no internal verification about i\_stPoseMatrix being a valid pose matrix.
