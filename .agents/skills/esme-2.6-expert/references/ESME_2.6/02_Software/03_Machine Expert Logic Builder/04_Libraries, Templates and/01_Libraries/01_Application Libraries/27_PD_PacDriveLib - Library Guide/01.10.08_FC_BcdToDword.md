Task

Converting BCD to binary code for DWORD.

Description

The function converts values in the decimal range 0..99999999 from BCD format into binary format.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_dwInput | DWORD | Eight-digit number in BCD format |
