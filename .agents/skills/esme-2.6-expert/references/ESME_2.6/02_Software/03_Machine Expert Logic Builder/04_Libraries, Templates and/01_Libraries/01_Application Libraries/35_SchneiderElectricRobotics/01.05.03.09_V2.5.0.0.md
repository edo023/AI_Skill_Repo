## New Functions

* Drive parameter UserDrivePeakCurren

  + Lexium P Robot (SH and ILM motors)

    The drive parameter UserDrivePeakCurrent can be set to a value greater than 5500 mA.

    The maximum value is 7500 mA.
  + Lexium T Robot (MH motors)

    The drive parameter UserDrivePeakCurrent can be set to a value greater than 8185 mA.

    The maximum value is 13000 mA.
  + Lexium T Robot (MH motors HD)

    The drive parameter UserDrivePeakCurrent can be set to a value greater than 8325 mA.

    The maximum value is 13000 mA.
  + Lexium T Robot (ILM motors)

    The drive parameter UserDrivePeakCurrent can be set to a value greater than 5800 mA.

    The maximum value is 7000 mA.

  If a value greater than the previous maximum value is configured, a current limit is calculated dynamically by using the drive parameter UserCurrentLimit.
