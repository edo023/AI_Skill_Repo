Task

Gets the feedback from last usage of [InitialiseFB](Function_Blocks_R_to_Z-15.htm#XREF_D_SE_0087356_1) and [CalculateRequiredMotion](Function_Blocks_R_to_Z-9.htm#XREF_D_SE_0087350_1).

Description

This method returns the feedback from last usage of [InitialiseFB](Function_Blocks_R_to_Z-15.htm#XREF_D_SE_0087356_1) and [CalculateRequiredMotion](Function_Blocks_R_to_Z-9.htm#XREF_D_SE_0087350_1).
