# Overview of the Application

![](images/logo.png)

Overview of the Application

Overview

The applications for both controllers are similar. Each application implements the following functions:

| Function | Communicates with |
| --- | --- |
| [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) [client](../glossary/glossary.htm#XREF_D_SE_0055638_5) | Communicates with the [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) server of the other controller. |
| TCP [server](../glossary/glossary.htm#XREF_D_SE_0055638_18) | Communicates with the [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) client of the other controller. |
| [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) [peer](../glossary/glossary.htm#XREF_D_SE_0055638_15) | Communicates with the [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) peer of the other controller. |

To control and to monitor the applications, each application implements a EcoStruxure Machine Expert visualization. The visualizations are available as web visualizations. They can be accessed with a compatible web browser from a PC which is connected to the same [Ethernet](../glossary/glossary.htm#XREF_D_SE_0055638_7) network as the controllers.
