# Initialize a P-Series Robot Collision Handler

## Overview

The following variables are used in the example:

  

```
etRobotType : SERP.ET_RobotPSeries;
fbRobotPSeries : SER.FB_RobotPSeries;
fbCollisionHandlerPSeries : SER.FB_CollisionHandlerPSeries;
stOptionalParameters : SER.ST_CollisionHandlerPSeriesOptionalParameter;
//diagnostics
etDiag : GD.ET_Diag;
etDiagExtSERP : SERP.ET_DiagExt;
etResultSER : SER.ET_Result;
sResultMsg : STRING(80);
```

## How to Initialize a P-Series Robot Collision Handler

| Step | Action |
| --- | --- |
| 1 | Initialize an object of type FB\_RobotPSeries:   ```       fbRobotPSeries.InitializeRobot(              i_etRobotType := etRobotType              i_ifDriveA := DRV_RobotA,              i_ifDriveB := DRV_RobotB,              i_ifDriveC := DRV_RobotC,              q_etDiag => etDiag,              q_etDiagExt => etDiagExtSERP,              q_sMsg => sResultMsg,              q_xRotationalAxisRequired =>        ); ```   In this case, DRV\_RobotA, DRV\_RobotB and DRV\_RobotC are the drives linked to the joints of the robot. |
| Optional:  Before calling SetParametersFromRobotPSeries, you can set the optional parameters:   ```        stOptionalParameters.lrCollisionMargin := 20.0; //[mm]       fbCollisionHandlerPSeries.SetOptionalParameters(              i_stData := stOptionalParameters,              q_xError => xError,              q_etResult => etResultSER,              q_sResultMsg => sResultMsg       ); ``` |
| 2 | After a successful initialization, you can initialize the collision handler as follows:   ```       fbCollisionHandlerPSeries.SetParametersFromRobotPSeries(              i_ifRobotPSeries:= fbRobotPSeries,              q_xError => xError,              q_etResult => etResultSER,              	q_sResultMsg => sResultMsg       ); ``` |
