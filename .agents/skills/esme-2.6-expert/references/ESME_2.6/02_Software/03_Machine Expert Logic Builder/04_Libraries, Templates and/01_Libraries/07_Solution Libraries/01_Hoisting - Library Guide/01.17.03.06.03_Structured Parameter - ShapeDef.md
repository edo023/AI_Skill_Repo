# Structured Parameter - ShapeDef

![](images/logo.png)

Structured Parameter - ShapeDef

Structure: ShapeDef

Structure of parameters related to the definition of polygonal restricted areas

| Parameter | Data type | Description |
| --- | --- | --- |
| xShapeActCmd | BOOL | Command to activate the shape  TRUE: Activate the shape  FALSE: Deactivate the shape |
| xShapeActStat | BOOL | Activation status of the shape  TRUE: Shape is active  FALSE: Shape is inactive |
| arShapeVertices | ARRAY[0..c\_iNoOfVert,0..1] OF REAL | Array of vertices |
| iVertexCount | INT | Number of defined vertices in the shape  Range: 1...10 |
| rLimitTop | REAL | Top limit of restricted area, measured from the trolley level (0 = no top limit)  Range: 0.0...3.438  Unit: m |
| rLimitBottom | REAL | Bottom limit of restricted area, measured from the trolley level (0 = no bottom limit)  Range: rLimitBottom...3.438 |

xShapeActCmd

Command to activate (TRUE) and deactivate (FALSE) the restricted area. The areas can be activated and deactivated by your application.

xShapeActStat

Activation status of the restricted area. This parameter is written by the function block.

arShapeVertices

Two-dimensional array of vertices defining the restricted area. The array is filled by the function block during the teaching process. The values are stored in structures of variables outside of the function block and may be manipulated by your application if it is necessary to change position of the vertices. They can also be saved and exported by the user program.

iVertexCount

Number of vertices defined for the shape. This parameter is written by the function block.

rLimitTop

Top limit of the restricted area. The distance between the topmost position of the hook and the top limit of the restricted area. If the value is 0, the restricted area is valid from the topmost hook position.

rLimitBottom

Bottom limit of the restricted area. The distance between the topmost position of the hook and the bottom limit of the restricted area. If the value is 0, the bottom limit is not defined and the height of the restricted area is not limited from the bottom.
