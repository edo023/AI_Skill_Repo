# ST\_PropertiesDisconnect

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_PropertiesDisconnect specifies the properties inside the DISCONNECT message sent by the MQTT client.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sReasonString | STRING[255] | Specifies a human readable string representing the reason for disconnecting from the MQTT broker. |
| astUserProperties | ARRAY[1..GPL.Gc\_uiMaxNumberOfUserProperties] OF ST\_UserProperty | Specifies the user properties to send to the MQTT broker.  NOTE: A user property with an empty name represents the end of the list. |
