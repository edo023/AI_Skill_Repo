Task

Display a single diagnostic message.

Functional Description

This visualization is used to display a single diagnostic message. The visualization must be instantiated within a frame visualization in the application.

Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_xDisplayHex | BOOL | If TRUE, the identifier of the diagnostic message is displayed in hexadecimal format. |
| iq\_stDiagMessage | [ST\_VisuDiagMessage](../Structures/Structures-8.htm#XREF_D_SE_0097712_1) | Used to assign the message to be displayed. |
