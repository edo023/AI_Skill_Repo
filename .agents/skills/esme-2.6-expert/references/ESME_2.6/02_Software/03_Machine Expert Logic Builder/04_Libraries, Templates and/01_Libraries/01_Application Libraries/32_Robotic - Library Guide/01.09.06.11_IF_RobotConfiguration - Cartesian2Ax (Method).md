## Task

Configuring a biaxial Cartesian robot.

## Description

With the method Cartesian2Ax(...), the robot can be configured as a Cartesian robot with two degrees of freedom.

The axes A and B move along the coordinate axes of the Cartesian coordinate system according to the configuration i\_etPlane.

NOTE: If a method was already called up successfully to configure a transformation (q\_etDiag = ET\_Diag.Ok AND q\_etDiagExt = ET\_DiagExt.Ok), then it is not possible to overwrite the parameterization by calling up another method to configure a transformation.
