Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

Description

Data structure for parameterization of dwell-dwell cams. The following dwell-dwell cams are available:

oPoly5

oModAccTr (modified acceleration trapezoid)

oInclSin (inclined sine)

oModSin (modified sine)

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| lrLambda | LREAL | Position of the inflexion point of the dwell-dwell cam.  Value range: 0.0...1.0 |
| lrStraight | LREAL | Straight section of the dwell-dwell cam.  Value range: 0.0...0.99 (99% straight section) |
