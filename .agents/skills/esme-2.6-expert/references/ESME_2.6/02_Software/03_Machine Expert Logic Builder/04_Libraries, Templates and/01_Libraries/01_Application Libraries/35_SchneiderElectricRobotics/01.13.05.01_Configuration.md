# Configuration

## Overview

| Step | Action |
| --- | --- |
| 1 | Use an instance of FB\_RobotPSeries to configure a Schneider Electric P-Series robot. |
| 2 | Forward this configuration to the robotic library using the methods ROB.IF\_RobotConfiguration.SchneiderElectricRobot or RM.IF\_Configuration.SchneiderElectricRobot. |
| 3 | Use the method SER.IF\_FastMove.Configure to enable and configure the FastMove functionality. |
| 4 | Set the correct payload and gripper mass using the method FB\_RobotPSeries.SetKinematicParameter.  NOTE: Adapt the value of the payload at runtime whenever the payload changes. |
| 5 | Configure a set of maximum motion parameters (velocity, acceleration and deceleration), that should not be exceeded by the robot, for example with the method ROB.IF\_RobotMotion.SetMotionParameter. |
| 6 | If necessary, configure a total acceleration on the TCP using the method ROB.IF\_RobotMotion.SetMaxAccelerationResultant. |
| 7 | When tuning is used, make sure that the tracking deviation allowed on the TCP is greater than the tracking deviation configured for FastMove, as the tuning process might exceed the given limit. |
