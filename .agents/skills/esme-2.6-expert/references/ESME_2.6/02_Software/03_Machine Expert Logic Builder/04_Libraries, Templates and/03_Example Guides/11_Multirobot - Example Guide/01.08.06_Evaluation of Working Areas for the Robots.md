# Evaluation of Working Areas for the Robots

## Overview

The demo includes the function FC\_EvaluateRobotCircularWorkspaceInTracking that is used to automatically evaluate the working area for each robot and each tracking system.

The function searches for an overlap between the maximum working area of the robot and the surface of a conveyor while fulfilling a set of geometrical constraints.

The function is used in a test scenario, when the layout of a RobotCell is still to be defined and the elements of the RobotCell can be frequently rearranged to find a good setup.

While running the project on a real machine, you must verify and adjust the values evaluated by the function to take into account possible obstacles or additional constraints within the working area of the robot.
