# ENCODER\_SUB\_FCT\_CAM\_DIRECTION

## Description

The ENCODER\_SUB\_FCT\_CAM\_DIRECTION enumeration defines the three modes of direction that are available for CAMs. For further information, refer to [CAM Mode Principle Description](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Exp_UG&topicID=CAMModePrincipleDescription_5EE78856).

## Enumeration Elements

The enumeration data type ENUM contains the following values:

| Name | Value | Description |
| --- | --- | --- |
| BOTH | 0 | The CAM is active in forward and reverse directions. |
| FORWARD | 1 | The CAM is active in forward direction. |
| REVERSE | 2 | The CAM is active in reverse direction. |
