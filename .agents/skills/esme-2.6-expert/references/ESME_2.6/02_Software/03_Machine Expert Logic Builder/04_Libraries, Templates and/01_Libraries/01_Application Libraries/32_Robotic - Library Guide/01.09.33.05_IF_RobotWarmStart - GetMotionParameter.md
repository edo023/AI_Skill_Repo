## Task

Reading the warm start motion parameters.

## Description

With the method GetMotionParameter(...), the warm start motion parameters can be read.

The parameters are:

* Maximum velocity (change of the position per time unit)
* Maximum acceleration (change of the velocity per time unit)
* Maximum deceleration (change of the velocity per time unit)
* Ramp (change of the acceleration/deceleration per time unit) with which the warm start motion of each component of the robot has to be executed.
