## Task

By calling this method, the RoboticModule is being registered to the application logger.

## Description

The RoboticModule is registered to the global application logger.

Further information can be found under [Using the ApplicationLogger](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.ApplicationLogger&topicID=D_SE_0077693) in this help.
