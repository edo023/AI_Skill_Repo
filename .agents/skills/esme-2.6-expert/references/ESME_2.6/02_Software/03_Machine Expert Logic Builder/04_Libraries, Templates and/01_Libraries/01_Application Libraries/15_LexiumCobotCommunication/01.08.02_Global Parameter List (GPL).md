# Global Parameter List (GPL)

## Overview

|  |  |
| --- | --- |
| Type: | Global parameter |
| Available as of: | V1.0.1.0 |

## Description

The Global Parameter List (GPL) contains the global parameter of the LexiumCobotCommunication  library.

## Global Constants

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_uiNumberOfBytesData | UINT | Gc\_uiDefaultNumberOfBytesData | Specifies the number of bytes of data which can be transmitted between the TCP client function blocks and the TCP server of the Lexium Cobot controller within one protocol.  Can be used to enlarge the number of bytes defined by the default value Gc\_uiDefaultNumberOfBytesData if necessary. |
