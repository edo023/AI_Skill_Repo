# FB\_VisApplicationManager - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.1.0.0 |

## Task

The instance of the function block FB\_VisApplicationManager is the communication interface to the visualization.

## Description

The function blocks FB\_VisApplicationManager and FB\_VisUnitManager are used to navigate through a framework structure in a visualization and to provide the data from the units. The instance of the FB\_VisApplicationManager is the communication interface to the visualization and also serves as register starting point for the unit visualization managers.

For more information on the use of the function block FB\_VisApplicationManager, refer to [Visualization Managers](VisualizationManagers-E0420982.html)

The function block FB\_VisApplicationManager must be adapted to the number of function blocks that are used in the visualization. If the function block FB\_VisApplicationManager is to manage seven different data function blocks, the declaration of the function block must appear as follows:  
  
`fbInstanceName : SE_AFU.FB_VisApplicationManager<7>;`
