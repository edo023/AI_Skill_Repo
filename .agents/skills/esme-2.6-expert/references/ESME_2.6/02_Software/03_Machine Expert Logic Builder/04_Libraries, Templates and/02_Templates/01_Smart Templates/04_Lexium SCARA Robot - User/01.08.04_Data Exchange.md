# Data Exchange

## Overview

If you want to exchange data with the robot, there are several ways to accomplish this.

## With Module Interface

Add a variable to Module Interface by following the following procedure:

| Step | Action |
| --- | --- |
| 1 | Open the Module Interface of your robot.    **1** STANDARD\_INTERFACE  **2** USER\_INTERFACE  **3** Textual view  **4** Tabular view  The Module Interface provides two sections: STANDARD\_INTERFACE (1) and USER\_INTERFACE (2). |
| 2 | Add variables in the Textual view (3) or Tabular view (4). |

NOTE: If you add VAR\_INPUT, VAR\_OUTPUT or VAR\_INPUT\_OUTPUT, you must update the call of the robot.

NOTE: If you use the [Code Generation Option for Non Template Robots](CodeGenerationOptionForNonTemplateR-2C6DE4C3.html), the call of the robot is updated automatically.

NOTE: If you need nonvolatile variables to save data, you must define the variables in the Module Interface.

## With Method

Right-click the Robot <Robot Name>-SCARA object to add, for example, a method.

You can use the method to set/get data of the robot.

NOTE: Variables defined in methods are volatile and are reinitialized with each call of the method.

## With Global Variables

Add a global variable to use it inside and outside of the robot.

As Data type you can use the standard data types (BOOL, LREAL, ...) or user-defined data types (for example ST\_MyStruct).
