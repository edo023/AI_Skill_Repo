# ET_MotionActiveVelType

![](images/logo.png)

ET\_MotionActiveVelType

ET\_MotionActiveVelType - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

Description

The enumeration describes all the velocity sources on the basis of which the q\_xMotionActive output of the FB\_UnwinderModule is set, if the velocity exceeds i\_lrMotionActiveVelLimit.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| RefVelocity | 1 | Reference velocity of the axis. |
| Velocity | 2 | Actual velocity of the axis. |
| MechRefVelocity | 3 | Dead time compensated reference velocity of the axis. |
| MechVelocity | 4 | Dead time compensated velocity of the axis. |
