## Task

Reading a configured robot coordinate system.

## Description

With the method GetCoordinateSystem(…), the parameters of a configured robot coordinate system can be read.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etName | [ET\_CoordinateSystem](D-SE-0075477.html#D-SE-0075477) | Name of the coordinate system from which the configured parameters have to be read. |
