# ET_ControllerMode

![](images/logo.png)

ET\_ControllerMode

ET\_ControllerMode - General Information

Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

Description

This enumeration defines the different modes of the FB\_Unwinder control movement.

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| PI | 1 | The PI controller is used. |
| ThreeStep | 2 | The ThreeStep controller is used. |
| Torque | 3 | The torque controller is used.  Note: This controller mode requires a special parametrization of the drive. See "Using the unwinder TorqueMode". |
