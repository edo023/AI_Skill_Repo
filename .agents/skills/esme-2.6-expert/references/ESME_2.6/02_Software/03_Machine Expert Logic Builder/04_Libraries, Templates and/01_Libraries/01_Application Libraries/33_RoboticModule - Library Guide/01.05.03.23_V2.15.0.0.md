## New Features

* **IF\_ConfigurationAdvanced - GetMotionProfileType()**

  Reading the configured motion profile type used for the selected robot component.
* **IF\_ConfigurationAdvanced - SetMotionProfileType()**

  Setting the motion profile type for the selected robot component.

The following property is added:

* **RM.ST\_ModuleInterface/iq\_ifConfigurationReadOnly**

  Reading the configuration data and advanced configuration data from the robot via the provided methods and properties.

The following parameters are available:

* **IF\_ConfigurationAdvanced.lrMinVelOverrideVariation**

  This parameter can be used to define a minimum variation threshold for the VelOverride.
* **IF\_ConfigurationAdvanced.lrVelOverrideZeroThreshold**

  This parameter can be used to define a minimum threshold when the VelOverride is treated as zero.
