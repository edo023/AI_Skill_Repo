Overview

|  |  |
| --- | --- |

| Available as of: | V1.1.0.0 |
| Support for: | PacDrive3 pilot template architecture |
| Versions: | Current version |

Task

Function block for initialization of an axis that is controlled by the [FB\_AxisModule](Function_Blocks-3.htm#XREF_D_SE_0077136_1) function block.

Description

 The rising edge of the i\_xResetHomeOk input will reset the xHomeOK signal of the axis specified by iq\_stAxisModuleItf. The i\_xResetHomeOk input has to be set false before it can be used again. This is a function block that will require an instance of it to be declared.

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xResetHomeOk | BOOL | Resets the xHomeOK signal of the associated axis |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stAxisModuleItf | ST\_ModuleInterface | The axis interface structure of the associated axis |
