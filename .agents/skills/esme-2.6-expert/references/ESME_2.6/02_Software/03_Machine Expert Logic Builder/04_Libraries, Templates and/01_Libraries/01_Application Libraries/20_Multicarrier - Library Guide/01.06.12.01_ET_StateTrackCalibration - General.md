# ET\_StateTrackCalibration - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.2.5.0 |

## Description

The enumeration ET\_StateTrackCalibration specifies the status of the function block [FB\_TrackCalibration](FB_TrackCalibGen-6314E4FB.html#FB_TrackCalibGen-6314E4FB) for calibrating a Lexium™ MC multi carrier track.

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| NotActive | 0 | The function block FB\_TrackCalibration is not enabled. |
| WaitForStart | 1 | Waiting for the start (i\_xStart) of the function block FB\_TrackCalibration. |
| WaitForStartCalibration | 2 | Waiting for the call of the calibration methods. |
| MoveToInitialPosition | 3 | The carrier is moving to the initial position for the calibration process. |
| MoveToNextSegment | 4 | The carrier is moving to the next segment. |
| CalculateSegmentValues | 5 | The calculation of the segment-related values is active. |
| CalculateTrackValues | 6 | The calculation of the track-related values is active. |
| TransferValuesToSegment | 7 | The transfer of the calibration values to the segment is active. |
| Disabling | 8 | The function block is being disabled. |
| ResetTrackCalibrationActive | 50 | The reset of the track calibration is active. |
| TrackCalibrationSuccessful | 100 | The track calibration was successful. |
| MeasurementSuccessful | 101 | The measurement was successful. |
| ResetTrackCalibrationSuccessful | 102 | The reset of the track calibration was successful. |
| Error | 1000 | An error has been detected. For more information, refer to the diagnostic outputs of the function block FB\_TrackCalibration. |
