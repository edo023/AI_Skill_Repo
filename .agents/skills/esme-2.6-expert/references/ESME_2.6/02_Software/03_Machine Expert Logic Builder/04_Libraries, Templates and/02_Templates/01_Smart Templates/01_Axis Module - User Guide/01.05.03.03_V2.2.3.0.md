# V2.2.3.0

## Description

* The Smart Template Axis Module contains the functionality to parametrize a Schneider Electric axis module in a graphical way and use it in a PacDrive 3 Template or in other EcoStruxure Machine Expert software architectures.
* The module is prepared and optimized to use the operation modes Homing, Manual, Endless, Positioning, MultiCam, and BrakeRelease of an axis module (AXM).
* The Smart Template Axis Module is based on the library PD\_AxisModule (AXM).
