## New Functions

* New function blocks:

  + FB\_CollisionHandlerPSeries

    The function block implements the collision handler of a T-Series robot.
  + FB\_CollisionHandlerTSeries

    The function block implements the collision handler of a P-Series robot.
* New interfaces:

  + IF\_CollisionHandlerPSeries
  + IF\_CollisionHandlerTSeries

## Enhancements

* Method CheckPositionInWorkEnvelope of IF\_RobotPSeries, IF\_RobotTSeries, IF\_RobotSSeries
* Property xRobotInWorkEnvelope of IF\_RobotPSeries, IF\_RobotSSeries, IF\_RobotTSeries and FB\_RobotPSeries, FB\_RobotSSeries, FB\_RobotTSeries

  When the property returns TRUE, the robot TCP is inside the work envelope. When the property returns FALSE, the robot TCP is outside the work envelope.
