# Output Pin Description

![](images/logo.png)

Output Pin Description

Output Pin Description

| Output | Data type | Description |
| --- | --- | --- |
| q\_xEn | BOOL | TRUE: The function block is enabled.  FALSE: The function block is disabled. |
| q\_xBridgeFwd | BOOL | Bridge travel forward command output  TRUE / FALSE |
| q\_xBridgeRev | BOOL | Bridge travel reverse command output  TRUE / FALSE |
| q\_xTrolFwd | BOOL | Trolley forward command output  TRUE / FALSE |
| q\_xTrolRev | BOOL | Trolley reverse command output  TRUE / FALSE |
| q\_xHoistFwd | BOOL | Hoist forward command output  TRUE / FALSE |
| q\_xHoistRev | BOOL | Hoist reverse command output  TRUE / FALSE |
| q\_xMoveRestr | BOOL | TRUE: Movement is blocked.  FALSE: Movement is not blocked. |
| q\_xTeachMode | BOOL | TRUE: Function block is in teach mode  FALSE: Function block is not in teach mode |
| q\_xBlockOvrrd | BOOL | TRUE: Override of movement blocking active  FALSE: Override of movement blocking not active |
| q\_xAlrm | BOOL | TRUE: Alarm detected.  FALSE: No alarm is detected. |
| q\_wAlrmId | WORD | Alarm status  Range: 0...511 |

q\_xEn

Status of the function block (enabled/disabled).

q\_xBridgeFwd, q\_xBridgeRev

Direction commands for the bridge travel.

q\_xTrolFwd, q\_xTrolRev

Direction commands for the trolley axis.

q\_xHoistFwd, q\_xHoistRev

Direction commands for the hoist axis.

q\_xMoveRestr

TRUE on this output signals that the point corresponding to the position of the trolley is inside a restricted area or within a clearance distance of a restricted area and a command to move towards the restricted area is currently active.

q\_xTeachMode

TRUE when the function block is in teach mode. The function block enters teach mode at rising edge of i\_xTeachMode input. The function block leaves teach mode at falling edge on the i\_xTeachMode input. The function block automatically leaves the teach mode after 60 minutes even if i\_xTeachMode is still TRUE.

The status of the teach mode must be indicated to the crane operator.

q\_xBlockOvrrd

TRUE when movement blocking is overridden using the i\_xBlockOvrrd. The override of movement blocking is activated at rising edge of i\_xBlockOvrrd input. The movement blocking override is deactivated at falling edge on the i\_xBlockOvrrd input. The movement blocking override automatically deactivates after 60 minutes even if the i\_xBlockOvrrd is still TRUE.

The status of the override must be indicated to the crane operator.

q\_xAlrm

The output indicates that an alarm is detected (TRUE = an alarm is detected). Detailed information on the detected alarm can be found in the description of output q\_wAlrmId.

In case of a detected alarm, the function block does not allow any movement until the cause of the alarm has been removed.

q\_wAlrmId

This output identifies the detected alarm.

| Alarm bit | Description |
| --- | --- |
| bit 0 | Nominal motor speed of bridge travel < 1 RPM  i\_stOarParam.iBridgeSpdNom < 1 and  i\_stOarParam.xBridgeAsw = FALSE |
| bit 1 | Linear speed of bridge travel ≤ 0  i\_stOarParam.rBridgeSpdLin ≤ 0.0 and  i\_stOarParam.xBridgeAsw = FALSE |
| bit 2 | Deceleration ramp time of the bridge travel motor = 0  i\_stOarParam.wBridgeDec = 0 and  i\_stOarParam.xBridgeAsw = FALSE |
| bit 3 | Nominal motor speed of trolley movement < 1 RPM  i\_stOarParam.iTrolSpdNom < 1 and  i\_stOarParam.xTrolAsw = FALSE. |
| bit 4 | Linear speed of trolley movement ≤ 0  i\_stOarParam.rTrolSpdLin ≤ 0.0 and  i\_stOarParam.xTrolAsw = FALSE |
| bit 5 | Deceleration ramp time of the trolley motor = 0  i\_stOarParam.wTrolDec = 0 and  i\_stOarParam.xTrolAsw = FALSE |
| bit 6 | Nominal motor speed of hoist movement < 1 RPM  i\_stOarParam.iHoistSpdNom < 1 |
| bit 7 | Linear speed of hoist movement ≤ 0  i\_stOarParam.rHoistSpdLin ≤ 0.0 |
| bit 8 | Deceleration ramp time of the hoist motor = 0  i\_stOarParam.wHoistDec = 0 |
| bit 9 | Modification of position of trolley axis and its motor speed have opposite signs. |
| bit 10 | Inconsistency between modification of position of the trolley axis and its motor speed detected. Verify the position measurement and actual motor speed values. |
| bit 11 | Inconsistency between modification of position of the trolley axis and its motor speed detected. Either the modification of position of the trolley axis or the actual speed of the trolley motor was zero while the other value changed. |
| bit 12 | Modification of position of bridge axis and its motor speed have opposite signs. |
| bit 13 | Inconsistency between modification of position of the bridge axis and its motor speed detected. Verify the position measurement and actual motor speed values. |
| bit 14 | Inconsistency between modification of position of the bridge axis and its motor speed detected. Either the modification of position of the bridge axis or the actual speed of the bridge motor was zero while the other value changed. |
