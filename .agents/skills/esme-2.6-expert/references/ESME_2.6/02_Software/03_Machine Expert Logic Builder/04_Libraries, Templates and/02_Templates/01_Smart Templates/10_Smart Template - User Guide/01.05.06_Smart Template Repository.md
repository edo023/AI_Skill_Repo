# Smart Template Repository

![](images/logo.png)

Smart Template Repository

Overview

The Smart Template Repository dialog manages the Smart Template modules and the plugins.

Open the Smart Template Repository dialog via Tools > Smart Template Repository.

A Smart Template Repository is a database for Smart Template modules and its referenced plugins which have been installed on the local system and are available for EcoStruxure Machine Expert projects.

The Smart Template Repository dialog provides two tabs:

oModules

oPlugins

Modules Tab

The Modules tab lists the Smart Template modules installed on your local system.

The modules can be grouped in folders by categories.

You can install, uninstall, and export Smart Template modules in this tab.

The Modules tab provides the following elements:

| Element | Description |
| --- | --- |
| Filters | |
| Full text search field | Enter a string to filter the list of the installed module descriptions. |
| Vendor list | Select a vendor from the list to display the installed module descriptions of the selected vendor. |
| Module properties | |
| Name | Name of the module or group of modules |
| Vendor | Module vendor |
| Version | Module version |
| Description | Module description |
| Buttons | |
| Install | Opens a dialog to select a single module description (\*.st-module) or a .zip file with several module descriptions.  A .zip file can include a bundle of module descriptions and plugins. The module descriptions and plugins, contained in the .zip file, are installed together. |
| Uninstall | Removes the selected module description from the Smart Template Repository. |
| Export | Opens a dialog to export one or several selected module descriptions. Several selected module descriptions can be saved as a bundle into a .zip file. |
| Details | Opens a dialog with detail information of the selected module description (at least Title, Version, Company, Repository, and Description). The Repository shows the Smart Template file location of the repository. |
| Close | Closes the dialog. |

Plugins Tab

The Plugins tab lists the plugins installed on your local system.

The plugins can be grouped in folders by categories.

You can install, uninstall, and export plugins in this tab.

The Plugins tab provides the following elements:

| Element | Description |
| --- | --- |
| Filters | |
| Full text search field | Enter a string to filter the list of the installed plugins. |
| Vendor list | Select a vendor from the list to display the installed plugins of the selected vendor. |
| Plugin properties | |
| Name | Name of the plugin or group of plugins |
| Vendor | Plugin vendor |
| Version | Plugin version |
| Description | Plugin description |
| Buttons | |
| Install | Opens a dialog to select a single plugin (\*.st-plugin.dll) or a .zip file with several plugins.  A .zip file can include a bundle of module descriptions and plugins. The module descriptions and plugins, contained in the .zip file, are installed together. |
| Uninstall | Removes the selected plugin from the Smart Template Repository. |
| Export | Opens a dialog to export one or several selected plugins. Several selected plugins can be saved as a bundle into a .zip file. |
| Details | Opens a dialog with detail information of the selected plugin (at least Title, Version, Company, Repository, and Description). The Repository shows the Smart Template file location of the repository. |
| Close | Closes the dialog. |

Project Archive

In the Project Archive editor (File > Project Archive > Save/Send Archive), the section Smart Template lists the Smart Template modules and plugins referenced in your project.
