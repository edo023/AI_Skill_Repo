# ST_IlsActValuesSlave

ST\_IlsActValuesSlave

ST\_IlsActValuesSlave - General Information

Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.3.0 |
| Inherits from: | - |
| Versions: | Current version |

Description

Data structure for the runtime values of the slave axes of an ILS axis system

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| xAccLimitation | BOOL | TRUE: an acceleration limit limits this slave axis of the master axes velocity for the moment. |
| xVelLimitation | BOOL | TRUE: a velocity limit limits this slave axes of the master axes velocity for the moment. |
| lrPosition | LREAL | Current position of the slave axis (calculated from the master axes position and the active curve). |
| lrVelocity | LREAL | Current velocity of the slave axis (calculated from the master axes position and the active curve). |
| lrAcceleration | LREAL | Current acceleration of the slave axis (calculated from the master axes position and the active curve). |
| lrJerk | LREAL | Current jerk of the slave axis (calculated from the master axes position and the active curve). |
| lrPower | LREAL | Current performance of the slave axis (calculated from the master axes position and the active curve; velocity \* acceleration without taking the moment of inertia into consideration). |
