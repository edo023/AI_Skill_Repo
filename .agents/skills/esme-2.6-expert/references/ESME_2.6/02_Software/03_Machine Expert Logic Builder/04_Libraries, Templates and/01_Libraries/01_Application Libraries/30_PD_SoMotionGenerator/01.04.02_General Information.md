General Information

Description

NOTE: Only Controller OnBoard I/O Touchprobes and drive Touchprobes are supported by the functionalities of this library. TM5 module Touchprobes are not supported.

Supported Master Types:

oAxis Types

oSincos- and Incremental Encoders connected via the Controller OnBoard Interface or the BT-4 Encoder Module

Not supported Master Types:.

oTM5 Encoder Input

oC2C Encoder Input

oEncoder Network Encoder Input

oVirtual Master Encoder

oSumMasterEncoder

Namespace of the Library

The default namespace of this library is SMG.

The POUs, data structures, enumerations and constants have to be addressed via this namespace.
