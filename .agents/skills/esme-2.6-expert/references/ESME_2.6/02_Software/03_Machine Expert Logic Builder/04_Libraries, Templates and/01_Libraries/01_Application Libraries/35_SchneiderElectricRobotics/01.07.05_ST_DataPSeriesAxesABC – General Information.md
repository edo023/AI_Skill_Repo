# ST\_DataPSeriesAxesABC – General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V2.14.0.0 |

## Description

Contains a list of parameters to read out axis related data from a Lexium P Robot.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| diGearIn | DINT | GearIn for axes A, B and C |
| diGearOut | DINT | GearOut for axes A, B and C |
| lrUserDrivePeakCurrentMin | LREAL | Minimum value for UserDrivePeakCurrent for axes A, B and C |
| lrUserDrivePeakCurrentDefault | LREAL | Default value for UserDrivePeakCurrent for axes A, B and C |
| lrUserDrivePeakCurrentMax | LREAL | Maximum value for UserDrivePeakCurrent axes A, B and C |
| xMotorDirection | BOOL | Motor direction for axes A, B and C |
| lrFeedConstant | LREAL | FeedConstant for axes A, B and C |
| diMotorTemperatureMonitoring | DINT | The way the motor temperature is monitored for axes A, B and C |
