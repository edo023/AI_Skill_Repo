## Description

This function is used to add information about a variable to be monitored to the required structure ST\_VariableToMonitor. The function creates the pointer to the variable, and determines the data type.

The return value is of type ST\_VariableToMonitor.

## Interface

| Inputs | Data Type | Description |
| --- | --- | --- |
| i\_anyVar | ANY | Variable to be monitored. |
| i\_sVariableName | WSTRING(GVL.Gc\_uiLengthVarNameUtf16) | Name of the variable. This name is used to identify the variable in the data message sent to the cloud application. Further, this is the display name of the variable in Machine Advisor cloud application. |
| i\_timTimeInterval | TIME := T#0s | Time interval for the monitoring option. This is an optional input. If the input is not assigned, the value is set to T#0s by default.  If the value is set to T#0s, the value of the variable is sampled when its value changes, otherwise the value is sampled in the given interval. |
