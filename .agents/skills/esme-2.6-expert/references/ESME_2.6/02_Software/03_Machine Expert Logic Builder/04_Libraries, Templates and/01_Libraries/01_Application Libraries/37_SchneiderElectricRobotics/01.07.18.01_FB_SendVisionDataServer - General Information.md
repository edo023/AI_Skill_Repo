## Task

Send a vision protocol with configured product data.

## Description

The function block FB\_SendVisionDataServer sends a specific vision protocol. The protocol is created by the function block. The data to be sent can be added by using the methods of the function block.

NOTE: The FB\_SendVisionDataServer can be used to send a vision protocol to the FB\_ReceiveVisionDataClient, for example to simulate a camera.
