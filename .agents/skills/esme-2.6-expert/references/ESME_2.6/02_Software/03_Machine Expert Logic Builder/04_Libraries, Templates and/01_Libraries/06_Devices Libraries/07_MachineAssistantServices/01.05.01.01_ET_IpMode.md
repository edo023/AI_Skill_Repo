# ET\_IpMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_IpMode specifies the IP address assignment of a detected device.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| UNKNOWN | 0 | The IP address assignment is undefined. |
| STATIC | 1 | The IP address of the device is set to a fixed value. |
| BOOTP | 2 | The IP address assignment is using the BOOTP protocol. |
| DHCP | 3 | The IP address assignment is using the DHCP protocol. |

## Used By

* ST\_PeerDeviceInfo
