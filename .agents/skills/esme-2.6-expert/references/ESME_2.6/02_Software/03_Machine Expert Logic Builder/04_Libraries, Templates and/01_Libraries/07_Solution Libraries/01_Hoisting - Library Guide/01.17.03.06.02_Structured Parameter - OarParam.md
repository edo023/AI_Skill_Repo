# Structured Parameter - OarParam

Structured Parameter - OarParam

Structure: OarParam

Structure of parameters related to the behavior of the crane

| Parameter | Data type | Description |
| --- | --- | --- |
| wBridgeSpdNom | WORD | Nominal speed of bridge travel motor  Range: 1...32767  Unit: RPM |
| rBridgeSpdLin | REAL | Linear speed of the bridge travel at nominal motor speed  Range: 0.1...3.438  Unit: m/s |
| wBridgeDec | WORD | Deceleration time of bridge travel motor  Range: 1...65535  Unit: 0.1 s |
| rBridgeClrnc | REAL | Bridge clearance  Range: 0.0...3.438  Unit: m |
| rBridgeRestrMarg | REAL | Distance to allow movement of the bridge travel out of the restricted area  Range: 0.0...3.438  Unit: m |

wBridgeSpdNom

Nominal speed of the bridge travel motor [RPM]. The value is used for calculation of the stopping distance when the bridge travel uses linear ramps. If the anti-sway function is active on the bridge travel, this parameter does not need to be configured.

rBridgeSpdLin

Linear speed of the bridge travel at nominal motor speed. The speed is measured in meters per second. The value is used for calculation of the stopping distance when the bridge travel uses linear ramps. If the anti-sway function is active on the bridge travel, this parameter does not need to be configured.

wBridgeDec

Deceleration ramp time of the bridge travel motor in 0.1 s resolution. The value is used for calculation of the stopping distance when the bridge travel uses linear deceleration ramps. If the anti-sway function is active on the bridge travel movement, the function block uses the value of i\_rBridgeDistToStopAsw instead.

rBridgeClrnc

Clearance of the bridge travel from a restricted area in meters. Defines how close the crane is allowed to move to the restricted area. If the value is low or zero, the crane may enter the restricted area.

The clearance value must be set according to the size of the load.

rBridgeRestrMarg

Defines a margin of the restricted area for the bridge travel in meters. If the point corresponding to the position of the center of the trolley moves into the restricted area, but still within the distance corresponding to this margin from the border, it is possible for the crane operator to move the crane out of the restricted area without usage of the i\_xBlockOvrrd input.

If the crane is deeper within the restricted area than the distance of this margin, the crane can move only after the i\_xBlockOvrrd input is set to TRUE.

xBridgeAsw

If xBridgeAsw is set to TRUE, the function accepts the value connected to i\_rBridgeDist­ToStopAsw input as the stopping distance.

wTrolSpdNom

Nominal speed of the trolley motor in RPM. The value is used for calculation of the stopping distance when the trolley uses linear ramps. If the anti-sway function is active on the trolley movement, this parameter does not need to be configured.

rTrolSpdLin

Linear speed of the trolley at nominal motor speed. The speed is measured in meters per second. The value is used for calculation of the stopping distance when the trolley movement uses linear ramps. If the anti-sway function is active on the trolley movement, this parameter does not need to be configured.

wTrolDec

Deceleration ramp time of the trolley axis in 0.1 s resolution. The value is used for calculation of the stopping distance when the trolley movement uses linear deceleration ramps. If anti-sway function is active on the trolley movement, the function block uses the value of i\_rTrolDist­ToStopAsw instead.

rTrolClrnc

Clearance of the trolley axis from a restricted area in meters. Defines how close the crane is allowed to move to the restricted area. If the value is low or zero, the crane may enter the restricted area.

The clearance value must be set according to the size of the load.

rTrolRestrMarg

Defines a margin of the restricted area for the trolley in meters. If the point corresponding to the position of the trolley moves into the restricted area, but still within the distance corresponding to this margin from the border, it is possible for the crane operator to move the crane out of the restricted area without usage of the i\_xBlockOvrrd input.

If the crane is deeper within the restricted area than the distance of this margin, the crane can move only after the i\_xBlockOvrrd input is set to TRUE.

xTrolAsw

If xTrolAsw is set to TRUE, the function block accepts the value connected to i\_rTrolDist­ToStopAsw input as the stopping distance.

wHoistSpdNom

Nominal speed of the hoist motor in RPM. The value is used for calculation of the stopping distance.

rHoistSpdLin

Linear speed of the hoist at nominal motor speed. The speed is measured in meters per second. The value is used for calculation of the stopping distance.

wHoistDec

Deceleration ramp time of the hoist axis in 0.1 s resolution. The value is used for calculation of the stopping distance.

rHoistClrnc

Clearance of the hoist axis from a restricted area in meters. Defines how close the crane is allowed to move to the restricted area. If the value is low or zero, the crane may enter the restricted area.

The clearance value must be set according to the size of the load.
