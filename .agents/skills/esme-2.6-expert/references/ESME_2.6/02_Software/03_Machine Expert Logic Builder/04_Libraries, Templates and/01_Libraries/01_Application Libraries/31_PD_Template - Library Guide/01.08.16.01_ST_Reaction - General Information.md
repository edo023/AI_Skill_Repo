# ST\_Reaction - General Information

## Overview

|  |  |
| --- | --- |

| Available as of: | V1.0.4.0 |

## Description

The structure shows a field in which reactions can be entered and forwarded to modules. The number of reactions can be set via the Gc\_udiMaxNumberOfReactions global parameter.
