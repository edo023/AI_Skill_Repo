Overview

Short Description

The EcoStruxure Machine Expert project LexiumPLCopenEtherNetIPExample.project provides an application example which shows how to implement a Lexium servo drive controlled via fieldbus [EtherNet/IP](../glossary/glossary.htm#XREF_D_SE_0024697_694). The application example is based on a programming framework supporting general machine control. The monitor and control functions for the Lexium drives are implemented using the [PLCopen](../glossary/glossary.htm#XREF_D_SE_0024697_476) [function blocks](../glossary/glossary.htm#XREF_D_SE_0024697_715) from the GMC Independent PLCopen MC

System Requirements and Limitations

| 1)   This example is based on the Modicon M251 Logic Controller, however, in principle, the concepts for the implementation of the Lexium drives on EtherNet/IP are the same for other EcoStruxure Machine Expert controllers. As a precondition, the controller must support the EtherNet/IP scanner and the required GMC Independent PLCopen MC library. | |
