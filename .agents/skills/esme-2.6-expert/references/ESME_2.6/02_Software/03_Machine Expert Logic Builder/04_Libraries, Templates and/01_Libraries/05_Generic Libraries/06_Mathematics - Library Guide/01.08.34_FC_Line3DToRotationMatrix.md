# FC\_Line3DToRotationMatrix

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

Based on the description of a line, the function defines an orientation taking into account the X axis, which points from i\_stLine stPoint1 to i\_stLine.stPoint2. The Z-axis points upwards (third component of the Z vector with a positive value) and the Y axis is oriented according to the right-hand rule. It is assumed that the rotation around the X axis is zero.

The orientation is described as a 3D rotation matrix.
