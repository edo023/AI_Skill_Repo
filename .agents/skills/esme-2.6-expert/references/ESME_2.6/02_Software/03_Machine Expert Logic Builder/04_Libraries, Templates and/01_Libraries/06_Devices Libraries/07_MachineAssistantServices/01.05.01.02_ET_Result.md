# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result contains the possible values that indicate the result of operations executed by the FB\_SendCommand function block.

Also the diagnostic codes of the functions FC\_Scan, FC\_ClearScanList and FC\_GetPeerScanData are covered by this enumeration.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| If q\_xError of the FB\_SendCommand or the FB\_ExtendedSendCommand is FALSE, one of the following status messages is shown. | | |
| Idle | 0 | The function block is ready to be executed. |
| OK | 1 | The function block has been executed successfully. |
| InProgress | 2 | The requested action is running on the controller but not finished yet. |
| One of the following status messages is shown if q\_xError of the FB\_SendCommand or the FB\_ExtendedSendCommand is TRUE or the return value of a function = FALSE. | | |
| InternalErrorDetected | 10 | An internal error is detected.  Possible reasons:   * No scan data read with the given search filter settings (verify and correct if necessary the filter settings, and execute the command again). * Internal database not cleared (contact your local Schneider Electric representative). * Scan not executed (contact your local Schneider Electric representative). |
| SendCommandNotAllowed | 11 | Another send command is executing.  Verify that only one instance of FB\_SendCommand or FB\_SendExtendedCommand is executed. |
| SearchTypeInvalid | 20 | The given search type is out of range.  Verify and correct if necessary the input i\_etSearchTypeFilter. |
| WrongSendCommandType | 21 | The given command type is out of range.  Verify and correct if necessary the input i\_stSendCommand.i\_etCommandType. |
| MacAddressInvalid | 22 | The given MAC address is invalid.  Verify and correct if necessary the syntax and length of the input i\_stSendCommand.i\_sMacAddr.  Example: 00:80:F4:0A:57:66 = 17 characters |
| IPv4AddressInvalid | 23 | The given IPv4 address is invalid.  Verify and correct if necessary the content of the input i\_stSendCommand.i\_sExtension1. |

| DhcpNetworkNameInvalid | 29 | The given DHCP network name is invalid.  Verify and correct if necessary the content of the input i\_stSendCommand.i\_sExtension1. |

NOTE: Only TRUE or FALSE are allowed. |

| InvalidInputParameter | 33 | No scan data is read with the input data provided by i\_etSearchTypeFilter and i\_sSearchTypeValue. | NodeNotFound | 34 | One of the following reasons can apply:   * A device specified by the given filter settings has not been found. * The end of the internal database of detected devices has been reached. * An internal database of detected devices is not available (for example, FC\_GetPeerScanData has been executed before the completion of FC\_Scan) . |
| UnexpectedFeedbackValue | 999 | Internal system firmware issue.  Contact your local Schneider Electric representative. |
