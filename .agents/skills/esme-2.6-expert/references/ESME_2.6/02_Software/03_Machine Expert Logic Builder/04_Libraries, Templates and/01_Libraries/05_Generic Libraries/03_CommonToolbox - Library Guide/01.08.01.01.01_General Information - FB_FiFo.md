# General Information - FB\_FiFo

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.2.9.0 |

## Task

Provides a First-In-First-Out (FiFo) stack.

## Description

The function block FB\_FiFo implements a First-In-First-Out stack with customizable stack size and object size.

## Interface

The function block provides two generic constants c\_uiRegisterSize and c\_uiObjectSize. The values for these constants must be defined during build time. They cannot be modified during runtime.

The value of uiRegisterSize specifies the size of the stack located inside the function block which contains the stored objects.

The value of c\_uiObjectSize specifies the size of the object to store in bytes.

The generic constants must be defined in the declaration of the instance of this function block.

For example:

```
VAR
       fbFifo : SE_CTBX.FB_Fifo<100, 2>;
END_VAR
```

In this example, the FiFo stack has a capacity of 100 values. The size of a single value is defined as two bytes.

The function block must not be called in the application.

## Methods

| Name | Description |
| --- | --- |
| Reset | Clears the stack and resets the value counter. |
| Retrieve | Reads and removes the earliest value stored in the stack. |
| Store | Adds a new value to the stack. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| uiCapacity | UINT | Read | Indicates the total capacity of the stack. |
| uiCount | UINT | Read | Indicates the actual number of the stored values inside the stack. |
