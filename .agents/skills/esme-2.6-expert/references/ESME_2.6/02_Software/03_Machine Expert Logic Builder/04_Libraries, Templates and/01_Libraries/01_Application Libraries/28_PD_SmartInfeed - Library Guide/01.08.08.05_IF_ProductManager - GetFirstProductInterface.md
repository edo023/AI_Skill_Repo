# IF_ProductManager - GetFirstProductInterface (Method)

![](images/logo.png)

IF\_ProductManager - GetFirstProductInterface (Method)

Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

Task

Returns the interface to the first product of the infeed distance

Description

The product that is closest to the end of the infeed distance is determined and its interface returned.
