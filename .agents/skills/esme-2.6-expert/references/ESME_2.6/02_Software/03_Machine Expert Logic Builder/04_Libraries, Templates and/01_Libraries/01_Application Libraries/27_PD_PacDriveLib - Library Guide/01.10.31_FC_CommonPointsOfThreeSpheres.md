# FC_CommonPointsOfThreeSpheres

![](images/logo.png)

FC\_CommonPointsOfThreeSpheres

FC\_CommonPointsOfThreeSpheres - General Information

Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Calculates the intersections of three spheres in the 3-dimensional space.

Description

Calculates the intersections of the spheres i\_stSphere1, i\_stSphere2 and i\_stSphere3. The sphere centers must not lie on a straight line.
