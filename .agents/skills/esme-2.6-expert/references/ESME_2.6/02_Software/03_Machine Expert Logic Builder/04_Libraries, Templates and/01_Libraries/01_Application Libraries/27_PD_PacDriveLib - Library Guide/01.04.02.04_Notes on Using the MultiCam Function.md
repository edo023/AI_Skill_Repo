# Notes on Using the "MultiCam" Function

![](images/logo.png)

Notes on Using the "MultiCam" Function

Description

This chapter contains information you must observe when using the "MultiCam" function.

NOTE: Errors in the cam definition can result in setpoint jumps. Thus, verify the parameterization (e.g. profile type, scaling factor) for all operating modes during commissioning.

|  |
| --- |
| Warning_Color.gifWARNING |
| UNINTENDED MOVEMENT OF THE SLAVE AXIS |
| Deactivate the POU that instructs the slave, or disconnect the connection from the master, if the slave axis stops independently from the master. |
| Failure to follow these instructions can result in death, serious injury, or equipment damage. |
