# FC_UnpackWord

FC\_UnpackWord

Overview

|  |  |

| Available as of: | V1.0.3.0 |
| Versions: | Current version |

Task

Break down WORD into 16 BOOL values.

Description

The function breaks down the WORD from the input into its bit components and delivers these as BOOL values.
