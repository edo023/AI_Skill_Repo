Task

Starts a new measurement

Description

The starting time of the next measurement is recorded. By calling up [FB\_RunTimeMea­surement.End](Function_Blocks_R_to_Z-19.htm#XREF_D_SE_0087361_1) after the codeblock to be examined, the measurement is evaluated.
