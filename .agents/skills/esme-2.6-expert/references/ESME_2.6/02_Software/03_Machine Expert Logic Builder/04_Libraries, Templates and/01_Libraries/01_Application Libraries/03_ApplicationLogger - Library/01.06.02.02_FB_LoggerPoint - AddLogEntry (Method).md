# FB\_LoggerPoint - AddLogEntry (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.4.0 |

## Description

This method has to be called to send a logger message to the Application Logger.

The information that is given to the logger point during the call of RegisterLoggerPoint is linked to every message that was sent using this logger point and does not have to be included inside the message.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etLogLevel | [ET\_LogLevel](D-SE-0077662.html#D-SE-0077662) | The logger level indicates the type and the priority of the message. |
| i\_etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | The diagnostic for the logger message. Default value of the input is GD.ET\_Diag.Ok. |
| i\_udiDiagExt | UDINT | The ET\_DiagExt of the logger message. |
| i\_sMessage | STRING[255] | The logger message. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | General, library-independent statement on the diagnostic.  A value unequal to GD.ET\_Diag.Ok corresponds to a diagnostic message. |
| q\_etDiagExt | [ET\_DiagExt](D-SE-0077660.html#D-SE-0077660) | POU-specific output on the diagnostic.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> diagnostic message |

| --- | --- |

| Enumeration value: | 22 |
