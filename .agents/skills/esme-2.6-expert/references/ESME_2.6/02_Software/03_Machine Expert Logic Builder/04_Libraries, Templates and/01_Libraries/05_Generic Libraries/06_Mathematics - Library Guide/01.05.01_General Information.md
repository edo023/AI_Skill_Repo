# General Information

## Library Overview

The Mathematics library offers a toolbox with various mathematical functions for robotics, motion control, and other applications.

## Characteristics of the Library

The following table indicates the characteristics of the library:

| Characteristic | Value |
| --- | --- |
| Library title | Mathematics |
| Company | Schneider Electric |
| Category | * Util * Application/Util |
| Component | Core Libraries |
| Default namespace | SE\_MATH |
| Language model attribute | [Qualified-access-only](../../../../../api/crossBook?lang=en-US&virtualBookName=SoLibref&topicID=D_SE_0081219) |
| Forward compatible library | Yes (FCL) |

NOTE: For this library, qualified-access-only is set. This means, that the POUs, data structures, enumerations, and constants have to be accessed using the namespace of the library. The default namespace of the library is SE\_MATH.
