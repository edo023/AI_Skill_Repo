# What is OPC DA?

What is OPC DA?

OPC DA ([Open Platform Communications Data Access](../glossary/glossary.htm#XREF_D_SE_0024697_834)) is a standardized interface for access to process data. It is based on the Microsoft standard COM/DCOM2 (Component Object Model / Distributed COM) that has been extended due to the requirements of data access in automation, where the interface is used to read data from or write data to the controller.

Whenever the short form OPC is used throughout this document, it is referring to [OPC DA](../glossary/glossary.htm#XREF_D_SE_0024697_834).
