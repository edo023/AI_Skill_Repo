# Adding Sercos Devices to the Devices Tree

## Adding the Sercos master

Prerequisite: A project based on the Modicon M262 Motion Controller must have been created in EcoStruxure Machine Expert Logic Builder.

| Step | Action |
| --- | --- |
| 1 | Display the **Devices tree** by clicking View > Navigators > Devices tree. |
| 2 | In the tree of the Modicon M262 Motion Controller, select the node **Ethernet\_1 (ETH1)**. |
| 3 | Click the white/green plus sign to the right of the node **Ethernet\_1 (ETH1)**.  You can also use Add device ... from the context menu of the white/green plus sign. |
| 4 | In the dialog box Add Device, select the Sercos master in the tree Fieldbuses > sercos > Master > Sercos Master. |
| 5 | Confirm with the button Add Device |

## Adding a Sercos Slave

Prerequisite: A Sercos master must have been added to the node Modicon M262 Logic/Motion Controller.

| Step | Action |
| --- | --- |
| 1 | Display the **Devices tree** by clicking View > Navigators > Devices tree |
| 2 | In the tree of the Modicon M262 Motion Controller, select the Sercos master **SercosMaster (Sercos Master)** below the Sercos port **Ethernet\_1 (ETH1)**. |
| 3 | Click the white/green plus sign to the right of the node **SercosMaster (Sercos Master)**.  You can also use Add device ... from the context menu of the node. |
| 4 | In the dialog box Add Device, select the Sercos slave to be added in the tree Fieldbuses > sercos > Slave > ••••••. |
| 5 | Confirm with the button Add Device |
| 6 | Repeat this procedure until you have added the physically connected slaves and the devices to be used as [simulated slaves](D-SE-0096550.html#D-SE-0096550). Do not exceed the number of device supported by your [Sercos master](D-SE-0096553.html#D-SE-0096553). |
