# CANopen Manager (Master) device tree

**In online mode, the device tree allows for a pinpointing of a pending diagnosis.**

* Green symbol when the device is OPERATIONAL.
* Error flag (red triangle): Hard error if, for example, the device is not running (example: CAN driver cannot be found, connection interruption, or missing module)
* Orange triangle: Device in state `PREOPERATIONAL`
* Diagnosis flag (red exclamation mark): Indicates that a diagnosis entry is currently available for exactly this device.
* Error-cleared flag (gray exclamation mark): Indicates that a previously pending error has been corrected.
