# Method: UpdateCommunicationSettings

The communication parameters can be updated only if the server is set to "Stop". The `Enable` property is used to stop and start the server.

Inputs (VAR\_INPUT)

| Name | Data Type | Initial Value | Description |
| --- | --- | --- | --- |
| `ipAdress` | `ARRAY [0..3] OF BYTE` |  | IP address of the server |
| `uiPort` | UNIT |  | Port of the server |

Outputs (VAR\_OUTPUT)

| Name | Data Type | Initial Value | Description |
| --- | --- | --- | --- |
| `UpdateCommunicationSettings` | `UDINT` |  | Leads to an error (return value != 0) if a TCP connection exists |
