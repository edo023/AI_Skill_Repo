# Diagnostics of the Sercos Master

## Diagnostics via the Sercos LED

The LED **S3** provides information on diagnostics related to Sercos.

|  | LED color | Description |
| --- | --- | --- |
|  | Off | State NRT (no Sercos communication) |
|  | Orange | Communication phase is CP0, CP1, CP2 or CP3 |
|  | Green | Communication phase is CP4 |
|  | Red | State error |

## Acknowledging Detected Errors

If an error is detected, remove the cause of the detected error. After that, use the parameter DesiredPhase of the parameter group **SercosPhaseChanger** to set the master to the communication state NRT and then back to the required communication phase.

## Diagnostics via Parameter

Use the parameter IDN S-0-0390 (Diagnostic Number) to read diagnostics information from the Sercos slaves.

## Additional Information in Loggers

The CODESYS Logger and the Message Logger provide additional messages. Refer to these logger entries for information regarding troubleshooting, commissioning, and diagnostics.
