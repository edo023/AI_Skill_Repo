# Dialog: Select item from object directory

The dialog shows a list of input and output objects that are defined in the device description of the module.

**The following options exist for adding new connections:**

* Select one of the standard connections that defined in the module description, and click **OK** to add it to the input or output configuration page.
* Select one of the standard connections and click in the blank area below the list. The displayed connection parameters are editable in the lower part of the view. You can change them and click **OK** to confirm.
* When none of the default entries are selected, the parameter fields are blank in the lower part of the dialog. You can configure an entirely new connection and click **OK** to add it to the configuration.

|  |  |
| --- | --- |

| **Name** | You can use or edit the existing name or define a new name. |
| **IDN** | **Identification number consisting of**   * **S**: standard data; **P**: product-specific data * **PSet**: Parameter set (0..7) * **FG**: Function group * **SI**: Structure instance * **SE**: Structure element |
| **Type** | Data type of the input or output. |
| **Supports CC** | : Select this option when the module uses a CC channel for communication. |
