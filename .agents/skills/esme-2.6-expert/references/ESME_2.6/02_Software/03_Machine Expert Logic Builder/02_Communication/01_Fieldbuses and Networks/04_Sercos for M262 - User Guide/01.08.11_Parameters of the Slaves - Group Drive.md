# Parameters of the Slaves - Group **Drive**

## Overview

The parameter group **Drive** provides a parameter indicating the active operating mode of the drive.

This parameter group is only available for Sercos drives.

## Parameter OperationModeStatus

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

With this parameter, you can read the active operating mode of the drive.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| Position | Active operating mode is velocity control with active position control loop in drive (Cyclic Synchronous Position). |
| Velocity | Active operating mode is Cyclic Synchronous Velocity |
| NotAvailable | The operating mode of the drive is not determinable.  This can be the case under the following conditions:   * Sercos is not in communication phase 4. * The operating mode of the drive is not Cyclic Synchronous Position or Cyclic Synchronous Velocity, but, for example, Homing. * A drive error has been detected. * The power stage of the drive is disabled. |
