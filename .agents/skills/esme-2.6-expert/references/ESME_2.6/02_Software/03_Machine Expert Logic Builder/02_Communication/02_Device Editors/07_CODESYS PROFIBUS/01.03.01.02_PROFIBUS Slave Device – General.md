# PROFIBUS Slave Device – General

The tab is used for configuring the main settings of a DP Slave for data exchange in the network.

Settings

|  |  |
| --- | --- |
| **Address** | Station address of the slave within the PROFIBUS network. Possible values: 1–126. Default: 126 |
| **Baud Rate** | Baud rate for data exchange. Possible settings: Auto (default; baud rate as predefined from the bus master), 9,600 / 19,200 / 31,250 / 45,450 / 93,750 / 18,750 / 500 / 1500 / 300 / 600 / 12000) |
