# Special settings for PROFIBUS

|  |  |
| --- | --- |
| **Scan for Devices** | New search for devices in the PROFIBUS Master network. |
| **Copy All Devices to Project** | Copies all selected slaves into the project. Existing slaves with the same address are overwritten. |

|  |  |
| --- | --- |
| **State** | **Scanned and projected slaves are assigned by station address.**   * : The detected device matches the configured device. * : No device was detected with the same station address. * : Assignment of the detected device to the configured device is possible, but the module configuration is different. |
|  | Unique assignment to the devices of a specific manufacturer is not possible with the available information (ID number for slaves, I/O configuration for modules). The **Add Device** dialog opens where you can assign the scanned devices manually to the devices available in the project. |
