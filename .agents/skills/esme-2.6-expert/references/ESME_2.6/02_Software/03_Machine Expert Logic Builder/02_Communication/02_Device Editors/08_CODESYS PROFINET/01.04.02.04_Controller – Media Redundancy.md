# Controller – Media Redundancy

Object: PROFINET Controller

This tab is used to configure the media redundancy protocol (MRP) for the PROFINET Devices addressed by this PROFINET Controller.

<group name>

|  |  |
| --- | --- |
| **Enable** | : Activates the MRP domain |
| **MRP domain** | Name of the MRP ring configuration |
| **Device** | Name of the device in the device tree or the name of the respective function block |
| **Station name** | The station name of the device |
| **MRP instance** | Index of the MRP ring (assigned automatically). Some devices support the participation in multiple MRP domains. |
| **MRP role** | **Role of the device in the MRP ring**   * **Off** * **Client** * **Manager** * **Manager\_Auto** |
| **Ring Port 1**  **Ring Port 2** | Ports of the ring that are part of the MRP ring. The neighboring port is displayed in brackets. |

For more information, see: [Media Redundancy (MRP)](_pnio_media_redundancy.html#_pnio_media_redundancy)
