# PROFIBUS Module – General

The parameters of the PROFIBUS Module are defined on this tab. The parameters are defined in the device description file, but they can be edited in this dialog.

Module Information

|  |  |
| --- | --- |
| **Configuration** | I/O configuration of the slave |
| **Input length** | Length of the input data (in bytes) as defined in the device description file (GSD file). |
| **Output length** | Length of the output data (in bytes) as defined in the device description file (GSD file). |

User Parameters

|  |  |
| --- | --- |
| **Symbolic values** | : When specified in the device description file (GSD file), symbolic names for the parameters can also be displayed |
| **Length of the user parameters (Byte)** | Added user parameters that are defined by the device description file. |
| **Defaults** | Returns the values to the default settings. |

**Tbit**: Time unit for transmitting a bit on the PROFIBUS; inverse value of the transmission rate (for example, 1 Tbit at 12 Mbaud = 1/12,000,000 bps = 83 ns).
