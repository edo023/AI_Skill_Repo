# Tab: Sercos Module – User Parameter

Object: **Sercos Module**

This tab is similar to the device editor tab of the superordinate Sercos slave. You can add even more parameters that are necessary for this special module and transferred one time to the bus system in the provided phase of the start routine. For this reason, you cannot change the command order of the user-defined parameters.
