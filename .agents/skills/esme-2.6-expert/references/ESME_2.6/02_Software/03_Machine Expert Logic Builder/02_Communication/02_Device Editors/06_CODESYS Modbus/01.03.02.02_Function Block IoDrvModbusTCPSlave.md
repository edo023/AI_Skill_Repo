# Function Block: IoDrvModbusTCPSlave

Library: `IoDrvModbusTCPSlave`

This block is generated for each Modbus TCP Server Device in the device tree.

Outputs (VAR\_OUTPUT)

| Name | Data Type | Initial Value | Description |
| --- | --- | --- | --- |
| xInternalError | `Bool` | `FALSE` | Unrecoverable internal error (for example, if `mem-aloc` or license is missing) |
| uiClientConnections | `Bool` | `FALSE` | Number of currently established client connections (TCP/IP) |
