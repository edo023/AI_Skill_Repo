# Device tree

**In online mode, the device tree allows for a pinpointing of a pending diagnosis.**

* Green symbol when the device is OPERATIONAL.
* Error flag (red triangle): If the device is not running (example: CAN driver cannot be found).
* Orange triangle: Device in state `PREOPERATIONAL`.
* Diagnosis flag (red exclamation mark): If an emergency error is active or a configuration error exists, but the slave is in Operational. More information about an error is displayed on the status page (**Status** tab).
* Error-cleared flag (gray exclamation mark): Indicates that a previously pending error has been corrected.
