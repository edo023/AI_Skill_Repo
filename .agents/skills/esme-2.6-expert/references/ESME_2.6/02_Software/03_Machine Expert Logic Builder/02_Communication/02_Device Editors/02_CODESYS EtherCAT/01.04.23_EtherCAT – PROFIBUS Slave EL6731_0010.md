# EtherCAT – PROFIBUS Slave EL6731\_0010 Terminal

CODESYS supports the EL6731\_0010 device from Beckhoff, This device establishes a gateway from EtherCAT to a PROFIBUS Slave. You insert the gateway below the EtherCAT EK1100 coupler. Then you can configure it like a typical EtherCAT Slave.

When you insert the gateway, a PROFIBUS Slave node is inserted automatically. This PROFIBUS Slave acts like a standard PROFIBUS Slave.

|  |  |
| --- | --- |
|  | (1) EtherCAT coupler  (2) Gateway EtherCAT PROFIBUS Slave  (3) PROFIBUS Slave  (4) PROFIBUS Module |

You insert the PROFIBUS modules below the PROFIBUS Slave. In the editor dialogs, you can set the typical PROFIBUS parameters and I/O channels.
