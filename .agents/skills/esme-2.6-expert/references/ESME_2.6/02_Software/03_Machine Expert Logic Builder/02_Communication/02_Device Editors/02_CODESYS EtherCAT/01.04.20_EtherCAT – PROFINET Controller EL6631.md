# EtherCAT – PROFINET Controller EL6631 Terminal

CODESYS supports the EL6631 device from Beckhoff, This device establishes a gateway from EtherCAT to a PROFINET Controller. You insert the gateway below the EtherCAT EK1100 coupler. Then you can configure it like a typical EtherCAT Slave.

When you insert the gateway, a PROFINET Controller node is inserted automatically. This PROFINET Controller acts like a standard PROFINET Controller.

|  |  |
| --- | --- |
|  | (1) EtherCAT coupler  (2) Gateway EtherCAT PROFINET Master  (3) PROFINET Master  (4) PROFINET Slave  (5) PROFINET Module |

Below the PROFINET Controller, insert the PROFINET devices and modules that you have installed to the device repository as a GSDML import. In the editor dialogs, you can set the typical PROFINET parameters and I/O channels.

TIP:

By clicking **Scan for Devices**, which is provided for the EL6631 device, you can automatically add PROFINET Devices below the PROFINET Controller in the device tree.
