# Device tree: EtherNet/IP Local Adapter

**In online mode, the device tree allows for a pinpointing of a pending diagnosis.**

* Green symbol: The device is in operation.
* Error flag (red triangle): The device is unavailable. It is possible that the network configuration is wrong.
* Orange triangle: The adapter is either not yet configured or has no active I/O connection. In the case of a configuration error, you will find more information on the log page.
* Diagnostic flag (red exclamation mark): The diagnostic flag indicates that there is currently a log entry for the adapter.
* Error-cleared flag (gray exclamation mark): The `error-cleared` flag indicates that a previously pending error has been corrected.
