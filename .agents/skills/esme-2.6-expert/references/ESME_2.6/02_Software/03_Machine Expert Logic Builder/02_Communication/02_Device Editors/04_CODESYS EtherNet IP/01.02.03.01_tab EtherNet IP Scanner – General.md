# Tab: EtherNet/IP Scanner – General

The tab in the configurator of the EtherNet/IP Scanner includes the basic settings. The network interface used by the scanner is configured in the settings of the Ethernet adapter.

Options

|  |  |
| --- | --- |
| **Auto-reestablish connections** | : The scanner always attempts to automatically re-establish an interrupted connection.  For example, if a timeout is detected for UDP I/O messages or if the TCP connection to the adapter is interrupted. If the option is enabled, then the scanner reconnects to the adapters to which it has lost the connection. |
| **Enable QoS** | : The UDP/TCP messages are sent with the priorities set in the object.  : The object is disabled in the device tree. This means that the attributes of the object can no longer be accessed. |
