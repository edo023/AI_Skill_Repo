# Runtime Configuration

For communication with the PROFINET RT protocol, the PROFINET IEC stacks use a special runtime interface (SysEthernet) that allows the transmission of RAW Ethernet frames.

This runtime component has to be activated in the configuration file of PLC.

```
[ComponentManager]
Component.1=CmpSysEthernet
```

TIP:

On the CODESYS Control RTE runtime, the components "CmpEt1000Drv" for Intel and "CmpRTL8169Mpd" for Realtek are required.

The special Ethernet device drivers supplied with the CODESYS Control RTE have to be installed in the Windows Device Manager. Operation is not possible with the standard drivers from the manufacturer.
