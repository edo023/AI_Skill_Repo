# Tab: Modbus TCP Client – General

The tab contains the basic settings for network communication in the network.

Modbus TCP

|  |  |
| --- | --- |
| **Response Timeout (ms)** | Time interval (in milliseconds) for the client to wait for the response from a server node. If the nodes do not respond within this time span, then an error is issued for the implicit server FB. The value specified for the time span is also the default value for each node. For each node, you can still set a specific value within its Modbus TCP Server configuration. |
| **Auto-reconnect** | : The client always attempts to automatically re-establish an interrupted connection. At this time, errors of the server are automatically confirmed. In the case of a TCP error, the errors of the sub-devices of the server are also automatically confirmed. |
