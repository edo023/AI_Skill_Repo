# Slave Devices Configuration on Start

## Description

To simplify device maintenance, you can send configuration data to slave devices.

At application start, you can send device configuration automatically by:

* [User parameters](D-SE-0056941.html#D-SE-0056941) when the application starts the connections.
* [Configuration assembly](D-SE-0056942.html#D-SE-0056942__D-SE-0056942.16) (for devices that can support this function)
