# Linux

The following configuration steps are necessary in Linux for a standard-compliant behavior (here as an example for Debian/Raspi) :

1. In the CODESYS project, disable the **Adjust operating system settings** option for the Ethernet adapter device.
2. If the "VLAN" Linux package is not installed on the Linux system by default, then install the package.

   ```
   sudo apt-get install vlan  # VLAN-Paket installieren
   sudo modprobe 8021q        # 8021q Kernelmodul laden
   ```

   The module can be loaded automatically by means of the command `sudo su -c 'echo "8021q" >> /etc/modules'` so you do not have to load it manually each time the system is restarted.
