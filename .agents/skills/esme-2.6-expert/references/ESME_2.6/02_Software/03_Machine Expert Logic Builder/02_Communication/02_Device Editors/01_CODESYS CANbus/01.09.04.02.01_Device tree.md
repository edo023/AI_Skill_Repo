# Device tree

**In online mode, the device tree allows for an exact pinpointing of a pending diagnosis.**

* Error flag (red triangle): Hard error, such as an incorrect/missing ECU or connection interruption.

  Note: The flag is set only in the case of an active communication watchdog.
* Diagnosis flag (red exclamation mark): Indicates that a DM1 has been received. For more information, see the status page.
* Error-cleared flag (gray exclamation mark): Indicates that a previously pending error has been corrected.
