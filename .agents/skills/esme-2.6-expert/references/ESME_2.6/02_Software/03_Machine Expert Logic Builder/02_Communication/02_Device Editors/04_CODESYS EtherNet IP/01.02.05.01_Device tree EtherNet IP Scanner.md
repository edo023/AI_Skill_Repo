# Device tree: EtherNet/IP Scanner

**In online mode, the device tree allows for a pinpointing of a pending diagnosis.**

* Green symbol: The device is in operation.
* Error flag (red triangle): The device is unavailable. It is possible that the network configuration is wrong.
* Orange triangle: The device is either not yet configured or the connection configuration is wrong. If the diagnostics flag is active, then you will find more information on the log page.
* Diagnostic flag (red exclamation mark): The diagnostic flag of the respective remote adapter indicates that there is currently a log entry for exactly this device.
* Error-cleared flag (gray exclamation mark): The `error-cleared` flag indicates that a previously pending error has been corrected.
