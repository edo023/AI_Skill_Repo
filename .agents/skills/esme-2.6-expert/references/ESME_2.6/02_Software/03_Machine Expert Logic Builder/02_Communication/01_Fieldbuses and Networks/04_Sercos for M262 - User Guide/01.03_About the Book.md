# About the Book

## Document Scope

The present User Guide provides information on Sercos and describes the process of setting up a Sercos bus with the Modicon M262 Motion Controller, adding devices, configuring the network communication, configuring the individual devices, and commissioning the system with EcoStruxure Machine Expert Logic Builder.

Consult the appropriate User Guides for information on hardware and installation of the Modicon M262 Motion Controller as well as other equipment such as Safety Logic Controllers, drives, I/O devices, and on programming the motion application and safety-related applications.
