# Programming Over Industrial Ethernet

## Overview

When the protocol manager is added, the EtherNet/IP Scanner library is automatically instantiated.

In addition, most Industrial Ethernet slave devices have a dedicated library containing function and function blocks.

Use these elements to facilitate the program writing.

EcoStruxure Machine Expert contains TVDA templates that can be used.

## Manage the Operating Modes of the Devices

The EtherNet/IP Scanner library contains these functions:

* EipControl: Start/Stop the EtherNet/IP scanner
* EipGetHealth: Read the Health Bit value

For more details, refer to [EtherNetIP Scanner Library](D-SE-0056498.html#D-SE-0056498).

For operational details, refer to [Mastering slave devices operating modes](D-SE-0056601.html#D-SE-0056601) and [Impact of the controller States on the Industrial Ethernet](D-SE-0057099.html#D-SE-0057099).

## Send Commands and Read Status from Devices

Cyclic data exchanges are used with generic devices that require deterministic data exchanges. Cyclic data exchanges are managed by the protocol manager.

To configure cyclic data exchanges, see [EtherNet/IP Cyclic Data Exchanges Configuration](D-SE-0056942.html#D-SE-0056942). To use cyclic data in your program, see [EtherNet/IP I/O Mapping](D-SE-0056943.html#D-SE-0056943).

In addition, you can send explicit messages.

For EtherNet/IP devices, with the EtherNet/IP Explicit Messaging library, you can use:

* Get\_Attribute\_All, refer to [Get\_Attribute\_All,Get All Attributes of an Object](D-SE-0057934.html#D-SE-0057934)
* Set\_Attribute\_All, refer to [Set\_Attribute\_All, Set All Attributes of an Instance or Class](D-SE-0057937.html#D-SE-0057937)
* Get\_Attribute\_Single, refer to [Get\_Attribute\_Single, Get an Attribute of an Object](D-SE-0057938.html#D-SE-0057938)
* Set\_Attribute\_Single, refer to [Set\_Attribute\_SIngle, Set an Attribute of an Object](D-SE-0057939.html#D-SE-0057939)

For EtherNet/IP devices, with the EtherNet/IP Scanner library, you can use EipDataExch for features not implemented in the [EtherNet/IP  Explicit messaging Library](D-SE-0057235.html#D-SE-0057235).

For operational details, refer to [Slave devices configuration on start](D-SE-0056604.html#D-SE-0056604) and [Data exchanges on demand](D-SE-0056602.html#D-SE-0056602).
