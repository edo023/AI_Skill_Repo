# Tab: CANopen Local Device – Object Directory

This dialog is available only in the device editor of a CANopen device of version V3.5.6.0 or higher. It shows all objects of the object directory and their parameters. In this dialog, you can add new objects and delete or edit existing objects.

|  |  |
| --- | --- |
| **Add Object** | Opens the **Add Object** dialog, showing all objects that belong to the device profile. In the dialog, you can select the objects that are to be added to the object directory. |
| **Add Subobject** | Available when a structured object is selected. The command opens the **Add Subobject** dialog for defining the sub-objects. Here you provide the **Start Subindex** and **Count** of the objects. |
| **Delete** | Deletes the selected entry from the list |
| **Clone** | Duplicates the selected object. The command opens a dialog for defining the start index and number of cloned objects. |
| **Import from EDS** | Imports the data from an existing EDS file |

The properties of the selected objects are shown on the right side of the tab. When several objects are selected, CODESYS displays the shared properties only. You can edit the values of these properties in this view.

TIP:

The values of the upper limit and lower limit are provided as information for the EDS file only. The values are not processes by the stack. The application must monitor the limits with the object directory interface.
