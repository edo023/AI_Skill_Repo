# Tab: Modbus Server Init

You use this tab to define initialization commands.

Initialization commands are executed one time when starting the bus or activating the server (setting the "Enabled" flag of the server instance).

When setting up or editing a server initialization value, the following parameters are available in the respective dialogs:

|  |  |
| --- | --- |
| **Move Up**  **Move Down** | The order of channels also determines the order of initialization. |
| **New** | Opens the **Initialization Value** dialog. The initialization commands are defined. |
