# Tab: J1939-ECU – TX Signals

This dialog shows the parameter groups that are transmitted to all other ECUs (broadcast) or to a specific ECU (P2P). In this dialog, you can activate and deactivate individual groups and modify their parameters. You can also add new groups or signals to the list.

|  |  |
| --- | --- |
| **Enable** | : The parameter group is transmitted.  : The parameter group is not transmitted for a local device. For a remote device, the PLC does not process this group. |
| **Type** | * **Broadcast**: The parameter group is transmitted to all ECUs. * **Peer-to-Peer P2P**: The parameter group is transmitted to a specific ECU. |
| **Add PG** | Opens the [Add Parameter Group](_can_dlg_j1939_add_parameter_group.html#_can_dlg_j1939_add_parameter_group) dialog. |
| **Add Signal** | Opens the [Add Signal](_can_dlg_j1939_add_signal.html#_can_dlg_j1939_add_signal) dialog. The button is enabled only if you have selected a parameter group or a signal of a parameter group. |
| **Data Layout Editor** | Opens the [Data Layout Editor](_can_dlg_j1939_data_layout_editor.html#_can_dlg_j1939_data_layout_editor) dialog.  The button is enabled only if you have selected a parameter group or a signal of a parameter group. |
