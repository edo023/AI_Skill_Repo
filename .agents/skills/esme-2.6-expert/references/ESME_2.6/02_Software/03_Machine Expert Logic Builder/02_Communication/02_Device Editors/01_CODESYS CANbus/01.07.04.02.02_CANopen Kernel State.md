# CANopen Kernel State

An interrogation of the CANopen kernel state can be used to detect a bus error, a configuration error, or a monitoring error.

The state can be interrogated by means of the property `<name of CANopen Manager in device tree>.KernelState` or the function block `GET_CANOPEN_KERNEL_STATE` (`CAA CiA 405` library).
