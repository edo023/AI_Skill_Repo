# Tab: EtherCAT Slave – Safety Diagnosis

The diagnosis data of the safety slave is displayed on the tab.

The FSoE address and the state of the FSoE connection are always given in pairs. For safety terminals (for example, EL1904), only one pair of values is displayed. For an FSoE master (for example, EL6900), a list for all FSoE connections to the FSoE slaves is displayed. This allows for a fast and easy check of the FSoE communication.

Requirement: You have to be logged in to the controller and the slave has to support FSoE (safety).

|  |  |
| --- | --- |
| **Update Data** | The diagnosis data is updated. |
| **Auto update** | The diagnosis data is updated automatically. |
| **Name** | FSoE address or FSoE connection |
| **Value** | Value of the FSoE address or diagnosis data of the connection |
