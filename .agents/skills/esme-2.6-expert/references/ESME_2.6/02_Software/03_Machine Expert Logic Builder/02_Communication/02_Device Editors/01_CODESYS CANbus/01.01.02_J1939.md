# J1939

The network protocol SAE 1939 also uses CAN as a physical transmission layer. The protocol is used to transfer controller information and diagnostic data in commercial vehicles.

You can download the CODESYS J1939 Manager SL in the [CODESYS Store](https://store.codesys.com/en/codesys-j1939-manager-sl.html). There you will also find the data sheet for the product.
