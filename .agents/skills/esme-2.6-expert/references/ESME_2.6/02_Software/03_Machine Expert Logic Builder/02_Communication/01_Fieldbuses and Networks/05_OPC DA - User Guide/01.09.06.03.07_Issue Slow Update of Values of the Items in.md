# Issue: Slow Update of Values of the Items in the Client

![](images/logo.png)

Issue: Slow Update of Values of the Items in the Client

Issue

The update of values of the items (variables) in the client is slow.

Cause 1

The transfer rate is too slow or the update intervals are too high.

The value for the Update rate (ms) [parameter](../OPC_Server_Configuration/OPC_Server_Configuration-2.htm#XREF_D_SE_0083829_11) of the OPC server is too high.

Solution 1

Adapt the value for the Update rate (ms) [parameter](../OPC_Server_Configuration/OPC_Server_Configuration-2.htm#XREF_D_SE_0083829_11) of the OPC server.

Cause 2

The value for the Update rate (ms) [parameter](../OPC_Server_Configuration/OPC_Server_Configuration-2.htm#XREF_D_SE_0083829_11) of the OPC client is too high.

Solution 2

Verify the setting of the update rate in your OPC client.
