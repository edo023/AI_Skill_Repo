# XML File Listing the Variables Accessible for the OPC Client

XML File Listing the Variables Accessible for the OPC Client

In the directory where you have stored your project, an XML file is created which describes the list of the variables which are accessible for the OPC client.

The XML file is used in SIMULATION3 mode. Also refer to the Project name [parameter](../OPC_Server_Configuration/OPC_Server_Configuration-2.htm#XREF_D_SE_0083829_12) for controller configuration. When the GATEWAY3 or ARTI3 mode is selected, the XML file is not used and cannot be entered as Project name in the OPC configuration tool. In these modes, the OPC client retrieves the variables shared by the symbol configuration via the OPC server from the controller.
