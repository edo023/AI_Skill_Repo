# Tab: EtherNet/IP Scanner – Network Overview

The tab gives you an overview of the connected remote adapters. You can view and change the essential settings of the current configurations. This makes it easier to monitor the adapters.

**Adapter**

|  |  |
| --- | --- |
| List of remote adapters connected to the scanner | |
| **Name** | Name of the adapter  The adapter is configured below an Ethernet/IP Scanner in the **Devices** view.  Example: `_1734_AENT_EtherNetIP_Adapter` |
| **IP Address** | IP address of the adapter  The address can be changed. Double-click to change it. |

**Connections**

|  |  |
| --- | --- |
| List of connections of the remote adapter selected (on the left in the **Adapter** table)  TIP: **"Bulk change" feature for the RPI**  If you have selected multiple connections, then you can change the RPIs for all selected connections at the same time by holding down the Shift key and clicking (multiselection). | |
| **Add** | Adds another connection An adapter must be selected for this.  For more information, see the following: [Dialog: New Connection, Add Connection](_enic_dlg_target_new_connection.html#_enic_dlg_target_new_connection) |
| **Delete** | Deletes the selected connection from the configuration |
| **Edit** | Opens the **Edit Connection** dialog with the configuration of the selected connection.  For more information, see the following: [Dialog: New Connection, Add Connection](_enic_dlg_target_new_connection.html#_enic_dlg_target_new_connection) |

|  |  |
| --- | --- |
| **Connection Name** | Example: `Assembly Exclusive Owner` |
| **RPI (ms)** | Example: `20` |
| **Connection Path** | Example: `20 04024 66 2C 64 2C 65` |
| **Connection Type** | `POINT2POINT` |
| **Adapter** | Name of the adapter  Example: `_1734_AENT_EtherNetIP_Adapter` |
