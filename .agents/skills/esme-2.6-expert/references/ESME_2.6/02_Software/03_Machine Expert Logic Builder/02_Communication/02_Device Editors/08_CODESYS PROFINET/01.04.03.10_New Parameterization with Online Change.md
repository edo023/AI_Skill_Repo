# New Parameterization with Online Change

PROFINET device settings can be changed and updated with an online change. For PROFINET devices which support the `Re-parameterization in run` feature, these parameters are applied in runtime mode without any interruption. Devices which do not support this feature will disconnect and reconnect with the new settings.

Only device settings can be changed whereas in the case of communication parameters (IP address, send interval) or changes to the structure of the device tree (for example, by adding a device) no online change is possible.
