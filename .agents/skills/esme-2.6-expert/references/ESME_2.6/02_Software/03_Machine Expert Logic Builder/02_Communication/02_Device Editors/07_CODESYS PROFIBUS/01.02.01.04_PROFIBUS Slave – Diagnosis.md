# PROFIBUS Slave – Diagnosis

The diagnosis data of the slave is displayed on the tab. This information is useful for error checking.

|  |  |
| --- | --- |
| **Station status** | DP\_StationStatus 1-3 according to PROFIBUS specification |
| **Master address** | Address of the station address of the PROFIBUS Master (if connected); otherwise 0xFF |
| **ID number** | ID number of the slave (as defined in associated GSD file) |
| **Extended diagnosis** | Extended diagnosis data according to PROFIBUS specification (IEC 61158-6). The channel-specific diagnosis is itemized using the GSD file and displayed as text. |
