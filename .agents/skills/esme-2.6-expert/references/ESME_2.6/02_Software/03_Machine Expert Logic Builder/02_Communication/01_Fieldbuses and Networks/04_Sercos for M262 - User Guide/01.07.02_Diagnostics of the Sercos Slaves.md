# Diagnostics of the Sercos Slaves

## Diagnostics via the Sercos LED

The LED **S3** provides information on diagnostics related to Sercos.

|  | LED color | Description |
| --- | --- | --- |
|  | Off | State NRT (no Sercos communication) |
|  | Steady orange | Communication phase is CP0 |
|  | One brief green flash followed by steady orange | Communication phase is CP1 |
|  | Two brief green flashes followed by steady orange | Communication phase is CP2 |
|  | Three brief green flashes followed by steady orange | Communication phase is CP3 |
|  | Steady green | Communication phase is CP4 |
|  | Orange and green flashing | Device is in hot-plug phase 0 |
|  | One brief orange flash followed by steady green | Device is in hot-plug phase 1 |
|  | Two brief orange flashes followed by steady green | Device is in hot-plug phase 2 |
|  | Flashing green | State of the real-time communication has changed from fast-forward to loopback |
|  | Red and orange flashing | Application error has been detected |
|  | Green and red flashing | MST transmission error has been detected (MST transmission ≥S-0-1003/2) |
|  | Steady red | Communication error has been detected |
|  | Flashing orange | State Identification |
|  | Flashing red | Watchdog error has been detected |

## Diagnostics via Parameter

Use the parameter IDN S-0-0390 (Diagnostic Number) to read diagnostics information from the Sercos slaves.

EIO0000003883.03
