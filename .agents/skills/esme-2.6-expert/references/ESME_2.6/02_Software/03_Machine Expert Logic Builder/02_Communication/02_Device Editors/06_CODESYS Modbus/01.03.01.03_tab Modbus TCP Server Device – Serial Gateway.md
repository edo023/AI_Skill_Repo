# Tab: Modbus TCP Server Device – Serial Gateway

TIP:

This functionality is not included in the standard scope of the license for the Modbus TCP Server. The Modbus Serial Client also has to be licensed.

The tab contains the settings for extending the device to a serial Modbus TCP/RTU Gateway. The Modbus TCP Server then forwards the requests of a TCP Client to Modbus RTU Servers which are connected to the local COM port of the PLC.

|  |  |
| --- | --- |
| **Serial Gateway Active** | : The TCP/RTU gateway functionality in the device is enabled. |
| **COM port** | Number of the COM port used |
| **Baud rate** | Transmission rate (in baud) |
