# Overview

Overview

You can run the OPC clients and the OPC servers on different computers. In order to establish a remote connection between the OPC client and the OPC server, call the tool MMC comexp.msc /32 via the command line interface, and enter the OPC server (CoDeSysOPCDA).

You will then be able to establish a remote connection to the OPC server.

NOTE: The requirement for such a connection is that an OPC server is also running on the local computer.
