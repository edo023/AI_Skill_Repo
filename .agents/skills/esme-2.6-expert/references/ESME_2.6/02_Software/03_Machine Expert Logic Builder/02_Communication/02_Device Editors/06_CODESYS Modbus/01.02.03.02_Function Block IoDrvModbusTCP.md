# Function Block: IoDrvModbusTCP

Library: `IoDrvModbusTCP`

This block is generated for each Modbus Client TCP in the device tree.

Inputs (VAR\_INPUT)

| Name | Data Type | Initial Value | Description |
| --- | --- | --- | --- |
| `xStop` | `Bool` | `FALSE` | Stops all TCP/IP communication |

Outputs (VAR\_OUTPUT)

| Name | Data Type | Initial Value | Description |
| --- | --- | --- | --- |
| `xSlaveError` | `Bool` | `FALSE` | Indicates that there is a server FB with an error. |
| **uiConnectedSlaves** | `UINT` | `0` | Number of servers connected by TCP/IP |
