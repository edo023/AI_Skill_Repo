# Device Tool Integration

Special configuration tools which are supplied by the device manufacturers are required for the parameterization of some PROFIsafe devices. These "device tools" can be called directly from CODESYS, as long as they comply with the standardized TCI interface (TCI = Tool Command Interface).

When the suitable device tool for a PROFIsafe device has been installed on the CODESYS developer PC, a **Device Tool Commands** category is displayed in the context menu of the device (right-click in the device tree).

The **Device Tools Commands** category contains commands which are supported for the respective device, typically for calculating the PROFIsafe "i-Parameter CRC". When the submenu item is selected, the device tool from the manufacturer which is registered on the system is started directly from CODESYS.
