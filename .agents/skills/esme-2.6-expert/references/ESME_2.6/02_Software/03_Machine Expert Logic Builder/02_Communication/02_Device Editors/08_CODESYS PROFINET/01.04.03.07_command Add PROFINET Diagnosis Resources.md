# Command: Add PROFINET Diagnosis Resources

**Function**: The command generates a text list with the diagnostic messages from the GSDML files.

**Call**: **Project** menu → **Add Object**; context menu in the **Devices** view

**Requirement**: the **Application** object is selected.

The manufacturers of the PROFINET devices define diagnostic messages and additional help texts for the numeric error codes (`ChannelError`, `ExtChannelError`) sent by the PROFINET device in the GSDML files. The command can be used to make these user-friendly help texts available in the application (for example, in a CODESYS visualization).

The diagnostic messages and the additional help texts from the GSDML file are stored in the generated text list. Various interfaces are available in the application which provide access to the PROFINET device diagnostics and the numeric error codes.

The `PVU.ProfinetDiagnosisResources` function block is used to assign the numeric error codes to the texts in the generated text list.

For more information, see: [Diagnosis in the Application](_pnio_f_master_diagnosis_application.html#_pnio_f_master_diagnosis_application)
