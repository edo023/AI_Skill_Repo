A PROFIBUS network consists of one or more masters and multiple slaves with modules. In this network, the CODESYS runtime system can be employed as a master or slave device.

Below a master (1), you can insert one or more slaves (2) with modules. Below the slave device (3), you can insert I/O modules (4).

PROFIBUS Devices are imported into CODESYS by means of device description files (GSD). These files are supplied with the hardware.

In CODESYS, it is possible to insert a USB interface in the device tree for PROFIBUS. This interface (PROFIusb by the company "Softing"), which is connected to the USB port of the computer, is used in the network as a PROFIBUS Master. It provides the full functionality of a PROFIBUS Master.
