# Tab: IO-Link – Parameters

Object: IO-Link Master

The tab can be used to read the parameters from the IO-Link device, to modify them and then to write them back to the device. The range of functions depends on the device description. The commands of the buttons are also available in the context menu of the selected row in the parameter table.

|  |  |
| --- | --- |
| **User role** | * Specialist * Maintenance * Observer   Depending on the selected user role, more or fewer details of the parameters are displayed in the table. |
| **Identification menu** | Information about the identification of the device |
| **Parameter menu** | Displays all parameters of the device |
| **Diagnosis menu** | Displays error messages of a module (for example, voltage error) |
| **Observation menu** | Displays observation messages (for example, "sensor dirty") |
| **Set to Defaults** | Resets the online values of the parameters to the default values as defined in the device description |
| **Read this Page** | Reads the currently displayed values of the parameters from the device and refreshes them in the table |
| **Write this Page** | Writes the currently displayed values of the parameters to the IO-Link device |
| **Save online values** | Applies the currently displayed **Online value** to the **Value** column |
| **Restore values** | Applies the values from the **Value** column to the **Online value** column |

The table lists different parameters (name) and parameter details depending on the set user role and selected menu.

|  |  |
| --- | --- |
| **Name** | Parameter name |
| **Value** | The currently valid parameter value is displayed in the **Value** column. |
| **Online value** | The **Online value** column is displayed only if the device supports a parameter download and if the parameters displayed in the configuration editor are also available on the device.  You can change the parameter values in this column and with **Write this page** write to the device without modifying the project and without having to download the application again. With **Save online values** you can use the current online value as **value** to save. |
| **r/w** | Permissions on the parameter values (`ro`: read only; `rw`: read and write) |
