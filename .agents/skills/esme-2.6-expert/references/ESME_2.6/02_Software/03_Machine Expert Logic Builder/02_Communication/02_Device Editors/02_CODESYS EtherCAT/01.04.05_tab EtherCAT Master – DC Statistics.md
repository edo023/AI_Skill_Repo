# Tab: EtherCAT Master – DC Statistics

This tab is used to diagnose the DC data and shows the temporal behavior of the runtime systems. The frequency of time deviations (e.g. jitter) is recorded for this purpose.

|  |  |
| --- | --- |
| **Reset values to zero** | Resets all counts to 0 |
| **Deviation [µ]** | Time ranges of the deviations.  The deviation describes the time between the actual time of the next task call and the expected time of the next task call.  Here, **<1** counts the deviations between 0 and 1 µs, **<2** counts the deviations between 1 µs and 2µs, and so on. |
| **Count (neg)** | Number of negative deviations |
| **Percent (neg)** | Percentage of negative counts in relation to the total number of cycles counted |
| **Percent (pos)** | Percentage of positive counts in relation to the total number of cycles counted |
| **Count (pos)** | Number of positive deviations |
