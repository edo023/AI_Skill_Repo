# Tab: EtherCAT Master – Safety FSoE Connections

This tab shows the FSoE connections of the EtherCAT network. The tab is visible only when the corresponding parameter is set in the device description.

The connections can be changed if the dimensions of the inputs and outputs match and an FSoE Slave is selected for a specific FSoE Master.

|  |  |
| --- | --- |
| **Clear all connections** | Clears all FSoE connections |
