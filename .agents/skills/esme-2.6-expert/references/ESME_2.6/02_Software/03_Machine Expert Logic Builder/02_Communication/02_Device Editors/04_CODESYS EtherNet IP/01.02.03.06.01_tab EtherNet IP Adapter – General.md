# Tab: EtherNet/IP Adapter – General

The tab in the device editor of the EtherNet/IP Adapter contains the basic settings for communication in the network.

Address Settings

|  |  |
| --- | --- |
| **IP address** | Address for the identification of the EtherNet/IP Adapter device. |

BOOTP

|  |  |
| --- | --- |
| Bootstrap Protocol  This option is available only for adapters under the NetX Scanner. | |
| **MAC address** | Device-specific MAC address of the slave |
| **Save IP address** | : The address of the slave is saved. However, the slave is required to support this function. This option is only available for the CIFX Scanner. |

Electronic Keying

|  |  |
| --- | --- |
| **Compatibility check** | : The adapter uses its own keying values to perform a compatibility check of the keying values from the EDS file. All keying values are sent to the device. The device then decides whether or not it is compatible with the received values.  : The adapter uses its own keying values to perform an exact check of the keying values from the EDS file. The user decides which keying information should be checked.   * Vendor ID * Device type * Product code * Major revision * Minor revision   If the check fails, then there is no I/O connection established to the device and an error message is displayed on the status page. |
| **Restore Default Values** | Only for generic devices. |
