# PROFIBUS Slave – General

The parameters of the PROFIBUS Slave are defined on this tab. The parameters are defined in the device description file, but they can be edited in this dialog.

Identification

|  |  |
| --- | --- |
| **Station Address** | Permitted range from 0 to 126.  Every new device inserted into the bus is assigned an incremented address automatically.  Note: Address 126 is the default DP Slave address. Addresses are checked for duplicates. |
| **ID number** | Unique number that was assigned to this device type from the PNO. This allows a unique reference between the DP Slave and the GSD file. |

Parameter

|  |  |
| --- | --- |
| **TSDR (Tbit):** | "Time Station Delay Responder". Earliest response time that the slave can respond to the master (min. 11 Tbit) |
| **Lock/Unlock:** | **The slave is blocked or released for other masters**   * **0 (TSDR unlock)**: min.TSDR and slave-specific parameters can be overwritten. * **1 (Will be unlocked)**: The slave is released for other masters. * **2 (Lock)**: The slave is blocked for other masters, and all parameters are accepted. * **3 (Unlock)**: The slave is released again from the masters. |

\* TBit: Time unit for transmission of a bit via PROFIBUS; reciprocal value of the transmission rate; e.g. 1 Tbit at 12MBaud = 1/12,000,000 bps=83ns

Monitoring

|  |  |
| --- | --- |
| **Watchdog control**: | : The defined monitoring time applies (watchdog, threshold monitoring, and base 10 ms). If the slave has not been accessed by the master within this time, it is reset to its initialization state. |
| **Time (ms)**: Monitoring time, relevant to the case when the "Watchdog control" option is active. | Monitoring time, relevant to the case when the **Watchdog control** option is active. |
| **Groups** | Opens the **Groups** dialog. |

User Parameters

|  |  |
| --- | --- |
| In addition to the base DP parameters, user parameters are the individual parameters of a DP Slave. When they are defined in the device description file (GSD file), they are displayed here. The parameter values can be edited by clicking the corresponding field in the **Value** column. | |
| **Symbolic values** | : When specified in the device description file (GSD file), symbolic names for the parameters can also be displayed |
| **Length of the user parameters (Byte)** | Added user parameters that are defined by the device description file. |
| **Defaults** | Returns the values to the default settings. |
