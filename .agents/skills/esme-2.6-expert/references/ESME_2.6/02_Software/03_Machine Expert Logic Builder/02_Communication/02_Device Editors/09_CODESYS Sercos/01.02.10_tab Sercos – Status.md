# Tab: Sercos – Status

Object: **Sercos Master, Sercos Slave, Sercos Module**

The device configurator tab displays status information (for example, "Running", "Stopped") and specific diagnostic messages from the respective device, including the card employed and the internal bus system.

|  |  |
| --- | --- |
| **Read S-0-0390** | For Sercos slave only.  Reads the IDN S-0-0390, and shows the diagnostic number (hexadecimal) and respective error text. |
