# Device tree

**In online mode, the device tree allows for an exact pinpointing of a pending diagnosis.**

* The error flag (red symbol) indicates a hard error, such as an incorrect/missing module or connection termination.
* The diagnosis flag (red exclamation mark) of the respective module indicates that a diagnosis entry is currently available for exactly this device.
* The error-cleared flag indicates that a previously pending error has been corrected.
