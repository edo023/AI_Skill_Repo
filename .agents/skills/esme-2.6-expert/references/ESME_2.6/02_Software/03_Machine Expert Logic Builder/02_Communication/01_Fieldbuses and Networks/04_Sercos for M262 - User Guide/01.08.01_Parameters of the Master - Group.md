# Parameters of the Master - Group **SercosPhaseChanger**

## Overview

The Sercos communication state machine uses the communication state NRT (Non-Real-Time) and five Communication Phases (CP) from CP0 to CP4 Refer to the chapter [Sercos Communication State Machine and Communication Phases](D-SE-0096491.html#D-SE-0096491) for details.

## Parameter ActualState

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

With this parameter, you can read the state of the Sercos master.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| NRT / -1 (default value) | The Sercos master is in state NRT. In the state NRT the Sercos communication is disabled. |
| Phase 0 / 0 | The Sercos master is in communication phase CP0. |
| Phase 1 / 1 | The Sercos master is in communication phase CP1. |
| Phase 2 / 2 | The Sercos master is in communication phase CP2. |
| Phase 3 / 3 | The Sercos master is in communication phase CP3. |
| Phase 4 / 4 | The Sercos master is in communication phase CP4. |
| PhaseInit / 10 | The Sercos master is in state PhaseInit. |
| PhaseError / 11 | The Sercos master is in state PhaseError. |

## Parameter DesiredPhase

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

With this parameter, you can set the Sercos master to the state NRT or to one of the communication phases. Refer to the chapter [Sercos Communication State Machine and Communication Phases](D-SE-0096491.html#D-SE-0096491) and the chapter Commissioning for additional information.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| NRT / -1 | Sets the Sercos master to the state NRT. In the state NRT, Sercos communication is disabled. |
| Phase 0 / 0 | Sets the Sercos master to communication phase CP0. |
| Phase 1 / 1 | Sets the Sercos master to communication phase CP1. |
| Phase 2 / 2 | Sets the Sercos master to communication phase CP2. |
| Phase 3 / 3 | Sets the Sercos master to communication phase CP3. |
| Phase 4 / 4 (default value) | Sets the Sercos master to communication phase CP4. |
