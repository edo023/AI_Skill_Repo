# EtherCAT – PROFIBUS Master EL6731 Terminal

CODESYS supports the EL6731 device from Beckhoff, This device establishes a gateway from EtherCAT to a PROFIBUS Master. You insert the gateway below the EtherCAT EK1100 coupler. Then you can configure it like a typical EtherCAT Slave.

When you insert the gateway, a PROFIBUS Master node (EL6731-PB Master) is inserted automatically. This PROFIBUS Master acts like a standard PROFIBUS Master.

|  |  |
| --- | --- |
|  | (1) EtherCAT coupler  (2) Gateway EtherCAT PROFIBUS Master  (3) PROFIBUS Master  (4) PROFIBUS Slave  (5) PROFIBUS Module |

Below the PROFIBUS Master, insert the PROFIBUS Slaves and Modules which you have installed to the device repository by means of GSD import. In the editor dialogs, you can set the typical PROFIBUS parameters and I/O channels.

TIP:

By clicking **Scan for Devices**, which is provided for the EL6731\_PB device, you can automatically add PROFIBUS Devices below the PROFIBUS Master in the device tree.
