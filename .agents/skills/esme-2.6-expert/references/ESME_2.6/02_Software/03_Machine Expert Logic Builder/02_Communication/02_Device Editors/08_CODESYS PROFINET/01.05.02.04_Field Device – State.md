# Field Device – State

Object: PROFINET Field Device

The tab shows the diagnosis data of the device and the subordinate modules. For the device, additional status information about the driver is shown.

|  |  |
| --- | --- |
| **Diagnosis Data** | The diagnosis entries that are currently in the diagnosis database of the device for the selected module. The entries of this database can be created or deleted from the device application by means of the `ProfinetCommon.UpdateDiagnosisEntry` function block. The entries are also shown in the remote Profinet controller and can be read via Profinet services. |
