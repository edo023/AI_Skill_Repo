# Tab: EtherCAT Master – Overview

The tab provides an overview of the states of all slaves, which are inserted below this master and have an address. Modules are not displayed.

|  |  |
| --- | --- |
| **Update Data** | The states of the slaves are refreshed. |
| **Auto update** | The states of the slaves are read cyclically. |
| **Name** | Icon and name of the device |
| **Address** | EtherCAT address (1001, 1002) |
| **Status** | **Current EtherCAT state of the slave devices**   * **INIT**: The slave is in the initialization. * **PREOP**: The slave is in the "Pre-Operational" state. * **SAFEOP**: The slave is in the "Safe-Operational" state. * **OP**: The slave is in the "Operational" state. * **BOOT**: The slave is in the "Bootstrap" state. * **ERR**: An error status is set in the slave. * **NO\_COMM**: Communication with the slave is not possible. * **LNK\_MIS**: A connection to the intended link is missing (Port = A, B, C, or D). * **LNK\_ADD**: Additional connection at the displayed link |
| **CRC** | CRC error counter for each active port |
