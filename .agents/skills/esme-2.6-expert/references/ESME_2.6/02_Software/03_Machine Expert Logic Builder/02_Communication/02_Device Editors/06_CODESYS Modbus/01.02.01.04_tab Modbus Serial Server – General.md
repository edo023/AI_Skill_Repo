# Tab: Modbus Serial Server – General

The tab includes the basic settings for communication via the serial port.

Modbus RTU/ASCII

|  |  |
| --- | --- |
| **Server address [1..247]** | Address of a serial Modbus device (value between 1 and 247) |
| **Response timeout (ms)** | Time interval for the client to wait for the response from a server node. It is configured especially for this server node and overwrites the general response timeout setting of the respective client. |
