# Controller – Status

Object: PROFINET Controller

The tab displays the current status of the controller, for example in the **Controller Status** field the phases of powering up.

The data in **Ethernet Statistic** can be used to diagnose the Ethernet configuration.

For more information, see: [Status dialog in the device configurator](602f1c933f382455bd887c15b30eaad6.html#UUID-b7126354-7f96-a498-15a5-8b6708e71ff0_section-idm43261146037984)
