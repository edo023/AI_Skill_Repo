# Issue: Message IOPCServer Interface not Found or Class not Registered is Displayed

![](images/logo.png)

Issue: Message IOPCServer Interface not Found or Class not Registered is Displayed

Issue

The OPC server displays the message IOPCServer Interface not Found or Class not Registered.

Cause 1

The OPC proxy is not registered or the OPC proxy has a not supported version.

Another OPC server has been installed with an earlier version of the proxy.

Solution 1

Reinstall the OPC server provided by Schneider Electric.
