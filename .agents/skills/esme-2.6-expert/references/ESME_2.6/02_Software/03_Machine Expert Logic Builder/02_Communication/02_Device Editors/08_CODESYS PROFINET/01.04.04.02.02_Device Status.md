# Device Status

The **Diagnosis Data** view displays the currently pending diagnosis of the PROFINET Module. The **Alarms** view displays the alarms assigned to this module which have been received by the controller since the application has started. The controller saves the alarm data in a ring buffer. When you click **Confirm**, the alarm data is deleted from the ring buffer and any pending Diag-Cleared-Flag is reset.
