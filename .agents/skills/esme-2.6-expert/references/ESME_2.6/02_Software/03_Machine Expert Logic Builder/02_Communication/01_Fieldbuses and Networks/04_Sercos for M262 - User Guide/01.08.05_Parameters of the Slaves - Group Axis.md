# Parameters of the Slaves - Group **Axis**

## Overview

The parameter group **Axis** comprises a parameter used for the reference position of the drive/axis and a parameter indicating the requested operating mode.

This parameter group is only available for Sercos drives.

## Parameter Position

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

This parameter contains the reference position of the drive/axis as calculated on the basis of the velocity values. The value is a read-only value.

## Parameter OperationMode

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

With this parameter, you can read the operating mode requested for the drive via the control word.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| Position (default value) | Requested operating mode is velocity control with active position control loop in drive (Cyclic Synchronous Position). |
| Velocity | Requested operating mode is Cyclic Synchronous Velocity. |
