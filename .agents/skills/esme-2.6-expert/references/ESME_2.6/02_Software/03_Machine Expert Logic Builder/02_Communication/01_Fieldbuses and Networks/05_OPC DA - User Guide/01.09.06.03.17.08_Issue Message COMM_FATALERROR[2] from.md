# Issue: Message COMM_FATALERROR[2] from <controller> (affects PacDrive M/ gateway V2.3) is Displayed

Issue: Message COMM\_FATALERROR[2] from <controller> (affects PacDrive M/ gateway V2.3) is Displayed

Cause 1: This error message is displayed in the OPCServer.log if the communication has been interrupted.

Solution 1: Verify the communication line (for example, by executing a ping) command.

Cause 2: The timeout for the configured in the OPC configuration tool has expired.

Solution 2: Verify the value configured for the Timeout (ms) [parameter](../OPC_Server_Configuration/OPC_Server_Configuration-2.htm#XREF_D_SE_0083829_12).
