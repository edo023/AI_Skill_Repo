# CODESYS EtherCAT

**The integration of the EtherCAT Master in the CODESYS IEC 61131-3 programming system includes the following:**

* Integrated configurator for the master and connected slaves
* Portable protocol stack in the form of a CODESYS library
* Runtime system component for direct access to the Ethernet interface
* Diagnosis functions in the programming system as well as for the runtime watchdog by the IEC 61131-3 application
* Optional support of EtherCAT Gateway modules for other fieldbus systems

For more general information about fieldbus support in CODESYS, see: Fieldbus Support

You can download the CODESYS EtherCAT Master from the [CODESYS Store](https://store.codesys.com/de/codesys-ethercat-master-sl.html). There you will also find the data sheet for the product.
