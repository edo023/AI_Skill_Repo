# Issue: Long Reconnection Time Between Visualization and Controller

![](images/logo.png)

Issue: Long Reconnection Time Between Visualization and Controller

Issue

After the connection between visualization (computer) and controller has been interrupted, reconnection takes long time. The first reconnection of the OPC server takes approximately 15 minutes, each further reconnection takes approximately 3 minutes.

Cause 1

Two network cards are available in the visualization computer running the OPC server and the gateway. Both network cards are active and connected.

Solution 1

Swap the allocation sequence of the network cards on the visualization computer by executing the command Control panel > Network and internet > Network and sharing center > Change adapter settings > Menu advanced > Advanced settings... > Register Adapter and allocations.
