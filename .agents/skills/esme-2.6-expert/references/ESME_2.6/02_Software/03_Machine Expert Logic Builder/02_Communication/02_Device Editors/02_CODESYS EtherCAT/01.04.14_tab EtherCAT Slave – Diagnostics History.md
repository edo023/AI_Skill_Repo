# Tab: EtherCAT Slave – Diagnostics History

The tab reads the diagnostics data (`0x10f3` object) of the slave and displays it in a tree structure. Moreover, the messages are issued in plain language.

Requirement: The application is in online mode. The attribute `DiagHistory` in the ESI file is set.

|  |  |
| --- | --- |
| **Update history** | The data is read again and the view is refreshed.  New messages are displayed in the **Flags** column with **N** (New). Acknowledged messages with **Q**. |
| **Auto update** | The data is updated automatically. |
| **New messages only** | Acknowledged messages are not displayed. |
| **Acknowledge messages** | All messages with the **N** flag are acknowledged. Then the flag changes to **Q**. |
