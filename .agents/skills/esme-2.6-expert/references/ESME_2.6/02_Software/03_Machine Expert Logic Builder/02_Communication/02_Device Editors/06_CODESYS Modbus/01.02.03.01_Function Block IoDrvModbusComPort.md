# Function Block: IoDrvModbusComPort

Library: `IoDrvModbus`

This block is generated for each serial Modbus Client in the device tree.

Inputs (VAR\_INPUT)

| Name | Data Type | Initial Value | Description |
| --- | --- | --- | --- |
| `xStop` | `Bool` | `FALSE` | If `TRUE`, then each new request to the servers is stopped. If `FALSE`, the process continues. |
| `xAcknowledge` | `Bool` | `FALSE` | Immediately closes and opens the COM port for a rising edge |

Outputs (VAR\_OUTPUT)

| Name | Data Type | Initial Value | Description |
| --- | --- | --- | --- |
| `uiNumberOfCommunicatingSlaves` | `int` | `0` | Displays the number of communication servers |
| `xAllSlavesOk` | `Bool` | `FALSE` | `TRUE` if all servers communicate.  `FALSE` if some servers have errors. |
