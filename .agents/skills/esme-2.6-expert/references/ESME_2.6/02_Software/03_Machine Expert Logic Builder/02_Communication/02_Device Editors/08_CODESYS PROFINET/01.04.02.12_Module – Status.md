# Module – Status

Object: PROFINET Module

The tab displays the diagnostic data and the alarms of the module.

|  |  |
| --- | --- |
| **Diagnosis Data** | The diagnostics currently located in the device. The current diagnostics data is reloaded from the status display as long as a link can be established to the device. |
| **Alarms** | All alarms of this module that have been reported by this device since the start of the application. The alarms are stored in a ring buffer on the controller. |
| **Confirm** | Deletes the alarms from the ring buffer. |
