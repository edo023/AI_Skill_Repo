# Tab: EtherNet Adapter – General

Object: Ethernet Adapter

You define the network interface on this tab.

|  |  |
| --- | --- |
| **Network interface** | : Opens the **Network Adapters** dialog. The Ethernet adapters can be selected here, which are available on the set target system. |
| **IP address**  **Subnet mask**  **Default gateway** | Settings of the selected network interface |
| **Adjust operating system settings** | You need to confirm this option again.  Note: The settings on the target system will be overwritten by the values above.  The IP settings of the operating system can be changed by the CODESYS runtime system only if this is permitted in the configuration file (`*.cfg`) of the runtime. The following entries are required for this:  `[SysSocket]`  `Adapter.0.Name="<adapter name>"` (example: `"LAN connection 4"`)  `Adapter.0.EnableSetIpAndMask=1` |

|  |  |
| --- | --- |
| **Redundant PLC Settings (PLC 2)** | When using a pair of redundant CODESYS controllers, these settings are used on the "backup PLC" (`PLC ID = 2`) for the network adapter. |
| **Network interface** | : Opens the **Network Adapters** dialog. The Ethernet adapters can be selected here, which are available on the set target system. |
| **IP address**  **Subnet mask**  **Default gateway** | Settings of the selected network interface |

IMPORTANT:

The settings for the redundant PLC are available only when you have inserted the `Redundancy Configuration` object below your application.
