# Logger

A filtered view of the [PLC logger](../../../../../../api/crossBook?lang=en-US&virtualBookName=../&topicID=) is displayed at both Controller level and Device level.

This shows only the entries that were created by the respective device, for example change of the J1939 state, DM1 messages, and monitoring errors.
