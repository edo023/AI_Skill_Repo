# __CommState Item

\_\_CommState Item

\_\_CommState items can be analyzed by different OPC clients and can have the following states:

oSTATE\_TERMINATE = -1

oSTATE\_PLC\_NOT\_CONNECTED = 0

oSTATE\_PLC\_CONNECTED = 1

oSTATE\_NO\_SYMBOLS = 2

oSTATE\_SYMBOLS\_LOADED = 3

oSTATE\_RUNNING = 4

oSTATE\_DISCONNECT = 5

oSTATE\_NO\_CONFIGURATION = 6
