# PROFINET and Older Runtime Versions

The PROFINET drivers require at least one runtime V3.5.7.0.

For the Windows RTE runtime, at least V3.5.10.0 is recommended (problems regarding reserved UDP ports from the PROFINET range).

The requirement for a certification according to the current standard (V2.35 or 2.4) is a RTE version >= SP16.1 or Linux > SP15.
