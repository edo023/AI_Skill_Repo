# Interfaces

Interfaces

The following interfaces are available for OPC communication:

| Interface designation | Description |
| --- | --- |
| ARTI | The CoDeSys V2.3 ARTI interface is used to establish connections via SymARTI. It is not supported by PacDrive M controllers. |
| GATEWAY | The GATEWAY interface is used to configure PacDrive M controllers – CoDeSys V2.3. |
| SIMULATION | The SIMULATION interface is used to configure PacDrive M projects – CoDeSys V2.3 – for the simulation mode. |
| GATEWAY3 | The GATEWAY3 interface is used to configure EcoStruxure Machine Expert controllers – CoDeSys V3. |
| SIMULATION3 | The SIMULATION3 interface is used to configure EcoStruxure Machine Expert projects – CoDeSys V3 – for the simulation mode. |
