# Parameters of the Slaves - Group **CoplaCommunicationModule**

## Overview

The parameter group **CoplaCommunicationModule** comprises the parameters related to the electronic nameplate of the communication module.

This parameter group is only available for Sercos drives with a COPLA communication module.

## Parameter FpgaRevision

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | No |

With this parameter, you can read the revision number of the FPGA stored in the electronic nameplate of the communication module.

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. If the device is simulated or if Sercos is not in phase CP2 or higher, an empty string is indicated here.
