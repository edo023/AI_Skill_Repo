# Parameters of the Slaves - Group **Cap•**

## Overview

The parameter groups **Cap1**, **Cap2**, and so on, comprise the parameters used for the Touch Probe function of the drive. Each parameter group corresponds to one Touch Probe input.

The parameter groups are only available for Sercos drives with Touch Probe inputs. The number of the parameter groups depend on the number of the Touch Probe inputs.

NOTE: The parameter groups are only visible if the Touch Probe function is activated via the checkboxes “TouchProbe/Cap•” on the tab “Feature Configuration”.

## Parameter CaptureEdge

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

With this parameter, you can specify the edge to be used for triggering a position capture.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| Falling edge | Position capture at falling edge. |
| Rising edge (default value) | Position capture at rising edge. |
| Both edges | Position capture at both edges. |

## Parameter RealTimeAccess

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

With this parameter, you can read whether the captured position can be mapped to the real-time channel.
