# tab: Sercos Module – Drive Inputs and Outputs

Object: **Sercos Module**

The tab appears instead of the function group dialog when the FSP type of the driver is FSP\_Drive (0x002...). In this way, you configure the input and output connections of the module.

|  |  |
| --- | --- |
| **Name** | : Activation of the respective connection. When you deactivate a connection, the respective I/O mapping is deleted. |
| **Type** | Data type of the input or output. |
| **IDN.SI.SE** | 32-bit identification number |
| **Add** | Opens the **Select item from object directory** dialog. You can use this to add new inputs or outputs or edit existing ones. |
| **Edit** |  |
| **Delete** | Removes the input or output from the list. |
