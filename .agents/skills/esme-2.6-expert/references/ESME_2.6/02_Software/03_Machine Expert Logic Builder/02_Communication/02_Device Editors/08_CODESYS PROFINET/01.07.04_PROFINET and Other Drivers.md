# PROFINET and Other Drivers

The PROFINET Controller and Device require exclusive access to the low-level Ethernet component (SysEthernet). As a result, for example, they cannot run parallel with EtherCAT or EtherNet/IP on one adapter. If multiple Ethernet adapters are available, then a separate PROFINET driver (Controller or Device) can be operated at each Ethernet adapter.
