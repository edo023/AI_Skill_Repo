# Controller – Device object has a red symbol.

It is imperative that you check the **Controller status** on the **[Status](602f1c933f382455bd887c15b30eaad6.html#UUID-b7126354-7f96-a498-15a5-8b6708e71ff0_section-idm43261146037984)** tab of the device.

**Causes**

* The IP address or the station name of the controller are doubled
* The fixed IP address deviates from the configured IP address.
* The driver does not have a license
* No winpcap or RTE Ethernet driver is installed.
* The specified Ethernet adapter is not found.
* Internal error
