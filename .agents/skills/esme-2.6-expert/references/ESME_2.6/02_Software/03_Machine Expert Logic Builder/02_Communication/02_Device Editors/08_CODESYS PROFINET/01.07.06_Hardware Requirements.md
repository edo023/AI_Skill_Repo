# Hardware Requirements

* PROFINET requires an Ethernet adapter that supports at least 100 mbps with full duplex. All applied network components (switches) also have to support this transmission mode.
* MAC addresses

  For some controllers with multiple Ethernet adapters, unique MAC addresses are not assigned. This can lead to serious malfunctions. In this case, the user can usually set a MAC address.
