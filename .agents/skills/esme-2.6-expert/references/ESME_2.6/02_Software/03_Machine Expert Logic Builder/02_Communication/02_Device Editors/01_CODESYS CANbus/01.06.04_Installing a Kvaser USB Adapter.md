# Installing a Kvaser USB Adapter

TIP:

The CmpKvaserCANDrv component is based on the Kvaser CANBus-API (CANlib), which supports the USB dongle and other hardware. See the company homepage of Kvaser ([www.kvaser.com](https://www.kvaser.com/)) for additional information.

Support for the Kvaser API is implemented only for CODESYS Control Win V3 and CODESYS Gateway V3.

**To do this, follow the steps below:**

1. Download the driver from the manufacturer website and install it on the Windows operating system.
2. Type the CAN network number in **Network** of the CANbus **General** tab.
