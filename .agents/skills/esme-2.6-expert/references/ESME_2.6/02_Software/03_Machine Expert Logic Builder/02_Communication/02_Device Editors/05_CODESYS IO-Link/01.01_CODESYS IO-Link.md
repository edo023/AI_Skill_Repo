IO-Link is a communication system for the linking of intelligent sensors and actuators to an automation system. An IO-Link system consists of an IO-Link master that provides the interface to the superordinate controller (PLC) and one or more IO-Link devices (sensors and actuators).

The associated IODD device description files must be installed in the device repository before the IO-Link modules are available in the 'Set device' dialog.

For more information, see: [Command: Plug Device](../../../../../../api/crossBook?lang=en-US&virtualBookName=/index.htm?rhmapid=) and [Command: Device Repository](../../../&topicID=)
