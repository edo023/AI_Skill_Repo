# EtherCAT – PROFINET EL6631\_0010 Device Terminal

CODESYS supports the EL6631\_0010 device from Beckhoff, This device establishes a gateway from EtherCAT to a PROFINET Device. You insert the gateway below the EtherCAT EK1100 coupler. Then you can configure it like a typical EtherCAT Slave.

When you insert the gateway, a PROFINET Device node is inserted automatically. This PROFINET Device acts like a standard PROFINET Device.

|  |  |
| --- | --- |
|  | (1) EtherCAT coupler  (2) Gateway EtherCAT PROFINET Device  (3) PROFINET Device  (4) PROFINET Module |

You insert the PROFINET Module below the PROFINET Device. In the editor dialogs, you can set the typical PROFINET parameters and I/O channels.
