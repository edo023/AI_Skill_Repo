# Configuring Addresses for IP Communication

## Overview

The IP address is required for non-cyclic Ethernet communication via the Unified Communication Channel (non-real-time). Refer to the chapter [Sercos Communication Cycle](D-SE-0096488.html#D-SE-0096488) for details.

Two types of addressing are available:

* Automatic IP address assignment (default)
* Manual IP address assignment (hardware)

With automatic IP address assignment, the IP address and the subnet mask are assigned automatically. The Sercos master uses the following default values:

* IP address: IP address configured for **Ethernet\_1 (ETH1)**
* Subnet mask: **255.255.255.0**

Consequently, the IP address of the device is set to 172.20.0.**•••**. The subnet mask is set to the subnet mask of the master.

With manual IP address assignment, you need to set the IP address, the subnet mask and the gateway address of the slave using the [Device Assistant](../../../../../api/crossBook?lang=en-US&virtualBookName=ServiceTool.DeviceAssistant&topicID=D_SE_0059220) tool.

## Configuration of the Addresses for IP Communication

| Step | Action |
| --- | --- |
| 1 | Display the **Device tree** by clicking View > Navigators > Device tree. |
| 2 | In the tree of the Modicon M262 Motion Controller below the node **SercosMaster (Sercos Master)**, double-click the Sercos slave to be configured. |
| 3 | Display the tab **Parameters** tab of the slave. |
| 4 | Expand the parameter group **Identification** by clicking the corresponding plus sign. |
| 5 | For addressing type **Automatic**   * Use the default setting of the parameter IPConfigMode (**Automatic IP address assignment / 0**)   For addressing type **Manual**:   * Set the IP address, the subnet mask and the gateway address of the slave using the [Device Assistant](../../../../../api/crossBook?lang=en-US&virtualBookName=ServiceTool.DeviceAssistant&topicID=D_SE_0059220) tool. * Set the parameter IPConfigMode to **Manual IP address assignment (hardware) / 1** * The parameters IpAddress, Subnetmask, and Gateway contain the address values currently used |

Refer to the chapter [Parameters of the Slaves - Group **Identification**](D-SE-0096455.html#D-SE-0096455) for additional information on the parameters.
