# Parameters of the Master - Group **SercosCycletimeConfig**

## Parameter CycleTime

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | No |
| Traceable | Yes |

With this parameter, you can set the Sercos cycle time.

The cycle time can be set in communication state NRT and in communication phase CP0. The configured value becomes effective as of communication phase CP2.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| 1000000 (default value) | Sets the cycle time to 1 ms (1000000 ns). |
| 2000000 | Sets the cycle time to 2 ms (2000000 ns). |
| 4000000 | Sets the cycle time to 4 ms (4000000 ns). |

NOTE: The maximum number of supported devices depends on the configured cycle time and on the type of motion controller used. Refer to the chapter [Device Limits And Cycle Time Limits M262 With Sercos](D-SE-0096553.html#D-SE-0096553) for details.
