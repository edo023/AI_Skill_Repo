# Command: Generate PROFINET Device from GSDML

**Function**: The command opens the **Generate PROFINET Device from GSDML** dialog. This allows for CODESYS devdesc.xml files to be generated from a self-created GSDML file for a PROFINET Device.

**Call**: **Tools** menu

Generate PROFINET Device from GSDML

|  |  |
| --- | --- |
| **Source GSDML** | GSDML file for generating the device description for a PROFINET Device |
| **DAP** | Data access point in the GSDML file |
| **Devdesc ID** | Unique ID for the device description in the device repository to be generated. The suggested value can be overwritten. |
| **Devdesc version** | Version for the device description in the device repository to be generated. The suggested value can be overwritten. |

TIP:

The device description of the CODESYS PROFINET Device contained in the setup is recommended as a template for the GSDML file. To do this, export the device description and use the GSDML file from the exported zip file. This GSDML file can by extended by its own modules and the identification data (vendor, product description) adapted.
