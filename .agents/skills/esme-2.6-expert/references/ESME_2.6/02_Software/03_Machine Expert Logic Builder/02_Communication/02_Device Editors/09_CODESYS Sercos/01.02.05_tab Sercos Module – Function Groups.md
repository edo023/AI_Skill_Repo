# Tab: Sercos Module – Function Groups

Object: **Sercos Module**

On this tab, you define the real-time data. Depending on the imported device description file, one or more function groups will be displayed here. These consist of inputs or outputs and can contain several more inputs and outputs.

Using the definitions here, CODESYS maps the input and output channels on the **Sercos Module I/O Mapping** tab.

Function Groups

|  |  |
| --- | --- |
| **Name** | : If the device description file allows this, the function groups can be activated and deactivated. Function groups that do not allow activation and deactivation are shown in gray. By deactivating individual function groups, the number of channels can be reduced. These function groups are not shown in the I/O mapping of the modules.  If defined in the device description, the function groups show the **IO Monitoring** group. If this is activated, then an additional bit is generated for each channel, showing the status of the channel. If this bit yields TRUE, then the channel data is valid. If not, then the data is invalid. |
| **Number of channels** | If defined in the device description, the number of channels and the channel width of DOUT, PDIN, DiagIn, and DiagOut can be changed. |

| **Type** | Data type of the function group |
