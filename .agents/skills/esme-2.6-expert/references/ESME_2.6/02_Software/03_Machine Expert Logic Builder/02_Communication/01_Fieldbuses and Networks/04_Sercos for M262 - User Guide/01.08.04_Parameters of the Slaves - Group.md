# Parameters of the Slaves - Group **SercosDiagnostics**

## Overview

The parameter group **SercosDiagnostics** comprises the parameters used for Sercos diagnostics and information related to the Sercos connection.

## Parameter DiagnosticNumber

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

With this parameter, you can read the diagnostic code of the device.

The parameter DiagnosticNumber corresponds to the Sercos parameter IDN S-0-0390.

Structure of the parameter IDN S-0-0390:

| Bit | Value | Designation | Notes |
| --- | --- | --- | --- |
| 31…30  Interpretation of the source type (bits 29 ... 24), the class (bits 19 ... 16), and the status code (bits 15 ... 0). | 00 bin | Manufacturer-specific status codes | Bits 29…16 (source type and class) are specified by Sercos.  Bits 15…0 (status code) are specified by the manufacturer. |
| 01 bin | Fully manufacturer-specific | Bits 29…0 are specified by the manufacturer. |
| 10 bin | Reserved | Bits 29…0 are specified by Sercos. |
| 11 bin | Standard | – |
| 29…24  Source type | 00 hex | FSP drive | – |
| 01 hex | FSP IO | Further detailed IO-specific information can be found in the following table S-0-1500.x.32. |
| 02 hex | GDP | – |
| 03 hex | SCP | – |
| 04 hex | CSoS | – |
| 05 hex | FSP encoder | – |
| 06 … 3E hex | Reserved | – |
| 3F hex | Unknown | – |
| 23…20  Reserved | – | – | – |
| 19…16  Class | 00 … 09 hex | Reserved | – |
| 0A hex | Operational state | **Priority 4 (lowest)**  For information about the operational state, regarding messages, or other information (for example, Drive STOP or Compatible replacement). |
| 0B hex | Reserved | – |
| 0C hex | Procedure command specific state | **Priority 3**  For information about the diagnostic events, regarding messages, or other information (for example, Drive STOP or Compatible replacement). |
| 0D hex | Reserved | – |
| 0E hex | Warning (C2D) | **Priority 2** |
| 0F hex | Error (C1D) | **Priority 1 (highest)**  Has to be confirmed/acknowledged when using the procedure command S-0-0099 Reset class 1 diagnostic |
| 15…0  Status code | – | – | – |

If diagnostic messages "C1D error" or "C2D warning" are triggered, this parameter can give a more precise explanation of what caused the error.

## Parameter ConnectionState

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

With this parameter, you can read the connection state of the device.

* Value Operational:

  Sercos state of the master is CP4, slave is present on Sercos network, real-time communication is running.
* Value PhasingUp:

  Sercos state of the master is CP0, CP1, CP2, or CP3.
* Value NoCommunication:

  Sercos state of the master is CP4, slave is not present on Sercos network.
* Value NoData:

  Sercos state of the master is CP4, slave is present on Sercos network, real-time communication is not running.
* Value NotRunning:

  Sercos state of the master is Phase Error or NRT.
