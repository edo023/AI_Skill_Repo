# Tab: EtherCAT Master – Parameters

The tab contains the master parameters which are defined in the device description file.

When the **[auto-configuration mode](_ecat_edt_master_master.html#_ecat_edt_master_master)** is selected in the **Master** dialog, the parameters are set here automatically according to the specifications from the device description file and the network topology. Nothing should be changed in the generic editor because an invalid configuration can be set here.

|  |  |
| --- | --- |
| **Value** | Editable: A change is effective only when the auto-configuration mode is disabled.  Whether or not the change becomes effective depends on the respective parameter. |
