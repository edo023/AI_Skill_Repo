# Issue: Error Message CoDeSysOPC.exe - Entry Point not Found Is Displayed

![](images/logo.png)

Issue: Error Message CoDeSysOPC.exe - Entry Point not Found Is Displayed

Issue

The error message CoDeSysOPC.exe - Entry point not found is displayed. The procedure entry point GSymEnterVarListAccess has not been found in the GSymbol.dll file.

Cause 1

The gateway server/Gateway Management Console is not updated.

The error message CoDeSysOPC.exe - Entry point not found is displayed if the OPC server has not been installed correctly.

Solution 1

Reinstall the OPC server.
