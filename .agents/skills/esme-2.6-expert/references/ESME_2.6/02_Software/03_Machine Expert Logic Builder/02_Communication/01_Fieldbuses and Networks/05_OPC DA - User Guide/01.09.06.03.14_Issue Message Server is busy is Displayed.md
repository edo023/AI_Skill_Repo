# Issue: Message Server is busy is Displayed

![](images/logo.png)

Issue: Message Server is busy is Displayed

Issue

After you have started the OPC client and you attempt to establish a connection to the OPC server, the operating system message Server is busy is displayed.

Cause 1

When starting the OPC server, a connection to the controller cannot be established immediately. In this case, the OPC server waits until the waiting time has elapsed. It then retries to connect to the client.

Solution 1

Reduce the value of the Wait time (s) [parameter](../OPC_Server_Configuration/OPC_Server_Configuration-2.htm#XREF_D_SE_0083829_12) in the controller configuration of the OPC configuration tool.

Cause 2

The virtual RAM of the computer running the visualization is not sufficient.

Solution 2

Increase the virtual RAM of the computer running the visualization.
