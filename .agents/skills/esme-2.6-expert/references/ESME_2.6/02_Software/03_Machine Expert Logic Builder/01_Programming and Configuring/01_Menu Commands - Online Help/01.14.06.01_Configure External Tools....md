# Configure External Tools...

## Overview

The Tools > External Tools > Configure External Tools... dialog box allows you to start external tools, such as Controller Assistant, and other tools from EcoStruxure Machine Expert.

In the Menu contents section of the External Tools dialog box, a list is provided that contains by default preconfigured Schneider Electric tools and commands. These entries cannot be deleted but can be copied and modified. Additionally, you can add tools and commands.

The following buttons are available for the Menu contents list:

| Button | Description |
| --- | --- |
| Add | Adds a new row to the bottom of the Menu contents list with the default name [New Tool [n]]. |
| Delete | Deletes the selected row from the Menu contents list. |
| Clone | Copies the selected row and creates a new row at the bottom of the Menu contents list with the same name extended by \_1. |
| Export | Allows to export the menu content that you have added in addition to the predefined tools. The default rows that are preconfigured for Schneider Electric tools cannot be exported.  Click the Export button to display a dialog box that requests the following export options:   * Click Yes to export the selected row. * Click No to export all rows that you have created. * Click Cancel to cancel the export operation and return to the External Tools dialog box.   If you click Yes or No, a Save as dialog box opens. Enter a file name and browse to the folder where you want to save the exported *\*.xml* file. |
| Import | Opens a File Open dialog box that allows you to browse for the *\*.xml* file you want to import. |
| Move Up | Moves the selected row up in the Menu contents list and in the menu configuration of EcoStruxure Machine Expert. |
| Move Down | Moves the selected row down in the Menu contents list and in the menu configuration of EcoStruxure Machine Expert. |

## Creating a Customized Entry

To add a customized entry to the Menu contents list, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Click the Add button to create a new entry from scratch or click the Clone button to copy an existing entry. |
| 2 | In the Title field below the list, enter a name for the entry. |
| 3 | In the Command line, click the browse button ... to browse for the executable file of the external tool.  The preconfigured Schneider Electric tools are provided in the subfolder Tools of the EcoStruxure Machine Expert installation folder. |
| 4 | In the Arguments line, click the right arrow button and select an entry from the list of available arguments or type another command line argument. |
| 5 | In the Initial directory line, click the right arrow button and select an entry from the list of available directories or type another directory. |
| 6 | Optionally, enter further information in the Comment field. |
| 7 | Click OK or Apply to create the entry. |
