# Compiler Error C0169

## Message

‘VAR\_GLOBAL’ declaration is only allowed in a global variable list

## Message Cause

‘VAR\_GLOBAL’ is used outside a global variable list.

## Solution

Use ‘VAR\_GLOBAL’ only in a global variable list.

## Error Example

```
PROGRAM PLC_PRG
VAR_GLOBAL
  i : INT;
END_VAR
```

--> C0169: 'VAR\_GLOBAL' declaration is only allowed in a global variable list
