# Route All Connections

## Overview

The CFC > Route All Connections command additionally is available in the menu behind the button ![](images/T003483113.gif) in the upper right corner of the [**CFC Editor**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083491) view.

The command affects a rerouting of all connections in the current CFC. Connection lines which have been modified manually, will be set back to the original automatically optimized routing, except for those still fixed by control points. For details refer to [routing in a CFC](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083494).
