# Compiler Error C0215

## Message

Direct address declaration is not possible in persistent list.

## Message Cause

Persistent variables are not allowed to have a direct address.

## Solution

Remove the direct address assignment in the persistent variable list.

## Error Example

```
VAR_GLOBAL PERSISTENT RETAIN
  directAddressVar AT %QB7 : BYTE;
END_VAR
```
