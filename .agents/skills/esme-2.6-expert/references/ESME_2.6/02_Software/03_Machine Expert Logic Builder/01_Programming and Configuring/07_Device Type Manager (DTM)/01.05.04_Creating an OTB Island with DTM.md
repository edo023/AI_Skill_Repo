# Creating an OTB Island with DTM

![](images/logo.png)

Creating an OTB Island with DTM

Overview

This chapter describes how to create an OTB island on the CANopen fieldbus with DTM.

NOTE: An OTB island is the association of a network interface module with expansion modules. It offers a modular and flexible I/O solution.
