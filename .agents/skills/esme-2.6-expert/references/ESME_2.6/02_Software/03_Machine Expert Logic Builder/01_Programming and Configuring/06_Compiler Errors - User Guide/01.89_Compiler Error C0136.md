# Compiler Error C0136

## Message

Ambiguous use of name ‘<variable name>’.

## Message Cause

A variable is declared in multiple GVLs.

NOTE: When this error is detected, the positions of multiple use are indicated in the Messagesview (C0181).

## Solution

Define the variable with the desired GVL.

## Error Example

```
PROGRAM PLC_PRG
VAR
  j : INT := g_i;
END_VAR
```

```
GVL1:
VAR_GLOBAL
  g_i : INT;
END_VAR
```

```
GVL2:
VAR_GLOBAL
  g_i : INT;
END_VAR
```

--> C0136: Ambiguous use of name 'g\_i'

## Error Correction

```
j : INT := GVL1.g_i;
```
