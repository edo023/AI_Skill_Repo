# Stop

## Overview

The Debug > Stop command stops the [application](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083435) program in the controller.

If the command is called from the Debug menu, the application is affected.

If you execute the command from the contextual menu when an application object is selected, then the selected application is affected, whether it is set to active status or not.

If the option Secure online mode is activated in the Communication Settings of the respective device, you have to confirm after calling this command.
