# Dialog: Create HMI Project: Choose Perspective

**Function**: The dialog is part of the wizard that opens when you create a new project with the **HMI project** template. It is used for selecting the perspective for the current project. The perspective can be changed at any time with the **Choose Perspective** command.

**Call**: The dialog opens automatically after the **Create HMI Project – Add Objects** dialog.

|  |  |
| --- | --- |
| **HMI Perspective**  **CODESYS Perspective** | : Perspective for the new project |
| **Preview** | Preview of the project organization in the new project according to the configuration in the wizard. |
| **Abort** | Applies the configuration and creates a new HMI project in CODESYS  Now the basic framework of an HMI project has been created. |
