# Display

|  |  |
| --- | --- |
| **Display type** | * **Bars** Data is displayed as bars * **Lines**: Data is displayed as lines * **Curve**: Interpolation of data into a curve. |
| **Line width** | Specified in pixels  Requirement: As the  **Display Type** is **Curve** selected. |
| **Show horizontal lines** | : Horizontal lines are drawn on the main scale  NOTE: Not all visualization styles have this element property. For visualization styles that have a striped background, for example **Flat style**, this element property does not exist. |
| **Relative bar width** | **Integer value between `1` and `100`**   * `1`: The bars are drawn as lines * `100`: Total width of the histogram is filled with the bars. |
