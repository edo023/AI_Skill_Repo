# Compiler Error C0077

## Message

Unknown type: ‘<data type>’

## Message Cause

Invalid data type in the declaration (maybe a syntax error).

## Solution

Only specify valid data types.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INTEGER;
END_VAR
```

--> C0077: Unknown type: 'INTEGER'

## Error Correction

```
i : INT;
```

EIO0000003933.04
