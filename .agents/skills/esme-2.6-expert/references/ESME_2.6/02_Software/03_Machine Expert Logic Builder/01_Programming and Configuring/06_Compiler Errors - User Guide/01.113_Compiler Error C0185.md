# Compiler Error C0185

## Message

It is not possible to perform a component access ‘.’, index access ‘[]’ or call ‘()’ on the result of a function call. Assign the result to help a variable first.

## Message Cause

A component or index access to the result of a function call is performed.

## Solution

Assign the result to a variable to access it.

## Error Example

```
PROGRAM PLC_PRG
VAR
    it : ITF;
END_VAR
```

```
POU_1()[0].METH1();
```

```
FUNCTION POU_1 : ARRAY[0..0] OF ITF
```

```
INTERFACE ITF
```

```
METHOD METH1
```
