# Change Device Name...

## Overview

The Change Device Name... command opens the same-named dialog box for changing the (present) name of the device as used in the tree structure of configured gateway channels in the Communication Settings [tab in classic mode](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0088469). You can enter the new name manually in the New: text field. Or you can select the standard option <devicename> (name of the device as used in the Devices tree of the project) from the selection list.
