## Contents

Default shortcut: Ctrl + Shift+ F1

Execute the Help > Contents command to open the online help in a web browser window.

## Download Online Help Files

Execute the Help > Download Online Help Files command to open the Schneider Electric Help Manager. Select a root node to choose a collection of documents (for example, Software or How To documents) or expand the root nodes and select individual documents from the sub nodes. Click DOWNLOAD to start downloading the selected online help file(s) to your PC for offline usage.

## ‘Machine Expert - Safety’ help...

Execute the Help > ‘Machine Expert - Safety’ help... command to open the EcoStruxure Machine Expert - Safety online help.

## Manage licenses...

Execute the Help > Manage licenses command to open the Manage licenses dialog box for managing your EcoStruxure Machine Expert licenses.

## Open Machine Advisor

Execute the Help > Open Machine Advisor command to open the EcoStruxure Machine Advisor in a web browser window.

## Open Machine Advisor Code Analysis

Execute the Help > Open Machine Advisor Code Analysis command to open the EcoStruxure Machine Advisor Code Analysis in a web browser window.

## Update Offline Help

This command is only applicable if you have downloaded the online help for offline usage by executing the command [Download Online Help Files](#D-SE-0096077__OpenUpdateOfflineHelpDialog-08662164). By default, a search for updates of the help books you have downloaded is performed in the background when you click the F1 key for the first time in the open instance of EcoStruxure Machine Expert. If outdated documents are detected, the Update Offline Help is displayed. Alternatively, you can execute the Help > Update Offline Help command to open the dialog box and to start the search for updated documents manually.

The Update Offline Help dialog box allows you to:

* Deactivate the automatic update process by deselecting the Check for Updates check box.
* Update the outdated online help documents that are listed in the Help Books area of the dialog box by clicking the Update All button.

  NOTE: The update process can take a long time depending on the number of books and number of languages that need to be updated. Do not close EcoStruxure Machine Expert as this will interrupt the update process and can result in an inconsistent state of the offline help. If you attempt to close the application while the update process is still running, a message is displayed indicating that this will interrupt the update process. In the case that the update process is interrupted, reopen the Update Offline Help dialog box and click the Update All button again.

  The progress of the update process is indicated in the Update Offline Help dialog box for each document and also as an overall progress bar for the entire update process. After you have started the update process you can close the dialog box. You can reopen it any time to see the progress. Additionally, the progress is indicated in the Offline Help Update category of the [Messages view](D-SE-0083922.html). The message Update of offline help finished. indicates that the procedure has been successfully completed.

## Video tutorials

Execute the Help > Video tutorials command to open a web browser providing a playlist of tutorials on Schneider Electric software.

## Training material and more Project examples

Execute the Help > Training material and more Project examples command to open the Schneider Electric web site providing a download link for training material on EcoStruxure Machine Expert.

## About...

Execute the Help > About... command to open a view providing information on the installed version of EcoStruxure Machine Expert.

## Technical Information...

Execute the Help > Technical Information... command to open the Machine Expert dialog box that provides copyright, license, and technical information.

## Collect and pack log files

Execute the Help > Collect and pack log files command to collect the different log files provided by the individual components of EcoStruxure Machine Expert and store them to one common \*.zip file. The file is stored to the temporary directory configured in the General directories tab of the [Tools > Options > Directories (Devices, Libraries, ...) dialog box](Directories-01C80778.html#Directories-01C80778__GeneralDirectoriesTab-01E68FEB).
