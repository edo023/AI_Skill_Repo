# Project Settings - Firmware Update

## Overview

EcoStruxure Machine Expert allows you to perform an automatic verification for the latest firmware version that is available on your computer whenever you make a connection to most controllers. You can enable or disable this function in the Project Settings dialog box for each project individually as described in the following paragraphs.

If the function is activated, the verification is performed automatically each time you intend to login to a supported controller. If a more recent firmware version is detected on your computer, this is indicated by a message in the Messages view or additionally by a separate message window that is displayed.

NOTE: The Firmware Update function searches the directory structure that was created during the installation of EcoStruxure Machine Expert to find the various firmware versions available. If you download a new firmware and do not store it in the correct directory, the Firmware Update function will not be aware that another version exists elsewhere on your machine.

## Supported Controllers

The automatic search for the latest firmware version is available for the following controllers:

| Controller | M258 / LMC058 | M241 / M251 | XBT Controllers |
| --- | --- | --- | --- |
| Verification for latest firmware version available | yes | yes | no |

## Activating / Deactivating the Verification for the Latest Firmware

To activate / deactivate the automatic verification for a more recent firmware of the supported controllers, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Open the project for which you want to activate / deactivate the verification. |
| 2 | Open the Program tab. |
| 3 | Select the menu Project > Project Settings.... |
| 4 | In the Project Settings dialog box select the option Firmware Update from the list on the left-hand side.  **Result**: The Firmware Update configuration dialog box is displayed: |
| 5 | Configure the firmware update function as follows:   * To activate the automatic verification for a more recent firmware of the supported controllers in this project, select the option Check if new Firmware is available. * To deactivate the automatic verification, select the option Do not check if new Firmware is available. * To prevent the software from opening a separate message box indicating that a more recent firmware has been found, select the option Do not show any pop-messages. In this case the more recent firmware version is only indicated in the Messages view. |

## Messages Indicating that a More Recent Firmware is Available

If the verification function is enabled and a more recent firmware version is detected, this is indicated in two different ways:

* A message in the Messages view lists the most recent firmware version that has been detected and the path where it is saved on your computer.
* A separate message box is opened that indicates the currently installed firmware version and the more recent version that has been detected and the path where it is saved on your computer. It also provides a link to the programming guide of the respective controller where you can find further information.

The separate message box is optional. You can prevent it from being created in the Project Settings dialog box via the Do not show any pop-messages option or by activating the Do not show this message again option in the message box itself.
