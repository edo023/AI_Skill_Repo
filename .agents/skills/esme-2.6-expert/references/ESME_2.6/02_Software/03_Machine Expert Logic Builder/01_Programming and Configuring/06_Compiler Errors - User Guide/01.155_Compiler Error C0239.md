# Compiler Error C0239

## Message

Interface <interface name 1> does not extend <interface name 2>

## Message Cause

The used interface does not extend another interface.

## Solution

Extend the interface.

## Error Example

```
PROGRAM PLC_PRG
VAR
  ITFref : ITF;
  ITFref2 : ITF2;
END_VAR
```

```
__QueryInterface(ITFref2,ITFref);
```

```
INTERFACE ITF
INTERFACE ITF2 EXTENDS ITF
```

--> C0239: Interface ITF2\_\_Union does not extend \_\_System.IQueryInterface.

For further information, refer to the [`__QUERYINTERFACE` operator](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083756) in the Programming Guide.

## Error Correction

```
INTERFACE ITF EXTENDS __System.IQueryInterface
```
