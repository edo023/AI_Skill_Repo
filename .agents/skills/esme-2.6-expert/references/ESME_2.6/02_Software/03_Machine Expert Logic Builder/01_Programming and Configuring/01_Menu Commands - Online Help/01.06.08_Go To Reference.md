# Go To Reference

## Overview

You can execute the Edit > Browse > Go To Reference command in online mode when a POU is open in the editor and the cursor is positioned on a pointer variable. If the referenced variable is stored in static memory, the declaration location opens.

The command is also available in the contextual menu of a pointer variable in the declaration part or in implementation code.

NOTE: If the pointer does not point exactly to the beginning of the variable, for example, if a POINTER TO BYTE variable points to the second byte of a WORD variable, a message is displayed when you execute the command. It indicates that the pointer address is inside a variable but is not equal to the start address. The declaration location remains open.

The message is not displayed if a POINTER TO X points to an element of an ARRAY OF X.
