# Context menu (right-click an object)

|  |  |
| --- | --- |
| **Compare** | Opens the compare dialog to compare the working copy with the top-level revision |
| **Compare with HEAD version** | Opens the compare dialog to compare the working copy with the HEAD revision |
| **Compare with Revision** | **The list entries are highlighted in colors according to the object status:**   * Blue: Modified * Green: Added * Dark red: Deleted * Red: Conflicted * Black: Non-versioned (not in SVN repository)  Note: These objects are displayed when the **Show non-versioned objects** option is selected. * Gray: Excluded from commit  Note: This is the case when the **Ignore during commit** option is selected for the object. |
| **Clear** | Discards your changes to the working copy  After that, the object corresponds to the revision in the SVN repository. |
| **Show log** | Shows the version history of the selected object |
| **Properties** | Opens the [SVN Properties](a43046eadca83df0a0dd11fa4ebb7082.html#UUID-396b58cb-c13c-1d99-1a28-686b7752a816_section-idm43257679000916) dialog  There the properties are displayed and you can edit them. |
| Move to change list | Note: The command has not been implemented yet. |
