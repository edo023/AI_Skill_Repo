# Compiler Warning C0441

## Message

Access to uninitialized VAR\_IN\_OUT variable.

## Message Cause

A VAR\_IN\_OUT variable is used as the initial value of another variable declaration.

## Solution

Remove the initial value and assign the value to the variable in the implementation part.

## Error Example

```
FUNCTION_BLOCK POU
VAR_INPUT
      i:INT;
END_VAR
VAR_OUTPUT
      b:BOOL := a[i];
END_VAR
VAR_IN_OUT
      a:ARRAY [1..5] OF BOOL;
END_VAR
```

--> C0441: Access to uninitialized VAR\_IN\_OUT variable.
