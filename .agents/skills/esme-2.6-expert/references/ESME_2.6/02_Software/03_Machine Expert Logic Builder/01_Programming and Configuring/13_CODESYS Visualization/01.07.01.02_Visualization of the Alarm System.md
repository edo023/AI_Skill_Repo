# Visualization of the Alarm System

The alarm system – which monitors your system, detects and saves faults, and generates messages – can be displayed via the visualization. The alarm messages are output there and confirmations are returned.

You can use the **Alarm Table** and **Alarm Banner** visualization elements to create and display a user interface for the alarm management of an application. The alarm table lists the alarms. The alarm banner is a simplified version of the alarm table. It visualizes a single alarm only. However, by adding scroll elements to the banner you can allow for switching the display from one active alarm to another active alarm.

The visual representation of the alarms in the visualization can be specified in the display settings of an alarm class. In that way, alarms of different alarm classes can be clearly separated from each other visually.
