# Configuration

Output to Test Report

|  |  |
| --- | --- |
| **Include in "Version information" header** | : The version information is included in the above table in the header of the test report; otherwise it appears below. |
| **Title in report** | Title for this information in the header table  If you do not specify a title, then a default title is derived from the project name.  Example: `Test report Project1` |

Output to Test Script Variables

|  |  |
| --- | --- |
| **Project URL** | : The URL of the project is output in the specified test script variable (output variable). |
| **Revision Summary** | : Outputs the tags of the revision summary from the `SVN_VERSION_INFO.VERSION` or `svnversion.exe`  Example: "12:34M": Mixed revision working copy with minimum revision 12, maximum revision 34, and local modifications |
