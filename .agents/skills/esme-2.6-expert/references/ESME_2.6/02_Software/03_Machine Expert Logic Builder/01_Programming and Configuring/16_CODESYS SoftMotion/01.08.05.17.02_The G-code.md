# The G-code

`CNC_PathSpeed`

```
N10 G01 X1000 F6000
N20 Y1000
```

In this G-code, the path velocity in block `N10` is set with `F6000` to 6000 mm/min. However, the CNC function blocks expect the path velocity in mm/s. Two linear movements are commanded, first to `X=1000` and then to `X=1000, Y=1000`.
