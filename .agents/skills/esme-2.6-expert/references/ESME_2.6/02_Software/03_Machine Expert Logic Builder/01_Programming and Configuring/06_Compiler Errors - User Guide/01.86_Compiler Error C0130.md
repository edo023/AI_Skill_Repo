# Compiler Error C0130

## Message

<object> ‘<object name>’ referenced without parentheses ‘()’

## Message Cause

A method is referenced without parentheses.

## Solution

Always reference methods by means of parentheses.

## Error Example

```
FUNCTION_BLOCK FB
VAR
END_VAR
```

```
METHOD METH1
VAR_INPUT
END_VAR
```

```
PROGRAM PLC_PRG
VAR
  inst : FB;
END_VAR
```

```
inst.METH1
```

--> C0130: METHOD 'METH1' referenced without parentheses '()'

## Error Correction

```
inst.METH1();
```
