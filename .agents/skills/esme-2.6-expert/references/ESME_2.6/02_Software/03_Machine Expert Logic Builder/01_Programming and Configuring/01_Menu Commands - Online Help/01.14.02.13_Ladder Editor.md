# Ladder Editor

## Overview

The Tools > Options > Ladder Editor dialog box allows settings concerning the editing in the Ladder editor.

## General Tab

|  |  |
| --- | --- |
| View | |
| Show network title | The network title is displayed in the upper left corner of a network if it is defined with [Ladder > Insert Network](InsertNetwork-4526CB92.html). |
| Show network comment | The network comment is displayed in the upper left corner line of a network if it is defined with [Ladder > Insert Network](InsertNetwork-4526CB92.html). If the network title is visible, the comment is displayed in the line below. |
| Show block icon | If a function block or a function available through a library or the [object properties](D-SE-0083921.html#D-SE-0083921) is provided with an icon (bitmap), this icon is displayed in the block in the Ladder editor. The standard operators are also provided with icons. |
| Show operand comment | The comment that can be assigned to an operand in the implementation part of the editor is displayed. The operand comment only refers to the usage of the operand in place, in contrast to the symbol comment, which is defined at the declaration of a variable. Operand comments can be edited in the editor.  This comment is truncated automatically depending on available space. |
| Show symbol comment | For each variable that has got a comment in its declaration, this comment is displayed above the identifier of the symbol. |
| Show symbol address | For each symbol (variable) the assigned address is displayed above the identifier of the symbol. |
| Operand sizing | |
| Maximum number of lines shown | Maximum number of lines of the operand name that can be displayed. |
| Maximum average characters per line | Maximum number of characters per line for displaying the operand name. |

## Monitoring Tab

|  |  |
| --- | --- |
| View | |
| Displayed digits for floating point numbers: `<n>` digits | Configure the number of digits to be displayed for floating point numbers.  Example representing the same number with different digits settings:   * `5` digits = `750.15` * `2` digits = `7.5e+02` |
| Displayed string length | Configure the maximum number of characters of a string. |
