Returns True if current expert function is main, False otherwise

Returns True if current expert function is a sub expert function, False otherwise

Get or Set name of expert function Name must not contain space

sub\_expert\_functions

Gets sub expert functions for reading and editing settings. This property is declared as an object and accepts an [IScriptEioExpertFunctions](IScriptEioExpertFunctions.html#iscripteioexpertfunctions_module_IScriptEioExpertFunctions) instance or a Python list for the values.

```
expert_function.sub_expert_functions = [{'type' : 'SubExpertFunctionType', 'name' : 'Expert_Function_Name'}]
```
