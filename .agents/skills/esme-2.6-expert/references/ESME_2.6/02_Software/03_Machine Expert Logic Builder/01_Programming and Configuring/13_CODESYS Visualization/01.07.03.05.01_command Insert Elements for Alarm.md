# Command: Insert Elements for Alarm Acknowledgement

Symbol: ![](images/uuid-f30cbda1-287e-44e9-e58f-914839.png)

**Function**: The command adds buttons automatically to the visualization for acknowledging alarms. It opens an assistant for inserting controls below the table.

**Call**: **Visualization** menu; context menu of the "Alarm Table" visualization element

**Requirement**: An "Alarm Table" visualization element is selected.
