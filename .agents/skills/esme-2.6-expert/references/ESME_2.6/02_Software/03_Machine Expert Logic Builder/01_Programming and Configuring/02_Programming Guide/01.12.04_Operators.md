# Operators

## Overview

All IEC operators are supported. In contrast to the standard functions, these operators are recognized implicitly throughout the project.

Besides the IEC operators, the following operators are supported which are not specified by the standard:

* `ANDN`
* `ORN`
* `XORN`
* `SIZEOF` (refer to [arithmetic operators](D-SE-0083688.html#D-SE-0083688))
* `ADR`
* `BITADR`
* content operator (refer to [address operators](D-SE-0083719.html#D-SE-0083719)
* some [scope operators](D-SE-0083762.html#D-SE-0083762)
* `__CURRENTTASK`
* `__COMPARE_AND_SWAP`
* `__XADD`
* `__TEST_AND_SET`
