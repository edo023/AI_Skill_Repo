# Compiler Error C0230

## Message

Type name ‘<data type>’ not expected in this place.

## Message Cause

The data type name of an enumeration is used at an invalid position.

## Solution

Verify whether the data type name is used correctly at this location. Maybe there is a spelling error.

## Error Example

```
TYPE MyEnum :
 (
    enum_member := 0
);
END_TYPE
```

```
PROGRAM PLC_PRG
VAR
    value : INT;
END_VAR
```

```
value := MyEnum;
MyEnum := value;
```

--> For PLC\_PRG, the error message is issued 2x: C0230: Type name ‘MyEnum’ not expected in this place.

## Error Correction

```
value := MyEnum.enum_member;
```
