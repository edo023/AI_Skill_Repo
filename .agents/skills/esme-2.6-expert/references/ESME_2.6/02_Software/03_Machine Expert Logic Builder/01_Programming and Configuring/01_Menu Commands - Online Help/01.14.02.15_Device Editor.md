# Device Editor

## Overview

The Tools > Options > Device Editor dialog box allows settings concerning the editing in a [device editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083384).

The View tab provides the following options:

| Option | Description |
| --- | --- |
| Show generic device configuration views | If this option is activated, the Configuration [view](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083387) is available in the device editor of the respective device. |
| Create Cross References for IEC addresses (Clean necessary) | If this option is activated, cross-references for unmapped inputs/outputs are created. For more information, refer to [Clean](D-SE-0083988.html#D-SE-0083988). |
| Communication page | Select the display mode for the Communication Settings tab of the device editor:   * Simple mode [view](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083386) * Classic mode [view](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0088469) * Controller selection mode [view](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083385) |
| Show implicit files for application download on the editor of a PLC | If this option is activated, the tab for synchronized files is available in the device editors. Synchronized files are downloaded to the controller at the time of application download. This can be external files that were added to the application, for example, a source code archive.  This function is not available for all supported controllers. Consult the *Programming Guide* specific to your controller. |
| Show access rights page | If this option is activated, the Access Rights [tab](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083877) is available in the device editors.  NOTE: Depending on the device, the device description may overwrite this setting. |
| Use horizontal tab pages | If this option is activated, you can create a new group of tabs as a sub-tab of the active view of the device editor. |
