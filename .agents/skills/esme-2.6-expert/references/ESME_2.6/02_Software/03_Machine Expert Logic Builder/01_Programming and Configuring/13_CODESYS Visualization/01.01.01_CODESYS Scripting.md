# CODESYS Scripting

With CODESYS Scripting, you can automate commands or complex program operations in CODESYS, which otherwise require manual mouse clicks and text input in the CODESYS user interface. You can access the (Iron)Python-based scripts via the scripting commands from CODESYS. Alternatively, it is also possible to start from the Windows command line.
