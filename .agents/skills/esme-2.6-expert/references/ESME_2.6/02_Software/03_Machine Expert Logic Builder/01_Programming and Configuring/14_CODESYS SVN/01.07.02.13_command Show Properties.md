# Command: Show Properties

Symbol: ![](images/uuid-35d1ca5b-3aa1-8133-154d-88c510.png)

**Function**: The command opens the **SVN Properties** dialog. There you can edit the properties of the versioned object.

**Call**: Context menu: **SVN**

**Requirement**: A versioned, unlocked object is selected.
