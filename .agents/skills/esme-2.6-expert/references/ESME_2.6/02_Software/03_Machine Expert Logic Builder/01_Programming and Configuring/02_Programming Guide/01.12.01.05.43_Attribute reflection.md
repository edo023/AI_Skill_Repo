# `Attribute reflection`

## Overview

Attach the pragma `{attribute 'reflection'}` to POUs containing variables that are tagged with the instance-path [attribute](D-SE-0083637.html#D-SE-0083637) or the is-connected [attribute](D-SE-0107038.html#D-SE-0107038).

## Syntax

{attribute 'reflection'}

## Example

Refer to the `attribute instance-path` [example](D-SE-0083637.html#D-SE-0083637__D-SE-0083637.4) and to the attribute is-connected [example](D-SE-0107038.html#D-SE-0107038__D-SE-0107038.6).
