# Mass Storage (USB or SDCard)

## Overview

The Mass Storage (USB or SDCard)... command in the Project menu allows you to save the file system image together with a file management script on a mass storage device. This function is only available for M241 and M251 logic controllers.

## Saving Data on a Mass Storage Device

To save your project to a mass storage device, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Connect the mass storage device to the PC running EcoStruxure Machine Expert. |
| 2 | From the Project menu, select the command Mass Storage (USB or SDCard)....  **Result**: EcoStruxure Machine Expert will automatically build the application. If errors are detected during the build process, an error detection message will be issued and the process will be canceled. If the build process is executed successfully, the Mass Storage (USB or SDCard) dialog box will be displayed. |
| 3 | In the Mass Storage (USB or SDCard) dialog box, click the plus sign to create a new row for a new command. |
| 4 | Select the commands for the script that will be executed on the controller from the Command list box or select already available macros from the Macros list. |
| 5 | Select the source for the commands from the Source list box or click the ... button to navigate to the folder that contains the source. |
| 6 | To change the Destination subfolder on the storage device, open the Destination list and select the suitable folder. |
| 7 | To create the script, click the Generate button at the bottom or the gearwheel icon from the toolbar at the top of the dialog box.  **Result**: A Browse For Folder dialog box opens. |
| 8 | Browse to the root directory of your storage device (do not open a subfolder) and click Open.  **Result**: The project and the file management script are saved to the USB mass storage device. |

## Installing the Project on a Controller

To install the EcoStruxure Machine Expert project on a controller, connect the mass storage device to the controller.

The script on the mass storage device will be executed automatically which is indicated by the LEDs of the controller as follows:

| Action Performed | LED Indication | Comment |
| --- | --- | --- |
| Mass storage device is plugged into the controller. | red LED is on | The red LED is on until the communication to the mass storage device has been established. |
| Script file is executed. | green LED is flashing | Do not remove the mass storage device while the script file is executing. |
| Script file execution has been completed successfully. | green LED is on | You can remove the mass storage device. |
| The script file could not be executed successfully. | red LED is on | See the log file on the mass storage device for reasons why the script file could not be executed successfully. |

NOTE: To then copy the application from the controller file system to executable memory, cycle power on the controller.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  Cycle power on the controller after installing an EcoStruxure Machine Expert project via a script file.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

## Log File

A log file is automatically saved in the folder of the script file.

It records every execution of the script file with header information (date, time, and machine information) and the result (successful, or not successful like: source file not available, incorrect destination, incorrect command, destination full).

Any new information is added to the file, information will not be overwritten.
