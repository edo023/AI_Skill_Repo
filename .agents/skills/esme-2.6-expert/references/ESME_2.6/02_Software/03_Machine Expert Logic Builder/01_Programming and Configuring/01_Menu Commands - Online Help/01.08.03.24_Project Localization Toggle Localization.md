# Project Localization > Toggle Localization

## Overview

The Project > Project Localization > Toggle Localization command is only available if a project is open and a Default Localization has been defined for the project in the Manage Localizations [dialog box](D-SE-0083941.html#D-SE-0083941__D-SE-0083941.3). Execute the command to switch between the Default Localization and the <Original Version>.
