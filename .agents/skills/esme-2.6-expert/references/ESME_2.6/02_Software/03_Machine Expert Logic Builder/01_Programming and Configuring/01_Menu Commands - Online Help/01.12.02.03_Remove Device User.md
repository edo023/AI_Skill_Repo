# Remove Device User

## Overview

The Online > Security > Remove device user... command is only available if you are logged in on the target system (controller). It serves to remove a user from the user management on this target system. The same dialog box is displayed as for adding a new user when executing the Add device user [command](D-SE-0084013.html#D-SE-0084013). Select the name of the user to be removed and click OK.

NOTE: You can handle the online user management via the Users and Groups view of the device editor ([**Users and Groups**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083876)). The commands of the menu Online > Security provide an additional simplified opportunity to protect the access on the target device.

NOTE: After you have executed this command, the user cannot log in anymore with this user account.

If the user is the last specified on the target, a message is displayed, indicating that this user cannot be removed.
