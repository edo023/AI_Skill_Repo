# Compiler Error C0013

## Message

At least one statement is expected

## Message Cause

At least one statement is expected at some positions in the code. For example, in the THEN and ELSE part of an IF statement or in the body of a FOR loop.

## Solution

Add at least one statement at the selected position. You can write a blank statement ”;”.

## Error Example

```
PROGRAM PLC_PRG
VAR
  bTest: BOOL;
END_VAR
```

```
IF bTest THEN
END_IF
```
