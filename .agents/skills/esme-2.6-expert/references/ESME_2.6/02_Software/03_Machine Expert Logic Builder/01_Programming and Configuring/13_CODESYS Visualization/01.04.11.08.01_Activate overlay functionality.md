# Activate overlay functionality

IMPORTANT:

You can execute a visualization, which should be displayed as a Remote TargetVisu variant on a Linux system, with the overly functionality only if your device supports this. For more information regarding this, contact your device manufacturer.

1. In your project, in the **Visualization Manager** editor, in the **General Settings** group, select the **Support client animations and overlay of native elements** option.
2. Adapt the configuration file of your **Remote TargetVisu** as follows:

   `[CmpVisuHandlerRemote]`

   `VisuClient.OverlayMode=1`

   * The **Overlay** functionality is activated.
