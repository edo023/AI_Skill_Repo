# Compiler Error C0041

## Message

VAR\_IN\_OUT parameter ‘<parameter name>’ of ‘<function name>’ needs variable with write access as input

## Message Cause

The passed parameter is not a variable with write access (but a constant for example).

## Solution

Pass a VAR\_IN\_OUT parameter with write access to the function.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  x : INT;
END_VAR
```

```
i := Test(31415);
```

```
FUNCTION TEST : INT
VAR_IN_OUT
  in_out: INT;
END_VAR
```

--> C0041: VAR\_IN\_OUT' parameter 'in\_out' of 'TEST' needs variable with write access as input

## Error Correction

```
i := Test(x);
```
