# Send to Front

## Overview

The CFC > Execution Order > Send to Front command affects that the selected elements will be moved to the front of the [execution order](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083494). The order within the group of selected as well as of the unselected elements is maintained. Also the order within the not selected elements will not be changed.
