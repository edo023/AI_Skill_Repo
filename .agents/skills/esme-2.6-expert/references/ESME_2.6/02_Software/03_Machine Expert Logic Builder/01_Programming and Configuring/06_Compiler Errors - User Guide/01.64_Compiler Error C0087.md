# Compiler Error C0087

## Message

There is no implementation for the method ‘<method name>’ defined in the interface ‘<interface name>’

## Message Cause

One of the methods specified by the interface has not been provided by the implemented function block.

## Solution

Implement all the methods that are specified by the interface.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : FB;
END_VAR
```

```
INTERFACE XY
METHOD METH1
VAR_INPUT
END_VAR
```

```
FUNCTION_BLOCK FB IMPLEMENTS XY
VAR
END_VAR
METHOD METH2
VAR_INPUT
END_VAR
```

--> C0087: There is no implementation for the method 'METH1' defined in the interface 'XY'

EIO0000003933.04
