# Object: Visualization

Symbol: ![](images/uuid-3ec4e119-aa58-69ee-0aca-957470.png)

The object represents a single visualization. You can insert a visualization below an application, or project-wide available below the root node of the view **Devices** or directly in the view **POUs**. Double-click on the object entry in the device tree or in the POUs view to open the visualization editor for editing.
