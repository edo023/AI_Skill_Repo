# General Information

## Overview

EcoStruxure Machine Expert and the supported devices are continuously improved. Therefore, new updates are released on a regular basis.

EcoStruxure Machine Expert provides, in most cases, a simple, and straight forward way to migrate projects created with previous versions to the present version.

NOTE:

* Launch the Schneider Electric Software Update (SESU) regularly to obtain the latest updates.
* Launch the Schneider Electric Software Installer to obtain information on new features available for installation.
* Also visit regularly the Schneider Electric [website](https://www.se.com) to download the latest device firmware available.

In specific situations, you may encounter compatibility concerns regarding these updates, and actions are required to establish consistency with your existing systems and project files for the following elements:

* Software files created with a previous version, for example, projects, archives, exports
* Project archives created with a previous EcoStruxure Machine Expert version can be opened.
* Projects or project archives converted to the file-based storage format.
* Projects or project archives created with EcoStruxure Machine Expert imported using the Machine Expert.
* Device firmware versions (for example, controllers, HMI terminals)
* Related software (for example, Vijeo-Designer)

## EcoStruxure Machine Expert Software

By installing a new software version, new component versions are available, which include:

| Component | Description |
| --- | --- |
| Compiler version | The compiler is the software that converts your project into the binary code that is downloaded and executed on the controller.  The compiler is called when you execute a Generate Code or Generate Code for all Applications command. The version of the compiler of your software version can be displayed by selecting: Project > Project Settings > Compile options. The latest version is selected and cannot be modified. |
| Device description version | The device description defines the properties of a device such as configurability, programmability, and possible connections to other devices.  When you insert a device inside a project, the software uses the device description to define the properties of the device.  The device description version is displayed in the Information [tab of the device editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083396). |
| Library version | A library is a container of variable lists, data structures (DUTs), functions, and function blocks (POU) that can be used in your projects. The versions of libraries used in an application are visible in the [**Library Manager**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoLibref&topicID=D_SE_0081220). |

The following types of project files are supported:

| File type | Description |
| --- | --- |
| *\*.project*  *\*.library*  *\*.fbs* | The version of a project/library file is determined by the project format version and the project categories.   * Project format version:  The version of the project or library file is defined by the version of EcoStruxure Machine Expert that saved this project or library file. It acts as a container that specifies the storage format for the content of the application. * Project categories:  Defines the versions of device descriptions, the versions of the libraries used, the compiler version, the visualization profile, and the visualization styles versions (refer to *Visualization* part of the online help). The project categories apply to the applications of the project/library.   NOTE: Files with the extensions  \*.project and \*.library are in binary file storage format. Files with the extension \*.fbs are in file-based storage format. |
| *\*.projectarchive* | A project archive can be created by using EcoStruxure Machine Expert and be saved on your PC as a *\*.projectarchive* file. This file contains one project and can contain download information files, referenced devices, referenced libraries, the visualization profile, and the visualization styles.  You can also do a source download to create an archive as an *Archive.prj* file in your controller. |

## Device Firmware

The firmware is the software embedded in the controller.

The firmware version is displayed in the Logic Builder Communication Settings [tab of the device editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083385) associated to your controller.

## Vijeo-Designer Software

The Vijeo-Designer software allows you to create your HMI applications.

The EcoStruxure Machine Expert installation includes the installation files of Vijeo-Designer and can be installed or updated without the EcoStruxure Machine Expert software, but cannot be used without the EcoStruxure Machine Expert software.

For compatibility information with Vijeo-Designer, refer to the section [*Compatibility of EcoStruxure Machine Expert and Vijeo-Designer Versions*](D-SE-0088871.html#D-SE-0088871).

## EcoStruxure Operator Terminal Expert Software

The EcoStruxure Operator Terminal Expert software allows you to create your HMI applications.

EcoStruxure Operator Terminal Expert can be installed from the Schneider Electric Software Installer by clicking Install new software and selecting the product EcoStruxure Operator Terminal Expert.

For compatibility information with EcoStruxure Operator Terminal Expert, refer to the section [Compatibility of EcoStruxure Machine Expert and EcoStruxure Operator Terminal Expert Versions](TPC_CompMigr_Compatibility_EOTE-952AFF28.html).

## EcoStruxure Machine Expert Twin Software

The EcoStruxure Machine Expert Twin software allows you to create simulations of your mechatronic objects.

The EcoStruxure Machine Expert installation includes the installation files of EcoStruxure Machine Expert Twin. EcoStruxure Machine Expert Twin can be used with EcoStruxure Machine Expert license.

NOTE: EcoStruxure Machine Expert Twin can also be installed as a standalone version from the Schneider Electric Software Installer by clicking Install new software and selecting the product EcoStruxure Machine Expert Twin. In this case, a separate license is required.

For compatibility information with EcoStruxure Machine Expert Twin, refer to the section [Compatibility of EcoStruxure Machine Expert and EcoStruxure Machine Expert Twin Versions](CompTwin-65CC0217.html).
