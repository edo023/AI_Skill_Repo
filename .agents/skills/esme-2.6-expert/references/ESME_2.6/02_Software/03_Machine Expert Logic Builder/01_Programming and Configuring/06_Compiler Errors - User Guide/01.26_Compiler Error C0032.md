# Compiler Error C0032

## Message

Cannot convert type ‘STRING’ to type ‘INT’

## Message Cause

A variable is assigned to another variable with an incompatible type.

## Solution

Use a type conversion function.

## Error Example

```
PROGRAM PLC_PRG
VAR
  test1: INT;
  test2: STRING;
END_VAR
```

```
test1 := test2;
```

-->C0032: Cannot convert type ‘STRING’ to type ‘INT’

## Error Correction

```
test1 := STRING_TO_INT(test2);
```
