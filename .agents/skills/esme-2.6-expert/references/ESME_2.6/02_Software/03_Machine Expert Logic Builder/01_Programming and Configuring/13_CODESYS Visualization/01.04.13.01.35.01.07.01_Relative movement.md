# Relative movement

The properties contains variables for moving the element. The reference point is the position of the element (**Position** property). The shape of the element can change.

|  |  |
| --- | --- |
| ****Movement point[0]****   * **X** * **Y** | Variable (numeric data type) for the number (in pixels) which the starting point of the line is moved.  Incrementing the X value moves the element to the right.  Incrementing the Y value moves the element to the down. |
| ****Movement point[0]****   * **X** * **Y** | Variable (numeric data type) for the number (in pixels) that the end point of the line is moved.  Incrementing the X value moves the element to the right.  Incrementing the Y value moves the element to the down. |
