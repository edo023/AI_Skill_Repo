# Travel commands and corresponding path elements

| Travel Command | Description | Path Element |
| --- | --- | --- |
| `G0` | Direct movement without tool operation; linear motion | [Positioning](_sm_cnc_din66025_positioning.html#_sm_cnc_din66025_positioning) |
| `G1` | Linear movement with tool operation | [Linear Motion](_sm_cnc_din66025_line.html#_sm_cnc_din66025_line) |
| `G2` | Circular segment or circle, clockwise | [Arc](_sm_cnc_din66025_arc.html#_sm_cnc_din66025_arc) |
| `G3` | Circular segment or circle, counterclockwise | [Arc](_sm_cnc_din66025_arc.html#_sm_cnc_din66025_arc) |
| `G4` | Dwell Time | [Dwell Time](_sm_cnc_din66025_delay.html#_sm_cnc_din66025_delay) |
| `G5` | Point of a 2D cardinal spline | [Spline](_sm_cnc_din66025_spline.html#_sm_cnc_din66025_spline) |
| `G6` | Parabola | [Parabola](_sm_cnc_din66025_parabel.html#_sm_cnc_din66025_parabel) |
| `G8` | Elliptical arc or ellipse, clockwise | [Ellipse](_sm_cnc_din66025_ellipse.html#_sm_cnc_din66025_ellipse) |
| `G9` | Elliptical arc or ellipse, counterclockwise | [Ellipse](_sm_cnc_din66025_ellipse.html#_sm_cnc_din66025_ellipse) |
| `G10` | Point of a 3D cardinal spline | [Spline](_sm_cnc_din66025_spline.html#_sm_cnc_din66025_spline) |
| `G15` | Switch to 2D | [3D Mode](_sm_cnc_din66025_3dmode.html#_sm_cnc_din66025_3dmode) |
| `G16` | Switch to 3D by activating 3D mode with the normal I/J/K vector to the plane | [3D Mode](_sm_cnc_din66025_3dmode.html#_sm_cnc_din66025_3dmode) |
| `G17` | Switch to 3D by activating 3D mode in the X/Y plane | [3D Mode](_sm_cnc_din66025_3dmode.html#_sm_cnc_din66025_3dmode) |
| `G18` | Switch to 3D by activating 3D mode in the Z/X plane | [3D Mode](_sm_cnc_din66025_3dmode.html#_sm_cnc_din66025_3dmode) |
| `G19` | Switch to 3D by activating 3D mode in the Y/Z plane | [3D Mode](_sm_cnc_din66025_3dmode.html#_sm_cnc_din66025_3dmode) |
| `G20` | Conditional jump to `L`, if `K` <> 0 | [Jump](_sm_cnc_din66025_jump.html#_sm_cnc_din66025_jump) |
| `G31` | Probing function (clear remaining distance) with decoding stop | [Probing function (clear remaining distance)](_sm_probe_movement.html#_sm_probe_movement) |
| `G36` | Write value `D` to variable `O` | [Changing the Values of Variables](_sm_cnc_din66025_change_variable_values.html#_sm_cnc_din66025_change_variable_values) |
| `G37` | Increment variable `O` by value `D` | [Changing the Values of Variables](_sm_cnc_din66025_change_variable_values.html#_sm_cnc_din66025_change_variable_values) |
| `G38` | Activate the extended preprocessing function | [Activating the Extended Preprocessing Function](_sm_cnc_din66025_activating_extended_preprocessor_functions.html#_sm_cnc_din66025_activating_extended_preprocessor_functions) |
| `G39` | Deactivate the extended preprocessing Function | [Activating the Extended Preprocessing Function](_sm_cnc_din66025_activating_extended_preprocessor_functions.html#_sm_cnc_din66025_activating_extended_preprocessor_functions) |
| `G40` | End of tool radius correction | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G41` | Start of tool radius correction, left of travel direction | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G42` | Start of tool radius correction, right of travel direction | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G43` | Starts the tool length correction | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G50` | End of corner rounding/smoothing | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G51` | Start of corner smoothing | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G52` | Start of corner rounding | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G53` | Ends the coordinate transformation and resets the decoder coordinate system to the original position (= machine coordinate system) | [Shifting, Rotating, and Scaling the Coordinate System](_sm_cnc_din66025_coordinate_shift.html#_sm_cnc_din66025_coordinate_shift) |
| `G54` | Absolute transformation of the coordinates | [Shifting, Rotating, and Scaling the Coordinate System](_sm_cnc_din66025_coordinate_shift.html#_sm_cnc_din66025_coordinate_shift) |
| `G55` | Relative transformation of the coordinates | [Shifting, Rotating, and Scaling the Coordinate System](_sm_cnc_din66025_coordinate_shift.html#_sm_cnc_din66025_coordinate_shift) |
| `G56` | Sets the current orientation, position, and scaling of the DCS is set as a reference point | [Shifting, Rotating, and Scaling the Coordinate System](_sm_cnc_din66025_coordinate_shift.html#_sm_cnc_din66025_coordinate_shift) |
| `G60` | End of loop suppression | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G61` | Start of loop suppression | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G70` | End of smoothing additional axes  See: `SMC_SmoothAddAxes` | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G71` | Start of smoothing additional axes  See: `SMC_SmoothAddAxes` | [Preprocessing](_sm_cnc_din66025_preprocessor.html#_sm_cnc_din66025_preprocessor) |
| `G75` | Timing synchronization with the interpolator | [Timing Synchronization with Interpolator](_sm_cnc_din66025_time_synchronisation_with_interpolator.html#_sm_cnc_din66025_time_synchronisation_with_interpolator) |
| `G90` | The coordinates (`X`/`Y`/`Z`/`A`/`B`/`C`/`P`/`Q`/`U`/`V`/`W`) are interpreted as absolute values. (This is the default setting.) | [Modes](_sm_cnc_din66025_modi.html#_sm_cnc_din66025_modi) |
| `G91` | The coordinates (`X`/`Y`/`Z`/`A`/`B`/`C`/`P`/`Q`/`U`/`V`/`W`) are interpreted as values relative to the current position. | [Modes](_sm_cnc_din66025_modi.html#_sm_cnc_din66025_modi) |
| `G92` | Positioning by jump | [Positioning](_sm_cnc_din66025_positioning.html#_sm_cnc_din66025_positioning) |
| `G98` | The axis midpoints (`I`/`J`/`K`) are interpreted as absolute values. | [Modes](_sm_cnc_din66025_modi.html#_sm_cnc_din66025_modi) |
| `G99` | The axis midpoints (`I`/`J`/`K`) are interpreted as values relative to the start position. (This is the default setting.) | [Modes](_sm_cnc_din66025_modi.html#_sm_cnc_din66025_modi) |
