# Command: Disconnect Project from Subversion

Symbol: ![](images/uuid-19492a01-613c-9b5c-16b7-a3275d.png)

**Function**: The command deletes all connections of the current project to SVN by converting the project into a non-versioned project.

**Call**: **Project → SVN** menu

TIP:

Because this operation cannot be undone, the operation has to be confirmed before the command is executed.

TIP:

Use the [Connect to existing project](_svn_cmd_connect_to_existing_project.html#_svn_cmd_connect_to_existing_project) command to connect to the SVN repository again at a later time.
