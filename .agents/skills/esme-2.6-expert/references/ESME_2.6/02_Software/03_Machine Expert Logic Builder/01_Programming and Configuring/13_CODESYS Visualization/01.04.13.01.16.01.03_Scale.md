# Scale

|  |  |
| --- | --- |
| **Show scale** | : The element has a visible scale.  NOTE: This property is available for the slider only. |
| **Scale start** | Least value of the scale and the lower limit of the value range for the element  Example: 0  TIP: In the value field, click the  symbol. Then the property is extended by the **Variable** property. |
| **Variable** | Variable (integer data type) for the scale start  Example: `PLC_PRG.iScaleStart` |
| **Scale end** | Greatest value of the scale and the upper limit of the value range for the element  Example: 100  TIP: In the value field, click the  symbol. Then the property is extended by the **Variable** property. |
| **Variable** | Variable (integer data type) for the scale end  Example: `PLC_PRG.iScaleEnd` |
| **Main scale** | Distance between two tick marks on the main scale.  Example: 10  TIP: In the value field, click the  symbol. Then the property is extended by the **Variable** property. |
| **Variable** | Variable (integer data type) for the spacing  Example: `PLC_PRG.iMainScale` |
| **Sub scale** | Distance between two tick marks on the fine scale  You can hide the fine scale by setting the value to 0.  Example: 2  TIP: In the value field, click the  symbol. Then the property is extended by the **Variable** property. |
| **Variable** | Variable (integer data type) for the spacing  Example: `PLC_PRG.iSubScale` |
| **Scale format (C syntax)** | Formatting the scale label  Example: `%d %s`  NOTE: This property is available for the slider only. |
| **Scale proportion** | Size of the scale (in %) of the total size |

NOTE:

A valid declaration in the application code is required for the variables used as an example in the table above.
