# Declaration Editor

## Overview

The Tools > Options > Declaration editor dialog box contains some default settings on the appearance of the [**declaration editor**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083518).

Define, which views of the declaration part should be available when an object is opened for editing.

|  |  |
| --- | --- |
| Textual only | – |
| Tabular only | – |
| Switchable between textual and tabular | If this setting is activated, 2 buttons will be available in the declaration editor window for switching between the textual and the tabular view of the declaration part. Choose one of the settings which defines where in both formats by Default the declaration editor at first will appear when an object is opened for editing:   * Always textual * Always tabular * Remember recent setting (per object): When reopening an object, the declaration editor will be opened in the same format as it had for this object during last editing phase. * Remember recent setting (globally): When opening any object, the declaration editor format recently used in any object will be used. |
