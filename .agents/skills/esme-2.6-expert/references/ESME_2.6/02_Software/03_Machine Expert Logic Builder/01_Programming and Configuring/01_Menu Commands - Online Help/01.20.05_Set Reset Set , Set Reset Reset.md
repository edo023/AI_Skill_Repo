# Set/Reset > Set, Set/Reset > Reset

## Overview

|  |  |
| --- | --- |
| Symbols | Set:  Reset: |
| Shortcut | You can use the keyboard combination Ctrl + M to toggle between Set, Reset, and no modifier on the selected element. |
| Call | * Ladder > Set/Reset > Set menu * Ladder > Set/Reset > Reset menu * Contextual menu |

## Function

Run the Set command to assign a `Set` modifier to the element selected in the editor. Run the Reset command to assign a `Reset` modifier to the element selected in the editor.

The command is only available at suitable positions.

## Examples

You can run the Set command on a [coil](D-SE-0084116.html) and a non-primary block output to convert it into a `set coil`.
