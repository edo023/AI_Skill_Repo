# Structured Text ST

## Overview

Structured Text is a textual high-level programming language, similar to PASCAL or C. The program code is composed of [expressions](D-SE-0083514.html#D-SE-0083514) and [instructions](D-SE-0083515.html#D-SE-0083515). In contrast to IL (Instruction List), you can use numerous constructions for programming loops, thus allowing the development of complex algorithms.

## Example

```
IF value < 7 THEN
 WHILE value < 8 DO
  value:=value+1;
 END_WHILE;
END_IF;
```
