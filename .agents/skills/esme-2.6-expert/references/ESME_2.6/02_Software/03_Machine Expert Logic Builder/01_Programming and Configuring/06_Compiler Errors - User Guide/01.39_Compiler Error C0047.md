# Compiler Error C0047

## Message

Cannot apply indexing with ‘[]’ to an expression of type ‘<data type>’

## Message Cause

A data type that is not an array is indexed with ‘[]’.

## Solution

Index data types with ‘[]’ only if they are declared as arrays.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
i[1];
```

--> C0047: Cannot apply indexing with '[]' to an expression of type 'INT'
