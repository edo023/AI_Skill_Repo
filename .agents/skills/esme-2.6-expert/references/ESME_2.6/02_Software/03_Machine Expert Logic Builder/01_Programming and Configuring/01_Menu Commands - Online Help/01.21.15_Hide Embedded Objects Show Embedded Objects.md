# Hide Embedded Objects / Show Embedded Objects

## Overview

The Hide Embedded Objects / Show Embedded Objects commands are available in the contextual menu of an SFC POU node in the Applications tree that contains embedded step or transition elements.

Execute the Hide Embedded Objects command to hide the subnodes of embedded step or transition elements below the SFC POU node in the Applications tree.

Execute the Show Embedded Objects command to show the subnodes of embedded step or transition elements below the SFC POU node in the Applications tree.
