# Save Project and Install Into Library Repository / Install into Library Repository

## Overview

The command Save Project and Install Into Library Repository is exclusive to \*.library files.

The command Install into Library Repository is exclusive to \*.fbslib files.

Use the commands to save a library and to install it into the System location of the Library Repository. The library is installed on the local system and thus is available for insertion in a project using the Library Manager [editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoLibref&topicID=D_SE_0081233).
