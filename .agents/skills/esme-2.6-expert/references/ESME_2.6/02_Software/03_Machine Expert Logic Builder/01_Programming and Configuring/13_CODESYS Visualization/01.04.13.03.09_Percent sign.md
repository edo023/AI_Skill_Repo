# Percent sign

Format definition for output of a percent sign

| Syntax | Example | Description | Example in Visualization |
| --- | --- | --- | --- |
| `&` | `%d%%` | Output of the percent sign | With a placeholder variable:  Code: `iPercentage : INT := 80;`  **Text** property: `Valid until %d%%.`  **Text variable** property: `PLC_PRG.iPercentage := 80;`  Output: `Valid until 80%` |
|  |  |  | **Text** property: `Valid until 90%%`  Output: `Valid until 90%` |
