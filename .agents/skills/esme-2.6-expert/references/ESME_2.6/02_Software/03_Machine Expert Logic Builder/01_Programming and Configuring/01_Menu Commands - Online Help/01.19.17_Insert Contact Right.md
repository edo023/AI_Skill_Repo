# Insert Contact Right

## Overview

Shortcut: CTRL + D

The FBD/LD/IL > Insert Contact Right command inserts a contact in an LD network. The same is valid as for command [**Insert Contact**](D-SE-0084119.html#D-SE-0084119), except that the new element will not be inserted left but right to the current cursor position in line.
