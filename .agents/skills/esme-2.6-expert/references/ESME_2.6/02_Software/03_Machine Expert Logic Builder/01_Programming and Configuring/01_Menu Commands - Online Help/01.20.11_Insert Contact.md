# Insert Contact

## Overview

|  |  |
| --- | --- |
| Symbol | In the editor: |
| Shortcut | Ctrl + K |
| Call | * Ladder > Insert Contact menu * Contextual menu |

## Function

The command inserts a contact in the network, to the left of the selected element. Possible insertion positions are indicated by the plus symbol at the cursor when you drag the element from the Toolbox to the implementation part.

## Requirements

A line or a contact is selected.

## Element: Contact

A contact passes the signal TRUE (ON) or FALSE (OFF) from left to right until the signal reaches a coil on the right side of the network. For this purpose, a Boolean variable containing the signal is assigned to the contact. To do this, replace the (`???`) placeholder above the contact with the name of a Boolean variable.

You can arrange multiple contacts both in series and in parallel. In the case of parallel contacts, only one must be set to TRUE in order for ON to be passed on to the right. When contacts are connected in series, all of them must be set to TRUE in order for ON to be passed on to the right by the last contact in the series. Thus, you can use LD to program parallel and series connections.

A negated contact ![](images/DD00896622.png) passes the TRUE signal when the variable value is FALSE. Use the [Ladder > Negate command](Negate-451D3E07.html) to enable or disable negation of an inserted contact.
