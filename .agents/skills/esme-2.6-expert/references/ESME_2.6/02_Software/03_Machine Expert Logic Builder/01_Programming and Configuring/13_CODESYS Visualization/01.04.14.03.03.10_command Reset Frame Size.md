# Command: Reset Frame Size

**Function**: This command resizes the selected frame element to the currently referenced image size.

**Call**: **Visualization** menu; context menu

**Requirement**: A [frame element](_visu_elem_frame.html#_visu_elem_frame) is selected.

TIP:

Reset the size only if the frame element should have the same size and the same scaling as the referenced visualization.
