# Context menu in the alarm table

Commands in the context menu

|  |  |
| --- | --- |
| **Add message column** | Adds a new message column |
| **Remove message column** | Removes the selected message column |
| **Add latch var column** | Inserts a latch variable. |
| **Remove latch var column** | Removes the selected variable |

TIP:

Multiple selected rows in the table can be edited at the same time by means of the context menu.

Commands for exporting and importing alarms to and from a CSV file

|  |  |
| --- | --- |
| **Export Alarms** | The command opens the default dialog for saving a file in the file system. The default file type is **Alarm Group Export File (\*.csv)**. The current alarm list can be exported to a CSV file there. |
| **Import Alarms** | The command opens the default dialog for searching for a file in the file system. The default file type is **Alarm Group Export File (\*.csv)** for describing alarm definitions. The selected file is imported. |
