# Command Icons

## Overview

The Command Icons tab of the Tools > Customize  dialog box allows you to define icons for the menu commands:

The dialog box shows the available Categories with the assigned Commands.

The following buttons are available:

| Button | Description |
| --- | --- |
| Assign | Opens a dialog box for selecting the new icon (\*.ico). |
| Remove | Removes the icon. The default icon is active again. |
| Reset | Resets the default settings of the command icons. |
| Load... | Loads the settings from a stored file (<file name>.opt.menu). |
| Save... | Saves the settings to a file (<file name>.opt.menu). |
