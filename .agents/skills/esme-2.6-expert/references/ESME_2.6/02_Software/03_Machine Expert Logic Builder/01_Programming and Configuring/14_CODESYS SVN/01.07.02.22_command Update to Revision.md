# Command: Update to Revision

Symbol: ![](images/uuid-8aa642bf-e534-0456-d3db-75f459.png)

**Function**: The command opens the **Update** dialog. In the dialog, the revision is defined for updating the project.

**Call**:

* **Project → SVN**
* **Context menu → SVN**

If you select nothing or the base node in the object tree, then the entire project is updated to a revision (**Update project to revision**). If you select one or more objects, then these objects are updated and their sub-objects are updated recursively (**Update to revision**). Optionally, you could define that the sub-objects are not updated.

The behavior of the updating process (for example, merging of conflicts) is similar to the **Update project** and **Update** commands.
