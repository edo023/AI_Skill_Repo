# Compiler Error C0131

## Message

‘<value>’ is not allowed as operand for ‘ADR’

## Message Cause

A constant is passed as an operand to the operator ADR.

## Solution

Only use valid operands for ADR.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  pt : POINTER TO INT;
END_VAR
```

```
pt := ADR(1);
```

--> C0131: '1' is not allowed as operand for 'ADR'

## Error Correction

```
pt := ADR(i);
```
