# Dialog: Revert

|  |  |
| --- | --- |
| **Group externals** | : The external definitions are displayed in groups according to their external locations. |
| **Keep Locks** | : The lock is retained for all files that are modified by the revert command. |
| **Select/Deselect All** |  |

When an external is deleted, CODESYS SVN cannot restore this data in SVN offline mode. The user is prompted for how to continue:

* Switch back to SVN online mode and call the externals.
* Connect now to the SVN server one time in order to complete the current operation, but afterwards switch back to SVN offline mode.
* Skip the retrieval of the externals. They can be fetched later by updating the project.
