# Compiler Warning C0355

## Message

A single bit cannot be referenced. A reference to the complete byte will be stored.

## Message Cause

An attempt is made to determine the address of a bit.

## Error Example

```
FUNCTION_BLOCK FBVAR
      pt : POINTER TO BOOL;
      b : BIT;
END_VAR
pt := ADR(b);
```
