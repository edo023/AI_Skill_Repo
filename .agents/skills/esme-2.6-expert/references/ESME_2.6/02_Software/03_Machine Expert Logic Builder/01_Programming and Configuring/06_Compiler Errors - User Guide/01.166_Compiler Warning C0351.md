# Compiler Warning C0351

## Message

The attribute <attribute name> is unknown and will be ignored by the compiler.

## Message Cause

There is a misspelling in an attribute name.

## Solution

If you are using a predefined attribute, ensure that the attribute is spelled correctly (in the example: `call_after_init)`.

## Error Example

```
{attribute 'call_after_ini'}
PROGRAM PLC_PRG
VAR
END_VAR
```

--> C0351: The attribute call\_after\_ini is unknown and will be ignored by the compiler.
