# CNC Example 16: Probing function (clear remaining distance) (G31)

See the `CNC16_G31.project` sample project in the installation directory of CODESYS under `..\CODESYS SoftMotion\Examples`.

The sample project shows how to use the probing function (clear remaining distance).

In the example, the machine moves in the X direction until a light barrier is reached at `X=50`. As soon as the light barrier has been reached, the machine has to stop. The remaining G-Code has to be processed starting from this position.
