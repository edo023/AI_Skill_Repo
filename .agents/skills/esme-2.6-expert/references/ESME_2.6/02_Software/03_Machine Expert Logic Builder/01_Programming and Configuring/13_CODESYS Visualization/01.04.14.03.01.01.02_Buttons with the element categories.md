# Buttons with the element categories

A button is displayed for each defined element category. When the element category is enabled (green), the assigned elements are displayed in the visualization element view.

|  |  |
| --- | --- |
| **Shift** + click on the category | Toggles between single selection and multiselection |
| Right click a category | Opens the context menu |

**Context menu of a button**

|  |  |
| --- | --- |
| **Disable category** | Hides the button |
| **Enable Category** | Filters by the active category (regardless of the selection type) and displays the elements of this category in the visualization element view |
| **Disable Category** | Hides the elements of this category in the visualization element view |
