# Compiler Error C0027

## Message

Size of string expected after ‘(‘

## Message Cause

The length of the string is not specified.

## Solution

Specify a string length between the parentheses.

## Error Example

```
PROGRAM PLC_PRG
VAR
  str : STRING();
END_VAR
```

--> C0027: size of string expected after '('

--> C0006: ';, :=, REF=, ( or [' expected instead of ')'

## Error Correction

```
str : STRING(100);
```
