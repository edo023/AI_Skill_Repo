# SVN Settings / SVN Settings (Advanced)

## Overview

For a description of the settings provided in the Tools > Options > SVN Settings dialog box, refer to the Software > Programming and Configuring > CODESYS SVN part of the online help.

The Tools > Options > SVN Settings (Advanced) dialog box provides the option Execute cleanup on project save. The option is by default disabled because an SVN cleanup can be time-consuming for large projects as it performs the following operations:

* Recursively cleaning the working copy.
* Removing working copy locks.
* Resuming interrupted operations.

To perform an SVN cleanup with every save operation, activate the Execute cleanup on project save option.
