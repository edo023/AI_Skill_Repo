# Dependency View (Filters)

## Overview

| Element | Description |
| --- | --- |
| Select filter for displayed graph | Select a filter from the list. The filters are defined in the Dependency (Filter) Queries tab of the Code Analysis Query Manager.  Refer to chapter [Dependency (Filter) Queries Tab](D-SE-0070589.html#D-SE-0070589). |
| Apply | Click this button to apply the filter to the displayed graph. |
| Types Quick Filters | This tree lists the Edge Types and Node Types based on the content in the Dependency Graph.  If you activate / deactivate Edge Types or Node Types in the list, the displayed graph is updated immediately. |
| Edge visualization options | * Show edge labels  Enables / disables the labels attached to the edges inside the Dependency Graph. * Display parallel Edges  Enables / disables to display parallel edges. |
