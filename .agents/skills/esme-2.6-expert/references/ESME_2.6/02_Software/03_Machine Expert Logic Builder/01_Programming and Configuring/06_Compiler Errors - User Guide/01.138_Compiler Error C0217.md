# Compiler Error C0217

## Message

Case label <case label> is also contained in range <case range begin> .. <case range end>

## Message Cause

A case label is part of the range of another case label.

## Solution

Make sure that there is no intersecting.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
CASE i OF
  3..5: i := i+2;
  4: i := i+2;
ELSE
  i := i+10;
END_CASE;
```

--> C0217: Case label 4 is also contained in range 3 .. 5
