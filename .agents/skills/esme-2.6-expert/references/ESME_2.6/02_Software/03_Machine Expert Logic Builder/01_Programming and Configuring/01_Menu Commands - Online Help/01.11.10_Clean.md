# Clean

## Overview

The Build > Clean command deletes the compile information for the application. The compile information was created and stored in a *\*.compileinfo* file during the last [**download**](D-SE-0083996.html#D-SE-0083996) of the application.

After a clean process, no [online change](D-SE-0083997.html#D-SE-0083997) will be possible for the respective application. The program has to be downloaded again.
