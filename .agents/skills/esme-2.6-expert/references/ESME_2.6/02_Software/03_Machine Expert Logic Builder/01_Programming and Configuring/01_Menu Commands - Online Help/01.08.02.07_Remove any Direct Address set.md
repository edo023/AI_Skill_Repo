# Remove any Direct Address set

## Overview

Right-click a device node in the Devices tree, and execute the Remove any Direct Address set command from the contextual menu to remove the direct addresses configured for the selected device and the related subdevices. For further information, [refer to the Programming Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083399_4).
