# Structure of the application

The application consists of a simple state machine in the `PLC_PRG` program and a trace which you can use to easily track the functionality of the function blocks.

* Enable the axes with `MC_Power`. `(STATE_POWER`)
* Start the synchronous movements and move the master axis continuously between position 0 and position 100 (`STATE_COMMAND_POS_0` and `STATE_COMMAND_POS_1`).
