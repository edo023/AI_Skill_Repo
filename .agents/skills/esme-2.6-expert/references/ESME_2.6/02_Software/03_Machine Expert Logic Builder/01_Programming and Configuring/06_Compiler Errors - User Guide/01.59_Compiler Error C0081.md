# Compiler Error C0081

## Message

Unexpected Pragma: ‘<pragma name>’ found without matching ‘if’

## Message Cause

The IF condition is missing when using the pragma.

## Solution

Complete the IF condition of the pragma.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
i := 5;
{END_IF}
```

--> C0081: Unexpected Pragma: 'END\_IF' found without matching 'if'

## Error Correction

```
{IF defined (i)}
i := 5;
{END_IF}
```
