# Compiler Error C0208

## Message

‘MOD’ is not defined for ‘REAL’

## Message Cause

An attempt is made to perform a modulo operation with a REAL type variable.

## Solution

Modulo operations are only possible with ANY\_INT type variables.

## Error Example

```
PROGRAM PLC_PRG
VAR
  r1 : REAL;
END_VAR
```

```
r1 := r1 MOD 2;
```

--> C0208: 'MOD' is not defined for 'REAL'
