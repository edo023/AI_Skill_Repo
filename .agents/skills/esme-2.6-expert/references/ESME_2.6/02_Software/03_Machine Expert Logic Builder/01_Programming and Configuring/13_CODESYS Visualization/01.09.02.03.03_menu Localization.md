# Menu: Localization

The commands affect the contents of the [Tab: Localization](_visu_html5_tab_localization.html#_visu_html5_tab_localization) tab.

|  |  |
| --- | --- |
| **Add Language** | Opens the **Add New Language** dialog  The dialog prompts you to enter the translations for the names of the properties in the new language column. |
| **Remove selected language** | Requirement: A cell is in focus.  Removes the column where the selected cell is located |
| **Rename selected language** | Opens the **Rename Language** dialog  Requirement: A cell is in focus.  The column where the cell is selected is renamed. This removes all previous translations. |

Dialog: **Add new language**

|  |  |
| --- | --- |
| **Name** | Name of the new language as a language code according to ISO 639-1  Examples: `de`, `en`, `es`, `it`, `fr`, `ja`, `ch-ZHS` |
| **Copy from existing** | All existing language columns are available for selection. The selected language is copied with all entered translations.  **<do not copy text>**: The new language receives a blank translation column. |
