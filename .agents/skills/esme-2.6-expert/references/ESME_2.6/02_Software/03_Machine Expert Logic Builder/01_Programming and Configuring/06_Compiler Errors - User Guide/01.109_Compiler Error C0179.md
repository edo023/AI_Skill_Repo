# Compiler Error C0179

## Message

‘<identifier>’ is no input of ‘<function block>’.

## Message Cause

The initialization of a function block instance must not contain VAR\_IN\_OUT variables.

## Solution

Use VAR\_IN\_OUT variables in function block calls only. When initializing a function block instance, only assign the inputs of a function block.

## Error Example

```
FUNCTION_BLOCK MyFB
VAR_IN_OUT
  inOut : INT;
END_VAR
```

```
PROGRAM PLC_PRG
VAR
    iValue : INT;
    fb : MyFB := (inOut := iValue);
END_VAR
```

-->C0179: ‘inOut’ is no input of ‘MyFB’.
