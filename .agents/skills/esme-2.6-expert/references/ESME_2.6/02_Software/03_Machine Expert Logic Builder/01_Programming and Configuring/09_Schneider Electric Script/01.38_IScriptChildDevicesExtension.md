# IScriptChildDevicesExtension

class

IScriptChildDevicesExtension.

IScriptChildDevicesExtension

Bases: object

Interface to get the child devices that can be appended to a device or project.

get\_allowed\_child\_devices

filterNewestVersions=True

Retrieves the child devices that can be appended to this device or project.

Parameters

filterNewestVersions (bool) -- If set to TRUE, only the newest version of each device is returned. (This parameter is optional, default = TRUE)

Returns

A device collection of possible child devices.

Return type

IScriptDeviceCollection

**Note**

Each entry in the list represents an IScriptDeviceDescription object. The members of this object can be found in CODESYS API description.

This example shows how to get a list with the allowed child devices from a device.

```
# Script creates an empty project and prints the possible devices that can be appended on top level
# We enable the new python 3 print syntax
from __future__ import print_function

import os

## Close project
if projects.primary:
	projects.primary.close()

## Create an empty project
proj = projects.create(os.path.expanduser("~\\Documents\\GetAllowedChildDevices.project"))

# Define the printing function
def printdevices(devicelist):
	# prints the devices
	for device in devicelist:
		print("Name: " + device.device_info.name + " Version: " + device.device_id.version)

# Get the possible devices on project level
devicelist = proj.get_allowed_child_devices()

# Call the printdevices function
printdevices(devicelist)
```
