# Compiler Warning C0543

## Message

The name <keyword> is a reserved keyword in the IEC 1131-3 standard. An error will be issued in future versions.

## Message Cause

A reserved keyword was assigned as the name of a variable.

NOTE: When violations are detected in compiled libraries, a text message (type Information) is issued instead of an alert (type Compiler Warning).

The following keywords are reserved:

* CHAR
* WCHAR
* ANY\_DERIVED
* ANY\_ELEMENTARY
* ANY\_MAGNITUDE
* ANY\_SIGNED
* ANY\_DURATION
* ANY\_CHARS
* CHAR\_TO
* TO\_CHAR
* WCHAR\_TO
* TO\_WCHAR
* ATAN2
* USING
* CLASS

## Solution

Rename the variable and use a name that is not a reserved keyword.

## Error Example

```
PROGRAM PLC_PRG
VAR
    char : BYTE;
END_VAR
```

--> C0543: The name 'char' is a reserved keyword in the IEC 1131-3 standard. An error will be issued in future versions.

## Error Correction

Rename the variable char to a non-keyword.
