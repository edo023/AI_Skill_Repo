# Compiler Error C0178

## Message

No external access to ‘VAR\_IN\_OUT’ parameter ‘<parameter name>’ of ‘<object name>’

## Message Cause

An attempt is made to access a ‘VAR\_IN\_OUT’ parameter remotely.

## Solution

Do not access ‘VAR\_IN\_OUT’ parameters remotely.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : FB;
  i : INT;
END_VAR
```

```
i := inst.in_out;
```

```
FUNCTION_BLOCK FB
VAR_IN_OUT
  in_out : INT;
END_VAR
```

--> C0178: No external access to 'VAR\_IN\_OUT' parameter 'in\_out' of 'FB'

EIO0000003933.04
