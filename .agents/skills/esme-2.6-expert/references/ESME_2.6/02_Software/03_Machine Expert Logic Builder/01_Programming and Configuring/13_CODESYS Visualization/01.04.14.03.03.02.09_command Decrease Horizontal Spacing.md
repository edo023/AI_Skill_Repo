# Command: Decrease Horizontal Spacing

Symbol: ![](images/uuid-7d6be400-9a8d-7dea-bc3c-e08956.png)

**Function**: The command aligns the selected visualization elements so that the blue element keeps its position and the other elements are positioned horizontally with less distance between them. The spacing decreases by one pixel.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Multiple elements are selected.
