# Import / Export Vijeo-Designer Project

## Overview

The Import Vijeo-Designer project command allows you to import a Vijeo-Designer project into a EcoStruxure Machine Expert project.

The Export Vijeo-Designer project command allows you to export the Vijeo-Designer project from an EcoStruxure Machine Expert project.

Regardless of the number of HMIs, only one EcoStruxure Machine Expert project is created.

NOTE: The imported Vijeo-Designer project will replace the existing HMI applications of the EcoStruxure Machine Expert project, if any exist.

## Importing a Vijeo-Designer Project into EcoStruxure Machine Expert

To import an already existing Vijeo-Designer project into an EcoStruxure Machine Expert project, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | In EcoStruxure Machine Expert, open the File menu and select the Import Vijeo-Designer project... command.  **Result**: A browse dialog box is displayed. |
| 2 | Browse to the folder that includes the Vijeo-Designer project you want to import, select the VDZ file, and click Open.  **Result**: The selected Vijeo-Designer project is merged with the already existing EcoStruxure Machine Expert project. |

## Exporting a Vijeo-Designer Project from EcoStruxure Machine Expert

To export the Vijeo-Designer project from an EcoStruxure Machine Expert project in order to create a single VDZ file, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | In EcoStruxure Machine Expert, open the File menu and select the Export Vijeo-Designer project... command.  **Result**: A browse dialog box is displayed. |
| 2 | Browse to the folder where you want to save the exported VDZ file and click Save.  **Result**: The extracted Vijeo-Designer project is saved as VDZ file and can be opened with Vijeo-Designer. |

NOTE: VDZ files only contain the HMI application component exported from the Vijeo-Designer application. Control elements of the device are not stored in the VDZ file format.

For HMI controllers, export and reimport the application components (such as POUs, tasks, symbol configuration) as well as fieldbus devices separately by using the Project > Export as XML and Project > Import from XML commands.
