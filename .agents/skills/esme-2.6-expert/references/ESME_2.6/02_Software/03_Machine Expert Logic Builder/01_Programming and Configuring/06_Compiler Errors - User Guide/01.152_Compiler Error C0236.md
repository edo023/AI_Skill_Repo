# Compiler Error C0236

## Message

Wrong type definition for VAR\_EXTERNAL <variable name>

## Message Cause

The variable is declared in VAR\_GLOBAL / VAR\_EXTERNAL as different types.

## Solution

Use the same type definition in VAR\_GLOBAL and VAR\_EXTERNAL.

## Error Example

```
PROGRAM PLC_PRG
VAR_EXTERNAL
  ig : STRING;
END_VAR
```

```
VAR_GLOBAL
  ig : INT;
END_VAR
```

--> C0236: Wrong type definition for VAR\_EXTERNAL ‘ig’

EIO0000003933.04
