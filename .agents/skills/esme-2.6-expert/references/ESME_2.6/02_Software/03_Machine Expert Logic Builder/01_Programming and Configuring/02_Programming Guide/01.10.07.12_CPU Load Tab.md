# CPU Load Tab

## Overview

The CPU Load tab of the Task Configuration is available in online mode for multicore controllers. The load of the individual processor cores is displayed in the trace editor. For further information, refer to [Displaying the CPU Load with DeviceTrace Objects in the Project](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=DisplayingTheCPULoadWithDeviceTrace_CFAC46BA).

To add more variables, open the trace configuration by clicking the Configuration link on the right-hand side.
