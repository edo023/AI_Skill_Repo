# `__POOL`

## Definition

This operator is not specified by the IEC 61131-3 standard.

The `__POOL` operator is used to reference objects which are managed in the global POUs pool. It directly accesses objects in the Global node of the Applications tree.

## Example

```
PROGRAM PLC_PRG
VAR
        svar_pou : STRING;
END_VAR

svar_pou := __POOL.POU();
```
