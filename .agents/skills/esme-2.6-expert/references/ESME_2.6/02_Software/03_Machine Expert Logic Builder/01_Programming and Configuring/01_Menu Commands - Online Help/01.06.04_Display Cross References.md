# Display Cross References

## Overview

The Edit > Browse > Display Cross References command searches for the locations where a selected variable is used and displays the results in the [Cross Reference List view](D-SE-0083917.html).

Two different types of the command are available that allow you to limit the search to the present function block instance or to search in all instances of the function block. The examples illustrate the commands using function block FB\_Test, variable i\_xTest and instance of the function block fb\_Test:

* Search for occurrences of the selected variable in the context of the present function block instance.

  Example:

  Display Cross References: fb\_Test.i\_xTest
* Search for occurrences of the selected variable of all instances of the function block.

  Example:

  Display Cross References: FB\_Test.i\_xTest

The commands are available if a POU is open in the editor and a variable is selected.
