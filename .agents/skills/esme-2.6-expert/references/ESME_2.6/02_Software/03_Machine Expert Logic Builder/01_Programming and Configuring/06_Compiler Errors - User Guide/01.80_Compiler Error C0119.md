# Compiler Error C0119

## Message

A ‘FB\_Init’-Method of a function block or structure needs two inputs, ‘bInitRetains’ and ‘bInCopyCode’ of type BOOL

## Message Cause

One or both of the BOOL type inputs ‘bInitRetains’ and ‘bInCopyCode’ are missing.

## Solution

Define the missing inputs.

## Error Example

```
PROGRAM PLC_PRG
VAR
inst : FB;
END_VAR
```

```
FUNCTION_BLOCK FB
METHOD FB_init
VAR_INPUT
END_VAR
```

--> C0119: A 'FB\_Init'-Method of a function block or structure needs two inputs, 'bInitRetains' and 'bInCopyCode' of type BOOL

## Error Correction

```
METHOD FB_init
VAR_INPUT
  bInitRetains : BOOL;
  bInCopyCode : BOOL;
END_VAR
```
