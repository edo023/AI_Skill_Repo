# Insert Negated Contact Parallel Below

## Overview

The FBD/LD/IL > Insert Negated Contact Parallel Below command inserts a negated parallel [contact](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083487). Alternatively, you can use a combination of the commands [**Insert Contact Parallel Below**](D-SE-0084121.html#D-SE-0084121) and [**Negation**](D-SE-0084130.html#D-SE-0084130) to insert a negated parallel contact.
