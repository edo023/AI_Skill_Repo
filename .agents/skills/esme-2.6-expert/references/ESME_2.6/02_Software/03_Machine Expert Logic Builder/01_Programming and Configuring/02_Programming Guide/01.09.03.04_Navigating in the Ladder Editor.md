# Navigating in the Ladder Editor

The following keys and key combinations switch between cursor positions in the editor. Switching is also possible across networks.

| Key or key combination | Effect |
| --- | --- |
| →  ← | Switch to the adjacent cursor position, along the signal flow (from left to right, and right to left). |
| ↑↓ | Switch to the next cursor position above or below the present position if this adjacent position belongs to the same logical group.  Example: The connections of a block form a logical group. If such a logical group does not exist, then change to the first cursor position in the next higher or lower adjacent element. In the case of parallel-connected elements, navigation is performed along the first branch. |
| Ctrl + Home | Switch to the first network and select it. |
| Ctrl + End | Switch to the last network and select it. |
| Page Up ↑ | Scroll up one page. The top network on this page is selected. |
| Page Down ↓ | Scroll down one page. The bottom network on this page is selected. |
