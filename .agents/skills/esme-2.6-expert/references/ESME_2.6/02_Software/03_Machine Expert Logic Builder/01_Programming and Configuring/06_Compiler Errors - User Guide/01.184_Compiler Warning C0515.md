# Compiler Warning C0515

## Message

Should not access protected property <POU name>.<property name>.

## Message Cause

A property of the POU, whose visibility is restricted with the `PROTECTED` keyword, is accessed from outside of the POU or its inheritance hierarchy.

## Solution

Modify the visibility of the property (for example, remove the `PROTECTED` keyword) or modify the declaration of the variables.

## Error Example

```
FUNCTION_BLOCK POU
VAR_INPUT
END_VAR
...

PROPERTY PROTECTED Prop : UDINT

PROGRAM PLC_PRG
VAR
      p:POU := STRUCT(Prop := 7);
END_VAR
```

--> C0515: Should not access protected property POU.Prop.
