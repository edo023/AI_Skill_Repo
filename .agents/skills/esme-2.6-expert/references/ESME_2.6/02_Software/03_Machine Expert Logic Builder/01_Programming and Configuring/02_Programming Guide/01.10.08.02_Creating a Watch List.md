# Creating a Watch List

## Procedure

Create a watch list by executing the following actions:

| Step | Action | Comment |
| --- | --- | --- |
| 1 | Execute the command View > Watch > Watch<n> to open a Watch view. | – |
| 2 | Double-click the field in the column Expression and enter the variable to be watched either manually or via the Input Assistant.  Syntax: `<device name>.<application name>.<object name>.<variable name>` | NOTE: If you enter the name of a structured variable, then in online mode the particular structure components are entered in further lines automatically. |
| 3 | Subsequently define all variables which you want monitored in the current watch list. You can modify the order of the entries via drag and drop. | The fields Execution point, Type, Address, Comment are filled automatically corresponding to the declaration of the variables. The symbol preceding the expression indicates whether it is an input variable , an output variable , or a normal variable . |

## Online Mode

In online mode, the current variable value is shown in the column Value. Like at other monitoring positions, you can define Prepared Values and force or write them. For further information, refer to the chapters [*Force Values, Prepare Value Dialog Box*](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0084023) and [*Monitoring*](D-SE-0083450.html#D-SE-0083450).

In online mode, you can add a currently focused expression to the watch list by executing the command Add Watch.
