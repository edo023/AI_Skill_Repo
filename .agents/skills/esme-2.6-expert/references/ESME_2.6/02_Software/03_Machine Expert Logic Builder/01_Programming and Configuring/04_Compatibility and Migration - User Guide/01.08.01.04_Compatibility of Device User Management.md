# Compatibility of Device User Management

## Overview

The user rights storage format is updated with EcoStruxure Machine Expert V2.0.

User rights included in controllers programmed with EcoStruxure Machine Expert versions prior to V2.0 must be recreated after updating their firmware version to EcoStruxure Machine Expert V2.0.

NOTE: When you have updated a project created with SoMachine V4.3 and you open the Users and Groups view of the device editor, a dialog box is displayed prompting you to modify the device user data (user rights). This dialog box is displayed even if the initial projects had no user rights.
