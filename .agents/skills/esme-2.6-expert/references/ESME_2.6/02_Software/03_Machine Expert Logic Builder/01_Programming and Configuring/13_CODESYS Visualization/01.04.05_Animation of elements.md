# Animation of elements

Animating a visualization element at runtime can be used to display value curves in a visualization in addition to strictly visual purposes. An animation is possible by means of a dynamic configuration of specific element properties. These properties are controlled by means of variables. See the following examples for possible animations.

The client can also execute the computation of animated movements and the visibility of visualization elements. The internal rendering engines of the web browser are used to show the animation quickly and smoothly.

TIP:

In order to use this functionality, you have to select the **Support client animations and overlay of native elements** option in the [Object: Visualization Manager](_visu_obj_manager.html#_visu_obj_manager).
