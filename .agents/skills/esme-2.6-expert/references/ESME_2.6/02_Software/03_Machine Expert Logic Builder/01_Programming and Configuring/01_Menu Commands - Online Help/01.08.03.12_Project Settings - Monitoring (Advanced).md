# Project Settings - Monitoring (Advanced)

## Overview

The Project Settings > Monitoring (Advanced) dialog box allows you to configure the behavior of device status monitoring for a project when switching to online mode. As device status monitoring is by default activated, this setting helps to reduce the network load if, for example, your project includes a large number of devices.

The following options are available:

| Option | Description |
| --- | --- |
| Active when going online | The device status monitoring is active when switching to online mode. |
| Inactive when going online | The device status monitoring is inactive when switching to online mode. |
| Remember last state | When switching to online mode, the last setting is used for device status monitoring as set with the command [Suspend Device Status Monitoring](D-SE-0108108.html). |
