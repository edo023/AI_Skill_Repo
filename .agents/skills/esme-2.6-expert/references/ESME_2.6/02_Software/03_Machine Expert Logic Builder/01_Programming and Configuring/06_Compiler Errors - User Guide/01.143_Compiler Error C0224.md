# Compiler Error C0224

## Message

Call recursion: <recursion>

## Message Cause

A function calls itself.

## Solution

Make sure that functions are not recursive.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
POU();
```

```
FUNCTION POU
VAR
END_VAR
```

```
POU();
```

--> C0224: Call recursion: POU -> POU

EIO0000003933.04
