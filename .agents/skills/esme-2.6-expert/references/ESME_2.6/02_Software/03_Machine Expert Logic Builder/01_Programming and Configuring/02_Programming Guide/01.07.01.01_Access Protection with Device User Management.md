# Access Protection with Device User Management

## Introduction

In order to meet constantly evolving cybersecurity requirements, the user rights management for EcoStruxure Machine Expert controllers is by default activated. Therefore, every EcoStruxure Machine Expert controller equipped with the latest firmware prompts you for user credentials whenever you attempt to gain access to it.

For general information regarding device user management, refer to the chapter describing the Users and Groups [view of the device editor](D-SE-0083876.html#D-SE-0083876).
