# Command: Add node

Symbol: ![](images/uuid-ca5e261c-1a47-3d20-5ec2-1d9968.png)

**Function**: Adds a new property at the end of the list

**Call**: **HTML5 Control Editor** menu

Double-click the new entry to open the input field so that you can specify a suitable name. After that, you can still position the property.
