# Compiler Error C0018

## Message

<expression> is not a valid assignment target

## Message Cause

An expression with no write permission is on the left side of an assignment. Example: a constant.

## Solution

Only assign to variables that have write access.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i: INT;
END_VAR
VAR CONSTANT
  j: INT := 0;
END_VAR
```

```
j := i;
```

--> C0018: 'j' is not a valid assignment target
