# Compiler Warning C0426

## Message

At least one statement is expected.

## Message Cause

A `CASE` statement contains an empty `CASE` label.

## Solution

Remove the empty `CASE` label or add a statement to this `CASE` label.

## Error Example

```
PROGRAM PLC_PRG
VAR
      nVar:INT;
END_VAR

CASE nVar OF
      1:
      2:
         nVar := nVar + 1;
END_CASE
```

--> C0426: At least one statement is expected.
