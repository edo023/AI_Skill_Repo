# Configuring Depictor

1. Add a "Depictor" object below the application. To do this, click **Project → Add Object → Depictor**.
2. Define the interface variables as follows:

   * **fb**: Depic.ScaraTrafo
   * **lrZ\_**: Drive1.fSetPosition
   * **lrZmin**: 10
   * **lrZmax**: -50
   * **lrSize**: Depic.LrSize
   * **xShowPlane**: 0
