# Creating an HTML5 control

1. You can now create the HTML5 control. The HTML5 control editor helps you with this.

   1. On the **[Tab: General](_visu_html5_tab_general.html#_visu_html5_tab_general)** tab, configure the new element.
   2. On the **[Tab: Control Properties](_visu_html5_tab_property.html#_visu_html5_tab_property)** tab, specify the desired properties.
   3. Create a new property of type **Variable**. In the **Method call** column, specify the desired method name.

      * This is how variables are transferred to methods of the control.

        TIP:

        To familiarize yourself with the capabilities, it is helpful to open the examples in the HTML5 Control Editor.

        For more information, see: [Reference: HTML5 Control Editor](_visu_ref_html5_editor.html#_visu_ref_html5_editor)
   4. On the [Tab: Localization](_visu_html5_tab_localization.html#_visu_html5_tab_localization) tab, localize the interface texts of the control for the various languages.

      * The texts are localized. The element can be localized in different languages.
   5. Save your configuration under the control name and install it in the CODESYS Visualization Element Repository.

      To do this, use the ![](images/uuid-12293cd2-ca15-3066-7ae7-aacd0f.png) command icon.

      For more information, see: <_visu_link_and_use_html5_element.html#_visu_link_and_use_html5_element__UUID-97eb390c-95a5-1d29-15de-fa85ef03769a_procedure-idm13309409181364>
   * You can now use the HTML5 control in a visualization.
