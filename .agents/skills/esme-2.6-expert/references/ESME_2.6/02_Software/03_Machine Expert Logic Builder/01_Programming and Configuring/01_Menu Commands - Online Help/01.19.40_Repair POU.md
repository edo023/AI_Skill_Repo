# Repair POU

## Overview

The FBD/LD/IL > Repair POU command will only be available if the editor has detected some internal inconsistency within the POU which possibly could be resolved automatically (for example, boxes without correct type). This situation can occur when you edit a project created with an earlier EcoStruxure Machine Expert version where this inconsistency was not yet detected as an error.
