# Compiler Error C0030

## Message

Direct address expected after ‘AT’ instead of ‘<identifier>’

## Message Cause

Either an invalid address or no address is assigned after ‘AT’.

## Solution

Specify a valid address.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i AT ABC : INT;
END_VAR
```

--> C0030: Direct address expected after 'AT' instead of 'ABC'

## Error Correction

```
i AT %IW0 : INT;
```
