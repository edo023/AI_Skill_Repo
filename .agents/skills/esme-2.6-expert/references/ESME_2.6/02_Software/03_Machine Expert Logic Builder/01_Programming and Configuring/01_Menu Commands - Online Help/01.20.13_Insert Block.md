# Insert Block

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | Ctrl + B |
| Call | * Ladder > Insert Block menu * Contextual menu |

## Function

The command inserts a block that is available at the end of the selected network. Possible insertion positions are indicated by the plus symbol at the cursor when you drag the element from the Toolbox to the implementation part. By default, an `EN/ENO` block is inserted. Use the EN and ENO commands to remove and add the `EN` and `ENO` modifiers.

NOTE: If the block has been defined in the project, update the usage in the Ladder editor after the block has been modified by running the [Update Parameters command](UpdateParams-45405777.html).

## Element: Block

A block element can represent additional functions, for example, IEC function blocks, IEC functions, library function blocks, or operators.

If the block provides an image file, then the block icon can be displayed inside the block.

Refer to the [Tools > Options > Ladder Editor dialog box](LadderEditor-44B3FC46.html) to enable visibility of the block with the Show block icon option.
