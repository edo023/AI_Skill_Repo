# Attribute Pragmas

## Overview

You can assign [attribute pragmas](D-SE-0083614.html#D-SE-0083614) to a signature in order to influence the compilation or pre-compilation that is the code generation.

There are [user-defined attributes](D-SE-0083620.html#D-SE-0083620), which you can use in combination with [conditional pragmas](D-SE-0083616.html#D-SE-0083616).

Attributes are defined within the declaration part. An exception is made for the action and transition objects which do not have a declaration part. You can define the attributes at the beginning of the implementation part.
