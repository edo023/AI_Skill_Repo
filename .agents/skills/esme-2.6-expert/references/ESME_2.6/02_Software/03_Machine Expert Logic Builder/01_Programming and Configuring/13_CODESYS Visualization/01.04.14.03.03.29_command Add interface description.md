# Command: Add interface description

Symbol: ![](images/uuid-9fcd79d9-5d8e-408c-f778-d5a9b4.png)

**Function**: Adds the **Element description** property at the top position

**Call**: Icon in the toolbar

There may be at most one element description in **Control Properties**. After that the command is inactive.
