# Adding a coupled drive device (EtherCAT)

Requirement: A SoftMotion-compatible controller is inserted into the project.

1. In the device tree, select the SoftMotion controller.
2. Click the **Add Device** button.

   * The device is added to the device tree.

     ![](images/uuid-06df1848-7199-b9f6-84af-5eaefc.png)

Double-clicking the device opens the corresponding device editor.
