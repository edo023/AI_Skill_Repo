# Compiler Error C0206

## Message

BIT is not allowed as a base type of an array

## Message Cause

An attempt is made to declare a BIT array.

## Solution

Do not declare BIT arrays.

## Error Example

```
PROGRAM PLC_PRG
VAR
  arr : ARRAY[1..2] OF BIT;
END_VAR
```

--> C0206: BIT is not allowed as a base type of an array
