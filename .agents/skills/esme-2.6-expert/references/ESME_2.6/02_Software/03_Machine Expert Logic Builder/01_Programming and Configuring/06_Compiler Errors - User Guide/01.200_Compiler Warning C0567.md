# Compiler Warning C0567

## Message

Interface <interface name 1> does not extend \_\_System.IQueryInterface.

## Message Cause

Every interface of an interface library must extend \_\_System.IQueryInterface. The interface used does not extend \_\_System.IQueryInterface.

## Solution

Extend the interface.

## Error Example

```
PROGRAM PLC_PRG
VAR
    ITFref : ITF;
    ITFref2 : ITF2;
END_VAR

INTERFACE ITF
INTERFACE ITF2 EXTENDS ITF
```

--> C0567: Interface ITF does not extend \_\_System.IQueryInterface.

## Error Correction

```
INTERFACE ITF EXTENDS __System.IQueryInterface
```
