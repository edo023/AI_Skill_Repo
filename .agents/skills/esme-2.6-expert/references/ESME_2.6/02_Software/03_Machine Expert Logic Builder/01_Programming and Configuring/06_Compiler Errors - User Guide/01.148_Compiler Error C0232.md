# Compiler Error C0232

## Message

Array initialisation expected.

## Message Cause

An array of arrays is initialized, but the initialization values are not nested.

## Solution

Use a nested array initialization as shown in the example below.

## Error Example

```
PROGRAM PLC_PRG
VAR
  value : ARRAY[0..2] OF ARRAY[0..2] OF
INT := [1,2,3];
END_VAR
```

-->C0232: Array initialisation expected.

## Error Correction

```
value : ARRAY[0..2] OF ARRAY[0..2] OF INT := [
[1,2,3],
[4,5,6],
[7,8,9]];
```
