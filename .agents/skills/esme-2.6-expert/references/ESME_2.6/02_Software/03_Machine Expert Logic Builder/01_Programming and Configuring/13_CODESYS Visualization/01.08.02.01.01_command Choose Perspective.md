# Command: Choose Perspective

**Function**: The command lets you switch between the CODESYS perspective and the HMI perspective.

**Call**: **View** menu.

|  |  |
| --- | --- |
| **CODESYS Perspective** | CODESYS is displayed in the usual tree oriented perspective in which IEC applications are typically developed. |
| **HMI Perspective** | CODESYS is displayed in the HMI perspective that supports the development of HMI applications.  The perspective is subdivided into individual views: **Visualizations**, **Data Sources**, and **HMI Features**. |
