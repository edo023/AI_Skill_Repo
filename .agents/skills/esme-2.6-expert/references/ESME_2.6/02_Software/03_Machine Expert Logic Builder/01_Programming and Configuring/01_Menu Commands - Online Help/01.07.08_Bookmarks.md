# Bookmarks

## Overview

The View > Bookmarks command opens the Bookmarks view which provides a list of the bookmarks available in the project. You can modify the order of the bookmarks by drag and drop. Double-clicking a row opens the Object in the editor and jumps to the selected bookmark.

Buttons in the Bookmarks view:

| Button | Description |
| --- | --- |
|  | Click the Previous Bookmark button to jump to the bookmark that is available in the row above the selected bookmark. The corresponding POU opens in the editor. |
|  | Click the Next Bookmark button to jump to the bookmark that is available in the row below the selected bookmark. The corresponding POU opens in the editor. |
|  | Deletes the selected bookmark from the table and from the corresponding POU. |

Information provided in the Bookmarks table:

| Column | Description |
| --- | --- |
| Bookmark | Name of the bookmark as assigned by the Logic Builder in ascending numerical order, such as Bookmark\_0, Bookmark\_2.  Select the Bookmark field to make it editable and to assign a name of your choice. |
| Object | Name and project path of the POU where the bookmark is set.  Example: `POU_Add [PLC_1: SPS-Logic: Application]` |
| Position | Position of the bookmark in the POU.  Example: Line 3, Column 1 (Impl)  (Impl): The bookmark is in the implementation part of the POU.  (Decl): The bookmark is in the declaration part of the POU. |
