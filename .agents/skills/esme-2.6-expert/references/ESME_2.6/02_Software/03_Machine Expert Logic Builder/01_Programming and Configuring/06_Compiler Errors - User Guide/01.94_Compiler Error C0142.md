# Compiler Error C0142

## Message

A local variable named ‘<variable name>’ is already defined in ‘<pou name>’

## Message Cause

The same variable name is used twice.

## Solution

Use different variable names.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  i : INT;
END_VAR
```

--> C0142: A local variable named 'i' is already defined in 'PLC\_PRG'
