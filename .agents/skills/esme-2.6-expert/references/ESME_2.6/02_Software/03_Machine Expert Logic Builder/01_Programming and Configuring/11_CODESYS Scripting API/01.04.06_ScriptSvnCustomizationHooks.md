Gets the collision pattern. It is used when the location created by the normalized name is already used by some other node, or unusable for another reason. It has to be a String.Format compatible format string getting one integer object as argument which is counted up until a non-colliding working copy location is found. For pattern matching on location resurrection, the format string also has to allow a “\*” to be passed.

SvnSelectionVerificationArgs

Bases: object

These args are used by the SvnUIServices, Svn Commands, and other Plugins to process selection extension or inversion.

For objects, this hook currently works on the granularity of SvnItemType.ObjectSynthesized and incoming Object IDs will be automatically extended to their synthesized counterparts. But this behaviour may change in the future, especially when the StatusControl learns to deal with unsynthesized IDs (similar to the working copy explorer).

This interface is considered obsolete. Its support may be phased out in a future version of the SVN plugin. You should use the new, RCS neutral RcsSelectionVerificationHook in the RevisionControlSystemHooks interface dll instead.

SvnSelectionVerificationArgsWithUrlAndRevision

Bases: object

Extension interface for [SvnSelectionVerificationArgs](#scriptsvncustomizationhooks_module_ScriptSvnCustomizationHooks__scriptsvncustomizationhooks_ScriptSvnCustomizationHooks_SvnSelectionVerificationArgs) since CoDeSys V3.5 SP1. This extension is used if [SvnSelectionVerificationArgs](#scriptsvncustomizationhooks_module_ScriptSvnCustomizationHooks__scriptsvncustomizationhooks_ScriptSvnCustomizationHooks_SvnSelectionVerificationArgs).Action is set to SvnSelectionVerificationAction.IncludeExternals to allow for the definition of URLs for the additionally selected :class:`SvnObjectId`s.

stReason

Requests the selection of the specified object, including an Url. For SvnSelectionVerificationAction.IncludeExternals, this has to be called instead of [SvnSelectionVerificationArgs](#scriptsvncustomizationhooks_module_ScriptSvnCustomizationHooks__scriptsvncustomizationhooks_ScriptSvnCustomizationHooks_SvnSelectionVerificationArgs).RequestSelection.

Parameters

* obj (SvnObjectId) – The object which the hooks wants to add to the selection.
* requiringObject (SvnObjectId) – The requiring object. (OIW, the object which causes the addition.)
* stReason (str) – The human readable reason (will be displayed in case of a conflict).
* url (Uri) – The URL for the object.
* revision (SvnRevision) – The revision. You can pass null if you donot want to specify a specific revision.
