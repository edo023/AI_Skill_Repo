# SmartTag Function

## Overview

The SmartTag function provides support in writing program code by suggesting commands suitable at the programming element. If the cursor is placed on a programming element with SmartTag function, the light bulb icon ![](images/T003225683.png) is displayed. Click the ![](images/T003225683.png) icon to display a list of commands available for this programming element.

Available smart tags:

* The [declare variable command](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0083913) is available for undeclared variables in the implementation part of the ST editor.
