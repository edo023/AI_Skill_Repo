# Compiler Warning C0571

## Message

FB\_EXIT of instances in VAR RETAIN is also called during Reset warm, but not FB\_INIT. Avoid retain declaration of function blocks with FB\_EXIT!

## Message Cause

An instance of a function block with an `FB_EXIT` method is declared in VAR RETAIN.

## Solution

Avoid retain declarations of function blocks with `FB_EXIT`.

## Error Example

```
FUNCTION_BLOCK POU
...
METHOD FB_Exit: BOOL
VAR_INPUT
      bInCopyCode : BOOL;
END_VAR
...
FUNCTION_BLOCK POU_1
VAR RETAIN
      inst2 : POU;
END_VAR
...
PROGRAM PLC_PRG
VAR RETAIN
      myFB : POU_1;
END_VAR
myFB();
```
