# Displaying rule violations in the message view

1. Click **View → Messages**.

   * The message view opens.
2. In the message view, in the list box, select the **Precompile** category.

   * In this category, only the rule violations are displayed, which have been detected during precompile and after a successful compile, and which you can resolve with Quickfix. The ![](images/uuid-b57a4bb3-a633-38bc-da78-b44b05.png) button provides the respective commands for this.
