# Watch

## Overview

The View > Watch command opens a submenu with commands Watch 1, Watch 2, Watch 3, Watch 4 and Watch all Forces. These are used to open the respective [watch list in a view window](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083543). Watch all Forces is a special watch view for the [currently forced values](D-SE-0084023.html#D-SE-0084023).
