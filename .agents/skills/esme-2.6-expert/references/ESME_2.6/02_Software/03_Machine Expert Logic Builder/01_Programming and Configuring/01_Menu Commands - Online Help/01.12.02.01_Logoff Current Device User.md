# Logoff Current Device User

## Overview

The Online > Security > Logoff current device user serves to explicitly log off the current online user from the target device. If there is still a connection to the target, this will be disconnected.

NOTE: The online user management can be handled via the tabs Users and Groups and Access Rights of the device editor ([**Users and Groups**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083876)). The commands of the menu Online > Security provide an additional simplified opportunity to protect the access on the target device.
