# Dialog: Export Trace Configuration

The dialog is used for saving the trace configuration to a text file that can be read by the runtime system.

|  |  |
| --- | --- |
| **File name** | Name of text file which will be created |
| **File type** | **Trace file (\*.trace)**: Format that the runtime system component `CmpTraceMgr` expects for reading |
