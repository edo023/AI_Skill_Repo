# Project Settings - Monitoring

## Overview

The Project Settings > Monitoring dialog box allows you to configure the following settings for displaying the variable values in [online monitoring mode](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083450).

|  |  |
| --- | --- |
| Display Mode for Integer Variables | |
| Binary | The value of the variable is displayed in the selected format in online mode.  This option corresponds to the setting of the Debug > Display Mode [command](D-SE-0084026.html#D-SE-0084026). |
| Decimal |
| Hexadecimal |

|  |  |
| --- | --- |
| Floating Point Variables | |
| Number of displayed digits | Configure the number of decimal places that are displayed in online mode for REAL values.  NOTE: The settings apply to the [watch list](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083546), the monitoring of the [declaration editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083520) and the [trace editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083568). The configuration for inline monitoring of the editor is set in the text editor options. |
| Structured display of the inheritance hierarchy | Select the option to display the inheritance hierarchy of function blocks and structures in a tree structure. The variables are displayed as subnodes of the function block or of the structure where they are declared.  If the option is not selected, a list is displayed. |
