# Insert Image

## Overview

The Image Pool > Insert Image command is available when you are working in the image pool editor. It inserts a new image file in a new line above the one where currently the focus is on. It opens the edit field for entering the path of the image file in the File name column (for example *D:\project1\icon\_xy.bmp*). Click the ... button to open the Select Image dialog box for this purpose. You can insert images in format BMP, EMF, GIF, ICO, JPG, PNG, SVG, and TIF.

On the usage of image pools, refer to the chapter [*Image Pool*](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083433).
