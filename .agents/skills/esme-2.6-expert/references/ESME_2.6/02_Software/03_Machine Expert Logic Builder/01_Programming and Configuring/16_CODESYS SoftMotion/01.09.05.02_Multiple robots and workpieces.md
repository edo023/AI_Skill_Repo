# Multiple robots and workpieces

The position and orientation of the coordinate systems are defined for each robot. When multiple robots are working together, it can be useful to define the world coordinate system (WCS) as a common base coordinate system. Based on this coordinate system, the machine coordinate systems (MCS) and product coordinate systems (PCS\_1, PCS\_2) of the individual robots can be shifted.
