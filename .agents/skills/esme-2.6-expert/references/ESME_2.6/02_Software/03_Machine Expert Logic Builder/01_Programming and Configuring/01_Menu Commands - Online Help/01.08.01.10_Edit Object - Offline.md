# Edit Object - Offline

## Overview

The Project > Edit Object - Offline command is available in online mode. After a login to the application has been performed, a double-click on an entry in the navigators will open the associated editor in online mode (if an instance of the selected object belongs to that application).

If you prefer to open the object in offline mode, select the object node by a single click in the navigator and execute the command Project > Edit Object (Offline).
