# Compiler Error C0065

## Message

There is no global definition for ‘<name>’.

## Message Cause

The value searched for is not a global variable, global POU, or other value that can be accessed globally.

## Solution

Declare ‘<name>’ as global variable.

## Error Example

```
PROGRAM PLC_PRG
```

```
.someValue := 5;
```

-->C0065: There is no global definition for ‘someValue’.

## Error Correction

```
VAR_GLOBAL
    someValue : INT;
END_VAR
```
