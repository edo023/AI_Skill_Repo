# Compiler Error C0175

## Message

‘RETAIN’ or ‘PERSISTENT’ is not allowed in this place

## Message Cause

‘RETAIN’ or ‘PERSISTENT’ are used in a function and do not have any effect in a function.

## Solution

‘RETAIN’ or ‘PERSISTENT’ do not have any effect in a function.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
POU_1();
```

```
FUNCTION POU_1
VAR RETAIN
END_VAR
```

--> C0175: 'RETAIN' or 'PERSISTENT' is not allowed in this place
