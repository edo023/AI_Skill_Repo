# Compiler Warning C0572

## Message

The uninitialized variable <variable1> is used for initialization of <variable2>. Use the 'global\_init\_slot' attribute to change the order of initialization.

## Message Cause

In the case of nested structures, the compiler warning C0572 is issued instead of the compiler error C0268 when an uninitialized variable exists in an inner structure.

## Error Example

```
GVL
{attribute 'qualified_only'}
VAR_GLOBAL
      x : INT := 7;
      y : INT := 9;
END_VAR

GVL_1
{attribute 'qualified_only'}
VAR_GLOBAL
      d1 : DUT_1;
END_VAR

TYPE DUT :
STRUCT
      a : INT;
      i : INT := GVL.x;
END_STRUCT
END_TYPE

TYPE DUT_1 :
STRUCT
      d : DUT;
END_STRUCT
END_TYPE

PROGRAM PLC_PRG
GVL_1.d1.d.a;
```

EIO0000003933.04
