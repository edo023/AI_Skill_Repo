# Compiler Warning C0228

## Message

No initial value for constant variable ‘<constant name>’

## Message Cause

A constant is not initialized.

## Solution

Initialize the constants.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
VAR CONSTANT
  k : INT;
END_VAR
```

--> C0228: No initial value for constant variable 'k'

## Error Correction

```
k : INT := 1;
```
