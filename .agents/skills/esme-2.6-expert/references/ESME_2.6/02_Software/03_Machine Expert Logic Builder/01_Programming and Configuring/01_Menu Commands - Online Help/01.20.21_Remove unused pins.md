# Remove unused pins

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | - |
| Call | * Ladder > Remove unused pins menu * Contextual menu |

## Function

The command deletes input and output pins of the selected [box](InsertBlock-4537C2CF.html) to which no variable and no value is assigned. However, the default inputs and outputs are retained..
