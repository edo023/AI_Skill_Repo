# Dialog: Create HMI Project: Add Objects

**Function**: The dialog is part of the wizard that opens when you create a new project with the **HMI project** template. This allows for adding usual objects conveniently.

**Call**: Automatically after the **Create HMI Project – Connect to HMI Device** dialog

|  |  |
| --- | --- |
| **Alarm Configuration** | : The object is added to the HMI application. |
| **Image Pool** |
| **Recipe Manager** |
| **Text List** |
| **Unit Conversion** |
| **Trend Recording Manager** |
| **Program** |
| **Next** | Takes over the selection and opens the **Create HMI Project – Choose Perspective** dialog |
| **Abort** | Applies the configuration and creates an HMI project |
