# Compiler Warning C0569

## Message

No matching instance path in VAR\_PERSISTENT list found for variable <variable name>. Use the command "Add All Instance Paths" to add all instance paths to the VAR\_PERSISTENT list.

## Message Cause

Persistent variables which have been declared outside of the persistent variable list must be added by running the Add All Instance Paths command for this list. This message is issued when the command is not used after a persistent variable has been declared outside of the persistent variable list.

## Solution

From the Declarations menu of the persistent variable list, run the Add All Instance Paths command.

For further information, refer to [Commands of the Persistent Variable Editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0084168).

## Error Example

A GVL with:

```
VAR_GLOBAL PERSISTENT
    giRetain : INT;
END_VAR
```

And a POU using this variable:

```
GVL.giRetain;
```

EIO0000003933.04
