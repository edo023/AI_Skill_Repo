# Insert Step / Insert Step After

## Overview

The Insert Step / Insert Step After  commands insert a new step before / after the selected point in the SFC editor.

The default name of the new step is `Step<n>`. `n` is an incremental number starting at 0 for the first step that is inserted in addition to the initial step. Click the name to edit it.

By default, the command is not available in the menus. You can add this command via the Tools > Customize [menu](D-SE-0084066.html#D-SE-0084066).
