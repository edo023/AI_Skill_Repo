# Step Over

## Overview

Default shortcut: F10

The Debug > Step Over command can be used for [stepping](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083451) through a program in online mode, for example for debugging purposes.

A single step will be executed. With one-level instructions this is equal to stepping with [step into](D-SE-0084031.html#D-SE-0084031) (that is, from instruction to instruction). If however, a POU call is reached, then Step Over effects that this POU will be executed completely within the current step. In SFC, a complete action will be executed.

If you would like to jump to the first instruction of a called POU, you have to use the Step Into [command](D-SE-0084031.html#D-SE-0084031).

See also Step Out [command](D-SE-0084033.html#D-SE-0084033).
