# Compiler Warning C0525

## Message

The type <type> cannot have a default value in this context.

## Message Cause

The initial value of a VAR\_INPUT has a custom data type.

## Solution

Remove the initial value.

## Error Example

```
FUNCTION Fun: INT
VAR_INPUT
      iPar : INT;
      xPar : BOOL;
      aInt : ARRAY [0..1] OF INT := [1, 2];
END_VAR

PROGRAM PLC_PRG
VAR
     aInt:ARRAY [0..1] OF INT := [1, 2];
END_VAR
Fun(1, TRUE, aInt);
```

--> C0525: The type ARRAY [0..1] OF INT cannot have a default value in this context.
