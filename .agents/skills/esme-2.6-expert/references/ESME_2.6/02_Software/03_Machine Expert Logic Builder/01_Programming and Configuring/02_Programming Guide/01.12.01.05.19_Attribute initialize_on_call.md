# `Attribute initialize_on_call`

## Overview

You can add the pragma {attribute initialize\_on\_call} to input variables. An input of a function block with this attribute will be initialized at any call of the function block. If an input expects a pointer and if this pointer has been removed due to an online change, then the input will be set to NULL.

## Syntax

{attribute 'initialize\_on\_call'}
