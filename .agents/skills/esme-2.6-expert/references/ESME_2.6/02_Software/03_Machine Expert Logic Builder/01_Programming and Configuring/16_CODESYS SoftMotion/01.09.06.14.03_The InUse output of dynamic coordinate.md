# The "InUse" output of dynamic coordinate system function blocks

Each dynamic coordinate system function block (`MC_TrackConveyorBelt`, `MC_TrackRotaryTable`, `MC_SetDynCoordTransform`, and `SMC_SetDynCoordTransformEx`) has an `InUse` output. Restarting a function block while the coordinate system is still in use is not allowed and will result in the error `SMC_AXIS_GROUP_PCS_STILL_IN_USE`. A dynamic coordinate system will become in use as soon as a movement to a position in the coordinate system is commanded. It will remain in use until a target position in a different coordinate system has been reached.
