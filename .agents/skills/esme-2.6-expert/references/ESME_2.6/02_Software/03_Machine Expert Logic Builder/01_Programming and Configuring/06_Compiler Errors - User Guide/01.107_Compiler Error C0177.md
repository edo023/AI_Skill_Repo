# Compiler Error C0177

## Message

‘<object>’ is of type ‘FUNCTION’ and cannot be instantiated

## Message Cause

An attempt is made to instantiate a function.

## Solution

Only instantiate objects that can be instantiated.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : POU;
END_VAR
```

```
FUNCTION POU
VAR
END_VAR
```

--> C0177: 'POU' is of type 'FUNCTION' and cannot be instantiated

EIO0000003933.04
