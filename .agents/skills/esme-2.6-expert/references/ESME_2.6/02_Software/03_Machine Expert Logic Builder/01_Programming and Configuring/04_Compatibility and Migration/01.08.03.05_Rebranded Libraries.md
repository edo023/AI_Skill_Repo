# Rebranded Libraries

Libraries that were manually added to the Library Manager and meet the following criteria are not automatically updated when a Project Update is performed:

* Libraries that were rebranded from CAA Technical Workgroup to CODESYS.
* Libraries that were rebranded and renamed:

  + IoDrvModbusSerialSlave > IoDrvModbusSerialServer
  + IoDrvModbusTCPSlave > IoDrvModbusTCPServer
  + ModbusTCP Slave > ModbusTCP Server

Update these libraries manually in the Library Manager.
