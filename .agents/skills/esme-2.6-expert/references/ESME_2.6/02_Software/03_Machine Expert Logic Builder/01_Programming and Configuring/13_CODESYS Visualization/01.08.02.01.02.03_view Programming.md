# View: Programming

The view contains objects of the selected application that often do not have to be modified for the creation of an HMI application. If this view is closed, then you have a better overview of your HMI application.

**Non-HMI relevant object types:**

* **DUT**
* **External file**
* **Global Variable List**
* **Global Variable List (Receiver)**
* **Global Variable List (Sender)**
* **Persistent Variables**
* **POU**
* **POU for Implicit Checks**
* **Symbol Configuration**
* **Trace**
* **Library Manager**
* **Task Configuration**
