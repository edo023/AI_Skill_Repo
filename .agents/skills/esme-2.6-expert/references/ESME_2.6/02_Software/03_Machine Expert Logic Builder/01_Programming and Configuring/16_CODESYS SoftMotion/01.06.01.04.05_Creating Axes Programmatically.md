# Creating Axes Programmatically

The following free drive units can also be created programmatically:

| Axes |  |
| --- | --- |
| SMC\_VIRTUAL\_AXIS | Creates a virtual axis without having to add it to the device tree  For more information, see the following: [Virtual Drive](_sm_drive_controller_virtual_axis.html#_sm_drive_controller_virtual_axis) |
| SMC\_Logical\_Axis | Creates a logical axis without having to add it to the device tree.  For more information, see the following: [Logical Drive](_sm_drive_controller_logical_axis.html#_sm_drive_controller_logical_axis) |
| SMC\_FreeEncoder | Creates an encoder axis without having to add it to the device tree  For more information, see the following: [Free Encoder](_sm_drive_controller_free_encoder.html#_sm_drive_controller_free_encoder) |
