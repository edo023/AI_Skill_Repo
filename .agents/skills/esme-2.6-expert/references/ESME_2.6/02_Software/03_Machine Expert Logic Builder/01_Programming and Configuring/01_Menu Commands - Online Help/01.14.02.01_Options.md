# Options

## Overview

The Tools > Options dialog box contains subdialogs for configuring the behavior and display of the EcoStruxure Machine Expert programming interfaces. The default settings are defined by the current profile.

The current settings, confirmed by clicking OK, will be saved on the local system. They overwrite the default settings which are provided with an installation of the programming system.
