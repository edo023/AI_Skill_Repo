# Adding a Cam Object

## Overview

To add a cam diagram, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | In EcoStruxure Machine Expert, right-click the Application node in the Tools tree and select Add Object > CamDiagram (classic)... |
| 2 | In the Add CamDiagram (classic) dialog box, enter a Name for the new cam diagram. |
| 3 | Click Add to add the cam object.  As a result, a Camdiagram node with the defined name is created below the Application node. By default, it contains a Segment subnode that is a child object of the diagram object. |
