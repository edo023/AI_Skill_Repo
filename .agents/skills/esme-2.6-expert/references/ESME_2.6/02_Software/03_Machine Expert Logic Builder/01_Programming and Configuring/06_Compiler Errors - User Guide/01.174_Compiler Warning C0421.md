# Compiler Warning C0421

## Message

Use keyword EXTENDS for inheritance of interfaces instead of IMPLEMENTS.

## Message Cause

An interface extends another interface but uses the `IMPLEMENTS` keyword.

## Solution

Replace the `IMPLEMENTS` keyword by the `EXTENDS` keyword.

## Error Example

```
INTERFACE ITF

INTERFACE ITF_1 IMPLEMENTS ITF
```
