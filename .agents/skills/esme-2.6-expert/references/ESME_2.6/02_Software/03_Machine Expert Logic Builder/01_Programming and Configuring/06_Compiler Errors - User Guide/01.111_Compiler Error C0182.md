# Compiler Error C0182

## Message

Return type is only possible for POUs of the type FUNCTION and METHOD

## Message Cause

An attempt is made to define a return value in a program.

## Solution

Define a return value only in methods and functions.

## Error Example

```
PROGRAM PLC_PRG : BOOL
VAR
END_VAR
```
