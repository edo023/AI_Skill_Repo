# SA0113: Variables with data type WSTRING

Determines variables of type `WSTRING`.

Justification: Not all systems support `WSTRING`. The code is more easily portable without `WSTRING`.

Importance: Low

**Example**

```
VAR
    wstrTemp : WSTRING;
END_VAR
```
