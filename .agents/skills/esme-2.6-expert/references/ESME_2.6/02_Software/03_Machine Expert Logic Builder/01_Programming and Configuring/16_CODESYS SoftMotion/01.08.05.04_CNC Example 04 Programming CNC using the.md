# CNC Example 04: Programming CNC using the Tabular Editor

See the `CNC04_table.project` sample project in the installation directory of CODESYS under `..\CODESYS SoftMotion\Examples`.

In this project, the functionality of the `CNC03_prepro.project` sample project is programmed with the tabular editor. In contrast with `CNC03_prepro.project`, no IEC variables are used in this project. Instead, it is calculated with fixed values. Otherwise, the functionality is identical.
