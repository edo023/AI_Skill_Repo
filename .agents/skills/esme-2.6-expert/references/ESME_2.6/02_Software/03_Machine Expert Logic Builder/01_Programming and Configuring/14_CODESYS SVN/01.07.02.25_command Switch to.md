# Command: Switch to

Symbol: ![](images/uuid-4ed77ef7-e42b-3043-1618-809528.png)

**Function**: The command opens the **SVN switch** dialog. In the dialog, you specify a URL in the SVN repository to which the current working copy of the project is updated. The command switches a project from one branch or tag to another.

**Call**: **Project → SVN** menu

**Requirement**: The project is versioned.
