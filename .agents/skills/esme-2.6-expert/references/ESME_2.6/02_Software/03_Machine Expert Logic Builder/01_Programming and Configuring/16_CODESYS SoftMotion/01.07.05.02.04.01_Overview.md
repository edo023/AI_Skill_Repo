# Overview

Open the cam editor by double-clicking the **Cam** object in the device tree.

**The editor consists of the following tabs:**

* [Tab: Cam](_sm_obj_cam_table_cam.html#_sm_obj_cam_table_cam)  tab: In this editor, you use a graphical editor to create a cam path. Here, you can display and modify the slave position, slave velocity, slave acceleration, and slave jerk. In the graphical editor, you recognize very quickly when you program a movement with high acceleration.
* [Cam table](_sm_obj_cam_table_cam_table.html#_sm_obj_cam_table_cam_table)  tab: In this editor, the base points are presented in a table. Here, you can specify the exact positions and velocities.
* [Tappets](_sm_obj_cam_table_tappets.html#_sm_obj_cam_table_tappets)  tab: In this editor, you program the tappets (switch points) in a diagram. This display provides a very good overview of the sequential order of the tappets.
* [Tappet table](_sm_obj_cam_table_tappet_table.html#_sm_obj_cam_table_tappet_table)  tab: In this editor, the switch points are presented in a table. Here, you can specify the exact switch points.

The tabs are split into an editor, as well as a **Toolbox** view and **Properties** view.
