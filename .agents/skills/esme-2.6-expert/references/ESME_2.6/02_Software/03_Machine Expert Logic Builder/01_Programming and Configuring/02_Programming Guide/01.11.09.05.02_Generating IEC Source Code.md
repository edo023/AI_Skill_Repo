# Generating IEC Source Code

## Generated IEC Source Code Tab

The Generated IEC Source Code tab provides the program code that was generated automatically based on the cam diagram and the target specified in the Configuration tab.

If you define a target structure (cam data structure), the cam data is automatically written into this target structure during program initialization.

You can also configure a target structure manually by copying this code and replacing `###` by the name of your variable if no target has been defined.

The program code displayed in this tab is read-only. When the cam diagram is modified, the code is regenerated.
