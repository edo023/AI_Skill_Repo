# Assignment in FBD/LD/IL

## Overview

Depending on the selected [cursor position](D-SE-0083469.html#D-SE-0083469) in FBD or LD, an assignment will be inserted directly in front of the selected input ([cursor position 2](D-SE-0083469.html#D-SE-0083469__D-SE-0083469.4)), directly after the selected output ([cursor position 4](D-SE-0083469.html#D-SE-0083469__D-SE-0083469.4)) or at the end of the network ([cursor position 6](D-SE-0083469.html#D-SE-0083469__D-SE-0083469.4)). In an LD network, an assignment will be displayed as a [coil](D-SE-0083488.html#D-SE-0083488). Alternatively, drag the assignment element from the [toolbox](D-SE-0083473.html#D-SE-0083473) or [copy or move](D-SE-0083467.html#D-SE-0083467) it by drag and drop within the editor view.

After insertion, the text string `???` can be replaced by the name of the variable that is to be assigned. For this, use the ... button to open the Input Assistant.

In [IL](D-SE-0083465.html#D-SE-0083465), an assignment is programmed via LD and ST instructions. Refer to [*Modifiers and Operators in IL*](D-SE-0083466.html#D-SE-0083466).
