# Compare with target device...

## Overview

Execute the command Online > Compare with target device... to compare the open project with the project that is running on a configured controller, whereas configured means that you must have downloaded the source code to the controller beforehand. For further information, refer to the Source Download to Connected Device [command](D-SE-0083999.html#D-SE-0083999).

NOTE: The comparison is performed with the sources that were downloaded beforehand, which may not be synchronized with the application running on the controller. For information on how to keep them synchronized, refer to the Project Settings > Source Download [dialog box](D-SE-0083956.html#D-SE-0083956).

In contrast to the Compare with device... command, the Controller selection dialog box is not displayed for this command. The comparison is directly performed with the project that is running on the controller that was connected last.

The result of the comparison is represented in a new view titled [**Project Comparison - Differences**](D-SE-0083939.html#D-SE-0083939).
