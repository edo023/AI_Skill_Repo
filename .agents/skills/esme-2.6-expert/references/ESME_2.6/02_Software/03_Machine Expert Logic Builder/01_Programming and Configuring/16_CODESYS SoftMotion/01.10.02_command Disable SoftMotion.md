# Command: Disable SoftMotion

**Function**: The command disables the components of SoftMotion.

**Call**: **Project** menu; context menu of a selected device in the device tree

**Requirement**: For the selected controller, SoftMotion was enabled by means of the command. SoftMotion cannot be disabled for controllers which are intended to support SoftMotion (example: CODESYS SoftMotion Win).

The command causes the **SoftMotion General Axis Pool** device to be removed from below the controller configuration. Moreover, the SoftMotion libraries (with prefix `SM3_`) are removed from the Library Manager.

You can no longer use the SoftMotion functionality.
