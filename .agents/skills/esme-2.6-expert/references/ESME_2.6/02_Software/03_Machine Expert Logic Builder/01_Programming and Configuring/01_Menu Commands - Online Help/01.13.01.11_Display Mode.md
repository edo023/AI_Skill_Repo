# Display Mode

## Overview

The optional command Debug > Display Mode can be used to choose which of the following formats should be used for the ([monitoring](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083450)) display. Click the desired option in the Display Mode submenu.

The selected option is marked by a check mark.

* Binary
* Decimal
* Hexadecimal

NOTE: The display modes Binary and Hexadecimal are unsigned formats. Decimal is a signed format.
