# IScriptTrack

class

IScriptTrack.

IScriptTrack

Bases: object

This object describes a track.

name

The track name. This information is only relevant, if the configuration contains more than one track.

position

Position of the track in the global coordinate system. This information is only relevant, if the configuration contains more than one track.

orientation

Orientation of the track in the global coordinate system. This information is only relevant, if the configuration contains more than one track. The orientation is defined by a sequence of intrinsic rotations in the order Z -> Y -> X. The individual values indicate the angle of rotation around the corresponding axis. They can be in the range of [-180 .. 180).

logical\_forward

The direction of the track.

Type

[ScriptDirection](ScriptDirection.html#scriptdirection_module_ScriptDirection)

topological\_start\_address

The start address for the address assignment of segment devices. 1..512: Topological address of first segment device -1: No address assignment of segment devices

segments

The IScriptList2 of segments.

Type

List of [IScriptSegment](IScriptSegment.html#iscriptsegment_module_IScriptSegment)

stations

The a IScriptList2 of stations.

Type

List of [IScriptStation](IScriptStation.html#iscriptstation_module_IScriptStation)
