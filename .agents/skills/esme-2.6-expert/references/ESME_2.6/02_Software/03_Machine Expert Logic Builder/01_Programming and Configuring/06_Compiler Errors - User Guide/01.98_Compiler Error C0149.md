# Compiler Error C0149

## Message

Variable declarations are not allowed in interfaces

## Message Cause

An attempt is made to define a variable in an interface.

## Solution

Do not define variables in interfaces.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : ITF;
END_VAR
```

```
INTERFACE ITF
VAR_INPUT
  i : INT;
END_VAR
```

--> C0149: Variable declarations are not allowed in interfaces
