# Structure of the application

As usual for CNC applications, the application consists of path processing (`CNC_PreparePath` program) and interpolation (`CNC` program). Because the interpolation is not different from the other examples, it will not be discussed further here.

Path processing is called cyclically every 20 ms in the low-priority `PathTask`. The interpolation is called cyclically every 4 ms in the high-priority `MotionTask`. The background task `VISU_TASK` is defined for the visualization.

You can start processing the G-code in the visualization. The G-code and the movement of the machine (3D gantry) are displayed on the right side of the visualization.
