# Toolbars

## Overview

The Toolbars tab of the Tools > Customize dialog box allows you to modify existing toolbars or to create new ones.

The editor shows the defined toolbars. The respective commands are listed indented below in the same top-down order as they are displayed in the EcoStruxure Machine Expert menu. They can be separated by a separator line.

## Buttons

The following buttons are available:

| Button | Description |
| --- | --- |
| Add toolbar... | In the toolbar structure box, select the line above which you want to insert a new toolbar and click the Add toolbar... button. In the empty toolbar, which is added to the toolbar structure,enter a toolbar name.  To rename a toolbar within the configuration, you can directly modify its name in the toolbar list. In order to get into edit mode, select the respective entry in the list and press the Spacebar or click the selected entry. |
| Add command... | In the toolbar structure box, select the line above which you want to insert a new command, and click the Add command... button.  In the Add Command dialog box, select the command to be added and confirm by clicking the OK button. |
| Add separator | Click the button to insert a separator line above the marked command in the toolbar structure box. |
| Reset | Click the button to reset the toolbar to the default settings. |
| Load...  Save... | You can save the settings to an external file (<filename>.opt.tbar) or reload it from previously saved setting. |
