# Compiler Error C0066

## Message

Cannot compare type ‘<data type>’ with type ‘<data type>’

## Message Cause

Two data types are compared which cannot be compared with each other.

## Solution

Only compare data types that can be compared with each other.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  re : REAL;
  str: STRING;
  b : BOOL;
END_VAR
```

```
b := i > str;
```

--> C0066: Cannot compare type 'INT' with type 'STRING'

## Error Correction

```
b := i > re;
```
