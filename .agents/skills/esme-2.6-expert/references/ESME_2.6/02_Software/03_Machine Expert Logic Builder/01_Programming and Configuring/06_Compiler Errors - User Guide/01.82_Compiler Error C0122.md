# Compiler Error C0122

## Message

Expression ‘SUPER’ is not allowed in this context

## Message Cause

“SUPER^” is used outside of derived function blocks.

## Solution

Only use “SUPER^” in function blocks.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
SUPER^.METH(TRUE, TRUE);
```
