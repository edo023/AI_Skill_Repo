# Add Object

## Overview

The Project > Add Object command opens a submenu providing the objects available for getting inserted at the currently selected position in the navigators (Devices tree, Applications tree or Tools tree). For example, you can only add an Action object if you have selected a POU object in the Applications tree.

Select the desired object type from the Project > Add Object submenu. In the dialog box, define a Name which should be used for the object in the navigators. Refer to the [Consideration for the Naming of identifiers](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083600) in order to get the name as unique as possible. Depending on the object type, further configuration settings can be available. For further information on this, see the respective help page of the particular object type or the corresponding editor.

To rename an object in the navigators, click the entry to open an edit frame or use the [**Properties**](D-SE-0083921.html#D-SE-0083921) dialog box that is available via the contextual menu.

NOTE: A device object is to be inserted via a separate command [**Add Device...**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083370) .
