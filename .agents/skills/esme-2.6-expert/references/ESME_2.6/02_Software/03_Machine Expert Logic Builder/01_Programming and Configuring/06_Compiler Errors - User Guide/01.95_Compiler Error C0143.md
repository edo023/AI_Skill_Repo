# Compiler Error C0143

## Message

The property ‘<property name>’ cannot be used in this context because it lacks the Get accessor

## Message Cause

The property does not have Get accessor.

## Solution

Make sure that the property has a Get accessor definition.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  inst: FB;
END_VAR
```

```
i := inst.Prop;
```

```
FUNCTION_BLOCK FB
VAR
END_VAR
```

```
PROPERTY Prop : INT
Set;
```
