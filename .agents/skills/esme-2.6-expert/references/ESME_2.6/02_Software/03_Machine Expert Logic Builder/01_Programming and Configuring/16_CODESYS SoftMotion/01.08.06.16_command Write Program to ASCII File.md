# Command: Write Program to ASCII File

**Function**: The command opens the default dialog for saving a file to the file system. The CNC program can be saved as an ASCII file.

**Call**: **CNC** menu

**Requirement**: A CNC path is open in the editor.

|  |  |
| --- | --- |
| **Name** | Default: Name of the CNC object  Example: `cncSpiral_A` |
| **File type** | File extension: `TXT`, `GCODE`, `CNC` |
