# Edit Page Size

## Overview

The CFC > Edit Page Size command opens the Edit Page Size dialog box for modifying the size and the margin of the working area of page-oriented CFCs.

| Parameter | Description |
| --- | --- |
| Width | Width of the page (minimum 24, maximum 1024). Elements which will be outside of the page due to a width reduction will be marked in red color |
| Height | Height of the page (minimum 24, maximum 1024). Elements which will be outside of the page due to a height reduction will be marked in red color. |
| Margin width | Width of the margin column (minimum 6, maximum 25% of the page width). |
| Set as default for new CFC objects | If this option is activated, the page size of new CFC objects will be set to the current settings by default. |
