# Compiler Warning C0139

## Message

The code <code> has no effect. Is this intended?

## Message Cause

The written code is syntactically correct but does not do anything.

## Solution

Only write code that has a purpose.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
i;
```

--> C0139: The code 'i;' has no effect. Is this intended?

EIO0000003933.04
