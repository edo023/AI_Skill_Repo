# Compiler Error C0168

## Message

‘VAR\_CONFIG’ declaration is only allowed in a VAR\_CONFIG - list

## Message Cause

‘VAR\_CONFIG’ is used outside a VAR\_CONFIG list.

## Solution

Use ‘VAR\_CONFIG’ only in a VAR\_CONFIG list.

## Error Example

```
PROGRAM PLC_PRG
VAR_CONFIG
  i : INT;
END_VAR
```

--> C0168: 'VAR\_CONFIG' declaration is only allowed in a VAR\_CONFIG - list
