# Adding an Alarm Configuration

1. In the device tree, select an application.
2. Click **Project → Add Object → Alarm Configuration** to insert an alarm configuration.

   * The `Alarm Configuration` object is displayed below the application in the device tree. The alarm configuration already contains has the predefined `Error`, `Info`, and `Warning` alarm classes, as well as the `AlarmStorage`.

     It is helpful to use the predefined objects. However, you are free to delete these and program your own or additional objects.
