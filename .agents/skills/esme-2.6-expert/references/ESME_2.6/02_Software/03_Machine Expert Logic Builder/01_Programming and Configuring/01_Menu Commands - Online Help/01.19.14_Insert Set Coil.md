# Insert Set Coil

## Overview

The FBD/LD/IL > Insert Set Coil command inserts a [set coil](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083484). Alternatively, you can use the combination of the commands [**Insert Coil**](D-SE-0084116.html#D-SE-0084116) and [**Set/Reset**](D-SE-0084132.html#D-SE-0084132) to insert a set coil.
