# Release Forcelist

## Overview

The Online > Release Forcelist command can be used to release the forcing for all variables of the currently active application. It is not available in any menu per default, but must be added to the menu commands. Refer to the [Tool Menu Commands](D-SE-0084067.html#D-SE-0084067__D-SE-0084067.5).

The variables will obtain the current value read from the controller. This corresponds to option Release the force, without modifying the value, which can be activated in the Prepare Value [dialog box](D-SE-0084023.html#D-SE-0084023) for a forced variable.

Failure to follow these instructions can result in death, serious injury, or equipment damage. |

If option Secure online mode is activated in the communication settings of the respective device, you have to confirm after calling this command.
