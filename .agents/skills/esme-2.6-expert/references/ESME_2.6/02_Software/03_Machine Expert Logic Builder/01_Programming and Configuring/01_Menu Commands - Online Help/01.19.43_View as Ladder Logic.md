# View as Ladder Logic

## Overview

Shortcut: CTRL + 2

The FBD/LD/IL > View as Ladder Logic command can be used in offline and online mode to switch to the LD view of the [FBD/LD/IL editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083462).

The FBD networks or the IL will be converted appropriately to LD networks. Keep in mind that there are some special elements, which cannot be converted and thus will only be available in the appropriate editor view. Additionally, some constructs cannot be converted unambiguously (refer to the chapter [View as Instruction List](D-SE-0084146.html#D-SE-0084146) for the respective remarks).

FBD elements which cannot be displayed as LD elements (for example, `XOR`) will be displayed as FBD boxes within an LD network.

To switch back to the FBD view, execute the command [View as Function Block Diagram](D-SE-0084144.html#D-SE-0084144) .

To switch back to the IL view, execute the command [View as Instruction List](D-SE-0084146.html#D-SE-0084146).

NOTE: A proper conversion presumes syntactically correct code. Otherwise parts of the implementation can become lost.

NOTE: Concerning the view options for the components of FBD, LD and IL networks, consider the FBD, LD and IL editor options.
