# Reset user rights management to default

## Reset with EcoStruxure Machine Expert

The Online > Security > Reset user rights management to default is also available in the contextual menu of the controllers. It allows you to reset the device user rights to the default settings.

| CAUTION | |
| --- | --- |
|  | NO ACCESS WITH FTP, HTTP, OPC UA  Set your individual user name and password when you have reset the user rights management, and, where necessary, allow access to FTP, HTTP and OPC UA servers.  Failure to follow these instructions can result in injury or equipment damage. |

Then, you can confirm the message with OK to reset the user rights management to the default settings.

## Reset with Webserver

The Modicon M241 Logic Controller, Modicon M251 Logic Controller, and the Modicon M262 Logic/Motion Controller support the reset of device user rights management via the embedded webserver: MAINTENANCE > USER MANAGEMENT > USER ACCOUNTS MANAGEMENT > RESET TO DEFAULT.

Consult the *Programming Guide* specific to your controller for further information:

* [Modicon M241 Logic Controller](../../../../../api/crossBook?lang=en-US&virtualBookName=m241prg&topicID=D_SE_0002960)
* [Modicon M251 Logic Controller](../../../../../api/crossBook?lang=en-US&virtualBookName=m251prg&topicID=D_SE_0002960)
* [Modicon M262 Logic/Motion Controller](../../../../../api/crossBook?lang=en-US&virtualBookName=m262prg&topicID=D_SE_0002960)

## Reset with Controller Assistant

With EcoStruxure Machine Expert V1.2 and later, the service tool Controller Assistant supports user rights management of the controllers. Refer to the [*Controller Assistant User Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=ContrAs&topicID=D_SE_0104135) for further information.
