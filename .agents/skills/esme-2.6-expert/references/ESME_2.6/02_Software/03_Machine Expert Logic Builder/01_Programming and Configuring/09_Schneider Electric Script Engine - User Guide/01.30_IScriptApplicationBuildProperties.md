# IScriptApplicationBuildProperties

class

IScriptApplicationBuildProperties.

IScriptApplicationBuildProperties

Bases: object

This interface can be used to read and modify the application build properties.

download\_application\_content

Retrieves or sets a boolean value indicating whether application content is downloaded or not.

Type

bool

use\_dynamic\_memory

Retrieves or sets a boolean value indicating whether to use dynamic memory allocation or not. Convenience property for setting [size\_for\_dynamic\_memory](#iscriptapplicationbuildproperties_module_IScriptApplicationBuildProperties__iscriptapplicationbuildproperties_IScriptApplicationBuildProperties_IScriptApplicationBuildProperties_size_for_dynamic_memory) size to [-1] or default value [1000].

Type

bool

size\_for\_dynamic\_memory

Retrieves or sets the maximum size of memory to use for dynamic memory allocation in bytes. Value must be greater or equal to 1000. A value of [-1] means that this feature is not active.

Type

int
