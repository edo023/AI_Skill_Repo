# Device Type Manager (DTM) Installation

![](images/logo.png)

Device Type Manager (DTM) Installation

Overview

The device-specific software component is called a Device Type Manager (DTM), which is supplied with the device by the manufacturer.

DTM installation in EcoStruxure Machine Expert is composed of two steps:

oInstall the DTMs on the PC with EcoStruxure Machine Expert. This makes the DTMs detectable by EcoStruxure Machine Expert.

oImport the DTMs in the Device Repository of EcoStruxure Machine Expert.

DTM Installation on the PC

A base set of DTMs is included with the EcoStruxure Machine Expert software delivery and can be installed during the EcoStruxure Machine Expert software installation by the Machine Expert Installer.

You can download other Schneider Electric DTMs from [www.schneider-electric.com](http://www.schneider-electric.com).

Refer to your DTMs provider to install third-party DTMs on your PC.

NOTE: The DTMs are installed/uninstalled on the PC that [runs](../glossary/glossary.htm#XREF_D_SE_0024697_591) EcoStruxure Machine Expert and are accessible in the Start > Control Panel > Add or Remove Programs window.

DTM Uninstallation on the PC

If you installed a DTM using the Machine Expert Installer, you must use the Machine Expert Installer to uninstall the DTM. If you installed a DTM not using the EcoStruxure Machine Expert Configuration Manager, you must use Start > Control Panel > Add or Remove Programs window to uninstall the DTM.
