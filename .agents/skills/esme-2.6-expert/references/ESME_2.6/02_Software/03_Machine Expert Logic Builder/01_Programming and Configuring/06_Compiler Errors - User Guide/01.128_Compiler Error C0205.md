# Compiler Error C0205

## Message

POINTER TO BIT is not allowed

## Message Cause

An attempt is made to declare a POINTER TO BIT.

## Solution

Do not declare a POINTER TO BIT.

## Error Message

```
PROGRAM PLC_PRG
VAR
  pt : POINTER TO BIT;
END_VAR
```

--> C0205: POINTER TO BIT is not allowed

EIO0000003933.04
