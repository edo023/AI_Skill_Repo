# Compiler Error C0015

## Message

Counter initialization expected

## Message Cause

Syntax error in a FOR loop. The counter variable is not initialized correctly.

## Solution

Correct the syntax of the FOR loop.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i: INT;
END_VAR
```

```
FOR i TO 10 DO
  ;
END_FOR
```

--> C0015: Counter initialization expected

## Error Correction

```
FOR i := 0 TO 10 DO
  ;
END_FOR
```
