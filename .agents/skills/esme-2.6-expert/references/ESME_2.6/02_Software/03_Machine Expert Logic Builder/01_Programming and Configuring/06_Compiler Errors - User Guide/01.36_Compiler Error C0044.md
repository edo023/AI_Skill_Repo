# Compiler Error C0044

## Message

Assignment to input is missing for parameter ‘<input variable name>’ in call of ‘<function block name>’

## Message Cause

A parameter is passed, although an input variable is not declared.

## Solution

Declare an input variable.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : FB;
END_VAR
```

```
inst(1);
```

```
FUNCTION_BLOCK FB
VAR_INPUT
END_VAR
```

--> C0044: Assignment to input is missing for parameter '1' in call of 'FB'

## Error Correction

```
FUNCTION_BLOCK FB
VAR_INPUT
  in : INT;
END_VAR
```
