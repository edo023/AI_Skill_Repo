# Add All Forces to Watchlist

## Overview

The Online > Add All Forces to Watchlist command is available in online mode when one of the [watch](D-SE-0083929.html#D-SE-0083929) views (Watch 1, Watch 2, Watch 3 or Watch 4) is active. It serves to add all currently prepared or already forced variables of the active application to this watchlist. The adding of force values to watch views is only valid for docked watch views.

NOTE: You can also use with the [watch view](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083546) Watch all Forces, which automatically contains all currently prepared or forced values and additionally provides commands for releasing the forces.
