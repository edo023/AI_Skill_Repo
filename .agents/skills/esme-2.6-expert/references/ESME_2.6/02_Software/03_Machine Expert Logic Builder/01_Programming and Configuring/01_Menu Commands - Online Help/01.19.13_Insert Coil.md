# Insert Coil

## Overview

Shortcut: CTRL + A

The FBD/LD/IL > Insert Coil command is used to insert a [coil](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083488) in parallel to the previous coils.

If the [marked position](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083469) is a connection between the [contacts](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083487) and the coils, then the new coil will be inserted as the last. If the marked position is a coil, then the new coil will be inserted directly above it.

By default, the coil is given the text `???`. You can click this text and change it to the desired variable. You can also use the Input Assistant for this purpose.

For the possibility of entering addresses, of linebreaks for variable names and of comments per coil, refer to the description of the Options [dialog box for FBD, LD, and IL editors](D-SE-0084056.html#D-SE-0084056).
