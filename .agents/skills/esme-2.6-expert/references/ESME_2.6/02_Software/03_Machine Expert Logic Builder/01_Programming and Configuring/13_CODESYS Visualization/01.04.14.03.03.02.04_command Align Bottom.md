# Command: Align Bottom

Symbol: ![](images/uuid-8da2f447-7855-d178-b7b6-eab7a0.png)

**Function**: The command aligns the selected visualization elements in a line at the top edge of the element which is positioned furthest up.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Multiple elements are selected.
