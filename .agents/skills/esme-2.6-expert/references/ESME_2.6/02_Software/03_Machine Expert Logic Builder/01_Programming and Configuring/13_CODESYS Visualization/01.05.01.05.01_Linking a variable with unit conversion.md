# Linking a variable with unit conversion

Requirement: A project with a visualization is open. In addition, the application contains the `UnitConversion` object with the `convert_A` rule.

1. Select an element.

   * The **Properties** view opens.
2. Compile, download, and start the application.

   * The application runs. The visualization opens. The unit conversion is applied.
