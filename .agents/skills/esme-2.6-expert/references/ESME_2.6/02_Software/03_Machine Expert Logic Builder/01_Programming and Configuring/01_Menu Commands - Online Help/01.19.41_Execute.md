# Execute

## Presentation

The FBD/LD/IL > Execute command inserts the `Execute` [block element](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083485) in an FBD or LD network for execution by ST code.

[Enter the desired ST code](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083485) after inserting the element into an ST editor integrated in the function block.
