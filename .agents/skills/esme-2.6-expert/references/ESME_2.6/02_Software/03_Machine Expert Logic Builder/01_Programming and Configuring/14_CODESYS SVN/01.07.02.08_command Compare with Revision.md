# Command: Compare with Revision

Symbol: ![](images/uuid-509a3782-0018-e77d-6f36-2e0c3f.png)

**Function**: The command opens the **Project log** dialog or **Log – <object>** where the version history is displayed from the project or an object of the CODESYS project. There you can select a revision. A tab opens and shows the result of the comparison of your working copy and the revision.

**Call**: Context menu: **SVN**

**Requirement**: The object is versioned and not conflicted.

Multiple tabs can be open at the same time with the comparison of different objects.

TIP:

**Comparison by object type**

The comparison dialog makes use of the functionality of the **Project → Compare** command in CODESYS. As a result, objects are compared according to their object type.
