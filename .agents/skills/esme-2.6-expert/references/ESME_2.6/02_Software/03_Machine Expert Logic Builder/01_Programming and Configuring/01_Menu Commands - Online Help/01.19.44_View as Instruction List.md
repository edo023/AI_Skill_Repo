# View as Instruction List

## Overview

Shortcut: CTRL + 3

The FBD/LD/IL > View as instruction List command can be used in offline and online mode to switch to the IL view of the [FBD/LD/IL editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083462).

NOTE: Some elements cannot get converted and thus the respective network will remain in the original editor view (for example, operators `CALC, CALCN`). In case of detected syntax errors, no conversion is possible; a corresponding error message will be generated.

Some constructs cannot be converted unambiguously and thus will be normalized after a conversion from IL to FBD/LD and back. This applies to negations and explicit/implicit assignment of function block in- and outputs.

To switch back to the FBD view, execute the command [View as Function Block Diagram](D-SE-0084144.html#D-SE-0084144).

To switch back to the LD view, execute the command [View as Ladder Logic](D-SE-0084145.html#D-SE-0084145).

NOTE: A proper conversion presumes syntactically correct code. Otherwise parts of the implementation can become lost.

NOTE: Concerning the view options for the components of FBD, LD and IL networks, consider the FBD, LD and IL editor options.
