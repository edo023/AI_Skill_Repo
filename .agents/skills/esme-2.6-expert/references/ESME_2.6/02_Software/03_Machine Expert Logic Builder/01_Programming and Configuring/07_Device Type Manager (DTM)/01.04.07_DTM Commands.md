# DTM Commands

![](images/logo.png)

DTM Commands

Overview

The availability of DTM commands depends on the device and the state of the device.

DTM commands are available by either:

oDouble-clicking the device node in the Devices tree to display the Configuration of the device tab, which has a toolbar containing a subset of the DTM commands.

oRight-clicking the device node in the Devices tree to display contextual menu commands.

If a dialog or function is currently not available, the corresponding menu command or toolbar icon is grayed out.

DTM Commands

This table describes the DTM commands:

| Toolbar Icon | Menu Command | Description |
| --- | --- | --- |
| N/A | Configuration | [Accesses the configuration user interface](DTMs_in_ESME-8.htm#XREF_D_SE_0057936_1), which allows you to configure the selected device.  NOTE: This interface is specific to the device and is provided by the DTM manufacturer. |
| G-SE-0053470.1.gif-high.gif | Edit Connection | Displays the [Edit Connection window](DTMs_in_ESME-7.htm#XREF_D_SE_0057935_1), which allows you to configure the connection parameters. |
| G-SE-0053468.1.gif-high.gif | Connect | Connects to the device. |
| N/A | Upload and Connect | Connects to the device and transfers the configuration from the selected device to the PC. |
| G-SE-0053466.1.gif-high.gif | Disconnect | Disconnects from the device. |
| G-SE-0053462.1.gif-high.gif | Upload from Device | Transfers the configuration from the selected device to the PC. |
| G-SE-0053464.1.gif-high.gif | Download to Device | Transfers the configuration from the PC to the selected device. |
| Device | Advanced configuration > Additional functions | Opens a submenu containing additional menu commands representing operations on the selected device.  NOTE:  These additional operations are specific to the device and are provided by the DTM manufacturer. If no additional operations are available, this menu command appears dimmed. |
| Documentation | Advanced configuration > Documentation | Opens a submenu containing a list of additional information provided by the DTM manufacturer for the selected device.  NOTE: If no additional information is provided, this menu command appears dimmed. |
