# Command: Make Same Size

Symbol: ![](images/uuid-50ec3cc5-59c8-8ce7-cadb-a161fc.png)

**Function**: The command adjusts the size of the selected visualization elements to the size of the blue selected element.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Multiple elements are selected. The first element is blue and the other elements are gray.

TIP:

The command does not work for lines or polygons.
