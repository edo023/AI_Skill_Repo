# Keywords

## Overview

Write keywords in uppercase letters in the editors.

The following strings are reserved as keywords. They cannot be used as identifiers for variables or POUs:

* `-`
* `&`
* `(`
* `)`
* `*`
* `,`
* `.`
* `..`
* `/`
* `:`
* `:=`
* `;`
* `[`
* `]`
* `^`
* `__CATCH`
* `__CHECKLICENSE`
* `__CHECKLICENSEBIT`
* `__DELETE`
* `__ENDTRY`
* `__FINALLY`
* `__ISVALIDREF`
* `__MEMORYBARRIER`
* `__NEW`
* `__QUERYINTERFACE`
* `__QUERYPOINTER`
* `__THROW`
* `__TRY`
* `__XADO`
* `Ι`
* `+`
* `<`
* `<=`
* `<>`
* `=`
* `>=`
* `>`
* `=>`
* `ABS`
* `ACOS`
* `ADR`
* `AND`
* `AND_THEN`
* `ASIN`
* `ATAN`
* `BITADR`
