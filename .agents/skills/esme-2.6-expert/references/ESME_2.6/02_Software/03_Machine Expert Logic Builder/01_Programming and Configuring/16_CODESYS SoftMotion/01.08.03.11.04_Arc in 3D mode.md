# Arc in 3D mode

In 3D mode, the system defines the arcs by the interpolation plane that is defined by means of G15-G19.

**Example**

**One semicircle in the X/Z-plane and one semicircle in the X/Y-plane**

```
N10 G1 X100 Y100
N15 G18
N20 G2 X200 R50
N25 G17
N30 G3 X100 R50
```
