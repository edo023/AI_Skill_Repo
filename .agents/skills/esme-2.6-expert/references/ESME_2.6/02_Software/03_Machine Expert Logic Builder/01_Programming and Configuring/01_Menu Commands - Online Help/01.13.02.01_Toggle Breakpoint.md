# Toggle Breakpoint

## Overview

Default shortcut: F9

The Debug > Toggle Breakpoints command basically toggles between status enabled and disabled of a [breakpoint](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083451). However, it also confirms that a new breakpoint will be set if one has not already been set at the current [breakpoint position](D-SE-0084030.html#D-SE-0084030). (For defining a new breakpoint consider also the possibility of using the [New Breakpoint](D-SE-0084029.html#D-SE-0084029) dialog box.)

NOTE: Each active breakpoint will obtain an inactive status when you leave online mode and login again afterwards.

An active breakpoint is indicated by the ![](images/T003483105.gif) symbol; an inactive one by the ![](images/T003483104.gif) symbol.

NOTE: See also the corresponding functions in the Breakpoint [dialog box](D-SE-0083918.html#D-SE-0083918).
