# Compiler Error C0241

## Message

Second operand of \_\_QueryPointer must be pointer

## Message Cause

Incorrect operands are passed to the operator \_\_QueryPointer.

## Solution

Pass a pointer.

## Error Example

```
FUNCTION_BLOCK FB
PROGRAM PLC_PRG
VAR
  b : INT;
  ITFref : ITF;
  pt : POINTER TO FB;
END_VAR
```

```
__QueryPointer(ITFref,b);
```

```
INTERFACE ITF EXTENDS __System.IQueryInterface
```

--> C0241: Second operand of \_\_QueryPointer must be pointer

## Error Correction

```
__QueryPointer (ITFref, pt);
```
