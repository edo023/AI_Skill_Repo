# Source Download...

## Overview

The File > Source download... command is used for creating and transferring an archive file of the actual project to any device.

After you have executed the command, the progress of the source download or upload will be indicated in the status bar. A Cancel button is available in the lower right corner to abort the download.

By default, the name of the archive file is *archive.prj*. You can reload the source code to the programming system in offline mode by executing the command [Source upload](D-SE-0083998.html#D-SE-0083998).

The default settings concerning destination device, content, and timing for the source download are defined in the Project Settings, [**Source Download**](D-SE-0083956.html#D-SE-0083956).
