# Command: Remove Horizontal Spacing

Symbol: ![](images/uuid-ff2ffce3-2cb1-3dde-b718-6e304d.png)

**Function**: The command aligns the selected visualization elements so that the blue element keeps its position and the other elements are positioned horizontally without space between them.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Multiple elements are selected.
