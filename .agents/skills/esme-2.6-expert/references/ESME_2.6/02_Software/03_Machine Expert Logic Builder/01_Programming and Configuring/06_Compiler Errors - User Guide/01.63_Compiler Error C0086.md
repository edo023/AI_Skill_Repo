# Compiler Error C0086

## Message

No definition found for the interface ‘<interface name>’

## Message Cause

An undefined interface is used.

## Solution

Define the interface.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : FB;
END_VAR
```

```
FUNCTION_BLOCK FB IMPLEMENTS XY
VAR
END_VAR
```

--> C0086: No definition found for the interface 'XY'

## Error Correction

```
INTERFACE XY
```
