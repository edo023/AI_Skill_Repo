# Compiler Warning C0344

## Message

The monitoring attribute is not supported for property <property name> and will be ignored.

## Message Cause

A property has the monitoring attribute `'call'` and returns a structure.

## Solution

Use the monitoring attribute `'variable'`.

## Error Example

```
TYPE ST_Test :STRUCT
      a,b : INT;  
END_STRUCT
END_TYPE

FUNCTION_BLOCK FB_Test
...

{attribute 'monitoring' := 'call'}
PROPERTY TestProp : ST_Test
```

--> C0344: The monitoring attribute is not supported for property 'TestProp' and will be ignored.
