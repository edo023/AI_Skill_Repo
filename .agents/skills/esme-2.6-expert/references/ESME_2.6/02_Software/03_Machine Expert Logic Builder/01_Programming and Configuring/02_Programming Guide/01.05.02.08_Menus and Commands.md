# Menus and Commands

## Overview

The following figure shows the default menu bar:

![](images/T003224965.gif)

Some commands are not visible in the default view. To add a command to a menu or reorganize the menu structures, use the Tools > Customize [dialog box](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0084066).

Specific commands, for a particular editor for example, are usually available in a corresponding menu. These commands are only visible when the editor is open. For example: when you edit an object in the SFC editor, the SFC menu is added to the menu bar.

For a description of the menus and commands, refer to the separate [Menu Commands Online Help](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0083893).
