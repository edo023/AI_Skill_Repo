# Watch View / Watch List Editor

## Overview

A watch list is a user-defined set of project variables. They are displayed in the watch view for [monitoring](D-SE-0083546.html#D-SE-0083546__D-SE-0083546.2) the values in a table. Also, writing and forcing of the variables is possible within the watch view.

Open a watch view via the Watch command submenu (by default in the View menu). It provides an editor for [creating watch lists](D-SE-0083545.html#D-SE-0083545).

By default, you can set up 4 individual watch lists in the watch views Watch 1, Watch 2, Watch 3, Watch 4. View Watch all Forces in online mode gets [filled](D-SE-0083546.html#D-SE-0083546__D-SE-0083546.4) automatically with all currently forced values of the active application.
