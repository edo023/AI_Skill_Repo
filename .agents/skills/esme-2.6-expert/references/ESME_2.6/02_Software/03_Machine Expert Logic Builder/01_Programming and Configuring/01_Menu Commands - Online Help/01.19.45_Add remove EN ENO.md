# Add/remove EN/ENO

## Overview

The Add/remove EN/ENO command is used to add/remove EN/ENO input/outputs to/from an existing function block box in the FBD/LD editor.

The command is available in the contextual menu when a box has been selected.

The already defined connections of inputs and outputs remain unchanged, except the one on the first output which is connected to the ENO output.
