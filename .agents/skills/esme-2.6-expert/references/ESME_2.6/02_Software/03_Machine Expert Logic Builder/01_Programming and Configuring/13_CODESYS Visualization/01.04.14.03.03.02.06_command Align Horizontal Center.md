# Command: Align Horizontal Center

Symbol: ![](images/uuid-69254dd7-8a71-9dae-09f8-c32ee2.png)

**Function**:The command aligns the selected visualization elements to their common horizontal center.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Multiple elements are selected.
