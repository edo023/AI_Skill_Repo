# SA0112: Reference variables

Detects variables of type `REFERENCE TO`

Justification: The IEC 61131-3 standard does not permit references.

Importance: Low

**Example**

```
VAR
    ref_int : REFERENCE TO INT;
    ref_dw : REFERENCE TO DWORD;
END_VAR
```
