# Visualization Element: Group Box

Symbol:

![](images/uuid-b6678ddd-7657-4182-0c0d-5e6d44.png)

Category: **Common Controls**

The element provides a visual grouping of visualization elements. The group box can have multiple levels of nesting.

TIP:

You can also use drag elements to add them to a **Group Box**. To do this, drag the element to the window area of the **Group Box**. The appearance of the cursor changes (a small plus sign is displayed). When you click the **Shift** key at the same time, the element is not added.

You can remove elements from the **Group Box** by dragging them out of the window area.
