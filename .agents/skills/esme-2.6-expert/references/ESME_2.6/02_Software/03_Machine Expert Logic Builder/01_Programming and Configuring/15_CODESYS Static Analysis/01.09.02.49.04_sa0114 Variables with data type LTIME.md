# SA0114: Variables with data type LTIME

Detects variables of type `LTIME`

Justification: Not all systems support `LTIME` The code is more easily portable without `LTIME`.

Importance: Low

**Example**

```
VAR
    ltVar : LTIME; // SA0114
END_VAR
```
