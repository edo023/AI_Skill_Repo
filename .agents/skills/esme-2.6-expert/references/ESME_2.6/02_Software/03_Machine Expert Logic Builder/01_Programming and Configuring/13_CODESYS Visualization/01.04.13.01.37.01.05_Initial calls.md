# Initial calls

These method calls are executed during initialization. They are executed in the first cycle only.

|  |  |
| --- | --- |
| **Method calls** | Button '**Create new**'  Creates a subnode below **Methods** with parameters for the method call. |
| **Methods** | ****[<number>]****   * **Method**: Name of the method * **Parameter**: Parameter passed at the method call * **Result parameter**: Optional variable for the return value of the method |
