# Compiler Error C0213

## Message

VAR\_GLOBAL or VAR\_CONFIG expected instead of <declaration>

## Message Cause

In a global variable list (GVL), there is a variable declaration outside of a `VAR_GLOBAL` block.

## Solution

Move the variable declaration to the `VAR_GLOBAL` or `VAR_CONFIG` block.

## Error Example

```
VAR_GLOBAL
END_VAR
i : INT;
```

--> C0213: VAR\_GLOBAL or VAR\_CONFIG expected instead of i:INT

EIO0000003933.04
