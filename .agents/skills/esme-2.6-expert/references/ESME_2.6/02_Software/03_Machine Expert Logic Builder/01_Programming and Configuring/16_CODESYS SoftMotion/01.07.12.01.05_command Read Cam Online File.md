# Command: Read Cam Online File

**Function**: The command reads an external file with cam data. The file extension is `CAM`. The cam is displayed in the cam editor.

**Call**: **Cam** menu

**Requirement**: The cam editor is open.

The [Write Cam Online File](_sm_cmd_cam_write_online_file.html#_sm_cmd_cam_write_online_file) command creates an appropriate file in `CAM` format.
