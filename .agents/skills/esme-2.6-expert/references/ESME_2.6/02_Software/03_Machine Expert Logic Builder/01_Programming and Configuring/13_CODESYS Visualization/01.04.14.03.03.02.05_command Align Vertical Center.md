# Command: Align Vertical Center

Symbol: ![](images/uuid-fb255ae4-e65a-88ef-974f-bd51f6.png)

**Function**: The command aligns the selected visualization elements to their common vertical center.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Multiple elements are selected.
