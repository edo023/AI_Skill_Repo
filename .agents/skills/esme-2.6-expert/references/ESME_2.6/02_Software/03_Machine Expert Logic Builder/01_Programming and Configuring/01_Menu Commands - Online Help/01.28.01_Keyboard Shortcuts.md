## Browse Menu

The following keyboard shortcuts are available for commands of the Browse menu:

| Keyboard shortcut | Command name | Description |
| --- | --- | --- |
| Ctrl + Shift + D | Go To Definition | Browses the project for the line or POU which contains the definition of the POU or variable corresponding to the selected identifier. |
| Ctrl + Shift + F | Go To Instance | Opens the instance of the selected function block in a new window when executed in online mode. |
| Ctrl + Shift + G | Go To Implementation | Opens the implementation of the selected method call or property when executed in online mode. |
| Ctrl + Shift + H | Go To Reference | Opens the declaration location of the variable referenced by the selected pointer variable when executed in online mode. |

## Bookmarks Menu

The following keyboard shortcuts are available for commands of the Bookmarks menu:

| Keyboard shortcut | Command name | Description |
| --- | --- | --- |
| F12 | Next Bookmark (Active Editor) | Goes to the next bookmark in the active editor. |
| Shift + F12 | Previous Bookmark (Active Editor) | Goes to the previous bookmark in the active editor. |
| Ctrl + F12 | Toggle Bookmark | Toggles the bookmark at the present editor position |

## Breakpoints Menu

The following keyboard shortcuts are available for commands of the Breakpoints menu:

| Keyboard shortcut | Command name | Description |
| --- | --- | --- |
| F8 | Step Into | If the present statement is a call statement, steps into the called POU. Otherwise, a step over is performed. |
| F9 | Toggle Breakpoint | Toggles the breakpoint at the present position. |
| F10 | Step Over | Steps over the next statement. |
| Shift + F10 | Step Out | Jumps back to the calling instruction or to the start of the application. |

## Shortcuts for Accessing Clipboard Functions

The following keyboard shortcuts are available for commands accessing Clipboard functions:

| Keyboard shortcut | Command name | Description |
| --- | --- | --- |
| Del | Delete | Deletes the present selection. |
| Shift + Ins | Paste | Pastes the Clipboard contents to the present position. |
| Shift + Del | Cut | Cuts the present selection out and moves it to the Clipboard. |
| Ctrl + Ins  Ctrl + C | Copy | Copies the present selection to the Clipboard. |
| Ctrl + A | Select All | Selects the entire content of the active editor. |
| Ctrl + V | Paste | Pastes the Clipboard contents to the present position. |
| Ctrl + X | Cut | Cuts the present selection out and moves it to the Clipboard. |

## Shortcuts for Editor View Commands

The following keyboard shortcuts are available in editor views:

| Keyboard shortcut | Command name | Description |
| --- | --- | --- |
| Ctrl + J | Navigate backward | Allows you to jump to the last cursor position within an editor view. As a history of last cursor positions is stored, use the shortcut again to move to previous positions. To move forward in the history, use the Navigate forward shortcut. |
| Ctrl + Shift + J | Navigate forward | Allows you to move forward within the history of cursor positions within editor views. To move backward within the history, use the Navigate backward shortcut. |

## Shortcuts for Online Operations

| Keyboard shortcut | Command name | Description |
| --- | --- | --- |
| F5 | Start | Starts the active application. |
| F7 | Force Values | Forces the prepared values to the controller. |
| Shift + F8 | Stop | Stops the active application. |
| Ctrl + F5 | Single Cycle | Performs one single cycle on the active application. |
| Ctrl + F7 | Write Values | Writes the prepared values to the controller. |

| Alt + F7 | Unforce Values | Unforce the variables that are being forced, keeping the present values. |
| Alt + F8 | Login | Connects to the active application and switches to online mode. |
