# Set Active Path

## Overview

The Set Active Path command is also available via the Set active path button in the Communication Settings [view of the device editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083386). This means that all actions concerning communication will refer to this channel. The command will also be executed by double-clicking the channel entry.

The current active path will be displayed bold in the list, and (active) will be appended:

![](images/T003483396.gif)

If a device has been set as the active path, but was not reachable during the last scan network, it will be displayed in italics.

NOTE: The information concerning the device that was recently set as active path is stored on the local system and can be automatically restored at a [login](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083444) without communication settings configured.
