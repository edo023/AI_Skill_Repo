# Parallel / Alternative

## Parallel

The SFC Editor > Parallel command transforms the currently selected [alternative branch](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083503) to a [parallel branch](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083503).

NOTE: After a branch transformation, check and adapt the chart appropriately. Arrange steps and transitions as required for the respective type of branching.

## Alternative

The SFC Editor > Alternative command transforms a [parallel branch](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083503) to an [alternative branch](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083503).

NOTE: After a branch transformation, check and adapt the chart appropriately. Arrange steps and transitions as required for the respective type of branching.
