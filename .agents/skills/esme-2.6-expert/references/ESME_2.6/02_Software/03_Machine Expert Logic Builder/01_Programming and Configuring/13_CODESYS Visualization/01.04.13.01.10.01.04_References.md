# References

|  |  |
| --- | --- |
| **Configure** Button | Opens the **Frame Configuration** dialog. The dialog allows you to manage the referenced visualizations. |
| **References** | Visualization references which have been selected in the **Frame Configuration** dialog |
| Name of the visualization reference | Example: `visRectangle` |
| **Heading** | Heading in the tab of the tab element  Example: `Panel` |
| **Image ID** | Image ID in the theme `<image pool name>.<ID>`  Example: `Imagepool_A.1` for the image with ID `1` in `Imagepool_A` |
| **Invisible** | Boolean variable to dynamically hide the corresponding tab  Example: `PLC_PRG.bInvisible`  `TRUE`: The tab is hidden.  Note: This function is not implemented for the integrated visualization (no `VISU_TASK)`. |
| Interface parameter of the visualization reference | Example: `iX`  If the visualization has an interface with variables or parameters, then they are listed here with the variable name (as child properties) with the initialization value (next to it under the **Value** column).  Hint: You can find the data type of the interface parameters in the [interface editor](_visu_cmd_interface_editor.html#_visu_cmd_interface_editor). |
