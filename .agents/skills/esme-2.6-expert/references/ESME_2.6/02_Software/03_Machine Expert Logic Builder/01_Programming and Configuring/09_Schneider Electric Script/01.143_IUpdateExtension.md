block\_project\_update\_dialog

block\_dialog

tool\_version=None

With this, the project update dialog box (only available in UI-mode) can be blocked for the loaded project. This block\_dialog-flag is only valid for the specified tool version.

Parameters

* block\_dialog (bool) -- If set to TRUE, the project update dialog box will not appear for given version. If set to FALSE, the dialog box will appear again if some components (devices, libraries,..) have to be updated.
* tool\_version -- If this parameter is not set, the version of the started instance is used. This is the python command to get this version:

engineering\_environment.build
