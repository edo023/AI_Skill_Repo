# Compiler Error C0145

## Message

Interfaces can only be implemented by function blocks

## Message Cause

An attempt is made to implement an interface outside a function block.

## Solution

Only implement interfaces in function blocks.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
POU();
```

```
FUNCTION POU IMPLEMENTS ITF
VAR
END_VAR
```

--> C0145: Interfaces can only be implemented by function blocks
