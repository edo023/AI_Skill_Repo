# Trend Recording

A trend element can be added to a visualization. For details regarding the configuration of trend visualization, refer to [Displaying Data Graphs with a Trend Element](../../../../../api/crossBook?lang=en-US&virtualBookName=CODESYS_Visualization&topicID=_visu_struct_display_data_progress_with_trend).

NOTE: A trend element can only be configured if a Webvisualisation object has been added to the Visu Manager object.
