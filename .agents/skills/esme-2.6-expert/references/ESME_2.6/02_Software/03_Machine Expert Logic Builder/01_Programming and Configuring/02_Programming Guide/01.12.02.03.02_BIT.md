# BIT

## Overview

You can use the BIT data type only for particular variables within [structures](D-SE-0083673.html#D-SE-0083673) or in a [function block](D-SE-0083417.html#D-SE-0083417). The possible values are TRUE (1) and FALSE (0).

A BIT element consumes 1 bit of memory space and allows you to address single bits of a structure by name (for further information, refer to the paragraph [*Bit Access in Structures*](D-SE-0083673.html#D-SE-0083673__D-SE-0083673.7)). Bit elements which are declared one after another will be combined in bytes. In contrast to [BOOL types](D-SE-0083662.html#D-SE-0083662__D-SE-0083662.3), where 8 bits are reserved in any case, the use of memory space can get optimized. On the other hand, the access to bits takes definitely more time. For that reason, use the BIT data type if you want to store several boolean pieces of information in a compact format.
