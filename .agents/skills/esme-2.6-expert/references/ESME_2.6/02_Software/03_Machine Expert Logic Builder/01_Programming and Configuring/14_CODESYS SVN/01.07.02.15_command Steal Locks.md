# Command: Steal Locks

Symbol: ![](images/uuid-fc641de1-ce60-6b77-04b5-109098.png)

**Function**: The command steals the lock of the object and all dependent objects when these are also locked. The **Lock Message** dialog opens for this purpose. However, the dependent objects are not displayed in this dialog.

**Call**: Context menu: **SVN**

**Requirement**: The versioned object is locked by someone else (overlaid with the ![](images/uuid-fc641de1-ce60-6b77-04b5-109098.png) symbol).
