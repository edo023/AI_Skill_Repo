# Compiler Warning C0508

## Message

Ambiguous use of name '<identifier>'.

## Message Cause

A local variable of a program has the same name as an action of the program.

## Solution

Change the name of the local variable.

## Error Example

```
FUNCTION_BLOCK FBRead_Var
VAR_INPUT
END_VAR
...

PROGRAM PLC_PRG
VAR
    read_input : FBRead_Var;
END_VAR
read_input(); // Call of the action read_input
```

--> C0508: Ambiguous use of name 'READ\_INPUT'.

EIO0000003933.04
