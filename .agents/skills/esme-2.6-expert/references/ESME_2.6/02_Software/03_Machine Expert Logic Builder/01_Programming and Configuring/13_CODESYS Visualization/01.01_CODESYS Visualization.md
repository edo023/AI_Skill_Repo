# CODESYS Visualization

The CODESYS Visualization add-on allows for creating user interfaces which are fully integrated in CODESYS Development System based on available visualization elements. These are displayed on a target device, on a PC, or in the web browser. This is also possible simultaneously based on the same record.

You can create structured user interfaces which consist of visualizations interconnected in any complex way.

You can monitor a user interface together with your application, or use it for testing and commissioning.

**Additional functionality:**

* Instantiation of function blocks
* Calling of IEC functions
* Array access
* Recording of real-time data
* Extension of the collection of visualization elements

You can download CODESYS Visualization from the [CODESYS Store](https://store.codesys.com/de/codesys.html). There you will also find the data sheet for the product.
