# Line width variable

Dynamic definition of the weight of a line element using a variable

|  |  |
| --- | --- |
| **Integer value** | Variable (integer data type) for the line width of the element (in pixels)  This overwrites the fixed value that is defined in **Appearance → Line weight**.  NOTE: The value 0 codes the same as 1 and sets the line weight to one pixel. |
