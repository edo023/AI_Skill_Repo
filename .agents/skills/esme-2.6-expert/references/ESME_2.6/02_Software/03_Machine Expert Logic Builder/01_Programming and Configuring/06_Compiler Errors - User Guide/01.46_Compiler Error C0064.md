# Compiler Error C0064

## Message

Dereferencing requires a pointer

## Message Cause

A variable that is not a pointer variable is dereferenced.

## Solution

Dereference only variables that are pointer variables.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  pi : POINTER TO INT;
END_VAR
```

```
i^:=1;
```

--> C0018: 'i^' is not a valid assignment target

--> C0064: Dereferencing requires a pointer

## Error Correction

```
pi := ADR(i);
pi^ := 1;
```
