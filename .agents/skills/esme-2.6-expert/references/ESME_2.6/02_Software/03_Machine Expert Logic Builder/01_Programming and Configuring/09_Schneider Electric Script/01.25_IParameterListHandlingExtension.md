# IParameterListHandlingExtension

class

IParameterListHandlingExtension.

IParameterListHandlingExtension

Bases: object

Extends a IScriptApplication with the ability to add new parameter list objects.

add\_parameter\_list

name

Adds a new parameter list to the application.

Parameters

name (string) -- The name of the new parameter list.

Reports

System.ArgumentException -- name is null or empty.

This example shows how to add a new parameter list to an existing application.

```
p = projects.primary
app = p.active_application

# create a new parameter list with the name 'ParameterList'
app.add_parameter_list('ParameterList')
```

Returns

A script object

Return type

IExtendedObject
