# Command: Connect to Existing Project

Symbol: ![](images/uuid-e5f62589-a484-31f1-c570-b518c8.png)

**Function**: The command opens the **Connect to SVN repository** dialog. In the dialog, you define the URL and the revision of the SVN repository with which the unversioned project is connected.

**Call**: **Project → SVN** menu

**Requirement**: The project is disconnected from SVN.

IMPORTANT:

Only users who have read access to the entire project (see the CODESYS user and access management) can import the project into the SVN repository or can link to an existing database project.

IMPORTANT:

This command functions reliably only when the project has already been imported into SVN and then disconnected by means of the [Disconnect project from SVN](_svn_cmd_disconnect_from_subversion.html#_svn_cmd_disconnect_from_subversion) command.
