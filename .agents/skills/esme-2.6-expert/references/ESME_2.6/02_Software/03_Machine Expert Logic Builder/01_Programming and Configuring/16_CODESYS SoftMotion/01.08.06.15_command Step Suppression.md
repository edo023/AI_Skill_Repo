# Command: Step Suppression

Symbol: ![](images/uuid-27378699-19df-f5f6-da04-774cdf.png)

**Function**: The command activates and deactivates the step suppression.

**Call**: **CNC** menu

**Requirement**: A CNC path is open in the editor. The CNC program is implemented as DIN66025.

If step suppression is activated, then blocks in the CNC program that begin with `/` are ignored.

If the CNC program is implemented as a table, then the command is not available.
