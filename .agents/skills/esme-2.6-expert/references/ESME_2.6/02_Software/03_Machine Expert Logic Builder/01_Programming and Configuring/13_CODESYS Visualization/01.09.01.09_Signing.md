# Signing

In the **Visualization Element Repository** dialog, you can sign an HTML5 element and manage its certificates.

TIP:

HTML5 controls can be displayed only on CODESYS WebVisus. For security reasons, they should always be signed with a valid certificate, which must be available in the local Microsoft Certificate Store.

**Signing an HTML5 Control**

1. Click **Tools → Visualization Element Repository**.

   TIP:

   The command is available only when there are no projects open in CODESYS.

   * The **Visual Element Repository** dialog opens.
2. Specify the URL of a public timestamp server and click **OK** to confirm your entries.

   * Your visualization element is certified. The green icon (![](images/uuid-4ca9ea03-3b5e-66ff-e729-d1a2eb.png)) is displayed below **Signature**.

For more information, see: Dialog: Certificate Selection
