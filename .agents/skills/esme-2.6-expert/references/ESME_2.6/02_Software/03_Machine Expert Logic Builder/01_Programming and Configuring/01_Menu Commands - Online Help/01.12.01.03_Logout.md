# Logout

## Presentation

The default shortcut is Ctrl + F8.

The Online > Logout command affects a log out of the [application](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083435). It disconnects the programming system from the target device (controller or simulation target) and thus changes into the offline mode.

If the command is called from the Online menu, the active application is affected.

If the command is called from the contextual menu when an application object is selected, then the selected application is affected, no matter which application object is active.

NOTE: The layout of the views in EcoStruxure Machine Expert can change when you disconnect from the target device: it is automatically switched from the Online [perspective](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083359) to the Offline perspective that had been used before login.
