# Dialog: Import DXF File

|  |  |
| --- | --- |
| **Layers** | All layers of the technical drawing; determined from the data in the DXF file  : The layer will be imported. |
| **Maximum gap size** | Distance between the end points of the geometric elements until where the points are seen as interconnected. Determined from the data in the DXF file.  Example: `0.001` |

|  |  |
| --- | --- |
| **Preview on CNC path** | The data of the selected DXF file is displayed in the way that they are imported into the CNC editor. The layer selected for viewing is also taken into consideration.  Example: |
| **View** | To switch the layer that is considered in the preview window. |
|  | x/y-plane; selected for viewing |
|  | x/z-plane |
|  | y/z-plane |

|  |  |
| --- | --- |
| Button '**Import**' | The data is loaded into the CNC editor and the existing CNC program is replaced. The block numbers are assigned automatically.  The CNC object retains its name. |
