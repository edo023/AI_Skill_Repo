# International Settings

## Overview

The Tools > Options > International Settings dialog box allows the following settings concerning the language used in the user interface and help system.

|  |  |
| --- | --- |
| User interface language | |
| Same as Microsoft Windows | If this option is activated, the language used in the user interface will be that which is currently set by Microsoft Windows on your computer. |
| Specific language | If this option is activated, the language currently selected from the list will be used. |

NOTE: Changing the user interface language is effective after a restart of EcoStruxure Machine Expert. Some components may not be available in the selected language and are displayed in their default language, which typically is English.

|  |  |
| --- | --- |
| Online help language | |
| Same as user interface language | This option is set by default. The online help will be displayed in the language which is set for the user interface. If no help is available in this language, English will be used. |
| Specific language | If this option is activated, the language currently selected from the list will be used. |
