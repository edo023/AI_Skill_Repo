# Compiler Error C0104

## Message

Out of global data memory: Variable ‘<variable name>’, <byte size> bytes.

## Message Cause

More memory for data is used than available on the controller. It is also possible that the memory is too fragmented, due to incremental builds.

## Solution

Use ‘Clean’ to de-fragment the memory. This will reallocate the data with the next compilation.

## Error Example

**For M241:**

```
PROGRAM POU
VAR
  arr1 : ARRAY[1...11000000] OF BYTE;
END_VAR
```
