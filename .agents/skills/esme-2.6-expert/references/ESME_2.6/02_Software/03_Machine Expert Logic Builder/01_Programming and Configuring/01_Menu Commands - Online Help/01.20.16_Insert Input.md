# Insert Input

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | Ctrl + Shift + I |
| Call | * Ladder > Insert Input menu * Contextual menu |

## Function

The command inserts an input at the selected position. Possible insertion positions are indicated by the plus symbol at the cursor when you drag the element from the Toolbox to the implementation part. Replace the initially displayed question marks (`???`) by a variable name or a constant.

## Element: Input

The element is used to assign a variable value to another element in the signal flow. It has a continuous line to the right.
