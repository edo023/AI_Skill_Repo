# Error diagnosis: The drive cannot be switched on via `MC_Power`

The generic axis expects the following bits in the status word in order to return `MC_Power.Status = TRUE`:

* Bit 0 (ready to switch on)
* Bit 1 (switched on)
* Bit 2 (operation enabled)
* Bit 5 (quick stop)
* Bit 12

  However, Bit 12 is not set by every drive. If the drive does not set the bit, then the `_bCheckBit12InPositionMode` parameter of the generic axis can be set to `FALSE`.
