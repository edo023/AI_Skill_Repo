# Command: Release Locks Recursively

Symbol: ![](images/uuid-8ecccf2c-823a-1e8c-8cd0-e5f594.png)

**Function**: The command releases the lock of an object explicitly with all of its subordinate objects.

**Call**: **Context menu → SVN**

**Requirement**: The object is locked.
