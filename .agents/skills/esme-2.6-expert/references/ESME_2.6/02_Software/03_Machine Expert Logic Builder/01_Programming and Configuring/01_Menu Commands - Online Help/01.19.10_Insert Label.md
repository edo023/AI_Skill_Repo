# Insert Label

## Overview

The FBD/LD/IL > Insert label command inserts a [label field](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083477) into the currently selected network. You can enter the name of a label providing a target for a [jump instruction](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083476). You can modify the default text string Label: according to your demands.
