# Compiler Error C0203

## Message

Only structures and function blocks can contain variables of the type BIT.

## Message Cause

An attempt is made to declare a variable of the type BIT outside of structures and function blocks.

## Solution

Only declare variables of the type BIT in structures and function blocks.

## Error Example

```
PROGRAM PLC_PRG
VAR
  b : BIT;
END_VAR
```
