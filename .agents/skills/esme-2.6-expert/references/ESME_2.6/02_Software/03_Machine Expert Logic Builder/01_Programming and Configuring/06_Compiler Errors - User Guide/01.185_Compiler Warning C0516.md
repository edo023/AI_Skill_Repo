# Compiler Warning C0516

## Message

Should not access internal variable <POU name> of library <library name>.

## Message Cause

A library contains a POU with a variable. The visibility of the POU is restricted to this library with the `INTERNAL` keyword. The variable is accessed from outside of this library.

## Solution

Modify the visibility of the POU (for example, remove the `INTERNAL` keyword) or modify the declaration of the variable.

## Error Example

```
PROGRAM PLC_PRG
VAR
    somevar1 : INT := MyLib.GVL.p.varIn;
END_VAR
```

–> C0516: Should not access internal variable InternalPOU.varIn of library MyLib, 1.0.0.0 (MyCompany).
