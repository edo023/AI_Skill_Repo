# ToolBox

## Overview

The View > Software Catalog > ToolBox command opens the toolbox for the currently used editor in a separate view. Typically toolboxes are available with graphic language editors or the visualization editor and provide graphic programming elements which can be inserted into the editor with drag and drop.

For an example, refer to the [*FBD/LD/IL Toolbox* chapter in the *Programming Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083473).
