# Compiler Error C0174

## Message

‘VAR\_TEMP’ declaration is not allowed in this place

## Message Cause

‘VAR\_TEMP’ is used outside a program or function block.

## Solution

Only use ‘VAR\_TEMP’ inside of programs and function blocks.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
FUN();
```

```
FUNCTION FUN
VAR_TEMP
END_VAR
```
