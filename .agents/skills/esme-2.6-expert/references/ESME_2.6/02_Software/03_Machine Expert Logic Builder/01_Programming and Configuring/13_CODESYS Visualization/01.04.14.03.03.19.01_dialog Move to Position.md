# Dialog: Move to Position

|  |  |
| --- | --- |
| **Tab position** | New position of the element within the "Tab Order"  If multiple entries were selected in the element list when opening the dialog, then the one with the lowest value in the tab order is set to the new position. The positions of the other selected elements are moved accordingly. If necessary, the positions of the remaining elements of the list are adjusted afterwards. |
