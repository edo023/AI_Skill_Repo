# Menu bar

**Filter**

|  |  |
| --- | --- |
| **Show all categories** | List of all element properties |
| **Default** | List of the most frequently used element properties |
| **Simple** | List of specific basic element properties, such as **Texts**, **Colors**, and **Input configuration** |
| **Animation** | List of element properties for animating with variables |
| **Colors** | List of element properties for configuring with color |
| **Texts** | List of element properties for configuring with text |
| **Input** | List of element properties for configuring the user input |

**Sort by**

|  |  |
| --- | --- |
| **Sort by type** | : Element properties are sorted by the original order of categories. |
| **Sort by name** | : Element properties are sorted in alphabetical order. |

**Sort order**

|  |  |
| --- | --- |
| **Sort ascending** | : The properties are sorted from A to Z. |
| **Sort descending** | : The properties are sorted from Z to A. |

|  |  |
| --- | --- |
| **Advanced** | : The table contains all properties. The **Filter → Show all categories** menu command is selected at the same time. |
