# Project Settings - Compiler Warnings

## Overview

This dialog box displays a list of the possible compiler messages.

Click the check box for each message to define whether it is displayed in the Messages view during a compilation and in which category:

* ![](images/DD01237891.png) The message is not displayed.
* ![](images/DD01237890.png) The message is displayed as an advisory.
* ![](images/DD01237889.png) The message is displayed as an error. It is not possible to download the application.

Configure the maximum number of messages to be displayed in the [**Compile options** tab](D-SE-0083949.html#D-SE-0083949).
