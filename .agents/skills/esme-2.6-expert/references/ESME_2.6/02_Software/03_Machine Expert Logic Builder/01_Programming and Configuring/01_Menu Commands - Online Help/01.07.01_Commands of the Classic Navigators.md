# Commands of the Classic Navigators

## Overview

The menu View > Classic Navigators provides the following commands:

* Devices
* POUs

By default, each view has an arrow down button in the upper right corner for opening a menu with the following commands and options:

| Action | Description |
| --- | --- |
| Open in editor | Opens the object in the appropriate editor |
| Find object... | Dialog Find Object opens, where you can search the tree for an object name. If a string is entered in the input field, the results will be displayed automatically with name and path. The command Open selects the result in the device tree and opens it in the editor. |
| Sort by type  Sort by name | The objects get sorted by type or name, and alphabetically. |
| Sort ascending  Sort descending | The above defined sorting is arranged in ascending or descending order. |
| Track active editor | If activated, the object which is displayed in the active editor will be tracked (selected) in the device tree. |

## Devices View

In the Devices view, the devices needed for the project are represented by nodes below the root node showing the project title. Below each PLC logic device, the respective applications and their associated objects or further subdevices are (to be) inserted.

## POUs View

In the POUs view, all programming units of the current project are organized and can be instantiated for the use in a specific application.
