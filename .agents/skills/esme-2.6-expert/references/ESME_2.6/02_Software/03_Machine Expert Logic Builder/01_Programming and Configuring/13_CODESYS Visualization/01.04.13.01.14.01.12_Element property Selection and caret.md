# Element property: Selection and caret configuration

|  |  |
| --- | --- |
| **Caret position** | Variable (integer data type) for the position of the caret |
| **Selection start** | Variable (integer data type) for the position of the first selected character  Example: `PLC_PRG.iSelStart` |
| **Selection end** | Variable (integer data type) for the position of the last selected character  Example: `PLC_PRG.iSelEnd` |
| **All selected** | Variable (BOOL) for toggling the selection of the specified element  `TRUE`: The text in the text field is completely selected.  `FALSE`: The selection starts with the value in **Selection start** and ends with **Selection end**. |
