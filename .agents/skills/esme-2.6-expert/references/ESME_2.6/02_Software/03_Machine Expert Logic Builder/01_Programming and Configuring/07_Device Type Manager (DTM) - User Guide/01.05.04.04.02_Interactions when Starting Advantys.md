# Interactions when Starting Advantys Configuration Software

Interactions when Starting Advantys Configuration Software

You can start the Advantys Configuration Software from EcoStruxure Machine Expert.

To do this, click the Start Advantys button ![G-SE-0002380.1.gif-high.gif](images/G-SE-0002380.1.gif-high.gif) in the configuration tab of the DTM.

Only 1 instance of the Advantys Configuration Software can be started at a time. For this reason, the following interactions with Advantys DTM are possible:

| Status | Action | Result |
| --- | --- | --- |
| Advantys Configuration Software is already running as a standalone application – started from the Start menu. | You click the Start Advantys button in EcoStruxure Machine Expert to start Advantys Configuration Software. | You are prompted to close the running instance before starting the software from EcoStruxure Machine Expert. |
| Advantys Configuration Software is already running – started from the EcoStruxure Machine Expert software. | You want to start Advantys Configuration Software from the Start menu. | You are prompted to close the running instance before Advantys Configuration Software is started as standalone application. |
