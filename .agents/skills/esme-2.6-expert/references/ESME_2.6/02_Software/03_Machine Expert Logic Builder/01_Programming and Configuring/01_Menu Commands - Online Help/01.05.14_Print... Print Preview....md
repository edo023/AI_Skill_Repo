# Print... / Print Preview...

## Print... Command

The File > Print command allows you to print the defined part of the project on a defined printer according to a defined layout.

The Print dialog box opens. It allows you to define the desired print range, the printer properties, and number of copies. If you activate the option Print to file, you can define an appropriate file instead of getting an output on a printer.

To define the print page layout, use the command [**Page Setup...**](D-SE-0083902.html#D-SE-0083902).

When you close the dialog box by clicking OK while you are printing, the printing will stop after the next page.

## Print Preview... Command

The File > Print Preview... command opens the dialog box Print Preview that allows you to verify the pages configured for printing in a preview.

For further information, refer to the description of the Print Preview dialog box in the [**Page Setup...**](D-SE-0083902.html#D-SE-0083902) chapter.
