# Track description

The track data is structured in the same way as the path data: VisuStruct3DTrack

|  |  |
| --- | --- |
| **Track data (VisuStruct3DTrack)** | Variable of type `VisuStruct3DTrack` which is declared in the IEC code  Example: `PLC_PRG.pc.vs3dt`  A description of the structure can be found in the Library Manager in the `VisuElem3DPath.library` library. |
| **Track color** | Color of the drawn track |
| **Track line thickness** | Track line thickness (in pixels)  Example: 2 |
