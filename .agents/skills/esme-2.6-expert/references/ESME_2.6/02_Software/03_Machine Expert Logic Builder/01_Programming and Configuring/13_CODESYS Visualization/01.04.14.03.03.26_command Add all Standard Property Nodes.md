# Command: Add all Standard Property Nodes

Symbol: ![](images/uuid-5e703a13-34ae-7ed7-5873-1cd24c.png)

**Function**: Adds all standard properties which have not yet been added to the frame configuration of the interface properties.

**Call**: **Visualization** menu; context menu

**Requirement**: The focus is in the **Frame configuration** tab.

Added standard properties are disabled in the tree view because they cannot be modified.

TIP:

In order to keep the frame interface of a visualization clear, we recommend that you pass as few as possible standard frame properties as parameters.

TIP:

With the [Command: Move node up](_visu_cmd_move_node_up.html#_visu_cmd_move_node_up) or [Command: Move node down](_visu_cmd_move_node_down.html#_visu_cmd_move_node_down) commands, you can position the lines in the frame configuration.
