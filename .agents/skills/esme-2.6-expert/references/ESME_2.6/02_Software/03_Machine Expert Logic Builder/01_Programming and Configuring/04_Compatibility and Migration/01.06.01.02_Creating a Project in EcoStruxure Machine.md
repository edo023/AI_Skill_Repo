# Creating a Project in EcoStruxure Machine Expert

## Overview

When you create a project:

* You may encounter [device firmware compatibility](D-SE-0088860.html#D-SE-0088860) concerns.
* You may not be able to open the project in SoMachine or SoMachine Motion because it detects that the project is incompatible.
* You may not be able to open the project with earlier versions of EcoStruxure Machine Expert because it detects that the project is incompatible.
