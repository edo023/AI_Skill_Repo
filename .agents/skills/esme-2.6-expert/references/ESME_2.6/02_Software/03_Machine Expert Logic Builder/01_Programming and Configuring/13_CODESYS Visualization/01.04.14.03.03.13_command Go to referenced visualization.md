# Command: Go to referenced visualization

**Function**: Opens the referenced visualization

**Call**: **Visualization** menu, context menu

**Requirement**: A frame element is selected.

If others are referenced in a visualization, then you can open them. In the element list (or in the Visualization Editor), select the referencing **Frame** or **Tabs** element. Then this command is available. The existing references are displayed in the following submenu. Click on the desired visualization and it will be opened.
