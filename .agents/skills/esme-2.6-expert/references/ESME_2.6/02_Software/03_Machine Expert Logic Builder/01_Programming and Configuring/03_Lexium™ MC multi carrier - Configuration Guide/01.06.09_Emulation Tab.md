# Emulation Tab

## Overview

The Emulation tab displays a 3-D emulation of your configured track and visualizes your application running virtually.

## Multi-track Mode

In multi-track mode, the Emulation tab is available in both editor types:

* In track editors, it emulates the track selected with this editor.
* In the Multicarrier Configuration editor, it emulates a scene that contains all tracks of the project.

## Creating Emulation Data

After you have built your Lexium™ MC12 multi carrier system, you have to create the emulation data.

| Step | Action |
| --- | --- |
| 1 | Activate the option Create Emulation Data in the Project > Project Settings > Multicarrier Configuration Editor > Code Generation tab.  For details, refer to [Menu Commands Online Help](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=TPC_MLS_Config_Tools_Options_D995A499) |
| 2 | Open the Track tab and click Update > To All to get the emulation code generated automatically.  Otherwise the emulation is not available in the EcoStruxure Machine Expert Twin. |
