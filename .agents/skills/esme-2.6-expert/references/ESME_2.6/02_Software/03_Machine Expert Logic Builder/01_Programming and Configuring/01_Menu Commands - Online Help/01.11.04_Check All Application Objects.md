# Check All Application Objects

## Overview

You can execute the Check All Application Objects command to start a [**Build**](D-SE-0083980.html#D-SE-0083980) operation for all objects of the active application, even for the POUs that are not used by the application. The errors that are detected - even in the unused objects - are displayed in the Messages [view](D-SE-0083922.html#D-SE-0083922).

By default, the command is not available in the menus. You can add this command via the Tools > Customize [menu](D-SE-0084066.html#D-SE-0084066).
