# Defining an alarm with **Alarm (API)** type

1. In `Alarm Configuration`, set an [alarm group template](_cds_obj_alarm_group_template.html#_cds_obj_alarm_group_template).
2. In the **Class** column, select for example the alarm class `Warning` and specify the desired alarm message under **Message**.

* The alarm `ID_0`, which is triggered via API, is defined.

TIP:

You can also define an API alarm in an [alarm group](_cds_obj_alarm_group.html#_cds_obj_alarm_group).
