# Command: Make Same Width

Symbol: ![](images/uuid-c593c835-08f1-aaae-40b5-c6f1e0.png)

**Function**: The command adjusts the width of the selected visualization elements to the width of the blue selected element.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Multiple elements are selected. The first element is blue and the other elements are gray.

TIP:

The command does not work for lines or polygons.
