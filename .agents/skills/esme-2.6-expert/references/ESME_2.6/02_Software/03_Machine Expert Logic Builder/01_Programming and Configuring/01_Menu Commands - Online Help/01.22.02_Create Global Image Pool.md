# Create Global Image Pool

## Overview

Use the Image Pool > Create Global Image Pool command to add the GlobalImagePool manually. It is an image pool for static images used in a visualization. Keep in mind that this pool is created automatically as soon as the first static image is defined in a project.

For a description of the image pool functionality, refer to the chapter [*Image Pool*](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083433).
