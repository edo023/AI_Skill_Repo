# Command: Un-Ignore on Commit

**Function**: The command removes an unversioned object from the "ignore list" so that the object is checked by default on commit.

**Call**: Context menu: **SVN**

**Requirement**: The [Ignore on commit](_svn_cmd_ignore_on_commit.html#_svn_cmd_ignore_on_commit) command has been executed for the object, and the object is marked with the ![](images/uuid-e260578d-a0c7-396e-c349-59e78a.png) symbol.
