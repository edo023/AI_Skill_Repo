# Compiler Error C0507

## Message

Value of 2nd operand of the UPPER\_BOUND operator must be 1 in this case.

## Message Cause

The value of the 2nd operand of the UPPER\_BOUND operator is not 1.

## Solution

Make sure that the value of the 2nd operand of the UPPER\_BOUND operator is 1.

## Error Example

```
FUNCTION_BLOCK POU
VAR_IN_OUT
  arrin : ARRAY [*] OF INT;
END_VAR
VAR
  arrtest : ARRAY [0..5] OF INT;
  test1: DINT;
  test2: DINT;
END_VAR
```

```
test1 := UPPER_BOUND(arrin, 0);
test2 := UPPER_BOUND(arrtest, 0);
```

```
fb1 : POU;
```

## Error Correction

```
test1 := UPPER_BOUND(arrin, 1);
test2 := UPPER_BOUND(arrtest, 1);
```
