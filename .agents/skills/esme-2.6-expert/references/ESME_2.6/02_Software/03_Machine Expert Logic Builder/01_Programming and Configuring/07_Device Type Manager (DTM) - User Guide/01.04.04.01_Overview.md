# Overview

Overview

The device-specific software component is called a Device Type Manager (DTM), which is supplied with the device by the manufacturer.

DTM installation in EcoStruxure Machine Expert is composed of two steps:

oInstall the DTMs on the PC with EcoStruxure Machine Expert. This makes the DTMs detectable by EcoStruxure Machine Expert.

oImport the DTMs in the Device Repository of EcoStruxure Machine Expert.
