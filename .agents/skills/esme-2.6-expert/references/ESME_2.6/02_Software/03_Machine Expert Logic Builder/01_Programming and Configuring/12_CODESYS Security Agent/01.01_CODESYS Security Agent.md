# CODESYS Security Agent

The CODESYS Security Agent is an add-on product for CODESYS. It facilitates the configuration and control of security-relevant aspects in the development system.

Currently, you can use the CODESYS Security Agent to configure and create certificates of the controller.

After installation, there are the following extensions:

* The **Security Screen** view in the development system has an additional **Device** tab. With this, you can configure encrypted communication with the controller, as well as encrypt the boot application, download, and online change.
* The **Communication Settings** tab of the device editor gets the **Security Settings** command in the **Device** menu, which takes you to the **Device Security Settings** dialog. Here you can edit settings and download them again to the controller.
