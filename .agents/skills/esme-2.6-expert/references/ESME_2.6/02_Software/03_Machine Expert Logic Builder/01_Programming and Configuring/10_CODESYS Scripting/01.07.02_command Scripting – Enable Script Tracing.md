# Command: Scripting – Enable Script Tracing

Symbol: ![](images/uuid-380e1d3b-d5ca-bb9e-4484-5dc331.png)

**Function**: The command gets CODESYS to print all commands from the script file to the message view. Use the command to monitor and debug scripts. A blue frame around the symbol indicates that the option is enabled.

**Call**: **Tools** menu

For more information see: [Scripting in CODESYS](_script_scripting_with_codesys.html#_script_scripting_with_codesys)
