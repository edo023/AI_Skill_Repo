# Compiler Error C0020

## Message

‘<statement>’ is no valid statement

## Message Cause

Syntax error (for example, insufficient or excessive number of characters).

## Solution

Make sure that the syntax is correct.

## Error Example

```
PROGRAM PLC_PRG
VAR
  x : INT;
END_VAR
```

```
x = 2;
```

--> C0020: '(x = 2); ' is no valid statement

## Error Correction

```
x := 2;
```
