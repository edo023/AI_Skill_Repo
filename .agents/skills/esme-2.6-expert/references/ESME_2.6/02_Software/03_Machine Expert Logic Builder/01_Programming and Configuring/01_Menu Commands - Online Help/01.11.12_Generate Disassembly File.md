# Generate Disassembly File

## Overview

The Build > Generate Disassembly File command creates a disassembly file <project name>.asm from the project and saves it to the file directory in the project folder.

By default, the command is not available in the menus. You can add this command via the Tools > Customize [menu](D-SE-0084066.html#D-SE-0084066).
