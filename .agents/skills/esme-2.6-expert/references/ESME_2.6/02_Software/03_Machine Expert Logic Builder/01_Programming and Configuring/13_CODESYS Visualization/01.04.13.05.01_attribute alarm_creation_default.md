# Attribute: alarm\_creation\_default

**Syntax**:

`{attribute 'alarm_creation_default' := true}`

`{attribute 'alarm_creation_default' := false}`

Insertion point: Instance or declaration of the function block or structure.

By default, the `{attribute 'alarm_creation_default'}` attribute pragma defines whether the alarm instance is created (`:= true`) or not (`:= false`). This is reflected in the table of alarm instances in the alarm configuration.

The attribute is not usually used. When no attribute is specified, the instances behave as if the attribute were specified as `TRUE`.

For more information, see the following: [Object: Alarm Configuration](_cds_obj_alarm_configuration.html#_cds_obj_alarm_configuration)
