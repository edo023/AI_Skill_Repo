# Add configuration setting...

## Overview

The Add configuration setting... command is available in the contextual menu and behind the Add button in the Gateway Configuration [dialog box](D-SE-0084177.html#D-SE-0084177__D-SE-0084177.3). It can be used for adding a further configuration setting for the selected driver (interface).

For a description of the gateway configuration, refer to the [*Configure the local gateway...* chapter](D-SE-0084177.html#D-SE-0084177__D-SE-0084177.3).
