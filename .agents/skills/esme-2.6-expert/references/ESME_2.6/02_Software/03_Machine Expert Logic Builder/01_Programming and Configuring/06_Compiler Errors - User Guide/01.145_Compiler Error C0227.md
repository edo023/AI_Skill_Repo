# Compiler Error C0227

## Message

Initialization of constant variable <constant name> is not constant

## Message Cause

A constant is initialized with a variable.

## Solution

Only initialize constants with constant values.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
VAR CONSTANT
  k : INT := i;
END_VAR
```

--> C0227: Initialization of constant variable 'k' is not constant

EIO0000003933.04
