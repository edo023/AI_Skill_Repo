# Triggering an event via an alarm instance

This is particularly relevant for library developers.

IMPORTANT:

If you observe the variable of a custom type (function block of structure) and you want to define an event for this POU, then you should preferably do this in an [alarm group template](_cds_obj_alarm_group_template.html#_cds_obj_alarm_group_template).

It is an advantage that both the POU and the alarm group template can become part of a development library.
