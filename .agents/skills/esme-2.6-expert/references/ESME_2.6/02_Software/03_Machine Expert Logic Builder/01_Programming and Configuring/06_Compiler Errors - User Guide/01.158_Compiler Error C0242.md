# Compiler Error C0242

## Message

Operand of \_\_DELETE must be pointer

## Message Cause

An incorrect operand is passed to the operator \_\_DELETE.

## Solution

Pass a pointer.

## Error Example

```
PROGRAM PLC_PRG
VAR
  a : INT;
  pt : POINTER TO INT;
END_VAR
```

```
__DELETE(a);
```

--> C0242: Operand of \_\_DELETE must be pointer

## Error Correction

```
__DELETE (pt);
```
