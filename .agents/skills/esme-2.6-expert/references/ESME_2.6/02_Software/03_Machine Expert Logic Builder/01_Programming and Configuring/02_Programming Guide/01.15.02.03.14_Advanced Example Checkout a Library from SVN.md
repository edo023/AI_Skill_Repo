# Advanced Example: Checkout a Library from SVN and Installation

## Overview

The following sample script can perform the checkout and installation of a library as part of a CT (Continuous Testing) environment so that they can be tested. The SVN Add-on must be installed on the PC and the SVN Add-on license must be activated.

## Sample Script

```
import tempfile

if projects.primary:
    projects.primary.close()

tempdir = tempfile.mkdtemp()
URL = "svn://localhost/testrepo/trunk/SvnTestLibrary/"

proj = svn.checkout(URL, tempdir, "testlibrary", as_library=True)
proj.save()

repo = librarymanager.repositories[0]
librarymanager.install_library(proj.path, repo, True)

proj.close()
```
