# Insert Return

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | Ctrl + R |
| Call | * Ladder > Insert Return menu * Contextual menu |

## Function

The command inserts a return element at the selected location. Possible insertion positions are indicated by the plus symbol at the cursor when you drag the element from the Toolbox to the implementation part.

## Element: Return

The element interrupts the execution of the Ladder block when the input of the return element becomes TRUE. You can place the return instruction parallel to or after the preceding elements.
