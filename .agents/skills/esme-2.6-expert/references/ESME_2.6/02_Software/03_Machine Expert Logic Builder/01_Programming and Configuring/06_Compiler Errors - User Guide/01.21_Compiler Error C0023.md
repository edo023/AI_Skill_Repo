# Compiler Error C0023

## Message

‘<operator>’ needs at least ‘<number of operands>’ operands

## Message Cause

An insufficient number of operands are assigned to an operator.

## Solution

Assign the required number of operands to the operator.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
i := MUX(30,40);
```

--> C0023: 'MUX' needs at least '3' operands

## Error Correction

```
i := MUX(30,40,50);
```
