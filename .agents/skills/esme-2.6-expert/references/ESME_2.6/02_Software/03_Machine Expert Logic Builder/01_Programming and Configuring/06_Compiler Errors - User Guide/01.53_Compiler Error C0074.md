# Compiler Error C0074

## Message

Unexpected array initialization

## Message Cause

Syntax error in the array initialization

## Solution

Correct the syntax

## Error Example

```
PROGRAM PLC_PRG
VAR
  arr1 : INT := [1,2,3,4,5,6];
END_VAR
```

--> C0074: Unexpected array initialization

--> C0032: Cannot convert type 'Unknown type: [1,2,3,4,5,6]' to type 'INT'

## Error Correction

```
arr1 : ARRAY [1..6] OF INT := [1,2,3,4,5,6];
```
