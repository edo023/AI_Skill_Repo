# `Attribute suppress_warning`

## Overview

The pragma `attribute 'suppress_warning'` has the effect that the compiler advisories specified in the definition of this pragma are suppressed.

## Syntax

{attribute 'suppress\_warning' := '<warning id1>',''<warning id2>',...}

Insert the `Attribute suppress_warning` in the line above the declaration part of a POU or DUT.

## Example

```
{attribute 'suppress_warning' := '0125'}
TYPE DUT_1 :
(
    enum_member1 := 0,
    enum_member2 := 0
);
END_TYPE
```

**Result**: The compiler message C0125 is suppressed.
