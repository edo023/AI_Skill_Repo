# Insert Box with EN/ENO

## Overview

Shortcut: CTRL + SHIFT + E

The FBD/LD/IL > Insert Box with EN/ENO command is available in the FBD and LD editor. You can use it to insert a box element with an EN input and an ENO output into a network.

## Operation of EN/ENO Boxes

If the value of EN is FALSE (0) when the box is called, the defined operations will not be executed. Otherwise, if the value of EN is TRUE (1), the operations defined by the box will be executed. The ENO output operates as a repeater of EN input.
