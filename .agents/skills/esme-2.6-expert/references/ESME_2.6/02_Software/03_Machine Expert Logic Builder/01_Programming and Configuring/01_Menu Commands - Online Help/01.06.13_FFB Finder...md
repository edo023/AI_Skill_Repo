# FFB Finder..

## Overview

The File > FFB Finder... command opens the FFB Finder dialog box. The FFB (function and function block) finder assists you in finding a specific function or function block even if you do not know its exact name.

For further information, refer to the chapter [*Function and Function Block Finder* in the *Programming Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083548).
