# I/O Summary

## Overview

Right-click an I/O node in the Devices tree, and execute the I/O Summary... command from the contextual menu to display a list of the local I/O configurations.

The command opens the I/O Summary dialog box that displays the Inputs and Outputs in separate lists. The Configuration column provides further information, for example, whether the input or output is associated with counters or pulse generator functions.
