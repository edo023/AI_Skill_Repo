# IScriptLoggerPoint

class

IScriptLoggerPoint.

IScriptLoggerPoint

Bases: object

Provides services on a logger point of the application logger.

log\_level

Gets or sets the logger level of this logger point.

name

Gets the name of the logger point. Every logger message that is sent via this logger point is connected to this logger point. The logger point represents a part of the machine or project or a special functionality.

source

Gets the source of the logger point. It specifies where the program part comes from. For example, which library contains the function block that provides the logger messages. This information can be used to identify non-unique information such as the enumeration ET\_DiagExt of a library.

type

Gets the type of the logger point. It specifies the kind of machine part (for example AxisModule, FillingModule) the logger point represents. This information is defined here since the name of the logger point does not necessarily provide the information.

id

Gets the unique identifier of the logger point.

parent\_id

Gets the unique identifier of the parent logger point.
