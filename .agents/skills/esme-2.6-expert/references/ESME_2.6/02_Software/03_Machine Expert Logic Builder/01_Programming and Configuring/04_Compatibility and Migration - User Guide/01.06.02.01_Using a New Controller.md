# Using a New Controller

## Overview

Schneider Electric updates the controller frequently providing new features and functions. Therefore, there may be a difference between the version you receive as a new product, the present version being put into stock from the factory, and the version of your EcoStruxure Machine Expert installation.

## Controller with an Earlier Firmware Version

You received a controller with a firmware version earlier than the one of the devices in your project.

In this case, updating the controller firmware is necessary. Controller firmware is provided inside the EcoStruxure Machine Expert installation or can be found on the Schneider Electric [website](https://www.se.com).

If you do not update your controller, you are able to log in to your controller only if [the compatibility rule is respected](D-SE-0088869.html#D-SE-0088869).

## Controller with a Later Firmware Version

You may receive a controller with a firmware version later than one of the devices in your project.

In this case, [updating the project](D-SE-0088893.html#D-SE-0088893__D-SE-0088893.10) is necessary.

Alternatively, consider downgrading the firmware of your controller.

In general, you are able to log in to your controller only if [the compatibility rule is respected](D-SE-0088869.html#D-SE-0088869).

## Controller Running a Solution Project (S-Type)

If your controller is running a SoMachine V3.1 solution project, use one of the following options:

* [Update](D-SE-0088893.html#D-SE-0088893__D-SE-0088893.10) your SoMachine V3.1 solution project to EcoStruxure Machine Expert, and update the controller firmware compatible to your EcoStruxure Machine Expert version. For details, refer to the EcoStruxure Machine Expert Release Notes.
* Change the firmware of the controller by the new firmware supporting the application libraries. Device firmware is provided with the EcoStruxure Machine Expert installation (managed by the Schneider Electric Software Installer) or can be found on the Schneider Electric [website](https://www.se.com).

Also refer to the [not supported application libraries](D-SE-0095991.html#D-SE-0095991).

Also refer to [Compatibility of Legacy S-Type Devices](D-SE-0088873.html#D-SE-0088873).
