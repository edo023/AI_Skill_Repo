Legacy interface for task objects. This remains available for compatibility reasons to earlier versions. The official API is now provided directly by CODESYS.

add\_cyclic\_task

name

Adds a new cyclic task to the application. The cyclic task will be processed cyclic according to the time definition (task cycle time) given in the property ITaskObject.Interval .

Parameters

name (string) -- The name of the new task.

Reports

System.ArgumentException -- name is null or empty.

This example shows how to add a new task to an existing application.

```
p = projects.primary
app = p.active_application

# create a new task with the name 'TASK_SR_Test'
app.add_cyclic_task('TASK_SR_Test')
```
