# Save prepared Parameters to Project

## Overview

The CFC > Save prepared Parameters to Project command is available when the CFC editor is open, and you are in offline mode, but parameter values of function block instances have been changed in online mode.

If the values of constants available on the controller differ from the values in the application, then this is indicated by a red asterisk next to the parameter field.

In order to save the controller values to the application, click the Save Prepared Parameters to Project button. For further information, refer to the chapter [Edit Parameters...](D-SE-0084092.html#D-SE-0084092).
