# Dialog: Trend Wizard

Each row of the table contains a control that can assigned to the trend. The elements are placed in the visualization next to the trend. The controls are saved in the **Assigned controls** property and can be modified there.

|  |  |
| --- | --- |
| **Attached control** | : The associated element is available in the visualization and connected with the trend via the property **Assigned Visu element**. The element is inserted into the visualization.  : Deactivating the option does **not** cause the element to be deleted from the visualization. |
| **Position** | Position of the control in relation to the trend. |
| **Type of element to insert** | List box with the installed types of the control |
| **Instance name** | Instance name of the control |
