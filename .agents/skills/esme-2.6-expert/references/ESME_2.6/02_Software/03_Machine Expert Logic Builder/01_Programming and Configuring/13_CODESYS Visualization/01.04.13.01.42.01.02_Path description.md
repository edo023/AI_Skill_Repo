# Path description

|  |  |
| --- | --- |
| **Path data (VisuStruct3DTrack)** | Variable of type `VisuStruct3DTrack` which is declared in the IEC code  Example: `PLC_PRG.pc.vs3dt`.  A description of the structure can be found in the Library Manager in the `VisuElem3DPath.library` library.  The data structure describes a path or track by a specific number of points. The points are determined and buffered by the application. The track typically displays the last n positions so that only a specific part of them is displayed at one time.  `VisuStruct3DTrack.pProjection` is a variable which is set by the visualization element and contains information about the path/track projection. It can (only) be read by the application. Moreover, the `Projection.Apply` or `.ApplyV` methods can be used to see if the transformed position is inside or outside of the visualization display area as defined by `Projection.ElementRect`. |
| **Path color** | Color of the drawn path |
| **Path line width** | Path line width (in pixels)  Example: `2` |
| **Edge point marking** | **Display of the points between two consecutive objects in the path**   * End points are not marked * End points are marked with a circle * End points are marked with a cross * End points are marked with a plus sign |
