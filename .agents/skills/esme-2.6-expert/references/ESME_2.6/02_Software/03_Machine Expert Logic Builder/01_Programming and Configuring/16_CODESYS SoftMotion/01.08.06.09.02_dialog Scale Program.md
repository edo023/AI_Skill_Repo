# Dialog: Scale Program

|  |  |
| --- | --- |
| **Scaling factor** | Default: `0.001`  Condition: Scaling factor >= 1e-3  For a scaling factor > 1, it is upscaled. |
| **Axes** | : The axis is scaled. |
| **D (tool radius)** | : The tool radius D is scaled. |
| **OK** | The scaling is performed. The block numbers are assigned automatically. |
