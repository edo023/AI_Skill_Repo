# Outcommented

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | Ctrl + O |
| Call | * Ladder > Outcommented menu * Contextual menu |

## Function

Run the command to switch a [network](InsertNetwork-4526CB92.html) between the commented out and uncommented states.

The contents of outcommented networks are displayed in gray and the texts in italics. The outcommented network is not taken into consideration during processing.
