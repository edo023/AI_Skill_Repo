# Compiler Error C0090

## Message

No definition found for the base class ‘<function block name>’

## Message Cause

The function block specified as the base does not exist or is not a function block.

## Solution

Use a function block as the base.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : FB;
END_VAR
```

```
FUNCTION_BLOCK FB EXTENDS POU
VAR
END_VAR
```

```
FUNCTION POU
VAR
END_VAR
```

--> C0090: No definition found for the base class 'POU'
