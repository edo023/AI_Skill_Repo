# Compiler Error C0078

## Message

Unsupported type: ‘<data type>’.

## Message Cause

The type used is not supported by the device.

## Solution

If possible, use a different type. For example, REAL instead of LREAL.

## Error Example

```
PROGRAM PLC_PRG
VAR
  value : LREAL;
END_VAR
```

-->C0078: Unsupported type: ‘LREAL’

## Error Correction

```
PROGRAM PLC_PRG
VAR
  value : REAL;
END_VAR
```
