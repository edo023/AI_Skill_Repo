# Compiler Error C0222

## Message

Outputs cannot be of the type ‘REFERENCE TO’

## Message Cause

An attempt is made to define REFERENCE TO as an output parameter.

## Solution

Do not use REFERENCE TO as an output parameter.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : FB;
END_VAR
```

```
FUNCTION_BLOCK FB
VAR_OUTPUT
  re : REFERENCE TO INT;
END_VAR
```

--> C0222: Outputs cannot be of the type 'REFERENCE TO'
