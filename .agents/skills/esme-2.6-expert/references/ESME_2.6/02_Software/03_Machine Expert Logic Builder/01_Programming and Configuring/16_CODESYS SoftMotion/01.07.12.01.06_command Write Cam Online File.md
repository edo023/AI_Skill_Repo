# Command: Write Cam Online File

**Function**: The command creates a file (`CAM` extension). The file contains the data of the cam that is active in the editor.

**Call**: **Cam** menu

**Requirement**: The cam editor is open and displays a cam.

The cam data is composed of a cam description and the positions and types of cams.

A CAM file can be read to the editor by means of the [Read cam online file](_sm_cmd_cam_read_online_file.html#_sm_cmd_cam_read_online_file) command.

In addition, an instance of the `SMC_ReadCAM` function block can read the file in order to load a cam table into the application at runtime.

For more information, see: [Data structures of cams](78736dbc7e16b676dd4a64e26b712607.html#UUID-327725b5-7947-997b-3347-a113293bee19_section-idm43433793858538)
