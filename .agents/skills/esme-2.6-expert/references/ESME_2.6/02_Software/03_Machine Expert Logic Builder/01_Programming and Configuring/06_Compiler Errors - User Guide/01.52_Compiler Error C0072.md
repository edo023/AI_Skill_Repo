# Compiler Error C0072

## Message

Operator <operator name>’ is not possible on type ‘<data type>’

## Message Cause

An operator is applied to an incompatible type.

## Solution

Only apply operators on compatible types.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  str : STRING;
END_VAR
```

```
str := ABS(str);
```

--> C0072: Operator 'Abs' is not possible on type 'STRING'

## Error Correction

```
i := ABS(i);
```
