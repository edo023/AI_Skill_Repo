# Smoothing of Additional Axis Motion using Multiple Objects

**G code**: `G70`, `G71`

**Function**: `G71` starts and `G70` ends the smoothing of motion on the additional axis that is given in L and travels over multiple objects.

Syntax

```
G70 L4
G71 L4
```

| G Code Word | Additional Axis |
| --- | --- |
| `L4` | A |
| `L5` | B |
| `L6` | C |
| `L7` | P |
| `L8` | Q |
| `L9` | U |
| `L10` | V |
| `L11` | W |
