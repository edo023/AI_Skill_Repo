# Error Handling

The `Append` method of the `CamBuilder` function block returns `TRUE` when a segment has been successfully added, and returns `FALSE` if an error has occurred. After an error, no more segments can be added and the `CamBuilder` function block has to be reinitialized using the `Init` method. The cause of the error can be determined using the `IsErrorPending` method:

```
camBuilder.IsErrorPending(errorId=> errorId);
```
