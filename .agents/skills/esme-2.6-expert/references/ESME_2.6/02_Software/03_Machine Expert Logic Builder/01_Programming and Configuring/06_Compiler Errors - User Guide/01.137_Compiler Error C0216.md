# Compiler Error C0216

## Message

Case label duplicate

## Message Cause

A case label is used multiple times.

## Solution

Use each case label only once.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
CASE i OF
  1: i := i+1;
  1: i := i+2;
ELSE
  i := i+10;
END_CASE;
```

--> C0216: Case label duplicate

EIO0000003933.04
