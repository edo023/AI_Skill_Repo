# Dialog: Gradient Editor

**Function**: Configures the gradient editor of the element

When you define two colors, the gradient is made between them. When you use only one color, the gradient is made within that color, changing its brightness. The detailed settings specify the starting position and angle of the gradient.

**Call**: Click in the value field of the **Gradient setting** property

**Requirement**: You have selected a visualization element in the editor which has the **Gradient setting** property.

|  |  |
| --- | --- |
| **Gradient** | * **Linear** * **Radial** * **Axial**: The color runs from one axis outwards on both sides. |
| **Color 1** | First color of the gradient |
| **Color 2** | Second color of the gradient |
| **Transparency** | Transparency of the respective color  Permitted values: Integer in the value range from `0` to `255`  The color is opaque at 0 and fully transparent at 255. |
| **Default linear** | Requirement: The gradient is set to **Linear**.  Default direction of the linear gradient |
| **Default radial** | Requirement: The gradient is set to **Radial**.  Default setting |
| **Default axial** | Requirement: The gradient is set to **Axial**.  Direction of the gradient |
| **Angle (degrees)** | Requirement: **Linear** or **Axial** gradient |
| **Center X (%)**: | Requirement: The gradient is set to **Radial**  X position of the center point (0–100 %) |
| **Center Y (%)**: | Requirement: The gradient is set to **Radial**.  Y-position of the center point (0–100 %) |
| **Use one color** | Gradient between **Color 1** and the same color with different brightness |
| **Brightness** | Requirement: The **Use one color** option is selected.  Setting from 0 (black) to 100 (white) |
| **Use two colors** | Gradient between the two selected colors: **Color 1** and **Color 2** |
