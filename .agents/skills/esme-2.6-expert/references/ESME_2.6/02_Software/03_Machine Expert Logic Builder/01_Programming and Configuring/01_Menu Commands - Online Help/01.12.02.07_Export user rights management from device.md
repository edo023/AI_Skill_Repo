# Export user rights management from device / Import user rights management on the device

## Overview

The Online > Security > Export user rights management from device / Import user rights management on the device commands are also available in the contextual menu of the controllers.

The commands support *Device user rights management* files of type \*.durm that include the settings made in the following editors:

* [Users and Groups: User management concerning device access during runtime.](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083876)
* [Access Rights: Configuration of the access rights on runtime objects and files for the particular user groups.](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083877)
* [Symbol Rights: Access rights of individual user groups to symbols (symbol sets) on the device.](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0106022)

## Exporting User Rights Management from Device

When you execute the Export user rights management from device command, the Export Device User Rights Management dialog box opens that provides the following elements:

| Element | Description |
| --- | --- |
| Path | Enter the path for saving the \*.durm file or click the browse (...) button to browse the file system for the folder. |
| Password of file (it is required on enabling or on importing the device user rights again) | Enter the password for the \*.durm file. It will be requested whenever you attempt to enable or import the device user rights configured in this file.  NOTE: Use a secure password. A secure password is one that has not been shared or distributed to any unauthorized personnel and does not contain any personal or otherwise obvious information.  The password must comply with the following rules:   * Length of 8...50 characters. * ASCII [33, 122] characters allowed. * Mix of three different types of characters (upper and lower case letters, numbers and special characters) required. |
| Confirm password | Enter the password a second time. Ensure that it is identical to the password entered in the Password of file (it is required on enabling or on importing the device user rights again) text box. |

## Importing User Rights Management to the Device

When you execute the Import user rights management on the device command, the Import Device User Rights Management dialog box opens that provides the following elements:

| Element | Description |
| --- | --- |
| Path | Enter the path to the \*.durm file you want to import or click the browse (...) button to browse the file system for the folder where it is located. |
| Password of file (It was set on disabling or exporting the device user rights) | Enter the password for the \*.durm file that was set when exporting the user rights management from device to the file. |

NOTE: For accessing the controller after you have imported the *Device user rights management* files of type \*.durm, the credentials of a user account configured in the imported file will be required.
