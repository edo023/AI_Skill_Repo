# DATE Constants

## Overview

Use these constants to enter dates.

## Syntax

d#<date declaration>

Instead of `d#` you can also use the following:

* `D#`
* `date#`
* `DATE#`

Enter the date declaration in format `<date keyword>#<year>-<month>-<day>`.

DATE values are internally handled as DWORD values, containing the time span in seconds since 01.01.1970, 00:00 clock, with an upper limit of DATE#2106-2-7.

Examples

```
DATE#1996-05-06
d#1972-03-29
```

For further information, refer to the description of the [Date and Time Data Types](D-SE-0083662.html#D-SE-0083662__D-SE-0083662.7).
