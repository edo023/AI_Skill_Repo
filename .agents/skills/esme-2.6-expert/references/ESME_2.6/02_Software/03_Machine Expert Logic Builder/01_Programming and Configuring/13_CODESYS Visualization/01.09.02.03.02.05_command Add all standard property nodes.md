# Command: Add all standard property nodes

Symbol: ![](images/uuid-80d750ca-9fcd-5b40-a58e-e813a3.png)

**Function**: Adds all standard property nodes to the end of the list

**Call**: **HTML5 Control Editor** menu

TIP:

For a clear design of the interface of the HTML5 control, we recommend that you define as few as possible default properties as parameters.

Standard properties are highlighted in gray in the list because they are not changeable. After that, you can still position the properties.
