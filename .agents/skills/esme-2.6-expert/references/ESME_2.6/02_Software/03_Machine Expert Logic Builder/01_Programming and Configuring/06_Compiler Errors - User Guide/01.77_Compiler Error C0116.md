# Compiler Error C0116

## Message

The label ‘<jump label>’ is a duplicate

## Message Cause

A label is defined multiple times.

## Solution

Define each label only once.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
JMP label;
label:
label:
```

--> C0116: The label 'LABEL' is a duplicate
