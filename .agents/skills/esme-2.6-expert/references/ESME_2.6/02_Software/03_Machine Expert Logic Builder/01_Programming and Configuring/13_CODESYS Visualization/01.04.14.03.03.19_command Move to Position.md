# Command: Move to Position

**Function**: The command opens the **Move to Position** dialog to input another position number within the **Tab Order** that is defined in the [Command: Visualization Element List](_visu_cmd_element_list.html#_visu_cmd_element_list). Multiple elements can be selected for this as long as they do not belong to different groups (different hierarchy levels).

**Call**: Element list in the visualization editor; context menu in the **Tab Order** column
