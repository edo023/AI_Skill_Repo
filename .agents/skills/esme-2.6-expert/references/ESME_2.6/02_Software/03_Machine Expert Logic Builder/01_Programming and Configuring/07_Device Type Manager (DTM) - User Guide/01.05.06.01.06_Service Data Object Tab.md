# Service Data Object Tab

Service Data Object Tab

This tab allows the manual configuration or modification of the SDOs:

| Element | Description |
| --- | --- |
| Move up | Change the PDO order. |
| Move down |
| New... | Add a SDO channel. |
| Delete... | Delete a SDO channel. |
| Edit... | Modify the SDO channel properties and objects contained in the SDO. |
| SDO Timeout (ms) | Configure the SDO timeout in ms (default 1000 ms). |
