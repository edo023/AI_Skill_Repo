# Paste Contacts: Paste Below

## Overview

Shortcut: CTRL + F

The FBD/LD/IL > Paste Contacts: Paste Below command is only available in the LD editor. It pastes the elements or the section of the network which has been put to the clipboard before by a Copy or Cut [command](D-SE-0083910.html#D-SE-0083910), below the currently selected contact element in the network. This corresponds to the common Paste command.

For a description on selecting in networks, refer to the chapter [*Working in the FBD and LD Editor*](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083467).
