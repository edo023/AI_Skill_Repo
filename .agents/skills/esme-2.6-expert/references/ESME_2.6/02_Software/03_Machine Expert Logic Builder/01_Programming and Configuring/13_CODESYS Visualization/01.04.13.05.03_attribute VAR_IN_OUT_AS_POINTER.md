# Attribute: VAR\_IN\_OUT\_AS\_POINTER

**Function**: The pragma `{attribute 'VAR_IN_OUT_AS_POINTER'}` allows for passing a data object reference to the interface variable of a visualization (type: dialog).

**Requirement**: The visualization has to be used as a dialog.

**Syntax:**

`{attribute 'VAR_IN_OUT_AS_POINTER'}`

IMPORTANT:

Uppercase and lowercase characters have to be respected.

Example: Declaration of an interface

```
VAR_IN_OUT
    {attribute 'VAR_IN_OUT_AS_POINTER'}
    itfController : ControlFB;
END_VAR
```
