# Change Password Device User

## Overview

The Online > Security > Change Password Device User command is only available if you are logged into the device and allows you to change your password.

NOTE: You can handle the online user management via the Users and Groups view of the device editor ([**Users and Groups**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083876)). The commands of the menu Online > Security provide an additional simplified opportunity to protect the access on the target device.

NOTE: After you have executed this command, you cannot log in anymore with the recent password.

NOTE: Use a secure password. A secure password is one that has not been shared or distributed to any unauthorized personnel and does not contain any personal or otherwise obvious information. Further, a mix of upper and lower case letters and numbers offer greater security. You should choose a password length of at least seven characters.
