# Compiler Error C0040

## Message

Function ‘<function name>’ requires exactly ‘<number of inputs>’ input

## Message Cause

An insufficient or an excessive number of parameters are passed to the called function.

## Solution

Pass exactly the expected number of parameters to the function.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
i := TEST(1,2);
```

```
FUNCTION TEST : INT
VAR_INPUT
  IN: INT;
END_VAR
```

--> C0040: Function 'TEST' requires exactly '1' input

## Error Correction

```
i := Test(1);
```
