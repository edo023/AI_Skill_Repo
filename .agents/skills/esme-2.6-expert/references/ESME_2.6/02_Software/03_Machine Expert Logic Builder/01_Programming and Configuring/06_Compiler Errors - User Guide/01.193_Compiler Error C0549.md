# Compiler Error C0549

## Message

Initialization of the static variable ‘<variable name>’ is not constant, or replaced constants is disabled.

## Message Causes

* VAR\_STAT and CONSTANT declarations are used together in a function block declaration.
* The Replaces constants compile option is disabled.

## Solution

In the Project > Project Settings dialog box, open the Compile Options tab and enable the Replaces constants compile option.

## Error Example

Example for the Replaces constants compile option being disabled.

```
VAR CONSTANT
        iMaxInst : INT := 2;
END_VAR
VAR_STAT
        iIDs : INT := iMaxInst - 1;
END_VAR
```

--> C0549: Initialization of the static variable 'iIDs' is not constant, or replaced constants is disabled.
