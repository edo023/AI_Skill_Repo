# Visualization

## User Management

Visualizations can be protected as described in [*CODESYS Visualization – Setting Up User Management*](../../../../../api/crossBook?lang=en-US&virtualBookName=CODESYS_Visualization&topicID=_visu_struct_user_management).

## Encrypted Communication

For communication between an Internet browser and a controller running a WebVisu, an HTTPS connection with encryption is available which helps to protect the integrity of the displayed data. This function is not available for all controllers. Consult the *Programming Guide* specific to your controller for further information about firewall script commands.
