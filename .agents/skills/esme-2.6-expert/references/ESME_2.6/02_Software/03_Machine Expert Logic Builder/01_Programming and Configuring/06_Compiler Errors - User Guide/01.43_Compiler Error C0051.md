# Compiler Error C0051

## Message

Single byte string expected for an attribute value instead of ‘<value>’.

## Message Cause

A character string does not appear at the displayed location as expected.

## Solution

Replace the value with a string.

## Error Example

```
PROGRAM PLC_PRG
{IF hasattribute(pou: MyPOU, MyAttribute)}
{END_IF}
```

-->C0051: Single byte string expected for an attribute value instead of ‘MyAttribute’.

## Error Correction

```
PROGRAM PLC_PRG
{IF hasattribute(pou: MyPOU, 'MyAttribute')}
{END_IF}
```
