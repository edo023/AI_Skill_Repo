# Write all values from <Device.Application>

## Overview

The Write all values from <Device.Application> command is available in online mode in the contextual menu of an application or in the contextual menu of a POU editor of the application. It resets the values of variables of the active application to predefined values on the controller. This writing process is performed once at the beginning of the next cycle that is started.

## Preparing Values for Writing

You can prepare values for writing as follows:

* Click the Prepared value field in the declaration part and enter the value. For Boolean variables, toggle the value by clicking the field.
* Click the inline monitoring field of the variable in the implementation part of the FBD/LD/IL editor.
* Click the Prepared value field in the monitoring view, and enter the value.
