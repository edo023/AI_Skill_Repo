# Compiler Error C0091

## Message

Recursion in base function block list: <function block name>

## Message Cause

A base function block is extended by itself. Such recursion in the base function block list is not possible.

## Solution

Do not attempt to extend the function block recursively.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : FB;
END_VAR
```

```
FUNCTION_BLOCK FB EXTENDS FB
VAR
END_VAR
```

--> C0091: Recursion in the base function block list: FB -> FB
