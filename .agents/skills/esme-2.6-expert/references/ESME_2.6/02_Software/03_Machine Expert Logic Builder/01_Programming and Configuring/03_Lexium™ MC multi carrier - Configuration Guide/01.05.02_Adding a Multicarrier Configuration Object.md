# Adding a Multicarrier Configuration Object to a Project

## Open or Create a Project

Create a new project with File > New Project > From Example > Multi carrier Example.

This example provides a pre-configured project that you can adapt to your needs.

For further information, refer to the [Lexium™ MC multi carrier Example Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=exMulCar&topicID=MulticarrierExGuide_28FFC567).

Alternatively, you can create a new project and add the Multicarrier Configuration object manually.

## Add a Multicarrier Configuration Object

NOTE: A PacDrive LMC Pro2 controller must be available in your project to be able to add a Multicarrier Configuration object. Only one Multicarrier Configuration object can be added as sub node of the Application node.

| Step | Action | Result / Comment |
| --- | --- | --- |
| 1 | Right-click the Application node in the Devices tree. | – |
| 2 | Execute the command Add Object > Multicarrier Configuration... from the contextual menu. | The Add Multicarrier Configuration dialog box is displayed. |
| 3 | Confirm with the Add button. The Name cannot be modified. | * A Multicarrier Configuration object is added as a sub node of the Application node in the Devices tree. * The Multicarrier Configuration editor is displayed. |

Alternatively, to create / open a Multicarrier Configuration object, you can add a Lexium MC Track device to the Sercos Master node (or a Lexium MC Carrier Pool device to the controller node) in the Devices tree and double-click the object. As a result, the device editor is displayed that contains a Multicarrier Configuration tab. This tab allows you to open or to create a Multicarrier Configuration object (if not already available).
