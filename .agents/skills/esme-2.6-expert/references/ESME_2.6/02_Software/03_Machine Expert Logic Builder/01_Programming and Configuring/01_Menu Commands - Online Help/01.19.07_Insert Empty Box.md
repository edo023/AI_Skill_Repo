# Insert Empty Box

## Overview

Shortcut: CTRL + SHIFT + B

The FBD/LD/IL > Insert Empty Box command is used to insert an empty [box element](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083478)  into a network.

For detailed information on inserting boxes in the FBD/LD/IL editor, refer to the [chapter Insert Box](D-SE-0084108.html#D-SE-0084108). In contrast to this command, when inserting an empty box, instead of the Input Assistant, the instance field above the box will open automatically.

Decide, which type of box you need:

* If the box should represent a function block, then enter the desired instance variable name and close the input by pressing the RETURN key. You can use the Input Assistant for entering the name of an already existing instance. After you have inserted an already declared instance variable, the function block will be displayed accordingly. If you enter a not yet known instance name, you have to specify the name of the desired function block. For this purpose, after closing the instance name input, the input focus automatically changes to the box type edit field within the box. In this case, the Input Assistant will only offer function blocks.
* If the box should represent an operator, program, function, or interface, press the ARROW DOWN key when the cursor is placed in the instance field. The input focus will change to the box type field (within the box), where you can enter the respective operator, program, function, or interface name, directly or via the input assistance. After having terminated this input, the box will be displayed accordingly.

In the IL editor, the corresponding instructions will be inserted.
