# Compiler Warning C0513

## Message

Should not access private property <POU name> <property name>.

## Message Cause

A property of the POU, whose visibility is restricted to the POU with the `PRIVATE` keyword, is accessed from outside of the POU.

## Solution

Modify the visibility of the property (for example, remove the `PRIVATE` keyword) or modify the declaration of the variable.

## Error Example

```
FUNCTION_BLOCK POU
VAR_INPUT
END_VAR
...

PROPERTY PRIVATE Prop : UDINT

PROGRAM
PLC_PRG
VAR
      p:POU := STRUCT(Prop := 7);
END_VAR
```

--> C0513: Should not access private property POU Prop.
