# Update Device...

## Overview

The update device function allows you to replace a device selected in the Devices tree

* by another version of the same device or
* by a different type of device.

For detailed information on the update device function, refer to the chapter [*Updating Devices* in the *Programming Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083377).
