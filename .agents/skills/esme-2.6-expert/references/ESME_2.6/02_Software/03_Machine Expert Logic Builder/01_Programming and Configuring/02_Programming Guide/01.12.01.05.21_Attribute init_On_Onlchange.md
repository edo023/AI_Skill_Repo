# `Attribute init_On_Onlchange`

## Overview

To initialize variables on online change, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Attach the pragma {attribute 'init\_on\_onlchange'} to a variable to initialize this variable with each [online change](D-SE-0083446.html#D-SE-0083446__D-SE-0083446.9). |
| 2 | Open the Build tab of the Properties [dialog box of the application](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0083921) and enter the string `no_fast_online_change` in the Compiler defines box. |

## Syntax

{attribute 'init\_on\_onlchange' }

## Insert Location

Insert this pragma in the line above the declaration of the variables.
