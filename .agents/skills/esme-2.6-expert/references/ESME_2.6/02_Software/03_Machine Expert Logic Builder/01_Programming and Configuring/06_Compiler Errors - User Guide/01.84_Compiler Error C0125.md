# Compiler Error C0125

## Message

The constant <constant value> is assigned to more than one enumeration

## Message Cause

The same value is assigned to two or more enumerations.

## Solution

Assign different values to the enumerations.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : DUT;
END_VAR
```

```
TYPE DUT :
(
  enum_member := 0,
  enum_member2 := 0
);
END_TYPE
```

--> C0125: The constant 0 is assigned to more than one enumeration

EIO0000003933.04
