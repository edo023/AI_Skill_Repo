# Export Symbol Configuration...

## Overview

The Project > Export Symbol Configuration... command opens a file save dialog box that allows you to save the symbol configuration as .xml file to a folder of your choice. For further information on the symbol configuration file, refer to [Symbol Configuration](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083587).
