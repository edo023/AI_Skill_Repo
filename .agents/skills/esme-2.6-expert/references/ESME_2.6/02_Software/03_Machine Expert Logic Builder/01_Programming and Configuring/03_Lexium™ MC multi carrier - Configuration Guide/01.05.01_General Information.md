# General Information

## Overview

The Lexium™ MC12 multi carrier is a transport system to be used in machines. It uses linear motion technology to move products individually through the machine. Carriers are moved on a configurable track consisting of arc and straight segments.

You can use the Multicarrier Configuration editor to layout tracks in single-track or multi-track mode or you can add multi carrier devices (tracks, segments, carriers) to your application in the Devices tree. You can also use both possibilities in parallel.

NOTE: Configuration of carrier movements is not performed in the Multicarrier Configuration editor. It has to be implemented by using the function blocks of the Multicarrier library. For further information, refer to the [Multicarrier Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=MLSLib&topicID=FunctionBlocks_5C05E475).

For a general product overview, refer to:

* The [Lexium™ MC12 multi carrier](https://www.se.com/ww/en/product-range/52642224-lexium%E2%84%A2-mc12-multi-carrier-/) website (including video presentations).

## Multicarrier Configuration

The Multicarrier Configuration editor is provided by Schneider Electric as a part of EcoStruxure Machine Expert.

It is used to configure a Lexium™ MC multi carrier from scratch or to modify an existing configuration. As a result, the Multicarrier Configuration editor adapts the device configuration in EcoStruxure Machine Expert, and generates or modifies code in the application.

You can use the Multicarrier Configuration editor to layout tracks in single-track or multi-track mode by adding and parameterizing the following components:

* Straight and curved segments (pre-defined type, length, topological address, start position, end position)
* Stations (name, waiting position, process position)
* Carrier instances (number, carrier compounds)
* Carrier compounds (name, description, properties)
* Tools (name, description, properties)
* Products (name, description, properties)

You can display the tracks and their components in simplified graphical representations.

## Editor Modes

EcoStruxure Machine Expert V2.5 and later versions support the multi-track feature that allows to control multiple tracks with the same controller. Consequently, the Multicarrier Configuration editor supports the single-track mode and the multi-track mode, where multiple tracks are available in one project. When a second track is added to a project, the Multicarrier Configuration editor switches to multi-track mode. The differences between the two modes are listed in the following table.

## EcoStruxure Machine Expert Twin

The EcoStruxure Machine Expert installation includes the installation files of EcoStruxure Machine Expert Twin. EcoStruxure Machine Expert Twin can be used with the EcoStruxure Machine Expert license.

NOTE: EcoStruxure Machine Expert Twin can also be installed as a standalone version from the Schneider Electric Software Installer by clicking Install new software and selecting the product EcoStruxure Machine Expert Twin. In this case, a separate license is required.

A viewer of EcoStruxure Machine Expert Twin is embedded in the Multicarrier Configuration editor. With this, you can display a single track or a scene with multiple tracks as a 3-D emulation and see your Lexium™ MC12 multi carrier system running virtually (by usage of a motion controller but without real mechanics).

This helps you to analyze your application behavior and to optimize your track design.

NOTE: Only one emulation can be active at a time.

## Parameters of Added Devices

Double-click a multi carrier device object (track, segment, carrier pool, carrier) in the Devices tree to open the device editor dialog box. Select the Parameters view to edit the device-specific parameters.

For details, refer to the Lexium™ MC multi carrier, Device Objects and Parameters Guide.

## Project Settings

The Project > Project Settings > Multicarrier Configuration Editor dialog box provides settings concerning the general behavior of the Multicarrier Configuration editor, and the pre-settings for the device and code generation.

For details, refer to [Menu Commands Online Help](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=TPC_MLS_Config_Tools_Options_D995A499)
