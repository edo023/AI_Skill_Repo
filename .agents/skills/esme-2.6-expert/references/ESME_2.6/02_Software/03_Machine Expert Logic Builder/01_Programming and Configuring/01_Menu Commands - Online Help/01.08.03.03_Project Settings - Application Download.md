# Project Settings - Application Download

## Overview

The Project Settings > Application Download dialog box allows you to change the procedure of [downloading an application to the controller](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083446) in such a way that the application running on the controller is stopped before the new application is downloaded. This may lead to a faster download.

By default, the application download is performed according to the following procedure:

1. Verifying the connection to the controller.
2. Transferring the application.
3. Stopping the controller.
4. Resetting the controller.
5. Starting the controller.

## Stopping the Application Before Downloading the New Application

Activate the option Stop running application before new application is downloaded to modify the sequence of the application download process as follows:

1. Verifying the connection to the controller.
2. Stopping the controller.
3. Transferring the application.
4. Resetting the controller.
5. Starting the controller.

NOTE: This function is not available for all controllers. The list Supported controllers inside this project provides the controllers used in your project that will be stopped before application download. For other controllers not mentioned in this list, the default application download procedure is applied.
