# Compiler Error C0036

## Message

Cannot call object of type ‘<type>’.

## Message Cause

An attempt has been made to call an object that does not support calls.

## Solution

Only functions, function blocks, programs, methods, and actions can be called.

## Error Example

```
VAR_GLOBAL GVL
  value : INT;
END_VAR
PROGRAM PLC_PRG
GVL();
```

-->C0036: Cannot call object of type ‘VAR\_GLOBAL’.
