# Running the HMI application

1. Click the ![](images/uuid-7c0bdf31-b663-b423-aac8-9c7d1a.png) symbol.

   * The application is compiled.
2. Click the start ![](images/uuid-76462a36-7abf-d99c-52e6-69de3c.png) symbol.

   * The HMI application is executed. The visualization starts.

     The **Alarm Table** element shows the active alarms of the remote PLC. You can acknowledge the alarms and also switch to the "History" view.
