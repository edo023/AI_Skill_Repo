# Command: Export Trace Configuration

**Function**: The command opens the **Export trace configuration** dialog.

**Call**: Context menu of the top node in the tree view of the trace configuration.

**Requirement**: The dialog **Trace Configuration** is active and the name of the trace configuration is selected in the tree view (example: `Visu_Trace1`).
