# Designing a visualization element with style properties

The set style includes style properties. These are provided in the **Properties** view of an element in the list box of the **Value** column. It is checked which style properties are appropriate for which property. For example, only style properties with color definitions are available for a color assignment.

TIP:

A style can have directly defined visualization element properties. If this style is used in the project, then these properties are not configurable anymore.

Requirement: A project with a visualization is open.

1. Double-click the visualization.
2. Select a style property.

   * The visualization shows the element according to the style.

**Example**

A visualization uses the style `CompanyStyle8`, which defines the colors `CompanyRed`, `CompanyBlue`, and `CompanyGreen`. An element is selected in the visualization. You can configure the element in the **Properties** view. By clicking into the value field of the **Color** property, you receive a list box with the entries `CompanyRed`, `CompanyBlue`, and `CompanyGreen`.
