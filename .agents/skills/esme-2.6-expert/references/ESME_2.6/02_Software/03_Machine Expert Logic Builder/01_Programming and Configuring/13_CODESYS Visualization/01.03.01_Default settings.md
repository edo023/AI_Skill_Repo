# Default settings

Below you can see which default settings are available for visualizations.

| Scope | Setting Location | Setting |
| --- | --- | --- |
| Throughout CODESYS | [Dialog: Options: Visualization](_visu_dlg_options.html#_visu_dlg_options)  [Dialog: Options: Visualization Styles](_visu_dlg_options_styles.html#_visu_dlg_options_styles) | * Visualization editor: display, handling * Paths of the basic text and image files * Visualization style |
| [Dialog: Customize – Menu](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0084067) | Adaptation of the visualization menu |
| Throughout the project | [Dialog: Project Settings: Visualization](_visu_dlg_project_settings.html#_visu_dlg_project_settings) | * Property handling for the visualization elements * Paths of the basic text and image files * Symbol libraries with prepared content * Visualization Profile |
| Project-specific | [Dialog: Project Environment: Visualization Styles](_visu_dlg_project_environment_visualization_style.html#_visu_dlg_project_environment_visualization_style)  [Dialog: Project Environment: Visualization Symbols](_visu_dlg_project_environment_visualization_symbols.html#_visu_dlg_project_environment_visualization_symbols) | Updates of the visualization profile, visualization styles, and visualization symbol libraries |
| Throughout the application | [Object: Visualization Manager](_visu_obj_manager.html#_visu_obj_manager) | * Unicode, CurrentVisu variable, multitouch, semi-transparency, memory size, data transmission, number of clients * Visualization style * Language setting, language-specific font * Default keyboard configuration * Visualizations and visualization references * Font for each language * User Management * Responsive design (concerns visualization size and its content) |
| Single visualization | [Dialog: Properties](_visu_dlg_object_properties.html#_visu_dlg_object_properties) | * Visualization type according to the purpose of use, area of use * Size definition   TIP: You can determine the visualization type on the **Visualization**  tab. |
| Display variant of an individual visualization | [Object: CODESYS WebVisu](_visu_obj_webvisu.html#_visu_obj_webvisu)  [Object: CODESYS TargetVisu](_visu_obj_targetvisu.html#_visu_obj_targetvisu)  [Object: CODESYS Remote TargetVisu](_visu_obj_remote_targetvisu.html#_visu_obj_remote_targetvisu) | * Start visualization, refresh rate, buffer size, HTM file name * Scaling Options * Presentation Options * Default Text Input |
