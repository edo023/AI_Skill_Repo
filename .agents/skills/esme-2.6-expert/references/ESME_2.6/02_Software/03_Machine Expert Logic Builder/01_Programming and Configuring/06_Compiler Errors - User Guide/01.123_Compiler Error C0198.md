# Compiler Error C0198

## Message

String constant ‘<string value>’ is too long for the destination type ‘<data type>’

## Message Cause

The string constant has an excessive number of characters.

## Solution

Use shorter string constants or declare larger strings.

## Error Example

```
PROGRAM PLC_PRG
VAR
  str : STRING(4) := '12345';
END_VAR
```

--> C0198: String constant '12345' is too long for the destination type 'STRING(4)'
