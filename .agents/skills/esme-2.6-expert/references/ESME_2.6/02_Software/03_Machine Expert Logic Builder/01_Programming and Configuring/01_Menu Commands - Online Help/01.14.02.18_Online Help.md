# Online Help

## Overview

The Tools > Options > Online Help dialog box allows you to configure your individual preferences concerning the online help.

The option Prefer Offline Help is applicable if you have [downloaded the online help for offline usage as described in the Installer User Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=Installer&topicID=D_SE_0095358):

* Activate the option to search your local copy of the EcoStruxure Machine Expert online help whenever a help topic is requested.

  If the requested help topic is not available in your local copy, an attempt is made to access the EcoStruxure Machine Expert online help through the Internet.
* Deactivate the option to access the EcoStruxure Machine Expert online help through the Internet whenever a help topic is requested. If the requested help topic is not available or access to the Internet is not provided, the help topic is loaded from your local copy of the online help.

## Adding Individual Help Files

The Directory for User Documentation field of the Tools > Options > Online Help dialog box allows you to link your own help files (in \*.chm or \*.html format). For a detailed description on the procedure, refer to the document [How To Implement a User Documentation](../../../../../api/crossBook?lang=en-US&virtualBookName=HowCHM&topicID=HowToUserDocu_E87BD6C3).
