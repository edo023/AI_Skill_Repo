# Convention: Uncommented Variable (In+Out+Global)

## User Description

This convention verifies whether uncommented variables defined in VAR\_GLOBAL, VAR\_INPUT, VAR\_OUTPUT, or VAR\_IN\_OUT exist in source code.

## Example

Declaration:

```
1: PROGRAM SR_Main
2: VAR_IN
3:     i_xCheck1: BOOL;//flag to identify
3:     i_uiMyVariable2: UINT;
4: END_VAR
2: VAR
3:     xFlag: BOOL;
4: END_VAR
```

i\_uiMyVariable2 is not commented. Therefore a convention violation is created.

EIO0000002710.08
