# Compiler Error C0189

## Message

‘;’ expected instead of ‘<token>’

## Message Cause

Syntax error

## Solution

Make sure that the syntax is correct.

## Error Example

```
PROGRAM PLC_PRG
VAR
  INT
END_VAR
```

--> C0009: Unexpected Token '<Token>' found

--> C0189: ';' expected instead of 'END\_VAR'
