# Configuration

## Configuration Tab

The Configuration tab allows you to select a target structure for its generation and to define the properties.

The Configuration tab allows you to do the following:

* Select a target structure for source code generation.
* Define properties for this structure.

| Element | Description |
| --- | --- |
| Targeted technology | The Targeted technology section provides two target structures (cam data structure) for source code generation:   * CMT.ST\_MultiCam (Library: CommonMotionTypes) * EXCAM.IF\_Cam (Library: ExtendedCam)   Cam data is initialized every time the application program is started.  If the expression is undeterminable, a diagnostic message is generated.  This function is not available for all controllers. Consult the *Programming Guide* specific to your controller for further information. |
| Code generation variant | The following options are available:   * Global variable instance (Variable name: ‘<YourCamDiagramName>’) * Function block type including initialization code (Type name: ‘<YourCamDiagramName>’) * Initialization function for SmartTemplate modules * Existing IEC structure: Enter a structure or click the browse button (...) to open the Input Assistant dialog box to browse for a structure. |
| CMT.ST\_MultiCam configuration | If Targeted technology > CMT.ST\_MultiCam (Library: CommonMotionTypes) is selected, configure the values of the following parameters:   * User table Id: Enter a positive integer (including 0). * Y - Period: Enter a positive floating-point number (including 0).   The default value for both parameters is 0. |
