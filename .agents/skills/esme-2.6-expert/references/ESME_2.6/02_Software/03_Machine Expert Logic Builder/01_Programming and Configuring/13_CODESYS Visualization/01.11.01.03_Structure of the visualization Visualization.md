# Structure of the visualization `Visualization`

This screen consists of control and display elements for the refrigerator.

![](images/uuid-7f98a42a-fa2f-4a87-10d7-bee809.png)

* ![](images/uuid-56937654-6436-89d4-229c-864584.png): Numeric display of the actual temperature
* ![](images/uuid-13916c51-cd03-dd8d-d783-2eb63e.png): Pointer to display of the actual temperature
* ![](images/uuid-b36dc361-fb0c-4605-659d-b973b7.png): Numeric display of the set temperature
* ![](images/uuid-88b4247e-0816-9eb0-e2ed-c1ce93.png): Potentiometer for setting the set temperature
* ![](images/uuid-608e0b83-feb2-b0cc-4f6f-eb8637.png): Label for compressor lamp
* ![](images/uuid-33fe82d4-4738-442e-0d58-59894d.png): Lamp for compressor on
* ![](images/uuid-72428975-b38e-26ee-d242-0a8af5.png): Label for signal lamp
* ![](images/uuid-e4e0ddaf-e618-c1a4-8256-16af16.png): Lamp for signal "Close doors"
* ![](images/uuid-92c61aa5-9832-75d4-3eff-6fa909.png): Switch for opening and closing the refrigerator door

1. Open the visualization `Visualization` in the editor.
2. Drag a **[Rocker Switch](_visu_elem_switch.html#_visu_elem_switch)** visualization element to the editor.

   **Change the following properties**

   * **Variable**: `Glob_Var.rDoorOpen`
