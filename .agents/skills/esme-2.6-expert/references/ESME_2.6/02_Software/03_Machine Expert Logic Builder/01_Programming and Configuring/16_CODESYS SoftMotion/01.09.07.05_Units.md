# Units

Forces and torques and their limits are basically specified in the application in newtons (N) or newton meters (Nm). Before set forces or set torques are sent to the drives, a rescaling may be done. For this, the CiA 402 standard requires, for example, that set torques are sent to the drive in thousandths of the rated torque (in mNm).

The unit for lengths is user-specific in SoftMotion. Because forces and torques are defined as a fixed unit (N or Nm), a conversion factor from a user-specific length unit to meters has to be configured (unless meters are used as length unit). This can be done with the `SMC_GroupSetUnits` function block.
