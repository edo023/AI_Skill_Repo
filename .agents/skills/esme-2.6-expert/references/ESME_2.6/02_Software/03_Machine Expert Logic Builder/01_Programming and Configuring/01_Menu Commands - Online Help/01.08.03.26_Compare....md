# Compare...

## Overview

Use the Project > Compare command to compare the actual project with another one (reference project). The Project Comparison editor allows you to specify the reference project and the compare options.

The Project Comparison editor contains the following parameters:

| Parameter | Description |
| --- | --- |
| Compare the currently open project with: | |
| Project on disk: | Enter the path to the reference project on the file system or click the ... button to browse for the project file. |
| Compare two selected objects | This option allows you to compare two objects (for example, two POU elements) that are selected in a navigator by running the Project > Compare command and clicking OK in the Project Comparison editor. |
| Comparison Options: | |
| Ignore Whitespace | If this option is selected, differences concerning blank spaces between the two projects are ignored except for semantically relevant spaces, for example, in string literals. |
| Ignore Comments | If this option is selected, comments in the programming code are ignored for the comparison. |
| Ignore Properties | If this option is selected, object properties are ignored for the comparison. |

The result of the comparison is represented in a new view titled [**Project Comparison - Differences**](D-SE-0083939.html#D-SE-0083939).
