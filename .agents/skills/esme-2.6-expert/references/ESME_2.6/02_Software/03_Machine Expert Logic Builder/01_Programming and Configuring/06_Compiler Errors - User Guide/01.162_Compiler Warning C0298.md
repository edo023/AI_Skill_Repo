# Compiler Warning C0298

## Message

Calculation of stack usage incomplete because of recursive calls, starting at '<recursive method name>'.

## Message Cause

The verification of the stack cannot be performed for methods with a recursive call because the stack usage cannot be determined.

## Solution

Use the `estimated-stack-usage` attribute to provide an estimated value (in bytes) for the stack size requirement to the method.

## Error Example

Following a method of a function block (FB) that is called by a POU, like, for example, `"fb.m_Recursive(255);"`:

```
METHOD PUBLIC m_Recursive : UDINT
    VAR_INPUT
            uiN : UINT;
    END_VAR
    VAR
    END_VAR
    m_Recursive := 1;
    IF      uiN > 1 THEN
            m_Recursive := uiN * THIS^.m_Recursive(uiN := (uiN - 1) );
            RETURN;
    ELSE
            RETURN;
    END_IF;
```

--> C0298: Calculation of stack usage incomplete because of recursive calls, starting at 'm\_Recursive'

Solution for the example: Decrease 255 to 254.
