# Calling Scripts from Menu Commands

Requirement: A valid Python script file `<file name>.py` is located in the file system. The CODESYS user interface is open.

1. (optional) To monitor the processing of individual commands used in the script, click **Tools → Scripting → Enable Script Tracing**.
2. In CODESYS, click **Tools → Scripting → Execute Script File**.

   * The statements in the script are executed and, if script tracing is activated, listed in the message view.
