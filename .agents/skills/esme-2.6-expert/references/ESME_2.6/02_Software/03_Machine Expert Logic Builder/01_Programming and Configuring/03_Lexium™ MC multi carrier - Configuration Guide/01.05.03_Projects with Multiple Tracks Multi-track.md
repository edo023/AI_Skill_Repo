# Projects with Multiple Tracks: Multi-track Mode

## Adding Tracks to the Project

By default, in single-track mode, the Multicarrier Configuration object in the Logic Builder includes one track that is not displayed in the tree structure. To add a second track and switch to multi-track mode, proceed as follows:

| Step | Action | Result / Comment |
| --- | --- | --- |
| 1 | Right-click the Multicarrier Configuration node. | – |
| 2 | Execute the command Add Object > Multicarrier Track... from the contextual menu. | The Add Multicarrier Track dialog box is displayed. |
| 3 | Modify the Name to, for example, `My_MC_Track`, and click Add. | * Two track subnodes are added to the Multicarrier Configuration node:    1. The <Track Base Name>\_<n> node represents the default track that is created for single track usage in the background and only becomes visible once a second track is created. The name of the node is defined with the parameter Track Base Name in the tab Device Generation of the Project > Project Settings > Multicarrier Configuration Editor dialog box (refer to [Project Settings - Multicarrier Configuration Editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=TPC_MLS_Config_Tools_Options_D995A499)).   2. The `My_MC_Track` node for the second track you just created. * The `My_MC_Track` editor opens for the second track you just created. For further information, refer to [Track Editor](#MultipleTracks-E4BEDD1B__TrackEditor-E4C5BAC5). * The Multicarrier Configuration editor switches to multi-track mode. Also refer to [Multicarrier Configuration Editor in Multi-track Mode](#MultipleTracks-E4BEDD1B__EditorInMultitrackMode-E7FFF7AF). |

## Track Editor

If your project contains more than one track, a track editor is provided for each track. It opens upon double-clicking the respective track object in the Applications tree. Track editors provide the following track-specific tabs:

* [Track Tab](TPC_MLS-Config_Tab_Track-D99C4272.html)
* [Stations Tab](TPC_MLS-Config_Tab_Stations-DA9DBC46.html)
* [Emulation Tab](TPC_MLS-Config_Tab_Visualization-DA963F21.html): To emulate the track selected with this track editor.

## Multicarrier Configuration Editor in Multi-track Mode

When a second track is added, the Multicarrier Configuration editor switches to multi-track mode. This has the following effects:

* An Overview tab is added. For a description of the tab, refer to [Overview Tab (Exclusive to Multi-track Mode)](OverviewTab-E4C6B98A.html).
* Track-independent tabs are displayed:

  + [Station Types Tab](StationTypeTab-F03E51E0.html)
  + [Carriers Tab](TPC_MLS-Config_Tab_Carriers-DAAA0843.html)
  + [Tools Tab](TPC_MLS-Config_Tab_Tools-DCC4C529.html)
  + [Products Tab](TPC_MLS-Config_Tab_Products-DDB83FF9.html)
  + [Emulation Tab](TPC_MLS-Config_Tab_Visualization-DA963F21.html): To emulate a scene that contains all tracks of the project.
* The track-specific tabs are no longer available in the Multicarrier Configuration editor. They are moved to the specific [track editors](#MultipleTracks-E4BEDD1B__TrackEditor-E4C5BAC5).
