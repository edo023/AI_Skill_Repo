# Command: View Standard Metrics

Symbol: ![](images/uuid-d005e4d0-9125-9204-b35b-a4584c.png)

**Function**: The command starts the static code analysis for the active application. The default metrics for all programming blocks are then calculated and displayed in a table on the **Standard Metrics** tab.

**Call**: **Build → Static Analysis** menu
