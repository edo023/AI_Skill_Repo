# Project Settings - Source Download

## Overview

The following settings are valid for a [**source download**](D-SE-0083901.html#D-SE-0083901) that is transferring the project files to the controller.

| Element | | Description |
| --- | --- | --- |
| Destination device | | By default, no device is predefined here. Select one from the list of programmable devices defined in the project. If a destination device is defined here, you cannot perform a source download if you are connected to another device. |
| Content area | | Predefines the selection of project files to be stored in the archive file for source download. |
|  | Use compact download | If this option is activated, only the source code of the device of the active application will be downloaded. The name of the copied project consists of the original name and the name of the device hosting the active application.  If this option is deactivated, the sources for all devices in the project will be downloaded.  NOTE: When you are logging in to the controller with the uploaded project (via the Source Upload command), you are prompted to perform a Login with online change, Login with download or a Login without any change as described for the [*Downloading an Application* process](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083446). Select the first option to harmonize the projects and not to be prompted at the next login. |
| Additional files... | Click this button to open a dialog box where additional files can be added to the source download. For example Boot projects, Download information files and so on. |
| Timing area | | Predefines when a source download will be done. |
|  | Implicitly at program download and online change | The source download is executed at each program download and online change (command [**Online \ Download <Application>**](D-SE-0083996.html#D-SE-0083996)) without user interaction. |
| Implicitly at creating bootproject | The source download is executed at each boot project creation (command [**Online \ Create boot application**](D-SE-0083995.html#D-SE-0083995)) without user interaction. |
| Implicitly at creating bootproject, download and online change | The source download is executed at each boot project creation (command [**Online \ Create boot application**](D-SE-0083995.html#D-SE-0083995)) and at each program download and online change (command [**Online \ Download <Application>**](D-SE-0083996.html#D-SE-0083996)) without user interaction. |
| Prompt at program download and online change | At each program download and online change, you will be asked whether a source download should be done. |
| Only on demand | The source download has to be initiated by command [**Source Download**](D-SE-0083901.html#D-SE-0083901). |

NOTE: Consider that depending on your choices in the Content and Timing area, these may impact the performance of the corresponding commands. For example, when you choose Implicity at program download and online change together with Referenced devices and Referenced libraries (under Addition files...) may significantly increase the time for performing an online change.
