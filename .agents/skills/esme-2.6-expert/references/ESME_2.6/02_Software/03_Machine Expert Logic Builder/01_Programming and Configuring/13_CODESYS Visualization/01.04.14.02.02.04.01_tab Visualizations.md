# Tab: Visualizations

|  |  |
| --- | --- |
| **Default behavior** | : All visualizations of the application are automatically downloaded to the visualization devices.  : The download behavior is defined explicitly for each visualization.  Hint: Use the explicit selection for referencing visualizations indirectly by means of IEC variables. |
| **Visualizations** | List of all visualizations of the application. This comprises visualizations in the device tree, POU view, and libraries. |
| **Dialogs** | All visualizations of type **Dialog** of the project and libraries are listed below.  Note: The number of dialog instances is displayed in the **Number of Instances** column. |

|  |  |
| --- | --- |
| **<name of RemoteTargetVisu>, <name of TargetVisu>, <name of WebVisu>,** | Engineered display variants of the application.  Example: `rtvDevice, tvDevice, wvBrowser`  : The visualization is downloaded to the display variants when the application is downloaded.  Hint: If you click **Deactivate Unused Visualizations**, then all visualizations not referenced in the application are deactivated. The command is available in the context menu of the check box (right-click). This reduces the memory requirement on the display variant. |
| **Number of Instances** | The dialog is downloaded with the specified number of instances and can be executed as often as specified.  Example: `2`  Note: This column is relevant for dialogs only. |
