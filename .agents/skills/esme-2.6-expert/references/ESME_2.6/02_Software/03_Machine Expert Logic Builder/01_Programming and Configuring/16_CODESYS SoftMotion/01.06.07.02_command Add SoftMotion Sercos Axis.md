# Command: Add SoftMotion Sercos Axis

**Function**: The command adds a general SoftMotionSercos axis below a Sercos Module.

**Call**: **Project** menu; context menu of the Sercos Slave

**Requirement**: An appropriate Sercos slave is selected in the device tree.

IMPORTANT:

Sercos axes are no longer supported for SoftMotion Version 4.17.0.0 and higher.

This command is used in order for any servo drive to work with CODESYS SoftMotion. The drive only has to support the Sercos standard. CODESYS uses a standard driver for communicating with the device. The way in which the drive works or not with the device depends on the implementation in the device itself. Therefore, there is no guarantee how the driver works or not with the device.

TIP:

The generic Sercos driver supports multi-axis devices up to eight axes.
