# Compiler Error C0219

## Message

Case contains overlapping range <case range 1 begin> .. <case range 1 end> and <case range 2 begin> .. <case range 2 end>

## Message Cause

Two branches of case markers have the same elements or subsets.

## Solution

Make sure that there is no intersecting.

## Error Message

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
CASE i OF
  3..5: i := i+2;
  1..4: i := i+2;
ELSE
  i := i+10;
END_CASE;
```

--> C0219: Case contains an overlapping range 1 .. 4 and 3 .. 5
