# Compiler Error C0061

## Message

Bit access on function call is not allowed

## Message Cause

Bit access is performed on a function.

## Solution

Use bit access only for supported data types.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
Test().2;
```

```
FUNCTION Test : INT
VAR_INPUT
```

```
END_VAR
```
