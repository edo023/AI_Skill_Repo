# Insert ST Block

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | Ctrl + Shift + S |
| Call | * Ladder > Insert ST Block menu * Contextual menu |

## Function

The command inserts an ST block before the selected element. Possible insertion positions are indicated by the plus symbol at the cursor when you drag the element from the Toolbox to the implementation part. Double-clicking the block opens a text editor for entering ST code.

## Element: ST Block

The element is used to add ST code in a Ladder POU. The code in the ST block is executed when the network is processed.
