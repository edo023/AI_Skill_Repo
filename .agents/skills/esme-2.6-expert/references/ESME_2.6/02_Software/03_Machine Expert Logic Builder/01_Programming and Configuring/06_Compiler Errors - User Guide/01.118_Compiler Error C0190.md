# Compiler Error C0190

## Message

‘;’ expected instead of end of POU

## Message Cause

Syntax error in the POU.

## Solution

Make sure that the syntax is correct.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
i := 5
```

--> C0190: ';' expected instead of end of POU
