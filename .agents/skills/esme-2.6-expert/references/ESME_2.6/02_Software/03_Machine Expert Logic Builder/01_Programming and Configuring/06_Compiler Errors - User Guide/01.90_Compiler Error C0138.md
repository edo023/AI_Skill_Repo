# Compiler Error C0138

## Message

No matching ‘FB\_Init’ method found for instantiation of POU.

## Message Cause

No FB\_Init method exists that accepts the passed parameters.

## Solution

Verify which arguments FB\_Init has to receive and adjust the passed arguments.

## Error Example

```
PROGRAM PLC_PRG
VAR
  myPOU : POU(arg1 := 1, arg2 := 2);
END_VAR
```
