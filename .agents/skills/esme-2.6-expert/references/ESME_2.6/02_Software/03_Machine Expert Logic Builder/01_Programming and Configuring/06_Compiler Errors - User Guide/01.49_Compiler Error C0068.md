# Compiler Error C0068

## Message

Compare is not possible on objects of the type ‘<data type>’

## Message Cause

Objects are being compared in which a comparison is not possible.

## Solution

Only compare data types in which a comparison is possible (INT, REAL, etc.).

## Error Example

```
PROGRAM PLC_PRG
VAR
  b : BOOL;
  arr1 : ARRAY [1..2] OF INT;
  arr2 : ARRAY [1..2] OF INT;
END_VAR
```

```
 b := arr1 > arr2;
```

--> C0068: Compare is not possible on objects of the type 'ARRAY [1..2]'

EIO0000003933.04
