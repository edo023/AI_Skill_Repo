# Compiler Error C0197

## Message

Implicit conversion from ‘<data type 1>’ to ‘<data type 2>’: possible loss of information

## Message Cause

An attempt is made to convert a variable from data type DINT or LINT to data type REAL.

## Solution

Use the data type LREAL for DINT and when converting from LINT to LREAL, make sure that the value of LINT does not exceed the capacity of LREAL.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : DINT;
  b : REAL;
END_VAR
```

```
b := i;
```

--> C0197: Implicit conversion from 'DINT' to 'REAL': possible loss of information
