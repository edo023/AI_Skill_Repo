# Compiler Warning C0566

## Message

The FB\_ReInit method of a function block or struct must have no inputs and a return value of type BOOL. The FB\_ReInit will not be called automatically.

## Message Causes

One or more inputs are defined for FB\_ReInit.

For further information, refer to the [FB\_Reinit method](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083611_6) in the Programming Guide.

## Solutions

Remove the inputs from the FB\_ReInit method.

## Error Example

```
METHOD FB_ReInit : BOOL
VAR_INPUT    
    input_var : INT; //C0566: unexpected input for FB_ReInit
END_VAR
```

## Error Correction

```
METHOD FB_ReInit : BOOL
// correction: inputs have been removed
END_VAR
```
