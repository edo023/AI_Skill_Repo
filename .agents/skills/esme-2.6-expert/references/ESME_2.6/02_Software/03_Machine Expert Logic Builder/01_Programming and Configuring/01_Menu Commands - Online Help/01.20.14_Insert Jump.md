# Insert Jump

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | Ctrl + L |
| Call | * Ladder > Insert Jump menu * Contextual menu |

## Function

The command inserts a jump element before the selected element. Possible insertion positions are indicated by the plus symbol at the cursor when you drag the element from the Toolbox to the implementation part.

## Requirements

A connector is selected.

## Element: Jump

In LD, a jump is inserted either directly before an input, directly after an output, or at the end of the network, depending on the cursor position.

Specify the jump label of a network as the jump destination directly after the jump element. For adding a jump label, refer to [Adding a Network Title, Network Comment or Jump Label](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=AddingANetworkTitleNetworkCommentOr_49F24553).
