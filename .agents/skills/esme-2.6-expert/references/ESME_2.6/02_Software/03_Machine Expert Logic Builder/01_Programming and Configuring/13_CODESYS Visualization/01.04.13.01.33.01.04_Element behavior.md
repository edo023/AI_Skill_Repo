# Element behavior

|  |  |
| --- | --- |
| **Element behavior** | * **Image toggler**: Every mouse click changes the switch and the  **Variable** value. * **Image tapper**: The switch is "on" and the **Variable** value is `TRUE` while the mouse button is pressed. |
| **Orientation** | * **At top**: The rotary switch turns from the top right to the top left. * **At side**: The rotary switch turns from the top right to the bottom right. |
| **Color change** | : The element changes in color when  **Variable** is `TRUE`. |
