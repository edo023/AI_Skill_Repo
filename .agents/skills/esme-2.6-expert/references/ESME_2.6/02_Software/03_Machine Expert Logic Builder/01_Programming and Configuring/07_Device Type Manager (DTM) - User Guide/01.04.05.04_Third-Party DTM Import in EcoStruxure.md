# Third-Party DTM Import in EcoStruxure Machine Expert with the DTM Repository

Third-Party DTM Import in EcoStruxure Machine Expert with the DTM Repository

| Step | Action |
| --- | --- |
| 1 | In the menu bar, select Tools > DTM Repository.  Result: The DTM Repository window is displayed. |
| 2 | Click Import third party DTMs.  Result: The Install DTM window is displayed. The PC is scanned to detect third-party DTMs installed and registered on the PC. All third-party DTMs which have been detected as installed and registered on the PC are listed. |
| 3 | Wait for the end of the [scan](../glossary/glossary.htm#XREF_D_SE_0024697_166).  Result: All third-party DTMs which are currently installed and registered on the PC are displayed (the list is an example):  G-SE-0029149.3.gif-high.gif |
| 4 | Click OK.  Result: The selected DTMs are now imported and available in the Device Repository window. |

NOTE: If you wish to use a third-party DTM with EcoStruxure Machine Expert software, contact your local Schneider Electric representative.
