# Command: Visualization Tags View

Symbol: ![](images/uuid-3ec4e119-aa58-69ee-0aca-957470.png)

**Function**: The **Visualization Tags View** dialog opens. The **Number of visualization tags** software metric is displayed in detail there.

**For more information, see the following:**

* Tab: Software metrics for license determination,
* [Licensing](_vis_licensing.html#_vis_licensing)

**Call**:

* Context menu of a visualization object

  **Requirement**: A visualization object is selected in the POU or device view.
* Select the command in the **Visualization** menu.

  **Requirement**: A visualization is open (active visualization editor).
