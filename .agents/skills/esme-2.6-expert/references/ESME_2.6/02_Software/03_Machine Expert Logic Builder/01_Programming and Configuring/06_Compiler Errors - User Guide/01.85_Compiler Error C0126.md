# Compiler Error C0126

## Message

The variable of the type ‘<data type>’ requires exactly 1 index

## Message Cause

Multiple indexes are assigned to a variable with one index.

## Solution

Only assign one index.

## Error Example

```
PROGRAM PLC_PRG
VAR
  pi : POINTER TO INT;
END_VAR
```

```
pi[0,1] := 0;
```

--> C0126: The variable of the type 'POINTER TO INT' requires exactly 1 index

## Error Correction

```
pi[0] := 0;
```
