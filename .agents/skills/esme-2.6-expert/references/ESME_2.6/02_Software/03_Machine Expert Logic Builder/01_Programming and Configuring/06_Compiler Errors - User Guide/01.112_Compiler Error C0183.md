# Compiler Error C0183

## Message

Global scope operation ‘.’ is not valid on expression ‘<expression>’.

## Message Cause

The ‘.’ operator is used to access a global variable. However, at this location it is not followed by a valid IEC identifier, but for example a character such as ‘;’ or a reserved identifier such as FUNCTION, or an operator such as TO\_STRING.

## Solution

Use a valid IEC identifier for a global variable.

## Error Example

```
PROGRAM PLC_PRG
...
iVar := .FUNCTION;// ERROR: C0183 because '.' is not a valid identifier
strVar := .TO_STRING;
```

-->C0183: Global scope operation ‘.’ is not valid on expression ‘<expression>’.

## Error Correction

```
global value is declared in a GVL.
PROGRAM PLC_PRG
iVar := .globalValue;
```
