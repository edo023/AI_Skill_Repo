# Library Packages

## Overview

A library package is a file with the extension .librarypackage that contains a set of libraries. To create a library package that includes the libraries of your choice, execute the File > Create a Library-Package... command. To install an already available library package, execute the File > Open Project... command.

## Creating a Library Package

To create a library package, proceed as follows:

| Step | Action | Comment |
| --- | --- | --- |
| 1 | Execute the File > Create a Library-Package... command. | **Result**: The Create Library-Package dialog box is displayed. |
| 2 | Select the libraries to include in the library package. | To filter the number of displayed libraries, you can use the following options:   * Enter text in the box in the upper part of the dialog box to display only those libraries that contain the text. * Select a Company from the list to display only libraries of this company. * Select the option Show all versions (for experts only) to display installed versions of the libraries indented below the selected library entry. By default, this option is deactivated and only the latest version of each library is displayed. |
| 3 | Click the OK button. | **Result**: The Save As dialog box is displayed. |
| 4 | Enter a file name, and click Save. | **Result**: The .librarypackage file is saved to the selected directory. |

## Installing a Library Package

To install a library package, proceed as follows:

| Step | Action | Comment |
| --- | --- | --- |
| 1 | Execute the File > Open Project... command. | **Result**:The Windows Open Project dialog box is displayed. |
| 2 | Browse to the .librarypackage file, select it, and click the Open button. | – |
