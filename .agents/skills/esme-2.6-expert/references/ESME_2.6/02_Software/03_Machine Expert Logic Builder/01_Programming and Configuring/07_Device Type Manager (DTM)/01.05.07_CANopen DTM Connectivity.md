# CANopen DTM Connectivity

![](images/logo.png)

CANopen DTM Connectivity

Executing an Online DTM Command over CANopen

Overview

To execute the four Online [DTM commands](../DTMs_in_ESME/DTMs_in_ESME-6.htm#XREF_D_SE_0008095_4), Connect, Upload and Connect, Upload from Device and Download to Device, log in to the controller.

How to Execute a Command

| Step | Action |
| --- | --- |
| 1 | Build the application and download it to the controller. |
| 2 | Log in to the controller. Refer to [Login in the EcoStruxure Machine Expert](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083444_1). |
| 3 | Execute the required DTM command. |
