# Executing as Online Visualization

The online visualization can be operated as usual in the online mode. In contrast to the integrated visualization, however, there is a visualization client below the Visualization Manager. If there is a visualization client and the `VISU_TASK` task below the task configuration, then this is the online visualization. The same functionality is then available as with the TargetVisu or WebVisu.

TIP:

When you select the **Support client animations and overlay of native elements** option in the Visualization Manager, a web browser which supports the overlay functionality is used for the online visualization.
