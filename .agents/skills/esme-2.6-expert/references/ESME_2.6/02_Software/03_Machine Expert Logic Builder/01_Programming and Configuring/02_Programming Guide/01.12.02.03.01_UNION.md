# UNION

## Overview

As an extension to the IEC 61131-3 standard, you can declare unions in user-defined types.

The components of a union have the same offset. Therefore, they occupy the same storage location. Thus, assuming a union definition as shown in the following example, an assignment to `utTest.udiValue` also manipulates `utTest.awValue`.

## Example

```
TYPE UT_MyUnion: UNION
    udiValue : UDINT;
    awValue : ARRAY [0..1] OF WORD;
END_UNION
END_TYPE

VAR
    utTest: UT_MyUnion;
END_VAR
```
