# Breakpoint-Related Commands

## Overview

Using breakpoints leads to an interruption of the corresponding running task in which the breakpoint is set.

| WARNING | |
| --- | --- |
|  | INTERRUPTION OF TASK EXECUTION  Use breakpoints only in a virtual or non-production/non-service environment.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |
