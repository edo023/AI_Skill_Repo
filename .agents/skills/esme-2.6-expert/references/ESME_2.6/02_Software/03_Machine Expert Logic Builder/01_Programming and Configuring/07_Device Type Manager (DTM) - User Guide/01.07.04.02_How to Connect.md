# How to Connect

How to Connect

| Step | Action |
| --- | --- |
| 1 | Build and download the application into the controller. |
| 2 | Configure the device IP address or the addressing method (DHCP, BOOTP). |
| 3 | Perform a device power cycle. It is necessary for some devices to take the correct network settings. |
| 4 | If needed you can [edit the connection](../DTMs_in_ESME/DTMs_in_ESME-7.htm#XREF_D_SE_0057935_1) for accessing the device. |
| 5 | Select the required DTM function in the contextual menu of the device (upload from device, download to device,...). Refer to [DTM Contextual Menu](../DTMs_in_ESME/DTMs_in_ESME-6.htm#XREF_D_SE_0008095_1). |

DTM on Modbus TCP and EtherNet/IP are only reachable via Ethernet directly.

NOTE: A transparent USB to Ethernet connection via the controller to the device is not supported.
