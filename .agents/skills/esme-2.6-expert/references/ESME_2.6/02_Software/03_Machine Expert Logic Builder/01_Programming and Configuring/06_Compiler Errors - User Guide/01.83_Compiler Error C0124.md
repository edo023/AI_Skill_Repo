# Compiler Error C0124

## Message

‘Initialization’ is no valid initialization for an enumeration

## Message Cause

A data type that is not ANY\_INT is used for the enumeration initialization.

## Solution

Only use ANY\_INT for enumeration initializations.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : DUT;
END_VAR
```

```
TYPE DUT :
(
enum_member := 1.5
) DWORD;
END_TYPE
```

--> C0032: Cannot convert the type 'LREAL' to type 'DUT'

--> C0124: 'Initialization' is no valid initialization for an enumeration
