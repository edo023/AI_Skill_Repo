# Customize

## Overview

The Tools > Customize dialog box contains tabs for the configuration of the user interface concerning structure and contents of menus and toolbars as well as of the keyboard functions.

You can restore the default settings at any time by clicking the button Reset to default. A dialog box opens allowing you to select the categories that will be restored.

The following tabs are available by default:

* [**Menu**](D-SE-0084067.html#D-SE-0084067)
* [**Toolbars**](D-SE-0084068.html#D-SE-0084068)
* [**Keyboard**](D-SE-0084069.html#D-SE-0084069)
* [**Command Icons**](D-SE-0084070.html#D-SE-0084070)

The current settings confirmed with OK are saved on the local system. You also have the possibility to save and reload specific configuration files as described in the next paragraph.

## How to Save and Reload the Settings Made in the Dialog Box

Click the Save button to store the settings of the currently opened tab in a file. The standard dialog box for saving a file will open, where you have to enter a filename without extension and confirm by clicking Save.

You can do this for each of the tabs and the following files can be stored:

|  |  |
| --- | --- |
| <filename>.opt.menu | for the menu settings |
| <filename>.opt.keyb | for the keyboard settings |
| <filename>.opt.tbar | for the toolbar settings |

To reload the settings from one of those files, click the Load button. A Load Menu dialog box opens. It allows you to browse for the desired menu file, whose settings then will be taken into the respective Customize dialog box.
