# Communication DTM

Communication DTM

Communication DTMs provide communication paths to a device for a certain fieldbus [protocol](../glossary/glossary.htm#XREF_D_SE_0024697_159) (for example [Modbus](../glossary/glossary.htm#XREF_D_SE_0024697_151)).

This table lists the communication DTM in EcoStruxure Machine Expert:

| DTM name | Supported function | Supported fieldbus |
| --- | --- | --- |
| Modbus Serial Line Manager | Establishes a Modbus serial communication for any Modbus serial line device | [Modbus SL](../glossary/glossary.htm#XREF_D_SE_0024697_317) |
