# Identifying a WebVisu

To be able to identify a WebVisu by means of the library POU `VisuFbClientTagDataHelper`, the WebVisu needs a name. To be able to address it directly in the application, you extend the URL call with the parameter `ClientName=<name>`.

Example:

```
http://localhost:8080/webvisu.htm?ClientName=VisClientxy
```
