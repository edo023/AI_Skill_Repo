# Creating a TM5/TM7 Distributed I/O Configuration with DTM

![](images/logo.png)

Creating a TM5/TM7 Distributed I/O Configuration with DTM

Overview

This chapter describes how to create a TM5/TM7 distributed I/O island on the CANopen field bus with DTM.

NOTE: An TM5/TM7 distributed island is the association of a network interface module with expansion modules. It offers a modular and flexible I/O solution.
