# Compiler Warning C0561

## Message

Call recursion: <POU> -> <POU>.

## Message Cause

The program code contains a recursive call.

## Solution

Remove the recursive call if this is unwanted.

## Error Example

```
PROGRAM Recursive
VAR
END_VAR
Recursive();
```

--> C0561: Call recursion: RECURSIVE -> RECURSIVE
