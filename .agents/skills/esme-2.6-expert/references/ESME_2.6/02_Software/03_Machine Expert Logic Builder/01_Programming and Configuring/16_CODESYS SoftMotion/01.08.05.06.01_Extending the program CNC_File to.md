# Extending the program CNC\_File to CNC\_File\_Path3D

1. Open the **Library Manager** and add the `SM3_CNC_Visu` library.
2. Configure other properties of **Path3D**. For example, configure the processed path elements to be displayed in gray:

   **Highlight → Highlight color**: `Gray`
