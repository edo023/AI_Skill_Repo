# Insert Transition / Insert Transition After

## Overview

The Insert Transition / Insert Transition After commands insert a new transition before / after the selected point in the SFC editor.

The default name of the new transition is `Trans<n>`. `n` is an incremental number starting at 0 for the first transition. Click the name to edit it.

By default, the command is not available in the menus. You can add this command via the Tools > Customize [menu](D-SE-0084066.html#D-SE-0084066).
