# Commands of the Messages View

## Messages View Commands

The Edit menu contains commands which serve to navigate between messages in the Messages view or between messages and the concerned position in the project.

## Next Message

Shortcut: F4

Use the Next Message command to set the focus to the next message in the Messages view.

## Previous Message

Shortcut: SHIFT+F4

Use the Previous Message command to set the focus to the previous message in the Messages view.

## Go To Source Position

If the cursor is placed in a line indicating an error or alert message in the Messages view, you can use the Go To Source Position command to find the object that provoked the message.
