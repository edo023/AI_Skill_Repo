# Keyboard

## Overview

The Keyboard tab of the Tools > Customize dialog box allows you to define keyboard functions:

The dialog box shows the available Categories with the assigned Commands.

|  |  |
| --- | --- |
| Shortcuts for selected command: | Displays the shortcut of the selected command. |
| Press shortcut keys: | If the cursor is set to this text field, shortcuts can be recorded. Combinations of Shift, Ctrl, Alt, and other keys are possible. You can assign the recorded shortcut to the selected command by clicking the Assign button. |
| Shortcut currently used by: | Displays the command the entered shortcut is assigned to. |

## Buttons

The following buttons are available:

| Button | Description |
| --- | --- |
| Reset | Click the button to reset the shortcuts to the default settings. |
| Load...  Save... | You can save the settings to an external file (<filename>.opt.keyb) or reload it from previously saved setting. |
