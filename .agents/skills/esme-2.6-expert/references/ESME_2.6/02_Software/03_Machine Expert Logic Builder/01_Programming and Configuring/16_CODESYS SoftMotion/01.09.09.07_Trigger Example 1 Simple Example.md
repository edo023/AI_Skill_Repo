# Trigger Example 1: Simple Example

The `Robotics_Trigger.project` sample project described here is located in the installation directory of CODESYS under `..\CODESYS SoftMotion\Examples`.

Triggers are used to find out exactly when an axis group reaches a specific point on the commanded path. This information can be used, for example, to switch a tool on or off at exactly the right time. This example shows how a single trigger is commanded on a movement and then read out cyclically. Based on this, the [Trigger Example 2: Gluing Process](_sm_example_digital_switches2.html#_sm_example_digital_switches2) project shows a realistic sample application.
