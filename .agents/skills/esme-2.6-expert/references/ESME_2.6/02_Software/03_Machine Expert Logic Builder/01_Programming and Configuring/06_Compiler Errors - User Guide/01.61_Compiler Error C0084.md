# Compiler Error C0084

## Message

‘<pragma operand>’ is no valid operand for a pragma

## Message Cause

Syntax error

## Solution

Use valid pragma operands.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
{IF defined(0)}
{END_IF}
```

--> C0084: 'defined (null)' is no valid operand for a pragma

## Error Correction

```
{IF defined (abc)}
```
