# Dialog: Create HMI Project: Connect to HMI Device

**Function**: The dialog is part of the wizard that opens when you create a new project with the **HMI project** template. This allows for optional setting of the communication parameters for the HMI device.

**Call**: Automatically after the **Create HMI Project – Choose HMI Device** dialog

|  |  |
| --- | --- |
| **Gateway** | List of known gateways that are active in the network |
| **Device** | Network name of the device to establish a connection |
| **Select Path by Network Scan** | By clicking this button, you start the network scan and a list of all active HMI devices appears in the **Select Device** dialog. |
| **Active communication path** | Network name and address of the selected device |
| **Next** | The **Create HMI Project – Add Objects** dialog opens. |
| **Abort** | The configuration and settings of an HMI project are ended. |
