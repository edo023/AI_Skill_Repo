# Compiler Error C0035

## Message

Program name, function or function block instance expected instead of ‘<invalid function>’

## Message Cause

A function is called that does not exist.

## Solution

Make sure that only existing program names, functions and function block instances are called.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
PLC_PRG.METH1();
```

```
METHOD METH
VAR_INPUT
END_VAR
```

--> C0004: 'METH1' is no component of 'PLC\_PRG'

--> C0035: Program name, function or function block instance expected instead of 'PLC\_PRG.METH1'

## Error Correction

```
PLC_PRG.METH();
```
