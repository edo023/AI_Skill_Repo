# Compatibility of Lexium 62 Servo Drives and Programming Software Versions

## Overview

Lexium 62 devices are available for use with SoMachine Motion or EcoStruxure Machine Expert.

Some of the new functions of Lexium 62 servo drives are only available in EcoStruxure Machine Expert as they require a hardware revision and the related product firmware version.

The compatibility table indicates the relation of hardware and programming software versions.
