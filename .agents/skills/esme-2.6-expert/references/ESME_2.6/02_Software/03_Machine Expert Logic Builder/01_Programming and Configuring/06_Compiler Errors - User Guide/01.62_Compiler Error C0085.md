# Compiler Error C0085

## Message

Define value expected instead of ‘<value>’.

## Message Cause

A string is expected instead of the value at the displayed location of the pragma.

## Solution

Replace the value with a string.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
{IF hasvalue(define, defineValue)}
{END_IF}
```

-->C0085: Define value expected instead of ‘defineValue’

## Error Correction

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
{IF hasvalue(define, '120')}
{END_IF}
```

EIO0000003933.04
