# Connect to Device / Disconnect from Device

## Connect to Device

The Online > Connect to Device command can be used to set up a connection to the device which is currently selected in the Devices tree. As a precondition, the [communication settings](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083386) of the device must be configured properly.

## Disconnect from Device

The Online > Disconnect from Device command can be used to disconnect from the device which is currently connected and selected in the Devices tree.
