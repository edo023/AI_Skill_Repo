# Compiler Warning C0033

## Message

Type ‘<pointer type>’ possibly not convertible to type ‘<data type>’.

## Message Cause

This message occurs only when using Check All Pool Objects. An attempt was made to convert a pointer to an integer. As the size of pointers in a library is not defined, errors may be detected when using the library.

## Solution

Use the type \_\_UXINT or \_\_XWORD for platform-independent calculations with pointers.

## Error Example

```
PROGRAM PLC_PRG
VAR
  ptr: POINTER TO INT;
  dw: DWORD;
END_VAR
```

```
dw := ptr;
```

-->C0033: Type ‘POINTER TO INT’ possibly not convertible to type ‘DWORD’.
