# Compiler Error C0101

## Message

Data recursion: ‘<recursion>’

## Message Cause

Recursive data initialization over two function blocks.

## Solution

Avoid recursions for data initializations.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst0 : FB1;
END_VAR
```

```
FUNCTION_BLOCK FB1
VAR
  inst1 : FB2;
END_VAR
```

```
FUNCTION_BLOCK FB2
VAR
  inst2 : FB1;
END_VAR
```

--> C0101: Data recursion: FB1->FB2->FB1

EIO0000003933.04
