# Style Configuration for Libraries and Global Visualizations

|  |  |
| --- | --- |
| **Use no visualization style** | : Visualizations in the **POUs** view or from libraries behave as if no style had been selected. |
| **Use the following visualization style** | : Visualizations in the **POUs** view or from libraries behave as if the selected style were active. |
| **Derive visualization style automatically** | : Visualizations in the **POUs** view or from libraries derive the style configuration from the Visualization Managers of the current project. If this is not possible or the result is unclear, then the style selected below is used as a fallback solution. |
| **Fallback if no visualization style could be derived** | Style from list box  Fallback solution in the case that no visualization style can be derived  Example: On the display variant, the automatically derived style results in a poor display due to the device. |
