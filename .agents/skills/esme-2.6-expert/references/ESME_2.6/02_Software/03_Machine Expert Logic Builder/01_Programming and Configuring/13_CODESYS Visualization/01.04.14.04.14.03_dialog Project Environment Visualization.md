# Dialog: Project Environment: Visualization Symbols

**Function**: The dialog lists the installed symbol libraries and allows you to assign symbol libraries to a project.

**Call**: **Project → Project Environment** menu, **Visualization Symbols** tab

**Requirement**: The open project contains a visualization and has been saved with a compiler version less than `3.5.7.0`. Symbol libraries are known in compiler version `3.5.7.0` and higher.

|  |  |
| --- | --- |
| **Symbol library** | List of all installed symbol libraries |
| **Active** | : The symbol library is selected for the project. As a result, their symbols are available in the **Visualization ToolBox** view.  : Symbol library is currently installed only in the library repository. |
