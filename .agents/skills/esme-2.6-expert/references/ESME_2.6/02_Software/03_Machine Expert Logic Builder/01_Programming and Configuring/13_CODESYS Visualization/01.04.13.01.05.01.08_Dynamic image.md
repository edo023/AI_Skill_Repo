# Dynamic image

You can use this element property for animating a series of image files.

|  |  |
| --- | --- |
| **Bitmap version** | Variable (integer data type) which includes the version of the image  If the variable changes, then the visualization re-reads the image referenced in the **Image ID** property and displays it.  The visualization displays animations when the image file on the controller is updated continuously, thus incrementing the version variable. The application must be programmed for this.  **Possible applications**   * Displaying graphics that are generated by the application * Displaying images that are refreshed by a camera |
