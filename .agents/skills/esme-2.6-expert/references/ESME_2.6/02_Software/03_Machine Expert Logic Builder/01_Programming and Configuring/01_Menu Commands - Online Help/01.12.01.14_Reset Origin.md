# Reset Origin

## Overview

The Online > Reset origin command is available in online mode.

It initiates a reset origin of the active application on the controller which has the following effects:

* The application code is deleted, and the [application](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083435) has no state.
* Variables are deleted, the values are no longer available.
* [Retain variables](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083608) are deleted, the values are no longer available.
* [Persistent variables](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083608) are deleted, the values are no longer available.
* [Breakpoints](D-SE-0083918.html#D-SE-0083918) that were set in the application are no longer available.

Also refer to [**Reset warm**](D-SE-0084003.html#D-SE-0084003), [**Reset cold**](D-SE-0084002.html#D-SE-0084002).
