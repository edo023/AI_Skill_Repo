# Radius setting

Visible only when **Rounded Rectangle** is selected in the **Type of element** property.

|  |  |
| --- | --- |
| **Radius** | Rounding of the corners  **From style**  **Relative to the element size**  **Explicit**: Allows for specifying a custom value in the **Value** setting |
| **Value** | Radius of the rounded corners (in pixels)  Example: `5`  Requirement: **Explicit** is selected in the **Radius** setting. |
