# Radio button settings

|  |  |
| --- | --- |
| Button '**Create new**' | Clicking this button creates a new selection button in the editor and lists an additional area in the properties editor. |
| **Radio button**   * **Areas**    + **[<n>]** | For each radio button, an area is visible that records the settings.   * [<n>]    + **[<n>]**: The number indicates the area. Clicking **Delete** will delete the associated radio button with its settings **Text**, **Tooltip**, and **Line spacing (in pixels)**. |
| **Areas: [<n>]** |  |
| **Text** | Name on the label of the radio button |
| **Tooltip** | Text to be displayed in a tooltip |
| **Line spacing (in pixels)** | Distance to the top button (in pixels) |
