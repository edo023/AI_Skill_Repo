# Configuring the file transfer mode

Usually, when the application is downloaded, files which the visualization requires for display are transferred to the respective visualization device. These are mainly image files or text list files.

Alternatively, you could also configure the visualization to access local files. No files will then be transferred during a download.

**To allow the visualization to access the local files, the following configuration is required:**

* The file paths for image files or text list files are relative.
* The link type for image files is **Link to File**.
