# Convert to Classic Format... (exclusive to \*.fbs projects)

## Overview

This function is exclusive to projects stored in file-based storage format.

The File > Convert to Classic Format... command converts a file-based storage \*.fbs project to a classic \*.project or \*.library file in classic file storage format. It opens a Save As dialog box that allows you to browse to a storage folder and to enter a file name.
