# IScriptEioObjectExtender

class

IScriptEioObjectExtender

Bases: object

Edge I/O NTS script extender for all IScriptObject

is\_eio\_device

Returns True if current object is inside an Edge I/O NTS project

is\_eio\_compatible\_scanner

Returns True if current object is a compatible I/O scanner to use Edge I/O NTS devices

is\_eio\_nim

Returns True if current object is an Edge I/O NTS NIM

is\_eio\_module

Returns True if current object is an Edge I/O NTS module (not a module receiver or a NIM)

is\_eio\_module\_receiver

Returns True if current object is an Edge I/O NTS module receiver

is\_eio\_io\_link\_master

Returns True if current object is an Edge I/O NTS IO-Link master module

is\_eio\_io\_link\_slot

Returns True if current object is an Edge I/O NTS IO-Link slave slot (can be empty)

is\_eio\_io\_link\_slave

Returns True if current object is an Edge I/O NTS IO-Link slave (False if slot is empty)

is\_eio\_serial\_master

Returns True if current object is an Edge I/O NTS serial line master

is\_eio\_serial\_slave

Returns True if current object is an Edge I/O NTS serial line slave

has\_eio\_expert\_function\_support

Returns True if current object use Edge I/O NTS expert function system

is\_eio\_expert\_function\_sub\_module

Returns True if current object is an Edge I/O expert function sub module

is\_eio\_power\_supply

Returns True if current object is an Edge I/O power supply
