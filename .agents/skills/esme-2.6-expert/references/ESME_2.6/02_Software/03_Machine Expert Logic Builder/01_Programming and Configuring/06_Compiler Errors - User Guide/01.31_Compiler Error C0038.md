# Compiler Error C0038

## Message

‘<invalid output>’ is no output of ‘<function name>’

## Message Cause

A local variable is handled as an output in a function call.

## Solution

Declare the variable as an output parameter.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  x : INT;
END_VAR
```

```
i := TEST(iVar => x);
```

```
FUNCTION TEST : INT
VAR
  iVar : INT;
END_VAR
```

--> C0038: 'iVar' is no output of 'TEST'

## Error Correction

```
VAR_OUTPUT
  iVar : INT;
END_VAR
```
