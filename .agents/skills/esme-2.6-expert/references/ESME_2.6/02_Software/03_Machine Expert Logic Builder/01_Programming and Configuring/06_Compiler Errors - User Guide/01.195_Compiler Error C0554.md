# Compiler Error C0554

## Message

No explicit calls of '<FB name>’ allowed. <value of attribute 'no\_explicit\_call'>

## Message Cause

A function block with the `no_explicit_call` attribute has been called.

## Solution

Do not call this function block with the `no_explicit_call` attribute.

## Error Example

```
{attribute 'no_explicit_call' := 'Call of this FB is not allowed'}
FUNCTION_BLOCK FB1
VAR_INPUT
END_VAR
VAR_OUTPUT
END_VAR
VAR
END_VAR
```

```
PROGRAM PLC_PRG
VAR
    myFB: FB1;
END_VAR

myFB();
```

--> C0554: No explicit calls of 'FB1' allowed. Call of this FB is not allowed.

## Error Correction

Delete the attribute pragma `no_explicit_call`.
