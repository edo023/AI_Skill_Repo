# Compiler Warning C0316

## Message

Method <method name> is already called implicitly

## Message Cause

The `FB_Exit` method of the base function block is called in the `FB_Exit` of a derived block.

## Solution

Remove the call in the `FB_Exit` method.

## Error Example

```
FUNCTION_BLOCK Base_FB...
METHOD FB_Exit: BOOL
VAR_INPUT
      bInCopyCode : BOOL;
END_VAR

FUNCTION_BLOCK FB EXTENDS Base_FB
...
METHOD FB_Exit : BOOL
VAR_INPUT
      bInCopyCode : BOOL;
END_VAR
SUPER^.FB_Exit(bInCopyCode);
```

--> C0316: Method 'FB\_Exit' is already called implicitly

EIO0000003933.04
