# Add Folder...

## Overview

The Project > Add Folder... command will be available if you are working in the navigators.

If you want to add a folder on the uppermost level in a tree, make sure that the window is active, but no existing object or folder is selected. If you want to add a folder on a lower level, select the entry in the tree below where the folder should be inserted.

Then execute the Add Folder... command to open the Add Folder dialog box. Define the name of the new folder. The name may contain spaces, digits, and special characters.

After confirming by clicking OK, the folder will be inserted in the tree preceded by the folder icon.

If the folder has been created below an existing tree entry, click the plus-sign which will now display before the entry to make the new folder visible.
