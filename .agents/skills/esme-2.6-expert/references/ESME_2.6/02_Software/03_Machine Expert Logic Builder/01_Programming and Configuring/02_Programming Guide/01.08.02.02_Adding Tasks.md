# Adding Tasks

## Introduction

You can add tasks to your application via the Applications tree.

## Procedure

| Step | Action |
| --- | --- |
| 1 | In the Applications tree, select the Task Configuration node, click the green plus button, and execute the command Task....  Alternatively, you can right-click the **Task Configuration** node, and select Add Object > Task... from the contextual menu.  **Result:** The **Add Task** dialog box opens. |
| 2 | In the **Add Task** dialog box, enter a name in the Name: text box.  **Note:** The name must neither contain any space nor exceed a length of 32 characters. |
| 3 | Click **Add**. |
