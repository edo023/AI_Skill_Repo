# Compiler Error C0098

## Message

The keyword “FUNCTIONBLOCK” is no longer supported. Use “FUNCTION\_BLOCK” instead.

## Message Cause

Syntax error

## Solution

Use the keyword “FUNCTION\_BLOCK”.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : FB;
END_VAR
```

```
FUNCTIONBLOCK FB
VAR
END_VAR
```
