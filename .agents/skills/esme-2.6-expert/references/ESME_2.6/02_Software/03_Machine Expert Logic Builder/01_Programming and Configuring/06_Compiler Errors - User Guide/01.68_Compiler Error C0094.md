# Compiler Error C0094

## Message

Interface of the overridden method ‘<method name>’ of the base ‘<function block name>’ does not match the declaration

## Message Cause

The signature of the method of the first interface does not match the signature of the method in the second interface, which is extended by the first.

## Solution

Align the signatures.

## Error Example

```
PROGRAM PLC_PRG
VAR
  xyz : XY2;
END_VAR
```

```
FUNCTION_BLOCK XY
METHOD METH1
VAR_INPUT
END_VAR
```

```
FUNCTION_BLOCK XY2 EXTENDS XY
METHOD METH1
VAR_INPUT
  iPar : BOOL;
END_VAR
```

--> C0094: Interface of the overridden method METH1 of the base XY does not match the declaration
