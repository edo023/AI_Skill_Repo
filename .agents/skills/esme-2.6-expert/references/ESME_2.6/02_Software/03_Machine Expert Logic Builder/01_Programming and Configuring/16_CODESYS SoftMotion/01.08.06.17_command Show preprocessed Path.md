# Command: Show preprocessed Path

Symbol: ![](images/uuid-84fa5ae9-828e-126a-fe40-5b4411.png)

**Function**: The command activates and deactivates the display of the CNC path with preprocessing.

**Call**: **CNC** menu

**Requirement**: A CNC path is open in the editor.

If the display is active, then the path is displayed with preprocessing and the original path is displayed in light gray in the background.

The preprocessing is configured in the **CNC settings** dialog (**[Path preprocessing](_sm_obj_cnc_settings.html#_sm_obj_cnc_settings)** tab). The preprocessing configured there is displayed. Then you can simulate the effect of the different function blocks (in series) on the preprocessing.

TIP:

The command does not have any effect on code generation.

Even if an SMC\_OutQueue file structure is generated when compiling the CNC program, the command does not have any effect on the contents of the file. The preprocessing is always taken into consideration when compiling.
