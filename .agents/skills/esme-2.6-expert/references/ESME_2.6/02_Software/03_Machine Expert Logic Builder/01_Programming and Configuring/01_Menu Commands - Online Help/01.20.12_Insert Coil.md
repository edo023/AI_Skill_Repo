# Insert Coil

## Overview

|  |  |
| --- | --- |
| Symbol | In the editor: |
| Shortcut | Ctrl + A |
| Call | * Ladder > Insert Coil menu * Contextual menu |

## Function

The command inserts a coil in the network. Possible insertion positions are indicated by the plus symbol at the cursor when you drag the element from the Toolbox to the implementation part.

## Requirements

A network, a coil, or a connector is selected. No block is selected.

## Element: Coil

A coil applies the value supplied from the left and saves it in the Boolean variable assigned to the coil. The input can have the value TRUE (ON) or FALSE (OFF).

In a negated coil ![](images/DD00896889.png), the negated value of the incoming signal is stored in the Boolean variable that is assigned to the coil.

You can use the [Set/Reset  commands](SetReset-4520A86A.html) to define a `set coil` or `reset coil`.

* `set coil`: When the value TRUE is supplied to a set coil, the coil retains the value TRUE. As long as the application is running, the value cannot be overwritten.
* `reset coil`: When the value TRUE is supplied to a reset coil, the coil retains the value FALSE. As long as the application is running, the value cannot be overwritten.
