# Compiler Error C0195

## Message

Implicit conversion from a signed type ‘<data type 1>’ to an unsigned type ‘<data type 2>’ : possible change of sign

## Message Cause

A sign conflict may have been missed in the implicit conversion.

## Solution

Implicitly convert data types with the same sign.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  b : UINT;
END_VAR
```

```
b := i;
```

--> C0195: Implicit conversion from a signed type 'INT' to an unsigned type 'UINT' : possible change of sign
