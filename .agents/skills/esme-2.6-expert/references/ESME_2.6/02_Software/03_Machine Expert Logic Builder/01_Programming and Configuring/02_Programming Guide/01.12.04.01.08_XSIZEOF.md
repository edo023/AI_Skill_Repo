# `XSIZEOF`

## Overview

This arithmetic operator is not specified by the standard IEC 61131-3.

You can use it to determine the number of bytes required by the passed variable or data type.

The `XSIZEOF` operator returns an unsigned value.

The data type of the return value `<return value>` is:

* ULINT on 64-bit platforms
* UDINT on platforms other than 64-bit

To generate code that runs on all platforms, the return value can be declared with the \_\_UXINT data type.

## Syntax

```
<return value> := XSIZEOF( <variable> );
```

## Example in ST

```
PROGRAM PLC_PRG
VAR
    uxiReturnValue : __UXINT;  (* data type for 64-bit platforms: ULINT *)
    aData_1 : ARRAY[0..4] OF INT; 
END_VAR
```

```
uxiReturnValue := XSIZEOF(aData_1);
```

Result:

`uxiReturnValue = 10`

NOTE: In the case of assignment to a variable of type \_\_UXINT, use the `XSIZEOF` operator instead of the `SIZEOF` operator because with `XSIZEOF`, the data type of the return value depends on the platform.
