# Command: Insert Elements for Trend Controlling

Symbol: ![](images/uuid-b946b76b-10cf-309c-cd33-bf3e63.png)

**Function**: When you execute this command in **Visualization**, the **Trend Wizard** dialog opens.

**Call**: **Visualization** menu; context menu of a **Trend** element in the visualization editor

**Requirement**: A trend is selected in the active visualization editor.
