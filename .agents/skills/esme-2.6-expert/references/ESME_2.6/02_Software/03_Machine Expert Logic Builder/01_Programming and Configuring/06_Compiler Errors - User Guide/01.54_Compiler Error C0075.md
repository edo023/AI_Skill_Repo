# Compiler Error C0075

## Message

Too many initializers for the array

## Message Cause

An excessive number of values are specified for the size of the array.

## Solution

The number of assigned values must correspond to the size of the array.

## Error Example

```
PROGRAM PLC_PRG
VAR
  arr1 : ARRAY [1..5] OF INT := [1,2,3,4,5,6];
END_VAR
```

--> C0075: Unexpected array initialization

## Error Correction

```
arr1 : ARRAY [1..6] OF INT := [1,2,3,4,5,6];
```
