# Command: Revert, Revert project

Symbol: ![](images/uuid-4561071c-ed48-4195-555c-ccc146.png)

**Function**: The command opens the **Revert** dialog. In the dialog, select the objects whose local changes should be reverted, and those that are reverted to the state of the base revision of the working copy.

**Call**:

* **Project → SVN** menu
* **Context menu → SVN**

If you select nothing or the main node in the device tree, then all modified objects are listed in this dialog (**Revert project**). If you selected one or more objects, then only the changes to this object are listed and recursively their sub-objects (**Revert**).
