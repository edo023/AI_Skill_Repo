# Set Execution Order

## Overview

The CFC > Execution Order > Set Execution Order command serves to redefine the element number of the currently selected element in order to change the position of this element within the [execution order](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083494).

The command opens the Set Execution Order dialog box. The current element number is displayed in the Current Execution Order field. You can enter the desired new element number in the New Execution Order text field. The possible values are displayed in brackets.
