# Adding Pins

## Add Input Pin

The CFC > Pins > Add Input Pin command is available in the contextual menu when an appropriate box is selected in the CFC editor. It adds an input pin to the selected box.

## Add Output Pin

The CFC > Pins > Add Output Pin command is available in the contextual menu when an appropriate box is selected in the CFC editor. It adds an output pin to the selected box.
