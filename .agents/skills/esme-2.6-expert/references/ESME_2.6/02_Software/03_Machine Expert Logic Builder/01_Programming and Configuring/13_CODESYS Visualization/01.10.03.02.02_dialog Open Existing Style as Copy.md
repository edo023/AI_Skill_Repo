# Dialog: Open Existing Style as Copy

**Function**: The dialog prompts you to specify data for copying a style file.

****Call**:**

* In CODESYS:

  In the editor of the **Visualization Manager** object, on the **Settings** tab, in the **Style Settings** group, click ![](images/uuid-385e07a3-6c77-2d0b-5e6e-a3d39d.png) for a list box). Click **Copy and Edit Style**.
* Visualization style editor:

  Menu: **File → Open as Copy**

|  |  |
| --- | --- |
| **Style** | Style to be copied  Example: `Basic Style, 3.5.9.0`  NOTE: You can also select a style from the repository. |
| **Destination** | Working directory for style editing |
| **OK** | A copy of the style is created and opened for editing in the Visualization Style Editor. |
