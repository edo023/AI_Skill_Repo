# Command: Make Horizontal Spacing Equal

Symbol: ![](images/uuid-3363fb7f-8a5a-8185-015a-88dc46.png)

**Function**: The command aligns the selected visualization elements so that the leftmost and rightmost elements keep their positions and the elements in between are positioned horizontally with equal spacing.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Three or more elements are selected. The first element is blue and the other elements are gray.
