# Compiler Warning C0514

## Message

Should not access internal property <POU name>.<property name> of library <library name>.

## Message Cause

A library contains a POU with a property, whose visibility is restricted to this library with the `INTERNAL` keyword. The property is accessed from outside of this library.

## Solution

Modify the visibility of the property (for example, remove the `INTERNAL` keyword) or modify the declaration of the variable.

## Error Example

```
PROGRAM PLC_PRG
VAR
      p1: MyLib.POU;
      somevar1 :INT := p1.Prop;
END_VAR
```

--> C0514: Should not access internal property POU.Prop of library MyLib, 1.0.0.0 (MyCompany).
