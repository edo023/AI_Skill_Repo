# Compiler Error C0118

## Message

The label ‘<jump label>’ has not been referenced

## Message Cause

A jump label is defined that is not referenced.

## Solution

Remove the unused jump labels.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
LABEL:
```

--> C0118: The label ' LABEL' has not been referenced
