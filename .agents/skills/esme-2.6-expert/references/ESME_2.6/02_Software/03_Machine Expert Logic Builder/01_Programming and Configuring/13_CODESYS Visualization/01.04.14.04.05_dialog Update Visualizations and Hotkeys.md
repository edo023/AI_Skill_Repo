# Dialog: Update Visualizations and Hotkeys

****Function**:**

**The dialog supports you in updating the configured permissions application-wide in the following places:**

* Per element in its visualization element **Access rights** property
* Per element in its visualization element **Input configuration** property in **Hotkeys**
* Per default hotkey, configured in the visualization manager in the [Tab: Default Hotkeys](_visu_obj_manager_hotkeys_configuration.html#_visu_obj_manager_hotkeys_configuration)

**This is helpful if the user management has been changed as follows:**

* Adding of new groups
* Renaming of groups
* Deletion of existing groups which are no longer used

**Call**: **Visualization Manager**, on **User Management** tab, on **Groups** tab via **Activate Visualizations/Hotkeys** button
