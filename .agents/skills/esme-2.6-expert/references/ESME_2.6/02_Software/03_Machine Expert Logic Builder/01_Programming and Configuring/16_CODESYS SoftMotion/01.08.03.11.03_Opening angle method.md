# Opening angle method

If you define the circle by the opening angle T, then a maximum of three full circles are permitted (T ≤ 1080).

The opening angle method is appropriate for an arc > 180°.

**Examples**

**Same semicircle as in the radius method; specified by means of opening angle**

```
N10 G1 X100 Y100
N15 G99
N20 G2 I50 J0 T180
```

**Helix (two full circles)**

```
N10 G1 X100 Y100
N15 G99
N20 G2 I50 J0 T720
```
