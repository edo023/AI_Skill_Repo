# Perspectives

## Overview

The Tools > Options > Perspectives dialog box allows you to manage perspectives. A perspective is used to save the layout of EcoStruxure Machine Expert views. It stores whether the Messages and Watch views are open and at which position the view windows are located (docked or independent windows). For further information, refer to the [description of perspectives](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083359).

The Perspectives dialog box consists of two tabs. The Perspectives tab provides a list of available perspectives. The active perspective can be recognized by a check mark in the Active column.

Elements of the Perspectives tab:

| Element | Description |
| --- | --- |
| Save current configuration as Perspective | Click this button to save the configuration of EcoStruxure Machine Expert as a perspective.  In the Save current view layout as perspective dialog box, enter a new Perspective Name to save it as a new perspective.  To overwrite the selected perspective, leave the Perspective Name unchanged, and click Ok. If the option Ask for confirmation before overwriting existing perspectives. is selected, a separate dialog box will be displayed, requesting you to confirm that the present perspective is overwritten. |
| Delete | Click the Delete button to delete the perspective that is selected in the list. |
| Rename | Click the Rename button to open the Rename Perspective dialog box that allows you to assign a new name to the perspective that is selected in the list. |
| Ask for confirmation before overwriting existing perspectives. | If this option is selected, a separate dialog box will be displayed asking you to confirm whether an existing perspective is to be overwritten. |
| Import | Click the Import button to open a browse dialog box. It allows you to browse for an XML file of a perspective you had exported from EcoStruxure Machine Expert before. |
| Export | Click the Export button to open a Save As dialog box. It allows you to save the selected perspective as an XML file. |
| Reset to Factory defaults | Click the Reset to Factory defaults button to restore the [default perspectives](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083359). |

Elements of the Behavior tab:

| Element | Description |
| --- | --- |
| Editor behavior when switching between Online and Offline Mode | |
| Codesys Default | Open editors remain open when you switch between online and offline mode. |
| Keep all Editors (incl. POU) open when going Offline | Open editors remain open when you switch to offline mode. |
| Separate Online / Offline View | Individual layouts with different open editors are managed for online and offline view. |
| Collapse all elements on opening configuration editor in online mode | When you open the configuration editor in online mode, all elements are collapsed. |
