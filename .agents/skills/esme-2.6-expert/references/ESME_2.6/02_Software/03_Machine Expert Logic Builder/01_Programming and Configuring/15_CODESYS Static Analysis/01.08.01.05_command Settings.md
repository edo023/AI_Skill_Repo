# Command: Settings

**Function**: The command opens the **[Static Analysis Settings](_san_dlg_settings_sa_naming_conventions.html#_san_dlg_settings_sa_naming_conventions)** dialog.

**Call**: **Build → Static Analysis** menu

**Requirement**:

* The package CODESYS Static Analysis is installed.
* A project is open.
