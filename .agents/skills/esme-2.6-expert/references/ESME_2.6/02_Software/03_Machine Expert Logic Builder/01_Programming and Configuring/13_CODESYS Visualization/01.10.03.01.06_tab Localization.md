# Tab: Localization

The tab makes it possible to translate the names of the style properties into other languages.

|  |  |
| --- | --- |
| **Name** | Lists the name of the style properties as they are defined on the **Style Properties** tab |
| **<language>** | Identification of the language name (as language code according to ISO 639-1) in which the style property name should be translated |
