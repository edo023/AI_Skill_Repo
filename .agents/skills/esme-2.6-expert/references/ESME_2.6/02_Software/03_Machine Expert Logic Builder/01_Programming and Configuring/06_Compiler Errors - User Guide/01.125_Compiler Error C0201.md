# Compiler Error C0201

## Message

Type ‘<data type 1>’ is not equal to type ‘<data type 2>’ of the VAR\_IN\_OUT ‘Variable’

## Message Cause

The data type that is passed to the function as a VAR\_IN\_OUT parameter does not match the data type defined in it.

## Solution

Pass a variable with the correct data type.

## Error Example

```
PROGRAM PLC_PRG
VAR
  Inst: POU;
  b : BOOL;
END_VAR
```

```
inst(in_out := b);
```

```
FUNCTION_BLOCK POU
VAR_IN_OUT
  in_out : INT;
END_VAR
```

--> C0201: Type 'BOOL' is not equal to type 'INT' of the VAR\_IN\_OUT 'in\_out'

EIO0000003933.04
