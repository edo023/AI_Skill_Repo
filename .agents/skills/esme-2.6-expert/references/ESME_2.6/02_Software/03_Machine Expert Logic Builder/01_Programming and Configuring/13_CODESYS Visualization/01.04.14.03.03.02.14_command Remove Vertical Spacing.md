# Command: Remove Vertical Spacing

Symbol: ![](images/uuid-8f1e1f63-2ac4-2a33-6e18-878520.png)

**Function**: The command aligns the selected visualization elements so that the blue element keeps its position and the other elements are positioned vertically without space between them.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Multiple elements are selected.
