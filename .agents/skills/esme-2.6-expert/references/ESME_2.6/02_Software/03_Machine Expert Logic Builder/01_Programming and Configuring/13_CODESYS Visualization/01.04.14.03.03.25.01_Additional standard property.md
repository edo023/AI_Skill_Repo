# Additional standard property

|  |  |
| --- | --- |
| **Forward inputs** | A frame element which references a visualization template forwards input events to the referenced visualization. Even if the frame has already processed the input.  Enter and Leave works for an element of the template visualization. For example, hovering can be implemented.  Requirement: This is implemented only for visualization templates with frame configuration (hierarchical editor). |
