# Compiler Error C0022

## Message

‘<operator>’ needs exactly ‘<number of operands>’ operands

## Message Cause

An insufficient or an excessive number of operands are assigned to an operator.

## Solution

Assign the required number of operands to the operator.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  pt: POINTER TO INT;
END_VAR
```

```
pt := ADR(i,1);
```

--> C0022: 'ADR' needs exactly '1' operands

## Error Correction

```
pt := ADR(i);
```
