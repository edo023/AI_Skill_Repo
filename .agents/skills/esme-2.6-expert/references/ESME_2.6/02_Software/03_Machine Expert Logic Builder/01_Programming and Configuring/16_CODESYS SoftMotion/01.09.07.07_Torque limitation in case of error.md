# Torque limitation in case of error

If an error is detected during the planning (for example, that a commanded movement will violate the workspace), then the axis group stops on the previously commanded path. In doing so, the torque limits are adhered to. If it is not possible to stop on the path (for example, because a drive reports an error), then also the torque limits cannot be adhered to.
