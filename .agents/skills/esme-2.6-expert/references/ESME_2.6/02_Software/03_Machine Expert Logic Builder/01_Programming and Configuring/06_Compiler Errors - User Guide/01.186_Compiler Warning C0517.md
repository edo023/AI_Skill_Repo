# Compiler Warning C0517

## Message

Should not access internal object <object name> of library <library name>.

## Message Cause

A library contains a POU whose visibility is restricted to this library with the `INTERNAL` keyword. For example, the POU in a `SIZEOF` operator is used from outside of this library.

## Solution

Modify the visibility of the POU (for example, remove the `INTERNAL` keyword) or modify the declaration of the variable.

## Error Example

```
PROGRAM PLC_PRG
VAR
    somevar1 : INT := SIZEOF(MyLib.POU);
END_VAR
```

--> C0517: Should not access internal object POU of library MyLib, 1.0.0.0 (MyCompany).
