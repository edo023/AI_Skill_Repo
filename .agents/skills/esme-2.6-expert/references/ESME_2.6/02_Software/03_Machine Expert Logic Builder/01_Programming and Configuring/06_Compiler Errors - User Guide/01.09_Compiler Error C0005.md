# Compiler Error C0005

## Message

Constant overflow in address ‘<address>’

## Message Cause

At least one component in the address does not fit into a 32-bit integer value.

## Solution

Use a valid address expression.

## Error Example

```
PROGRAM PLC_PRG
VAR
  X: BYTE;
END_VAR
```

```
X := %QB5555555555;
```
