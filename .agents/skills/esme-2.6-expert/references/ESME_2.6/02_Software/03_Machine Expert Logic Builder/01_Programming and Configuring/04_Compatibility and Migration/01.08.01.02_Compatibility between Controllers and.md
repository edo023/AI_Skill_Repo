# Compatibility between Controllers and Modicon Edge I/O NTS Devices

## Introduction

This section provides the compatibilities between the different devices:

* Controllers
* Modicon Edge I/O NTS Network Interface Modules
* Modicon Edge I/O NTS Modules
* Modicon Edge I/O NTS Field Device Master
* Modicon Edge I/O NTS Power Supplies

For more information regarding the supported configuration, refer to the Release Notes of your controller.
