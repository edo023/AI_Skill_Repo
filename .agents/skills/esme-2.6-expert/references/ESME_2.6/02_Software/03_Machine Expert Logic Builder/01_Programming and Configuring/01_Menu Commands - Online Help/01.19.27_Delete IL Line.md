# Delete IL Line

## Overview

Shortcut: CRTL + DEL

The FBD/LD/IL > Delete IL Line command is only available in the IL editor. It is used to delete the line where the cursor is currently positioned.

NOTE: Concerning the view options for the components of FBD, LD and IL networks, consider the FBD, LD and IL editor options.
