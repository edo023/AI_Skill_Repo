# Notes

* Modulo axes are also supported as additional axes. They are always operated in `shortest` mode for absolute additional axis movements.

  For more information, see the following: MC\_DirectionMC\_Direction
* The maximum number of additional axes is limited by the library parameter `MAX_NUM_ADDITIONAL_AXES` from the `SM3_CPKernelDefaults` library. This parameter can be changed in the Library Manager.
* For an example on how to use additional axes, see the following: [Using Additional Axes in Robotics](_sm_example_additional_axes.html#_sm_example_additional_axes)
