# Compiler Error C0238

## Message

No initial value allowed for VAR\_EXTERNAL <variable name>

## Message Cause

An attempt is made to initialize a variable in VAR\_EXTERNAL.

## Solution

Do not initialize variables in VAR\_EXTERNAL.

## Error Example

```
PROGRAM PLC_PRG
VAR_EXTERNAL
  ig : INT := 2;
END_VAR
```

```
VAR_GLOBAL
  ig : INT;
END_VAR
```

--> C0238: No initial value allowed for VAR\_EXTERNAL ig
