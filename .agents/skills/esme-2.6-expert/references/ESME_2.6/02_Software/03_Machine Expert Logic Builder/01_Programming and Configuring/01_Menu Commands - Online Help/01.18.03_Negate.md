# Negate

## Overview

The CFC > Negate command is used to negate inputs, outputs, jumps, or RETURN commands. The symbol for the negation is a small circle on the connection. To assign a negation, select the respective input or output pins of the element and execute the command. Refer to Cursor Positions in CFC for the possible [cursor positions](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083492) for selection.

A negation can be deleted by negating again.
