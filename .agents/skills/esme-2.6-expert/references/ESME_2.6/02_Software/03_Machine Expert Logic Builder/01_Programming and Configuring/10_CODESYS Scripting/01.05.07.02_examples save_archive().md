# Examples: `save_archive()`

Various examples of `save_archive()` with the parameter `additional_categories` are presented below:

**Default behavior**

If `additional_categories` are not explicitly set, then the default categories of the CODESYS interface are also archived.

**No categories**: `additional_categories=None`

If `additional_categories` is explicitly set to `None` in the script, then no categories will be archived.

**Multiple categories**

Multiple categories can be specified in the parameter `additional_categories`. Only the specified categories will be included in the archive.

**Example**

```
# Muliple categories
projects.save_archive(path=r"D:\Tests", comment="A new project", additional_files=[], additional_categories=[ArchiveCategories.libraries, ArchiveCategories.devices])
```
