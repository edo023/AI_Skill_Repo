# Unforce Values

## Overview

The Debug > Unforce values command is available in online mode. It serves to release the forcing for all variables of the application. The variables revert to the value as determined by the logic of the application.

Additionally you have the option Release the force, without modifying the value which can be enabled in the [**Prepare Value**](D-SE-0084023.html#D-SE-0084023) dialog box for a forced variable.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  * You must have a thorough understanding of how the removing of forcing will affect the outputs relative to the tasks being executed. * Do not attempt to remove all forcing of the application, or all applications, without knowledge and confirmation of which I/O are in a forced state, and the effect the removal of the forcing would have on the application(s) concerned.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

NOTE: Command Unforce values [all applications], which affects all applications of a project, is not inserted in any menu per default. (If needed, see the [**Customize**](D-SE-0084067.html#D-SE-0084067) dialog box).
