# Insert Output

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | Ctrl + Shift + O |
| Call | * Ladder > Insert Output menu * Contextual menu |

## Function

The command inserts an output at the selected position. Possible insertion positions are indicated by the plus symbol at the cursor when you drag the element from the Toolbox to the implementation part. Replace the initially displayed question marks (`???`) by a variable name or a constant.

## Element: Output

The element is used to assign a signal coming from the left to the variable associated with the element. It has an input line coming from the left.
