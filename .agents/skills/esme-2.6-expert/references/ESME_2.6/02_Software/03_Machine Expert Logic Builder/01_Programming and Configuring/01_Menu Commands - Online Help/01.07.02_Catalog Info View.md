# Catalog Info View

## Overview

The View > Catalog Info View command opens the Catalog Info View. This separate view window provides further information on the item that is currently selected in the catalog view.

This view is small, by default, but you can resize the view within the framd window or you can move it to another position on your desk top as described in the chapter [*Customizing the User Interface*](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083359).
