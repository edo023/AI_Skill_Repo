# Device Type Manager (DTM) Import

![](images/logo.png)

Device Type Manager (DTM) Import

Overview

NOTE: The DTM installed with Machine Expert Installer are automatically imported in EcoStruxure Machine Expert.

There are 3 methods to import DTMs in EcoStruxure Machine Expert:

oDTM Repository to import DTMs officially supported by EcoStruxure Machine Expert

oDTM Repository for third-party DTM: to import other Schneider Electric and third-party DTMs (see note below)

oDevice Repository to import all DTMs (it can be slow if many DTMs are installed on the computer because no filter is used during the DTM discovery process).

NOTE: If you wish to use a third-party DTM with EcoStruxure Machine Expert software, contact your local Schneider Electric representative.

DTM Import into EcoStruxure Machine Expert with the DTM Repository

NOTE: The first time EcoStruxure Machine Expert Logic Builder is opened, the DTM Repository is automatically displayed if DTMs are installed on the computer and not yet imported into the Device Repository.

| Step | Action |
| --- | --- |
| 1 | In the menu bar, select Tools > DTM Repository.  Result: The DTM Repository window is displayed. |
| 2 | Select the DTMs to import into the Device Repository:  G-SE-0029142.4.gif-high.gif      Window description:  oScan: if unchecked, the DTM will not be imported in the Device Repository.  oDTM: name of the DTM.  oVersion: version of the DTM installed on the PC.  oStatus:  oNot Imported: The DTM is not imported in the Device Repository.  oImported: The DTM is imported in the Device Repository.  oUpdate Needed: The DTM update is required.  oIgnore in future: if checked, the DTM will no longer appear in subsequent calls of the DTM Repository.  oInclude DTMs already imported to the Device Repository: if checked, the DTMs that have already been imported to the Device Repository are displayed.  oCheck for new DTM versions at Logic Builder startup: if checked and there are new DTM versions, causes an automatic launch of the DTM Repository at EcoStruxure Machine Expert Logic Builder startup. |
| 3 | Click Import Selected DTMs.  Result: The Install DTM window is displayed. The PC is scanned to detect DTMs installed and registered on the PC. All DTMs officially supported by EcoStruxure Machine Expert which have been detected as installed and registered on the PC are listed. |
| 4 | Wait for the end of the [scan](../glossary/glossary.htm#XREF_D_SE_0024697_166).  Result: All DTMs officially supported by EcoStruxure Machine Expert which are currently installed and registered on the PC are displayed:  G-SE-0005754.5.gif-high.gif |
| 5 | Click OK.  Result: The selected DTMs are now imported and available in the Device Repository window and the Hardware Catalog. |

NOTE: The imported DTMs are available for all your existing and future projects.

DTM Import in EcoStruxure Machine Expert with the Device Repository

| Step | Action |
| --- | --- |
| 1 | Launch EcoStruxure Machine Expert Logic Builder. |
| 2 | In the menu bar, select Tools > Device Repository.  Result: The Device Repository window is displayed. |
| 3 | Click Install DTM.  Result: The Install DTM window is displayed. The PC is scanned to detect DTMs installed and registered on the PC. All DTMs which have been detected as installed and registered on the PC are listed. |
| 4 | Wait for the end of the [scan](../glossary/glossary.htm#XREF_D_SE_0024697_166).  Result: All DTMs which are currently installed and registered on the PC are displayed:  G-SE-0005754.5.gif-high.gif |
| 5 | Click OK.  Result: The selected DTMs are now imported and available in the Device Repository window and the Hardware Catalog. |
| 6 | Click Close. |

NOTE: For more information, refer to [Device Repository](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0084041_1).

The imported DTMs are available for all your existing and future projects.

Third-Party DTM Import in EcoStruxure Machine Expert with the DTM Repository

| Step | Action |
| --- | --- |
| 1 | In the menu bar, select Tools > DTM Repository.  Result: The DTM Repository window is displayed. |
| 2 | Click Import third party DTMs.  Result: The Install DTM window is displayed. The PC is scanned to detect third-party DTMs installed and registered on the PC. All third-party DTMs which have been detected as installed and registered on the PC are listed. |
| 3 | Wait for the end of the [scan](../glossary/glossary.htm#XREF_D_SE_0024697_166).  Result: All third-party DTMs which are currently installed and registered on the PC are displayed (the list is an example):  G-SE-0029149.3.gif-high.gif |
| 4 | Click OK.  Result: The selected DTMs are now imported and available in the Device Repository window. |

NOTE: If you wish to use a third-party DTM with EcoStruxure Machine Expert software, contact your local Schneider Electric representative.
