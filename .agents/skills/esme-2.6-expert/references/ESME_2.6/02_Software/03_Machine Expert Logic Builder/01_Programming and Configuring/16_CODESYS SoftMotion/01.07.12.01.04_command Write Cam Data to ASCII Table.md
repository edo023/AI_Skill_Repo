# Command: Write Cam Data to ASCII Table

**Function**: The command creates an `ASCII` file (`TXT` extension) on the development system. A specified number of X/Y-values of the active cam is saved in this file. A default dialog opens first and then the **Number of points** dialog.

**Call**: **Cam** menu

**Requirement**: The cam editor is open and displays a cam.
