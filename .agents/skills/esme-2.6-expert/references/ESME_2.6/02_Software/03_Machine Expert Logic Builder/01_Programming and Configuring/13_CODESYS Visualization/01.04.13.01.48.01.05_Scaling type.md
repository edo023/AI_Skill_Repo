# Scaling type

|  |  |
| --- | --- |
| **Scaling type** | Defines the scaling for height and width of the element   * **Anisotropic**:  The background image is scaled to the size of the element The height and width are scaled independently of each other. * **Isotropic**:  The background image is scaled to the size of the element, retaining its proportion. The proportion of height and width is fixed. |
