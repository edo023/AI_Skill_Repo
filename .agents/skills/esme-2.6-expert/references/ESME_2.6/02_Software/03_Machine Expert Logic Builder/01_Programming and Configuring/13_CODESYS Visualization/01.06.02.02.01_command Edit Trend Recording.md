# Command: Edit Trend Recording

Symbol: ![](images/uuid-19c42046-a5a2-103a-a5fe-3312b1.png)

**Function**: The command opens a [Trend Recording](_cds_obj_trend_recording.html#_cds_obj_trend_recording) object.

**Call**: **Visualization** menu; context menu of a **Trend** element in the visualization editor in the **Trend recording** property

**Requirement**: A **Trend Recording** element is selected in the visualization editor.
