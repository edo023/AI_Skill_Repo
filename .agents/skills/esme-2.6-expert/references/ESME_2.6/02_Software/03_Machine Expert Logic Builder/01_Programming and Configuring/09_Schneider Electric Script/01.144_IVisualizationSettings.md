# IVisualizationSettings

class

IVisualizationSettings.

IVisualizationSettings

Bases: object

Class provides methods to show and modify the visualization settings.

active\_profile\_name

Retrieves or sets the name of the active profile. The name is the directory name of a profile stored in visual elements repository.

Type

string

This example shows how to use the active\_profile\_name property to set or get the visualization profile.

```
# We enable the new python 3 print syntax
from __future__ import print_function

proj = projects.primary

# Print the active profile
print("Visual profile: " + visualization_settings.active_profile_name)

# Get the available visualization profiles
profile_names = visualization_settings.get_all_visual_profile_names()

# Print the profiles
for visual_profile in profile_names:
    print (" - " + visual_profile)

# Set the profile to V1.35.12.0
visualization_settings.active_profile_name = "V1.35.20.0"

# Get the active profile
profile_name = visualization_settings.active_profile_name

# Print the active profile
print("New visual profile: " + profile_name)

# Save project
projects.primary.save()
```

get\_all\_visual\_profile\_names

Retrieves the visual profile names.

Returns

A list of available profile names.

Return type

string[]

This example shows how to get the visual profiles.

```
# We enable the new python 3 print syntax
from __future__ import print_function

proj = projects.primary

# Get visualization profile
profile_name = visualization_settings.active_profile_name

# Print visualization profile
print("Current visual profile: " + profile_name)

# Get the available visualization profiles
profile_names = visualization_settings.get_all_visual_profile_names()

# Print the profiles
for visual_profile in profile_names:
    print (" - " + visual_profile)

# Save project
projects.primary.save()
```
