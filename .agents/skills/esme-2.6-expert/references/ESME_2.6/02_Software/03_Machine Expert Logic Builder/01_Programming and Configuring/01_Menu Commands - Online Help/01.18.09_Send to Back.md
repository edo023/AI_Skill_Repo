# Send to Back

## Overview

The CFC > Execution Order > Send to Back effects that the selected elements will be moved to the end of the [execution order](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083494). The order within the group of selected as well as of the unselected elements is maintained. Also, the order within the elements not selected will not be changed.
