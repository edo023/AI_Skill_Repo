# Add Module

## Overview

As a prerequisite, an application is selected in the Modules tree view.

The Project > Add Module command opens the Add Module dialog box that displays a list of available Smart Template modules that you can add to your controller application.

The search text box above the list allows you to enter a string and to filter the list according to the entered string. Right to the search text box, a Vendor list allows you to select a vendor and filter the Smart Template modules list according to the selected vendor.

Below the list of available Smart Template modules, two check boxes are provided:

| Check box | Description |
| --- | --- |
| Group by category | Select the check box to group the Smart Template modules according to their category. |
| Show all module versions | Select the check box to display not only the newest but all available versions of the Smart Template modules in the list.  Color-coding in the Version cells indicates the following incompatibility issues:   * Yellow marked Version cells indicate that this version of the Smart Template module is incompatible to the open project (for example, due to a referenced library being unavailable). * Grey marked Version cells indicate that this version of the Smart Template module cannot be added because there is already another version of this module used in the project. A combination of different versions of the same Smart Template module in one project is not allowed. |
