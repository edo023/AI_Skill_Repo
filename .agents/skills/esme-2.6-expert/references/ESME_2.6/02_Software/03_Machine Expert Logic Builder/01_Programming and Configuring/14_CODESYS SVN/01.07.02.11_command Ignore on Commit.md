# Command: Ignore on Commit

**Function**: The command identifies an object and adds it to the "ignore-on-commit” list. Then it is deactivated in the commit dialog by default.

**Call**: **SVN** menu

**Requirement**: At least one object is available that is not in the change list `ignore-on-comment`.

Objects of the "ignore-on-commit” list are overlaid with the ![](images/uuid-57dabac7-0ba2-74bb-5907-66eeb4.png) symbol in the object tree. By default, they are not selected in the commit dialog, unless a dependency of a selected object requires it. These objects can always be selected manually in the dialog.
