# Compiler Error C0237

## Message

No global definition found for VAR\_EXTERNAL ‘<variable name>’

## Message Cause

An attempt is made to declare a variable in VAR\_EXTERNAL which does not exist in VAR\_GLOBAL.

## Solution

Make sure that the identifiers match.

## Error Example

```
PROGRAM PLC_PRG
VAR_EXTERNAL
  i : INT;
END_VAR
```

```
VAR_GLOBAL
  ig : INT;
END_VAR
```

--> C0237: No global definition found for VAR\_EXTERNAL 'i'
