# EN/ENO > ENO

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | Ctrl + Shift + N |
| Call | * Ladder > EN/ENO > ENO menu * Contextual menu |

## Function

The command adds an `ENO` output to a block with an `EN` input. The `ENO` output has the same value as the `EN` input.

## Requirements

A block with an `EN` input is selected.
