# View as Function Block Diagram

## Overview

Shortcut: CTRL + 1

The FBD/LD/IL > View as Function Block Diagram command can be used in offline and online mode to switch to the FBD view of the [FBD/LD/IL editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083462).

The LD networks or the IL will be converted appropriately to FBD networks. Keep in mind that there are some special elements, which cannot be converted and thus will only be available in the appropriate editor view. Additionally, some constructs cannot be converted unambiguously (refer to the chapter [View as Instruction List](D-SE-0084146.html#D-SE-0084146) for the respective remarks).

To switch back to the LD view, execute the command [View as Ladder Logic](D-SE-0084145.html#D-SE-0084145) .

To switch back to the IL view, execute the command [View as Instruction List](D-SE-0084146.html#D-SE-0084146).

NOTE: A proper conversion presumes syntactically correct code. Otherwise parts of the implementation can become lost.

NOTE: Concerning the view options for the components of FBD, LD and IL networks, consider the FBD, LD and IL editor options.
