# Variable

|  |  |
| --- | --- |
| **Variable** | The array index of the list entry that the user clicks is saved at runtime.   * Variable (integer data type)  Example: `PLC_PRG.iIndexComboboxEntry` |
| **Data array** | Displayed as a combo box. Every array component becomes a combo box entry.   * Array variable (`ARRAY[...] OF`)  Example: `PLC_PRG.astrCombobox`  Declaration: `astrCombobox : ARRAY[0..4] OF STRING := ['First', 'Second', 'Third', 'Fourth'];` |
