# Compiler Error C0008

## Message

Unexpected end-of-file found: ‘<operator 1>’, ‘<operator 2>’, or ‘<operator 3>’ expected

## Message Cause

Syntax error

## Solution

Make sure that the syntax is correct.

## Error Example

```
PROGRAM PLC_PRG
VAR
  x: INT;
  bTest : BOOL;
END_VAR
```

```
IF bTest THEN
  x := 9;
```

--> C0008: Unexpected end-of-file found: 'ELSIF', 'ELSE' or 'END\_IF' expected
