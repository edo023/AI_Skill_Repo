# Set/Reset

## Overview

Shortcut: CTRL + M

The FBD/LD/IL > Set/Reset command is used in FBD or LD to define boolean outputs or coils as `Set` or `Reset` outputs or coils. It is not available in the IL editor, where the corresponding [modifiers](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083466) have to be used appropriately.

For information on the set/reset command, refer to the [*Set/Reset in FBD/LD/IL* chapter](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083483).

With multiple executions of the command, the output will alternate between set, reset, and normal output.

NOTE: Concerning the view options for the components of FBD, LD and IL networks, consider the FBD, LD and IL editor options.
