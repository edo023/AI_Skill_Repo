# Compiler Error C0070

## Message

‘INI’ operator needs a function block instance or data unit type instance

## Message Cause

Neither a function block instance nor a DUT instance is applied to the INI operator.

## Solution

Only pass function block instances or DUT instances to the INI operator.

## Error Example

```
PROGRAM PLC_PRG
VAR
  b : BOOL;
  inst : FB;
END_VAR
```

```
b := INI(b, TRUE);
```

```
FUNCTION_BLOCK FB
VAR
END_VAR
```

--> C0070: 'INI' operator needs a function block instance or data unit type instance

## Error Correction

```
b := INI(inst, TRUE);
```
