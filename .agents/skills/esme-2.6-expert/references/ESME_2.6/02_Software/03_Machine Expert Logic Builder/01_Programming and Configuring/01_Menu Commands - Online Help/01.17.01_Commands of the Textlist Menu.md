# Commands of the Textlist Menu

## Textlist Commands

Some commands for working with [text lists](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083432) are by default provided in the Textlist menu and in the contextual menu when working in a text list, and the appropriate ones also in the Visualization menu. Those commands that are not displayed by default, are available in the Tools > Customize [dialog box](D-SE-0084066.html#D-SE-0084066) and can be added to the menus manually.

## Insert Text

The Textlist > Insert Text command is also available in the contextual menu when working in the Text List editor. It serves for adding a new text in a new line above that one where the focus is on. It opens an edit field in the Standard column and by default enters the string NewText, which you can modify immediately.

On the usage of text lists, see [**Text List**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083432).

## Create Global Text List

The Textlist > Create Global Text List command can be used to create a global text list explicitly. The GlobalTextList [object](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083432) will be added to the Global node of the Applications tree.

GlobalTextList is created automatically as soon as the first text is defined in the configuration of a visualization. Refer to [text lists](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083432) for detailed information on text lists.

## Add Language

The Textlist > Add Language command is used to add a new language column to a text list.

Open the text list, execute the command, and enter the language name in the Choose language dialog box. After confirming by clicking OK the language column will be added rightmost in the present list.

Compare with [**Remove language**](#D-SE-0084076__D-SE-0084076.6).

## Remove Language

The Textlist > Remove Language command is used to remove this language column from a text list. Open the text list, set the cursor in the respective column and execute the command.

## Rename Language

The Textlist > Rename Language command is used to assign a new name to a language that is displayed in the text list as column heading. Open a text list or a global text list, select the language column heading that should be renamed, and execute the command.

## Check Visualization Text Ids

If a static text is modified within a visualization element, the visualization and eventually the GlobalTextList as well must have write permission (see [**GlobalTextList**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083432)). If modifications are performed though the write permission is missing, it may be that the text IDs no longer correspond to the texts of their visualization elements.

By use of the Textlist > Check Visualization Text Ids command, such detected errors can be detected in the visualizations.

## Update Visualization Text Ids

If a static text is modified within a visualization element, the visualization and eventually the GlobalTextList as well must have write permission (see [**GlobalTextList**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083432)). If modifications are performed though the write permission is missing, it may be that the text IDs no longer correspond to the texts of their visualization elements.

By use of the Textlist > Update Visualization Text Ids command, these errors can be corrected automatically. Therefore, all affected visualizations and the GlobalTextList must have write permission.

## Add Text List Support

Right-click a DUT object and execute the Add Text List Support command to assign text list support to the selected DUT object of type enumeration.

Text list support allows the localization of the enumeration component identifiers and the display of the symbolic component value in a text output of a visualization.

By default, the command is not available in the menus. You can add this command via the Tools > Customize [menu](D-SE-0084066.html#D-SE-0084066).

## Remove Text List Support

Right-click a DUT object and execute the Remove Text List Support command to remove text list support from the selected enumeration object.

By default, the command is not available in the menus. You can add this command via the Tools > Customize [menu](D-SE-0084066.html#D-SE-0084066).

## Remove Unused Text List Entries

The Textlist > Remove Unused Text List Entries command serves to delete texts no longer being used in a visualization element from the GlobalTextList.
