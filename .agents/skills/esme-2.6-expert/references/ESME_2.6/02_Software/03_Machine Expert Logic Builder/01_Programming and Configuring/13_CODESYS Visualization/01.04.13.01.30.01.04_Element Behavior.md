# Element Behavior

|  |  |
| --- | --- |
| **Element behavior** | * **Image switcher**: The image in switches with each mouse click. * **Image tapper**: While a visualization user holds down the mouse button, the image of the **Image on** property is displayed. At the same time, the value `TRUE` is assigned to the **Variable** property. |
| **Tap FALSE** | : : While the mouse button is pressed, the image of the **Image** property is displayed and the **Variable** property gets the value `FALSE` instead of the value `TRUE`, and back.  Requirement: **Image tapper** is selected in the **Element behavior** property. |
