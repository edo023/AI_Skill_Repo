# FBS

(*file-based storage*) A file format for saving EcoStruxure Machine Expert projects. The project is stored as a folder structure and each object of the project is stored as an individual file in a textual (editable) format.

The folder structure can consist of several folder levels with long folder names which can lead to long path names. To allow long path names, perform the following actions on the computer running EcoStruxure Machine Expert:

* In the Windows Settings, activate the option Enable long paths. Verify whether this is allowed by your company rules or adapt the Group Policy accordingly.
* Adapt the Git settings with the following command line argument:

  ```
  git config --global core.longpaths true
  ```
