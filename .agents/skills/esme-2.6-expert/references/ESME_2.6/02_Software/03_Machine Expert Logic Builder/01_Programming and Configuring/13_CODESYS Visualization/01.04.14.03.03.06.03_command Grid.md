# Command: Grid

**Symbol**: ![](images/uuid-8f7f0bb2-fbc8-c00d-526b-2be754.png)

**Function**: Enables positioning along the displayed grid

**Call**: Context menu

When the grid function is enabled, the element is automatically set to a grid point when it is inserted and moved. The element is aligned with the grid.

TIP:

You can adapt the command in the [Dialog: Options: Visualization](_visu_dlg_options.html#_visu_dlg_options) dialog. The grid size is adjustable. In addition, the grid function can be disabled. This causes only the grid points to be displayed. However, the element can be positioned freely.
