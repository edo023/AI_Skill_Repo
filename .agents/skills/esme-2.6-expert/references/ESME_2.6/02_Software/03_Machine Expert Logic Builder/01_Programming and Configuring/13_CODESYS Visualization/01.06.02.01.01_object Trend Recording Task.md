# Object: Trend Recording Task

Symbol: ![](images/uuid-af8af368-7d74-5d51-dfe7-6682d6.png)

When you design a visualization with a **Trend** element, CODESYS TargetVisu automatically extends the **Task Configuration** with a **Trend Recording Task**. The task is below an application one time at most and calls the `VisuTrendStorageAccess.GlobalInstances.g_TrendRecordingManager.CyclicCall` program to run the trend recording manager.
