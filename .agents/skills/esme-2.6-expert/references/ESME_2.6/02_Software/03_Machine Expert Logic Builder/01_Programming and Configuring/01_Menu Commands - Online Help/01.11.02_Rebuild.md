# Rebuild

## Overview

By default, the command is not available in the menus. You can add this command via the Tools > Customize [menu](D-SE-0084066.html#D-SE-0084066).

The Build > Rebuild command starts the [build](D-SE-0083980.html#D-SE-0083980) process for the currently active application, even if it had been built before successfully.

NOTE: The currently used compiler version is defined in the Project Settings > Compile options [dialog box](D-SE-0083949.html#D-SE-0083949).
