# Colors

The properties contain fixed values for setting colors.

|  |  |
| --- | --- |
| **Color** | Color of the line in normal state.  NOTE: The alarm state is in effect if the expression in the [Color variables → Toggle color](_visu_prp_color_variables_line.html#_visu_prp_color_variables_line) property has the value `FALSE`. |
| **Alarm color** | Color of the line in alarm state.  NOTE: The alarm state is in effect if the expression in the [Color variables → Toggle color](_visu_prp_color_variables_line.html#_visu_prp_color_variables_line) property has the value `TRUE`. |
| **Transparency** | Value (0 to 255) for defining the transparency of the selected color  `255`: The color is opaque.  `0`: The color is completely transparent.  NOTE: If the color is a style color and already has a transparency value, then this property is write-protected. |
