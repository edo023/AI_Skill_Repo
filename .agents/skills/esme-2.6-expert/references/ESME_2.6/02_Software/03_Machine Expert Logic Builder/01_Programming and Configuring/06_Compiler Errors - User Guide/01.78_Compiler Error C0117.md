# Compiler Error C0117

## Message

No such label ‘<jump label>’ within the scope of the ‘JMP’ statement

## Message Cause

A jump is made to a label that does not exist.

## Solution

Define the label that you specify as destination.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
JMP A;
```

--> C0117: No such label 'A' within the scope of the 'JMP' statement

EIO0000003933.04
