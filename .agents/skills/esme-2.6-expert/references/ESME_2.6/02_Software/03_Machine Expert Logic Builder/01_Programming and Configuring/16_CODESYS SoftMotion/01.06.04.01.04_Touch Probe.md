# Touch Probe

The `MC_TouchProbe` function block requires a `TRIGGER_REF` as an input. `TRIGGER_REF.iTriggerNumber` has the following meaning for the generic CiA402 axis:

| TRIGGER\_REF.iTriggerNumber | Meaning |
| --- | --- |
| 0 | Touch probe 1, positive edge |
| 1 | Touch probe 1, negative edge |
| 2 | Touch probe 2, positive edge |
| 3 | Touch probe 2, negative edge |
