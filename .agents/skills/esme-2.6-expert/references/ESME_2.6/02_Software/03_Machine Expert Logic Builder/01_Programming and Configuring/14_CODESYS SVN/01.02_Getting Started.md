# Getting Started

The following steps are required in order to develop your CODESYS project with CODESYS SVN with [Version Control](_svn_version_control.html#_svn_version_control) by Apache™ Subversion®:

1. Install the CODESYS SVN package in CODESYS.
2. To edit and further develop the project ([working copy](_svn_use_working_copy.html#_svn_use_working_copy)) under SVN version control, the project is edited in CODESYS and then committed to the SVN project repository.

NOTE:

As of version 4.4.0.0, in the case of **storage profile inconsistencies** between the working copy and the project in the SVN repository, corresponding instructions will guide you further. Under certain circumstances, commit actions at the object level are then no longer possible.
