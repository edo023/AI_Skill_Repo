# Compiler Warning C0371

## Message

Access to VAR\_IN\_OUT '<variable name>' declared in '<function block name>' from external context '<POU name>'.

## Message Cause

A VAR\_IN\_OUT of the POU is accessed in a method of a POU.

## Solution

Either change the declaration of the variable or remove the assignment.

## Error Example

```
FUNCTION_BLOCK FB_Test
VAR_IN_OUT
      bInOut : BOOL;
END_VAR

METHOD METH: BOOL
VAR
      bBool : BOOL;
END_VAR
bInOut := bBool;
```

–> C0371: Access to VAR\_IN\_OUT 'bInOut' declared in 'FB\_Test' from external context 'METH'
