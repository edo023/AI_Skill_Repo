# Command: Get Lock

Symbol: ![](images/uuid-2fbf635c-8604-07b8-6313-d3fded.png)

**Function**: The command locks the object explicitly for you. The **Lock Message** dialog opens for this purpose.

**Call**: Context menu: **SVN**

**Requirement**: The versioned object is not locked (not overlaid with the ![](images/uuid-9f018a6b-cee8-fd40-f3b1-20b199.png) symbol).
