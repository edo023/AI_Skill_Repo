# Create Control Point

## Overview

The CFC > Create Control Point serves to place a control point on a connection line in a CFC chart in order to define a line segment which then can be shifted by drawing it with the mouse. This means, that the originally automatically created routing of the connection will be modified manually. See [routing of connection lines](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083494) for details.

The command corresponds to dragging-and-dropping a control point element from the [CFC elements](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083493) to the desired position on a connection line.
