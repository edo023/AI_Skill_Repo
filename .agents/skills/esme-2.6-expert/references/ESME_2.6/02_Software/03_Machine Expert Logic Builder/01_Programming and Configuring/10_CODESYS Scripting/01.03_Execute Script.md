# Execute Script

You can execute Python script files (<file name>.py), which contain a sequence of commands for activating CODESYS functionalities, in the following way:

* In the CODESYS user interface by means of commands in the **Tools → Scripting** menu
* From the CODESYS user interface by means of a customized, configured toolbar
* From the Windows command-line
