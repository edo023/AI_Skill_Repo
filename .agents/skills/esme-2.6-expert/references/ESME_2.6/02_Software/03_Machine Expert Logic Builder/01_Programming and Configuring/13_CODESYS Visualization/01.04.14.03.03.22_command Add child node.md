# Command: Add child node

Symbol: ![](images/uuid-a35c15eb-40f4-610e-dcdb-3cc1a3.png)

**Function**: Adds a new property in the frame configuration, in the tree view of the interface properties, under the selected category

**Call**: **Visualization** menu, context menu

**Requirement**: A category is selected in the tree view.

The property is classified under the category. Double-click the new entry to open the line editor so that you can specify a suitable name. Afterwards, you can use the **Move node up** command to position the property.
