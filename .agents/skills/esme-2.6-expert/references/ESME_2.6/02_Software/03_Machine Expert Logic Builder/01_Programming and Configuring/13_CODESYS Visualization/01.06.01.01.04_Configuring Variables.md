# Configuring Variables

IMPORTANT:

The number of variables is limited for trend recording. But you can adjust the number per trend recording. To do this, open the object. You will find the **Maximum number of variables** setting below **Record Settings**.

For more information, see: [Object: Trend Recording](_cds_obj_trend_recording.html#_cds_obj_trend_recording)
