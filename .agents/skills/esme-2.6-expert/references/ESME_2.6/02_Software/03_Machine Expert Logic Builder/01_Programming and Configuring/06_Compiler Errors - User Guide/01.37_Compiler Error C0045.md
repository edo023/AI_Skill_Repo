# Compiler Error C0045

## Message

Using ‘THIS’ is not allowed in this context

## Message Cause

In order to be assigned to the instance, THIS can be used only in a method, action and transition, or in the body of a function block. This error message appears for all other positions.

## Solution

Use THIS only in an allowed context.

## Error Example

```
PROGRAM PLC_PRG
VAR
  test1: INT;
END_VAR
```

```
THIS^.test1 := 19;
```

--> C0018: 'THIS^.test1' is not a valid assignment target

--> C0062: 'THIS^' is not a structure variable

--> C0045: Using 'THIS' is not allowed in this context
