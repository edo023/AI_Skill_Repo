# Import from XML

## Overview

Use the Project > Import from XML command to import one or several objects which are available in the form of an export file in XML format.

A dialog box for browsing such files is displayed. By default, the filter is set to files with extension *.export*. You can create export files by exporting objects from a project executing the command [**Export as XML**](D-SE-0083968.html#D-SE-0083968).

Choose the desired export file and while the import dialog box is opened, select the position in the navigator where the imported objects should be inserted. The object list only shows objects which can be inserted at that position.

The imported objects are added to the open project. If objects with the same name are already available, messages will be displayed prompting you to decide whether objects should be overwritten or not.

NOTE: To import DTMs, right-click the device node in the Devices tree and click Advanced Configuration. Run the command Import DTM from the contextual menu.
