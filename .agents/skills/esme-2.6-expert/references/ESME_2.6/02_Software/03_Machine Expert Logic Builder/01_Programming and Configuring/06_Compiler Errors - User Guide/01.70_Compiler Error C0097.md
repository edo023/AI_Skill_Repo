# Compiler Error C0097

## Message

Duplicate definition of the variable ‘<variable name>’ in the function block ‘<function block name>’ and in the base ‘<base function block name>’

## Message Cause

A variable is declared with the same name in a function block and its base.

## Solution

Use different variable names.

## Error Example

```
PROGRAM PLC_PRG
VAR
  fb : FB;
END_VAR
```

```
FUNCTION_BLOCK FB EXTENDS FB2
VAR
  i : INT;
END_VAR
```

```
FUNCTION_BLOCK FB2
VAR
  i : INT;
END_VAR
```

--> C0097: Duplicate definition of the variable 'i' in the function block 'FB' and in the base 'FB2'
