# Creating an Alarm Class

Basic settings such as the priority, the acknowledgement method, the notification actions, and the display options are configured in an [Alarm Class](_cds_obj_alarm_class.html#_cds_obj_alarm_class) object.

**The following predefined alarm classes are available:**

* Alarm class: **Error**
* Alarm class: **Info**
* Alarm class: **Warning**
