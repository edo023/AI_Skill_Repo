# Compiler Error C0026

## Message

Identifier expected instead of ‘<invalid identifier>’

## Message Cause

An invalid identifier is passed to a method.

## Solution

Use valid identifiers.

## Error Example

```
METHOD 123
VAR_INPUT
END_VAR
```

--> C0243: The name used in the signature is not identical to the object name

--> C0026: Identifier expected instead of '123'

## Error Correction

```
METHOD METH123
```

EIO0000003933.04
