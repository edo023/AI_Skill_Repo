# Directories

## Overview

The Tools > Options > Directories (Devices, Libraries, ...) dialog box allows you to set the paths to the repository and temporary directories.

## Repository directories Tab

The Repository directories tab allows you to set the root path for the repositories by clicking the browse button (...) to browse for the folder or by entering the complete path to the directory in the Look for repositories in the following location: text box.

The default path for EcoStruxure Machine Expert is C:\ProgramData\EcoStruxure Machine Expert.

This path applies to the following repositories:

* Devices (System)
* Libraries (System)
* Visual elements (System)
* Visual styles (System)
* Device and function templates (System)
* Devices (User)
* Visual elements (User)
* Visual styles (User)
* Libraries (User)
* Device and function templates (User)

(System) repositories contain elements of the categories (devices, libraries, visual elements, visual styles and device and function templates) that are installed by the Schneider Electric Software Installer.

(User) repositories are empty by default. They are filled with the elements that you install, for example, via the Device Repository dialog box or with elements that you extract from a projectarchive file.

The two options below the repositories list allow you to use elements from legacy repositories that originate from SoMachine or SoMachine Motion installed in parallel to EcoStruxure Machine Expert on the same PC:

* Include legacy repositories
* Include SoMachine / SoMachine Motion

NOTE: The content of the (User) repositories can considerably increase over time and thus can have an impact on the performance of EcoStruxure Machine Expert (such as startup time). If you are sure that the content has become unnecessary or unsuitable for the present installation of EcoStruxure Machine Expert, the Clear user repositories button allows you to remove the contents of the (User) repositories. If, however, the repositories contain elements that are referenced in your projects the projects can no longer be built afterwards.

| NOTICE | |
| --- | --- |
|  | LOSS OF INFORMATION  Backup the directories listed in the Directories (Devices, Libraries, ...) dialog box before clicking the Clear user repositories button.  Failure to follow these instructions can result in equipment damage. |

NOTE: Clicking the Clear user repositories button also removes DTMs from EcoStruxure Machine Expert repositories. They can be restored by executing the Tools > DTM Repository... command.

The Reset repository locations button allows you to reset the repository paths to the default values (*C:\Program Data\EcoStruxure Machine Expert\V•.•\...*). If the repositories in the former path contain elements that are referenced in your projects, the projects can no longer be built afterwards.

| NOTICE | |
| --- | --- |
|  | LOSS OF INFORMATION  Record the existing paths before clicking the Reset repository locations button in the event that a path must be restored.  Failure to follow these instructions can result in equipment damage. |

## General directories Tab

The General directories tab allows you to set the path to the directory that is used for temporary files. The default path is *C:\Users\Admin\AppData\Local\Temp\*. You might want to specify another directory, for example, to exclude those temporary files from virus scans which may help improve the performance.
