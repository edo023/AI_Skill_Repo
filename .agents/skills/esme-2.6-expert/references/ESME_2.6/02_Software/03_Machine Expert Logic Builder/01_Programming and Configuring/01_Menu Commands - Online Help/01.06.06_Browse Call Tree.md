# Browse Call Tree

## Overview

The Browse > Browse Call Tree command opens the Call Tree [view](D-SE-0083923.html#D-SE-0083923). It is a static representation of the caller and the calls of a specific POU.

The command is available if a POU is open in the editor and the cursor is placed in a variable, or if a POU is selected in the Devices tree or in the POUs tree.
