# Object: Visualization Manager

Symbol: ![](images/uuid-91532721-61be-6952-6384-83e433.png)

The Visualization Manager is responsible for the configuration settings for all display variants of the visualizations of the current application.

The object is inserted below the application when a visualization object is inserted. Double-click to open a configuration dialog with several tabs. By default, the **Settings** tab opens first.

NOTE:

If the device in use supports display variants of the visualization, then the Visualization Manager provides respective objects automatically (for example, CODESYS WebVisu, CODESYS TargetVisu).
