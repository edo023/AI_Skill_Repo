# Positioning

|  |  |
| --- | --- |
| **Horizontal offset** | Distance (in pixels) from the scale or bar to the horizontal element frame  Used for achieving the exact position relative to the background image |
| **Vertical offset** | Distance (in pixels) from the scale or bar to the vertical element frame  Used for achieving the exact position relative to the background image |
| **Horizontal scaling** | Horizontal division of the scale (in pixels)  Used for achieving the exact positioning relative to the background image |
| **Vertical scaling** | Vertical division of the scale (in pixels)  Used for achieving the exact positioning relative to the background image |
