# Display Global Cross References

## Overview

The Edit > Browse > Display Global Cross References command allows you to display the locations where all variables with the same name are used within the whole project in the [Cross Reference List view](D-SE-0083917.html).

Example for the variable i\_xTest:

Display Global Cross References: i\_xTest

The command is available if a POU is open in the editor and a variable is selected.
