# Compiler Error C0011

## Message

No ‘CASE’ label found

## Message Cause

Syntax error in a CASE statement. A statement in a CASE statement is not assigned to a CASE label.

## Solution

Add a CASE label.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i: INT;
  x: INT;
END_VAR
```

```
CASE i OF
  x := 9;
END_CASE
```

--> C0011: No 'CASE' label found

## Error Correction

```
CASE i OF
0:
  x := 9;
END_CASE
```
