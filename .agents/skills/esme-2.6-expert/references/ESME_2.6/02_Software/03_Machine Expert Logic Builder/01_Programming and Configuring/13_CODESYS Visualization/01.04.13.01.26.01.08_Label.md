# Label

|  |  |
| --- | --- |
| **Label** | * **Outside**: Scale values are placed outside of the scale. * **Inside**: Scale values are placed inside of the scale. |
| **Unit** | Text which is displayed in the element  Example: Display of unit in m/s |
| **Font** | Font for labels (example: scale numbering).  Selection from the list box or by clicking the  button. |
| **Scale format (C syntax)** | Values scaled in `printf` syntax  Examples: `%d, %5.2f` |
| **Max. text width of labels** | Optional value that defines the maximum width of the scale label  The value is set automatically.  TIP: Change this value only if the automatic adjustment does not yield the expected result. |
| **Text height of labels** | Optional value that defines the maximum height of the scale label  The value is set automatically.  TIP: Change this value only if the automatic adjustment does not yield the expected result. |
| **Font color** | Selection from the list box or by clicking the  button |
