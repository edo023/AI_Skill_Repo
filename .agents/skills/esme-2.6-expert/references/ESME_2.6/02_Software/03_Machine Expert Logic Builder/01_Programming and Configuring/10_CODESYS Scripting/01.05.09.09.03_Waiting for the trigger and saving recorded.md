# Waiting for the trigger and saving recorded data

The following code snippet is based on the "[Downloading and starting Trace](e496235e78c491496fa3dee26bbd6321.html#UUID-7cdc3fea-17fb-1c0d-4407-f1812b4b2bd0_id_KopievonBeispieleSnippetsBestPractices-Downloadandstarttrace)" snippet and "Configuring a \* trigger".

```
triggerstate = editor.get_trigger_state()
while triggerstate != TriggerState.TriggerReached:
        system.delay(200)
        triggerstate = editor.get_trigger_state()

editor.save(r"<File path to csv file>")
```
