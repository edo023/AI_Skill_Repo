# Dialog: Create HMI Project: Choose HMI Device

**Function**: The dialog is part of the wizard that opens when you create a new project with the **HMI project** template. It allows for the selection of the device type for the new project.

**Call**: **File → New Project**, **HMI Project** template

|  |  |
| --- | --- |
|  | Window containing information about the selected device |
|  | List of installed HMI devices (with version) |
| **Show latest versions only** | : All versions for each device are listed for selection. |
| **Next** | Takes over the selection and opens the **Create HMI Project – Connect to HMI Panel** dialog  Requirement: The button is available only when you have selected an HMI device. |
| **Abort** | Applies the configuration and creates an HMI project |
