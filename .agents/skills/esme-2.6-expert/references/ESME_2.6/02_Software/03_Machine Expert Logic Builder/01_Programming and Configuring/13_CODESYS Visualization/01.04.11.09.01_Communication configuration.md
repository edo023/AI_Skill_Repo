# Communication configuration

The CFG file `CODESYSControl.cfg` is located on the PLC, for example in the folder  `Platforms/Linux/Projects`. There you can configure the communication as follows.

|  |  |
| --- | --- |
| `[CmpVisuHandlerRemote]` |  |
| `Communication.EncryptionMode=0` | Connection via plain text communication  Default |
| `Communication.EncryptionMode=1` | Connection via encrypted communication  If you start the Remote TargetVisu with `Communication.EncryptionMode=1` without having previously installed a certificate, then a dialog prompt with the certificate information is displayed after the start.When you click **Yes** to acknowledge the dialog, the Remote TargetVisu starts with encrypted communication.  By default, the certificate, which can then be installed., is stored in the folder `\cert\export`. |

For more information see: [Displaying a Visualization as a Variant of Remote TargetVisu](_visu_execute_remote_target_visu.html#_visu_execute_remote_target_visu)
