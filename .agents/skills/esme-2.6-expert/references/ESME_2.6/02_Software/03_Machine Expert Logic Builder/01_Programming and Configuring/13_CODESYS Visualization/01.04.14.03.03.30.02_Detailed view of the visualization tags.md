# Detailed view of the visualization tags

|  |  |
| --- | --- |
| **Visualization Path** | Visualization (indented: visualizations or visualization elements used)  This results in a multi-level hierarchy of calling visualizations or visualization elements. This layout represents the visualization path. |
| **Element ID** | Example: 20  Click the column title to invert the sort order. |
| **Variable Name** | Variable whose usage hierarchy is displayed as a **Visualization Path**  Example: `PLC_PRG.xRecord`  Click the column title to invert the alphabetical sort order. |
| **Tags** | Number of tags generated by a variable  A variable is counted at most one time. The tag number is displayed at the first occurrence. For all other occurrence locations, 0 is displayed.  Click the column title to invert the sort order. |
| **All Tags** | Number of tags which this variable generates on this visualization element, whereby duplicate occurrences are not taken into consideration  Click the column title to invert the sort order. |
