# Creating and updating alarm instances

You can update the alarm configuration so that all declared instances across the application are listed there.

1. In CODESYS, open the **Alarm Configuration** object below the application.
2. Click the **Create or update alarm instances** button.

   * All instances with alarm group templates of the application are currently listed in the **Alarm Instances** table.

For more information, see the following: [Using API Calls to Control Alarms Programmatically](_visu_trigger_alarms_programmatically_with_api.html#_visu_trigger_alarms_programmatically_with_api) and [Using API Calls to Trigger Events Programmatically](_visu_trigger_events_programmatically_with_api.html#_visu_trigger_events_programmatically_with_api)
