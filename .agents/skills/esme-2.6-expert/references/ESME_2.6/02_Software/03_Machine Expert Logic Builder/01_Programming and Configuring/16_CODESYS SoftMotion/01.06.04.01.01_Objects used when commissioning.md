# Objects used when commissioning

When commissioning, the following objects are accessed if they are supported by the device.

| Object | Read/Write Access | (M)andatory  (O)ptional | Comment |
| --- | --- | --- | --- |
| `0x1000:0` | Read | M | The lower value has to be 402. Otherwise cancel. |
| `0x1018:1` | Read | O |  |
| `0x1018:2` | Read | O |  |
| `0x1018:3` | Read | O |  |
| `0x1018:4` | Read | O |  |
| `0x6502:0` | Read | O |  |
| `0x605A:0` | Read | O |  |
| `0x60C2:1` | Write | If parameter `Set60C2 = TRUE` |  |
| `0x60C2:2` | Write | If parameter `Set60C2 = TRUE` |  |
| `0x6076:0` | Read | O |  |
