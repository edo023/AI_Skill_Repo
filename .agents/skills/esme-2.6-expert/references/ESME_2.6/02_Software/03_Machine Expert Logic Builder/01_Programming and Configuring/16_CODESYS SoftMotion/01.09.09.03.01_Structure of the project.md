# Structure of the project

* CODESYS SoftMotion Controller
* Axis group **Tripod**: The axis group defines the kinematics and the mapping to the individual axes **Tripod1**, **Tripod2**, and **Tripod3**.
* **DepictorCalculations (PRG)**, **Environment (PRG)**, **Ring (FB)**: Programs for controlling the complete sequence and visualization. These POUs are not relevant for understanding PLCopen blocks, except the definition of the product coordinate system (PCS\_1 and PCS\_2).
* **Robot (PRG)**: Program for controlling the movement sequences. A detailed description is located in the next paragraph.
* **Visualization**: Animated visualization for clarifying the sequence.
* **Tripod1**, **Tripod2**, **Tripode3**: Linear axes of the tripod.
* **DriveRotaryTable**: Axis for the rotary table.
* **DriveConveyorBelt**: Axis for the conveyor belt.
