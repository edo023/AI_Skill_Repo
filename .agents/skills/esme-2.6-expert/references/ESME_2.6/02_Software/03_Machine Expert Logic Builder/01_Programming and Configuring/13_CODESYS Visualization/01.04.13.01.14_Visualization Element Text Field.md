# Visualization Element: Text Field

Symbol:

![](images/uuid-ca125d9f-b4e5-02a2-859e-7c7d31.png)

Category: **Common Controls**

**The element can output text statically.**

* The contents of a variable can be part of the text.
* Show tooltips. The text is managed as static text and can also be defined so that the contents of a variable are also displayed.
* Display text dynamically. Texts of a text list are displayed dynamically.
* Used for a text input. For example, a user can input a number or a text literal.
