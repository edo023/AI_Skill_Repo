# Command: Make Vertical Spacing Equal

Symbol: ![](images/uuid-ba21e0cd-3a2e-9d29-2b57-e57012.png)

**Function**: The command aligns the selected visualization elements so that the top and bottom elements keep their positions and the elements in between are positioned vertically with equal spacing.

**Call**: **Visualization** menu; context menu

**Requirement**: Three or more elements are selected. The first element is blue and the other elements are gray.
