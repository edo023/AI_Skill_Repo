# Command: Scroll Path View

**Function**: The command activates and deactivates the scrolling of the CNC path.

**Call**: **CNC** menu

**Requirement**: A CNC path is open in the editor.

If scrolling is activated, then the view of the CNC path is scrolled and zoomed in the graphical editor so that a path element that is selected in the CNC program is completely visible.
