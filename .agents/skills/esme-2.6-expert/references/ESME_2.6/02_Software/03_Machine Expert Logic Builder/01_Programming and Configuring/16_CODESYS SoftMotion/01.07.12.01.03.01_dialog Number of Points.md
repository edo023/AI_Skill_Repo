# Dialog: Number of Points

|  |  |
| --- | --- |
| **Number of Points** | Number of points used for interpolation.  Preset: According to the number of X/Y-values that are stored in the read file. Example: `256`  You can decrease the preset value in order to determine the cam with fewer interpolation points. When determining the interpolation points, their X-values are distributed equidistantly. |

TIP:

As the cam is interpolated using a 5th degree polynomial, a larger number of interpolation points may cause oscillations.
