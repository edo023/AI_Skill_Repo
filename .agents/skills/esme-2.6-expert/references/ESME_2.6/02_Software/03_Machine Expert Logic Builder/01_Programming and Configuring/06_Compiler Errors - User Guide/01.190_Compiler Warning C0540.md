# Compiler Warning C0540

## Message

Attribute 'no\_assign' missing for POU '<POU>'. The type of the variable '<variable>' is attributed with 'no\_assign'.

## Message Cause

A POU X contains a variable, whose type (POU Y) has the `'no_assign'` attribute. POU X has no `'no_assign'` attribute.

## Solution

Also assign the `'no_assign'` attribute to the POU X.

## Error Example

```
{attribute 'no_assign'}
FUNCTION_BLOCK FB_NotAssignable
VAR
      pBool : POINTER TO BOOL;
END_VAR

FUNCTION_BLOCK FB_WrapperOfNotAssignable
VAR
      fbNotAssignable : FB_NotAssignable;
END_VAR
```

The declaration of the variable `fbNotAssignable` leads to the following message when instantiating `'WrapperOfNotAssignable'` in a `PRG`.

--> C0540: Attribute 'no\_assign' missing for POU 'FB\_WrapperOfNotAssignable'. The type of the variable 'fbNotAssignable' is attributed with 'no\_assign'.

## Error Correction

Also assign the `'no_assign'` attribute to the `FB_WrapperOfNotAssignable POU`.
