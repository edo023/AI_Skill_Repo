# Identify display variant

The `VisuFbClientTagDataHelper` library POU of the `VisuElemBase` library is provided to programmatically identify a display variant. The library itself is referenced in `VisuElems`. The library POU is typically called with `VisuElems.VisuFbClientTagDataHelper`.

For more information about this library POU, see its documentation in the Library Manager.
