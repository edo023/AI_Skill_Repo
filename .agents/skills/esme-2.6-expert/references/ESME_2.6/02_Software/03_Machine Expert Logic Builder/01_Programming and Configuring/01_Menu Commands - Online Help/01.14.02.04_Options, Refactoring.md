# Options, Refactoring

## Overview

The Tools > Options > Refactoring dialog box serves to define at which renamings in the project an automatic refactoring should be offered.

|  |  |
| --- | --- |
| Auto declare | If the option On adding or removing variables, or on changing the scope is enabled, the Refactoring dialog box opens when you delete variable names in the Auto Declare dialog box and click the OK button. The Refactoring [dialog box](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083455) allows you to remove the variable throughout the project.  If the option On renaming variables is enabled, the Refactoring dialog box opens when you specify variable names in the Auto Declare dialog box and click the OK button. The Refactoring [dialog box](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083455) allows you to rename the variable throughout the project. |
| Mapping editor | If the option On renaming variables is enabled, Automatic Refactoring is suggested for the rename action when you change a variable name in the I/O Mapping [tab of the device editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083399). |
| Navigator | If the option On renaming objects is enabled, Automatic Refactoring is suggested for the rename action when you change an object name in a [navigator](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083356). |
| Tabular declaration editor | If the option On renaming variables is enabled, Automatic Refactoring is suggested for the rename action when you change the name of a variable in the [tabular declaration editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083519). |
| Unit conversion editor | If the option On renaming of unit conversions is enabled, Automatic Refactoring is suggested for the rename action when you change the name of a conversion in the [editor for unit conversion](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083583). |
