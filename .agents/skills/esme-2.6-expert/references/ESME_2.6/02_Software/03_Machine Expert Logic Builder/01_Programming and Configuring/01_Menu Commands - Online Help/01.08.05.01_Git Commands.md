# Git Commands

The following Git commands are provided in Git-managed \*.fbs projects upon right-clicking an element in the tree structure of the Logic Builder.

| Command | Prerequisite for the command to be enabled | Function |
| --- | --- | --- |
| Git Compare | One modified file is right-clicked in the tree structure. | A side-by-side view of the modifications indicated in the rendered files is displayed. |
| Git Compare (raw file contents) | One modified file is right-clicked in the tree structure. | A side-by-side view of the modifications indicated in the files as they are stored on the disk is displayed. |
| Git Resolve Conflicts | One conflicted file is right-clicked in the tree structure. | A side-by-side editor opens to resolve the conflict. |
| Git Restore | One or more modified files are right-clicked in the tree structure. | Discards the modifications made to the selected file or files since the last Commit command.  Before the modifications are discarded, another confirmation message is displayed that requests confirmation by clicking Yes. |
| Git Stage Changes | One or more modified files are right-clicked in the tree structure. | The selected file or files is / are added to the staging state displayed in the [Status & Staging view](StatusStaging-2CE9FB60.html). |
| Git Unstage Changes | One or more staged files are right-clicked in the tree structure. | The selected file or files is / are removed from the staging state displayed in the [Status & Staging view](StatusStaging-2CE9FB60.html). |
