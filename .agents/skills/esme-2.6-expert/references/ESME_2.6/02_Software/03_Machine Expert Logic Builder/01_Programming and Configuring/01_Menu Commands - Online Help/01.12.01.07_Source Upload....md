# Source Upload...

## Overview

The Online > Source Upload... command is used for uploading a project from a controller. For this purpose, a project archive file must be available there, possibly generated by the [**Source Download**](D-SE-0083901.html#D-SE-0083901) command.

The command opens the Select Device dialog box, where you have to choose the network path to the controller as in the Communication Settings [view](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0088469) . Select the appropriate entry in the tree of available devices and click OK.

The Project Archive dialog box opens, where you can configure which contents of the archive should be extracted for upload and to which path they should be copied. The usage of this dialog box corresponds to that of the Project Archive/Extract Archive function. After confirmation by clicking OK, the files are copied. If a file is already available in the specified path, you are asked whether it should be overwritten

Then a dialog box is displayed, asking whether the extracted project should be opened in the programming system.
