# Command: Calculate and Export Standard Metrics

Function: Calculates and exports the standard metrics directly without displaying the metrics table in the editor. Specify a suitable name for the CSV export file in the standard file dialog.

Call: This command is not available in any menu by default.

Alternatively, you could also use the [Show standard metrics](../../../../../../api/crossBook?lang=en-US&virtualBookName=CODESYS_Static_Analysis&topicID=_san_cmd_view_standard_metrics) command to display the metrics in the editor. Among other things, the editor provides the **Export Table** context command.

TIP:

**Customizing the Menu**

Providing the command

1. Click **Tools → Customize**.

   * The **Menu** tab opens. First you need to define a menu position for the command. In this case, it makes sense to select an item under the **Build → Static Analysis** menu.
2. Select the **Static Analysis** category on the left side and then the desired command on the right side. Click **OK** to close the dialog. Then click **OK** again to close the **Customize** dialog.

   * The **Calculate and Export Standard Metrics** command is now available in the **Build → Static Analysis** menu.
