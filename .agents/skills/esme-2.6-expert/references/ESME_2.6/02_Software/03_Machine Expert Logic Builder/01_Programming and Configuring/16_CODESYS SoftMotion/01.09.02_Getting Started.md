# Getting Started

To program the axis movements for a robot, you first need to create an axis group below the application in your project. For more information about this, see: [How to Create an Axis Group](_sm_creating_an_axis_group.html#_sm_creating_an_axis_group).

You configure the created axis group with the desired kinematics. You insert real or virtual axes below the device and link these axes to the axis group. For more information, see the help pages in [Kinematics](_sm_robotics_kinematics.html#_sm_robotics_kinematics):

* [Axis Group States](_sm_robotics_state_machine.html#_sm_robotics_state_machine)
* [Combination of Position and Orientation Kinematics](_sm_robotics_combining_position_and_tool_kinematics.html#_sm_robotics_combining_position_and_tool_kinematics)
* [Rotary Joints and Modulo Axes](_sm_robotics_rotary_joints_modulo_axes.html#_sm_robotics_rotary_joints_modulo_axes)
* [Rotary Axes with a Value Range over 360°](_sm_robotics_rotation_axis_range_bigger_360.html#_sm_robotics_rotation_axis_range_bigger_360)

How to create a program for the axis control after creating and configuring the axes is shown here as an example: [How to Create a Program for Controlling the Axis Group](_sm_creating_an_axis_group_program.html#_sm_creating_an_axis_group_program)

For information about how to implement the various motion control demands, see the additional help pages in [Motion Control](_sm_f_motion_control.html#_sm_f_motion_control).

* [Buffering and Blending of Movements](_sm_robotics_blending.html#_sm_robotics_blending)
* [Jogging of Axis Groups](_sm_cartesian_jogging.html#_sm_cartesian_jogging)
* [Interruption and Continuation of Movements](_sm_robotics_stop_continue_movements.html#_sm_robotics_stop_continue_movements)
* [Synchronization with a Moving Coordinate System](_sm_robotics_dynamic_tracking.html#_sm_robotics_dynamic_tracking)
* [Configuring a Tool Offset](_sm_configure_tool_offset.html#_sm_configure_tool_offset)

Background information for understanding how specific movements are performed can be found here:

* [Orientation Interpolation for CP Movements](_sm_robotics_orientation_interpolation.html#_sm_robotics_orientation_interpolation)

For advanced users:
