# Menu: File

|  |  |
| --- | --- |
| **New HTML5 control** | Opens the **Create a New HTML5 Control** dialog |
| **Open** | Opens the device selection dialog  Select an HTML5 control to edit there. |
| **Close** | Closes the control which is actively open in the editor |
| **Save** | Saves the changes in the description file |
| **Save as** | Opens the file selection dialog to save the HTML5 control under a new name |
| **Save and Install** | Saves and installs the HTML5 control |
| **Recently opened files** | Lists the most recently used HTML5 controls |
| **Exit** | Exits the  **HTML5 Control Editor** |
