# Compiler Error C0082

## Message

‘<invalid pragma>’ is no valid condition for a pragma

## Message Cause

When using a pragma, an invalid expression is used in the IF condition.

## Solution

Use valid pragma conditions.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
{IF abc}
i := 5;
{END_IF}
```

--> C0082: '!!!ERROR!!!' is no valid condition for a pragma

## Error Correction

```
{IF defined (abc)}
```
