# Compiler Error C0004

## Message

‘<variable>’ is not a component of ‘<structure>’

## Message Cause

Component access with ”.” to a variable that is not a structured value or does not exist as a component of the structure.

## Solution

Access a defined component, or change the definition of the component in the data type. The input assistance “List components” provides the valid access to this position.

## Error Example

```
TYPE DUT:
STRUCT
  x, y : INT;
END_STRUCT
END_TYPE
```

```
PROGRAM PLC_PRG
VAR
  test1 : DUT;
  test2 : INT;
END_VAR
```

```
test2 := test1.z;
```

--> C0032: Cannot convert type 'Unknown type: 'test1.z'' to type 'INT'

--> C0004: 'z' is not a component of 'DUT'
