# Compiler Error C0120

## Message

A ‘FB\_Exit’-Method of a function block or structure needs an input ‘bInCopyCode’of type BOOL

## Message Cause

The BOOL type input ‘bInCopyCode’ is missing.

## Solution

Define the input.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : FB;
END_VAR
```

```
FUNCTION_BLOCK FB
METHOD FB_exit
VAR_INPUT
END_VAR
```

--> C0120: A 'FB\_Exit'-Method of a function block or structure needs an input 'bInCopyCode' of type BOOL

## Error Correction

```
METHOD FB_exit
VAR_INPUT
  bInCopyCode : BOOL;
END_VAR
```
