# Compiler Warning C0526

## Message

Default value is not constant.

## Message Cause

The initial value of a VAR\_INPUT does not use a constant, but the result of a function call.

## Solution

Use a constant as the initial value.

## Error Example

```
FUNCTION F2: INT

FUNCTION F1: INT
VAR_INPUT
      iPar : INT := F2();
      xPar : BOOL;
END_VAR
```

The declaration of the variable `iPar` leads to the following message when calling F1 from a `PRG`.

--> C0526: Default value is not constant.

EIO0000003933.04
