# Continuous Function Chart (CFC) Language

## Overview

The Continuous Function Chart is an extension to the IEC 61131-3 standard, and is a graphical programming language based on the [Function Block Diagram (FBD) language](D-SE-0083463.html#D-SE-0083463). However, in contrast to the FBD language, there are no networks. CFC allows the free positioning of graphic elements, which in turn allows for feedback loops.

For creating CFC programming objects in EcoStruxure Machine Expert, see the description of the [CFC editor](D-SE-0083491.html#D-SE-0083491).
