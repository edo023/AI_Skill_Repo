# Element appearance

The properties contain fixed values for defining the look of the element.

|  |  |
| --- | --- |
| **Border line width** | Value (in pixels)  Example: `2`  Note: The values `0` and `1` both result in a line weight of one pixel. If no line should be displayed, then the **Border line style** property must be set to the option **Invisible**. |
| **Border line style** | * **Solid** * **Dash** * **Dots** * **Dash Dot** * **Dash Dot Dot** * **Invisible** |
| **Frame line color** | * Style color from the list box. Example: `Black` * Fixed value that is selected in the color dialog. Example: `0; 0; 0` |
