# ICompilerSettings

class

ICompilerSettings.

ICompilerSettings

Bases: object

Class provides methods to show and change the compiler version.

active\_compiler\_version

Retrieves or sets the compiler version.

Type

string

**Note**

This property returns / requires the version format mapped by OEM. Use MapFromInternalToOEMVersion() and MapFromOEMToInternalVersion() to convert between CODESYS and OEM version.

This example shows how to use the active\_compiler\_version property to set or get the compiler version.

```
# We enable the new python 3 print syntax
from __future__ import print_function

# Load the project
proj = projects.primary

# Set the project as primary project
proj = projects.primary

print("All compiler versions")

# Get all compiler versions (filtered)
compiler_versions = compiler_settings.get_all_compiler_versions()

# Print all compiler versions (filtered)
for version in compiler_versions:
    print (" - OEM mapped version: " + version)

# Get active compiler version
compiler_version = compiler_settings.active_compiler_version

print("compiler version: " + compiler_version)

# Set new compiler version
compiler_settings.active_compiler_version = "1.36.35.0"

print("New compiler version: " + compiler_settings.active_compiler_version)

# Save project
projects.primary.save()
```

map\_from\_internal\_to\_oem\_version

version

Maps from internal to OEM version.

Parameters

version (string) -- The internal version string.

Returns

The mapped OEM version, null if not mapped.

Return type

string

This example shows how to verify the mapping between an internal and OEM compiler version.

```
# We enable the new python 3 print syntax
from __future__ import print_function

# Set the project as primary project
proj = projects.primary

print("All compiler versions (not filtered)")

# Get all compiler versions (not filtered)
compiler_versions = compiler_settings.get_all_compiler_version_not_filtered()

# Print all compiler versions (not filtered)
for version in compiler_versions:
    print (" - CODESYS based version: " + version)

    # Verify if CODESYS compiler version is mapped to an OEM version
    mapped_version = compiler_settings.map_from_internal_to_oem_version(version)
    
    if mapped_version == None:
        print (" - Not OEM mapped!")
    else:
        print (" - OEM mapped version: " + mapped_version)
```

map\_from\_oem\_to\_internal\_version

version

Maps from OEM to internal version.

Parameters

version (string) -- The OEM version string.

Returns

The mapped internal version, null if not mapped.

Return type

string

This example shows how to verify the mapping between an OEM and internal compiler version.

```
# We enable the new python 3 print syntax
from __future__ import print_function

proj = projects.primary

print("All compiler versions")

# Get compiler versions (filtered)
compiler_versions = compiler_settings.get_all_compiler_versions()

# Print compiler versions (filtered)
for version in compiler_versions:
    print (" - OEM mapped version: " + version)

    # Verify if OEM version is mapped to a CODESYS compiler version
    mapped_version = compiler_settings.map_from_oem_to_internal_version(version)

    if mapped_version == None:
        # No mapping available
        print (" - No CODESYS based version mapped!")
    else:
        # Mapping available, print mapped version
        print (" - CODESYS based version: " + mapped_version)
```

get\_all\_compiler\_versions

Gets the compiler versions.

Returns

Returns a filtered list with the compiler versions.

Return type

string[]

**Note**

The versions are based on the mapped OEM compiler version.

**Note**

Since Machine Expert 2.2 (CODESYS V3.5 SP19) only one compiler version is supported.

This example shows how to get a list of OEM-filtered compiler versions.

```
# We enable the new python 3 print syntax
from __future__ import print_function

proj = projects.primary

print("All compiler versions")

# Get compiler versions (filtered)
compiler_versions = compiler_settings.get_all_compiler_versions()

# Print compiler versions (filtered)
for version in compiler_versions:
    print (" - OEM mapped version: " + version)
```
