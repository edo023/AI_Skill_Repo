# IScriptApplicationLoggerObject

class

IScriptApplicationLoggerObject.

IScriptApplicationLoggerObject

Bases: object

Provides services on the application logger.

xml\_import

path

Imports the logger points and logger messages from an external XML file with the given path into this instance.

Parameters

path (string) -- The full path of the file in XML format.

xml\_export

path

Exports the state of this instance in XML format into a file with the given path.

**Note**

The state depends on previous calls of [load\_logger\_points](#iscriptapplicationloggerobject_module_IScriptApplicationLoggerObject__iscriptapplicationloggerobject_IScriptApplicationLoggerObject_IScriptApplicationLoggerObject_load_logger_points) and / or [load\_logger\_messages\_and\_logger\_points](#iscriptapplicationloggerobject_module_IScriptApplicationLoggerObject__iscriptapplicationloggerobject_IScriptApplicationLoggerObject_IScriptApplicationLoggerObject_load_logger_messages_and_logger_points) .

Parameters

path (string) -- The full path of the file in XML format.

load\_logger\_points

Loads the logger points from the controller into this instance.

**Note**

Preconditions: The application on the controller must run and the application logger must be registered before. Registering can be done via IEC program by calling 'APL.G\_ifApplicationLogger.RegisterCommunicationService();'.

load\_logger\_messages\_and\_logger\_points

Loads logger messages and logger points from the controller into this instance.

**Note**

Preconditions: The application on the controller must run and the application logger must be registered before. Registering can be done via IEC program by calling 'APL.G\_ifApplicationLogger.RegisterCommunicationService();'.

save\_log\_levels

Saves the logger levels of the logger points of this instance to the controller.

get\_logger\_points

Gets a read only list of the logger points.

**Note**

The log level can be modified on each logger point and updated to the controller using [save\_log\_levels](#iscriptapplicationloggerobject_module_IScriptApplicationLoggerObject__iscriptapplicationloggerobject_IScriptApplicationLoggerObject_IScriptApplicationLoggerObject_save_log_levels) .

Returns

A read only list of the logger points.

Return type

[IScriptList](IScriptLoggerPoint.html#iscriptloggerpoint_module_IScriptLoggerPoint)
