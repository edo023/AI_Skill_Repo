# CFC Editor

## Overview

The Tools > Options > CFC Editor dialog box provides settings concerning the editing in a CFC [editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083491) (continuous function chart).

The settings will be applied to the opened editor views as soon as they have been confirmed (with the dialog box closed) by clicking OK.

## General Tab

|  |  |
| --- | --- |
| Enable AutoConnect | If you enable this option: when you drop elements somewhere on the canvas, then unconnected pins touching each other are automatically connected. This enables quick editing. |
| Prepare values in implementation part | If this option is enabled, in online mode, you can prepare variable values for writing and forcing in the implementation part of the CFC POU. Besides that, the prepared values are displayed in the inline monitoring boxes of the variables in angle brackets. |

## View Tab

|  |  |
| --- | --- |
| Display grid points | If this option is enabled, a grid (10 points) is displayed in the editor window and the elements can only be positioned along this grid and be moved in spans of 10 points. |
| Show box icon | If this option is enabled, the following is true:  If a function block or a function available via a library or via the object properties is provided with an icon (bitmap), then this icon is displayed in the box in the CFC editor. The usual operators are provided with icons. |
| Edit Line Colors | This button opens an editor for defining the colors for the connection lines depending on the applied data type. The lines appear in these colors in offline and online mode, except they are overwritten by the thick black and blue lining indicating boolean data flow. Click the button Add type > to open a selection list where you can choose a data type. After that, the dialog box for selecting a color is displayed. |

## Print Tab

The Print tab allows you to configure the printing settings.
