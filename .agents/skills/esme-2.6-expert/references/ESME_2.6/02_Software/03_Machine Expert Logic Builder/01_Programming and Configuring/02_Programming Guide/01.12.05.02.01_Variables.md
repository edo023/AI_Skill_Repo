# Variables

## Overview

You can declare variables either locally in the declaration part of a POU or in a global variable list (GVL) or in a persistent variables list or in the I/O mapping of devices.

Refer to the chapter [*Variables Declaration*](D-SE-0083597.html#D-SE-0083597) for information on the declaration of a variable, including the rules concerning the variable identifier and multiple use.

It depends on the [data type](D-SE-0083658.html#D-SE-0083658) where a variable can be used.

You can access available variables through the Input Assistant.

## Accessing Variables for Arrays, Structures, and POUs

The table lists the respective syntax for accessing arrays, structures, and POUs:

| Syntax | Access to |
| --- | --- |
| <array name>[Index1, Index2] | 2-dimensional [array](D-SE-0083672.html#D-SE-0083672) components |
| <structure name>.<variable name> | [structure](D-SE-0083673.html#D-SE-0083673) variables |
| <function block name>|<program name>.<variable name> | function block and program variables |
