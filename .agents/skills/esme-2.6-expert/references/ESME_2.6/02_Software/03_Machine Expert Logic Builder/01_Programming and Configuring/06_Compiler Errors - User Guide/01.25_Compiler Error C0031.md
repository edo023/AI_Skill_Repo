# Compiler Error C0031

## Message

Type definition expected instead of ‘<no data type>’

## Message Cause

An invalid type definition is assigned to the identifier.

## Solution

Specify a valid type definition.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : 0;
END_VAR
```

--> C0031: Type definition expected instead of '0'

## Error Correction

```
i : INT := 0;
```
