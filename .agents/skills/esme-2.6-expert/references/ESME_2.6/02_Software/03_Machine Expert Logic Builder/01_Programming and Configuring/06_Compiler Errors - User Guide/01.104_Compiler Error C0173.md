# Compiler Error C0173

## Message

‘<keyword>’ not allowed in this place.

## Message Cause

A declaration keyword, such as VAR\_INPUT, VAR\_OUTPUT, or VAR is not allowed at this location.

## Solution

Correct the declaration: Inputs and outputs are not useful or necessary in type definitions or global variable lists.

## Error Example

```
TYPE DUT :
STRUCT
    VAR_INPUT;
        member : INT;
    END_VAR
END_STRUCT
END_TYPE
```

```
PROGRAM PLC_PRG
VAR
  myDUTVar;
END_VAR
```

-->C0173: ‘VAR\_INPUT’ not allowed in this place.

## Error Correction

```
TYPE DUT :
STRUCT
  member : INT;
END_STRUCT
END_TYPE
```
