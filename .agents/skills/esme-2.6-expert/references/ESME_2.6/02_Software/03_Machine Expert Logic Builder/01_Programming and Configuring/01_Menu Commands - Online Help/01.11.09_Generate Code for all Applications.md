# Generate Code for all Applications

## Overview

The Generate Code for all Applications command performs two operations:

* It compiles the controller applications that are available in the Applications tree.
* It builds the HMI target applications.

## Procedure

Execute the Generate Code for all Applications command as follows:

| Step | Action |
| --- | --- |
| 1 | From the Build menu, run the command Generate Code for all Applications.  **Result**: The applications are compiled and the HMI target applications are built. This is indicated in the Messages [view](D-SE-0083922.html#D-SE-0083922). |
| 2 | In the Messages view, select Build from the list in the upper left side of the view to see the results of the individual compile processes. |
| 3 | To display a summary of the messages that occurred during the execution of the Generate Code for all Applications command, select Vijeo-Designer from the list in the upper left side of the view. |
| 4 | To verify the messages, you can start Vijeo-Designer directly by double-clicking the Vijeo-Designer message in the Messages view. |
