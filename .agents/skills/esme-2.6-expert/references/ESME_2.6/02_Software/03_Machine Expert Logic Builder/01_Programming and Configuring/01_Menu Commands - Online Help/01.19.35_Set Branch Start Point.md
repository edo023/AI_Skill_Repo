# Set Branch Start Point

## Presentation

The FBD/LD/IL > Set Branch Start Point command is available in LD editor. It sets the start point of a branch.

You can execute the command if you have selected a connection line. The start point can also be set in the contextual menu or with the space bar. A defined start point is marked by a red square. The possible end points are marked by a light blue square. You can cancel the command by pressing the Esc key.

If you use the element branch start/end from the [toolbox](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083473) to drag the start point to the editor, the possible insert positions are indicated by gray position markers.

To insert a new branch into the network, execute the command Set Branch End Point.

For further information, refer to the chapter [*Insert Branch*](D-SE-0084134.html#D-SE-0084134).
