# Command: Group

Symbol: ![](images/uuid-1c2f1faf-8af7-426a-fa7a-19f246.png)

**Function**: The command groups the selected visualization elements and displays the as one element.

**Call**: **Visualization** menu; context menu

**Requirement**: At least two elements are selected.

To select multiple elements, draw a selection rectangle around the elements. As an alternative, you can hold down the **Shift** key and click the elements.

To select all elements, click [Command: Select None](_visu_cmd_select_none.html#_visu_cmd_select_none) in the context menu of the visualization editor.

TIP:

You can also use drag elements to add them to a group. To do this, hold down the **Shift** key and drag the elements to the group. The appearance of the cursor changes (a small plus sign is displayed).
