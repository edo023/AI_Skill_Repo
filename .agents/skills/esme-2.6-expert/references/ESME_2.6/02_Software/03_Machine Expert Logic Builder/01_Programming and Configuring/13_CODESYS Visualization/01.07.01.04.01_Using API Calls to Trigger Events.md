# Using API Calls to Trigger Events Programmatically

| NOTICE | |
| --- | --- |
|  | Example: AlarmManager |

Events are alarms of observation type **Event (API)**. They are triggered programmatically in the application code using the functionality of the `AlarmManager` library and the [IArarmHandler7](../../../../../../api/crossBook?lang=en-US&virtualBookName=AlarmManager&topicID=IAlarmHandler7) interface.

Events are defined directly in an alarm group or in an alarm group template as an alarm instance.

For more information, see Example: AlarmManager
