# Getting Started with CODESYS Softmotion Basic

To program a movement with CODESYS Softmotion Basic, at least one axis needs to be configured in the device tree. This includes setting the axis type (**modulo** or **finite**), the velocity ramp type, the software limit switches, and the error reaction.

An overview of the supported movement types can be found in the [Single-Axis Movements](_sm_basic_single_axis.html#_sm_basic_single_axis) chapter.

For information about how to program a single-axis movement, see the [Controlling the Movement of Single Axes](_sm_example_single_axis_motion_control.html#_sm_example_single_axis_motion_control) chapter.

For more information about the basic use of cams, see the [Controlling a Cam Drive with a Virtual Time Axis](_sm_example_drive_control_virtual_time_axis.html#_sm_example_drive_control_virtual_time_axis) chapter.

For information about buffering, blending, and superimposing movements, see the [Commanding Multiple Movements](_sm_basic_mulitple_movements.html#_sm_basic_mulitple_movements) section.
