# Element properties displayed in tabular form

|  |  |
| --- | --- |
| **Property** column | Properties of the selected element |
| **Value** column | The assigned value is applied in the editor view. |

|  |  |
| --- | --- |
| Double-click in the **Value** column | A line editor, list box, or dialog opens for editing the value.  opens the **Input Assistant** dialog for help (for example, when assigning variables or image references). |
| Single-click for a selected field |  |
| **Blank** for a selected field |  |

TIP:

A style selected in the Visualization Manager can include individual, predefined element properties. As a result, these are not listed in this view because a fixed value is already assigned to them. They do not require any more configuration.

TIP:

Visualizations can be configured with device-specific restrictions that block the availability of element properties.

**Device-specific restrictions :**

* Fewer available elements
* Fewer available fonts
* Fewer displayable fonts
* Fewer displayable image formats
* Maximum number of visualization elements
* Maximum number of visualizations below the device
