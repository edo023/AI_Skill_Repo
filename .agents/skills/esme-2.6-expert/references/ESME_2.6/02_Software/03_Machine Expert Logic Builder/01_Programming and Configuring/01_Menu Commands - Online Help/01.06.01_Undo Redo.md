# Undo/Redo

## Undo

Default shortcut: CTRL + Z

The Undo command undoes the action which was most recently executed in the open editor or view.

Repeated use undoes all actions back to the time that the view was opened. This applies to all actions in the editors for POUs, data types, visualizations, and global variables and in the object organizer.

With the Redo command, you can restore an action which you have undone before.

## Redo

Default shortcut: CTRL + Y

The Redo command restores an action in the open editor or view, which has been undone before.

You can carry out the Redo command as often as you have previously executed the Undo command.
