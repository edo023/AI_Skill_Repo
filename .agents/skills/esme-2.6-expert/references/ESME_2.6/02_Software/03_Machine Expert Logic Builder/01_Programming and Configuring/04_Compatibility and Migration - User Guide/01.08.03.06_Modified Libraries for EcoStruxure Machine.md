# Modified Libraries for EcoStruxure Machine Expert V2.6

The following libraries were subject to change from EcoStruxure Machine Expert V2.5 to V2.6. The table lists the affected libraries with the modifications that were performed and the issues this can cause. If you use one or more of the libraries in your project, perform the relevant steps listed below the table to migrate to V2.6.

Perform the following steps to migrate:

1. Remove the library reference and open the Add library dialog box to add the library with the new name.

   Expected entry:

   ```
   [placeholdername] = [library name], [version] (Schneider Electric)
   ```
2. Adapt your program code.

   Replace the occurrences of the namespace `[old namespace]` with the `[new namespace]`.
3. Open the Library Manager and verify that the library is referenced using the placeholder `[placeholdername]`.

   If this is not the case, remove the library reference and add the library again using the Add library dialog box.

   Expected entry:

   ```
   [placeholdername] = [library name], [version] (Schneider Electric)
   ```
4. Verify the effective resolution of the placeholder SE\_PLCSystem.

   This change is relevant for library projects where placeholders are not resolved by device descriptions.

   SE\_PLCSystem is the common placeholder for the controller-specific system library. The device description of the controller provides the correct resolution.
