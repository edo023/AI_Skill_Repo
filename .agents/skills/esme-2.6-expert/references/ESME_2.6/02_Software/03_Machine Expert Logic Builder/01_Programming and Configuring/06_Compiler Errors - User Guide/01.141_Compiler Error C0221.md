# Compiler Error C0221

## Message

Direct address ‘<address>’ malformed

## Message Cause

An address is not displayed completely.

## Solution

Make sure that the address is displayed correctly.

## Error Example

```
PROGRAM PLC_PRG
VAR
  xVar : BOOL;
END_VAR;
```

```
xVar := %IX0;
```

--> C0221: Direct address '%IXO' malformed

## Error Correction

```
xVar := %IX0.2;
```

EIO0000003933.04
