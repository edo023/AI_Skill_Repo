# Compiler Error C0141

## Message

Reference assign needs a variable with write access

## Message Cause

A constant is assigned to the reference assignment.

## Solution

Assign a writable variable.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
  I_r : REFERENCE TO INT;
END_VAR
```

```
I_r REF= 314;
```

--> C0141: Reference assign needs a variable with write access

## Error Correction

```
I_r REF= i;
```
