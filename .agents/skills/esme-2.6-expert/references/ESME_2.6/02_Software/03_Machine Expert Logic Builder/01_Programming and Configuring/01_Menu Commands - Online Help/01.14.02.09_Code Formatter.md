# Code Formatter

## Overview

The Tools > Options > Code Formatter dialog box allows you to configure the code formatting feature for programming code written in the ST (Structured Text) [editor](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083510).

The settings are applied while executing the Format Code command from the Edit menu or from the contextual menu that opens upon right-clicking in the declaration or implementation part of the ST editor view.

## General Tab

| Element | Description |
| --- | --- |
| Indentation style section | |
| Tab | Select this option to use a tab as indentation character. |
| Spaces | Select this option to use spaces as indentation characters and enter the number of space characters.  Default value: 4  Value range: 0...16 |
| Automatic Linebreaks section | |
| Max. line length | Configure the maximum number of characters per line after which an automatic linebreak is inserted.  Default value: 150 |
| Min. line length | Configure the minimum number of characters of a line.  Default value: 50 |
| Max. POU call parameters in single line | Configure the maximum number of parameters that are allowed per function.  Default value: 6 |

## Alignment Tab

| Element | Description |
| --- | --- |
| Enable Alignment | Select this option to execute the alignment on the programming code. |
| Alignment Settings section | |
| Max. length for deviation | Configure the maximum deviation of the length that is allowed.  Default value: 40 |
| Prevent alignment beyond line length of | Configure the minimum length of a line that prevents alignment.  Default value: 130 |
| Enable variable declaration alignment | Select this option to execute the alignment on variable declarations. |
| Enable ENUM declaration alignment | Select this option to execute the alignment on ENUMs. |
| Enable POU call parameter alignment | Select this option to execute the alignment on parameters. |
| Enable assignment alignment | Select this option to execute the alignment on assignments. |

## Empty Lines Tab

| Element | Description |
| --- | --- |
| New Lines at end of File section | |
| Force a single empty line | Select this option to insert one empty line at the end of the programming code. |
| Remove all empty lines | Select this option to remove empty lines at the end of the programming code. |
| Keep as it is | Select this option to leave the end of the programming code unchanged. |
