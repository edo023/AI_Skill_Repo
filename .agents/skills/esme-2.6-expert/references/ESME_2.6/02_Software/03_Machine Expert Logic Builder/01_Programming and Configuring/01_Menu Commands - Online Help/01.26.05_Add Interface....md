# Add Interface...

## Overview

The Add Interface... command is available in the contextual menu and behind the Add button in the Gateway Configuration [dialog box](D-SE-0084177.html#D-SE-0084177__D-SE-0084177.3). It can be used for adding one or more main interfaces for communication.

For a description of the gateway configuration, refer to the [*Configure the local gateway...* chapter](D-SE-0084177.html#D-SE-0084177__D-SE-0084177.3).
