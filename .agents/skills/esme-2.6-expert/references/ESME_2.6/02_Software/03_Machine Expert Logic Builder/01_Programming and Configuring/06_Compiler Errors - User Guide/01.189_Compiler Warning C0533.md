# Compiler Warning C0533

## Message

The default value for a VAR\_OUTPUT is not used in abstract or interface methods.

## Message Cause

In a method of an abstract POU, an initial value is assigned to a VAR\_OUTPUT.

In the case of interface methods and abstract methods, the initial value at the output has no effect, in contrast to initial values at the input.

## Solution

Remove the initial value from the affected VAR\_OUTPUT.

## Error Example

```
FUNCTION_BLOCK ABSTRACT POU

METHOD METH: BOOL
VAR_INPUT
      iPar : INT;
      xPar : BOOL;
END_VAR
VAR_OUTPUT
      iOut : INT;
      xOut : BOOL := TRUE;
END_VAR
```

The declaration of the variable `xOut` leads to the following message when extending the function block `POU` and instantiating this one in a `PRG`.

--> C0533: The default value for a VAR\_OUTPUT is not used in abstract or interface methods.

EIO0000003933.04
