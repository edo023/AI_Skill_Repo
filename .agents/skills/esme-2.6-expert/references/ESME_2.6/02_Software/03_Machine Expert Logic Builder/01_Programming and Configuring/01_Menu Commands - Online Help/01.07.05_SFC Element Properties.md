# SFC Element Properties

## Overview

The View > Element properties command opens the Properties view for the currently selected element. The properties are displayed in a structured table. To edit them, click the values field due to the type of the element.

For details on the particular element properties, refer to the [*SFC Element Properties* chapter](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083502).
