# Introduction

Robots often have rotary axes with a value range over 360°. For example, the last axis (orientation axis A3) of a SCARA robot can often travel more than one rotation, from about -540° to 540°.

This means that the same position and orientation can be approached in multiple ways. Considering SCARA for example, the orientation and position of the tool is identical, regardless of whether the last axis is at -170° or at 190° (= -170° + 360°). As a result, the conversion of Cartesian position and orientation to axis values (the kinematic inverse transformation) is not unique, but has several solutions.

This can be used, for example, in Pick&Place applications to save time by moving from 170° to 190° and not to -170°.

Note: The movement of the SCARA orientation axis can quickly become a limiting factor for the cycle time in these applications.
