# Monitoring Range

## Overview

The Monitoring Range dialog box is available in online mode when a POU with a variable of data type ARRAY is being monitored. It can be used to restrict the range of array elements whose values are displayed during monitoring.

To open the dialog box, click in the cell of the column Data Type that belongs to the array variable.

| Element | Description |
| --- | --- |
| Valid range | The validity range of the array elements that are monitored.  Example of a three-dimensional array:   ``` [1..10][-3..3][-10..10] ``` |
| Maximum number of array elements | Number of elements of the array variables.  Example: `1470` |
| When you edit one of the settings “Start” , “End” , or “Scroll range of 1000 elements” , both of the other settings are adapted automatically. | |
| Start | Index of the first array element whose value is displayed. |
| End | Index of the last array element whose value is displayed. |
| Scroll range of 1000 elements | Scrollbar for selecting a range from the set of array elements. |
