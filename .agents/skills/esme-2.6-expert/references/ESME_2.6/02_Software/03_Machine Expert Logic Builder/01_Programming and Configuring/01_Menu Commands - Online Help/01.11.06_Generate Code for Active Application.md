# Generate Code for Active Application

## Overview

The Build > Generate Code for Active Application command allows compilation of the application of a library project for test purposes. By executing this command, code generation and source code tests are performed but the code is not downloaded to the controller. Thus, you can verify the code for compilation errors.

The following prerequisites must be fulfilled:

* A library project is open.
* The library project contains an application (that can be achieved by adding a controller for test purposes to the library).
