# Insert Network

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | Ctrl + I |
| Call | * Ladder > Insert Network menu * Contextual menu |

## Function

The command inserts an additional network in the Ladder editor. Possible insertion positions are indicated by the plus symbol at the cursor when you drag the element over the implementation part.

## Requirements

No block is selected.

## Element: Network

A network is the core unit of an LD program. In the Ladder editor, networks are arranged in a list. Each network is provided with a consecutive network number on the left-hand side and can include logical and arithmetic expressions, program/function/function block calls, jumps, or return statements.

You can assign the following elements to a network:

* Title: Double-click the first line at the top of a network to specify the network title.
* Comment: Double-click the second line at the top of a network to specify a network comment.
* Jump label: Double-click the third line at the top of a network to define a jump label. The jump label can then be specified as a target in the [jump element](InsertJump-453AE50E.html#InsertJump-453AE50E__ElementJump-453B375C).

Refer to the [Tools > Options > Ladder Editor dialog box](LadderEditor-44B3FC46.html) to enable the visibility of the elements.
