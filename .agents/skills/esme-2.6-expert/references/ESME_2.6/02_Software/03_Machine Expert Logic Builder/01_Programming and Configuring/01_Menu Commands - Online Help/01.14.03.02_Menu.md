# Menu

## Overview

The Menu tab of the Tools > Customize dialog box allows defining the structure and commands of the menus.

The defined menus, submenus, and the assigned commands are displayed. A menu/sub-menu title is indicated by an icon ![](images/T003483281.gif). Indented below the respective commands and submenus are listed in the same top-down order as they are displayed in the EcoStruxure Machine Expert menu. They can be separated by a separator line.

## Buttons

The following buttons are available:

| Button | Description |
| --- | --- |
| Add command... | In the menu structure box, select that line above which you want to insert a new command and click the Add command... button.  In the Add command... dialog box, select the command to be added and confirm by clicking the OK button. |
| Add separator | Click the button to insert a separator line above the marked command in the menu structure box. |
| Add Popup Menu...  Edit Popup Menu... | In the menu structure box, select that line above which you want to insert a new popup menu.  Click the button Add popup menu... to insert a new popup menu before the selected line. Select a popup menu, and click Edit popup menu... to edit it. The Edit Popup Menu dialog box allows you to define or to modify menu titles in various languages. |
| Reset | Click the button to reset the menu to the default settings. |
| Load...  Save... | You can save the settings to an external file (<filename>.opt.menu) or reload it from previously saved setting. |

## Add / Edit Popup Menu

You can enter the default text of the new popup menu in the first text field. Click the Add language > button to add new languages. After you have added a new language, you can enter the language-dependent text in the table. Click text entries to modify them.

The new popup menu will be displayed in the menu bar if you have assigned at least one command. The menu title will be displayed in the [**currently set user interface language**](D-SE-0084048.html#D-SE-0084048) if a localized text has been defined for that language.
