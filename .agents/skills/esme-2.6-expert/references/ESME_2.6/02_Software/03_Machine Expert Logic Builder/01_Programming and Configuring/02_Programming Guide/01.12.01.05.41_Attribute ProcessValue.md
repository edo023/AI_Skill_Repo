# `Attribute ProcessValue`

## Overview

The `ProcessValue` attribute marks a component of a structure. In the CFC editor, you can execute the Use attributed member as input [command](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0105996) to connect this structure to a scalar type input.

## Syntax

```
{attribute 'ProcessValue'}
```

## Insert Location

Insert the `ProcessValue` attribute in the line above the structure variable.

## Example

```
TYPE QINT :
STRUCT
    Status : STRING;
    {attribute 'ProcessValue'}
    Value1 : INT;
    Value2 : INT;
END_STRUCT
END_TYPE
```
