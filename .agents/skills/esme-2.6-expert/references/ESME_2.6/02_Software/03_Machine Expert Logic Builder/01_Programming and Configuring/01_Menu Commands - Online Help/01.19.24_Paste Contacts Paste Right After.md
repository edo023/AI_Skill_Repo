# Paste Contacts: Paste Right After

## Overview

Shortcut: CTRL + G

The FBD/LD/IL > Paste Contacts: Paste Right After command is only available in the LD editor. It pastes the elements or the section of the network which has been put to the clipboard before by a Copy or Cut [command](D-SE-0083910.html#D-SE-0083910), right to the currently selected contact element in the network.

For a description on selecting in networks, refer to the chapter [*Working in the FBD and LD Editor*](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083467).
