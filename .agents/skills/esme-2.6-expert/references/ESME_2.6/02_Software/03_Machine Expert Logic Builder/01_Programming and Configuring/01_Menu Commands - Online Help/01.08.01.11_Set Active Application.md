# Set Active Application

## Overview

Use the Project > Set Active Application command to define which of the available [applications](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083435) defined in the Applications tree should be the active one. If an application is selected in the Applications tree, the command is available in the contextual menu and by default in the Project menu. After you have executed the command, the chosen application will display as a bolded entry in the Applications tree.

Keep in mind that online activities only concern the active application.
