# Run to Cursor

## Overview

The Debug > Run to Cursor command can be used (for example for debugging purposes) in order to execute the program up to a temporary definable position.

This corresponds to setting a [breakpoint](D-SE-0084029.html#D-SE-0084029) at the desired next stop position and stepping there.

Place the cursor at the desired next break position and perform the command. The instructions between the current break position and the cursor position will be executed.
