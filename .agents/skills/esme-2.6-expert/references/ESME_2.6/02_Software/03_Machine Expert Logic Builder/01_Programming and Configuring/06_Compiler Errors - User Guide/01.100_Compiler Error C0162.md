# Compiler Error C0162

## Message

Number <number of array values> of array initializations is no constant value

## Message Cause

The initialization [Value1,NumberValue2(Value2)] only works with a constant for NumberValue2.

## Solution

Only use constants.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT := 3;
  arr1 : ARRAY[1..4] OF INT := [1,i(7)];
END_VAR
```

--> C0162: Number 'i' of array initializations is no constant value

## Error Correction

```
arr1 : ARRAY[1..4] OF INT := [1,3(7)];
```
