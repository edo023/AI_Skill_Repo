# Overview

This section provides an overview of movements for which one axis is moved synchronously with another axis (depending on time or position.

* Cams: Depending on a drive (master), another drive (slave) should execute any kind of defined movements. For more information, see: [Cams](_sm_f_cam.html#_sm_f_cam)
* Synchronization of two axes with a defined transmission ratio using MC\_GearIn, MC\_GearOut, and MC\_GearInPos. For more information about these functions, see the documentation for the respective function block.
* Phase offset between a master and slave axis using MC\_Phasing. For more information about this function, see the documentation of the function block.
* Compensation of backlash between mechanical components (for example, in a gearbox) using SMC\_BacklashCompensation. For more information about this function, see the documentation of the function block.
