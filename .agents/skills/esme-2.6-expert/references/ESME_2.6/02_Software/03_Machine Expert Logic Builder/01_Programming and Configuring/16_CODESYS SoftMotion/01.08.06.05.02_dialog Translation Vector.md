| **OK** | This moves the CNC path. In addition, a G0 command (relative positioning command) is inserted with the coordinates of the translation vector in the CNC program before the first positioning command. The positioning commands are G1, G2, G3, G5, G6, G8, G9, G10, G11, and G92.  If G54, G55, or G56 commands shift the coordinate system in the CNC program, then the arguments are adapted in these commands. The blocks up to G53 no dot require any further adaptation.  If a G56 command is the first positioning command, then only that command is adapted.  The block numbers are assigned automatically. |
