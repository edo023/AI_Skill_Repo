# Import and Export Options

## Overview

The Tools > Import and Export Options dialog box allows you to configure the export and import of selected settings of the EcoStruxure Machine Expert options. The settings are saved to an XML file with the default extension (options.xml).

The Import and Export Options dialog box provides the following options:

| Option | Description |
| --- | --- |
| Export selected options | From the Select options list, select the categories of User or Machine options whose settings will be exported to the XML file.  In the File box, enter the path to the export file on the local file system, or click the ... browse button to browse for the folder where the file will be created (for example, D:\system1.options.xml). The file type *option export* (\*.options.xml) is preset. |
| Import selected options | In the File box, enter the path to the options export file whose contents is to be imported, or click the ... browse button to browse the local file system for the folder where the file of type *option export* (\*.options.xml) is located. |
| OK | Click OK to close the dialog box and to perform the export or import operation as specified in the dialog box. |
