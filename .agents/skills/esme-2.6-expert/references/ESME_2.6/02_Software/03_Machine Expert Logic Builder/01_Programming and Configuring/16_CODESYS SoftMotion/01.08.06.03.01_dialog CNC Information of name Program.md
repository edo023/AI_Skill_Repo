# Dialog: CNC Information of <name> Program

The dialog provides information about the specified CNC program.

|  |  |
| --- | --- |
| **Program name** | Example: `Workpiece_1` |
| **Version of function blocks** | Example: `4.2.0.0` |
| **Number of lines** | Example: `5` |
| **Number of objects** | Example: `3` |
| **Path length [units]** | Example: `261.803398874992` |
| **Duration [s]** | Time (in seconds) for traveling the preprocessed path  Example: `10.158`  Requirement: Path preprocessing without errors and successfully generated interpolation points |
| **Show generated code** | The **Generated Code** dialog opens.  Requirement: The program contains no errors. |
