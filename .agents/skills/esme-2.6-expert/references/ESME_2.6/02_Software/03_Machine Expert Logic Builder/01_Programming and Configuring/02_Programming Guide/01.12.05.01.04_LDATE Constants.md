# LDATE Constants

## Overview

Use the LDATE constant to specify a date.

## Syntax

ld#<date declaration>

Instead of `ld#` you can also use the following:

* `LD#`
* `ldate#`
* `LDATE#`

Enter the date declaration in format `<date keyword>#<year>-<month>-<day>`.

LDATE values are 64-bit types and contain the time span in nanoseconds since 01.01.1970, 00:00 clock until 21.07.2554.

Examples

```
LDATE#2018-8-8
LD#1996-05-06
```

For further information, refer to the description of the [Date and Time Data Types](D-SE-0083662.html#D-SE-0083662__D-SE-0083662.7).
