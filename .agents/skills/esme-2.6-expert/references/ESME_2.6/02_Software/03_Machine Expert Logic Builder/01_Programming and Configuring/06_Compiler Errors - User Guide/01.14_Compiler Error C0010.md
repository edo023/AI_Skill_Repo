# Compiler Error C0010

## Message

Unexpected end-of-file found: ‘<token>’ expected

## Message Cause

Syntax error

## Solution

Make sure that the syntax is correct.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i: INT;
END_VAR
```

```
FOR i := 0 TO 2 DO
;
```

--> C0010: Unexpected end-of-file 'END\_FOR' found

EIO0000003933.04
