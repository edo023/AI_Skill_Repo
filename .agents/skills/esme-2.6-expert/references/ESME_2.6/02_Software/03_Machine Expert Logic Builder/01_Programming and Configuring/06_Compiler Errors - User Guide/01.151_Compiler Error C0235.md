# Compiler Error C0235

## Message

Second operand of \_\_QueryInterface must be an interface reference

## Message Cause

Incorrect operands are passed to the operator \_\_QueryInterface.

## Solution

Pass an interface reference.

## Error Example

```
PROGRAM PLC_PRG
VAR
  a : INT;
  ITFref : ITF;
  ITFref2 : ITF2;
END_VAR
```

```
__QueryInterface(ITFref2, a);
```

```
INTERFACE ITF EXTENDS __SYSTEM.IQueryInterface
INTERFACE ITF2 EXTENDS ITF
```

--> C0235: Second operand of \_\_QueryInterface must be an interface reference

## Error Correction

```
__QueryInterface(ITFref2, ITFref);
```
