# Image ID variable

|  |  |
| --- | --- |
| **Image ID** | Variable (`STRING`) for the image ID  Example: `PLC_PRG.stImageID`  Application code: `PLC_PRG.stImageID := 'ImagePool_A.Bild3';`  The variable value formally refers to a specific image of an image pool and corresponds to the value in the **Static ID** property. |
