# Tab: Default Hotkeys

Symbol: ![](images/uuid-d0cfc073-2468-d7ad-ce83-bf6a35.png)

The tab includes a list of configured hotkeys that are valid for all visualizations available throughout the application. Therefore, the tab is the central location for defining hotkeys for all visualizations that are below an application.

Hotkeys of the standard keyboard handling are **not** listed here.

The tab is similar to the [Keyboard Configuration](_visu_cmd_hotkeys_editor.html#_visu_cmd_hotkeys_editor) tab and provides the same editing options.
