# Dialog: Update

|  |  |
| --- | --- |
| **HEAD** | : The command behaves the same as the **Update** and **Update project** commands. |
| **Revision** | : The revision, to which the last updated was made, is selected by the revision number.  : Opens the **Log** dialog for selecting the revision |
| **Date** | : The revision, to which the last update was made, is selected by the modification date.  **Use UTC time**: : The date is displayed in universal time. |
| **Recursive** | : Default setting  The selected part is updated recursively. This means that all elements below the selected object are also updated. |
| **Omit externals** | : Externals are not updated. |
