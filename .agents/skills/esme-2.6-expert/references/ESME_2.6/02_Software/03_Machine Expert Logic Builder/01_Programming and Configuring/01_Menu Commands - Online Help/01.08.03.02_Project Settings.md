# Project Settings

## Overview

The Project Settings command is by default available in the Project menu. The object Project Settings is by default available in the Global node of the Tools tree of a new project.

The Project Settings dialog box provides subdialogs for various settings, such as project encryption, user, and access right management, version handling, layout definitions for printouts, and so on.

On the left side of the dialog box, the categories of possible settings are listed. On the right side, the corresponding parameters will appear. The different categories are explained in the following chapters.
