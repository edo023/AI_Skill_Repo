# Add New Gateway

## Overview

The Add New Gateway command opens the Gateway dialog box, where you can define a gateway to be added to the present configuration.

You can also open this dialog box by clicking the Add gateway... button in the Communication Settings [tab in classic mode](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0088469).

Enter a symbolic Name, the Driver type, and driver-specific parameters (for example, IP Address and Port) for that gateway. To enter the parameter values, double-click the respective column field to open an edit frame. Some information on the concerned parameter each is provided in the lower part of the dialog box.

After having closed the dialog box by clicking OK, the new gateway entry will be added to the tree structure of configured gateway channels in the left part of the Communication Settings dialog box.

You can also use the Gateway dialog box to modify the gateway entry later (refer to the Edit Gateway... [command](D-SE-0084181.html#D-SE-0084181)).
