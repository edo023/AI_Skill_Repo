# Alarm Configuration

An alarm element can be added to a visualization. For details regarding the configuration of visualization alarm management, refer to [Visualizing Alarm Management](../../../../../api/crossBook?lang=en-US&virtualBookName=CODESYS_Visualization&topicID=_cds_f_visualize_alarms).

NOTE: An alarm element can only be configured if a Webvisualisation object has been added to the Visu Manager object.
