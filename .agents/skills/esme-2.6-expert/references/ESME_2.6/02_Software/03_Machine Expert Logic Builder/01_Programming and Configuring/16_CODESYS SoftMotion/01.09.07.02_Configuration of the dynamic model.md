# Configuration of the dynamic model

An existing dynamic model in the form of a function block which implements the `ISMDynamics` interface can be assigned to an axis group by means of the `SMC_GroupSetDynamics` function block. Besides the dynamic model, the function block also has to be used to configure the direction and magnitude of the gravitational acceleration in the form of a vector in the machine coordinate system.
