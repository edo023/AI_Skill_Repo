# Compiler Error C0115

## Message

The second parameter of a conditional call (CALC) has to be a valid call statement.

## Solution

Specify the call of a function, method, or function block in the second parameter of the conditional (CALC) call.

## Error Example

```
PROGRAM PLC_PRG
VAR
  condition : BOOL;
END_VAR
```

```
CALC(condition, 1+2);
```

-->C0115: The second parameter of a conditional call has to be a valid call statement.

## Error Correction

```
CALC(condition, MyFunction(1,2))
```
