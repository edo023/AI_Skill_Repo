# Compiler Warning C0357

## Message

POU '<POU name>' has been marked as obsolete: <message text>.

## Message Cause

A POU is assigned the attribute `'obsolete'`, but it is still used in the program.

## Solution

Remove the attribute from the POU or do not use the POU.

## Error Example

```
{attribute 'obsolete':= 'datatype fb not valid!'}
FUNCTION_BLOCK FB_Test

PROGRAM PLC_PRG
VAR
      inst : FB_Test;
END_VAR
```

--> C0357: POU 'FB\_Test' has been marked as obsolete: datatype fb not valid!

EIO0000003933.04
