# Compiler Error C0144

## Message

Inheritance is only allowed in function blocks, interfaces and structures

## Message Cause

An attempt is made to use inheritance in an object that does not permit inheritance.

## Solution

Only use EXTENDS in function blocks, interfaces and structures.

## Error Example

```
PROGRAM PLC_PRG
VAR
  inst : DUT_1;
END_VAR
```

```
TYPE DUT:
(
  enum_member := 0
);
END_TYPE
```

```
TYPE DUT_1 EXTENDS DUT:
(
  enum_memberX := 0
);
END_TYPE
```

--> C0144: Inheritance is only allowed in function blocks, interfaces and structures
