# Connect Selected Pins

## Overview

The CFC > Connect Selected Pins command is only enabled when exactly one output pin and one or several input pins are selected. When executed, a connection between the output pin and the input pin will be established.

For further information, see [connections in CFC](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083494).
