# Add Object > Parameter List...

## Introduction

The Project > Add Object > Parameter List... command is available if the Application node is selected in the Devices tree. Execute the command to add ParameterList nodes below the selected Application node, and to open the ParameterList editor.

The ParameterList editor displays devices with selected device parameters in one table. You can adjust the displayed parameter values. The input assistant helps you to select devices and parameters.

## Functions

The ParameterList editor provides the following functions:

* Fast adding of the devices of one device category when creating a device list
* Adding, modifying, deleting devices and parameters
* Using multiple parameter lists in one project
* Cut/copy/paste, undo changes/restore functions for the devices and parameters
* Modifying the position of a device or parameter in the list
* Visibility/modifiability of parameters regarding device configuration (comparable to the device editor)
* Printing parameter lists
* Modifying parameter values in online and offline mode
* Optionally writing values modified in online mode as offline values into the project
* Supporting text search for device and parameter names
* Updating device names if they have been changed outside the parameter list

## Creating a ParameterList

| Step | Action | Result |
| --- | --- | --- |
| 1 | Select an Application node in the Devices tree and execute the Project > Add Object > Parameter List... command. | The Add Parameter List dialog box is displayed. |
| 2 | Enter a Name for the parameter list to create. | – |
| 3 | From the Device Types list, select the items (for example, Digital Inputs and Outputs) that will be included in the parameter list.  As an alternative, click the Select all button. | – |
| 4 | Click the Add button. | The Add Parameter List dialog box closes.  The ParameterList editor opens showing the selected items. |

## ParameterList Editor

The ParameterList editor displays the selected parameters in a list.

It provides the following buttons to customize the list:

| Button | Description |
| --- | --- |
| Add device | Adds a new line at the bottom of the table that allows you to add a device to the list. |
| Add parameter | Adds a new column to the table that allows you to add a new parameter to the list. |
| Add missing devices | Opens the Add missing devices dialog box. You can select devices to add from the Device Types list. |
| Move device up | Moves up the selected device one line in the table. |
| Move device down | Moves down the selected device one line in the table. |
| Move parameter left | Moves the selected parameter one column to the left in the table. |
| Move parameter right | Moves the selected parameter one column to the right in the table. |
| Delete | Deletes the selected row from the table. |

To use [smart coding](D-SE-0084050.html#D-SE-0084050) for selecting devices and parameters, proceed as follows:

| Step | Action | Result |
| --- | --- | --- |
| 1 | Double-click the title cell of a parameter column.  Alternatively, click to activate it, and enter characters. | – |
| 2 | Press the Ctrl + Spacebar keys. | A list of parameters of the selected device is displayed. |
| 3 | Select the parameter you want to insert. | – |
| 4 | Press the Enter key. | The selected parameter is inserted in the table. |
