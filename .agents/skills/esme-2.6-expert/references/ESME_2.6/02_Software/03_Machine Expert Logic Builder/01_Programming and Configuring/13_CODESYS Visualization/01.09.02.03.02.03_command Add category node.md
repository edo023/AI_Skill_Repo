# Command: Add category node

Symbol: ![](images/uuid-93105fc4-29dc-535d-ea7d-7e33d8.png)

**Function**: Adds a new category at the end of the list

**Call**: **HTML5 Control Editor** menu

Double-click the new entry to open the input field so that you can specify a suitable category name. After that, you can position the category.
