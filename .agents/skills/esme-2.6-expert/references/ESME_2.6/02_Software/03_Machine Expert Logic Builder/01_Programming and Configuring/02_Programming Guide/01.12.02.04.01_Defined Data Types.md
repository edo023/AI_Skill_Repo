# Defined Data Types

## Overview

Additionally, to the standard data types, you can define special data types within a project.

You can define them via creating [DUT (Data Unit Type) objects](D-SE-0083427.html) in the POUs tree or Devices tree or within the declaration part of a POU.

See the [Best practices for the Naming of Identifiers](D-SE-0083600.html#D-SE-0083600) in order to make it as unique as possible.

See the following user-defined data types:

* [arrays](D-SE-0083672.html#D-SE-0083672)
* [vector](D-SE-0108123.html)
* [structures](D-SE-0083673.html#D-SE-0083673)
* [enumerations](D-SE-0083674.html#D-SE-0083674)
* [subrange types](D-SE-0083675.html#D-SE-0083675)
* [alias](Alias-53AD6A64.html)
