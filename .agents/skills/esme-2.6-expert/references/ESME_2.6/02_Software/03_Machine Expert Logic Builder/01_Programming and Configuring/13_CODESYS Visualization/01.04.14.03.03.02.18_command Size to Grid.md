# Command: Size to Grid

Symbol: ![](images/uuid-61b1047d-c55c-88d5-45d4-2d3f54.png)

**Function**: The command aligns the selected visualization elements to their size and position on the grid.

**Call**: **Visualization → Alignment** menu; context menu

**Requirement**: Multiple elements are selected.

TIP:

The command does not work for lines or polygons.
