# ETEST Commands

## Overview

The ETEST menu provides commands for managing the ETEST function.

ETEST is a framework to automate unit tests for program code with EcoStruxure Machine Expert.

Automated tests help to verify the correctness of the implemented functions. The consequential execution of unit tests after every modification of the program code helps to improve the quality. The ETEST framework helps simplify the definition and execution of reusable tests.

For a detailed description of the ETEST framework of the Logic Builder, refer to the [ETEST *User Guide*](../../../../../api/crossBook?lang=en-US&virtualBookName=ETEST&topicID=D_SE_0066310).
