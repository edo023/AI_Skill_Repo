# Accessing the SVN repository

SVN repository URLs

|  |  |
| --- | --- |
| `file:///` | Direct access to an SVN repository (on local hard drive) |
| `http://` | Access via WebDAV protocol to Apache server that is supported by SVN |
| `https://` | As `http://`, but with SSL encryption |
| `svn://` | Access via own protocol to an `svnserve` server |
| `svn+ssh://` | As `svn://`, but tunneled via SSH |
