# Compiler Error C0037

## Message

‘<invalid input>’ is no input of ‘<function name>’

## Message Cause

A local variable is defined in a function call.

## Solution

Declare the variable as an input parameter.

## Error Example

```
PROGRAM PLC_PRG
VAR
  i : INT;
END_VAR
```

```
i := TEST(iVar := 1);
```

```
FUNCTION TEST : INT
VAR_INPUT
  iVarx : INT;
END_VAR
```

--> C0037: 'iVar' is no input of 'TEST'

## Error Correction

```
VAR_INPUT
  iVar : INT;
END_VAR
```
