# Forward and Inverse Transformation

CODESYS SoftMotion provides function blocks for the mathematical forward and inverse transformation for many popular kinematics.

For each kinematic design, two function blocks are available for the transformation. The POU named `SMC_TRAFO_<kinematics>` calculates the axis position from the position in space (inverse transformation). The POU named `SMC_TRAFOF_<kinematics>` calculates the position in space from the axis position (forward transformation).

IMPORTANT:

You can quickly and easily link the instances of the forward transformation POUs to visualization templates in order to visualize the kinematics.
