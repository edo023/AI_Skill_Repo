# Compiler Warning C0266

## Message

Loop exit condition <condition>' is constant FALSE. Possible endless loop.

## Message Cause

In a `FOR` loop, the iteration is performed within the entire value range of the control variable.

## Solution

In accordance with the IEC 61131 standard, the statements within the `FOR` loop are executed when the value of the control variable exceeds the final value. Therefore, use a data type of sufficient length for the control variable that can map the final value +1.

## Error Example

```
PROGRAM PLC_PRG
VAR
      b : BYTE;
      i : INT;
END_VAR
FOR b := 0 TO 255 BY 1 DO
      i := i + 1;
END_FOR;
```

--> C0266: Loop exit condition 'b > 255' is constant FALSE. Endless loop possible.

Solution for the example: Decrease 255 to 254.
