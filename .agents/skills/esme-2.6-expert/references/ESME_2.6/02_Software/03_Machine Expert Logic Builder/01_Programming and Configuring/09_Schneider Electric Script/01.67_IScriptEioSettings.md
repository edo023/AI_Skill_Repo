# IScriptEioSettings

class

IScriptEioSettings.

IScriptEioSettings

Bases: object

Edge I/O NTS script object to manage grid settings on device

get\_properties\_names

Provides all properties available on this object

Returns

List of property names

Return type

IList

**Note**

This object provides dynamic properties to edit Edge I/O NTS parameters.

| Each property have its type. Supported types are : | * + bool * + int * + float * + str * + bool * + enum |

Enums are prefixed by EIO\_. Exemple : EIO\_HEAD\_CYCLE\_TIME.\_10ms

Can be used to read values :

```
nim = projects.primary.find("NTSNEC1200", True)[0]
nim.eio_settings.cycle_time
# returns EIO_HEAD_CYCLE_TIME._10ms
```

Can be used to write values :

```
nim = projects.primary.find("NTSNEC1200", True)[0]
nim.eio_settings.cycle_time = EIO_HEAD_CYCLE_TIME._4ms
```

Channel settings are group on a dedicated [IScriptEioSettings](#iscripteiosettings_module_IScriptEioSettings) :

```
ddi = projects.primary.find("NTSDDI0602", True)[0]
ddi.eio_settings.channel0.channel_enable
#returns True
ddi.eio_settings.channel0.channel_enable = false
```

Can be used to duplicate parameters of a channel to another :

```
ddi = projects.primary.find("NTSDDI0602", True)[0]
# All parameters of channel0 are copied into channel1
ddi.eio_settings.channel1 = ddi.eio_settings.channel0
```

Can be used to set parameters of a channel via python dictionary :

```
ddi = projects.primary.find("NTSDDI0602", True)[0]
ddi.eio_settings.channel0 = {'channel_enable' : True, 'filter' : 0.8}
```
