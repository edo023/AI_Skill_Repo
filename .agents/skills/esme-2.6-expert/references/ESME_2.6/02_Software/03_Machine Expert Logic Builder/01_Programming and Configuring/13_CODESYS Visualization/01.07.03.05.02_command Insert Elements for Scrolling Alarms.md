# Command: Insert Elements for Scrolling Alarms

Symbol: ![](images/uuid-f30cbda1-287e-44e9-e58f-914839.png)

**Function**: The command opens the **Alarm Banner Wizard** for adding buttons to scroll the display of the active alarms in the alarm banner.

**Call**: **Visualization** menu; context menu of the ****Alarm Banner**** visualization element

**Requirement**: An **Alarm Banner** visualization element is selected.
