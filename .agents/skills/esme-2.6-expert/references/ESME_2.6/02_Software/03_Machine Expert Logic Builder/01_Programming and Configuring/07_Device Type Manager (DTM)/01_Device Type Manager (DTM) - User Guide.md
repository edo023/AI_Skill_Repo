![](images/logo.png)

Device Type Manager (DTM) - User Guide

Device Type Manager (DTM) - User Guide

This manual provides a summary on FDT/ DTM standards and explains how to set devices configured with DTMs in association with EcoStruxure Machine Expert. This document will help you to install DTMs within EcoStruxure Machine Expert and how to access those DTMs.

For information specific to the DTMs each DTM has its own online help system.
