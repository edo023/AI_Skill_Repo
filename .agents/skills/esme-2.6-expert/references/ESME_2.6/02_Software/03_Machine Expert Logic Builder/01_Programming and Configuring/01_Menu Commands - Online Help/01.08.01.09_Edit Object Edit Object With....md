# Edit Object / Edit Object With...

## Edit Object

Use the Project > Edit Object command if you want to edit or view an object which is available in the navigators. Select the object in the navigators. The command will then be available in the contextual menu and by default in the Project menu.

The command opens the object in the appropriate editor. If used in online mode, a dialog box is displayed asking you in which view the object should be opened ([**Select Online State**](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083360) dialog box).

## Edit Object With...

If multiple editors are available for editing an object, the Edit Object With... command opens a dialog box for selecting the appropriate editor.

If only one editor is available (which is the regular case), this command works as does the Edit Object command. It opens the object in the available editor.
