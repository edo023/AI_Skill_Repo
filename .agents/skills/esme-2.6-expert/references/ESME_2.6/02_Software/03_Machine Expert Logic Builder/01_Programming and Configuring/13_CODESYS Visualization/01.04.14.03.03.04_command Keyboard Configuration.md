# Command: Keyboard Configuration

Symbol: ![](images/uuid-c2fe6144-59c3-936c-e27f-fc0a4f.png), Shortcut: **Alt**+**F6**

**Function**: The command opens and closes the **Keyboard Configuration** tab above the visualization editor.

**Call**: **Visualization** menu

**Requirement**: A visualization is open and active in the visualization editor.
