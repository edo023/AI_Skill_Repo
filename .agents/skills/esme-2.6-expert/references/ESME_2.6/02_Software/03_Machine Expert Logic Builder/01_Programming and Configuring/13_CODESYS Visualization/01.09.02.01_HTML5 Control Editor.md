# HTML5 Control Editor

Symbol: ![](images/uuid-e47a923c-4c0e-999a-9b58-b2c195.png)

**Function**: Allows you to create, edit, and localize an HTML5 control for use in CODESYS WebVisu

**Call in CODESYS** :

1. Click **Tools → Visualization Element Repository**.

   TIP:

   The command is available only when there are no projects open in CODESYS.

   * The **Visual Element Repository** dialog opens.
2. In the file selection dialog which opens, select the description file of an HTML5 control (`*.html5control.xml`).

   * The ![](images/uuid-e47a923c-4c0e-999a-9b58-b2c195.png) **HTML5 Control Editor** opens and displays the data of the element.

**Call at system level** :

Windows Start Menu > Installation folder from CODESYS > ![](images/uuid-e47a923c-4c0e-999a-9b58-b2c195.png) **HTML5 Control Editor**
