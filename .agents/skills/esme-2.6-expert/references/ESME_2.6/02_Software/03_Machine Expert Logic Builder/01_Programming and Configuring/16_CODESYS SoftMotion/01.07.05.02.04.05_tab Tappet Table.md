# Tab: Tappet Table

In this tabular table, you can also configure the tappet paths as an alternative to the graphical editor ([Tab: Tappets](_sm_obj_cam_table_tappets.html#_sm_obj_cam_table_tappets)). A tappet path defines one or more tappets depending on the master position. In the table, the lines with the definitions of the respective tappets are listed below each line that defines a tappet path.

You can switch between the tabular editor and the graphical editor at any time.
