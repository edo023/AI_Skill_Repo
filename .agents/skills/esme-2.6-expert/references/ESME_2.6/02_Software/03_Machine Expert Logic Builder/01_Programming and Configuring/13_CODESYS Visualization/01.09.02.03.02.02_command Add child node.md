# Command: Add child node

Symbol: ![](images/uuid-a35c15eb-40f4-610e-dcdb-3cc1a3.png)

**Function**: Adds a new property to the selected category

**Call**: **HTML5 Control Editor** menu

**Requirement**: A category is selected in the list.

A new property is added to the category. Double-click the new entry to open the input field so that you can specify a suitable name. After that, you can still position the property.
