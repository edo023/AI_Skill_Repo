# Go To Definition

## Overview

You can use the (Edit > Browse) command when the cursor is currently positioned on an identifier in an editor view. It will browse the project for the line or POU, which contains the definition of the corresponding POU or variable and will open the respective POU in an editor view. If the definition is within a compiled library which is included in the project, the respective POU will be opened in the Library Manager editor.

The command corresponds to the Zoom (to POU) command available in earlier versions of EcoStruxure Machine Expert and can be used in offline and in online mode. If applicable, the ([Select Online State](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083360)) dialog box will be opened. It allows you to decide whether to open the online or offline view of the POU and - in case of function or function block instances - whether you want to view the instance or implementation.

## Examples

The following POU contains a function instance definition (`fbinst`), a program call (`prog_y`) and a function call (`fbinst.out`):

```
VAR
 fbinst:fb1;
 ivar:INT;
END_VAR
prog_y();
ivar:=prog_y.y;
res1:=fbinst.out;
```

If you put the cursor on `prog_y`, the command will open program `prog_y` in the editor view.

If you put the cursor on `fbinst`, the command will set the focus up to the declaration window to the line `fbinst:fb1;`.

If you put the cursor on `out`, the command will open function block `fb1` in the editor view.
