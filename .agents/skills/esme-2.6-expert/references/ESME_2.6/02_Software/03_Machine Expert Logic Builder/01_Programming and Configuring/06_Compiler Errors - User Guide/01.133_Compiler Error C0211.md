# Compiler Error C0211

## Message

Variable declaration expected instead of <expression>

## Message Cause

Syntax error

## Solution

Make sure that the syntax is correct.

## Error Example

```
PROGRAM PLC_PRG
VAR
  VAR
```

```
  END_VAR
END_VAR
```

--> C0211: Variable declaration expected instead of VAR END\_VAR

EIO0000003933.04
