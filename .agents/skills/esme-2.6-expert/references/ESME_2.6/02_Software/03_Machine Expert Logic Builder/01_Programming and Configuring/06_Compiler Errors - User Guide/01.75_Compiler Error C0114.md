# Compiler Error C0114

## Message

Invalid destination <jump label> for ‘JMP’

## Message Cause

Syntax error or typographical error in the JMP destination.

## Solution

Correct the typographical or syntax error.

## Error Example

```
PROGRAM PLC_PRG
VAR
END_VAR
```

```
JMP 0;
```

--> C0114: Invalid destination 0 for 'JMP'

EIO0000003933.04
