# Set Branch End Point

## Presentation

The FBD/LD/IL > Set Branch End Point command is available in LD editor. It sets the end point of a branch.

The command is available if the command Set Branch Start Point has been executed previously. You can define the end point in the contextual menu or by clicking the desired position.

If you use the element branch start/end from the [toolbox](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083473) to drag the end point to the editor, the possible insert positions are indicated by light blue position markers.

For further information, refer to the chapter [*Insert Branch*](D-SE-0084134.html#D-SE-0084134).
