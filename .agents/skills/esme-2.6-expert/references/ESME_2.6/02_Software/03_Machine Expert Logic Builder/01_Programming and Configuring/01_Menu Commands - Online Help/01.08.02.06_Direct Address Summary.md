# Direct Address Summary

## Overview

Execute the command Project > Direct Address Summary to display a dialog box that lists the modules of the connected controller where direct addresses are configured (which differ from the default address). For further information, [refer to the Programming Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083399_4).
