# Add input pin

## Overview

|  |  |
| --- | --- |
| Symbol |  |
| Shortcut | - |
| Call | * Ladder > Add input pin menu * Contextual menu |

## Function

The command adds an additional input pin to an extendable [box](InsertBlock-4537C2CF.html) (ADD, OR, ADD, MUL, SEL) above the selected input pin. To insert a new input pin below the existing input pins, select the block.

EIO0000002860.10
