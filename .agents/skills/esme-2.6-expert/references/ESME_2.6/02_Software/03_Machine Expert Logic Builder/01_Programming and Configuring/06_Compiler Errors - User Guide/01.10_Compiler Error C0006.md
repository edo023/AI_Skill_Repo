# Compiler Error C0006

## Message

‘<operator>’ expected instead of ‘<token>’

## Message Cause

Syntax error

## Solution

Make sure that the syntax is correct.

## Error Example

```
PROGRAM PLC_PRG
VAR
  x: INT;
  bTest : BOOL;
END_VAR
```

```
 IF bTest
  x := 9;
END_IF
```

--> C0006: 'THEN' expected instead of 'x'

EIO0000003933.04
