# Presentation

Presentation

The Tools > Options > FDT Options dialog box allows selection of options concerning the Device Type Manager ([DTM](../glossary/glossary.htm#XREF_D_SE_0024697_680)) functionality.

FDT Options dialog box

![G-SE-0005771.2.gif-high.gif](images/G-SE-0005771.2.gif-high.gif)

This table describes the FDT Options:

| Option | Description |
| --- | --- |
| Open the default DTM [function](../glossary/glossary.htm#XREF_D_SE_0024697_714) when the editor is opened | By double-clicking the device in the Devices tree:  oIf activated: the DTM configuration tab is opened.  oIf deactivated: only the usual EcoStruxure Machine Expert tabs are opened. |
| Close all DTM functions when the editor is closed | Not used. |
| Disable undo after deleting a DTM (performance optimization) | If activated, the [Undo](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0083915_2) command is not available after deleting a DTM. Consequently, the EcoStruxure Machine Expert project size increases if a DTM is added to the project, and decreases if the DTM is subsequently deleted.  If deactivated, the Undo command is available after deleting a DTM. Consequently, the EcoStruxure Machine Expert project size increases if a DTM is added to the project, but does not decrease if the DTM is subsequently deleted. |
| Advanced settings | Opens the second FDT Options window for advanced settings. |
