# Compiler Error C0002

## Message

‘<operator 1>’ or ‘<operator 2>’ expected instead of ‘<tag>’

## Message Cause

Syntax error

## Solution

Make sure that the syntax is correct.

## Error Example

```
PROGRAM PLC_PRG
```

```
Fun(1;
```

--> C0002: ',' or ')' expected instead of ';'
