# Command: Reset to Default

**Function**: The command resets the position value of the element within the **Tab Order** to the original value.

**Call**: Element list in the visualization editor; context menu in the **Tab Order** column

The **Tab Order**, which is defined in the [element list](_visu_cmd_element_list.html#_visu_cmd_element_list), originally results from the order of the elements on the individual display layers of the visualization (numbering in the **Type** column). When positions have been changed (see [Command: Move to Position](_visu_cmd_move_to_tab_order.html#_visu_cmd_move_to_tab_order)), they can be reset by means of the **Reset to Default** command. This is possible simultaneously for a multiselection of elements.
