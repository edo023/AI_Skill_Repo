# SMC\_ScaleQueue3D (FB)

FUNCTION\_BLOCK SMC\_ScaleQueue3D

This function will stretch the path contained in poqDataIn by factor fScaleFactor.

The input variable poqDataIn is a pointer to the structure [SMC\_OUTQUEUE](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-outqueue.html#cah0ife6nckjzp5ef3qpy_filyk_smc_outqueue_smc_outqueue_fb) , which describes the path to get scaled. At first, the initialization value FALSE of input variable bEnable will bar the scaling of the path. As soon as bEnable is set to TRUE all [SMC\_GEOINFO](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-geoinfo.html#cah0ife6nckjzp5ef3qpy_filyk_smc_geoinfo_smc_geoinfo_struct) objects found in poqDataIn will be processed. When bEnable is reset to FALSE, the module will not execute any further modifications.

The initialization of input variable bReset with FALSE effects, that the [SMC\_GEOINFO](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-geoinfo.html#cah0ife6nckjzp5ef3qpy_filyk_smc_geoinfo_smc_geoinfo_struct) objects currently found in poqDataIn will not get rotated, but only those that will be added.
