# ST\_ToolEndButtons

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.1.0.0 |
| Inherits from: | - |

## Description

This structure contains the status of tool end buttons of the Lexium Cobot arm.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| xPoint | BOOL | The **POINT** button of the Lexium Cobot arm is addressed. |
| xFree | BOOL | The **FREE** button of the Lexium Cobot arm is addressed. |
| xPlayPause | BOOL | The **PLAY/PAUSE** button of the Lexium Cobot arm is addressed. |
