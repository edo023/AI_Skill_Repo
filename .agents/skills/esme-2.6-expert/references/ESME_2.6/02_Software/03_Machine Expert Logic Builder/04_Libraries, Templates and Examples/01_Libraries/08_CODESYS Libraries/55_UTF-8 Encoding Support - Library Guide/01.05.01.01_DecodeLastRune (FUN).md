# DecodeLastRune (FUN)

FUNCTION DecodeLastRune : [RUNE](Tdg6Ql3l9Me28nf8SHpLBXgb98M_rune.html#tdg6ql3l9me28nf8shplbxgb98m_rune_rune_alias)

Unpacks the last UTF-8 encoding in pbyData and returns the rune and its position (udiIndex) and length (udiLength) in bytes.

If pbyData is empty it returns ([c\_diRuneError](bsynvdeBnJ4tsWpfW9WWyKK0oOk_constants.html#bsynvdebnj4tswpfw9wwykk0ook_constants_constants_gvl__bsynvdebnj4tswpfw9wwykk0ook_constants_constants_c_diruneerror), udiLength=0). Otherwise, if the encoding is invalid, it returns ([c\_diRuneError](bsynvdeBnJ4tsWpfW9WWyKK0oOk_constants.html#bsynvdebnj4tswpfw9wwykk0ook_constants_constants_gvl__bsynvdebnj4tswpfw9wwykk0ook_constants_constants_c_diruneerror), *udiLength=1`*). Both are impossible results for correct, non-empty UTF-8.

An encoding is invalid if it is incorrect UTF-8, encodes a rune that is out of range, or is not the shortest possible UTF-8 encoding for the value. No other validation is performed.
