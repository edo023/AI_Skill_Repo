# i_wDrvSpdRefAna

i\_wDrvSpdRefAna

This input pin allows you to control the speed of the drive using an analog input channel. It is the speed reference from analog input device like a joystick. i\_wDrvSpdSel input must be set to zero to select analog input as reference.

NOTE: You should scale the analog reference input to RPM units. The ScaleInput function block can be used for this purpose.
