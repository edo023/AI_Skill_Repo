## Simple One-To-One Relationship

* One action ([IAsyncActionProvider](CZue5CN2wzRfuhta_JgtNFmpobk_iasyncactionprov.html#czue5cn2wzrfuhta_jgtnfmpobk_iasyncactionprov_iasyncactionprovider_itf)) is assigned to one background task.
* The current foreground task fills the parameter queue with parameter sets.
* The result of the background task is available via the AsyncResult property.
