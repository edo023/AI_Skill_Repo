# CRC32 (FUN)

FUNCTION CRC32 : DWORD

Function calculates the CRC32 checksum used in PKZIP, AUTODIN II, Ethernet, FDDI.

Example:

```
sTest : STRING(10) := '123456789' ;
->
MEM.CRC32(ADR(sTest), 9) = 16#CBF43926
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | CRC32 | DWORD | function returns the CRC32 checksum of “MemoryBlock” | | Input | pMemoryBlock | POINTER TO BYTE | address of memory Block | | uiLength | UINT | length of MemoryBlock in byte | |
