# Overview

Overview

The Module Manager is used to manage the versions of the Smart Template modules.

The Module Manager is available beneath the Application (root node) in the Modules view (View > Modules). It is automatically inserted by adding the first Smart Template module and cannot be removed as long at least one module is available in your project.

| Step | Action |
| --- | --- |
| 1 | Double-click the Module Manager in the Modules view to open it.  Result: The Smart Template modules available in your project are listed with:  oTitle  oVendor  oVersion (used at present) |
| 2 | In the Version column of the module you want to edit, open the list of available versions and select the version you want to use in your project.  NOTE: Versions that are not compatible with your project are displayed in red. |
