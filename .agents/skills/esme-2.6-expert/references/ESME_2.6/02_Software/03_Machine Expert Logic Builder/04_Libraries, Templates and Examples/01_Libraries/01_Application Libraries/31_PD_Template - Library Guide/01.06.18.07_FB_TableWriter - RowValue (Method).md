# FB\_TableWriter - RowValue (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | 1.1.2.0 |
| Versions: | Current version |

## Task

Writing the content of individual cells of a row

Each cell must be prepared for writing values using RowStart() and finished using RowEnd().

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiColumnNumber | UDINT | ID of the column that was returned by HeaderAddColumn() |
| i\_sValue | STRING(255) | Cell content displayed in PacDrive Diagnostics |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | General, library-independent statement on the diagnostic.  A value unequal to GD.ET\_Diag.Ok corresponds to an error message. |
| q\_etDiagExt | ET\_DiagExt | POU-specific output on the diagnostic.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> error message |
| q\_sMsg | STRING(80) | Event-triggered message which gives more detailed information on the diagnostic state. |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Returns TRUE if successful, FALSE in the case of errors |
