# Functional Description

Functional Description

The function block FB\_ParameterUpload is used to read the values of the parameters defined in the [Index File](../Specifications/Specifications-2.htm#XREF_D_SE_0081722_1) from the drive. The values are transferred to the controller and saved to the [Parameter File](../Specifications/Specifications-4.htm#XREF_D_SE_0081727_1)  in the controller.

To read or write CSV files from or to the controller, the library FileFormatUtility is used. For further information, refer to the CSV Function Blocks chapter of the [FileFormatUtility Library Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=formatLi&topicID=D_SE_0080776_1).
