# Description

Description

Clock-pulse controlled RS flip flop with reset priority.

If the clock pulse is TRUE, the clock-pulse controlled RS flip flop with reset priority behaves as follows:

i\_xS = 0 , i\_xR = 0 -> q\_xQ = unchanged

i\_xS = 0 , i\_xR = 1 -> q\_xQ = 0 (flip flop is reset)

i\_xS = 1 , i\_xR = 0 -> q\_xQ = 1 (flip flop is set)

i\_xS = 1 , i\_xR = 1 -> q\_xQ = 0 (flip flop is reset)
