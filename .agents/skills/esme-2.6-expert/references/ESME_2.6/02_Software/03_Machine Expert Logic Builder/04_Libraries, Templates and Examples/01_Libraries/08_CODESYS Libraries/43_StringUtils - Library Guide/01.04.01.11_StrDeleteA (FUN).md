# StrDeleteA (FUN)

FUNCTION StrDeleteA

Deletes |iLength| characters of |pby|, beginning at the |iPosition|-th character position.

**Note**

|iPosition| = 1 is the first character.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | pby | POINTER TO BYTE | Pointer to the string that shall be edited | | iLength | INT | Number of characters to be deleted | | iPosition | INT | Position within the string where deletion shall start | |
