# MessageToSend (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.2.0 |
| Inherits from: | - |

## Description

When the interface is registered as callback, this method is executed before the message is sent to the proxy server. It allows to inspect and to modify the message.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_pbyMessage | POINTER TO BYTE | Pointer to the message which is ready to send. |
| i\_uiActLength | UINT | Indicates the actual length of the message to send. |
| i\_uiMaxLength | UINT | Indicates the maximum possible length of the message to send. |

NOTE: The input i\_uiMaxLength indicates the size of the buffer the pointer i\_pbyMessage points to. To help prevent an access violation while a read or write access to the message is performed via the pointer, use the value of i\_uiMaxLength to limit the buffer size.

Example:

```
SysMem.SysMemCpy(pDest := ADR(sMessage), pSrc := i_pbyMessage, udiCount := MIN(SIZEOF(sMessage), i_uiMaxLength));
```
