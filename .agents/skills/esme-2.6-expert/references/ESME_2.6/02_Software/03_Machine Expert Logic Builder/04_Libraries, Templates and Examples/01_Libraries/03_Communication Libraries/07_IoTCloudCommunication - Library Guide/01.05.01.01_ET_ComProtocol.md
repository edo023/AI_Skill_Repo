# ET\_ComProtocol

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_ComProtocol is used to specify the communication protocol to be used by the communication client.

The conversion operator TO\_STRING is supported for this enumeration.

## Enumeration Elements

| Name | Value (INT) | Description |
| --- | --- | --- |
| NotSet | 0 | Protocol is not set. |
| Https | 1 | HTTPS is selected. |
| Mqtts | 2 | MQTTS is selected. |
