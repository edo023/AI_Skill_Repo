# Description

Description

This section provides a short code example which is used to move a vehicle to a new destination (position-based).

Assuming the MagneMotion system is ready to use, you have to prepare a command as represented in the following:

astVehicle[1].i\_etCommand := TSC.ET\_VehicleCommand.MoveToPosition;   
astVehicle[1].iq\_stDestination.i\_etMovementDirection := TSC.ET\_MovementDirection.Downstream;   
astVehicle[1].iq\_stDestination.i\_rAcceleration := 1.0;   
astVehicle[1].iq\_stDestination.i\_rVelocity := 1.0;   
astVehicle[1].iq\_stDestination.i\_rPosition := 0.5;   
astVehicle[1].iq\_stDestination.i\_uiId := 1;   
astVehicle[1].i\_xExecute := TRUE;
