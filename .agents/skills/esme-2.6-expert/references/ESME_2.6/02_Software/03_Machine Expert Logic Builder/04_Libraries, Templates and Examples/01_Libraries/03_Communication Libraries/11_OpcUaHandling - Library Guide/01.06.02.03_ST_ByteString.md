# ST\_ByteString

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.2.4.0 |
| Inherits from: | - |
| Attribute: | ``` 'pack_mode' := '1' ``` |

## Description

The structure ST\_ByteString is used to assign the variable specified inside your application to an argument of type ByteString of a method.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| pbyBuffer | POINTER TO BYTE | Address of the buffer allocated in the application. |
| udiBufferSize | UDINT | Size of the buffer allocated in your application.  NOTE: Use the `SIZEOF` operator to get the size of the variable inside the application. |
