# LimitAlarm\_LREAL (FB)

FUNCTION\_BLOCK LimitAlarm\_LREAL

This function block checks whether the value ![](images/423ccff0d86c4c1bb2fd9ea24dc9f639.jp.jpg) lies between an lower bound ![](images/d79dabca5ad54ff9a999b2a1c6e56b56.jp.jpg) and an upper bound ![](images/a6bfa09f1a034a85a0ecbc9a9b5b804a.jp.jpg).

The output value will be set according to the result.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | lrValue | LREAL | value  to be tested | | lrLimitPos | LREAL | upper bound | | lrLimitNeg | LREAL | lower bound | | Output | xExceedsPos | BOOL | TRUE: If  FALSE: Else | | xExceedsNeg | BOOL | TRUE: If  FALSE: Else | | xInLimits | BOOL | TRUE: If  FALSE: Else, also in the atypical case | |
