# ET\_DiagExt - General Information

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |
| Versions: | Installed version |

## Description

The enumeration ET\_DiagExt specifies the POU-specific diagnostic and status messages.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Disabled | 0 | The POU is disabled. |
| Initializing | 1 | The POU is being initialized. |
| Working | 2 | The POU is active. |
| WaitForExecute | 3 | POU waits for the execution. |
| WaitForSercos | 4 | POU waits for Sercos. |
| Executing | 5 | Version |
| TorqueFeedForwardConfigured | 6 | The torque feed forward is configured. |
| DriveDoesNotSupportThisFunction | 7 | The drive does not support the function. |
| SercosCommunicationNotPossible | 8 | Sercos communication not possible. |
| ReadingConfigList | 10 | The configuration list is being read. |
| ParameterNotConfigured | 11 | The parameter is not configured. |
| Ok | 12 | Ok |
| ConfigurationNotDone | 13 | The configuration has not been performed. |
| WaitForConfiguration | 14 | POU waits for the configuration. |
