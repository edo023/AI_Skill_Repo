# Time and Date

Modules of this library support reading and setting of the realtime clock. These modules, however, might not take into account the time zone and a possible switch between summer and standard time. If an automatic switch is desired all the same, the function blocks of the CAA\_DTUtility library can be used for reading and setting the realtime clock. In this case the time switch is done via the CAA\_DTUtility library. If the runtime system supports the automatic switch between summer and standard time, the calls of the library will be forwarded directly to the corresponding functions of library CAA\_RealTimeClock and the clock switch will be done by the runtime system.
