# IF\_MultiAxisCustomJobCallback

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The interface IF\_MultiAxisCustomJobCallback is the central extension point of the library. It is designed to be implemented by a user-defined function block containing the motion logic for the multi-axis custom job.

The methods of this interface are called by the function block [FB\_MultiAxisCustomJob](FB_MultiAxisCustomJob-4CFA57F9.html) at specific points during the initialization and execution of the multi-axis group. This allows the user algorithm to run within the RTP cycle of the controller.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).

## Methods

| Name | Description |
| --- | --- |
| [Calculate](IF_MultiAxisCustomJobCallback-Calcu-4D67945D.html#IF_MultiAxisCustomJobCallback-Calcu-4D67945D) | Called cyclically in every RTP cycle to perform the custom motion calculations and write target values to the subordinate axes. |
| [Prepare](IF_MultiAxisCustomJobCallback-Prepa-4D679954.html#IF_MultiAxisCustomJobCallback-Prepa-4D679954) | Called once during the initialization phase to set up the custom algorithm before the cyclic execution begins. |
