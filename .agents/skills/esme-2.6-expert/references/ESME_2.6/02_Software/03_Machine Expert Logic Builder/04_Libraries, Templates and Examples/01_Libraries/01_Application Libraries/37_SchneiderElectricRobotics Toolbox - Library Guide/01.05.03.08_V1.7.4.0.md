## New Features

* Enumeration elements User3Ax and User6Ax added to ET\_RobotType. The elements can now be selected as robot type in ST\_RobotEntityData.etType.
* IF\_EntitiesHandler now EXTENDS \_\_System.IQueryInterface.

## Modifications

ST\_RobotEntityData.etType is now verified during the call of FB\_EntitiesHandler.AddRobot().

An invalid value is reported by the diagnostic message q\_etDiag = GD.ET\_Diag.InputParameterInvalid / q\_etDiagExt.RobotTypeInvalid.

## Mitigated Anomalies

ROB.ET\_OrientationConvention.ZXY is fully supported now.

It can be selected as orientation convention on target generation and on adding robots and conveyors to the function block FB\_EntitiesHandler now.
