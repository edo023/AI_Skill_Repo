## Description

The structure ST\_FeedbackControlStandardStation contains feedback on the methods of the interface [IF\_ControlStandardStation](CtrlGroupStation-EED16FBE.html#CtrlGroupStation-EED16FBE).

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| xDriveProcessEmptyExecuted | BOOL | Indicates TRUE if the carriers in the process have received the command to leave the station. |
| xStopProcessExecuted | BOOL | Indicates TRUE if the carriers on the way to the process have received the stop command. |
| xHoldProcessActive | BOOL | Indicates TRUE if the process is on hold and no carrier is sent into the process. |
| xWaitForControlCommand | BOOL | Indicates TRUE if the function block is on hold and waits for the control command [Restart](Restart-EED689B5.html#Restart-EED689B5), [DriveProcessEmpty](DriveProcEmpty-EED35245.html#DriveProcEmpty-EED35245) or [StopProcess](StopProcess-EED58DEE.html#StopProcess-EED58DEE). |
