## Task

The FB\_UserManagement function block is used to manage user accounts for SNMPv3 communications.

## Functional Description

The FB\_UserManagement function block is used to manage the user accounts for the user-based security model for version 3 of the Simple Network Management Protocol (SNMPv3). It needs to be instantiated inside the user application. To preserve the user data which are saved in an array inside the function block instance after a controller reset, this can also be performed inside a retain memory.

The function block implements the [IF\_UserManagement](IF_UserMan-7F53CD30.html) interface. Therefore, it can be referenced by the [FB\_SnmpManager](D-SE-0081046.html) function block using the i\_ifUserManagement input.
