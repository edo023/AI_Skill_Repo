# ET\_StateComClient

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_StateComClient is used to indicate the state of the communication client.

The conversion operator TO\_STRING is supported for this enumeration.

## Enumeration Elements

| Name | Value (INT) | Description |
| --- | --- | --- |
| Idle | 0 | Idle state, the function block is ready for operation. |
| DnsClientInProgress | 1 | The integrated DNS client is in progress. |
| Connecting | 2 | The connection to the server is in progress. |
| Connected | 3 | The client is connected to the server. |
| Disconnecting | 4 | The disconnection from the server is in progress. |
| WaitForAutoReconnect | 5 | Attempt to reconnect is pending and is performed after the configured waiting time. |
| Initializing | 6 | The initializing of the function block is in progress. |
| Error | -1 | An error was detected. |
