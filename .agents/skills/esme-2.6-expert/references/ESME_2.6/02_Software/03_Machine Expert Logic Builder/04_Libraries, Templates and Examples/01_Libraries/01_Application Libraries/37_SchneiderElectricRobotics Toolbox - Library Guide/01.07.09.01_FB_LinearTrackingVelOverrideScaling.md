## Task

Scale the velocity override of a robot based on a reference target position and on the state of a linear tracking system.

## Description

This function block can be used during the motion of a robot toward a target in a linear tracking system. The aim of the function block is to scale the velocity override of the robot based on the reachability of the target.

* The supported tracking configurations for this feature are linear tracking and tracking source. Rotative tracking is not supported.
* If the selected tracking system is configured as a tracking source, the linear motion is assumed to be along the X direction and only the X component of the velocity is considered. So, the motion implemented by the tracking source should be linear and limited along the X direction of the tracking system.
