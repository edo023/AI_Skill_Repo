# IReconfigureProvider.ReconfigureAsync (METH)

METHOD ReconfigureAsync : [PROC\_STATE](Wk7sVif4qdICA8XUp7Q627WapoY_proc-state.html#wk7svif4qdica8xup7q627wapoy_proc_state_proc_state_enum)

Reconfigure reads the configuration of the corresponding node and applies it. Afterwards it calls ReconfigureAsync for all child [IReconfigureProvider](qG5KZxShRzJP0Witj1Yz7sQso84_ireconfigureprov.html#qg5kzxshrzjp0witj1yz7sqso84_ireconfigureprov_ireconfigureprovider_itf). After Reconfiguring, all changed configuration values (Enable flag, Parameter changes,…) take effect. If the node doesn’t support reconfiguring, method returns [PROC\_STATE.ERROR](Wk7sVif4qdICA8XUp7Q627WapoY_proc-state.html#wk7svif4qdica8xup7q627wapoy_proc_state_proc_state_enum__wk7svif4qdica8xup7q627wapoy_proc_state_proc_state_error) and eError contains [ERROR.NOT\_SUPPORTED](Wk7sVif4qdICA8XUp7Q627WapoY_error.html#wk7svif4qdica8xup7q627wapoy_error_error_enum__wk7svif4qdica8xup7q627wapoy_error_error_not_supported).
