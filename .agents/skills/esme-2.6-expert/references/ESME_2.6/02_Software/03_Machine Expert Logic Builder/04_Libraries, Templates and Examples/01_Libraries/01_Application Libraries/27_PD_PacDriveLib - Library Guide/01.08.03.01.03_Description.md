# Description

Description

JK flip flop with Set and Reset inputs.

If i\_xSet is set, output q\_xQ is forced to TRUE independently of the clock pulse. If i\_xReset is set, output q\_xQ is forced to FALSE independently of the clock pulse. If the inputs i\_xSet and i\_xReset are equal, output q\_xQ is set as defined by the JK flip flop.

In the case of a rising edge of the clock pulse, the JK flip flop shows the following behavior:

i\_xJ = 0 , i\_xK = 0 -> q\_xQ = unchanged

i\_xJ = 0 , i\_xK = 1 -> q\_xQ = 0 (flip flop is reset)

i\_xJ = 1 , i\_xK = 0 -> q\_xQ = 1 (flip flop is set)

i\_xJ = 1 , i\_xK = 1 -> q\_xQ = State is changed (toggle)
