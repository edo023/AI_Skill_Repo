# EncoderInvalid

EncoderInvalid

|  |  |
| --- | --- |
| Enumeration name: | EncoderInvalid |
| Enumeration value: | 51 |
| Description: | The Encoder is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The logical encoder of the axis at the input Master is invalid. | It must be made sure that the axis at the input Master was initialized with the POU [FC\_DriveInit](../Functions/Functions-2.htm#XREF_D_SE_0090869_1).  When calling up the POU [FC\_DriveInit](../Functions/Functions-2.htm#XREF_D_SE_0090869_1) for the axis at the input Master a logical encoder must have been indicated. |
