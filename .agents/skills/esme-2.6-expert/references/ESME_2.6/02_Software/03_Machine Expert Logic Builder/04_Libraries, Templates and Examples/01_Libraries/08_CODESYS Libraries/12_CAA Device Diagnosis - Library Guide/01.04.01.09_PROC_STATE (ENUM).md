# PROC\_STATE (ENUM)

TYPE PROC\_STATE :

This enumeration is used to issue the current operation state of asynchronous methods.

| Attributes: | qualified\_only |
| InOut: | | Name | Initial | Comment | | --- | --- | --- | | DORMAND | 0 | The operation is in IDLE state. | | BUSY |  | The operation is busy. | | DONE |  | The operation is done. | | ABORTED |  | The operation has been aborted. | | ERROR |  | An error occured. | |
