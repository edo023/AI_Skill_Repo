# MaxNumberOfLoggerPointsExceeded

MaxNumberOfLoggerPointsExceeded

|  |  |
| --- | --- |
| Enumeration name: | MaxNumberOfLoggerPointsExceeded |
| Enumeration value: | 28 |
| Description: | Maximum number of logger points is exceeded. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | It was attempted to register more than APL.Gc\_udiMaxNumberOfLoggerpoints logger points in the Application Logger. | The global parameter APL.Gc\_udiMaxNumberOfLoggerPoints must be increased. |
