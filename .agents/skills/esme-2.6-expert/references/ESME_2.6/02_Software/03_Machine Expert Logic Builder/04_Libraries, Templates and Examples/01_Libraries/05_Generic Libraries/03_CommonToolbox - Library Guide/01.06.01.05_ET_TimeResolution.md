# ET\_TimeResolution

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.2.9.0 |

## Description

The enumeration ET\_TimeResolution specifies the type of resolution that is used for indicating the values provided by the function blocks[FB\_RuntimeMeasurement](D-SE-0105432.html) and [FB\_RuntimeMeasurement2](FB_RunTimeMeasurement2-482D7DA6.html).

## Enumeration Elements

| Name | Data type | Value | Description |
| --- | --- | --- | --- |
| Nanoseconds | INT | 0 | The measured value is indicated in nanoseconds. |
| Microseconds | INT | 1 | The measured value is indicated in microseconds. |
| Milliseconds | INT | 2 | The measured value is indicated in milliseconds. |
| Seconds | INT | 3 | The measured value is indicated in seconds. |
