# IsValid (FUN)

FUNCTION IsValid : BOOL

Reports whether pbyData consists entirely of valid UTF-8-encoded runes.

If the checked area contains only characters with Unicode less than 16#80 (ASCII), the output xASCII is set accordingly

| InOut: | | Scope | Name | Type | Initial | Comment | | --- | --- | --- | --- | --- | | Return | IsValid | BOOL |  |  | | Input | pbyData | POINTER TO BYTE |  | Pointer to the first byte of the memory | | udiSize | UDINT |  | Size of the memory in bytes | | Output | xASCII | BOOL | TRUE |  | |
