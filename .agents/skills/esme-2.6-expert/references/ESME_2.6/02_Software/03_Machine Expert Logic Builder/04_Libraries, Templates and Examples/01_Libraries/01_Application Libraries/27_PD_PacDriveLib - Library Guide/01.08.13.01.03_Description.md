# Description

Description

The function implements a P controller. A positive signal at i\_xStart starts the function block. The input signal i\_lrInput is calculated from the difference i\_lrInput := (reference value - actual value). i\_lrKP is to be chosen according to the desired gain. The output control value is calculated according to the following formula:

q\_lrOutput:= i\_lrKP \* i\_lrInput
