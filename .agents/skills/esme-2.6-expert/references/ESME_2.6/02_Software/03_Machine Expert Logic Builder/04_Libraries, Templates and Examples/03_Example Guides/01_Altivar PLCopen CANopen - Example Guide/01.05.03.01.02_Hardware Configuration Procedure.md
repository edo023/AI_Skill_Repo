# Hardware Configuration Procedure

Hardware Configuration Procedure

| Step | Action | Comment |
| --- | --- | --- |
| 1 | In the Devices tree, double-click the CAN interface and configure the parameter in the device editor. | Configure the transmission rate for the CANopen communication. |
| 2 | In the Devices tree, add the CANopen Manager under the CAN interface of the controller. | Execute the Add Device … command from the contextual menu of the CAN interface. In the Add Device dialog box, double-click the CANopen Manager to add it to the project. |
| 3 | Select the CANopen Manager node in the Devices tree. | The Add Device dialog box is updated and lists the devices which can be added. |
| 4 | Double-click the Altivar 320 to add it under the CANopen Manager node. | – |
| 5 | Double-click the Altivar 340 to add it under the CANopen Manager node. | – |
| 6 | Double-click the Altivar 6xx to add it under the CANopen Manager node. | – |
| 7 | Double-click the Altivar 9xx to add it under the CANopen Manager node. | – |
| 8 | Close the Add Device dialog box. | – |
| 9 | In the Devices tree, double-click the previously added device nodes. Adjust the CANopen address for each drive in the respective device editor. | Further configuration of the devices is not necessary for the described application example. |
