# Rename (FB)

FUNCTION\_BLOCK Rename EXTENDS CBM.ETrig

This function block modifies the file name and is optional for the different targets.

Contrary to the original CAA specification the file may be renamed even if it is opened via ”FILE.[Open](VIPLMVOpanJfjqQXFKWGQlzGjxw_open.html#viplmvopanjfjqqxfkwgqlzgjxw_open_open_fb_prefix_filop)”.

This behavior is dependent of the underlying operating system and file system.

Whether this function block is implemented or not is explicitly stated and encoded in the return value of the function ”FILE.[GetProperty](RFfZ6qcMtbnrwad-bdR1UipBxBA_getproperty.html#rffz6qcmtbnrwad_bdr1uipbxba_getproperty_getproperty_fun)”.

The function block cannot be used to move directories!
