# UserMgrObjectGetGroupRights (FUN)

FUNCTION UserMgrObjectGetGroupRights : RTS\_IEC\_RESULT

Get the rights and the denied rights of a group on a specified object

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | UserMgrObjectGetGroupRights | RTS\_IEC\_RESULT | | Error code | * ERR\_OK: Access rights could be retrieved * ERR\_PARAMETER: Invalid handles * ERR\_NO\_ACCESS\_RIGHTS: User not authorized | | | Input | hUser | RTS\_IEC\_HANDLE | Handle to authorized user | | hObject | RTS\_IEC\_HANDLE | Handle to the object | | hGroup | RTS\_IEC\_HANDLE | Handle to the group which access rights are requested | | Output | ulRights | DWORD | Rights of the group. See category “User rights” for details | | ulDeniedRights | DWORD | Denied rights of the group. See category “User rights” for details | |
