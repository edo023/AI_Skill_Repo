# Description

Description

Based on the array i\_astCamSegments, this method generates a segmented cam with i\_diNumberOfSegments segments. Ensure that the values [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrXEnd and [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrYEnd of a segment correspond to the values [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrXStart and [ST\_Cam](../Structures/Structures-2.htm#XREF_D_SE_0089480_1).lrYStart of the following segment.
