## Task

The method RegisterCallbackEnhancedAuthentication allows for registering a callback to process enhanced authentication.

## Functional Description

If an AUTH message is received, the method EnhancedAuthentication specified by the interface IF\_EnhancedAuthentication is called. Inside the method, it is possible to process the information contained in the received AUTH message and to specify the information to send to the MQTT broker in a subsequent AUTH message.
