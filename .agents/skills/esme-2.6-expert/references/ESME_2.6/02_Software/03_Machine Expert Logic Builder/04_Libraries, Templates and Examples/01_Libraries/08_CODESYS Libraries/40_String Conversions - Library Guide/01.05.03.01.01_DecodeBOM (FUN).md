# DecodeBOM (FUN)

FUNCTION DecodeBOM : UDINT

Attempts to recognize a *BOM* in the first bytes of byData and then returns the corresponding [ENCODING](QS9IiJwK2G4tvWpsSL8DUp7vIyA_encoding.html#qs9iijwk2g4tvwpssl8dup7viya_encoding_encoding_alias) and its length in bytes.
