# IF\_EnhancedAuthentication

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V2.4.4.0 |

## Description

The interface IF\_EnhancedAuthentication is implemented by a function block defined in the application. It can be passed to the function block FB\_MqttClient through the method RegisterCallbackEnhancedAuthentication to register a callback and process enhanced authentication. If registered, the method EnhancedAuthenticaton is called by the FB\_MqttClient each time the MQTT broker sends an AUTH message.
