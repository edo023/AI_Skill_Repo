## Task

Setting a set of parameters for a specific tracking system.

## Description

With the method SetTrackingParameters(…), a set of parameters can be applied to a specific tracking system.

Also refer to [*Minimizing the Influence of a Tracking System on the Trajectory of a Robot*](D-SE-0101026.html#D-SE-0101026).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etTrackingSystem | [ET\_CoordinateSystem](D-SE-0075477.html#D-SE-0075477__D-SE-0075477.4) | Tracking system for which the parameters should be set.  Valid value: Identifier of a configured tracking system. |
| i\_stParameters | [ST\_TrackingParameters](D-SE-0100785.html#D-SE-0100785__D-SE-0100785.4) | Parameters that should be applied to the tracking system transferred at input i\_etTrackingSystem. |
