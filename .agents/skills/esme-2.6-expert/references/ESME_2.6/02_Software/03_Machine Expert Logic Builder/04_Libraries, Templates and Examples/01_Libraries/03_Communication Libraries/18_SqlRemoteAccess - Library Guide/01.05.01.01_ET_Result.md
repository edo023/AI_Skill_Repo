# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result contains the possible values that indicate the result of operations executed by the function block.

## Enumeration Elements

| Name | Data type | Value | Description |

| If q\_xError of a [function block](D-SE-0080901.html#D-SE-0080901) is FALSE, the following status message is shown. | | | |
| Ok | UDINT | 0 | Operation completed successfully. |

| Disabling | UDINT | 3 | Ongoing disabling process of the function block (only set by FB\_SqlDbRequest). |
