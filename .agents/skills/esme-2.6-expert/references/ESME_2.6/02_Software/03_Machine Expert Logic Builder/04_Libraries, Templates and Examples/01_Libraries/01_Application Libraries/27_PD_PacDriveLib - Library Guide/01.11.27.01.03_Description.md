# Description

Description

Calculates the absolute extreme of the grade 5 polynomial and its 1st and 2nd derivation to the interval [i\_lrX0, i\_lrX1] given by the coefficients array i\_stCoefficients ("curve tracing").

If the absolute extreme values of a polynomial, which meets specific boundary conditions, are to be calculated, the coefficient array of the polynomial can first be calculated by means of the function [FC\_Poly5Coefficients](Functions_K_to_Z-28.htm#XREF_D_SE_0087591_1).
