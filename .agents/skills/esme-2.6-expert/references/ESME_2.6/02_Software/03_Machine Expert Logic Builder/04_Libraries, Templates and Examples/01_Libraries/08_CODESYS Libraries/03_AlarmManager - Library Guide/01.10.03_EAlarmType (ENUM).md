# EAlarmType (ENUM)

TYPE EAlarmType :

This enumeration is used to distinguish between alarms and events.

| Attributes: | qualified\_only |
| InOut: | | Name | Initial | Comment | | --- | --- | --- | | Undefined | 0 | Undefined, resp. represents the absence of a value | | Alarm | 1 | An alarm. | | Event | 2 | An event, that must be acknowledged by the user (has acknoledge method ACK) | | ImplicitlyAcknowledgedEvent | 3 | An event with acknowledge method REP, that is implicitly acknowledged and only written into the alarm storage (not visible in the list of active alarms). | | AlarmAPI | 4 | An alarm that can only be raised programmatically | |
