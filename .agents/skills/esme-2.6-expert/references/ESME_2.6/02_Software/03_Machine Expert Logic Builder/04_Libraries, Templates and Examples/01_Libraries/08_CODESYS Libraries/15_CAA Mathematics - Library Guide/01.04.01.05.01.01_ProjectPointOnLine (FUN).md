# ProjectPointOnLine (FUN)

FUNCTION ProjectPointOnLine : BOOL

This function will project the point ![](images/162a232a49e14f7a982303beb260c1bf.jp.jpg) onto a straight line: thereby, the direction vector of the straight line is taken as normal vector of a plane that is moved along the line as long as it touches the point ![](images/85b7d2eefdc94ab6acec49d1f112c93e.jp.jpg). The intersection point of the plane with the line corresponding to this location is the projected point.
