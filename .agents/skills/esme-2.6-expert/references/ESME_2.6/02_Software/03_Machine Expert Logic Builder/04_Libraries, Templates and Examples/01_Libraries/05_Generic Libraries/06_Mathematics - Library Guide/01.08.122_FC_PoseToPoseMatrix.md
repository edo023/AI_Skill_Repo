# FC\_PoseToPoseMatrix

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

The function converts from a Cartesian pose represented as (X, Y, Z) and (roll, pitch, yaw) to the same Cartesian pose represented as a homogeneous transformation matrix.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stPose | [ST\_CartesianPose](ST_CartesianPose-GeneralInformation-10190D63.html#ST_CartesianPose-GeneralInformation-10190D63) | A Cartesian pose. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |

## Return Value

| Data type | Description |
| --- | --- |
| [ST\_Matrix4D](ST_Matrix4D-GeneralInformation-0C050CBD.html#ST_Matrix4D-GeneralInformation-0C050CBD) | The function returns the conversion from a Cartesian pose represented as (X, Y, Z) and (roll, pitch, yaw) to the same Cartesian pose represented as a homogeneous transformation matrix. |
