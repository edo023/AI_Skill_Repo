The negative limit switch ([ST\_Main.i\_xHwLimitNeg](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) = TRUE) is still active. However, a Touchprobe signal has been detected or the maximum travel distance ([ST\_Home](../Structures/Structures-5.htm#XREF_D_SE_0077219_1).stTouchProbe.i\_lrMaxTravel) has been reached. | Check the signal at the [ST\_Main](../Structures/Structures-7.htm#XREF_D_SE_0077223_1).i\_xHwLimitNeg input.  Check the value for the maximum travel distance [ST\_Home](../Structures/Structures-5.htm#XREF_D_SE_0077219_1).stTouchProbe.i\_lrMaxTravel.  Ensure that the sensor of the negative limit switch is not dirty. |
