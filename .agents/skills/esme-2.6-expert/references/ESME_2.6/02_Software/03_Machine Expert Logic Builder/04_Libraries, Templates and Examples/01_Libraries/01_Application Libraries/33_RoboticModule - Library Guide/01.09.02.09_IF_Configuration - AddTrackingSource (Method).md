## Task

Adding a configured tracking source to the robot.

## Description

With the method AddTrackingSource(…), a tracking source can be added to the robot.

The tracking sources must be added to the robot before the method IF\_Configuration.ConfigDone is called.

NOTE: The usage of tracking sources cannot be combined with linear and rotative tracking systems. It is not possible to add these systems when a tracking source was configured and it is not possible to use a tracking source when linear or rotative tracking system has already been defined.

The following methods are not compatible and will return an exception when called after the configuration of a tracking source:

* IF\_Configuration.AddLinearTrackingSystem: Configure a tracking source and use it instead.
* IF\_Configuration.AddLinearTrackingSystem2: Configure a tracking source and use it instead.
* IF\_Configuration.AddLinearTrackingSystem3: Configure a tracking source and use it instead.
* IF\_Configuration.AddRotativeTrackingSystem: Configure a tracking source and use it instead.
* IF\_Configuration.SetTrackingStopParameters: Configure the tracking motion parameters for each tracking source with the method IF\_TrackingSource.SetMotionParameters.
* IF\_ConfigurationAdvanced.SetTrackingParameters: Configure the tracking parameters for each tracking source with the method IF\_TrackingSource.SetOptionalParameters.
* ROB.IF\_RobotMotion.SetMaxAccelerationResultant: Configure the maximum acceleration with the input i\_lrMaxAcceleration of the method IF\_TrackingSource.SetMotionParameters.
