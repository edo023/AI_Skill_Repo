# i_rDrvTrqMax

i\_rDrvTrqMax

This is the torque threshold input that will trigger an overload alarm. It represents the torque of the drive at 100% of the nominal load of the crane. If the measurement state of the Overload\_EN15011 FB calculates an actual torque above this value the upwards movement is stopped and the overload alarm indicators q\_xLampRed, q\_xHorn are enabled.

If using a load cell this input is not used.
