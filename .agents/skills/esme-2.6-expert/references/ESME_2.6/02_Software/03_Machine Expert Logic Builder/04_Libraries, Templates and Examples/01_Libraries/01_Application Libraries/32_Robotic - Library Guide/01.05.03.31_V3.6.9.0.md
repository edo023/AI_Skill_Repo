## New Features

The interface IF\_AdditionalComponentsTransformation is added. Interface to implement a transformation for the movements of additional components.

To the interface IF\_RobotConfiguration, the following methods are added:

* AddDrive - Adds an additional drive to the robot to be used for additional components.
* AdditionalComponentsTransformation  - Adds an additional transformation for robot components.
* ConfigureTool - Modifies a tool in the robot configuration.
* GetTool - Reads the parameters of a configured tool.
* MapComponentToDrive - Maps the movement of a robot component directly to a drive.

To the interface IF\_RobotConfiguration, the following properties are added:

* raxCartesianComponentAvailable - Reads the available cartesian components.
* udiNumberOfConfiguredTools - Reads the number of configured tools.
* raudiConfiguredToolIds - Array of configured tool IDs.

To the interface IF\_RobotConfigurationReadOnly, the following method added:

* GetTool - Reads the parameters of a configured tool.

To the interface IF\_RobotConfigurationReadOnly, the following properties are added:

* udiNumberOfConfiguredTools - Reads the number of configured tools.
* raudiConfiguredToolIds - Array of configured tool IDs.

To the interface IF\_RobotMotion, the following methods are added:

* SetTool - Selects a configured tool.
* TransformCoordinate2 - Transforms a coordinate from one set of coordinate system and tool to another set of coordinate system and tool.
* TransformOrientation2 - Transforms an orientation from one set of coordinate system and tool to another set of coordinate system and tool.
* TransformPose - Transforms a cartesian pose from one set of coordinate system and tool to another set of coordinate system and tool.

To the Enumeration ET\_CoordinateSystem, the following enumeration element is added:

* TrackingSystemOffset - The enumeration element TrackingSystemOffset is added to the enumeration ET\_CoordinateSystem.

## Modifications

The default orientation convention for the mechanical coordinate system is set to ET\_OrientationConvention.ZYX for the transformations:

* IF\_RobotConfiguration.Articulated2Ax
* IF\_RobotConfiguration.Cartesian2Ax
* IF\_RobotConfiguration.Cartesian3Ax
* IF\_RobotConfiguration.Delta2Ax
* IF\_RobotConfiguration.Delta3Ax
* IF\_RobotConfiguration.Gantry2Ax
* IF\_RobotConfiguration.User3Ax

## Enhancements

* The feedback IF\_RobotFeedback.xInCoordinateSystem is now correctly changing back to TRUE in case you request the robot to change from CSR to CSR with no tracking systems configured.
* The diagnostic messages of functionality VelocityLimitationAxes now contains more information when the position deviation is exceeded.
* The position deviation feedback IF\_RobotFeedbackVelocityLimitation.rstRefPositionDeviationTCP and IF\_RobotFeedbackVelocityLimitation.lrRefPositionDeviation no longer contains the tracking offset.
