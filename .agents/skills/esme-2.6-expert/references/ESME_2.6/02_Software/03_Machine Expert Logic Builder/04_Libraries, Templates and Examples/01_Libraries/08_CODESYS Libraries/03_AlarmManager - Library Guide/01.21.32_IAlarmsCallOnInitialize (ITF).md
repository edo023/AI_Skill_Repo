# IAlarmsCallOnInitialize (ITF)

INTERFACE IAlarmsCallOnInitialize EXTENDS VisuShared.ICallOnInitializeBase

Notifies when initialization of alarms is finished. Register() must be called after global init slot 30000 (=GVL “AlarmGlobals”). Init() is called when all alarms are initialized and before evaluation of all alarms.
