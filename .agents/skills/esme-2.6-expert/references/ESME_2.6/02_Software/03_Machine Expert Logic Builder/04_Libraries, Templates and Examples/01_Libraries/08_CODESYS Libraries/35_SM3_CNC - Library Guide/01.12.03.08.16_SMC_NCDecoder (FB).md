# SMC\_NCDecoder (FB)

FUNCTION\_BLOCK FINAL SMC\_NCDecoder

This function block is used to convert a CNC program (Din 66025, G-code), into a list of [SMC\_GEOINFO](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-geoinfo.html#cah0ife6nckjzp5ef3qpy_filyk_smc_geoinfo_smc_geoinfo_struct) objects. In each cycle, one line of the program is decoded.

**Note**

The maximum number of M-functions in a row is 64. If the G-Code contains more than 64 M-functions in a row, the error SMC\_DEC\_OUTQUEUE\_RAN\_EMPTY occurs. This restriction can be circumvented by inserting a G75 previous to the first M-function of the sequence. G4-elements count as M-functions, too.

**Note**

As from V3.5 SP2 of the programming system, iLineNumberDecoded stores the just coded 0-based line number of the CNC file. In earlier versions, in particular CODESYS V2.3, the sentence number (= value of the N-word) had been stored there.

**Note**

This function block has been superseeded by [SMC\_NCInterpreter](VVFvsggjXKxZ6qmLw10MvZPswnU_smc-ncinterprete.html#vvfvsggjxkxz6qmlw10mvzpswnu_smc_ncinterprete_smc_ncinterpreter_fb) and is kept only for compatibility. It is recommended to use [SMC\_NCInterpreter](VVFvsggjXKxZ6qmLw10MvZPswnU_smc-ncinterprete.html#vvfvsggjxkxz6qmlw10mvzpswnu_smc_ncinterprete_smc_ncinterpreter_fb) (along with [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb)), which has additional features like mathematical expressions or sub programs in g-code
