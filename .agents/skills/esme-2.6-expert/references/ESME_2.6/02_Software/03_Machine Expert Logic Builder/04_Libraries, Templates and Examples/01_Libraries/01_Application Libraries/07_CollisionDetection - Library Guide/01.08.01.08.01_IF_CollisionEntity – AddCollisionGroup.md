## Task

Adds a collision group.

## Description

Adding a collision group to the collision entity.

## Interface

Return type: UDINT

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifCollisionGroup | [IF\_CollisionGroup](IF_CollisionGroupGeneralInformation-A28F23D4.html#IF_CollisionGroupGeneralInformation-A28F23D4) | Collision group to add to the entity. |

| Output | Data type | Description |
| --- | --- | --- |

| q\_etResult | [ET\_Result](ET_ResultEnumerator-9BCEF714.html#ET_ResultEnumerator-9BCEF714) | POU-specific output on the diagnostic; q\_xError = FALSE -> Status message; q\_xError = TRUE -> Diagnostic message. |
| q\_sResultMsg | STRING(80) | Event-triggered message that gives additional information on the diagnostic state. |

## Return Value

The returned value is the index of the collision group added to the list. It can be used to refer to such group through the property raifCollisionGroups.
