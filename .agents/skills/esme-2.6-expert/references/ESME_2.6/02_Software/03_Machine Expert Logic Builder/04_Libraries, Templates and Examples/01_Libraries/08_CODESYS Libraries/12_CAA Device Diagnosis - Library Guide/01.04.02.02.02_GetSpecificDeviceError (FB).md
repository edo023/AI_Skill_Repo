# GetSpecificDeviceError (FB)

FUNCTION\_BLOCK GetSpecificDeviceError EXTENDS ETrigA

This function block delivers the fieldbus specific device error.

**Note**

The interpretation of pData is fieldbus dependent and will be specified by the associated fieldbus driver.

**Note**

This feature may not be supported by all fieldbus drivers. eError will return [ERROR.NOT\_SUPPORTED](Wk7sVif4qdICA8XUp7Q627WapoY_error.html#wk7svif4qdica8xup7q627wapoy_error_error_enum__wk7svif4qdica8xup7q627wapoy_error_error_not_supported) if not supported.
