# Output Pin Description

Output Pin Description

| Output | Data Type | Description |
| --- | --- | --- |
| q\_xEn | BOOL | This displays the status of the enable parameter.  TRUE: Function block enabled  FALSE: Function block disabled |
| q\_xOvldAlrm | BOOL | This is the overload alarm output. Refer also the q\_xOvldAlrm description below this table.  Range: –3000...3000  Scaling/Unit: 0.1% |
| q\_rDistOvld | REAL | This is the traveled distance under overload. Refer also the q\_rDistOvld description below this table  Range: 1.175494957e-38F...3.402829466e+38F) |
