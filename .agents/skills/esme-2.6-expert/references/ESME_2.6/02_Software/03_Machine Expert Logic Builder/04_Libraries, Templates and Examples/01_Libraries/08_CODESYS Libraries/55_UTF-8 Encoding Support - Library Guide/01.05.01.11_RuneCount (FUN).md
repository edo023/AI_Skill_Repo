# RuneCount (FUN)

FUNCTION RuneCount : UDINT

Returns the number of runes in pbyData.

Erroneous and short encodings are treated as single runes of length 1 byte.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | RuneCount | UDINT |  | | Input | pbyData | POINTER TO BYTE | Pointer to the first byte in memory | | udiSize | UDINT | Memory size in byte | |
