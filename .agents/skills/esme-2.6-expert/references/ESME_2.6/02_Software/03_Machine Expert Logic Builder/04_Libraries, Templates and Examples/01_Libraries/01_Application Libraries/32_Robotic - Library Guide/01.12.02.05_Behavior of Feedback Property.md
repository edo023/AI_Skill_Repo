# Behavior of Feedback Property IF\_RobotFeedback.xOnPath

## General

If the robot is enabled the first time, or if a cold start was performed, the feedback property IF\_RobotFeedback.xOnPath is TRUE until a move command MoveL, MoveC, MoveS or MoveAsync is sent and the robot and the TCP (Tool Center Point) is jogged away from its position.
