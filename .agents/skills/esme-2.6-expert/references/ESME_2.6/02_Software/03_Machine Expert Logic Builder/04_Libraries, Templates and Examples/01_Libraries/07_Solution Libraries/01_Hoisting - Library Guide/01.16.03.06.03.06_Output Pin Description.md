# Output Pin Description

Output Pin Description

| Output | Data Type | Description |
| --- | --- | --- |
| q\_xEn | BOOL | Displays the status of the enable parameter input i\_xEn. If the output stays FALSE although i\_xEn is TRUE, an EcoStruxure Machine Expert controller platform cannot be detected.  TRUE: Function block enabled.  FALSE: Function block disabled. |
| q\_xHrsPral | BOOL | Status of the pre-alarm of the remaining operating hours. If remaining operating hours are equal or smaller than 10% of i\_stMDS.wHstDrvOphrMax, the output is set to TRUE.  TRUE: Alarm detected  FALSE: No alarm detected |
| q\_xHrsAlrm | BOOL | Status of the alarm of the remaining operating hours. If remaining operating hours are equal or smaller than zero the output is set to TRUE.  TRUE: Alarm detected  FALSE: No alarm detected |
| q\_wAlrmId | WORD | Detected alarm identification.  Range: 0...127  Refer to [Notifications](#XREF_D_SE_0021093_8). |
| q\_xAlrm | BOOL | Detected alarm bit.  TRUE: Alarm detected  FALSE: No alarm detected |
| q\_rHstDrvOphrRm | WORD | Value of remaining hours until servicing. That means, nominal hours according to FEM class minus actual operating hours based on real load.  Range: 0...i\_stMDS.wHstDrvOphrMax  Scaling/Unit: 1 hour |
| q\_rHstDrvOphrActl | WORD | Value of actual operating hours since the start of the measurement based on real load.  Range: 0...i\_stMDS.wHstDrvOphrMax  Scaling/Unit: 1 hour |
| q\_wHst300Ophr | WORD | Hours with more than 300 operations/hour.  Range: 1...65535  Scaling/Unit: 1 hour |
| q\_wHst600Ophr | WORD | Hours with more than 600 operations/hour.  Range: 1...65535  Scaling/Unit: 1 hour |
