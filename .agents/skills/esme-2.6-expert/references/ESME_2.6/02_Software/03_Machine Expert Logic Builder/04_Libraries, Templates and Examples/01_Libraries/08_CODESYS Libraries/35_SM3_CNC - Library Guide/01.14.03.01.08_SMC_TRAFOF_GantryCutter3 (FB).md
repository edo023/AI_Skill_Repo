# SMC\_TRAFOF\_GantryCutter3 (FB)

FUNCTION\_BLOCK SMC\_TRAFOF\_GantryCutter3

Forward transformation for three-dimensional gantries with a cutter knife. The rotation axis is controlled in such a way that the knife is aligned along the path tangent.

**Note**

For detailed information on transformations see overview to [SM\_Trafo\_POUs](uXtGJT17gvW9uxS1De45C-gyyqc_fld-SM_Trafo_POUs.html#uxtgjt17gvw9uxs1de45c_gyyqc_fld_sm_trafo_pous_sm_trafo_pous).
