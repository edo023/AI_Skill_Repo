# ET\_HomingCmd

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.3.1.0 |

## Description

The enumeration ET\_HomingCmd contains the commands that can be executed by the function block FB\_Home. This enumeration is used by FB\_Home.ifCmd.stGetCmdState.

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| None | 0 | No command has been selected. |
| HomingToDigitalInput | 1 | Home the axis to a reference switch. |
| SetPosition | 2 | Set the axis position to a parameterizable value. |
| WriteAxisEncoder | 3 | Write the axis encoder in connection with setting the axis position. |
| RestorePosFromAxisEncod | 4 | Restore the axis position from the axis encoder value. |
| HomingToTouchProbe | 5 | Home the axis using the signal of a touch probe input. |
