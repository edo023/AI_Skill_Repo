## Task

Interface with parameters from the move command [MovePosAndSync](IF_MovePosAndSync-4526BC87.html#IF_MovePosAndSync-4526BC87) for channel B.

## Description

The feedback interface provides parameters from the movement of the carrier with the move command MovePosAndSync for channel B.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| etPosMode | ET\_PosMode | Read | For the positioning modes, refer to the enumeration [ET\_PosMode](ET_PosMode-GeneralInformation-6D8695BB.html). |
| lrEndTargetPosition | LREAL | Read | The target position (end target) of the motion in length units.  The end target is set by calling the method [IF\_MovePosAndSync - StartPositioning](MovePosAndSync-StartPos-45306F20.html). |
| lrMaxVelocity | LREAL | Read | Indicates the maximum velocity (change of position per time unit). |
| lrMaxAcceleration | LREAL | Read | Indicates the maximum acceleration (change of velocity per time unit). |
| lrMaxDeceleration | LREAL | Read | Indicates the maximum deceleration (change of velocity per time unit). |
| lrMaxAbsJerk | LREAL | Read | Indicates the maximum jerk (change of acceleration per time unit). |
| lrYStep | LREAL | Read | Indicates the step movement of the carrier, executed with the method [StartSyncStep](MovePosAndSync-StartSyncStep-46B83CD1.html#MovePosAndSync-StartSyncStep-46B83CD1) or [StartSyncStepMaxVelocity](StartSyncStepMaxVel-48C43828.html#StartSyncStepMaxVel-48C43828). |
