# Description

Description

The library "Sercos3PlcOpenMc" is a collection of PLCopen-conform function blocks for activating servo drives.

The library is an alternative to the [PacDriveLib](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0090101_1) library. If no PLCopen-conform function blocks are required, it is recommended to use the [PacDriveLib](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0090101_1) library.

Axes of type Lexium LXM62 Drive, Lexium LXM52 Drive and Lexium ILM62 Drive Module are supported via Sercos3.
