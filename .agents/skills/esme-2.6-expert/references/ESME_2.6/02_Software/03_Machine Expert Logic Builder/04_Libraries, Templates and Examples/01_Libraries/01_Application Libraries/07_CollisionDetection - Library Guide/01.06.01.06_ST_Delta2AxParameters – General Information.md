## Description

A set of parameters describing the kinematics and the geometry of a Delta2Ax robot. For more information, refer to [ST\_Delta2AxKinematics](ST_Delta2AxKinematics-9F63EFEB.html) and [ST\_Delta2AxGeometry](ST_Delta2AxGeometryGeneralInformati-9EE7E971.html).

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stKinematics | ST\_Delta2AxKinematics | A set of parameters used by the kinematics of the robot. |
| stGeometry | ST\_Delta2AxGeometry | A set of parameters describing the geometry of the robotic structure. |
