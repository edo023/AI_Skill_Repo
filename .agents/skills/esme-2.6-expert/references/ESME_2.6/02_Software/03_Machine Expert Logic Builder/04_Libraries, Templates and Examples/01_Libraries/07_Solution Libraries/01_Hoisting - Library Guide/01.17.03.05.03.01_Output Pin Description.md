# Output Pin Description

Output Pin Description

| Output | Data type | Description |
| --- | --- | --- |
| q\_xEn | BOOL | TRUE: The function block is enabled.  FALSE: The function block is disabled. |
| q\_xBridgeFwd | BOOL | Bridge travel forward command output  TRUE / FALSE |
| q\_xBridgeRev | BOOL | Bridge travel reverse command output  TRUE / FALSE |
| q\_xTrolFwd | BOOL | Trolley forward command output  TRUE / FALSE |
| q\_xTrolRev | BOOL | Trolley reverse command output  TRUE / FALSE |
| q\_xHoistFwd | BOOL | Hoist forward command output  TRUE / FALSE |
| q\_xHoistRev | BOOL | Hoist reverse command output  TRUE / FALSE |
| q\_xMoveRestr | BOOL | TRUE: Movement is blocked.  FALSE: Movement is not blocked. |
| q\_xTeachMode | BOOL | TRUE: Function block is in teach mode  FALSE: Function block is not in teach mode |
| q\_xBlockOvrrd | BOOL | TRUE: Override of movement blocking active  FALSE: Override of movement blocking not active |
| q\_xAlrm | BOOL | TRUE: Alarm detected.  FALSE: No alarm is detected. |
| q\_wAlrmId | WORD | Alarm status  Range: 0...511 |
