# ET\_Parameter

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Parameter specifies the drive parameter that is sampled for a channel.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Disable | 0 | No parameter of the drive is sampled on the channel. |
| MechPosition | 1 | Position parameter of the drive is sampled on the channel. |
| MechVelocity | 2 | Velocity parameter of the drive is sampled on the channel. |
| Current | 3 | Current parameter of the drive is sampled on the channel. |
| FeedbackCurrent | 4 | Feedback current parameter of the drive is sampled on the channel. |
