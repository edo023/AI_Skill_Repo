## Description

This method is used to set the value for the header field Proxy Authorization in the HTTP CONNECT request sent to the proxy server.

If this method successfully sets a non-null value, the request created with the ConnectToRemoteServer method will be as follows:

`CONNECT <iq_sHostname>:<i_uiServerPort> HTTP/1.1`  
`Proxy-Authorization: <Value for Proxy-Authorization>`

NOTE: The header field Proxy Authorization is added only if a non-null value is set with the SetValueProxyAuthorization method.

The return value of type BOOL indicates TRUE if the method execution was completed successfully. Otherwise, refer to the values of Result and ResultMsg to get detailed information about the detected error.
