## Description

The method DefineUnitModeWithHandler has the same functionality as the method DefineUnitMode(). In addition, however, the method DefineUnitModeWithHandler offers the possibility of passing an instance of the function block FB\_StateModelHandlerBase. This gives the FB\_UnitModeManager2 access to this instance and using the ExecuteCurrentState() method, the corresponding state method of the instance is invoked.
