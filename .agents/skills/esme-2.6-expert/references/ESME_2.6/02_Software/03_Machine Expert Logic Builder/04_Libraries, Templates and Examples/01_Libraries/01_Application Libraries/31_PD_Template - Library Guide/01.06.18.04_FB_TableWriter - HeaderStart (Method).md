# FB\_TableWriter - HeaderStart (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | 1.1.2.0 |
| Versions: | Current version |

## Task

Prepares writing of the table header data

The header data then are written using HeaderAddColumn(). After inserting all table header data, writing must be finished using HeaderFinish().

Every XML table output must be started using this command. TableEnd() must be called up to end a table.
