# SMC\_OutQueueIsSyncing (FUN)

FUNCTION SMC\_OutQueueIsSyncing : BOOL

Returns whether the queue is syncing, i.e. if it is to be emptied completely before being refilled. This is done if a G75 command is issued, which is stored as a m-function with m-value -2.

task-safety: may be called from both tasks
