# ST\_ConfigDataRobotPSeries

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | ST\_ConfigDataGeneric |

## Description

The structure ST\_ConfigDataRobotPSeries contains additional parameters to configure a Lexium P robot in EcoStruxure Machine Expert Twin extending the structure [ST\_ConfigDataGeneric](ST_ConfigDataGeneric-6D5DB8E3.html#ST_ConfigDataGeneric-6D5DB8E3).

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| xRotAxRotationLeft | BOOL | Rotational direction of the rotational axis.  By default, xRotAxRotationLeft is set to FALSE which corresponds to clockwise rotation.  If xRotAxRotationLeft is set to TRUE, the rotational direction is counterclockwise. |
| lrTcpPlateSize | LREAL | The size (radius) of the TCP (Tool Center Point) plate.  The default value is `0.0` which corresponds to the default radius of 50.0 mm (1.97 in.).  Value = lrLengthABC3  For further information, refer to the [Robotic Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=D_SE_0075525). |
