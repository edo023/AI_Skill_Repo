# IecTaskEnableWatchdog (FUN)

FUNCTION IecTaskEnableWatchdog : RTS\_IEC\_RESULT

Enable watchdog for the specified task

**Note**

* You have to disable the watchdog of the task before with [IecTaskDisableWatchdog](IecTaskDisableWatchdog.html#iectaskdisablewatchdog_iectaskdisablewatchdog_fun) and \_not\_ with [IecTaskDisableWatchdog2](IecTaskDisableWatchdog2.html#iectaskdisablewatchdog2_iectaskdisablewatchdog2_fun), because they act on different task flags!
* The watchdog is enabled only at the next IEC cycle and \_not\_ immediately after calling this function!
* If you disable with [IecTaskDisableWatchdog](IecTaskDisableWatchdog.html#iectaskdisablewatchdog_iectaskdisablewatchdog_fun) and forgot to enable it, at least at the next cycle, the watchdog is automatically enabled!
