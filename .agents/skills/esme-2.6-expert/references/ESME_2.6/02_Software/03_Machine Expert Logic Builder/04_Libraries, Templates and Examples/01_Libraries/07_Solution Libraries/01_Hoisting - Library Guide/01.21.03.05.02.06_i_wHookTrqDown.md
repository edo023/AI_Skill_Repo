# i_wHookTrqDown

i\_wHookTrqDown

You are advised to run the drive without a load on the hook and record the percentage (%) load from the Monitoring menu on the drive or by monitoring the corresponding communication parameters.

When the Hoist is moving down and if the actual torque (i\_iDrvTrqActl) is less than or equal to 70% of the hook torque (i\_wHookTrqDown), then the target speed (q\_wDrvSpdTarg) is set to minimum speed reference (i\_wDrvSpdLsp) and (q\_wSpdTargMax) is set to (i\_wDrvSpdNom).
