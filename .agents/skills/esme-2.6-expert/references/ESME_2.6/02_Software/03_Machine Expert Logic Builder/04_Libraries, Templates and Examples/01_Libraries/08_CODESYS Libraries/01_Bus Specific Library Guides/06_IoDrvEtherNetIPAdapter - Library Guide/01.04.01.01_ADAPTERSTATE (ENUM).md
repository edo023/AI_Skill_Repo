# ADAPTERSTATE (ENUM)

TYPE ADAPTERSTATE :

An output of IoDrvEtherNetIPAdapter.

| Attributes: | qualified\_only |
| InOut: | | Name | Initial | Comment | | --- | --- | --- | | UPDATE\_CONFIGURATION | 0 | Start phase | | NOT\_CONFIGURED | 1 | Loading parameters | | DISABLED | 2 | Disabled in device tree | | CONFIGURED | 3 | Create CIP objects | | IP\_CONFIG | 4 | Configure TCP object and waiting for ethernet node | | IMPLICITMESSAGING\_CONFIG | 5 | Open ports UDP | | EXPLICITMESSAGING\_CONFIG | 6 | Open ports TCP | | RUNNING | 7 | Stack is running and scanner is connected | | STOPPED | 8 | Ethernet node is inactiv, waiting for ethernet node to come back (pull cable) | | RESET | 9 | Closing the connections opened in |IMPLICITMESSAGING\_CONFIG| and |EXPLICITMESSAGING\_CONFIG| | | SCANNER\_EXTENSION | 10 | This state is active, if a scanner registered this adapter as an extension for I/Os | | ERROR | 11 | Fatal error: no license, no memory | | NO\_CONNECTION | 12 | Stack has started, but no scanner is connected | | BUS\_ERROR | 13 | If the ethernet interface is not ready or available | |
