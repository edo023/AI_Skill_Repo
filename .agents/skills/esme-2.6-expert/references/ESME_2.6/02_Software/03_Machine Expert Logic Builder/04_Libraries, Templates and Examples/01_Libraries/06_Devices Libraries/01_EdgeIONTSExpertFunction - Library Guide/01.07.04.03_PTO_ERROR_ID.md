# PTO\_ERROR\_ID

## Description

The PTO\_ERROR\_ID enumeration describes the types of PTO errors that can be detected:

* Detected axis errors are in the range from 1000...1999
* Detected motion state errors are in the range from 2000...2999
* Detected range errors are in the range from 3000...3999

## Enumeration Elements

The enumeration data type ENUM contains the following values. For parameters that are configured in the Edge I/O NTS Editor of the Logic Builder, [refer to the *PTO Configuration*](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Exp_UG&topicID=PTOInterfaceConfiguration_827F6FBC).

| SwPositionLimitP | 1004 | The software positive position limit is reached. |
| SwPositionLimitN | 1005 | The software negative position limit is reached. |
| ApplicationStopped | 1006 | Application execution has been stopped. |
| OutputProtection | 1007 | Short-circuit output protection is active on the PTO channels. |
| INTERNAL\_FIELD\_POWER\_SUPPLY | 1008 | Internal field power supply is not detected. |
| Fallback | 1009 | Outputs are in fallback state. |
| WarningVelocityValue | 1100 | Commanded Velocity parameter is out of range. |
| WarningAccelerationValue | 1101 | Commanded Acceleration parameter is out of range. |

| InvalidBufferModeValue | 3005 | The [buffer mode](MC_BUFFERMODE-91D8C3A7.html) does not correspond to a valid enumeration value. |

| InvalidParameter | 3008 | The parameter number does not exist for the specified axis. |

| ReadOnlyParameter | 3010 | The parameter is read-only. |
