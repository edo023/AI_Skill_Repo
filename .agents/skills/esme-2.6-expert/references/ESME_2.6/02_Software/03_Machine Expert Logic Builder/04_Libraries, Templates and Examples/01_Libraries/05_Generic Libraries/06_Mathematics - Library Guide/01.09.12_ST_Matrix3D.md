# ST\_Matrix3D

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |

## Description

The structure ST\_Matrix3D represents a quadratic matrix with three lines and three columns.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| alrElement | ARRAY [1..3, 1..3] OF LREAL | Two-dimensional field of the matrix with the primary field representing the lines and the secondary field representing the columns. |
