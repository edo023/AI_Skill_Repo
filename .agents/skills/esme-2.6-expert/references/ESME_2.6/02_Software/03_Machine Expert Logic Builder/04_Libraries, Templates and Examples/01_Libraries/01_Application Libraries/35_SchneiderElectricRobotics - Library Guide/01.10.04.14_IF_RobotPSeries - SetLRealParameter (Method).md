## Task

Set/overwrite a parameter for a Lexium P Robot.

## Description

By calling the method, a parameter contained in the enumeration ET\_Parameter can be parametrized.

NOTE: The method must be called after the method [FB\_RobotPSeries.InitializeRobot](FB_RobotPSeries-InitializeRobotMeth-D258A1C9.html) has been called but before IF\_Configuration.SchneiderElectricRobot (SER.FB\_RobotPSeries) of Robotic/RoboticModule is called.
