# BeltIdRange

BeltIdRange

|  |  |
| --- | --- |
| Enumeration name: | BeltIdRange |
| Enumeration value: | 11 |
| Description: | BeltId is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid value has been transferred at the input i\_udiBeltId. | At the i\_udiBeltId input, the number of the belt has to be specified whose feed forward velocity is to be changed. The number must be between one and the number of parameterized belts. |
