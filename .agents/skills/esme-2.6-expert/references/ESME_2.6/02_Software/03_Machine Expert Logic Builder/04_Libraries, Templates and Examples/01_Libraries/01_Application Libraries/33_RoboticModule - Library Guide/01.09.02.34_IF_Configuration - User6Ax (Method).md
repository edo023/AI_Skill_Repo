## Task

Configuring an up to six axial user-configured transformation robot.

## Description

Next to the use of default kinematics, user-specific kinematics can be defined by using the User6Ax(…) method.

The calculation of the direct and inverse transformation must be realized in a user POU that implements the *IF\_UserTransformation6Ax* interface.

Kinematics with up to six axes and up to six degrees of freedom are possible.

A transformation for up to six axes can be parameterized.

NOTE: If a method to configure a transformation was already called up successfully (q\_etDiag = ET\_Diag.Ok AND q\_etDiagExt = ET\_DiagExt.Ok), then it is not possible to overwrite the parameterization by calling another method to configure a transformation.
