# IAlarm3.GetMessage2 (METH)

METHOD GetMessage2 : CharBufferPtr

Returns a message text, that has been configured for the alarm.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | GetMessage2 | CharBufferPtr |  | | Input | stringType | \_\_SYSTEM.TYPE\_CLASS | A \_\_SYSTEM.TYPE\_CLASS indicating, which type of string has to be retrieved | | iWhichMessage | INT | The 1-based index of the message text. If this parameter is 1 then the return value will be the return value of method GetMessage. For all greater parameter values the additional message texts will be returned | |
