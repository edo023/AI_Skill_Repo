## Task

Parameterize and generate a list of targets.

## Description

The function block FB\_RandomTargetsGenerator contains a list of methods to parameterize and to generate a list of targets.

## Methods

| Name | Description |
| --- | --- |
| SetRandomSeed | Set the internal seed, used for the generation of the random sequence of targets, to a constant value provided. |
| GetRandomSeed | Retrieve the present random seed value used by the function block for the generation of random targets. |
| SetTargetsInPlane | Define a set of constraints for the generation of random targets contained in a selected working plane. |
| GetTargetsInPlane | Get a set of constraints for the generation of random targets contained in a selected working plane. |
| SetTargetsInPlaneRotationList | Define a set of constraints for the generation of random targets contained in a selected working plane. |
| GetTargetsInPlaneRotationList | Get a set of constraints for the generation of random targets contained in a selected working plane. |
| SetTargetMatrixInPlane | Define a set of constraints for the generation of a matrix of targets contained in a selected working plane. |
| GetTargetsMatrixInPlane | Get a set of constraints for the generation of a matrix of targets contained in a selected working plane. |
| SetTargetMatrixInPlaneRotationList | Define a set of constraints for the generation of a matrix of targets contained in a selected working plane. |
| GetTargetMatrixInPlaneRotationList | Get a set of constraints for the generation of a matrix of targets contained in a selected working plane. |
| Generate | Generate a set of targets. Depending on the type of generation selected, the result is different. |
