# ST\_Polytope

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.1.0.0 |

## Description

The structure ST\_Polytope represents a convex polytope in three-dimensional space. This polytope is given by the convex envelope of a list of points.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| diNumberOfVertices | DINT | Number of defining points of the polytope.  Value range: 1 ... Gc\_diMaxNumberOfVertices |
| astListOfVertices | ARRAY[1..Gc\_diMaxNumberOfVertices] OF [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | List of the defining points of the polytope |
