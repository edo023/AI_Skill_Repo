# SMC\_NC\_PrependDefaults (FB)

FUNCTION\_BLOCK SMC\_NC\_PrependDefaults

This function block is used internally by |SMC\_ReadNCFileFromStream| to prepend a sentence containing the given defaults, e.g. ‘N0 G17 F25 E1000 E-2000’. If |SMC\_ReadNCFileFromStream| or [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb) is used, there is not need to use this function block.

Important: this function block will prepend the defaults again after a jump command in the out queue. So it should be used before any buffering function blocks such the jump buffer. It should be called before [SMC\_NCNavigator](TW1SQNUTI52-XXX1HJ6us52paR8_smc-ncnavigator.html#tw1sqnuti52_xxx1hj6us52par8_smc_ncnavigator_smc_ncnavigator_fb) as well, because the navigator will handle the jumps to unknown jump targets.
