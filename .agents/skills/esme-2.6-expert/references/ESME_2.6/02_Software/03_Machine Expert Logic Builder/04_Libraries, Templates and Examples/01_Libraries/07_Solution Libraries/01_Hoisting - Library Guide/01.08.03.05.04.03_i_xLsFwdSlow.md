# i_xLsFwdSlow

i\_xLsFwdSlow

This signal is used to assist the slow down at the slow down limit switch in a forward direction. This input does not modify the speed reference. It sets shorter ramps in order to slow the movement down in a shorter distance while still using the Anti-sway function. The speed reference required in the slow area must be given to the input i\_wDrvSpdRef from an external source (e.g. SpeedSelect function block).
