# Description

Description

Calling up the function destroys the transferred semaphore. Prior to deleting, a Take() is executed on the semaphore. In the event that the semaphore is not free, the call up task is blocked.

For restrictions for use see [Notes on Using the "SemaphoreMutualExclusion" Function](../Presentation_of_the_Library/Presentation_of_the_Library-8.htm#XREF_D_SE_0088285_1).
