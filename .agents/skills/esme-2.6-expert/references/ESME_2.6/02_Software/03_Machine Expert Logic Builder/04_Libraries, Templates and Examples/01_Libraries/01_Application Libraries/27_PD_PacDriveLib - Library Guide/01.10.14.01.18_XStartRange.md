# XStartRange

XStartRange

|  |  |
| --- | --- |
| Enumeration name: | XStartRange |
| Enumeration value: | 119 |
| Description: | XStart is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | At the input i\_lrXStart, a number >= that at the input i\_lrXEnd has been transferred. | Verify i\_lrXStart and set it to a value not exceeding i\_lrXEnd.  Verify i\_lrXEnd and set it to a value exceeding i\_lrXStart. |
| - | At the input i\_lrXStart, a number which is smaller than the start of the cam has been transferred. | Verify i\_lrXStart and transfer it into the range of the cam.  Verify the applied cam and define it such that i\_lrXStart lies within the cam. |
