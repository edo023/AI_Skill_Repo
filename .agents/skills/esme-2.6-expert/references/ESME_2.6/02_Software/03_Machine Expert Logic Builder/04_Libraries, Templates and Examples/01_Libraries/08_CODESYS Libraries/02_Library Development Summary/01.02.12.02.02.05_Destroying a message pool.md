### Destroying a message pool

> ```
> hPool : MBM.HANDLE;
> xExit : BOOL;
>
> IF xExit THEN
>     MBM.XChgDelete(hXChg:=hPool);
>     // The pool is now locked and MsgReceive will return an error
>     // The pool memory will be released with the last MsgRelease
>     hPool := MBM.gc_hINVALID;
>     xExit := FALSE;
> END_IF
> ```
