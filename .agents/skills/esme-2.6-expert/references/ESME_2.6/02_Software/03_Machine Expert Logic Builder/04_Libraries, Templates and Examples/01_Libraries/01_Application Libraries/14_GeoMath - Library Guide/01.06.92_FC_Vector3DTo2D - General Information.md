## Description

Given an input 3D vector and an input plane, the function returns a 2D vector with the original components copied on the elements selected by the plane (the third element is ignored).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stVector | SE\_MATH.ST\_Vector3D | A 3D vector. |
| i\_etPlane | ET\_CartesianPlane | A Cartesian plane. |
