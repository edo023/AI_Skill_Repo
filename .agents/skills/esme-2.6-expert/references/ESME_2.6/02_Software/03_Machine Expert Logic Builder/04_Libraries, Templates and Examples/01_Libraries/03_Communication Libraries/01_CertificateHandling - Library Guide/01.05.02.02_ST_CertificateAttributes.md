# ST\_CertificateAttributes

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

The structure ST\_CertificateAttributes is used to provide the attributes of a certificate.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sOrganization | STRING[64] | The name of the organization. |
| sOrganizationUnit | STRING[64] | The name of the organization unit. |
| sLocality | STRING[128] | The name of the locality. |
| sStateProvince | STRING[128] | The name of the state or province. |
| sCountry | STRING[2] | The country code consisting of two characters. |
| stAlternativeNames | ST\_AlternativeNames | The structure containing the alternative names. |
| etExtendedKeyUsage | ET\_ExtendedKeyUsage | The enumeration that contains the values describing the usage of the certificate.  NOTE: Exclusive to self-signed certificates. |

## Used By

* FB\_CreateCSR
* FB\_CreateSelfSignedCertificate
