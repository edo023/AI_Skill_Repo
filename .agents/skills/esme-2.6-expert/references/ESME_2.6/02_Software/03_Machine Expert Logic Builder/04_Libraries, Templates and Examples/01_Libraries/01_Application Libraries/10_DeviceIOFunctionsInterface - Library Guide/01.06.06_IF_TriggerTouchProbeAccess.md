# IF\_TriggerTouchProbeAccess

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The access interface IF\_TriggerTouchProbeAccess allows you to access the user function interface [IF\_TriggerTouchProbe](IF_TriggerTouchProbe-EBF48A34.html#IF_TriggerTouchProbe-EBF48A34) for position capture with a touchprobe.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| triggerCap | [IF\_TriggerTouchProbe](IF_TriggerTouchProbe-EBF48A34.html#IF_TriggerTouchProbe-EBF48A34) | Get | Accesses the user function interface IF\_TriggerTouchProbe. |
