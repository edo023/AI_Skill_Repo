# SR\_<Conveyor Name> - Configuration (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V2.2.0.0 |

## Functional Description

* In case of node type 'PacDrive 3 Template' and node type 'Non Template' the method Configuration is called only once before the initialization step of the module is executed.
* In case of node type 'Physical encoder' the method Configuration is called once after i\_xEnable is set to TRUE.

## Interface

* In case of node type 'PacDrive 3 Template' and node type 'Non Template':

  | Input/Output | Data type | Description |
  | --- | --- | --- |
  | iq\_stConveyorModuleItf | [AXM. ST\_ModuleInterface](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077227) | Adapt the parameter for the different operation modes. |
* In case of node type 'Physical encoder' the interface has no inputs and outputs.
