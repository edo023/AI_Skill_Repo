# Global Parameter List (GPL)

## Overview

|  |  |
| --- | --- |
| Type: | Global parameters |
| Available as of: | V2.0.3.0 |

## Description

The global parameter list (GPL) contains global constants which are used by certain components of this library. The parameters can be edited individually for each application where the library is used. The modification must be done within the Library Manager of the project where the library is referenced.

## Global Parameters

| Variable | Data type | Default value | Description |
| --- | --- | --- | --- |
| Gc\_uiMaxNumberOfAvatars | UINT | 20 | The value indicates the maximum number supported avatars. |
| Gc\_uiMaxNumberOfOrders | UINT | 20 | The value indicates the size of the FIFO for acyclic requests. |
| Gc\_uiLengthOfAvatarName | UINT | 20 | The value indicates the number of characters of the avatar name. |
| Gc\_uiLengthOfDeviceName | UINT | 20 | The value indicates the number of characters of the device name. |
