# SMC\_GExpToken\_Iterator (FB)

FUNCTION\_BLOCK FINAL SMC\_GExpToken\_Iterator

This iterator allows to iterate all g-expression tokens of a g-word. It can be retrieved from a gword-iterator ([SMC\_GWord\_Iterator.GetGExpTokenIterator](ntF018iBVnK9SXQgapHsJMDHy9c_getgexptokeniter.html#ntf018ibvnk9sxqgaphsjmdhy9c_getgexptokeniter_smc_gword_iterator_getgexptokeniterator_meth)). For each token, the additional data can be queried.
