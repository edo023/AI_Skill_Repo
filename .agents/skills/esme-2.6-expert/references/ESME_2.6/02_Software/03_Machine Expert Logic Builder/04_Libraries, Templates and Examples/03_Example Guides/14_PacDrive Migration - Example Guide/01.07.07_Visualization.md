# Visualization

## Overview

To use the full functionality of the visualization, you must adapt the visualization file.

## Exporting the Visualization

| Step | Action |
| --- | --- |
| 1 | Go to Project > Export .  **Result:** An export dialog is displayed. |
| 2 | In the devices tree, select  Application > TemplateFullProgrammingFramework > Visualization and select the check boxes for the objects you wish to export. |
| 3 | Click Ok.  **Result:** A browse dialog is displayed. |
| 4 | Browse to the folder where you want to save the visualization file and confirm with Save. |

## Importing the Visualization File

| Step | Action |
| --- | --- |
| 1 | In the devices tree, select  Application > TemplateFullProgrammingFramework > Visualization. Right-click Visualization and select Delete in the contextual menu. |
| 2 | In the devices tree, select  Application > TemplateFullProgrammingFramework.  Then go to Project > Import.  **Result:** An import dialog is displayed. |
| 3 | Browse to the folder where you have saved the visualization file. Select the xml file and confirm with Save.  **Result:** A visualization folder is added in the devices tree structure. |
