# Connect (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method is used to initiate a connection to the configured cloud server. Once this method is called, the connection establishment is processed by executing the method Cyclic().

The integrated DNS client is executed during the connection process, if required.

A connection can be initiated only from state Idle.

The return value of type ET\_Result indicates if the connection is initiated.

This method has no inputs and outputs.
