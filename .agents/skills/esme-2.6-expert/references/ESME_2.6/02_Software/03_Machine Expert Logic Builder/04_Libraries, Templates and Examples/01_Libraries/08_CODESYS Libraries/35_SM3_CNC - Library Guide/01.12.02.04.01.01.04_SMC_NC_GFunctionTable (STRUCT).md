# SMC\_NC\_GFunctionTable (STRUCT)

TYPE SMC\_NC\_GFunctionTable : STRUCT

A table of functions of type [SMC\_NC\_IFunction](N1iY2oaH809ZTO1uQFP4HHVDcCg_smc-nc-ifunction.html#n1iy2oah809zto1uqfp4hhvdccg_smc_nc_ifunction_smc_nc_ifunction_itf). The function blocks [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb) and [SMC\_ReadNCFromStream](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfromst.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfromst_smc_readncfromstream_fb) accept such a table to add additional functions that can be used in expressions in the G-Code.

Do not change. | | numFunctions | UDINT |  | The number of functions to be added | | pFunction | POINTER TO [SMC\_NC\_GFunction](N1iY2oaH809ZTO1uQFP4HHVDcCg_smc-nc-gfunction.html#n1iy2oah809zto1uqfp4hhvdccg_smc_nc_gfunction_smc_nc_gfunction_struct) |  | Pointer to an array of SMC\_NC\_GFunction, containing the functions to be added | |
