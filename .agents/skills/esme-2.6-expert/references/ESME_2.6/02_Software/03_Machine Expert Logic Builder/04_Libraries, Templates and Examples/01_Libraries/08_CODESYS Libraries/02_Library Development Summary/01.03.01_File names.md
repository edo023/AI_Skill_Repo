# File names

## Rule #1:

Select a nice file name ([Required](glossary.html#glossary_glossary__glossary_term_Required))

* Common Libraries: <name>.library
* Container Libraries: <name>\_Cnt.library
* Interface Libraries: <name>\_Itfs.library

The file name of a library should be a short, significant term, referring to the content of the library.

* A [Common Library](library_types.html#library_types_common_libraries) must not have any special extension. (e.g. Storage.library)
* A [Container Library](library_types.html#library_types_container_libraries) has usually the string \_Cnt appended to the “normal” file name (e.g. SysTime\_Cnt.library).
* A [Interface Library](library_types.html#library_types_interface_libraries) has usually the string \_Itfs appended to the “normal” file name (e.g. AlarmManager\_Itfs.library).
