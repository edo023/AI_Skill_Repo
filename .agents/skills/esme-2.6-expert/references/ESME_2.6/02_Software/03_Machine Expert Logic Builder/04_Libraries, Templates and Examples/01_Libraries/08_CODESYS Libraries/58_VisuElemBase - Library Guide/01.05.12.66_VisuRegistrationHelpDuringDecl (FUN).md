# VisuRegistrationHelpDuringDecl (FUN)

FUNCTION VisuRegistrationHelpDuringDecl : BYTE

The intention of this function is to allow the registration of globally (generated) instances to a pointer declared within the library within the declaration itself. That means it allows to write the following code without the need of an additionally generated assignment: generatedInst : LibType := (a:=1, b:=2, dummy:= VisuRegistrationHelpDuringDecl(VisuGlobals.g\_pMyInst, ADR(generatedInst))
