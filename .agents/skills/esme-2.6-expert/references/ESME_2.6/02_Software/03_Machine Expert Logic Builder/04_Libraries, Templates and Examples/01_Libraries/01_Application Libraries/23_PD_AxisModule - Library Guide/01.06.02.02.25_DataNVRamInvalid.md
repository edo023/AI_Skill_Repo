# DataNVRamInvalid

DataNVRamInvalid

|  |  |
| --- | --- |
| Enumeration name: | DataNVRamInvalid |
| Enumeration value: | 75 |
| Description: | The non-volatile memory (NVRam) does not contain any more valid data. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The data stored in the NVRam are invalid. Thus, the position cannot be restored from [ST\_HomeSetPos.iq\_pstRestorePosRetain](../Structures/Structures-6.htm#XREF_D_SE_0077221_1). | Check the NVRam, check the buffer battery and replace if necessary.  New homing of the drive required. |
