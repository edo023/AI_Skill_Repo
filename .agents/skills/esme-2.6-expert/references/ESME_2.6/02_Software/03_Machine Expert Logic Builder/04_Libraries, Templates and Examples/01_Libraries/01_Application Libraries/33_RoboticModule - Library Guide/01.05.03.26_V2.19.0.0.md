## New Features

The following method is added to the interface IF\_ConfigurationAdvanced:

* SetOrientationConvention - Setting the orientation convention for the robot.

## Modifications

The default orientation convention was changed from XYZ to ZYX.

The method IF\_ConfigurationAdvanced.SetOrientationConvention() can be used to set the orientation convention to the desired value.
