# AccRange

AccRange

|  |  |
| --- | --- |
| Enumeration name: | AccRange |
| Enumeration value: | 12 |
| Description: | Acc is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | In the parameter lrAcc of the current table step, an invalid value has been transferred. (The table steps are transferred at the input i\_pstMultiPosStep) | The following must hold: 0 < lrAcc < drive parameter MaxAcc  For the valid value range for lrAcc, see output q\_sMsg |
