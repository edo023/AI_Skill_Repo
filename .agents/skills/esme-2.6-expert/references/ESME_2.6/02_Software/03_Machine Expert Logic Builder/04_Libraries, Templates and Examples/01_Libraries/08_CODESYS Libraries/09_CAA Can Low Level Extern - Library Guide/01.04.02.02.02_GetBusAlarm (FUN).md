# GetBusAlarm (FUN)

FUNCTION GetBusAlarm : BOOL

This function returns if the CAN chip is in Bus Alarm state. In this state sending and receiving messages is not possible. A Bus Alarm can be reset by [ResetBusAlarm](8gqnJ6B5M46-D0DeyJx7cbK99BA_resetbusalarm.html#8gqnj6b5m46_d0deyjx7cbk99ba_resetbusalarm_resetbusalarm_fun) which leads to a re-initialization of the CAN chip. Diagnostic Information can be also read by [GetDiagnosis](8gqnJ6B5M46-D0DeyJx7cbK99BA_getdiagnosis.html#8gqnj6b5m46_d0deyjx7cbk99ba_getdiagnosis_getdiagnosis_fun).
