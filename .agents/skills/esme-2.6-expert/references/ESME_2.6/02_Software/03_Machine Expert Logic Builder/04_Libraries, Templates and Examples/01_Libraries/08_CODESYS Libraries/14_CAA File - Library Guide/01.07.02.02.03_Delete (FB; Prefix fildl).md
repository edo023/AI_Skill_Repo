# Delete (FB; Prefix fildl)

FUNCTION\_BLOCK Delete EXTENDS CBM.ETrig

This function block deletes a file.

Contrary to the original CAA specification the file may be deleted even if it is opened via ”FILE.[Open](VIPLMVOpanJfjqQXFKWGQlzGjxw_open.html#viplmvopanjfjqqxfkwgqlzgjxw_open_open_fb_prefix_filop)”.

This behavior is dependent of the underlying operating system and file system.
