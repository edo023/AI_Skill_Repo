## Description

The method ForceStop of the interface [IF\_MultiAxisCustomJob](IF_MultiAxisCustomJob-4D7A4899.html) terminates the execution of the multi-axis custom job. When this method is called, the cyclic execution of the Calculate method is stopped.

The subordinate axes are released from the control of the custom job and are decelerated to a standstill. The deceleration and jerk values used for this stop are defined by the ErrorStopDec and ErrorStopJerk parameters of each respective subordinate axis. After the axes have come to a standstill, the interface transitions to the Idle state.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
