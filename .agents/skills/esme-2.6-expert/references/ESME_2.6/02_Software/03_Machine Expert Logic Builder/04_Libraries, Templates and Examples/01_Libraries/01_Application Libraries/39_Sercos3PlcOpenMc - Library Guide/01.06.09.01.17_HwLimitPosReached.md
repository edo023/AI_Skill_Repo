# HwLimitPosReached

HwLimitPosReached

|  |  |
| --- | --- |
| Enumeration name: | HwLimitPosReached |
| Enumeration value: | 66 |
| Description: | Positive limit switch has been reached. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The positive hardware limit switch has been found when searching for the reference signal. | Verify the signal at the input i\_xHwLimitPos.  Ensure that the sensor of the positive limit switch is not dirty. |
