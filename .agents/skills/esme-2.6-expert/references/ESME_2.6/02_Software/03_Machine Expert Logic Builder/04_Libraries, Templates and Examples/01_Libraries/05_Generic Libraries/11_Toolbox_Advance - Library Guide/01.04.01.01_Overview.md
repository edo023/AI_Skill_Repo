# Overview

Overview

The Toolbox\_Advance library offers 2 functions that can be used in a Program Organisation Unit (POU) to get (GetCurrentTaskCycle) and set (SetCurrentTaskCycle) the interval of the attached Cyclic Task.

NOTE: These functions can be used in POUs attached to Cyclic Tasks only. For further information about the Tasks configuration and POUs attachment, refer to the Tasks chapter of your Controller Programming Guide.
