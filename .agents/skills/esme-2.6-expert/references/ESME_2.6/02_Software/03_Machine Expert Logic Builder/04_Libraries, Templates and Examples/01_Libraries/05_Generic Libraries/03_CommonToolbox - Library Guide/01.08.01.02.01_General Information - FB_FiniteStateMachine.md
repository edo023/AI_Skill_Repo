# General Information - FB\_FiniteStateMachine

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.2.9.0 |

## Task

Facilitates the implementation of a finite state machine.

## Description

This function block supports you on implementing a finite state machine in your application.

## Example

The following example shows how the finite state machine can be implemented:

```
PROGRAM SR_Main
VAR
        fbFiniteStateMachine : SE_CTBX.FB_FiniteStateMachine;
        fbTransitionLogger : SE_CTBX.FB_FsmTransitionLogger<50>;
END_VAR
```

```
IF NOT fbFiniteStateMachine.xInitialized THEN
        //Add trasition logger to finite state machine
        fbFiniteStateMachine.AddTransitionLogger(
                i_ifTransitionLogger := fbTransitionLogger
        );
        //Set initial state of finite state machine applied after first update
        fbFiniteStateMachine.diInitialState := 10;
END_IF

//Cyclic call of update method to initialize finite state machine and to process detected state change requests
fbFiniteStateMachine.Update();

CASE fbFiniteStateMachine.diState OF
        10: //State 10
                IF fbFiniteStateMachine.xEntryAction THEN
                        //Executed first cycle after changing state
                        ;
                END_IF
                IF fbFiniteStateMachine.timElapsedTimeInState > T#2S THEN
                        fbFiniteStateMachine.SetNextState(i_diState := 20);
                END_IF
        20: //State 20
                IF fbFiniteStateMachine.timElapsedTimeInState > T#2S THEN
                        IF fbFiniteStateMachine.diPreviousState <> fbFiniteStateMachine.diState THEN
                                fbFiniteStateMachine.SetNextState(i_diState := 20, i_sEvent := 'Restart state');
                        ELSE
                                fbFiniteStateMachine.SetNextState(i_diState := 30);
                        END_IF
                END_IF
        30: //State 30
                IF fbFiniteStateMachine.timElapsedTimeInState > T#2S THEN
                        fbFiniteStateMachine.SetNextState(i_diState := 10);
                END_IF
                IF fbFiniteStateMachine.diNextState <> fbFiniteStateMachine.diState THEN
                        //Executed before changing to next state (comparable to an exit action)
                        ;
                END_IF
ELSE
        //Source code to be executed in case of unknown state
        ;
END_CASE
```

## Methods

| Name | Description |
| --- | --- |
| Reset | Resets the finite state machine. |
| SetNextState | Requests the transition to a different state. |
| Update | Performs the requested transition from the present state to the next state. |
| AplRegisterLoggerPoint | Registers a logger point into the Application Logger. |
| AddStateConversion | Adds a custom string conversion for the state values to the finite state machine. |
| AddTransitionLogger | Adds a transition logger to the finite state machine. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| diInitialState | DINT | Read / write | Initial state that is active after initialization of the function block done by the first call of the method Update.  NOTE: The set value is maintained after calling method Reset. |
| diPreviousState | DINT | Read | Indicates the state that was active before the last state transition. |
| diNextState | DINT | Read | Indicates the requested state. |
| diState | DINT | Read | Indicates the present state. |
| timElapsedTimeInState | TIME | Read | Indicates the time elapsed in the present state. |
| xEntryAction | BOOL | Read | Indicates that the entry action for the present state is active. Is TRUE after a state transition until the next call of the Update() method without a state transition. |
| xInitialized | BOOL | Read | Indicates if the finite state machine is initialized. If not, call the method Update to initialize the finite state machine. |
| uiNumberOfTransitions | UINT | Read | Indicates the number of transitions since the finite state machine was initialized. |
| etAplLogLevelLoggerPoint | APL.ET\_LogLevel | Read / write | Log level of the logger point registered at the Application Logger.  Default value: APL.ET\_LogLevel.StatusMessage |
