# FB\_MotorDataFileCreate2 - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.7.4.0 |

## Task

This function block creates a binary motor data file (.mdf). Also refer to the chapter [Asynchronous Motor](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.3rdPartyMotors&topicID=D_SE_0088217).

NOTE: Incorrect machine encoder typeplate data has a direct influence on the behavior of the controller.

| DANGER | |
| --- | --- |
|  | UNINTENDED OPERATION OF THE MOTOR  * The technical data of the motor manufacturer must be observed and respected. * The motor data of the parameterized variables must concur with the motor data of the manufacturer. * If the definition of the motor data does not correspond to the motor, you must convert it to the required motor data. * Ensure that no one is in the operating zone during the startup.  Failure to follow these instructions will result in death or serious injury. |

NOTE: A deactivated monitoring function can lead to a possible undetected overheating of the motor.

| DANGER | |
| --- | --- |
|  | OVERHEATING OF THE MOTOR  * The technical data of the motor manufacturer must be observed and respected. * The motor data of the parameterized variables must concur with the motor data of the manufacturer. * If the definition of the motor data does not correspond to the motor, you must convert it to the required motor data. * Use appropriate methods to verify thermal overload of the motor. * Ensure that no one is in the operating zone during the startup.  Failure to follow these instructions will result in death or serious injury. |

## Description

This file can be written to an axis by using [FB\_MotorDataWrite2](FB_MotorDataWrite2-GeneralInformati-493DCEE8.html). You must specify the name of the file and a structure that contains the motor data (refer to [ST\_UserMotorData](D-SE-0090544.html)).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xExecute | BOOL | On a rising edge, the file creation process is started. |
| i\_sFilename | STRING[255] | Name of the file that is created on the file system of the controller. |
| i\_stUserMotorData | [ST\_UserMotorData](D-SE-0090544.html) | Structure that contains the application data that is written into the file. |
| i\_xOverwrite | BOOL | If set to TRUE, an existing file is overwritten. |

| Output | Data type | Description |

| q\_xBusy | BOOL | Indicates TRUE, as long the process to create the motor data file is in progress. |
| q\_xDone | BOOL | Indicates TRUE, if the motor data file was created successfully. |
| q\_xError | BOOL | Indicates TRUE, if an error is detected while creating the motor data file. |
| q\_etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | General library-independent statement on the diagnostic.  A value unequal to GD.ET\_Diag.Ok corresponds to a diagnostic message. |
| q\_etDiagExt | [ET\_DiagExt](D-SE-0090520.html#D-SE-0090520) | POU-specific output on the diagnostic.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> diagnostic message |
| q\_sMsg | STRING[80] | Event-triggered message which gives more detailed information on the diagnostic state. |

## BrakeInvalid

| Cause | Solution |
| --- | --- |
| uiBrake has an invalid value. Only the values 0 and 1 are permitted. | Set uiBrake to a valid value. |

## ContinuousStallCurrentInvalid

| Cause | Solution |
| --- | --- |
| The standstill rated current is not within a valid range. | Adapt the value for the standstill rated current. The value must be > 0. |

## ContinuousStallTorqueInvalid

| Cause | Solution |
| --- | --- |
| The standstill torque is not within a valid range. | Adapt the value for the standstill rated torque. The value must be > 0. |

## CouldNotCreateFile

| Cause | Solution |
| --- | --- |
| There is already a file with the same name. | Use a different file name or enable overwriting of the existing file. |
| There is not enough memory available on the flash disk. | Clear disk space or use a flash disk with more memory. |
| The flash disk is inoperable. | Replace the flash disk. |

## InputStringTooLarge

| Cause | Solution |
| --- | --- |
| At least one of the parameters sMotorname, sMotorArticleNumber, or sMotorSerialnumber exceeds the maximum length of 20 characters. | Adapt the string length. A maximum of 20 characters is permitted. |
