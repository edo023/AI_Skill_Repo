## Common Libraries

In most cases, this is the standard type of a library.

* A common library should be referenced by other libraries by using a [placeholder](placeholder.html#placeholder_placeholder).
* A common library has been marked with the [property](glossary.html#glossary_glossary__glossary_term_Property) Placeholder (Text) := <Placeholder name>.
* A common library should be decorated with the [property](glossary.html#glossary_glossary__glossary_term_Property) LanguageModelAttribute(Text) := 'qualified-access-only'.
* A common library has been assigned a [prefix](glossary.html#glossary_glossary__glossary_term_Prefix) or [namespace](glossary.html#glossary_glossary__glossary_term_Namespace).
* The [file name](filename.html#filename_file_names) contains no type specific suffix.
