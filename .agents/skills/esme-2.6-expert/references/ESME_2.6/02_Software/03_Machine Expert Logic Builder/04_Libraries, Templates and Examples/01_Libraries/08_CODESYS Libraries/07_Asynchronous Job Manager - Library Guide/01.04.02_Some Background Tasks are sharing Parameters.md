## Some Background Tasks are sharing Parameters and Results

* Every instance of a [BackgroundTask](rJ9NnPzj8kCZsoW9jcVtGOgoM-k_backgroundtask.html#rj9nnpzj8kczsow9jcvtgogom_k_backgroundtask_backgroundtask_fb) function block is connected to one and the same instance of an [IAsyncActionProvider](CZue5CN2wzRfuhta_JgtNFmpobk_iasyncactionprov.html#czue5cn2wzrfuhta_jgtnfmpobk_iasyncactionprov_iasyncactionprovider_itf).
* One parameter queue is connected to the group of [BackgroundTask](rJ9NnPzj8kCZsoW9jcVtGOgoM-k_backgroundtask.html#rj9nnpzj8kczsow9jcvtgogom_k_backgroundtask_backgroundtask_fb)s. Each task will fetch one of the next parameter sets to its context for executing the [IAsyncActionProvider.AsyncAction](s0xPGJcx02wfS1E4gBcJR4aats8_asyncaction.html#s0xpgjcx02wfs1e4gbcjr4aats8_asyncaction_iasyncactionprovider_asyncaction_meth) with this set of parameters.
* The results of these joint efforts are available for the foreground task through the common SHD.ISharedArea.
