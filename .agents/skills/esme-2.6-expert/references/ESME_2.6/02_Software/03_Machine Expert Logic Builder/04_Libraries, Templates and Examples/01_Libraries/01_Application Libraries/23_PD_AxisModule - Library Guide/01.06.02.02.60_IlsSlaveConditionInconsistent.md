# IlsSlaveConditionInconsistent

IlsSlaveConditionInconsistent

|  |  |
| --- | --- |
| Enumeration name: | IlsSlaveConditionInconsistent |
| Enumeration value: | 66 |
| Description: | The Ils slave specification is inconsistent. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.MultiCam](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected, however, the specifications whether its is an IIs slave are contradictory. | Ensure that in [ST\_MultiCam.i\_udiSlaveNumber](../Structures/Structures-10.htm#XREF_D_SE_0077229_1) the slave number of the IIs slave is entered. If it is not an IIs slave, a 0 must be entered.  Ensure that the [ST\_MultiCam.pstIlsData](../Structures/Structures-10.htm#XREF_D_SE_0077229_1) pointer is assigned with the address of the Ils data. If it is not an IIs slave, a 0 must be entered.  Both the conditions mentioned earlier must be met simultaneously. |
