# ET\_RobotType - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.4.1.0 |

## Description

This enumeration type specifies the selected robot type. ET\_RobotType is used for EntitiesHandler.
