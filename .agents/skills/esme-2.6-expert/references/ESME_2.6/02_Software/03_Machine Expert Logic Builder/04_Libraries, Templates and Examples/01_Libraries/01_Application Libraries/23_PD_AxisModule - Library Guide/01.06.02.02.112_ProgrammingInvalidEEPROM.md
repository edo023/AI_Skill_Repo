ProgrammingInvalidEEPROM

|  |  |
| --- | --- |
| Enumeration name: | ProgrammingInvalidEEPROM |
| Enumeration value: | 90 |
| Description: | An error occurred while writing on the encoder EEPROM. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.WriteAxisEncoder](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).  Writing is already being performed on the encoder. | Please retry later. |
