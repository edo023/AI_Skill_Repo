# Call Robot in Your Program

## Overview

You must call the robot in your project.

## PacDrive 3 Template

| Step | Action |
| --- | --- |
| 1 | Copy the code snippets of the Code snippets tab (refer to [Explorer](Explorer-F6FADFEC.html#Explorer-F6FADFEC)) to the desired location in your application code. |
| 2 | If you are using PacDrive 3 Template, it is usually inside SubModules\_Action (of SR\_MainMachine or equivalent in a node module) which generally uses the FBD language. To use the code snippets of Configuration Data, you can use the EXECUTE box to add code in structured text. |
| 3 | Add the in/out variables specific to your application. The variables used are only an example. |

## Non Template

| Step | Action |
| --- | --- |
| 1 | Copy the code snippets of the Code snippets tab (refer to [Explorer](Explorer-F6FADFEC.html#Explorer-F6FADFEC)) to the desired location in your application code. |
| 2 | Copy the code snippets of the Code snippets tab (refer to [Explorer](Explorer-F6FADFEC.html#Explorer-F6FADFEC)) to the desired location in your application code. |

NOTE: If you add VAR\_INPUT, VAR\_OUTPUT or VAR\_INPUT\_OUTPUT to the ModuleInterface you must update the call of the robot.
