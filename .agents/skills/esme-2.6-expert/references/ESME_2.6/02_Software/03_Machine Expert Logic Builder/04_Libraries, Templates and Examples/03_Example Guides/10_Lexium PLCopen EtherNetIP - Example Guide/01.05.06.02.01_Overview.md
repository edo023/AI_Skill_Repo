# Overview

Overview

To run the application example, the following preconditions must be fulfilled:

oThe application was downloaded and is running on the associated controller.

oThe Lexium drives are configured for the control over the associated fieldbus. Further configuration of the drive is compatible with the connected motor and driven mechanical parts.

oA PC, laptop, or mobile device with a compatible web browser is connected to the same network as the controller so that the web visualization can be accessed.
