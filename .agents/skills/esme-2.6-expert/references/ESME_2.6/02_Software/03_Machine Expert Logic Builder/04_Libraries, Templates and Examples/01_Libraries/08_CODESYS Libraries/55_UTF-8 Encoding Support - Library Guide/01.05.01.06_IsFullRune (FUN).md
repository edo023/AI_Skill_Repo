# IsFullRune (FUN)

FUNCTION IsFullRune : BOOL

Reports whether the bytes in pbyData begin with a full UTF-8 encoding of a rune.

An invalid encoding is considered a full rune since it will convert as a error rune.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | IsFullRune | BOOL |  | | Input | pbyData | POINTER TO BYTE | Pointer to the first byte of the memory | | udiSize | UDINT | Size of the memory in bytes | |
