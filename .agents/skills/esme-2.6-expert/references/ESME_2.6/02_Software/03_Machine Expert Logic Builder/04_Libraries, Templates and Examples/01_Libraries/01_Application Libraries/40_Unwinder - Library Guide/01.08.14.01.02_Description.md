# Description

Description

This structure contains the parameters and feedback data for the start movement behavior of the FB\_Unwinder for a splicing procedure.

After starting the unwinder the bobbin is at stand still and the foil of the bobbin is not connected to the feeder and the dancer is on its minimum position (where the maximum amount of foil is inside the dancer). The master moves the foil from another unwinder. When the splicing is done, the foil of this unwinder is connected to the feeder (and cut from the other unwinder). Thereby the foil is tense and the dancer starts to move. If the position of the dancer exceeds i\_lrStart­Threshold, then the bobbin starts to accelerate. The acceleration is calculated such, so that the bobbin will have a circumference velocity identical to the velocity of the master when the dancer reached its reference position.
