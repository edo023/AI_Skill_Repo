## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.8.0 |
| Versions: | Current version |

## Task

Transfer and execute user-specific code to a task running asynchronously to the program.

NOTE: The interface IF\_AsyncManager2 extends the interface IF\_AsyncManager providing additional information about the created tasks and the status of asynchronous jobs in the queue.

## Description

The interface IF\_AsyncManager2 is implemented by this library and is used for the following purposes:

* Initialize the AsyncManager
* Assign the program part which has to be executed asynchronously to the calling task
* Retrieve information about created tasks, pending jobs, and active jobs

NOTE: It is not required to implement the interface IF\_AsyncManager2 in your application.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| udiActiveJobs | UDINT | Read | Indicates the number of running jobs. |
| udiCreatedTasks | UDINT | Read | Indicates the number of created asynchronous tasks. |
| udiPendingJobs | UDINT | Read | Indicates the number of pending jobs in the queue. |
| xInitDone | BOOL | Read | Signals whether initialization of the AsyncManager has been executed successfully. |
