# VisuFctSelectElement (FUN)

FUNCTION VisuFctSelectElement : DWORD

In the case of VISU\_SELECTION\_AT the input variable pPosition should be considered. In the case of VISU\_SELECTION\_ELEMENT the input variable iVisualElement should be considered.
