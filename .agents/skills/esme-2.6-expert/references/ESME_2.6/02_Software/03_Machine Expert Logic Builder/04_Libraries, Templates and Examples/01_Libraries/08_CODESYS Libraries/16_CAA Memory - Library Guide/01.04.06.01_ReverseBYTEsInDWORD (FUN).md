# ReverseBYTEsInDWORD (FUN)

FUNCTION ReverseBYTEsInDWORD : DWORD

Function returns the DWORD “dwInput” with a reversed byte-order.

Example:

```
dwInput = 16#01020304
->
MEM.ReverseWORDsInDWORD(dwInput) = 16#04030201
```
