## Task

Sets the owner of a slot inside a container target.

## Description

The method SetSlotOwnerInTarget allows you to set the owner of a slot inside a container target.

If the owner of the slot was set to ET\_SystemEntity.None and, as an effect of the call of this method, it is now set to a value different from ET\_SystemEntity.None, the value of uiNumberOfSlotsWithOwner inside the structure of type ST\_RobotTargetWithSlots linked to the container target is automatically increased.

If the owner of the slot was set to a value different from ET\_SystemEntity.None and, as an effect of the call of this method, it is now set to a value equal to ET\_SystemEntity.None, the value of uiNumberOfSlotsWithOwner inside the structure of type ST\_RobotTargetWithSlots linked to the container target, is automatically decreased.
