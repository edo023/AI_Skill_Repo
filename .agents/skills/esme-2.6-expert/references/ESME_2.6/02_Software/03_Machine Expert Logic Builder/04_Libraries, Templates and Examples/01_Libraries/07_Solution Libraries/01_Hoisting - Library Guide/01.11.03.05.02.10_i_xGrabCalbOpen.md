# i_xGrabCalbOpen

i\_xGrabCalbOpen

Rising edge on this input calibrates position difference corresponding to open state of the grab. You must perform calibration for the soft limit switch function to work. You can perform a new calibration without re-setting the previously calibrated value. If desired position cannot be reached because of previous calibration, you can reset the calibration values by rising edge on input i\_xGrabCalbRst.
