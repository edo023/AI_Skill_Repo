# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| q\_uiPathId | UINT | Path ID of present position |
| q\_rPosition | REAL | Position on present path in [m] calculated from start of path. |
| q\_xIsObstructed | BOOL | TRUE when vehicle is blocked by another vehicle. |
| q\_xIsJammed | BOOL | TRUE when vehicle is blocked by an object that is not under the management of the transport system controller. |
| q\_iPidSet | INT | Number relating to predefined parameter sets for the PID-Controllers when moving a vehicle:  0 - unloaded PID values,  1 - loaded PID values |
