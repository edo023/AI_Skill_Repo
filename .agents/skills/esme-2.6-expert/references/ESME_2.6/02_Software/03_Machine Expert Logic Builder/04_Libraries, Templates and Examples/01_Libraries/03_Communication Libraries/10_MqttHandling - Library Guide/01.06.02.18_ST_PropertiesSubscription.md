## Description

The structure ST\_PropertiesSubscription specifies the properties sent by the MQTT client inside the SUBSCRIBE and UNSUBSCRIBE messages.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stSubscribe | ST\_PropertiesUnSubscribe | Structure containing properties sent by the MQTT client to the MQTT broker inside the SUBSCRIBE message when subscribing to a topic. |
| stUnsubscribe | [ST\_PropertiesUnSubscribe](ST_PropertiesUnSubscr-D8FCAC2A.html) | Structure containing properties sent by the MQTT client to the MQTT broker inside the UNSUBSCRIBE message when unsubscribing from a topic. |
