# Description

Description

Returns the inverse matrix of a quadratic matrix of the dimension i\_diDim x i\_diDim. The pointer i\_plrMatrix must point to the start of the memory area in which the matrix is stored. The pointer i\_plrInvMatrix must point to the beginning of the memory area from where the result of the calculation, i.e. the inverse matrix should be copied. If a matrix cannot be inverted, it is referred to as a singular matrix.
