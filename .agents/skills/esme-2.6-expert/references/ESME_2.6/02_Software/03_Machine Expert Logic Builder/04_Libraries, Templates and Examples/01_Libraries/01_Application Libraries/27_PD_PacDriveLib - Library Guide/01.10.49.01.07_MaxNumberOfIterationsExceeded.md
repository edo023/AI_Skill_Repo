# MaxNumberOfIterationsExceeded

MaxNumberOfIterationsExceeded

|  |  |
| --- | --- |
| Enumeration name: | MaxNumberOfIterationsExceeded |
| Enumeration value: | 96 |
| Description: | The maximum number of iteration steps has been exceeded. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | i\_lrAccuracy has been chosen too small. The numerical approximation method converges to such an exact result in not less than [Gc\_diMaxNumberOfIterations](../Global_Elements/Global_Elements-2.htm#XREF_D_SE_0087806_1) iteration steps. | A higher value must be chosen for i\_lrAccuracy. |
