# LastVelBeltIndexRange

LastVelBeltIndexRange

|  |  |
| --- | --- |
| Enumeration name: | LastVelBeltIndexRange |
| Enumeration value: | 97 |
| Description: | LastVelBeltIndex is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid value has been transferred at the input iq\_stParameter.udiLastVelBeltIndex. | It must be made sure that a value is transferred at the iq\_stParameter.udiLastVelBeltIndex input that is greater than 0 and smaller than the number of parameterized belts of the infeed instance transferred at the i\_ifInfeed input. |
