# AutoTune Reports: Axis Position Out of Range (lResult - -10053)

AutoTune Reports: Axis Position Out of Range (lResult = -10053)

The position of the drive is not within the permitted range.

NOTE: It might not be possible to optimize drives with a highly limited position range with AutoTune!

| Cause | Removal |
| --- | --- |
| The start parameters were selected unfavorable for the optimization. Oscillations occurred on the drive that only allow an insufficient control over the compliance of the position range. | Start the optimization with default parameters.  Click on Default parameter in the visualization Results.  Then restart the optimization. |
| The controller parameters were not optimized or not optimized successfully, with the result that oscillation occurs at the drive which does not allow sufficient control over the observation of the position range. | Optimize the controller parameters and ensure that the optimization is completed successfully. |
| The position range defined in the visualization by min. Position and max. Position is too small. To optimize the axis a greater position range is required. | Increase the position range with min. Position and max. Position in the visualization. Ensure that the maximum position range permitted on the drive is not exceeded. |
