# xEncType

xEncType

Encoder type:

oTRUE = absolute

oFALSE = incremental

After a power cycle, if the configured encoder type does not correspond to the physical encoder type, an invalid position may be determined.

| --- |
| Warning_Color.gifWARNING |
| LOSS OF POSITION AFTER POWER CYCLE |
