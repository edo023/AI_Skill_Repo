# DiagnosisDataReader.InitData (METH)

METHOD InitData

Use this method to inititialize the FB, if the source of the diagnosis data is an acyclic read service (CommFB.RDREC).

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | pbyDiagBuffer | POINTER TO BYTE | Pointer to the ARRAY[0..n] of byte that contains the received data. (CommFB.RDREC.RECORD). | | udiDiagBufferSize | UDINT | Length of the received acyclic data (CommFB.RDREC.LEN) | |
