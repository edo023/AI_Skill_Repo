# ET\_EventSource

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_EventSource contains the sources of the event.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Device | 0 | The event is generated by an IO-Link device. Events generated by an IO-Link device are IO-Link standard compliant. |
| Master | 1 | The event is generated by the IO-Link master. |
