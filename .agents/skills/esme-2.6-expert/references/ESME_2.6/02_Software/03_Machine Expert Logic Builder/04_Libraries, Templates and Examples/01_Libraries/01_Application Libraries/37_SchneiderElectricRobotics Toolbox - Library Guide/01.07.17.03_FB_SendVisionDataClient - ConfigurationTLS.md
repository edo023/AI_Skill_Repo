## Task

Configure the IP address and the port of the server and the TLS settings.

## Description

With the method ConfigurationTLS, the IP address and the port can be configured for the server to monitor and the TLS (Transport Layer Security) settings that are used.

The configuration of the function block with this method will establish a secure connection, the client also must support a TLS connection.

If the server is not supporting TLS, the [ConfigurationNoTLS](#FB_SendVisionDataClient-Configurati-51168C06) method must be used.
