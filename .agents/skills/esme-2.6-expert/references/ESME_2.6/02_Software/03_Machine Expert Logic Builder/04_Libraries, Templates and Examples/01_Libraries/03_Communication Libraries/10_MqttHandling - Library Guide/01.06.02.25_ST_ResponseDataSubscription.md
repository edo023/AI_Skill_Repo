# ST\_ResponseDataSubscription

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_ResponseDataSubscription contains information extracted from the response sent by the MQTT broker after subscribing or unsubscribing or receiving a message to a subscribed topic.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| etSubackReasonCode | [ET\_ReasonCode](ET_ReasonCode-D887BF12.html) | Indicates the reason code from the last received message of type SUBACK sent by the MQTT broker. |
| etSubackProperties | [ST\_PropertiesUnSuback](ST_PropertiesUnSuback-D8FB1205.html) | Indicates the properties of the last received message of type SUBACK sent by the MQTT broker. |
| etUnSubackReasonCode | [ET\_ReasonCode](ET_ReasonCode-D887BF12.html) | Indicates the reason code in the last received message of type UNSUBACK sent by the MQTT broker. |
| etUnSubackProperties | [ST\_PropertiesUnSuback](ST_PropertiesUnSuback-D8FB1205.html) | Indicates the properties of the last received message of type UNSUBACK sent by the MQTT broker. |
| stPublishProperties | [ST\_PropertiesPublish](ST_PropertiesPublish-D8E854FB.html) | Indicates the properties of the last published message sent by the MQTT broker that matches the subscribed topic. |
| sTopicName | STRING[GPL.Gc\_uiMaxLengthOfReceivedTopicName] | Indicates the topic name of the last published message sent by the MQTT broker that matches the subscribed topic.  NOTE: If the length of the topic name exceeds the value defined with Gc\_uiMaxLengthOfReceivedTopicName, it is truncated at the front. |
