# BackgroundTask (FB)

FUNCTION\_BLOCK FINAL BackgroundTask EXTENDS FBF.InstanceBase IMPLEMENTS [IBackgroundTask](CZue5CN2wzRfuhta_JgtNFmpobk_ibackgroundtask.html#czue5cn2wzrfuhta_jgtnfmpobk_ibackgroundtask_ibackgroundtask_itf), ICmpEventCallback

Encapsulates all the details of managing a iec task in the context of the runtime system.

A running background task can be controlled via the methods of [IBackgroundTask](CZue5CN2wzRfuhta_JgtNFmpobk_ibackgroundtask.html#czue5cn2wzrfuhta_jgtnfmpobk_ibackgroundtask_ibackgroundtask_itf).

**Note**

| Watchdog Sensitivity: | If the actual cycle time of a task exceeds the currently udiWatchdogTime, the currently defined usiWatchdogSensitivity value is taken into account:   * Sensitivity 0, 1 => exception in cycle 1 * Sensitivity 2 => exception in cycle 2 * Sensitivity n => exception in cycle n * Single Timeout => Exception if the actual cycle time is longer than usiWatchdogSensitivity x udiWatchdogTime. |
