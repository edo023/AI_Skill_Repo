# DisturbanceValueRange

DisturbanceValueRange

|  |  |
| --- | --- |
| Enumeration name: | DisturbanceValueRange |
| Enumeration value: | 64 |
| Description: | DisturbanceValue is outside of the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The feedback signal origin mode ET\_FeedbackSignalOrigin.Simulation is chosen and iq\_stParameter.stFeedbackSignal.stSimulation.iq\_xCauseDisturbance is TRUE while an invalid value is chosen for the input iq\_stParameter.stFeedbackSignal.stSimulation.i\_lrDisturbanceValue. | At the input iq\_stParameter.stFeedbackSignal.stSimulation.i\_lrDisturbanceValue a value greater or equal than -100.0 and lower or equal to 100.0 must be specified. |
