# FB\_InitMachineEncoder - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.3.4.0 |
| Inherits from: | – |
| Implemented: | – |

## Task

This function block initializes the machine encoder with a typeplate. The parameters are defined in i\_stUserMachineEncoderData.

NOTE: Execute this POU only when required for application needs. The encoder memory area has a limited number of write cycles.

NOTE: Incorrect machine encoder typeplate data has a direct influence on the behavior of the controller.

| DANGER | |
| --- | --- |
|  | UNINTENDED OPERATION OF THE MOTOR  * Verify assignment of machine encoder data to the motor before writing. * Ensure that no one is in the operating zone during the startup.  Failure to follow these instructions will result in death or serious injury. |

## Requirements

Sercos must be in phase 2.

## Functional Description

This POU initializes the machine encoder with the parameters which are defined in i\_stUserMachineEncoderData. The machine encoder data is stored to machine encoder.

The machine encoder data is permanently stored. Thus, the machine encoder data must be written only once.

NOTE: The machine encoder data must be rewritten when the following applies:

* The machine encoder is replaced.
* Parameterization is interrupted (for example, by Sercos diagnostic messages or by power outage).
* The POU FB\_InitMachineEncoder is not successfully executed (diagnostic messages are issued).

NOTE: The POU has a long processing time (> 1 s). Thus, it must be executed in a task that is not time-critical and with an appropriate watchdog time.

## Disabled

| Cause | Solution |
| --- | --- |
| The POU is disabled. | Set the i\_xEnable input from FALSE to TRUE to enable the POU. |

## DriveInvalid

| Cause | Solution |
| --- | --- |
| No valid drive was applied at i\_ifDrive of FB\_InitMachineEncoder | A valid drive must be transferred to the i\_ifDrive. |

## DriveVirtual

| Cause | Solution |
| --- | --- |
| The WorkingState of the connected drive is not real. | Set WorkingMode of the connected drive to real. |

## Done

The motor data has been deleted successfully. New motor data may be written to the drive.

## EncoderLinesPerRevolutionInvalid

| Cause | Solution |
| --- | --- |
| An invalid value was transferred at the ST\_UserMachineEncoderData. uiEncoderLinesPerRevolution input. | A value between MTP.Gc\_uiEncoderLinesPerRevolutionMinValue and MTP.Gc\_uiEncoderLinesPerRevolutionMaxValue must be transferred to the input. |

## EncoderMaxSpeedInvalid

| Cause | Solution |
| --- | --- |
| An invalid value was transferred at the ST\_UserMachineEncoderData.uiEncoderMaxSpeed input. | A value between MTP.Gc\_uiEncoderMaxSpeedMinValue and MTP.Gc\_uiEncoderMaxSpeedMaxValue must be transferred to the input. |

## EncoderMaxTempInvalid

| Cause | Solution |
| --- | --- |
| An invalid value was transferred at the ST\_UserMachineEncoderData.uiEncoderMaxTemp input. | A value between MTP.Gc\_uiEncoderMaxTempMinValue and MTP.Gc\_uiEncoderMaxTempMaxValue must be transferred to the input. |

## EncoderNumberOfTurnsInvalid

| Cause | Solution |
| --- | --- |
| An invalid value was transferred at the ST\_UserMachineEncoderData.uiEncoderNumberOfTurns input. | A value between MTP.Gc\_uiEncoderNumberOfTurnsMinValue  and MTP.uiEncoderNumberOfTurnsMaxValue  must be transferred to the input. |

## EncoderTempSensorInvalid

| Cause | Solution |
| --- | --- |
| An invalid value was transferred at the ST\_UserMachineEncoderData.uiEncoderTempSensor input. | The value 0: no temperature sensor or 1: with temperature sensor must be transferred to the input. |

## Executing

The POU is in the executing phase.

## FunctionNotSupportedByThisDevice

| Cause | Solution |
| --- | --- |
| Invalid object type | Verify the selected object. |

## InvalidEncoderType

| Cause | Solution |
| --- | --- |
| An invalid value was transferred at the ST\_UserMachineEncoderData.etEncoderType input. | The value ET\_EncoderType.SincosHiperface must be transferred to the input. |

## InvalidStorageLocation

| Cause | Solution |
| --- | --- |
| There is no Hiperface encoder connected to the physical machine encoder input. | Connect a Hiperface encoder to the physical machine encoder input. |
| The encoder does not provide sufficient memory or there is no encoder. | Connect an encoder with Hiperface interface with a minimum of 2 Kb of physical memory. |

## MEncInObjectNotAppendedToDrive

| Cause | Solution |
| --- | --- |
| There is no MEncIn object appended to the drive. | Append the MEncIn object to the drive in the controller configuration. |
| The drive in controller configuration does not support a machine encoder object. | Set the drive in controller configuration to Lexium 62 Advanced Plus and append the MEncIn object to the drive in the controller configuration. |

## NoValidMachineEncoderConnected

| Cause | Solution |
| --- | --- |
| There is no SinCos Hiperface encoder connected to the machine encoder hardware input. | Connect a SinCos Hiperface encoder to the machine encoder hardware input. |

## Prepare

The POU is preparing the execution when q\_xBusy is TRUE.

## ReinitializationFailed

| Cause | Solution |
| --- | --- |
| Reinitialization was not successful. | Verify the Sercos phase. |
| The status of the procedure command Reinit cannot be verified. | Increase the value of the property timDelayPCMDReinitCheckStatus. |

## SercosCommunicationNotPossible

| Cause | Solution |
| --- | --- |
| Sercos communication is not possible. | Verify the Sercos phase and the wiring. |

## SercosNotInPhaseTwo

| Cause | Solution |
| --- | --- |
| Sercos communication is not possible. The Sercos needs to be in phase 2 to be able to write to the machine encoder. | Set Sercos to phase 2. |

## WaitForExecute

The POU is active and ready for execution. q\_xReady is TRUE.

## Examples

```
VAR
    fbInitMachineEncoder     : MTP.FB_InitMachineEncoder;
    stUserMachineEncoderData : MTP.ST_UserMachineEncoderData;
    xInitMachineEncoderData  : BOOL := TRUE;
    xEnable                  : BOOL;
    xExecute                 : BOOL;
END_VAR
```

```
IF xInitMachineEncoderData THEN
    stUserMachineEncoderData.etEncoderType := MTP.ET_EncoderType.SincosHiperface;
    stUserMachineEncoderData.uiEncoderMaxSpeed           := 12000;	    stUserMachineEncoderData.uiEncoderMaxTemp            := 130;
    stUserMachineEncoderData.uiEncoderTempSensor         := 1;
    stUserMachineEncoderData.uiEncoderNumberOfTurns      := 4096;	    stUserMachineEncoderData.uiEncoderLinesPerRevolution := 128;
END_IF
```

```
fbInitMachineEncoder(
    i_xEnable                    := xEnable,
    i_xExecute                   := xExecute,
    i_ifDrive                  := DRV_Lexium62AdvancedPlus,
    i_stUserMachineEncoderData := stUserMachineEncoderData,
   );
```
