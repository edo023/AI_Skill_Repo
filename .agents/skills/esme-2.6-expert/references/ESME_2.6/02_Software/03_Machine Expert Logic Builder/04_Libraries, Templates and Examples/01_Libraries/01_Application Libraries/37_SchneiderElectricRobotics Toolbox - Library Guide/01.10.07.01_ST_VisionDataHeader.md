# ST\_VisionDataHeader

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

Structure to read the header data of a protocol.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sHeaderId | STRING[7] | Contains the string Vision indicating that data is provided by a camera. |
| uiProtocolLength | UINT | Size of the protocol length. |
| udiProtocolIndex | UDINT | Index of the data protocol. As each data protocol has an unique index, it can be used to detect unsuccessful communications. |
| uiVisionId | UINT | ID of the vision system; it can be used to identify a vision system if several are used. |
