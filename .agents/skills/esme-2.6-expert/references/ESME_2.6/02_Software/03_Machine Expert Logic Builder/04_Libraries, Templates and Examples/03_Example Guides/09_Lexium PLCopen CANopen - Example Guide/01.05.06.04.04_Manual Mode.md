# Manual Mode

Manual Mode

The following operations are available:

oClick the Manual Mode Request button to enter the manual mode.

oClick the Manual Mode Exit button to leave the manual mode.

oSelect the drives you want to operate in Jog-Mode.

oSelect the speed for jog operation.

oClick the JogFwd or JogRev buttons to start a movement of connected motor in the according direction.

NOTE: Only if manual mode is in progress the buttons are used to control the drives.

The movement ends as soon you release the respective button.
