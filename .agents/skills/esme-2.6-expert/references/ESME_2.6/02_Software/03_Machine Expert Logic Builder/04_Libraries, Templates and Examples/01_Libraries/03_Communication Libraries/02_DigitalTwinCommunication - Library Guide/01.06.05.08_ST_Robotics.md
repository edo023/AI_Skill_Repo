## Description

The structure ST\_Robotics contains arrays of robot objects that can be built and emulated in EcoStruxure Machine Expert Twin.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| astRobotPSeries | ARRAY [1..GCL.Gc\_udiMaxNumberOfRobotPSeries] OF ST\_DataRobotPSeries | The Lexium P robots that can be built and emulated. |
| astRobotTSeries | ARRAY [1..GCL.Gc\_udiMaxNumberOfRobotTSeries] OF ST\_DataRobotTSeries | The Lexium T robots that can be built and emulated. |
| astToolDefault | ARRAY [1..GCL.Gc\_udiMaxNumberOfToolsDefault] OF ST\_DataToolDefault | The default tools for robots that can be built and emulated. |
| astRobotSCARA | ARRAY [1..GCL.Gc\_udiMaxNumberOfRobotSCARA] OF ST\_DataRobotSCARA | The SCARA robots that can be built and emulated. |
