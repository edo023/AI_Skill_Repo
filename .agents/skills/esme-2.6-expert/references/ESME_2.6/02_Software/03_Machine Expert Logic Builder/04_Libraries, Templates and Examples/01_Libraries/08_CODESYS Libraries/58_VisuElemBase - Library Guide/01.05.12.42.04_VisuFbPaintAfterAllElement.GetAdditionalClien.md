# VisuFbPaintAfterAllElement.GetAdditionalClientDataSize (METH)

METHOD GetAdditionalClientDataSize : DINT

Returns the additional number of bytes, that a visual element needs to register in the client specific data in order to store element specific data. If no additional memory is needed, 0 has to be returned. For the correct alignment an instance of VisuFbAdditionalClientDataHelper should be used to compute the data size, where the data is aligned correctly.
