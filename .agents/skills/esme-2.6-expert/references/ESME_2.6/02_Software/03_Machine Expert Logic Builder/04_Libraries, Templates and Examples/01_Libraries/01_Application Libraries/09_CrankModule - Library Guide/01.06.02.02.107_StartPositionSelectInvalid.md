# StartPositionSelectInvalid

StartPositionSelectInvalid

|  |  |
| --- | --- |
| Enumeration name: | StartPositionSelectInvalid |
| Enumeration value: | 142 |
| Description: | Using StartPositionSelect is not allowed. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The [ST\_MultiCam](../Structures/Structures-9.htm#XREF_D_SE_0088766_1).iq\_stExt.xStartPositionSelect parameter is set to TRUE. | The module does not support the function. Set the parameter [ST\_MultiCam](../Structures/Structures-9.htm#XREF_D_SE_0088766_1).iq\_stExt.xStartPositionSelect to FALSE.  Use the operation mode Positioning to move the axis to a position before you change to the operation mode MultiCam. |
