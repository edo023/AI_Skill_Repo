## Description

A set of parameters describing the kinematics and the geometry of a SCARA4Ax robot. For more information, refer to [ST\_SCARA4AxKinematics](ST_SCARA4AxKinematicsGeneralInforma-9F77E50A.html) and [ST\_SCARA4AxGeometry](ST_SCARA4AxGeometryGeneralInformati-9F6C0310.html).

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stKinematics | ST\_SCARA4AxKinematics | A set of parameters used by the kinematics of the robot. |
| stGeometry | ST\_SCARA4AxGeometry | A set of parameters describing the geometry of the robotic structure. |
