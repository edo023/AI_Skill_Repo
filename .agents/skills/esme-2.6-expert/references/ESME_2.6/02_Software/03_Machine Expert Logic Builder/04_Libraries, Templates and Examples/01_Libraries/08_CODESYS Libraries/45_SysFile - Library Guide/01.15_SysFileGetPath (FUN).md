# SysFileGetPath (FUN)

FUNCTION SysFileGetPath : RTS\_IEC\_RESULT

Get the path of this file.

If a path is specified in the filename, the path will be extracted from the filename.

If no path is specified in the filename, the standard path for this file extension type will be returned.

Can contain an absolute or relative path | | Inout | szPath | STRING | Path for this file | | Input | diMaxLen | DINT | Maximum size in bytes of path length | |
