# ST\_UserProperty

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_UserProperty specifies the name and value pair required for a user property.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sName | STRING[GPL.Gc\_uiMaxLengthOfUserProperties] | Indicates the name of the user property.  NOTE: If the maximum length is exceeded, the string is truncated and the output q\_xTruncatedResponseData is set to TRUE. |
| sValue | STRING[GPL.Gc\_uiMaxLengthOfUserProperties] | Indicates the value of the user property.  NOTE: If the maximum length is exceeded, the string is truncated and the output q\_xTruncatedResponseData is set to TRUE. |
