# PeriodRange

PeriodRange

|  |  |
| --- | --- |
| Enumeration name: | PeriodRange |
| Enumeration value: | 26 |
| Description: | Period is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid value has been applied at the [ST\_EndlessFeed.i\_lrPeriod](../Structures/Structures-4.htm#XREF_D_SE_0077217_1) input. | A value greater than or equal to 1.0 must be transferred to the [ST\_EndlessFeed.i\_lrPeriod](../Structures/Structures-4.htm#XREF_D_SE_0077217_1) input. |
| - | A negative value has been applied at the [ST\_Manual.i\_lrPeriod](../Structures/Structures-8.htm#XREF_D_SE_0077225_1) input. | A value greater than 0 must be transferred to the [ST\_Manual.i\_lrPeriod](../Structures/Structures-8.htm#XREF_D_SE_0077225_1) input. |
| - | The operating mode [ET\_OpMode.MultiCam](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected and the Period parameter of the logical encoder at the [ST\_Main.iq\_lencEncoder](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) input has been set to a value unequal to 0. | Ensure that the Period parameter of the logical encoder at the [ST\_Main.iq\_lencEncoder](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) input is set to 0.  Disable the check of the period of the logical encoder by setting the [ST\_MultiCam.i\_xCheckEncoderPeriod](../Structures/Structures-10.htm#XREF_D_SE_0077229_1) parameter to FALSE. |
