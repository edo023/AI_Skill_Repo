# ET\_State

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.4.0 |

## Description

The enumeration ET\_State describes the elements used as states in the state machine of the function blocks.

In the state machine diagrams, states are represented in the upper part of the yellow boxes (also refer to the chapter [Legend of State Machine Diagrams](StateMachTPC-D1DD728B.html)).

## Enumeration Elements

| Name | Value (INT) | Description |
| --- | --- | --- |
| Idle | 0 | The function block is in idle state. |
| Activating | 1 | The function block is activating. |
| Ready | 2 | The function block is ready for execution. |
| WaitForExecute | 3 | The function block is waiting to execute. |
| Executing | 4 | The function block is executing functions. |
| Resetting | 5 | The function block is resetting. |
| Deactivating | 6 | The function block is deactivating. |
| Aborting | 7 | Execution of the function block is aborting. |
| WaitForReset | 8 | The function block is waiting to be reset. |
| WaitForExecuteFalse | 9 | The function block is waiting for a falling edge of the input i\_xExecute. |
| WaitForEnableFalse | 10 | The function block is waiting for a falling edge of the input i\_xEnable. |
