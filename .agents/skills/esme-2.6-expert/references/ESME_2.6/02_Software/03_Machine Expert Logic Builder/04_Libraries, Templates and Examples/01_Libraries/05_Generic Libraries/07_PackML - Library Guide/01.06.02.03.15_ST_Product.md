# ST\_Product

## Overview

|  |  |
| --- | --- |
| Type | Data structure |
| Available as of | V1.0.1.0 |

## Description

The structure ST\_Product is a collection of tags used to describe the product that the machine is making.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| ProductID | DINT | ID of the product, used to indicate to the machine which product it is producing. |
| ProcessVariable | ARRAY [0..GPL.Gc\_uiNumberOfProcessVariable] OF ST\_Descriptor | Structured array describes the specific process variables, which are needed or were used by the machine to process the specified product. |
| Ingredients | ARRAY [0..GPL.Gc\_uiNumberOfIngredients] OF ST\_Ingredient | Structured array, holds the information about the raw materials, which are needed or were used by the machine to process the specified product. |
