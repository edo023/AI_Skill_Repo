# XStartRange

XStartRange

|  |  |
| --- | --- |
| Enumeration name: | XStartRange |
| Enumeration value: | 119 |
| Description: | XStart is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | At the input i\_lrXStart, a number that lies outside of the cam has been applied. | Verify i\_lrXStart and set it to a value exceeding i\_lrXOffset.  Verify i\_lrXStart and set it to a value smaller than i\_lrXOffset + i\_lrXFactor - i\_lrXIncr. |
