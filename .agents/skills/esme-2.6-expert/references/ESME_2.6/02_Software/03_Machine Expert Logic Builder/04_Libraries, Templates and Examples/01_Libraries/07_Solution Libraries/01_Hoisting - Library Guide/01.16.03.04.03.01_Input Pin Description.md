# Input Pin Description

Input Pin Description

| Input | Data Type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block.  Refer also to [detailed description](#XREF_D_SE_0021101_5) of i\_xEn. |
| i\_xFwd | BOOL | Forward command from the hoist axis.  NOTE: It is not possible to run forward (i\_xFwd) and reverse (i\_xRev) at the same time. The FB accepts the first command in time if both inputs are TRUE.  TRUE: Forward  FALSE: Not forward |
| i\_xRev | BOOL | Reverse command from the hoist axis.  NOTE: It is not possible to run forward (i\_xFwd) and reverse (i\_xRev) at the same time. The FB accepts the first command in time if both inputs are TRUE.  TRUE: Reverse  FALSE: Not reverse |
| i\_xRstOps | BOOL | Reset input to clear the following retained values on a rising edge:  oq\_rOpHrs  oq\_dwOpNb  oq\_dwPlsNb  oq\_dwBtrkNb  TRUE: Active  FALSE: Inactive |
