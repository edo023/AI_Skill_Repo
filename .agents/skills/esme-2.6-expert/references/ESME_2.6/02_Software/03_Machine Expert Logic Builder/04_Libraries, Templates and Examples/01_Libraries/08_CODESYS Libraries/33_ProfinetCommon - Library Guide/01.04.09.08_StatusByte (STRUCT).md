# StatusByte (STRUCT)

TYPE StatusByte : STRUCT

The Status Byte is contained in each safety PDU of a (Sub)Module transmitted from a device to its controller

| InOut: | | Name | Type | Comment | | --- | --- | --- | | iPar\_OK | BIT | Bit0 F-(Sub)Module acknowledges iParameter assignment | | Device\_Fault | BIT | Bit1 at F\_Passivation = 0: Device\_Fault - Fault in F-(Sub)Module  Bit1 at F\_Passivation = 1: ChF\_Ack\_Req - F-(Sub)Module channel fault cleared | | CE\_CRC | BIT | Bit2 Communication fault: CRC | | WD\_timeout | BIT | Bit3 Communication fault: WD-timeout | | FV\_activated | BIT | Bit4 Fail-safe values (FV) activated | | Toggle\_d | BIT | Bit5 Toggle Bit | | cons\_nr\_R | BIT | Bit6 MonitoringNumber has been reset | | BaseID\_Active\_d | BIT | Bit7 Optional feature BaseID is active | |
