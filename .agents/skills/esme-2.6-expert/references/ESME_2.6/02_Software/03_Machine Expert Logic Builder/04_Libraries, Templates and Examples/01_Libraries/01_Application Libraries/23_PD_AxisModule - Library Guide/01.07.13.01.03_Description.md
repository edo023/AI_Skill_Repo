# Description

Description

This function will link an axis specified by i\_ifDrive to a master axis and a logical encoder. The links are stored in the axis’s interface structure specified by iq\_stAxisModuleItf.

The velocity at which an axis is considered moving is specified with the i\_lrMotionActiveV­elLimit input. The xAxisMove output of the axis’s interface structure is TRUE whenever the absolute value of the i\_etMotionActiveVelType reference velocity is greater than this value.

Normally, the logic encoder is compensated for one SERCOS clock cycle because it’s assumed the logical encoder will follow a virtual or real axis. The dead time compensation of the associated logical encoder (LE.delay) is thus changed when the associated master axis is a physical encoder. The delay parameter is set to the associated axis’s delay (- Drive.ShaftRefDelay).
