# i_xGrabCalbClosed

i\_xGrabCalbClosed

Rising edge on this input calibrates position difference corresponding to closed state of the grab. Perform calibration for the soft limit switch function to work. You can perform a new calibration without re-setting the previously calibrated value. If desired position cannot be reached because of previous calibration, you can reset the calibration values by rising edge on input i\_xGrabCalbRst.
