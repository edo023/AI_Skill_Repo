# PackBytesToDword (FUN)

FUNCTION PackBytesToDword : DWORD

Function pack four BYTEs into a DWORD.

Example:

```
MEM.PackBytesToDword(16#01,16#02,16#03,16#04) = 16#01020304
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | PackBytesToDword | DWORD | functions returns the DWORD defined by the BYTE inputs | | Input | byHHByte | BYTE | HH byte of the Word | | byHLByte | BYTE | HL byte of the Word | | byLHByte | BYTE | LH byte of the Word | | byLLByte | BYTE | LL byte of the Word | |
