# Methods

Methods

| Name | Description |
| --- | --- |
| [Calculate](Function_Blocks-4.htm#XREF_D_SE_0089451_1) | Calculating the CSpline reference positions is part of the [IF\_CamReferenceGenerator](../Interfaces/Interfaces-3.htm#XREF_D_SE_0089474_1) interface and does not have to be called up by the user. It can be used to issue slave positions, depending on the master positions. |
| [PrepareEqDisX](Function_Blocks-5.htm#XREF_D_SE_0089452_1) | Generating a CSpline with equidistant X values. |
| [PrepareVarDisX](Function_Blocks-6.htm#XREF_D_SE_0089453_1) | Generating a CSpline with discretionary X values. |
| [Reset](Function_Blocks-7.htm#XREF_D_SE_0089454_1) | This method releases the dynamic memory that was used for the last generated CSpline. When a new CSpline is generated this method does not have to be called up. The used dynamic memory is already released automatically. This method has to be called up if no further CSplines have to be generated. |
