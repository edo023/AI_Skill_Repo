# SMC\_NCNavigator (FB)

FUNCTION\_BLOCK SMC\_NCNavigator

This function block selects one of a given number of G-sentence queues. The selection is done based on the called subprogram and controlled by [SMC\_NCInterpreter](VVFvsggjXKxZ6qmLw10MvZPswnU_smc-ncinterprete.html#vvfvsggjxkxz6qmlw10mvzpswnu_smc_ncinterprete_smc_ncinterpreter_fb).

Additionally to subprogram calls and returns, this function block also handles jumps to sentence numbers, jumps to jump targets, and step suppression.

**Note**

This function block is included in SMC\_ReadNCFile2 and is needed in CNC applications only in special cases.
