# IAlarmHandler6.AcknowledgeAllWithComment (METH)

METHOD AcknowledgeAllWithComment

Acknowledges all active alarms, whose alarm class hasn’t the flag “acknowledge separately” set. Optionally a comment and the user id can be provided.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Inout Const | wsComment | WSTRING | The comment for the state transition | | wsUserId | WSTRING | The user who provides the comment | |
