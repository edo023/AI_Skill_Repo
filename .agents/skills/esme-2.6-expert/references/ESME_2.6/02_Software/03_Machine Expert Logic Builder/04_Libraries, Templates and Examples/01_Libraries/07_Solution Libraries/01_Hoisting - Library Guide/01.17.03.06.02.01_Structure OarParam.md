# Structure: OarParam

Structure: OarParam

Structure of parameters related to the behavior of the crane

| Parameter | Data type | Description |
| --- | --- | --- |
| wBridgeSpdNom | WORD | Nominal speed of bridge travel motor  Range: 1...32767  Unit: RPM |
| rBridgeSpdLin | REAL | Linear speed of the bridge travel at nominal motor speed  Range: 0.1...3.438  Unit: m/s |
| wBridgeDec | WORD | Deceleration time of bridge travel motor  Range: 1...65535  Unit: 0.1 s |
| rBridgeClrnc | REAL | Bridge clearance  Range: 0.0...3.438  Unit: m |
| rBridgeRestrMarg | REAL | Distance to allow movement of the bridge travel out of the restricted area  Range: 0.0...3.438  Unit: m |
| xBridgeAsw | BOOL | TRUE: The function block accepts the value connected to i\_rBridgeDistToStopAsw input as the stopping distance.  xBridgeAsw is TRUE if the anti-sway function on the bridge movement is active. |
| wTrolSpdNom | WORD | Nominal speed of trolley motor  Range: 1...32767  Unit: RPM |
| rTrolSpdLin | REAL | Linear speed of the trolley at nominal motor speed  Range: 0.1...3.438  Unit: m/s |
| wTrolDec | WORD | Deceleration time of trolley motor  Range: 1...65535  Unit: 0.1 s |
| rTrolClrnc | REAL | Trolley clearance  Range: 0.0...3.438  Unit: m |
| rTrolRestrMarg | REAL | Distance to allow movement of the trolley out of the restricted area  Range: 0.0...3.438  Unit: m |
| xTrolAsw | BOOL | TRUE: The function block accepts the value connected to i\_rTrolDistToStopAsw input as the stopping distance.  xTrolAsw is TRUE if the anti-sway function on the trolley movement is active. |
| wHoistSpdNom | WORD | Nominal speed of hoist motor  Range: 1...32767  Unit: RPM |
| rHoistSpdLin | REAL | Linear speed of the hoist at nominal motor speed  Range: 0.1...3.438  Unit: m/s |
| wHoistDec | WORD | Deceleration time of hoist motor  Range: 1...65535  Unit: 0.1 s |
| rHoistClrnc | REAL | Hoist clearance  Range: 0.0...3.438  Unit: m |
