# ID (FUN)

FUNCTION ID : DWORD

This function converts the physical identification of a slot or a DP-slave to a handle which can be used with Communication Function Blocks.

| InOut: | | Scope | Name | Type | Initial | Comment | | --- | --- | --- | --- | --- | | Return | ID | DWORD |  |  | | Input | EN | BOOL | TRUE |  | | MASTER | BYTE |  | ID of the DP system | | SEGMENT | BYTE |  | Number of the DP segment | | STATION | BYTE |  | Number of the DP-slave(station address) | | SLOT | BYTE |  | Slot number | |
