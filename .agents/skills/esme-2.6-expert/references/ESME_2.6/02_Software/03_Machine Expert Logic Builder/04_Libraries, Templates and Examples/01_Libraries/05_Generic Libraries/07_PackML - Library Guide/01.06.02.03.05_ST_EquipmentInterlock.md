# ST\_EquipmentInterlock

## Overview

|  |  |
| --- | --- |
| Type | Data structure |
| Available as of | V1.1.0.0 |

## Description

The structure ST\_EquipmentInterlock indicates whether the downstream and upstream system is the reason for the suspended state of a machine.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| Blocked | BOOL | TRUE: Indicates that a downstream system is not able to accept product. |
| Starved | BOOL | TRUE: Indicates that an upstream system is not able to supply product |
