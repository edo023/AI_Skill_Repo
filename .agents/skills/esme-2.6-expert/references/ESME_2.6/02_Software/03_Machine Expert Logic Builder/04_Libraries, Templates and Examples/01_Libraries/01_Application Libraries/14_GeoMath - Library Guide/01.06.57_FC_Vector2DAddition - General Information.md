## Description

Given two 2D input vectors, the function returns the element-wise addition of their elements.

![](images/0000611905.png)

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stVectorA | SE\_MATH.ST\_Vector2D | First 2D vector to be added. |
| i\_stVectorB | SE\_MATH.ST\_Vector2D | Second 2D vector to be added. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-93D70399.html#ET_Result-GeneralInformation-93D70399) | Provides diagnostic and status information. The enumeration ET\_Result contains the possible values of the POU operation results. |
