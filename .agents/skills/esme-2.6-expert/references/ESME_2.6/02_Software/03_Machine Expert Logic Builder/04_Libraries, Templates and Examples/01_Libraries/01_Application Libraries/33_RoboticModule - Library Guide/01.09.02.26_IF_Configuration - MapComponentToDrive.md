## Task

Maps the movement of a robot component directly to a drive.

## Description

The method can be used to enable an additional robot component and map its movement directly to a drive that has been added by using the method [AddDrive](IF_RobotConfiguration-AddDriveMetho-74AA4031.html).

To be able to do the mapping, the main transformation must be configured.

Only cartesian and orientation components that are not used by the configured main transformation can be enabled and mapped with this method.

The mapping must target a drive that has been added to the robot by using the method [AddDrive](IF_RobotConfiguration-AddDriveMetho-74AA4031.html) and is not used for another component.

A combination of a direct mapping in combination with a transformation for the additional components (Refer to the method [AdditionalComponentTransformation](IF_Configuration-AdditionalComponen-74A6A539.html)) is not possible. In case there are components which require a transformation and components with a direct mapping, all components must be handled in the transformation.
