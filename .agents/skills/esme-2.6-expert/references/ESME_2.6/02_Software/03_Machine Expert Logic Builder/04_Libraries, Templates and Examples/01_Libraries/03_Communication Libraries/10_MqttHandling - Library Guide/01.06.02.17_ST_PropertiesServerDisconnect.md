# ST\_PropertiesServerDisconnect

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_PropertiesServerDisconnect indicates the properties inside the DISCONNECT message received from the MQTT broker.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sReasonString | STRING[255] | Indicates a human readable string representing the reason for disconnecting. NOTE: The length of the value is limited to 255 characters. If it exceeds the maximum length, it is truncated and the output q\_xTruncatedResponseData is set to TRUE. |
| sServerReference | STRING[255] | Indicates the reference to another server to be used by the client. NOTE: The length of the value is limited to 255 characters. If it exceeds the maximum length, it is truncated and the output q\_xTruncatedResponseData is set to TRUE. |
| astUserProperties | ARRAY[1..GPL.Gc\_uiMaxNumberOfUserProperties] OF ST\_UserProperty | Indicates the user properties received from the MQTT broker.  NOTE: A user property with an empty name represents the end of the list. |
