# AccRange

AccRange

|  |  |
| --- | --- |
| Enumeration name: | AccRange |
| Enumeration value: | 14 |
| Description: | Acc is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operation mode Endless was selected and the i\_lrAcc parameter of the [ST\_EndlessFeed](../Structures/Structures-3.htm#XREF_D_SE_0088754_1) structure was allocated with an invalid value. | The following must apply: 0 < i\_lrAcc < MaxAcc parameter of the drive. |
| - | The operation mode Positioning was selected and the i\_lrAcc parameter of the [ST\_Positioning](../Structures/Structures-10.htm#XREF_D_SE_0088768_1) structure was allocated with an invalid value. | The following must apply: 0 < i\_lrAcc < MaxAcc parameter of the drive. |
| - | The operation mode EndlessIIs was selected and the i\_lrAcc parameter of the [ST\_EndlessFeed](../Structures/Structures-3.htm#XREF_D_SE_0088754_1) structure was allocated with an invalid value. | The following must apply: i\_lrAcc > 1. |
