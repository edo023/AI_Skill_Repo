# IVisualizationClient (ITF)

INTERFACE IVisualizationClient EXTENDS \_\_SYSTEM.IQueryInterface

Shows the information, needed to choose a client (with the client filter). See also IVisualizationClientRaw if the pointer to the VisuStructClientData is needed.

**Note**

This interface is not intended for being implemented by functionblocks outside this library as it might be extended in the future.
