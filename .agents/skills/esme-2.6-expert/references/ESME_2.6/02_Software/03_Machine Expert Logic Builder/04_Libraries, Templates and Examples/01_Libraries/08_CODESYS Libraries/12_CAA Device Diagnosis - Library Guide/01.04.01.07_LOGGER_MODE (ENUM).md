# LOGGER\_MODE (ENUM)

TYPE LOGGER\_MODE :

This enumeration lists all possible logger modes for a bus logger.

| Attributes: | qualified\_only |
| InOut: | | Name | Comment | | --- | --- | | LINEAR | Logger uses a linear logging buffer and stops logging if buffer is full. | | CIRCULAR | Logger uses a circular logging buffer and overwrites old entries if buffer is full. | |
