# SysFileGetTime (FUN)

FUNCTION SysFileGetTime : RTS\_IEC\_RESULT

Get file time of the specified file.

A standard path will be added to the filename, if no path is specified.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | SysFileGetTime | RTS\_IEC\_RESULT | The runtime system error code (see CmpErrors.library) | | Input | szFileName | STRING | File name. File name can contain an absolute or relative path to the file. Path entries must be separated with a Slash (/) and not with a Backslash (\)! | | ptFileTime | POINTER TO [SYS\_FILETIME](SYS_FILETIME.html#sys_filetime_sys_filetime_struct) | Pointer to get the file time results. | |
