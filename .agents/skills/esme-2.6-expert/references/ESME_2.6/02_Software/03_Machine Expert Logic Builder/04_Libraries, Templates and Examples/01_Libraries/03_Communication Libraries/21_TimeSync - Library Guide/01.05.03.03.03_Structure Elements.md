# Structure Elements

Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| q\_stSyncTimestamp | SysTimeRtc.RTS\_SYSTIMEDATE | Contains the synchronized time stamp (including the offset i\_diTimeZone) presented in a readable format. Structure provided by the SysTimeRtc [library](../../../../../../api/crossBook?lang=en-US&virtualBookName=RTCmanag&topicID=D_SE_0067239_1). |
| q\_uliUnixTimestamp | ULINT | Synchronized time stamp (including the offset i\_diTimeZone) as UNIX time stamp. |
| q\_liRtcOffset | LINT | Calculated clock offset (in ms) between local [RTC](../glossary/glossary.htm#XREF_D_SE_0024697_729) of the controller and the (S)NTP time server.  Negative values are possible, for example -2 h = -7200000. |
| q\_udiRoundTripDelay | UDINT | Calculated round-trip delay (in ms).  Example: 30 = 30 ms |
