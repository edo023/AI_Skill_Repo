# i_rKpDrft

i\_rKpDrft

The parameter contains the proportional gain of the drift controller. Optimal value depends on the crane mechanics. Set this value after the i\_rKpSkew has been configured. Setting this input to 0 disables the drift correction, but leaves the skew correction active.

For more information on configuration of controller gains, check the [Commissioning Procedure](Anticrab_2-13.htm#XREF_D_SE_0033860_1) section.
