## Functional Description

The method Trigger is called automatically when the property xTrigger is set to TRUE. It allows you to process trigger relevant actions, for example setting an output to send a hardware trigger signal to a sensor via a cable.

An input value is required to trigger the sensor target generation.

**Property xTrigger:**

* With the property xTrigger a signal is generated that triggers the target simulation.
* You can add code to the method Trigger that is called when the property xTrigger is set to TRUE, for example, to set an output to send a hardware trigger signal to a sensor via a cable.
* The usage of the xTrigger property is optional for the Trigger types Timer,Distance and TouchProbeCapture, but mandatory for the type External.

## Interface

The method has no input or output parameters.
