# Output Parameters

Output Parameters

| Parameter | Type | Comment |
| --- | --- | --- |
| xDone | BOOL | The reset after detected error(s) has been completed successfully. |
| xBusy | BOOL | The function block is active. |
| xError | BOOL | TRUE: NETVARRESETERROR function block execution error has been detected.  FALSE: No NETVARRESETERROR function block execution error has been detected. |
