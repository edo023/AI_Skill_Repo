# Interface

Interface

| Output | Data type | Description |
| --- | --- | --- |
| q\_xActive | BOOL | Indicates that the POU is activated and has to be selected. |
| q\_xReady | BOOL | Indicates the POU is enabled and working correctly |
| q\_etDiag | GD.ET\_Diag | Diagnostic class of the POU |
| q\_etDiagExt | ET\_DiagExt | Diagnosis code of the POU |
| q\_sMsg | STRING[80] | Message of the POU |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stMachineMotionItf | TPL.ST\_StandardMotionInterface | Specifies the standard motion interface for the machine |
| iq\_stMachineItf | TPL.ST\_StandardModuleInterface | Specifies the standard module interface for the machine |
| iq\_stExceptionList | TPL.ST\_ExceptionList | Specifies the global error detection list |
| iq\_stLogDataList | TPL.ST\_LogDataList | Specifies the global logging list |
