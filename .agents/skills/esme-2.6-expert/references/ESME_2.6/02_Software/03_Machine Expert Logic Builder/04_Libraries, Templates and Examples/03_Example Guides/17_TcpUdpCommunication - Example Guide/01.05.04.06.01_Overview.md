# Overview

Overview

The program SR\_TcpServer implements the program code for the [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) [server](../glossary/glossary.htm#XREF_D_SE_0055638_18) functionality. For controlling and monitoring this functionality, the visualization screen VisuTcpServer is provided in the example project.

The variables which link the visualization with the program code are defined in the global variable list GVL\_VisuTcpServer.
