# Write (FB; Prefix filwr)

FUNCTION\_BLOCK Write EXTENDS CBM.ETrigToA

This function block writes data into the file, which was previously opened via ”FILE.[Open](VIPLMVOpanJfjqQXFKWGQlzGjxw_open.html#viplmvopanjfjqqxfkwgqlzgjxw_open_open_fb_prefix_filop)”. The contents of the memory area indicated by pointer ”pBuffer” should not be modified during the write action! The size of the structure of the memory containing the bytes to be written as well as the number of bytes to be written will not be checked.

If the stability of the pointer on the data structures and their contents cannot be guaranteed in case of an online change, an online change must be avoided with the help of functions from the ”CAA\_Application library”.
