# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| etJobType | [ET\_MotionJobType](../Enumerations/Enumerations-7.htm#XREF_D_SE_0089438_1) | Type of the positioning job to be assigned |
| diJobId | DINT | Identification number of the motion job. Serves for checking which positioning job is active. diJobId can either be defined manually by the user or automatically by FB\_SoMotionGenerator. In this case, diJobId := -1 must be set. |
| xTerminateCurrentJob | BOOL | TRUE := If this job is the first in the job queue, the currently active job is canceled and replaced by this job. |
| xClearBufferedJobs | BOOL | TRUE := Deletes all jobs in the job queue (except for the currently active job). |
| etSetposMode | [ET\_SetPosMode](../Enumerations/Enumerations-11.htm#XREF_D_SE_0089446_1) | Defines the SetPos type to be executed onto the axis [i\_ifDrive](../Function_Blocks/Function_Blocks-19.htm#XREF_D_SE_0089466_1). The SetPos is executed upon activation of the motion job. |
| lrSetposValue | LREAL | Position (difference) of the SetPos to be executed |
| stPositioning | [ST\_Positioning](Structures-11.htm#XREF_D_SE_0089498_1) | Substructure for parameterization of positioning jobs |
| stCam | [ST\_Cam](Structures-2.htm#XREF_D_SE_0089480_1) | Substructure for parameterization of cam jobs |
