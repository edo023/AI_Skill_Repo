# IVisuUserEventManager.SetGestureEventHandler (METH)

METHOD SetGestureEventHandler : BOOL

This method assigns the eventhandler for gesture events. The method will return <c>TRUE</c> when there was no previous handler, and <c>FALSE</c> when there was a previous event handler.
