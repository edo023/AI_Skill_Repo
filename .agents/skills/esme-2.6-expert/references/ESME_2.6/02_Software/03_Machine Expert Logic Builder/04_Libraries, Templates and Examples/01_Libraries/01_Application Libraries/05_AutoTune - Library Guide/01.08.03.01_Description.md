| Step | Action |
| --- | --- |
| 1 | Open the ControllerOptimizationSelectAxis visualization.  Result: The "Select Axis" visualization is displayed  See also chapter [Drive Selection (Select Axis)](../Visualization/Visualization-3.htm#XREF_D_SE_0091186_1). |
| 2 | Select the axis.  All configured axes are automatically displayed in the "SelectAxis" visualization window. To select the axis to be optimized, you click the corresponding name. The selected axis is indicated by the green button with the axis name. The axis name also is displayed with Axis, and the optimization status of the axis is displayed with Status. The optimization is automatically set to the corresponding axis. |
| 3 | Click the button Main Configuration to switch to the Main Configuration visualization.  Result: The Main Configuration visualization opens.  See also chapter [Drive Configuration (Main Configuration)](../Visualization/Visualization-4.htm#XREF_D_SE_0091187_1). |
