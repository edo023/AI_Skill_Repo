# diInPosWdow

diInPosWdow

This parameter defines the required accuracy of positioning.

Positioning is considered successfully finished when the difference between the actual and target position of the axis is lower than i\_stPosPara.diInPosWdow for the time defined by i\_stPosPara.wInPosTime. The parameter i\_stPosPara.diInPosWdow is defined in encoder pulses.

Excessively low value will cause reports of detected error on positioning.

Zero value disables the in-position window function. Target position is considered reached immediately after profile generator reaches the target position.
