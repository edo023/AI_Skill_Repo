# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| lrXStart | LREAL | Left marginal point of the X interval |
| lrDeltaX | LREAL | Length of the X interval. |
| alrCoefficients | ARRAY[0..5] OF LREAL | List of the coefficients, with the element 5 being the coefficient of the highest order and the element 0 being the coefficient of the lowest order. |
