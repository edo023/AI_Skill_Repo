# Defining a Collision Object

## Overview

The following steps are required to define a collision object:

| Step | Action |
| --- | --- |
| 1 | Within the variable declaration, create an instance of the desired collision object type. For example, if a Sphere is required, define it as follows:   ``` VAR       //a Sphere function block       fbSphere : COD.FB_Sphere;       stCenter : SE_MATH.ST_Vector3D;       lrRadius : LREAL; END_VAR ``` |
| 2 | The variables stCenter and lrRadius will be set to the desired values. |
| 3 | You can now configure the new Sphere object within the body of the code calling a proper Set method; for example:   ``` fbSphere.SetCenterRadius(       i_stCenter := stCenter,       i_lrRadius := lrRadius,       q_xError => xError,        q_etResult => etResult,        q_sResultMsg=> sResultMsg );	 ``` |
| 4 | You can verify if a collision object has been properly configured by verifying that its property xConfigured = TRUE. This is a mandatory condition in order to be able to perform a collision or distance query on a collision object. |

NOTE:

Each Set method of the collision objects can be called multiple times to reconfigure the object.
