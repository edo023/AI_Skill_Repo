# ExecutionAborted

ExecutionAborted

|  |  |
| --- | --- |
| Enumeration name: | ExecutionAborted |
| Enumeration value: | 12 |
| Description: | The further execution of a program section aborted. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The execution requirements are not fulfilled. | Obtain more detailed diagnostic information (e.g. output q\_etDiagExt) for a more detailed analysis and to be able to derive appropriate measures. |
| - | During the execution impermissible deviations occurred. | Obtain more detailed diagnostic information (e.g. output q\_etDiagExt) for a more detailed analysis and to be able to derive appropriate measures. |
