# Notifications

Notifications

| Alarm Bit q\_wAlrmld | Description |
| --- | --- |
| 0 | Acceleration speed levels are not set in ascending order (i\_wDrvSpdRef1,2,3) |
| 1 | Deceleration speed levels are not set in descending order (i\_wDrvSpdRef4,5,6) |

NOTE:

On detection of alarm:

oq\_wDrvAcc is set to the i\_wDrvAccLvl4.

oq\_wDrvDec is set to the i\_wDrvDecLvl4.
