# Properties

## Properties

Properties provided by the FB\_ModbusTcpServer function block:

| Property | Data type | Access | Description |
| --- | --- | --- | --- |
| NumberOfConfiguredAccessRules | UINT | Get | This property indicates the number of rules configured for access control. |
| IsAccessControlActive | BOOL | Get | This property indicates TRUE if access control is activated. |
| NumberOfRefusedConnections | UDINT | Get | This property indicates the number of client connections which have been refused due to no matching IP address in the access rules.  This counter is reset when the function block is disabled. |
| NumberOfRejectedRequests | UDINT | Get | This property indicates the number of Modbus requests which have been rejected due to a request that is not included in the address range of the access rule.  This counter is reset when the function block is disabled. |
| IsAutoReopenActive | BOOL | Get / Set | This property indicates TRUE if the automatic reopen function of the server is activated.  The automatic reopen function closes and opens the server if an interface error (ET\_Result.InterfaceError) is detected. The reopening of the server is performed in intervals of 30 seconds. |
