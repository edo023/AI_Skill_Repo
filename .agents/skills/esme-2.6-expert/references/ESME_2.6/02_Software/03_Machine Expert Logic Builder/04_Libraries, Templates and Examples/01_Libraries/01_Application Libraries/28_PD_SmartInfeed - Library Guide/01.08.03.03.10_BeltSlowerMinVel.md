# BeltSlowerMinVel

BeltSlowerMinVel

|  |  |
| --- | --- |
| Enumeration name: | BeltSlowerMinVel |
| Enumeration value: | 81 |
| Description: | The belt is slower than the minimum velocity. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The belt velocity is below FB\_Infeed.ifCorrection.lrMinVel. | The belt velocity must be increased to FB\_Infeed.ifCorrection.lrMinVel.  FB\_Infeed.ifCorrection.lrMinVel must be reduced to a value that corresponds to the lowest base velocity of the belt. |
