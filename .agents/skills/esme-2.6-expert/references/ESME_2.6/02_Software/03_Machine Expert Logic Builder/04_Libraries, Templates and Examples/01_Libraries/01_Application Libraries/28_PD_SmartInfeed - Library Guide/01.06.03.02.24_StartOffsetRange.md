# StartOffsetRange

StartOffsetRange

|  |  |
| --- | --- |
| Enumeration name: | StartOffsetRange |
| Enumeration value: | 85 |
| Description: | StartOffset is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid value has been transferred at the input iq\_stSimParameter.stProductGeneration.lrStartOffset. | It must be made sure that a value greater than or equal to 0 is transferred at the input iq\_stSimParameter.stProductGeneration.lrStartOffset. |
| - | An invalid value has been transferred at the input iq\_stSimParameter.stTargetSignalGeneration.lrStartOffset. | It must be made sure that a value greater than or equal to 0 is transferred at the input iq\_stSimParameter.stTargetSignalGeneration.lrStartOffset. |
