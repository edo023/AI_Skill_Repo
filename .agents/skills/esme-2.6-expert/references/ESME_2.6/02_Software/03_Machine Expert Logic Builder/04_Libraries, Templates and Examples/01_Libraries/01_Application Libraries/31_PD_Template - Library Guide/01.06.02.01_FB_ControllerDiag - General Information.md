## Task

Function block used to decode the PacDrive diagnostic data to provide the user with detailed information on the pending diagnostic message and the affected object.

## Description

In the template this function block is used exclusively to determine whether the DiagCode can be attributed to a drive. Reactions to drive errors must take place in the submodules.

## Interface

NOTE: **POU-body not addressable!**

The POU itself must not be called up.

It is used exclusively by means of its methods and properties.
