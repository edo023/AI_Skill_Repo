## TeX Live for Windows

**Note**

For generating perfect formated PDF documents a full blown TeX installation is necessary.

Please download the [TeX Live installer](https://www.tug.org/texlive/windows.html) from the TeX Users Group web site.

A huge amount of information is out there in the net about TeX, LaTeX, …

One Example of a “short” introduction in to LaTeX: [https://tobi.oetiker.ch/lshort/lshort-a5.pdf](../../../../../../api/crossBook?lang=en-US&virtualBookName=lshort-a5)
