# GET\_CANOPEN\_KERNEL\_STATE (FB)

FUNCTION\_BLOCK GET\_CANOPEN\_KERNEL\_STATE EXTENDS CiA405Base

Gets the current state of the CANopen Kernel. As long as CANopen network is running correctly the function block returns [CANOPEN\_KERNEL\_ERROR.CANOPEN\_KERNEL\_NO\_ERROR](WkZ9when7NogvVdipw-bWj0hVTA_canopen-kernel-e.html#wkz9when7nogvvdipw_bwj0hvta_canopen_kernel_e_canopen_kernel_error_enum__wkz9when7nogvvdipw_bwj0hvta_canopen_kernel_e_canopen_kernel_error_canopen_kernel_no_error) for output STATE. The function block detects bus errors, configuration errors, guarding errors and internal system errors (e.g. out of system memory).
