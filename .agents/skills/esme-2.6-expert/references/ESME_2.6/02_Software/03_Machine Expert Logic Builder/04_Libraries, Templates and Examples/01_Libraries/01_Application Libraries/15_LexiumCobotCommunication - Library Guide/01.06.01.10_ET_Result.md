# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

This enumeration type is used to address a specific result of an operation.

It is used at the diagnostic output q\_etResult of function blocks, functions and methods.
