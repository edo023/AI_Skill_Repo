# IOL\_PQI (STRUCT)

TYPE IOL\_PQI : STRUCT

PROFINET Port qualifier information (PQI) The “Port qualifier information” provides status information of the IO-Link Port or the status of the Device respectively

| InOut: | | Name | Type | Comment | | --- | --- | --- | | NewPar | BIT | 0 : No update of Device parameter detected | | SubstDev | BIT | 0 : No substitute Device detected (identical SerialNumber) | | PortActive | BIT | 0 : Port de-activated via Port function | | DevCom | BIT | 0 : No Device available | | DevErr | BIT | 0 : No error/warning occurred | | PQ | BIT | 0 : Invalid IO Process Data from Device | |
