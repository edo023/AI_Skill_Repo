# SYS\_FILE\_STATUS (ENUM)

TYPE SYS\_FILE\_STATUS :

File status

Actual file status of the specified file.

| InOut: | | Name | Comment | | --- | --- | | FS\_OK | File could be opened | | FS\_NO\_FILE | No file available | | FS\_ILLEGAL\_POS | Illegal position in the file | | FS\_FULL | No more space on the filesystem | | FS\_EOF | End of file reached | |
