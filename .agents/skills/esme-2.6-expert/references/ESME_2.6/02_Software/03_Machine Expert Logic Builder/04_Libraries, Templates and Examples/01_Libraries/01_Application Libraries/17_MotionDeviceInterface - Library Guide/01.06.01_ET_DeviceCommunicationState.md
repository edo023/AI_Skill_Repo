# ET\_DeviceCommunicationState

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | UINT |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_DeviceCommunicationState of the library MotionDeviceInterface contains communication states of a Sercos device.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Operational | 0 | Device is in Sercos communication phase CP 4. |
| PhasingUp | 1 | Device is in Sercos communication phase CP 0, 1, 2, or 3. |
| NoCommunication | 2 | Device is in Sercos communication phase CP 4, but has not returned a valid state for at least two Sercos cycles, or the Sercos bus is in state Error. |
| NoData | 3 | Device is in Sercos communication phase CP 4, but an error has been detected. |
| NotRunning | 4 | The Sercos communication is in state NRT. |
