# diPosDiffMax

diPosDiffMax

The parameter is defined in encoder pulses. Drive running in [closed loop](../glossary/glossary.htm#XREF_D_SE_0024697_737) motor control mode will allow for lower value of i\_stPosPara.diPosDiffMax because it can follow the target speed more accurately. When the drive is running in [open loop](../glossary/glossary.htm#XREF_D_SE_0024697_746) motor control mode, it is necessary to increase the value to help preventing alarms.
