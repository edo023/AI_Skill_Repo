# Notifications

Notifications

The alarm ID when q\_xAlrm is TRUE. The ID is indicated by a bit set in this output.

The bit values have the following meaning:

| Alarm Bit q\_wAlrmId | Description |
| --- | --- |
| 0 | wrong order of limit switch signals detected |
| 1 | i\_wDrvSpdNom is zero although i\_rDistStop>0 |
| 2 | i\_rMotSpdLin is zero although i\_rDistStop>0 |
| 3 | i\_wScalFact is zero although i\_rDistStop>0 |
| 4 | i\_rMotSpdLin is negative |
