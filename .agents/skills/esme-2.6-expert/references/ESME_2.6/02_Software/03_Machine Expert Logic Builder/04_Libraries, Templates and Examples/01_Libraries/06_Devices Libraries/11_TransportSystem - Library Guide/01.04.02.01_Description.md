# Description

Description

The library "TransportSystemCommunication" provides a functional interface for monitoring and controlling 3rd party linear transport systems, using PacDrive Logic Motion Controllers (LMC) of Schneider Electric.

Note: Currently, the library only supports one transport system.

| Characteristic | Value |
| --- | --- |
| Library title | TransportSystemCommunication |
| Company | Schneider Electric |
| Category | Devices > TransportSystemCommunication |
| Component | PacDrive  (Application Libraries and Templates > PacDrive) |
| Default name space | TSC |
| Language model attribute | Qualified-access-only |
| Forward compatible library | Yes |

NOTE: For this library, qualified-access-only is set. This means, that the POUs, data structures, enumerations, and constants have to be addressed by using the name space of the library. The default name space of the library is TSC.
