# General

General

ET\_Cmd.Endless

The command changes to operating mode Endless.

ET\_Cmd.Endlesslls

This command changes to operating mode Endlesslls (Intelligent line shaft).

ET\_Cmd.Manual

This command changes to operating mode Manual.

ET\_Cmd.MultiCam

This command changes to operating mode Cam and prepares for a warm or cold start.

ET\_Cmd.DriveEnableOff

This command switches off the position control.

ET\_Cmd.DriveEnableOn

This command switches on the position control.

NOTE: The following service commands are available and used based on individual customer needs: Brake\_release, Hold, DriveEnable On, and DriveEnableOff.

A combination of positioning modes and commands can serve as a solution to get a working application.
