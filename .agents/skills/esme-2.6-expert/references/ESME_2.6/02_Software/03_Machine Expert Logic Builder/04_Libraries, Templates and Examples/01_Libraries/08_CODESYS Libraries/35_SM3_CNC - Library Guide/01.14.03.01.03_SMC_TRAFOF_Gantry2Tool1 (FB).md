# SMC\_TRAFOF\_Gantry2Tool1 (FB)

FUNCTION\_BLOCK SMC\_TRAFOF\_Gantry2Tool1

Forward transformation for two-dimensional gantries (X/Y) with tool offset, i.e. with the tool axis differing from the z-axis. The tool is approximated by a line.

**Note**

For detailed information on transformations see Overview to [SM\_Trafo\_POUs](uXtGJT17gvW9uxS1De45C-gyyqc_fld-SM_Trafo_POUs.html#uxtgjt17gvw9uxs1de45c_gyyqc_fld_sm_trafo_pous_sm_trafo_pous).
