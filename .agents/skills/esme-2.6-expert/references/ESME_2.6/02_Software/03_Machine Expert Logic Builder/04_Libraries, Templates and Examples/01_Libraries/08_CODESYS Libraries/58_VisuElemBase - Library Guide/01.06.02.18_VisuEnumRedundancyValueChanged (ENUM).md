# VisuEnumRedundancyValueChanged (ENUM)

TYPE VisuEnumRedundancyValueChanged :

Enum for different possible actions to do when in visu redundancy we change a visu variable on the active device.

| Attributes: | qualified\_only |
| InOut: | | Name | Initial | Comment | | --- | --- | --- | | NoChange | 0 | no action is done on the passive device | | OnlySyncArea |  | the value is changed on the passive device if the variable belongs to one of the synchronized areas | | Always |  | the value is always changed on the passive device | |
