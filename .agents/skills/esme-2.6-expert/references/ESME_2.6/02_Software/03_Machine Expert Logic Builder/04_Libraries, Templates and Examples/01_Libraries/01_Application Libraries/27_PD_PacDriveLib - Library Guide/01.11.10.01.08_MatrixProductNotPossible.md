# MatrixProductNotPossible

MatrixProductNotPossible

|  |  |
| --- | --- |
| Enumeration name: | MatrixProductNotPossible |
| Enumeration value: | 98 |
| Description: | The matrix product cannot be calculated. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The matrices transferred at the inputs i\_plrMatrix1 and i\_plrMatrix2 do not match. | The number of columns of matrix 1 i\_diCols1 must correspond to the number of lines of matrix 2 i\_diRows2. |
