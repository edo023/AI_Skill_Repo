# EFilterLatchContent (ENUM)

TYPE EFilterLatchContent :

This enumeration describes how to interpret the value of filter by latch variable content

| Attributes: | qualified\_only |
| InOut: | | Name | Initial | Comment | | --- | --- | --- | | NOT\_SET | -1 | Filter is not already set | | DISABLE | 0 | Filter is disabled | | STR | 1 | Variable for filtering contains a literal value: e.g. ‘ABCD’ ‘123’ “ABCD” “123” | | IEC | 2 | Variable for filtering contains a string that has to be interpret as IEC notation number: e.g. t#2s | | NUMBER | 3 | Variable for filtering contains a string that has to be interpret as number: e.g. “123”, “456.4” | |
