# Setting Up the QuickMotionProgramming Example for Training Purpose Using Virtual Axes

## General

The following modifications are only for training purpose with virtual axes to get in touch with program functionality with only an LMC controller available.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  Verify that no physical axes are connected to your test system.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |
