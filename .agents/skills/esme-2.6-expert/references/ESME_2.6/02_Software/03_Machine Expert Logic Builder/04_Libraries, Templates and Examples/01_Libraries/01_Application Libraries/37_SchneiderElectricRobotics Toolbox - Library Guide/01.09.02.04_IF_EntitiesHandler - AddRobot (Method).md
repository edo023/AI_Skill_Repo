## Task

Add a new robot entity.

## Description

The method AddRobot is used to add a new robot entity.

Once a robot has been stored with a specific ID, it is not possible to use such ID to store a new robot entity since every entity ID must be unique in the system.

The interface stores the value inside i\_stData.stKinematicParameters but does not perform any validation of them.

| Enumeration value: | 135 |
| Description: | The maximum number of configurable robots is reached. |

| Issue | Cause | Solution |
| --- | --- | --- |
| A robot has not been added to the list. | It is not possible to add a new robot to the function block because the maximum number of robot Gc\_udiMaxNumberOfRobots has been already stored. | Ensure to add a maximum of Gc\_udiMaxNumberOfRobots robots to the function block. |
