## Description

The structure UALocalizedText corresponds to the built-in data type LocalizedText as defined in the specification [*OPC 10000-3: UA Part 3: Address Space Model*](D-SE-0099923.3.html#D-SE-0099923.3__D-SE-0099923.13).

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| Locale | STRING [6] | Language and country/region identifier.  Format: `<language>-<country/region>`  Example: `en-US; zh-CHS` |
| Text | STRING [255] | Indicates localized text as a string. |
