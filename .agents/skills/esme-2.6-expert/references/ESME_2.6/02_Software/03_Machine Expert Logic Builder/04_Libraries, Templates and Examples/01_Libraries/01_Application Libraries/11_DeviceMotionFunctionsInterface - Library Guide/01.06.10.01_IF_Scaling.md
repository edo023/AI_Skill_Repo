# IF\_Scaling

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_Scaling of the library DeviceMotionFunctionsInterface allows you to access the user function Scaling of a drive in order to read and write scaling parameters.

## Methods

| Name | Description |
| --- | --- |
| [ScaleIncrementsToPosition](IF_Scaling-ScaleIncrementsToPositio-4F9246E3.html#IF_Scaling-ScaleIncrementsToPositio-4F9246E3) | Scales increment values to position values. |
| [ScalePositionToIncrements](IF_Scaling-ScalePositionToIncrement-4F9271BC.html#IF_Scaling-ScalePositionToIncrement-4F9271BC) | Scales position values to increment values. |

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| GearIn | UDINT | Get/Set | Gear factor for scaling. The value of the parameter GearIn is divided by the value of the parameter GearOut. |
| GearOut | UDINT | Get/Set |
| IncrementResolution | LINT | Get/Set | Resolution in increments of a motor revolution. |
| InvertDirection | BOOL | Get/Set | Inverts the direction of movement. |
| PositionResolution | LREAL | Get/Set | Modulo position of the axis. For scaling, the value of the parameter IncrementResolution is divided by the value of the parameter PositionResolution. |
