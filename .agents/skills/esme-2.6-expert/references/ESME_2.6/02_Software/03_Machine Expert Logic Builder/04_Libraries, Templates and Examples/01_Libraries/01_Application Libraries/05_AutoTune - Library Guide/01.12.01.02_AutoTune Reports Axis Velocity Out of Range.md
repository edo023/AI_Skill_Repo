# AutoTune Reports: Axis Velocity Out of Range (lResult - -10051)

AutoTune Reports: Axis Velocity Out of Range (lResult = -10051)

The permitted maximum velocity of the drive has been exceeded.

NOTE: It might not be possible to optimize drives with a highly limited velocity with AutoTune!

| Cause | Removal |
| --- | --- |
| The start parameters were selected unfavorable for the optimization. Oscillations occurred on the drive which increased the velocity excessively. | Start the optimization with default parameters.  Click on Start with default control loop parameters in the visualization Advanced Configuration.  Then restart the optimization. |
| The velocity limit max. Velocity selected in the visualization is too low. To optimize the axis, higher speeds are required. | Increase the velocity limit max. Velocity selected in the Main Configuration visualization window. Ensure that the maximum velocity permitted at the drive is not exceeded. |
