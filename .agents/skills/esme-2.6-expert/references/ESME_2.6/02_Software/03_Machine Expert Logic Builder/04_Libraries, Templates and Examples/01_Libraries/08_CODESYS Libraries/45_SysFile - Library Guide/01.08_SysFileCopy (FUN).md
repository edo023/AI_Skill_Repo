# SysFileCopy (FUN)

FUNCTION SysFileCopy : RTS\_IEC\_RESULT

Copy one file to another.

A standard path will be added to the filename, if no path is specified.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | SysFileCopy | RTS\_IEC\_RESULT | The runtime system error code (see CmpErrors.library) | | Input | szDestFileName | STRING | Destination file name. File name can contain an absolute or relative path to the file. Path entries must be separated with a Slash (/) and not with a Backslash (\)! | | szSourceFileName | STRING | Source file name. File name can contain an absolute or relative path to the file. Path entries must be separated with a Slash (/) and not with a Backslash (\)! | | pulCopied | POINTER TO \_\_XWORD | Number of bytes copied | |
