# CmdUnknown

CmdUnknown

|  |  |
| --- | --- |
| Enumeration name: | CmdUnknown |
| Enumeration value: | 2 |
| Description: | Unknown command. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The command sent to the module does not belong to the list of commands for the [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0088728_1) crank module. | Ensure that only commands from the [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0088728_1) list are sent to the module. |
