# SMC\_DEC\_STATUS (ENUM)

TYPE SMC\_DEC\_STATUS :

This enumeration variable indicates the current state of the FB instance. Possible states:

| InOut: | | Name | Initial | Comment | | --- | --- | --- | | WAIT\_PROG | 0 | Program not found yet | | READ\_WORD | 1 | Word read | | PROG\_READ | 2 | End of program reached | |
