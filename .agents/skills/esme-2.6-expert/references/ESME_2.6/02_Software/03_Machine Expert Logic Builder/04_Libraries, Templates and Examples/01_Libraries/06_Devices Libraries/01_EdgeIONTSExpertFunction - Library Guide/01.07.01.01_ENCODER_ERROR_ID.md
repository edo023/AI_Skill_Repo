# ENCODER\_ERROR\_ID

## Description

The ENCODER\_ERROR\_ID enumeration describes the types of errors that can be detected on an encoder function block.

## Enumeration Elements

The enumeration data type ENUM contains the following values:

| Name | Value | Description |
| --- | --- | --- |
| NO\_ERROR | 0 | No error is detected. |
| INTERNAL\_FIELD\_POWER\_SUPPLY | 1 | Internal field power supply is not detected. |
| POWER | 2 | The power monitoring of the encoder does not detect power return of the encoder. |
| FAULT | 3 | Power cannot be supplied to the encoder. |
| SHORTCUT | 4 | A short-circuit is detected in the channel. |
| FALLBACK | 5 | An implicit communication issue is detected. The module is in fallback state. The expert function is stopped. |
| QUADRATURE | 10 | The signals from the incremental encoder are incorrect. |
| FRAME\_FORMAT | 20 | Loss of communication between module and encoder. |
| FRAME\_PARITY | 21 | Loss of communication between module and encoder. |
| FRAME\_START | 22 | Loss of communication between module and encoder. |
| FRAME\_STUCK\_AT\_1 | 23 | Loss of communication between module and encoder. |
| TIMEOUT | 30 | No reply received from the encoder to a communication request. |
| PROTOCOL | 31 | Internal error detected in the encoder. |
| CRC | 32 | Communication between module and encoder is established but the content is incorrect. |
| CONF\_NOT\_SUPPORTED | 33 | The encoder used is not supported by the module. |
| SINCOS\_SIGNAL | 40 | The signals received from the encoder are incorrect. |
| SINCOS\_SYNCHRO | 41 | A synchronization issue between the number of period and the angle in the period has been detected. The position cannot be calculated. |
| NOT\_CONFIGURED | 50 | The encoder is not configured. The encoder reference is invalid. |
| ENC\_NOT\_ENABLED | 80 | The encoder is not enabled. Use the Encoder\_Main\_NTS function block to enable the encoder. |
| COMM\_ERR | 100 | The module is unavailable or the Network Interface Module (NIM) is unavailable or the GCS (Group Cyclic Status) value is incorrect. |
