# Read (FB; Prefix filrd)

FUNCTION\_BLOCK Read EXTENDS CBM.ETrigToA

This function block reads the file, which was previously opened via ”FILE.[Open](VIPLMVOpanJfjqQXFKWGQlzGjxw_open.html#viplmvopanjfjqqxfkwgqlzgjxw_open_open_fb_prefix_filop)”. If less characters can be read than specified in ”szBuffer”, the function block returns an active ”xDone” and indicates the current number of characters in ”szSize”. The size of the target memory structure for the bytes to be read and the number of bytes to be read will not be checked.

If the stability of the pointer on the data structures and their contents cannot be guaranteed in case of an online change, an online change must be avoided.
