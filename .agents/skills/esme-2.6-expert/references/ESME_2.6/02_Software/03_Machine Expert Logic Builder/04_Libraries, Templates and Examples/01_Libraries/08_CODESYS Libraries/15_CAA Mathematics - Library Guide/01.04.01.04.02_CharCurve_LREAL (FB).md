# CharCurve\_LREAL (FB)

FUNCTION\_BLOCK CharCurve\_LREAL

This function block will evaluate a piecewise linear function (the characteristic curve) at an integral point ![](images/a689ad51c2ba45e0938746d14630cb84.jp.jpg). The characteristic curve is specified by a defined number of integral sampling points ![](images/227a6d72cbc9475b932a02c3d2efe834.jp.jpg)
