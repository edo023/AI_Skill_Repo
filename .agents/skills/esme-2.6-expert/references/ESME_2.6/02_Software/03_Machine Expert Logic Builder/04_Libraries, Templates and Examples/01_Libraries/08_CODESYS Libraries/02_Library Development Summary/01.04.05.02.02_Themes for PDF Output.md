### Themes for PDF Output

Although there is a general method for PDF output from the Sphinx source files, Sphinx does not generate PDFs by itself. Sphinx provides a LaTeX builder which generates a collection of [LaTeX](http://latex-project.org/intro.html) source files. When using LaTeX for creating PDFs, it is necessary to customize the LaTeX output by means of options in the Sphinx config file. This method is less convenient than customizing the HTML themes that Sphinx uses for HTML-based output. Please refer to the Sphinx documentation of the [Sphinx LaTeX-Builder](http://sphinx-doc.org/config.html#options-for-latex-output).

Another option for creating PDF files from a Sphinx project is to use the Sphinx extension from the [rst2pdf](https://code.google.com/p/rst2pdf/) project.
