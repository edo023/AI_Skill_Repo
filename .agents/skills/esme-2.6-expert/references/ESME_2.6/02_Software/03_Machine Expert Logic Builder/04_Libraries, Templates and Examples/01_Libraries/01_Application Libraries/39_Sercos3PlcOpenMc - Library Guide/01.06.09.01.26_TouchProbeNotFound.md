# TouchProbeNotFound

TouchProbeNotFound

|  |  |
| --- | --- |
| Enumeration name: | TouchProbeNotFound |
| Enumeration value: | 58 |
| Description: | No Touchprobe was found. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The Touchprobe sensor does not transmit a signal. | Verify the power supply of the Touchprobe sensor.  Make sure that the Touchprobe sensor is not dirty.  Make sure that the Touchprobe sensor can be triggered by the mechanical system.  Verify the electrical connection of the Touchprobe sensor and of the Touchprobe input. |
