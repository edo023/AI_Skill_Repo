# ProfinetController (FB)

FUNCTION\_BLOCK ProfinetController

This Function Block represents the Profinet Controller.

One instance is created for the Profinet Controller device object that is added to the device tree.

For changing the bus status, use [ProfinetControllerDiag.SetCommunicationState](3STKY79N_7lYrCw5QAb9ysVJ51w_setcommunication.html#3stky79n_7lyrcw5qab9ysvj51w_setcommunication_profinetcontrollerdiag_setcommunicationstate_meth). This derived instance is created instead, if the device diagnosis is activated.
