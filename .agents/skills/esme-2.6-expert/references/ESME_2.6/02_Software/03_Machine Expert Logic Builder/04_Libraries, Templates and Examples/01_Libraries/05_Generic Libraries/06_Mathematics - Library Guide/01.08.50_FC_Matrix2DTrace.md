# FC\_Matrix2DTrace

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

Given a 2D input matrix, the function returns the trace of such matrix. The trace of a squared matrix is defined as the sum of the elements on its main diagonal.
