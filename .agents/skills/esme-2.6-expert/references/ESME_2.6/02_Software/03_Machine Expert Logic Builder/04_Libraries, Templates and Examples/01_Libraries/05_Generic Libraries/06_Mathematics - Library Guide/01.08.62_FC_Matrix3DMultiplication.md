# FC\_Matrix3DMultiplication

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

Given two 3D matrices, the function evaluates a new 3D matrix resulting from their multiplication.

| Input | Data type | Description |
| --- | --- | --- |
| i\_stMatrixA | [ST\_Matrix3D](ST_Matrix3D-GeneralInformation-0C04E9BF.html#ST_Matrix3D-GeneralInformation-0C04E9BF) | First 3D matrix to multiply. |
| i\_stMatrixB | [ST\_Matrix3D](ST_Matrix3D-GeneralInformation-0C04E9BF.html#ST_Matrix3D-GeneralInformation-0C04E9BF) | Second 3D matrix to multiply. |
