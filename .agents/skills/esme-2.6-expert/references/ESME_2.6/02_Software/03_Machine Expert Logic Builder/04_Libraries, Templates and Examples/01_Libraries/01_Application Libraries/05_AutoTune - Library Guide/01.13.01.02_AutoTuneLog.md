# AutoTuneLog

AutoTuneLog

A log file is generated automatically if

othe optimization was completed successfully,

othe user stops the optimization, or

othe optimization is stopped because of an error.

This log file is saved as "AutoTunelog" on the flash disk of the PacDrive Controller. It contains the following data:

oDate and time

oEthernet address

oName of the axis that has to be optimized

oType of the axis that has to be optimized (LXM62, LXM52 or ILM62)

oParameters from the Main Configuration and Advanced Configuration visualizations, for documenting the configuration.

oParameters of the PLC configuration of the axis being optimized, located in the Controller and Mechanics folders that resulted from the optimization

oCut-off frequencies of the speed and position control loops, as results of the optimization.

oStatus of the optimization

This file can be used as documentation of optimization results. This file also is very useful when searching for problems with AutoTune, and always should be included with corresponding queries.
