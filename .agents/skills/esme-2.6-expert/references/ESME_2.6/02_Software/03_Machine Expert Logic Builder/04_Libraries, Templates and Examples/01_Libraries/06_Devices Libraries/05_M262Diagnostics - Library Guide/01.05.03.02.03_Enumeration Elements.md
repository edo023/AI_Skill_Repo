# Enumeration Elements

Enumeration Elements

| Name | Value  (INT) | Description |
| --- | --- | --- |
| GetDiagData | 0 | By this command, the function block gathers the diagnostic information provided by the controller and stores them into the GVL.G\_astDiagTable. |
| ClearDiagTable | 1 | By this command, the content of the array GVL.G\_astDiagTable is erased. |
| CreateFileFromDiagTable | 2 | By this command, the file usr/Syslog/M262Diagnostic.txt on file system of the controller is created. The file contains the diagnostic messages from the array GVL.G\_astDiagTable. |
