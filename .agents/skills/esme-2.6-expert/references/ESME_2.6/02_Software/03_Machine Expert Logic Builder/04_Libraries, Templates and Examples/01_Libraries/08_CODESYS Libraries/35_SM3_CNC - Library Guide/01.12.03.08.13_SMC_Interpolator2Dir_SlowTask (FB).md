# SMC\_Interpolator2Dir\_SlowTask (FB)

FUNCTION\_BLOCK SMC\_Interpolator2Dir\_SlowTask

This FB is used for generating the reversed-path. It has been split off from [SMC\_Interpolator2Dir](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-interpolator1.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_interpolator1_smc_interpolator2dir_fb) in order to be called from the task that reads the g-code and performs the path preprocessing.
