# ST\_Credentials

|  |  |
| --- | --- |

| Available as of: | V1.0.0.0 |

## Description

The structure ST\_Credentials contains the credentials to authorize the MQTT client on the MQTT server. Username and password authentication are optional and depend on the server configuration. Leave the user name empty if no authentication is required.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sUsername | STRING[255] | Username to connect to the MQTT server. |
| sPassword | STRING[255] | Password to connect to the MQTT server. |
