# GetBusload (FUN)

FUNCTION GetBusload : USINT

This function is not implemented by all drivers. It returns the current busload in percent. Diagnostic Information can be also read by [GetDiagnosis](8gqnJ6B5M46-D0DeyJx7cbK99BA_getdiagnosis.html#8gqnj6b5m46_d0deyjx7cbk99ba_getdiagnosis_getdiagnosis_fun).
