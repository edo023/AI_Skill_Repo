# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_sIpAddress | STRING[15] | IP address from transport system controller |
| i\_timControllerResponseTimeout | TIME | Defines how long to wait for a response from transport system controller after sending a command before assuming a communication timeout.  Default value is 0.5 s. |
| i\_timBootOperationTimeout | TIME | Defines how long to wait for transport system controller to be operational before assuming a communication timeout.  Default value is 0.5 s. |
| i\_timPathResetTimeout | TIME | Defines how long to wait for a response from transport system controller after sending "Reset All Paths" during startup routine, before assuming a communication timeout.  Default value is 5.0 s. |
| i\_udiNumberOfPaths | UDINT | Defines how many paths the system has. |
| i\_udiNumberOfVehicles | UDINT | Defines how many vehicles the system has. |
