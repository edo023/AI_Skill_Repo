## Mitigated Anomalies

* **Lexium P Robot now supports triple axis**

  The Lexium P Robot with rotational axis now supports triple axis instead of double axis.
* **UserDrivePeakCurrent of axis D for Lexium S Robot**

  The maximum value for the parameter UserDrivePeakCurrent of axis D for Lexium S Robot is adapted.

  + Previous maximum value = 1050
  + New maximum value = 1600
