## Description

The method GetMovementValues of the interface [IF\_MasterChannel](IF_MasterChannel-4D60C9AB.html) retrieves the current movement data of the master channel (position, velocity and acceleration). These values reflect the tracked movement of the connected master axis plus any offset defined via the SetPosition method.

If the movement of the master axis cannot be determined (for example, due to a communication interruption), the method returns a detected error indicating that the data is invalid.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
