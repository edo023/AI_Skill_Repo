# SetLengthRange

SetLengthRange

|  |  |
| --- | --- |
| Enumeration name: | SetLengthRange |
| Enumeration value: | 45 |
| Description: | SetLength is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The mode [ET\_InTakeMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0080269_1).TpLeadEdgeSetLength or [ET\_InTakeMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0080269_1).TpTrailEdgeSetLength has been selected at the input FB\_Infeed.raifBelt[x].etInTakeMode.  An invalid value has been applied at the input FB\_Infeed.raifBelt[x].lrSetLength. | It must be made sure that a value greater than 0 is transferred at the input FB\_Infeed.raifBelt[x].lrSetLength.  An InTake mode must be selected that does not take this factor into account. |
| - | The mode [ET\_CheckForProductQueues](../Enumerations/Enumerations-3.htm#XREF_D_SE_0080265_1).ProductLengthFactorBelt or [ET\_CheckForProductQueues](../Enumerations/Enumerations-3.htm#XREF_D_SE_0080265_1).ProductLengthFactorBeltAndFrontBelt has been selected at the input FB\_Infeed.raifBelt[x].etCheckForProductQueues.  An invalid value has been transferred at the input FB\_Infeed.raifBelt[x].lrSetLength. | It must be made sure that a value greater than 0 is transferred at the input FB\_Infeed.raifBelt[x].lrSetLength.  A mode must be selected that does not take this factor into account. |
