# Libraries

## Overview

The following libraries have been added to the Library Manager of the application:

| Library | Company | Function |
| --- | --- | --- |
| SqlRemoteAccess | Schneider Electric | Provides SQL (Structured Query Language) client function blocks that allow your controller to connect to an SQL database through an SQL Gateway in order to run SQL requests for reading and writing data. |
| Standard64 | System | Provides several string functions for wide character strings (datatype: WSTRING). |
| SysTimeRtc | System | Provides the components to manage the real time clock (RTC) of the controller. |
| SysMem | System | Provides components to access the dynamic memory management of the runtime system. |

The SqlRemoteAccess library provides parameters to configure the supported functions. You can access these parameters in the global parameter list (GPL) available in the Library Manager of the application.

In this application example, the parameters are used with their default values.
