# V1.0.2.0

## Overview

NOTE: This library is for projects exclusively.

The library RoboticModule V1.0.2.0 is released for projects exclusively.

`Gc_xLibraryForProject : BOOL := TRUE;`

Using the library with other versions of software or firmware may have results other than described in the present documentation.

## Supported Hardware

* PacDrive 3 Controller LMC101C
* PacDrive 3 Controller LMC201C
* PacDrive 3 Controller LMC300C
* PacDrive 3 Controller LMC400C
* PacDrive 3 Controller LMC600C

## Software Requirements

* SoMachine Motion V4.1

## Firmware Requirements

PacDrive 3 V4.1

* PacDrive LMC x00C V1.51.10.6 or greater
* PacDrive LMC x01C V1.51.10.6 or greater

## New Functions

* None

## Modifications

* Default value of ET\_SignalType changed

  The default value of ET\_SignalType is ET\_SignalType.NormallyOpen instead of ET\_SignalType.NormallyClosed.

## Enhancements

* Behavior of module command ET\_Cmd.Start

  The behavior of module command ET\_Cmd.Start is fully supported in the present version after the module command ET\_Cmd.Stop has been sent.
* Behavior of hardware (IF\_HardwareLimit) and software limits (IF\_SoftwareLimit)

  The behavior of hardware (IF\_HardwareLimit) and software limits (IF\_SoftwareLimit) are fully supported in the present version.
