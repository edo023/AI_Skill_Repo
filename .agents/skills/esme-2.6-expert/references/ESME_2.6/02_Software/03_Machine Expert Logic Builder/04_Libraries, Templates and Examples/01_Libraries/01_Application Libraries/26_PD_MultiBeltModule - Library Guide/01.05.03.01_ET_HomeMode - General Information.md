# ET\_HomeMode - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.4.0 |
| Versions: | Current version |

## Description

This enumeration contains the homing modes of the FB\_MultiBeltModule.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| PosToStation | 1 | All trains are moved to a station. The station is determined using the uiStation parameter of the ST\_HomeParameter. i\_stPosToStationParameter structure. |
| HomeOnTp | 2 | All trains are moved to a station. In doing so, the axes are referenced for each axis via a Touchprobe. The station is determined using the uiStation parameter of the ST\_HomeParameter.i\_stOnTpParameter structure. |
| SetPos | 3 | Homing by setting the position of all axes. See [Homing with Setpos](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087290). |
| RestoreFromAxisEncoder | 4 | Homing by reading out an absolute encoder. See [Multiturn encoder homing](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087278). |
| WriteToAxisEncoder | 5 | Writing the absolute encoder. See [Multiturn encoder homing](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087278).  stMultiBeltItf.stMain.i\_rstParameter.lrBeltLength is used as i\_lrUserPeriod for the [Multiturn encoder Homing](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087278). It must be an integer factor of the encoder path (EncoderRange). If not, this homing mode must not be used. |
| RestoreFromRetain | 6 | Homing by reading out the position in the retain area. See [Homing from Retain](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087621). |
