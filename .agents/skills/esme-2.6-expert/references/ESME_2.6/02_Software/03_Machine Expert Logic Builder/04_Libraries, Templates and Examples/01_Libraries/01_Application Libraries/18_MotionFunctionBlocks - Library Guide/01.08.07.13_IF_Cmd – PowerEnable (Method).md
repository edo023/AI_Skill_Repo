# IF\_Cmd – PowerEnable (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.1.0 |
| Versions: | Installed version |

## Description

The functionality of this method corresponds to the functionality of MC\_Power for PLCopen, with TRUE at the input Enable. Refer to [M262 Synchronized Motion Control Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PLCO&topicID=D_SE_0086556).

## Inputs

| Input | Data type | Description |
| --- | --- | --- |
| i\_diJobId | DINT | Identifier for the command. |

## Outputs

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | The output is set to TRUE if an error has been detected during the execution. |
| q\_etResult | ET\_Result | POU-specific output on the diagnostic; q\_xError = FALSE -> Status message; q\_xError = TRUE -> Diagnostic message. |
| q\_sResultMsg | STRING[80] | Event-triggered message that gives additional information on the diagnostic state. |
