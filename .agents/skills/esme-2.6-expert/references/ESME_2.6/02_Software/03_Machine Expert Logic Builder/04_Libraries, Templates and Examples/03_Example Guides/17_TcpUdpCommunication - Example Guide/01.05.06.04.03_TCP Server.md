# TCP Server

TCP Server

Open a configurable TCP [port](../glossary/glossary.htm#XREF_D_SE_0055638_16) to allow incoming client connections to be accepted. Data can be received from the client through the established TCP connection. Depending on the content of a received message, a response is sent to the TCP [client](../glossary/glossary.htm#XREF_D_SE_0055638_5). The status of the [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) [server](../glossary/glossary.htm#XREF_D_SE_0055638_18) and the exchanged data are displayed in the visualization.
