# SMC\_NCSymbolResult (ENUM)

TYPE SMC\_NCSymbolResult :

The possible results of a symbol table lookup.

| Attributes: | qualified\_only |
| InOut: | | Name | Comment | | --- | --- | | FOUND | The signature has been found | | LOOKING | The signature will be retrieved and can be queried again in the next cycles. | | ERROR | An error ocurred while looking up the signature, e.g. if the subprogram could not be found. | |
