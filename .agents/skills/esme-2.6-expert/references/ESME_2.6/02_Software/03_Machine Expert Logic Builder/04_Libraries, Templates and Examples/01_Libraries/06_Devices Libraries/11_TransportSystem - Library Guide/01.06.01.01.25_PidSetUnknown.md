# PidSetUnknown

PidSetUnknown

|  |  |
| --- | --- |
| Enumeration name: | PidSetUnknown |
| Enumeration value: | 20 |
| Description: | The configuration of the PID set cannot be interpreted. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An unsupported set of PID values has been used in an attempt to move a vehicle to a new position. | Use only the following, supported values:  0 - unloaded PID set and 1 - loaded PID set |
