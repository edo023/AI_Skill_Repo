# EncodeSurrogate (FUN)

FUNCTION EncodeSurrogate

Returns the UTF-16 surrogate pair diRune1, diRune2 for the given diRune. If the rune is not a valid Unicode code point or does not need encoding, EncodeRune returns U+FFFD (c\_diRuneError).
