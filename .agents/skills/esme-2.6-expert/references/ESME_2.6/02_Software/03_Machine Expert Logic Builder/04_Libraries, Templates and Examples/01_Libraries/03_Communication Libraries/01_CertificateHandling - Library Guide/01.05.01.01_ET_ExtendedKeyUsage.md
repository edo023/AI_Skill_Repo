# ET\_ExtendedKeyUsage

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_ExtendedKeyUsage contains the values describing the usage of the certificate.

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| None | 0 | The usage of the certificate is not specified. |
| ClientAuthentication | 1 | The certificate is used to authenticate a client. |
| ServerAuthentication | 2 | The certificate is used to authenticate a server. |

## Used By

* FB\_CreateCSR
* FB\_CreateSelfSignedCertificate
