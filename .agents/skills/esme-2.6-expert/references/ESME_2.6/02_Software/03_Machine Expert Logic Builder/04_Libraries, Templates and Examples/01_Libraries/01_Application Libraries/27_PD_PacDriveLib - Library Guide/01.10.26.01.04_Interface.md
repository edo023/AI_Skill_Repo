| Input | Data type | Description |

| q\_diNumberOfCommonPoints | DINT | Number of intersections  Possible values:  0: There is no intersection, the straight lines are parallel and do not coincide.  1: There is exactly one intersection.  99: The straight lines coincide, i.e. there is an infinite number of intersections. |
| q\_stCommonPoint | [ST\_Vector2D](../Structures/Structures-47.htm#XREF_D_SE_0087800_1) | Intersection of the straight line, if q\_diNumberOfCommonPoints = 1. |

| q\_lrParameterLine2 | LREAL | Parameter of the intersection q\_stCommonPoint in the above parameter representation of the straight line 2 (only if q\_diNumberOfCommonPoints = 1) |
