# Overview of the Application

## Overview

The controller application implements the MQTT client for sending (publish) and receiving (for subscribed topics) messages via the connected MQTT server. In addition, the application implements a visualization component by which the communication can be controlled and monitored.

The following features are supported:

* Establishing a connection to an MQTT server
* Auto reconnect in case of communication interruption
* Publishing of a message to a dedicated topic
* Auto publishing with an adjustable time interval
* Subscribing to a topic to receive corresponding messages from the server
