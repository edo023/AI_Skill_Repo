# Alarm Fallback State, Disable Fallback State

Alarm Fallback State, Disable Fallback State

In the case of an alarm the lamp outputs q\_xLampYell and q\_xLampRed are flashing and the hoist movement is disabled by setting the output speed to zero.

If the FB is disabled the outputs q\_xOvldAlrm, q\_xOvldWrn, q\_xLampYell, q\_xLampRed, q\_xHorn and q\_xAlrm are set to FALSE.

The input target speed i\_wDrvSpdRef is transferred to the output q\_wDrvSpdRef to allow operation and the q\_wAlrmId, q\_wStat as well as q\_wActWght is zero.
