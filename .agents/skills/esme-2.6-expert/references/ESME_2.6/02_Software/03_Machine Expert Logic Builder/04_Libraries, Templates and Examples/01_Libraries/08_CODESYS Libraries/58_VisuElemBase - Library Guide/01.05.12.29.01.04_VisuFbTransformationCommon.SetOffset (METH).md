# VisuFbTransformationCommon.SetOffset (METH)

METHOD SetOffset

Method to set the values of VisuFbTransformInformation explicitly Be careful with SetOffset. SetOffset does not change the \_optimizedTransform member. When the transformation of drawing context has to be changed the method SetTransformation must be used. pClientData^.GlobalData.DrawingContext.SetTransformation(newTransform);
