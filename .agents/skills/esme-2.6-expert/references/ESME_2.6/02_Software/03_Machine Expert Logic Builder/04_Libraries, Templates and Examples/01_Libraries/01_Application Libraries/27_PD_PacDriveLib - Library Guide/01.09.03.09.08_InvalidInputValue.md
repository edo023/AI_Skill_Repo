# InvalidInputValue

InvalidInputValue

|  |  |
| --- | --- |
| Enumeration name: | InvalidInputValue |
| Enumeration value: | 324 |
| Description: | The input value is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The input for the allowed distance error or time error is not positive. | Apply a positive value to input i\_lrAllowedDistanceError and input i\_lrAllowedTimeErrorInSeconds. |
