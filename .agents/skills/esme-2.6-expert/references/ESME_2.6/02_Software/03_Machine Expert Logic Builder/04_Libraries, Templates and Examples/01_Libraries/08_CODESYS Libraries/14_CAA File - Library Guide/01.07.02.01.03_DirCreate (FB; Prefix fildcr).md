# DirCreate (FB; Prefix fildcr)

FUNCTION\_BLOCK DirCreate EXTENDS CBM.ETrig

This function block creates a sub-directory in the standard directory. If the sub-directory already exists, an error message is generated. There may be restrictions concerning the specification of the directory name, e.g. only capital letters allowed, for different targets. The function ”FILE.[GetProperty](RFfZ6qcMtbnrwad-bdR1UipBxBA_getproperty.html#rffz6qcmtbnrwad_bdr1uipbxba_getproperty_getproperty_fun)” shows these restrictions.
