# ReducedVelocityLimit

ReducedVelocityLimit

|  |  |
| --- | --- |
| Enumeration name: | ReducedVelocityLimit |
| Enumeration value: | 9 |
| Description: | The reduced velocity has been reached. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The parameter [ST\_Main.stReducedVelocity.i\_xEnableReducedVelocity](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) is TRUE.  The MechVelocity of the axis is greater than [ST\_Main.stReducedVelocity.i\_lrMaxReducedVelocity](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) \* [ST\_Main.stReducedVelocity.i\_lrReducedVelocityCheckFactor](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) | Check the motion parameters of the selected mode and/or change the values [ST\_Main.stReducedVelocity.i\_lrMaxReducedVelocity](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) or [ST\_Main.stReducedVelocity.i\_lrReducedVelocityCheckFactor](../Structures/Structures-7.htm#XREF_D_SE_0077223_1). |
