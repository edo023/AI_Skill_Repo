## Description

The method GetMovementValues of the interface [IF\_SlaveChannel](IF_SlaveChannel-4D6988AD.html) retrieves the movement values (position, velocity, and acceleration) from the subordinate axis connected to the channel.

In the first RTP cycle of execution, while the Prepare method is being executed, these values reflect the state of the axis from the cycle immediately preceding the start of the job. During the cyclic execution of the Calculate method, these values provide the movement values of the axis as recorded in the previous RTP cycle.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
