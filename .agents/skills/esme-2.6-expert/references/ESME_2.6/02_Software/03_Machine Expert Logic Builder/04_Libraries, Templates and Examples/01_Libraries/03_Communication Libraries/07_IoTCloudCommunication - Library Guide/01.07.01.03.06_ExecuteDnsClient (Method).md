# ExecuteDnsClient (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method is used to initiate the execution of the integrated DNS client. The specified DNS server is requested to resolve the required hostname to an IPv4 address. If no hostname need to be resolved, the communication to the DNS server is not established.

Host name resolution is required under the following conditions:

* The usage of the integrated proxy server is configured, and the address of the proxy server is specified by its hostname. In this case, the hostname of the proxy server is attempted to be resolved.
* The usage of a proxy server is not configured, and the address of the cloud server is specified by its hostname. In this case, the hostname of the cloud server is attempted to be resolved.

The execution of the DNS client is only supported in state Idle.

The return value of type ET\_Result indicates whether the execution of the DNS client could be initiated. Once this method is called, the execution of the request to the DNS server is processed by executing the method Cyclic().

This method has no inputs and outputs.
