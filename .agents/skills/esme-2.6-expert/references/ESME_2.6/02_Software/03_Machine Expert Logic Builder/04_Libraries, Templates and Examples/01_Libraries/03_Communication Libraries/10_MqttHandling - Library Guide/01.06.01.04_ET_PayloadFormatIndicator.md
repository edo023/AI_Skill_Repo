# ET\_PayloadFormatIndicator

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V2.4.4.0 |

## Description

The enumeration ET\_PayloadFormatIndicator specifies the format of the payload of a message.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Unspecified | 0 | The format of the payload of a message is unspecified (binary data). |
| UTF8 | 1 | The format of the payload of a message is UTF-8 encoded string. |
