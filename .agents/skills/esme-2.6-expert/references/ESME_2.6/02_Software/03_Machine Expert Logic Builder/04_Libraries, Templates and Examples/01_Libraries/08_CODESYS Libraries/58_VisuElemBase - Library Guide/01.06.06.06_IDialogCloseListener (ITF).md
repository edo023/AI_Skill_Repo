# IDialogCloseListener (ITF)

INTERFACE IDialogCloseListener EXTENDS \_\_SYSTEM.IQueryInterface

This interface has to be implemented by function blocks, that should get a notification, when a dialog is closed. To get this notification, the dialog has to be opened with method OpenDialog5
