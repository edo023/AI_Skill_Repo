# IF\_Configuration – SetErrorStopRamp (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.1.0 |
| Versions: | Installed version |

## Task

Sets deceleration and jerk for stopping in response to a detected error. The parameter lrErrorStopDec and lrErrorStopJerk have preset values which are overridden by the input parameters of the interface.

## Description

This method sets the deceleration and jerk for stopping in response to a detected error.

Settings can only be modified when the axis is disabled. The default values for the parameters are defined at the axis object.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrDeceleration | LREAL | Specifies the deceleration in units/s2. |
| i\_lrJerk | LREAL | Specifies the jerk in units/s3. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | The output is set to TRUE if an error has been detected during the execution. |
| q\_etResult | ET\_Result | POU-specific output on the diagnostic; q\_xError = FALSE -> Status message; q\_xError = TRUE -> Diagnostic message. |
| q\_sResultMsg | STRING[80] | Event-triggered message that gives additional information on the diagnostic state. |
