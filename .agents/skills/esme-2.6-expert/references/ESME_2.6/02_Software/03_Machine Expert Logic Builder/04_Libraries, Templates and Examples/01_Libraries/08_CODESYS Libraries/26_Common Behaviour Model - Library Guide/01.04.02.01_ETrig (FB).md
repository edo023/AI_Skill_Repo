This function block is a base implementation of a *PLCopen Edge Triggered Behavior Model* (see: [IETrig](c-dVK8bVY1F-S7PaIINI_aQKZks_ietrig.html#c_dvk8bvy1f_s7paiini_aqkzks_ietrig_ietrig_itf)). For use, this block should be extended and the specific inputs and outputs must be supplemented. The output eErrorID of the application-specific enum type ERROR must be added.
