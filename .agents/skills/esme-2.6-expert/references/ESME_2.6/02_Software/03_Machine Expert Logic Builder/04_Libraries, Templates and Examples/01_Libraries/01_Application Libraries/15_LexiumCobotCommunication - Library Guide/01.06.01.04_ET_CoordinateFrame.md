# ET\_CoordinateFrame

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

This enumeration type is used to address a specific coordinate frame of the Lexium Cobot.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| World | 0 | The world coordinate frame of the Lexium Cobot is addressed. |
| Joint | 1 | The joint coordinate frame of the Lexium Cobot is addressed. |
| Tool | 2 | The tool coordinate frame of the Lexium Cobot is addressed. |
