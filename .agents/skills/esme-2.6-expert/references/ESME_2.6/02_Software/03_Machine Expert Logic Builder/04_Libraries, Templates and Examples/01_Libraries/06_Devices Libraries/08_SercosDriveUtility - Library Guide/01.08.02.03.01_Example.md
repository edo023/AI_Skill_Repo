# Example

Example

Example of an Index File that is intended to read parameters from an ATV340S:

ATV340S  
Name;LogicAddress;IDN;LenBytes;DataType  
CMI;8504;P-0-2055.0.4;2;WORD (BitString16)  
ACC;9001;P-0-2060.0.1;2;UINT (Unsigned16)  
DEC;9002;P-0-2060.0.2;2;UINT (Unsigned16)  
…  
CD1;8423;P-0-2054.0.23;2;WORD (Enumeration)  
FR1;8413;P-0-2054.0.13;2;WORD (Enumeration)

Each line of the Index File is a data record that corresponds to one parameter.

Each parameter is thus defined by five fields that are separated by semicolons.
