## New Features

To the interface IF\_Configuration, the following methods are added:

* AdditionalComponentsTransformation - Adding an additional transformation which calculates the kinematics for the additional components.
* AddDrive - Adding an additional drive to the robot to be used for additional components.
* ConfigureTool - Creates or modifies a tool in the robot configuration.
* GetTool - Reading the parameters of a configured tool.
* MapComponentToDrive - Mapping the movement of a robot component directly to a drive.

To the interface IF\_Configuration, the following properties are added:

* raudiConfiguredToolIds - Array of configured tool IDs.
* raxCartesianComponentAvailable - Reads the available cartesian components.
* udiNumberOfConfiguredTools - Reads the number of configured tools.

## Enhancements

* Jogging of AuxAx is now possible with parametrized IF\_ConfigurationAdvanced.AdditionalTransformationTCP or IF\_ConfigurationAdvanced.AdditionalTransformationAxes without ST\_ModuleInterface.q\_xHomeOk.
* The behavior of homing modes using hardware limit switches (IF\_HardwareLimit) is improved.
