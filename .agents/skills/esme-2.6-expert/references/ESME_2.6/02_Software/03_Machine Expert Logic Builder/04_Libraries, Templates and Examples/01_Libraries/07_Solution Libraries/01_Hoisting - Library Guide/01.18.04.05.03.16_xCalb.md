# xCalb

xCalb

This is the switch to enable calibration. If the input i\_rDrvTrqMax is left at zero the result of the calibration is used to analyze the load. If any value unequal to zero is entered at the i\_rDrvTrqMax input, it will override the calibrated result.

Refer to the calibration [procedure description](Overload_EN15011-14.htm#XREF_D_SE_0020993_1).
