# UserCamHasBeenDeleted

UserCamHasBeenDeleted

|  |  |
| --- | --- |
| Enumeration name: | UserCamHasBeenDeleted |
| Enumeration value: | 169 |
| Description: | A user-defined Cam profile has been deleted. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.RestorePosFromRetain](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.RestorePosFromRetain](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1) or  [PDL.ET\_HomeMode.WriteAxisEncoder](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).  An invalid value was transferred to the [ST\_Home.stSetPos.i\_lrUserPeriod](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) parameter. | The following must apply: 0 <= i\_lrUserPeriod |
| - | The value of the parameter [ST\_Home.stSetPos.i\_lrUserPeriod](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) does not fit into the encoder period. | Allocate the parameter [ST\_Home.stSetPos.i\_lrUserPeriod](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) with 0 to not consider any offset.  The parameter [ST\_Home.stSetPos.i\_lrUserPeriod](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) must be selected such that the following formula is fulfilled:  ABS(lrEncoderPeriode / i\_lrUserPeriod - TRUNC\_INT(lrEncoderPeriode / i\_lrUserPeriod)) = 0  with lrEncoderPeriode := EncoderRange \* FeedConstant \* DINT\_TO\_LREAL(GearIn) / DINT\_TO\_LREAL(GearOut)  EncoderRange, FeedConstant, GearIn and GearOut are parameters of the axis. |
