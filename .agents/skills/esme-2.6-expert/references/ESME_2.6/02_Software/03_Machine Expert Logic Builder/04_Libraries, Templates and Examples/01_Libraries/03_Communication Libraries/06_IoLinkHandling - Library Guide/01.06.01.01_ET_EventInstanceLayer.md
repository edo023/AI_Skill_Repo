# ET\_EventInstanceLayer

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_EventInstanceLayer contains the instance layers of the event.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Unknown | 0 | Indeterminable event |
| Hardware | 1 | Hardware |
| DataExchangeLayerOfTheIoLinkDevice | 2 | Data exchange layer of the IO-Link device. |
| ApplicationLayerOfTheIoLinkDevice | 3 | Application layer of the IO-Link device. |
| Application | 4 | Application |
