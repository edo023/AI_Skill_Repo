# FC\_Matrix3DSetDiagonal

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

Given three scalar values, the function returns a new 3D diagonal matrix that has such elements on its main diagonal.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrE11 | LREAL | First value to insert in the main diagonal. |
| i\_lrE22 | LREAL | Second value to insert in the main diagonal. |
| i\_lrE33 | LREAL | Third value to insert in the main diagonal. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
