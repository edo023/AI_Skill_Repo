# IecTaskDisableWatchdog2 (FUN)

FUNCTION IecTaskDisableWatchdog2 : RTS\_IEC\_RESULT

Disable watchdog for the specified task

**Note**

* You have to enable the watchdog of the task with [IecTaskEnableWatchdog2](IecTaskEnableWatchdog2.html#iectaskenablewatchdog2_iectaskenablewatchdog2_fun) and \_not\_ with [IecTaskEnableWatchdog](IecTaskEnableWatchdog.html#iectaskenablewatchdog_iectaskenablewatchdog_fun), because they act on different task flags!
* The watchdog is disabled until [IecTaskEnableWatchdog2](IecTaskEnableWatchdog2.html#iectaskenablewatchdog2_iectaskenablewatchdog2_fun) is called!!! So this is a security issue, if you never enable the watchdog!
* As a consequence, you always have to call [IecTaskDisableWatchdog2](#iectaskdisablewatchdog2_iectaskdisablewatchdog2_fun) and [IecTaskEnableWatchdog2](IecTaskEnableWatchdog2.html#iectaskenablewatchdog2_iectaskenablewatchdog2_fun) symmetrical
* Can be called nested. First call disables the watchdog.
