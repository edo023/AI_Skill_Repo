# Input Pin Description

Input Pin Description

| Input | Data Type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block.  Refer to [detailed description](#XREF_D_SE_0021124_4) below this table. |
| i\_xDevAlrm | BOOL | Alarm event input monitored by the FB. On a rising edge the monitoring is started until a falling edge is detected.  TRUE: Recording active  FALSE: Recording inactive |
| i\_wAlrmNb | WORD | Number of incident to be displayed at the output.  Range: 0...20  Refer to [detailed description](#XREF_D_SE_0021124_5) below this table. |
| i\_xRst | BOOL | Resets all retained values.  A rising edge on this input resets the following retained values:  oq\_wAlrmNb is set to zero  oq\_dtAlrm [1...20] will be set to 1977-07-07-07:07:07  oq\_wAlrmDur [1...20] will be set to zero  TRUE: Resets function block  FALSE: No action |
