# SMC\_DIRECTION (ENUM)

TYPE SMC\_DIRECTION :

This enumeration variable indicates the current interpolation direction of the FB instance. Possible states:

| InOut: | | Name | Initial | Comment | | --- | --- | --- | | IPO\_positive | 1 | Interpolation is ongoing in positive direction (along the original path) | | IPO\_standstill | 0 | Interpolation paused | | IPO\_negative | -1 | Interpolation is ongoing in negative direction (in opposite direction to the original path) | |
