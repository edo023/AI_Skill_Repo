# Description

Description

This structure contains the parameters and feedback data for the start movement behavior of the FB\_Unwinder for the harmless TorqueMode.

NOTE: For this control mode the parameters of the bobbin drive have to be set a special way. See "Using the unwinder TorqueMode".

With this start movement a wind-up of the PI controller when starting the torque control mode can be prevented. This wind-up can happen, if the foil is loose when starting the torque controller. Then the torque controller moves with a maximum torque, but does not have any feedback from the tension measurement system. When the foil is tense, then the PI controller has to reduce his integrated value first before it can react to the new feedback. This can lead to a crack of the foil.

This start movement moves the bobbin already in torque control, but with a velocity control loop. The bobbin is moved with the specified velocity until the feedback value from the tension measurement system exceeds i\_lrStartThreshold. Then the start movement is finished and the control mode takes over.
