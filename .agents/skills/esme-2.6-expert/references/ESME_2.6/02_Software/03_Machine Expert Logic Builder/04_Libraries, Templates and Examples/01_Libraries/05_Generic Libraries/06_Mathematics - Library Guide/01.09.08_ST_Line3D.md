# ST\_Line3D

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |

## Description

The structure ST\_Line3D represents a straight line in the three-dimensional space which passes through the points stPoint1 and stPoint2. The points stPoint1 and stPoint2 must be different.

The following parameter representation is used for the straight line:

Line(Parameter) = Line.stPoint1 + Parameter \* (Line.stPoint2 - Line.stPoint1)

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stPoint1 | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | First point of the straight line |
| stPoint2 | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | Second point of the straight line |
