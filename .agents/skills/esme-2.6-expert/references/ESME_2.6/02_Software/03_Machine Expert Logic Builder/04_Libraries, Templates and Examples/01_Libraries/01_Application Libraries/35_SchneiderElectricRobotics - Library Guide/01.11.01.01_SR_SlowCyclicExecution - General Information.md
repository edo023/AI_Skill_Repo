## Task

Program to verify the parameters and execute the calibration.

## Description

The program verifies cyclically whether the parameters are valid and executes the calibration. Even if there are several Schneider Electric robots used in a project the program only has to be called once.

The program must be called in a task with an interval of 10 ms.

The priority of that task must be parametrized to the application needs to manage a cycle time overrun.

NOTE:

For each call of SetControlLoopParameter, the cycle verification values are modified for which entries are made in the logger. Depending on the frequency, the logger may cover just a short period of time.
