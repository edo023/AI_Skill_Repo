# FC\_Distance3DPointPlane

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the distance of a point i\_stPoint from a plane i\_stPlane in a three-dimensional space. The sign indicates in which of the half spaces defined by the plane i\_stPlane the point i\_stPoint is located. In addition, the perpendicular of the point to the plane is calculated.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stPoint | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | Point |
| i\_stPlane | [ST\_Plane](ST_Plane-GeneralInformation-0C057E9D.html#ST_Plane-GeneralInformation-0C057E9D) | Plane |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
| q\_lrDistance | LREAL | Calculated distance of the point i\_stPoint from the plane i\_stPlane. |
| q\_lrParameter1 | LREAL | First parameter of the perpendicular q\_stPerpendicular in the parameter representation of the plane i\_stPlane (see ST\_Plane). |
| q\_lrParameter2 | LREAL | Second parameter of the perpendicular q\_stPerpendicular in the parameter representation of the plane i\_stPlane (see ST\_Plane). |
| q\_stPerpendicular | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | Perpendicular of the point i\_stPoint onto the plane i\_stPlane |

## DefinitionPlane

|  |  |
| --- | --- |
| Enumeration name: | DefinitionPlane |
| Enumeration value: | 45 |
| Description: | The definition of Plane is not correct. |

| Cause | Solution |
| --- | --- |
| The inputs i\_stPlane.stSpanPoint1 and i\_stPlane.stBasePoint have been set to the same point. | The distance between i\_stPlane.stSpanPoint1 and i\_stPlane.stBasePoint must be at least Gc\_lrZeroTolerance. |
| The inputs i\_stPlane.stSpanPoint2 and i\_stPlane.stBasePoint have been set to the same point. | The distance between i\_stPlane.stSpanPoint2 and i\_stPlane.stBasePoint must be at least Gc\_lrZeroTolerance. |
| The points i\_stPlane.stSpanPoint1, i\_stPlane.stSpanPoint2 and i\_stPlane.stBasePoint are on a straight line. | The points i\_stPlane.stSpanPoint1, i\_stPlane.stSpanPoint2 and i\_stPlane.stBasePoint must not be on a common straight line. |
