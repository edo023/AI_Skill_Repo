# ET\_RandomTargetsType - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.1.0.0 |

## Description

This enumeration type specifies how the random targets are generated.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No target type selected. |
| TargetsInPlane | 1 | Generation of a list of random targets with coordinates contained in a working plane. |
| TargetsInPlaneRotationList | 2 | Generation of a list of random targets with coordinates contained in a working plane and rotation randomly selected from a list. |
| TargetMatrixInPlane | 3 | Generation of a matrix of targets with coordinates contained in a working plane. |
| TargetMatrixInPlaneRotationList | 4 | Generation of a matrix of targets with coordinates contained in a working plane and rotation randomly selected from a list. |
