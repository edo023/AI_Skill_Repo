## New Functions

**Lexium P Robot Feedback**

The property ifFeedback of type IF\_RobotPSeriesFeedback has been added to FB\_RobotPSeries.

It contains, for example, indicators for tracking deviation, motor overload, and load of the axes of a Lexium P Robot.

It can be used, for example, to simulate a robotics application to get feedback about its feasibility.
