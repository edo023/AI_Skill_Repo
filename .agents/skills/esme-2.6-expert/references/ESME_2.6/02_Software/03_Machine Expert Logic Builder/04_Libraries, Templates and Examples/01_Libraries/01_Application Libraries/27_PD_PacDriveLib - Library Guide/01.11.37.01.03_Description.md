# Description

Description

The function starts with a linear search from i\_lrXStart with the step distance i\_lrXIncr. If a sign reversal is detected in the difference of i\_lrYPos – Y(Xi), the section containing the solution has been found.

A binary iterative procedure is then used in this section in order to find the result q\_lrXPos.

The function is ended after either

|i\_lrYpos – Y(XPos)| <= i\_lrAccuracy or after a maximum of 20 passes.
