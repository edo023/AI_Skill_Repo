# TpDistanceToBeltStartRange

TpDistanceToBeltStartRange

|  |  |
| --- | --- |
| Enumeration name: | TpDistanceToBeltStartRange |
| Enumeration value: | 27 |
| Description: | TpDistanceToBeltStart is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The mode [ET\_InTakeMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0080269_1).TpLeadEdgeMeasure, [ET\_InTakeMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0080269_1).TpTrailEdgeMeasure, [ET\_InTakeMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0080269_1).TpLeadEdgeSetLength or ET\_InTakeMode.TpTrailEdgeSetLength has been selected at the input FB\_Infeed.raifBelt[x].etInTakeMode.  An invalid value has been applied at the input FB\_Infeed.raifBelt[x].ifTpFilter.lrTpDistanceToBeltStart. | It must be made sure that a value greater than or equal to 0 is transferred at the input FB\_Infeed.raifBelt[x].ifTpFilter.lrTpDistanceToBeltStart.  An InTake mode must be selected that does not require a Touchprobe. |
| - | The input FB\_Infeed.ifTargetManager.xEnableTargetGenerator has been set to TRUE.  The mode [ET\_TargetGeneratorMode](../Enumerations/Enumerations-9.htm#XREF_D_SE_0080277_1).TouchProbe, has been selected at the input FB\_Infeed.ifTargetManager.ifTargetGenerator.etMode.  An invalid value has been applied at the input FB\_Infeed.ifTargetManager.ifTargetGenerator.ifTpFilter.lrTpDistanceToBeltStart. | It must be made sure that a value greater than or equal to 0 is transferred at the input FB\_Infeed.ifTargetManager.ifTargetGenerator.ifTpFilter.lrTpDistanceToBeltStart.  A target generation must be selected that does not require a Touchprobe. |
