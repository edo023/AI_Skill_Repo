# YEqualCheckFailed

YEqualCheckFailed

|  |  |
| --- | --- |
| Enumeration name: | YEqualCheckFailed |
| Enumeration value: | 46 |
| Description: | YEqualCheck was unsuccessful. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | If [ST\_MultiCam](../Structures/Structures-9.htm#XREF_D_SE_0088766_1).iq\_stExt.xYEqualCheck = TRUE: Then the verification whether the new cam follows the old cam directly was unsuccessful. | Configure the new cam so that its start deviates by less than 0.0001 units from the end of the old cam.  Deactivate the verification by setting [ST\_MultiCam](../Structures/Structures-9.htm#XREF_D_SE_0088766_1).iq\_stExt.xYEqualCheck = FALSE. |
