## Task

Implementation for a Capsule collision object.

## Description

A Capsule is defined by two points, A and B, and a radius. For more information, refer to the method [SetPointsRadius](FB_Capsule-SetPointsRadiusMethod-C49A472E.html#FB_Capsule-SetPointsRadiusMethod-C49A472E).

Extends: FB\_CollisionObject
