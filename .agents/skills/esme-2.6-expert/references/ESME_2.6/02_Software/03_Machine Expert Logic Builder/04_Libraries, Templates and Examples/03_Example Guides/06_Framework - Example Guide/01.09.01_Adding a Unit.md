# Adding a Unit

To add a single axis unit example from the Framework project to an existing project that already includes an SR\_Application entry point, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Select and copy, for example, the U200\_SingleAxisUnitExample folder node including the elements and sub nodes from [the Applications tree](Project-A98179A4.html#Project-A98179A4__ApplicationTree-AD178B72). |
| 2 | Paste the folder into a new project and rename it to, for example, U500\_MySingleAxisUnit. |
| 3 | Refactor the `PRG` and `STRUCTs` with meaningful names according to the context of your project.  NOTE: It is a good practice to enter preceding numbers to the unit name to sort the folders in the Applications tree. |
| 4 | Initialize and call the new unit in SR\_Application or the leading unit that controls the new subunit. |
