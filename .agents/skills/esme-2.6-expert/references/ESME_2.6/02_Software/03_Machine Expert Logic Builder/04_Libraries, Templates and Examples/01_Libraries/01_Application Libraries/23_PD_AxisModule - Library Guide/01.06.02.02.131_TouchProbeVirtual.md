# TouchProbeVirtual

TouchProbeVirtual

|  |  |
| --- | --- |
| Enumeration name: | TouchProbeVirtual |
| Enumeration value: | 97 |
| Description: | The connected Touchprobe is virtual. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.PosDirectionPosEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionPosEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionNegEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.PosDirectionNegEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  The used Touchprobe is not real. | Ensure that the Touchprobe does not belong to a virtual device. (e.g. virtual Lexium62) |
