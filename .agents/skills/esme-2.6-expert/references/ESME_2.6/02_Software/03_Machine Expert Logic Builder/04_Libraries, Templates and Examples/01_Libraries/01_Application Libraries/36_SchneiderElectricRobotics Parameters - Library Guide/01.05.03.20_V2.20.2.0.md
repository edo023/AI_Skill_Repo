## New Features

The Lexium SCARA Robot is supported:

* The function block [FB\_RobotSCARAParameters](FB_RobotSCARAParameters-GeneralInfo-109B5EF1.html) contains the Lexium SCARA Robot parameters.
* The enumeration [ET\_RobotSCARA](ET_RobotSCARA-GeneralInformation-109A165A.html) contains the different Lexium SCARA Robot types.
* The function [FC\_EtRobotSCARAToString](FC_EtRobotSCARAToString-GeneralInfo-54A084F3.html) converts an element of [ET\_RobotSCARA](ET_RobotSCARA-GeneralInformation-109A165A.html) to string.
