## Functional Description

By calling up the method, the kinematic parameter of a Lexium P Robot can be read.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etName | [SER.ET\_KinematicParameter](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRobotics&topicID=D_SE_0075190) | Type of kinematic parameter. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_lrValue | LREAL | Mass of the selected kinematic parameter [kg]. |
| q\_stCenterOfMass | PDL.ST\_Vectro3D | Center of mass for the selected kinematic parameter [mm]. |
| q\_etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | General, library-independent statement on the diagnostic.  A value not equal to ET\_Diag.Ok corresponds to a diagnostic message. |
| q\_etDiagExt | [SER.ET\_DiagExt](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRobotics&topicID=D_SE_0075188) | POU-specific output for the diagnostic.  q\_etDiag = ET\_Diag.Ok -> Status message  q\_etDiag <> ET\_Diag.Ok -> Diagnostic message |
| q\_sMsg | STRING[80] | Event-triggered message that gives additional information on the diagnostic state. |

## KinematicParameterInvalid

|  |  |
| --- | --- |
| Enumeration name: | KinematicParameterInvalid |
| Enumeration value: | 15 |
| Plain text: | The kinematic parameter is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| Reading the kinematic parameter was not successful. | The value transferred at the input i\_etName is invalid. | Verify that at the input i\_etName a valid kinematic parameter has been transferred, and adjust if necessary. |
