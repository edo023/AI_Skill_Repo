# AdditionalWaitForStart

AdditionalWaitForStart

|  |  |
| --- | --- |
| Enumeration name: | AdditionalWaitForStart |
| Enumeration value: | 138 |
| Description: | The external POU is waiting for a start signal. |

The operating mode AdditionalMode is selected and the activated external POU is expecting a start signal which is sent with the Start command from the list of the [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) commands.
