## New Features

The method Scara4AxCoupling was added to the interface IF\_Configuration. With this method the robot can be configured as a SCARA Robot with four axes which uses a coupling for Z (translation movement along the mechanical Z axis) and OrientationZ (rotation about the mechanical Z axis).

## Modifications

* Calculating the motion profile is improved.
* Transforming a pose when referring to the mounting flange is improved.
* Diagnosis of the method [GetLinearTrackingSystem](D-SE-0075518.html) is improved in case it is called with a system ID that is configured as another tracking system type (for example, tracking source).
