# SlaveMasterRatioRange

SlaveMasterRatioRange

|  |  |
| --- | --- |
| Enumeration name: | SlaveMasterRatioRange |
| Enumeration value: | 22 |
| Description: | SlaveMasterRatio is outside of the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The PreControlmode ET\_PreControllerMode.Master is chosen and at the input iq\_stParameter.stPreController.stMaster.i\_lrSlaveMasterRatio an invalid value has been transferred. | For iq\_stParameter.stPreController.stMaster.i\_lrSlaveMasterRatio, a value unequal to 0 must be specified. |
