# Description

Description

The target of a motion started e.g. via [MC\_MoveAbsolute](../../../../../../api/crossBook?lang=en-US&virtualBookName=Function_Blocks-10.htm#XREF_D_SE_0090907_1) or [MC\_MoveRelative](Function_Blocks-12.htm#XREF_D_SE_0090911_1) is offset relatively by Distance by calling up the function on the new target using the function [SystemInterface.FC\_PosStartJerk](../../../PD.Lib.SystemInterface&topicID=D_SE_0085245_1).

If the function block in the axis status ‘ContinuousMotion’ is enabled, the specified relative distance is added to the position at the time of the execution.
