# ParameterNotConfigured

ParameterNotConfigured

|  |  |
| --- | --- |
| Enumeration name: | ParameterNotConfigured |
| Enumeration value: | 69 |
| Description: | Parameter is not configured. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The control mode ET\_ControllerMode.Torque is chosen, and the bobbin axis is not parameterized correctly. | The Sercos IDN P-0-1069.0.0 has to be added to the AdditionalIDNListMDT in the parameters of the drive for the bobbin axis.  The Sercos bus has to perform a startup (phase 0=> phase 4) after setting the needed parameter, in order to make the parameter work. |
