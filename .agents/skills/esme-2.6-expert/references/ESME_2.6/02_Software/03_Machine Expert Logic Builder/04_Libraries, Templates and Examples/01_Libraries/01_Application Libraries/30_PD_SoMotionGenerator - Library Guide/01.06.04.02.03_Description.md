# Description

Description

This standard function block serves for external generation and calculation of segmented cams. For this purpose, a number of existing cams is composed to form a new cam. In order to generate a segmented cam, an instance of the function block must be created within the user program, then the method Prepare() must be called up.

NOTE: POU is not addressable!

The POU must not be called up.

The function block can be used by calling it up over methods and properties.
