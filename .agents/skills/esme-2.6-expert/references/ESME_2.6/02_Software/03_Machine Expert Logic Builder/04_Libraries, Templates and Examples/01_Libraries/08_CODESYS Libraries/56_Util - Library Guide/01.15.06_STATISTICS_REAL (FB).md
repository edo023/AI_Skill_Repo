# STATISTICS\_REAL (FB)

FUNCTION\_BLOCK STATISTICS\_REAL

Calculates minimum, maximum, and average of an input value

The calculation is done over time and can be reset in order to begin a fresh caluclation/statistic.

| InOut: | | Scope | Name | Type | Initial | Comment | | --- | --- | --- | --- | --- | | Input | IN | REAL |  | Input value | | RESET | BOOL |  | TRUE: AVG set to 0, MN and MX to minimum and maximum possible real values | | Output | MN | REAL | 3.402823466E+38 | Minimum value | | MX | REAL | -3.402823466E+38 | Maximum value | | AVG | REAL |  | Average value | |
