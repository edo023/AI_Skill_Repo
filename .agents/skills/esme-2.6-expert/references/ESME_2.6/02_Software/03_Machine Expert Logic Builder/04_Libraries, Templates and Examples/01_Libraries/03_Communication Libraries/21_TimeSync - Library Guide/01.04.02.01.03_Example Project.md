# Example Project

Example Project

In conjunction with the library, the example project FB\_SntpClient.project is provided. The example project shows how to implement the components from the TimeSync library.

| Step | Action | Comment |
| --- | --- | --- |
| 1 | In the EcoStruxure Machine Expert Logic Builder, execute the command New Project. | – |
| 2 | In the New Project dialog box, select From Example from the Project type list. | – |
| 3 | On the right-hand side of the New Project dialog box, select the controller (1) from the Controller list. | Result: Available examples are listed in the Matching Examples text box. |
| 4 | Select your example from the Matching Examples list. | – |
| 5 | Enter a name for the new project, and select the file location. | – |
| 6 | Click the OK button. | Result: A new project is created based on the selected example. |
