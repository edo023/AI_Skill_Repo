# Input Pin Description

Input Pin Description

| Input | Data type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block. |
| i\_xDrvRun | BOOL | TRUE: Hoist is running.  FALSE: Hoist is not running. |
| i\_rLoadNom | REAL | Nominal load  Range: 0...3.438  Unit: kg |
| i\_rLoadAct | REAL | Actual load  Range: 0...3.438  Unit: kg |
| i\_xStandard | BOOL | Used standard  TRUE: ISO  FALSE: FEM |
