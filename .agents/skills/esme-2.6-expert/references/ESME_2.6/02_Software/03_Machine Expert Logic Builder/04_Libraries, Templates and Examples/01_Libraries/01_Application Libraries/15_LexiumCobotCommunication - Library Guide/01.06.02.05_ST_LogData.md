## Description

This structure describes the log data provided by the Lexium Cobot controller.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| xUpdated | BOOL | TRUE: The array of log entries is updated. New log data was provided by the Lexium Cobot controller.  FALSE: The array of log entries is not updated. |
| astEntry | ARRAY [1...Gc\_uiMaxNumberOfLogDataEntries] OF [ST\_LogDataEntry](ST_LogDataEntry-7C53BEE9.html) | Log entries provided by the Lexium Cobot controller. |
