# Description

Description

The POUs in this section are used to calculate assigned motion positioning instructions for analysis purposes.

For example for the program logic (total moving time) or to derivate parameters for motion function blocks (required acceleration).

They cannot be used to start an axis.

The function block FB\_RgPosStartOptimized relates to FB\_PathForJerkPositioning.

They are used to define a motion operation:

o[FB\_RgPosStartOptimized](../Function_Blocks_R_to_Z/Function_Blocks_R_to_Z-8.htm#XREF_D_SE_0087349_1) is used to calculate the velocity and acceleration over a specified travel distance.

It defines the maximum velocity, accelerations, and jerk.

o[FB\_PathForJerkPositioning](../Function_Blocks_I_to_Q/Function_Blocks_I_to_Q-19.htm#XREF_D_SE_0087320_1) is used to calculate the positioning of a jerk. It provides the position, velocity, and acceleration.

FB\_RgPosStartOptimized can transfer its results to FB\_PathForJerkPositioning.
