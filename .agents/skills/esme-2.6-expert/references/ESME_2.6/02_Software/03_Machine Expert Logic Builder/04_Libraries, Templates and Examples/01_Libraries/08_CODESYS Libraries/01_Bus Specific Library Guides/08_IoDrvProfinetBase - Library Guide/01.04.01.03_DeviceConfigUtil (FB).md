# DeviceConfigUtil (FB)

FUNCTION\_BLOCK DeviceConfigUtil

This function block can be used to adapt some parameters of the initially downloaded profinet device configuration.

For example the slot numbers of teh modules that are configured for this Profinet device can be adapted at runtime.

The FB is typically used with the Reconfigure-feature (see DED.Reconfigure function block).
