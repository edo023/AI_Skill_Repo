# ST\_Line2D

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |

## Description

The structure ST\_Line2D represents a straight line in the two-dimensional space which passes through the points stPoint1 and stPoint2. The points stPoint1 and stPoint2 must be different.

The following parameter representation is used for the straight line:

Line(Parameter) = Line.stPoint1 + Parameter \* (Line.stPoint2 - Line.stPoint1)

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stPoint1 | [ST\_Vector2D](ST_Vector2D-GeneralInformation-0BFF6B0C.html#ST_Vector2D-GeneralInformation-0BFF6B0C) | First point of the straight line |
| stPoint2 | [ST\_Vector2D](ST_Vector2D-GeneralInformation-0BFF6B0C.html#ST_Vector2D-GeneralInformation-0BFF6B0C) | Second point of the straight line |
