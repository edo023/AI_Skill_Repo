# LTON (FB)

FUNCTION\_BLOCK LTON

Timer on delay. Q is TRUE, after the delay defined in PT and IN had a rising edge.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | IN | BOOL | starts timer with rising edge, resets timer with falling edge | | PT | LTIME | time to pass, before Q is set | | Output | Q | BOOL |  | | ET | LTIME | elapsed time since rising edge at IN | |
