## Functional Description

With the method RegisterLoggerPoint, the program SR\_<RobotCell> can be registered to the Application Logger. The method RegisterLoggerPoint of the SubModules for example robots, conveyors, is automatically called and registered to the Application Logger.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifParent | APL.IF\_LoggerPoint | Parent logger point under which the SR\_<RobotCell> is registered. |
| I\_sName | STRING[80] | The name of the SR\_<RobotCell> logger point that is shown in the Application Logger is declared here. |
