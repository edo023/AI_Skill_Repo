# FAST\_IO\_ERROR\_ID

## Description

The FAST\_IO\_ERROR\_ID enumeration describes the types of errors that can be detected on the [FastIO\_React\_NTS function block](FastIOReactNTS-1633EB8D.html).

## Enumeration Elements

The enumeration data type ENUM contains the following values:

| Name | Value | Description |
| --- | --- | --- |
| NO\_ERROR | 0 | No error is detected. |
| INTERNAL\_FIELD\_POWER\_SUPPLY | 1 | Internal field power supply is not detected. |
| SHORTCUT | 2 | A short-circuit is detected in the channel. |
| FALLBACK | 3 | An implicit communication issue is detected. The module is in fallback state. The expert function is stopped. |
| COMM\_ERR | 100 | The module is unavailable or the NIM is unavailable or the GCS (Group Cyclic Status) value is incorrect. |
