# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_etMode | ET\_ControllerMode | Controller type selection. |
| stPI | ST\_ControllerPi | The parameters for the PI controller. |
| stThreeStep | ST\_ControllerThreeStep | The parameters for the ThreeStep controller. |
| q\_lrControlPosition | LREAL | Actual position of the bobbin based on the controller. This output can be used to analyze and optimize the control behavior of the unwinder. |
| stTorque | ST\_ControllerTorque | The parameters for the torque controller. |
