## LibDoc Scripting Collection

The LibDoc Scripting Collection is provided as a file called LibDoc-<version>.zip. To install the [scripting collection](../_downloads/750fe93928dfd90773e35bb6583e6257/LibDoc-3.5.11.0.zip) please execute the following command.

```
(LibDoc) > pip install LibDoc-<version>.zip
```

To test all this installations, please execute the following lines of code:

```
> Scripts\activate
(LibDoc) > pip list
docopt (0.6.1)
LibDoc (0.0.1a4)
docutils (0.11)
Jinja2 (2.7.1)
pip (1.5)
setuptools (2.1)
Sphinx (1.2)
(LibDoc) > python
>>> from libdoc import __version__
>>> print __version___
<version>
>>> exit()
(LibDoc) > libdoc --version
<version>
```
