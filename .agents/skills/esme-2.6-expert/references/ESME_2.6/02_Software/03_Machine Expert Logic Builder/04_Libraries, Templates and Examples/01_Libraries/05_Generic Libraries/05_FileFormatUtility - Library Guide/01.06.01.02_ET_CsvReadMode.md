# ET\_CsvReadMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.8.0 |

## Description

The enumeration ET\_CsvReadMode defines the content to be read from the CSV file using the function block FB\_CsvRead.

## Enumeration Elements

| Name | Value (INT) | Description |
| --- | --- | --- |
| AllValues | 0 | All values are read from the CSV file.  The file must be dimensioned within the limits imposed by the number of records and the number of columns as specified under uiNumOfRows and uiNumOfColumns in structure ST\_CsvTable. |
| OneRow | 1 | One row (record) is read from the CSV file.  Row number must be specified with udiNumOfRow of ST\_CsvReadParameter. Row number can be greater than the uiNumOfRows (structure ST\_CsvTable). The number of columns of the file must be dimensioned within uiNumOfColumn in structure ST\_CsvTable. |
| OneColumn | 2 | One column is read from the CSV file.  Column number must be specified with udiNumOfColumn of ST\_CsvReadParameter. Column number can be greater than uiNumOfColumns (structure ST\_CsvTable). The number of rows of the file must be dimensioned within uiNumOfRows in structure ST\_CsvTable. |
| OneValue | 3 | One single value is read from the CSV file.  Row and column number must be specified with udiNumOfRow and udiNumOfColumn of ST\_CsvReadParameter. Both values can be greater than uiNumOfRows and uiNumOfColumns in structure ST\_CsvTable. |
| GetFileInformation | 4 | Only the information about the content of the file is retrieved. No value is read.  The complete file is scanned to determine the total number of rows and columns. There is no restriction in the function block concerning number of rows or number of columns of the file. |
