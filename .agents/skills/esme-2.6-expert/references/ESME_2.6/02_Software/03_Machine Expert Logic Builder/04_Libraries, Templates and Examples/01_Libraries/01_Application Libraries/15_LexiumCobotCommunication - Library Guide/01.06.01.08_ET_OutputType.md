# ET\_OutputType

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

This enumeration type is used to address a specific output type of the Lexium Cobot.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Cabinet | 0 | The outputs of the Cabinet of the Lexium Cobot are addressed. |
| ToolEnd | 1 | The outputs of the ToolEnd of the Lexium Cobot are addressed. |
