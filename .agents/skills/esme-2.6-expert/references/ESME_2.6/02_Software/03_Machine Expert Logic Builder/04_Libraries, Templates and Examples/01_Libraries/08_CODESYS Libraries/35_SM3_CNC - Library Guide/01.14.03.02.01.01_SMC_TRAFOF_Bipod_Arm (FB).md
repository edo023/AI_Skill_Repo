# SMC\_TRAFOF\_Bipod\_Arm (FB)

FUNCTION\_BLOCK SMC\_TRAFOF\_Bipod\_Arm

The transformation computes the forward transformation of a bipod with two rotary axes. See [SMC\_TRAFO\_Bipod\_Arm](YMZ8X3rM5laUkqR8DeCMAtAm7kI_smc-trafo-bipod.html#ymz8x3rm5laukqr8decmatam7ki_smc_trafo_bipod_smc_trafo_bipod_arm_fb) for a description.

**Note**

For more information on the supported transformations, see the chapter on CNC transformations.
