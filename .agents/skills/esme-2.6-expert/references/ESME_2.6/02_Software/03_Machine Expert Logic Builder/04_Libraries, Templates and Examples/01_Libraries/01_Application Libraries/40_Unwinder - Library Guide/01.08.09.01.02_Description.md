# Description

Description

Use this structure to define parameters when the precontrol movement is commanded by an external master (for instance the feeder axis or the machine master itself).

For this movement, the bobbin is connected to the master via a Straight CAM type.

The slope is defined by the radius of the bobbin.

When the bobbin radius changes, an additional movement of type Poly5 CAM is performed to ensure a smooth transition between both slopes.
