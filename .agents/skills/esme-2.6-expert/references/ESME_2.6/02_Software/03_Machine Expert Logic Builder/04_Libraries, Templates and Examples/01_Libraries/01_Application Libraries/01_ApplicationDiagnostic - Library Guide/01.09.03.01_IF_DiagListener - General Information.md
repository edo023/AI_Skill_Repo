## Task

Interface to retrieve the diagnostic messages in conjunction with a diagnostic handler.

## Description

The interface IF\_DiagListener is implemented by the listener which in conjunction with a diagnostic handler retrieves the diagnostic messages. This listener can be registered to a diagnostic handler using the method RegisterListener.

The interface IF\_DiagListener is implemented by the [FB\_ActiveDiagnostics](FB_ActiveDiagnostics-GeneralInforma-44A3BD07.html) function block.

The interface can also be implemented by a user-defined function block declared inside the application. The methods of the function block that must be implemented according to the definition of the IF\_DiagListener are called by the diagnostic handler to provide information related to set diagnostic messages or diagnostic quit status. They can be processed according to the requirements of the application.
