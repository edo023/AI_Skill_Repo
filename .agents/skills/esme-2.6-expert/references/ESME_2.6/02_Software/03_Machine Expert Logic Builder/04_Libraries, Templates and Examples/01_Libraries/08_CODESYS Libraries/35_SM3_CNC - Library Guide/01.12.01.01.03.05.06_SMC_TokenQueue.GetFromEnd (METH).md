# SMC\_TokenQueue.GetFromEnd (METH)

METHOD GetFromEnd : POINTER TO SMC\_Token

Returns a pointer to the n-last token, or 0 if the queue has less than n+1 tokens. The pointer is valid until the next call to RemoveFirst. Only to be called safely from the writing task, or when it is made sure that no tokens are added concurrently.
