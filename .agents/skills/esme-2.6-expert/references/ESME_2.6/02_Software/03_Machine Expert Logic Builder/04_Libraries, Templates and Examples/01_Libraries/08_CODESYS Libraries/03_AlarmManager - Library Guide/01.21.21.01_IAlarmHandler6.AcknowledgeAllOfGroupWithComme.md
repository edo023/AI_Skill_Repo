# IAlarmHandler6.AcknowledgeAllOfGroupWithComment (METH)

METHOD AcknowledgeAllOfGroupWithComment

Acknowledges all active alarms of an alarm group, whose alarm class hasn’t the flag “acknowledge separately” set. Optionally a comment and the user id can be provided.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | usiAlarmGroupID | [ALARMGROUP\_ID](ALARMGROUP_ID.html#alarmgroup_id_alarmgroup_id_alias) | The ID of the alarm group | | Inout Const | wsComment | WSTRING | The comment for the state transition | | wsUserId | WSTRING | The user who provides the comment | |
