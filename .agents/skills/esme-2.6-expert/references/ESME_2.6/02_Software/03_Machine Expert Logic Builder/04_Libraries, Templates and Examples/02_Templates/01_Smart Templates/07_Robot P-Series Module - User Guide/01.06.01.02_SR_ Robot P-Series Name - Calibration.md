# SR\_<Robot P-Series Name> - Calibration (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.1.2.0 |

## Functional Description

Execute the calibration for a Robot P-Series Module.

For more detailed information, refer to [Calibration / Configuration / Logic Method and ModuleInterface](D-SE-0080393.html#D-SE-0080393).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifCalibration | SER.IF\_RobotPSeriesCalibration | Interface to execute calibration modes. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xRequest | BOOL | TRUE: Calibration is requested. If the Smart Template Robot P-Series Module is disabled, the interface at the input i\_ifCalibration is validated.  FALSE: Calibration is not requested. |
