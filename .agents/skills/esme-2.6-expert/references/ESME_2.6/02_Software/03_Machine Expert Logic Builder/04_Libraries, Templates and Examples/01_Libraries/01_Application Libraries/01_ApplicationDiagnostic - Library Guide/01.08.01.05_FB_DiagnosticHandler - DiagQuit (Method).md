## Task

Triggers a diagnostic quit command.

## Description

The method DiagQuit is used to trigger a diagnostic quit command. The command is forwarded to all children.

NOTE: A call of this method resets the properties stDiagMessage, stDiagMessageTriggeringReaction, stDiagMessageFirst, and etReaction. It sets the xDiagQuitInProgress property to TRUE if the associated diagnostic handler or one of its children has an active diagnostic message or reaction.

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Indicates TRUE if after calling the method DiagQuit, a diagnostic process is in progress. |
