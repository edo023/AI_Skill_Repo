# NumberOfAsyncTasksRange

NumberOfAsyncTasksRange

|  |  |
| --- | --- |
| Enumeration name: | NumberOfAsyncTasksRange |
| Enumeration value: | 13 |
| Description: | NumberOfAsyncTasks is outside of the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The internal used AsyncManager is not able to work because there are no tasks defined for him. | The global parameter inside the PacDriveLib library Gc\_udiNumOfAsyncTasks has to be set to a value greater than 0 and lower or equal to 10. |
