## Task

Confirms that the processing of the diagnostic quit command is completed.

## Description

The method DiagQuitDone is used to confirm that the diagnostic quit process within your application is completed. This can be successfully done only if the diagnostic quit process of all children is also completed and their property xDiagQuitInProgress indicates FALSE. By setting the i\_xRecursiveToTop input to TRUE after a successful call, the DiagQuitDone method of the parent is called recursively.

NOTE: A successful call of this method sets the property xDiagQuitInProgress to FALSE.
