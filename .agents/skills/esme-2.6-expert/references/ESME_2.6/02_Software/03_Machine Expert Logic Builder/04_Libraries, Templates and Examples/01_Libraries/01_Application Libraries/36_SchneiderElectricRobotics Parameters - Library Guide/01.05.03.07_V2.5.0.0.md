## New Functions

* Lexium S Robot:

  + Function block FB\_RobotSSeriesParameters containing the Lexium S Robot parameters.
  + Enumeration ET\_RobotSSeries containing the different Lexium S Robot types.
  + Function FC\_EtRobotSSeriesToString to convert an element of ET\_RobotSSeries to string.
* New robot types added to ET\_RobotPSeries:

  + P4 robots: VRKP4S0FWF01000
  + P6 robots: VRKP6S0C1000000, VRKP6S0FWF01000
