# ST\_RobotTargetWithSlots

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.4.1.0 |
| Inherits from: | - |

## Description

Structure that extends ST\_RobotTarget with information about the slots contained inside a container target (for example, the place positions inside a tray).

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| astSlotAttributes | ARRAY [1...Gc\_uiMaxNumberOfSlots] OF ST\_RobotTargetSlotAttributes | Array of ST\_RobotTargetSlotAttributes describing the slots related to the target. |
| uiNumberOfSlots | UINT | Number of configured slots. |
| uiNumberOfSlotsWithOwner | UINT | Number of slots with an assigned owner. |
| uiNumberOfProcessedSlots | UINT | Number of slots set as processed. |
