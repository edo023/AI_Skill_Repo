# BrakeRelease

BrakeRelease

The operation mode BrakeRelease is selected. The slave axes of the infeed distance managed by the module are not moved. Only the holding brakes of the axes are released and engaged.

The operation mode is parameterized in the structures [ST\_BrakeRelease](../Structures/Structures-3.htm#XREF_D_SE_0080849_1) and [ST\_Main](../Structures/Structures-4.htm#XREF_D_SE_0080851_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [PDL.FB\_Brake](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087255_1) are being activated.

Via the array [ST\_BrakeRelease](../Structures/Structures-3.htm#XREF_D_SE_0080849_1).i\_axDriveSelect, the axes are selected at which the brake is to be released. The first element selects the drive of the first belt, the second element the drive of the second belt, etc. Only the elements of those belts are checked that are driven by an axis managed by the module.

The diagnostic messages of all managed slave axes are monitored.

Possible state commands

o[Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The selected holding brakes are released.

o[Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> All holding brakes are engaged.

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is enabled.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is disabled.

Possible operation mode commands

o[Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [NoOp](#XREF_D_SE_0080863_4) takes place.

o[Manual](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Manual](#XREF_D_SE_0080863_5) takes place.

o[Auto](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Automatic](#XREF_D_SE_0080863_6).
