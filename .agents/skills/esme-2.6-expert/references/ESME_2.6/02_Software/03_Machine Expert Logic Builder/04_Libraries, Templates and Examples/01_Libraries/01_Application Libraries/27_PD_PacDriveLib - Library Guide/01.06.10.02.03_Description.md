# Description

Description

The axis is moved at a constant velocity while its position is held within a defined period.

NOTE: The stop behavior of the axis for i\_xStart (TRUE -> FALSE) depends on the parameters i\_lrDec, i\_lrJerk and i\_lrStopPosition.

Depending on the time of i\_xStart (TRUE -> FALSE) an axis stop can occur in the period or can last for multiple periods.
