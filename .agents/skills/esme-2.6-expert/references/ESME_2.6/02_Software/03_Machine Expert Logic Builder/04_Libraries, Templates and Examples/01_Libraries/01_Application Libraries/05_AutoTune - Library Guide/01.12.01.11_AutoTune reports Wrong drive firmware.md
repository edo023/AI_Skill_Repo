# AutoTune reports: Wrong drive firmware version (lResult - -10200)

AutoTune reports: Wrong drive firmware version (lResult = -10200)

The optimization cannot be performed with the current firmware version of the drive.

| Cause | Removal |
| --- | --- |
| To be able to use the AutoTune\_V0020xx library, a software version according to chapter [System Requirements](../System_Requirements/System_Requirements.htm#XREF_D_SE_0091176_4) is required for the drive. | Install a software version according to chapter [System Requirements](../System_Requirements/System_Requirements.htm#XREF_D_SE_0091176_4) on the drive (LXM62, LXM52 or ILM62).  Install a version of the AutoTune library via the library repository that matches the firmware version in your drive. |
