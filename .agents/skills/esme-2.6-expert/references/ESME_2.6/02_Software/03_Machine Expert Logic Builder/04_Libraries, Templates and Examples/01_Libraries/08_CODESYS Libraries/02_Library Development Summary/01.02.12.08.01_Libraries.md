## Libraries

The structure for a library file server corresponds to the following scheme:

\CODESYSLibs\<VendorName>\<LibName>\<Version>

In addition, a folder of each level provide an index file, which contains a list of file names of the current folder.

The bottom level contains the following files – the library and the index.

The server for missing libraries must be made known in CODESYS. The address of the web server and URL of the library files must be specified so that the download of missing libraries can work.
