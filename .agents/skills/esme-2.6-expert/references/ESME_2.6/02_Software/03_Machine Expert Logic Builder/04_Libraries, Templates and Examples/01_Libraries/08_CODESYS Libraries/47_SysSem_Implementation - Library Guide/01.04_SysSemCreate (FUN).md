# SysSemCreate (FUN)

FUNCTION SysSemCreate : RTS\_IEC\_HANDLE

This function can be used to create a semaphore (synchronization object).

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | SysSemCreate | RTS\_IEC\_HANDLE | Handle to the semaphore or RTS\_INVALID\_HANDLE if failed. | | Input | pResult | POINTER TO RTS\_IEC\_RESULT | Pointer to runtime system error code (see CmpErrors.library) | |
