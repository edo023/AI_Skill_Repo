# DirRename (FB; Prefix fildrn)

FUNCTION\_BLOCK DirRename EXTENDS CBM.ETrig

This function block can be used to rename a directory and is optional for the different targets.

Contrary to the original CAA specification the directory may be renamed even if it is opened via ”FILE.[DirOpen](ApmELEjJXGYoLzYq0bSC23LAm_0_diropen.html#apmelejjxgyolzyq0bsc23lam_0_diropen_diropen_fb_prefix_fildop)”.

This behavior is dependent of the underlying operating system and file system.

Whether this function block is implemented or not is explicitly stated and encoded in the return value of the function ”FILE.[GetProperty](RFfZ6qcMtbnrwad-bdR1UipBxBA_getproperty.html#rffz6qcmtbnrwad_bdr1uipbxba_getproperty_getproperty_fun)”.

It is not recommended to use this function block to move files (it may work but is system dependent). Please use the appropriate function block ”FILE.[Rename](VIPLMVOpanJfjqQXFKWGQlzGjxw_rename.html#viplmvopanjfjqqxfkwgqlzgjxw_rename_rename_fb)” instead.
