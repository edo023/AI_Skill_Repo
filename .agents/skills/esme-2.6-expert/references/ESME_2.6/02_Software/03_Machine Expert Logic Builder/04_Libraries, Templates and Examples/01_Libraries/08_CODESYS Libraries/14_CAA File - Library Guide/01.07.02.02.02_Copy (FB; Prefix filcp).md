# Copy (FB; Prefix filcp)

FUNCTION\_BLOCK Copy EXTENDS CBM.ETrigToA

This function block copies a file. The target file is created or - if already existent - overwritten. The overwrite mode can be defined with the input ”xOverWrite”. If the overwrite mode is activated, no write protection may be activated in the existing file.

The function block is optional for different targets. Whether this function block is implemented or not is explicitly stated and encoded in the return value of the function ”FILE.[GetProperty](RFfZ6qcMtbnrwad-bdR1UipBxBA_getproperty.html#rffz6qcmtbnrwad_bdr1uipbxba_getproperty_getproperty_fun)”.
