## Description

A Cartesian pose describes both a position and an orientation with reference to a coordinate system. A structure ST\_CartesianPose containing the following structure elements is added to the library.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stPosition | *SE\_MATH.ST\_Vector3D* | Representation of a Cartesian position as a 3D vector. |
| stOrientation | *SE\_MATH.ST\_Vector3D* | Representation of a Cartesian orientation in the form of roll, pitch, and yaw angles. |
| etOrientationConvention | SE\_MATH.ET\_OrientationConvention | Enumerator containing the list of rotation orders supported by the library when describing an orientation as roll, pitch, and yaw. |
