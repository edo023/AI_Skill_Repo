# HighByte (FUN)

FUNCTION HighByte : BYTE

Function returns the high byte of the input “wValue”.

Example:

```
wValue : WORD := 16#1234 ;
->
MEM.HighByte(wValue) = 16#12
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | HighByte | BYTE | function returns high byte of “wValue” | | Input | wValue | WORD | word with high and low byte | |
