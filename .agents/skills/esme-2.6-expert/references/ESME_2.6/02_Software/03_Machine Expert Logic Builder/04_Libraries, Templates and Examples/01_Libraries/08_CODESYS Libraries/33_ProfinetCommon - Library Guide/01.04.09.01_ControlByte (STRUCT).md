# ControlByte (STRUCT)

TYPE ControlByte : STRUCT

The Control Byte is being sent with each safety PDU for a (Sub)Module from the IO-Controller to the device

| InOut: | | Name | Type | Comment | | --- | --- | --- | | iPar\_EN | BIT | Bit0 iParameter assignment enabled | | OA\_Req | BIT | Bit1 Operator acknowledge requested (indication) | | R\_cons\_nr | BIT | Bit2 Reset MonitoringNumber | | Use\_TO2 | BIT | Bit3 Use F\_WD\_Time\_2 (secondary watchdog) | | activate\_FV | BIT | Bit4 Fail-safe values (FV) to be activated | | Toggle\_h | BIT | Bit5 Toggle Bit | | ChF\_Ack | BIT | Bit6 Operator acknowledge after cleared channel fault | | Loopcheck | BIT | Bit7 at F\_CRC\_Seed = 0: Loopcheck - Loop-back check (“1”) is active  Bit7 at F\_CRC\_Seed = 1: BaseID\_Active\_h - Optional feature BaseID is active | |
