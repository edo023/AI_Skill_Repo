# General Information

## Description

The library "PD\_Template" is an optional upgrade of the default libraries. It contains recurring functionalities required for machine programming.

The following areas are taken into account in the "PD\_Template" library and facilitate project planning:

* Functions and structures
* Mapping mechatronic functions
* Design, modular and scalable architecture
* Machine flow control
* Exception handling (EMERGENCY OFF, exceptions, warnings …)
* Commissioning (visualizations)
* Diagnostic functionalities

## Namespace of the Library

The default namespace of this library is TPL.

The POUs, data structures, enumerations and constants have to be addressed via this namespace.
