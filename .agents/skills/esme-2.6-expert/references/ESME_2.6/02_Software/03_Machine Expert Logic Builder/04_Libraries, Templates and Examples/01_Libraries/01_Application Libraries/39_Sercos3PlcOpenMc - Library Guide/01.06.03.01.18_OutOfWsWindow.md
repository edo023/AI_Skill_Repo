# OutOfWsWindow

OutOfWsWindow

|  |  |
| --- | --- |
| Enumeration name: | OutOfWsWindow |
| Enumeration value: | 47 |
| Description: | The position is outside the restore window. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The value 3 was applied at the input StartMode .  The axis is located further away than i\_lrStartWindow from its start position. | The axis must be moved up to i\_lrStartWindow to the start position.  i\_lrStartWindow must be enlarged.  Another StartMode must be selected. |
