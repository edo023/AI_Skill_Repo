# SMC\_TRAFOF\_Gantry3D (FB)

FUNCTION\_BLOCK SMC\_TRAFOF\_Gantry3D

Forward transformation for three dimensional gantry systems. Each instance of SMC\_TRAFOF\_GANTRY3D can be connected to a visualization template named SMC\_VISU\_GANTRY3D\_XY or SMC\_VISU\_GANTRY3D\_YZ.

**Note**

For detailed information on transformations see overview to [SM\_Trafo\_POUs](uXtGJT17gvW9uxS1De45C-gyyqc_fld-SM_Trafo_POUs.html#uxtgjt17gvw9uxs1de45c_gyyqc_fld_sm_trafo_pous_sm_trafo_pous).
