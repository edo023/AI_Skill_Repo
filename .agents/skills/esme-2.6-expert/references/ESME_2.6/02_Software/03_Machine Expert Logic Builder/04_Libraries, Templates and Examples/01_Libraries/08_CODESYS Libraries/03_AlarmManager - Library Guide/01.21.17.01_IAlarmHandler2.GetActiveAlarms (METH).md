# IAlarmHandler2.GetActiveAlarms (METH)

METHOD GetActiveAlarms : UDINT

Returns the alarms, that are currently active, according to given filter criteria of the IAlarmManagerClient instance. The return value indicates, whether the processing was successful or not: \* ERR\_OK: Output parameters of method are valid, client was found \* ERR\_NO\_OBJECT: Client was not found, has been not yet added to the vector \* ERR\_NOTINITIALIZED: No client has been registered at the alarm manager
