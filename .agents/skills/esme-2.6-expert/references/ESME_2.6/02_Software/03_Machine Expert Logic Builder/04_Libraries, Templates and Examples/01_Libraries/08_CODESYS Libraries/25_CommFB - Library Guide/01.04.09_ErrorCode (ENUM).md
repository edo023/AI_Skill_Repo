# ErrorCode (ENUM)

TYPE ErrorCode :

The STATUS output has the data type DWORD which is interpreted as a packed array of four bytes.

Byte 0: Function\_Num

Byte 1: Error\_Decode

Byte 2: Error\_Code\_1

Byte 3: Error\_Code\_2

The Elements are used as specified in IEC 61158-6 (Profibus DP, Profinet IO)

Exception: If Function\_Num has a value of 16#40, no protocol element is used. This is the case for example if the validation of one input parameter failed. In this case Error\_Decode value of 16#82..16#8F reports an error at parameter number 2 - 15.

Other fixed error codes are defined in this enum.
