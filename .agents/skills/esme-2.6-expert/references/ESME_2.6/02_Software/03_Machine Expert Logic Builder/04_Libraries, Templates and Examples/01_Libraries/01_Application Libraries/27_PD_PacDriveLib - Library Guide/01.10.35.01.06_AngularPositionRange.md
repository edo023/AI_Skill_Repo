# AngularPositionRange

AngularPositionRange

|  |  |
| --- | --- |
| Enumeration name: | AngularPositionRange |
| Enumeration value: | 65 |
| Description: | AngularPosition is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The outputs q\_lrLinearPosition and q\_lrForce cannot be calculated for the value defined for i\_lrAngularPosition. | Verify i\_lrAngularPosition  Verify all parameters that are part of the calculation: i\_lrRadius, i\_lrPushRod, i\_lrYOffset |
