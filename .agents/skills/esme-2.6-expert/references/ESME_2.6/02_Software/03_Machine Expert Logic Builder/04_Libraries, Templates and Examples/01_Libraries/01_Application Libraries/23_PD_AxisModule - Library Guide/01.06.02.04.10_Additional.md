# Additional

Additional

The operation mode Additional is selected. In this operating mode an external user POU is activated. The axis module takes over the position control of the axis and the command handling.

The operating mode is parameterized in the structures [ST\_AdditionalMode](../Structures/Structures-2.htm#XREF_D_SE_0077213_1) and [ST\_Main](../Structures/Structures-7.htm#XREF_D_SE_0077223_1).

The technology POU [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) is being activated.

Basic monitoring systems are enabled:

oMonitoring the diagnostic messages of the axis

oMonitoring of the hardware limit switches

oMonitoring of the reduced velocity

Possible state commands

o[Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The q\_xStart bit of the [ST\_AdditionalMode](../Structures/Structures-2.htm#XREF_D_SE_0077213_1) structure is being set to TRUE. If the operating mode was entered via the [AdditionalCs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command, a cold start is executed. The q\_xWsSelect bit of the [ST\_AdditionalMode](../Structures/Structures-2.htm#XREF_D_SE_0077213_1) structure remains on FALSE.

 If the operating mode was entered via the [AdditionalWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command, a warm start is preselected. The q\_xWsSelect bit of the [ST\_AdditionalMode](../Structures/Structures-2.htm#XREF_D_SE_0077213_1) structure is being set to TRUE. Subsequently, a [WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command is waited for.

o[WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> After the warm start has been prepared by means of the commands [AdditionalWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) - [Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1), the warm start motion is started via [WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1). The q\_xWsStart bit of the [ST\_AxisModu­leAdditionalMode](../Structures/Structures-2.htm#XREF_D_SE_0077213_1) structure is being set to TRUE.

o[Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The external POU is stopped. The q\_xStart bit of the [ST\_AxisModuleAdditionalMode](../Structures/Structures-2.htm#XREF_D_SE_0077213_1) structure is being set to FALSE.

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The position control of the axis is activated.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The position control of the axis is being disabled.

Possible operating mode commands

o[Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [NoOp](#XREF_D_SE_0077326_3) takes place.

o[Homing](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Homing](#XREF_D_SE_0077326_4) takes place.

o[CamCs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [MultiCam](#XREF_D_SE_0077326_6) takes place and a cold start is preselected.

o[CamWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [MultiCam](#XREF_D_SE_0077326_6) takes place and a warm start is preselected.

o[Manual](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Manual](#XREF_D_SE_0077326_5) takes place.

o[Endless](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Endless](#XREF_D_SE_0077326_7) takes place.

o[EndlessIls](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [EndlessIls](#XREF_D_SE_0077326_8) takes place.

o[Positioning](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Positioning](#XREF_D_SE_0077326_9) takes place.

o[BrakeRelease](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [BrakeRelease](#XREF_D_SE_0077326_10) takes place.
