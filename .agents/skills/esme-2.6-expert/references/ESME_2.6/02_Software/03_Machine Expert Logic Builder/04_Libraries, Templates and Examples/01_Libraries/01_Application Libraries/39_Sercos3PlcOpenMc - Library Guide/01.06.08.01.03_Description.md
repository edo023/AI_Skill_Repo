Description

If the slave axis executes a synchronous motion, the job buffer of the axis is cleared via [SystemInt­erface.FC\_JobClear](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085167_1) and the axis set to standstill via Axis.ControllerStopSet= TRUE.
