# HwLimitNegStillActive

HwLimitNegStillActive

|  |  |
| --- | --- |
| Enumeration name: | HwLimitNegStillActive |
| Enumeration value: | 275 |
| Description: | HwLimitNeg is still active. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The negative limit switch (i\_xHwLimitNeg = TRUE) is still enabled.  However, the maximum travel distance (i\_lrMaxTravel) has been reached. | Verify the signal at the input i\_xHwLimitNeg.  Verify the value for the maximum travel distance at the i\_lrMaxTravel input.  Ensure that the sensor of the negative limit switch is not dirty. |
