# FB\_VisUnitManager - Init (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.1.0.0 |

## Description

The method is used for assigning the parent node.

For more information, refer to [Providing Data for Visualization](ProvidingDataForVis-DCA4DB61.html).

## Inputs

| Name | Data type | Description |
| --- | --- | --- |
| i\_ifParent | IF\_VisManager | Specifies the visualization manager function block under which the function block FB\_VisUnitManager is registered. |

## Outputs

| Name | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | TRUE: The object assigned on the input i\_ifParent is invalid. FALSE: The object assigned on the input i\_ifParent is valid. |
| q\_etResult | [ET\_Result](ET_Result-DD9CD87C.html) | Provides diagnostic and status information as a numeric value. If q\_xError = FALSE, q\_etResult provides status information. If q\_xError = TRUE, q\_etResult provides diagnostic/error information. |
