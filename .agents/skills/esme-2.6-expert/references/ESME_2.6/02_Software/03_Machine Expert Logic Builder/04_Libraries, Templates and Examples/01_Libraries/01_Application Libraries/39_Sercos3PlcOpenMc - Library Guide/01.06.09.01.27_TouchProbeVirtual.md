# TouchProbeVirtual

TouchProbeVirtual

|  |  |
| --- | --- |
| Enumeration name: | TouchProbeVirtual |
| Enumeration value: | 64 |
| Description: | The connected Touchprobe is virtual. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The used Touchprobe is not real. | Make sure that the Touchprobe does not belong to a virtual device. (e.g. virtual Lexium62) |
