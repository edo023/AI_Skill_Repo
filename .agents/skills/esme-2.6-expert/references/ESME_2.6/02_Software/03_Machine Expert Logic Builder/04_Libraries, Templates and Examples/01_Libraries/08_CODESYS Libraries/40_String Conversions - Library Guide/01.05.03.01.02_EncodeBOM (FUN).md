# EncodeBOM (FUN)

FUNCTION EncodeBOM : UDINT

Writes into pbyData (which must be large enough) a *BOM* that matches the [ENCODING](QS9IiJwK2G4tvWpsSL8DUp7vIyA_encoding.html#qs9iijwk2g4tvwpssl8dup7viya_encoding_encoding_alias)

It returns the number of bytes written.
