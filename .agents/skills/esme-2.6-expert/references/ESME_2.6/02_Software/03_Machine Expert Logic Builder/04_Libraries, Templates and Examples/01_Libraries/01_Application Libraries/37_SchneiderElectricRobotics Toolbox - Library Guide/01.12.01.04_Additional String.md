# Additional String

## Overview

The difference between additional strings and user-specific strings is, that user-specific strings can be added to each product data, an additional string can be added only once at the end of a protocol.

## Additional String

| Designation | Content | Description |
| --- | --- | --- |
| Value Id | `A` | Indicates an additional string. |
| String length | `80`\* | Size/length of the additional string.  Range: 00...80 |
| Content | `#` | String to be transmitted. |
| \* A0: No data in additional string | | |

## Example

`A10IJKLmnop34`

String length = `10`

Content = `IJKLmnop34`
