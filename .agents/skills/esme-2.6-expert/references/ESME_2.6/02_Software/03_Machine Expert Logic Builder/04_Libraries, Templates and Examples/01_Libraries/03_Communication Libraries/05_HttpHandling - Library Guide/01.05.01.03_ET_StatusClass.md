# ET\_StatusClass

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_StatusClass contains the possible values that indicate the status class of the response. The status classes are defined according to RFC 2616 which describes the HTTP V1.1.

## Enumeration Elements

| Name | Value  INT | Description |
| --- | --- | --- |
| Informational | 1 | The request was received successfully, the process is continuing. |
| Success | 2 | The action was successfully received and accepted. |
| Redirection | 3 | Further action must be taken in order to complete the request. |
| ClientError | 4 | The request contains incorrect syntax or cannot be fulfilled. |
| ServerError | 5 | The server was not able to fulfill the request. |
