# DEVICE\_STATE (ENUM)

TYPE DEVICE\_STATE :

This enumeration lists all possible states of a device. The mapping to the according fieldbus specific states is defined by the associated fieldbus stack.

| Attributes: | qualified\_only |
| InOut: | | Name | Comment | | --- | --- | | UNKNOWN | The device state is unknown.  Example: No supervision mechanism active. | | STOPPED | The device is stopped. | | RUNNING | The device is running. | | ERROR | The device is in error state.  [GetDeviceError](jyrNEZTYIzIrX1TeqIWUiva0Awk_getdeviceerror.html#jyrneztyizirx1teqiwuiva0awk_getdeviceerror_getdeviceerror_fb) may supply a specific error code. | | DISABLED | The device is disabled in device tree. | | NOT\_CONFIGURED | The device has not yet been configured by the stack.  Example: Configuration phase not started yet. | | CONFIGURED | The device has been configured by the stack.  Example: Configuration phase finished but device is not in running state. | | NOT\_FOUND | The device was not found on bus. | |
