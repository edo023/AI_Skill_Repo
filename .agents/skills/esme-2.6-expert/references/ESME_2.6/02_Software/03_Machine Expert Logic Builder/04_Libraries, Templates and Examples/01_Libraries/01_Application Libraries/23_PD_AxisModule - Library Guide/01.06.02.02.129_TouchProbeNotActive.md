# TouchProbeNotActive

TouchProbeNotActive

|  |  |
| --- | --- |
| Enumeration name: | TouchProbeNotActive |
| Enumeration value: | 95 |
| Description: | The Touchprobe is not active. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.PosDirectionPosEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionPosEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionNegEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.PosDirectionNegEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  The Touchprobe transferred to the [ST\_Home.stTouchProbe.i\_ifTouchProbe](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) parameter is not active.  For instance, this is an input at a Lexium62. | Ensure that the parameter IOx\_Mode of the input group (LXM62IO\_InOutTP) has been set to Touchprobe / 1. |
