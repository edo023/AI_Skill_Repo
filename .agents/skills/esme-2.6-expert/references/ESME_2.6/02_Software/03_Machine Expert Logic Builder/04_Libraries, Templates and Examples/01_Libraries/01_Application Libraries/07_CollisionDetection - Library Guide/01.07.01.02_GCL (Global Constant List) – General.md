# GCL (Global Constant List) – General Information

## Overview

|  |  |
| --- | --- |
| Type: | Global parameters |
| Available as of: | V1.0.0.0 |

## Description

Set of global constant variables.

## Global Parameters

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_udiNumberOfBoxVertices | UDINT | 8 | Number of vertices of a box. |
| Gc\_udiMaxNumberOfCollisionObjectVertices | UDINT | MAX(Gc\_udiUserMaxNumberOfCollisionObjectVertices, Gc\_udiNumberOfBoxVertices) | Maximum number of vertices used by a collision object. This must be ≥ Gc\_udiNumberOfBoxVertices. |
| Gc\_udiMaxNumberOfCollisionGroupObjects | UDINT | 150 | Maximum number of collision object inside a collision group. |
| Gc\_udiMaxNumberOfCollisionEntityGroups | UDINT | 20 | Maximum number of collision entity groups. |
| Gc\_udiDelta3AxDefaultNumberOfCollisionGroups | UDINT | 4 | By default, a Delta3Ax robot has one collision group for each chain and a collision group for the TCP, for a total of four groups. |
| Gc\_udiDelta3AxNumberOfJoints | UDINT | 3 | Number of joints for a Delta3Ax. |
| Gc\_udiDelta2AxDefaultNumberOfCollisionGroups | UDINT | 3 | By default, a Delta2Ax robot has one collision group for each chain and a collision group for the TCP, for a total of three groups. |
| Gc\_udiDelta2AxNumberOfJoints | UDINT | 2 | Number of joints for a Delta2Ax. |
| Gc\_udiSCARA4AxDefaultNumberOfCollisionGroups | UDINT | 2 | By default, a SCARA4Ax robot has one collision group for the whole arm and a collision group for the TCP, for a total of two groups. |
| Gc\_udiSCARA4AxNumberOfJoints | UDINT | 4 | Number of joints for a SCARA4Ax. |
