# FormatTimestamp2 (FUN)

FUNCTION FormatTimestamp2 : BOOL

This method extends the format timestamp to support the utc offset conversion calling FB\_Exit does not hurt because so far only on CharBufferString instances that operate on local buffers (ie. nothing done in FB\_Exit at all)
