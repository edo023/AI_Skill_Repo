# Description

Description

GPL (Global Parameter List) contains the global constants of the CrankModule library which can be overwritten project-specific in the library manager.

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_udiMaxNumberOfCamTables | UDINT | 4 | The highest index of the cam table which has to be switched in the axis module by using the TabSwitch function. Counting starts at 0. |
