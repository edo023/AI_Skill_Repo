# wClsClosedPosOfst

wClsClosedPosOfst

Offset of closed position for [closing on stack](../glossary/glossary.htm#XREF_D_SE_0024697_780). This parameter allows exceeding the closed position during closing on stack. When controlling clamshell grabs, it is usually required to exceed the closed position during closing on stack to prevent spillage of material during lifting. Start with a zero value and increase it if the grab spills during lifting. Spider (orange peel) grabs do not close fully when they are filled with material and therefore it is not necessary to use this parameter. Set this value to zero when controlling a spider grab.
