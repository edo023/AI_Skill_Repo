## Task

Interface with return values for the active connected path.

## Description

This interface is part of IF\_RobotFeedback and contains data on the active connected path.

A connected path consists of motion commands IF\_RobotMotion.MoveL, IF\_RobotMotion.MoveS or IF\_RobotMotion.MoveC, which were connected with each other by a blending zone.

## Properties
