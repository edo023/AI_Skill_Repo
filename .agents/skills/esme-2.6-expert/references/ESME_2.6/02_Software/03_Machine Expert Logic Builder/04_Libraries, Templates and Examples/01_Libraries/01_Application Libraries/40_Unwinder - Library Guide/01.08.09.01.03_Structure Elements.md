# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_ifMaster | IF\_Master | The interface of the master for the unwinder can be the feeder axis or the machine master. |
| i\_lrSlaveMasterRatio | LREAL | Factor with which the movement of the feeder is multiplied. For example, if the feeder has to have a defined feed constant of 360. Negative values invert the movement direction of the master. (Default value is 1). |
| i\_lrAcceleration | LREAL | Scaling factor to determine how long the transition phase is when the radius of the bobbin changes. The higher this number, the shorter the transition will be and conversely.  NOTE: This parameter has to be positive (i\_lrAcceleration > 0.0) and should be very small (i\_lrAcceleration < 0.0001) for a smooth transition. |
