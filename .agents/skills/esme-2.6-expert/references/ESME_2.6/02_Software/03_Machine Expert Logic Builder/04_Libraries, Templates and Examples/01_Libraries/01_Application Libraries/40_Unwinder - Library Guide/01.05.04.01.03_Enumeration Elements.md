# Enumeration Elements

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| InputVariable | 0 | The feedback value of the sensor is taken from the parameter ST\_FeedbackSignal.i\_lrSignalValue. |
| AnalogInput | 1 | The feedback value is read out of the given analog input. If high cyclic controllers are used, for example the PI, then the sensor feedback should be given here, in order to have an sensor value for every control call. |
| Simulation | 2 | The feedback value of the sensor is simulated in the behavior of a dancer. |
