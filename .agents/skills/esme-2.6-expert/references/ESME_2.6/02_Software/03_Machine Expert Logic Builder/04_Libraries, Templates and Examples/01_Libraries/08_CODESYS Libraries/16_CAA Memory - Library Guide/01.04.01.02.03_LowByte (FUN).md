# LowByte (FUN)

FUNCTION LowByte : BYTE

Function returns the low byte of the input “wValue”.

Example:

```
wValue : WORD := 16#1234 ;
->
MEM.LowByte(wValue) = 16#34
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | LowByte | BYTE | function returns low byte of “wValue” | | Input | wValue | WORD | word with high and low byte | |
