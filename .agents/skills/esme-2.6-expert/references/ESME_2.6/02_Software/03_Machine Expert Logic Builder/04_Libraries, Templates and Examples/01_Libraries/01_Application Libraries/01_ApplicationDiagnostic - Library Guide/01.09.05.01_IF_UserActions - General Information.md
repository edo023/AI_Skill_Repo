## Task

The interface IF\_UserActions is used to register a callback at the FB\_DiagnosticHandler.

## Description

The interface IF\_UserActions is used to register a callback on the FB\_DiagnosticHandler. To do this, the interface must be implemented in your own function block. As soon as a supported event occurs, the corresponding interface method is called by the FB\_DiagnosticHandler. To register the interface at the instance of the FB\_DiagnosticHandler, the method [RegisterUserActions](FB_DiagnosticHandler-RegisterUserAc-418AB150.html#FB_DiagnosticHandler-RegisterUserAc-418AB150) must be used.
