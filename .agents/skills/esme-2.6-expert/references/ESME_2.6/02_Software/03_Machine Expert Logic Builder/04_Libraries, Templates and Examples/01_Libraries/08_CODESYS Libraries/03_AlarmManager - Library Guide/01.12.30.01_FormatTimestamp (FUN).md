# FormatTimestamp (FUN)

FUNCTION FormatTimestamp

Takes a SYSTIME value and formats it. If the SYSTIME value is 0, an empty string is used as result.

| InOut: | | Scope | Name | Type | Initial | Comment | | --- | --- | --- | --- | --- | | Input | liTimestamp | [Timestamp](TIMESTAMP.html#timestamp_timestamp_alias) |  | The timestamp to format | | cbsResult | REFERENCE TO CharBufferString |  | The string, where the result will be stored | | sDateFormatPattern | STRING | ‘’ | optional date format | | sTimeFormatPattern | STRING | ‘’ | optionale time format | |
