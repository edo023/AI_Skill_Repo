# STATISTICS\_INT (FB)

FUNCTION\_BLOCK STATISTICS\_INT

Calculates minimum, maximum, and average of an input value

The calculation is done over time and can be reset in order to start a fresh caluclation/statistic.

| InOut: | | Scope | Name | Type | Initial | Comment | | --- | --- | --- | --- | --- | | Input | IN | INT |  | Input value | | RESET | BOOL |  | TRUE: AVG set to 0, MN and MX to minimum and maximum possible integer values | | Output | MN | INT | 32767 | Minimum value | | MX | INT | -32768 | Maximum value | | AVG | INT |  | Average value | |
