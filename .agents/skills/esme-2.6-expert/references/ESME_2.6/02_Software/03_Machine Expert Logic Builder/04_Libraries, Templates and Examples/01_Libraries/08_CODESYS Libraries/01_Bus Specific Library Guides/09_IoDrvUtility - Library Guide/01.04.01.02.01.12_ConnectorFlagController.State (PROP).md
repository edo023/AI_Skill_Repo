# ConnectorFlagController.State (PROP)

PROPERTY State : [ConnectorState](t9juu3iN-PnbrtWUCZ0YBTaJuMw_connectorstate1.html#t9juu3in_pnbrtwucz0ybtajumw_connectorstate1_connectorstate_enum)

Sets or gets the actually state of the connector, which is set in [ConnectorFlagController.init](EpDUA0ToSNz8wQXwG7EAxhZ0DJo_init1.html#epdua0tosnz8wqxwg7eaxhz0djo_init1_connectorflagcontroller_init_meth). For each state in [ConnectorState](t9juu3iN-PnbrtWUCZ0YBTaJuMw_connectorstate1.html#t9juu3in_pnbrtwucz0ybtajumw_connectorstate1_connectorstate_enum) expect [ConnectorState.DISABLED](t9juu3iN-PnbrtWUCZ0YBTaJuMw_connectorstate1.html#t9juu3in_pnbrtwucz0ybtajumw_connectorstate1_connectorstate_enum__t9juu3in_pnbrtwucz0ybtajumw_connectorstate1_connectorstate_disabled) a fixed bit combination is set in the |CONNECTOR\_FLAG\_STATES\_RS.STATE\_BITS| of dwFlags with the representation in |CONNECTOR\_FLAG\_STATES\_RS|. The mask |CONNECTOR\_FLAG\_STATES\_RS.STATE\_BITS| ensures to write the correct bits. If the masked bits have no valid combination, then [ConnectorState.UNDEFINED](t9juu3iN-PnbrtWUCZ0YBTaJuMw_connectorstate1.html#t9juu3in_pnbrtwucz0ybtajumw_connectorstate1_connectorstate_enum__t9juu3in_pnbrtwucz0ybtajumw_connectorstate1_connectorstate_undefined) is returned in get.
