OutOfRestoreWindow

|  |  |

| Enumeration name: | OutOfRestoreWindow |
| Enumeration value: | 195 |
| Description: | The position is outside the restore window. |

Has one of the following modes been selected at the i\_etMode input of [FB\_Home](#XREF_D_SE_0087275_1)?

o[ET\_HomeMode](../Enumerations/Enumerations-9.htm#XREF_D_SE_0087221_1).RestorePosFromRetain

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The axis has been moved too far since the last time the position has been saved. | Push the axis position back into the window.  Extend the limits of the i\_lrRestoreWindow window (stSetPos property of [FB\_Home](#XREF_D_SE_0087275_1)). |

If none of the modes mentioned above has been selected, please inform the support team about this detected error.
