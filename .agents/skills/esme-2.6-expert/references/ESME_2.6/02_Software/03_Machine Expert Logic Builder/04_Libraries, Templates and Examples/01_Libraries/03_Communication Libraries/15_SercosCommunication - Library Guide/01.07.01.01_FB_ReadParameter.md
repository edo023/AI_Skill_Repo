# FB\_ReadParameter

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.2.1.0 |

## Functional Description

The function block FB\_ReadParameter reads a data block element of a Sercos parameter with the service channel.

NOTE: Clear the memory area (assigned by the input i\_pbyData) after execution before the next use to avoid retention of data that would be subsequently invalid.

NOTE: Verify that the parameter access is completed even in case of a timeout (data which is assigned with the input i\_pbyData may be overwritten).

| WARNING | |

|  | UNINTENDED EQUIPMENT OPERATION OR LOSS OF DATA  Ensure that the function block is disabled in the application before using the Online Change command or the Log in with online change option.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xExecute | BOOL | A rising edge of the input i\_xExecute starts the execution of the function block. |
| i\_ifDevice | CMI.IF\_AxisIdentification | Reference to the Sercos device from which the parameter is read.  The following types are supported:   * SystemConfigurationItf.IF\_Drive (used by LMC Eco/Pro/Pro2 controllers) * MOIN.IF\_Axis (used by M262 controllers) |
| i\_dwParamterIDN | DWORD | Data service IDN  The input i\_dwParamterIDN can be set in different ways:  * Directly (with predefined list) * With FC\_BuildIDN * With FC\_IdnStringToDword |
| i\_etDataBlockElement | ET\_DataBlockElement | Defines which element of the parameter (given by the IDN number) is read. |
| i\_pbyData | POINTER TO BYTE | Pointer to the data memory, filled with the number of bytes defined by i\_uiDataLength |
| i\_uiDataLength | UINT | Number of data bytes to read |
| i\_timTimeOut | TIME | Maximum allowed execution time  Default value = 100 ms |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xBusy | BOOL | If this output is set to TRUE, the function block execution is in progress. |
| q\_xDone | BOOL | If this output is set to TRUE, the execution is completed successfully. |
| q\_xError | BOOL | The output q\_xError indicates TRUE if an error was detected. For details, refer to q\_etResult and q\_sResultMsg. |
| q\_etResult | ET\_Result | Provides diagnostic and status information as an enumeration value. |
| q\_sResultMsg | STRING [80] | Provides additional diagnostic and status information as a text message. |
| q\_uiReadDataLength | UINT | Output for reading the data length of a Sercos parameter (only valid, if input i\_etDataBlockElement = ET\_DataBlockElement.OperationDataVarLength, otherwise this value is 0)  NOTE: This output displays the length of the read Sercos parameter. It is independent of the input i\_uiDataLength. |
| q\_uiMaxReadDataLength | UINT | Output for reading the maximum data length of a Sercos parameter (only valid, if input i\_etDataBlockElement = ET\_DataBlockElement.OperationDataVarLength, otherwise this value is 0)  NOTE: This output displays the maximum length of the read Sercos parameter. It is independent of the input i\_uiDataLength. |
