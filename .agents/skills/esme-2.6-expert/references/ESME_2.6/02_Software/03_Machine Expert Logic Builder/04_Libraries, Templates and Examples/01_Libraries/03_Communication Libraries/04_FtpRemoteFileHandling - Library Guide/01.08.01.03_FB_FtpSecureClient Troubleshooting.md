# FB\_FtpSecureClient Troubleshooting

If an error is detected by the function block, verify the value of the output q\_sResultMsg. It provides detailed information about the detected error.

| Symptom | Possible cause | Solution |
| --- | --- | --- |
| The function block indicates a detected error and the value of q\_etResult is UnableToEstablishConnection. | The specified FTP server is not reachable. | Verify the connection parameters and try to reach the server with its IP address. |
| The server certificate is not trusted. | Verify whether the server certificate or the corresponding CA (Certificate Authority) certificate is trusted. For more information, refer to the [Programming Guide of your controller](../../../../../api/crossBook?lang=en-US&virtualBookName=m262prg&topicID=FTPServerCertificateVerification_1E1CDE8B). |
