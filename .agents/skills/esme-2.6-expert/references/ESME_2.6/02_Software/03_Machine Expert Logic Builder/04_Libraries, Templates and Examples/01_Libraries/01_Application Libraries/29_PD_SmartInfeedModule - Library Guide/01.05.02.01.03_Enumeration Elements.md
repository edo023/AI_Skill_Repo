# Enumeration Elements

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Manual | 710 | Change to operation mode Manual. |
| Auto | 720 | Change to operation mode Auto. |
| BrakeRelease | 730 | Change to operation mode Brake. |
| Start | 740 | Start the selected operation mode. |
| Stop | 750 | Cancel the command and return to the operation mode NoOp. |
| Hold | 760 | Cancel a start command. |
| WsStart | 780 | Trigger a warm start. |
| DriveEnableOn | 770 | Enable the position control of all axes. |
| DriveEnableOff | 775 | Disable the position control of all axes. |
