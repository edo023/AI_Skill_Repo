# Description

Description

The enumeration describes all velocity sources on the basis of which the outputs [q\_xMotionActive](../Structures/Structures-4.htm#XREF_D_SE_0080851_1) and [q\_axMotionActiveBelt](../Structures/Structures-4.htm#XREF_D_SE_0080851_1) of the [FB\_SmartInfeedModule](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080825_1) are set if the velocity exceeds [i\_lrMotionActiveVelLimit](../Structures/Structures-4.htm#XREF_D_SE_0080851_1).
