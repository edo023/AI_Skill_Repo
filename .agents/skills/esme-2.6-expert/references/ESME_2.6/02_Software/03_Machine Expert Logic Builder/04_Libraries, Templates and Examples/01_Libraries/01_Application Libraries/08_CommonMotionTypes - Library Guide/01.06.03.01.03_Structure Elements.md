# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| diNumberOfCamPoints | DINT | Number of cam points |
| diUserTableId(1) | DINT | Id for identifying the table during runtime |
| lrYPeriod(1) | LREAL | If this value is <> 0, then this value is assumed to be the Y period of the motion profile. If this is not the case, the Y period of the motion profile is determined as follows: astCamPoint[diNumberOfCamPoints - 1].lrY - astCamPoint[0].lrY |
| astCamPoint | ARRAY [0..SystemInterface.MAX\_NO\_OF\_SEG] OF [ST\_CamPoint](Structures-2.htm#XREF_D_SE_0096122_1) | List of cam points |
| (1) This parameter is only supported by PacDrive controllers. The parameter is ignored if you use Modicon M262 Logic/Motion Controller. | | |
