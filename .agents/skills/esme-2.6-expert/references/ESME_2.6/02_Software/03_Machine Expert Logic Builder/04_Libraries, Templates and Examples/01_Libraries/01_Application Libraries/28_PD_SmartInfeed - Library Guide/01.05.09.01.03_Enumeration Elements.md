# Enumeration Elements

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Equidistant | 0 | Targets with the same distance to [lrDistance](../Interfaces/Interfaces-28.htm#XREF_D_SE_0080342_1) are created. |
| EquidistantGroups | 1 | Groups of targets are created with [udiTargetsInGroup](../Interfaces/Interfaces-28.htm#XREF_D_SE_0080342_1) targets per group at a distance of [lrDistance](../Interfaces/Interfaces-28.htm#XREF_D_SE_0080342_1). A distance of [lrDistanceGroup](../Interfaces/Interfaces-28.htm#XREF_D_SE_0080342_1) is created between the groups. |
| TouchProbe | 2 | The targets are created by the filtered signals of [ifTouchProbe](../Interfaces/Interfaces-34.htm#XREF_D_SE_0080349_1). The targets detected by [ifTouchProbe](../Interfaces/Interfaces-34.htm#XREF_D_SE_0080349_1) are located [lrTpDistanceToTargetBeltStart](../Interfaces/Interfaces-28.htm#XREF_D_SE_0080342_1) before the beginning of the target belt. |
| External | 3 | No targets are created automatically. The user can specify targets using the methods [AddTargetDistance](../Interfaces/Interfaces-30.htm#XREF_D_SE_0080344_1) and [AddTargetPos](../Interfaces/Interfaces-30.htm#XREF_D_SE_0080344_1). |
