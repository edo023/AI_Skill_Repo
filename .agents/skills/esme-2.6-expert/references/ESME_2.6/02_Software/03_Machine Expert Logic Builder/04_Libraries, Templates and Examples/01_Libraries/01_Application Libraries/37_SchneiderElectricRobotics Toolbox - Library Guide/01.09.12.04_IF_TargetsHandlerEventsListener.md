## Task

Provides user-specific diagnostics when a target is removed from the list.

## Description

The OnRemoveTarget method is called by the targets handler where the implementation of IF\_TargetsHandlerEventsListener has been subscribed every time a target is removed from the list.

The following methods trigger the event:

* RemoveTarget
* RemoveAllTargets
* The method is also triggered every time a target is automatically removed from the list. This can occur if FB\_TargetsHandler is configured to remove the targets once they reach a maximum position (i\_xAutoRemoveTargets input of the Configuration method set to TRUE).

The diagnostics provided as output depend on your specific implementation. They are ignored by the library.
