# StrCaseCmpA (FUN)

FUNCTION StrCaseCmpA : INT

Does a case insensitive comparison of two strings

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | StrCaseCmpA | INT | * 0 -> the strings are equal * -1 -> |pByte1| is less than |pByte2| * 1 -> |pByte1| is larger than |pByte2| * -2 -> Error: An invalid string (ie. Null-Pointer is given) | | Input | pByte1 | POINTER TO BYTE | Pointer to the first string to compare | | pByte2 | POINTER TO BYTE | Pointer to the second string to compare | |
