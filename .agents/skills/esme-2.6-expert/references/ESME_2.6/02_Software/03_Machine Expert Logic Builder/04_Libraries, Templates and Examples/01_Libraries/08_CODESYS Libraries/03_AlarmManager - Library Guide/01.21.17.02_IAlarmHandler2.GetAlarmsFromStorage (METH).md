# IAlarmHandler2.GetAlarmsFromStorage (METH)

METHOD GetAlarmsFromStorage : UDINT

Reads alarms from the alarm storage using the filter criteria of the passed IAlarmManagerClient instance. The result will be stored in the array the IAlarmManagerClient instance passed in the call of method RegisterClient. The return value indicates, whether the processing was successful or not: \* ERR\_OK: Alarms from storage have been read successfully \* ERR\_NO\_OBJECT: Client was not found, has been not yet registered at the alarm manager \* ERR\_NOTINITIALIZED: No client has been registered at the alarm manager \* ERR\_NO\_ALARM\_STORAGE: No access to the alarm storage possible, because no array for

> storing alarms from storage has been passed during registering the client
