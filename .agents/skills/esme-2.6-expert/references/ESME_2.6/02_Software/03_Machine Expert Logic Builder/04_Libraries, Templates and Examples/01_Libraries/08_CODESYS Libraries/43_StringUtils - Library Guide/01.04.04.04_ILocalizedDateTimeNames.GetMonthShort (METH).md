# ILocalizedDateTimeNames.GetMonthShort (METH)

METHOD GetMonthShort : [CharBufferPtr](hfzJmOS2ddAPvg6yR-C9-xk02nE_charbufferptr.html#hfzjmos2ddapvg6yr_c9_xk02ne_charbufferptr_charbufferptr_alias)

returns the localized text of the month as short name. If no localization exists, a defaulttext (textid) should be returned. The stored value, the returnpointer is set to, will be overridden by any next call of one of the Get-methods of the implementing functionblock in VisuElemBase. Therefore, the result should be used at once. The pointer should not be stored.
