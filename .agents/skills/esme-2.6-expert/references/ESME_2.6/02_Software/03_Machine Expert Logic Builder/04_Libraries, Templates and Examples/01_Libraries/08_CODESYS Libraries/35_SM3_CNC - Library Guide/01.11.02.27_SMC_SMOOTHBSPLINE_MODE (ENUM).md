# SMC\_SMOOTHBSPLINE\_MODE (ENUM)

TYPE SMC\_SMOOTHBSPLINE\_MODE :

The mode determines how [SMC\_SmoothBSpline](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-smoothbsplin.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_smoothbsplin_smc_smoothbspline_fb) determines the knot vectors.

| InOut: | | Name | Comment | | --- | --- | | SMC\_SMOOTHBSPLINE\_UNIFORM | Uniform knot vector | | SMC\_SMOOTHBSPLINE\_CHORDLENGTH | Distance of knots proportional to chord length between control points | | SMC\_SMOOTHBSPLINE\_CENTRIPETAL | Distance of knots proportional to square root of chord length between control points | |
