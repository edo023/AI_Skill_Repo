## Description [[1]](#index_indices_and_tables__fn_index_id2)

This library contains a full implementation of the Common Behaviour Model as an object-oriented CODESYS library. The function block [BehaviourModel](n5IjNnDzG-Use9HSQOX0ejVKCSA_behaviourmodel.html#n5ijnndzg_use9hsqox0ejvkcsa_behaviourmodel_behaviourmodel_fb) implements everything expected from a BehaviourModel in a centralized manner.

This implementation is parameterized with, among other things, an [IActionProvider](D4ipCmCdfWAt64OLkZLrmdRGPIg_iactionprovider1.html#d4ipcmcdfwat64olkzlrmdrgpig_iactionprovider1_iactionprovider_itf) instance. An ActionProvider implements the operations for the methods [IStartActionProvider.StartAction (METH)](TgIRWKfoebcxOSGEX9MrVIrovXg_startaction.html#tgirwkfoebcxosgex9mrvirovxg_startaction_istartactionprovider_startaction_meth), [ICyclicActionProvider.CyclicAction (METH)](xCk-2KCNpmg0Syfw_jlg92dhEDc_cyclicaction.html#xck_2kcnpmg0syfw_jlg92dhedc_cyclicaction_icyclicactionprovider_cyclicaction_meth), [ICleanupActionProvider.CleanupAction (METH)](H78ShPlrryaTl2Picf-U2GaQm3E_cleanupaction.html#h78shplrryatl2picf_u2gaqm3e_cleanupaction_icleanupactionprovider_cleanupaction_meth) or the [IResetActionProvider.ResetAction (METH)](tjekeDi9GjZtgy2nGAt-VOtWZxY_resetaction.html#tjekedi9gjztgy2ngat_votwzxy_resetaction_iresetactionprovider_resetaction_meth). Depending on the required behaviour the ActionProvider can be decorated with a decorator such as [ILevelControlled](7Dy_vkh4hU084C7tLiDR95ZL0SQ_ilevelcontrolled.html#7dy_vkh4hu084c7tlidr95zl0sq_ilevelcontrolled_ilevelcontrolled_itf) or [IEdgeTriggered](7Dy_vkh4hU084C7tLiDR95ZL0SQ_iedgetriggered.html#7dy_vkh4hu084c7tlidr95zl0sq_iedgetriggered_iedgetriggered_itf). For example, in order to build a function block with a [LConC](LXh6gD4blZNWros8dlCwRnZiw1g_lconc.html#lxh6gd4blznwros8dlcwrnziw1g_lconc_lconc_fb) behaviour, its ActionProvider has to be decorated with the Interfaces [ILevelControlled](7Dy_vkh4hU084C7tLiDR95ZL0SQ_ilevelcontrolled.html#7dy_vkh4hu084c7tlidr95zl0sq_ilevelcontrolled_ilevelcontrolled_itf) and [IHasContinuousBehaviour](7Dy_vkh4hU084C7tLiDR95ZL0SQ_ihascontinuousbe.html#7dy_vkh4hu084c7tlidr95zl0sq_ihascontinuousbe_ihascontinuousbehaviour_itf). The StateMachine in the BehaviourModel then forces the ActionProvider to the appropriate [LConC](LXh6gD4blZNWros8dlCwRnZiw1g_lconc.html#lxh6gd4blznwros8dlcwrnziw1g_lconc_lconc_fb) behaviour. In this case, [STATE.EXECUTING](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_executing) can only be left through [STATE.ABORTED](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_aborted) or [STATE.ERROR](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_error). [STATE.DONE](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_done) is not accessible.

Furthermore the function block [BehaviourModel](n5IjNnDzG-Use9HSQOX0ejVKCSA_behaviourmodel.html#n5ijnndzg_use9hsqox0ejvkcsa_behaviourmodel_behaviourmodel_fb) implements the interface [IBehaviourModel](HxFoZ2ysCUMXYKT8P2lbmhX-bw0_ibehaviourmodel.html#hxfoz2yscumxykt8p2lbmhx_bw0_ibehaviourmodel_ibehaviourmodel_itf). This interface provides all methods and properties which are necessary for controlling a [BehaviourModel](n5IjNnDzG-Use9HSQOX0ejVKCSA_behaviourmodel.html#n5ijnndzg_use9hsqox0ejvkcsa_behaviourmodel_behaviourmodel_fb) instance via interface reference variables. When operating via an interface reference variable it should be possible to decide exactly when the state machine is called. For this purpose the methods [BehaviourModel.StartModel (METH)](torqZTIDHZx2YVTSqQsHZ-XN4hg_startmodel.html#torqztidhzx2yvtsqqshz_xn4hg_startmodel_behaviourmodel_startmodel_meth), [BehaviourModel.AbortModel (METH)](torqZTIDHZx2YVTSqQsHZ-XN4hg_abortmodel.html#torqztidhzx2yvtsqqshz_xn4hg_abortmodel_behaviourmodel_abortmodel_meth), [BehaviourModel.ResetModel (METH)](torqZTIDHZx2YVTSqQsHZ-XN4hg_resetmodel.html#torqztidhzx2yvtsqqshz_xn4hg_resetmodel_behaviourmodel_resetmodel_meth) and [BehaviourModel.GetModelState (METH)](torqZTIDHZx2YVTSqQsHZ-XN4hg_getmodelstate.html#torqztidhzx2yvtsqqshz_xn4hg_getmodelstate_behaviourmodel_getmodelstate_meth) have the parameter xCommit. Calling one of these methods with the value xCommit:=FALSE will only prepare the state changes and will not trigger the StateMachine to execute the related actions for changing the state. Calling one of these methods with the value xCommit:=TRUE will prepare the state changes and will then trigger the StateMachine to execute the necessary action in order to reach the next stable state. Sometimes the StateMachine needs more than one invocation to transit from one state to the next stable state. It is really important to retrieve the StateMachine only once per cycle.

The function blocks [ETrig](d5Ek35equq6Geb4iMYOV1LabLwI_etrig.html#d5ek35equq6geb4imyov1lablwi_etrig_etrig_fb), [LCon](OuY6VWj545vhZVQukdhGMUJjDWk_lcon.html#ouy6vwj545vhzvqukdhgmujjdwk_lcon_lcon_fb) and [LConC](LXh6gD4blZNWros8dlCwRnZiw1g_lconc.html#lxh6gd4blznwros8dlcwrnziw1g_lconc_lconc_fb) each use a [BehaviourModel](n5IjNnDzG-Use9HSQOX0ejVKCSA_behaviourmodel.html#n5ijnndzg_use9hsqox0ejvkcsa_behaviourmodel_behaviourmodel_fb) instance. They also implement the interface [IBehaviourModel](HxFoZ2ysCUMXYKT8P2lbmhX-bw0_ibehaviourmodel.html#hxfoz2yscumxykt8p2lbmhx_bw0_ibehaviourmodel_ibehaviourmodel_itf). They define an own FB-Interface while delegating as much as possible to the implementation of the [BehaviourModel](n5IjNnDzG-Use9HSQOX0ejVKCSA_behaviourmodel.html#n5ijnndzg_use9hsqox0ejvkcsa_behaviourmodel_behaviourmodel_fb) (See the corresponding \_Execute method).

These function blocks then serve as basis for the inheritance of the function blocks [ETrigA](d5Ek35equq6Geb4iMYOV1LabLwI_etriga.html#d5ek35equq6geb4imyov1lablwi_etriga_etriga_fb), [ETrigATl](d5Ek35equq6Geb4iMYOV1LabLwI_etrigatl.html#d5ek35equq6geb4imyov1lablwi_etrigatl_etrigatl_fb), [ETrigATo](d5Ek35equq6Geb4iMYOV1LabLwI_etrigato.html#d5ek35equq6geb4imyov1lablwi_etrigato_etrigato_fb), [ETRigATlTo](d5Ek35equq6Geb4iMYOV1LabLwI_etrigatlto.html#d5ek35equq6geb4imyov1lablwi_etrigatlto_etrigatlto_fb), [ETrigTlTo](d5Ek35equq6Geb4iMYOV1LabLwI_etrigtlto.html#d5ek35equq6geb4imyov1lablwi_etrigtlto_etrigtlto_fb), [ETRigTo](d5Ek35equq6Geb4iMYOV1LabLwI_etrigto.html#d5ek35equq6geb4imyov1lablwi_etrigto_etrigto_fb), [ETRigTl](d5Ek35equq6Geb4iMYOV1LabLwI_etrigtl.html#d5ek35equq6geb4imyov1lablwi_etrigtl_etrigtl_fb) originating from [ETrig](d5Ek35equq6Geb4iMYOV1LabLwI_etrig.html#d5ek35equq6geb4imyov1lablwi_etrig_etrig_fb), the function blocks [LConTl](OuY6VWj545vhZVQukdhGMUJjDWk_lcontl.html#ouy6vwj545vhzvqukdhgmujjdwk_lcontl_lcontl_fb), [LConTo](OuY6VWj545vhZVQukdhGMUJjDWk_lconto.html#ouy6vwj545vhzvqukdhgmujjdwk_lconto_lconto_fb), [LConTlTo](OuY6VWj545vhZVQukdhGMUJjDWk_lcontlto.html#ouy6vwj545vhzvqukdhgmujjdwk_lcontlto_lcontlto_fb) originating from [LCon](OuY6VWj545vhZVQukdhGMUJjDWk_lcon.html#ouy6vwj545vhzvqukdhgmujjdwk_lcon_lcon_fb) and the function block [LConTlC](LXh6gD4blZNWros8dlCwRnZiw1g_lcontlc.html#lxh6gd4blznwros8dlcwrnziw1g_lcontlc_lcontlc_fb) originating from [LConC](LXh6gD4blZNWros8dlCwRnZiw1g_lconc.html#lxh6gd4blznwros8dlcwrnziw1g_lconc_lconc_fb).

The implementation of an [IActionController](HxFoZ2ysCUMXYKT8P2lbmhX-bw0_iactioncontrolle.html#hxfoz2yscumxykt8p2lbmhx_bw0_iactioncontrolle_iactioncontroller_itf) can influence the conditions under which the [BehaviourModel](n5IjNnDzG-Use9HSQOX0ejVKCSA_behaviourmodel.html#n5ijnndzg_use9hsqox0ejvkcsa_behaviourmodel_behaviourmodel_fb) is allowed to execute the methods [IStartActionProvider.StartAction](TgIRWKfoebcxOSGEX9MrVIrovXg_startaction.html#tgirwkfoebcxosgex9mrvirovxg_startaction_istartactionprovider_startaction_meth), [ICyclicActionProvider.CyclicAction](xCk-2KCNpmg0Syfw_jlg92dhEDc_cyclicaction.html#xck_2kcnpmg0syfw_jlg92dhedc_cyclicaction_icyclicactionprovider_cyclicaction_meth), [ICleanupActionProvider.CleanupAction](H78ShPlrryaTl2Picf-U2GaQm3E_cleanupaction.html#h78shplrryatl2picf_u2gaqm3e_cleanupaction_icleanupactionprovider_cleanupaction_meth), [IExitActionProvider.ExitAction](QNpX716ZUUJkDdB7vmhXRHi0PLA_exitaction.html#qnpx716zuujkddb7vmhxrhi0pla_exitaction_iexitactionprovider_exitaction_meth) or [IResetActionProvider.ResetAction](tjekeDi9GjZtgy2nGAt-VOtWZxY_resetaction.html#tjekedi9gjztgy2ngat_votwzxy_resetaction_iresetactionprovider_resetaction_meth) of the corresponding [IActionProvider](D4ipCmCdfWAt64OLkZLrmdRGPIg_iactionprovider1.html#d4ipcmcdfwat64olkzlrmdrgpig_iactionprovider1_iactionprovider_itf).
