Use [IDevice2.CheckCurrentSupportedCommunicationState](C6k4Fa3i7pbW6AzB9qINy0B7Cc0_checkcurrentsupp.html#c6k4fa3i7pbw6azb9qiny0b7cc0_checkcurrentsupp_idevice2_checkcurrentsupportedcommunicationstate) to check which [DEVICE\_TRANSITION\_STATE](Wk7sVif4qdICA8XUp7Q627WapoY_device-transitio.html#wk7svif4qdica8xup7q627wapoy_device_transitio_device_transition_state_enum) is possible in the current state.
