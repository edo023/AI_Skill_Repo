## Task

FB to control a Lexium T Robot.

## Description

Function block to control a Lexium T Robot.

The initialization is performed with the inherited method [SERP.InitializeRobot](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRoboticsParameters&topicID=D_SE_0074979) and SetKinematicParameter.
