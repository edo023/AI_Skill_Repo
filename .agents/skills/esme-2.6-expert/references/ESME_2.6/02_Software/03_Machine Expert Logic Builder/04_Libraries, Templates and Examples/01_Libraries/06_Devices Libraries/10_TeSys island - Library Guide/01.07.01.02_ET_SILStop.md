# ET\_SILStop

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V2.0.3.0 |

## Description

The enumeration ET\_SILStop describes the status of the commands to the SIL group controlled by a Safety Interface Module (SIM).

## Enumeration Elements

| Name | Value (USINT) | Description |
| --- | --- | --- |
| SILNotPresent | 0 | SIL group not present in system configuration. |
| AvatarDeviceEvent | 1 | SIL group impacted by AvatarDeviceEvent. |
| SILStartersNotOpen | 2 | SIL group stop command received; SIL starters are not open yet. |
| AllSILStartersOpen | 3 | SIL group stop command successfully issued; SIL starters are open. |
| CommandOnlyOneTerminal | 4 | SIL group stop command issued to only one SIM input channel (jumper or SIM input wiring is causing an issue), but SIL starters did successfully open. |
| NormalOperation | 5 | Normal operation, SIL starters can be open or closed. |
