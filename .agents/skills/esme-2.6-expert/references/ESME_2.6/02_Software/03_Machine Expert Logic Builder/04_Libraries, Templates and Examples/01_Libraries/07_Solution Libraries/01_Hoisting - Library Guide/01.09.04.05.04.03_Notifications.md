# Notifications

Notifications

| Alarm Bit q\_wAlrmId | Description |
| --- | --- |
| 0 | Number of configured program executions i\_wExeCntCfg equals 0. |
| 1 | Configured task execution time i\_wTaskTime is out of the allowed range between 1 and 1000 ms. |
| 2 | Configured task jitter limit i\_wTaskJittLim is greater than 200 ms. |
| 3 | Configured filter time for the communication test delay i\_wComTestFltrTimeMs must not exceed 1000 ms. |
