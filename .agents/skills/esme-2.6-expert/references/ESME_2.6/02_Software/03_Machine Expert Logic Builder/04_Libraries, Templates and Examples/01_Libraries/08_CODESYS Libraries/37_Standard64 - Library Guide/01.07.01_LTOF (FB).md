# LTOF (FB)

FUNCTION\_BLOCK LTOF

Timer of delay. Q is FALSE, after the delay defined in PT and IN had a faling edge.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | IN | BOOL | starts timer with falling edge, resets timer with rising edge | | PT | LTIME | time to pass, before Q is reset | | Output | Q | BOOL | is FALSE, PT seconds after IN had a falling edge | | ET | LTIME | elapsed time since falling edge at IN | |
