# GPL (Global Parameter List)

## Overview

|  |  |
| --- | --- |
| Type: | Global parameters |
| Available as of: | V1.0.0.0 |

## Description

The global parameter list (GPL) contains global constants which are used by certain components of this library. The parameters can be edited individually for each application where the library is used. The modification must be done within the Library Manager of the project where the library is referenced.

NOTE: The variables are read-only. If you want to adjust the value of a variable, you need to adjust the corresponding value in the **ApplicationLoggerBase** library.

## Global Parameters

| Variable | Data type | Value | Description |
| --- | --- | --- | --- |
| Gc\_etDefaultLogLevel | ET\_LogLevel | APL\_Base.GPL.Gc\_etDefaultLogLevel (Default value: Nothing) | The default logger level of the Application Logger. |
| Gc\_udiMaxNumberOfLogEntries | UDINT | APL\_Base.GPL.Gc\_udiMaxNumberOfLogEntries (Default value: 500) | The maximum number of messages that are stored in the ring buffer of the Application Logger on the controller. |
| Gc\_udiMaxNumberOfLoggerPoints | UDINT | APL\_Base.Gc\_udiMaxNumberOfLoggerPoints (Default value: 100) | The maximum number of logger points that can be registered to the Application Logger. |

NOTE: If the parameter Gc\_udiMaxNumberOfLogEntries is set too low, messages in the ring buffer can be overwritten and get lost. For the Application Logger to automatically load new messages, the setting in Gc\_udiMaxNumberOfLogEntries must be greater than 5s/interval of log entries.
