# OutOfRestoreWindow

OutOfRestoreWindow

|  |  |
| --- | --- |
| Enumeration name: | OutOfRestoreWindow |
| Enumeration value: | 195 |
| Description: | The position is outside the restore window. |

Has one of the following modes been selected at the i\_etMode input of [FB\_HomeSetPos](#XREF_D_SE_0087290_1)?

o[ET\_HomeSetPosMode](../Enumerations/Enumerations-10.htm#XREF_D_SE_0087223_1).RestorePosFromRetain

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The axis has been moved too far since the last time the position has been saved. | Push the axis position back into the window.  Extend the limits of the window i\_lrRestoreWindow. |

If none of the modes mentioned above has been selected, please inform the support team about this detected error.
