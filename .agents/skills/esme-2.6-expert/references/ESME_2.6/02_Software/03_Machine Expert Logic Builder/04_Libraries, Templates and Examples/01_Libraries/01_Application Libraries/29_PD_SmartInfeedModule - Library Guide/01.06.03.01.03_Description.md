# Description

Description

Jogging jobs of the inputs i\_xJogForw and i\_xJogBack of the [Standard motion interface](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078574_1) are transferred to the module defined at i\_udiAxisSelect. If an exception is active in the exception list, all jogging jobs are canceled.
