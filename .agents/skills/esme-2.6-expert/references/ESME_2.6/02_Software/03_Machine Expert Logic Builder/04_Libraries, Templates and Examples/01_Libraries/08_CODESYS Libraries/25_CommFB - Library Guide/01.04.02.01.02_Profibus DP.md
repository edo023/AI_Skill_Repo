### Profibus DP

The input parameter ID of the Communication Function Blocks addresses one slot of a DP-slave or a DP-slave (Slot 0).

This handle may be generated by using one of the following functions:

* [ID](W4KfW0iFq1qBGJqo0M1JpzN9GHA_id1.html#w4kfw0ifq1qbgjqo0m1jpzn9gha_id1_id): Conversion of a physical address of a DP-slave to the handle
* [ADDR](W4KfW0iFq1qBGJqo0M1JpzN9GHA_addr1.html#w4kfw0ifq1qbgjqo0m1jpzn9gha_addr1_addr_fun): Conversion of a handle to the physical address of a DP-slave
* [SLOT\_ID](W4KfW0iFq1qBGJqo0M1JpzN9GHA_slot-id1.html#w4kfw0ifq1qbgjqo0m1jpzn9gha_slot_id1_slot_id_fun): Addressing a slot of a DP-slave
