# VelRange

VelRange

|  |  |
| --- | --- |
| Enumeration name: | VelRange |
| Enumeration value: | 11 |
| Description: | Vel is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The axis cannot move at the velocity of a positioning step. | For every parameter lrVel of a positioning step of the list i\_pstMultiPosStep, a value between 0 and the axis parameter MaxVel must be specified. |
