# ST\_PropertiesUnSuback

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_PropertiesUnSuback contains information extracted from the response sent by the MQTT broker after subscribing or unsubscribing to a topic.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sReasonString | STRING[255] | Specifies a human readable string indicated in the received message. NOTE: The length of the value is limited to 255 characters. If it exceeds the maximum length, it is truncated and the output q\_xTruncatedResponseData is set to TRUE. |
| astUserProperties | ARRAY[1..GPL.Gc\_uiMaxNumberOfUserProperties] OF ST\_UserProperty | Indicates the user properties received from the MQTT broker.  NOTE: A user property with an empty name represents the end of the list. |
