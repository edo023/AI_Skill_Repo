# Disconnect (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method is used to initiate the disconnection from the cloud server. The disconnection is executed even if there are send requests in the queue. The send requests which remain in the queue are processed when the client is reconnected. Once this method is called, the disconnection is processed by executing the method Cyclic().

This method has no inputs and outputs.
