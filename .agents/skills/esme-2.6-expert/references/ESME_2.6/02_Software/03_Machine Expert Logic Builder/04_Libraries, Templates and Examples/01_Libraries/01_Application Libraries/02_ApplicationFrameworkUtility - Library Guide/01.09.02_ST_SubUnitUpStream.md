# ST\_SubUnitUpStream

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

The structure ST\_SubUnitUpStream contains the data that are sent from a subunit to the function block FB\_PackMlSubUnitHandler.

## Structure Elements

| Input | Data type | Description |
| --- | --- | --- |
| etState | [PackML.ET\_States](../../../../../api/crossBook?lang=en-US&virtualBookName=PackMLli&topicID=D_SE_0077937) | Provides the PackML state of the subunit to the function block FB\_PackMlSubUnitHandler. |
| xIgnoreSubUnit | BOOL | TRUE: The function block FB\_PackMlSubUnitHandler does not generate commands for the subunit and does not monitor the state of the subunit. FALSE: The function block FB\_PackMlSubUnitHandler generates commands for the subunit and monitors the state of the subunit. |
