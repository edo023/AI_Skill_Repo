# Overview

The project implements an application example which shows how to implement email handling in a controller application using the EmailHandling library.

## System Requirements and Limitations

The application example has been created and tested with, or supported by, the components listed in the table:

| Component | Type and Version |
| --- | --- |
| Software | EcoStruxure Machine Expert V2.x |
| Controller | * Modicon M241 Logic Controller(1) * Modicon M251 Logic Controller * Modicon M258 Logic Controller(1) * Modicon LMC058 Motion Controller(1) * Modicon M262 Logic/Motion Controller * Modicon M660 Motion Controller (exclusive to EcoStruxure Machine Expert V2.5 and greater) * PacDrive LMC 1xx / 2xx(1) * PacDrive LMX 300 / 4xx/ 600 / 802C(1) |
| Additional devices | A PC with a running Email server and an additional Email client. |
| Additional requirements | The Email server is running on a PC and can be reached from the controller through the network. |
| **(1)** The concepts presented in the application example are in principle, transferable to other controllers supporting the controller firmware released in conjunction with EcoStruxure Machine Expert version 2.0 or greater. | |
