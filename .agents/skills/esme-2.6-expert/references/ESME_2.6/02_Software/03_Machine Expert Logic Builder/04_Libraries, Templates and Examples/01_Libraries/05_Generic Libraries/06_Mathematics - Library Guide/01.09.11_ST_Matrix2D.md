# ST\_Matrix2D

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |

## Description

The structure ST\_Matrix2D represents a quadratic matrix with two lines and two columns.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| alrElement | ARRAY [1..2, 1..2] OF LREAL | Two-dimensional field of the matrix with the primary field representing the lines and the secondary field representing the columns. |
