# IGestureEventHandler.HandleFlickGesture (METH)

METHOD HandleFlickGesture : BOOL

This method will be called when a pan gesture was detected by the visualiuation. RETURN: TRUE - When the handler has handled this event and it should not be handled by someone else FALSE - When the event is not handled by this handler

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | HandleFlickGesture | BOOL |  | | Input | start | VisuStructPoint | The point on the screen where this gesture was started | | offset | VisuStructPoint | In fact not a point but an offset. The distance, the gesture was moved on the screen. | | velocityPxPerSecond | UDINT | The velocity of this flick gesture in pixels per second | | pClient | POINTER TO VisuStructClientData | A pointer to the client structure were the event occurred. | |
