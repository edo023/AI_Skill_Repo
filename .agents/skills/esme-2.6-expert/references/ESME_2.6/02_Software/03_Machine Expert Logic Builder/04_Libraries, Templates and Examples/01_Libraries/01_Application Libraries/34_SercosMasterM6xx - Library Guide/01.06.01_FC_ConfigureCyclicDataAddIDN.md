# FC\_ConfigureCyclicDataAddIDN

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

The FC\_ConfigureCyclicDataAddIDN function of the library SercosMasterM6xx adds a Sercos parameter (IDN) to the cyclic data of a specified connection.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_wTopoAddr | WORD | Topology address of the Sercos device. |
| i\_etConnectionId | [SMI.ET\_ConnectionID](../../../../../api/crossBook?lang=en-US&virtualBookName=SercosMasterInterface&topicID=ET_ConnectionID_5343DAEE) | Type of Sercos telegram (AT, MDT) determining the direction of communication. |
| i\_dwConfigureIDNForCyclicCommunication | DWORD | IDN to be configured for cyclic communication. |
| i\_uiIDNDataLength | UINT | Data length of the IDN. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_udiIDNDataPosition | UDINT | Position of the IDN in cyclic communication. |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -461: Error detected: Incorrect Sercos communication phase. |
