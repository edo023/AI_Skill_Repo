# LIMITALARM (FB)

FUNCTION\_BLOCK LIMITALARM

Monitors if an input value is between a lower and a upper limit

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | IN | INT | Input value | | HIGH | INT | Upper limit | | LOW | INT | Lower limit | | Output | O | BOOL | TRUE: IN > HIGH  FALSE: Else | | U | BOOL | TRUE: IN < LOW  FALSE: Else | | IL | BOOL | TRUE: Neither O nor U are TRUE | |
