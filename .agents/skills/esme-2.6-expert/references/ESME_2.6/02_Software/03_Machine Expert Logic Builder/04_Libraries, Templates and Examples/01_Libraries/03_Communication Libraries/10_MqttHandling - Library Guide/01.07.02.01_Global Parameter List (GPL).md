# Global Parameter List (GPL)

## Overview

|  |  |
| --- | --- |
| Type: | Global parameters |
| Available as of: | V1.0.0.0 |

## Description

The global parameter list (GPL) contains global constants which are used by certain components of this library. The parameters can be edited individually for each application where the library is used. The modification must be done within the Library Manager of the project where the library is referenced.

## Global Parameters

| Variable | Data type | Default value | Range | Description |
| --- | --- | --- | --- | --- |
| Gc\_uiMaxNumberOfBytesToExchange | UDINT | 1000 | 100... 1000000 | The value indicates the sum of the maximum size of an application message in bytes. This value considers, beside the topic and application data, also the overhead required by the MQTT protocol. |
| Gc\_uiMaxNumberOfMessagesToHandlePerCycle | UINT | 20 | 1...100 | The value indicates the maximum number of received messages which the MQTT client has to handle in one cycle. |
| Gc\_uiMaxNumberOfQoS1PublishPerClient | UDINT | 25 | 1...50 | The value indicates the maximum number of simultaneously published messages with QoS1 per client. |
| Gc\_uiMaxNumberOfSubscribePerClient | UINT | 25 | 1...100 | The value indicates the maximum number of subscribed topics per client. |
| Gc\_uiMaxNumberOfSubscribeTotal | UINT | 100 | 10...500 | The value indicates the maximum number of subscribed topics inside the application. |
| Gc\_uiMaxNumberOfQoS1MessagesPerSubscribe | UINT | 10 | 1...50 | The value indicates the maximum number of received and unhandled messages with QoS1 per subscribed topic. |
| Gc\_uiMaxLengthOfClientID | UINT | 255 | 23...2000 | The value indicates the maximum length of the client identifier. |
| Gc\_uiMaxLengthOfCorrelationData | UINT | 81 | 80...255 | The value indicates the maximum length of the correlation data. |
| Gc\_uiMaxNumberOfUserProperties | UINT | 3 | 1...10 | The value indicates the maximum number of user properties in one message. |
| Gc\_uiMaxLengthOfUserProperties | UINT | 80 | 10...255 | The value indicates the maximum length of the name or value of a user property. |
| Gc\_uiMaxLengthOfReceivedTopicName | UINT | 255 | 80...1000 | The value indicates the maximum length of the topic name indicated by the function block FB\_MqttSubscribe. |
| Gc\_uiMaxNumberOfQoS1PublishTotal | UINT | 25 | 10...500 | The value represents the maximum number of simultaneously published messages with QoS1 inside the application. |
