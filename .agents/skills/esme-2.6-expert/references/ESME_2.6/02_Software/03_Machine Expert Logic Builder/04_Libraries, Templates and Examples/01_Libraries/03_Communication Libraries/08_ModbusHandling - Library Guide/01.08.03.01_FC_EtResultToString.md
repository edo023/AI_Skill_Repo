|  |  |
| --- | --- |
| Type: | Function |

## Task

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etVar | ET\_Result | Enumeration to be converted. |

## Return Value

| Data type | Description |
| --- | --- |
