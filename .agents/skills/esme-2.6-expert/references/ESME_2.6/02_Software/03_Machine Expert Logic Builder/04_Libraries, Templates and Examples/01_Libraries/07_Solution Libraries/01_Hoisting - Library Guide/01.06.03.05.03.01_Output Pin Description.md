# Output Pin Description

Output Pin Description

| Output | Data type | Description |
| --- | --- | --- |
| q\_xEn | BOOL | TRUE: The function block is enabled.  FALSE: The function block is disabled. |
| q\_astSyncGroup | ARRAY OF SyncGroupOut | Output structure of synchronization groups  For detailed description, refer to [SyncGroupOut](Advanced_Position_Synchronization-12.htm#XREF_D_SE_0091381_2). |
| q\_astAxisIntf | ARRAY OF AxisIntfOut | Output structure of axis interfaces  For detailed description, refer to [AxisIntfOut](Advanced_Position_Synchronization-10.htm#XREF_D_SE_0091382_2). |
| q\_xAlrm | BOOL | TRUE: Alarm detected.  FALSE: No alarm is detected. |
