# TimeoutDuringBootOperation

TimeoutDuringBootOperation

|  |  |
| --- | --- |
| Enumeration name: | TimeoutDuringBootOperation |
| Enumeration value: | 33 |
| Description: | A timeout occurred during the boot operation. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The transport system controller is unable to complete its initialization. | Increase the default value of i\_timBootOperationTimeout in [ST\_ControllerParameter](../Structures/Structures-2.htm#XREF_D_SE_0090700_1) to allow the transport system controller more time to complete initialization. |
