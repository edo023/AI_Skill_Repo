# SMC\_TRAFOV\_GantryCutter2 (FB)

FUNCTION\_BLOCK SMC\_TRAFOV\_GantryCutter2

Reverse transformation for two-dimensional gantry systems, taking into account the path velocity and path direction as control variables for the axis.

**Note**

For detailed information on transformations see overview to [SM\_Trafo\_POUs](uXtGJT17gvW9uxS1De45C-gyyqc_fld-SM_Trafo_POUs.html#uxtgjt17gvw9uxs1de45c_gyyqc_fld_sm_trafo_pous_sm_trafo_pous).
