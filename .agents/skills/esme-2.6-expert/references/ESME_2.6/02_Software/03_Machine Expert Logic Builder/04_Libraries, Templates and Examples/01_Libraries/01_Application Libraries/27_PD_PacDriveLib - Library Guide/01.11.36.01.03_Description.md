# Description

Description

The positive part is defined as follows:

oi\_lrValue >= 0.0 -> i\_lrValue

oi\_lrValue < 0.0 -> 0.0

Together with the function [FC\_NegativePart](Functions_K_to_Z-14.htm#XREF_D_SE_0087563_1), the following identity applies to all real lrValues:

lrValue = FC\_PositivePart(lrValue) - FC\_NegativePart(lrValue)
