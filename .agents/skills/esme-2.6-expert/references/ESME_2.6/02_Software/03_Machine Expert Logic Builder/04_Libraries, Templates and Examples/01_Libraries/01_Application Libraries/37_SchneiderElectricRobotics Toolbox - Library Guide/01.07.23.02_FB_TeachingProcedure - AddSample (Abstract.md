## Task

Adds a new sample.

## Description

With the method AddSample(...), a new sample for a teaching procedure is acquired.

NOTE: This method is abstract for the base FB\_TeachingProcedure implementation, so the description of its behavior, including the diagnostics, is delegated to the specific implementation of the procedures.

Access: PUBLIC

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrSampleTolerance | LREAL | Tolerance value used to verify the consistency of the samples.
