## Task

The function block combines various homing functionalities within one function block to help improve the scalability in an application.

## Description

The function block FB\_Home is used to perform a homing operation on an axis. It supports homing modes that are provided by the underlying axis or motion controller. The intended mode is selected and executed using the available command methods.

For detailed information on how to use the function block, refer to [Using the Function Block FB\_Home](FB_Home-ProgrammingExample-461AEFF7.html).

## ApplicationLogger - Logger Point

The function block implements a logger point that can be registered with the global Application Logger.

When registered, the function block [FB\_Home](FB_Home-459E662D.html#FB_Home-459E662D) generates logger entries on following events:

| Event | Log Level |
| --- | --- |
| Initialization of the function block is triggered. | APL2.ET\_LogLevel.UserAction |
| Initialization of the function block failed. | APL2.ET\_LogLevel.Exception |
| Logger Point successfully registered. | APL2.ET\_LogLevel.UserAction |
| Logger Point registration failed. | APL2.ET\_LogLevel.Exception |
| Homing command triggered. | APL2.ET\_LogLevel.UserAction |
| Homing command denied. | APL2.ET\_LogLevel.Exception |
| Homing command aborted. | APL2.ET\_LogLevel.Exception |

|  |  |
| --- | --- |
| Execution of command SetHomeOk failed. | APL2.ET\_LogLevel.Exception |

With the method [Register](IF_LoggerPointConfigurationRegister-4599E66B.html), the function block FB\_Home is registered as a logger point in the global Application Logger. The values for the LoggerPoint parameters type and source are set to FB\_Home and MFB, respectively. For more information refer to the ApplicationLogger Library Guide.
