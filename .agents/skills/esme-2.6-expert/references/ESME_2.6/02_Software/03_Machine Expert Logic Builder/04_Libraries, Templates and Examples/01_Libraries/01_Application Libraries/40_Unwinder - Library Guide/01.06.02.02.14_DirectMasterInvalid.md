# DirectMasterInvalid

DirectMasterInvalid

|  |  |
| --- | --- |
| Enumeration name: | DirectMasterInvalid |
| Enumeration value: | 63 |
| Description: | DirectMaster is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The feedback signal origin mode ET\_FeedbackSignalOrigin.Simulation is chosen and iq\_stParameter.stFeedbackSignal.stSimulation.i\_xUseDirectMaster is TRUE while the object connected at the input iq\_stParameter.stFeedbackSignal.stSimulation.i\_ifDirectMaster is no logical encoder. | At the input iq\_stParameter.stFeedbackSignal.stSimulation.i\_ifDirectMaster a logical encoder which is connected to the master object and is not used somewhere else must be specified. |
