# IBehaviourModel.StartModel (METH)

METHOD StartModel : [ERROR](e8R4--mzfNvSIX3lkxCmoWVrX_8_error.html#e8r4__mzfnvsix3lkxcmowvrx_8_error_error_enum)

The parameter xCommit allows to call the state machine whenever it is needed. When xCommit is FALSE, the state change to [STATE.STARTING](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_starting) is being prepared but not executed yet. When xCommit is TRUE, the state change to [STATE.STARTING](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_starting) is being prepared as well as executed.
