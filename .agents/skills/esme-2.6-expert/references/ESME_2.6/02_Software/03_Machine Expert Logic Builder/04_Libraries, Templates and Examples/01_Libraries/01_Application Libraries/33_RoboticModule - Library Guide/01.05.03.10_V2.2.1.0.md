## Enhancements

* A configured period of auxiliary axes is fully supported for the following homing modes:

  + IF\_Homing.LimitSwitch
  + IF\_Homing.MoveOnPos
  + IF\_Homing.RestorePosFromAxisEncoder
  + IF\_Homing.RestorePosFromRetain
  + IF\_Homing.Sensor
  + IF\_Homing.SetPosAxisPosition
  + IF\_Homing.Torque
  + IF\_Homing.TouchProbe
  + IF\_Homing.WritePos
* Configured hardware limits of auxiliary axes are fully supported during the execution of homing mode IF\_Homing.MoveOnPos.
* A configured RestoreOnInit is fully supported even if another homing mode is selected for the operation mode ET\_OpMode.Homing. This applies for the following homing modes:

  + IF\_Homing.RestorePosFromAxisEncoder
  + IF\_Homing.RestorePosFromRetain
* A configured RestoreWindow is fully supported for the homing mode IF\_Homing.RestorePosFromRetain.
* A configured MaxDistance is fully supported for jogging an auxiliary axis into negative direction.
