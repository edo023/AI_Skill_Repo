# Enumeration Elements

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | Products may overlap. The position of the products is determined according to the motions of the belts. |
| StrictFrontProduct | 1 | Overlapping is prevented. If two products have to overlap due to the motions of the belts, the preceding product is assumed as fixed. The front edge of the subsequent product must not move beyond the rear edge of the preceding product. |
