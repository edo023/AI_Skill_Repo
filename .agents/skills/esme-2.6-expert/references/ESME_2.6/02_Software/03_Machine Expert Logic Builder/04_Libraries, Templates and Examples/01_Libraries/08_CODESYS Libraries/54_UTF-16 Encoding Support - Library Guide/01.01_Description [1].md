## Description [[1]](#index_indices_and_tables__fn_index_id2)

Implements functions and constants to support text encoded in UTF-16. It includes functions to translate between runes (See: [RUNE](zNAnMkgR5a17w3XOB1HSBNNzcXw_rune.html#znanmkgr5a17w3xob1hsbnnzcxw_rune_rune_alias)) and UTF-16 byte sequences. See <https://en.wikipedia.org/wiki/UTF-16>
