# FB\_ApplicationLoggerHandler - EnableLogger (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.4.0 |

## Description

If the logger was disabled using the method DisableLogger, it can be re-enabled with this method. A logger message is created to indicate the enabling.

This method is unnecessary to be called at the start of the application execution because the logger is enabled by default.

| Output | Data type | Description |
