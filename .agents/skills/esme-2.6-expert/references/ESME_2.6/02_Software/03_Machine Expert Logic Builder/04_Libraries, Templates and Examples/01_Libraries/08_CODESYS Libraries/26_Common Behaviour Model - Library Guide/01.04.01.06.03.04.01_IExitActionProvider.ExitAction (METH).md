# IExitActionProvider.ExitAction (METH)

METHOD ExitAction

This method is intended for releasing external resources (like RTS handles). Do not call other internal function block instances in the context of this method. Call only external functions defined inside the RTS. For cleaning up other resources then external ones, please use the CleanupAction. This method should be callable in the context of FB\_EXIT for example before downloading a new application or before reseting an application.
