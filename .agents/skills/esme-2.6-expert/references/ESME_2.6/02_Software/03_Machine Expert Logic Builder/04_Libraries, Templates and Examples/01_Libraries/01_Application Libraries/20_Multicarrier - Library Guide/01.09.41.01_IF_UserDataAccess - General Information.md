## Task

Interface that must be implemented in a user-defined function block to connect this function block to the carrier.

## Description

When the interface IF\_UserDataAccess is implemented in a user-defined function block, you can use the method SetIfUserDataAccess to connect this function block to the carrier.

For more information, refer to the method [SetIfUserDataAccess](SetUserData-E49E382C.html#SetUserData-E49E382C).
