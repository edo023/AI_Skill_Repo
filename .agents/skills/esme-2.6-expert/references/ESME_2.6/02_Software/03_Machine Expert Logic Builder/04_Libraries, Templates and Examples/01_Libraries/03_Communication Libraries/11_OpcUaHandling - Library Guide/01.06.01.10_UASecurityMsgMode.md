# UASecurityMsgMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration UASecurityMsgMode specifies the message security mode applied to the connection to the OPC UA server.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| UASMM\_BestAvailable | 0 | Highest available message security mode to the OPC UA server. The client receives the available message security modes from the server and selects the highest. Note that this could also result in level UASMM\_None. |
| UASMM\_None | 1 | No security is applied to messages. |
| UASMM\_Sign | 2 | Messages are signed but not encrypted. |
| UASMM\_SignEncrypt | 3 | Messages are signed and encrypted. |
