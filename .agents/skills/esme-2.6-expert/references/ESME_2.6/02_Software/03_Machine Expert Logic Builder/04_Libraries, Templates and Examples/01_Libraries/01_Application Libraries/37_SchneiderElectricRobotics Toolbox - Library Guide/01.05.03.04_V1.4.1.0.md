## New Features

* **Entities Handler**

  FB\_EntitiesHandler

  A function block used to store a global description of the layout of a robotic cell. Such description includes robots, conveyor belts, tracking systems and types of target.
* **Targets Handler**

  FB\_TargetsHandler

  A function block used to represent a flow of targets (for example, a flow of products). The function block handles a list of targets and, through methods, it is possible to add, remove, or interact with them.
* **Balancing Strategies**

  FB\_WeightedBalancing, FB\_ListBalancing, FB\_ConveyorWidthBalancing

  Implementation of algorithms for the balancing of the workload among a group of robots.
* **Target Selection Strategies**

  FB\_FIFOInsideArea, FB\_LIFOInsideArea, FB\_NearestTargetInsideArea, FB\_FarthestTargetInsideArea

  Implementation of algorithms for the selection of a valid target for a single robot starting from a list of possible targets.
