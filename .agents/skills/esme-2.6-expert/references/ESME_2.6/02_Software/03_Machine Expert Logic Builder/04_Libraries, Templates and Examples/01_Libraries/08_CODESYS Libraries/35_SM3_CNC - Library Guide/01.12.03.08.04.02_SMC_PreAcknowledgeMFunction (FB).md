# SMC\_PreAcknowledgeMFunction (FB)

FUNCTION\_BLOCK SMC\_PreAcknowledgeMFunction

This function block serves to acknowledge an M-function even before the interpolator reaches it. If an M-function is acknowledged in advance, a stop at the M-function is avoided. The function block should be executed in the same task as the interpolator. It operates on the queue that is fed into [SMC\_CheckVelocities](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-checkvelocit.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_checkvelocit_smc_checkvelocities_fb) .

**Note**

To pre-acknowledge a G75 command, set input iM to -2.
