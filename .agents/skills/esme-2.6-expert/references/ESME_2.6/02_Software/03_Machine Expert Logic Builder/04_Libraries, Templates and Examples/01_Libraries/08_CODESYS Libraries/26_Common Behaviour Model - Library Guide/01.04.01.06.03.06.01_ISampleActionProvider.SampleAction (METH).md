# ISampleActionProvider.SampleAction (METH)

METHOD SampleAction

After leaving the state [STATE.DORMANT](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_dormant) the SampleAction is running one time then the state [STATE.STARTING](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_starting) is reached. If there is a iErrorID ≠ 0 (Zero) the state [STATE.CLEANING](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_cleaning) is reached instead.
