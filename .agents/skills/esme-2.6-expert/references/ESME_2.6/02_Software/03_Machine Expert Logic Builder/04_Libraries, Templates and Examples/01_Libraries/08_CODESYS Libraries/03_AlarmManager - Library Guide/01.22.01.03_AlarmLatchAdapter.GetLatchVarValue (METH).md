# AlarmLatchAdapter.GetLatchVarValue (METH)

METHOD GetLatchVarValue : AlarmManager\_Interfaces.ITypedValue

Returns a ITypedValue instance, describing the current value of an optional latch variable. This method is an enhancement/a replacement of methods IAlarm.GetLatchVar1Value and IAlarm.GetLatchVar2Value.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | GetLatchVarValue | AlarmManager\_Interfaces.ITypedValue |  | | Input | iWhichLatchVar | INT | The 1-based index of the latch variable. Specify 1 to obtain the same result like method IAlarm.GetLatchVar1Value and 2 to obtain the same result like method IAlarm.GetLatchVar2Value. All greater values correspond to the optional additional latch variables. | |
