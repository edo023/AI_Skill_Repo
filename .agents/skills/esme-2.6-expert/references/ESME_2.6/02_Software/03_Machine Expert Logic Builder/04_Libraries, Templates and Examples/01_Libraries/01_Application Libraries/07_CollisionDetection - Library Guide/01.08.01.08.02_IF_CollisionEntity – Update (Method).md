## Task

This method is used to update the internal representation of the collision entity.

## Description

On a call of this method, there are two possible behaviors that depend from the value of the input i\_xUpdateGroups.

* If on a call of the Update method i\_xUpdateGroups = FALSE, all the groups added to the entity must already be successfully updated (xUpdated = TRUE for each group).
* If on a call of the Update method i\_xUpdateGroups = TRUE, the Update method of each group added to the entity is called and must be successful; otherwise, the Update call generates an error.
