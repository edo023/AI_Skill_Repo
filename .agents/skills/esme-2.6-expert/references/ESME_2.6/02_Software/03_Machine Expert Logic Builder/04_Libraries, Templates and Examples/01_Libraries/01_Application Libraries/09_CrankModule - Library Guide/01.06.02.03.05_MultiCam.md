# MultiCam

MultiCam

The operation mode MultiCam is selected. The axis is attached as slave to a master via cams.

The operation mode is parameterized in the structures [AXM.ST\_MultiCam](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077229_1) and [AXM.ST\_Main](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077223_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [PDL.FB\_MultiCam](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087313_1) are being activated.

If the axis shall be activated in an ILS network, the pstIlsData parameter of the ST\_MultiCam structure must be linked to the (global) stIlsData structure, to which also the pstIlsData parameter of the ST\_EndlessFeed structure of the master module was linked. Subsequently, the technology POU [PDL.FB\_PreCalcMultiCam](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087330_1) is called up first.

Basic monitoring systems are enabled:

oMonitoring the diagnostic messages of the axis

oMonitoring the hardware limit switches

oMonitoring the reduced velocity

Possible state commands

o[Start](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> If the operation mode was entered via the [CamCs](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) command, a coldstart is executed.

 If the operation mode was entered via the [CamWs](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) command, a warmstart is preselected. Subsequently, a WsStart command is waited for.

o[WsStart](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> After the warmstart has been prepared via the commands [CamWs](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) - [Start](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1), the warmstart motion is started via [WsStart](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1).

o[Hold](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> The start command is canceled.

o[DriveEnableOn](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> The position control of the axis is activated.

o[DriveEnableOff](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> The position control of the axis is disabled.

Possible operation mode commands

o[Stop](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [NoOp](#XREF_D_SE_0088804_3) takes place.

o[Homing](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [Homing](#XREF_D_SE_0088804_4) takes place.

o[Manual](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [Manual](#XREF_D_SE_0088804_5) takes place.

o[Endless](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [Endless](#XREF_D_SE_0088804_7) takes place.

o[Positioning](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [Positioning](#XREF_D_SE_0088804_8) takes place.

o[BrakeRelease](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [BrakeRelease](#XREF_D_SE_0088804_9) takes place.
