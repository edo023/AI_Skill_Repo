# SDO\_WRITE\_DATA (FB)

FUNCTION\_BLOCK SDO\_WRITE\_DATA EXTENDS CiA405Base

This function block is used for writing a specific object in object dictionary of a DEVICE. If DATA was written without error CONFIRM becomes TRUE. DATA has to be in Little Endian byte order. If an error occurs ERROR becomes unequal 0. In case of a SDO abort ERROR is set to [CANOPEN\_KERNEL\_ERROR.CANOPEN\_KERNEL\_OTHER\_ERROR](WkZ9when7NogvVdipw-bWj0hVTA_canopen-kernel-e.html#wkz9when7nogvvdipw_bwj0hvta_canopen_kernel_e_canopen_kernel_error_enum__wkz9when7nogvvdipw_bwj0hvta_canopen_kernel_e_canopen_kernel_error_canopen_kernel_other_error) and ERRORINFO contains the associated abort code (in Little endian) as defined by CiA 301.
