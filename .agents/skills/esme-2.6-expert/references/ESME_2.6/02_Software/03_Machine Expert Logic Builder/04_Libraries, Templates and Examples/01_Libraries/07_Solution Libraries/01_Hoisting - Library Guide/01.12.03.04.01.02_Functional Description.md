# Functional Description

Functional Description

This function block uses information about actual load. The actual load value is measured using a load cell. Alternative load measurement methods are possible. However, lower accuracy of actual load input may negatively influence the calculated values.

The function block calculates usage for days on which the hoist was used. Days on which the crane was either not powered or powered on, but without any movement of the hoist does not influence the calculated mechanical class.

Therefore, the average usage in the last 7 and 30 days calculated by the function block is not in calendar days, but in working days.

The function block keeps track of the real time and can handle situations when the control system of the crane is kept powered constantly, gets powered down overnight, gets powered down for prolonged periods of time exceeding one day or is powered down and up repeatedly within one day.
