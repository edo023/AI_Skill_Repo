# IExitActionProvider (ITF)

INTERFACE IExitActionProvider EXTENDS \_\_SYSTEM.IQueryInterface

Optinal element of an [IActionProvider](D4ipCmCdfWAt64OLkZLrmdRGPIg_iactionprovider1.html#d4ipcmcdfwat64olkzlrmdrgpig_iactionprovider1_iactionprovider_itf) for the handling of external resources. The StateMachine of the BahaviourModel will call its ExitAction once while switching from [STATE.CLEANING](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_cleaning) to [STATE.DONE](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_done), [STATE.ERROR](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_error) or [STATE.ABORTED](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_aborted). The implementation should provide the code conditioned for the additional call in the context of FB\_EXIT.
