# Enumeration Elements

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| SetSlaveAndMasterPosToCamStart | 0 | The positions of slave and master are set to the start point of the cam. |
| NoPosChangeStartCamWithActPos | 1 | No position offset. The cam is started using the positions of slave and master. |
| AutoConnectedToPreviousCam | 2 | The start of the cam is set to the end of the preceding cam. |
| MoveSlaveAxisToCamWhileMasterStandstill | 3 | The slave axis is moved to the start position of the cam. During this time, the master must be at standstill. |
