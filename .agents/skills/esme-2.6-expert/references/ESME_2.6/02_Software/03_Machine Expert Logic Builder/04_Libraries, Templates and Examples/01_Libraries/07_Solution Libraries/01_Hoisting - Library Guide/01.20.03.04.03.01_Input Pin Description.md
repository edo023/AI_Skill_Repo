# Input Pin Description

Input Pin Description

| Input | Data Type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block.  Refer to detailed description below this table. |
| i\_iDrvSpdActl | INT | Actual speed input of drive  Range: -32768...32767 RPM |
| i\_wDrvAccLvl1...4 | WORD | Drive acceleration for ramp Level 1...4  Range: 1...9999  Scaling/Unit: 0.1 s |
| i\_wDrvDecLvl1...4 | WORD | Drive deceleration for ramp Level 1...4  Range: 1...9999  Scaling/Unit: 0.1 s |
| i\_wDrvSpdRef1...6 | WORD | Drive pre-select speed 1...6  Range: 0...6000 RPM |
| i\_xRst | BOOL | Resets detected alarms  TRUE: Active  FALSE: Inactive |
