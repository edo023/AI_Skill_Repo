# ST\_Arguments

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.2.4.0 |
| Inherits from: | - |
| Attribute: | ``` 'pack_mode' := '1' ``` |

## Description

The structure ST\_Arguments contains the list of arguments used in a method call.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| uiNumberOfArguments | UINT | The number of arguments in the astArguments array. It must not exceed the value of [GPL.Gc\_uiMaxArgumentsMethod](D-SE-0100002.html#D-SE-0100002__D-SE-0100002.7). |
| astArguments | ARRAY [1..GPL.Gc\_uiMaxArgumentsMethod] OF ST\_Argument | Array containing the arguments. |
