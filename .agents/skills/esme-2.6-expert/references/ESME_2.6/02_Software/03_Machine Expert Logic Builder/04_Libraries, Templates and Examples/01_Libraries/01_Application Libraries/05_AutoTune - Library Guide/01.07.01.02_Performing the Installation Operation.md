# Performing the Installation Operation

Performing the Installation Operation

If the AutoTune library was not installed automatically during your EcoStruxure Machine Expert installation, you must complete the following steps manually:

| Step | Action |
| --- | --- |
| 1 | Open the library repository via Tool > Library Repository.  Resut: The window Library Repository opens. |
| 2 | Click on Install.  Resut: The window Select Library opens. |
| 3 | Select the directory in which the AutoTune library is stored. |
| 4 | Select the file with the desired AutoTune version. |
| 5 | Click on Open.  Resut: The window Select Library is closed.  Resut: In the window Library Repository, the AutoTune library with the respective version is displayed in the list of installed libraries. |
| 6 | Click on Close.  Resut: The window Library Repository is closed. |
