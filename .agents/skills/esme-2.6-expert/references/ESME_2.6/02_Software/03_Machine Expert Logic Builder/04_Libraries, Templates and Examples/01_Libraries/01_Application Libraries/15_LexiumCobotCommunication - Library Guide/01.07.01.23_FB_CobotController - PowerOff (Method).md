## Task

The method is used to turn off the Lexium Cobot arm.

## Description

This method is used to power off the Lexium Cobot arm.

Preconditions to execute this command:

* The control source of the Lexium Cobot must be Remote. For further information, refer to the method [GetControlSource(Method)](FB_CobotController-GetControlSource-7FECE4FA.html).
* The Lexium Cobot arm must be disabled. FB\_CobotControllerFeedback.rstData.xEnabled reports the value FALSE.

Result of this command:

* The Lexium Cobot arm is powered off. FB\_CobotControllerFeedback.rstData.xPoweredOn reports the value TRUE.
* The indicator light of the Lexium Cobot arm turns off.

The sent command is acknowledged by the Lexium Cobot controller either as successful or as unsuccessful.

* A successful execution of the command is reported by the property xCommandAcknowledged.
* An unsuccessful execution of the command is reported by the property xCommandFailed. For detailed information verify the diagnostic outputs of the function block and the log data of FB\_CobotControllerFeedback.rstData.stLogData.
