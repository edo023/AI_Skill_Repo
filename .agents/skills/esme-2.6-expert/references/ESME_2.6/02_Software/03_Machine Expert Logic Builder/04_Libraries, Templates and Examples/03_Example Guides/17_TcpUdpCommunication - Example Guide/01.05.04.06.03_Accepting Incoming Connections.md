# Accepting Incoming Connections

Accepting Incoming Connections

After the [port](../glossary/glossary.htm#XREF_D_SE_0055638_16) has been opened, it is monitored by the [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) [server](../glossary/glossary.htm#XREF_D_SE_0055638_18) for incoming connections and receiving data from connected clients.

Each incoming [client](../glossary/glossary.htm#XREF_D_SE_0055638_5) connection is accepted. The number of connected clients is shown in the visualization screen VisuTcpServer.

Whether a connection to the TCP client is encrypted using TLS (Transport Layer Security) depends on the controller used.
