# IF\_ConnectionManager.CloseAllInactiveConnections (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

The method CloseAllInactiveConnections closes the inactive connections that are held by the global Modbus TCP connection manager.

## Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| CloseAllInactiveConnections | BOOL | The return value indicates TRUE if the method was executed successfully. |
