# PackBitsToWord (FUN)

FUNCTION PackBitsToWord : WORD

Function packs 16 Bits “xBit0” .. “xBit15” into a WORD.

Example:

```
PackBitsToWord(FALSE,FALSE,FALSE,FALSE,FALSE,FALSE,FALSE,TRUE,
FALSE,FALSE,FALSE,FALSE,FALSE,FALSE,TRUE,FALSE) = 16#0102
```
