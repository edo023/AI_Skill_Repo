# ST\_SetIpRequest - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.2.0 |
| Inherits from: | - |

## Description

Structure for setting the IP configuration of a Sercos device with [FC\_SetIpRequest](FC_SetIpRequest-GeneralInformation-091CB01E.html).

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| sNodeIdentifier | STRING[17] | Node identifier |
| sMacAddress | STRING[17] | MAC address of the interface from which the response has been sent |
| byDHCPMode | BYTE | Values:  * 0: Disable DHCP * 1: Enable DHCP client * 2: Enable DHCP server * 3: Enable DHCP relay client |
| sIpAddress | STRING[15] | IP address of Sercos device |
| sSubnet | STRING[15] | Subnet mask of Sercos device |
| sGateway | STRING[15] | Gateway of Sercos device |
| bPersistent | BOOL | TRUE: Store settings persistently |
| sHostName | STRING[80] | Name to identify the node (see parameter S-0-1039) |
