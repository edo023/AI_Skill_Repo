# ET\_ConnectionState

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

This enumeration type is used to address a specific connection state of the TCP connection to a Lexium Cobot controller.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Idle | 0 | The connection state Idle is addressed.  No connection to a Lexium Cobot controller is established. |
| Connecting | 1 | The connection state Connecting is addressed.  A connection to a Lexium Cobot controller is being established. |
| Connected | 2 | The connection state Connected is addressed.  A connection to a Lexium Cobot controller is established. |
| Disconnecting | 3 | The connection state Disconnecting is addressed.  An established connection to a Lexium Cobot controller is being disconnected. |
| Shutdown | 4 | The connection state Shutdown is addressed.  An established connection to a Lexium Cobot controller was shutdown. |
