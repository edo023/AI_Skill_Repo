# SMC\_GCODE\_TEXT (STRUCT)

TYPE SMC\_GCODE\_TEXT : STRUCT

This data type is used to transmit the string of the last interpreted line of the CNC file from the function block [SMC\_NCInterpreter](VVFvsggjXKxZ6qmLw10MvZPswnU_smc-ncinterprete.html#vvfvsggjxkxz6qmlw10mvzpswnu_smc_ncinterprete_smc_ncinterpreter_fb) to the function block [SMC\_GCodeViewer](5udZX4_f3oFG5rPy6LFF4FCbGCU_smc-gcodeviewer.html#5udzx4_f3ofg5rpy6lff4fcbgcu_smc_gcodeviewer_smc_gcodeviewer_fb).

| InOut: | | Name | Type | Comment | | --- | --- | --- | | str | STRING(80) | String representing the current line | | iLineNumber | DINT | Line number (from N word) | | bNewLine | BOOL | TRUE when a new line has been decoded in this cycle; FALSE else | | bClearList | BOOL | TRUE when the interpreter has been (re-) started. This implies that a buffer that may store previous lines should be emptied. | | objectId | [SMC\_GCodeObjectId](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-gcodeobjecti.html#cah0ife6nckjzp5ef3qpy_filyk_smc_gcodeobjecti_smc_gcodeobjectid_struct) | The object id of the object created by the given line. | |
