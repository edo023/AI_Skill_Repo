# FB\_DatFileToCam - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | 1.2.0.0 |
| Inherits from: | – |

## Description

The function block FB\_DatFileToCam reads the file providing CAM data that is generated by MotionSizer.

The reading is processed in an asynchronous task. Once the asynchronous task is initialized and started, the following steps are performed:

1. Open the file.
2. Parse the file content and store the data in the structure assigned to iq\_stMultiCamData.
3. Close the file.

NOTE: In the event of an error detected during the file reading, the process is aborted and the file is closed. Any subsequent errors, therefore, do not affect the outputs.

## Troubleshooting

This table describes the possible issues and their solutions:

| Issue  (error information (q\_xError = True) | Cause | Solution |
| --- | --- | --- |
| q\_etResult = FileNotFound | The specified file was not found. | Verify whether the file exists on the controller.  Verify the file path defined on the input i\_sFilename. |
| q\_etResult = FileOpenFailed | An internal error was detected while opening the file. | Execute the function block again. If the issue persists, contact your local Schneider Electric service representative. |
| q\_etResult = FileCloseFailed | An internal error was detected while closing the file. | Execute the function block again. If the issue persists, contact your local Schneider Electric service representative. |
| q\_etResult = ReadFileFailed | An internal error was detected while reading the file. | Execute the function block again. If the issue persists, contact your local Schneider Electric service representative. |
| q\_etResult = InvalidCamData | Plausibility verification for read data unsuccessful. The expected character for line termination could not be found. | Verify whether the file is in the correct format. Only files created by MotionSizer are supported. |
| q\_etResult = TimeoutReadFile | Timeout: The end of line was not found in the file. | Verify whether the file has the correct format and has not been changed after export from MotionSizer. |
| q\_etResult = AsyncManagerInitFailed | An internal error was detected during initialization of the AsyncManager. | Execute the function block again. If the issue persists, contact your local Schneider Electric service representative. |
| q\_etResult = AsyncManagerStartJobFailed | An internal error was detected during start of the asynchronous job. | Execute the function block again. If the issue persists, contact your local Schneider Electric service representative. |
