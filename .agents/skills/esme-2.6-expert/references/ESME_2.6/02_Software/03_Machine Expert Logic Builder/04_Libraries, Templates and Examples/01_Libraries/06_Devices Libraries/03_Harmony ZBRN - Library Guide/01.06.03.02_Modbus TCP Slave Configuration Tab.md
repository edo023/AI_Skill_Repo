# Modbus TCP Slave Configuration Tab

Modbus TCP Slave Configuration Tab

| Step | Action |
| --- | --- |
| 1 | In the Devices tree, double-click Generic\_Modbus\_TCP\_Slave.  Result: The configuration window is displayed.  G-SE-0031832.1.gif-high.gif |
| 2 | Enter a Slave IP Address value (by default 192.168.0.1). |
| 3 | Enter a [Health Timeout](../glossary/glossary.htm#XREF_D_SE_0024697_782) (ms) value (by default 1000).  This represents the delay (in ms) between a request of the Modbus TCP IOScanner and a response from the slave. When the health timeout expires, the associated [health bit](../glossary/glossary.htm#XREF_D_SE_0024697_781) values change to 0. Health bit values can be visualized in the [IOScanner I/O Mapping tab](MB_TCP_Conf-2.htm#XREF_D_SE_0036289_4). |
