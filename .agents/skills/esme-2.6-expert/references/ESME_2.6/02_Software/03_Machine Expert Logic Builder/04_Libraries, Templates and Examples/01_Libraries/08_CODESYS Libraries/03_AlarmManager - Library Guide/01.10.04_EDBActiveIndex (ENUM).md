# EDBActiveIndex (ENUM)

TYPE EDBActiveIndex :

This enumeration defines the db active index DO NOT change the sequence, it’s used in Test Manager as number

| Attributes: | qualified\_only |
| InOut: | | Name | Initial | Comment | | --- | --- | --- | | NOT\_DEFINED | 0 |  | | TIMESTAMP |  | Index “timestamp last, descending” | | GROUP |  | Index “group id ascending, timestamp last, descending” | | EDB\_CLASS |  | Index “class id ascending, timestamp last, descending” (CLASS cannot be used as enumvalue, as it is a IEC-keyword. Therefore changed to EDB\_CLASS ) | | LATCH1 |  | Index “Latch1, timestamp last descending” | |
