# ST\_HomeCmdState

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.3.1.0 |
| Inherits from: | — |

## Description

The structure ST\_HomeCmdState indicates the state of a homing command. The structure is used by FB\_Home.

## Structure Elements

| Name | Data Type | Description |
| --- | --- | --- |
| etCmdType | ET\_HomingCmd | Indicates the type of homing command. |
| xBusy | BOOL | Indicates whether the command is ongoing. |
| xDone | BOOL | Indicates whether the command has been successfully completed. |
| xAborted | BOOL | Indicates whether the command has been aborted. |
| xError | BOOL | Indicates whether the command has been stopped due a detected error. |
| etResult | ET\_Result | Provides diagnostic and status information as a numeric value.   * If xError = FALSE, etResult provides status information. * If xError = TRUE, etResult provides diagnostic or error information. |
| sResultMsg | STRING(255) | Provides the additional diagnostic and status information as a text message. |
