## Task

Configures a teaching procedure.

## Description

The method Configuration(...) is used to provide a basic configuration of a teaching procedure.

NOTE: By calling this method, all the stored samples are removed. Each procedure may require additional configuration data that are set with the method SetProcedureData. i\_ifRobotFeedback and i\_ifRobotConfigurationReadOnly must refer to the same robot.

Access: PUBLIC

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stReferenceSystemBasePose | [ST\_CartesianPose](D-SE-0080396.html) | The cartesian pose of the system used as reference for the teaching. This pose refers either to the robot or to the other system involved in the teaching, as defined by the input i\_xIsRobotReferenceSystem. |
| i\_xIsRobotReferenceSystem | BOOL | A procedure is always performed between a robot and another system.  TRUE: The input i\_stReferenceSystemBasePose is the global pose of the robot and the pose estimated by the procedure is the one of the other system.
