# BUS\_STATE (ENUM)

TYPE BUS\_STATE :

This enumeration lists all possible states of a fieldbus. The mapping to the according fieldbus specific states is defined by the associated fieldbus stack.

| Attributes: | qualified\_only |
| InOut: | | Name | Comment | | --- | --- | | UNKNOWN | The bus state is unknown. | | STOPPED | The fieldbus is stopped. No I/O transfer can be processed. | | RUNNING | The fieldbus is running. | | ERROR | The fieldbus is in error state. [GetBusError](_i3Rclu0CxGIST3QKKbewmL3Y4o_getbuserror.html#_i3rclu0cxgist3qkkbewml3y4o_getbuserror_getbuserror_fb) supplies the specific error code. | | DISABLED | Bus is disabled in device tree. | |
