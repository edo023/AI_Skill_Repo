# Structure: SyncGroupIn

Structure: SyncGroupIn

Structure of parameters related to the synchronization of groups

| Parameter | Data type | Description |
| --- | --- | --- |
| xSync | BOOL | Synchronization command  TRUE: All axes fulfilling the conditions to enter synchronization are added to the synchronization group.  FALSE: No axes are added to the synchronization group. |
| xDrvFwdSync | BOOL | Forward direction command for the synchronization group  TRUE: Enables the forward direction command for all axes within the synchronization group.  FALSE: Disables the forward direction command for all axes within the synchronization group. |
| xDrvRevSync | BOOL | Reverse direction command for the synchronization group  TRUE: Enables the reverse direction command for all axes within the synchronization group.  FALSE: Disables the reverse direction command for all axes within the synchronization group. |
| wDrvSpdRefSync | WORD | Speed reference for the synchronization group  Range: 0...6000  Unit: RPM |
| rKpPosSync | REAL | Proportional gain of position controller  Range: 0...3.4 38 |
| rMaxPosDevSync | REAL | Maximum position deviation  Range: 0...3.4 38  Unit: m or in. |
| wDrvSpdNomSync | WORD | Nominal speed corresponding to the synchronization group  Range: 1...6000  Unit: RPM |
| wDrvAccSync | WORD | Acceleration time in synchronized mode  Range: 1...999  Unit: 0.1 s |
| wDrvDecSync | WORD | Deceleration time in synchronized mode  Range: 1...999  Unit: 0.1 s |
| xFlexDir | BOOL | Flexible direction of movement  TRUE: Allows axes synchronized within given synchronization group to move towards their respective preset offset positions in an arbitrary direction as long as any direction command is present.  FALSE: Allows axes to move towards their respective preset offset positions only in the direction specified by the direction command. |
