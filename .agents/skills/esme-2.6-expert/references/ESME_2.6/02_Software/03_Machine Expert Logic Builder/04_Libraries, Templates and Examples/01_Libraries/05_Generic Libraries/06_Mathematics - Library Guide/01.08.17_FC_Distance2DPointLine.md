# FC\_Distance2DPointLine

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the distance of a point i\_stPoint from a straight line i\_stLine in a two-dimensional space. The sign indicates in which half-plane defined by the straight line i\_stLine the point i\_stPoint is located. In addition, the perpendicular of the point to the straight line is calculated.

| --- | --- | --- |
| i\_stPoint | [ST\_Vector2D](ST_Vector2D-GeneralInformation-0BFF6B0C.html#ST_Vector2D-GeneralInformation-0BFF6B0C) | Point |
| i\_stLine | [ST\_Line2D](ST_Line2D-GeneralInformation-0C04692F.html#ST_Line2D-GeneralInformation-0C04692F) | Straight line |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |

| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
| q\_lrDistance | LREAL | Calculated distance of the point i\_stPoint from the straight line i\_stLine |

| Enumeration value: | 31 |
