# AddSubElement (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.3.2.0 |

![](images/T003225655.png)

## Functional Description

This method is used to add a new sub-element under the presently selected element. After the method is executed successfully, the new element is selected for further operations.

The return value of type BOOL indicates TRUE if a new element was added successfully.

A call of this method returns either Ok, InvalidInput, NoElementSelected, MaxNumOfParentsExceeded, or BufferFull. Use the property Result to obtain the result of the method.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sElementName | STRING [Gc\_uiXmlLengthOfString] | Name of the sub-element to be added. |

## Example
