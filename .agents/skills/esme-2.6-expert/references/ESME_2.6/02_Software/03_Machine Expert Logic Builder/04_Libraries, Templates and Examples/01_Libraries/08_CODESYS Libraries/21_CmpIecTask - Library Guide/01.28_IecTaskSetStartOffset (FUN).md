# IecTaskSetStartOffset (FUN)

FUNCTION IecTaskSetStartOffset : RTS\_IEC\_RESULT

Set the start offset of a task NOTES: - Can only be called out of FB\_Init sequence! If you call this function during runtime of the application, error code ERR\_OPERATION\_DENIED is returned. - Only supported on Linux targets yet! On all other targets, setting this offset has no effect!

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | IecTaskSetStartOffset | RTS\_IEC\_RESULT | Error code | | Input | sIecTaskName | STRING | IEC task name | | startOffsetUs | DINT | Start offset in microseconds from calculated task start | |
