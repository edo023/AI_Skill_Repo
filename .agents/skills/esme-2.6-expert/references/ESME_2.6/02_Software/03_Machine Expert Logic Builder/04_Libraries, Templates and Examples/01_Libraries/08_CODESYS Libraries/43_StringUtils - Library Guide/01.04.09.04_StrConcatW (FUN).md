# StrConcatW (FUN)

FUNCTION StrConcatW : BOOL

This function concatenates |pstFrom| to |pstTo| if the |iBufferSize| from |pstTo| is big enough. The function will do its work either with native IEC-Strings or with strings coming from external functions.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | StrConcatW | BOOL | It will return FALSE and does no concatenation if one of the strings is NULL or the buffer size is not big enough. TRUE otherwise. | | Input | pstFrom | [CharBufferPtr](hfzJmOS2ddAPvg6yR-C9-xk02nE_charbufferptr.html#hfzjmos2ddapvg6yr_c9_xk02ne_charbufferptr_charbufferptr_alias) | Pointer to the string that shall be added to |pstTo|. | | pstTo | [CharBufferPtr](hfzJmOS2ddAPvg6yR-C9-xk02nE_charbufferptr.html#hfzjmos2ddapvg6yr_c9_xk02ne_charbufferptr_charbufferptr_alias) | Pointer to the string where |pstFrom| shall be added to. | | iBufferSize | INT | The size of |pstTo| in BYTE. | |
