# FC\_Circle3DThroughThreePoints

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates a circle in the three-dimensional space that intersects three points. The three points which must not be located on a straight line are specified as input. This function calculates the center, the radius, and the normal vector of the plane in which the circle lies which intersects these three points.

| Input | Data type | Description |

| i\_stPoint2 | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | Point 2 which the circle is to intersect |
| i\_stPoint3 | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | Point 3 which the circle is to intersect |
