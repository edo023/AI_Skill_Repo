# DriveInvalid

DriveInvalid

|  |  |
| --- | --- |
| Enumeration name: | DriveInvalid |
| Enumeration value: | 26 |
| Description: | The connected drive is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The drive transferred at the input Master does not support all required functionalities. | The drive must support the SystemConfigurationItf.IF\_RefActualValuesVelocity interface. |
| - | The drive transferred at the input Slave does not support all required functionalities. | The drive must support the SystemConfigurationItf.IF\_ControllerStop interface. |
