# GetConnectionString (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.7.0 |

## Description

This method is used to obtain the connection string. The return value of type ET\_Result indicates the result of the method execution. If Ok is indicated, the value is successfully copied to the assigned buffer.

## Interface

| Inputs | Data Type | Description |
| --- | --- | --- |
| i\_pbyBuffer | POINTER TO BYTE | Pointer to the buffer to store the value. |
| i\_uiSize | UDINT | Size of the buffer. |

NOTE: To prevent access violation caused by invalid pointer access (out of bounds) to the memory, use the arithmetic operator SIZEOF in conjunction with the targeted buffer to determine the value for i\_uiSize.
