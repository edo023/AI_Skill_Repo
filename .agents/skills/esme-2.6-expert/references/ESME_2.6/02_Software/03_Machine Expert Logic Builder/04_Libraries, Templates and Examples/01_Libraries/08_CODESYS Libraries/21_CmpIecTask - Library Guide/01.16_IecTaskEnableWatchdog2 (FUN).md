# IecTaskEnableWatchdog2 (FUN)

FUNCTION IecTaskEnableWatchdog2 : RTS\_IEC\_RESULT

Enable watchdog for the specified task

**Note**

* You have to disable the watchdog of the task before with [IecTaskDisableWatchdog2](IecTaskDisableWatchdog2.html#iectaskdisablewatchdog2_iectaskdisablewatchdog2_fun) and \_not\_ with [IecTaskDisableWatchdog](IecTaskDisableWatchdog.html#iectaskdisablewatchdog_iectaskdisablewatchdog_fun), because they act on different task flags!
* The watchdog is enabled only at the next IEC cycle and \_not\_ immediately after calling this function! But if you never enable the watchdog after calling [IecTaskDisableWatchdog2](IecTaskDisableWatchdog2.html#iectaskdisablewatchdog2_iectaskdisablewatchdog2_fun), the watchdog is disabled forever!
* As a consequence, you always have to call [IecTaskDisableWatchdog2](IecTaskDisableWatchdog2.html#iectaskdisablewatchdog2_iectaskdisablewatchdog2_fun) and [IecTaskEnableWatchdog2](#iectaskenablewatchdog2_iectaskenablewatchdog2_fun) symmetrical
* Can be called nested. Last call enables the watchdog.
