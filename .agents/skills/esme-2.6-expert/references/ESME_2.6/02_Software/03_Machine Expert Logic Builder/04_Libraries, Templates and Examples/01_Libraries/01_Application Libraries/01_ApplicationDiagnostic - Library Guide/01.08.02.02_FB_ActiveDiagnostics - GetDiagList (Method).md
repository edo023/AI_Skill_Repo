## Task

Provides the list of active diagnostic messages.

## Description

The method GetDiagList  is used to obtain a list of active diagnostic messages form the diagnostic handler where the listener is registered.

## Return Value

| Data type | Description |
| --- | --- |
| UINT | The return value indicates the diagnostic messages written into the array at the in\_out parameter iq\_astDiagMsgs. Depending on the used filters, this number can be lower than the absolute number of active diagnostic messages indicated by the property uiNumOfActiveDiagMsgs. |

## Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_astDiagMsgs | ARRAY[\*] OF ST\_DiagMessageListener | Provides the list of active diagnostic messages. |
