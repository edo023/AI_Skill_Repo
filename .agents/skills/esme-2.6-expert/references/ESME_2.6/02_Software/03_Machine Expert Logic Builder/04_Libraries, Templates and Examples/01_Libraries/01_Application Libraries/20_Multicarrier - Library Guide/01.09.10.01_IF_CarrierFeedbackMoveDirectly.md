## Task

Interface with status values for the move command [MoveDirectly](IF_MoveDirectly-5B82D7F8.html#IF_MoveDirectly-5B82D7F8).

## Description

The feedback interface provides status values for the movement of the carrier with the move command MoveDirectly.
