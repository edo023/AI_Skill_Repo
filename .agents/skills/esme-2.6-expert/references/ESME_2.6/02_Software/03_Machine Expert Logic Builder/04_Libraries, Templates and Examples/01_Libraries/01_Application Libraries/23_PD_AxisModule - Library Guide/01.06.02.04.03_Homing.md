# Homing

Homing

The operating mode Homing is selected. The axis is homed via the mode selected in i\_etMode.

The operating mode is parameterized in the structures [ST\_Home](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) and [ST\_Main](../Structures/Structures-7.htm#XREF_D_SE_0077223_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and PDL.FB\_Homing are being activated.

Basic monitoring systems are enabled:

oMonitoring the diagnostic messages of the axis

oMonitoring of the hardware limit switches

oMonitoring of the reduced velocity

Possible state commands

o[Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Homing of the axis is started.

o[Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Homing of the axis is stopped.

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The position control of the axis is activated.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The position control of the axis is being disabled.

Possible operating mode commands

o[Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [NoOp](#XREF_D_SE_0077326_3) takes place.

o[CamCs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [MultiCam](#XREF_D_SE_0077326_6) takes place and a cold start is preselected.

o[CamWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [MultiCam](#XREF_D_SE_0077326_6) takes place and a warm start is preselected.

o[Manual](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Manual](#XREF_D_SE_0077326_5) takes place.

o[Endless](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Endless](#XREF_D_SE_0077326_7) takes place.

o[EndlessIls](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [EndlessIls](#XREF_D_SE_0077326_8) takes place.

o[Positioning](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Positioning](#XREF_D_SE_0077326_9) takes place.

o[BrakeRelease](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [BrakeRelease](#XREF_D_SE_0077326_10) takes place.

o[AdditionalCs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Additional](#XREF_D_SE_0077326_11) takes place and a cold start is preselected.

o[AdditionalWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Additional](#XREF_D_SE_0077326_11) takes place and a warm start is preselected.
