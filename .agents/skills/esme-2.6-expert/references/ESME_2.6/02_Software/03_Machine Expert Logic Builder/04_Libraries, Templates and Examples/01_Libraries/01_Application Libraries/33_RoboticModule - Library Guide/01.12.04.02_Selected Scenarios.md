# Selected Scenarios

## Reaction to Software and Hardware Limit Switches

Software and hardware limit switches report by default an exception with a reaction SyncStopEL. The robot performs a synchronous stop with emergency parameters and the tracking also stops with its emergency parameters.

## Reaction when SchneiderElectricRobotics Detects an Exceedance

The SchneiderElectricRobotics library verifies whether the estimated stop position of the robot is within the working envelope. If the position exceeds the working envelope, an active tracking is stopped immediately and a stop for the path is issued. Both stops are dependent on the value of the parameter [IF\_ConfigurationAdvanced.xUseEStopParameterForEstimatedStopPosition](D-SE-0076933.html):

If the estimated stop position is calculated with motion parameters (xUseEStopParameterForEstimatedStopPosition = FALSE), then the regular motion parameters are also used for the stop.

If xUseEStopParameterForEstimatedStopPosition is TRUE and the estimated stop position is calculated with emergency parameters, then the emergency parameters are used for the stop.

For more information, refer to:

* [rstEstimatedStopPosition (Part 1)](https://product-help.schneider-electric.com/Machine%20Expert/V2.1/en/PD.Lib.Robotic/index.htm#t=PD.Lib.Robotic%2FD-SE-0075806.html)
* [rstEstimatedStopPosition (Part 2)](https://product-help.schneider-electric.com/Machine%20Expert/V2.1/en/PD.Lib.Robotic/index.htm#t=PD.Lib.Robotic%2FD-SE-0102139.html) with xUseEstopParameters = FALSE
* [rstEstimatedStopPosition (Part 3)](https://product-help.schneider-electric.com/Machine%20Expert/V2.1/en/PD.Lib.Robotic/index.htm#t=PD.Lib.Robotic%2FD-SE-0102140.html) with xUseEstopParameters = TRUE

## ExternalPositionSource

When the robot moves with an external position source, the selection of the parameters used for the stop is the same as during regular operation. The major difference is that the stop parameters are applied to the individual cartesian movements. This implies that the total delay can be a factor of up to 1.75 higher than the configured value, depending on which component is moving and whether the maximum delay is reached during the stop.

For more information, refer to [General Behaviour of ExternalPositionSource](https://product-help.schneider-electric.com/Machine%20Expert/V2.1/en/PD.Lib.Robotic/index.htm#t=PD.Lib.Robotic%2FD-SE-0096791.html) in the Robotic Library Guide.
