# SMC\_ArticulatedRobot\_6DOF\_J4\_State (ENUM)

TYPE SMC\_ArticulatedRobot\_6DOF\_J4\_State :

The possible configuration states of joint 4 of the 6DOF robot arm.

| InOut: | | Name | Comment | | --- | --- | | SMC\_AR\_HANDDOWN | The hand is pointing “down”, i.e. q4 > 0 | | SMC\_AR\_HANDUP | The hand is pointing “up”, i.e. q4 < 0 | | SMC\_AR\_HANDSTRAIGHT | The hand is straight (q4 ~ 0), singular case | |
