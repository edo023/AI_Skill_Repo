# CanReconfigure (FUN)

FUNCTION CanReconfigure : BOOL

The function returns TRUE if itfNode can perform a Reconfigure-Action. Only nodes implementing [IReconfigureProvider](qG5KZxShRzJP0Witj1Yz7sQso84_ireconfigureprov.html#qg5kzxshrzjp0witj1yz7sqso84_ireconfigureprov_ireconfigureprovider_itf) can be used as input of [Reconfigure](HUr-HPtekCI-al68iX7szHFsAss_reconfigure1.html#hur_hptekci_al68ix7szhfsass_reconfigure1_reconfigure_fb)
