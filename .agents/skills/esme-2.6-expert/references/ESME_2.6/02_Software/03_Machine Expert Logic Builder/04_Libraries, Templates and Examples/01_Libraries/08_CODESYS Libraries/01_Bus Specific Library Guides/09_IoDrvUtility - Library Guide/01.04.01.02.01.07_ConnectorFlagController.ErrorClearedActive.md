# ConnectorFlagController.ErrorClearedActive (PROP)

PROPERTY ErrorClearedActive : BOOL

Sets or resets the Error cleared bit. The bit could be used to show the user that an error, warning or diagnostic message was automatically reset and the problem does not exist anymore. If [ConnectorFlagController.AutoSetErrorClearedFlag](EpDUA0ToSNz8wQXwG7EAxhZ0DJo_autoseterrorclea1.html#epdua0tosnz8wqxwg7eaxhz0djo_autoseterrorclea1_connectorflagcontroller_autoseterrorclearedflag_prop) is TRUE, this flag will be set automatically by the ConnectorFlagController. In this case you only have to implement the reset functionality.
