# SMC\_TRAFO\_Tripod (FB)

FUNCTION\_BLOCK SMC\_TRAFO\_Tripod

The transformation module is provided to execute backward transformation of a tripod.

**Note**

For detailed information on transformation modules see the overview on kinematic transformations in the CNC online help.
