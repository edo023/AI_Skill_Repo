# IF\_ControlStandardStation - Restart (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

![](images/0000760267.png)

## Task

Restarting the process of the station.

## Description

With the method Restart, you can restart the process.

The method Restart must be called after a call of the methods [DriveProcessEmpty](DriveProcEmpty-EED35245.html#DriveProcEmpty-EED35245), [HoldProcess](HoldProcess-EED4C135.html#HoldProcess-EED4C135) or [StopProcess](StopProcess-EED58DEE.html#StopProcess-EED58DEE).

## Inputs

The method has no inputs.

## Outputs

The method has no outputs.
