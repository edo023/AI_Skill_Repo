# FC\_VectorGetElement

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.12.0 |

## Task

The function reads a value from a specified position of a one dimensional LREAL array (vector).

## Description

The function validates the provided row index against the runtime bounds of the vector. If the index is out of range, it reports an error and returns an InputValueInvalid result. If the index is valid, the function returns the value of the specified vector element.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_uiRow | UINT | Row index of the element to be read. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_alrVector | ARRAY [\*] OF LREAL | Array into which the element must be read. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |

## Return Value

| Data type | Description |
| --- | --- |
| LREAL | When successful, the function returns the requested element value; otherwise, it returns NaN. |

## Troubleshooting

| Issue | Cause | Solution |
| --- | --- | --- |
| Execution ends with a detected error and result indicates InputValueInvalid. | The given value for i\_uiRow is out of range. | Verify that the value of i\_uiRow lies within the bounds of the assigned array. |
