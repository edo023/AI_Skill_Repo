# IoDrvCIFXProfibus.Sync (METH)

METHOD Sync : UDINT

Method Sync is part of the interface ICmpIoDrvProfibus. By use of a SYNC telegram a master causes a slave or a group of slaves to write the actual output data and to freeze the output afterwards. CARE: Use either udiRemAdr, or udiGroupSelect. Do not assign values <> 0 to both inputs at the same time.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | Sync | UDINT |  | | Input | udiRemAdr | UDINT | Remote address OF slave (==127, IF all slaves are concerned). 0 if one ore more groups shall be selected. | | udiGroupSelect | UDINT | Number of group. This is a bit-field. Example: Use 1 for first Group, 4 for third Group, or 5 to address group 1 and 3. 0 if no group should be selected. | |
