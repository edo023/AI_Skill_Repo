# AnalogInputInvalid

AnalogInputInvalid

|  |  |
| --- | --- |
| Enumeration name: | AnalogInputInvalid |
| Enumeration value: | 39 |
| Description: | AnalogInput is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The feedback source ET\_FeedbackSignalOrigin.AnalogInput is choosen and at the input iq\_stParameter.stFeedbackSignal.i\_ifAnalogInput an invalid value is given. | At the input iq\_stParameter.stFeedbackSignal.i\_ifAnalogInput the analog input of the controller has to be connected.  Another feedback source has to be chosen. |
