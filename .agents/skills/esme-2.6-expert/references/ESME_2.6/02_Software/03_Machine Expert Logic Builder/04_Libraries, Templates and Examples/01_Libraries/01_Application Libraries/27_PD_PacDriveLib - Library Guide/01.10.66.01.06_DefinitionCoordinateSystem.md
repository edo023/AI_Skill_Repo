# DefinitionCoordinateSystem

DefinitionCoordinateSystem

|  |  |
| --- | --- |
| Enumeration name: | DefinitionCoordinateSystem |
| Enumeration value: | 95 |
| Description: | The definition of the coordinate system is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The vectors i\_stV1 and i\_stV2 are parallel, they do not span a 2-dimensional coordinate system. | The vectors i\_stV1 and i\_stV2 must be linearly independent, i.e. they must not be parallel. |
