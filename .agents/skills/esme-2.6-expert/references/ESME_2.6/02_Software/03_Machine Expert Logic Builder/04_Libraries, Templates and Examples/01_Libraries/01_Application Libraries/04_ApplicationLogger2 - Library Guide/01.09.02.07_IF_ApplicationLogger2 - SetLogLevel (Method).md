# IF\_ApplicationLogger2 - SetLogLevel (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method sets the log level of a logger point.

In addition, you can also set whether the log level should also be set for the subordinate logger points.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiLoggerPointId | UDINT | Identification number of the logger point whose logger level is to be set or modified. |
| i\_etLoglevel | [ET\_LogLevel](ET_LogLevel-37599385.html#ET_LogLevel-37599385) | Log level to be set. |
| I\_xRecursively | BOOL | If this input is set to TRUE, the subordinate logger points are also set to the value of i\_etLoglevel. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-375A58EE.html#ET_Result-375A58EE) | Provides diagnostic and status information as a numerical value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
