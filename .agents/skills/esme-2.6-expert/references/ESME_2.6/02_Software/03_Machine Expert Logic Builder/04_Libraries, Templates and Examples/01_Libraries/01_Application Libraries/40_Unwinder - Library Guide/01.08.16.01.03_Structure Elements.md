# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_lrStartThreshold | LREAL | Level the feedback signal has to cross in order to start the precontrol- and control mode of the unwinder. |
| i\_timTimeOut | TIME | The maximum time the bobbin is moved to search for the feedback value i\_lrStartThreshold. |
| i\_lrVelocity | LREAL | The circumferential velocity of the bobbin to search for the feedback value, based on the specified radius. |
| i\_lrVelPGain | LREAL | The proportional gain of the velocity controller. |
| i\_timVelGain | TIME | The integral gain of the velocity controller. |
