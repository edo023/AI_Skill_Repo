# FB\_Statistics

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.1.0.0 |
| Inherits from: | - |
| Implements: | - |

## Description

The function block calculates the minimum, maximum and average values as well as the variance (= square of the standard deviation) of the input values. The value series is extended by one value each time the method Put is called. The value series can be deleted by calling the method Reset.

## Interface

The function block has no interface. Use the function block via the methods and properties.

The function block must not be called in the application.

## Methods

| Name | Description |
| --- | --- |
| [Put](FB_Statistics-PutMethod-0E8DB000.html#FB_Statistics-PutMethod-0E8DB000) | Adds a new value to the value series. |
| [Reset](FB_Statistics-ResetMethod-0E8DE1F7.html#FB_Statistics-ResetMethod-0E8DE1F7) | Deletes value series. |

## Properties

| Name | Data type | Accessing | Description |

| lrMinimum | LREAL | Read | Lowest value of the value series |
