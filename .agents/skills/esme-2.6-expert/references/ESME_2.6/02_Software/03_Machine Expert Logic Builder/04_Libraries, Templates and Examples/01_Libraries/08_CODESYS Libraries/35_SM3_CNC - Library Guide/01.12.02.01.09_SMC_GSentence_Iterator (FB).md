# SMC\_GSentence\_Iterator (FB)

FUNCTION\_BLOCK FINAL SMC\_GSentence\_Iterator

This iterator allows to iterate all sentences of a sentence queue. It can be retrieved from the sentence-queue ([SMC\_GSentenceQueue.GetSentenceIterator](sPgcZ_mXTWZCf5xiluEOzwuwUdY_getsentenceitera.html#spgcz_mxtwzcf5xilueozwuwudy_getsentenceitera_smc_gsentencequeue_getsentenceiterator_meth)). For each sentence, a g-word iterator ([SMC\_GWord\_Iterator](0AOpftHaTjaOBKEDUp2z_aWxY1Y_smc-gword-iterat.html#0aopfthatjaobkedup2z_awxy1y_smc_gword_iterat_smc_gword_iterator_fb)) can be created.
