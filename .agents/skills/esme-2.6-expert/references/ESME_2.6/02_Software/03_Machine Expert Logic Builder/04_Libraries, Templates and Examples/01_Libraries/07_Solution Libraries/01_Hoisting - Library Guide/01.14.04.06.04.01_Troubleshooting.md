# Troubleshooting

Troubleshooting

| Issue | Cause | Solution |
| --- | --- | --- |
| Alarm is signaled:  q\_xAlrm = TRUE  q\_wAlrmId bit 0 = TRUE | An alarm has been detected for an unexpected input signal combination. (q\_wLsStat) | oCheck the position of movement in regards to the limit switch signals indicated by the LimitSwitch\_AR function block.  oClear the alarm signal or reboot the system (Power OFF and Power ON).  oIn the case of incorrect position signals, set the input to TRUE for intermediate operation and verify the wiring or limit switch sensors. |
| Alarm is signaled:  q\_xAlrm = TRUE  q\_wAlrmId bit 1, 2, 3, 4 = TRUE | Incorrect parameterization of the FB. | Correct the input parameters. |
