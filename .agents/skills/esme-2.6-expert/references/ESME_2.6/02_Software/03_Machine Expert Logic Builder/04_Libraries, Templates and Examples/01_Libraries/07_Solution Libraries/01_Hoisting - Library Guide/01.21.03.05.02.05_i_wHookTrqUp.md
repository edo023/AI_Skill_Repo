# i_wHookTrqUp

i\_wHookTrqUp

You are advised to run the drive without a load on the hook and record the percentage (%) load from the Monitoring Menu on the drive or by monitoring the corresponding communication parameters.

When the Hoist is moving up and if the actual torque (i\_iDrvTrqActl) is less than or equal to 70% of the hook torque (i\_wHookTrqUp), then the target speed (q\_wDrvSpdTarg) is set to minimum speed reference (i\_wDrvSpdLsp) and (q\_wSpdTargMax) is set to (i\_wDrvSpdNom).

NOTE:

Following are the points to be respected while using this input:

oThe load on the motor at hook level must be at minimum 10% of the nominal load. Otherwise, the function cannot work correctly due to the efficiency of the gear box. If the load torque is below the minimum 10%, the further reduction by the gear box creates the scenario in which the Altivar drive is not able to determine between generator and motor mode. The result is a non-optimized movement with limited speed.

oWhen the i\_wHookTrqUp input is set to zero, Rope slack detection and correction while moving up is disabled.

oWhen the i\_wHookTrqDown input is set to zero, Rope slack detection and correction while moving down is disabled.
