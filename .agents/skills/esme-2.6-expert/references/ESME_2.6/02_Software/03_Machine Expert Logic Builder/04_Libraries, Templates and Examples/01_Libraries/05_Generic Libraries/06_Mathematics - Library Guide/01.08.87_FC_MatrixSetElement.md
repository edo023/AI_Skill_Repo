# FC\_MatrixSetElement

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.12.0 |

## Task

The function writes a given value into a specified position of a two dimensional LREAL array (matrix).

## Description

The function validates the provided row and column indices against the runtime bounds of the matrix. If the indices are out of range, it reports an error and returns an InputValueInvalid result. If the indices are valid, the function updates the specified matrix element and returns TRUE.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_uiRow | UINT | Row index of the element that is written. |
| i\_uiCol | UINT | Column index of the element that is written. |
| i\_lrValue | LREAL | Value that is written to the specified matrix position. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_alrMatrix | ARRAY [\*, \*] OF LREAL | Matrix into which the element must be written. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | When successful, the function returns TRUE. |

## Troubleshooting

| Issue | Cause | Solution |
| --- | --- | --- |
| Execution ends with a detected error and result indicates InputValueInvalid. | The given value for i\_uiRow is out of range. | Verify that the value of i\_uiRow lies within the bounds of the first dimension of the assigned array. |
| The given value for i\_uiCol is out of range. | Verify that the value of i\_uiCol lies within the bounds of the second dimension of the assigned array. |
