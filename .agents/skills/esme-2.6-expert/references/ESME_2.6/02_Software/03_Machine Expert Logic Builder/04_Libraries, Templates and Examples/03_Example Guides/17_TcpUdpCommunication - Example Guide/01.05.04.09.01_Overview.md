# Overview

Overview

In addition to the MAST task, the task configuration of the example application contains the following tasks:

| Task | Description |
| --- | --- |
| TCP\_UDP\_Communication | Manages the calls of the POUs which are related to the [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) and [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) communication. The task type is Cyclic and the Priority is lower than that of the MAST task. |
| VISU\_TASK | This task is added automatically when adding the web visualization. The default settings are left unchanged. |
