# ST\_Cumulative\_Times

## Overview

|  |  |
| --- | --- |
| Type | Data structure |
| Available as of | V1.4.2.0 |

## Description

The structure ST\_Cumulative\_Times is a collection of tags providing information for timer values.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| AccTimeSinceReset | DINT | Accumulated time since last reset |
| ModeStateTimes | ARRAY [1..Gc\_ uiMaxNumOfModes] OF ST\_ModeState\_Times | Structured array of mode and state time values for each cumulative time tracker |
