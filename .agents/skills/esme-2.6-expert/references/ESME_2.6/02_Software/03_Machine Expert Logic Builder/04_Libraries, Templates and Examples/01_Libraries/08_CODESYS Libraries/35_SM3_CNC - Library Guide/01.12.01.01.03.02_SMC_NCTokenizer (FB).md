# SMC\_NCTokenizer (FB)

FUNCTION\_BLOCK FINAL SMC\_NCTokenizer

Reads a CNC program from an [SMC\_ITextStream](Gtvyd6BrvwKKahBDJTzi-4MJq2k_smc-itextstream.html#gtvyd6brvwkkahbdjtzi_4mjq2k_smc_itextstream_smc_itextstream_itf) and converts it into a [SMC\_TokenQueue](H68tKFuIllH6eV3voqhxDkSWkNU_smc-tokenqueue.html#h68tkfuillh6ev3voqhxdkswknu_smc_tokenqueue_smc_tokenqueue_fb) of [SMC\_Token](H68tKFuIllH6eV3voqhxDkSWkNU_smc-token.html#h68tkfuillh6ev3voqhxdkswknu_smc_token_smc_token_fb), which ultimately will be processed by [SMC\_NCTokenParser](VKaHy6dlyRAODh6pHpRDreSTG3k_smc-nctokenparse.html#vkahy6dlyraodh6phprdrestg3k_smc_nctokenparse_smc_nctokenparser_fb) (which repleaces [SMC\_ReadNCTokens](H68tKFuIllH6eV3voqhxDkSWkNU_smc-readnctokens.html#h68tkfuillh6ev3voqhxdkswknu_smc_readnctokens_smc_readnctokens_fb)). Delimiters for tokens of type *string*, *comment* and *variable* can be reconfigured using the method [SMC\_NCTokenizer.SetDelimiters](isrxvP1l1eEJyAmgeopmb0oqgXY_setdelimiters.html#isrxvp1l1eejyamgeopmb0oqgxy_setdelimiters_smc_nctokenizer_setdelimiters_meth).
