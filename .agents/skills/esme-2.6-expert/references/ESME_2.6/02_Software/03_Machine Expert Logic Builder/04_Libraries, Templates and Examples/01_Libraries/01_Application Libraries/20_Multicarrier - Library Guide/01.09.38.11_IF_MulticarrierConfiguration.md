|  |  |
| --- | --- |

## Task

Reading out the carrier pool of IF\_MulticarrierCarrierPool.

## Description

With the method GetMulticarrierCarrierPool, you can read out the carrier pool of IF\_MulticarrierCarrierPool, defined by the method [ConfigureMulticarrierCarrierPool](ConfigCarrPool-3B197819.html).

## Inputs

The method has no inputs.

| q\_ifMulticarrierCarrierPool | SystemConfigurationItf.IF\_MulticarrierCarrierPool | Provides the carrier pool of IF\_MulticarrierCarrierPool. |
| q\_xError | BOOL | Indicates TRUE if an error has been detected. For details, refer to q\_etResult and q\_sResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-509D6EF3.html#ET_Result-509D6EF3) | Provides diagnostic and status information as a numeric value. If q\_xError = FALSE, q\_etResult provides status information. If q\_xError = TRUE, q\_etResult provides diagnostic/error information. |
