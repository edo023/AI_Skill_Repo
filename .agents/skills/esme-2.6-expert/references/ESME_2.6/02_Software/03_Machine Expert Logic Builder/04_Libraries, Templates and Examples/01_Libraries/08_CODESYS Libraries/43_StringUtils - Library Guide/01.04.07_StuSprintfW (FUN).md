# StuSprintfW (FUN)

FUNCTION StuSprintfW : INT

Does a sprintf for the given format string and the given variable. Will not return NULL!

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | StuSprintfW | INT |  | | Input | wstFormat | POINTER TO WSTRING | the formatstring | | pVarAdr | POINTER TO DWORD | the address of the variable that shall be printed | | udiVarType | \_\_SYSTEM.TYPE\_CLASS | the type of the variable as a \_\_SYSTEM.TYPE\_CLASS value | | pBuffer | POINTER TO BYTE | the buffer where the text is to be printed | | dwBufferSize | UDINT | the size of the buffer in bytes, not in wstring characters! | |
