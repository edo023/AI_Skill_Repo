# Structure: stMotPara

Structure: stMotPara

Structure of parameters related to the motor configuration

| Parameter | Data Type | Description |
| --- | --- | --- |
| diRampAcc | DINT | Acceleration ramp. This parameter is common to Positioning, Jog and Homing modes.  Increasing value of i\_stMotPara.diRampAcc makes the axis accelerate faster. Decreasing it makes the axis accelerate slower.  Range: 1...2147483647  Default value: 100  Scaling/Unit: RPM/s |
| diRampDec | DINT | Deceleration ramp. This parameter is common to Positioning, Jog and Homing modes.  Increasing value of i\_stMotPara.diRampDec makes the axis decelerate faster.  Decreasing it makes the axis decelerate slower.  Range: 1...2147483647  Default value: 100  Scaling/Unit: RPM/s |
| diRampDecQuickStop | DINT | This parameter defines deceleration of the axis for Quick stop. Increasing value of i\_stMotPara.diRampDecQuickStop makes the axis decelerate faster. The input i\_diRampQuickStop deceleration is used in case of an alarm of the FB. The value should be higher than i\_stMotPara.diRampDec.  Range: 0...2147483647  Default value: 100  Scaling/Unit: RPM/s |
| wMotFreqNom | WORD | Nominal motor frequency.  Range: 0...3000  Default value: 500  Scaling/Unit: 0.1 Hz (at 50 Hz: i\_stMotPara.wMotFreqNom=500) |
| wMotFreqMax | WORD | Maximum motor frequency used for movement for the axis.  Range: 0...3000  Default value: 500  Sacaling/Unit: 0.1 Hz (50 Hz: i\_stMotPara.wMotFreqMax=500)  This value must not exceed the high speed configured inside the drive. |
| wMotSpdNom | WORD | Nominal motor speed. Assign the value of nominal speed of the motor from the motor nameplate to this input.  Range: 0...6000  Default value: 1500  Scaling/Unit: RPM |
