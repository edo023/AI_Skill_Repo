## Functional Description

The method SetMatrixTargetsNoRotation sets the parameter for matrix targets without rotation.

## Interface

The interface is identical to FB\_RandomTargetsGenTar.SetTargetsMatrixInPlane(), except for the inputs i\_lrMinRotation, i\_lrMaxRotation which do not exist for the method SetMatrixTargetsNoRotation.

For more information, refer to FB\_RandomTargetsGenerator in the SchneiderElectricRobotics Toolbox Library Guide Online Help.

NOTE: The input i\_etOrientationConvention is no longer input of the method, because it is set to the value ROB.ET\_OrientationConvention.ZYX.
