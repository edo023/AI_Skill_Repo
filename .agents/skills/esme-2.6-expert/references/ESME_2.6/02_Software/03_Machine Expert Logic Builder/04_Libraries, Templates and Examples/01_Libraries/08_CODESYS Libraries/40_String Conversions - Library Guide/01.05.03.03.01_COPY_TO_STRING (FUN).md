# COPY\_TO\_STRING (FUN)

FUNCTION COPY\_TO\_STRING : UDINT

Copies an [IStringRepresentation](QS9IiJwK2G4tvWpsSL8DUp7vIyA_istringrepresent.html#qs9iijwk2g4tvwpssl8dup7viya_istringrepresent_istringrepresentation_alias) into a memory block of a string.

Type of the itfString can be IBuilder, IString or just an IStringRepresentation.
