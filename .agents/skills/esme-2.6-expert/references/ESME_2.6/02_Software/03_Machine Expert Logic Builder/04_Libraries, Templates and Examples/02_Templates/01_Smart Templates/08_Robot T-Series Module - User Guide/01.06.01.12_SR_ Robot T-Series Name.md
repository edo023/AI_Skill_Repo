## Functional Description

By calling the method, the kinematic parameter can be adapted.

For information on how to use the i\_stCenterOfMass, refer to [*FB\_RobotTSeries - SetKinematicParameter (Method)*](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRobotics&topicID=D_SE_0075206) in the SchneiderElectricRobotics Library Guide.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etName | [SER.ET\_KinematicParameter](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRobotics&topicID=D_SE_0075190) | Select the kinematic parameter. |
| i\_lrValue | LREAL | Mass of the selected kinematic parameter [kg]. |
| i\_stCenterOfMass | PDL.ST\_Vector3D | Set the center of mass for the selected kinematic parameter [mm]. |

| q\_etDiagExt | [SER.ET\_DiagExt](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRobotics&topicID=D_SE_0075188) | POU-specific output for the diagnostic.  q\_etDiag = ET\_Diag.Ok -> Status message  q\_etDiag <> ET\_Diag.Ok -> Diagnostic message |
