# IVisuStreamHandler.StartOperation (METH)

METHOD StartOperation : SysTypes.RTS\_IEC\_RESULT

This method will be called when a read or write operation on the current instance operation is started. The return value can either be:

> * ERR\_OK: operation successfully started
> * any other error code: starting an operation is not possible (at the moment)
