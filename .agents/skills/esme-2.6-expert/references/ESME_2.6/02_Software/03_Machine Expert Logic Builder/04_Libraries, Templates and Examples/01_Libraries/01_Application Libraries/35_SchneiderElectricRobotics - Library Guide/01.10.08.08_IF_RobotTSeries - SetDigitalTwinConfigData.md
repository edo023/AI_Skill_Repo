## Task

Set the configuration data for the EcoStruxure Machine Expert Twin communication of the Lexium T Robot.

## Description

By calling the method, the provided configuration data are verified, stored, and transferred into the OPC UA data structure of the Lexium T Robot.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stData | DTC.ST\_ConfigDataRobotTSeries | Configuration data for EcoStruxure Machine Expert Twin.  NOTE: i\_stData.sCategory and i\_stData. xParallelLinkageLeft are overwritten by the value stored inside the function block. |
| i\_udiArrayIndex | UDINT | The array index number indicates the robot related OPC UA data structure, for example, DTC.GVL\_OpcUaRobotics.G\_stROB.astRobotTPSeries[i\_udiArrayIndex]. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | General, library-independent statement on the diagnostic.  A value not equal to ET\_Diag.Ok corresponds to a diagnostic message. |
