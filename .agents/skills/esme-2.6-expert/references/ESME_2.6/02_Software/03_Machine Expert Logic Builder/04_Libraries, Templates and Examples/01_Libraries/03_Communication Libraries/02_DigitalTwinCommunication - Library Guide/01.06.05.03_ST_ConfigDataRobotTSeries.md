# ST\_ConfigDataRobotTSeries

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | ST\_ConfigDataGeneric |

## Description

The structure ST\_ConfigDataRobotTSeries contains additional parameters to configure a Lexium T robot in EcoStruxure Machine Expert Twin extending the structure [ST\_ConfigDataGeneric](ST_ConfigDataGeneric-6D5DB8E3.html#ST_ConfigDataGeneric-6D5DB8E3).

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| xParallelLinkageLeft | BOOL | Position where the parallel linkage is mounted.  By default, xParallelLinkageLeft is set to FALSE which means that the parallel linkage is mounted on the right.  If xParallelLinkageLeft is set to TRUE, the parallel linkage is mounted on the left. |
