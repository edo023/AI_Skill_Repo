# ST\_WillMessage

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

The structure ST\_WillMessage specifies the Will message for the last will and testament feature.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sTopic | STRING[255] | The topic to which the specified Will message belongs to. |
| sData | STRING[255] | The Will message data which is published by the MQTT server in case of disconnection of the client. |
| xRetain | BOOL | Specifies if the Will message must be retained on the MQTT server. |
| etQualityOfService | ET\_QualityOfService | Specifies the QoS (Quality of Service) level which must be used while publishing the Will message. |
