# VisuFbClientTagDataHelper (FB)

FUNCTION\_BLOCK VisuFbClientTagDataHelper

This functionblock can be used to access the client tag data optionally contained in a visualization client object. As compiling this function block requires a device of version 3.5.5 at least, instances should be created only when such a device is used.
