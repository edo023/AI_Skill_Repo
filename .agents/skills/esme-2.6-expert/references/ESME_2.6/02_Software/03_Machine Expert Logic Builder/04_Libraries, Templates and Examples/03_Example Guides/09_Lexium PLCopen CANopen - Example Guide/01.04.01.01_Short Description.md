# Short Description

Short Description

The EcoStruxure Machine Expert project LexiumPLCopenCANopenExample.project provides an application example which shows how to implement a Lexium servo drive controlled via fieldbus [CANopen](../glossary/glossary.htm#XREF_D_SE_0024697_650). The application example is based on a programming framework supporting general machine control. The monitor and control functions for the Lexium drives are implemented using the [PLCopen](../glossary/glossary.htm#XREF_D_SE_0024697_476) [function blocks](../glossary/glossary.htm#XREF_D_SE_0024697_715) from the libraries GMC Independent PLCopen MC and Lexium 28.
