# CAAReconfigureBase (FB)

FUNCTION\_BLOCK CAAReconfigureBase IMPLEMENTS [IReconfigureProvider](qG5KZxShRzJP0Witj1Yz7sQso84_ireconfigureprov.html#qg5kzxshrzjp0witj1yz7sqso84_ireconfigureprov_ireconfigureprovider_itf)

This function block provides a base implementation for [IReconfigureProvider](qG5KZxShRzJP0Witj1Yz7sQso84_ireconfigureprov.html#qg5kzxshrzjp0witj1yz7sqso84_ireconfigureprov_ireconfigureprovider_itf). It is important to call [CAAReconfigureBase.Initialize\_Diag](Gg3z1PB3R1_5J9tcreCbLRBTJFM_initialize-diag.html#gg3z1pb3r1_5j9tcrecblrbtjfm_initialize_diag_caareconfigurebase_initialize_diag_meth) once before using this function block. I/O driver nodes just delegate all [IReconfigureProvider](qG5KZxShRzJP0Witj1Yz7sQso84_ireconfigureprov.html#qg5kzxshrzjp0witj1yz7sqso84_ireconfigureprov_ireconfigureprovider_itf) methods and properties to this function block. There is no additional code necessary. For other nodes a specific implementation is necessary. For this [CAAReconfigureBase.BeginReconfigure](Gg3z1PB3R1_5J9tcreCbLRBTJFM_beginreconfigure.html#gg3z1pb3r1_5j9tcrecblrbtjfm_beginreconfigure_caareconfigurebase_beginreconfigure_meth), [CAAReconfigureBase.ReconfigureChildrenAsync](X8lnxNxTZZvAWu9FTCE_CAUXaW8_reconfigurechild.html#x8lnxnxtzzvawu9ftce_cauxaw8_reconfigurechild_caareconfigurebase_reconfigurechildrenasync_meth) and [CAAReconfigureBase.EndReconfigure](Gg3z1PB3R1_5J9tcreCbLRBTJFM_endreconfigure.html#gg3z1pb3r1_5j9tcrecblrbtjfm_endreconfigure_caareconfigurebase_endreconfigure_meth) can be used.
