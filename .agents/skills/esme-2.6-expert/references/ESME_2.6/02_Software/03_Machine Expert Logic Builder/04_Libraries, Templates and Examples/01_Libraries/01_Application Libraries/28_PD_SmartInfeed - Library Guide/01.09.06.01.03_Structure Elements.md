# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| lrPartsPerMinuteInfeed | LREAL | The measured number of products which are accepted per minute by the feed belt specified with [udiBeltId](Structures-7.htm#XREF_D_SE_0080366_1). |
| lrTargetsPerMinuteOutfeed | LREAL | The measured number of targets that change to the last belt per minute |
| lrFeedForwardOutfeed | LREAL | The calculated velocity at which the target belt has to move so that lrPartsPerMinuteInfeed is equal to lrTargetsPerMinuteOutfeed |
| lrFeedForwardInfeed | LREAL | The calculated velocity at which the belt specified by [udiBeltId](Structures-7.htm#XREF_D_SE_0080366_1) has to move so that lrPartsPerMinuteInfeed is equal to lrTargetsPerMinuteOutfeed |
