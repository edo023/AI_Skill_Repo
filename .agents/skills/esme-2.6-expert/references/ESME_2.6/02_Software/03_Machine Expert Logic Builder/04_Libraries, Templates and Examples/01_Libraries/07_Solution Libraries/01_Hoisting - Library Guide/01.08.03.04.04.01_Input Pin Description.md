# Input Pin Description

Input Pin Description

| Input | Data Type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block. |
| i\_rPosHigh | REAL | This input is the cable length for the high position.  Range: 0...40 m  Accuracy: 0.01 m |
| i\_rPosMid | REAL | This input is the cable length for the middle position.  Range: 0...40 m  Accuracy: 0.01 m |
| i\_rPosLow | REAL | This input is the cable length for the low position.  Range: 0...40 m  Accuracy: 0.01 m |
| i\_rLoadLen | REAL | This input is the load length value.  Range: 0...10 m  Accuracy: 0.01 m |
| i\_xSel1 | BOOL | This input is the cable length selector position 1.  TRUE: Low position  FALSE: Medium or high position |
| i\_xSel2 | BOOL | This input is the cable length selector position 2.  TRUE: Medium position  FALSE: High position; depends on i\_xSel1 |
| i\_xLoadSel | BOOL | This input is used to add the load length to the total cable length calculation.  TRUE: Offset used  FALSE: Offset not used |
