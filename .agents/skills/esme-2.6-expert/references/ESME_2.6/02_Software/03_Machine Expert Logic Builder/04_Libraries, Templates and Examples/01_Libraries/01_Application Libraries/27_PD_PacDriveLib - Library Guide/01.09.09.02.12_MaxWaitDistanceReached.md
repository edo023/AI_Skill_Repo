# MaxWaitDistanceReached

MaxWaitDistanceReached

|  |  |
| --- | --- |
| Enumeration name: | MaxWaitDistanceReached |
| Enumeration value: | 170 |
| Description: | MaxWaitDistance has been reached. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The encoder has covered a distance of more than i\_lrMaxWaitDistance without receiving a Touchprobe signal. | Verify the Touchprobe sensor  Verify whether the correct touchprobe has been connected to the i\_ifTouchProbe connection.  Disable the monitoring by setting the input i\_lrMaxWaitDistance to 0. |
