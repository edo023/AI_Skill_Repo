# GVL

## Overview

|  |  |
| --- | --- |
| Type: | Global variables |
| Available as of: | V1.3.0.0 |

## Description

The global variable list contains the global variables of the FtpRemoteFileHandling library.

This global variable is automatically used by the function block FB\_FtpClient.

NOTE: It is not used by the FB\_FtpSecureClient function block.

## Global Variables

| Variable | Data type | Description |
| --- | --- | --- |
| G\_xDisableAsyncExecFtpClient | BOOL | If TRUE, the asynchronous execution of the commands which can be executed by the FB\_FtpClient function block is disabled.(1)  Default value: TRUE |
| **(1)** **Note:** Disabling the asynchronous execution means that the program code is executed during the calling task. This leads to an increased execution time for at least one task cycle during the process. | | |
