# AsyncInitNotFinished

AsyncInitNotFinished

|  |  |
| --- | --- |
| Enumeration name: | AsyncInitNotFinished |
| Enumeration value: | 189 |
| Description: | Init has not been completed yet. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | Init has not been completed yet. | Call up the Init() before you call up Start() for the first time.  After you have called up the Init, wait until Init has been executed successfully (Init.q\_xDone = TRUE). |
