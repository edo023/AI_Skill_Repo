# DataNVRamInvalid

DataNVRamInvalid

|  |  |
| --- | --- |
| Enumeration name: | DataNVRamInvalid |
| Enumeration value: | 194 |
| Description: | The non-volatile memory (NVRam) does not contain any more valid data. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The data stored in the NVRam are invalid. Therefor, the position cannot be restored from i\_stRestorePosition. | Verify the NVRam, verify the buffer battery and replace if necessary.  New homing of the drive required. |
