# NumberOfVerticesPolytope2Range

NumberOfVerticesPolytope2Range

|  |  |
| --- | --- |
| Enumeration name: | NumberOfVerticesPolytope2Range |
| Enumeration value: | 57 |
| Description: | NumberOfVertices of polytope 2 is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | At the input iq\_stPolytope2.diNumberOfVertices, a value smaller than 1 or greater than [Gc\_diMaxNumberOfVertices](../Global_Elements/Global_Elements-2.htm#XREF_D_SE_0087806_1) has been applied. | For iq\_stPolytope2.diNumberOfVertices, a value between 1 and [Gc\_diMaxNumberOfVertices](../Global_Elements/Global_Elements-2.htm#XREF_D_SE_0087806_1) must be chosen. |
