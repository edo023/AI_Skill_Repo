# SMC\_Seg (STRUCT)

TYPE SMC\_Seg : STRUCT

This structure is used internally by the [SMC\_Interpolator](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-interpolator.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_interpolator_smc_interpolator_fb) . It represents a path segment of fixed length in which a certain acceleration shape is used.
