## Description [[1]](#index_indices_and_tables__fn_index_id2)

Implements functions and constants to support text encoded in UTF-8. It includes functions to translate between runes (See: [RUNE](Tdg6Ql3l9Me28nf8SHpLBXgb98M_rune.html#tdg6ql3l9me28nf8shplbxgb98m_rune_rune_alias)) and UTF-8 byte sequences. See <https://en.wikipedia.org/wiki/UTF-8>
