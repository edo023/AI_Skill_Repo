# PackBytesToWord (FUN)

FUNCTION PackBytesToWord : WORD

Function packs low and high BYTE into a WORD.

Example:

```
MEM.PackBytesToWord(16#01,16#02) = 16#0102
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | PackBytesToWord | WORD | function returns the WORD defined by the BYTE inputs | | Input | byHighByte | BYTE | high byte of the Word | | byLowByte | BYTE | low byte of the Word | |
