# UAMonitoringParameter

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.0.0.0 |
| Inherits from: | - |
| Attribute: | ``` 'pack_mode' := '1' ``` |

## Description

The structure UAMonitoringParameter provides parameters to configure the monitoring.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| SamplingInterval | TIME | The interval (in milliseconds) the server polls the underlying data source for changes. |
| QueueSize | UINT | The queue size for the monitoring item.  Default value: 1 |
| DiscardOldest | BOOL | Determines the discard policy in case of queue overflow. |
| DeadbandType | UADeadbandType | Indicates the type of deadband that is applied, if any. |
| Deadband | REAL | The deadband value (in %). |
