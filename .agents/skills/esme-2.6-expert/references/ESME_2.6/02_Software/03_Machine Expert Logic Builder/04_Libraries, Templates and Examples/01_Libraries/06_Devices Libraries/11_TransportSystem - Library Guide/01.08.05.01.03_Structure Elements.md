# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_uId | UINT | Destination where vehicle is commanded to move to.  “MoveVehicleToStation” => Station ID  “MoveVehicleToPosition” => Path ID |
| i\_rPosition | REAL | “MoveVehicleToStation” => <not needed>  “MoveVehicleToPosition” => Position on path in [m] calculated from start of path. |
| i\_rVelocity | REAL | Determine max. velocity for movement in [m/s] |
| i\_rAcceleration | REAL | Determine max. acceleration for movement in [m/s2] |
| i\_etMovementDirection | [ET\_MovementDirection](../Enumerations/Enumerations-4.htm#XREF_D_SE_0090678_1) | Determine movement direction |
