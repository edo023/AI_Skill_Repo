# Open (FB; Prefix filop)

FUNCTION\_BLOCK Open EXTENDS CBM.ETrig

This function block opens an already existing file or creates a new one. The return value is a file handle, which can then be used as an input ”hFile” in the function blocks ”FILE.[Write](VIPLMVOpanJfjqQXFKWGQlzGjxw_write.html#viplmvopanjfjqqxfkwgqlzgjxw_write_write_fb_prefix_filwr)”, ”FILE.[Read](VIPLMVOpanJfjqQXFKWGQlzGjxw_read.html#viplmvopanjfjqqxfkwgqlzgjxw_read_read_fb_prefix_filrd)”, ”FILE.[Close](VIPLMVOpanJfjqQXFKWGQlzGjxw_close.html#viplmvopanjfjqqxfkwgqlzgjxw_close_close_fb_prefix_filcl)”. There may be restrictions concerning the specification of the directory name, e.g. only capital letters allowed, for different targets. The function ”FILE.[GetProperty](RFfZ6qcMtbnrwad-bdR1UipBxBA_getproperty.html#rffz6qcmtbnrwad_bdr1uipbxba_getproperty_getproperty_fun)” shows these restrictions.
