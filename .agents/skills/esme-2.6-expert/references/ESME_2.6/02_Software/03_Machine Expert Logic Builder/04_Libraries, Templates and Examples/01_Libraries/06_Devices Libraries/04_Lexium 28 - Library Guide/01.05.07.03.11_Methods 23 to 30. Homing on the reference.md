# Methods 23 to 30. Homing on the reference switch

Methods 23 to 30. Homing on the reference switch

These methods use a reference switch which is active over only portion of the travel.

Methods 23 to 26:

The initial movement is in the positive direction if the reference switch is inactive.

The initial movement is in the negative direction if the reference switch is active.

If the initial movement leads away from the reference switch, the drive reverses on encountering the positive limit switch.

oMethod 23:

The home position is in the negative direction where the reference switch becomes inactive.

oMethod 24:

The home position is in the positive direction where the reference switch becomes active.

oMethod 25:

The home position is in the negative direction where the reference switch becomes active.

oMethod 26:

The home position is in the positive direction where the reference switch becomes inactive.

Methods 27 to 30:

The initial movement is in the negative direction if the reference switch is inactive.

The initial movement is in the positive direction if the reference switch is active.

If the initial movement leads away from the reference switch, the drive reverses on encountering the negative limit switch.

oMethod 27:

The home position is in the positive direction where the reference switch becomes inactive.

oMethod 28:

The home position is in the negative direction where the reference switch becomes active.

oMethod 29:

The home position is in the positive direction where the reference switch becomes active.

oMethod 30:

The home position is in the negative direction where the reference switch becomes inactive.
