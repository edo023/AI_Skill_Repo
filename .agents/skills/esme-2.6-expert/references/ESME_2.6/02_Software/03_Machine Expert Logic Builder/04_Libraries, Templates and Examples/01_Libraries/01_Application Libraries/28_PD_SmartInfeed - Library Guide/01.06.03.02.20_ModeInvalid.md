# ModeInvalid

ModeInvalid

|  |  |
| --- | --- |
| Enumeration name: | ModeInvalid |
| Enumeration value: | 65 |
| Description: | An invalid Mode has been selected. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid value has been transferred at the input iq\_stParameter.stProductGeneration.etMode. | It must be made sure that an element of [ET\_SimGenerateMode](../Enumerations/Enumerations-7.htm#XREF_D_SE_0080273_1) is transferred at the input iq\_stParameter.stProductGeneration.etMode. |
| - | An invalid value has been transferred at the input iq\_stParameter.stTargetSignalGeneration.etMode. | It must be made sure that an element of [ET\_SimGenerateMode](../Enumerations/Enumerations-7.htm#XREF_D_SE_0080273_1) is transferred at the input iq\_stParameter.stTargetSignalGeneration.etMode. |
