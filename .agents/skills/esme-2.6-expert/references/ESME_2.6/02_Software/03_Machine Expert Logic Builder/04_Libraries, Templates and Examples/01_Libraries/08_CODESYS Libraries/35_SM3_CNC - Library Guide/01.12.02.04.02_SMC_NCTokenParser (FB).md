# SMC\_NCTokenParser (FB)

FUNCTION\_BLOCK SMC\_NCTokenParser

The parser between the [SMC\_NCTokenizer](H68tKFuIllH6eV3voqhxDkSWkNU_smc-nctokenizer.html#h68tkfuillh6ev3voqhxdkswknu_smc_nctokenizer_smc_nctokenizer_fb) and the [SMC\_NCInterpreter](VVFvsggjXKxZ6qmLw10MvZPswnU_smc-ncinterprete.html#vvfvsggjxkxz6qmlw10mvzpswnu_smc_ncinterprete_smc_ncinterpreter_fb). The parser transforms the stream of tokens to a sequence of g-code sentences ready to be interpreted.

**Note**

Usually, [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb) or [SMC\_ReadNCFromStream](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfromst.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfromst_smc_readncfromstream_fb) should be used, which contain the parser. It is not necessary to use this function block directly in most cases.
