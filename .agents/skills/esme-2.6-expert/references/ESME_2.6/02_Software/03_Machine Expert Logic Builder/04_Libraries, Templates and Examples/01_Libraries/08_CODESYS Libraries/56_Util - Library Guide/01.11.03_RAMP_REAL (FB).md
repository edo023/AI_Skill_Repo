# RAMP\_REAL (FB)

FUNCTION\_BLOCK RAMP\_REAL

Limits the slope of a value to a certain value

The function block is similar to RAMP\_INT with the difference that the inputs IN, ASCEND, DESCEND and the output OUT are of the type REAL. RAMP\_REAL takes more computing time but calculates more precisely than RAMP\_INT.

**Note**

A reset of the function block ( RESET = TRUE) stops the calculation and the last output value OUT is maintained. So if afterwards the ramp gets restarted by RESET = FALSE, it will start with this last output value!

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | IN | REAL | Input value | | ASCEND | REAL | Maximum positive slope | | DESCEND | REAL | Maximum negative slope (non-negative!) | | TIMEBASE | TIME | Reference for ASCEND/DESCEND :  t#0s: ASCEND/DESCEND defined per call  Else : ASCEND/DESCEND defined per specified time | | RESET | BOOL | Resets the function block | | Output | OUT | REAL | Fnction value with limited slope | |
