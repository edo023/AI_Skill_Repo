# IVisualizationClientRaw.ClientDataPointer (PROP)

PROPERTY ClientDataPointer : POINTER TO VisuElemBase.VisuStructClientData

Returns the ClientDataPointer which is needed for other functionalities for which there is no alternative in VisuUtils. Normally this pointer should only be passed to other functions. The internal data should not be accessed.
