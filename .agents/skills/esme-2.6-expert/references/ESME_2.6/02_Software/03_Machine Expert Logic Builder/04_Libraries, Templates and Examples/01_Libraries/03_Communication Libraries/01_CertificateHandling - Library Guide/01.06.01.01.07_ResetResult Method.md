# ResetResult Method

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.1.2.0 |

![](images/0000807858.png)

## Functional Description

This method is used to set the value of the property Result to ET\_Result.Ok.

## Interface

| Return value | Data type | Description |
| --- | --- | --- |
| ResetResult | BOOL | If this output is set to TRUE, the value of the method Result is reset to Ok. |
