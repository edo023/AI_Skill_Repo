|  |  |
| --- | --- |

## Task

Deleting the velocity limitation for the movement of a carrier.

## Description

The method DeleteVelocityLimitation deletes the values configured for the velocity limitation of a carrier.

## Inputs

The method has no inputs.

| --- | --- | --- |
