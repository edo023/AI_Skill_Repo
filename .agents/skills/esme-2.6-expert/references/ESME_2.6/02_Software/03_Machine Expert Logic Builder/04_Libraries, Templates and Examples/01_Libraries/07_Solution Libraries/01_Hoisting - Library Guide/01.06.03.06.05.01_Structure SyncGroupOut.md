# Structure: SyncGroupOut

Structure: SyncGroupOut

Structure of parameters related to the synchronization of groups

| Parameter | Data type | Description |
| --- | --- | --- |
| xSyncState | BOOL | Synchronization state  TRUE: Synchronization is active.  FALSE: Synchronization is inactive. |
| diActPosSync | DINT | Actual position of the synchronization group  Range: 0...65535  Unit: in. |
| iActSpdSync | INT | Actual speed of the synchronization group  Range: -32768...32767  Unit: RPM |
| iAxisCount | INT | Number of axes within the synchronization group  Range: 0...32767 |
| wAlrmId | WORD | Alarm identification  Range: 0...65535 |
