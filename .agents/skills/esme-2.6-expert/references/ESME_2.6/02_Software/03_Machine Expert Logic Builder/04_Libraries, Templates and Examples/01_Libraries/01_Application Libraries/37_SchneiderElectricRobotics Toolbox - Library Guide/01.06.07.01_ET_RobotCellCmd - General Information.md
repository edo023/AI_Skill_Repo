# ET\_RobotCellCmd - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.6.0.0 |

## Description

The ET\_RobotCellCmd enumerator is used to send commands to the robot cell.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| NoCmd | 0 | No command. |
| Prepare | 710 | Command Prepare can be used to execute homing mode. |
| Manual | 720 | Command Manual can be used to execute manual mode. |
| Auto | 730 | Command Auto can be used to execute automatic mode. |
| BrakeRelease | 740 | Command BrakeRelease can be used to execute BrakeRelease mode. |
| User1 | 750 | Command User1 can be used for commands of a user. |
| User2 | 760 | Command User2 can be used for commands of a user. |
| DriveEnableOn | 791 | Command DriveEnableOn can be used to enable all drives. |
| DriveEnableOff | 793 | Command DriveEnableOff can be used to disable all drives. |
| Stop | 794 | Command Stop can be used for cancelling the current mode. |
| Start | 795 | Command Start can be used to start the current mode. |
| Hold | 796 | Command Hold can be used for cancelling a start command. |
| Abort | 798 | Command Abort can be used to abort the operation mode. |
