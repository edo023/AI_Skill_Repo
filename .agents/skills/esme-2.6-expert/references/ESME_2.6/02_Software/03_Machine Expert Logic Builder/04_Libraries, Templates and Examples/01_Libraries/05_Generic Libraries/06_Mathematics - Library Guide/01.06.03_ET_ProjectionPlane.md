|  |  |
| --- | --- |

## Description

Indication of a plane of the coordinate system.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No projection plane selected. |
| XY | 1 | Plane along the X and the Y axis. |
| XZ | 2 | Plane along the X and the Z axis. |
| YZ | 3 | Plane along the Y and the Z axis. |
