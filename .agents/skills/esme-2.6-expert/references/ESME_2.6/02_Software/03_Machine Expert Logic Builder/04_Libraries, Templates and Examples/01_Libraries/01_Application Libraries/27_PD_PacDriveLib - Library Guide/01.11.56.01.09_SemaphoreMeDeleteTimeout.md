# SemaphoreMeDeleteTimeout

SemaphoreMeDeleteTimeout

|  |  |
| --- | --- |
| Enumeration name: | SemaphoreMeDeleteTimeout |
| Enumeration value: | 270 |
| Description: | Timeout when deleting the SemaphoreMe |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An attempt was made to destroy a semaphore in use that has not been released within a given time out. | Ensure that the semaphore with [FC\_SemaphoreMeGive](Functions_K_to_Z-57.htm#XREF_D_SE_0087649_1) has been released and was reserved with [FC\_SemaphoreMeTake](Functions_K_to_Z-59.htm#XREF_D_SE_0087653_1). |
