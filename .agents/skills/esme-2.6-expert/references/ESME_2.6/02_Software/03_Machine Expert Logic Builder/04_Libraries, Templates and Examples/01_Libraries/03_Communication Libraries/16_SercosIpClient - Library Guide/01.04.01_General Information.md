# General Information

## Description

The library SercosIpClient provides enumerations, structures, and functions for non-real-time communication via the Sercos bus.

The Sercos Internet Protocol (S/IP) provides services via the non-real-time communication of the Sercos bus. The Internet Protocol Services (IPS) can be used to get information on Sercos devices connected to a Sercos bus. The controller acts as an S/IP client, the devices connected to the Sercos bus such as drives and bus couplers act as S/IP servers. PacDrive Sercos 3 devices use User Datagram Protocol (UDP) for the S/IP services with the exception of firmware management and update which uses the Trivial File Transfer Protocol (TFTP). The Transmission Control Protocol (TCP) is not supported.
