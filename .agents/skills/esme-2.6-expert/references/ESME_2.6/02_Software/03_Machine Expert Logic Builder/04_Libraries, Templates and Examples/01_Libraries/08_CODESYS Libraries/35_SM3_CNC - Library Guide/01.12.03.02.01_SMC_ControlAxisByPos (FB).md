# SMC\_ControlAxisByPos (FB)

FUNCTION\_BLOCK SMC\_ControlAxisByPos

The function block writes the set position fSetPosition to the drive structure Axis and monitors Axis for jumps. SMC\_ControlAxisByPos is mostly used with CNC and an instance of the [SMC\_Interpolator](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-interpolator.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_interpolator_smc_interpolator_fb).
