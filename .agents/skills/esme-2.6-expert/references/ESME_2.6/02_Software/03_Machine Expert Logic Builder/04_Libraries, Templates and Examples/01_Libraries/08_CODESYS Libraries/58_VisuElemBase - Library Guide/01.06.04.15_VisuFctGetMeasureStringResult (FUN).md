# VisuFctGetMeasureStringResult (FUN)

FUNCTION VisuFctGetMeasureStringResult : BOOL

Function, to get the result of MeasureString. Call the function from HandleInput of the element function block when the EventTag = VISU\_ET\_MEASURESTRINGRESULT Returns true, when the measuring result is available, else false. After a successful result, the calling element should reset its iTextSizeIndex to avoid a second usage with VisuFctGetMeasureStringResult, where measuring was started for another element.
