# Hysteresis\_LREAL (FB)

FUNCTION\_BLOCK Hysteresis\_LREAL

This function block will set the output variable to TRUE, if the input value ![](images/cf1d7f065ccd4844adbc73529d1e0bd4.jp.jpg) is smaller than a lower bound ![](images/ec35c36e58934f5cbb217b9c42d0be40.jp.jpg).

It will set the output variable to FALSE, if the input value ![](images/2ca6a34230874d84a73924dd8bd39393.jp.jpg) exceeds the upper bound ![](images/8dfa7fd9b9a1494ea2af5eab96eefa16.jp.jpg).

If ![](images/37cb7c61bad443f2bea05af73167b044.jp.jpg) lies between the lower and the upper bound, the value of the output variable will rest unchanged by the module.
