# i_xLsRevSlow

i\_xLsRevSlow

This signal is used to assist the slow down at the slow down limit switch in the reverse direction. This input does not modify the speed reference. It sets shorter ramps in order to slow the movement down in a shorter distance while still using the Anti-sway function. The speed reference required in the slow area must be given to the input i\_wDrvSpdRef from an external source (e.g. SpeedSelect function block).
