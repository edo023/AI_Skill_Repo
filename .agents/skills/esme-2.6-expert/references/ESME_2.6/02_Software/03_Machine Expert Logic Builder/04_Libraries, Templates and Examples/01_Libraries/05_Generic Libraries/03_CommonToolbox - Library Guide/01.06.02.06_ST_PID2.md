# ST\_PID2

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.2.3.0 |

## Description

The structure ST\_PID2 provides the parameters for the PID controller for FB\_HeatingControl2.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| rKp | REAL | Value of proportional gain.  Range: 0…10000  Default value: 2 |
| rTn | REAL | Value of integral time in milliseconds.  Range: 0...60000  Default value: 200 |
| rTv | REAL | Value of derivative time in milliseconds.  Range: 0…60000  Default value: 0 |
