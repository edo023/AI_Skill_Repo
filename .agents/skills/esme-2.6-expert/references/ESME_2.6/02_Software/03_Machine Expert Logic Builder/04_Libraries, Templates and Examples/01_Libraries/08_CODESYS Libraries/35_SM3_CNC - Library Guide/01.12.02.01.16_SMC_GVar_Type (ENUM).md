# SMC\_GVar\_Type (ENUM)

TYPE SMC\_GVar\_Type :

The supported types of variables, formal subprogram parameters, and expressions in G-Code. Note that there is only one numeric type, LREAL. All numeric values are represented as an LREAL. See [SMC\_NC\_GArgumentValue](N1iY2oaH809ZTO1uQFP4HHVDcCg_smc-nc-gargument1.html#n1iy2oah809zto1uqfp4hhvdccg_smc_nc_gargument1_smc_nc_gargumentvalue_union), a union that holds the value of a variable or expression.
