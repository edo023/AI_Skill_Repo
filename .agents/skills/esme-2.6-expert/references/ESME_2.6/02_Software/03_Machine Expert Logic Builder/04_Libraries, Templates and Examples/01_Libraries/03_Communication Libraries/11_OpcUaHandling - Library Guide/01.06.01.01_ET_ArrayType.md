# ET\_ArrayType

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V2.2.4.0 |

## Description

The enumeration ET\_ArrayType specifies the type of the variable used as argument in a method call.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| UATypeScalar | 16#0 | The variable is of type scalar. |
| UATypeArray | 16#1 | The variable is of type array. |
| UATypeMatrix | 16#2 | The variable is of type matrix. |
