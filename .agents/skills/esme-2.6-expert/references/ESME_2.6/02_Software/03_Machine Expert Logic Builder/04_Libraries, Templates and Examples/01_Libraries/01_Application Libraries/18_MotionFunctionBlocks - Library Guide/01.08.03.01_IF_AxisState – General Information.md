# IF\_AxisState – General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.1.0 |
| Inherits from: | — |

## Description

With the interface IF\_AxisState, it is possible to read the operating state of the axis according to PLCopen state machine.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| etAxisState | MOIN.ET\_AxisState | Get | This property is used to read the state of the axis according to PLCopen state machine. |
| xContinuousMotion | BOOL | Get | TRUE: The drive is in operating state Continuous Motion. |
| xDisabled | BOOL | Get | This property indicates if the power of the drive is disabled.  TRUE: The power stage is disabled. |
| xDiscreteMotion | BOOL | Get | TRUE: The drive is in operating state Discrete Motion. |
| xErrorstop | BOOL | Get | TRUE: The drive is in operating state Errorstop. |
| xStandStill | BOOL | Get | TRUE: The drive is in operating state StandStill. |
| xStopping | BOOL | Get | TRUE: The drive is in operating state Stopping. |
| xSynchronizedMotion | BOOL | Get | TRUE: The drive is in operating state SynchronizedMotion. |
