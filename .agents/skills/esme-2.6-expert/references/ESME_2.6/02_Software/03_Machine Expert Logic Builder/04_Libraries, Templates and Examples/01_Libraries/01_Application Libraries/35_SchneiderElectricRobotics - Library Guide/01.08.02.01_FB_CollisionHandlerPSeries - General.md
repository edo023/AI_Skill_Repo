## Task

Implements the collision handler of a P-Series robot.

## Description

The function block is an extension of COD.FB\_CollisionHanlderDelta3Ax and allows a simplified configuration of the collision handler of a Delta3Ax starting from a configured instance of FB\_RobotPSeries.

## Methods

| GetOptionalParameters | Get the optional parameters for a P-Series collision handler. |
