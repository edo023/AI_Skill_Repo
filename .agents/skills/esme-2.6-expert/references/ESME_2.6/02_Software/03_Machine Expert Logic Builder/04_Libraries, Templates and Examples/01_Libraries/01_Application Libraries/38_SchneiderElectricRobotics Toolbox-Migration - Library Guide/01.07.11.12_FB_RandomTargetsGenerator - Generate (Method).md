## Task

Generate a set of targets.

## Description

The method Generate allows you to generate a set of targets. Depending on the type of generation selected, the result is different.

|  |  |
| --- | --- |
| Enumeration name: | MaxOverlapCheckIterationsRange |
| Enumeration value: | 55 |
| Description: | Maximum number of iterations out of range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| Maximum number of iterations out of range. | The configured maximum number of iterations described by i\_udiMaxOverlapCheckIterations is reached and it is not possible to generate a list of i\_udiNumberOfTargets non-overlapping targets according to the constraints provided by you. | * Verify that i\_udiNumberOfTargets targets can fit the spawn area with no overlaps. * Try to increase the value of i\_udiMaxOverlapCheckIterations. |
