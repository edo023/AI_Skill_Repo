# Overview

## Short Description

The project implements an application example which demonstrates how to implement the OPC UA client functionality in your application using the function blocks out of the OpcUaHandling library.

## System Requirements and Limitations

The application example has been created and tested with the components listed in the table:

| Component | Type and Version |
| --- | --- |
| Software | EcoStruxure Machine Expert V2.x |
| Controller | Modicon M262 Motion Controller TM262M25MESS8T and PacDrive LMC Pro/Pro2 |

NOTE: The concepts presented in the application example are, in principle, transferable to the other M262 or PacDrive LMC Eco controllers that support the OPC UA library and supporting the controller firmware released with EcoStruxure Machine Expert version 2.0 or greater.
