# Outputs

Outputs

| Output | Data Type | Description |
| --- | --- | --- |
| Valid | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  FALSE: Execution has not been started or an error has been detected. The values at the outputs are not valid.  TRUE: Execution has been completed without an error detected. The values at the outputs are valid and can be further processed. |
| Busy | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  FALSE: Execution of the function block has not been started or not been terminated.  TRUE: Function block is being executed. |
| Error | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  FALSE: Execution of the function block is running, no error has been detected.  TRUE: An error has been detected in the execution of the function block. |
| Errorstop | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  TRUE: The movement has been canceled by a detected error. |
| Disabled | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  oFALSE: Power stage is enabled.  oTRUE: Power stage is disabled. |
| Stopping | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  TRUE: The function block is being executed or the movement is being stopped. |
| Homing | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  TRUE: The operating mode Homing has been started. |
| StandStill | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  TRUE: The movement hast been stopped. |
| DiscreteMotion | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  TRUE: The operating mode Profile Position has been started. |
| ContinuousMotion | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  TRUE: The operating mode Profile Velocity or Profile Torque has been started. |
| SynchronizedMotion | BOOL | Value range: FALSE, TRUE.  Default value: FALSE.  TRUE: A synchronized movement at a constant velocity is performed. (for example, in the operating mode Electronic Gear) |
