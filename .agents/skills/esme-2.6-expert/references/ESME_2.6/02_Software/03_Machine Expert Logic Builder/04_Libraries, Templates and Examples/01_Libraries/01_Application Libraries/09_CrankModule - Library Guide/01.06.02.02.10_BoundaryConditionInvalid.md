# BoundaryConditionInvalid

BoundaryConditionInvalid

|  |  |
| --- | --- |
| Enumeration name: | BoundaryConditionInvalid |
| Enumeration value: | 34 |
| Description: | The boundary condition is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | When defining the motion profile in iq\_stMultiCamData in the [ST\_MultiCam](../Structures/Structures-9.htm#XREF_D_SE_0088766_1) structure, it is not possible to comply with the set boundary conditions of the segments. | Check the definition of the motion profile and particularly the boundary conditions of the sub-segments.  As an alternative, select different profiles for the sub-segments to be able to comply with all the desired boundary conditions. |
