# MaintenanceDataStorage_2 Function Block

MaintenanceDataStorage\_2 Function Block

The MaintenanceDataStorage\_2 function block is used to record statistical information about the hoist drive and calculates the remaining hours before maintenance, including:

oHours spent by the hoist drive working with the actual load

oRemaining hours that the hoist drive can work with the actual load

oHours 300 class of hoist (amount of hours with more than 300 operations per hour)

oHours 600 class of hoist (amount of hours with more than 600 operations per hour)

For detailed description refer to the [function block description](Monitoring_Data_Storage_Function-13.htm#XREF_D_SE_0021092_7).
