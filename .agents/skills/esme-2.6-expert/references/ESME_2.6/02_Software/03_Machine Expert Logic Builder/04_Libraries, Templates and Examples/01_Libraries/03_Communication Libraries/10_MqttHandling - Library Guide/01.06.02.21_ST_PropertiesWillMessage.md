# ST\_PropertiesWillMessage

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_PropertiesWillMessage specifies the properties associated with the Will message.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| udiWillDelayInterval | UDINT | Specifies the delay before the Will message is published. |
| etPayloadFormatIndicator | [ET\_PayloadFormatIndicator](ET_PayloadForInd-D886A098.html) | Specifies the format of the payload of the Will message. |
| diMessageExpiryInterval | DINT | Specifies the lifetime of the Will message in seconds. It is sent as the publication expiry interval when the MQTT broker publishes the Will message. |
| sContentType | STRING[255] | Specifies the content of the Will message. |
| sResponseTopic | STRING[255] | Specifies the topic name for a response message to the Will message. |
| abyCorrelationData | ARRAY[0..GPL.Gc\_uiMax LengthOfCorrelationData-1]OF BYTE | Specifies the correlation data used to identify the response message. |
| uiCorrelationDataLength | UINT | Specifies the length of the correlation data in bytes. |
| astUserProperties | ARRAY[1..GPL.Gc\_uiMaxNumberOfUserProperties] OF ST\_UserProperty | Specifies the user properties to send to the MQTT broker.  NOTE: A user property with an empty name represents the end of the list. |
