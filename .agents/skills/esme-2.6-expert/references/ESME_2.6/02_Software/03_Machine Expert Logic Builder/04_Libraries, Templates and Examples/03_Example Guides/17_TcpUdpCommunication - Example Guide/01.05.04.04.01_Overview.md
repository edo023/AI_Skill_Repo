# Overview

Overview

The protocols [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) and [UDP](../glossary/glossary.htm#XREF_D_SE_0055638_21) are used for the [socket](../glossary/glossary.htm#XREF_D_SE_0055638_2)-based network communication. The data transmitted via these protocols is assembled based on a specific application protocol. Both [peers](../glossary/glossary.htm#XREF_D_SE_0055638_15) must use the same application protocol for successful communication.
