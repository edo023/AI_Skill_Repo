# SMC\_GWord\_Iterator (FB)

FUNCTION\_BLOCK FINAL SMC\_GWord\_Iterator

This iterator allows to iterate all g-words of a g-sentence. It can be retrieved from the sentence-queue for the first sentence ([SMC\_GSentenceQueue.GetGWordIterator](sPgcZ_mXTWZCf5xiluEOzwuwUdY_getgworditerator.html#spgcz_mxtwzcf5xilueozwuwudy_getgworditerator_smc_gsentencequeue_getgworditerator_meth)), or for any sentence from a sentence iterator ([SMC\_GSentence\_Iterator.GetGWordIterator](UongzXGQ2XmfiOMzQF_ggqEk2gE_getgworditerator.html#uongzxgq2xmfiomzqf_ggqek2ge_getgworditerator_smc_gsentence_iterator_getgworditerator_meth)). For each sentence, a g-expression token iterator ([SMC\_GExpToken\_Iterator](0AOpftHaTjaOBKEDUp2z_aWxY1Y_smc-gexptoken-it.html#0aopfthatjaobkedup2z_awxy1y_smc_gexptoken_it_smc_gexptoken_iterator_fb)) can be created.
