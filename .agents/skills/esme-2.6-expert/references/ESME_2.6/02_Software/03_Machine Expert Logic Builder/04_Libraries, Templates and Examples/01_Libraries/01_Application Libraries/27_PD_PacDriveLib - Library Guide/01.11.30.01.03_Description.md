# Description

Description

This function fulfils the same task as the function [FC\_Poly5Coefficients](Functions_K_to_Z-28.htm#XREF_D_SE_0087591_1). The only difference is that a transition to a standardized X space is made before calculating the polynomial coefficients. This is effected by the following transformation:

P(X) = Pnorm( (X - i\_lrX0) / (i\_lrX1 - i\_lrX0) )

Pnorm is a polynomial with an X space standardized to the interval [0, 1], for which the following holds:

P(i\_lrX0) = Pnorm(0)

P(i\_lrX1) = Pnorm(1)

Compared to [FC\_Poly5Coefficients](Functions_K_to_Z-28.htm#XREF_D_SE_0087591_1), this way of proceeding has the advantage of a reduced sensitivity in relation to rounding errors (in particular if i\_lrX0 is very large). A disadvantage, however, is that, in addition to the polynomial coefficients, the values i\_lrX0 and i\_lrX1 must be stored. This is effected in the extended polynomial coefficient structure [ST\_Poly5NormalizedCoefficients](../Structures/Structures-38.htm#XREF_D_SE_0087782_1).
