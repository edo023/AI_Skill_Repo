# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| lrRefVelocity | LREAL | Current velocity of the master axis. |
| lrRefAcceleration | LREAL | Current acceleration of the master axis. |
| lrRefJerk | LREAL | Current jerk of the master axis. |
| xVelLimitationGlobal | BOOL | TRUE: a velocity limit limits at least one of the slave axes of the master axes velocity. Which following axis (axes) this refers to can be obtained from the actual value of the slave axes. |
| xAccLimitationGlobal | BOOL | TRUE: an acceleration limit limits at least one of the slave axes of the master axes velocity. Which following axis (axes) this refers to can be obtained from the actual value of the slaves. |
| lrCycleTime | LREAL | Period of the last master axes cycle. Only releases a valid value when the cycle time is stable. |
