# Structure: AxisIntfOut

Structure: AxisIntfOut

Structure of parameters related to the axis interface

| Parameter | Data type | Description |
| --- | --- | --- |
| xSync | BOOL | TRUE: Synchronization is active  FALSE: Synchronization is inactive |
| wActSyncGroup | WORD | Number of the synchronization group the axis is assigned to  Range: 1...65535 |
| xDrvFwd | BOOL | Forward direction command  TRUE / FALSE |
| xDrvRev | BOOL | Reverse direction command  TRUE / FALSE |
| wDrvSpdTarg | WORD | Speed reference  Range: 0...6000  Unit: RPM |
| wDrvAcc | WORD | Acceleration ramp time  Range: 1...999  Unit: 0.1 s |
| wDrvDec | WORD | Deceleration ramp time  Range: 1...999  Unit: 0.1 s |
| rEncPosDev | REAL | Position deviation  Range: 1.4 -45...3.4 +38  Unit: m or in. |
| wAlrmId | WORD | Alarm identification  Range: 0...32 |
