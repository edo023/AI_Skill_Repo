# SysFileWrite (FUN)

FUNCTION SysFileWrite : \_\_XWORD

Write number of bytes to the file. File must be opened with [AM\_WRITE](ACCESS_MODE.html#access_mode_access_mode_enum__access_mode_access_mode_am_write) or [AM\_APPEND](ACCESS_MODE.html#access_mode_access_mode_enum__access_mode_access_mode_am_append).

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | SysFileWrite | \_\_XWORD | Number of bytes written to the file. 0=if failed | | Input | hFile | RTS\_IEC\_HANDLE | Handle of the file | | pbyBuffer | POINTER TO BYTE | Pointer to buffer with data to write to file | | ulSize | \_\_XWORD | Number of bytes to write in the file. Must be less or equal the buffer size! | | pResult | POINTER TO RTS\_IEC\_RESULT | Pointer to runtime system error code (see CmpErrors.library) | |
