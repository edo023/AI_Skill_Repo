## Task

Interface for a manual motion of the robot.

## Description

With this interface, the motion parameters for a jogging motion and the jogging motions themselves can be started and stopped again.

## Methods

| Name | Description |
| --- | --- |
| AdvanceToNextConnectedPath | Advancing to the next connected path. |
| GetMotionParameter | Reading the jogging motion parameters. |
| SetMaxAcceleration | Setting the maximum jogging acceleration. |
| SetMaxDeceleration | Setting the maximum jogging deceleration. |
| SetMaxVelocity | Setting the maximum jogging velocity. |
| SetMotionParameter | Setting the jogging motion parameters. |
| SetRamp | Setting the ramp. |
| Start | Starting a jogging motion. |
| Stop | Stopping a jogging motion. |
