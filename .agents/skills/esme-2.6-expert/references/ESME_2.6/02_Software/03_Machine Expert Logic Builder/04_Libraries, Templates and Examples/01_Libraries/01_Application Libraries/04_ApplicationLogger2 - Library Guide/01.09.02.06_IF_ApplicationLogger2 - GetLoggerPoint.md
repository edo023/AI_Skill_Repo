# IF\_ApplicationLogger2 - GetLoggerPoint (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method returns the interface to a registered logger point.

By registration to the Application Logger, the logger points are provided with an identification number.

The identification number of the Application Logger is 1. The other logger points get their identification number according to the order in which they are registered.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiId | UDINT | The identification number of the logger point. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-375A58EE.html#ET_Result-375A58EE) | Provides diagnostic and status information as a numerical value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
