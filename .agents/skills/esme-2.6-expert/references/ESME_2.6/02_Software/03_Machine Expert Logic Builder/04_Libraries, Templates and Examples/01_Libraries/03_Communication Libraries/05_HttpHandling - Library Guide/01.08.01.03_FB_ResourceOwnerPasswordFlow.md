## Functional Description

The function block FB\_ResourceOwnerPasswordFlow stores the credentials used for the authorization using the resource owner password flow. It does not provide any parameters (inputs and outputs) and must not be called in the application. Use the [properties to specify the credentials](PropertiesFBResOwnPw-53C8BFBC.html).

The function block implements the interface [IF\_OAuthCredentials](IF_OAuthCredentials-53CAB4F3.html). To use the OAuth protocol, this interface must be passed to the function block FB\_HttpClient through the [property OAuthCredentials](D-SE-0095571.html).

NOTE: Instantiate the function block in nonvolatile memory to preserve the credentials after a reboot.
