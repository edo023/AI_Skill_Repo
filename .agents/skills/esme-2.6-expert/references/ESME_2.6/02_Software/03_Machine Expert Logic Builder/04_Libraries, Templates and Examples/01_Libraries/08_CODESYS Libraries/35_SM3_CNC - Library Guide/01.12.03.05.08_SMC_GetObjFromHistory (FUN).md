# SMC\_GetObjFromHistory (FUN)

FUNCTION SMC\_GetObjFromHistory : POINTER TO SMC\_GEOINFO

Returns an object that has been deleted but not yet overwritten. For N >= 0, SMC\_GetObjFromHistory(poq, N) = SMC\_GetObj(poq, N). For negative N, the objects from the history are retrieved. (N=-1 gets the newest object from history, smaller N get older elements). The valid range for N is -SMC\_GetHistoryCount(poq) ... SMC\_GetCount(poq) - 1.

task-safety: call from consumer task only
