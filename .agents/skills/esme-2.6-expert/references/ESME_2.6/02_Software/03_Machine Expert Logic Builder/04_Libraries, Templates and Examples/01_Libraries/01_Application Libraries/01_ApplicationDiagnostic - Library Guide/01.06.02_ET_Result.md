# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result contains the library diagnostic results.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Ok | 0 | Success |
| MaxNumberOfChildrenExceeded | 1 | The maximum number of children for this parent is exceeded. |
| InvalidInterface | 2 | The specified interface is not valid. |
| IdAlreadyExists | 3 | Another diagnostic handler with the same ID is already registered at the specified parent. |
| SeverityOutOfRange | 4 | The specified severity is out of range. |
| ReactionLevelOutOfRange | 5 | The specified reaction level is out of range. |
| NotReady | 6 | Not possible to execute this command. |
| MaxNumberOfHandlerExceeded | 7 | The specified group exceeds the number of maximum handlers. |
| MaxNumberOfDiagMessagesExceeded | 8 | The number of the diagnostic handlers of the group exceeds the size of the array. |
| AlreadyInitialized | 9 | The diagnostic handler is already initialized and added to a parent. |
| InvalidId | 10 | The ID of the diagnostic handler is not valid. |
| InvalidName | 11 | The name of the diagnostic handler is not valid. |
| NotFound | 12 | The specified child cannot be found. |
