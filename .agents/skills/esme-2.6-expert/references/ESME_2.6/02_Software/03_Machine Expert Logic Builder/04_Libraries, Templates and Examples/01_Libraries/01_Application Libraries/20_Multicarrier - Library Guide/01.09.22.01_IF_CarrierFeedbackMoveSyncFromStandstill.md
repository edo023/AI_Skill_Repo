|  |  |
| --- | --- |

## Task

Interface with reference and status values for the move command [MoveSyncFromStandstill](IF_MoveSyncPathFromStandstill-5B839E78.html#IF_MoveSyncPathFromStandstill-5B839E78).

## Description

The feedback interface provides reference and status values for a synchronized movement of the carrier with the move command MoveSyncFromStandstill.
