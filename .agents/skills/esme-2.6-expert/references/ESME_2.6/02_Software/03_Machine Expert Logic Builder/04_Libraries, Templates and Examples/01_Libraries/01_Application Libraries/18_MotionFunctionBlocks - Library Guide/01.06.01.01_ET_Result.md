# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

The enumeration ET\_Result is used by all POUs of the library. It contains the values that indicate the result or the status of a POU.
