# Manual

Manual

The operation mode Manual is selected. The slave axes of the infeed distance managed by the module can be activated in jogging mode.

The operation mode is parameterized in the structures [ST\_Manual](../Structures/Structures-5.htm#XREF_D_SE_0080853_1) and [ST\_Main](../Structures/Structures-4.htm#XREF_D_SE_0080851_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [PDL.FB\_Jogging](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087305_1) are being activated.

Via the array [ST\_Manual](../Structures/Structures-5.htm#XREF_D_SE_0080853_1).i\_axBeltSelect, the axes are selected that are to be moved in jogging mode. The first element selects belt one, the second element belt two, etc. Only the elements of those belts are checked that are driven by an axis managed by the module.

The diagnostic messages of all managed slave axes are monitored.

Possible state commands

oi\_xJogForw and i\_xJogBack of the [ST\_Manual](../Structures/Structures-5.htm#XREF_D_SE_0080853_1) structure move the selected axes in positive and in negative direction.

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is enabled.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is disabled.

Possible operation mode commands

o[Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [NoOp](#XREF_D_SE_0080863_4) takes place.

o[Auto](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Automatic](#XREF_D_SE_0080863_6).

o[BrakeRelease](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [BrakeRelease](#XREF_D_SE_0080863_7) takes place.
