# ET\_RobotFeedback - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V3.2.0.0 |

## Description

Enumeration type to address the robot feedback.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| All | 1 | All feedback options that are part of this enumeration type, are selected. |
| ConnectedPathRemainingTime | 2 | The robot feedback property lrRemainingTime of the interface IF\_RobotFeedbackConnectedPath is selected. |
| TrackingRemainingTimePathPositionEnd | 3 | The robot feedback property lrRemainingTimePathPositionEnd of interface IF\_RobotFeedbackTracking is selected. |
