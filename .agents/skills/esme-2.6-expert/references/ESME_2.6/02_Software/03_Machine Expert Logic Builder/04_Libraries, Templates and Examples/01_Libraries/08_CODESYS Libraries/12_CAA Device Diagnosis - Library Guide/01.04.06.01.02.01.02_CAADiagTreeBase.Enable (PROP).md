# CAADiagTreeBase.Enable (PROP)

PROPERTY Enable : BOOL

Set: This property is used for enabling/disabling a node. Changes will take effect after doing a Reconfigure (see [Reconfigure](HUr-HPtekCI-al68iX7szHFsAss_reconfigure1.html#hur_hptekci_al68ix7szhfsass_reconfigure1_reconfigure_fb)).

Get: Returns the current Enable value
