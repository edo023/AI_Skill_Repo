# StartPositionSelectInvalid

StartPositionSelectInvalid

|  |  |
| --- | --- |
| Enumeration name: | StartPositionSelectInvalid |
| Enumeration value: | 142 |
| Description: | The use of StartPositionSelect is not permitted. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The [ST\_MultiCam.iq\_stExt.xStartPositionSelect](../Structures/Structures-10.htm#XREF_D_SE_0077229_1) parameter is set to TRUE. | The module does not support the function. Set the parameter [ST\_MultiCam.iq\_stExt.xStartPositionSelect](../Structures/Structures-10.htm#XREF_D_SE_0077229_1) to FALSE.  Use the operating mode Positioning to move the axis to a position before you change to the operating mode MultiCam. |
