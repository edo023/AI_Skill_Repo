# CmdImpossibleOverwrite

CmdImpossibleOverwrite

|  |  |
| --- | --- |
| Enumeration name: | CmdImpossibleOverwrite |
| Enumeration value: | 3 |
| Description: | The command cannot be overwritten. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | A command was being processed by the module while a new command was sent that was not [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0088728_1).Stop or [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0088728_1).BrakeRelease.  The module is in the operation mode Positioning and is currently executing an [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0088728_1).Start command while it receives a command that is not [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0088728_1).Hold. | Use the command tables of the template to send commands to the module.  Make certain that the module is not processing any commands ([ST\_StandardModuleInterface](../Structures/Structures-8.htm#XREF_D_SE_0088764_1).q\_xCmdActive = FALSE) when you send a new command to the module. |
