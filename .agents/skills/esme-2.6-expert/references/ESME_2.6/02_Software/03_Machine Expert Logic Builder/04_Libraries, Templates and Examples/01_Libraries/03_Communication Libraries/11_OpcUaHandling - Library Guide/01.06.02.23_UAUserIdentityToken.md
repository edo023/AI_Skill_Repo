# UAUserIdentityToken

## Overview

|  |  |
| --- | --- |
| Type: | Structure |

## Description

The structure UAUserIdentityToken is used to provide the information to authenticate at the OPC UA server while establishing connection.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| UserIdentityTokenType | [UAUserIdentityTokenType](D-SE-0099991.html#D-SE-0099991__D-SE-0099991.5) | Identity token to authenticate a user during the creation of a session.  Default value: UAUITT\_Username |
| TokenParam1 | STRING [255] | Parameter for token type:   * UAUITT\_Username: Username * UAUITT\_x509: The given value is ignored. |
| TokenParam2 | STRING [255] | Parameter for token type:   * UAUITT\_Username: Password * UAUITT\_x509: Name of the certificate file   Also refer to [Authentication by X.509 Certificates](D-SE-0099991.html#D-SE-0099991__X509Certificates-C80622FE). |
