# DriveNotReady

DriveNotReady

|  |  |
| --- | --- |
| Enumeration name: | DriveNotReady |
| Enumeration value: | 41 |
| Description: | Drive is not ready. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The drive is not ready to perform motion commands. | The Drive has to be enabled and in position control (AxisState 3) while the function block is called.  The Sercos has to be in state 4 while the function block is called. |
