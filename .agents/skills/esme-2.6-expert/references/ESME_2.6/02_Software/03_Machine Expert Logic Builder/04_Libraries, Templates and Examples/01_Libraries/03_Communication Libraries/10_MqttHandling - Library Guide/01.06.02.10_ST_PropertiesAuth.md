# ST\_PropertiesAuth

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_PropertiesAuth specifies the properties of an AUTH message exchanged between the MQTT client and the MQTT broker.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| sAuthenticationMethod | STRING[255] | Indicates the authentication method used and how to interpret the authentication data.  NOTE: The length of the value is limited to 255 characters. If it exceeds the maximum length, it is truncated and the input i\_xTruncatedProperties is set to TRUE. |
| uiAuthenticationDataLength | UINT | Specifies the length of the authentication data. |
| abyAuthenticationData | ARRAY[0..254] OF BYTE | Indicates the authentication data. |
