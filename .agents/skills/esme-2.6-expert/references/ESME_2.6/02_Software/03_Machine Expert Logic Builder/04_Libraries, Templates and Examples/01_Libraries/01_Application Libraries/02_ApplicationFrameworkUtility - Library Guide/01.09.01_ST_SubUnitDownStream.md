# ST\_SubUnitDownStream

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

The structure ST\_SubUnitDownStream contains the data output of the function block FB\_PackMlSubUnitHandler for a single subunit.

## Structure Elements

| Input | Data type | Description |
| --- | --- | --- |
| etCmd | [PackML.ET\_Cmd](../../../../../api/crossBook?lang=en-US&virtualBookName=PackMLli&topicID=D_SE_0077934) | The PackML command that is to be sent to the subunit. |
| xAligned | BOOL | TRUE: The subunit shows a PackML state that is congruent with the state of the leading unit. FALSE: The subunit shows a PackML state that is incongruent with the state of the leading unit. |
| xActingStateConcluded | BOOL | If the leading unit is in a PackML waiting state, xActingStateConcluded is FALSE.  If the leading unit is in a PackML acting state: FALSE indicates that the function block FB\_PackMlSubUnitHandler sends commands to the subunit or waits until the subunit has reached the target state. TRUE indicates that all necessary steps for the subunit are finished. The Completed signal for the corresponding subunit can be sent to the PackML state machine. |
