# SMC\_ReadNCFile (FB)

FUNCTION\_BLOCK SMC\_ReadNCFile

SMC\_ReadNCFile can read a NC-ASCII-file from the file system of the controller in order to make it available for the [SMC\_NCDecoder](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-ncdecoder.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_ncdecoder_smc_ncdecoder_fb). Thus, a NC program can be read in and decoded at runtime.

**Note**

This function block has been superseeded by [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb) and is kept only for compatibility. It is recommended to use [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb), which has additional features like mathematical expressions or sub programs in g-code
