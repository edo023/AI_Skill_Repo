# SMC\_NCInterpreter (FB)

FUNCTION\_BLOCK FINAL SMC\_NCInterpreter

This function block is used to interpret a CNC program (Din 66025, G-code). The program is converted into a list of [SMC\_GeoInfo](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-geoinfo.html#cah0ife6nckjzp5ef3qpy_filyk_smc_geoinfo_smc_geoinfo_struct) objects and stored in a queue of type [SMC\_OutQueue](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-outqueue.html#cah0ife6nckjzp5ef3qpy_filyk_smc_outqueue_smc_outqueue_fb). In each call, usually one line of the program is interpreted.

This function block should be called in a separate task with a priority lower than the priority of the bus task. It should be called together with the function blocks for reading the G-Code (e.g. [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb)) and for pre-processing the path (e.g. [SMC\_CheckVelocities](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-checkvelocit.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_checkvelocit_smc_checkvelocities_fb)). It can be called multiple times per cycle.

**Note**

* The interpreter must be called in the same task as the function block that fills the SMC\_GSentenceQueue (input sentences). Usually, the sentences are provided by [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb), [SMC\_ReadNCFromStream](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfromst.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfromst_smc_readncfromstream_fb), or [SMC\_NCTokenParser](VKaHy6dlyRAODh6pHpRDreSTG3k_smc-nctokenparse.html#vkahy6dlyraodh6phprdrestg3k_smc_nctokenparse_smc_nctokenparser_fb).
* The maximum number of M-functions in a row is 64. If the G-Code contains more than 64 M-functions in a row, the error SMC\_DEC\_OUTQUEUE\_RAN\_EMPTY occurs. This restriction can be circumvented by inserting a G75 previous to the first M-function of the sequence. G4-elements count as M-functions, too.
