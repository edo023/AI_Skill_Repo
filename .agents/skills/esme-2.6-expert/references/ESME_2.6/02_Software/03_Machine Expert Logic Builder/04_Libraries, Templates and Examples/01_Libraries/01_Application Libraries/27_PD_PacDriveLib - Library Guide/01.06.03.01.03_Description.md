# Description

Description

NOTE: The function block FB\_AsciiWriter only works correctly if the used data structures are saved with Pack\_Mode = 1.

The function block writes the number of variables of the type specified in i\_pbyStructInfo which are entered in i\_iCount from memory at the address i\_dwBufferAdr to a file.

Writing is started with a positive edge at i\_xExecute and stopped after an error is detected or after all data has been written successfully. During writing, no parameter (other than i\_xExecute) may be changed. Also, that memory area which i\_dwBufferAdr points to may not be written to.

The file that is created can be read with any editor.
