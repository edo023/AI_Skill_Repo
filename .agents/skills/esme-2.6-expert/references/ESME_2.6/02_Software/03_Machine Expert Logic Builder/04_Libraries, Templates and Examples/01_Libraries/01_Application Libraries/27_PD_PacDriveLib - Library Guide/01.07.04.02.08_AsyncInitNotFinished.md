# AsyncInitNotFinished

AsyncInitNotFinished

|  |  |
| --- | --- |
| Enumeration name: | AsyncInitNotFinished |
| Enumeration value: | 290 |
| Description: | Init has not been completed yet. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | PDL.G\_ifAsync.Init() method has not been completed yet. | Call up the Init() method before you call up FB\_HomeForce for the first time.  After you have called up the Init method, wait until Init has been executed successfully. (Init.q\_xDone = TRUE) |
