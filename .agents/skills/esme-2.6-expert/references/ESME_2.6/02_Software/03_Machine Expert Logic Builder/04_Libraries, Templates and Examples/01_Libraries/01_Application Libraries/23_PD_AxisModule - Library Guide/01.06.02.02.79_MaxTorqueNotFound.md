# MaxTorqueNotFound

MaxTorqueNotFound

|  |  |
| --- | --- |
| Enumeration name: | MaxTorqueNotFound |
| Enumeration value: | 84 |
| Description: | MaxTorque has not been reached. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.PosDirectionMaxTorque](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1) or  [PDL.ET\_HomeMode.NegDirectionMaxTorque](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).  The stop was not found. | Check the state of the limit stop.  Check the input [ST\_Home.stTorque.i\_lrMaxTorque](../Structures/Structures-5.htm#XREF_D_SE_0077219_1).  Check the [ST\_Home.stTorque.i\_lrMaxTravel](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) input.  Check the axis state. |
