# DriveNotInitialised

DriveNotInitialised

|  |  |
| --- | --- |
| Enumeration name: | DriveNotInitialised |
| Enumeration value: | 27 |
| Description: | The connected drive is not initialized. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The function [FC\_DriveInit](../Functions/Functions-2.htm#XREF_D_SE_0090869_1) was not activated for the drive at the input Axis. | Before using the POU, the drive must be initialized using the function [FC\_DriveInit](../Functions/Functions-2.htm#XREF_D_SE_0090869_1). |
