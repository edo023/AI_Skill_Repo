# rTrolRestrMarg

rTrolRestrMarg

Defines a margin of the restricted area for the trolley in meters. If the point corresponding to the position of the trolley moves into the restricted area, but still within the distance corresponding to this margin from the border, it is possible for the crane operator to move the crane out of the restricted area without usage of the i\_xBlockOvrrd input.

If the crane is deeper within the restricted area than the distance of this margin, the crane can move only after the i\_xBlockOvrrd input is set to TRUE.
