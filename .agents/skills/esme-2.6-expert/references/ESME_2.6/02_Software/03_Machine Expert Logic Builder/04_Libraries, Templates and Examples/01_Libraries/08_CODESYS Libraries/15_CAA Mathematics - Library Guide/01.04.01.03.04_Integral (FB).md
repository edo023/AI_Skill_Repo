# Integral (FB)

FUNCTION\_BLOCK Integral

This function block will approximate the integral function of the fuction ![](images/945988b0727345eea288874674ffa4a4.jp.jpg) over the time interval between the first function call ![](images/449ce1eb72a74523b5b75e55479257fc.jp.jpg) and the actual time ![](images/8384c07d88cf4bbd8c487462c1df1ba8.jp.jpg): ![](images/78cf112e64b34c5ebdf249ce041fc7e0.jp.jpg). The size of the time intervals ![](images/05b097ee705945a2be21fef329202601.jp.jpg) are integers and measured in micro seconds. The approximation is carried out by use of the explicit (![](images/dc0ba954740a497bae37270867091f98.jp.jpg)) resp. implicit (![](images/fc23d67d22b64643a00186f5e77d79e2.jp.jpg)) Euler method:

![](images/fde78e336d2443c5b24f68bfa21fd3c8.jp.jpg)
