# ET\_DataBlockElement

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.2.1.0 |

## Description

The enumeration ET\_DataBlockElement defines the Sercos parameter element which is to be read (written).

NOTE: As of version V1.4.0.0, this data type is an alias of the data type SERC\_CT.ET\_DataBlockElement defined in the controller independent common library SercosCommunicationCommonTypes.

| Name | Value | Description |
| --- | --- | --- |
| Name | 2 | IDN name (up to 240 bytes) |
| Attribute | 3 | Contains information on write protection, decimal places, data type, and display format, procedure command, data length. |
| Unit | 4 | Unit (UTF-8-coded) |
| MinimumValue | 5 | Smallest numerical value for the operation data |
| MaximumValue | 6 | Largest numerical value for the operation data |
| OperationData | 7 | Operation data of an IDN with a fixed length  Supported data length are 2,4 or 8 |
| OperationDataVarLength | 8 | Operation data of an IDN with variable length  Supported data length are 1,2,4,8,9 ... 200 |
