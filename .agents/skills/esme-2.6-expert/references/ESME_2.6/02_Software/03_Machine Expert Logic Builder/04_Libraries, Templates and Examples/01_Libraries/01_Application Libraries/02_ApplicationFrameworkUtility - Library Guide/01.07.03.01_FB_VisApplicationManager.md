# FB\_VisApplicationManager - GetVisConnectorReferences (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.1.0.0 |

## Description

The method GetVisConnectorReferences is used for transferring pointer arrays to the visualization manager.

For more information, refer to [Providing Data for Visualization](ProvidingDataForVis-DCA4DB61.html).

## Inputs/Outputs

| Name | Data type | Description |
| --- | --- | --- |
| iq\_apbyVisConnectorReferences | ARRAY[\*] OF POINTER TO BYTE | A variable that is connected to the input iq\_apbyVisConnectorReferences maps the addresses of the function blocks which were assigned to an array by the method [UpdateVisConnectorReference](VisAppManager-UpdateVis-DC0B31C4.html#VisAppManager-UpdateVis-DC0B31C4). |
| iq\_axVisConnectorValid | ARRAY[\*] OF BOOL | A variable that is connected to the input iq\_axVisConnectorValid indicates whether the address of a function block in the array of the input iq\_apbyVisConnectorReferences is valid. |
