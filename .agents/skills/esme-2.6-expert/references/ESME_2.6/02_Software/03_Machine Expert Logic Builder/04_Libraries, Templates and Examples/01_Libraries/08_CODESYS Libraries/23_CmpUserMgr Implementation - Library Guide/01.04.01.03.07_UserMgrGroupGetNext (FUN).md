# UserMgrGroupGetNext (FUN)

FUNCTION UserMgrGroupGetNext : RTS\_IEC\_HANDLE

Iteration interface to get next user group. Must be started with UserMgrGroupGetFirst()

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | UserMgrGroupGetNext | RTS\_IEC\_HANDLE | Handle to the next user group or RTS\_INVALID\_HANDLE if failed | | Input | hUser | RTS\_IEC\_HANDLE | Handle to authorized user | | hPrevGroup | RTS\_IEC\_HANDLE | Handle to the previous user group retrieved via UserMgrGroupGetFirst() or subsequent calls of UserMgrGroupGetNext() | | Output | Result | RTS\_IEC\_RESULT | | Error code | * ERR\_OK: Next user group available * ERR\_NO\_OBJECT: No next user group available * ERR\_NO\_ACCESS\_RIGHTS: User not authorized | | |
