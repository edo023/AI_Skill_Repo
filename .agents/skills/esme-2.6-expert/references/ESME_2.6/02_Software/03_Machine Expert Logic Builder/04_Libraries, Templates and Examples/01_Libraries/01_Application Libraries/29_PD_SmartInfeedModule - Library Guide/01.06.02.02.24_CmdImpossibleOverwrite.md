# CmdImpossibleOverwrite

CmdImpossibleOverwrite

|  |  |
| --- | --- |
| Enumeration name: | CmdImpossibleOverwrite |
| Enumeration value: | 24 |
| Description: | The command cannot be overwritten. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | A command was being processed by the module while a new command has been sent that was not [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1).Stop or [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1).BrakeRelease. | Use the command tables of the template to send commands to the module.  Make sure that the module is not processing any commands (ST\_StandardModuleInterface.q\_xCmdActive = FALSE) when you send a new command to the module. |
