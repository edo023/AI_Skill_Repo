# GetNextNode (FUN)

FUNCTION GetNextNode : [INode](qG5KZxShRzJP0Witj1Yz7sQso84_inode.html#qg5kzxshrzjp0witj1yz7sqso84_inode_inode_itf)

The function implements a depth-first search. Nodes will be returned in top to bottom order of the device tree

**Example**

Following example code counts all nodes in the device tree.

```
PROGRAM PLC_PRG
VAR
    itfNode : DED.INode;
    uiNodes : UINT;
END_VAR

itfNode := DED.GetRoot();
REPEAT
     itfNode := DED.GetNextNode(itfNode);
   uiNodes := uiNodes + 1;
UNTIL
   itfNode = 0
END_REPEAT
```
