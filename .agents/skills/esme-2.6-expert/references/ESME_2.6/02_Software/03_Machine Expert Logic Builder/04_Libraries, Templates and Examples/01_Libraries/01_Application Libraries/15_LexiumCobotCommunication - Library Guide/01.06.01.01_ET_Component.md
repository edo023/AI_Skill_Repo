# ET\_Component

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

This enumeration type is used to address a specific component of the Lexium Cobot.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Joint1 | 0 | Joint 1 of the Lexium Cobot arm is addressed. |
| Joint2 | 1 | Joint 2 of the Lexium Cobot arm is addressed. |
| Joint3 | 2 | Joint 3 of the Lexium Cobot arm is addressed. |
| Joint4 | 3 | Joint 4 of the Lexium Cobot arm is addressed. |
| Joint5 | 4 | Joint 5 of the Lexium Cobot arm is addressed. |
| Joint6 | 5 | Joint 6 of the Lexium Cobot arm is addressed. |
| PositionX | 10 | The cartesian component X of the Lexium Cobot is addressed. |
| PositionY | 11 | The cartesian component Y of the Lexium Cobot is addressed. |
| PositionZ | 12 | The cartesian component Z of the Lexium Cobot is addressed. |
| OrientationX | 13 | The rotational component RX of the orientation of the Lexium Cobot is addressed. |
| OrientationY | 14 | The rotational component RY of the orientation of the Lexium Cobot is addressed. |
| OrientationZ | 15 | The rotational component RZ of the orientation of the Lexium Cobot is addressed. |
