# InputExceedsGplLimits

InputExceedsGplLimits

|  |  |
| --- | --- |
| Enumeration name: | InputExceedsGplLimits |
| Enumeration value: | 31 |
| Description: | The input exceeds the limit set in [GPL](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0090712_1). |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | Number of existing paths and/or vehicles used in the system exceeds the limit defined in the [GPL](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0090712_1). | Verify whether [ST\_ControllerParameter](../Structures/Structures-2.htm#XREF_D_SE_0090700_1).i\_udiMaxNumberOfPaths is smaller or equal to [GPL](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0090712_1).Gc\_udiMaxNumberOfPaths and [ST\_ControllerParameter](../Structures/Structures-2.htm#XREF_D_SE_0090700_1).i\_udiMaxNumberOfVehicles is smaller or equal to [GPL](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0090712_1).Gc\_udiMaxNumberOfVehicles.  If not, increase the respective values in the [GPL](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0090712_1). |
