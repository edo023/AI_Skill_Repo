# Moving To the Reference Position

Moving To the Reference Position

This start mode is for dancer mechanics. It brings the dancer into the middle position while the feeder stands still. The unwinder moves the bobbin to tense the foil. If the dancer does not react, then a diagnostic message occurs. When the dancer reaches the middle position the start movement is finished.

The user can also move the dancer to the minimum or maximum positions first, to measure the radius of the bobbin (xRadiusMeasurement=TRUE). First the dancer is moved to the maximum position. The positions of the bobbin and the dancer are stored. Afterwards the dancer moves to the minimum position and the dancer and bobbin positions are stored again. Now the radius of the bobbin is calculated and set as the start radius for the automatic mode and the dancer is moved to the middle position.
