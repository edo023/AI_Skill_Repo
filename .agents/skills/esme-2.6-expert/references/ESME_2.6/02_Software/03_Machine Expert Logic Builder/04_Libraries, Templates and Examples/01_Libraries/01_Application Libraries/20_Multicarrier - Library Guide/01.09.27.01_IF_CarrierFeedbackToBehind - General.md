# IF\_CarrierFeedbackToBehind - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Task

Interface with feedback for the carrier behind.

## Description

The feedback interface provides status values from the carrier behind the selected carrier.

(For more information on the carrier positions, refer to the [general description](IntroMC_MovDir-10BB46E9.html#IntroMC_MovDir-10BB46E9__InFrontBehind-10BB584B) of a Lexium™ MC multi carrier track.)

## Properties

| Property | Data type | Accessing | Description |
| --- | --- | --- | --- |
| lrDistanceToCarrierBehind | LREAL | Read | Indicates the path distance of the selected carrier to the carrier behind (from the center point of one carrier to the center point of the other carrier). |
| lrGapToCarrierBehind | LREAL | Read | Indicates the gap of the selected carrier to the carrier behind. |
| lrRefAccelerationCarrierBehind | LREAL | Read | Indicates the reference acceleration of the carrier behind. |
| lrRefMinGapFromCarrierBehind | LREAL | Read | Indicates the reference minimum gap of the carrier behind to the selected carrier. |
| lrRefMinGapToCarrierBehind | LREAL | Read | Indicates the reference minimum gap of the selected carrier to the carrier behind. For more information, refer to the method [SetRefMinGapToCarrierBehind](IF_Motion-SetRefMinGapToCarrierBehi-534E0D23.html#IF_Motion-SetRefMinGapToCarrierBehi-534E0D23). |
| lrRefPositionCarrierBehind | LREAL | Read | Indicates the reference position of the carrier behind. |
| lrRefVelocityCarrierBehind | LREAL | Read | Indicates the reference velocity of the carrier behind. |
| udiCarrierIndexCarrierBehind | UDINT | Read | Indicates the carrier index of the carrier behind. |
| xJobActiveCarrierBehind | BOOL | Read | Indicates TRUE if a job has been assigned to the carrier behind and if this job is active. |
| etCurrentMoveCommandFromCarrierBehind | ET\_MoveCommand | Read | Indicates the active move command for the carrier behind. For more information, refer to the enumeration [ET\_MoveCommand](ET_MoveComm-DA370FC4.html#ET_MoveComm-DA370FC4). |
| ifFeedbackConfigurationFromCarrierBehind | [IF\_CarrierFeedbackConfiguration](CarrFeedbConf-E1D3F75B.html#CarrFeedbConf-E1D3F75B) | Read | Access to the feedback interface IF\_CarrierFeedbackConfiguration that provides status values for the configured parameters of the carrier behind. |
