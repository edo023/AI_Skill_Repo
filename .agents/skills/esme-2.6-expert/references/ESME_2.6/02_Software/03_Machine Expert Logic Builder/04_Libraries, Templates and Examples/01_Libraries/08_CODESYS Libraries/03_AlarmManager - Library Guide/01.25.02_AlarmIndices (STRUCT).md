# AlarmIndices (STRUCT)

TYPE AlarmIndices : STRUCT

This structure contains indices the alarm table in the HMI needs.

| InOut: | | Name | Type | Comment | | --- | --- | --- | | m\_iAlarmGroupIndex | INT | The index of the alarm group within all alarm groups the alarm belongs to | | m\_iAlarmIndex | INT | The index of the alarm within all alarms of the alarm group | | m\_iCallerIndex | INT | The index the alarm table uses internally for storing the alarm | |
