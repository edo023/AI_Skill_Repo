# ParentNotRegistered

ParentNotRegistered

|  |  |
| --- | --- |
| Enumeration name: | ParentNotRegistered |
| Enumeration value: | 27 |
| Description: | Parent is not registered. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The logger point transferred at the input i\_ifParent is not registered in the Application Logger. | The logger point transferred at the input i\_ifParent must be registered in the Application Logger first by calling his RegisterLoggerPoint method. |
