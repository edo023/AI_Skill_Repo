# MaxNrOfProductsReached

MaxNrOfProductsReached

|  |  |
| --- | --- |
| Enumeration name: | MaxNrOfProductsReached |
| Enumeration value: | 68 |
| Description: | The maximum number of manageable products has been reached. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | There was an attempt to add more than the maximum number of products to the infeed distance. | At no time more than [Gc\_udiMaxNumberOfProducts](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0080374_1) products may be located on the infeed distance.  The global parameter [Gc\_udiMaxNumberOfProducts](../Global_Elements/Global_Elements-3.htm#XREF_D_SE_0080374_1) must be set to a value that is greater than the maximum number of products on the infeed distance. |
