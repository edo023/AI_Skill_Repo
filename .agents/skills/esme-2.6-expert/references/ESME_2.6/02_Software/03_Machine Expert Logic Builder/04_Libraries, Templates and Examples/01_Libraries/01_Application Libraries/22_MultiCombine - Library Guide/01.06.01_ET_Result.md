# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | UDINT |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result contains the values that indicate the result of operations executed by the POUs of this library.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Ok | 0 | No error detected. |
| UnexpectedReturnValue | 1 | Indeterminable return value from system.  Contact your Schneider Electric representative. |
| AxisInvalid | 2 | No axis is specified for the input Slave or specified axis does not support the required function block. |
| NoBusCommunication | 3 | Communication via the motion bus is interrupted. |
| BufferModeInvalid | 4 | A buffer mode other than Aborting is active. The function block must be started with the buffer mode set to Aborting. |
| AxisInErrorStop | 5 | The function block was re-executed while already active, or the subordinate axis is in the operating state ErrorStop.  Function block cannot be executed because an axis error has been detected and the axis is in operating state ErrorStop.  Verify that the axis is not in the operating state ErrorStop when attempting to start a new function block. |
| JobInvalid | 6 | An attempt has been made to use bundling with a PLCO function block that is not supported by the function block.  Supported PLCO function blocks are MC\_MoveRelative, MC\_MoveAbsolute, MC\_Halt, MC\_MoveVelocity, MC\_CamIn (RampIn), MC\_Stop. |
| AxisNotInStandstill | 7 | The axis was not in operating state Standstill when the function block was attempted to start.  Verify that the axis is in operating mode Standstill before executing this function block. |
| NotSupportedWithFeedbackAxis | 8 | An attempt has been made to execute the function block with an encoder axis.  The function block does not support encoder axes. |
| MasterSlaveCascadeFormsLoop | 9 | Master axis/subordinate axis function blocks create a loop (the master itself as subordinate axis, or a following subordinate axis with the initial master as subordinate axis).  Resolve the loop. |
| OperationModeInvalid | 10 | The subordinate axis is in an operating mode that does not allow synchronized motion.  Select operating mode Position or Velocity. |
| NotSupportedWithLogicalEncoder | 11 | An attempt has been made to execute the function block with a logical encoder axis.  The function block does not support logical encoder axes. |
| InvalidChannelNumber | 12 | The maximum number of channels is exceeded.  Reduce the number of channels. Refer to [FB\_MultiCombine](FB_MultiCombine-58411E70.html) |
| InvalidChannelState | 13 | The operating state of the channel is invalid.  The function block can only be started if the operating state of the channel is Standstill or Disabled. |
