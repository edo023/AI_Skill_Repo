## Task

Setting the working plane of a two-dimensional robot for calculating spline points for an elliptic spline.

## Description

While configuring a two-dimensional robot (IF\_RobotConfiguration.Cartesian2Ax(…), IF\_RobotConfiguration.Delta2Ax(…), etc.), you can define the working plane of the robot. This working plane must also be configured with this method for an elliptic spline to help avoid the diagnostic message q\_etDiag = GD.ET\_Diag.InputParameterInvalid / q\_etDiagExt = ET\_DiagExt.TargetInvalid on sending the motion job.
