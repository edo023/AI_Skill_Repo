# ReduceSegment (FUN)

FUNCTION ReduceSegment : UDINT

This fuction reduces a segment to get a segment with only complete utf8 runes in it.

It returns the number of the necessary bytes for the utf8 runes.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | ReduceSegment | UDINT |  | | Input | pbyData | POINTER TO BYTE | Pointer to the first byte of segment | | udiSize | UDINT | Size of the segment in bytes | |
