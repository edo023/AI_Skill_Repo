# Input Pin Description

Input Pin Description

This function block requires metric input units.

| Input | Data Type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block. |
| i\_rSenIputActl | REAL | Actual torque input from the drive or load cell input.  Range: -30000...+100000  Scaling/Unit: 0.01% for torque, 1 kg for load cell  Refer to detailed description below this table. |
| i\_rDrvTrqMax | REAL | Drive torque at 100% nominal load of the crane.  Range: 0...30000  Scaling/Unit: 0.01%  Refer to detailed description below this table. |
| i\_iDrvSpdActl | INT | Actual drive speed.  Range: -6000...+6000  Scaling/Unit: 1 RPM |
| i\_wDrvSpdRef | WORD | Target speed to the drive.  Range: 0...6000  Scaling/Unit: 1 RPM |
| i\_wEncPos | WORD | Encoder input from the drive.  Range: 0...65535 |
| i\_stOVLD | OVLD | Refer to the [Structured Variable Description](Overload_EN15011-9.htm#XREF_D_SE_0020178_1). |
