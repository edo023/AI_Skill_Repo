# Functional Description

Functional Description

The HoistPositionSync\_2 function block for industrial cranes is designed to synchronize positions of 2 axes for simultaneous handling of large objects. It can synchronize axes with identical or different motors, gears, and encoders. The block retains information about positions of both axes and synchronization status when the crane is switched off. This means that the synchronization can continue after a power cycle of the crane.
