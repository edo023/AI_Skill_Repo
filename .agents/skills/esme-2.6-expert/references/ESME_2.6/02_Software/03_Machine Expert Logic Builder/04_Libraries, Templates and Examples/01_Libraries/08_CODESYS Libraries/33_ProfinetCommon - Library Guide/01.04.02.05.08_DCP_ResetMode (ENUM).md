# DCP\_ResetMode (ENUM)

TYPE DCP\_ResetMode :

The DCP-Reset Mode defines which data is cleared from the device.

| Attributes: | qualified\_only |
| InOut: | | Name | Initial | Comment | | --- | --- | --- | | FactoryReset | 0 | Default, reset all stored data | | ResetApplicationData | 1 | Reset data which have been stored permanent in submodules and modules to factory values (e.g. I&M1-3). | | ResetCommunicationParameter | 2 | All parameters active for the interface or the ports and the ARs shall be set to the default values and if permanently stored reset (e.g. Stationname, IP-Suite, Port-Data) | | ResetEngineeringParameter | 3 | Reset engineering parameters, e.g. an application program or configuration loaded by an engineering system shall be reset | | ResetAllStoredData | 4 | ResetApplicationData + ResetCommunicationParameter + ResetEngineeringParameter | | ResetDevice | 8 | Same as ResetAllStoredData applied to ALL interfaces | | ResetAndRestoreData | 9 | Reset installed software revisions to factory images. | |
