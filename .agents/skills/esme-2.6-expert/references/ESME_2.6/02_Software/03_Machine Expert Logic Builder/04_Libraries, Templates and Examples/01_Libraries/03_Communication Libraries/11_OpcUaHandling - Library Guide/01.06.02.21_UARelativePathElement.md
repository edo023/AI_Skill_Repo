## Description

The structure UARelativePathElement is used to provide information about a relative path element.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| ReferenceTypeID | [UANodeID](D-SE-0099967.html#D-SE-0099967__D-SE-0099967.4) | Identifier for the reference type. |
| IsInverse | BOOL | If TRUE, the inverse references will be evaluated. |
| IncludeSubtypes | BOOL | Defines whether subtypes of the reference type should be returned.  Default value: TRUE |
| TargetName | [UAQualifiedName](D-SE-0100082.html) | Defines the name of the target element. |
