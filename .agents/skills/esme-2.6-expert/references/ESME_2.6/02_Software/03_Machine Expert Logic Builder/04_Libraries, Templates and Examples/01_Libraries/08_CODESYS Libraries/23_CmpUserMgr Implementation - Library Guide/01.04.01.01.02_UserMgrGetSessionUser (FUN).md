# UserMgrGetSessionUser (FUN)

FUNCTION UserMgrGetSessionUser : RTS\_IEC\_HANDLE

Retrieves the user handle assigned to a CODESYS online session.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | UserMgrGetSessionUser | RTS\_IEC\_HANDLE | Handle of the user or RTS\_INVALID\_HANDLE if not available. | | Input | sessionId | UDINT | Session id of which the user handle should be read. | | pResult | POINTER TO RTS\_IEC\_RESULT | | Pointer to error code. One of: | * ERR\_OK: User is available * ERR\_NOTINITIALIZED: The session management is not initialized * ERR\_NO\_OBJECT: The session is not available | | |
