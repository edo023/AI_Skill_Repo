# MaxDistanceRange

MaxDistanceRange

|  |  |
| --- | --- |
| Enumeration name: | MaxDistanceRange |
| Enumeration value: | 11 |
| Description: | MaxDistance is outside of the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode ET\_OpMode.Manual has been selected and ST\_Manual.i\_lrMaxDistance contains an invalid value | ST\_Manual.i\_lrMaxDistance has to be greater than 0 |
