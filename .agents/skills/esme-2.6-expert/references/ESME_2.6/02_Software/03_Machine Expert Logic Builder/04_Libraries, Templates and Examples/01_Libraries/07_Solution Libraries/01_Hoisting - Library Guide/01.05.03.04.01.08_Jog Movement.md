# Jog Movement

Jog Movement

Jog movement is designed to manually change a position of the axis. If configured (i\_stJogPara.diJogDist>0), the axis performs a defined distance movement at a defined speed after a rising edge on a Jog command input (for example, one motor revolution). After this movement, the FB waits for a defined period (i\_stJogPara.wJogDelay) and then, if the Jog command is still present, starts a continuous movement until the Jog command is set to FALSE or a limit switch is reached.
