# DriveInvalid

DriveInvalid

|  |  |
| --- | --- |
| Enumeration name: | DriveInvalid |
| Enumeration value: | 26 |
| Description: | The connected drive is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | No drive has been applied at the input i\_ifAxis. | A valid drive must be transferred to the input i\_ifAxis. |
| - | Too many axes were applied. | Increase the parameter constant Gc\_uiMaxNumberOfDrives. |
