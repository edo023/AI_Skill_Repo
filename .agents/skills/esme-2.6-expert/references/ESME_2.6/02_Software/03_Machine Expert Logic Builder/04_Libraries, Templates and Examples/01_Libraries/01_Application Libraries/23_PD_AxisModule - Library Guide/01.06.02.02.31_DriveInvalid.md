# DriveInvalid

DriveInvalid

|  |  |
| --- | --- |
| Enumeration name: | DriveInvalid |
| Enumeration value: | 10 |
| Description: | The connected drive is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | No valid drive was applied at the [ST\_Main.i\_ifDrive](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) input. | A valid drive must be transferred to the [ST\_Main.i\_ifDrive](../Structures/Structures-7.htm#XREF_D_SE_0077223_1) input. |
| - | The connected drive does not support all required functionalities. | The q\_sMsg output of the exception shows which functionalities are not supported by the drive.  Use a drive that supports all required functionalities. |
