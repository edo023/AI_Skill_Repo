## Task

Method to process and/or modify an incoming diagnostic message.

## Description

The interface method ConfigDiagMsgAndReactionLevel is called by the instance of the FB\_DiagnosticHandler in the event a new diagnostic message is set.

All information about the incoming diagnostic message is provided with the method parameters and can be processed within the method. In addition, it is possible to modify the diagnostic message itself and the associated reaction level.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifSource | IF\_DiagHandlerIdentification | Indicates the source reporting the diagnostic message. If it is 0, the diagnostic messages is set through the methods SetDiagMsgAdvisory and/or SetDiagMsgError. |
| i\_ifRootSource | IF\_DiagHandlerIdentification | Indicates the root source the diagnostic message is set. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stDiagMsg | ST\_DiagMessageError | Indicates the incoming diagnostic message.  You can adapt it to meet your requirements. |
| iq\_uiReactionLevel | UINT | Indicates the proposed reaction level according to the mapping table.  You can adapt it to meet your requirements.  NOTE: Only the diagnostic messages of type error can trigger a reaction level > 0. If you change this to an invalid value, a diagnostic message is returned or the origin value is applied. |
