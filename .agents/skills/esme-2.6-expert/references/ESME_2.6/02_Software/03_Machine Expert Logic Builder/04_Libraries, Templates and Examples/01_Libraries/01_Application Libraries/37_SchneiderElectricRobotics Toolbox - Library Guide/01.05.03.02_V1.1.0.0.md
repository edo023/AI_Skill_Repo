## New Functions

* FB\_RandomPoseGenerator

  The function block provides several methods for the generation of a random Cartesian pose that is typically employed for the simulation of a random flow of targets (for example, products) for a robot.

  Methods to parametrize the generation are:

  + SetPose(…)
  + SetPoseOrientationList(…)
  + SetPoseInPlane(…)
  + SetPoseInPlaneRotationList(…)

  By calling the method Generate(…), a random Cartesian pose is generated.

  Methods to retrieve the set values are:

  + GetPose(…)
  + GetPoseOrientationList(…)
  + GetPoseInPlane(…)
  + GetPoseInPlaneRotationList(…)
* FB\_RandomTargetsGenerator

  The function block provides several methods for the generation of a random list of targets for a robot.

  Methods to parametrize the generation are:

  + SetTargetsInPlane(…)
  + SetTargetsInPlaneRotationList(…)
  + SetTargetsMatrixInPlane(…)
  + SetTargetsMatrixInPlaneRotationList(…)

  By calling the method Generate(…), a random list of targets is generated.

  Methods to retrieve the set values are:

  + GetTargetsInPlane(…)
  + GetTargetsInPlaneRotationList(…)
  + GetTargetsMatrixInPlane(…)
  + GetTargetsMatrixInPlaneRotationList(…)
