# AdditionalModeException

AdditionalModeException

|  |  |
| --- | --- |
| Enumeration name: | AdditionalModeException |
| Enumeration value: | 67 |
| Description: | The external POU reports an exception. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The activated external POU reports an exception via the i\_etDiag diagnostic input of the [ST\_AdditionalMode](../../../../../../api/crossBook?lang=en-US&virtualBookName=Structures/Structures-2.htm#XREF_D_SE_0077213_1) structure by transferring a value unequal to [GD.ET\_Diag.Ok](../../../PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1). | Find out why the external POU reports an exception. The information of the diagnostic inputs i\_udiDiagExt, i\_sDiagExt and i\_sMsg at the time of the exception is contained in the diagnostic message of the module.  Ensure that the value [GD.ET\_Diag.Ok](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228_1) is applied at the i\_etDiag diagnostic input of the [ST\_AdditionalMode](../Structures/Structures-2.htm#XREF_D_SE_0077213_1) structure when the external POU does not report an error. |
