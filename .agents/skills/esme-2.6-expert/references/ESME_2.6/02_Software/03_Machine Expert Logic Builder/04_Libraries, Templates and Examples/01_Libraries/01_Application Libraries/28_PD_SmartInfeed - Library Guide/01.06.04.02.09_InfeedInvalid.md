InfeedInvalid

|  |  |
| --- | --- |
| Enumeration name: | InfeedInvalid |
| Enumeration value: | 94 |
| Description: | An invalid infeed POU has been transferred. |

| Issue | Cause | Solution |
| --- | --- | --- |
