# ET\_MqttVersion

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V2.4.4.0 |

## Description

The enumeration ET\_MqttVersion is used to specify the version of the MQTT prorocol when establishing a connection from the MQTT client to the MQTT broker.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| V3\_1\_1 | 0 | MQTT standard version V3.1.1 is used for the connection from the MQTT client to the MQTT broker. |
| V5\_0 | 1 | MQTT standard version V5.0 is used for the connection from the MQTT client to the MQTT broker. |
