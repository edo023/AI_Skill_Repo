## Task

Interface with reference and status values for the move command [MovePosAndSync](IF_MovePosAndSync-4526BC87.html#IF_MovePosAndSync-4526BC87).

## Description

The feedback interface provides reference and status values for a synchronized movement of the carrier with the move command MovePosAndSync on channel B.
