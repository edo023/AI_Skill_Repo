# SMC\_TranslateQueue3D (FB)

FUNCTION\_BLOCK SMC\_TranslateQueue3D

This function will translate the path stored in poqDataIn to the vector given by vec, which is of structure type SMC\_VECTOR3D .

The input variable poqDataIn is a pointer to the structure [SMC\_OUTQUEUE](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-outqueue.html#cah0ife6nckjzp5ef3qpy_filyk_smc_outqueue_smc_outqueue_fb), which describes the path to get rotated. The initialization value FALSE of input variable bEnable will bar the rotation of the path. As soon as bEnable is set to TRUE all [SMC\_GEOINFO](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-geoinfo.html#cah0ife6nckjzp5ef3qpy_filyk_smc_geoinfo_smc_geoinfo_struct) objects found in poqDataIn will be processed. When bEnable is reset to FALSE, the module will not execute any further modifications.

The initialization of input variable bReset with FALSE effects, that the [SMC\_GEOINFO](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-geoinfo.html#cah0ife6nckjzp5ef3qpy_filyk_smc_geoinfo_smc_geoinfo_struct) objects currently found in poqDataIn will not get rotated, but only those that will be added.

As, during the processing, a sudden change of the characteristic parameter vec of the translation might cause the path to be broken, modifications to the corresponding input will not be taken into account, until [SMC\_OUTQUEUE](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-outqueue.html#cah0ife6nckjzp5ef3qpy_filyk_smc_outqueue_smc_outqueue_fb) will be empty at poqDataIn or until a Reset (bReset) will be done.
