## Task

Triggers the diagnostic quit command on the listener.

## Description

The method DiagQuit is used to trigger the diagnostic quit command on the listener. If the listener is registered to a diagnostic handler, the method is called by the diagnostic handler as soon as a diagnostic quit command at the diagnostic handler is triggered.
