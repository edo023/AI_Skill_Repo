# StrFindW (FUN)

FUNCTION StrFindW : INT

Does a case sensitive search to find |pst2| inside of |pst1|

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | StrFindW | INT | * | -1 in case of invalid input (e.g. a null pointer) | + 0 in case |pst2| is not found + otherwise the position relative to 1 of |pst2| within |pst1| | | | Input | pst1 | POINTER TO WSTRING(255) | Pointer to the string to look in | | pst2 | POINTER TO WSTRING(255) | Pointer to the string to look for | | uiSearchStart | UINT | A position relative to 1, where the search starts | |
