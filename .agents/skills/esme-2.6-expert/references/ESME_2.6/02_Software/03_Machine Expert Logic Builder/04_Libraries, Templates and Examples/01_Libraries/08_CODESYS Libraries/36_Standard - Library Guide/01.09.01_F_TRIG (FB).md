# F\_TRIG (FB)

FUNCTION\_BLOCK F\_TRIG

Detects a falling edge of a boolean signal

```
(* Example declaration *)
FTRIGInst : F_TRIG ;

(* Example in ST *)
FTRIGInst(CLK := VarBOOL1);
VarBOOL2 := FTRIGInst.Q;
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | CLK | BOOL | Boolean signal to be checked | | Output | Q | BOOL | TRUE: Falling edge detected | |
