# Interface

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifMaster | SystemConfigurationItf.IF\_Master | The master object e.g. a master axis. |
| i\_ifDrive | SystemConfigurationItf.IF\_Drive | Symbolic name of the axis to initialize |
| i\_lrMotionActiveVelLimit | LREAL | Velocity at which the axis is considered moving in units/sec |
| i\_etMotionActiveVelType | ET\_MotionActiveVelType | The velocity mode that is to be monitored for i\_lrMotionActiveVelLimit. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| i\_lencLogicalEncoder | L\_Enc | Symbolic name of the associated logical encoder |
| iq\_stAxisModuleItf | ST\_ModuleInterface | The axis interface structure of the associated axis |
