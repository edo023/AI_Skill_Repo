# Prepare Mode

## Overview

In the template operation mode Prepare, the carriers are moved into the first station and they are added to the first station in the physical order. If the Lexium™ MC12 multi carrier is virtual, a setpos motion is executed on the carrier. If a Smart Carrier is present (the variable [GVL\_Project.Gc\_xSmartCarrier](Add-GVLs-2B9A08D6.html#Add-GVLs-2B9A08D6__GVL_ProjectConstantVar-311A43D2) is set to TRUE), the pairing process is started and the carriers are paired again.
