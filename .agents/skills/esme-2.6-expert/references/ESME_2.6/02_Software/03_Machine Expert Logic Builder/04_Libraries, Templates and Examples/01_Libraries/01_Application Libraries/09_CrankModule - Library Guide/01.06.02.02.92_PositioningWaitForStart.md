# PositioningWaitForStart

PositioningWaitForStart

|  |  |
| --- | --- |
| Enumeration name: | PositioningWaitForStart |
| Enumeration value: | 131 |
| Description: | The operation mode Positioning is waiting for a start command. |

The operation mode Positioning is selected and is expecting the command [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0088728_1).Start, [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0088728_1).StartTrig or [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0088728_1).StartTrigWaitInPos.
