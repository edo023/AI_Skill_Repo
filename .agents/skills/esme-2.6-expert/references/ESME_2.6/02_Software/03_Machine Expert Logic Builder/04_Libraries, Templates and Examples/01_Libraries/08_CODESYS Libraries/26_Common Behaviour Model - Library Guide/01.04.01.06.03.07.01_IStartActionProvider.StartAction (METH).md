# IStartActionProvider.StartAction (METH)

METHOD StartAction

The StartAction is running until either the output xComplete is TRUE or iErrorID ≠ 0 (Zero). If xComplete is TRUE the state [STATE.EXECUTING](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_executing) is reached. If there is a iErrorID ≠ 0 (Zero) the state [STATE.CLEANING](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_cleaning) is reached.

**Note**

To sample the inputs in the first invocation of a function block the method SampleAction is the solution of choice. See: [ISampleActionProvider.SampleAction](6_IsVOI2gxF_ufsAUp4Hm-q58CM_sampleaction.html#6_isvoi2gxf_ufsaup4hm_q58cm_sampleaction_isampleactionprovider_sampleaction_meth)
