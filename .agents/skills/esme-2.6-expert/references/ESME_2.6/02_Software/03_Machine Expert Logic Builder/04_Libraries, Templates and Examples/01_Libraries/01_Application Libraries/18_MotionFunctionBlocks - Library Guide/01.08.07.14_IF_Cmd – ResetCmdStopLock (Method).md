# IF\_Cmd – ResetCmdStopLock (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.1.0 |
| Versions: | Installed version |

## Task

Resets the lock from a stop command to allow for new commands.

## Description

The method is used to reset the lock from a stop command to allow for new commands. After calling a Stop command, new commands cannot be executed until the method ResetCmdStopLock is called. The property xCmdStopLockActive shows if the lock is active.
