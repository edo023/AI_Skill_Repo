# Integrating the SR_AutoTune Program

Integrating the SR\_AutoTune Program

| Step | Action |
| --- | --- |
| 1 | Right-click the Task Configuration and select Add object > Task.  Result: The window "Add Task" opens. |
| 2 | Enter the name of the new task (e.g. TASK\_SR\_AutoTune) and confirm by clicking Open. |
| 3 | Open the new task object. |
| 4 | Set the cycle time to 100 ms and disable the watchdog.  G-SE-0071535.1.gif-high.gif |
| 5 | Click Add Call-up..., select AUT > AutoTune > SR\_AutoTune and confirm by clicking OK in the input assistant.  G-SE-0071536.1.gif-high.gif |
