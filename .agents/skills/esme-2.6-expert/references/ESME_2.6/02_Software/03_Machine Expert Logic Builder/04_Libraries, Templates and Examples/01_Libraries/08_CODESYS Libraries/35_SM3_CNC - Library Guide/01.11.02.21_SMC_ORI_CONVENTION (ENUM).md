# SMC\_ORI\_CONVENTION (ENUM)

TYPE SMC\_ORI\_CONVENTION :

Represents different interpretations of an orientation given by three Euler angles.

| Attributes: | qualified\_only |
| InOut: | | Name | Comment | | --- | --- | | ADDAXES | A/B/C are treated as additional axes and not as an orientation; | | ZYZ | Rotation around the axes z, y’, z’’; | | ZYX | Rotation around the axes z, y’, x’’; also known as Yaw-Pitch-Roll; | | XYZ | Rotation around the axes x, y’, z’’; | |
