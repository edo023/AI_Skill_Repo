# FC\_Vector3DGetCoordinates

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the coordinates of a point in a three-dimensional coordinate system that is spanned by three linearly independent vectors:

i\_stPoint = q\_lrC1 \* i\_stVectorA + q\_lrC2 \* i\_stVectorB + q\_lrC3 \* i\_stVectorC

The coordinates of the point i\_stPoint are converted to the coordinates of the coordinate system which is spanned by the vectors i\_stVectorA, i\_stVectorB, and i\_stVectorC. The vectors i\_stVectorA, i\_stVectorB, and i\_stVectorC must be linearly independent and must not be in the same plane.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stPoint | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | Point for which the coordinates are to be converted. |
| i\_stVectorA | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | Vector A which spans the new coordinate system. |
| i\_stVectorB | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | Vector B which spans the new coordinate system. |
| i\_stVectorC | [ST\_Vector3D](ST_Vector3D-GeneralInformation-0FB413FF.html#ST_Vector3D-GeneralInformation-0FB413FF) | Vector C which spans the new coordinate system. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
| q\_lrC1 | LREAL | Coordinate of the point i\_stPoint on the axis i\_stVectorA. |
| q\_lrC2 | LREAL | Coordinate of the point i\_stPoint on the axis i\_stVectorB. |
| q\_lrC3 | LREAL | Coordinate of the point i\_stPoint on the axis i\_stVectorC. |

|  |  |
| --- | --- |
| Enumeration name: | DefinitionCoordinateSystem |
| Enumeration value: | 95 |
| Description: | The definition of the coordinate system is invalid. |

| Cause | Solution |
| --- | --- |
| The vectors i\_stVectorA, i\_stVectorB, and i\_stVectorC are in the same plane, they do not span a three-dimensional coordinate system. | The vectors i\_stVectorA, i\_stVectorB, and i\_stVectorC must be linearly independent and must not be in the same plane. |
