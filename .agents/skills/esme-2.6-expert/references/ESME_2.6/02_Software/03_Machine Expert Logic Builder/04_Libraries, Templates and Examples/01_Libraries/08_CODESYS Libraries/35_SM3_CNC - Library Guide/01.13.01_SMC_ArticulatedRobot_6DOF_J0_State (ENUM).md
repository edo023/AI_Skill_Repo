# SMC\_ArticulatedRobot\_6DOF\_J0\_State (ENUM)

TYPE SMC\_ArticulatedRobot\_6DOF\_J0\_State :

The possible configuration states of joint 0 of the 6DOF robot arm.

| InOut: | | Name | Comment | | --- | --- | | SMC\_AR\_ARMFORWARD | The arm is pointing towards the wrist center | | SMC\_AR\_ARMBACKWARDS | The arm is pointing opposite to the wrist center | | SMC\_AR\_ARMSTRAIGHT | The wrist center is directly above joint 0 (singular case) | |
