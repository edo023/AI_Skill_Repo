# ETrigA (FB)

FUNCTION\_BLOCK ETrigA EXTENDS [ETrig](d5Ek35equq6Geb4iMYOV1LabLwI_etrig.html#d5ek35equq6geb4imyov1lablwi_etrig_etrig_fb) IMPLEMENTS [IETrigA](c-dVK8bVY1F-S7PaIINI_aQKZks_ietriga.html#c_dvk8bvy1f_s7paiini_aqkzks_ietriga_ietriga_itf)

This function block is a base implementation of a *PLCopen Edge Triggered Behavior Model* (see: [IETrigA](c-dVK8bVY1F-S7PaIINI_aQKZks_ietriga.html#c_dvk8bvy1f_s7paiini_aqkzks_ietriga_ietriga_itf)). For use, this block should be extended and the specific inputs and outputs must be supplemented. The output eErrorID of the application-specific enum type ERROR must be added.
