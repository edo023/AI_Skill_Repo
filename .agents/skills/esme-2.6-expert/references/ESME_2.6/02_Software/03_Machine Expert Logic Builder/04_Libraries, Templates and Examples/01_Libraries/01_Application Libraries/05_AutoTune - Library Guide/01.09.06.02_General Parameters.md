| Rectangle Cut-off frequency | The Cut-off frequency parameter specifies a cut-off frequency in Hertz; up to this frequency, the transfer performance of the position or velocity control loop is good. The higher this cut-off frequency is, the greater the absolute dynamic response of the drive. These values are related only indirectly to the relative dynamic setting from the visualization.  The cut-off frequencies are intended to give you a feeling for the travel cycles which can be achieved with the parameters set. The cut-off frequencies are typically between 50 Hz and 400 Hz for the speed regulator and between 10 Hz and 100 Hz for the position controller. |
