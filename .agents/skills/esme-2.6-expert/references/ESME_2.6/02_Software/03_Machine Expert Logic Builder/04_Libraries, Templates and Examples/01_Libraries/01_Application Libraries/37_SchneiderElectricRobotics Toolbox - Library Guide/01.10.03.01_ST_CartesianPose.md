## Description

A Cartesian pose describes both a position and an orientation with reference to a coordinate system. A structure ST\_CartesianPose containing the following structure elements is added to the library.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stPosition | *[PDL.ST\_Vector3D](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087802)* | Representation of a Cartesian position as a 3D vector. |
| stOrientation | *[PDL.ST\_Vector3D](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087802)* | Representation of a Cartesian orientation in the form of roll, pitch, and yaw angles. |
| etOrientationConvention | *[ROB.ET\_OrientationConvention](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Robotic&topicID=D_SE_0075485)* | Enumerator containing the list of rotation orders supported by the library when describing an orientation as roll, pitch, and yaw. |
