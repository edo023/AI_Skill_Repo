# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| None |  | No delay. Positioning is started as soon as the motion job is activated. |
| Absolute |  | Positioning is started as soon as the job timer [ST\_RealtimeData](../Structures/Structures-15.htm#XREF_D_SE_0089506_1).lrAbsoluteJobTimer has reached the value [ST\_PosStart](../Structures/Structures-12.htm#XREF_D_SE_0089500_1).lrStartTime. |
| Relative |  | Positioning is started as soon as the time [ST\_PosStart](../Structures/Structures-12.htm#XREF_D_SE_0089500_1).lrRelativeDelay has elapsed after the activation of the job. |
