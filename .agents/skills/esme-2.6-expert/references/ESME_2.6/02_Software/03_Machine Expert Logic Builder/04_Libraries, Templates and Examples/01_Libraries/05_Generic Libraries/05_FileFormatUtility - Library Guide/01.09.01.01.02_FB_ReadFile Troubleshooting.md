# FB\_ReadFile Troubleshooting

## Troubleshooting

This table describes some general issues and their solutions:

| Output values of q\_etResult when q\_xError = TRUE | Cause | Solution |
| --- | --- | --- |
| FilePathInvalid | * The specified file is not available. * The given file path has an invalid syntax.  For example, FilePath must not be empty. | * Verify that the file exists. In case the file path targets extended memory (such as an SD memory card), verify that it is available. * Verify that the syntax used is supported by your controller.  For example, your controller may support '`\`' as a separator while another controller supports '`/`' instead. |
| BufferInvalid | The value of the input i\_pbyBuffer is 0. | Assign the pointer to your buffer to the input i\_pbyBuffer. |
| BufferSizeInvalid | The value of the input i\_udiBufferSize is 0. | Assign the size of your buffer to the input i\_udiBufferSize. |
| BufferTooSmall | * The value of the input i\_udiBufferSize is less than the file size. * The value of the input i\_udiBufferSize is less than the value of the input i\_udiBytesToRead. | * Increase the buffer size. * Limit the maximum number of bytes to be read to the size of the buffer using the parameter i\_udiBytesToRead. |
| OffsetInvalid | The offset position to be read from is greater than the file size. | Configure a value for i\_udiOffset that is equal to or less than the file size. |
| EndOfFileReached | The end of the file is reached before the requested number of bytes is read. | Configure values for i\_udiBytesToRead and i\_udiOffset that do not exceed the end of the file. |
| FileNotExists | The file does not exist. | Verify whether the specified file is present. |
| MaxNumOfSimultaneousFileOperationsExceeded | Too many file operations have been performed simultaneously. | Do not perform more than 20 file operations in parallel. |
| GetFileSizeFailed | An internal error was detected in the execution of the function block while retrieving the file size. | Retry to read the file. If the issue persists, contact your local Schneider Electric support representative. |
| FileOpenFailed | An internal error was detected in the execution of the function block while opening the file. | Retry to read the file. If the issue persists, contact your local Schneider Electric support representative. |
| SetPosFailed | An internal error was detected in the execution of the function block while setting the offset position. | Retry to read the file. If the issue persists, contact your local Schneider Electric support representative. |
| FileReadFailed | An internal error was detected in the execution of the function block while reading the file. | Retry to read the file. If the issue persists, contact your local Schneider Electric support representative. |
| FileCloseFailed | An internal error was detected in the execution of the function block while closing the file. | Retry to read the file. If the issue persists, contact your local Schneider Electric support representative. |
| InvalidInput | The value of at least one input parameter is invalid. | Details about the invalid parameter are provided in the output q\_sResultMsg. Assign a valid value to the corresponding input and re-execute the function block. |
