# Function Block Description

Function Block Description

The AlarmDataStorage\_2 function block can be used to record an arbitrary alarm.

The function block records the following data:

oThe number of alarm events detected.

oThe number of hours since the last detected alarm event occurred.

oDuration of the detected alarm event.

oMaintains a historical record (YYYYMMDD HHMMSS format) of the alarm event detected in the system.

This data is retained after both normal and unexpected power down.
