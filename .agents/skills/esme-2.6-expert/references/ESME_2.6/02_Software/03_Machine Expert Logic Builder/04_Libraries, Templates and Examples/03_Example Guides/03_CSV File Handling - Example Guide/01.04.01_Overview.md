# Overview

## Short Description

This document describes an application example for implementing read and write access to a file in CSV format using the function blocks from the FileFormatUtility library.

## System Requirements and Limitations

The application example has been created with the components listed in the table:

| Component | Type and Version |
| --- | --- |
| Software | * EcoStruxure Machine Expert V1.1 |
| Controller | * PacDrive LMC300 / 400 / 420 / 600 / 802C * Modicon M262 Logic/Motion Controller * Modicon M241 Logic Controller / Modicon M251 Logic Controller |

NOTE: The concepts presented in the application example are, in principle, transferable to other controllers supporting the controller firmware released in conjunction with EcoStruxure Machine Expert V1.1.
