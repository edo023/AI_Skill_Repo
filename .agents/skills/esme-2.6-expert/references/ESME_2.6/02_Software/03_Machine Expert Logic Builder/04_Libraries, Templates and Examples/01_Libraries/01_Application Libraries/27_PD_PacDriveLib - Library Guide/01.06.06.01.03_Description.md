# Description

Description

The function implements a flashing light. A positive signal at i\_xStart enables the flashing sequence. The output signal q\_xOutput is first FALSE for the duration of i\_timLow, then TRUE for the duration of i\_timHigh, and then starts this cycle over again until the signal at i\_xStart is removed. At that point, the output signal is maintained until it begins again at FALSE upon a positive signal at i\_xStart. This function block must be called up cyclically with the period MIN (i\_timLow, i\_timHigh) (or more frequently).
