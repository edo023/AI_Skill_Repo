# IRectangleProvider (ITF)

INTERFACE IRectangleProvider

A visualization element (IVisualElement) can optionally implement this interface too if it needs to pass rectangles to the currently handled visualization client. Elements that handle nested elements (like groups, frames etc.) should implement this interface at least to forward according calls to their “child” elements.
