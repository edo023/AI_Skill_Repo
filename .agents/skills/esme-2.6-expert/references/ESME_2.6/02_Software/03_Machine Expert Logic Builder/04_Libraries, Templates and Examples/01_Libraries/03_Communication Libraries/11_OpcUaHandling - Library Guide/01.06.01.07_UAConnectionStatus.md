# UAConnectionStatus

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration UAConnectionStatus specifies the connection state between the OPC UA server and the OPC UA client.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| UACS\_Connected | 0 | OPC UA client is connected to OPC UA server. |
| UACS\_ConnectionError | 1 | For the connection from the OPC UA client to the OPC UA server, an error was detected. |
| UACS\_Shutdown | 2 | The OPC UA client was disconnected from the OPC UA server. |
