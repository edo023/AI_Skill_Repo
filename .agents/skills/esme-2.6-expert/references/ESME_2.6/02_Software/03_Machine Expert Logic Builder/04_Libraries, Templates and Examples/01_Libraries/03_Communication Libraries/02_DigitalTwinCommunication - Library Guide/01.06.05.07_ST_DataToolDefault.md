# ST\_DataToolDefault

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

The structure ST\_DataToolDefault contains parameters to configure and operate a tool, for example, a gripper, vacuum, or screw tool, in EcoStruxure Machine Expert Twin.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| stConfigData | [ST\_ConfigDataGeneric](ST_ConfigDataGeneric-6D5DB8E3.html) | Structure that contains the general configuration data of the tool. |
| udiSystemId | UDINT | Reserved |
| xActuator | BOOL | Tool actuator:  If set to TRUE, the actuator is on.  If set to FALSE, the actuator is off. |
| xSensor | BOOL | Tool sensor feedback (this variable is updated by EcoStruxure Machine Expert Twin):  If set to TRUE, a target is detected by the sensor.  If set to FALSE, the sensor does not detect any target. |
