# q_xTeachMode

q\_xTeachMode

TRUE when the function block is in teach mode. The function block enters teach mode at rising edge of i\_xTeachMode input. The function block leaves teach mode at falling edge on the i\_xTeachMode input. The function block automatically leaves the teach mode after 60 minutes even if i\_xTeachMode is still TRUE.

The status of the teach mode must be indicated to the crane operator.
