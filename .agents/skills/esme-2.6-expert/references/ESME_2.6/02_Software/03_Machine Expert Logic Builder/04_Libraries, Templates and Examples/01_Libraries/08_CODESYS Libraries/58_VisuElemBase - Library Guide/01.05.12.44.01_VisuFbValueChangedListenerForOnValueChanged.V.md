| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | ValueChanged | BOOL |  | | Input | pClient | POINTER TO VisuStructClientData | A pointer to the client structure were the event occurred. pClient can be 0 when eType = ConditionVariable | | itfVisualization | IVisualisation | The visualization from which the value changed event is called.
