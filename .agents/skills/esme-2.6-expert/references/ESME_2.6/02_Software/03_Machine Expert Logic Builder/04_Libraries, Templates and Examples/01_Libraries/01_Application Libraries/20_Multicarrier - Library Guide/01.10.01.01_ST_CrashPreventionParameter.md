## Description

The structure ST\_CrashPreventionParameter contains parameters used by the function block FB\_CrashPrevention that is intended to help prevent collisions of carriers.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| lrCrashDistance | LREAL | The minimum distance between two carriers. |
| xDisableCrashPreventionForMoveGapControl | BOOL | If xDisableCrashPreventionForMoveGapControl is TRUE, a carrier moved by the move command [MoveGapControl](IF_MoveGapControl-5B81ACFA.html#IF_MoveGapControl-5B81ACFA) is not considered for carrier collision prevention. |
