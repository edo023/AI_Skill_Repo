# DateTimeFromWeek (FUN)

FUNCTION DateTimeFromWeek : ULINT

Combines the ISO week date parts to a timestamp

* uiYear := 2009 uiWeek :=1 eWeekday:=WEEKDAY.MONDAY => 29.12.2008
* uiYear := 2009 uiWeek :=53 eWeekday:=WEEKDAY.SUNDAY => 03.01.2010

See: <https://en.wikipedia.org/wiki/ISO_week_date>
