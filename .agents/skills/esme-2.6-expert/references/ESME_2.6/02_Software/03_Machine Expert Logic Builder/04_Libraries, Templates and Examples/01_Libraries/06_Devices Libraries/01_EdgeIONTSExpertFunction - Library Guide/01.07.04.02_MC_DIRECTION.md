# MC\_DIRECTION

## Description

The MC\_DIRECTION enumeration provides the motion direction.

## Enumeration Elements

The enumeration data type ENUM contains the following values:

| Name | Value | Description |
| --- | --- | --- |
| mcNegativeDirection | -1 | CCW (Counter Clock Wise), backward, reverse, negative motion |
| mcPositiveDirection | 1 | CW (Clock Wise), forward, positive motion |
| mcCurrentDirection | 2 | Motion is performed in the direction that was last used. |
