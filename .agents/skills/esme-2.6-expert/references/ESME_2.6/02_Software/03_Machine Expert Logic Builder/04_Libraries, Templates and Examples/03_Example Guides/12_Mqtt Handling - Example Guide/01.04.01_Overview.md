# Overview

## Short Description

The project implements an application example which shows how to implement an MQTT client in a controller application using the MqttHandling library. Beyond this, the example shows how to create an application message in JSON format using the function block FB\_CreateJsonFormattedString from the FileFormatUtility library.

## System Requirements and Limitations

The application example has been created and tested with the components listed in the table:

| Component | Type and Version |
| --- | --- |
| Software | * EcoStruxure Machine Expert V2.0.2 |
| Controller | * PacDrive LMC * Modicon M262 Logic/Motion Controller |

NOTE: The concepts presented in the application example are, in principle, transferable to other controllers supporting the controller firmware released in conjunction with EcoStruxure Machine Expert V2.0.2.
