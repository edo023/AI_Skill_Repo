# SensorNotFound

SensorNotFound

|  |  |
| --- | --- |
| Enumeration name: | SensorNotFound |
| Enumeration value: | 92 |
| Description: | No sensor signal was found. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.PosDirectionPosEdgeSensor](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionPosEdgeSensor](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionNegEdgeSensor](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.PosDirectionNegEdgeSensor](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.PosDirectionPosEdgeHWLimitPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionPosEdgeHWLimitNeg](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionNegEdgeHWLimitNeg](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1) or  [PDL.ET\_HomeMode.PosDirectionNegEdgeHWLimitPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).  The sensor signal is not transmitted to the module. | Ensure that the signal has been applied at the [ST\_Home.stSensor.i\_xSensor](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) variable. |
| - | The sensor does not transmit a signal. | Check the power supply of the sensor.  Ensure that the sensor is not dirty.  Ensure that the sensor can be triggered by the mechanical system.  Check the electrical connection of the sensor and of the digital input. |
