# FC\_AxisConverterCnc - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.3.0 |

## Description

This function converts values from MOIN.IF\_Axis to SM3\_Basic.AXIS\_REF\_SM3 for use by the library SM3\_CNC.

NOTE: In the case of coupled axes, there is a delay of one cycle between the leading axis and the following axis.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxis | MOIN.IF\_Axis | Reference to the axis (refer to [Motion Interface Library Guide](AboutTheBook-91929F63.html#AboutTheBook-91929F63__RelatedDocuments-9192A39A) for details). |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stSM3\_Axis | SM3\_Basic.AXIS\_REF\_SM3 | Softmotion drive interface |
