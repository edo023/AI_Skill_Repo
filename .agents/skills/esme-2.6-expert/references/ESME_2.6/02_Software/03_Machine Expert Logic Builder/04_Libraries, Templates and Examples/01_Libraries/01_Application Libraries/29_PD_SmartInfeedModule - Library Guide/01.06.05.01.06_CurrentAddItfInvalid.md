# CurrentAddItfInvalid

CurrentAddItfInvalid

|  |  |
| --- | --- |
| Enumeration name: | CurrentAddItfInvalid |
| Enumeration value: | 95 |
| Description: | The current additional interface is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid interface has been transferred at the i\_dwCurrentAddItf input (0). | Make sure that the address of the additional interface of the selected module is transferred at the i\_dwCurrentAddItf input. |
