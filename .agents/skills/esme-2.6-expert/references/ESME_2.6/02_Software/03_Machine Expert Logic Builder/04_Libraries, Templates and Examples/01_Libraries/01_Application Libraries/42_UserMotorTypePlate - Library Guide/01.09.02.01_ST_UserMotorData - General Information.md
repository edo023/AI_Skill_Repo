# ST\_UserMotorData - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |

## Description

This structure contains general motor data for the various motor types.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| etMotorType | [ET\_MotorType](D-SE-0090524.html#D-SE-0090524) | Motor types:   * Rotary\_PMSM * Linear\_PMSM * Rotary\_ACIM   Required parameter |
| sMotorname | STRING[80] | Name of the motor (is displayed in the MotorType parameter), string with a maximum of 20 characters.  Not required |
| sMotorSerialNumber | STRING[80] | Serial number of the motor (displayed in the SerialNumberMotor parameter).  Not required |
| sMotorArticleNumber | STRING[80] | Part number of the motor (displayed in the PartNumberMotor parameter).  Not required |
| stMotorDataACIM | [ST\_UserMotorDataACIM](D-SE-0090546.html#D-SE-0090546) | Motor data for asynchronous motors. This structure must be filled in if etMotorType is set to the value Rotary\_ACIM. |
| stMotorDataPMSM | [ST\_UserMotorDataPMSM](D-SE-0090548.html#D-SE-0090548) | Motor data for synchronous motors. This structure must be filled in if etMotorType is set to the value Rotary\_PMSM or Linear\_PMSM. |
