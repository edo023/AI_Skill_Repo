## Task

The method is used to set a coordinate system of the Lexium Cobot.

## Description

This method is used to modify a coordinate system of the Lexium Cobot.

Preconditions to execute this command:

* The control source of the Lexium Cobot must be Remote. For further information, refer to the method [GetControlSource(Method)](FB_CobotController-GetControlSource-7FECE4FA.html).

The sent command is acknowledged by the Lexium Cobot controller either as successful or as unsuccessful:

* A successful command is reported by the property xCommandAcknowledged.
* A unsuccessful command is reported by the property xCommandFailed. For detailed information, verify the diagnostic outputs of the function block and the log data of [FB\_CobotControllerFeedback.rstData.stLogData](FB_CobotControllerFeedback-GeneralI-27E4BC7A.html#FB_CobotControllerFeedback-GeneralI-27E4BC7A__Properties-27E35EB1).

A coordinate system consists of its index, name, and offset.

By using this command, the name and offset can be modified.
