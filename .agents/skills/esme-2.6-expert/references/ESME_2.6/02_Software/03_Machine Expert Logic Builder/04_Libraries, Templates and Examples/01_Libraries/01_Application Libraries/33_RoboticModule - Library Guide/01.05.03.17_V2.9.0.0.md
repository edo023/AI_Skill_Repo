## New Features

* [**Additional Output Variables q\_lrPeriodStart and q\_lrPeriodLength for IF\_Configuration.GetConfiguredDrive**](D-SE-0076924.html#D-SE-0076924__D-SE-0076924.5)

  To read the configured period data for an auxiliary axis, the outputs q\_lrPeriodStart and q\_lrPeriodLength were added.
* [**IF\_ConfigurationAdvanced.xDisableTrackingEncoderCheck**](D-SE-0076933.html#D-SE-0076933__D-SE-0076933.6)

  The parameter verifications of FeedConstant, GearIn, GearOut, and Direction for the tracking encoder against the corresponding parameters of the velocity source can be disabled.
* [**IF\_ConfigurationAdvanced.xSetTrackingEncoderParameters**](D-SE-0076933.html#D-SE-0076933__D-SE-0076933.6)

  The parameters FeedConstant, GearIn, GearOut, and Direction of the tracking encoder are set automatically fitting to the corresponding parameters of the velocity source.
* [**IF\_Configuration.User6Ax(…)**](D-SE-0098634.html#D-SE-0098634__D-SE-0098634.2)

  With this method, the robot can be configured as a six axial user-configured robot with six degrees of freedom.
* **Operation Mode Service added**

  ET\_OpMode.Service was added. This operation mode was added for Schneider Electric robots.

## Mitigated Anomalies

* IF\_Homing.xResetHomeOk

  xHomeOk is only reset if IF\_Homing.xResetHomeOk is called with the value TRUE.
