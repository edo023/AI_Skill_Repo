# Input Pin Description

Input Pin Description

| Input | Data Type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block.  Refer to [detailed description](#XREF_D_SE_0021131_4) below this table. |
| i\_xCalbMode | BOOL | This input enables the calibration.  TRUE: Calibration active.  FALSE: Calibration inactive.  Refer to [detailed description](#XREF_D_SE_0021131_5) below this table. |
| i\_wPulsEnc | WORD | This is the encoder pulses input from the drive (PUC).  Range: 0 to 65535 |
| i\_xHighLim | BOOL | This is the forward stop limit switch input of the hoisting axis.  NOTE: It is mandatory to use a limit switch which is normally closed ([NC](../glossary/glossary.htm#XREF_D_SE_0024697_85) contact).  TRUE: Limit inactive  FALSE: Limit active |
| i\_rLoadLen | REAL | This input is the input for an additional load length if needed.  Range: 0...100  Scaling/Unit: 1 m |
| i\_stCLE | CLE | Refer to [Sub-structure description](#XREF_D_SE_0021131_6). |
