# FB\_DummyCarrierFeedback - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.0.0.0 |

![](images/0000757650.png)

## Description

The function block FB\_DummyCarrierFeedback can be used to help avoid access exceptions in user applications via interface null-pointers, for example in case of:

* initialization of carrier feedback data (MCR.IF\_CarrierFeedback).
* initialization of trace data

The instance of the function block FB\_DummyCarrierFeedback does not need to be called and has no other effect.

## Inputs

The function block has no inputs.

## Outputs

The function block has no outputs.
