# ST\_InputsOutputs

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |
| Inherits from: | - |

## Description

This structure contains the status of digital inputs and outputs as well as the value of analog inputs and outputs of the Lexium Cobot.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| auiAnalogInputsCabinet | AnalogInputsCabinet | The analog inputs of the Lexium Cobot cabinet are addressed.  Unit: [%] |
| auiAnalogInputsToolEnd | AnalogInputsToolEnd | The analog inputs of the Lexium Cobot Tool End are addressed.  Unit: [%] |
| auiAnalogOutputsCabinet | AnalogOutputsCabinet | The analog outputs of the Lexium Cobot cabinet are addressed.  Unit: [%] |
| axDigitalInputsCabinet | DigitalInputsCabinet | The digital inputs of the Lexium Cobot cabinet are addressed. |
| axDigitalInputsToolEnd | DigitalInputsToolEnd | The digital inputs of the Lexium Cobot Tool End are addressed. |
| axDigitalOutputsCabinet | DigitalOutputsCabinet | The digital outputs of the Lexium Cobot cabinet are addressed. |
| axDigitalOutputsToolEnd | DigitalOutputsToolEnd | The digital outputs of the Lexium Cobot Tool End are addressed. |
