# ET\_StateMoveSyncFromStandstill - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_StateMoveSyncFromStandstill specifies the status of the MoveSyncFromStandstill command for the synchronized movement of a carrier (refer to [IF\_MoveSyncFromStandstill](IF_MoveSyncPathFromStandstill-5B839E78.html#IF_MoveSyncPathFromStandstill-5B839E78)).

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| NotActive | 0 | The move command MoveSyncFromStandstill is not active. |
| WaitForPhasingZoneReached | 1 | The carrier is waiting for the master to reach the phasing zone.  The phasing zone is defined by the parameters i\_lrXStartPhasingToMaster and i\_lrXEndPhasingToMaster that are specified in the method [StartSyncToExternalMaster](MoveSyncExtMaster-080435F3.html#MoveSyncExtMaster-080435F3). |
| PhasingToMaster | 2 | The carrier is in the phasing zone and is synchronizing to a master. |
| InPhaseToMaster | 3 | The carrier is synchronized to a master by a one-to-one cam. |
