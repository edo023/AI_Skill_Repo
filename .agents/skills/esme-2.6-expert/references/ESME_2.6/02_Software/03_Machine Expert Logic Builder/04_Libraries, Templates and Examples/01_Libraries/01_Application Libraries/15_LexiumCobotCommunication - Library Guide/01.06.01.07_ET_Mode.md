# ET\_Mode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.1.0 |

## Description

This enumeration type is used to address a specific mode of the Lexium Cobot.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Manual | 1 | The manual mode of the Lexium Cobot is addressed. |
| Automatic | 2 | The automatic mode of the Lexium Cobot is addressed. |
| Drag | 4 | The hand-guide (Drag) mode of the Lexium Cobot is addressed. |
