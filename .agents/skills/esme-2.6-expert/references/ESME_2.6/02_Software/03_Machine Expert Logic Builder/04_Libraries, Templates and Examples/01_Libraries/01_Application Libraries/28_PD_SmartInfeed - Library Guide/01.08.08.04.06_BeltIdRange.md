# BeltIdRange

BeltIdRange

|  |  |
| --- | --- |
| Enumeration name: | BeltIdRange |
| Enumeration value: | 11 |
| Description: | BeltId is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid value has been transferred at the input i\_udiBeltId. | The number of the belt, from which the product is to be removed, must be specified at the i\_udiBeltId input. The number must be between one and the number of parameterized belts. |
