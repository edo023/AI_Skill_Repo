# AlarmStorageLatchVariable (STRUCT)

TYPE AlarmStorageLatchVariable : STRUCT

This structure bundles the information of a latch variable stored in the alarm storage.

| InOut: | | Name | Type | Comment | | --- | --- | --- | | m\_byType | BYTE | The type of the latch variable. A TypeClass value. | | m\_liValue | LINT | The latch value as “raw” value | | m\_xValueNULL | BOOL | Indicates, whether NULL is stored in the alarm storage | |
