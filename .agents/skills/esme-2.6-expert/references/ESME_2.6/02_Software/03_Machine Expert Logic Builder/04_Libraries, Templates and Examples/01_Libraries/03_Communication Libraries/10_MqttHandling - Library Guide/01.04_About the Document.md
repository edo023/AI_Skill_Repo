# About the Document

## Document Scope

This document describes the library MqttHandling.

The MqttHandling library provides the MQTT client functionality in order to exchange data with other devices over the network through an MQTT message server.

## Validity Note

This document has been updated for the release of EcoStruxure™ Machine Expert V2.5.

## Product Related Information
