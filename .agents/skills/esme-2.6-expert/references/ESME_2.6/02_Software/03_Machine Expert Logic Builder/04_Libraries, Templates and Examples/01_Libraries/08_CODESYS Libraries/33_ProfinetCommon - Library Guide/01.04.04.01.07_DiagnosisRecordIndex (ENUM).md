# DiagnosisRecordIndex (ENUM)

TYPE DiagnosisRecordIndex :

The diagnosis information that is read by an acyclic read-service (RDREC) can be filtered to reduce the returned diagnosis information.

For filtering the record data index is used.

This enumeration defines some indices for filtering by the scope (e.g module-level, device-level)

Within this scope, these filters return all types of diagnosis information (maintenance, qualified and status).
