### Destroying a block pool

> ```
> hPool : MBM.HANDLE;
> eError : MBM.ERROR;
> xExit : BOOL;
>
> IF xExit THEN
>     eError := MBM.PoolDelete(hPool:=hPool);
>     // The pool is now locked and PoolGetBlock will return an error
>     // The pool memory will be released with the last PoolPutBlock
>     hPool := MBM.gc_hINVALID;
>     xExit := FALSE;
> END_IF
> ```
