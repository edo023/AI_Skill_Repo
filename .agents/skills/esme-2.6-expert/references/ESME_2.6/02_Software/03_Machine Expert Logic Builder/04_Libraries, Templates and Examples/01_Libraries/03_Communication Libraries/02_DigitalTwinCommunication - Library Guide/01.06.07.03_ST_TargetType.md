## Description

The structure ST\_TargetType contains target type variables.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| udiTargetType | UDINT | An ID which identifies the target type. |
| udiGeometricShape | UDINT | The geometric shape of the target, defined with [ET\_GeometricShape](ET_GeomShape-21B66940.html#ET_GeomShape-21B66940). |
| stGeometricSize | [stGeometricSize](ST_GeomSize-24CC3236.html) | The size of the target. |
| stOffsetPosition | SE\_MATH.ST\_Vector3D | The offset position of the target. |
| stOffsetOrientation | SE\_MATH.ST\_Vector3D | The offset orientation of the target. |
