# Structure: ShapeDef

Structure: ShapeDef

Structure of parameters related to the definition of polygonal restricted areas

| Parameter | Data type | Description |
| --- | --- | --- |
| xShapeActCmd | BOOL | Command to activate the shape  TRUE: Activate the shape  FALSE: Deactivate the shape |
| xShapeActStat | BOOL | Activation status of the shape  TRUE: Shape is active  FALSE: Shape is inactive |
| arShapeVertices | ARRAY[0..c\_iNoOfVert,0..1] OF REAL | Array of vertices |
| iVertexCount | INT | Number of defined vertices in the shape  Range: 1...10 |
| rLimitTop | REAL | Top limit of restricted area, measured from the trolley level (0 = no top limit)  Range: 0.0...3.438  Unit: m |
| rLimitBottom | REAL | Bottom limit of restricted area, measured from the trolley level (0 = no bottom limit)  Range: rLimitBottom...3.438 |
