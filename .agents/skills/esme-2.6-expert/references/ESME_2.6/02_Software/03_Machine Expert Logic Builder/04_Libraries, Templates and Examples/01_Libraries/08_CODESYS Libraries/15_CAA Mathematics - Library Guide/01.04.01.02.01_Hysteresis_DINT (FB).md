# Hysteresis\_DINT (FB)

FUNCTION\_BLOCK Hysteresis\_DINT

This function block will set the output variable to TRUE, if the integral value ![](images/7f8453f800ce44bda5d2fc88f510348d.jp.jpg) is smaller than an integral lower bound ![](images/bcec42bab0d94c578dbf5d511c0eb761.jp.jpg)

It will set the Boolean output variable to FALSE, if the input value ![](images/3687c2ea19f0449882421a52bb63dc74.jp.jpg) exceeds the integral upper bound ![](images/242ab2308cbc45349e0efe3f881a9fb3.jp.jpg).

If ![](images/f8d39670cb434c8fb8107dd1c3e47951.jp.jpg) lies between the lower and upper bound, the value of the output will rest unchanged by the module.
