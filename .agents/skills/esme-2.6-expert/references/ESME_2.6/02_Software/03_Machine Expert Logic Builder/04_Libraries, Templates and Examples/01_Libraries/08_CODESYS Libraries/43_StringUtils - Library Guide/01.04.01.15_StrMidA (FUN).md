# StrMidA (FUN)

FUNCTION StrMidA

Copies |iLength| characters of |pst| to |pstResult|, beginning at the |iPosition|-th character position. .. note:

```
|iPosition| = 1 is the first character.
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | pst | POINTER TO BYTE | Pointer to the source string | | uiInputBufferSize | UINT | The size of the buffer, where |pst| points to | | iLength | INT | The number of characters that shall be copied | | iPosition | INT | The position from where copying shall start | | pstResult | POINTER TO BYTE | The buffer, where the result will be returned | | uiResultBufferSize | UINT | The size of the buffer (in bytes), where the result will be returned | |
