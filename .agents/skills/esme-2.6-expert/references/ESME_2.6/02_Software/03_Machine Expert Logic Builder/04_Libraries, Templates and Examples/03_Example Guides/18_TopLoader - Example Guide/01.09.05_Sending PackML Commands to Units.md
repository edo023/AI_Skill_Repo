# Sending PackML Commands to Units

## Overview

To send PackML commands directly to a specific [software unit](ProjectStructure-7AD3B398.html#ProjectStructure-7AD3B398__SoftwareUnitsRepresentingMechanicUn-7B34E3C0), proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Select the tab of the desired unit (ProductBelt, CartonBelt, PickPlace, Sealing) from the top of the visualization window.  **Result**: The corresponding PackML state machine is displayed below the tab. |
| 2 | From the bottom of the visualization window, click the button activate debug Cmds and click one of the buttons to send the command to the selected unit. |

NOTE: The command from the visualization is sent in parallel to other possible command sources. As no priority is assigned to one source, the last command received is executed.
