# AbortDataTransmission (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method is used to abort a requested data transmission. If the request is in the internal queue, it is removed from it. The request is identified by the interface assigned to the input i\_ifRequester. The same interface must be assigned, which was used for creating the send request with the method SendData().

The return value of type BOOL indicates TRUE if the request was removed from the queue. If FALSE, the interface was not found in the queue and the method is completed without affecting the queue.
