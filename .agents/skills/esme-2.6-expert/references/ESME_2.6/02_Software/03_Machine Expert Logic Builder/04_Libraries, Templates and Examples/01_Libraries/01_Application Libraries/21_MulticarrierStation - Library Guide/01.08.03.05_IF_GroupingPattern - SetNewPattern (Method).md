## Task

Starting to create a new carrier group pattern.

## Description

The method SetNewPattern is a mandatory method for creating a group pattern. When starting to create a new group pattern, you must first call the method SetNewPattern.

The return value SetNewPattern of type BOOL indicates TRUE if a pattern has been created successfully and is ready to add groups of carriers (see method [AddNewGroupAndFirstCarrier](AddNewGroup-EEC3AB17.html#AddNewGroup-EEC3AB17)).

## Inputs

The method has no inputs.
