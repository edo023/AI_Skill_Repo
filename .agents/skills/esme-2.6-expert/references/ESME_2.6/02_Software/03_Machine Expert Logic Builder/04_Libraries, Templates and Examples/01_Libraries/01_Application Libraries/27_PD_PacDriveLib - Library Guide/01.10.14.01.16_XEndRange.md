# XEndRange

XEndRange

|  |  |
| --- | --- |
| Enumeration name: | XEndRange |
| Enumeration value: | 192 |
| Description: | XEnd is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | At the input i\_lrXEnd, a value that lies outside of the cam has been applied. | i\_lrXEnd must lie within the cam defined at iq\_stMultiCam. |
