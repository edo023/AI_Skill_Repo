# Description

Description

The profiles described in iq\_stMultiCamData are loaded, adapted where necessary and scaled such that, using the function FC\_ProfileSegLoadById, a segmented profile is generated which is normalized to 1.

This function reserves one profile Id respectively for each segment and for the entire profile. Using the function [FC\_DelSegProfile](../Functions_A_to_J/Functions_A_to_J-38.htm#XREF_D_SE_0087459_1), a segmented profile which has been created using the function described above can be deleted.
