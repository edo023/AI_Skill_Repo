# Overview

Overview

The program SR\_TcpClient implements the program code for the [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) [client](../glossary/glossary.htm#XREF_D_SE_0055638_5) functionality. For control and monitoring of this functionality, the visualization screen VisuTcpClient is provided in the example project.

The variables which link the visualization with the program code are defined in the global variable list GVL\_VisuTcpClient.
