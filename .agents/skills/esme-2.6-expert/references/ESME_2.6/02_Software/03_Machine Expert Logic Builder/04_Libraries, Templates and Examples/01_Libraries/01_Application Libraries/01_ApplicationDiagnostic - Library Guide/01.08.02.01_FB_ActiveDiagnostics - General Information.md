## Task

The function block implements a listener that provides a list of active diagnostic messages of an instance of the FB\_DiagnosticHandler.

## Description

The FB\_ActiveDiagnostics function block implements a listener that provides a list of active diagnostic messages of an instance of the FB\_DiagnosticHandler. The instance of the function block must be registered to an instance of FB\_DiagnosticHandler using the method RegisterListener.

NOTE: Any instance of FB\_ActiveDiagnostics should be registered with only one instance of FB\_DiagnosticHandler.

## Interface

The function block provides the generic constant c\_uiMaxNumberOfDiagMessages that specifies the maximum number of diagnostic messages stored inside the function block. The value of the generic constant must be defined during build time and cannot be modified during runtime.

The generic constant must be defined in the declaration of the instance of this function block.

For example:

```
VAR  
	fbActiveDiagnostics : ApDiag.FB_ActiveDiagnostics<100>  
END_VAR
```

The function block must not be called in the application.
