# Enumeration Elements

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Create | 0 | A new file is created. A file having the same name will be overwritten. |
| Append | 1 | If a file of the same name exists, it will be opened and the data to be written will be appended to the end of the file. If no file of this name exists already, it will be created. |
