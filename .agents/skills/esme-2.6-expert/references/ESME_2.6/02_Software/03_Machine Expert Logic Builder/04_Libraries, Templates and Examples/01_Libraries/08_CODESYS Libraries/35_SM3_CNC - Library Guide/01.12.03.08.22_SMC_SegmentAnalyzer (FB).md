# SMC\_SegmentAnalyzer (FB)

FUNCTION\_BLOCK SMC\_SegmentAnalyzer

This function block analyzes the different segments (acceleration constant velocity, deceleration) of a CNC path, independently of the interpolator function block. With every call, the SMC\_SegmentAnalyzer outputs basic information about the current path segment and adds up the time that will be needed for its interpolation.

Restrictions:

The complete path must fit into the input path queue. The execution on the output path queue by the next function block (usually [SMC\_Interpolator](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-interpolator.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_interpolator_smc_interpolator_fb) ) may only begin, when SMC\_SegmentAnalyzer has analyzed the queue completely.
