# GVL\_OpcUaRobotics

## Overview

|  |  |
| --- | --- |

| Available as of: | V1.0.2.0 |

## Description

The global variable list GVL\_OpcUaRobotics contains variable declarations for the build and emulation of the Lexium robots via OPC UA communication by using the Symbol Configuration.

Verify that in the [Symbol Configuration](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083586) the emulation data are selected and the access rights are in line with the application needs.
