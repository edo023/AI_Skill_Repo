## New Features

* Configuration for SetMotionProfile (ET\_MotionProfile.MoveSyncMaxVelocity).
* Support of EcoStruxure Machine Expert Twin. OPC\_UA structure is filled automatically.
* For 3D view of single robot the emulation using EcoStruxure Machine Expert Twin and the OPC/UA communication between EcoStruxure Machine Expert Twin and the controller can be used.
* New methods and properties GetRobotData, GetDigitalTwinConfigData and ifCollisionHandlerPSeries have been added.

## Modifications

Set default value for UserDrivePeakCurrent for P-Series P2-P6Ex to 7500.

## Mitigated Anomalies

* In a multi-application project the devices were not added correctly to the Devices tree. Multi-application project is now fully supported.
* Configuration of internal exception handler has been improved.
