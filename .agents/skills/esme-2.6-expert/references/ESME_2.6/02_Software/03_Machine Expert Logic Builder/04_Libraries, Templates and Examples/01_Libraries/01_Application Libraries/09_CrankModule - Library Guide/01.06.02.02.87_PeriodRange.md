# PeriodRange

PeriodRange

|  |  |
| --- | --- |
| Enumeration name: | PeriodRange |
| Enumeration value: | 26 |
| Description: | Period is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid value was applied at the [ST\_EndlessFeed](../Structures/Structures-3.htm#XREF_D_SE_0088754_1).i\_lrPeriod input. | A value greater or equal to 1.0 must be transferred to the [ST\_EndlessFeed](../Structures/Structures-3.htm#XREF_D_SE_0088754_1).i\_lrPeriod input. |
| - | A negative value was applied at the [ST\_Manual](../Structures/Structures-7.htm#XREF_D_SE_0088762_1).i\_lrPeriod input. | A value greater than 0 must be transferred to the [ST\_Manual](../Structures/Structures-7.htm#XREF_D_SE_0088762_1).i\_lrPeriod input. |
| - | The operation mode [ET\_OpMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0088734_1).MultiCam was selected and the Period parameter of the logical encoder at the [ST\_Main](../Structures/Structures-6.htm#XREF_D_SE_0088760_1).iq\_lencEncoder input was set to a value unequal to 0. | Ensure that the Period parameter of the logical encoder at the [ST\_Main](../Structures/Structures-6.htm#XREF_D_SE_0088760_1).iq\_lencEncoder input is set to 0.  Disable the verification of the period of the logical encoder by setting the [ST\_MultiCam](../Structures/Structures-9.htm#XREF_D_SE_0088766_1).i\_xCheckEncoderPeriod parameter to FALSE. |
