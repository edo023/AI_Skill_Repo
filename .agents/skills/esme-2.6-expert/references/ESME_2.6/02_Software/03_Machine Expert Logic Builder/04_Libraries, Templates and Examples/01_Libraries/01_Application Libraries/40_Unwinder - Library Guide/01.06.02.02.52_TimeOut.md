# TimeOut

TimeOut

|  |  |
| --- | --- |
| Enumeration name: | TimeOut |
| Enumeration value: | 71 |
| Description: | A TimeOut has occured. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The start movement mode ET\_StartMovementMode.Torque is chosen and the feedback signal does not rise above iq\_stParameter.stStartMovement.stTorque.i\_lrStartThreshold within the given time iq\_stParameter.stStartMovement.stTorque.i\_timTimeOut. | Check that the feedback signal from the tension sensor is transmitted to the POU.  Check that the foil is tense when the bobbin winds up the foil. (e.g. the foil is not cracked.)  Increase the time to reach the feedback signal iq\_stParameter.stStartMovement.stTorque.i\_timTimeOut.  Decrease the threshold for the feedback signal iq\_stParameter.stStartMovement.stTorque.i\_lrStartThreshold. |
