# VisuFbDateTimeNamesLocalizer.GetDayShort (METH)

METHOD GetDayShort : CharBufferPtr

returns the localized text of the day as short name. If no localization exists, a defaulttext (textid) should be returned. The stored value, the returnpointer is set to, will be overridden by any next call of one of the Get-methods of this functionblock. Therefore, the result should be used at once. The pointer should not be stored.
