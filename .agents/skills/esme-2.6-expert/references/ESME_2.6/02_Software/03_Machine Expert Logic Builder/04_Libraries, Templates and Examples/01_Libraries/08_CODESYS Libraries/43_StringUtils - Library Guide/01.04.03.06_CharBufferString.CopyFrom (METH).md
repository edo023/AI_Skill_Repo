# CharBufferString.CopyFrom (METH)

METHOD CopyFrom

Does a strcpy from the source string cbsSource into our instance at given positions and a given length. Note: No terminating 0 will be appended.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | rcbsSource | REFERENCE TO [CharBufferString](hfzJmOS2ddAPvg6yR-C9-xk02nE_charbufferstring.html#hfzjmos2ddapvg6yr_c9_xk02ne_charbufferstring_charbufferstring_fb) | The source string, where the characters will be copied from | | diSourceOffset | DINT | The 1 relative offset within the source string, where the copying will start | | diTargetOffset | DINT | The 1 relative offset within the source string, where the copying to will start | | diCountCharacters | DINT | The number of characters to copy | |
