# Overview of the Hardware Configuration

Overview of the Hardware Configuration

The project example implements a Modicon M251 Logic Controller and the two different Lexium servo drives: Lexium 32M and integrated Lexium ILA2K. The Lexium drives are linked to the controller via EtherNet/IP fieldbus as [EtherNet/IP](../glossary/glossary.htm#XREF_D_SE_0024697_694) targets. The controller is the EtherNet/IP scanner and implements the logic to control and monitor the drives over the fieldbus.
