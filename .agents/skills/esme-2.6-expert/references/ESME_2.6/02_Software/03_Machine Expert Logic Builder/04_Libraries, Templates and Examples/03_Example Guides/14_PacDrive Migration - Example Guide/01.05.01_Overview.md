# Overview

## Short Description

The PacDrive Migration Example Guide describes the functionalities of the PacDrive Migration project that is used to migrate PacDrive projects to the Modicon M660 Motion Controller platform.

## System Requirements and Limitations

The application example has been created and tested with the components listed in the table:

| Component | Type and Version |
| --- | --- |
| Software | EcoStruxure Machine Expert V2.5 |
| Controller | Modicon M660 Motion Controller |
