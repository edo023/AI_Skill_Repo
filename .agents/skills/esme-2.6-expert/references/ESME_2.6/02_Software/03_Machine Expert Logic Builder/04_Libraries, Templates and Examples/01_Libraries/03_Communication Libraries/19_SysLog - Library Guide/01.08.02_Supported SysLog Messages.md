# Supported SysLog Messages

## Message ID List

All messages presented in this chapter share the following parameters:

* Host name: FQDN
* APP name: Ref. Com.

The following structured messages are supported by this library. Modicon M262 Logic/Motion Controller produces messages which are documented in the Modicon M262 Logic/Motion Controller Programming Guide, see SysLog Messages from M262 Logic/Motion Controller.
