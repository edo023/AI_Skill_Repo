# FC\_SercosWriteServiceDataByTopAddr

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

The FC\_SercosWriteServiceDataByTopAddr function of the library SercosMasterM6xx writes data through the Sercos service channel to the Sercos device addressed with the specified topology address.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  * Verify that the correct Sercos device is addressed. * Verify that the correct Sercos parameter (IDN) is specified and that the parameter values to be written are correct.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_wTopoAddr | WORD | Topology address of the Sercos device |
| i\_dwIDN | DWORD | IDN to be written |
| i\_wType | WORD | Type of action through service channel   * 0: Write name * 1: Write attribute * 2: Write unit * 3: Write minimum value * 4: Write maximum value * 5: Write user data * 6: Write user data with variable length * 7: Execute command * 8: Start command * 9: Verify command * 10: Stop command |
| i\_lwPointerToData | LWORD | Pointer to memory buffer |
| i\_wMaxDataLen | WORD | Length of bytes to write |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: Error detected: The topology address is invalid.  -431: Error detected: Error detected during service request (for example, timeout).  -445: Error detected: Service timeout  -461: Error detected: In the active Sercos communication phase, writing parameters through the service channel is not supported.  -464: Error detected: Invalid service transfer. |
