## Overview

| --- | --- |

## Description

The method Calculate is called cyclically by the function block [FB\_MultiAxisCustomJob](FB_MultiAxisCustomJob-4CFA57F9.html) in every RTP cycle, starting with the cycle in which the Prepare method was executed.

This method is the primary location for your custom motion algorithm for the multi-axis custom job. In this method, you calculate the target position, velocity and acceleration for each subordinate axis. You must then apply these values to the respective axes by calling the WriteMovementValues method of the corresponding instances of the function block FB\_SlaveChannel instances.

The method provides the output q\_xDone. When your algorithm sets q\_xDone to TRUE, the multi-axis custom job is completed. In this case, the Calculate method is no longer called, and all subordinate axes in the group transition to the operating state Standstill.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
