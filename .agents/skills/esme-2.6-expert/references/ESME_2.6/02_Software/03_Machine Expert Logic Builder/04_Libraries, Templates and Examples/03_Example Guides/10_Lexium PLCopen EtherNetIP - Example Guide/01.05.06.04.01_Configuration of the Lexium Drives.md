# Configuration of the Lexium Drives

Configuration of the Lexium Drives

To operate the Lexium drives via EtherNet/IP, the communication parameters have to be set for the device. Further configuration settings are dependent on your application and on the installation.

For information about the configuration of the Lexium drives, refer to the associated manuals. A list of documents can be found in the section [Related Documents](../front/front-4.htm#XREF_D_SE_0088451_12).
