# IF\_TriggerTouchProbe

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | IF\_Trigger |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_TriggerTouchProbe allows you to perform position capture with a touchprobe.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| CaptureCount | UINT | Get | Reads the number of capture events during the last Sercos cycle. |
| CaptureEdge | [MOIN.ET\_CaptureEdge](../../../../../api/crossBook?lang=en-US&virtualBookName=MOIN&topicID=D_SE_0096920) | Get | Reads the type of the capture edge configured for position capture (rising edge, falling edge, both edges) via the enumeration MOIN.ET\_CaptureEdge of the MotionInterface library. |
| CaptureEdgeSet | [MOIN.ET\_CaptureEdgeSet](../../../../../api/crossBook?lang=en-US&virtualBookName=MOIN&topicID=MOIN::/ET_CaptureEdgeSet_EC27EB54) | Get/Set | Reads or writes the configured edge of the touchprobe input via the enumeration MOIN.ET\_CaptureEdgeSet of the MotionInterface library. |
| CaptureEdgeSetDelayed | BOOL | Get | Reads whether there is a delay on change of CaptureEdge.  Value FALSE (default): The change of the edge is set at the start of the function.  Value TRUE: The change of the edge is delayed from the start of the function. |
| CaptureOk | BOOL | Get | Indicates that the configured edge has been detected. |
| CaptureState | [MOIN.ET\_CaptureState](../../../../../api/crossBook?lang=en-US&virtualBookName=MOIN&topicID=ET_CaptureState_EC295B9D) | Get | Reads the state of the position capture function. |
| CaptureValue | LREAL | Get | Reads the value of a captured position. |
| SensorDelay | LREAL | Get/Set | Reads or writes the delay of a position sensor to be considered in position capture. |
| Value | BOOL | Get | Reads the voltage value at the touchprobe input.  FALSE corresponds to 0 V (input state LOW).  TRUE corresponds to 24 V (input state HIGH). |
