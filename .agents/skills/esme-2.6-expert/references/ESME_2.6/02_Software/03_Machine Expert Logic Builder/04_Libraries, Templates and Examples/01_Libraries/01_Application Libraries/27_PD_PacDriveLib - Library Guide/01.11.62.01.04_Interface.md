# Interface

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_plrMatrixStart | POINTEr TO LREAL | Pointer to the start address of the matrix to be processed |
| i\_diNumCols | DINT | Number of columns contained in the matrix |
| i\_diRow | DINT | Line containing the matrix element to be written |
| i\_diCol | DINT | Column containing the matrix element to be written |
| i\_lrValue | LREAL | The value which is to be written into the addressed matrix element. |
