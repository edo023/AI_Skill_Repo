## Task

Reading the emergency parameters.

## Description

With the method GetEmergencyParameter(…), the configured data for an emergency stop of the robot can be read.

NOTE: With the method GetEmergencyParameter, only data can be read:

* Written with the method SetEmergencyParameter or
* Written with the method SetEmergencyParameter2 and addressed with ET\_RobotComponent.All.
