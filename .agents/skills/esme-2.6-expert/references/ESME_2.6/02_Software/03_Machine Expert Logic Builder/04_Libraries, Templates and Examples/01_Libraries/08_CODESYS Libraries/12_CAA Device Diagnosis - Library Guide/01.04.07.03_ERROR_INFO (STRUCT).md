# ERROR\_INFO (STRUCT)

TYPE ERROR\_INFO : STRUCT

This structure contains an error code for a bus or device. The error information can be retrieved by [GetBusError](_i3Rclu0CxGIST3QKKbewmL3Y4o_getbuserror.html#_i3rclu0cxgist3qkkbewml3y4o_getbuserror_getbuserror_fb) and [GetDeviceError](jyrNEZTYIzIrX1TeqIWUiva0Awk_getdeviceerror.html#jyrneztyizirx1teqiwuiva0awk_getdeviceerror_getdeviceerror_fb).
