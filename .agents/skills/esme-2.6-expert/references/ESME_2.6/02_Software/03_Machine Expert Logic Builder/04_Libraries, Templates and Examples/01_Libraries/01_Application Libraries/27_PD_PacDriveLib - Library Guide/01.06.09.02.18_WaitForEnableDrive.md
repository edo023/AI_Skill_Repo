# WaitForEnableDrive

WaitForEnableDrive

|  |  |
| --- | --- |
| Enumeration name: | WaitForEnableDrive |
| Enumeration value: | 136 |
| Description: | Waiting for the drive to be enabled. |

The initialization of the function block is complete. Via the input i\_xEnableDrive, it is now possible to enable the position control for the axis.
