# Description

Description

Data structure for parameterization of the phase generator of a channel. The phase generator enables the user to change the phasing between the master object of a cam and the logical master position. The effect can be compared to that of a SetPos. Since the change is effected continuously, it can also be executed with a running cam at any point in time.

The change of phasing is introduced by a change of the phase generator reference position. The process is equivalent to a positioning with defined jerk.
