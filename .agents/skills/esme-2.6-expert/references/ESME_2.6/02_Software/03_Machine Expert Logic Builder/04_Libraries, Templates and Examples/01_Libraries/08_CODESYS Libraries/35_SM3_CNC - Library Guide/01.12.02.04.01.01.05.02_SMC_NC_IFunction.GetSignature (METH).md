# SMC\_NC\_IFunction.GetSignature (METH)

METHOD GetSignature

Returns the signature of the function, consisting of the type of the result, the number of arguments, and type of each argument.

Note that the type [SMC\_GVar\_Type](0AOpftHaTjaOBKEDUp2z_aWxY1Y_smc-gvar-type.html#0aopfthatjaobkedup2z_awxy1y_smc_gvar_type_smc_gvar_type_enum).T\_OTHER can be used for the return type and for the argument types. This means that an arbitrary type may be used (T\_BOOL, T\_LREAL, or T\_STRING), and that it must be the same type for all uses of T\_OTHER in this signature.
