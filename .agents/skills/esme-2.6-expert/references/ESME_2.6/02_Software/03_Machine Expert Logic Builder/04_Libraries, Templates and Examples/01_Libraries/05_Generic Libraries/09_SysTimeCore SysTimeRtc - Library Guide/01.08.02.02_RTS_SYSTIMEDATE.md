# RTS\_SYSTIMEDATE

## Data Type Description

The data type RTS\_SYSTIMEDATE is a structure which implements the elements needed to represent a high resolution date and time value.

This structure contains detailed information on the date and time, presented in a readable format (in contrast to the time stamp).

| Name | Type | Description |
| --- | --- | --- |
| wYear | UINT | Year |
| wMonth | UINT | Month, values 1 to12 (1=January, 12=December) |
| wDay | UINT | Day of month, values 1 to 31 |
| wHour | UINT | Hour of the day, values 0 to 23 |
| wMinute | UINT | Minute of the hour, values 0 to 59 |
| wSecond | UINT | Second of the minute, values 0 to 59 |
| wMilliseconds | UINT | Millisecond of the second, values 0 to 999 |
| wDayOfWeek | UINT | Day of the week, values 0 to 7 (1=Monday, 6=Saturday, 0 or 7=Sunday**\***)  **\*** The representation of this value depends on the controller. |
| wYday | UINT | Day of the year, values 1 to 366 (1=January 1st, 365 or 366=December 31st) |
