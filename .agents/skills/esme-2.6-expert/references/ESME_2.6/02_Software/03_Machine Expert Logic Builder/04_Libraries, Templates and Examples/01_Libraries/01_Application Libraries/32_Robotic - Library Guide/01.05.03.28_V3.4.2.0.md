## New Features

* **IF\_RobotConfigurationReadOnly**

  Reading the configuration data from the robot via the provided methods and properties.
* **IF\_RobotConfigurationAdvancedReadOnly**

  Reading the advanced configuration data from the robot via the provided methods and properties.
* **IF\_RobotConfigurationAdvanced - SetMotionProfileType()**

  Setting the motion profile type for the selected robot component.
* **IF\_RobotConfigurationAdvanced - GetMotionProfileType()**

  Reading the configured motion profile type used for the selected robot component.

The following properties are added:

* FB\_Robot./IF\_Robot.ifConfigurationReadOnly

  Reading the configuration data and advanced configuration data from the robot via the provided methods and properties.

## Modifications

IF\_RobotMotion.lrVelOverride: A new value of lrVelOverride is only accepted if the last value is fully processed (typically 3 Sercos cycles). In case the value is not accepted, an application logger message is generated.
