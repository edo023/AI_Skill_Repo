# ST\_Plane

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.0.1.0 |

## Description

The structure ST\_Plane represents a plane in a three-dimensional space. The plane is defined by the position vectors of the three points stBasePoint, stSpanPoint1, and stSpanPoint2. These points must not be on a straight line.

The following vectors are used as direction vectors for the parameter representation of the plane:

stDirection1 := stSpanPoint1 - stBasePoint

stDirection2 := stSpanPoint2 - stBasePoint

With the parameters Lambda1 and Lambda2, the parameter equation of the plane is:

Plane(Lambda1, Lambda2) := stBasePoint + Lambda1 \* stDirection1 + Lambda2 \* stDirection2
