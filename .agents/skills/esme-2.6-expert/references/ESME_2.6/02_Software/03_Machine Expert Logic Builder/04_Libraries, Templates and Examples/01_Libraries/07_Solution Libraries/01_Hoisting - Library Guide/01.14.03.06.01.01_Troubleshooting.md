# Troubleshooting

Troubleshooting

| Issue | Cause | Solution |
| --- | --- | --- |
| The bit 0 of q\_wAlrmId is TRUE. | Wrong order of the limit switch detected. | Check for the wiring of the limit switches. The function block is designed for the NC type of input signals from the limit switches only. |
| The bits 1...3 of q\_wAlrmId are TRUE. | The parameterization of the function block is incorrect if the stop on distance function is enabled. | Check the parameterization of the function block. |
| Raised alarm in first cycle for Limit switch error bit 0. | Information about limit switch status is transferred through the fieldbus. | Wait until the state of the limit switch is updated after power cycle of the controller before enabling the FB. |
