# i_xScrapEn

i\_xScrapEn

TRUE on this input enables the scraping function. Scraping is active only during automatic [closing on stack](../glossary/glossary.htm#XREF_D_SE_0024697_780). Ideally it is directly controlled by a modifier button on the joystick. It can share the button with the function of rapid opening. When the button gets released during scraping, the FB automatically changes to standard closing on stack.
