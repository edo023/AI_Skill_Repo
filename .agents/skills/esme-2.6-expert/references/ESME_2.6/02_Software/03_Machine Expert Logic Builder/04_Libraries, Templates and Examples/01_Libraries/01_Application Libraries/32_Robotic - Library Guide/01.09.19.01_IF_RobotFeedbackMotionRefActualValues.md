## Task

Interface with return values of the reference and actual values of a motion.

## Description

This interface is used by IF\_RobotFeedbackMotionCartesian as well as IF\_RobotFeedbackMotionAuxiliaryAxis in order to indicate reference and actual values of a motion from the respective Cartesian space direction and auxiliary axis.
