# DirRemove (FB; Prefix fildrm)

FUNCTION\_BLOCK DirRemove EXTENDS CBM.ETrigToA

This function block deletes a directory.

Contrary to the original CAA specification the directory may be removed even if it is opened via ”FILE.[DirOpen](ApmELEjJXGYoLzYq0bSC23LAm_0_diropen.html#apmelejjxgyolzyq0bsc23lam_0_diropen_diropen_fb_prefix_fildop)”.

This behavior is dependent of the underlying operating system and file system.
