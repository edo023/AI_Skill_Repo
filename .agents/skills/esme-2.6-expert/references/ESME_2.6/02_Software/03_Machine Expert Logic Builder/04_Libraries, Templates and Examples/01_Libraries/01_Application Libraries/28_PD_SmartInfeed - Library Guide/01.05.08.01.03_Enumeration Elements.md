# Enumeration Elements

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| VirtualTp | 0 | The position at which the simulated products would trigger a Touchprobe is calculated and transferred to the [FB\_Infeed](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080279_1). No cable bridges are required at the controller. |
| SignalsTp | 1 | The signals that would be triggered by the simulated products at a Touchprobe are determined and transferred to a digital output. In order for the [FB\_Infeed](../Function_Blocks/Function_Blocks-3.htm#XREF_D_SE_0080279_1) POU to receive the signals, the digital outputs must be connected to the Touchprobe inputs using cable bridges. |
| ShowTp | 2 | No signals due to simulated products are generated. The real signals are displayed in the visualization. |
