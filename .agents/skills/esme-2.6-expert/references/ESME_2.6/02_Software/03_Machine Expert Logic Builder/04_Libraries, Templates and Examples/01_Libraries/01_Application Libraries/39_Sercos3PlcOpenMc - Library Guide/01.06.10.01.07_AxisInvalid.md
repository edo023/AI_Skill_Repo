# AxisInvalid

AxisInvalid

|  |  |
| --- | --- |
| Enumeration name: | AxisInvalid |
| Enumeration value: | 25 |
| Description: | The connected drive is not supported. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | No drive has been applied at the input Axis. | A valid drive must be transferred to the input Axis. |
