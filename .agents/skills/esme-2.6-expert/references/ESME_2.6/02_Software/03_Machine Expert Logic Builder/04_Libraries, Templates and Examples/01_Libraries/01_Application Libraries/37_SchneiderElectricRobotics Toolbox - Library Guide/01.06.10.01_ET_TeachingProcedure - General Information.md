# ET\_TeachingProcedure - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.8.0.0 |

## Description

This enumeration type identifies a teaching procedure.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No teaching procedure is selected. |
| FixedSystem | 1 | Teaching procedure for a fixed coordinate system. |
| LinearMotionSystem | 2 | Teaching procedure for a linear motion coordinate system.  A conveyor belt is an example of such kind of system. |
