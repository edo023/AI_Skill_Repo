# INTEGRAL (FB)

FUNCTION\_BLOCK INTEGRAL

determines approximately the integral over time.

Example: Function block in FBD

| InOut: | | Scope | Name | Type | Initial | Comment | | --- | --- | --- | --- | --- | | Input | IN | REAL |  | Input value | | TM | DWORD |  | Time since last call in msec | | RESET | BOOL |  | TRUE: OUT is set to zero and OVERFLOW to FALSE | | Output | OUT | REAL |  | Value of the integral. This is done by summing all part integrals IN \* TM | | OVERFLOW | BOOL | FALSE | TRUE: The value of OUT is out of range of REAL variables. he function is blocked till it is new initialised by input RESET. | |
