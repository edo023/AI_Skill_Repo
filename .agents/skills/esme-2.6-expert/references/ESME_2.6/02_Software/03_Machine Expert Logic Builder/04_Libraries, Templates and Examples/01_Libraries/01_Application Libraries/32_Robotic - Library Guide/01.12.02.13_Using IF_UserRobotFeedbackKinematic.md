# Using IF\_UserRobotFeedbackKinematic

## Example

The example of using the IF\_UserRobotFeedbackKinematic interface is based on a user-defined transformation function block that implements the IF\_UserTransformation3Ax interface.

| Step | Action |
| --- | --- |
| 1 | In addition to the IF\_UserTransformation3Ax interface, implement the IF\_UserRobotFeedbackKinematic interface for the user-defined transformation function block.  The IF\_UserRobotFeedbackKinematic interface enables the function block to forward the reference position of the robot joints. This information is provided at the robot feedback interface IF\_RobotFeedback.ifKinematic.ralrRefPositionJoints. |
| 2 | Define the methods Direct and Inverse with the interface IF\_UserTransformation3Ax.  Define the property ralrRefPositonJoint with the interface IF\_UserRobotFeedbackKinematic. |
| 3 | Add a local variable with name lrRefPositionJoint of type ARRAY [ET\_RobotComponent.AxisA .. ET\_RobotComponent.AxisAll+Gc\_udiMaxNumberOfAxes] OF LREAL to the function block. |
| 4 | In the get-accessor, add the code `ralrRefPositionJoint REF= THIS^.alrRefPositionJoint`. |
| 5 | Add the code to calculate the values for the RefPositionJoint feedback to the Direct and Inverse methods so that the feedback is available in the enabled and disabled cases of the robot. |

Also refer to [IF\_UserRobotFeedbackKinematic - General Information](IF_UserRobotFeedbackKinematic-Gener-B50D2975.html#IF_UserRobotFeedbackKinematic-Gener-B50D2975).
