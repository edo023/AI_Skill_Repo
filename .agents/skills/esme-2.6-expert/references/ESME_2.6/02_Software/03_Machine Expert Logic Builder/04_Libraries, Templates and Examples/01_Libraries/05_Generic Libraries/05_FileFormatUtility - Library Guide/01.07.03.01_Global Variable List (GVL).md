# Global Variable List (GVL)

## Overview

|  |  |
| --- | --- |
| Type: | Global variables |
| Available as of: | V1.0.0.0 |

## Description

The global variables list contains the global variables of the library.

## Global Variables

| Variable | Data Type | Default value | Description |
| --- | --- | --- | --- |
| `Gc_xOnlineChangeAllowed` | BOOL | FALSE | If TRUE, the online change detection is successfully initialized. Execute an online change while the POU is busy only if Gc\_xOnlineChangeAllowed = TRUE. |
