# WORD\_AS\_STRING (FUN)

FUNCTION WORD\_AS\_STRING : STRING(2)

Converts one word into its ASCII string representation

Example:

```
Input:  '16#48_46'
Byte order: FALSE
->
String: 'FH'
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | WORD\_AS\_STRING | STRING(2) | A STRING(2) representing the HEX code of the input W | | Input | W | WORD | Input value | | ORDER | BOOL | TRUE: Swaps high and low byte | |
