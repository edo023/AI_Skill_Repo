# P5RangeHighRange

P5RangeHighRange

|  |  |
| --- | --- |
| Enumeration name: | P5RangeHighRange |
| Enumeration value: | 158 |
| Description: | P5RangeHigh is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The overlapping of the polynomial intervals was selected so that the X part of a polynomial becomes 0. | The inputs iq\_stCrankData.lrP5Pole1IntervalLow, iq\_stCrankData.lrP5Pole1IntervalHigh, iq\_stCrankData.lrP5Pole2IntervalLow and iq\_stCrankData.lrP5Pole2IntervalHigh must be selected so that not the entire interval range for iq\_stCrankData.lrP5Pole1IntervalHigh and iq\_stCrankData.lrP5Pole2IntervalHigh is forced into a straight line due to overlapping. |
