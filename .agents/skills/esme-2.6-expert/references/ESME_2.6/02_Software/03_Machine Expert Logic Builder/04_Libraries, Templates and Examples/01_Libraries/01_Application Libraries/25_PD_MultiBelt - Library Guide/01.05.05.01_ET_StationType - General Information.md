# ET\_StationType - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V1.0.0.0 |
| Versions: | Current version |

## Description

This enumeration contains the different station types that can be used.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Indexed | 1 | The station works in indexed mode. The train stops and waits for a start signal. If the start signal is recognized, the train makes one step and waits for the next signal. See [Indexed station](D-SE-0077880.html#D-SE-0077880). |
| Synchronous | 2 | The station works in synchronous mode. The trains are moving to a master synchronously. They can move over start signals or phase synchronously. See [Synchronous station](D-SE-0077882.html#D-SE-0077882). |
| PassBy | 3 | The station works in run-through mode. The train is not stopped, it is moved through the station with the Move parameters. The motion cannot be influenced by external signals. See [PassBy station](D-SE-0077881.html#D-SE-0077881). |
