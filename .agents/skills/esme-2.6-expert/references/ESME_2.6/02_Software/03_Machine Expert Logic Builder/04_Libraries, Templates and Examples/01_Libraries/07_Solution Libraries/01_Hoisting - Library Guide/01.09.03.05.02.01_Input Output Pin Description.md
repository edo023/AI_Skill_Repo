# Input/Output Pin Description

Input/Output Pin Description

| Input/Output | Data Type | Description |
| --- | --- | --- |
| iq\_wExeCntActl | WORD | This pin is used for testing the number of executions of sub-programs within one cycle of a main cyclic task. Connect a global variable to this input. The global variable must be incremented by one within each called sub-program. The number of sub-programs must correspond to the value of i\_wExeCntCfg. The DiagnosticCoverage FB may be executed at any position in the program.  Range: 0...65535 |
