# FC\_CommonCircle3DTwoSpheres

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function calculates the intersection circle of two spheres in the three-dimensional space, that is, the circle in which the spherical surfaces intersect. If the spheres intersect in one point, the radius of the intersection circle is 0. If the spheres do not intersect or if they have a common center, an error is detected.
