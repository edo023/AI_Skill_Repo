# Visu\_FbWebServerFileProtection.FB\_INIT (METH)

METHOD FB\_INIT : BOOL

FB\_Init is always available implicitly and it is used primarily for initialization. The return value is not evaluated. For a specific influence, you can also declare the methods explicitly and provide additional code there with the standard initialization code. You can evaluate the return value.
