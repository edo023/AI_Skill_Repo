# Custom-Tuning (Heating) - FB\_HeatingControl

## Overview

This procedure describes how to set the PID parameter manually.

| Step | Action |
| --- | --- |
| 1 | Set iq\_etSelectParameterSetPID to ET\_ParameterSetPID.Custom. |
| 2 | Set rTn, rTv, and rTd to 0.  Also refer to [ST\_PID](D-SE-0106257.html#D-SE-0106257). |
| 3 | Increase rKp until the system begins to oscillate. |
| 4 | Decrease rKp until the system does not oscillate. |
| 5 | Decrease rKp by another few points. |
| 6 | Set rTn, beginning with a large value for time, for example 300 s (system dependent) and verify that the system does not oscillate. |
| 7 | Decrease rTn until the system is stable and without oscillation.  **Result**: In the figure below a typically correct regulation is shown. |

Also refer to chapter [*FB\_HeatingControl*](D-SE-0106246.html#D-SE-0106246).
