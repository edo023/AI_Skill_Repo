# RemoteAdapter\_diag.CheckSupportedCommunicationState (METH)

METHOD CheckSupportedCommunicationState : BOOL

This method checks if a specific DEVICE\_TRANSITION\_STATE is supported by the node in general.

**Note**

Use IDevice2.CheckCurrentSupportedCommunicationState to check which DEVICE\_TRANSITION\_STATE is possible in the current state.
