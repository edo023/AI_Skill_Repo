# SMC\_NC\_IFunction (ITF)

INTERFACE SMC\_NC\_IFunction

Interface for functions that can be called from G-Code. By implementing a function block of this type, it is possible to add additional functions that can be called in expressions in the G-Code. These function blocks need to be passed to the [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb) or [SMC\_ReadNCFromStream](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfromst.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfromst_smc_readncfromstream_fb) via the input pCustomFunTable.
