# FC\_EtHomeModeToString - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.4.0 |
| Versions: | Current version |

## Task

Convert the ET\_HomeMode enumeration type of the FB\_MultiBeltModule to a string.

## Description

The i\_etHomeMode input from ET\_HomeMode is converted to a string.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etHomeMode | [ET\_HomeMode](D-SE-0079096.html#D-SE-0079096) | Homing mode of the FB\_MultiBeltModule |

| Output | Data type | Description |
| --- | --- | --- |
| q\_etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | General library-independent statement on the diagnostic.  A value unequal to GD.ET\_Diag.Ok corresponds to a diagnostic message. |
| q\_etDiagExt | [ET\_DiagExt](D-SE-0079094.html#D-SE-0079094) | POU-specific output on the diagnostic.  q\_etDiag = GD.ET\_Diag.Ok -> status message  q\_etDiag <> GD.ET\_Diag.Ok -> diagnostic message |

## Return Value

| Data type | Description |
| --- | --- |
| STRING [25] | HomeMode of the function block as a string |

## Diagnostic Messages

| q\_etDiag | q\_etDiagExt | Enumeration value | Description |
| --- | --- | --- | --- |
| OK | Ok | 0 | Ok |
| InputParameterInvalid | HomeModeUnknown | 12 | The homing mode is invalid. |

## HomeModeUnknown

|  |  |
| --- | --- |
| Enumeration name: | HomeModeUnknown |
| Enumeration value: | 12 |
| Description: | The homing mode is invalid. |

| Issue/Cause | Solution |
| --- | --- |
| An indeterminable homing mode was applied at the i\_etHomeMode input. | Ensure that only elements from the ET\_HomeMode list are transferred at the i\_etHomeMode input. |

## Ok

|  |  |
| --- | --- |
| Enumeration name: | Ok |
| Enumeration value: | 0 |
| Description: | Ok |

The HomeMode has successfully been converted to a string.
