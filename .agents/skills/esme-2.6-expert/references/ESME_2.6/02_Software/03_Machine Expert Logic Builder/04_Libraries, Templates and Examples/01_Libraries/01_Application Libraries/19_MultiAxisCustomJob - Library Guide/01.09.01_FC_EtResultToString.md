# FC\_EtResultToString

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

The function FC\_EtResultToString allows you to convert an enumeration element of type ET\_Result to a variable of type STRING.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etResult | [ET\_Result](ET_Result-4C5F7ABA.html#ET_Result-4C5F7ABA) | Enumeration with the result. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error was detected. |
| q\_etResult | [ET\_Result](ET_Result-4C5F7ABA.html#ET_Result-4C5F7ABA) | Result of the function block execution. Refer to [ET\_Result](ET_Result-4C5F7ABA.html). |

## Return Value

| Data type | Description |
| --- | --- |
| STRING(80) | String converted from the ET\_Result element. |
