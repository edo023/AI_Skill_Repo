# Runtime Class

Runtime Class

There are nine run time classes (V 0.06 to V 5 for FEM and T 0 to T 8 for ISO). These classes define the mean time of usage of the hoist per day.

| Run time class | | Mean daily run time [h] |
| --- | --- | --- |
| FEM 9.511 | ISO 4301-1 |
| V0.06 | T 0 | ≤0.12 |
| V 0.12 | T 1 | ≤0.25 |
| V 0.25 | T 2 | ≤0.5 |
| V 0.5 | T 3 | ≤1 |
| V 1 | T 4 | ≤2 |
| V 2 | T 5 | ≤4 |
| V 3 | T 6 | ≤8 |
| V 4 | T 7 | ≤16 |
| V 5 | T 8 | > 16 |
