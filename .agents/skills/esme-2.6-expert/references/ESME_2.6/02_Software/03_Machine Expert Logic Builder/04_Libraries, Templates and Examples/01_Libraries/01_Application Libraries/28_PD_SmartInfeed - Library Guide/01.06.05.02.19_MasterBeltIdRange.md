# MasterBeltIdRange

MasterBeltIdRange

|  |  |
| --- | --- |
| Enumeration name: | MasterBeltIdRange |
| Enumeration value: | 52 |
| Description: | MasterBeltId is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The belt configuration of the infeed POU transferred at the i\_ifInfeed input is not correct. | The belts from iq\_stParameter.udiFirstVelBeltIndex to iq\_stParameter.udiLastVelBeltIndex of the infeed POU transferred at the i\_ifInfeed input must all have been allocated the last belt of the infeed distance as cam master. |
