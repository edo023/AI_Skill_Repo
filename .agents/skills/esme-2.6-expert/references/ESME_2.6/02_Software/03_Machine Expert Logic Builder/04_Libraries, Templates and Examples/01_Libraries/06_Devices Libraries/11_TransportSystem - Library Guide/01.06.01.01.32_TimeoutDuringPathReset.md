# TimeoutDuringPathReset

TimeoutDuringPathReset

|  |  |
| --- | --- |
| Enumeration name: | TimeoutDuringPathReset |
| Enumeration value: | 35 |
| Description: | A timeout occurred during path reset. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The system is unable to complete the command. | Increase the default value of i\_timControllerResponseTimeout in [ST\_ControllerParameter](../Structures/Structures-2.htm#XREF_D_SE_0090700_1) to allow the transport system controller more time to respond to incoming commands. |
