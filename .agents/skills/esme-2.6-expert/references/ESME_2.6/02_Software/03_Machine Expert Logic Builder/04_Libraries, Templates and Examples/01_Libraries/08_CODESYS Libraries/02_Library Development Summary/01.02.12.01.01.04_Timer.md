### Timer

The handling of a time duration at the basis of the data types TIME and LTIME are also the kernel of the timer function blocks in the libraries standard (for TIME) and standard64 (for LTIME).

**Example**

```
PROGRAM TIMER
VAR
    tDelta : TIME := T#2s;
    T1 : TON;
    T2 : TP;
    T3 : TOF;
END_VAR
```
