# General

## Overview

The library provides several visualization frames.

You can use these frames to display information about:

* Alarms
* Machine operation modes and states
* Production data

## FB\_VisController Function Block

The main task of the function block FB\_VisController is to process data provided by the administration tags and status tags and to provide these to the visualization frames via the visualization interface input/output iq\_stVisInterface of type ST\_VisInterface.

## Visualization Frames

* FR\_Alarm
* FR\_AlarmHistory
* FR\_CurrentModeAndStateTime
* FR\_DynStateModel
* FR\_ModeAndStateTime
* FR\_ProdConsumedCount
* FR\_ProdDefectiveCount
* FR\_ProdProcessedCount
* FR\_Warning
* FR\_StopReason
* FR\_StatesDisabled
