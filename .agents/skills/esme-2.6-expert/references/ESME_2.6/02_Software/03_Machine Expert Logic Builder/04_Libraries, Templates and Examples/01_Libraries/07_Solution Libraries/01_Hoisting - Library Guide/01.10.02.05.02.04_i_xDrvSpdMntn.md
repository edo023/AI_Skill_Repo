# i_xDrvSpdMntn

i\_xDrvSpdMntn

Command to maintain the actual speed reference. This input is used in four-wire mode. In three-wire mode it does not have any effect.

If this input is TRUE together with i\_xDrvAcc and a direction command, the FB is ramping up the speed reference in requested direction.

If this input is TRUE but i\_xDrvAcc is FALSE and direction command in actual direction of movement is present, the FB keeps the speed reference.

If this input is FALSE and direction command in actual direction of movement is present, the FB ramps the speed reference towards i\_wDrvSpdLsp.
