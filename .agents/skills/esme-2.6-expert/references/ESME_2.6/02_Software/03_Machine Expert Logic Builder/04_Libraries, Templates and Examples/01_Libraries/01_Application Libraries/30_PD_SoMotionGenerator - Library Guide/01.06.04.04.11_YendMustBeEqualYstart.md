# YendMustBeEqualYstart

YendMustBeEqualYstart

|  |  |
| --- | --- |
| Enumeration name: | YendMustBeEqualYstart |
| Enumeration value: | 69 |
| Description: | Yend and Ystart must be equal. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | In the case of two consecutive segments of a segmented cam lrYend <> lrYStart. | Set lrYend of the previous segment equal to lrYStart of the subsequent segment.  The q\_sMsg output shows which segments are incorrect. |
