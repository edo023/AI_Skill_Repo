# ErrorCode1\_RW (ENUM)

TYPE ErrorCode1\_RW :

Some encodings of ErrorCode1 of STATUS (byte 3) that are use for acyclic read-/write services.

Could be used for access errors in [RCVREC](QAQf2JmdvASMLRmr7NVT6FU1WYM_rcvrec1.html#qaqf2jmdvasmlrmr7nvt6fu1wym_rcvrec1_rcvrec_fb) or [PRVREC](QAQf2JmdvASMLRmr7NVT6FU1WYM_prvrec1.html#qaqf2jmdvasmlrmr7nvt6fu1wym_prvrec1_prvrec_fb) function blocks, for example.
