# FC\_GenericResponse - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.2.0 |

## Task

This function receives the response to an S/IP service from a Sercos device.

## Functional Description

This function receives the response to an S/IP service from a Sercos device to which a request had been sent via FC\_GenericRequest. Allocated memory must be provided to the function via the parameter iq\_stData.

The function copies the number of bytes specified in uiLength of [ST\_Data](ST_Data-GeneralInformation-089965B9.html) to the reserved memory of pbyData.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiTransactionID | UDINT | Transaction ID of corresponding request |
| i\_diTimeoutMs | DINT | Timeout in milliseconds for waiting for the response |

| Output | Data type | Description |
| --- | --- | --- |
| q\_udiBytesReceived | UDINT | Bytes to be received by Sercos device (needs to be equal to or less than the length uiLength in the structure [ST\_Data](ST_Data-GeneralInformation-089965B9.html)). |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stData | ST\_Data | Data to be received by response. See [ST\_Data](ST_Data-GeneralInformation-089965B9.html) for additional information. |
