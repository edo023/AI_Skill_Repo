# TwidoEmulation­Support Library

![](images/logo.png)

TwidoEmulation­Support Library

TwidoEmulationSupport Library Guide

This document describes the TwidoEmulationSupport library, which provides the functions and function blocks to convert projects from EcoStruxure Machine Expert - Basic / TwidoSoft / TwidoSuite into a compatible implementation in EcoStruxure Machine Expert. For more information on Twido, refer to [Converting EcoStruxure Machine Expert - Basic and Twido Projects](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083381_1) and to the [Controller Feature Sets for Migration lists](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083878_1).
