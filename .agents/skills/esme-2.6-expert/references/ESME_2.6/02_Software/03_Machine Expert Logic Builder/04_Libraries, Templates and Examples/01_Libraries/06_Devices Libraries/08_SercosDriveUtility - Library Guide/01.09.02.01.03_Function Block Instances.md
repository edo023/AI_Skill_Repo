# Function Block Instances

Function Block Instances

Even if it is possible to create several instances of FB\_ParameterUpload and/or FB\_Parame­terDownload, it is suggested working with only one function block instance for each function.

You can execute several instanced function blocks in parallel (to handle diffrent drives at the same time). However, due to the asynchronous communication restrictions of Sercos, there is no significant performance improvement.

Executing several function blocks in parallel also requires independent CSV files due to the restricted file access. Refer to File Handling below.
