# DEVICE\_STATE (ENUM)

TYPE DEVICE\_STATE :

Describes the state of the CANopen network layer, as defined in /DS-301/. The states INIT, RESET\_COMM, RESET\_APP, PRE\_OPERATIONAL, STOPPED, OPERATIONAL correspond to the same states in /DS-301/. The state UNKNOWN is used, if the actual state of the device is not known (for example, if no guarding is performed). The state NOT\_AVAIL is used, if it is known, that the device is not available (for example, if guarding is performed and the device does not answer).
