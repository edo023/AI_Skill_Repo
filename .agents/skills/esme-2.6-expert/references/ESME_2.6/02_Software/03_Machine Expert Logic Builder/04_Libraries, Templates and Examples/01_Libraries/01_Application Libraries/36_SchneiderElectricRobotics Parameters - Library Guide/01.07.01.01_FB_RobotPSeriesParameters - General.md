## Task

The POU contains the parameters for a Lexium P Robot.

## Description

The POU contains applicable parameters for a Lexium P Robot that has been selected by the methods InitializeRobot, InitializeRotationalAxis and OptionalEquipment.

## Methods

| Name | Description |
| --- | --- |
| InitializeRobot | Initialize the robot and its axes. |
| InitializeRotationalAxis | Initialize the rotational axis of the robot, if necessary. |
| OptionalEquipment | Set optional equipment of the robot. |
| SetLRealParameter | Set/overwrite a parameter for a Lexium P Robot. |
