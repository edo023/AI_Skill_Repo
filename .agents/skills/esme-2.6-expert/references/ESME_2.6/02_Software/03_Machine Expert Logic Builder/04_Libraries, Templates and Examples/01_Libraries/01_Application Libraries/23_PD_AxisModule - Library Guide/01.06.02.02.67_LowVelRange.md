# LowVelRange

LowVelRange

|  |  |
| --- | --- |
| Enumeration name: | LowVelRange |
| Enumeration value: | 83 |
| Description: | LowVel is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.PosDirectionPosEdgeHWLimitPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1) ,  [PDL.ET\_HomeMode.NegDirectionPosEdgeHWLimitNeg](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionNegEdgeHWLimitNeg](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1) or  [PDL.ET\_HomeMode.PosDirectionNegEdgeHWLimitPos](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).   An invalid value was transferred to the [ST\_Home.stLimitSwitch.i\_lrLowVel](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) parameter. | A value greater than 0 and smaller than or equal to the MaxVel parameter of the drive must be transferred to the [ST\_Home.stLimitSwitch.i\_lrLowVel](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) parameter. |
