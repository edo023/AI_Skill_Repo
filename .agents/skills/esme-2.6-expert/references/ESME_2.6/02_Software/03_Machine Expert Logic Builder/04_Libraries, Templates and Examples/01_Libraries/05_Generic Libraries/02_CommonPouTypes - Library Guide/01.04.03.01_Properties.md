# Properties

The CommonPouTypes library provides the following common properties:

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| etPreviousState | [ET\_State](ET_State-CC1E0BCE.html) | Read | Provides the previous state of the internal state machine. The property is updated once the state changes. |
| timTimeout | TIME | Read, Write | Reading and writing the time value for the timeout function. |
| xTimeoutExpired | BOOL | Read | Reading the status of the timeout function. If set to TRUE, the timeout has expired. |

The table provides an overview of which property is provided by which base function block:

| Property | FB\_EnableReady | FB\_Execute | FB\_ExecuteAbort | FB\_EnableExecute | FB\_EnableExecuteQuit |
| --- | --- | --- | --- | --- | --- |
| etPreviousState | X | X | X | X | X |
| timTimeout | - | X | X | X | X |
| xTimeoutExpired | - | X | X | X | X |
