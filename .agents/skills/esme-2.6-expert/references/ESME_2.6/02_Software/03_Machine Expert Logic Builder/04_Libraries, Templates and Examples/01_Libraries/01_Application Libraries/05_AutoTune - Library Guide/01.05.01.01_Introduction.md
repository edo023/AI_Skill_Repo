# Introduction

Introduction

The PacDrive™3 automation system provides the optimal solution for electronic packaging machines, both in terms of technology and economy. It offers all advantages required by fast, dynamic systems.

The usual requirements are short acceleration and braking times with the broadest possible range of speed settings. The operating performance must always be as smooth and vibration-free as possible when connected to large external masses, whether connected with an elastic connection, such as a belt, or a general coupling.

The PacDriveTM3 System provides the appropriate operating behavior for a majority of applications even with the default settings or by slightly modifying the controller parameters.

In high-dynamic systems or systems with a wide range of resonance frequencies in the control circuit, it can be very difficult to establish an overall configuration which meets the requirements.

The automatic control optimization aims to solve this with the following goals:

oAutomatic adjustment of the controllers to different mechanical systems

oStability of the control circuits

oDefined transient response (dynamics)

oDamping of resonance frequencies

The AutoTune library described in the following is a powerful automatic control optimization system, which determines optimal control parameters simply and reliably.
