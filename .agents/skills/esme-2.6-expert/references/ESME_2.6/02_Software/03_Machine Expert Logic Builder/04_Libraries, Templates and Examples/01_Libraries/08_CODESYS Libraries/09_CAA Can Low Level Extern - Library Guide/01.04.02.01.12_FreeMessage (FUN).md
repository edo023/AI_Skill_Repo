# FreeMessage (FUN)

FUNCTION FreeMessage : [ERROR](gBaR_WSLC7pYiPIcZ7U3-TDNYt0_error.html#gbar_wslc7pyipicz7u3_tdnyt0_error_error_enum)

This function releases a message that has been created with [CreateMessage](UzaP1MMIvsGx5wgQfvzFSZKw87g_createmessage.html#uzap1mmivsgx5wgqfvzfszkw87g_createmessage_createmessage_fun), read with [Read](UzaP1MMIvsGx5wgQfvzFSZKw87g_read.html#uzap1mmivsgx5wgqfvzfszkw87g_read_read_fun) or duplicated with [CloneMessage](UzaP1MMIvsGx5wgQfvzFSZKw87g_clonemessage.html#uzap1mmivsgx5wgqfvzfszkw87g_clonemessage_clonemessage_fun).

**Note**

Do not free a message after it has been passed to the CL2 layer.

Example: Creating a message, then passing the message handle to CL2 by calling [Write](UzaP1MMIvsGx5wgQfvzFSZKw87g_write.html#uzap1mmivsgx5wgqfvzfszkw87g_write_write_fun) and releasing it afterwards may result in unexpected behaviour.
