# Description

Description

The function implements a PI controller. A positive signal at i\_xStart enables the function block. The input signal i\_lrInput is calculated from the difference i\_lrInput:= (reference value - actual value). The parameters i\_lrKP and i\_timTI must be chosen according to the enabled power stage. The output setpoint is calculated according to the following formula from the current input parameters and the previous progression.

q\_lrOutput := KP \* ( i\_lrInput + I(i\_lrInput) / TN)

Where i() is the integral over time.
