## Task

This method sets the base pose of the entity in the Cartesian space with reference to a global coordinate system.

## Description

The Cartesian pose is defined as a position, described as a 3D-vector and an orientation, described with a 3D-rotation matrix.

The function GEM.FC\_OrientationToRotationMatrix can be used for the conversion of the orientation from a roll, pitch and yaw representation to a 3D-rotation matrix representation required by i\_stBaseOrientation.
