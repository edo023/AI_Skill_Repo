# UASecurityPolicy

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration UASecurityPolicy specifies the security profile applied to the connection to the OPC UA server.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| UASP\_BestAvailable | 0 | Highest available security connection to the OPC UA server. The client receives the available policies from the server and selects the highest. Note that this can also result in level UASP\_None. |
| UASP\_None | 1 | No security profile is applied to the connection to the OPC UA server. |
| UASP\_Basic128Rsal5 | 2 | Basic128Rsa15 profile. |
| UASP\_Basic256 | 3 | Basic256 profile. |
| UASP\_Basic256Sha256 | 4 | Basic256Sha256 profile. |
