# FeedbackMaxReached

FeedbackMaxReached

|  |  |
| --- | --- |
| Enumeration name: | FeedbackMaxReached |
| Enumeration value: | 58 |
| Description: | Maximum feedback is reached. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The feedback signal received via the parameters in ST\_FeedbackSignal was for more than the timespan iq\_stParameter.stFeedbackSignal.i\_timMaxLevelTime in the zone iq\_stParameter.stFeedbackSignal.i\_lrMaxLevelZone around the maximum level of the feedback range. This means, the function block is not able to control the bobbin within the parametrized tolerances. | The parameters which influence the control dynamic of the function block has to be adapted to the application during the commissioning of the machine.  Decrease the tolerance zone i\_lrMaxLevelZone.  Increase the tolerance time i\_timMaxLevelTime. |
