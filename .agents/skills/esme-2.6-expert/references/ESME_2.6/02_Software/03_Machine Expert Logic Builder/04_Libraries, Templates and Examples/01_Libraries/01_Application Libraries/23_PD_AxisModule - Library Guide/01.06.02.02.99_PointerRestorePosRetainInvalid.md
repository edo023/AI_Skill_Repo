# PointerRestorePosRetainInvalid

PointerRestorePosRetainInvalid

|  |  |
| --- | --- |
| Enumeration name: | PointerRestorePosRetainInvalid |
| Enumeration value: | 88 |
| Description: | The RestorePosRetain pointer is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.RestorePosFromRetain](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).  An invalid pointer has been transferred to the [ST\_Home.stSetPos.iq\_pstRestorePosRetain](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) parameter. | The memory address that contains the structure in which the axis positions are saved must be transferred to the [ST\_Home.stSetPos.iq\_pstRestorePosRetain](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) parameter. |
