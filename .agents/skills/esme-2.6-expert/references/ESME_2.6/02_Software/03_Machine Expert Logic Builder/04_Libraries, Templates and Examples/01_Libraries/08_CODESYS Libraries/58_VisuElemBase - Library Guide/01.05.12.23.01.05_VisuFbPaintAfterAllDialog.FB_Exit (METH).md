# VisuFbPaintAfterAllDialog.FB\_Exit (METH)

METHOD FB\_Exit : BOOL

FB\_Exit must be implemented explicitly. If there is an implementation, then the method is called before the controller removes the code of the function block instance (implicit call). The return value is not evaluated.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | FB\_Exit | BOOL |  | | Input | bInCopyCode | BOOL | TRUE: the exit method is called in order to leave the instance which will be copied afterwards (online change). | |
