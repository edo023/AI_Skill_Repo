# RS (FB)

FUNCTION\_BLOCK RS

Realizes a bistable reset-dominat latch

```
(* Example declaration *)
RSInst : RS ;

(* Example in ST *)
RSInst(SET:= VarBOOL1 , RESET1:=VarBOOL2 );
VarBOOL3 := RSInst.Q1 ;
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | SET | BOOL | Rising edge: Set Q1 to TRUE | | RESET1 | BOOL | Rising edge: Reset Q1 to FALSE (dominant) | | Output | Q1 | BOOL | Output value | |
