# R\_TRIG (FB)

FUNCTION\_BLOCK R\_TRIG

Detects a rising edge of a boolean signal

```
(* Example declaration *)
RTRIGInst : R_TRIG ;

(* Example in ST *)
RTRIGInst(CLK := VarBOOL1);
VarBOOL2 := RTRIGInst.Q;
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | CLK | BOOL | Boolean signal to be checked | | Output | Q | BOOL | TRUE: Rising edge detected | |
