# MissingExtInit

MissingExtInit

|  |  |
| --- | --- |
| Enumeration name: | MissingExtInit |
| Enumeration value: | 174 |
| Description: | The additional interface has not been initialized correctly. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The initialization of the additional interfaces Ext is unavailable. | Ensure that interfaces of the modules are initialized.  Ensure that the [FC\_UpdateAddInterfaceStruct](../Functions/Functions-3.htm#XREF_D_SE_0088750_1) function is called up for the module. |
