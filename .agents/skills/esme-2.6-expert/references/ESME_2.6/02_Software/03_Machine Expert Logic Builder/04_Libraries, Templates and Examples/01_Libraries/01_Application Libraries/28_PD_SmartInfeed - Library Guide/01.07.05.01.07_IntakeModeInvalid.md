# IntakeModeInvalid

IntakeModeInvalid

|  |  |
| --- | --- |
| Enumeration name: | IntakeModeInvalid |
| Enumeration value: | 121 |
| Description: | An invalid InTakeMode has been selected. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | A number has been transferred at the input i\_etIntakeMode the value of which is not allocated to any element of [ET\_InTakeMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0080269_1). | An element of the enumeration [ET\_InTakeMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0080269_1) must be transferred at the input i\_etIntakeMode. |
