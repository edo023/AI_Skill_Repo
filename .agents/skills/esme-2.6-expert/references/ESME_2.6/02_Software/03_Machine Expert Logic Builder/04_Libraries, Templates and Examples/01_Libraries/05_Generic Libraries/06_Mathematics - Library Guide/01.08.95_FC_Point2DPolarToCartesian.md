# FC\_Point2DPolarToCartesian

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function converts planar polar coordinates into two-dimensional Cartesian coordinates. The pole of the polar coordinate system corresponds to the origin of the Cartesian coordinate system, and the horizontal axis of the polar coordinate system corresponds to the X axis of the Cartesian coordinate system.
