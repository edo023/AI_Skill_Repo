# ST\_StandardMotionInterface - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.4.0 |
| Inherits from: | - |

## Description

This interface can be used for the structured integration of the listed variables into an additional interface.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_udiModuleSelect | UDINT | Module ID |
| i\_xJogForw | BOOL | TRUE: The selected module shall move forward in jog mode. |
| i\_xJogBack | BOOL | TRUE: The selected module shall move backward in jog mode. |
