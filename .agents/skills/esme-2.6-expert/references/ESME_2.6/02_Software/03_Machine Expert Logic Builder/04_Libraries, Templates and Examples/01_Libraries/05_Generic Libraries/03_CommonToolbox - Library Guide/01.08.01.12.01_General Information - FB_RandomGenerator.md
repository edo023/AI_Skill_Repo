# General Information - FB\_RandomGenerator

## Overview

|  |  |
| --- | --- |
| Type: | Function block |
| Available as of: | V1.0.0.0 |

## Task

The function block FB\_RandomGenerator generates a pseudo-random number, herein referred to as a random number.

## Description

The function block FB\_RandomGenerator generates a random number that is within an interval with defined limits upon each call of the GetLreal or GetDint methods.

The random numbers are generated according to the linear congruence method. The required start value diNStart is generated during the initialization of the application from the system clock. This can be any integer superior to 0 which is located within the value range of data type DINT. The initialization of the function block can be triggered again using the Init method. This methods supports the definition of your diNStart value.

The linear congruence method generates a sequence of numbers in accordance with the following rules:

`z(i+1) = (a * z(i)) MOD m`

The variables `a` and `m` must be suitable integers. `z(0)` corresponds to the start value diNStart. The random generator uses the values `a = 16807` and `m = 2^31 - 1` (Mersenne prime number). This generates a sequence of numbers having the period `2^31`, that is, the values only repeat after more than 2 billion numbers generated. These may practically be used as random numbers in many computations for specific purposes.

## Interface

The function block does not provide any input or output parameters. An instance of the function block must not be called in your application program. To control and monitor, the function block provides methods.

## Methods

| Name | Description |
| --- | --- |
| Init | Is used to initialize the function block. |
| GetDint | Generates a random number of the DINT type. |
| GetLreal | Generates a random number of the LREAL type. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| diNStart | DINT | Read | Indicates the start value, also known as seed value, that is used for generating the random numbers. |
