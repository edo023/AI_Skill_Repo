# SMC\_CheckVelocities (FB)

FUNCTION\_BLOCK SMC\_CheckVelocities

This function block checks the track speed of the particular path segments. If the OutQueue has not been created by the editor, but by the IEC program (e.g. by [SMC\_NCInterpreter](VVFvsggjXKxZ6qmLw10MvZPswnU_smc-ncinterprete.html#vvfvsggjxkxz6qmlw10mvzpswnu_smc_ncinterprete_smc_ncinterpreter_fb)), this function block has to be called straight before each call to the interpolator.

The main task of this function is to check the path for sharp bends and to reduce the velocity to zero at these bends.
