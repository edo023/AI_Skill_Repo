# ST\_NodeIdentifiers - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.2.0 |
| Inherits from: | - |

## Description

Structure for node identifiers of Sercos devices. The MAC address is used as the node identifier of a Sercos device.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| asNodeIdentifiers | ARRAY [0..84] OF STRING(17) | Array of node identifiers of Sercos devices (MAC address) |
| uiNumber | UINT | Number of node identifiers in the array |
