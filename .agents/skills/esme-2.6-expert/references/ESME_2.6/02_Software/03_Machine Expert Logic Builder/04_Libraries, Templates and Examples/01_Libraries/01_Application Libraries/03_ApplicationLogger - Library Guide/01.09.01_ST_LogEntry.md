# ST\_LogEntry

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.4.0 |
| Inherits from: | — |

## Description

Structure for the information of one log entry of the Application Logger.

NOTE: As of version V2.0.0.0, the timestamp of a created message is in UTC. Use the function [FC\_ConvertLogEntryTimeStampUtcToLocal](FC_ConvertLogEntryTimeStampUtcToLoc-1E998D97.html#FC_ConvertLogEntryTimeStampUtcToLoc-1E998D97) to convert the timestamp from UTC to the local time zone of the controller.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| udiId | UDINT | Identification number of the logger message. |
| udiLoggerPointId | UDINT | Identification of the logger point that sent the message. |
| etLoglevel | [ET\_LogLevel](D-SE-0077662.html#D-SE-0077662) | Logger level for which the message has been sent. |
| etDiag | [GD.ET\_Diag](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.GlobalDiagnostic&topicID=D_SE_0076228) | ET\_Diag that was sent for this logger message. |
| udiEtDiagExt | UDINT | ET\_DiagExt number that was sent for this logger message. |
| dtDate | DATE | Date when the message was created. |
| todTime | TIME\_OF\_DAY | Time when the message was created. |
| sMessage | STRING[255] | The logger message. |
