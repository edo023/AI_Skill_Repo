# SMC\_ABC\_SLOPES\_NO\_STOP (ENUM)

TYPE SMC\_ABC\_SLOPES\_NO\_STOP :

Determines how [SMC\_RecomputeABCSlopes](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-recomputeabc.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_recomputeabc_smc_recomputeabcslopes_fb) sets the slopes of A/B/C at the transtion between two elements, when there is not stop.

| Attributes: | qualified\_only |
| InOut: | | Name | Comment | | --- | --- | | SET\_SMOOTH\_CARDINAL | The slope at the end of element i is set as (AddAxisValue(i+1) - AddAxisValue(i-1)) / (ElementLength(i) + ElementLength(i+1)). The slope at the start of element i+1 is set to the same value. | | DONT\_CHANGE | The slope is not changed. | |
