## Task

Interface with reference and status values for the move command [MoveSyncFromStandstillSuperimposedChannelC](MoveSyncFrStandstSuperimpC-2F9D684A.html#MoveSyncFrStandstSuperimpC-2F9D684A).

## Description

The feedback interface provides reference and status values for a superimposed movement of the carrier with the move command MoveSyncFromStandstillSuperimposedChannelC.
