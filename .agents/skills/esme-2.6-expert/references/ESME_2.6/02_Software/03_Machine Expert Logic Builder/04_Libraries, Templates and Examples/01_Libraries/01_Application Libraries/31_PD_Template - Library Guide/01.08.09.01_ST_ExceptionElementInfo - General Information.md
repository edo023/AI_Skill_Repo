# ST\_ExceptionElementInfo - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.4.0 |
| Inherits from: | - |

## Description

This is a file type in which additional information for the respective exceptions can be saved.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| xQuitMarker | BOOL | TRUE: The exception can be deleted as it is no longer active.  FALSE: The bit was reset by the FC\_SetException function. The exception is still active. |
