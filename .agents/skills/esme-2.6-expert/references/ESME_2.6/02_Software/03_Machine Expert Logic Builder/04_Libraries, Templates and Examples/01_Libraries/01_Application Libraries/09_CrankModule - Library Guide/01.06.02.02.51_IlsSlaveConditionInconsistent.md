# IlsSlaveConditionInconsistent

IlsSlaveConditionInconsistent

|  |  |
| --- | --- |
| Enumeration name: | IlsSlaveConditionInconsistent |
| Enumeration value: | 66 |
| Description: | The Ils slave specification is inconsistent. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operation mode [ET\_OpMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0088734_1).MultiCam was selected but the specifications whether it is an IIs slave are contradictory. | Ensure that the slave number of the IIs slave is entered in [ST\_MultiCam](../Structures/Structures-9.htm#XREF_D_SE_0088766_1).i\_udiSlaveNumber. If it is not an IIs slave, a 0 must be entered.  Both conditions mentioned earlier must be met simultaneously. |
