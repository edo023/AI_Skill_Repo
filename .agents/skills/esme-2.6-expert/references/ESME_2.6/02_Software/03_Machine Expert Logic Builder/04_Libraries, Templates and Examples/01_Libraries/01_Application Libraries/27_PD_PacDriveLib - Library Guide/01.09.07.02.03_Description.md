# Description

Description

The function calculates minimum, maximum, expected value (=average) and variance (=square of the default deviation) of the input series of measurements. The series of measurements is extended by one value each time the [Put](Function_Blocks_R_to_Z-25.htm#XREF_D_SE_0087368_1) method is called up. The series of measurements can be deleted by calling up the [Reset](Function_Blocks_R_to_Z-26.htm#XREF_D_SE_0087369_1) method.
