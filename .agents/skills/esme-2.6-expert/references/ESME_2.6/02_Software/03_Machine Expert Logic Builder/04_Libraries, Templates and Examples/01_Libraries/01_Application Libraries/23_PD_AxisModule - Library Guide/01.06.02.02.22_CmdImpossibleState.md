# CmdImpossibleState

CmdImpossibleState

|  |  |
| --- | --- |
| Enumeration name: | CmdImpossibleState |
| Enumeration value: | 4 |
| Description: | The command cannot be executed in the current state. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The module is in the state  [ET\_State.NoOpAxisDisabled](../Enumerations/Enumerations-7.htm#XREF_D_SE_0077134_1) or  [ET\_State.NoOpAxisEnabled](../Enumerations/Enumerations-7.htm#XREF_D_SE_0077134_1)  when it is sent a command  [ET\_Cmd.Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1),  [ET\_Cmd.WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1),  [ET\_Cmd.StartTrig](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) or  [ET\_Cmd.StartTrigWaitInPos](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1)  is send to it.  The module is not in the state  [ET\_State.MultiCamWaitForWsStart](../Enumerations/Enumerations-7.htm#XREF_D_SE_0077134_1),  [ET\_State.MultiCamRunning](../Enumerations/Enumerations-7.htm#XREF_D_SE_0077134_1)  or in the operating mode  [ET\_OpMode.Additional](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1)  when it is sent a command  [ET\_Cmd.WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1)  is send to it.  The module is in the state  [ET\_State.MultiCamWaitForWsStart](../Enumerations/Enumerations-7.htm#XREF_D_SE_0077134_1)  and a warm start mode  [PDL.ET\_MultiCamWsMode.StoredCamShowCamPosition](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087233_1) or  [PDL.ET\_MultiCamWsMode.NewCamShowCamPosition](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087233_1)  has been selected when it is sent a command  [ET\_Cmd.WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1)  is send to it.  The module is in the operating mode  [ET\_OpMode.Manual](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1)  when it is sent a command  [ET\_Cmd.Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) or  [ET\_Cmd.Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1),  is send to it.  The module is not in the operating mode  [ET\_OpMode.Positioning](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1)  when it is sent a command  [ET\_Cmd.StartTrigWaitInPos](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1)  is send to it.  The module is not in the operating mode  [ET\_OpMode.Positioning](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1),  [ET\_OpMode.Endless](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1),  [ET\_OpMode.EndlessIls](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) or  [ET\_OpMode.MultiCam](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1)  when it is sent a command  [ET\_Cmd.StartTrig](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1)  is send to it. | Before you send a command, ensure that the module can execute the command in the current state. |
