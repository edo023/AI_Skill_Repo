## Task

Sets the data for the procedure.

## Description

The method SetProcedureData (...) is used to set the data required by the procedure.

NOTE: When the method is called, all the stored samples are removed.

NOTE: The logical encoder described by i\_stData.stLEncLogicalAddress must be allocated for the teaching procedure and not manipulated from outside the function block.

Access: PUBLIC
