# ProgrammingInProgress

ProgrammingInProgress

|  |  |
| --- | --- |
| Enumeration name: | ProgrammingInProgress |
| Enumeration value: | 89 |
| Description: | Writing is being performed on the encoder. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.WriteAxisEncoder](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).  You cannot write on the encoder because a writing process is already active. | Wait until the active writing process is completed and try again.  If this diagnostic message occurs even after retrying it several times, please inform the support team about this detected error. |
