# DecRange

DecRange

|  |  |
| --- | --- |
| Enumeration name: | DecRange |
| Enumeration value: | 13 |
| Description: | Dec is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | In the parameter lrDec of the current table step, an invalid value has been transferred. (The table steps are transferred at the input i\_pstMultiPosStep) | The following must hold: 0 < lrDec < parameter MaxAcc of the drive  For the valid value range for lrDec, see output q\_sMsg |
