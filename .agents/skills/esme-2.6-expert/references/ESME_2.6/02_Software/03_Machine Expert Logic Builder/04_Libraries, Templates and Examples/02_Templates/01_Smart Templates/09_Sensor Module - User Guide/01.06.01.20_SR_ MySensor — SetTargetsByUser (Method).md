## Functional Description

The method SetTargetsByUser writes targets data to the sensor module.

NOTE: Targets data can only be written, if xNewTargetsAvailable = FALSE. If xNewTargetsAvailable = TRUE, the method GetTargets can be used to acquire the targets. On a call of GetTargets, the property xNewTargetsAvailable is set to FALSE.
