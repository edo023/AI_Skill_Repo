System Requirements and Limitations

The application example has been created with the components listed in the table:

| Component | Type and Version |
| --- | --- |
| Software | EcoStruxure Machine Expert V1.1 |
| Controller | TM241CEC24T1) |
| Additional devices | oLexium 32A with a compatible BSH or BMH motor  oLexium 28 with a compatible BCH2 motor  oLexium 32i (integrated drive) |
| 1)   This example is based on the Modicon M241 Logic Controller, however, in principle, the concepts for the implementation of the Lexium drives on CANopen are the same for other EcoStruxure Machine Expert controllers. As a precondition, the controller must support the CANopen manager and the required libraries. | |
