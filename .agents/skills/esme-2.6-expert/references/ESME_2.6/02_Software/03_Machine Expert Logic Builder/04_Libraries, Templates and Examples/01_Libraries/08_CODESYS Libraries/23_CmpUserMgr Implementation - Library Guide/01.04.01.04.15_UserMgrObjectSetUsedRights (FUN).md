# UserMgrObjectSetUsedRights (FUN)

FUNCTION UserMgrObjectSetUsedRights : RTS\_IEC\_RESULT

Sets the used access rights for the object specified by handle.

The rights are additionally stored in an internal list. In case USERDB\_RIGHT\_NONE is passed for ulUsedRights the previously stored rights are retrieved for this object.

Limited to the application subtree.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | UserMgrObjectSetUsedRights | RTS\_IEC\_RESULT | | Error code | * ERR\_OK: Rights are successfully set * ERR\_PARAMETER: Invalid object handle * ERR\_NO\_OBJECT: Object was not found for rights retrieval * ERR\_NO\_ACCESS\_RIGHTS: No access rights in IEC context | | | Input | hObject | RTS\_IEC\_HANDLE | Handle to the object | | ulUsedRights | DWORD | Used access rights or USERDB\_RIGHT\_NONE for retrieval | |
