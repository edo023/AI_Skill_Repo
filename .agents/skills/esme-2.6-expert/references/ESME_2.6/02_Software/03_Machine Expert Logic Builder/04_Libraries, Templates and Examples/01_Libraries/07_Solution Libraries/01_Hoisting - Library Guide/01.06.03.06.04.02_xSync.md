# xSync

xSync

Synchronization command for the axis group. All axes fulfilling the conditions to enter synchroni­zation are added to the synchronization group when i\_astSyncGroup[i].xSync is TRUE. Axes can be added to synchronization groups only while they are ready and no alarm state is detected. All axes must be in standstill in order to enter or leave a synchronization group, unless they have the parameter i\_astAxisIntf[i].xInstSync set to TRUE.

There must be at least one axis within a synchronization group to allow the group to enter and maintain the synchronous state.
