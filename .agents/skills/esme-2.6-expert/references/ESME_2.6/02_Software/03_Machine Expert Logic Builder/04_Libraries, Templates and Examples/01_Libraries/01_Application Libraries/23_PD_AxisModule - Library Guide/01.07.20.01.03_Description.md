# Description

Description

This function is used to specify that an axis, when homed, will be homed by seeking i\_lrMaxTorque value from the axis’s motor. The i\_etMode input specifies the homing direction:

| i\_etMode | Home Action |
| --- | --- |
| PDL.ET\_HomeMode.PosDirectionMaxTorque | Move in the positive direction until the i\_lrMaxTorque is reached |
| PDL.ET\_HomeMode.NegDirectionMaxTorque | Move in the negative direction until i\_lrMaxTorpque is reached |

All the possible values of i\_etHomingMode are enumerated in PDL.ET\_HomeMode; however, the function returns FALSE if none of the above values is used.
