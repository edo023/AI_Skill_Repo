# Status Display

Status Display

The status of each group is displayed in the last line of the group. This allows you to see whether and how successfully the individual areas have been optimized.

The following conditions are defined for the status:

| Element | Description |
| --- | --- |
| Rectangle Not optimized | The area has not been optimized yet. |
| Rectangle Optimizing | The adjustment of the parameters for this area has been started but is not yet completed. |
| Rectangle No parameters found | No parameters were found for this area in the current optimization. Pending optimizations are no longer performed. |
| Rectangle Parameters that have to be checked | For this area parameters were found by the current optimization that probably provide a good controller behavior but have deviations in the required dynamic, damping or accuracy. This is why they definitely should be checked again. |
| Rectangle Parameters found | Parameters were found for this area in the current optimization. In general the desired controller behavior is reached with these parameters. |
