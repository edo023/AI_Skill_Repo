# Custom-Tuning (Heating) - FB\_HeatingControl2

## Overview

This procedure describes how to set the P and I parameters manually.

| Step | Action |
| --- | --- |
| 1 | Set iq\_etSelectParameterSetPID to ET\_ParameterSetPID.Custom. |
| 2 | Set rTn andrTv to 0.  Also refer to [ST\_PID2](ST_PID2-6CCF1B18.html). |
| 3 | Increase rKp until the system begins to oscillate. |
| 4 | Decrease rKp until the system does not oscillate. |
| 5 | Decrease rKp by another few points. |
| 6 | Set rTn, beginning with a large value for time, for example 100 s (system dependent) and verify that the system does not oscillate. |
| 7 | Decrease rTn until the system is stable and without oscillation.  **Result**: The figure displays a typically correct regulation. |

Also refer to [FB\_HeatingControl2](FB_HeatingControl2-771217B0.html).
