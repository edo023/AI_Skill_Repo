# YOffsetNotEnabled

YOffsetNotEnabled

|  |  |
| --- | --- |
| Enumeration name: | YOffsetNotEnabled |
| Enumeration value: | 172 |
| Description: | The YOffset generator is not activated. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The Y offset encoder of the axis is disabled. | The parameter YOffsetEnable of the axis must be TRUE for a YOffset correction. |
