## Task

Interface with general feedbacks of the Lexium™ MC multi carrier transport system.

## Description

The interface provides feedback properties for the Lexium™ MC multi carrier transport system.

## Properties

| Property | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ifFeedbackConfiguration | IF\_MulticarrierFeedbackConfiguration | Read | Reading the results of the configuration of the Lexium™ MC multi carrier transport system.  For more information, refer to [IF\_MulticarrierFeedbackConfiguration](FeedbConfig-D619B88F.html#FeedbConfig-D619B88F) |
| xJobActiveAnyCarrier | BOOL | Read | Indicates TRUE if at least one carrier on the track has a job assigned to it. |
| xMotionActiveAnyCarrier | BOOL | Read | Indicates TRUE if at least one carrier on the track is moving. |
