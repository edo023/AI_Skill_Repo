# General Information

## Description

The library includes the function blocks and functions to build, parse, read, and write Sercos IDNs (Identification Numbers).

| Characteristic | Value |
| --- | --- |
| Library title | SercosCommunication |
| Company | Schneider Electric |
| Category | Communication |
| Component | Core Repository |
| Default namespace | SERC |
| Language model attribute | qualified-access-only |
| Forward compatible library | Yes (FCL) |

NOTE: For this library, qualified-access-only is set. This means, that the functions have to be accessed using the namespace of the library. The default namespace of the library is **SERC**.

## Overview of the Function Blocks

The library contains the following function blocks:

| Function blockss | Use |
| --- | --- |
| [FB\_ReadParameter](TPC_SercosCommLib_FB_ReadParameter-E3E2B8EE.html) | Reads a data block element of a Sercos parameter via the service channel. |
| [FB\_WriteParameter](TPC_SercosCommLib_FB_WriteParameter-E3E53A21.html) | Writes a data block element of a Sercos parameter via the service channel. |

## Overview of the Functions

The library contains the following functions:

| Functions | Use |
| --- | --- |
| [FC\_BuildIDN](D-SE-0055488.html) | Builds the structured IDN with the input values from the symbolic addressing (`<S/P>-<PS>-<DBN>.<SI>.<SE>`). |
| [FC\_IdnStringToDword](TPC_SercosCommLib_FC_IdnStringToDword-E3DBE177.html) | Builds the structured IDN with the symbolic addressing input given as STRING. |
| [FC\_ParseIDN](D-SE-0055489.html) | Parses an IDN given as DWORD to the single elements. It has no return value. |

## Overview of the Enumerations

| Enumeration | Use |
| --- | --- |
| [ET\_DataBlockElement](TPC_SercosCommLib_ET_DataBlockElement-E3DFAEAA.html) | Defines the Sercos parameter element which is to be read (written). |
| [ET\_Result](D-SE-0055490.html) | The enumeration ET\_Result contains the possible values that indicate the result of operations executed by the POUs of this library. |

## Hardware Platforms

The SercosCommunication library is compatible with:

* LMC Eco
* LMC Pro/Pro2
* M262
