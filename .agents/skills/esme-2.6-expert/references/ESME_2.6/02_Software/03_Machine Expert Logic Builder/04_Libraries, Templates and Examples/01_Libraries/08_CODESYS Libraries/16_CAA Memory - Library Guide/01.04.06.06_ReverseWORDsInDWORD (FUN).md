# ReverseWORDsInDWORD (FUN)

FUNCTION ReverseWORDsInDWORD : DWORD

Function returns the DWORD “dwInput” with a reversed word-order (Low-Word<->High-Word).

Example:

```
dwInput = 16#01020304
->
MEM.ReverseWORDsInDWORD(dwInput) = 16#03040102
```
