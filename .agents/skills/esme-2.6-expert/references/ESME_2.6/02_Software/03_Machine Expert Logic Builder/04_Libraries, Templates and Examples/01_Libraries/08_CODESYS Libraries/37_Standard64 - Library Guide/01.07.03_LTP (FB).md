# LTP (FB)

FUNCTION\_BLOCK LTP

Timer Pulse. Q produces a High-Signal with the length of PT on every rising edge on IN.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | IN | BOOL | Trigger for Start of the Signal | | PT | LTIME | The length of the High-Signal | | Output | Q | BOOL | The pulse | | ET | LTIME | elapsed time since pulse start | |
