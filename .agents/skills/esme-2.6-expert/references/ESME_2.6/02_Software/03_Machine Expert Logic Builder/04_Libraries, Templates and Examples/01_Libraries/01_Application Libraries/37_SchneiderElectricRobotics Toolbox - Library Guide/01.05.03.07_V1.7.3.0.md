## New Features

New function block:

* **FB\_LinearTrackingVelOverrideScaling**

  The function block can be used to scale the velocity override of a robot based on a reference target position and on the state of a linear tracking system.
* **FB\_ReceiveVisionDataClient** and **FB\_ReceiveVisionDataServer**

  The function blocks collect the data for a vision protocol and analyze the data.

  The function blocks are supporting Transport Layer Security (TLS).
* **FB\_SendVisionDataClient** and **FB\_SendVisionDataServer**

  The function blocks send a specific vision protocol. The protocol is created by the function blocks. The data to be sent can be added using the methods of the function blocks.

  The function blocks are supporting Transport Layer Security (TLS).

New interface:

* **IF\_LinearTrackingVelOverrideScaling**

  The interface can be used to scale the velocity override of a robot based on a reference target position and on the state of a linear tracking system.
