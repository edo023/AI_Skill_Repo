# SMC\_TC\_STATUS (ENUM)

TYPE SMC\_TC\_STATUS :

This enumeration indicates the current state of the FB [SMC\_ToolCorr](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-toolcorr.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_toolcorr_smc_toolcorr_fb).
