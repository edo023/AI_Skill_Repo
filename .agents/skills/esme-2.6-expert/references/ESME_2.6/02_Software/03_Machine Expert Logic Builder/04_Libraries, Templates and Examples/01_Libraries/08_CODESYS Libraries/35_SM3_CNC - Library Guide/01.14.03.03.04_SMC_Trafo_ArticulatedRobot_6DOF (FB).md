# SMC\_Trafo\_ArticulatedRobot\_6DOF (FB)

FUNCTION\_BLOCK SMC\_Trafo\_ArticulatedRobot\_6DOF

SMC\_Trafo\_ArticulatedRobot\_6DOF implements the mathematic inverse transformation of an articulated robot with six rotary axes used for calculation of the tool center point coordinates from the axes position, the tool orientation and the states of joint.

Robots with the following denavit-hartenberg configuration are supported: (the values of d\_i and a\_i are supplied in the configuration)

| x | joint offset | link offset d\_i | link length a\_i | link twist |
| --- | --- | --- | --- | --- |
| 1 | 0° | d1 | a1 | -90° |
| 2 | 90° | 0 | a2 | 0° |
| 3 | 0° | d3 | a3 | 90° |
| 4 | 0° | d4 | 0 | 90° |
| 5 | 0° | 0 | 0 | -90° |
| 6 | 0° | d6 | 0 | 0° |

| Assumptions: | * | All lengths in config are positive | (greater than g\_fSMC\_CNC\_EPS) | * | The valid range for axis 0, 1, 3, and 4 | is contained in ]-180°, 180°[ | * | The valid range for axis 2 (dQ2) | is contained in ]-90°, 180°[ | * | The valid range for axis 5 (dQ5) | is not constrained and may be greater than 360° | |
