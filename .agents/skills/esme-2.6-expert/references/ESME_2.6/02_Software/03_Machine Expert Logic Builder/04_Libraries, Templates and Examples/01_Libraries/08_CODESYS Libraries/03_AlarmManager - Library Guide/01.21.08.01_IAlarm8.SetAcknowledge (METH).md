# IAlarm8.SetAcknowledge (METH)

METHOD SetAcknowledge : BOOL

Function to directly acknowledge an alarm with additional information as an alternative to IAlarmHandler.AcknowledgeOneOfGroup. Add a comment from a certain user or leave both fields empty if no comment is needed.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | SetAcknowledge | BOOL |  | | Input | pwsComment | POINTER TO WSTRING | Optional: Provide additional information about the reason of the acknowledgement. | | pwsUserId | POINTER TO WSTRING | Optional: Provide a user that made the comment. | |
