## Task

Setting the offset vector for the tool center point.

## Description

With the method SetToolOffset, you can specify the offset vector from the center point of the carrier to the center point of the tool.

| Top view | Side view |
| --- | --- |
|  |  |
| * The crossing red lines indicate the center of the carrier in X and Y direction. * The blue box symbolizes the tool installed at the carrier. | * The red line indicates the height of the carrier in Z direction. * The blue box symbolizes the tool installed at the carrier. |

For more information on the coordinate system of the carrier, refer to [Carrier Coordinate System](IntroMC_CarrCenter-16E8092C.html#IntroMC_CarrCenter-16E8092C__CarrierCoordinateSystem-16E820B3).

The tool offset is used for calculating the rear offset and the front offset of the carrier based on the [ToolDimensions](IF_CarrierConfiguration-SetToolDime-51BFC8FC.html#IF_CarrierConfiguration-SetToolDime-51BFC8FC) and the ToolOffset as well as the [ProductDimensions](IF_CarrierConfiguration-SetProductD-514499A8.html#IF_CarrierConfiguration-SetProductD-514499A8) and the [ProductOffset](IF_CarrierConfiguration-SetProductO-51C0FECB.html#IF_CarrierConfiguration-SetProductO-51C0FECB).

For more details on the properties lrRearOffset and lrFrontOffset, refer to the interface [IF\_CarrierFeedbackConfiguration](CarrFeedbConf-E1D3F75B.html#CarrFeedbConf-E1D3F75B).
