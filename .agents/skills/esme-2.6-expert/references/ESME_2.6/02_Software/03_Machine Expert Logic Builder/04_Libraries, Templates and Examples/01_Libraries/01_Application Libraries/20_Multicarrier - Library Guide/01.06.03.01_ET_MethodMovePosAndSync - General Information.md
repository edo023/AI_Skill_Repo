# ET\_MethodMovePosAndSync - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.5.10.0 |

## Description

The enumeration ET\_MethodMovePosAndSync specifies the methods for the move command [MovePosAndSync](IF_MovePosAndSync-4526BC87.html#IF_MovePosAndSync-4526BC87).

## Enumeration Elements

| Name | Value (DINT) | Description |
| --- | --- | --- |
| None | 0 | No method assigned. |
| StartPositioning | 1 | Method for starting the movement of a carrier on the selected channel to a given target position. |
| Stop | 2 | Method for stopping the carrier movement controlled by the move command MovePosAndSync. |
| StopChannel | 3 | Method for stopping the carrier movement on a selected channel, controlled by the move command MovePosAndSync. |
| StartSyncFromStandstill | 101 | Method for the synchronization of the selected carrier to an external master using a cam job. |
| StartSyncStep | 102 | Method for executing a step movement of the carrier synchronized with an external master. |
| StartSyncOnTheFly | 103 | Method for the on-the-fly synchronization of the selected carrier to an external master or to a different gear factor. |
| SetposRelativeChannelABC | 901 | Method for assigning a setpos motion job to the channels A, B and C of the SoMotionGenerator (SMG), using the setpos mode Relative.  For more information on the SoMotionGenerator, refer to the [PD\_SoMotion Generator library](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SoMotionGenerator&topicID=). |
