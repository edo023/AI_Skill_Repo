# General Information - IF\_FsmStateConversion

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.2.9.0 |

## Description

The interface IF\_StateConversion must be implemented by a function block declared in the application which allows to convert the numerical value representing the state to a string. The interface is used to register the function block instance to the FB\_FiniteStateMachine through the method AddStateConversion.

Once an instance of the interface is added to the FB\_FiniteStateMachine, the interface method ToString is called whenever a state transition is processed.
