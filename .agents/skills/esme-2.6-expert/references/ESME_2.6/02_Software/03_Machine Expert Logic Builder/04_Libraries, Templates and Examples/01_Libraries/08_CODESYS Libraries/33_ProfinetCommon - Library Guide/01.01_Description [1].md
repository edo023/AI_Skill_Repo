## Description [[1]](#index_indices_and_tables__fn_index_id2)

**Common Profinet datatypes and functions:**

* DCP: Function Blocks for Name- and IP-Assignment or Network-Scan
* Diagnosis: Function Blocks for evaluating Profinet Diagnosis Data received by RALARM or RDREC
* IO-Link: Function Block IOL\_CALL for read/write IO-Link parameters
