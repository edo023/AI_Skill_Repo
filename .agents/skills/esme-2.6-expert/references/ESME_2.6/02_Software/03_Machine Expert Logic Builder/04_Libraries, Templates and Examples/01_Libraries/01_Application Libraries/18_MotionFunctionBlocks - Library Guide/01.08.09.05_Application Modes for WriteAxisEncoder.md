# Application Modes for WriteAxisEncoder (Method)

## Periodic Application

A periodic application is active when i\_lrUserPeriod > 0.

* The axis reference position after homing is calculated back to the interval [0 … i\_lrUserPeriod].
* Values outside this range are automatically converted back to the valid interval.
* The encoder offset parameter i\_lrAbsEncOffset is ignored.

## Non-Periodic Applications

A non-periodic application is active when i\_lrUserPeriod = 0.

* The parameter i\_lrAbsEncOffset must be greater than 0.
* The axis position after homing is within the interval: [-i\_lrAbsEncOffset … EncoderPeriod – i\_lrAbsEncOffset]
* Values outside this interval are wrapped back into the valid range.
