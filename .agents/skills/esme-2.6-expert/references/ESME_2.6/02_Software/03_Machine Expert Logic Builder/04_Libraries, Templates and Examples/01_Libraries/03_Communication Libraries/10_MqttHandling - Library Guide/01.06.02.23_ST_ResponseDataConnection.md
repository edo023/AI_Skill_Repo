# ST\_ResponseDataConnection

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V2.4.4.0 |
| Inherits from: | - |

## Description

The structure ST\_ResponseDataConnection contains information related to the connection indicated in the response message sent by the MQTT broker while connecting and disconnecting.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| etConnackReasonCode | [ET\_ReasonCode](ET_ReasonCode-D887BF12.html) | Indicates the reason code in the CONNACK message sent by the MQTT broker while establishing a connection. |
| stConnackPropertiesProperties | [ST\_PropertiesConnack](ST_PropertiesConnack-D8E1CD7A.html) | Indicates the properties of the CONNACK message sent by the MQTT broker while establishing a connection. |
| etDisconnectReasonCode | [ET\_ReasonCode](ET_ReasonCode-D887BF12.html) | Indicates the reason code in the DISCONNECT message sent by the MQTT broker. |
| stDisconnectProperties | [ST\_PropertiesServerDisconnect](ST_PropertiesServerDisc-D8E75664.html) | Indicates the properties of the DISCONNECT message sent by the MQTT broker. |
