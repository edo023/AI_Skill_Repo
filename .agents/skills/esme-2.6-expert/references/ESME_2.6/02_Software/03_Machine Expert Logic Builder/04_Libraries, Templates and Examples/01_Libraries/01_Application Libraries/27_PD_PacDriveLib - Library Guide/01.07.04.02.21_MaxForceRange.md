# MaxForceRange

MaxForceRange

|  |  |
| --- | --- |
| Enumeration name: | MaxForceRange |
| Enumeration value: | 314 |
| Description: | MaxForce is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | At the input i\_lrMaxForce, an invalid value has been applied. | At the input i\_lrMaxForce, a value must be transferred which fulfills the following formula:  0.0 < (i\_lrMaxForce / lrForceConstant \* 10000000.0 / lrMaxDrivePeakCurrent / lrLimCurrent) < 100.0  with lrLimCurrent := lrUserDrivePeakCurrent \* 100.0 / lrMaxDrivePeakCurrent  with lrForceConstant, lrMaxDrivePeakCurrent and lrUserDrivePeakCurrent being the parameters of the axis. |
