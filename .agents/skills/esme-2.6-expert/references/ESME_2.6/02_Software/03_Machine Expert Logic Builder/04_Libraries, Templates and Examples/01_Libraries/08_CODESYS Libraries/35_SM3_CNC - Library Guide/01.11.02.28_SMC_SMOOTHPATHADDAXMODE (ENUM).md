# SMC\_SMOOTHPATHADDAXMODE (ENUM)

TYPE SMC\_SMOOTHPATHADDAXMODE :

Mode for handling the additional axes by [SMC\_SmoothPath](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-smoothpath.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_smoothpath_smc_smoothpath_fb)

| InOut: | | Name | Comment | | --- | --- | | SPAA\_LATE | Shift additional axes’ position to end position of inserted spline | | SPAA\_BROAD | Shift additional axes’ position at the first spline to start, at last spline to the end | | SPAA\_NONE | Do not consider additional axes | | SPAA\_EVEN | Distribute positions of additional axes proportional to element lengths | | SPAA\_EXACT | Cut ABC axes’ position functions like X/Y/Z and connect smoothly | |
