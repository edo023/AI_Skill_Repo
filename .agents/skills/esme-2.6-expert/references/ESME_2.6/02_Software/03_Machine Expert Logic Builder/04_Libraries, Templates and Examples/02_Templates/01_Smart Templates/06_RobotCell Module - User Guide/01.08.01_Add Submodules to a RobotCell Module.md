# Add Submodules to a RobotCell Module

## Add a RobotCell Module to a Project

| Step | Action |
| --- | --- |
| 1 | To create a new project, use File > New Project > Default Project . |
| 2 | Click View > Modules to open the Modules view. |
| 3 | Right-click Application (root node). |
| 4 | Select Add Module… from the contextual menu.  **Result:** TheAdd Module dialog box is displayed. The Smart Template modules installed in your project are listed. |
| 5 | Select the RobotCell Module and click Next. |
| 6 | Select the values for Orientation Convention, Surface Plane, Direction, and activate the Add SmartVisu connection code (FB\_VisuBridge) check box. |
| 7 | Click Add Module.  **Result:**   * The Module Manager is inserted beneath the Application (root node). This is done automatically once (by adding the first Smart Template module). * The module is inserted in parallel to the Module Manager node. * The Configuration data node is displayed beneath the module. * The Module Interface node is displayed beneath the module * The Configuration data > General tab of the inserted module is displayed. |

## Add Submodules to a RobotCell Module

| Step | Action |
| --- | --- |
| 1 | Right-click the RobotCell Module in the Modules tree. |
| 2 | Select Add Module… from the contextual menu.  **Result:** The Add Module dialog box is displayed. The Smart Template modules installed in your project are listed. |
| 3 | Select the Smart Template to be added (for example a robot, a conveyor, a camera, or a sensor module). |
| 4 | Click Next, edit the module-specific properties and click Add Module…. |
| 5 | Repeat steps 1 to 4 until you have added the desired submodules to your RobotCell Module. |

## Set Data of the SubModules

To use the submodules within the RobotCell Module, configure the submodules.

| Step | Action |
| --- | --- |
| 1 | In the Modules tree under your RobotCell Module, double-click the Smart Template module you want to configure (for example a robot, a conveyor, a camera, or a sensor module).  **Result:** The Configuration data of the module is displayed. |
| 2 | For how to configure the modules, refer to the respective [tab descriptions](ConfigurationData-GeneralInformatio-66BA43FA.html). |
