# FC\_NegativePart

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |

## Description

This function determines the negative part of a value in the following way:

* i\_lrValue >= 0.0 -> 0.0
* i\_lrValue < 0.0 -> ABS(i\_lrValue)

In conjunction with the function FC\_PositivePart, the following identity applies to the real values of lrValue:

lrValue = FC\_PositivePart(lrValue) - FC\_NegativePart(lrValue).
