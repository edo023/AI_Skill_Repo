# CharBufferString.FromString (METH)

METHOD FromString : BOOL

Converts an IEC string value into the internally used representation. The result of the method indicates, whether the processing was successful or not (e.g. CharBufferString.m\_Type <> ANY\_STRING). If the internally used buffer is not large enough the result is TRUE and only the used buffer size – 1 will be copied.
