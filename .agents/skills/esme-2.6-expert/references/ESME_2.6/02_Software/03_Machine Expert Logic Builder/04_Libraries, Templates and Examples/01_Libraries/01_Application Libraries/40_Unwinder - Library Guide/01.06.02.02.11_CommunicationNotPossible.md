# CommunicationNotPossible

CommunicationNotPossible

|  |  |
| --- | --- |
| Enumeration name: | CommunicationNotPossible |
| Enumeration value: | 68 |
| Description: | Communication is not possible. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The control mode ET\_ControllerMode.Torque is chosen, and the Sercos bus is not in phase 4 while the unwinder POU is active. | The Sercos bus has to be in phase 4 while the unwinder POU is active. |
