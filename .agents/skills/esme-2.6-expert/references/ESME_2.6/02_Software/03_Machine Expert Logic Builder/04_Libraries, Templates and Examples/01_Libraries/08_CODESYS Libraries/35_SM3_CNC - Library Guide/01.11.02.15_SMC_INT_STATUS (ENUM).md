# SMC\_INT\_STATUS (ENUM)

TYPE SMC\_INT\_STATUS :

Status of an interpolator instance.

| InOut: | | Name | Initial | Comment | | --- | --- | --- | | IPO\_UNKNOWN | 0 | Internal state that may not occur after a complete pass of the [SMC\_Interpolator](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-interpolator.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_interpolator_smc_interpolator_fb). | | IPO\_INIT | 1 | Initialization state; movement not started yet | | IPO\_ACCEL | 2 | Interpolation ongoing; currently accelerating | | IPO\_CONSTANT | 3 | Interpolation ongoing with constant velocity | | IPO\_DECEL | 4 | Interpolation ongoing; currently decelerating | | IPO\_FINISHED | 5 | Interpolation done | | IPO\_WAIT | 6 | Interpolation ongoing; currently waiting for example by a stop input or a M-command | | IPO\_INCREASING\_ACCEL | 7 | Interpolation ongoing; currently building up the acceleration | | IPO\_DECREASING\_ACCEL | 8 | Interpolation ongoing; currently ramping down the acceleration | | IPO\_INCREASING\_DECEL | 9 | Interpolation ongoing; currently building up the deceleration | | IPO\_DECREASING\_DECEL | 10 | Interpolation ongoing; currently ramping down the deceleration | |
