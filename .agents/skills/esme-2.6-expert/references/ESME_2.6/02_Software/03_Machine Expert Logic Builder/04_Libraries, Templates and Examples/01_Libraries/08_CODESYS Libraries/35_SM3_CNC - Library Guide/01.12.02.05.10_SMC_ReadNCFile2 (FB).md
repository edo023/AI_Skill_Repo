# SMC\_ReadNCFile2 (FB)

FUNCTION\_BLOCK SMC\_ReadNCFile2

SMC\_ReadNCFile2 can read an NC-ASCII-file from the file system of the controller in order to make it available for the [SMC\_NCInterpreter](VVFvsggjXKxZ6qmLw10MvZPswnU_smc-ncinterprete.html#vvfvsggjxkxz6qmlw10mvzpswnu_smc_ncinterprete_smc_ncinterpreter_fb). Thus, an NC program can be read in and interpreted at runtime.
