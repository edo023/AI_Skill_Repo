# TON (FB)

FUNCTION\_BLOCK TON

Implements a timer with a turn-on delay

```
(* Example declaration *)
TONInst : TON ;

(* Example in ST *)
TONInst(IN := VarBOOL1, PT:= T#5s);
VarBOOL2 := TONInst.Q;
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | IN | BOOL | Rising edge: starts delay counter  Falling edge: resets delay counter | | PT | TIME | Time for the delay counter [ms] | | Output | Q | BOOL | FALSE if IN is FALSE  TRUE if IN is TRUE and delay time PT elapsed | | ET | TIME | Elapsed time since rising edge at IN | |
