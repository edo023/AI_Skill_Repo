# ST\_Product\_Data

## Overview

|  |  |
| --- | --- |
| Type | Data structure |
| Available as of | V1.4.2.0 |

## Description

The structure ST\_Product\_Data is a collection of tags providing information for product stream data.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| ID | DINT | ID number for each product stream |
| Name | STRING | Name of product stream |
| Unit | STRING[6] | Structured array of ingredients process variable information for values with LREAL data type |
| PrimaryQty | REAL | Product primary quantity value of each product stream |
| ConsumedCount | DINT | Number of consumables used of each product stream |
| ProcessedCount | DINT | Number of produced goods of each product stream |
| DefectiveCount | DINT | Number of products marked as not acceptable of each product stream |
| AccConsumedCount | DINT | Accumulated number of consumables used of each product stream since last reset |
| AccProcessedCount | DINT | Accumulated number of produced goods of each product stream since last reset |
| AccDefectiveCount | DINT | Accumulated number of products marked as not acceptable of each product stream since last reset |
