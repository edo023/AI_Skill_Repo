# BeltIdRange

BeltIdRange

|  |  |
| --- | --- |
| Enumeration name: | BeltIdRange |
| Enumeration value: | 11 |
| Description: | BeltId is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | An invalid value has been transferred at the input i\_udiBeltId. | At the input i\_udiBeltId, the number of the belt has to be specified to which a connection is to be established. The number must be between one and the number of parameterized belts. |
