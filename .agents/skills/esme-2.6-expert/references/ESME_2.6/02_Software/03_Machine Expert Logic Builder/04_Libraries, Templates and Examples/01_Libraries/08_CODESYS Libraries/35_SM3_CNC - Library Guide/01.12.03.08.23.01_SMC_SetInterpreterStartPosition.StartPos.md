# SMC\_SetInterpreterStartPosition.StartPos (PROP)

PROPERTY StartPos : [SMC\_POSINFO](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-posinfo.html#cah0ife6nckjzp5ef3qpy_filyk_smc_posinfo_smc_posinfo_struct)

Returns the start position for the interpreter.

If the start position has not been set yet (by calling the function block in the bus task), a zero position is returned.

To be called from the background task that also calls the interpreter.
