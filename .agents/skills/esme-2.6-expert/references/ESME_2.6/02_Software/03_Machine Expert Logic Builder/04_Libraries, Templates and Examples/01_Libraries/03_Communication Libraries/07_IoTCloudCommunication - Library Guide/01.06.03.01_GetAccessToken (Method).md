# GetAccessToken (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method is used to obtain the access token for the connection to the Machine Advisor application interface. The return value of type ET\_Result indicates the result of the method execution. If Ok is indicated, the value is successfully copied to the assigned buffer.

## Inputs

| Paramerter | Data Type | Description |
| --- | --- | --- |
| i\_pbyBuffer | POINTER TO BYTE | Pointer to the buffer to store the value. |
| i\_uiSize | UDINT | Size of the buffer. |
