# ICyclicActionProvider.CyclicAction (METH)

METHOD CyclicAction

The CyclicAction is running until either xComplete is TRUE or iErrorID ≠ 0 (Zero). If xComplete is TRUE or iErrorID ≠ 0 (Zero) the state [STATE.CLEANING](e8R4--mzfNvSIX3lkxCmoWVrX_8_state.html#e8r4__mzfnvsix3lkxcmowvrx_8_state_state_enum__e8r4__mzfnvsix3lkxcmowvrx_8_state_state_cleaning) is reached. With itfTimingController ≠ 0 (Zero) it is possible to check the current invocation time (see: [ITimingController.ControllerCheckTiming](dtFLVhuzIH9lcVq7nrUOVkfKC_c_controllercheckt.html#dtflvhuzih9lcvq7nruovkfkc_c_controllercheckt_itimingcontroller_controllerchecktiming_meth)). Function blocks with a udiTimeLimit input variable (see: [IETrigTl](c-dVK8bVY1F-S7PaIINI_aQKZks_ietrigtl.html#c_dvk8bvy1f_s7paiini_aqkzks_ietrigtl_ietrigtl_itf), [IETrigATl](c-dVK8bVY1F-S7PaIINI_aQKZks_ietrigatl.html#c_dvk8bvy1f_s7paiini_aqkzks_ietrigatl_ietrigatl_itf), [IETrigTlTo](c-dVK8bVY1F-S7PaIINI_aQKZks_ietrigtlto.html#c_dvk8bvy1f_s7paiini_aqkzks_ietrigtlto_ietrigtlto_itf), [IETrigATlTo](c-dVK8bVY1F-S7PaIINI_aQKZks_ietrigatlto.html#c_dvk8bvy1f_s7paiini_aqkzks_ietrigatlto_ietrigatlto_itf), [ILConTl](WTeZ7YPBU-0n7x7253IdPMO6GZk_ilcontl.html#wtez7ypbu_0n7x7253idpmo6gzk_ilcontl_ilcontl_itf), [ILConTlTo](WTeZ7YPBU-0n7x7253IdPMO6GZk_ilcontlto.html#wtez7ypbu_0n7x7253idpmo6gzk_ilcontlto_ilcontlto_itf), [ILConTlC](FK06OFgXbi1-1qaBZEChV9WIZQ4_ilcontlc.html#fk06ofgxbi1_1qabzechv9wizq4_ilcontlc_ilcontlc_itf)) should be implement in such a way that the current invocation is exited when the consumed time for this invocation has exceeded the settings from udiTimeLimit.

**Example**

```
REPEAT
    // working to reach the ready condition
    // ⇒ xComplete := TRUE
    // if the maximum invocation time is reached
    // ⇒ xTimeLimit := TRUE
    // if the maximum operating time is reached
    // ⇒ xTimeOut := TRUE
    // if an error condition is reached
    // ⇒ set iErrorID to a value other than 0 (Zero)
    itfTimingController.ControllerCheckTiming(
        xTimeOut=>xTimeOut,
        xTimeLimit=>xTimeLimit
    );

    xComplete := TRUE;
    iErrorID := ERROR.NO_ERROR;

UNTIL xComplete OR
      xTimeOut OR xTimeLimit OR
      iErrorID <> ERROR.NO_ERROR
END_REPEAT
```
