## Description

Given an input matrix pose and a 3D vector, the function applies the transformation described by the pose matrix to the vector. The resulting vector is then returned.

NOTE: There is no internal verification about i\_stPoseMatrix being a valid pose matrix.

NOTE: The multiplication between the elements is handled internally.
