# GET\_STATE (FB)

FUNCTION\_BLOCK GET\_STATE EXTENDS CiA405Base

Function block returns the CANopen state of device with NodeID DEVICE. The ourput STATE is defined as follows:

* If guarding/heartbeat is not active for the selected DEVICE:

  + [DEVICE\_STATE.NOT\_AVAIL](WkZ9when7NogvVdipw-bWj0hVTA_device-state.html#wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_enum__wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_not_avail) is returned if DEVICE doesn’t respond in configuration phase.
  + [DEVICE\_STATE.UNKNOWN](WkZ9when7NogvVdipw-bWj0hVTA_device-state.html#wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_enum__wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_unknown) is returned in all other cases even if a boot-up message was received.
* If guarding/heartbeat is not active but at least one cyclic synchronous TPDO is configured for the selected DEVICE:

  + [DEVICE\_STATE.NOT\_AVAIL](WkZ9when7NogvVdipw-bWj0hVTA_device-state.html#wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_enum__wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_not_avail) is returned if DEVICE doesn’t respond in configuration phase.
  + [DEVICE\_STATE.OPERATIONAL](WkZ9when7NogvVdipw-bWj0hVTA_device-state.html#wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_enum__wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_operational) is returned if cyclic synchronous TPDOs are received correctly
  + [DEVICE\_STATE.UNKNOWN](WkZ9when7NogvVdipw-bWj0hVTA_device-state.html#wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_enum__wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_unknown) is returned in all other cases.
* If guarding/heartbeat is active:

  + If there is no guarding/heartbeat error (due to timeout) received NMT state is returned.
  + [DEVICE\_STATE.NOT\_AVAIL](WkZ9when7NogvVdipw-bWj0hVTA_device-state.html#wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_enum__wkz9when7nogvvdipw_bwj0hvta_device_state_device_state_not_avail) is returned if device is not responding or in case of huarding/heartbeat error.
