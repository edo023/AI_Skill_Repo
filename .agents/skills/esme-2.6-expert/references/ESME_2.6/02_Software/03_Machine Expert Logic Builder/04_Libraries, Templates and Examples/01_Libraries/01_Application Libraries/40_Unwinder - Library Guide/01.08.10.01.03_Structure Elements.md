# Structure Elements

Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| i\_lrVelocity | LREAL | The (middle) velocity of the feeder. |
| i\_lrAcceleration | LREAL | The value to accelerate to the master velocity. |
| i\_lrDeceleration | LREAL | The value to decelerate to the master velocity. |
| i\_lrJerk | LREAL | The jerk to accelerate and decelerate the master velocity. |
