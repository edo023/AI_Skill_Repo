# Description

Description

Define number of values from which the sliding-type mean value can be calculated. See [FB\_MovingAverage](Function_Blocks_I_to_Q-9.htm#XREF_D_SE_0087308_1)

The setting of the number of values to calculate the sliding-type mean value can be requested via the [uiOrder](Function_Blocks_I_to_Q-9.htm#XREF_D_SE_0087308_1) property.
