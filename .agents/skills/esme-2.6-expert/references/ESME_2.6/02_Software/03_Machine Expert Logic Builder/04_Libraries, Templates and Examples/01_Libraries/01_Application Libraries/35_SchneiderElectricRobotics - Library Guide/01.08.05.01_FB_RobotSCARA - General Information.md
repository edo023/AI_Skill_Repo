## Task

To effect the control of a Lexium SCARA Robot.

## Description

Function block to control a Lexium SCARA Robot.

The initialization is performed using the inherited method [SERP.FB\_RobotSCARAParameters.InitializeRobot()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SchneiderElectricRoboticsParameters&topicID=DD00814941).
