# Troubleshooting

Troubleshooting

| Issue | Cause | Solution |
| --- | --- | --- |
| ScaleInput: Alarm bit 0 | The sensor output is outside the hardware range. | Check the sensor and replace if it is damaged.  OR  Change the range of device-limit-high and device-limit-low. |
| ScaleInput: Alarm bit 1 | Device hardware alarm detected. The alarm bit from the sensor is high. | Check the sensor and replace if it is damaged. |
| ScaleInput: Alarm bit 2 | Device input is lower than the threshold of input-range-low. | This indicates that you should lower the value of low input range if possible (i\_wDevMinThsh). |
| ScaleInput: Alarm bit 3 | Device input is higher than the threshold of input-range-high. | This indicates that you should increase the value of high input range if possible (i\_wDevMaxThsh). |
