# IF\_HomeCmd – General Information

|  |  |
| --- | --- |

## Description

The interface IF\_HomeCmd defines the command methods for the homing modes supported by the function block [FB\_Home](FB_Home-GeneralInformation-459EBABF.html#FB_Home-GeneralInformation-459EBABF) which can be triggered on the assigned axis.

## Methods

| Name | Description |
| --- | --- |
| [SetPosition](IF_HomeCmdSetPositionMethod-44AB4AB9.html#IF_HomeCmdSetPositionMethod-44AB4AB9) | Sets the position of the axis. |
| [HomingToDigitalInput](IF_HomeCmdHomingToDigitalInputMetho-45135217.html#IF_HomeCmdHomingToDigitalInputMetho-45135217) | Sets the position of the axis based on a digital signal. |
| [WriteAxisEncoder](IF_HomeCmdWriteAxisEncoderMethod-451E54DB.html#IF_HomeCmdWriteAxisEncoderMethod-451E54DB) | Writes the axis encoder in connection with setting the axis position |
| [RestorePosFromAxisEncoder](IF_HomeCmdRestorePosFromAxisEncoder-452D28DC.html#IF_HomeCmdRestorePosFromAxisEncoder-452D28DC) | Restores the axis position from the axis encoder value. |
| [HomingToTouchProbe](IF_HomeCmdHomingToTouchProbeMethod-45326E03.html#IF_HomeCmdHomingToTouchProbeMethod-45326E03) | Homes the axis using the signal of a touchprobe input. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| stGetCmdState | REFERENCE TO ST\_HomeCmdState | Get | Returns the state of the last triggered homing command.  NOTE: If a command is rejected because the function block is already processing a homing command, the command state continues to indicate the status of the ongoing homing process. |
