# PackBitsToByte (FUN)

FUNCTION PackBitsToByte : BYTE

Function packs 8 Bits “xBit0” .. “xBit7” into a BYTE.

Example:

```
PackBitsToByte(FALSE,FALSE,FALSE,FALSE,FALSE,FALSE,FALSE,TRUE) = 16#01
```
