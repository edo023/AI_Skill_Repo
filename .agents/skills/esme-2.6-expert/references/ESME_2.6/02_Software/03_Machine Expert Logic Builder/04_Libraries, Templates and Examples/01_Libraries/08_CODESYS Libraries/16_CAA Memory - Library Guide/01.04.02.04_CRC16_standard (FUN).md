# CRC16\_standard (FUN)

FUNCTION CRC16\_standard : WORD

Function calculates the default CRC16 checksum.

Example:

```
sTest : STRING(10) := '123456789' ;
->
MEM.CRC16_standard(ADR(sTest), 9) = 16#BB3D
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | CRC16\_standard | WORD | function returns the default CRC16 checksum of “MemoryBlock” | | Input | pMemoryBlock | POINTER TO BYTE | address of memory Block | | uiLength | UINT | length of MemoryBlock in byte | |
