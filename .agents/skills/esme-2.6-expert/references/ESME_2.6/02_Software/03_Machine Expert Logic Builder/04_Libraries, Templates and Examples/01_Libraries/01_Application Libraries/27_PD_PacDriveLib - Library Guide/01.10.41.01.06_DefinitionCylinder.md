# DefinitionCylinder

DefinitionCylinder

|  |  |
| --- | --- |
| Enumeration name: | DefinitionCylinder |
| Enumeration value: | 42 |
| Description: | The definition of Cylinder is not correct. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | At the input i\_stCylinder.lrRadius, a value <= 0 has been transferred. | The radius of the cylinder must be greater than zero. |
| - | At the inputs i\_stCylinder.stAxisStart and i\_stCylinder.stAxisEnd, the same point has been transferred. | The distance between i\_stCylinder.stAxisStart and i\_stCylinder.stAxisEnd must be at least [Gc\_lrZeroTolerance](../Global_Elements/Global_Elements-2.htm#XREF_D_SE_0087806_1). |
