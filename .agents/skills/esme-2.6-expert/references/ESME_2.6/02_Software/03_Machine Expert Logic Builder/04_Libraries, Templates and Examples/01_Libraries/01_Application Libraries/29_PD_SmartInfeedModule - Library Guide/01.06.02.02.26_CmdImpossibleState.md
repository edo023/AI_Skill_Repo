|  |  |

| Enumeration name: | CmdImpossibleState |

| Issue | Cause | Solution |

The module is in the operation mode  [ET\_OpMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0080821_1).Manual  when it is sent a command  [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1).Start or  [ET\_Cmd](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1).Hold  is send to it. | Before you send a command, ensure that the module can execute the command in the current state. |
