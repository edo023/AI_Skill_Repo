# LRealToStr (FUN)

FUNCTION LRealToStr : STRING(32)

Converts LREAL values to a decimal representation, taking into account special values like NaN and infinity. Based on LREAL\_TO\_STRING.

**Note**

This function also handles the case LREAL = REAL.
