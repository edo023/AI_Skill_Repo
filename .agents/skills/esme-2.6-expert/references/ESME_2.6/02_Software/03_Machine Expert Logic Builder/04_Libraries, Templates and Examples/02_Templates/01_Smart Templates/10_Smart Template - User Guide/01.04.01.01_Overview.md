# Overview

Overview

Smart Template is an innovative programming approach for EcoStruxure Machine Expert. In principle, it is usable across different platforms and covers a broad range of functional modules.

In addition, Smart Template removes many functional restrictions that came with the usage of EDESIGN or Robotics Projects.

The main differences are:

oSmart Template can be used in default projects (no longer a special project type needed).

oSmart Template can be used in combination with SVN (software versioning and revision control system).

oFolders can be used for structuring.

oSmart Template can be used by more than one controller in the project.

oSmart Template can be used in an existing project.

oA new navigator (Modules [view](../../../../../../api/crossBook?lang=en-US&virtualBookName=SmrtTplt_Using_Smart_Template_Modules/SmrtTplt_Using_Smart_Template_Modules-2.htm#XREF_D_SE_0089675_4)) for managing the Smart Template modules can be used as a supplement to the [Classic Navigators](../../../SoMMenu&topicID=D_SE_0083920_1) (Devices, POUs) or the multi-tabbed [Navigators](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083356_1) (Devices tree, Applications tree, Tools tree, Functional tree).

oA new repository dialog is available (Smart Template [Repository](../SmrtTplt_Using_Smart_Template_Modules/SmrtTplt_Using_Smart_Template_Modules-6.htm#XREF_D_SE_0089677_1)) to manage the installation base for Smart Template modules.
