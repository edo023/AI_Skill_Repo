# Global Variable List (GVL)

| Global variable list | Description |
| --- | --- |
| GVL | Contains variable declarations for retain and common variables used to configure the application at initialization. |
| GVL\_Visu | Contains variable declarations for variables linked to the visualization. Variables linked to the visualization are declared exclusively in this list. |
| GVL\_SimulationConfigData | Contains variable declarations to configure the simulation. |
| GVL\_Server | Contains variable declarations for the data provided by the OPC UA server using the Symbol Configuration. |
