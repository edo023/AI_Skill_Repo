# FC\_UpdateCmdTableListElementSimpleTen - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.0.0 |
| Support for: | PacDrive pilot template architecture |

## Task

Function for updating the command table list.

## Description

This function mainly provides the same functionality as the FC\_UpdateCmdTableListElementSimpleOne function. The only difference is that you can update 10 command tables at the same time using this function.
