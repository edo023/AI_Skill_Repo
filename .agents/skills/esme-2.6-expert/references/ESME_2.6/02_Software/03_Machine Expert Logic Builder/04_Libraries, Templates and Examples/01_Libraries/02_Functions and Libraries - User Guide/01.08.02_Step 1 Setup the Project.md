# Step 1: Setup the Project

## Overview

In EcoStruxure Machine Expert ≤V2.3.x, create a new library project by executing the command File > New Project....

In EcoStruxure Machine Expert V2.5 and greater versions, use the EcoStruxure Machine Expert Portal (Platform) to execute this function.

In the New Project dialog box, select the option Library from the Project type list, and define the name and details as described in the *New Project* section of the [Menu Commands Online Help](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0083904).
