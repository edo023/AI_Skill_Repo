# GetHoursIn12hFormat (FUN)

FUNCTION GetHoursIn12hFormat : UINT

Converts the hours into 12h format. For the 12h format: hours 1 - 12 are valid. The 0 from the 24h format has to be converted as well as values 13 - 23.
