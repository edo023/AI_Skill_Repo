# ST\_CmdTable - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.4.0 |
| Inherits from: | - |

## Description

This data type reflects a command table. Command sequences can be created with a table.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| sTitle | STRING[40] | Title of the command table. |
| diCmd | DINT | Command for this table. |
| udiNumberOfCmdSteps | UDINT | Number of steps of the command table. |
| pstCmdTable | POINTER TO [ST\_Cmd](D-SE-0078534.html#D-SE-0078534) | Point of entry (pointer onto first command) of the command table. |
