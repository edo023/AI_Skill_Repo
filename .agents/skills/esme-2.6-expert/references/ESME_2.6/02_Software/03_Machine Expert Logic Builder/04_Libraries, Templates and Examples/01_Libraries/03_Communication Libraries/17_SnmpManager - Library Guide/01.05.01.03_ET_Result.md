# ET\_Result

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_Result contains the possible values that indicate the result and extended error codes of operations executed by the function block.

If detected errors are received in the response telegram from the agent, the outputs iq\_stRequestInfo.q\_stResponse.sOID and iq\_stRequestInfo.q\_stResponse.sAgentIp are still set with the data of the request that caused the error.

In the event of recurring timeouts and other communication errors, verify:

* the physical connections of controller, SNMP agent, and so on.
* the connection configuration of controller and SNMP agent.
* the communication route between controller and SNMP agent. For example, switch and router settings, firewall settings (UDP communication and SNMP ports need to be allowed).
* the provided IP address and SNMP port in iq\_stRequestInfo.i\_stRequest.
* the provided community name in iq\_stRequestInfo.i\_stRequest.
