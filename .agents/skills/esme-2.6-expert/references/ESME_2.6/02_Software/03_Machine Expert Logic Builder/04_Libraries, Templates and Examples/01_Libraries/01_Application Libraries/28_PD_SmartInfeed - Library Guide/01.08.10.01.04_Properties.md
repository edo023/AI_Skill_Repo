# Properties

Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| udiId | UDINT | Read | The identification number of the target |
| lrPosition | LREAL | Read | The distance of the belt to the beginning of the last belt  Negative values indicate that the target is located before the beginning of the last belt.  Positive values indicate that the target is located behind the beginning of the last belt. |
| xValid | BOOL | Read | TRUE: The interface refers to a valid target.  FALSE: The interface does not refer to a valid target. Standard values are returned for the properties. |
| ifProduct | [IF\_Product](Interfaces-14.htm#XREF_D_SE_0080325_1) | Read/Write | The user can allocate a product to a target, for instance to ensure in the algorithm that the product is moved always to the same target. |
