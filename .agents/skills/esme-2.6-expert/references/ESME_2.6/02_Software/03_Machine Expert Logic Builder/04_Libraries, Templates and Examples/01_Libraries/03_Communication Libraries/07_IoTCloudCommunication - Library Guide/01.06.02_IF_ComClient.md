# IF\_ComClient

## Overview

This interface is used to control and monitor a communication client function block. It is implemented by the function block FB\_ESMAClientMonitoringApi.

## Properties

| Name | Data Type | Accessing | Description |
| --- | --- | --- | --- |
| State | [ET\_StateStateDataTransmission](ET_StateComClient-828FEBFE.html) | Get | This property serves to provide information about the state of the client function block. |
| IsConnected | BOOL | Get | This property serves to indicate whether the client function block is connected or not. |
