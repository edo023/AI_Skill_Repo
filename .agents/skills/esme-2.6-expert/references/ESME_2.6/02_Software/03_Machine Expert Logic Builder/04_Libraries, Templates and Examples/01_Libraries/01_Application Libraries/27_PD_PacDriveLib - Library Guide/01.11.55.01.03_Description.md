# Description

Description

The function initializes a data structure of the type [ST\_SemaphoreMe](../Structures/Structures-44.htm#XREF_D_SE_0087794_1). It creates a semaphore of the type MutualExclusion with the settings SEM\_Q\_PRIORITY and SEM\_INVERSION\_SAFE. The selected settings ensure that tasks waiting for the semaphore are enabled in their order of priority. In addition, this avoids the problem of "priority inversion".

For restrictions for use see [Notes on Using the "SemaphoreMutualExclusion" Function](../Presentation_of_the_Library/Presentation_of_the_Library-8.htm#XREF_D_SE_0088285_1).
