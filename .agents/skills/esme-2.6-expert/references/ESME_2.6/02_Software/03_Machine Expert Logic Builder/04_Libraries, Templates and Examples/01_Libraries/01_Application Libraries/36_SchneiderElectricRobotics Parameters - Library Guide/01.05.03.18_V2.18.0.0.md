## Modifications

Lexium S Robot is no longer supported:

* The function block FB\_RobotSSeriesParameters containing the Lexium S Robot parameters is removed.
* The enumeration ET\_RobotSSeries containing the Lexium S Robot types is removed.
* The function FC\_EtRobotSSeriesToString to convert an element of ET\_RobotSSeries to string is removed.
