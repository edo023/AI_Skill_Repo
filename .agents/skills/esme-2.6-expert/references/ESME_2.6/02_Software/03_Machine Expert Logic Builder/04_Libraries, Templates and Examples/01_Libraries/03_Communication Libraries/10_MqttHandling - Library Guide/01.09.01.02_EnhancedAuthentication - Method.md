# EnhancedAuthentication - Method

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V2.4.4.0 |

## Task

Callback method for processing the information contained in the AUTH message received from the MQTT server.

## Functional Description

This method is called by the FB\_MqttClient if the interface was registered and an enhanced authentication is applied for the connection between MQTT client and server. The method provides the information received from the server and allows to define the data that will be sent as response.

## Interface

| Input / Output | Data type | Description |
| --- | --- | --- |
| i\_stResponseDataAuth | [ST\_ResponseDataAuth](ST_ResponseDataAuth-D8FEB42D.html) | Provides the information contained in the AUTH message received from the MQTT server. |
| i\_xTruncatedProperties | BOOL | When TRUE, at least one property received in the response data has been truncated or more user properties are received than defined with [Gc\_ uiMaxNumberOfUserProperties](D-SE-0086551.html#D-SE-0086551__D-SE-0086551.7). |
| iq\_stProperties | [ST\_PropertiesAuth](ST_PropertiesAuth-D900B69B.html) | Specifies the properties of the AUTH message that will be sent as response to the MQTT server. |
