## Description

The method SetPosition of the interface [IF\_SlaveChannel](IF_SlaveChannel-4D6988AD.html) is used to adjust the position reference of the subordinate channel. This allows the user-defined algorithm to apply an absolute or relative offset to the position value of the subordinate axis as managed within the multi-axis group.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrPosition | LREAL | The new position value to be applied to the subordinate channel. |
| i\_xRelative | BOOL | If set to TRUE, the value of i\_lrPosition is added to the current position of the subordinate channel. If set to FALSE, the position is set as an absolute value. |
