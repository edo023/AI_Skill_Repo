# SMC\_GSentenceJumpBuffer (FB)

FUNCTION\_BLOCK SMC\_GSentenceJumpBuffer

This function block buffers g-code sentences in order to avoid a full restart of the g-code file on jumps (G20). It must be used before the [SMC\_NCNavigator](TW1SQNUTI52-XXX1HJ6us52paR8_smc-ncnavigator.html#tw1sqnuti52_xxx1hj6us52par8_smc_ncnavigator_smc_ncnavigator_fb) function block. Note: this function block is part of [SMC\_ReadNCFile2](TW1SQNUTI52-XXX1HJ6us52paR8_smc-readncfile2.html#tw1sqnuti52_xxx1hj6us52par8_smc_readncfile2_smc_readncfile2_fb), so it is not necessary to instantiate it by hand.
