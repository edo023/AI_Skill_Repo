# IF\_Administrative – SetPosition (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.1.0 |
| Versions: | Installed version |

## Task

Set a position value at the position of the motor.

## Description

The method SetPosition sets a position value at the position of the motor and can be called at any time.

NOTE: The method does not affect the motion buffer and can be executed at any time.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrPosition | LREAL | Value range: -2147483648...2147483647  Default value: 0  Position in user-defined units  Value for position setting |
| i\_xRelative | BOOL | Value range: FALSE, TRUE  Default value: FALSE   * FALSE: The position is set to the value of the input Position. * TRUE: The value of the Position is added to the position. If the absolute position is set to TRUE, the flag xHomed of the axis is set to TRUE. Refer to [IF\_AxisInfo](IF_AxisInfoGeneralInformation-431DFDD5.html). |
