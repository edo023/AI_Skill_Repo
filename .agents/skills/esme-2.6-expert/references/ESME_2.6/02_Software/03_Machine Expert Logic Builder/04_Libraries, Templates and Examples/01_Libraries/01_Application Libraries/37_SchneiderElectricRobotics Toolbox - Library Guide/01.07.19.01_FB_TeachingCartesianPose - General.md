## Task

Function block to estimate a cartesian pose starting from a set of sampled poses.

## Description

The function block acquires a set of samples and then uses this data to estimate the relative cartesian pose between a robot and a target coordinate system.

Each sample includes a TCP pose referred to the robot coordinate system and a position representing the same point in space with reference to the target coordinate system.

The estimated pose depends on the arrangement of the samples in space, and it is referred to the robot coordinate system. The resulting pose describes the translation and rotation from the robot coordinate system to the target coordinate system.

NOTE: The Z axis of the teached coordinate system is always assumed to be pointing up, meaning that it must have a positive Z coordinate.

## Methods

| Name | Description |
| --- | --- |
| AddSample | Adds a new sample to the active set. |
| GetSample | Gets a previously stored sample. |
| EstimateCartesianPose | Estimates a cartesian pose based on sampled data. |
| RemoveAllSamples | Removes the stored samples. |
| RemoveLastSample | Removes the last stored sample. |
| SetNumberOfSamplesPerSet | Sets how many samples must be stored inside a set. |
