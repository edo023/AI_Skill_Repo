# Using the Framework Project

This chapter indicates how to use the units of the Framework project as a template for your customized project. It explains core functionalities and how to modify the units.

As a first step of creating a new project, make a clear plan on how to divide the machine into units.

The example described in this document is intended for learning purposes only. It must not be used directly on products that are part of a machine or a process.

This document and its related project file focus on specific functions and function blocks of the Schneider Electric libraries provided with EcoStruxure Machine Expert V2.5 and greater versions, and on specific features related to these libraries. They are intended to help you understand how to develop, test, commission, and integrate applicative software of your own design on control systems.
