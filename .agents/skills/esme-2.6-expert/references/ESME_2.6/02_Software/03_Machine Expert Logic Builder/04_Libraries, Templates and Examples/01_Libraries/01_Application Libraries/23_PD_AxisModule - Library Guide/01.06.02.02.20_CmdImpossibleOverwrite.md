# CmdImpossibleOverwrite

CmdImpossibleOverwrite

|  |  |
| --- | --- |
| Enumeration name: | CmdImpossibleOverwrite |
| Enumeration value: | 3 |
| Description: | The command cannot be overwritten. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | A command was being processed by the module while a new command has been sent that was not [ET\_Cmd.Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) or [ET\_Cmd.BrakeRelease](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1).  The module is in the operating mode Positioning and is currently executing an [ET\_Cmd.Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command while it receives a command that is not [ET\_Cmd.Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1). | Use the command tables of the template to send commands to the module.  Make certain that the module is not processing any commands (ST\_StandardModuleInterface.q\_xCmdActive = FALSE) when you send a new command to the module. |
