## New Functions

* The following methods were added to FB\_TargetsHandler and IF\_TargetsHandler

  :
  + ConfigurationRotative to configure the function block by linking it to a rotative moving system
  + GetConfigurationRotative to retrieve the information on a rotative conveyor entit
  + Reset to reset the configuration of the function block
* The following methods were added to the function block FB\_EntitiesHandler and IF\_EntitiesHandler:

  + AddTrackingSystem to parameterize a tracking system
  + AddTargetGenerator to add a new target generator
  + AddRotativeConveyor to add a new rotative conveyor entity to the function block
  + GetTrackingSystem to read parameter of a tracking system
  + GetTargetGenerator to retrieve the information on a target generator entity
  + CalcTargetGeneratorPoseInTrackingSystem to transform a Cartesian pose referred to a target generator coordinate system
  + CalcEntityPoseInEntitySystem to transform a Cartesian pose in the base frame of a source entity to a Cartesian pose in the base frame of a target entity
  + Reset to reset the configuration of the function block
* New function blocks:

  + FB\_FlowEstimation

    The function block can be used to estimate a flow of products detected by a sensor.
  + FB\_CognexTelnet

    The function block allows you to change the online state of the Cognex Vision System and change the job.
  + FB\_TelnetCommunication

    The function block establishes the communication via Telnet to a camera and provides functionality to send command and receive data.
