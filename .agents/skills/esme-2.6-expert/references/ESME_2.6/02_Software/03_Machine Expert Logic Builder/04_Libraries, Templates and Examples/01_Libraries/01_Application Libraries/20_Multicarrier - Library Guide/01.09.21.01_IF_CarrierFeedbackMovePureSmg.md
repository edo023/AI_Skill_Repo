## Task

Interface with return values for the move command [MovePureSmg](IF_MovePureSmg-5B83BBA5.html#IF_MovePureSmg-5B83BBA5).

## Description

The feedback interface provides reference and status values for the interface IF\_MovePureSmg that allows you to execute carrier movements using a motion job from the SoMotionGenerator (SMG).
