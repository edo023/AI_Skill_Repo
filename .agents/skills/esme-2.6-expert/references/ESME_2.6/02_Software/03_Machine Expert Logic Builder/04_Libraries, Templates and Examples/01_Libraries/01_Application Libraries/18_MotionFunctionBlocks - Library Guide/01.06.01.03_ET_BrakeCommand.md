# ET\_BrakeCommand

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | 1.2.0.0 |

## Description

The enumeration ET\_BrakeCommand contains the possible values for the input command of the brake.

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| NotSet | 0 | No command has been selected. |
| Couple | 1 | Command for coupling the brake. |
| Release | 2 | Command for releasing the brake. |
