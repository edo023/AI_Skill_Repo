# V2.4.1.0

## Overview

This library contains function blocks to parametrize a Schneider Electric robot.

## System Requirements

Using the library with other versions of software or firmware may have results other than those described in the present documentation.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  * Ensure that the software and firmware are of the versions supported by this library. * Contact your Schneider Electric service representative for compatibility information.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

## Supported Hardware

* PacDrive LMC Eco
* PacDrive LMC Pro
* PacDrive LMC Pro2

## Software Requirements

* SoMachine Motion V4.4

## Firmware Requirements

PacDrive 3 V4.4

* PacDrive LMC Eco V1.56.15.12 or greater
* PacDrive LMC Pro V1.56.15.12 or greater
* PacDrive LMC Pro2 V1.56.15.12 or greater

## New Functions

* Lexium S Robot

  + Function block FB\_RobotSSeries to parametrize a Lexium S Robot.
  + Enumeration ET\_RobotSSeriesComponent to address a specific axis of a Lexium S Robot.
  + Function FC\_EtRobotSSeriesComponentToString to convert an element of ET\_RobotSSeriesComponent to string.
  + Interface IF\_RobotSSeriesCalibration to calibrate a Lexium S Robot.
