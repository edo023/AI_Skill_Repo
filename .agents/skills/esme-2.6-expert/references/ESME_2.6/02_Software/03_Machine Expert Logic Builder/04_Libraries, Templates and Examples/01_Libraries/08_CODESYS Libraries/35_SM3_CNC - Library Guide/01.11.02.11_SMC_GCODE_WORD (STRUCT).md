# SMC\_GCODE\_WORD (STRUCT)

TYPE SMC\_GCODE\_WORD : STRUCT

This data type includes the data of one G-code word like ‘N20’ or ‘G1’.

| InOut: | | Name | Type | Initial | Comment | | --- | --- | --- | --- | | fValue | LREAL | 0 | Value as LREAL. This value is used by [SMC\_NCDecoder](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-ncdecoder.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_ncdecoder_smc_ncdecoder_fb). If the SMC\_GCODE\_WORD is created programatically, this value has to be set for all g-words (e.g. for ‘M5’, set it to 5.0). | | diValue | DINT | 0 | Value as DINT . Not used any more. If the SMC\_GCODE\_WORD is created programatically, we recommend to set this value to TO\_DINT(fValue). | | pAdr | POINTER TO BYTE | 0 | If <> 0, the value of this word is specified by a variable. pAdr points to this variable. | | byLetter | BYTE | 0 | ASCII-code of the letter of the word, e.g. 69 for ‘F’ | | byVarType | BYTE | 0 | If < 32, contains the variable type as defined in [SMC\_VARTYPE](ObV4AuTnEd4qdCdTi1H9MI9BbW8_smc-vartype.html#obv4autned4qdcdti1h9mi9bbw8_smc_vartype_smc_vartype_enum).  If >= 32, contains 32 + the variable type as defined in IBaseLibrary.TypeClass | |
