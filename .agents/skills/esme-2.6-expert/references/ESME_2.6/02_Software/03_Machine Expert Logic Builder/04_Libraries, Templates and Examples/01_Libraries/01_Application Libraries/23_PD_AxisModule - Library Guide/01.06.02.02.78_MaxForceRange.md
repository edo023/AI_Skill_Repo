# MaxForceRange

MaxForceRange

|  |  |
| --- | --- |
| Enumeration name: | MaxForceRange |
| Enumeration value: | 160 |
| Description: | MaxForce is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) was selected with the mode    [ST\_Home.i\_etMode](../../../../../../api/crossBook?lang=en-US&virtualBookName=Structures/Structures-5.htm#XREF_D_SE_0077219_1) = [PDL.ET\_HomeMode.PosDirectionMaxForce](../../../PD.Lib.PacDriveLib&topicID=D_SE_0087221_1) or  [ST\_Home.i\_etMode](../../../../../../api/crossBook?lang=en-US&virtualBookName=Structures/Structures-5.htm#XREF_D_SE_0077219_1) = [PDL.ET\_HomeMode.NegDirectionMaxForce](../../../PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).    An invalid value was transferred to the [ST\_Home.stForce.i\_lrMaxForce](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) parameter. | The following must apply:  0.0 < ([ST\_Home.stForce.i\_lrMaxForce](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) / lrForceConstant \* 10000000.0 / lrMaxDrivePeakCurrent / lrLimCurrent) < 100.0    with lrLimCurrent := lrUserDrivePeakCurrent \* 100.0 / lrMaxDrivePeakCurrent,  whereat lrForceConstant, lrMaxDrivePeakCurrent and lrUserDrivePeakCurrent are parameters of the axis. |
