# Output Pin Description

Output Pin Description

| Output | Data Type | Description |
| --- | --- | --- |
| q\_xEn | BOOL | TRUE: Function block enabled  FALSE: Function block disabled |
| q\_xDrvFwd | BOOL | Drive forward run command.  TRUE: Forward  FALSE: Not forward |
| q\_xDrvRev | BOOL | Drive reverse run command.  TRUE: Reverse  FALSE: Not reverse |
| q\_xDrvFwdSlow | BOOL | Drive forward slow-down command to a safe speed.  TRUE: Forward slow  FALSE: Not forward slow |
| q\_xDrvRevSlow | BOOL | Drive reverse slow-down command to a safe speed.  TRUE: Reverse slow  FALSE: Not reverse slow |
| q\_wLsStat | WORD | Status of limit switch management FB.  Range: 0, 1, 2, 3, 4  Refer to detailed description below this table. |
| q\_rDistTrvl | REAL | Actual distance traveled in slow-down area.  Range: 1.175494351e-38...3.402823466e+38  Scaling/Unit: 1 m |
| q\_xAlrm | BOOL | Detected alarm bit.  TRUE: Alarm detected  FALSE: No alarm detected  Refer to detailed description below this table. |
| q\_wAlrmId | WORD | Detected alarm identification.  Refer to [Notifications](#XREF_D_SE_0020241_5).  Range: 0...15 |
