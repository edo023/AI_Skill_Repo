# SMC\_ControlAxisByPosVel (FB)

FUNCTION\_BLOCK SMC\_ControlAxisByPosVel

This function block writes the set position fSetPosition and the set velocity fSetVelocity to the axis drive structure and monitors axis for jumps in a similar way to [SMC\_ControlAxisByPos](RjOhgD1RHWXzClvCTVcAUflF8nc_smc-controlaxisb.html#rjohgd1rhwxzclvctvcauflf8nc_smc_controlaxisb_smc_controlaxisbypos_fb). SMC\_ControlAxisByPosVel is mostly used with CNC and an instance of the [SMC\_Interpolator](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-interpolator.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_interpolator_smc_interpolator_fb).

**Note**

For direct checking of the set value of the axis, one of the SMC\_Follow\* blocks should be used.

See also: [SMC\_ControlAxisByPos](RjOhgD1RHWXzClvCTVcAUflF8nc_smc-controlaxisb.html#rjohgd1rhwxzclvctvcauflf8nc_smc_controlaxisb_smc_controlaxisbypos_fb)
