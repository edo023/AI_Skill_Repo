# FC\_MatrixGetElement

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.12.0 |

## Task

The function reads a value from the specified position of a two dimensional LREAL array (matrix).

## Description

The function validates the provided row and column indices against the runtime bounds of the matrix. If the indices are out of range, it reports an error and returns an InputValueInvalid result. If the indices are valid, the function returns the value of the specified the matrix element.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_uiRow | UINT | Row index of the element that is read. |
| i\_uiCol | UINT | Column index of the element that is read. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_alrMatrix | ARRAY [\*, \*] OF LREAL | Matrix from which the element is read. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-GeneralInformation-0C182C26.html#ET_Result-GeneralInformation-0C182C26) | Provides diagnostic and status information as a numeric value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |

## Return Value

| Data type | Description |
| --- | --- |
| LREAL | When successful, the function returns the requested element value; otherwise, it returns NaN. |

## Troubleshooting

| Issue | Cause | Solution |
| --- | --- | --- |
| Execution ends with a detected error and result indicates InputValueInvalid. | The given value for i\_uiRow is out of range. | Verify that the value of i\_uiRow lies within the bounds of the first dimension of the assigned array. |
| The given value for i\_uiCol is out of range. | Verify that the value of i\_uiCol lies within the bounds of the second dimension of the assigned array. |
