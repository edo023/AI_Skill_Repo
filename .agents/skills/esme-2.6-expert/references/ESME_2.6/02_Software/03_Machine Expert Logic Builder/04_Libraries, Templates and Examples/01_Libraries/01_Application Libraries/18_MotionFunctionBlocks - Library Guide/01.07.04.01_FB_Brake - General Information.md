| Inherits from: | – |

## Description

The function block FB\_Brake is used to control a motor brake during machine commissioning. Controlling the brake using this function block is only possible when the axis is in state Disabled.

The following preconditions apply for the function block:

* The controlled axis implements the interface DDI.IF\_Brake.
* The function block is supported for physical and virtual axes.

NOTE: This function block has no effect if the property BrakeMode of the interface DDI.IF\_Brake is set to **Control brake by user program / 3**.

## ApplicationLogger - Logger Point

The function block implements a logger point that can be registered with the global Application Logger.

When registered, the function block generates logger entries on the following events:

| Event | Logger level |
| --- | --- |
| Command release/couple the brake is triggered. | APL2.ET\_LogLevel.UserAction |
| Command release/couple the brake successfully completed. | APL2.ET\_LogLevel.StatusMessage |
| Command release/couple the brake unsuccessful. | APL2.ET\_LogLevel.StatusMessage |

With the method [Register](IF_LoggerPointConfigurationRegister-4599E66B.html), the function block FB\_Brake is registered as a logger point in the global Application Logger. The values for the LoggerPoint parameters type and source are set to FB\_Brake and MFB, respectively. For more information refer to the ApplicationLogger Library Guide.
