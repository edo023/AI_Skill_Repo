# IoDrvEthernetDiag.UpdateConfiguredIPSettings (METH)

METHOD UpdateConfiguredIPSettings : [ErrorCode](XYuqQn58m-Ao0rMJIF3SueTsKW8_errorcode.html#xyuqqn58m_ao0rmjif3suetskw8_errorcode_errorcode_enum)

This method updates the configured ip address, subnet mask and standard gateway of the ethernet node. The device has to be in disabled state (via [Enable](m1wzSsf8oZufwhKQ0elXJ-drqSc_enable.html#m1wzssf8ozufwhkq0elxj_drqsc_enable_iodrvethernetdiag_enable_prop), see also DED.Reconfigure), else the update is rejected.
