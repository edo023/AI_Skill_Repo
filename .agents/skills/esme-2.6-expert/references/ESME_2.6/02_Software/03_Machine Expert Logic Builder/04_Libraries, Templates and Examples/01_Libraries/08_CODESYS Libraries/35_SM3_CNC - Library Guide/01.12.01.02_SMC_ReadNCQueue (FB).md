# SMC\_ReadNCQueue (FB)

FUNCTION\_BLOCK SMC\_ReadNCQueue

This FB is used for files created with Codesys V2.3; To write the queue to a file or to read the queue from a file: the FBs SMC\_WriteQueueToFile and SMC\_ReadQueueFromFile from SM3\_Debug library can be used. This function block will read an [SMC\_OUTQUEUE](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-outqueue.html#cah0ife6nckjzp5ef3qpy_filyk_smc_outqueue_smc_outqueue_fb) file, which has been created by the CNC editor, from the PLC file system and provide an OutQueue structure, which typically is processed by the decoder.
