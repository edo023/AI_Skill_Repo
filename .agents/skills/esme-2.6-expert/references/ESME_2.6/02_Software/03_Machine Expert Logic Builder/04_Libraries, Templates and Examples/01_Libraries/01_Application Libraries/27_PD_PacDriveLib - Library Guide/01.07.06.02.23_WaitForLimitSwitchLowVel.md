WaitForLimitSwitchLowVel

|  |  |
| --- | --- |
| Enumeration name: | WaitForLimitSwitchLowVel |
| Enumeration value: | 224 |
| Description: | Waiting for signal from LimitSwitch during travel at LowVel velocity. |
