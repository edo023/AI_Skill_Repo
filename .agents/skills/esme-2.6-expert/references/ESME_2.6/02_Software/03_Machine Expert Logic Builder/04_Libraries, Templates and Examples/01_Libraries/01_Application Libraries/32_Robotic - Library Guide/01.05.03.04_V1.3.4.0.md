## New Functions

* IF\_RobotConfiguration.SchneiderElectricRobot

  With this method, the robot can be configured as a Schneider Electric robot.

## Modifications

* Required License Points

  25 license points are required for a Schneider Electric Lexium P Robot configured by IF\_RobotConfiguration.SchneiderElectricRobot.

  400 license points are required for a triaxial Delta robot configured by IF\_RobotConfiguration.Delta3Ax.
