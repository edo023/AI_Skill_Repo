## Task

Interface with parameters from the move command [MoveSyncFromStandstill](IF_MoveSyncPathFromStandstill-5B839E78.html#IF_MoveSyncPathFromStandstill-5B839E78).

## Description

The feedback interface provides parameters from a synchronized movement of the carrier with the move command MoveSyncFromStandstill.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| lrDistanceToMasterAtStartSync | LREAL | Read | Indicates the distance from the reference position of the carrier on channel A to the reference position of the master at the start of the synchronization. NOTE: The superimposed parts of the movement on channel B and C are not considered for the distance. For more information on superimposed movements, refer to [IF\_MoveSyncFromStandstillSuperimposedChannelC](MoveSyncFrStandstSuperimpC-2F9D684A.html#MoveSyncFrStandstSuperimpC-2F9D684A). |
