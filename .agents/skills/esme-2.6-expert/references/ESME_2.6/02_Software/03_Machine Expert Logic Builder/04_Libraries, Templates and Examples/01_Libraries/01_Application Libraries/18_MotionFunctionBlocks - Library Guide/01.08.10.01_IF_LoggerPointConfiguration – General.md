# IF\_LoggerPointConfiguration – General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.3.1.0 |
| Inherits from: | — |

## Description

The interface provides the connection to a logger point. Using this interface the configuration of the logger point can be done.

## Method

| Name | Description |
| --- | --- |
| [Register](IF_LoggerPointConfigurationRegister-4599E66B.html#IF_LoggerPointConfigurationRegister-4599E66B) | Registers a logger point to the Application Logger. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| etLogLevel | APL2.ET\_LogLevel | Get / Set | Log level of the logger point registered at the **Application Logger**.  If the LoggerPoint is not registered, the value is not valid and a new value is ignored.  NOTE: The LogLevel of the LoggerPoint can be modified also from the global object Application Logger. |
| xIsRegistered | BOOL | Get | Verifies whether the LoggerPoint has been registered. |
| udiId | UDINT | Get | Get the identification number of the logger point for further processing. |
