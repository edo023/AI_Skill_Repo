# UnpackByte (FB)

FUNCTION\_BLOCK UnpackByte

Function block unpacks “byValue” and returns the 8 bits “xBit0” .. “xBit7”.

Prefix: “memuby”.

Example:

```
byInput = 16#01 = 2#00000001
->
xBit0 = TRUE
xBit1..xBit7 = FALSE
```
