## Description

This library is a collection of POUs for controlling robot kinematics.

Kinematics that can be controlled:

* Triaxial Delta robot - IF\_RobotConfiguration.Delta3Ax
* Triaxial Cartesian robot - IF\_RobotConfiguration.Cartesian3Ax
* Biaxial Cartesian robot - IF\_RobotConfiguration.Cartesian2Ax
* Up to triaxial user-defined robot - IF\_RobotConfiguration.User3Ax

Motion interpolation in order to move to a point in two or three-dimensional space:

* Linear interpolation - IF\_RobotMotion.MoveL
* Circular interpolation - IF\_RobotMotion.MoveC
* Spline interpolation - IF\_RobotMotion.MoveS
