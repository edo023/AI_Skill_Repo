# ID\_TO\_ADDR (FB)

FUNCTION\_BLOCK ID\_TO\_ADDR

The function block ID\_TO\_ADDR converts the handle used for the Communication Function Blocks to the physical identification of a slot of a PROFIBUS DP-slave or subslot of a PROFINET IO Device.

**Code-Example: see [RALARM](oxvisIujE0YbnEvay2vz2A6FFPI_ralarm1.html#oxvisiuje0ybnevay2vz2a6ffpi_ralarm1_ralarm_fb)**
