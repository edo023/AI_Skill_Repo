# Interface

Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrTarget | LREAL | Specifies the axis’s position after the homing process is complete in units |
| i\_lrVelocity | LREAL | Specifies the homing velocity unless reduced velocity is enabled in units/sec |
| i\_lrAcceleration | LREAL | Specifies the homing acceleration in units/sec^2 |
| i\_lrDeceleration | LREAL | Specifies the homing deceleration in units/sec^2 |
| i\_lrPeriod | LREAL | The position of the axis is kept between zero and period. The axis will not move if period = 0 |
| i\_xRotativeSystem | BOOL | Specifies how the axis will seek the position if the axis is already there |
| i\_lrJerk | LREAL | Specifies the jerk for homing in units/sec^3 |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stAxisModuleItf | ST\_ModuleInterface | The axis interface structure of the associated axis |
