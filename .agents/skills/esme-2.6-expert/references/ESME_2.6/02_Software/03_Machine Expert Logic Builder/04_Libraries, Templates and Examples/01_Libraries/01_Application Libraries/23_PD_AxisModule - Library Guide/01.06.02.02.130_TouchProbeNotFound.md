# TouchProbeNotFound

TouchProbeNotFound

|  |  |
| --- | --- |
| Enumeration name: | TouchProbeNotFound |
| Enumeration value: | 96 |
| Description: | No Touchprobe was found. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.PosDirectionPosEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionPosEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.NegDirectionNegEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  [PDL.ET\_HomeMode.PosDirectionNegEdgeTp](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1),  The Touchprobe sensor does not transmit a signal. | Check the power supply of the Touchprobe sensor.  Ensure that the Touchprobe sensor is not dirty.  Ensure that the Touchprobe can be triggered by the mechanical system.  Check the electrical connection of the Touchprobe sensor and of the Touchprobe input. |
