# wBrakDly

wBrakDly

This is an alternative to the feedback signal from a brake (i\_xBrakFbck). If the brake feedback signal is not available and the i\_stPAS.wBrakDly value is not zero, an internal timer is used to enable the speed profile generation instead.

If there is a command to run present (i\_xDrvFwd or i\_xDrvRev) and other conditions to start the movement are fulfilled, the corresponding direction output command (q\_xDrvFwd or q\_xDrvRev) is set to TRUE immediately. (in order to switch the drive to RUN state) Ramp generation starts after the time specified by i\_stPAS.wBrakDly has elapsed.
