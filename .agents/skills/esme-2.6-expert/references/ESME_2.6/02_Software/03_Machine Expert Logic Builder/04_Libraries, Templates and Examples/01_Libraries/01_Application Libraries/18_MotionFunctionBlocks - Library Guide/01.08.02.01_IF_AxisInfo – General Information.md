# IF\_AxisInfo – General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.1.0 |
| Inherits from: | – |

## Description

The interface provides status information on the axis.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| xCommunicationReady | BOOL | Get | The property indicates if the communication to the drive is ready.  TRUE: The fieldbus is ready for communication. |
| xIsHomed | BOOL | Get | The property indicates if the axis is homed.  TRUE: The reference point is valid (axis homed). |
| xPowerOn | BOOL | Get | The property indicates if the power of the drive is enabled.  In order to ensure that the drive is ready to move, also verify whether the axis state is in state standstill.  TRUE: The power stage is enabled. |
| xReadyForPowerOn | BOOL | Get | The property indicates if the drive is ready to enable the power stage. Additionally verify whether the drive InverterEnable is ready.  TRUE: The drive is ready to enable power stage. |
| xIsModulo | BOOL | Get | The property indicates if the axis limits are defined as modulo.  TRUE: The axis limits are defined as modulo.   * If the axis position value is less than 0, it is increased by the axis period value. * If the axis position becomes equal to or greater than the axis period value, it is reduced by the axis period value.   This position jump has no effect on the physical movement of the drive that controls the axis. |
| lrModuloPeriod | LREAL | Get | The property indicates the modulo period of the axis.  If the axis limits are defined as modulo, the axis position is kept in a range of [0...lrModuloPeriod]. Otherwise, the value of this parameter is zero. |
| lrNegativeDirectionLimit | LREAL | Get | If the axis is defined as linear with limited movement range, the negative position limit is provided. Otherwise, the value is 0.   * The axis performs an emergency stop if the position of the axis is moved in a negative direction beyond this limit. * If the axis position value is less than this limit, only movement commands which increase the axis position values are allowed. |
| lrPositiveDirectionLimit | LREAL | Get | If the axis is defined as linear with limited movement range, the positive position limit is provided. Otherwise, the value is 0.   * The axis performs an emergency stop if the position of the axis is moved in a positive direction beyond this limit. * If the axis position value is less than this limit, only movement commands which decrease the axis position values are allowed. |
| xIsLimited | BOOL | Get | TRUE: The axis limits are defined as linear axis with limited movement range. If the axis position value exceeds the limits, an error stop movement is executed. |
| xSimulation | BOOL | Get | The property indicates if the axis is simulated.  TRUE: The axis is simulated.  FALSE: The axis is not simulated. |
