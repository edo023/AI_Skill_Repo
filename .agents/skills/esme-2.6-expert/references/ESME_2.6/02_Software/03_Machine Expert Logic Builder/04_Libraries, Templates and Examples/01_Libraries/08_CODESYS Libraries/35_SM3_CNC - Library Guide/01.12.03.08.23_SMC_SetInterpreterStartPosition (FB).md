# SMC\_SetInterpreterStartPosition (FB)

FUNCTION\_BLOCK SMC\_SetInterpreterStartPosition

This function block copies the actual position from the bus task to the task of the [SMC\_NCInterpreter](VVFvsggjXKxZ6qmLw10MvZPswnU_smc-ncinterprete.html#vvfvsggjxkxz6qmlw10mvzpswnu_smc_ncinterprete_smc_ncinterpreter_fb), in a task safe manner. Usually, the position is computed based on the actual position of the drives and the kinematic forward transformation.

To be called from the bus task.
