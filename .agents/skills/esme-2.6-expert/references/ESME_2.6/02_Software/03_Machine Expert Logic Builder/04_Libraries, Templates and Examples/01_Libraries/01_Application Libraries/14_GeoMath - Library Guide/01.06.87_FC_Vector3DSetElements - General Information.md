## Description

Given a set of input elements, the function returns a new 3D vector that contains such elements.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrX | LREAL | First element to set. |
| i\_lrY | LREAL | Second element to set. |
| i\_lrZ | LREAL | Third element to set. |
