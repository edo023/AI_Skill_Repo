# MESSAGE (STRUCT)

TYPE MESSAGE : STRUCT

This structure defines a CANL2 message. Only use this structure if you are using unmanaged CANL2 functions (functions with underscore). For unmanaged CL2 functions the returned handle hMsg can be interpreted as a pointer to this structure because no resource manager is involved. For interpreting the structure members use the bit masks defined in [MASKS](Q4p3QYOxX2Yu5WhJkTflLOyfYRg_masks.html#q4p3qyoxx2yu5whjktflloyfyrg_masks_masks_gvl).
