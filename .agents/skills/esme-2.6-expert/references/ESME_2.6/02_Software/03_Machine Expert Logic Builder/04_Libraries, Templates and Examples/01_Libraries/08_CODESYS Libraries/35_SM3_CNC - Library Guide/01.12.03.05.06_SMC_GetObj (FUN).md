# SMC\_GetObj (FUN)

FUNCTION SMC\_GetObj : POINTER TO SMC\_GEOINFO

Returns the n-th object of an [SMC\_OUTQUEUE](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-outqueue.html#cah0ife6nckjzp5ef3qpy_filyk_smc_outqueue_smc_outqueue_fb). For N=0 the first element will be returned. If the number of elements is less than n+1, zero is returned.

task-safety: call from consumer task only
