# ET\_CollisionHandlerType – General Information

## Overview

|  |  |
| --- | --- |

## Description

The elements of ET\_CollisionHandlerType identify the type of collision handler implemented by an object.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | - |
| Delta2Ax | 1 | A Delta2Ax robot |
| Delta3Ax | 2 | A Delta3Ax robot |
| SCARA4Ax | 3 | A SCARA4Ax robot |
| PSeries | 4 | A P-Series robot |
| TSeries | 5 | A T-Series robot |
