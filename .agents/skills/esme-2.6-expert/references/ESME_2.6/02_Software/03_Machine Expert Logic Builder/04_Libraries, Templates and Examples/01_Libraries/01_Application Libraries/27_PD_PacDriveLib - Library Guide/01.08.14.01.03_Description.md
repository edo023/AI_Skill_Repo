# Description

Description

The function implements a PD controller. A positive signal at i\_xStart enables the function block. The input signal i\_lrInput is calculated from the difference i\_lrInput := (reference value - actual value). The parameters i\_lrKP and i\_timTD must be chosen according to the enabled power stage. The output setpoint is calculated according to the following formula from the current input parameters and the previous progression.

q\_lrOutput := i\_lrKP \* ( i\_lrInput + d(i\_lrInput) \* i\_timTD )

Where d() is the derivative with respect to time.
