# MaxTorqueRange

MaxTorqueRange

|  |  |
| --- | --- |
| Enumeration name: | MaxTorqueRange |
| Enumeration value: | 85 |
| Description: | MaxTorque is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operating mode [ET\_OpMode.Homing](../Enumerations/Enumerations-5.htm#XREF_D_SE_0077130_1) has been selected with the [ST\_Home.i\_etMode](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) mode equal to  [PDL.ET\_HomeMode.PosDirectionMaxTorque](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1) or  [PDL.ET\_HomeMode.NegDirectionMaxTorque](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).  An invalid value was transferred to the [ST\_Home.stTorque.i\_lrMaxTorque](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) parameter. | The following must apply: 0.0 < ([ST\_Home.stTorque.i\_lrMaxTorque](../Structures/Structures-5.htm#XREF_D_SE_0077219_1) / lrTorqueConstant \* 10000000.0 / lrMaxDrivePeakCurrent / lrLimCurrent) < 100.0  with lrLimCurrent := lrUserDrivePeakCurrent \* 100.0 / lrMaxDrivePeakCurrent  where lrTorqueConstant, lrMaxDrivePeakCurrent and lrUserDrivePeakCurrent are parameters of the axis. |
