# GetTransmitPoolSize (FUN)

FUNCTION GetTransmitPoolSize : CAA.COUNT

Returns the number of free transmit messages in the transmit message pool of the current driver. Should be greater 0. Otherwise no more messages can be sent. Diagnostic Information can be also read by [GetDiagnosis](8gqnJ6B5M46-D0DeyJx7cbK99BA_getdiagnosis.html#8gqnj6b5m46_d0deyjx7cbk99ba_getdiagnosis_getdiagnosis_fun).
