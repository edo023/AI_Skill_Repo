# ET\_SqlRequest

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V2.0.0.0 |

## Description

The enumeration ET\_SqlRequest defines which request (read/write) is to be executed.

## Enumeration Elements

| Name | Data type | Value | Description |
| --- | --- | --- | --- |
| Read | INT | 0 | Performs SQL requests that read data from the SQL database. |
| Write | INT | 1 | Performs SQL requests that update or modify the SQL database. |
