# DriveInvalid

DriveInvalid

|  |  |
| --- | --- |
| Enumeration name: | DriveInvalid |
| Enumeration value: | 6 |
| Description: | Connected drive is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The interface ST\_Main.i\_ifDrive is empty | A valid interface must be transferred to ST\_Main.i\_ifDrive |
| - | The connected drive does not support all required functionalities | Consider q\_sMsg from the exception structure to identify what feature the drive is unavailable  Connect a drive that supports all required functionalities |
