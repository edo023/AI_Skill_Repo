# General

General

Command Processing

A successful executed command is displayed as follows:

q\_xCMDActive = FALSE

q\_diCMDActive = last command

q\_xCMDDone = TRUE

iq\_diCMD = 0

 Operating mode commands are not possible if an error reaction [TPL.ET\_Reaction.AsyncStop](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1), [TPL.ET\_Reaction.SyncStopEL](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1) or [TPL.ET\_Reaction.SyncStopEH](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1) is pending.

You cannot enable the position control via the [DriveEnableOn](../../../../../../api/crossBook?lang=en-US&virtualBookName=Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command if an error reaction [TPL.ET\_Reaction.AsyncStop](../../../PD.Lib.Template&topicID=D_SE_0078354_1), [TPL.ET\_Reaction.SyncStopEL](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.Template&topicID=D_SE_0078354_1) is pending.
