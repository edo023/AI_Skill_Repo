# ST\_Administration

## Overview

|  |  |
| --- | --- |
| Type | Data structure |
| Available as of | V1.0.1.0 |

## Description

The structure ST\_Administration represents the list of PackTags of type Administration as defined in ANSI/ISA TR88.00.02-2022.

Administration tags are used to describe the quality and alarm information of the unit machine.
