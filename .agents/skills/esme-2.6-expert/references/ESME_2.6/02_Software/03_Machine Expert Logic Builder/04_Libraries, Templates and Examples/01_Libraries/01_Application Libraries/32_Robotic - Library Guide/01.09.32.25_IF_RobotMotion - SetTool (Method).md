## Task

Selecting a configured tool.

## Description

With the method SetTool a configured tool can be selected as TCP for the robot motion.

In case the input i\_udiToolId is set to zero an active tool is deactivated, so the commands and the feedback refers to the robot flange.

The tool can only be switched when no move commands are active or loaded for the robot.

After a tool switch, one RTP cycle is necessary to process the tool change. You must call the method SetTool and the Move commands in two different IEC-cycles.
