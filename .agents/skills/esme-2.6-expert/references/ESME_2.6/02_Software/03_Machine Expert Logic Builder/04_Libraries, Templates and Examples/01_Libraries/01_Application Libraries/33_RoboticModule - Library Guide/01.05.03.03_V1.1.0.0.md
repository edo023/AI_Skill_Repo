## Software Requirements

* SoMachine Motion V4.1

## Firmware Requirements

PacDrive 3 V4.1

* PacDrive LMC x00C V1.51.10.6 or greater
* PacDrive LMC x01C V1.51.10.6 or greater

## New Functions

* IF\_Configuration.Delta2Ax

  With this method, the robot can be configured as a biaxial Delta robot with two degrees of freedom.
* Diagnostic concept in Online Help

  The Diagnostic concept in the Online Help of the RoboticModule library is fully supported in the present version.

## Modifications

* None

## Enhancements

* IF\_ConfigurationAdvanced.ModifyCoordinateSystem

  In case of a two-dimensional transformation of the robot, the method verifies, whether it is possible to move or rotate the coordinate system or not.

  It is not possible to move the coordinate system in Z direction if the working plane of the robot is XY.
