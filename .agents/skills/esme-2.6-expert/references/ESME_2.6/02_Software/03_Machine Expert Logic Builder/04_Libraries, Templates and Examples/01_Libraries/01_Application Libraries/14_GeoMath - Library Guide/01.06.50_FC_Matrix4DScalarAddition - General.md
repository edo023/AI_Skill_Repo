## Description

Given a 4D matrix and a scalar value, the function returns a new 4D matrix resulting from adding the scalar value to each element of the input matrix.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stMatrix | SE\_MATH.ST\_Matrix4D | A 4D matrix. |
| i\_lrScalar | LREAL | A scalar value to add to the elements of i\_stMatrix. |

| Output | Data type | Description |
| --- | --- | --- |
