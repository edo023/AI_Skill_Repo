## New Features

The Lexium SCARA Robot is supported:

* With the function block [FB\_RobotSCARA](FB_RobotSCARA-GeneralInformation-0E9E82B9.html) and the interface [IF\_RobotSCARA](IF_RobotSCARA-GeneralInformation-30873EA7.html) , it is possible to configure a Lexium SCARA Robot.
* The enumeration [ET\_RobotSCARAComponent](D-SE-0088976.html) addresses the axes of a Lexium SCARA Robot either individually or collectively.
* The function [FC\_EtRobotSCARAComponentToString](FC_EtRobotSCARAComponentToString-Ge-3AEB404B.html) converts an element of [ET\_RobotSCARAComponent](D-SE-0088976.html) to string.
