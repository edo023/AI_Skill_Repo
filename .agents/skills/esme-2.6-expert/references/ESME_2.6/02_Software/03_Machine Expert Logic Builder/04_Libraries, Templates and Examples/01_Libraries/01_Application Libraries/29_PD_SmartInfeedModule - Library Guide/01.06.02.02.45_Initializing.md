# Initializing

Initializing

|  |  |
| --- | --- |
| Enumeration name: | Initializing |
| Enumeration value: | 6 |
| Description: | The POU is being initialized. |

The POU is being initialized and thus is not yet ready to receive commands at its inputs.

The POU will signalize that it is ready for operation with the signal q\_xReady = TRUE.
