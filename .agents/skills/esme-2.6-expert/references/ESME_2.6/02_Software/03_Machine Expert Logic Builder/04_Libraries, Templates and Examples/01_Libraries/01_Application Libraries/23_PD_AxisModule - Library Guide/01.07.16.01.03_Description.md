# Description

Description

This function is used to specify that an axis, when homed, will be homed by seeking an input or sensor signal. The i\_etMode input specifies the homing direction:

| i\_etHomingMode | Home Action |
| --- | --- |
| PDL.ET\_HomeMode.PosDirectionPosEdgeSensor | Move in the positive direction until the rising edge is detected. |
| PDL.ET\_HomeMode.NegDirectionPosEdgeSensor | Move in the negative direction until the rising edge is detected. |
| PDL.ET\_HomeMode.NegDirectionNegEdgeSensor | Move in the negative direction until the falling edge is detected. |
| PDL.ET\_HomeMode.PosDirectionNegEdgeSensor | Move in the positive direction until the falling edge is detected. |

All the possible values of i\_etMode are enumerated in PDL.ET\_HomeMode; however, the function returns FALSE if none of the above four values is used.

The i\_xRotativeSystem input specifies how the axis will search for the boolean signal if the sensor is activated when the homing starts. An example for the PosDirectionPosEdgeSensor mode: If i\_xRotativeSystem = FALSE, the axis will move in the negative direction until the input turns off (falling edge). The axis then reverses direction and begins a new. If i\_xRotativeSystem = TRUE, the axis will move in the positive direction until the rising edge of the input is found.
