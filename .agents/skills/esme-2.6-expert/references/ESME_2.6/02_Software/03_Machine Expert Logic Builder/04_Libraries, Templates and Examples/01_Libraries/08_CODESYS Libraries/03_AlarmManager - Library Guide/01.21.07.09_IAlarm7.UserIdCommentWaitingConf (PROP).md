# IAlarm7.UserIdCommentWaitingConf (PROP)

PROPERTY UserIdCommentWaitingConf : POINTER TO WSTRING

Optional user id that can be provided with the comment for the state “Waiting for confirmation” or “Inactive, Unacknowledged” if the option “Acknowledge instead of confirm” is active.
