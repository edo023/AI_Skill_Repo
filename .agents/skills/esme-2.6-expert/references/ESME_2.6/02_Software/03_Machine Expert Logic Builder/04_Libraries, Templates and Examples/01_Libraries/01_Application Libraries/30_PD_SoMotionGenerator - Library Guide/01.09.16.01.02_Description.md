# Description

Description

Data structure for parameterization of cams that have been generated using the system function [SystemInterface.FC\_ProfileLoad()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085263_1)

NOTE: Used user profiles must be available on the flash disk. When replacing the flash disk, make sure that all profile data are copied.
