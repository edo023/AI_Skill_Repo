# Why Use the DiagnosticCoverage_2 Function Block?

Why Use the DiagnosticCoverage\_2 Function Block?

The function block compares application signature and firmware version with configured values. It also monitors the execution of selected POUs, the measured cycle time of a cyclic task and provides a means for cross-checking of two controllers using two 16-bit pseudo-random variables. Internally the function block tests parts of the variable memory, boolean and floating point operations.
