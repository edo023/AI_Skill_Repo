# RecipeManCommands.CreateRecipe (METH)

METHOD CreateRecipe : DWORD

Creates a new recipe in the given recipe definition

The method reads the current PLC values to the new recipe and saves them as a recipe file with the default name. The default name is <recipe>.<recipe definition>.<recipe extension>. At the end the new recipe would be stored in the default file.
