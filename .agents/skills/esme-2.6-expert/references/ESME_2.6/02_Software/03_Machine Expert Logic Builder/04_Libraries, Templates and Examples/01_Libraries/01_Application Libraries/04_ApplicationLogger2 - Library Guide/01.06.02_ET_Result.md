|  |  |
| --- | --- |

## Description

The enumeration ET\_Result specifies the POU-specific diagnostic and status messages.

NOTE: This data type is an alias of the data type APL\_Base.ET\_Result defined in the **ApplicationLoggerBase** library. The **ApplicationLoggerBase** library contains the basic implementations of the logger functions.
