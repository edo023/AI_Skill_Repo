# ConnectorState (ENUM)

TYPE ConnectorState :

Represents the different states of a connector, which can be set and get over [ConnectorFlagController.state](EpDUA0ToSNz8wQXwG7EAxhZ0DJo_state1.html#epdua0tosnz8wqxwg7eaxhz0djo_state1_connectorflagcontroller_state_prop). The effect of the different states are shown in the image in [ConnectorFlagController](lSMfQ0Hb1zMjuPbr0XbLf94JY9o_connectorflagcon1.html#lsmfq0hb1zmjupbr0xblf94jy9o_connectorflagcon1_connectorflagcontroller_fb).
