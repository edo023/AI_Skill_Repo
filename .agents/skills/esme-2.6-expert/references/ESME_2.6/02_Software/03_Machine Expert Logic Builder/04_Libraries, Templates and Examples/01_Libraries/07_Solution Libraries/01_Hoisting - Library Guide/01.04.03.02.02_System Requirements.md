# System Requirements

System Requirements

The targeted controllers for Hoisting applications are:

oModicon M241 Logic Controller

oModicon M251 Logic Controller

oModicon M258 Logic Controller

oModicon M262 Logic/Motion Controller

oModicon LMC058 Motion Controller

NOTE: Modicon LMC058 Motion Controller can be used with 2 [CANopen](../glossary/glossary.htm#XREF_D_SE_0024697_650) fieldbusses. In this case, the [CANmotion](../glossary/glossary.htm#XREF_D_SE_0024697_649) bus is configured as a CANopen fieldbus.
