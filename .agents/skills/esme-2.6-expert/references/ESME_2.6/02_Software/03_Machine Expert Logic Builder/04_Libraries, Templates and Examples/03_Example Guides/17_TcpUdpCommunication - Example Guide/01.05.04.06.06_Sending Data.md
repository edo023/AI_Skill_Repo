# Sending Data

Sending Data

Whenever valid data is received from the [TCP](../glossary/glossary.htm#XREF_D_SE_0055638_20) [client](../glossary/glossary.htm#XREF_D_SE_0055638_5), a response is sent.

The response is created depending on the received data. If the text message GetTimestamp has been received, the response data contains the time stamp of the system ([RTC](../glossary/glossary.htm#XREF_D_SE_0024697_729) of the controller) as text message. If a text message other than GetTimestamp has been received, the response data contains the text message UnknownCommand.
