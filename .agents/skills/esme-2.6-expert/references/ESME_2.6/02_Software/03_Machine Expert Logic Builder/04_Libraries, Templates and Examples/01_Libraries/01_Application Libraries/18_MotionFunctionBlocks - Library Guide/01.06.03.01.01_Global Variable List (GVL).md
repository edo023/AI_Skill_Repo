# Global Variable List (GVL)

## Overview

|  |  |
| --- | --- |
| Type: | Global variables |
| Available as of: | V1.3.1.0 |

## Description

The global variables list contains the global variables of the library.

## Global Variables

| Variable | Data Type | Default value | Description |
| --- | --- | --- | --- |
| `Gc_xOnlineChangeAllowed` | BOOL | FALSE | If TRUE, the online change detection is successfully initialized. While the POU is busy, only execute an online change when Gc\_xOnlineChangeAllowed = TRUE. |

| NOTICE | |
| --- | --- |
|  | Invalid reference of the digital input signal  Do not perform an online change when the variable GVL.G\_xOnlineChangeAllowed is FALSE and the homing mode HomingToDigitalInput is in progress.  Failure to follow these instructions can result in equipment damage. |
