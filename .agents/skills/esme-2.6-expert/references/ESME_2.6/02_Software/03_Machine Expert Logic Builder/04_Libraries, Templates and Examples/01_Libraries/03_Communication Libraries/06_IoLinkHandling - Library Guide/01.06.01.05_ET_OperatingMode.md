# ET\_OperatingMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_OperatingMode contains the operating modes of an IO-Link channel.

NOTE: After writing the Operating Mode, verify the channel status of the IO-Link device. This parameter is available in the device editor of the IO-Link master.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Inactive | 0 | Operating mode is Inactive. |
| DigitalOutput | 1 | Operating mode DigitalOutput. |
| DigitalInput | 2 | Operating mode DigitalInput. |
| Operate | 3 | Operating mode Operate. |
