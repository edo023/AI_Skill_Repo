# Preconditions

Preconditions

oMake sure to have the correct syntax and the correct extension (some Windows tools add the extension \*.txt)

oThe values within a parameter file are integers (no enumeration, no floating point values). During download, the drive adapts them to the correct format. Consider this if you change values within a parameter file manually.

There is no consistency or limit verification of the read data. Each value is inserted into the Parameter File and thereafter written to the drive without a verification.

NOTE: A new parameter upload may be required.

oMake sure that the IDNs in the Index File have read and write access. Furthermore, each IDN must be accessible via the Sercos fieldbus.

oUp- or Downloading of parameters is only possible if the Sercos bus is in phase 4 and the drive is not enabled (AxisState = 1).

oSome drive parameters can only be written if one (or more) other parameter was previously restored. Consider this when creating the Index File.

oExample for ATV340S:

Parameter CD1: This parameter can only be accessed if parameter CHCF is set to Separate or I/O profile.

oDepending on the number of IDNs in the Index File, you must adapt GPL.Gc\_uiMaxNumber­OfRowsRead and/or GPL.Gc\_uiMaxExecutionTime.
