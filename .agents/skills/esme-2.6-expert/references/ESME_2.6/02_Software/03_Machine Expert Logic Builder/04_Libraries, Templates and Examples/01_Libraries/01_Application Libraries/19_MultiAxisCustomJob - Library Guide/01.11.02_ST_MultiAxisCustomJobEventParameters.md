# ST\_MultiAxisCustomJobEventParameters

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | 1.0.0.0 |

## Description

The structure ST\_MultiAxisCustomJobEventParameters is used to pass parameters during event-based communication in a multi-axis group. It acts as an alias for MOIN.ST\_MultiAxisGroupEventParameters, providing compatibility with the event-handling mechanisms defined in the MotionInterface library.

In the context of the EventCallback method, this structure contains the information for the system to identify which group or axis event is being processed.

For details on the interaction of function blocks, interfaces and methods of a multi-axis custom job, refer to [Multi-Axis Custom Jobs](Multi-AxisCustomJobs-572EC46B.html).
