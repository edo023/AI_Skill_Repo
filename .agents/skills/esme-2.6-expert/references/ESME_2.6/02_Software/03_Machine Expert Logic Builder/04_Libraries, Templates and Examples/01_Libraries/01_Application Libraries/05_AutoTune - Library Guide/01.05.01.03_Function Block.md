# Function Block

Function Block

The functions of the function block are explained briefly in this chapter.

The AUT.G\_xAutoTuneEnable input is the only IEC interface of the POU.

The function block is the link for data transfer between the drive and the ready-made visualization. The parameterization of the function block is performed only via the visualization.

The visualization comprises the following 5 visualization windows:

oSelectAxis

oMainConfiguration

oAdvancedConfiguration

oResults

oTest

See also chapter [Visualization](../Visualization/Visualization-1.htm#XREF_D_SE_0091184_1).
