| Available as of: | V1.0.1.0 |

## Description

The method FB\_AxisMotion.Cycle indicates the detected error.

NOTE: The method FB\_AxisMotion.Cycle must be called cyclically.

There are two cases:

* A command method is called and an error is detected. This error is indicated on the command method and on the outputs of the FB\_AxisMotion.Cycle method.
* During a running motion command, an error is detected and indicated on the outputs of the cycle methods.

## Interface

| Output | Data type | Description |
