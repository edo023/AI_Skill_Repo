# EncNumber

EncNumber

|  |  |
| --- | --- |
| Enumeration name: | EncNumber |
| Enumeration value: | 50 |
| Description: | The number of encoders is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | A sum encoder was used as master. | The number of encoders must be greater or equal 1 and less or equal 8. |
