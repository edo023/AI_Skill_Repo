DefinitionCoordinateSystem

|  |  |
| --- | --- |
| Enumeration name: | DefinitionCoordinateSystem |
| Enumeration value: | 95 |
| Description: | The definition of the coordinate system is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The vectors i\_stV1, i\_stV2 and i\_stV3 lie in the same plane, they do not span a 3-dimensional coordinate system. | The vectors i\_stV1, i\_stV2 and i\_stV3 must be linearly independent, i.e. they must not lie in the same plane. |
