# Output Pin Description

## Output Pin Description

This table describes the output pins of the `Operation_Mode` function block:

| Output | Data Type | Description |
| --- | --- | --- |
| `q_xAut` | `BOOL` | TRUE: Automatic mode enabled  FALSE: Disabled |
| `q_xMan` | `BOOL` | TRUE: Manual mode enabled  FALSE: Disabled |
| `q_xLoc` | `BOOL` | TRUE: Local mode enabled  The operating mode is not changeable by the HMI and function blocks cannot be operated from the HMI.  FALSE: Disabled |
| `q_xHmiCtrl` | `BOOL` | TRUE: Operating mode is given by the HMI.  `q_xHmiCtrl` is overwritten by local mode.  FALSE: Disabled |
| `q_xErr` | `BOOL` | TRUE: Detected error is active.  FALSE: No detected error |
| `q_dwStat` | `DWORD` | Status bits to be displayed in the HMI  Range: 0...4294967295  Refer to the [status word description](D-SG-0026849.html#D-SG-0026849). |
