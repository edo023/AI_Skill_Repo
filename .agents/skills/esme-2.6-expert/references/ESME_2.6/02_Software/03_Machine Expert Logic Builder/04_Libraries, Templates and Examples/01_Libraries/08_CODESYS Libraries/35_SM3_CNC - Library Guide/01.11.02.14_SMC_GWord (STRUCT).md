# SMC\_GWord (STRUCT)

TYPE SMC\_GWord : STRUCT

This data type represents the data of one G-code word like ‘N20’ or ‘G1’. It replaces the older structure [SMC\_GCODE\_WORD](cAH0ifE6NckjZp5eF3Qpy-filyk_smc-gcode-word.html#cah0ife6nckjzp5ef3qpy_filyk_smc_gcode_word_smc_gcode_word_struct), which is kept only for compatibility.
