# FdDriverOpenP (FUN)

FUNCTION FdDriverOpenP : CAA.HANDLE

See [FdDriverOpenH](kh367MosQmNbzGw63h7hukNE_dU_fddriveropenh.html#kh367mosqmnbzgw63h7hukne_du_fddriveropenh_fddriveropenh_fun) for details. In contrast to [FdDriverOpenH](kh367MosQmNbzGw63h7hukNE_dU_fddriveropenh.html#kh367mosqmnbzgw63h7hukne_du_fddriveropenh_fddriveropenh_fun) no dynamic memory will be allocated. Memory specified by pMemory and szMemSize will be used. Use [FdDriverGetSize](kh367MosQmNbzGw63h7hukNE_dU_fddrivergetsize.html#kh367mosqmnbzgw63h7hukne_du_fddrivergetsize_fddrivergetsize_fun) to calculate the required memory size. Please note that required memory size can change with new runtime version.

**Note**

pMemory should be aligned to a 4 byte address on 32 bit systems and an 8 byte address on 64 bit systems. To ensure this alignment, you can declare the memory with ARRAY [0..n] of \_\_XWORD.

> Please keep in mind that the required memory size is different for 32 and 64-bit systems.
