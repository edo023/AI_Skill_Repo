## Task

Resetting the clamping station.

## Description

With the method CleanUp, you can delete the carriers from the clamping station and reset the state machine of the clamping station.

The return value CleanUp of type BOOL indicates TRUE if the clean-up has been done successfully.

NOTE: The method CleanUp is automatically called when the instance of the station function block FB\_ClampingStation is deleted.

## Inputs

The method has no inputs.

## Outputs

The method has no outputs.
