# Description

Description

The function is an extension of the arithmetic operator MOD.

oModulo Division of a real number is possible.

oSpecification of the interval lower limit is possible. (At MOD the interval lower limit is always 0.)

As a result, this function always provides the remainder of the division which is located within the interval [ i\_lrIntervalStart, (i\_lrIntervalStart + i\_lrIntervalLength) ].

Calculation:

Result = ( (i\_lrValue - i\_lrIntervalStart) MOD i\_lrIntervalLength) + i\_lrIntervalStart
