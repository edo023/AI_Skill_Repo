# Homing

Homing

The operation mode Homing is selected. The axis is homed via the mode selected in i\_etMode.

The operation mode is parameterized in the structures [AXM.ST\_Home](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077219_1) and [AXM.ST\_Main](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077223_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [PDL.FB\_Home](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087275_1) are being activated.

Basic monitoring systems are enabled:

oMonitoring the diagnostic messages of the axis

oMonitoring the hardware limit switches

oMonitoring the reduced velocity

Possible state commands

o[Start](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Homing of the axis is started.

o[Hold](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Homing of the axis is stopped.

o[DriveEnableOn](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> The position control of the axis is activated.

o[DriveEnableOff](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> The position control of the axis is disabled.

Possible operation mode commands

o[Stop](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [NoOp](#XREF_D_SE_0088804_3) takes place.

o[CamCs](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [MultiCam](#XREF_D_SE_0088804_6) takes place and a coldstart is preselected.

o[CamWs](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [MultiCam](#XREF_D_SE_0088804_6) takes place and a warmstart is preselected.

o[Manual](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [Manual](#XREF_D_SE_0088804_5) takes place.

o[Endless](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [Endless](#XREF_D_SE_0088804_7) takes place.

o[Positioning](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [Positioning](#XREF_D_SE_0088804_8) takes place.

o[BrakeRelease](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.AxisModule&topicID=D_SE_0077124_1) -> Change to operation mode [BrakeRelease](#XREF_D_SE_0088804_9) takes place.
