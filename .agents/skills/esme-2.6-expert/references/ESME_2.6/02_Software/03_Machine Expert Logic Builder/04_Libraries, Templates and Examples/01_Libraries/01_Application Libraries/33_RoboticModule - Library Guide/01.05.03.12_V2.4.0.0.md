## New Functions

* Linear Tracking System

  + IF\_Configuration.AddLinearTrackingSystem3(…)

    By using this method to configure a linear tracking system, it is possible to define the system ID.
* Additional Transformation TCP

  + IF\_ConfigurationAdvanced.AdditionalTransformationTCP(…)

    An additional user-defined transformation for the TCP can be configured.
* Resultant Acceleration Limitation Parameters

  + IF\_ConfigurationAdvanced.SetResAccLimitParameters(…)

    Additional configuration parameters for the functionality to limit the resultant acceleration of a robot space movement can be set.

    IF\_ConfigurationAdvanced.GetResAccLimitParameters(…)

    Additional configuration parameters for the functionality to limit the resultant acceleration of a robot space movement can be read.
