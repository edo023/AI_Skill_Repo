# Commissioning the HoistDutyRating Function Block

Commissioning the HoistDutyRating Function Block

| Step | Action |
| --- | --- |
| 1 | Instantiate the HoistDutyRating function block. |
| 2 | Enter the value of the nominal load (i\_rLoadNom) and the inputs coming from the field: drive run discrete signal (i\_xDrvRun) and actual load value (i\_rLoadAct). |
| 3 | Make sure that the 3 variables connected to the [input/output pins](Hoist_Duty_Rating_2-7.htm#XREF_D_SE_0092074_1) (iq\_arMeanEffectLoad, iq\_alrRunTime, iq\_stHoistDutyCtrl) of the function block are defined as either persistent or retained. |
| 4 | Verify the accuracy of the actual load value during commissioning and maintenance cycles. |
