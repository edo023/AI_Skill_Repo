# UABrowseDirection

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V2.0.0.0 |

## Description

The enumeration UABrowseDirection specifies the direction of references to return.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| UABD\_Forward | 0 | Select forward references. |
| UABD\_Inverse | 1 | Select inverse references. |
| UABD\_Both | 2 | Select forward and inverse references. |
