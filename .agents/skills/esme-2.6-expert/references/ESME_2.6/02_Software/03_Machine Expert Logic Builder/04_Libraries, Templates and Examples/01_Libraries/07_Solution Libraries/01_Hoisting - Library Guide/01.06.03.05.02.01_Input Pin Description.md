# Input Pin Description

Input Pin Description

| Input | Data type | Description |
| --- | --- | --- |
| i\_xEn | BOOL | TRUE: Enables the function block.  FALSE: Disables the function block.  Refer to detailed description below this table. |
| i\_astSyncGroup | ARRAY OF SyncGroupIn | Array of parameters and commands for synchronization of groups.  For detailed description, refer to [SyncGroupIn](Advanced_Position_Synchronization-11.htm#XREF_D_SE_0091263_1). |
| i\_astAxisIntf | ARRAY OF AxisIntfIn | Array of parameters and commands for the axis interface.  For detailed description, refer to [AxisIntfIn](Advanced_Position_Synchronization-9.htm#XREF_D_SE_0091268_1). |
| i\_xRst | BOOL | Resets detected alarms on rising edge, provided the cause of the alarm has been remedied.  TRUE: Active  FALSE: Inactive |
