# FeedbackMaxReached

FeedbackMaxReached

|  |  |
| --- | --- |
| Enumeration name: | FeedbackMaxReached |
| Enumeration value: | 72 |
| Description: | FeedbackMax reached its limit. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The function block is unable to control the bobbin with the parameters since the feedback value ST\_Auto.stFeedbackSignal.i\_lrMaxLevelZone was longer present than the allowed time span ST\_Auto.stParameter.stFeedbackSignal.i\_timMaxLevelTime. | Decrease ST\_Auto.stFeedbackSignal.i\_lrMaxLevelZone.  Increase ST\_Auto.stParameter.stFeedbackSignal.i\_timMaxLevelTime. |
