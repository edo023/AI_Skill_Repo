# MaxNrOfProductsReached

MaxNrOfProductsReached

|  |  |
| --- | --- |
| Enumeration name: | MaxNrOfProductsReached |
| Enumeration value: | 90 |
| Description: | The maximum number of manageable products has been reached. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | There was an attempt to add more than the maximum number of products to the infeed distance. | At no time more than [SI.Gc\_udiMaxNumberOfProducts](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SmartInfeed&topicID=D_SE_0080374_1) products may be located on the infeed distance.  The global parameter [SI.Gc\_udiMaxNumberOfProducts](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SmartInfeed&topicID=D_SE_0080374_1) must be set to a value that is greater than the maximum number of products on the infeed distance. |
