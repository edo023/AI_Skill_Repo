# Enumeration Elements

Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| PI | 1 | The PI controller is used. |
| ThreeStep | 2 | The ThreeStep controller is used. |
| Torque | 3 | The torque controller is used.  Note: This controller mode requires a special parametrization of the drive. See "Using the unwinder TorqueMode". |
