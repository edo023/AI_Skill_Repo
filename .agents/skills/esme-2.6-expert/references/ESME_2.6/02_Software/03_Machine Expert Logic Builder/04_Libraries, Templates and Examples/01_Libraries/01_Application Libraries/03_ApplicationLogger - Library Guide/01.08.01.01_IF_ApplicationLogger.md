| --- | --- |

## Description

This interface extends the interface IF\_LoggerPoint. It represents the Application Logger in the project. This interface is used to encapsulate the functionalities of the Application Logger. The Application Logger is also a logger point that represents the root of the logger points tree. The name of this logger point is Application Logger, the type is IF\_ApplicationLogger and the default logger level is given by the global parameter Gc\_etDefaultLogLevel. The ID of this logger point is 1.
