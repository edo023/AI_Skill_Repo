# GetAlarmIDName (FUN)

FUNCTION GetAlarmIDName : STRING

This function gets the ID the user entered in the AlarmGroup for a given Alarm If the alarm can not be found an empty string is returned.

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | GetAlarmIDName | STRING |  | | Input | uiAlarmId | [ALARM\_ID](ALARM_ID.html#alarm_id_alarm_id_alias) | The (internal) AlarmID that can f.i. be retrieved via the selected variable at the alarm table | | sAlarmGroup | STRING | The name of the AlarmGroup that can f.i. be retrieved via the selected variable at the alarm table | |
