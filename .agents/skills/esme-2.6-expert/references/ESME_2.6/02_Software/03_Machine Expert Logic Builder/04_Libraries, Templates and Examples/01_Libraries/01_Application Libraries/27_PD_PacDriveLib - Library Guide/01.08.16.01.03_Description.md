# Description

Description

The function implements a PID controller. A positive signal at i\_xStart starts the function block. The input signal i\_lrInput is calculated from the difference i\_lrInput := (reference value - actual value). The parameters KP, TI and TD must be chosen according to the controlled power stage. The output setpoint is calculated according to the following formula from the current input parameters and the previous progression.

Output := i\_lrKP \* ( i\_lrInput + d(i\_lrInput) \* i\_timTD + I(i\_lrInput) / i\_timTI )

Where d() is the derivative with respect to time and i() is the integral over time.
