# XstartGreaterXend

XstartGreaterXend

|  |  |
| --- | --- |
| Enumeration name: | XstartGreaterXend |
| Enumeration value: | 65 |
| Description: | XStart is greater than XEnd. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The assigned value of the stCam.lrXStart variable is too high. | The value of the stCam.lrXStart variable must be less than the value of the stCam.lrXEnd variable. |
