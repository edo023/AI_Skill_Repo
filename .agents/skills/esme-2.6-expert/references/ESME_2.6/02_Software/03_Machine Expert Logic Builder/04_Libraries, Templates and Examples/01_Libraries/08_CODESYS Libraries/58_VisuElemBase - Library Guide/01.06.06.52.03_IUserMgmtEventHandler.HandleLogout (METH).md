# IUserMgmtEventHandler.HandleLogout (METH)

METHOD HandleLogout : BOOL

This method will be called when a logout from the visu user management is performed

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | HandleLogout | BOOL | TRUE - When the handler has handled this event and it should not be handled by someone else FALSE - When the event is not handled by this handler | | Input | pClient | POINTER TO VisuStructClientData | A pointer to the client structure were the event occurred. The name of the user, that logged out, is stored in pClient^.GlobalData.CurrentUserName | |
