# SMC\_NC\_SourcePosition (STRUCT)

TYPE SMC\_NC\_SourcePosition : STRUCT

Represents a source position consisting of a line number, a column number (both zero-based), and a length.

| InOut: | | Name | Type | Initial | Comment | | --- | --- | --- | --- | | diLine | DINT | -1 | The zero-based line number. -1 if unknown. | | diColumn | DINT | -1 | The zero-based column number. -1 if unknown. | | diLength | DINT | -1 | The length. -1 if unknown. | |
