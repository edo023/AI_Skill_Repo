# CreateMessage (FUN)

FUNCTION CreateMessage : CAA.HANDLE

This function allocates a new message from the data pool of the driver and makes it available via its handle hMessage. In addition, the function sets the CAN ID, data length and RTR flag of the message. Such a message must either be sent with [Write](UzaP1MMIvsGx5wgQfvzFSZKw87g_write.html#uzap1mmivsgx5wgqfvzfszkw87g_write_write_fun) or released with [FreeMessage](UzaP1MMIvsGx5wgQfvzFSZKw87g_freemessage.html#uzap1mmivsgx5wgqfvzfszkw87g_freemessage_freemessage_fun). It can also be duplicated by use of [CloneMessage](UzaP1MMIvsGx5wgQfvzFSZKw87g_clonemessage.html#uzap1mmivsgx5wgqfvzfszkw87g_clonemessage_clonemessage_fun).

**Example**

See [Write](UzaP1MMIvsGx5wgQfvzFSZKw87g_write.html#uzap1mmivsgx5wgqfvzfszkw87g_write_write_fun).
