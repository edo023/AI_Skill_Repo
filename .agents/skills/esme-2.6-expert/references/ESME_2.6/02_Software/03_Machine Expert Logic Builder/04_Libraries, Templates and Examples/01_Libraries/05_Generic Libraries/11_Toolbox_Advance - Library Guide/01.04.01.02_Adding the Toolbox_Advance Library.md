# Adding the Toolbox_Advance Library

Adding the Toolbox\_Advance Library

To have access to the Toolbox\_Advance functions, it is necessary to manually add the library:

| Step | Action |
| --- | --- |
| 1 | Double-click the Library Manager  node of the controller Application node in the Devices window. |
| 2 | Click Add Library.  Result: The Add Library dialog box opens. |
| 3 | Browse to Util, visible when the Company filter is set to Schneider Electric or All Company.  Result: The Util libraries list is displayed. |
| 4 | Select the Toolbox\_Advance library and press OK.  Result: The library is added to the Library Manager list. |
