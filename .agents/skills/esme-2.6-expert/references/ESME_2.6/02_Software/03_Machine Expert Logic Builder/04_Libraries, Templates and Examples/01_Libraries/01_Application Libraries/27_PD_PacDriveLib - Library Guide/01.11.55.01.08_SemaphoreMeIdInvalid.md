# SemaphoreMeIdInvalid

SemaphoreMeIdInvalid

|  |  |
| --- | --- |
| Enumeration name: | SemaphoreMeIdInvalid |
| Enumeration value: | 272 |
| Description: | The Id of the SemaphoreMe is invalid. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The semaphore at the iq\_stSemMe input is already in use. | Use a different structure to create a second semaphore.  Delete the semaphores with [FC\_SemaphoreMeDelete](Functions_K_to_Z-56.htm#XREF_D_SE_0087647_1) before creating the new semaphores. |
