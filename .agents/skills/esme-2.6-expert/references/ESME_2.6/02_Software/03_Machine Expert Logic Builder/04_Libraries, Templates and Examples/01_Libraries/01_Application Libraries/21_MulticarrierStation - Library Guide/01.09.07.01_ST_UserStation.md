# ST\_UserStation

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.3.0.0 |
| Inherits from: | - |

## Description

The structure ST\_UserStation is used as input/output of the visualization frame [FR\_UserStation](FRUserStation-E1CD17A3.html#FRUserStation-E1CD17A3).

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| diTimer | DINT | The variable diTimer is used for auto-triggering the station. It contains information about the time that has elapsed since the station is in a state for enabling the triggering of a process. |
| diTimeState | DINT | The variable diTimeState is used for auto-triggering the station. It contains information about the trigger state and determines which action is executed, based on the timing defined in diTimer. |
| xAutoTrigger | BOOL | If xAutoTrigger is set to TRUE, the station automatically triggers the next process when the specified process time has elapsed. If xAutoTrigger is set to FALSE, you must activate the station manually. |
| xStart | BOOL | If xStart is set to TRUE, the station is triggered. When the station has been triggered, xStart is set to FALSE. |
| xStationReadyForJob | BOOL | Indicates TRUE if the station can be triggered. Indicates FALSE if the station cannot be triggered. |
