# MaxTorqueRange

MaxTorqueRange

|  |  |
| --- | --- |
| Enumeration name: | MaxTorqueRange |
| Enumeration value: | 85 |
| Description: | MaxTorque is outside the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The operation mode [ET\_OpMode](../Enumerations/Enumerations-5.htm#XREF_D_SE_0088734_1).Homing was selected with the [ST\_Home](../Structures/Structures-4.htm#XREF_D_SE_0088756_1).i\_etMode mode equal to  [PDL.ET\_HomeMode](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).PosDirectionMaxTorque or  [PDL.ET\_HomeMode](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087221_1).NegDirectionMaxTorque.  An invalid value was transferred to the [ST\_Home](../Structures/Structures-4.htm#XREF_D_SE_0088756_1)..stTorque.i\_lrMaxTorque parameter. | The following must apply: 0.0 < ([ST\_Home](../Structures/Structures-4.htm#XREF_D_SE_0088756_1)..stTorque.i\_lrMaxTorque / lrTorqueConstant \* 10000000.0 / lrMaxDrivePeakCurrent / lrLimCurrent) < 100.0  with lrLimCurrent := lrUserDrivePeakCurrent \* 100.0 / lrMaxDrivePeakCurrent  where lrTorqueConstant, lrMaxDrivePeakCurrent and lrUserDrivePeakCurrent are parameters of the axis. |
