# VisuFbDateTimeNamesLocalizer (FB)

FUNCTION\_BLOCK VisuFbDateTimeNamesLocalizer IMPLEMENTS ILocalizedDateTimeNames

This function block implements functionality, used for Stu.DateTimeFormatter With this function block, the DateTimeFormatter is able to format values like ddd, dddd, MMM and MMMM. (Formats, that can be localized, using textlist system)
