## Overview

|  |  |
| --- | --- |

| Available as of: | V1.0.0.0 |

## Task

Interface with status values for the interface [IF\_Jogging](IF_Jogging-5B8373DE.html#IF_Jogging-5B8373DE).

## Description

The feedback interface provides status values for the manual movement of the carrier in jogging mode.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| xJogActivated | BOOL | Read | The jogging mode is started and the carrier is ready for jogging with the motion properties xJogSelect, xJogForward, and xJogBackward (see  [IF\_Jogging](IF_Jogging-GeneralInformation-57F1DE3C.html#IF_Jogging-GeneralInformation-57F1DE3C)). |
| xJobActive | BOOL | Read | Indicates TRUE if a motion job is active. |
| xJogBackwardActive | BOOL | Read | Indicates TRUE if the carrier moves backward. |
| xJogForwardActive | BOOL | Read | Indicates TRUE if the carrier moves forward. |
