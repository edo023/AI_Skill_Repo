# FC\_WriteDataResponse - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.2.0 |

## Task

This function returns the response to the WriteData service from a Sercos device. This service writes a parameter (IDN).

## Functional Description

This function returns the response to the WriteData service from a Sercos device to which a request had been sent via FC\_WriteDataRequest.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_udiTransactionID | UDINT | Transaction ID of the corresponding request |
| i\_diTimeoutMs | DINT | Timeout in milliseconds for waiting for the response |

## Return Value

| Data type | Description |
| --- | --- |
| ET\_Result | Refer to [ET\_Result](ET_Result-GeneralInformation-08184BB1.html). |
