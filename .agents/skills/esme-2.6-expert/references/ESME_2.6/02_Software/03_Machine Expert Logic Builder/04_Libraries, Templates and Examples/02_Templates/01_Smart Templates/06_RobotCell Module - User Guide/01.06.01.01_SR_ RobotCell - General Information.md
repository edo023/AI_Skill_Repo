## Functional Description

SR\_<RobotCell> is used to configure a robot cell.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xDoNotUseDefaultCmdTables | BOOL | FALSE: The default command table is used.  TRUE: You must write your own command table.  For more information, refer to [Using i\_xDoNotUseDefaultCmdTables](UsingI_xDoNotUseDefaultCmdTables-68F39A2B.html#UsingI_xDoNotUseDefaultCmdTables-68F39A2B). |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_astConveyorModuleInterface | ARRAY [SERT.ET\_SystemEntity.Conveyor1...SERT.ET\_SystemEntity.ConveyorOffset+SERT.Gc\_udiMaxNumberOfConveyors] OF AXM.ST\_ModuleInterface (refer to EcoStruxure Machine Expert, PD\_AxisModule, Library Guide) | Array of conveyor interface of all conveyors configured in the RobotCell. |
| iq\_astRobotModuleInterface | ARRAY [SERT.ET\_SystemEntity.Robot1...SERT.ET\_SystemEntity.RobotOffset+SERT.Gc\_udiMaxNumberOfRobots] OF RM.ST\_ModuleInterface (refer to EcoStruxure Machine Expert, RoboticModule, Library Guide) | Array of RoboticModule interface of all robots configured in the RobotCell. |
| iq\_stStandardModuleItf | TPL.ST\_StandardModuleInterface (refer to EcoStruxure Machine Expert, PD\_Template, Library Guide) | Standard module interface |
| iq\_stExceptionList | TPL.ST\_ExceptionList (refer to EcoStruxure Machine Expert, PD\_Template, Library Guide) | Exception list |
| iq\_stLogDataList | TPL.ST\_LogDataList (refer to EcoStruxure Machine Expert, PD\_Template, Library Guide) | Log data list |

| Output | Data type | Description |
| --- | --- | --- |
| q\_aifTargetsHandler | ARRAY [ROB.ET\_CoordinateSystem.Tracking1 ...ROB.ET\_CoordinateSystem.Tracking30] OF SERT.IF\_TargetsHandler | Read the interface for an array of a TargetsHandler. For every target handler that is added to the robot cell in the Targets Handler tab, the interface to this targets handler is set and can be used, for example, to read and set the data. |
| q\_ifEntitiesHandler | SERT.IF\_EntitiesHandler | Interface of the EntitiesHandler in which the RobotCell adds the entities.  NOTE: If external EntitiesHandler is used the interface is passed through this output. |
