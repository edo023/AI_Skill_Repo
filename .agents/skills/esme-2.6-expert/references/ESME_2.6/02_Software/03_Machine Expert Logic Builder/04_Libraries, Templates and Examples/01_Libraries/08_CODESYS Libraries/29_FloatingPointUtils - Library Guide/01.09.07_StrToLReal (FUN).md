# StrToLReal (FUN)

FUNCTION StrToLReal : LREAL

Converts a string to an LREAL value, taking into account the special values ‘NaN’, ‘+Inf’, and ‘-Inf’. Based on STRING\_TO\_LREAL.

**Note**

This function also handles the case LREAL = REAL.
