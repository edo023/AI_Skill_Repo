## New Features

* **IF\_Configuration.Delta4Ax**

  Method to configure a four-axis Delta robot.
* **IF\_ConfigurationAdvanced.GetExternalPositionSourceAccLimit()**

  Method for reading the acceleration limit for a specific component.
* **IF\_ConfigurationAdvanced.SetExternalPositionSourceAccLimit()**

  Method for setting a limit for the acceleration of the external position source.
* **xEnableAxisJoggingDelta4Ax**

  Property to enable the manual mode of robot main axes of a Delta4Ax robot.
