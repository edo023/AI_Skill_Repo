# ET\_OperatingMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_OperatingMode is used to define the [operating mode](OperatingModes-44B9543B.html).

## Enumeration Elements

| Name | Value (USINT) | Description |
| --- | --- | --- |
| Continuous | 0 | The operating mode Continuous is selected for processing products with a constant length. Also refer to:   * [Operating Mode Continuous for FlyingShear](OpMContinuous-44B9D396.html) * [Operating Mode Continuous for RotaryKnife](OpModeContinuousRK-E76E49E1.html) |
| ContinuousWithCorrection | 1 | The operating mode ContinuousWithCorrection is selected for processing products with small differences in length. It includes an automatic correction of the product length, calculated from a position capture. Also refer to:   * [Operating Mode ContinuousWithCorrection for FlyingShear](OpMContWCorrect-44BA1BDD.html) * [Operating Mode ContinuousWithCorrection for RotaryKnife](OpMContWCorrRK-E771413E.html) |
| CutOnTouchProbe | 2 | The operating mode CutOnTouchProbe is selected for processing products with variable length and format, calculated from a position capture. Also refer to:   * [Operating Mode CutOnTouchProbe for FlyingShear](OpMCutOnTouchProbe-44BA9270.html) * [Operating Mode CutOnTouchProbe for RotaryKnife](OpMCutOnTPRK-E773EE16.html) |
