# MultiCam

MultiCam

The operating mode MultiCam is selected. The axis is appended as slave to a master via cams.

The operating mode is parameterized in the structures [ST\_MultiCam](../Structures/Structures-10.htm#XREF_D_SE_0077229_1) and [ST\_Main](../Structures/Structures-7.htm#XREF_D_SE_0077223_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [PDL.FB\_MultiCam](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087313_1) are being activated.

If the axis shall be activated in an ILS network, the pstIlsData parameter of the ST\_MultiCam structure must be linked to the (global) stIlsData structure, to which also the pstIlsData parameter of the ST\_EndlessFeed structure of the master module was linked. Subsequently, the technology POU [PDL.FB\_PreCalcMultiCam](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087330_1) is called up first.

Basic monitoring systems are enabled:

oMonitoring the diagnostic messages of the axis

oMonitoring of the hardware limit switches

oMonitoring of the reduced velocity

Possible state commands

o[Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> If the operating mode was entered via the [CamCs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command, a cold start is executed.

 If the operating mode was entered via the [CamWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) command, a warm start is preselected. Subsequently, a WsStart command is waited for.

o[WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> After the warm start has been prepared via the commands [CamWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) - [Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1), the warm start motion is started via [WsStart](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1).

o[Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The start command is canceled.

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The position control of the axis is activated.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> The position control of the axis is being disabled.

Possible operating mode commands

o[Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [NoOp](#XREF_D_SE_0077326_3) takes place.

o[Homing](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Homing](#XREF_D_SE_0077326_4) takes place.

o[Manual](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Manual](#XREF_D_SE_0077326_5) takes place.

o[Endless](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Endless](#XREF_D_SE_0077326_7) takes place.

o[EndlessIls](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [EndlessIls](#XREF_D_SE_0077326_8) takes place.

o[Positioning](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Positioning](#XREF_D_SE_0077326_9) takes place.

o[BrakeRelease](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [BrakeRelease](#XREF_D_SE_0077326_10) takes place.

o[AdditionalCs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Additional](#XREF_D_SE_0077326_11) takes place and a cold start is preselected.

o[AdditionalWs](../Enumerations/Enumerations-2.htm#XREF_D_SE_0077124_1) -> Change to operating mode [Additional](#XREF_D_SE_0077326_11) takes place and a warm start is preselected.
