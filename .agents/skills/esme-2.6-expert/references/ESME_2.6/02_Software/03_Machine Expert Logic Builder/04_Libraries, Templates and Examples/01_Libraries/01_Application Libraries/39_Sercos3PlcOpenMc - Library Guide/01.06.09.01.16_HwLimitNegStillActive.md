# HwLimitNegStillActive

HwLimitNegStillActive

|  |  |
| --- | --- |
| Enumeration name: | HwLimitNegStillActive |
| Enumeration value: | 56 |
| Description: | HwLimitNeg is still active. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The negative limit switch i\_xHwLimitNeg is still active. However, a Touchprobe signal has been detected or the maximum travel distance i\_lrMaxTravel has been reached. | Verify the signal at the input i\_xHwLimitNeg.  Verify the value for the maximum travel distance at the input i\_lrMaxTravel.  Ensure that the sensor of the negative limit switch is not dirty. |
