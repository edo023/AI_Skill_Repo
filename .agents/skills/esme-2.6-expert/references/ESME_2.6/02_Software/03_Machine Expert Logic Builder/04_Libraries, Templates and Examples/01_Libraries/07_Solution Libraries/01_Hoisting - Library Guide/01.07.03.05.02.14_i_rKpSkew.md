# i_rKpSkew

i\_rKpSkew

The parameter contains the proportional gain of the skew controller. Optimal value depends on the crane mechanics. Set this value first before i\_rKpDrft. Setting this input to 0 disables both skew and drift correction.

For more information on configuration of controller gains, check the [Commissioning Procedure](Anticrab_2-13.htm#XREF_D_SE_0033860_1) section.
