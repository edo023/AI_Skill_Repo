# q_wAlrmId

q\_wAlrmId

This output identifies the detected alarm.

| Alarm bit | Description |
| --- | --- |
| bit 0 | Nominal motor speed of bridge travel < 1 RPM  i\_stOarParam.iBridgeSpdNom < 1 and  i\_stOarParam.xBridgeAsw = FALSE |
| bit 1 | Linear speed of bridge travel ≤ 0  i\_stOarParam.rBridgeSpdLin ≤ 0.0 and  i\_stOarParam.xBridgeAsw = FALSE |
| bit 2 | Deceleration ramp time of the bridge travel motor = 0  i\_stOarParam.wBridgeDec = 0 and  i\_stOarParam.xBridgeAsw = FALSE |
| bit 3 | Nominal motor speed of trolley movement < 1 RPM  i\_stOarParam.iTrolSpdNom < 1 and  i\_stOarParam.xTrolAsw = FALSE. |
| bit 4 | Linear speed of trolley movement ≤ 0  i\_stOarParam.rTrolSpdLin ≤ 0.0 and  i\_stOarParam.xTrolAsw = FALSE |
| bit 5 | Deceleration ramp time of the trolley motor = 0  i\_stOarParam.wTrolDec = 0 and  i\_stOarParam.xTrolAsw = FALSE |
| bit 6 | Nominal motor speed of hoist movement < 1 RPM  i\_stOarParam.iHoistSpdNom < 1 |
| bit 7 | Linear speed of hoist movement ≤ 0  i\_stOarParam.rHoistSpdLin ≤ 0.0 |
| bit 8 | Deceleration ramp time of the hoist motor = 0  i\_stOarParam.wHoistDec = 0 |
| bit 9 | Modification of position of trolley axis and its motor speed have opposite signs. |
| bit 10 | Inconsistency between modification of position of the trolley axis and its motor speed detected. Verify the position measurement and actual motor speed values. |
| bit 11 | Inconsistency between modification of position of the trolley axis and its motor speed detected. Either the modification of position of the trolley axis or the actual speed of the trolley motor was zero while the other value changed. |
| bit 12 | Modification of position of bridge axis and its motor speed have opposite signs. |
| bit 13 | Inconsistency between modification of position of the bridge axis and its motor speed detected. Verify the position measurement and actual motor speed values. |
| bit 14 | Inconsistency between modification of position of the bridge axis and its motor speed detected. Either the modification of position of the bridge axis or the actual speed of the bridge motor was zero while the other value changed. |
