# Examples

Examples

Example with Single-Turn-Encoder (Encoder period standardized to 0 ... 360° standardized)

| i\_lrHomePos | i\_lrAbsEncOffset | i\_lrUserPeriod | Axis position according to FB\_HomeAbs or FB\_HomeSetPos | Run range |
| --- | --- | --- | --- | --- |
| 50 | 1 | 0 | 50 | -1 ... 359 (Delta 360) |
| 1000 | 100 | 0 | -80 | -100 ... 260 (Delta 360) |
| 0 | 100 | 0 | 0 | -100 ... 260 (Delta 360) |
| -1000 | 100 | 0 | 80 | -100 ... 260 (Delta 360) |
| 1000 | 0 | 360 | 280 | 0 ... 360 |
| -1000 | 0 | 360 | 80 | 0 ... 360 |
| 1 | 0 | 36 | 1 | 0 ... 36 |
| 35 | 0 | 36 | 35 | 0 ... 36 |
| 54 | 0 | 36 | 18 | 0 ... 36 |
| -54 | 0 | 36 | 18 | 0 ... 36 |
| -55 | 0 | 36 | 17 | 0 ... 36 |

If in the example above it shall be possible to reconstruct outside the encoder period (e.g. travel range -1000 ... -640), the [FB\_WriteDriveEncoder](../Function_Blocks_R_to_Z/Function_Blocks_R_to_Z-38.htm#XREF_D_SE_0087383_1) POU can be used alternatively in connection with [FC\_ReadDriveEncoder](../Functions_K_to_Z/Functions_K_to_Z-41.htm#XREF_D_SE_0087617_1).
