# UnpackWord (FB)

FUNCTION\_BLOCK UnpackWord

Function block unpacks “wValue” and returns the 2 bytes contained “byLowByte” and “byHighByte” and the 16 bits “xBit0” .. “xBit15”.

Prefix: “memuw”.

Example:

```
wValue = 16#0102 = 2#00000001 00000010
->
byLowByte = 16#02
byHighByte = 16#01
xBit1, xBit8 = TRUE
remain bits = FALSE
```
