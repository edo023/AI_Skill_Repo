# SMC\_AxisSettingsFromAxes (FB)

FUNCTION\_BLOCK SMC\_AxisSettingsFromAxes

Reads the axis settings from the given axes and stores them in the output AxisSettings. Note: not all inputs must be connected. For example, if the kinematics has 3 axes, then only input Axis0, Axis1, and Axis2 have to be connected. Intended for use with [SMC\_Trafo\_Wrapper](yu2A8Eb0pHPjsoioWnisaw2o8yk_smc-trafo-wrappe.html#yu2a8eb0phpjsoiownisaw2o8yk_smc_trafo_wrappe_smc_trafo_wrapper_fb) and [SMC\_TrafoF\_Wrapper](yu2A8Eb0pHPjsoioWnisaw2o8yk_smc-trafof-wrapp.html#yu2a8eb0phpjsoiownisaw2o8yk_smc_trafof_wrapp_smc_trafof_wrapper_fb).
