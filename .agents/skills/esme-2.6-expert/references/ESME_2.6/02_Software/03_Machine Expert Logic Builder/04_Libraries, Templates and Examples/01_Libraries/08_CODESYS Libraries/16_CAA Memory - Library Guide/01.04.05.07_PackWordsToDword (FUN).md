# PackWordsToDword (FUN)

FUNCTION PackWordsToDword : DWORD

Function packs low and high WORD into a DWORD.

Example:

```
MEM.PackWordsToDword(16#0102, 16#0304) = 16#01020304
```

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Return | PackWordsToDword | DWORD | function returns the DWORD defined by the WORD inputs | | Input | wHighWord | WORD | high word of the Dword | | wLowWord | WORD | low word of the Dword | |
