# SetSegment (FUN)

FUNCTION SetSegment : [IString](RcUUcqZ0iUdpu4tpDwADYSBDZbA_istring.html#rcuucqz0iudpu4tpdwadysbdzba_istring_istring_itf)

Initializes the passed [IString](RcUUcqZ0iUdpu4tpDwADYSBDZbA_istring.html#rcuucqz0iudpu4tpdwadysbdzba_istring_istring_itf) instance and uses the specified memory area for this purpose

Example for a 80 byte IString instance:

```
abyMemory : ARRAY[0..79) OF BYTE;
   myString : _UTF8String;
itfString : IString := SetSegment(myString, ADR(abyMemory), SIZEOF(abyMemory), 0);
```
