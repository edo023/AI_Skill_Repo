## Task

Sets a diagnostic message inside the listener.

## Description

The method SetDiag is used to set a diagnostic message inside the listener. If the listener is registered to a diagnostic handler, the method is called by the diagnostic handler as soon as a diagnostic message is set at the diagnostic handler.

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Returns TRUE if a diagnostic message is successfully set. |

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stDiagMsg | ST\_DiagMessageListener | Specifies the diagnostic message to set inside the listener. |
| i\_uiReactionLevel | UINT | Specifies the reaction level of the diagnostic message to set inside the listener. |

| Output | Data type | Description |
| --- | --- | --- |

| q\_etResult | ET\_Result | Provides diagnostic and status information as a numeric value.  If q\_xError  = FALSE, q\_etResult provides status information. If q\_xError  = TRUE, q\_etResult provides diagnostic/error information. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text value. |
