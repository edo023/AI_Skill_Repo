# Automatic

Automatic

The operation mode Automatic is selected. The axes are moved in the infeed mode according to the algorithm.

The operation mode is parameterized in the structures [ST\_Automatic](../Structures/Structures-2.htm#XREF_D_SE_0080847_1) and [ST\_Main](../Structures/Structures-4.htm#XREF_D_SE_0080851_1).

The technology POUs [PDL.FB\_DriveEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.PacDriveLib&topicID=D_SE_0087260_1) and [SI.FB\_Infeed](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SmartInfeed&topicID=D_SE_0080279_1) are being activated.

If [ST\_Automatic](../../../../../../api/crossBook?lang=en-US&virtualBookName=Structures/Structures-2.htm#XREF_D_SE_0080847_1).i\_xEnableFeedForw is set to TRUE, the technology POU [SI.FB\_MasterVel­FeedForward](../../../PD.Lib.SmartInfeed&topicID=D_SE_0080286_1) is additionally activated.

If [ST\_Simulation](../../../../../../api/crossBook?lang=en-US&virtualBookName=Structures/Structures-7.htm#XREF_D_SE_0080857_1).i\_xEnableSimulaiton is set to TRUE, the technology POU [SI.FB\_InfeedSim­ulation](../../../PD.Lib.SmartInfeed&topicID=D_SE_0080283_1) is additionally activated.

The diagnostic messages of all managed slave axes are monitored.

Possible state commands

o[Start](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Infeed is started.

o[Hold](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The infeed is stopped.

o[DriveEnableOn](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is enabled.

o[DriveEnableOff](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> The position control of all managed slave axes is disabled.

Possible operation mode commands

o[Stop](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [NoOp](#XREF_D_SE_0080863_4) takes place.

o[Manual](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [Manual](#XREF_D_SE_0080863_5) takes place.

o[BrakeRelease](../Enumerations/Enumerations-2.htm#XREF_D_SE_0080815_1) -> Change to operation mode [BrakeRelease](#XREF_D_SE_0080863_7) takes place.
