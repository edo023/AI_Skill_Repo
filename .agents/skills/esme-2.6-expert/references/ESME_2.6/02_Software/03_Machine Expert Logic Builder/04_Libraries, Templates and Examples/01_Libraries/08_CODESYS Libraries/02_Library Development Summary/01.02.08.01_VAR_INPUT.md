## VAR\_INPUT

The data in this area is provided by the caller (writer) and should by the called POU only be read. See rule SA0037 of the CODESYS Static Analysis “Write access to input variables”. The transfer method used is **by value**. The CODESYS compiler will initialize the input variables of a PROGRAM or FUNCTION\_BLOCK with default values. So the initialisation of the input variables by the caller is optional and it is possible to use “unconnected” input variables in this context.
