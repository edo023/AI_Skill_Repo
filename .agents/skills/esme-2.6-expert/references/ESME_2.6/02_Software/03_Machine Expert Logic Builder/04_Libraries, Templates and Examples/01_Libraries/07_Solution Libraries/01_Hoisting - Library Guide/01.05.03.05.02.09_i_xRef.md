# i_xRef

i\_xRef

This input defines the normally open (N/O) reference switch signal used in homing methods 23...30. Usage of this input is not mandatory when homing methods 23...30 are not used. For details, refer to the description of the [structure](AdvancedPositioning-12.htm#XREF_D_SE_0034423_1) stHmngPara.
