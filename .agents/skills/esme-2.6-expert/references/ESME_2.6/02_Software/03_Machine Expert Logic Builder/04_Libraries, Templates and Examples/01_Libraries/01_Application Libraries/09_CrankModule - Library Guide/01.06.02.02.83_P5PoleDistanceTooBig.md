# P5PoleDistanceTooBig

P5PoleDistanceTooBig

|  |  |
| --- | --- |
| Enumeration name: | P5PoleDistanceTooBig |
| Enumeration value: | 157 |
| Description: | P5PoleDistance is too big. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | The polynomial intervals overlap in all cases. | The inputs iq\_stCrankData.lrP5Pole1IntervalLow, iq\_stCrankData.lrP5Pole1IntervalHigh, iq\_stCrankData.lrP5Pole2IntervalLow and iq\_stCrankData.lrP5Pole2IntervalHigh must be selected so that the range for the polynomial intervals from SlideLogPos min. to SlideLogPos max. only overlap once. |
