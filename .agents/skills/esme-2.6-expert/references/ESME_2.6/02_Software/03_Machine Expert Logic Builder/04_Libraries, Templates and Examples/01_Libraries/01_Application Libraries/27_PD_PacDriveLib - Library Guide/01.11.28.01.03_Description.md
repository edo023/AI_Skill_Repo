# Description

Description

Calculates the coefficients of a polynomial P which meets the following boundary conditions:

oP(i\_lrX0) = i\_lrY0

P(i\_lrX1) = i\_lrY1

oP’(i\_lrX0) = i\_lrM0

P’(i\_lrX1) = i\_lrM1

oP’’(i\_lrX0) = i\_lrK0

P’’(i\_lrX1) = i\_lrK1

The coefficients are returned in a structure of the type ARRAY[0...5] OF LREAL which may serve e.g. as an input of the function [FC\_Poly5Value()](Functions_K_to_Z-34.htm#XREF_D_SE_0087603_1) in order to calculate the function values of the polynomial. The following must hold: i\_lrX1 > i\_lrX0.
