# DTConcat (FUN)

FUNCTION DTConcat : DATE\_AND\_TIME

The function converts year, month, hour, minute and second to the IEC 6-1131-3 format DATE\_AND\_TIME. In case of invalid inputs DT#1970-01-01-00:00 will be returned.
