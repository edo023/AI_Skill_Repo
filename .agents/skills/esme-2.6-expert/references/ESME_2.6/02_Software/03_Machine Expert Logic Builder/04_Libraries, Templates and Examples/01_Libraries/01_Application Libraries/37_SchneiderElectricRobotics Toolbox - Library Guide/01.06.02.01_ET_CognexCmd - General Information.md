# ET\_CognexCmd - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration |
| Available as of: | V1.6.0.0 |

## Description

This enumeration type specifies valid commands to send to a Cognex Vision System.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No command selected. |
| GetOnlineState | 10 | Get online state of the Cognex Vision System. |
| ChangeOnlineState | 20 | Change online state of the Cognex Vision System. |
| GetCurrentJob | 30 | Get current running job on the Cognex Vision System. |
| LoadJob | 40 | Load the selected job on the Cognex Vision System. |
| GetJobList | 50 | Get the list of jobs stored on the Cognex Vision System. |
| SendNativeCmd | 60 | Send a native command to the Cognex Vision System. |
