# ST\_Event

## Overview

|  |  |
| --- | --- |
| Type: | Structure |
| Available as of: | V1.1.0.0 |
| Inherits from: | – |

## Description

The structure ST\_Event contains information related to the event type defined in the enumeration [ET\_EventType](ET_EventType-21B5606A.html#ET_EventType-21B5606A).

NOTE: For generating event data, use the [FB\_EventWriter](FB_EventWriter-264D8B60.html#FB_EventWriter-264D8B60) to help avoid invalid data.

## Structure Elements

| Name | Data type | Description |
| --- | --- | --- |
| udiEventId | UDINT | The ID of an event. |
| udiEventType | UDINT | Identifies the type of the event.  For more information, refer to [ET\_EventType](ET_EventType-21B5606A.html#ET_EventType-21B5606A). |
| udiObjectId | UDINT | The ID of the object. |
| udiObjectType | UDINT | The type of the object. |
| udiParentId | UDINT | The ID of the parent object. |
| stPosition | SE\_MATH.ST\_Vector3D | Offset position relative to the connection point of the (sub)object, for example, the Tool Center Point (TCP) of a robot.  NOTE: In EcoStruxure Machine Expert, the position values are based on a right-hand coordinate system. |
| stOrientation | SE\_MATH.ST\_Vector3D | Offset orientation relative to the connection point of the (sub)object, for example, the Tool Center Point (TCP) of a robot.  NOTE: In EcoStruxure Machine Expert, the orientation values use the orientation convention ZYX, based on a right-hand coordinate system. |
| lrReferencePosition | LREAL | Reference position of the system identified in udiParentId when the event was created. The reference position is used to compensate for communication time delays. |
| udiColorId | UDINT | Color information of the object, based on the enumeration [ET\_EmulationColors](ET_EmulationColors-66CA3B7A.html). |
