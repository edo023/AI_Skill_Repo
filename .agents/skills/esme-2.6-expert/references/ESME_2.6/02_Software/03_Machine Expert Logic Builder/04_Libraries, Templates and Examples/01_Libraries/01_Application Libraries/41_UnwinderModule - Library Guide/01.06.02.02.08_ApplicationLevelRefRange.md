# ApplicationLevelRefRange

ApplicationLevelRefRange

|  |  |
| --- | --- |
| Enumeration name: | ApplicationLevelRefRange |
| Enumeration value: | 66 |
| Description: | ApplicationLevelRef is outside of the valid range. |

| Issue | Cause | Solution |
| --- | --- | --- |
| - | ST\_Auto.stParameter.stFeedbackSignal.i\_lrApplicationLevelRef does not match selection of ST\_Auto.stParameter.stFeedbackSignal.i\_lrApplicationValueMax and ST\_Auto.stParameter.stFeedbackSignal.i\_lrApplicationValueMin. | ST\_Auto.stParameter.stFeedbackSignal.i\_lrApplicationLevelRef has to be greater than or equal to ST\_Auto.stParameter.stFeedbackSignal.i\_lrApplicationValueMin and smaller than or equal to ST\_Auto.stParameter.stFeedbackSignal.i\_lrApplicationValueMax. |
