# IF\_LoggerPoint2 - AddLogEntry (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V1.0.0.0 |

## Description

This method has to be called to send a logger message to the Application Logger.

The information that is given to the logger point during the call of RegisterLoggerPoint is linked to every message that was sent using this logger point and does not have to be included inside the message.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etLogLevel | [ET\_LogLevel](ET_LogLevel-37599385.html#ET_LogLevel-37599385) | The logger level indicates the type and the priority of the message. |
| i\_udiDiagCode | UDINT | Diagnostic code of the logger message. |
| i\_sMessage | STRING[255] | The logger message. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If this output is set to TRUE, an error has been detected. For details, refer to q\_etResult and q\_etResultMsg. |
| q\_etResult | [ET\_Result](ET_Result-375A58EE.html#ET_Result-375A58EE) | Provides diagnostic and status information as a numerical value. |
| q\_sResultMsg | STRING[80] | Provides additional diagnostic and status information as a text message. |
