# NMT (FB)

FUNCTION\_BLOCK NMT EXTENDS CiA405Base

This function block provides NMT services. The NMT request will be sent to the NodeID specified in input DEVICE. If NMT request should be addressed to all devices (including NMT master and sending device) set DEVICE to 0 . If all devices but not NMT master and sending device should be addressed use the additional flag [TRANSITION\_STATE.ALL\_EXCEPT\_NMT\_AND\_SENDER](WkZ9when7NogvVdipw-bWj0hVTA_transition-state.html#wkz9when7nogvvdipw_bwj0hvta_transition_state_transition_state_enum__wkz9when7nogvvdipw_bwj0hvta_transition_state_transition_state_all_except_nmt_and_sender).
