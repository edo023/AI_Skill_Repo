# HEXinASCII\_TO\_BYTE (FUN)

FUNCTION HEXinASCII\_TO\_BYTE : BYTE

Converts one word in HEX code representing its ASCII value into one byte of binary code

The higher byte in the input represents the higher bits in the output. Use [BYTE\_TO\_HEXinASCII](a2M_YG6_8tw_qPOw5eUVQXcrEeg_byte-to-hexinasc.html#a2m_yg6_8tw_qpow5euvqxcreeg_byte_to_hexinasc_byte_to_hexinascii_fun) function for the reverse conversion.
