# ReverseBYTEsInWORD (FUN)

FUNCTION ReverseBYTEsInWORD : WORD

Function returns the WORD “wInput” with a reversed byte-order (Low-Byte<->High-Byte).

Example:

```
wInput = 16#0102
->
MEM.ReverseBYTEsInWORD(wInput) = 16#0201
```
