# DriverOpenH (FUN)

FUNCTION DriverOpenH : CAA.HANDLE

Opens a CAN interface and allocates memory from heap. Not every hardware supports each baud rate. If a baud rate is not supported or if the interface cannot be opened due to different reasons, the function returns CAA.gc\_hINVALID. The function can be called several times on the same interface, so that multiple parts of a program can work on it. You can set uiBaudrate to 0 if the interface is already opened or if the default baudrate from configuration file should be used. See following section in configuration file:

[CmpCAACanL2]

PersistentBaudrate=1

Net.0.DefaultBaudrate=1000
