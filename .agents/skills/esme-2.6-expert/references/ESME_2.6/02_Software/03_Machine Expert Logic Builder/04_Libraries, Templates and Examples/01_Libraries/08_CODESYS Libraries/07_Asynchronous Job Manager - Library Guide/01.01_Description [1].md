## Description [[1]](#index_indices_and_tables__fn_index_id2)

A Manager for asynchronous job processing.

It mainly offers a function block [BackgroundTask](rJ9NnPzj8kCZsoW9jcVtGOgoM-k_backgroundtask.html#rj9nnpzj8kczsow9jcvtgogom_k_backgroundtask_backgroundtask_fb). This allows certain job to be executed in the background independently of the foreground task. This should normally shorten the cycle time of the foreground task.

For handling dynamic generation of BackgroundTask instances the additional function blocks [BackgroundTaskFactoryBase](rJ9NnPzj8kCZsoW9jcVtGOgoM-k_backgroundtaskfa1.html#rj9nnpzj8kczsow9jcvtgogom_k_backgroundtaskfa1_backgroundtaskfactorybase_fb) and [BackgroundTaskFactoryArgs](rJ9NnPzj8kCZsoW9jcVtGOgoM-k_backgroundtaskfa.html#rj9nnpzj8kczsow9jcvtgogom_k_backgroundtaskfa_backgroundtaskfactoryargs_fb) are provided.

Further details are described under [AsyncManager](pH5EC-zcjrPC-lLEiLF3wRgoTBQ_fld-AsyncManager.html#ph5ec_zcjrpc_lleilf3wrgotbq_fld_asyncmanager_asyncmanager).
