# IAlarmManagerClient2.EventOccurred (METH)

METHOD EventOccurred

Notification, that an event occurred, that is not visible in the list of active alarms, but is only written into the alarm storage. Remark: This method is only called for those event, that matches the filter criteria returned by method IAlarmManagerClient.GetFilterCriteria
