## Task

Interface to access the diagnostic handler functionalities.

## Description

The interface IF\_DiagHandlerAccess is implemented by the FB\_DiagnosticHandler and provides access to the diagnostic handler functionalities, for example setting a diagnostic message or triggering a diagnostic quit process.
