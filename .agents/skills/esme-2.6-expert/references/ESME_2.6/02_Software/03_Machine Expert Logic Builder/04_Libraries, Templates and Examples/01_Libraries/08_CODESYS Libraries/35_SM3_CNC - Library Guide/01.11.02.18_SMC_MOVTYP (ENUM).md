# SMC\_MOVTYP (ENUM)

TYPE SMC\_MOVTYP :

Indicates the geometrical type of the object. Possible states:

| InOut: | | Name | Initial | Comment | | --- | --- | --- | | LIN | 1 | Straight movement (G01) | | CLW |  | Circle in clockwise direction (G02) | | CCLW |  | Circle in counterclockwise direction (G03) | | SPLINE | 5 | Spline, parabola (G05, G06) | | ELLCLW | 8 | Clockwise ellipse (G08) | | ELLCCLW | 9 | Counterclockwise ellipse (G09) | | SPLINE3D | 10 |  | | SPLINE3D\_5 | 11 |  | | LINPOS | 100 | Straight positioning (G00) | | LINNODELTA | 101 | Straight movement without movement in X/Y but Z | | LINADDAXES | 102 | Straight movement without movement in X/Y/Z | | INITPOS | 110 | Blind positioning (start point still unknown, will be added by [SMC\_Interpolator](8OGX-fND8b3kPGnX4Q3neKp1m4c_smc-interpolator.html#8ogx_fnd8b3kpgnx4q3nekp1m4c_smc_interpolator_smc_interpolator_fb)) | | MCOMMAND | 120 | Additional option, M-option | |
