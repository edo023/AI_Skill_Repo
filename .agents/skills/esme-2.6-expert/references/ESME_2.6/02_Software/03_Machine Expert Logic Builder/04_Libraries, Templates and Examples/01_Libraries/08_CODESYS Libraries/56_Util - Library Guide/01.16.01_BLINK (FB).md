# BLINK (FB)

FUNCTION\_BLOCK BLINK

Simulates a blinking signal (turning on and off for specific durations)

| InOut: | | Scope | Name | Type | Comment | | --- | --- | --- | --- | | Input | ENABLE | BOOL | TRUE: Starts blinking  FALSE: Stops blinking whereas OUT keeps its value | | TIMELOW | TIME | Time for OUT to be FALSE | | TIMEHIGH | TIME | Time for OUT to be TRUE | | Output | OUT | BOOL | Output value, starts with FALSE and switches between TRUE and FALSE for the given high and low times | |
