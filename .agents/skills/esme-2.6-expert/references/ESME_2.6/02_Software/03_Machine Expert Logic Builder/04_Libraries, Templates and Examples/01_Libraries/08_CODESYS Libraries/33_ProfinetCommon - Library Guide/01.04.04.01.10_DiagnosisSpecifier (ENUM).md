# DiagnosisSpecifier (ENUM)

TYPE DiagnosisSpecifier :

In the context of a received diagnosis alarm, the specifier indicates whether a disgnosis entry appears of disappears.

| Attributes: | qualified\_only |
| InOut: | | Name | Initial | Comment | | --- | --- | --- | | All\_subsequent\_disappears | 0 | The Diagnosis ASE contains no longer any entries (of any severity) for this channel | | Appears | 1 | An event appears and/or exists further | | Disappears | 2 | An event disappears and/or exists no longer | | Disappears\_but\_other\_remain | 3 | The Diagnosis ASE contains no longer any entries of the same severity for this channel An event disappears The Diagnosis ASE still contains other entries of the same severity for this channel | |
