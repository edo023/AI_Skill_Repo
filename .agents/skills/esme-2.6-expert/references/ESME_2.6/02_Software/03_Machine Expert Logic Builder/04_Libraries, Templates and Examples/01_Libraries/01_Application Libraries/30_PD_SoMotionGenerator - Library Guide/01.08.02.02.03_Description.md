# Description

Description

It is possible to transfer external cam jobs to the SoMotionGenerator. For this purpose, the user has to transfer a function block to the structure [ST\_ExternalCam](../Structures/Structures-4.htm#XREF_D_SE_0089484_1) via the parameter ifExternal­CamGenerator which implements the interface IF\_CamReferenceGenerator.

The calculation of reference values is executed in the method Calculate() and is called up by the SoMotionGenerator in the real-time cycle.
