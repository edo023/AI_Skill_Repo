# VisuFbAnalyzeTextVarsDateTimeOnly.GetAsLWORD (METH)

METHOD GetAsLWORD : LWORD

Takes the current variable value and converts it into a LWORD value. This method can be only used if the variable’s type is DATE, DT, LDATE or LDT.
