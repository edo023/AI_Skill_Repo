## New Features

* [**IF\_RobotFeedbackKinematic.ralrRefPositionJoint**](D-SE-0075549.html#D-SE-0075549__D-SE-0075549.5)

  Feedback for joint positions of robot kinematics.
* [**ET\_OrientationConvention.ZXY**](D-SE-0075485.html#D-SE-0075485__D-SE-0075485.4)

  ET\_OrientationConvention.ZXY was added and can be used, for example, to configure a linear tracking system.
* [**Additional Output Variables q\_lrPeriodStart and q\_lrPeriodLength for IF\_RobotConfiguration.GetConfiguredDrive**](D-SE-0075529.html#D-SE-0075529__D-SE-0075529.2)

  To read the configured period data for an auxiliary axis, the outputs q\_lrPeriodStart and q\_lrPeriodLength were added.
* [**IF\_RobotConfigurationAdvanced.xDisableTrackingEncoderCheck**](D-SE-0075531.html#D-SE-0075531__D-SE-0075531.6)

  The parameter verifications of FeedConstant, GearIn, GearOut, and Direction for the tracking encoder against the corresponding parameters of the velocity source can be disabled.
* [**IF\_RobotConfigurationAdvanced.xSetTrackingEncoderParameters**](D-SE-0075531.html#D-SE-0075531__D-SE-0075531.6)

  The parameters FeedConstant, GearIn, GearOut, and Direction of the tracking encoder are set automatically fitting to the corresponding parameters of the velocity source.
* [**IF\_Configuration.User6Ax(…)**](D-SE-0098607.html#D-SE-0098607__D-SE-0098607.2)

  With this method, the robot can be configured as a six axial user-configured robot with six degrees of freedom.

## Mitigated Anomalies

* In case of ResultantAccelerationLimitation and PreAnalysis = TRUE, a StopOnPath is retained.

  In case a motion profile must be modified due to limit ResultantAccelerationLimitation a StopOnPath is executed.
