# Modicon M258 Logic Controller

## Overview

Modicon M258 Logic Controllers are supported by EcoStruxure Machine Expert V1.2.•.

Software configuration and hardware description can be found in the following manuals:

* Modicon M258 Logic Controller [programming guide](../../../../../api/crossBook?lang=en-US&virtualBookName=m258prg&topicID=D_SE_0003112)
* Modicon M258 Logic Controller [hardware guide](../../../../../api/crossBook?lang=en-US&virtualBookName=tm5hw258&topicID=D_SE_0003145)

The Schneider Electric Modicon M258 Logic Controller is a controller with a variety of powerful features. It can control a wide range of applications.

The software configuration is described in the [EcoStruxure Machine Expert Programming Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SG_0026478).

## Key Features

The EcoStruxure Machine Expert software compatible with Modicon M258 Logic Controller provides the following IEC61131-3 programming languages:

* IL: Instruction List
* ST: Structured Text
* FBD: Function Block Diagram
* SFC: Sequential Function Chart
* LD: Ladder Diagram

EcoStruxure Machine Expert software can also be used to program the controller using CFC (Continuous Function Chart) language.

All controllers include:

* CANopen Master
* Ethernet
* Serial Line
* Expert functions (counting, reflex outputs...)
* Embedded I/Os

All controllers support up to 20 tasks with the following limits:

* 4 cyclic tasks: one is configured by default (Mast)
* 1 freewheeling task
* 8 software event driven tasks
* 8 hardware event driven tasks
