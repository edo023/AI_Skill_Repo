# TM5 Transmitter and Receiver Modules

## Introduction

The following table lists the transmitter and receiver expansion modules with their descriptions.

The transmitter and receiver modules are described in the Modicon [TM5 Transmitter and Receiver Modules Hardware Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=tm5bushw&topicID=D_SE_0003232).

## TM5 Transmitter and Receiver Modules

| Module | Module Description |
| --- | --- |
| [TM5SBET1](../../../../../api/crossBook?lang=en-US&virtualBookName=tm5bushw&topicID=D_SE_0002829) | Transmits the TM5 data bus. |
| [TM5SBET7](../../../../../api/crossBook?lang=en-US&virtualBookName=tm5bushw&topicID=D_SE_0011044) | Transmits the TM7 data bus and provides the TM7 power bus to the TM7 expansion I/O blocks. |
| [TM5SBER2](../../../../../api/crossBook?lang=en-US&virtualBookName=tm5bushw&topicID=D_SE_0002828) | Receives the TM5 data bus, provides power to the 24 Vdc I/O power segment and provides the TM5 power bus to the TM5 expansion I/O modules. |
