# Modicon M251 Logic Controller

## Overview

Software configuration and hardware description can be found in the following manuals:

* Modicon M251 Logic Controller - [Programming Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=m251prg&topicID=D_SE_0035747)
* Modicon M251 Logic Controller - [Hardware Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=m251hw&topicID=D_SE_0035746)
* Industrial Ethernet User Guide

The Modicon M251 Logic Controller has various powerful features and can service a wide range of applications.

Software configuration, programming, and commissioning is accomplished with the software described in the [EcoStruxure Machine Expert Programming Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SG_0026478).

## Programming Languages

The M251 Logic Controller is configured and programmed with the software, which supports the following IEC 61131-3 programming languages:

* IL: Instruction List
* ST: Structured Text
* FBD: Function Block Diagram
* SFC: Sequential Function Chart
* LD: Ladder Diagram

The software can also be used to program these controllers using CFC (Continuous Function Chart) language.

## Embedded Communication Features

The following types of communication ports are available on the front panel of the controller, depending on the controller reference:

* CANopen Master
* Ethernet
* USB Mini-B
* Serial Line
