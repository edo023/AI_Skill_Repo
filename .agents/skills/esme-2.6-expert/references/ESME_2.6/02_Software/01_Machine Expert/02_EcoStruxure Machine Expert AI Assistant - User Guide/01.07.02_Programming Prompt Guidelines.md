# Programming Prompt Guidelines

To help improve clarity, consistency, and reusability in AI-assisted programming prompts, use the following structure when defining a prompt:

1. **Essential Context Information**

   Start your prompt by specifying the following information:

   * **Controller Type**: Specify the controller type (M262 or PacDrive).
   * **Drive Type** (if applicable): Specify the drive type used in the application.
   * **Library** (if known): Specify the relevant library or function block being used.
2. **Prompt Structure**

   * Focus on one functionality per prompt to maintain clarity and modularity.
   * Build incrementally: start with a basic version of the functionality and progressively expand it in subsequent prompts.
