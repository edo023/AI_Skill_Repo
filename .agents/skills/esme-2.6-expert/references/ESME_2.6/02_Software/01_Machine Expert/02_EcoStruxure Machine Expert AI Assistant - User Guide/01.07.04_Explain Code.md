# Explain Code

There are two ways to generate an explanation of the generated code:

1. **Manual Input**

   To generate an explanation of the generated code, follow these steps:

   | Step | Action |
   | --- | --- |
   | 1 | Copy the code output from the Generate Code window.  NOTE: You can use the  icon to copy the code. |
   | 2 | Using the arrow in the bottom left menu, select Explain code. |
   | 3 | Paste the code in the text entry box and click Send or press Enter.  The explanation of the code: * Provides a visual representation of the code structure and flow. * Breaks down the code into logical and functional sections. Each section is described in natural language, highlighting its purpose and key operations. |
2. **Direct Interaction**

   Click Explain this code.

The explanation of the code:

* Provides a visual representation of the code structure and flow.
* Breaks down the code into logical and functional sections. Each section is described in natural language, highlighting its purpose and key operations.
