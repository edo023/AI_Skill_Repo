# Create Test Case

There are two ways to test the generated content:

1. **Manual Input**

   To test the generated content, follow these steps:

   | Step | Action |
   | --- | --- |
   | 1 | Copy the generated content from the Generate Code window.  NOTE: You can use the  icon to copy the code. |
   | 2 | Using the arrow in the bottom left menu, select Create test case. |
   | 3 | Paste the code in the text entry box and click Send or press Enter. |
2. **Direct Interaction**

   Click Create a test case for this code.

**Result:** The test case code is displayed. You can copy and paste the test case into the targeted programming environment for execution and testing.
