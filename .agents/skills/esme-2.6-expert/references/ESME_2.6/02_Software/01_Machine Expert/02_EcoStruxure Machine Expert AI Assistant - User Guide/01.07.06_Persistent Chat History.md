# Persistent Chat History

The EcoStruxure Machine Expert AI Assistant provides Persistent Chat History, allowing users to access their previous prompts and responses.

To access the Persistent Chat History, follow these steps:

| Step | Action |
| --- | --- |
| 1 | Click the Persistent Chat History icon on the top left corner of the EcoStruxure Machine Expert AI Assistant interface. |
| 2 | Upon clicking on the Persistent Chat History icon, a side pane appears containing the list of previous conversations between the EcoStruxure Machine Expert AI Assistant and the user.  NOTE: The chat history is only visible to the user who is signed in with their User profile. Other users chat history will not be visible. |
| 3 | Hover over the message to delete it from the chat history, click the icon. |
