# About the Document

## Document Scope

This document describes how to install and use EcoStruxure Machine Expert AI Assistant feature of EcoStruxure Machine Expert. With EcoStruxure Machine Expert AI Assistant, you can generate and explain the code, view references, and generate test code.

For further information, refer to the separate documents provided in the EcoStruxure Machine Expert Help.

## Validity Note

This document has been updated for the release of EcoStruxure Machine Expert V2.6.
