# Logging In

| Step | Action |
| --- | --- |
| 1 | Log in to EcoStruxure Machine Expert.  For more information, see the [EcoStruxure Machine Expert - User Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=esmepug&topicID=FirstStartSteps_1F00F201). |
| 2 | Click the EcoStruxure Machine Expert – AI Assistant icon in the upper right corner of the EcoStruxure Machine Expert home page. |
| 3 | In the Activate AI Assistant window, enter your subscription credentials and click Submit.  **Result:** You are redirected to EcoStruxure Machine Expert AI Assistant. |
