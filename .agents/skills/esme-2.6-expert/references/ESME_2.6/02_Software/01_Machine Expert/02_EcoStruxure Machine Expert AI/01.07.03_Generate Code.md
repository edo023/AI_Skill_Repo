# Generate Code

To Generate Code, follow these instructions:

| Step | Action |
| --- | --- |
| 1 | Using the arrow in the bottom left menu, select Generate Code. |
| 2 | In the entry box, enter controller-related prompts.  Example prompt:  Generate Structured Text (ST) code using the PLC open Motion Control library for an indexing and sawing application controlled by a Modicon M262 Motion Controller. The application consists of a Lexium 32S servo drive controlling a linear actuator that feeds a tube for cutting based on a user‑defined length and a user‑defined speed. |
| 3 | Submit your prompt by clicking Send or press Enter. |
| 4 | After submitting the prompt, EcoStruxure Machine Expert AI Assistant generates a response based on your input. NOTE: EcoStruxure Machine Expert AI Assistant-generated content must be reviewed for accuracy before use. |

With the generated code, you can:

* Click the copy icon ![](images/DD01362934.png) to copy the code output and paste it into the Explain code tab for an explanation. For more information, see [Explain Code](Explain-9192A36A.html#Explain-9192A36A).
* Click the copy icon ![](images/DD01362934.png) to copy the code output and paste it into the Create test case tab to create a test case. For more information, see [Create Test Case](Test-9192A43E.html#Test-9192A43E).
