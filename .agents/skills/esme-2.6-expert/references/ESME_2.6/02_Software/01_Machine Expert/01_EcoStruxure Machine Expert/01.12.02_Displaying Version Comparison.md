# Displaying Version Comparison

## Introduction

To compare commits, in the Repository window, select the branch you want to compare, and then in the History tab select the commit to compare.

The Compare view displays the differences between the selected commit and the content of the branch. It provides a comparison displaying the added, removed, and modified lines.

The page is divided into two vertical sections:

* Previous commit on the left side.
* Current commit on the right side.

## Left Side - Previous Commit

The Previous section displays the content of the baseline commit you selected in the History tab. It represents the state before the changes.

This section contains:

* The file content as it existed in the selected Previous commit.
* Removed or modified lines highlighted in red.
* The commit identifier to track the source revision.

## Right Side - Current Commit

The Current section displays the content of the branch or commit you selected. It represents the state after the changes.

This section displays:

* The updated file content from the selected commit.
* Added or modified lines highlighted in green.
* The commit identifier to track the source revision.
