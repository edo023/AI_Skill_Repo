# Displaying the History

To display the History of a solution, follow these steps:

| Step | Action |
| --- | --- |
| 1 | In the Repository window, select the branch for which you want to display the History. |
| 2 | In the left pane, click the History tab.  **Result**: the commits are displayed with the following information:  * Title of the commit * ID of the commit * Clicking  allows you to:    + Reset branch to this commit: The operation updates your local files to match the content of this commit and discards uncommitted changes.   + Revert this commit: Creates a new commit that reverts the changes introduced by the selected commit. The operation also modifies your local files to apply the reverted content. |
| 3 | Click a commit to [display the Compare tab](ViewingVersionComparaison-E5646147.html). |
