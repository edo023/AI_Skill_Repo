# Select Solution

## Introduction

The Select solution of EcoStruxure Machine Expert is used to manage the solutions for customers as per their project requirements. The Select solution displays the local stored solutions last used by the user.

A solution is a container that organizes and groups together related EcoStruxure Machine Expert projects and files. The purpose of a solution is to provide a structured way to manage multiple projects, such as machine projects or multi‑project configurations.

## Select Solution Window

The Select solution window is accessed directly after opening EcoStruxure Machine Expert.

In the Select solution window, the following commands are available:

| Command / Icon | Description |
| --- | --- |
|  | Allows you to find a solution by entering its name, in whole or in part. |
|  | The solution selection card displays:  * The solution name * When the solution was last modified * The solution file path  The  button allows you to:  * Open the solution * Copy local path of the solution |
|  | The Get started section allows you to:  * [Open from disk](#LaunchingPortalHomePageAndAutomatio-01C65C3B__OpeningASolutionFromDisk-419DD86E) * [Open from repository](#LaunchingPortalHomePageAndAutomatio-01C65C3B__OpeningASolutionFromARepository-419DE60A) * [Create new solution](#LaunchingPortalHomePageAndAutomatio-01C65C3B__CreatingASolution-41FD6C80) |

## Creating a Solution

To create a solution, follow these steps:

| Step | Action |
| --- | --- |
| 1 | Click Create new solution. |
| 2 | In the Create new solution window:   1. Enter the Title of your solution. 2. Enter or browse the Local path where you want to store the solution. 3. Click Create solution.   **Result**: The created solution appears. |

After you have created a solution, the Projects page is displayed to create or import new Projects and list the existing Projects.

## Opening a Solution from Disk

To open a solution from a disk, follow these steps:

| Step | Action |
| --- | --- |
| 1 | Click Open from disk. |
| 2 | Locate the solution to open in the explorer. |
| 3 | Click Open to open the solution.  **Result**: The opened solution appears. |

## Opening a Solution from a Repository

To open a solution from a repository, follow these steps:

| Step | Action |
| --- | --- |
| 1 | Click Open from repository. |
| 2 | In the Open from repository window:   1. Enter the Remote URL of your solution remote repository. Refer to [Configuring a Git Clone](ConfiguringAGitRepository-459A8B23.html). 2. Enter the Title of your solution. 3. Enter or browse the Local path where you want to store the solution. 4. Click Open solution.   **Result**: The opened solution appears. |
