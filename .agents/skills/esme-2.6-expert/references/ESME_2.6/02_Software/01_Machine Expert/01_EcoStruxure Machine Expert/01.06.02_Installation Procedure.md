# Installation Procedure

## Prerequisites

Before starting the installation, ensure that the following prerequisites are met:

* **SESI Installation**

  The installation of [SESI application package](InstallationOverview-1EFFF225.html) (referred to herein as the installer) is mandatory.
* **Administrative Rights**

  The installation and uninstallation processes must be run with administrative rights.

## Installation Steps

The following table describes how to install EcoStruxure Machine Expert:

| Step | Action |
| --- | --- |
| 1 | Launch SESI. Click Install New Software. |
| 2 | Click the Release Notes option to read the Release Notes. |
| 3 | Select the source as Online.  As product, choose EcoStruxure Machine Expert.  Select the latest version and click Next. |
| 4 | Review the license agreement, and if you agree, click Start Installation. |
| 5 | Wait for the installation process to complete. Upon completion, a confirmation message appears. Click OK, then click Next. |
| 6 | Click Finish to exit SESI. |

If the installation process stops, review the error logs, address the issues mentioned in the logs and start the installation process again.

## Uninstallation Steps

The following table describes how to uninstall EcoStruxure Machine Expert:

| Step | Action |
| --- | --- |
| 1 | Launch SESI. Click Modify Installed Software. |
| 2 | As product, choose EcoStruxure Machine Expert.  Select the installed version. |
| 3 | Under Choose an action, select the Uninstall version option.  Then click Next. |
| 4 | Wait for the uninstallation process to complete. Upon completion, a confirmation message appears. Click OK, then click Next. |
| 5 | Click Finish to exit SESI. |
