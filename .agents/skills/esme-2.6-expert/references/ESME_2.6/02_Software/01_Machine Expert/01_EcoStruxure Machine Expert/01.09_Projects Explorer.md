# Projects Explorer

## Introduction

A solution is a collaborative repository of the data needed to manage the different projects. The Project explorer allows you to access the Projects within a solution.

## Creating a Project

To create a Project follow these steps:

| Step | Action |
| --- | --- |
| 1 | In the Select solution window, click a solution to access the Projects explorer. |
| 2 | Click  and choose Create > New project.... |
| 3 | In the Create project window:   1. Enter the Title of the project. 2. Select the Version of the project. 3. Click Create.   **Result**: The application required for the chosen project type starts. |
| 4 | Create your project in the application. For more information, refer to [EcoStruxure Machine Expert, Programming Guide](https://product-help.se.com/ESME/latest/VLP_Software/VLP_Programming/en-US/SoMProg?t=SoMProg%2FSoMProg_D-SG-0026478.html&theme=Help).  NOTE: After the project is created in the application, refresh the Projects explorer page to make it visible. |

## Importing a Project

To import Projects follow these steps:

| Step | Action |
| --- | --- |
| 1 | In the Select solution window, click a solution to access the Projects explorer. |
| 2 | Click  and choose Import > Existing project.... |
| 3 | In the Import project window:   1. Enter or browse a Local path. 2. Enter the Title of the project. 3. Select the Version of the project. 4. Click Import.   **Result**: The application required for the chosen project type starts. |
| 4 | Import your project in the application. For more information, refer to [EcoStruxure Machine Expert, Programming Guide](https://product-help.se.com/ESME/latest/VLP_Software/VLP_Programming/en-US/SoMProg?t=SoMProg%2FSoMProg_D-SG-0026478.html&theme=Help).  NOTE: After the project is created in the application, refresh the Projects explorer page to make it visible. |

## Logic Menu

When you click a project, the Logic menu is displayed.

This menu displays:

* On the left: the complete solution structure in the explorer, where you can
  + Navigate through the files by collapsing and expanding them.
  + Search branch.
  + [Create new branch](BranchManagement-48F430A2.html).
* On the right: the details of the selected file, where you can
  + View the Content and History of the selected file.
  + [Edit the Content](BasicStructuredTextWebEditor-AED07826.html).

  NOTE: You can open multiple files in the detailed view, and they appear as tabs.
