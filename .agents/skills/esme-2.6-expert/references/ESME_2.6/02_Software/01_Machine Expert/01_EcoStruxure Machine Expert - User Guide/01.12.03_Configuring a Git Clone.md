# Configuring a Git Clone

To configure a Git Clone, follow these steps:

| Step | Action |
| --- | --- |
| 1 | In the Select solution, click the solution for which you want to configure a Git Clone. |
| 2 | Click . |
| 3 | In the Git repository section, enter the remote URL of the Remote repository, for example *https://github.<your company>.com/<organization>/<repository name>.git*.  NOTE: For the connection to the remote repository, EcoStruxure Machine Expert uses the Windows Credential Manager. Add a generic credential with the development platform address (it must start with git, for example git:https://github.<your company>.com), user name and password/personal access token. |
| 4 | Click Configure.  **Result**: The Git repository is configured. It appears as Configured. |
