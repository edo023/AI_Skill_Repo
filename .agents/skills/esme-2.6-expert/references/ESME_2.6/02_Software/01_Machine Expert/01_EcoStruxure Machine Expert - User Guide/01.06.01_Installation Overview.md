# Installation Overview

## Introduction

Schneider Electric Software Installer (SESI) allows you to install and update several Schneider Electric application packages.

SESI is a [prerequisite](DesktopInstallation-1F007943.html#DesktopInstallation-1F007943__Prerequisites-25DA16A4) for EcoStruxure Machine Expert. To download and install SESI, visit the Schneider Electric download center [www.se.com/ww/en/download/](https://www.se.com/ww/en/download/).
