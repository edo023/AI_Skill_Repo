# EcoStruxure Machine Expert Overview

EcoStruxure Machine Expert is a collaborative and unified engineering environment that supports automation engineers to manage the life cycle of a solution.

EcoStruxure Machine Expert provides a set of functionalities to help your user experience and collaboration:

* Solution management: repository for solutions providing centralized access to existing projects
* Collaborative work environment: share and collaborate on solutions
* Project view to configure multiple projects and coordinate projects
* Source control and change tracking
