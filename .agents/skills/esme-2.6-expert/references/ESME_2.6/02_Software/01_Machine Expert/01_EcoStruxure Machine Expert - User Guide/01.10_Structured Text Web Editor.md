# Structured Text Web Editor

## Introduction

The Structured Text (ST) Web Editor allows you to edit your applications without opening the EcoStruxure Machine Expert application.

## Editing Code with the ST Web Editor

To edit code with the ST Web Editor, follow these steps:

| Step | Action |
| --- | --- |
| 1 | After you created your EcoStruxure Machine Expert project in the EcoStruxure Machine Expert Logic Builder (refer to [First Steps](FirstSteps-4280897B.html)), click your project. |
| 2 | Expand the tree structure by clicking the triangle icons. |
| 3 | Click the tree structure element.  **Result**: The code is displayed in a tab in the right pane window. |
| 4 | Click Edit to allow the modification of the code. |
| 5 | Edit the code, then click Commit changes.  **Result**: A confirmation window appears.  NOTE: To discard changes, click Discard changes. The Discard changes window appears, select the I confirm I want to discard changes. check box, then click Yes. |
| 6 | You can add an explanation on what was modified in the code in the Commit message text box. Then click Commit changes to confirm.  **Result**: The code is edited and integrated to the application. |

NOTE: In the EcoStruxure Machine Expert application, click Synchronize Changes in the Source Control window to display the modifications implemented in the ST Web Editor.
