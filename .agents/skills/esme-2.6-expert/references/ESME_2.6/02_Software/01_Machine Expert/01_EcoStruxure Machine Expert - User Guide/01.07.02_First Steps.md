# First Steps

When EcoStruxure Machine Expert is launched, the Select solution window appears to create a solution.

To create a first solution, follow these steps:

| Step | Action |
| --- | --- |
| 1 | Click Create solution or use the [Get started section](LaunchingPortalHomePageAndAutomatio-01C65C3B.html#LaunchingPortalHomePageAndAutomatio-01C65C3B__SelectSolutionWindow-419D7E1F). |
| 2 | In the Create solution window:   1. Enter the Title of your solution. 2. Enter or browse a Local path. 3. Click Create solution.   **Result**: The solution is created, the Projects explorer is displayed. |
| 3 | You can [create or import Projects](AutomationSolutionExplorer-4BEC913D.html). |
