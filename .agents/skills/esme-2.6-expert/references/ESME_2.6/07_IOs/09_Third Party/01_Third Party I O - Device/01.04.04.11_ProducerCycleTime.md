# ProducerCycleTime

General

| Type | ED |
| Devices supporting the parameter | Safe Logic Controller TM5CSLCx00FS,  Bus Interface TM5NS31,  Sercos I/O |
| Traceable | – |

Functional Description

Displays the cycle time in ns at which the cyclic data of a Sercos device are compared to the terminal. This parameter is taken over during Sercos run-up.

This parameter must be an integer multiple of the Sercos cycle time for Sercos I/O devices. Safe Logic Controller TM5CSLCx00FS and Bus Interface TM5NS31 only support the ProducerCy­cleTime if it equals the Sercos cycle time.

Up to PacDrive LMC x01C/PacDrive LMC x00C version V1.36.2.6:

If an invalid value is entered, diagnostic message 8994 is triggered.

As of PacDrive LMC x01C/PacDrive LMC x00C version V1.36.2.7:

If an invalid value which is higher than the Sercos cycle time is entered, diagnostic message 8994 is triggered.

If an invalid value which is smaller than the Sercos cycle time is entered, the Sercos cycle time is automatically used as ProducerCycleTime.

Example:

ProducerCycleTime = 1,000,000 ns

SERCOS III.CycleTime = 4,000,000 ns

Used ProducerCycleTime during run-up: 4,000,000 ns

No diagnostic message is triggered.

If a value which is not supported by the Sercos device is entered, error 8508 SERCOS run-up not possible occurs with the ExtDiagMsg C\_Time=20000.
