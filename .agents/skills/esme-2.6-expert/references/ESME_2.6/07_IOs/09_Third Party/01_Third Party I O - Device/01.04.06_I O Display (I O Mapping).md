# I/O Display (I/O Mapping)

![](images/logo.png)

I/O Display (I/O Mapping)

I/O Display (I/O Mapping)

General

The input and output channels of a module can be mapped to project variables that are used by the application.

Whether an I/O display can be configured for a certain module depends on the respective module.

For further information concerning the I/O display, refer to the online help under EcoStruxure Machine Expert > EPAS > Device Editors > Generic Device Editor > I/O Display.
