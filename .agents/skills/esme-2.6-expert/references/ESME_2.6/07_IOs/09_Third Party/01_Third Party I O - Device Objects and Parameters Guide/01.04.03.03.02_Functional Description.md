# Functional Description

Functional Description

With this command, you can set the working mode of the device.

| Value | Data type | Meaning |
| --- | --- | --- |
| virtual / 0 | DINT | The device is operated virtually.  Reference position values are not transferred to a physical device. Actual position values are copied from the reference values in each motion cycle. |
| real / 1 | DINT | The device is being searched for in the Sercos network  oDevice found: -> device is used.  oDevice not found: -> diagnostic message; Sercos phase up not possible. |
| deactivated / 2 | DINT | Position value calculation is completely deactivated. This behavior reduces the cycle load. |
| optional / 3 | DINT | The device is being searched for in the Sercos network:  oDevice found: -> device is used.  oDevice not found: -> device is operated in virtual / 0 working mode. |
