# Ethernet Services

## Overview

The Modicon TM5 EtherNet/IP Fieldbus Interface supports the following services:

* Ethernet/IP adapter
* Web server
* DHCP client

## Ethernet Protocols

The Modicon TM5 EtherNet/IP Fieldbus Interface supports IP (Internet Protocol).

## Connections

The Modicon TM5 EtherNet/IP Fieldbus Interface is equipped with 2 RJ45 connectors.
