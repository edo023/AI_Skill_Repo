# Adding the TM5SE4IOL Module and IO-Link Devices to Your Application

## Adding the TM5SE4IOL Module to Your Application

| Step | Action |
| --- | --- |
| 1 | Launch the programming software EcoStruxure Machine Expert. |
| 2 | Create a new project with File > New Project with a PacDrive LMC Eco/PacDrive LMC Pro/Pro2 or Modicon M262 Logic/Motion controller. |
| 3 | In the Devices tree, right-click the corresponding node and select Add Device from the contextual menu. |
| 4 | In the Add Device dialog box, select the bus Interface TM5NS31 or the bus interface TM5NEIP1 and confirm with Add Device. |
| 5 | In the Devices tree, right-click the TM5NS31 node or the TM5NEIP1 node and select Add Device from the contextual menu. |
| 6 | In the Add Device dialog box, select TM5 Communication Modules > TM5SE4IOL or TM5 Communication Modules > TM5NEIP1 and confirm with Add Device. |
| 7 | In the Devices tree, double-click the TM5SE4IOL node to open its module editor where you can edit the module parameters. |

After you have added the TM5SE4IOL module, four slots for IO-Link devices are available. Empty slots are identified by the entry <Empty>. Slots to which an IO-Link device have been assigned have the name of the IO-Link device.

## Importing IO-Link Devices Into Your Application

| Step | Action |
| --- | --- |
| 1 | Download the IODD (IO-Link Device Description) XML file for your IO-Link device from the IO-Link website at <https://ioddfinder.io-link.com/#/>. |
| 2 | In EcoStruxure Machine Expert, select Tools > Device Repository. |
| 3 | In theDevice Repository window, click the Install ... button. |
| 4 | In the dialog box, navigate to the storage location of the IODD XML file of your IO-Link device, select the XML file and click the Close button. |

## Adding an IO-Link Device to Your Application

| Step | Action |
| --- | --- |
| 1 | In the Devices tree below the TM5SE4IOL node, right-click the first <Empty> slot and select Plug Device from the contextual menu.  NOTE: If you right-click a slot that is not <Empty>, the IO-Link device already assigned to this slot is replaced by the new IO-Link device to be added. |
| 2 | In the Plug Device window, select <All Vendors> from the Vendor listbox. |
| 3 | Select the IO-Link device you want to add and click the Plug Device button. |

NOTE: To modify parameter values of an IO-Link device, double-click the corresponding device entry below the TM5SE4IOL node in the Devices tree. Refer to [IO-Link Devices - Parameter Configuration](Devices-ParameterConfiguration-0829F7EF.html) for details.

## Removing an IO-Link Device From Your Application

| Step | Action |
| --- | --- |
| 1 | Right-click the IO-Link device to be removed from your application and select Delete from the contextual menu. |
