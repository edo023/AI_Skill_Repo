# Sensor Type and Channel Deactivation

Sensor Type and Channel Deactivation

The block is designed for various sensor types. The sensor type must be specified because of the different adjustment values. To save time, individual channels can be switched off.

This table shows the code corresponding sensor types:

| Sensor Types | Digital Value | Temperature °C (°F) | Resolution °C (°F) |
| --- | --- | --- | --- |
| Sensor type N | -2700...2372 | -270...1300 (-454...2372) | 0.1 (0.18) |
| Sensor type R | -500...1664 | -50...1664 (-58...3027) | 0.1 (0.18) |
| Sensor type J | -2200...12000 | -220...1200 (-364...2192) | 0.1 (0.18) |
| Sensor type K | -2700...13720 | -270...1372 (-454...2501) | 0.1 (0.18) |
| Sensor type S | -500...17680 | -50...1768 (-58...3214) | 0.1 (0.18) |
| Measurement range of ±32.767 mV | -32768...32767 | – | 1 µV |
| Measurement range of ±65.534 mV | -32768...32767 | – | 2 µV |
