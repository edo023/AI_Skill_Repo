Status LEDs

The following figure shows the status LEDs of the TM7BDI16B block:

![G-SE-0006623.2.gif](images/G-SE-0006623.2.gif)

1   TM7 bus status LEDs, set of two LEDs: 1.1 (green) and 1.2 (red)

2   Channel LEDs, composed of sixteen LEDs (green)

3   Block status LEDs, set of two LEDs: 3.1 (green) and 3.2 (red)

The table below provides the TM7 bus status LEDs of the TM7BDI16B block:

| TM7 bus status LEDs | | Description |
| --- | --- | --- |
| LED 1.1 | LED 1.2 |
| OFF | OFF | No power supply on TM7 bus |
| ON | ON | TM7 bus in preoperational state:  opower supply on TM7 bus and  oblock not initialized |
| ON | OFF | TM7 bus in operational state |
| OFF | ON | TM7 bus error detected |

The table below provides the input status LEDs of the TM7BDI16B block:

| Channel LEDs | State | Description |
| --- | --- | --- |
| 1 to 16 | OFF | Corresponding input deactivated |
| 1 to 16 | ON | Corresponding input activated |
