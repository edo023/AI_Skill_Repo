# TM7 Power Distribution Block (PDB)

![](images/logo.png)

TM7 Power Distribution Block (PDB)

Introduction

The TM7SPS1A expansion block is a 24 Vdc power distribution block for internal I/O supply.

For further information, refer to [TM7SPS1A TM7 Power Distribution Block](../../../../../../api/crossBook?lang=en-US&virtualBookName=m258pig&topicID=D_SE_0009375_1).

I/O Configuration

There is no parameter configuration for this block.
