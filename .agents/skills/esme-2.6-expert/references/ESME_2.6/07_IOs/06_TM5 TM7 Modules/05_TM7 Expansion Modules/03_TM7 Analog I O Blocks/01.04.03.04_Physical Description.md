Introduction

The TM7 System consists of IP67 I/O blocks along with field bus, expansion, sensor/actuator and power cables.

A TM7 I/O block can be a:

odigital I/O block, [for details, refer to the Modicon TM7 Digital I/O Blocks Hardware Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=tm7diohw&topicID=D_SE_0008132_9)

oanalog I/O block

oPower Distribution Block (PDB), [for details, refer to the chapter TM7SPS1A Power Distribution Block (PDB)](../../../../../../api/crossBook?lang=en-US&virtualBookName=m258pig&topicID=D_SE_0009375_1) in the [Modicon TM5 / TM7 Flexible System - System Planning and Installation Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=m258pig&topicID=D_SE_0001952_9)
