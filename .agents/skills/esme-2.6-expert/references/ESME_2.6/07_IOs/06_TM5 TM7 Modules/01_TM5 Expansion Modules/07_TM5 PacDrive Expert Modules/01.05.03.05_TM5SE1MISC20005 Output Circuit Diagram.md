# TM5SE1MISC20005 Output Circuit Diagram

![](images/logo.png)

TM5SE1MISC20005 Output Circuit Diagram

Output Circuit

The output circuit is a symmetrical +5 Vdc, with pull-down resistors on the channels 1-3 and pull-up resistors on the primes of channels 1-3.

Connection Options

Wiring of the incremental encoder output:

| Channel | Signal | Meaning |
| --- | --- | --- |
| 0 (output) | A0 | Output state of counter output A |
| 1 (output) | B0 | Output state of counter output B |
| 2 (output) | R0 | Output state of reference pulse R |
| 3 (input) | DI0 | Digital input (24 V asymmetrical) |
| 4 (input) | DI1 | Digital input (24 V asymmetrical) |
