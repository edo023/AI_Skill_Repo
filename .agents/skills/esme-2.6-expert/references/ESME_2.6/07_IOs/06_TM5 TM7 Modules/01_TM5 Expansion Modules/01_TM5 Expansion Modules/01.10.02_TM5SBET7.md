# TM5SBET7

## Introduction

The TM5SBET7 expansion electronic module is a Transmitter electronic module which transmits the TM7 data bus and provides the TM7 power bus to the TM7 expansion I/O blocks.

For further information, refer to [TM5SBET7 Transmitter Electronic Module](../../../../../api/crossBook?lang=en-US&virtualBookName=tm5bushw&topicID=D_SE_0011044).

## TM5 Module I/O Mapping Tab

Variables can be defined and named in the TM5 Module I/O Mapping tab. Additional information such as topological addressing is also provided in this tab.

Refer [Status Mapping](#D-SE-0011041__D-SE-0011041.10), for the status bits configuration details.

For further generic descriptions, refer to [TM5 Module I/O Mapping Tab Description](D-SE-0005771.html#D-SE-0005771__D-SE-0005771.9).
