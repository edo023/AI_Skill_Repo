# StainGauge_Error: Error Codes

![](images/logo.png)

StainGauge\_Error: Error Codes

Enumerated Type Description

The StainGauge\_Error enumeration data type contains the following values:

| Enumerator | Value | Description |
| --- | --- | --- |
| CALIBRATION\_OK | 00 hex | Indicates that the measure is valid. |
| OVERFLOW\_VALUE | 01 hex | Indicates that the average raw value is above of the maximum value. |
| UNDERFLOW\_VALUE | 02 hex | Indicates that the average raw value is below of the minimum value. |
| INVALID\_VALUE | 03 hex | Indicates that the average raw value is invalid or the module is busy. |
| INVALID\_CYCLETIME | 04 hex | Indicates that the function block input Cycle\_number = 0. |
| MULTIPLE\_COMMAND | 05 hex | Indicates that 2 of the function block inputs Taring\_enable, Ref1\_enable or Ref2\_enable are set to 1. |
| INVALID\_CALIBRATION\_PARAMETERS | 06 hex | Indicates that the [calibration parameters](../TM5SEAISG_Programming/TM5SEAISG_Programming-8.htm#XREF_D_SE_0021072_1) are invalid. |
| INVALID\_MODULE\_REF | 07 hex | Indicates that the the module reference on the Module\_Ref input is incorrect. |
