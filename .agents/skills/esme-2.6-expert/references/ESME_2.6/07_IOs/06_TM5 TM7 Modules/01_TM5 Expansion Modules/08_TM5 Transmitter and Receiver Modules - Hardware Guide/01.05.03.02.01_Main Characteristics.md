Main Characteristics

The table below describes the main characteristics of the TM5SBET7 electronic module:

| Main Characteristics | |
| --- | --- |
| Function | Transmits the TM7 data bus and provides the TM7 power bus to the TM7 expansion I/O blocks. |
| Maximum bus length | 2500 m (8202.1 ft) |
| Maximum cable distance between Transmitter and Receiver | 100 m (328 ft) |
