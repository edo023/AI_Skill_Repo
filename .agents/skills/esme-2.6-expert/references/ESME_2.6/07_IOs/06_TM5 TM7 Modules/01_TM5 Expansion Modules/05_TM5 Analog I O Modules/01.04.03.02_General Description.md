# General Description

General Description

Introduction

The range of analog electronic modules includes:

• [Analog input](../glossary/glossary.htm#XREF_D_SE_0024697_624) electronic modules

• Analog temperature input electronic modules

• [Analog output](../glossary/glossary.htm#XREF_D_SE_0024697_625) electronic modules

The TM5 analog I/O electronic modules need to be associated with a bus base and a [terminal block](../glossary/glossary.htm#XREF_D_SE_0024697_420). Every analog electronic module channel has a status [LED](../glossary/glossary.htm#XREF_D_SE_0024697_302).

Analog Input Electronic Module Features

Analog input electronic modules convert measured values (voltages, currents) into numeric values, which can be processed by the controller.

Each channel in an electronic module is capable of converting either current or voltage signals. There are different terminal connection points for each type of signal. Since current and voltage require different adjustment values, it is also necessary to configure the desired type of input signal:

o-10...+10 Vdc voltage signal (default).

o0...20 mA current signal.

o4...20 mA current signal.

Analog Temperature Electronic Module Features

Temperature measurement values are converted into numeric values which can be processed by the controller using temperature electronic modules. For temperature measurements, the temperature electronic module returns the measured value in 0.1°C steps.

The step value 0.1°C (0.18°F) is supported as standard by all temperature electronic modules.
