# StrainGaugeParameter: Calibration Parameters

![](images/logo.png)

StrainGaugeParameter: Calibration Parameters

Enumerated Type Description

The StrainGaugeParameter enumeration data type contains the following values:

| Enumerator | Value | Description |
| --- | --- | --- |
| Calibrate\_Tare | DINT | Offset of the calibrated value. |
| Calibrate\_Ref1 | DINT | Calibrated value at reference point 1 of the calibration line. |
| Calibrate\_Ref2 | DINT | Calibrated value at reference point 2 of the calibration line. |
| Raw\_Ref1 | REAL | Average raw value at reference point 1 of the calibration line. |
| Raw\_Ref2 | REAL | Average raw value at reference point 2 of the calibration line. |
