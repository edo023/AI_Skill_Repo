Ordering Information

The following illustration shows the slice with a TM5SE1SC10005:

![G-SE-0000922.2.gif-high.gif](images/G-SE-0000922.2.gif-high.gif)

The table below shows the model numbers for the terminal block and bus base associated with the TM5SE1SC10005:

| Number | Model Number | Description | Color |
| --- | --- | --- | --- |
| 1 | TM5ACBM11  or  TM5ACBM15 | Bus base    Bus base with address setting | White    White |
| 2 | TM5SE1SC10005 | Electronic module | White |
| 3 | TM5ACTB12 | Terminal block, 12 pins | White |
