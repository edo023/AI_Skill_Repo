# Main Characteristics

Main Characteristics

The table below describes the main characteristics of the TM5SAI6TH module:

| Main Characteristics | | |
| --- | --- | --- |
| Number of input channels | 6 | |
| Measurement type | Temperature | Voltage |
| Input sensor type | J, K, N and S thermocouple sensors |  |
| Input range | Type J: -210...1200°C (-346...2192°F)  Type K: -270...1372°C (-454...2501°F)  Type N: -270...1300°C (-454...2372°F)  Type S: -50...1768°C (-58...3214°F)  1 µV per bit(1)  2 µV per bit(1)  Sensor type R(1)  Sensor type B(1) | ± 65.534 mV |
| Resolution | 16-bit | |
| (1)   Designed for use with Sercos III (TM5NS31) and EtherNet/IP (TM5NEIP1). | | |

The thermocouple modules are configured as a whole for the same type of thermocouple sensor. You cannot mix thermocouple sensor types on the same module, otherwise the temperature readings will not be correct.
