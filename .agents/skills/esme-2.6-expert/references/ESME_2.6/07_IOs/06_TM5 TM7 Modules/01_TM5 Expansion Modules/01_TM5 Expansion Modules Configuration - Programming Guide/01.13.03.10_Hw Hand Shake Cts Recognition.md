# Hw Hand Shake Cts Recognition

## General

This register configures the hardware handshake line CTS. Correct wiring to the outstation at an active CTS prompt must be observed.

| Name | Meaning |
| --- | --- |
| Hw Hand Shake Cts Recognition | 0 / disabled = CTS line is ignored; data can be sent.  1 / flow control = CTS line active and is used for flow control and transmission release from the outstation. |
