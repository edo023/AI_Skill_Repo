# TM5SDI6D Presentation

## Main Characteristics

The table below describes the main characteristics of the TM5SDI6D electronic module:

| Main Characteristics | |
| --- | --- |
| Number of input channels | 6 |
| Input type | Type 1 |
| Signal type | Sink |
| Rated input voltage | 24 Vdc |
