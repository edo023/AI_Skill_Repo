# TM5SBER2

## Introduction

The TM5SBER2 expansion electronic module is a Receiver electronic module which receives the TM5 data bus.

For further information, refer to [TM5SBER2 Receiver Electronic Module](../../../../../api/crossBook?lang=en-US&virtualBookName=tm5bushw&topicID=D_SE_0002828).

## TM5 Module I/O Mapping Tab

Variables can be defined and named in the TM5 Module I/O Mapping tab. Additional information such as topological addressing is also provided in this tab.

Refer [Status Mapping](#D-SE-0005871__D-SE-0005871.10), for the status bits configuration details.

For further generic descriptions, refer to [TM5 Module I/O Mapping Tab Description](D-SE-0005771.html#D-SE-0005771__D-SE-0005771.9).
