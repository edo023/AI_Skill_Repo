# Overview

Overview

The analog 4AI ±10 V / 0-20 mA electronic module is equipped with 4 12-bit inputs.

Each channel is capable of converting either current or voltage. There are different terminal connection points for each type of signal.

It is necessary to configure the desired type of input signal:

o-10...+10 Vdc voltage signal (default)

o0...20 mA current signal
