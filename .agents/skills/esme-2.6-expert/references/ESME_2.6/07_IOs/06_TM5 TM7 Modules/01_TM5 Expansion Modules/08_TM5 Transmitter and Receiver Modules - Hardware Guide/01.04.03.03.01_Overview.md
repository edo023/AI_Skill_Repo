# Overview

Overview

The TM5 System provides a virtual rack system through a decentralized backplane. The decentralized backplane contains a local configuration and several remote configurations connected together using specific expansion bus cables. The TM5 Transmitter and Receiver electronic modules handle the communication between remote electronic modules via expansion bus cables.

The TM5 Transmitter and Receiver electronic modules need to be associated with a [bus base](../glossary/glossary.htm#XREF_D_SE_0024697_644) and a terminal block.
