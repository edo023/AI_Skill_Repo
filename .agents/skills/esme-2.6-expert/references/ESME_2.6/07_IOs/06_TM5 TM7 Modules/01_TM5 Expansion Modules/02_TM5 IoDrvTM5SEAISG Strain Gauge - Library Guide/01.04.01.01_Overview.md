# Overview

Overview

The full-bridge strain gauge sensor provides an electrical signal that the TM5SEAISG module converts to a raw numerical value. The raw value is then processed with the use of the [StrainGa­ugeExt function block](../TM5SEAISG_Programming/TM5SEAISG_Programming-4.htm#XREF_D_SE_0032982_25) which provides a calibrated value.

This function block has 3 functions:

omake an average measure of the TM5SEAISG input in a defined period

odefine a linear calibration to match the needs of your process

oprovide a calibrated mesure

NOTE: The Strain Gauge function block is not automatically declared when you add TM5SEAISG to the controller.

The average raw value is calculated with all the measures done by the TM5SEAISG module during a define number of [task](../glossary/glossary.htm#XREF_D_SE_0024697_175) cycles. The number of task cycles is set with the Cycle\_number input of the function block.
