# TM5SDI2DF Presentation

## Main Characteristics

The table below describes the main characteristics of the TM5SDI2DF electronic module:

| Main Characteristics | |
| --- | --- |
| Number of input channels | 2 |
| Input frequency | 50 kHz maximum |
| Input function | Function event counter / gate measurement |
| Input type | Type 1 |
| Signal type | Sink |
| Rated input voltage | 24 Vdc |
