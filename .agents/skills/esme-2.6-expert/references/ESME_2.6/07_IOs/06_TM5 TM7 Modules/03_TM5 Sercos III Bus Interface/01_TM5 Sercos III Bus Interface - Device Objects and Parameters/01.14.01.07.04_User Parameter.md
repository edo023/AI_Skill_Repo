# User Parameter

## General

The change over between numerical and symbolical value takes place via the check box Symbolic values.

NOTE: Modifications to the parameter CentralControl\_DigitalOutput\_xx only become effective after a restart of the controller or after phasing up of Sercos.

| Parameter | Meaning | Value symbolical (numerical) | Default value | Data type |
| --- | --- | --- | --- | --- |
| CentralControl\_DigitalOutput\_01 | Specifies how the output channel can be addressed by the application.  **Central**  The output channel can be addressed directly by the application. The signal 'DigitalOutput\_01' is available in I/O mapping.  **Direct**  The output channel cannot be addressed directly by the application. The signal 'DigitalOutput\_01' is not available in I/O mapping. Addressing the output channel by the application is only possible via the communication channels from application logic controller to the Safety Logic Controller. | Central/Direct (0/1) | Central (0) | BYTE |
| CentralControl\_DigitalOutput\_ 02 | Specifies how the output channel can be addressed by the application.  **Central**  The output channel can be addressed directly by the application. The signal 'DigitalOutput\_02' is available in I/O mapping.  **Direct**  The output channel cannot be addressed directly by the application. The signal 'DigitalOutput\_02' is not available in I/O mapping. Addressing the output channel by the application is only possible via the communication channels from application logic controller to the Safety Logic Controller. | Central/Direct (0/2) | Central (0) | BYTE |
| CentralControl\_DigitalOutput\_0102 | Specifies how the output channel can be addressed by the application.  **Central**  The output channel can be addressed directly by the application. The signal 'DigitalOutput\_0102' is available in I/O mapping.  **Direct**  The output channel cannot be addressed directly by the application. The signal 'DigitalOutput\_0102' is not available in I/O mapping. Addressing the output channel by the application is only possible via the communication channels from application logic controller to the Safety Logic Controller. | Central/Direct (0/4) | Central (0) | BYTE |
