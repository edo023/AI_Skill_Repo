General

To generate an encoder signal with the Incremental Encoder Output object, the object must be coupled to a velocity source in the application.

Initialization of the Encoder Output

| Step | Action |
| --- | --- |
| 1 | Set the WorkingMode parameter of the parent bus coupler to real  / 1. |
| 2 | Perform a Sercos phase up. |
| 3 | Set the OutputsActiveSet parameter of the parent bus coupler to on / 1. |
| 4 | Couple the Incremental Encoder Output object to a producing encoder as source. |

The encoder output module behaves like an encoder as soon as the startup procedure is finished.

Coupling of the Incremental Encoder Output Object and the Virtual Master Encoder

The coupling of the Incremental Encoder Output object must be done with the FC\_SetMaster­Encoder() SystemInterface function as follows:

diRetSetMasterEnc := FC\_SetMasterEncoder(

i\_stLEncId := IME\_EncOut.stLogicalAddress,

i\_stMasterId := VME\_Enc.stLogicalAddress);

After this call, the virtual master encoder VME\_Enc and the Incremental Encoder Output object are coupled via velocity.
