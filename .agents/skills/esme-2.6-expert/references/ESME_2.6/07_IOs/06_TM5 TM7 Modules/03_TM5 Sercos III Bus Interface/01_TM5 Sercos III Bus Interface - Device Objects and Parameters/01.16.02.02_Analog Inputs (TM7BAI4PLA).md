# Analog Inputs (TM7BAI4PLA)

## General

The converted analog values are output by the module as temperature 1 to temperature 4. Consider the following so that a defined output value is available.

* Until the first conversion, 16#8000 is output.
* After the switchover of the sensor type, 16#8000 is output until the first conversion.
* If the input is not switched on, 16#8000 is output.
* For the determination of the measured temperature, at least one terminal temperature sensor is required for J, K, S, N, and R thermocouple sensors. Otherwise, 16#7FFF is output generally.

## Raw Value Measurement

The raw value measurement works with and without the terminal temperature measurement.

If a different sensor type than J, K, S, N, and R is used, the terminal temperature has to be measured at least at one input.

By using this value, you have to perform a terminal temperature compensation.

## Temporal Coordination

The measured value recording is coordinated temporally via the converter hardware.

Each conversion cycle, the switched on inputs are converted. In addition, the terminal temperature is measured.

Unneeded inputs can be switched off ([DisabledChannel0x](D-SE-0069175.html#D-SE-0069175__D-SE-0069175.3)), whereby the refresh time is reduced. The shutdown can be temporarily.

The measurement of the terminal temperature cannot be switched off.

The savings per input depend on the filter time.

| Filter | Filter time | Saving per input | Digital transducer resolution |
| --- | --- | --- | --- |
| 50 Hz | 20 ms | 75 ms | 16 bit |
| 60 Hz | 16.67 ms | 65 ms | 16 bit |
| 250 Hz | 4 ms | 27 ms | 13 bit |
| 500 Hz | 2 ms | 21 ms | 10 bit |
