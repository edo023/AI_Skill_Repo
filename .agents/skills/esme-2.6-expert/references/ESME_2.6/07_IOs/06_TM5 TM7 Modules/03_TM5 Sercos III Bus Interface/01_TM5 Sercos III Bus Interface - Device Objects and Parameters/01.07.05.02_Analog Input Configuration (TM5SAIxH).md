# Analog Input Configuration (TM5SAIxH)

## General

| Bit | Meaning |
| --- | --- |
| 0 | Terminal selector  0 = Voltage terminal for ±10 Vdc  1 = Current terminal for 0...20 mA |
| 1 | Gain selector  0 = Voltage ±10 Vdc  1 = Current 0...20 mA |
| 2 | Reserved |
| 3 | Reserved |
| 4 | Filtering active (only if bit 7 = 0)  0 = Inactive  1 = Active |
| 5 | Minimum / maximum analysis (only if bit 7 = 0)  0 = Inactive  1 = Active |
| 6 | Error monitoring active (only if bit 7 = 0)  0 = Inactive  1 = Active |
| 7 | Enable input  0 = Input enabled  1 = Input disabled |
