# Upper or Lower Limit Value

## General

The default values are 0x8001 (-32767) and 0x7fff (+32767). For current measurements, the value is limited to a minimum of 0.

From the point in time one of the two registers is written on, and if the limit value is exceeded or fallen short of, the corresponding error status bit is set and a limitation to the respective limit value is executed.

Measured value < lower limit or measured value > upper limit.

This also applies for the current measurement which then allows negative values, for example if 4...20 mA is set by currents < 4 mA (for example, 0 mA equate -8192).
