# General

## Description

Two analog outputs ±10 V / 0...20 mA, resolution 12

Output module 24 Vdc with 2x12 bit analog outputs (voltage ±10 Vdc / current 0...20 mA, differential)

## Configuration Register

| Register | Meaning | Default | Data type |
| --- | --- | --- | --- |
| 0 | Output 01 | ±10 Vdc | INT |
| 2 | Output 02 | ±10 Vdc | INT |
| 18 | [Channel type (TM5SAOxL)](D-SE-0070399.html#D-SE-0070399) | 0 | USINT |

For further information, see the TM5 analog I/O modules hardware manual.
