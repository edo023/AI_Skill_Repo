# Functional Description

Functional Description

A synchronization error detection on coupled machines is recognized in the following cases:

o8601 Master Encoder signal out of range (SinCos master encoder)

o8782 Master Encoder signal out of range (incremental encoder input)

o8522 C2C encoder error detected (C2C encoder input)

The parameter ErrorDeceleration can bring the system to a standstill with a defined velocity deceleration. When a track error is detected, the velocity is set to 0 with the defined deceleration.

If 0 is entered in ErrorDeceleration, then no velocity deceleration is created by the system.
