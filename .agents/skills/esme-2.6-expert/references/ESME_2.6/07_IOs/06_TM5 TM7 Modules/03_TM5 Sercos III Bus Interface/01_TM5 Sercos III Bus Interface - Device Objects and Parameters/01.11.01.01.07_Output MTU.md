# Output MTU

## General

This register configures the number of bytes available in the cyclic output data area. This output MTU area is used to transfer data to be sent and control bytes to the module. The output MTU size is not related to the permitted segment size or the actual serial frame size; it just defines the size of the transfer area on the bus.

The size of the output MTU is set to 15 bytes.
