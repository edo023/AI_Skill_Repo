# Channel Type for Analog Outputs 1–2 (TM5SAO2L)

## General

The selection voltage (0 = Default) or current (1) is required due to the different reference values for current and voltage.

| Bit | Description |
| --- | --- |
| 0 | 0 = Output 01: voltage signal -10...+10 Vdc  1 = Output 01: current signal 0...20 mA |
| 1 | 0 = Output 02: voltage signal -10...+10 Vdc  1 = Output 02: current signal 0...20 mA |
| 2–7 | 0 |

| Selection current/voltage | Output |
| --- | --- |
| 0000000x | 01 |
| 000000x0 | 02 |
