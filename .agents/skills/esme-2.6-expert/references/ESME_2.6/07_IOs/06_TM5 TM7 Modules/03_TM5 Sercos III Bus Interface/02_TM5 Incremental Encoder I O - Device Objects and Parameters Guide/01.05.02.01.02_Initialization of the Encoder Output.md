# Initialization of the Encoder Output

Initialization of the Encoder Output

| Step | Action |
| --- | --- |
| 1 | Set the WorkingMode parameter of the parent bus coupler to real  / 1. |
| 2 | Perform a Sercos phase up. |
| 3 | Set the OutputsActiveSet parameter of the parent bus coupler to on / 1. |
| 4 | Couple the Incremental Encoder Output object to a producing encoder as source. |

The encoder output module behaves like an encoder as soon as the startup procedure is finished.
