# TM5CableLength

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Bus Interface TM5NS31 |
| Traceable | Yes |

## Functional Description

With this command, you can enter the cable length [in meters] for the TM5 bus.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0–25300  (default: 0) | UINT | Cable length of the TM5 bus [m] |
