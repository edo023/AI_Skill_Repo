# Digital Input Filter

## General

Used only as asynchronous configuration register (see also [Configuration of the Modules](D-SE-0068182.html#D-SE-0068182)).

The filter value can be adjusted in steps of 0.1 ms. Since the input signals are sampled at 0.2 ms intervals, it is a logical choice to enter the values in steps of 2. The value can range from 0 (no filter) to 250 (25 ms). At cycle times < 0.15 ms, filtering is switched off.

| Value | Filter |
| --- | --- |
| 0 | No filter |
| 2 | 0.2 ms |
| 4 | 0.4 ms |
| … | … |
| 250 | 25 ms (higher values are limited to this value) |
