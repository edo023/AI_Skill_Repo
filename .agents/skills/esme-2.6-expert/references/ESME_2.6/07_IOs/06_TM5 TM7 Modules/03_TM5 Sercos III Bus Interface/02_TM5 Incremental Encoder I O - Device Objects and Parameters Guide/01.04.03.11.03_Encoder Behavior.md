# Encoder Behavior

Encoder Behavior

Encoder behavior when ErrorDeceleration = 0 and a track error has been detected:

oSinCos Encoder Input (physical master encoder): The velocity remains constant.

oIncremental Encoder Input: The velocity follows the received incremental encoder input signals. If the connection to the encoder is interrupted, the velocity is reduced to 0.

oSynchronization encoder output (encoder network): The last known velocity is maintained.

oC2C Encoder Input: The last known velocity is maintained.
