# Minimum Cycle Time (Analog Module)

## General

The minimum cycle time is the minimum time needed for the TM5-bus cycle to be shut down without a communication error or function-impairments occurring. Please note that the remaining time to deal with monitoring, diagnostic and anti-cyclical commands is reduced by very fast cycles.

The cycle time can be set at the bus interface. Least cycle time = 250 µs.

The jitter free output of the analog values at the beginning of every cycle is only possible if the I/O cycle times are ≥ 400 µs. If the I/O cycle times are < 400 µs, the output of the analog values might happen one cycle later due to the internal calculation time.
