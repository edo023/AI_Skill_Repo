# Resolution

![](images/logo.png)

Resolution

General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC Pro |
| Traceable | No |

Functional Description

Specifies the resolution of the incremental encoder output.

NOTE: Modify the Resolution parameter value at encoder standstill and when the program is inactive. Otherwise, the [velocity signal](EncoderOutput-13.htm#XREF_D_SE_0069682_1) appears incorrect, which can cause the detection of other errors.

Example of Calculating Maximum Resolution

Motor with 6,000 revolutions/minute -> 100 revolutions/second

Maximum resolution = 1,000,000 increments/second / 100 revolutions/second =10,000 increments/revolution

NOTE: Maintain a reserve of at least 10 % for the calculated maximum resolution; otherwise you could receive the diagnostic message 8780 "Encoder output frequency > 1 MHz".

The ZeroTrack is generated after each revolution. For details on the ZeroTrack limitation, refer to [Limitation of ZeroTrack Generation on High Speeds](../Limitations/Limitations.htm#XREF_D_SE_0073875_4).
