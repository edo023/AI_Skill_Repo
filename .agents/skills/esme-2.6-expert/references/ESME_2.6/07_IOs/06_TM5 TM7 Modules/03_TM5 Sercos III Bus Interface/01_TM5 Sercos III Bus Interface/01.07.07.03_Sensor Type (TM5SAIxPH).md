| 8–11 (1) | Input 03:  0000 = Reserved  0001 = Reserved  0010 = Sensor type PT100  0011 = Sensor type PT1000  0100 = Channel inactive  0101 = Resistance measurement 0.1...4500 μ  0110 = Resistance measurement 0.05...2250 μ  0111 = Channel inactive | –  2  2  4  2  –  2  4  – |
| 12–15 (1) | Input 04:  0000 = Reserved  0001 = Reserved  0010 = Sensor type PT100  0011 = Sensor type PT1000  0100 = Channel inactive  0101 = Resistance measurement 0.1...4500 μ  0110 = Resistance measurement 0.05...2250 μ  0111 = Channel inactive | –  2  2  4  2  –  2  4  – |
