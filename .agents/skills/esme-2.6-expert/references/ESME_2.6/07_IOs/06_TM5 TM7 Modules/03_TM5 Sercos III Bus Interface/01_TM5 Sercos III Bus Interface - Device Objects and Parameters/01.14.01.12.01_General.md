# General

## Description

Safety digital input module, 24 Vdc

* Eight safety-related inputs
* Two non-safety-related digital inputs
* Two safety-related test outputs
* Two non-safety-related digital outputs
* Configurable input filter, 0...500 ms

## Configuration

| Tab | Meaning |
| --- | --- |
| [TM7 Module I/O Mapping](D-SE-0070448.html#D-SE-0070448) | Mapping of channels to project variables |
| [TM7 Module Parameters](D-SE-0070449.html#D-SE-0070449) | Safety-related configuration parameters |

For further information, see the TM5/TM7 Digital I/O Safety Modules Hardware Guide.
