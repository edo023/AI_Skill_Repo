# Number of I/Os

## Sercos III Restriction

The Sercos III master verifies during phase up (Sercos processing from phase 0 to phase 4) whether the configured Sercos III devices exceed the maximum amount of data.

If the amount of data exceeds the maximum, a message logger entry is written and the Sercos phase up is canceled.

## TM5 Bus Interface (TM5NS31) Restrictions

The TM5 bus interface (TM5NS31) restricts the number of modules connected based upon the cycle time of the bus and the data length of the individual modules attached thereto. The controller provides a message logger entry if this maximum is determined to be exceeded.
