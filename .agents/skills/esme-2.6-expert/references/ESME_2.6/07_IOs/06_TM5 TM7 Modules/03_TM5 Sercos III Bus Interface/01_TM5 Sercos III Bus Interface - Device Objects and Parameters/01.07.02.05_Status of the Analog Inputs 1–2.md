# Status of the Analog Inputs 1–2

## General

| Bit | Description |
| --- | --- |
| 0–1 | Input 1  00 = No detected error  01 = Lower limit value exceeded  10 = Upper limit value exceeded  11 = Wire break (not with current) |
| 2–3 | Input 2  00 = No detected error  01 = Lower limit value exceeded  10 = Upper limit value exceeded  11 = Wire break (not with current) |
| 4–7 | 0 |
