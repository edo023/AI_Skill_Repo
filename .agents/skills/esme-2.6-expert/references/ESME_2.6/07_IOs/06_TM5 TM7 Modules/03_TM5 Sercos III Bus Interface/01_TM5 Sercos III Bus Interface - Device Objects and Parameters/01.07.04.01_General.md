# General

## Description

Two thermocouple temperature inputs

Analog temperature input module 24 Vdc with 2x16 bit thermocouple inputs (thermocouple J/K/N/S)

## Configuration Register

| Register | Meaning | Default | Data type |
| --- | --- | --- | --- |
| 0 | Input 01 | Inactive | INT |
| 2 | Input 02 | Inactive | INT |
| 12 | Terminal temperature 1 | – | INT |
| 14 | Terminal temperature 2 | – | INT |
| 24 | [Filter time/conversion rate (TM5SAIxPH, TM5SAIxTH)](D-SE-0070375.html#D-SE-0070375) | 3 | USINT |
| 26 | [Sensor type (TM5SAIxTH)](D-SE-0070376.html#D-SE-0070376) | 0 | USINT |
| 27 | [Disable/enable inputs 1–2](D-SE-0070377.html#D-SE-0070377) | 0 | USINT |
| 28 | I/O cycle counter | Ein | USINT |
| 30 | [Status of the analog inputs 1–2](D-SE-0070378.html#D-SE-0070378) | – | USINT |

For further information, see the TM5 analog I/O modules hardware manual.
