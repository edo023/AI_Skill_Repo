# General

## Description

Four analog inputs ±10 V / 0...20 mA, resolution 12 bit

Analog input module 24 Vdc with 4x12 bit analog inputs (voltage ±10 Vdc / current 0...20 mA or 4...20 mA, differential)

## Configuration Register

| Register | Meaning | Default | Data type |
| --- | --- | --- | --- |
| 0 | Analog input 01 | ±10 Vdc | INT |
| 2 | Analog input 02 | ±10 Vdc | INT |
| 4 | Analog input 03 | ±10 Vdc | INT |
| 6 | Analog input 04 | ±10 Vdc | INT |
| 16 | [Analog input filter](D-SE-0070379.html#D-SE-0070379) | Filter disabled, no limitation | USINT |
| 18 | [Channel type for analog inputs 1–4](D-SE-0070380.html#D-SE-0070380) | 0 | USINT |
| 20 | Lower limit value channel 1–4 ([Upper or lower limit value](D-SE-0070381.html#D-SE-0070381)) | -32767 | INT |
| 22 | Upper limit value channel 1–4 ([Upper or lower limit value](D-SE-0070381.html#D-SE-0070381)) | 32767 | INT |
| 30 | [Status of analog inputs 1–4](D-SE-0070382.html#D-SE-0070382) | – | USINT |

For further information, see the TM5 analog I/O modules hardware manual.
