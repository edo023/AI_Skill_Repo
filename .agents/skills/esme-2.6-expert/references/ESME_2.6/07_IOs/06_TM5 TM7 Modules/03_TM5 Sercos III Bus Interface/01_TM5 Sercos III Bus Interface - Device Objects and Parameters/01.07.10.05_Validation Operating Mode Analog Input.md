# Validation Operating Mode Analog Input

## General

Using this register, the validation operating mode of an analog output is configured.

Options per output:

* Validation timer data type: general selection 16 bits or 32 bits
* Validation window: within the data type, the maximum permissible value can be limited further
* Timer assignment: a separate timer is provided for every output. However, it is also possible to configure the outputs with the same validation timer. In this case, the data type and window settings must be the same in the TimeMode registers of the outputs.

| Bit | Description |
| --- | --- |
| 0–4 | Validation window [2 n μs]  0 = Disabled, output is not switched off  1...2 μs  2...4 μs  3...8 μs  ...  15...32786 μs  ...  31...2147483648 μs |
| 5–7 | 0 |
| 8 | Validation timer:  0 = Timer 1 is used with preset NetTime  1 = Timer 2 is used with preset NetTime |
| 9–14 | 0 |
| 15 | Validation timer data type:  0...16 bit  1...32 bit |
