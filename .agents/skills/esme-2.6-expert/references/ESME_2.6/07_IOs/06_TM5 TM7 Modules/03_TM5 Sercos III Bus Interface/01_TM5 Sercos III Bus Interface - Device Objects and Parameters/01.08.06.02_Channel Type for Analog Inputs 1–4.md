# Channel Type for Analog Inputs 1–4

## General

The selection voltage (0 = Default) or current (1) is required due to the different reference values for current and voltage.

| Bit | Nibble | Description |
| --- | --- | --- |
| 0 | Low-order nibble (4 bit) | 0 = Channel 1: voltage signal  1 = Channel 1: current signal |
| 1 | 0 = Channel 2: voltage signal  1 = Channel 2: current signal |
| 2 | 0 = Channel 3: voltage signal  1 = Channel 3: current signal |
| 3 | 0 = Channel 4: voltage signal  1 = Channel 4: current signal |
| 4 | Higher-quality nibble (4 bit) | 0 = Channel 1: 0...20 mA current signal |
| 5 | 0 = Channel 2: 0...20 mA current signal  1 = Channel 2: 4...20 mA current signal |
| 6 | 0 = Channel 3: 0...20 mA current signal  1 = Channel 3: 4...20 mA current signal |
| 7 | 0 = Channel 4: 0...20 mA current signal  1 = Channel 4: 4...20 mA current signal |

NOTE: By setting the corresponding bit in the higher-quality nibble, the selection of the 4...20 mA range is possible for the current pre-selection.

General description for the bit 7-0 assignment

| Selection current/voltage  Bit 7...Bit 0 | Meaning |
| --- | --- |
| 0000000x | Configure the output 01 in the low-order nibble:  Bit 0 is set to 0: Voltage signal  Bit 0 is set to 1: Current pre-selection |
| 000000x0 | Configure the output 02 in the low-order nibble:  Bit 1 is set to 0: Voltage signal  Bit 1 is set to 1: Current pre-selection |
| 000x0001 | Configure the output 01 by the current pre-selection in the higher-quality nibble:  Bit 4 is set to 0: 0x01...0...20 mA  Bit 4 is set to 1: 0x11...4...20 mA |
| 00x00010 | Configure the output 02 by the current pre-selection in the higher-quality nibble:  Bit 5 is set to 0: 0x02...0...20 mA  Bit 5 is set to 1: 0x22...4...20 mA |

Examples for the bit 7-0 assignment

| Selection current/voltage  Bit 7 ... Bit 0 | Meaning |
| --- | --- |
| 00000000 | Output 01 to 04: Configured voltage signal +/–10 Vdc |
| 00000001 | Output 01: Configured current signal 0...20 mA  Output 02 to 04: Configured voltage signal +/–10 Vdc |
| 00010001 | Output 01: Configured current signal 4...20 mA  Output 02 to 04: Configured voltage signal +/–10 Vdc |
| 00000110 | Output 02 and 03: Configured current signal 0...20 mA  Output 01 and 04: Configured voltage signal +/–10 Vdc |
| 01100110 | Output 02 and 03: Configured current signal 4...20 mA  Output 01 and 04: Configured voltage signal +/–10 Vdc |
