# General

## Description

4 DI, 2 DA, 1 AI, 1 AO

Four digital inputs 24 Vdc, sink, 1-wire technology, two digital outputs 0.5 A, source, 1-wire technology. One analog input ±10 V or 0...20 m/A or 4...20 m/A, 12-bit resolution. One analog output, ±10 V or 0...20 m/A 12-bit resolution, two counters as event counter or gate measurement.

TM5 expert module, 24 Vdc with eight channels, optional configurable as in- or output via the software:

* Six digital channels, two analog channels optional configurable as in- or output
* Digital and analog input filter
* The outputs are equipped with a single channel diagnostic
* Extensive additional state information

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the device tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the device tree.  **Result:** The device editor is opened and the tabs TM5 Module I/O display, User-defined parameters, and Information are displayed. |

The check box Symbolic values is used for switching between the numeric and the symbolic value.
