# General

## Description

Two analog inputs, two analog outputs, 0...20 mA, resolution 12 bit, LEDs to indicate the status

The analog mixed module 24 Vdc is equipped with two inputs and two outputs with 12-bit digital transducer resolution. The input/output signal is 0...20 mA.

* Two analog inputs, two analog outputs 0...20 mA
* Wirebreak detection at the inputs
* Adjustable digital input filters
* Short cycle time
* Ideal shield discharge on the channels

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM7 Module I/O display, User-defined parameters, and Information are displayed. |

## Tab TM7 Module I/O Display

The input and output channels of the module are mapped to project variables that are used by the application.

| Channel | Meaning | Default value | Data type |
| --- | --- | --- | --- |
| Analog input 1 | [Analog input 1](D-SE-0069862.html#D-SE-0069862) | – | INT |
| Analog input 2 | Analog input 2 | – | INT |
| Analog output 3 | [Analog output 3](D-SE-0069863.html#D-SE-0069863) | – | INT |
| Analog output 4 | Analog output 4 | – | INT |
| DiagIn 1 | [Status of the analog inputs](D-SE-0069864.html#D-SE-0069864) | – | BYTE |

## Tab User-Defined Parameters

The values are set via a selection list.

The check box Symbolic values is used for switching between the numeric and the symbolic value.

| Name | Meaning | Numeric/symbolic value | Default value | Data type |
| --- | --- | --- | --- | --- |
| Input filter | [Filter stages of the analog input filter](D-SE-0069865.html#D-SE-0069865) | 0 / off  1 / level 2  2 / level 4  3 / level 8  4 / level 16  5 / level 32  6 / level 64  7 / level 128 | 0 / off | BYTE |
| InputLimitation | [Input ramp limit](D-SE-0069865.html#D-SE-0069865__D-SE-0069865.5) | 16 / 16383  32 / 8191  48 / 4095  64 / 2047  80 / 1023  96 / 511112 / 255 | 16 / 16383 | BYTE |
