# General

## Description

Counter, 1xSSI, 5 V, 1 Mbit/s

1-channel high speed counter module 1 Mbit/s / 5 Vdc or 24 Vdc with 1x32 bit input channel (sink, 3 wires, SSI absolute encoder)

## Configuration Register

| Register | Meaning | Default |
| --- | --- | --- |
| 0 | CFG channel#0 | 0 |
| 2 | CFG channel#1 | 115 |
| 8 | CFG channel#4 | 0 |
| 10 | CFG channel#5 | 0 |
| 64 | DISPLAY LED#0 source | 32 |
| 66 | DISPLAY LED#1 source | 36 |
| 74 | DISPLAY LED#5 source | 37 |
| 256 | DIRECTIO output clear mask | 0 |
| 258 | DIRECTIO output set mask | 0 |
| 7168 | SSI#0 event ID write | 225 |
| 7176 | SSI#0 config | 0 |

For further information, see the TM5 expert module (high-speed counter) hardware manual.
