# General

## Description

Eight digital outputs, 24 Vdc, 2.0 A, output state re-readable, LEDs to indicate the status.

Digital output module 24 Vdc with eight digital outputs for source output wiring, M8 connection technology

* 8 A total current per module
* The outputs with single channel diagnostic for short circuit or overload
* Short cycle times
* Integrated protection of switching inductances

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM7 Module I/O display, User-defined parameters, and Information are displayed. |

## Tab TM7 Module I/O Display

The input and output channels of the module are mapped to project variables that are used by the application.

| Channel | Meaning | Default value | Data type |
| --- | --- | --- | --- |
| Digital outputs 1–8 | [Digital output 00](D-SE-0069904.html#D-SE-0069904)  Digital output 01  Digital output 02  Digital output 03  Digital output 04  Digital output 05  Digital output 06  Digital output 07 | – | BYTE |
| Diag In 1 | [Status of digital outputs 00–07](D-SE-0069905.html#D-SE-0069905) | – | BYTE |

For further information, see the TM7 digital I/O POUs hardware manual.
