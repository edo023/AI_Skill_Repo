# ApplicationType

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos Drive,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  C2C Slave,  Logic Motion Controller Device |
| Traceable | No |

## Functional Description

Displays the application type read from the physical Sercos device. The parameter ApplicationType can only be modified by manually writing the Sercos parameter IDN S-0-1302.0.3. For this purpose, the function *[FC\_SercosWriteServiceData](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085305)* can be used.
