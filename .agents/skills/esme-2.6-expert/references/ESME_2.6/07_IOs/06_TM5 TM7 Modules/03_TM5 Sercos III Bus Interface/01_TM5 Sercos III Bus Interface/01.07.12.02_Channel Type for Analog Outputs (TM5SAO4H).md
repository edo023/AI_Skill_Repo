# Channel Type for Analog Outputs (TM5SAO4H)

## General

| Bit | Description |
| --- | --- |
| 0–7 | 0 |
| 8 | 0 = Channel 1: voltage signal (-10...+10 Vdc)  1 = Channel 1: current signal (0...20 mA) |
| 9 | 0 = Channel 2: voltage signal (-10...+10 Vdc)  1 = Channel 2: current signal (0...20 mA) |
| 10 | 0 = Channel 3: voltage signal (-10...+10 Vdc)  1 = Channel 3: current signal (0...20 mA) |
| 11 | 0 = Channel 4: voltage signal (-10...+10 Vdc)  1 = Channel 4: current signal (0...20 mA) |
| 12–15 | 0 |
