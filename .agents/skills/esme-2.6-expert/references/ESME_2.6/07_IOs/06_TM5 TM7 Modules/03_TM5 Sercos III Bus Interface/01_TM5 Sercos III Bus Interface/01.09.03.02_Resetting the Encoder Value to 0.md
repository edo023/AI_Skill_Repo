# Resetting the Encoder Value to 0 (TM5SE1IC02505)

## General

To enable the referencing process, the parameters in the User Parameter tab have to be set in the right way.

**Example configuration:** Rising edge of reference impulse sets EncoderPosition to 0.

| Parameter | Value |
| --- | --- |
| PresetABR01\_32Bit | 0 |
| ReferenceEdge01 | Rising (not default!) |
| ReferenceEnableSwitch01 | Low active |
| ReferenceEnableSwitch01 | Disabled |

## User Parameter Tab

| Parameter | Meaning | Value | Default value | Data type |
| --- | --- | --- | --- | --- |
| PresetABR01\_32 bit | Allows you to apply a preset home position.  The value set here is applied to the counter value upon completion of the referencing process. | 0 | 0 | BitArea |
| ReferenceEdge01 | Off: reference process is disabled  Falling: reference process is completed on the negative edge of the reference pulse  Rising: reference process is completed on the positive edge of the reference pulse | Off  Falling  Rising | Off | BitArea |
| ReferenceEnableSwitch01 | Low active: reference process is enabled when DigitalInput00 is at 0 Vdc  High active: reference process is enabled when DigitalInput00 is at 24 Vdc | Low active  High active | Low active | Byte |
| ReferenceEnableSwitch01 | Disabled: reference process is possible like configured in ReferenceEdge  Enabled: reference process is only enabled when the DigitalInput00 has the level set with low active/high active | Disabled  Enabled | Disabled | Byte |
