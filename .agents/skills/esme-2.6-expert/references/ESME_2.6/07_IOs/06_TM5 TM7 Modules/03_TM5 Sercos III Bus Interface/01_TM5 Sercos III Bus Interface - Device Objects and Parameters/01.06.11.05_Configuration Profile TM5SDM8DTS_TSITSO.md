# Configuration Profile TM5SDM8DTS\_TSITSO

## Data Usage

If the module is used with the configuration profile TM5SDM8DTS\_TSITSO, the following data size is used on the Sercos III bus:

* Sercos III MDT (LMCx0xC -> TM5NS31): 26 bytes
* Sercos III AT (TM5NS31 -> LMCx0xC): 24 bytes

NOTE: MDT = Master Data Telegram (set point values) / AT = Acknowledged Telegram (actual values)

The module uses 93 registers in the TM5 bus interface. This influences the maximum number of modules per bus interface (refer to [Number of I/Os](D-SE-0068978.html#D-SE-0068978).
