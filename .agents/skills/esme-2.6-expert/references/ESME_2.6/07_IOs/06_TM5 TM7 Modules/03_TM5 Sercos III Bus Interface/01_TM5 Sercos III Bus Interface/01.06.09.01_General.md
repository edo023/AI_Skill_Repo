# General

## Description

16 digital inputs, 24 Vdc, sink, configurable input filter, 1-wire technology, LEDs to indicate the state

Digital input module 24 Vdc:

* 16 digital inputs
* Sink wiring
* 1-wire technology
* Adjustable SW input filter

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs Sercos III Module I/O display, User-defined parameters, and Information are displayed. |
