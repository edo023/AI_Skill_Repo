# Coupling of the Incremental Encoder Input Object and the Logical Encoder

Coupling of the Incremental Encoder Input Object and the Logical Encoder

The coupling of the Incremental Encoder Input object can be done with the FC\_SetMaster­Encoder() SystemInterface function as follows:

diRetSetMasterEnc := FC\_SetMasterEncoder(

i\_stLEncId := LE\_LogicalEncoder.stLogicalAddress,

i\_stMasterId := IME\_EncIn.stLogicalAddress);

After this call, the logical encoder LE\_LogicalEncoder and the Incremental Encoder Input object are coupled via velocity.
