# Terminal Temperature Compensation

## General

* A terminal temperature compensation is possible for the inputs 1–4
* A sensor for the measurement of the terminal temperature is integrated into the connector casing (X67AC9A02).
* The module recognizes on the basis of such a connector independently, that a terminal temperature compensation is desired.
* For the determination of the measured temperature, at least one terminal temperature sensor is required for the J, K, S, N, and R thermocouple sensor, otherwise, 16#7FFF is output generally.

## Examples for Possible Configurations:

| Connector with sensor on input | Description |
| --- | --- |
| 1 | The terminal temperature compensation is performed for the four inputs with the temperature measured at input 1. |
| 1 and 3 | The terminal temperature compensation is performed for the inputs 1 and 2 with the temperature measured at input 1.  For the inputs 3 and 4, the terminal temperature compensation is performed with the temperature measured at input 3. |
| 1–4 | The terminal temperature compensation is performed with the temperature measured at the respective input. |

The terminal temperatures are output by the module into the registers 8, 10, 12 and 14 in steps of 0.1 °C.

Consider the following so that a defined output value is available.

* Until the first conversion, 16#8000 is output.
* If not all terminal temperature sensors are populated, the value 16#7FFF is output at the inputs that are not populated.
* If no terminal temperature sensor is populated, the value 16#7FFF is output generally.
