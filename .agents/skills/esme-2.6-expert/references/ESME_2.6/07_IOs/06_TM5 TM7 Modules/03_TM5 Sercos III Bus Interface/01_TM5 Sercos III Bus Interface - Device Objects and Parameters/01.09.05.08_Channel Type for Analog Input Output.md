# Channel Type for Analog Input/Output

## General

The individual channels are designed for current and voltage signals.

Differentiation is made by different terminals and due to different adjustment values for current and voltage, selecting the output signal is also required.

| Bit | Description |
| --- | --- |
| 0–1 | 00 = Analog input: voltage signal -10...+10 Vdc  01 = Analog input: current signal 0...20 mA  11 = Analog input: current signal 4...20 mA |
| 2–3 | 0 |
| 4 | 0 = Analog output: voltage signal -10...+10 Vdc  1 = Analog output: current signal 0...20 mA |
| 5–7 | 0 |

Corresponding bit set in the range of 4 to 5 **and** 0 to 1 means channel selection current 4...20 mA.

The measuring range 4...20 mA is calculated from the value of the 0...20 mA range (value - 0x1999) \* 1.25.

4 mA correspond to 0x0000 and 20 mA correspond to 0x7fff.

Due to the increased spreading, the number of unavailable codes is increased.
