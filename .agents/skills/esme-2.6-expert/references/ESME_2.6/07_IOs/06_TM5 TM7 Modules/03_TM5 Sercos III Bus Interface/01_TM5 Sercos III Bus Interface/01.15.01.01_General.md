# General

## Description

16 digital inputs, 24 Vdc, sink, input filter 1 ms, M12 connection technology, high packaging density, LEDs to view the status

Digital input module 24 Vdc with 16 digital inputs, 8xM12 with two channels per outlet

* 16 digital inputs
* For standard sensors with M12 connection
* 1:1 replacement of passive distributors
* Short cycle times
* Integrated short circuit proof sensor supply

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM7 Module I/O display, User-defined parameters, and Information are displayed. |

## Tab TM7 Module I/O Display

| Channel | Meaning | Default value | Data type |
