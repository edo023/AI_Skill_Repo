# Channel Type for Analog Inputs 1–2

## General

The selection voltage (0 = Default) or current (1) is required due to the different reference values for current and voltage.

| Bit | Description |
| --- | --- |
| 0 | 0 = Channel 1: voltage signal  1 = Channel 1: current signal |
| 1 | 0 = Channel 2: voltage signal  1 = Channel 2: current signal |
| 2–3 | 0 |
| 4 | 0 = Channel 1: 0...20 mA current signal  1 = Channel 1: 4...20 mA current signal |
| 5 | 0 = Channel 2: 0...20 mA current signal  1 = Channel 2: 4...20 mA current signal |
| 6–7 | 0 |

Corresponding bit set in the range of 4 to 5 **and** 0 to 1 means channel selection current 4...20 mA.

The measuring range 4...20 mA is calculated from the value of the 0...20 mA range (value–0x1999) \* 1.25.

4 mA correspond to 0x0000 and 20 mA correspond to 0x7fff.

Due to the increased spreading, the number of unavailable codes is increased.
