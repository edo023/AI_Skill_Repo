# Configuration Profile TM5SDM8DTS\_OIOO

## Data Usage

If the module is used with the configuration profile TM5SDM8DTS\_OIOO, the following data size is used on the Sercos III bus:

* Sercos III MDT (LMCx0xC -> TM5NS31): 7 bytes
* Sercos III AT (TM5NS31 -> LMCx0xC): 8 bytes

NOTE: MDT = Master Data Telegram (set point values) / AT = Acknowledged Telegram (actual values)

The module uses 42 registers in the TM5 bus interface. This influences the maximum number of modules per bus interface (refer to [Number of I/Os](D-SE-0068978.html#D-SE-0068978)).
