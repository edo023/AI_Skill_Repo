# Channel Type for Analog Outputs 1–2

## General

The selection voltage (0 = Default) or current (1) is required due to the different reference values for current and voltage.

| Bit | Description |
| --- | --- |
| 0 | 0 = Channel 1 (analog output 1): voltage signal -10 Vdc to +10 Vdc  1 = Channel 1 (analog output 1): current signal 0...20 mA |
| 1 | 0 = Channel 2 (analog output 2): voltage signal -10 Vdc to +10 Vdc  1 = Channel 2 (analog output 2): current signal 0...20 mA |
| 2–7 | 0 |

## Example

|  |  |
| --- | --- |
| 00000001 | Analog output 1: current signal 0...20 mA  Analog output 2: voltage signal -10 Vdc to +10 Vdc |
| 00000010 | Analog output 1: voltage signal -10 Vdc to +10 Vdc  Analog output 2: current signal 0...20 mA |

Both available analog outputs can be configured independently from each other via the adjustable channel type.
