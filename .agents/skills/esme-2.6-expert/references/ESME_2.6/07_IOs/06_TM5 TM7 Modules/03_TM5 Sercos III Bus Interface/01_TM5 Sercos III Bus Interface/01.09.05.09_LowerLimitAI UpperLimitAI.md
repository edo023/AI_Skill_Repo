# LowerLimitAI / UpperLimitAI

## LowerLimitAI

Here, the lower limit value of the analog values can be set. If the limit value is undershot, the analog value is frozen to this value and the respective error status bit is set.

NOTE: The default value of -32768 corresponds to the minimum standard value of -10 Vdc. For current measurements, this value should be set to 0. For configuration 4...20 mA, the value can be set to -6553 (corresponds to 0 mA) to display values < 4 mA.

## UpperLimitAI

Here, the upper limit value of the analog values can be set. When exceeding the limit value, the analog value is frozen at this value and the respective error status bit is set.

NOTE: The default value of 32767 corresponds to the maximum standard value at 20 mA or +10 Vdc.

EIO0000002196.02
