# Digital Outputs 1–4

## General

| Bit | Description |
| --- | --- |
| 0 | 0 = Digital output 1 reset  1 = Digital output 1 set |
| 1 | 0 = Digital output 2 reset  1 = Digital output 2 set |
| 2 | 0 = Digital output 3 reset  1 = Digital output 3 set |
| 3 | 0 = Digital output 4 reset  1 = Digital output 4 set |
