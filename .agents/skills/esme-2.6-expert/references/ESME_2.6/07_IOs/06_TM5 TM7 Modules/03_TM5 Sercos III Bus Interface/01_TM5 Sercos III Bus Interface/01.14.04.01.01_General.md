# General

## Description

Safety digital counter-module, 24 Vdc

* One safety-related input counter-channel
* Configurable input filter
* Function modes: A-A, A-B, A-A/-B-B/
* Maximum input frequency: 7 kHz
* Rated output current: 80 mA

## Configuration

| Tab | Meaning |
| --- | --- |
| [TM5 Module I/O Mapping](D-SE-0070462.html#D-SE-0070462) | Mapping of channels to project variables |
| [TM5 Module Parameters](D-SE-0070463.html#D-SE-0070463) | Safety-related configuration parameters |

For further information, see the TM5/TM7 Digital I/O Safety Modules Hardware Guide.
