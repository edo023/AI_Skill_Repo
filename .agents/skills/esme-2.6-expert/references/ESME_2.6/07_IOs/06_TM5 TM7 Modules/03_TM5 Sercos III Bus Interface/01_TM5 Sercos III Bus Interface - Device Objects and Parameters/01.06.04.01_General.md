# General

## Description

Four digital inputs, 100...240 Vac, 50 / 60 Hz

Digital input module 100...240 Vac with four digital inputs (2-wire, type 1)

## Configuration Register

| Register | Meaning | Default | Data type |
| --- | --- | --- | --- |
| 0 | [Digital inputs 1–4](D-SE-0070318.html#D-SE-0070318) | – | USINT |
| 18 | [Digital input filter](D-SE-0070319.html#D-SE-0070319) | 10 | USINT |
