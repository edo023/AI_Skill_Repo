# Disable / Enable Inputs 1–2

## General

If a bit is set, the respective input channel is deactivated:

| Bit | Description |
| --- | --- |
| 0 | Input 01:  0 = Disabled  1 = Enabled |
| 1 | Input 02:  0 = Disabled  1 = Enabled |

By disabling/enabling inputs during operation, the disabled input registers are reset to invalid value and the status flags are deleted.
