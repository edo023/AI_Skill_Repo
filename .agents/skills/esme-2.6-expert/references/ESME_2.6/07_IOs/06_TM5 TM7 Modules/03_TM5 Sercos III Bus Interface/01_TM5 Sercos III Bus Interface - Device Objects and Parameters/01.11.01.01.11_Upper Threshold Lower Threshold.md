# Upper Threshold / Lower Threshold

## Upper Threshold

Using this register, the upper threshold of the receive FIFO is configured. If the number of free bytes in the receive FIFO exceeds this upper threshold, the receive status is set to active. This means, depending on the handshake configuration, for example, an XOn character can be sent or the RTS line can be set to active.

| Name | Value range |
| --- | --- |
| Upper Threshold | 0–4095 |

## Lower Threshold

Using this register, the lower threshold of the receive FIFO is configured. If the number of free bytes in the receive FIFO undershoots the lower threshold, the receive status is set to passive. This means, depending on the handshake configuration, for example, an XOff character can be sent or the RTS line can be set to passive.

| Name | Value range |
| --- | --- |
| Lower Threshold | 0–4095 |
