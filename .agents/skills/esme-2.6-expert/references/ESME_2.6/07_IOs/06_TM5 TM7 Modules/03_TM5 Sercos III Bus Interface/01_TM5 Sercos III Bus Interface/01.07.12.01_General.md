# General

## Description

Four analog outputs ±10 V / 0...20 mA, resolution 16 bit

Analog output module 24 Vdc with 4x16 bit analog outputs (voltage ±10 Vdc / current 0...20 mA, differential)

## Configuration Register

| Register | Meaning | Default | Data type |
| --- | --- | --- | --- |
| 2 | Analog output 01 | ±10 Vdc | INT |
| 4 | Analog output 02 | ±10 Vdc | INT |
| 6 | Analog output 03 | ±10 Vdc | INT |
| 8 | Analog output 04 | ±10 Vdc | INT |
| 0 | [Channel type for analog outputs (TM5SAO4H)](D-SE-0070400.html#D-SE-0070400) | 0 | UINT |

For further information, see the TM5 analog I/O modules hardware manual.
