# Channel Type for Analog Outputs 1–4 (TM5SAO4L)

## General

The selection voltage (0 = Default) or current (1) is required due to the different reference values for current and voltage.

| Bit | Nibble | Description |
| --- | --- | --- |
| 0 | Low-order nibble (4 bit) | 0 = Output 01: Voltage signal ±10 Vdc  1 = Output 01: Current signal 0...20 mA |
| 1 | 0 = Output 02: Voltage signal ±10 Vdc  1 = Output 02: Current signal 0...20 mA |
| 2 | 0 = Output 03: Voltage signal ±10 Vdc  1 = Output 03: Current signal 0...20 mA |
| 3 | 0 = Output 04: Voltage signal ±10 Vdc  1 = Output 04: Current signal 0...20 mA |
| 4–7 | Higher-quality nibble (4 bit) | 0 |

General description for the bit 7-0 assignment

| Selection current/voltage  Bit 7...Bit 0 | Meaning |
| --- | --- |
| 0000000x | Configure the output 01 in the low-order nibble:  Bit 0 is set to 0: Voltage signal  Bit 0 is set to 1: Current signal |
| 000000x0 | Configure the output 02 in the low-order nibble:  Bit 1 is set to 0: Voltage signal  Bit 1 is set to 1: Current signal |

Examples for the bit 7-0 assignment

| Selection current/voltage  Bit 7...Bit 0 | Meaning |
| --- | --- |
| 00000000 | Output 01 to 04: Configured voltage signal ±10 Vdc |
| 00000001 | Output 01: Configured current signal 0...20 mA  Output 02 to 04: Configured voltage signal ±10 Vdc |
| 00000110 | Output 02 and 03: Configured current signal 0...20 mA  Output 01 and 04: Configured voltage signal ±10 Vdc |
