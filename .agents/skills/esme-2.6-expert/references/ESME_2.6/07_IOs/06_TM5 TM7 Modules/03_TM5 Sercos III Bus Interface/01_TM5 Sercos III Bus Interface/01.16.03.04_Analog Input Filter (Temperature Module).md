# Analog Input Filter (Temperature Module)

## General

The filtering of the analog inputs is parameterized via this tab.

| Value | Filter | Filter time | Digital transducer resolution |
| --- | --- | --- | --- |
| 0 | 50 Hz | 20 ms | 16 bit |
| 1 | 60 Hz | 16.67 ms | 16 bit |
| 2 | 250 Hz | 4 ms | 13 bit |
| 3 | 500 Hz | 2 ms | 10 bit |
