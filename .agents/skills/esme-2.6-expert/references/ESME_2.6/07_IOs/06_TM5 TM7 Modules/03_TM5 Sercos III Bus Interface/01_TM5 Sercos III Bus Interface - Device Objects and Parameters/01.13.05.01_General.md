# General

## Description

The module TM5SD000 is used for the following options:

* Placeholder for later system upgrade
* Used as terminal support
* Module without electrical function

When respective accessory parts are used, the module TM5SD000 has the following functions:

* Terminal block, sixfold, 24 V coded
* Terminal block, twelvefold, 24 V coded
* Supply bus module, internal I/O supply isolated to the left
* Bus module 24 V coded, internal I/O supply wired through

For further information, see the TM5/TM7 System Planning and Installation Guide.
