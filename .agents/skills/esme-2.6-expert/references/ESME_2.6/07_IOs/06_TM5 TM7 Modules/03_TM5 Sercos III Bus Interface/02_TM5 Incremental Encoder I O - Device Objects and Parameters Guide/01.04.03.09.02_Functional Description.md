# Functional Description

Functional Description

Disables the encoder monitoring.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOLEAN | Encoder monitoring active. |
| yes / 1 | BOOLEAN | Encoder monitoring (track and cable monitoring) not active. |

NOTE: For BT-4/ENC1: If the encoder simulation is operated in internal master mode, the external encoder input 0 is ignored. This input does not issue any error detection.
