# General

## Description

Safety analog input module, 24 Vdc

* 2x two analog safety-related inputs for thermocouples
* Configurable input filter and switching threshold
* Measurement range: -270...1768 °C (-454...3214.4 °F)
* Input sensor type: J, K, N, S, R, C, T thermocouple sensors
* Input range: -65...+65 mV (voltage measurement range)
* Terminal temperature Compensation: 2x one analog safety-related inputs for PT100/PT1000 measurement
* Digital converter resolution: 24 bits
