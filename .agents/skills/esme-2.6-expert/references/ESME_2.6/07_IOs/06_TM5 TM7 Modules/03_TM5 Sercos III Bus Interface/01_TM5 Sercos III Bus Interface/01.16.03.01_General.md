# General

## Description

Four inputs resistance measuring, 2- and 4- wire measuring, PT100, PT1000, KTY10,KTY84, resolution 0.1 K, LEDs to indicate the status.

The temperature input module can be operated with resistance temperature detectors of type KTY10, KTY84, PT100, and PT1000. The sensors can be connected with 2- or 4-conductor technology. The sensor type can be adjusted selectively for each input.

It is equipped with 16-bit digital transducer resolution.

* Four inputs for resistance temperature measurement
* For the sensor types PT100, PT1000 and others
* In addition, direct resistance measurement
* Sensor type adjustable per channel
* 2- and 4-wire measuring

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM7 Module I/O display, User-defined parameters, and Information are displayed. |
