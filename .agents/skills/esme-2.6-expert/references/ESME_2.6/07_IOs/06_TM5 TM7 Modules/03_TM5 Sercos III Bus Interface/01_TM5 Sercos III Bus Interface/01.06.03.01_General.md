# General

## Description

Two digital inputs 24 Vdc, sink, IEC 61131-2, type 1, counter

Digital input module 24 Vdc with two digital inputs (sink, 1 / 2 or wires, type 1) and additional function 50 kHz event counting / gate measurement

NOTE: Gate measurement can be performed at one channel only.
