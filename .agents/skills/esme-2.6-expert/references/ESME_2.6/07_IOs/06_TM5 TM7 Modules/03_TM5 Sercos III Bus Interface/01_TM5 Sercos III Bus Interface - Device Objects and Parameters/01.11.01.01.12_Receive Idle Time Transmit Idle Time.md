# Receive Idle Time / Transmit Idle Time

## Receive Idle Time

Using this register, the Receive Idle Time (receive timeout) is configured. If this time has elapsed after a receive block and no new data has been received, the frame stored so far is considered as completed and transferred to the superordinated system.

The time is specified in characters to generate the same behavior, irrespective of the set Baudrate.

| Name | Meaning |
| --- | --- |
| Receive Idle Time | 1–65535 |

## Transmit Idle Time

Using this register, the Transmit Idle Time (transmit timeout) is configured. If several frames are located in the transmit buffer, a transmission break can be generated between the frames. This helps the outstation to detect the end of the frame via a Receive Idle Time. The time is specified in characters to generate the same behavior, irrespective of the set Baudrate.

| Name | Meaning |
| --- | --- |
| Transmit Idle Time | 1–65535 |
