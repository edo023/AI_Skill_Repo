# Digital Inputs (Expert Module)

## Unfiltered

The input status is registered with a fixed offset in relation to the network cycle and transferred in the same cycle.

## Filtered

The filtered status is registered with a fixed offset in relation to the network cycle and transferred in the same cycle. The filtering proceeds asynchronously to the network in a raster of 200 μs with a jitter due to the network of up to 50 μs.
