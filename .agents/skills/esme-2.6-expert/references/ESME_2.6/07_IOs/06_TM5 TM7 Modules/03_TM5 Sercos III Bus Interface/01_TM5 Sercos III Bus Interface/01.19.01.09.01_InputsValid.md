# InputsValid

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Safe Logic Controller TM5CSLCx00FS,  Bus Interface TM5NS31,  Sercos I/O |
| Traceable | – |

## Functional Description

Indicates whether the input values are valid.

| Value | Data type | Meaning |
| --- | --- | --- |
| TRUE | BOOL | The input values are valid. |
| FALSE | BOOL | The input values are invalid. |

EIO0000002196.02
