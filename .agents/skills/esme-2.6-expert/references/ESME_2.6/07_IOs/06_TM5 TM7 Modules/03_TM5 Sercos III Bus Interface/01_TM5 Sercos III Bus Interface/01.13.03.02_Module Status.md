# Module Status

## General

The following module states are monitored:

* Micro-fuse state
* Voltage monitoring 24 V externally

| Bit | Description |
| --- | --- |
| 0 | 0 = Fuse OK  1 = Fuse not OK |
| 1 | 0 = Level of fed voltage OK  1 = Level of fed voltage not OK |
| 2–7 | Reserved |
