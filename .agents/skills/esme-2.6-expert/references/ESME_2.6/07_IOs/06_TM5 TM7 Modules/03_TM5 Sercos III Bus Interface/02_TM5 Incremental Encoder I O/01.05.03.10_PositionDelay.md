# PositionDelay

![](images/logo.png)

PositionDelay

General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Incremental Encoder Input  Incremental Encoder Output |
| Traceable | No |

Functional Description

Displays how recent was the last read of the position (parameter [Position](EncoderOutput-11.htm#XREF_D_SE_0069681_1)), in μs. There is a delay because the encoder position is transferred from the module via the TM5 bus and the Sercos back to the controller.
