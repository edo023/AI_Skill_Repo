# Regular Digital Outputs

The regular digital output states of the configuration profile TM5SDM8DTS\_HFOIDO can be written to the [I/O mapping channel](D-SE-0069001.html#D-SE-0069001) DigitalOutput01-08.

Each bit represents one physical channel on the module.

The following bits represent the values of the four outputs:

* Bit 2: physical channel 3
* Bit 3: physical channel 4
* Bit 6: physical channel 7
* Bit 7: physical channel 8

The bits 0, 1, 4 and 5 of the I/O mapping channel DigitalOutput01-08 are reserved for system use.
