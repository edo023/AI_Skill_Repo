# Input MTU

## General

This register configures the size of the MTU available in the input data area, in bytes. This MTU area is used to read the received data and the control bytes from the module. The MTU size is not related to the permitted segment size or the actual serial frame size; it just specifies the size of the transfer area on the bus.

The size of the input MTU is set to 15 bytes.
