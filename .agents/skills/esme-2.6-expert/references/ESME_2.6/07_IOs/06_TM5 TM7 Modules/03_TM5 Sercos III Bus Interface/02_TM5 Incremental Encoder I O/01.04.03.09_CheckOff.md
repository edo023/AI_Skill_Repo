# CheckOff

![](images/logo.png)

CheckOff

General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Incremental Encoder Input |
| Traceable | No |

Functional Description

Disables the encoder monitoring.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOLEAN | Encoder monitoring active. |
| yes / 1 | BOOLEAN | Encoder monitoring (track and cable monitoring) not active. |

NOTE: For BT-4/ENC1: If the encoder simulation is operated in internal master mode, the external encoder input 0 is ignored. This input does not issue any error detection.

Interplay of Track Monitoring Parameters

Interplay of track monitoring parameters CheckOff and ZeroTrackCheckOff:

| CheckOff | ZeroTrackCheckOff | Active monitorings |
| --- | --- | --- |
| no / FALSE | no / FALSE | Track A, B, and zero track. |
| no / FALSE | yes / TRUE | Track A and B. |
| yes / TRUE | no / FALSE | No monitoring. |
| yes / TRUE | yes / TRUE | No monitoring. |
