# Enable

![](images/logo.png)

Enable

General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Incremental Encoder Output |
| Traceable | No |

Functional Description

The following parameters are edge-sensitively initialized and reset:

ooff -> on: [Position](EncoderOutput-11.htm#XREF_D_SE_0069681_1) = 0; Velocity = 0; Output frequency = 0 Hz

oon -> off: [Velocity](EncoderOutput-13.htm#XREF_D_SE_0069682_1) = 0; Output frequency = 0 Hz

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOLEAN | The encoder is not processed by the real-time process. |
| on / 1 | BOOLEAN | The encoder is processed by the real-time process. |
