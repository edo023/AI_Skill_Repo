# General

## Description

Safety digital input module, 24 Vdc

* Two safety-related inputs
* Two safety-related test outputs
* Configurable input filter can be set by channel

## Configuration

| Tab | Meaning |
| --- | --- |
| [TM5 Module I/O Mapping](D-SE-0070412.html#D-SE-0070412) | Mapping of channels to project variables |
| [TM5 Module Parameters](D-SE-0070413.html#D-SE-0070413) | Safety-related configuration parameters |

For further information, see the TM5/TM7 Digital I/O Safety Modules Hardware Guide.
