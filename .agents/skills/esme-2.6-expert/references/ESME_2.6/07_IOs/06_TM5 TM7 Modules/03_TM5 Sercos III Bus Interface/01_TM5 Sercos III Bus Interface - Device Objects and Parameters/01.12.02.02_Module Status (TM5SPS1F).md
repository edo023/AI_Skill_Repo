# Module Status (TM5SPS1F)

## General

The following module states are monitored:

* Bus supply voltage
* 24 Vdc I/O supply voltage
* Fuse status

| Bit | Description |
| --- | --- |
| 0 | 0 = No detected error  1 = Bus supply alert in the case of undervoltage (<4.7 V) |
| 1 | 0 = Fuse OK  1 = Fuse defective |
| 2 | 0 = I/O supply exceeding the alert limit of 20.4 V  1 = I/O supply below alert limit of 20.4 V or fuse defective |
| 3–x | 0 |
