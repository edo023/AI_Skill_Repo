# TM5CycleTime

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Bus Interface TM5NS31 |
| Traceable | Yes |

## Functional Description

With this command, you can set the cycle time [in μs] for the TM5 bus.

| Value | Data type | Meaning |
| --- | --- | --- |
| 4000 μs / 0  3500 μs / 1  3000 μs / 2  2500 μs / 3  2000 μs / 4  1500 μs / 5  1000 μs / 6 (default)  500 μs / 7  250 μs / 8 | UINT | Cycle time for the TM5 bus [μs] |
