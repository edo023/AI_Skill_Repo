# Digital Inputs 1–6 (TM5SDI6U)

## General

| Bit | Description |
| --- | --- |
| 0 | Status of digital input 1 |
| 1 | Status of digital input 2 |
| 2 | Status of digital input 3 |
| 3 | Status of digital input 4 |
| 4 | Status of digital input 5 |
| 5 | Status of digital input 6 |
| 6 | 0 |
| 7 | 0 = Supply voltage too low  1 = Supply voltage > 80 Vac |
