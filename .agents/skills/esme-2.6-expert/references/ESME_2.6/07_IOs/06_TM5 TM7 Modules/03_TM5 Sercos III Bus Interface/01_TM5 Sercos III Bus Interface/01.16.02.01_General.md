# General

## Description

Four inputs for thermocouples, type J, K, S, N, R resolution 0.1 K, LEDs to indicate the status

The temperature input module can be operated with thermocouple sensors of type J, K, S, N, and R. It is equipped with 16-bit digital transducer resolution. The selected sensor type is used for the four inputs.

* Four inputs for thermocouples
* For sensor types J, K, S, N, R
* In addition, direct measurement of the raw values for other sensor types
* Terminal temperature compensation

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM7 Module I/O display, User-defined parameters, and Information are displayed. |

The values are set via a selection list.

The check box Symbolic values is used for switching between the numeric and the symbolic value.
