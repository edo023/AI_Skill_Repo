# General

## Description

Safety analog input module, 24 Vdc

* 2x two analog safety-related inputs
* Configurable input filter and switching threshold
* Input range: 4...20 mA (valid measurement range) 0.5...25 mA (input range)
* Digital converter resolution: 24 bits

## Configuration

| Tab | Meaning |
| --- | --- |
| [TM5 Module I/O Mapping](D-SE-0070453.html#D-SE-0070453) | Mapping of channels to project variables |
| [TM5 Module Parameters](D-SE-0070454.html#D-SE-0070454) | Safety-related configuration parameters |

For further information, see the TM5/TM7 Digital I/O Safety Modules Hardware Guide.
