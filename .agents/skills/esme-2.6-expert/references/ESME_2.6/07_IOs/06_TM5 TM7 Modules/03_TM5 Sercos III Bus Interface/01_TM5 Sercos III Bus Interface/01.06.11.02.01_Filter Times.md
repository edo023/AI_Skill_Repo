# Filter Times

## General

The inputs of the module have a hardware filter with a 155 kHz frequency (HW filter first order with cut-off frequency = 155 kHz).

This implies a filter time of 6 μs.

For touchprobing this means that a signal change has to be present at least for 6 μs to be detected as an edge.

Additional software filter times are not supported by this hardware.
