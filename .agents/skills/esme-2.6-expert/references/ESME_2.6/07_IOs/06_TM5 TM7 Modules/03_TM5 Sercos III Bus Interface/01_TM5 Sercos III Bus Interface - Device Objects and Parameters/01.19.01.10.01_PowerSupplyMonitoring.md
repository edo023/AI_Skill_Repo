# PowerSupplyMonitoring

## General

|  |  |
| --- | --- |
| Type | ED |
| Devices supporting the parameter | Bus Interface TM5NS31 |
| Traceable | Yes |

## Functional Description

Enables/disables the supply voltage monitoring for the TM5 power supply units connected to the TM5 bus interface (TM5NS31).

The parameter can be read and written from the application during runtime and is configurable by using the EcoStruxure Machine Expert during programming.

## Activated / 1

If the supply voltage monitoring is Enabled / 1 for a TM5 bus interface and the I/O power supply warning parameter of one of the connected power supply units (TM5SPS1, TM5SPS1F, TM5SPS2, TM5SPS2F) is set to TRUE, diagnostic message "[8998 TM5/TM7 Supply voltage too low](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064122)" is triggered.

The I/O Data valid parameter is monitored for the TM7 output modules. If the parameter is set to TRUE, diagnostic message "[8998 TM5/TM7 Supply voltage too low](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064122)" is triggered.

If an analog TM5/TM7 module is connected to the power supply unit which lost supply voltage, diagnostic message "[8163 SERCOS Slave C1D Error detected](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063884)" is triggered.

## Message / 2

If the supply voltage monitoring is Message / 2 for a TM5 bus interface and the I/O power supply warning parameter of one of the connected power supply units (TM5SPS1, TM5SPS1F, TM5SPS2, TM5SPS2F) is set to TRUE, diagnostic message "[8999 TM5/TM7 Warning supply voltage](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064123)" is triggered.

The I/O Data valid parameter is monitored for the TM7 output modules. If the parameter is set to TRUE, diagnostic message "[8999 TM5/TM7 Warning supply voltage](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064123)" is triggered.

If an analog TM5/TM7 module is connected to the power supply unit which lost the supply voltage, diagnostic message "[8163 SERCOS Slave C1D Error detected](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063884)" is not triggered.

## Deactivated / 0

If the supply voltage monitoring is Disabled / 0 for a TM5 bus interface and the I/O power supply warning parameter of one of the connected power supply units (TM5SPS1, TM5SPS1F, TM5SPS2, TM5SPS2F) or the I/O Data valid parameter on a connected TM7 output module is set to TRUE, no diagnostic message is triggered.

If an analog TM5/TM7 module is connected to the power supply unit which lost supply voltage, **no** diagnostic message is triggered.

| Value | Data type | Meaning |
| --- | --- | --- |
| Enabled / 1 (default) | BOOL | The power supply unit is monitored and the detected errors regarding the power supply unit are reported as detected errors. |
| Message / 2 | BOOL | The power supply unit is monitored and the detected errors regarding the power supply unit are displayed as messages. |
| Deactivated / 0 | DINT | No detected errors regarding the power supply unit are reported. |
