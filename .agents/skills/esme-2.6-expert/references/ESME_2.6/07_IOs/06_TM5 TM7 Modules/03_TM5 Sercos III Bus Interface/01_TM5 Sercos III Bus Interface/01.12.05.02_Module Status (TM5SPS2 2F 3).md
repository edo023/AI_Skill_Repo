# Module Status (TM5SPS2/2F/3)

## General

The following module states are monitored:

* Bus supply current
* Bus supply voltage
* 24 Vdc I/O supply voltage

| Bit | Description |
| --- | --- |
| 0 | 0 = No detected error  1 = Bus supply alert in the case of overcurrent (> 2.3 A) or undervoltage (<4.7 V) |
| 1 | 0 |
| 2 | 0 = I/O supply exceeding the alert limit of 20.4 V  1 = I/O supply below the alert limit of 20.4 V |
| 3–x | 0 |
