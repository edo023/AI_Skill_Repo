# Functional Description

Functional Description

Displays the encoder position with a delay (represented by the parameter [PositionDelay](EncoderInput-15.htm#XREF_D_SE_0073771_1)). The value range runs from 0 to [FeedConstant](EncoderInput-7.htm#XREF_D_SE_0069537_1). With negative direction of rotation, clockwise rotation causes the value range to run in decreasing order.

At system startup, the encoder position is set to 0. If the incremental encoder is connected to a bus terminal, the Position is not 0 after the controller has been reset. 0 is only displayed if the controller is also reset.

For TM5 encoder: The parameter [PositionDelay](EncoderInput-15.htm#XREF_D_SE_0073771_1) indicates how recent was the last read of the position. (There is a delay because the encoder position is transferred from the module via the TM5 bus and the Sercos back to the controller.)

Use the parameter [State](EncoderInput-6.htm#XREF_D_SE_0069517_1) to verify the availability or validity of the encoder position data.
