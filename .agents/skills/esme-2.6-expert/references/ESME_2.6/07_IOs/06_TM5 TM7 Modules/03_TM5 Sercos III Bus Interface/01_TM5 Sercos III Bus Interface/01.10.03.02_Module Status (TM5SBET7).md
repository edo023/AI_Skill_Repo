# Module Status (TM5SBET7)

## General

The following module states are monitored:

* TM7 bus supply current
* TM7 bus supply voltage
* 24 Vdc I/O supply voltage

| Bit | Description |
| --- | --- |
| 0 | 0 = No detected error  1 = TM7 bus supply alert in the case of overcurrent (>0.4 A) or undervoltage (<18 V) |
| 1 | 0 |
| 2 | 0 = I/O supply exceeding the alert limit of 20.4 V  1 = I/O supply below the alert limit of 20.4 V |
| 3–x | 0 |
