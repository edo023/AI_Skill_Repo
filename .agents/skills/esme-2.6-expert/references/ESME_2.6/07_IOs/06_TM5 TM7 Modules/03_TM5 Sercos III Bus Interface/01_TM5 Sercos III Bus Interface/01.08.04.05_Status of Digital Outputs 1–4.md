# Status of Digital Outputs 1–4

## General

| Bit | Description |
| --- | --- |
| 0 | 0 = Output 1: no error  1 = Output 1: short circuit or overload |
| 1 | 0 = Output 2: no error  1 = Output 2: short circuit or overload |
| 2 | 0 = Output 3: no error  1 = Output 3: short circuit or overload |
| 3 | 0 = Output 4: no error  1 = Output 4: short circuit or overload |
