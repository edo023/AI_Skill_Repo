# General

## Description

24 Vdc supply module for the TM7 power bus

The TM7SPS1A module (PDB, Power Distribution Block) feeds the TM7 power bus.

The module is in Logic Builder not visible and thus cannot be inserted into the device tree and cannot be parameterized.

It is only connected physically with the system.

Main characteristics of the module

| Characteristic | Value |
| --- | --- |
| Rated output power | 15 W |
| Rated input voltage | 24 Vdc (18...30 Vdc) |
| Rated output voltage | 20 Vdc |
| Rated output current | 750 mA |
| Connection type for TM7 bus | M12, B-coded, connector of the type plug connector and socket |
| Connection type for power supply | M8, 4-pin, connector of the type plug connector and socket |

For further information, see the TM5/TM7 System Planning and Installation Guide.
