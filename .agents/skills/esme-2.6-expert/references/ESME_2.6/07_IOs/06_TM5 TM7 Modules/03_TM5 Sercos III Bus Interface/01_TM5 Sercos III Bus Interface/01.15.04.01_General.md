# General

## Description

16 digital channels (as input or output), 24 Vdc, 0.5 A, inputs for sink-wiring, output for source-wiring, input filter configurable, M12 connection technology, high packing density, LEDs to view the status

Digital mixed module 24 Vdc with 16 channels, via software configurable as input or output, 8x M12 A-coded with two channels per outlet

* 16 digital channels, configurable as input or output
* Replacement of passive distributors
* Adjustable digital input filters
* The outputs are equipped with a single channel diagnostic
* Extensive additional state information

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM7 Module I/O display, User-defined parameters, and Information are displayed. |
