# Description

TM5 interface module RS-232

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM5 Module I/O display, User-defined parameters, and Information are displayed. |

## Tab TM5 Module I/O Display

The input and output channels of the module are mapped to project variables that are used by the application.

| Channel | Meaning | Default value | Data type |
| --- | --- | --- | --- |
| [OutputSequence](D-SE-0069258.html#D-SE-0069258) | Transmitter sequence number  Transmitter status  Feedback receiver sequence number  Receiver status | – | BYTE |
| [TxByte1–TxByte15](D-SE-0069259.html#D-SE-0069259) | Control byte in sending direction or transmission data | – | BYTE |
| [InputSequence](D-SE-0069260.html#D-SE-0069260) | Receiver sequence number  Receiver status  Feedback transmitter sequence number  Transmitter status | – | BYTE |
| [RxByte1–RxByte15](D-SE-0069261.html#D-SE-0069261) | Control byte in receiving direction or received data | – | BYTE |
