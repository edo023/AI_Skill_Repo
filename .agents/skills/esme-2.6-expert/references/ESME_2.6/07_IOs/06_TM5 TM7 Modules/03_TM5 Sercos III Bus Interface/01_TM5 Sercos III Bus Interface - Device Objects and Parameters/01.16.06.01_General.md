# General

## Description

Two analog inputs, two analog outputs, ±10 V, resolution 12 bit

The analog mixed module 24 Vdc is equipped with two inputs and two outputs with 12-bit digital transducer resolution. The input/output signal is ±10 V.

* Two analog inputs, two analog outputs ±10 V
* Wirebreak detection at the inputs
* Adjustable digital input filters
* Short cycle time
* Ideal shield discharge on the channels

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM7 Module I/O display, User-defined parameters, and Information are displayed. |

## Tab TM7 Module I/O Display

The input and output channels of the module are mapped to project variables that are used by the application.

| Channel | Meaning | Default value | Data type |

| [Analog input 1](D-SE-0069868.html#D-SE-0069868) | Analog input 1 | – | INT |

| Analog output 4 | Analog output 4 | – | INT |
