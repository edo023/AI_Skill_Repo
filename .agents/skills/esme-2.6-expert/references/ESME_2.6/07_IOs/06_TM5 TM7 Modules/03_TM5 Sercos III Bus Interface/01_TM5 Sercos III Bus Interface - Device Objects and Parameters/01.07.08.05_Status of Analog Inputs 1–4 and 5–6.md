# Status of Analog Inputs 1–4 and 5–6

## Status Register Inputs 1–4

| Bit | Description |
| --- | --- |
| 0–1 | Input 1  00 = No detected error  01 = Lower limit value exceeded  10 = Upper limit value exceeded  11 = Wire break (not with current) |
| 2–3 | Input 2  00 = No detected error  01 = Lower limit value exceeded  10 = Upper limit value exceeded  11 = Wire break (not with current) |
| 4–5 | Input 3  00 = No detected error  01 = Lower limit value exceeded  10 = Upper limit value exceeded  11 = Wire break (not with current) |
| 6–7 | Input 4  00 = No detected error  01 = Lower limit value exceeded  10 = Upper limit value exceeded  11 = Wire break (not with current) |

## Status Register Inputs 5–6

| Bit | Description |
| --- | --- |
| 0–1 | Input 5  00 = No detected error  01 = Lower limit value exceeded  10 = Upper limit value exceeded  11 = Wire break (not with current) |
| 2–3 | Input 6  00 = No detected error  01 = Lower limit value exceeded  10 = Upper limit value exceeded  11 = Wire break (not with current) |
| 4–5 | – |
| 6–7 | – |
