# Analog Inputs (TM7BAI4TLA)

## General

The converted analog values are output by the module as RTD input 1 to RTD input 4.

Consider the following should be noted so that a defined output value is available.

* Until the first conversion, 16#8000 is output.
* After the switchover of the sensor type, 16#8000 is output until the first conversion.
* If the input is not switched on, 16#8000 is output.

## Temporal Coordination

The measured value recording is coordinated temporally via the converter hardware.

Each conversion cycle, the switched on inputs are converted.

Unneeded inputs can be switched off, whereby the refresh time is reduced. The shutdown can be temporarily.

The savings per input depend on the filter time.

| Filter | Filter time | Saving per input | Digital transducer resolution |
| --- | --- | --- | --- |
| 50 Hz | 20 ms | 75 ms | 16 bit |
| 60 Hz | 16.67 ms | 65 ms | 16 bit |
| 250 Hz | 4 ms | 27 ms | 13 bit |
| 500 Hz | 2 ms | 21 ms | 10 bit |
