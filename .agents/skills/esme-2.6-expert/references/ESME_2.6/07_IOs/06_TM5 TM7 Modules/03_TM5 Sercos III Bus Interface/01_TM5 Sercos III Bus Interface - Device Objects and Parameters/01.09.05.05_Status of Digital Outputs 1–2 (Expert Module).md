# Status of Digital Outputs 1–2 (Expert Module)

## Status Digital Output 1–2

| Bit | Description |
| --- | --- |
| 0 | Input state digital input 1 |
| 1 | Input state digital input 2 |
| 2 | Input state digital input 3 |
| 3 | Input state digital input 4 |
| 4 | 0 = Digital output channel 1: no detected error  1 = Digital output channel 1: short circuit or overload |
| 5 | 0 = Digital output channel 2: no detected error  1 = Digital output channel 2: short circuit or overload |
| 6 | Reserved |
| 7 | Reserved |
