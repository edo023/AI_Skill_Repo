# Functional Description

Functional Description

When [zero track detection](EncoderInput-17.htm#XREF_D_SE_0069585_1) is enabled, this bit indicates whether a zero track has been detected.

| Value | Data type | Meaning |
| --- | --- | --- |
| false / 0 | BOOLEAN | No zero track detected. |
| true / 1 | BOOLEAN | Zero track detected. |

NOTE: When a zero track is detected, the [Position](EncoderInput-14.htm#XREF_D_SE_0069589_1) of the incremental encoder input is set to 0. The detection of a zero impulse has no effect on the Position parameter of any incremental encoder simulation output that may be present.
