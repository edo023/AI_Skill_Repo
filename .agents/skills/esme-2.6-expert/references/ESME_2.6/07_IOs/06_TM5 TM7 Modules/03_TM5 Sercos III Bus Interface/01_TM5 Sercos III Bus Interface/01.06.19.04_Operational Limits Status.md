# Operational Limits Status

## General

The output supply of the module is monitored. An I/O supply voltage < 20.4 V is indicated as alert message.

| Bit | Description |
| --- | --- |
| 0–1 | 0 |
| 2 | 0 = I/O supply exceeding the alert limit of 20.4 V  1 = I/O supply below the alert limit of 20.4 V |
| 3–7 | 0 |
