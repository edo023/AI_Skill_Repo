# Limitation to Updating I/O Variables

Limitation to Updating I/O Variables

The output channels of the TM5SE1IC20005 / TM5SE1MISC20005 modules must not be written by the application. To prevent the output channels from being written by the application, directly mapping the output channels of the device to the I/O variables is not possible.

Mapping the output channels of a device to the I/O variables used by the application is done on the I/O Mapping index card in the device editor of the module. (The device editor for the module is opened by double-clicking the module in the Devices tree.) To prevent the output channels of the TM5SE1IC20005 / TM5SE1MISC20005 modules from being written, the output channels are not visible on the I/O Mapping index cards of the TM5 modules.

How the I/O variables (and the mapped output channels) are updated in the bus cycle task is defined with the Always update variables option. This option can be set in the module-specific I/O Mapping index card and also the PLC Settings index card of the parent device. (The parent device of the TM5SE1IC20005 / TM5SE1MISC20005 modules is the LMC device.)

For the TM5SE1IC20005 / TM5SE1MISC20005 modules, there are two settings for the Always update variables option that are incompatible:

oDo not set the option on the module-specific I/O Mapping index card to Enabled 1 or Enabled 2.

oDo not set the Always update variables option on the PLC Settings index card of the parent device to Enabled 1 or Enabled 2 and the option on the module-specific I/O Mapping index card to Use parent device setting.

In both cases, a compiler message is generated indicating that the Always update variables must not be enabled for the device.

Example of the compiler message for TM5SE1MISC20005:

[ERROR] TM5SE1MISC20005:

TM5SE1MISC20005 [LMC\_PacDrive: Sercos\_Master: BC\_TM5NS31]: 'Always update variables' must not be enabled for the device 'TM5SE1MISC20005'. Check 'Always update variables' in the device and inside 'PLC settings' of the parent controller.

For a detailed description of the Always update variables options, see the descriptions of the corresponding index cards in the EcoStruxure Machine Expert online help.
