# Sw Handshake Period

## General

This register configures a repeat time for resending the current XON/XOFF character. This can remove blockages of the interface.

| Name | Meaning |
| --- | --- |
| Sw Handshake Period | 0 = Automatic status repetition disabled (default)  500...10000 = Repeat time (ms) |
