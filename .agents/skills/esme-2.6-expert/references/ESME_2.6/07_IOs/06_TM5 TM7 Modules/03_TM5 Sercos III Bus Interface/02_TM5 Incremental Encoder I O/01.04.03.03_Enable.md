# Enable

![](images/logo.png)

Enable

General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | SinCos Encoder Input,  Incremental Encoder Input |
| Traceable | Yes |

Functional Description

Enables or disables the processing of the encoder by the real-time process.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOLEAN | The encoder is not processed by the real-time process.  The physical master encoder always deactivates when it is not connected. Detection takes place once the system is started. |
| on / 1 | BOOLEAN | The encoder is processed by the real-time process. |
