# General

## Description

Two analog outputs ±10 V / 0...20 mA, resolution 16 bit

Analog output module 24 Vdc with 2x16 bit analog outputs (voltage ±10 Vdc / current...20 mA, differential)

## Configuration Register

| Register | Meaning | Default | Data type |
| --- | --- | --- | --- |
| 0 | [Channel type for analog outputs (TM5SAO2H)](D-SE-0070396.html#D-SE-0070396) | 0 | UINT |
| 2 | Analog output 01 | ±10 Vdc | INT |
| 4 | Analog output 02 | ±10 Vdc | INT |
| 257 | [Error states analog outputs](D-SE-0070395.html#D-SE-0070395) | – | USINT |
| 457 | [SDCLifeCount](D-SE-0070398.html#D-SE-0070398) | – | SINT |
| 594 | [Validation operating mode analog output 01](D-SE-0070397.html#D-SE-0070397) | 0 | UINT |
| 598 | Validation operating mode analog output 02 | 0 | UINT |
| 802 | Validation timer 01, 16 bits | Off | INT |
| 804 | Validation timer 01, 32 bits | Off | DINT |

| 812 | Validation timer 02, 32 bits | Off | DINT |
| 833 | [Disable/enable validation](D-SE-0070394.html#D-SE-0070394) | 0 | USINT |
