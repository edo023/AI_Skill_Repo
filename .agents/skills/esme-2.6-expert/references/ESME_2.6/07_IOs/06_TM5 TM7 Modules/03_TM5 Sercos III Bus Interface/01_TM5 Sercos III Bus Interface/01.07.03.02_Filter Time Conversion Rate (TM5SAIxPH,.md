# Filter Time / Conversion Rate (TM5SAIxPH, TM5SAIxTH)

## General

| HW input filter adjustable | Conversion rate |
| --- | --- |
| Filter value 0 | 15 S/Sec (66.7 mSec) |
| Filter value 1 | 25 S/Sec (40 mSec) |
| Filter value 2 | 30 S/Sec (33.3 mSec) |
| **Filter value 3 (default)** | **50 S/Sec (20 mSec)** |
| Filter value 4 | 60 S/Sec (16.7 mSec) |
| Filter value 5 | 100 S/Sec (10 mSec) |
| Filter value 6 | 500 S/Sec (2 mSec) |
| Filter value 7 | 1000 S/Sec (1 mSec) |

Changing the filter time during operation will trigger a calibration of the current conversion, thus the conversion time is doubled.
