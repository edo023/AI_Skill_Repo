# General

## Description

4 AI; 4 AI;–; 4 AO; 4 AO

Compact I/O module 24 Vdc with 2x4 analog inputs and 2x4 analog outputs, ±10 V 12 bits

The module consists of the following five inseparable submodules:

* One submodule (slot 1 + slot 2) with four analog voltage inputs each
* One dummy module (slot 3)
* One submodule (slot 4 + slot 5) with four analog voltage outputs each

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM5 Module I/O display, User-defined parameters, and Information are displayed. |

## Tab TM5 Modules I/O Display

The input and output channels of the module are mapped to project variables that are used by the application.

| Channel | Meaning | Default value | Data type |
| --- | --- | --- | --- |
| Analog inputs (slot 1+2) | SL1 analog input 01  SL1 analog input 02  SL1 analog input 03  SL1 analog input 04  SL2 analog input 01  SL2 analog input 02  SL2 analog input 03  SL2 analog input 04 | – | INT |
| Dummy module (slot 3) | – | – |  |
| Analog outputs (slot 4+5) | SL4 analog output 01  SL4 analog output 02  SL4 analog output 03  SL4 analog output 04  SL5 analog output 01  SL5 analog output 02  SL5 analog output 03  SL5 analog output 04 | – | INT |
| [Status of the analog inputs](D-SE-0070189.html#D-SE-0070189) SL1 | Status of the analog inputs SL1 | – | INT |
| [Status of the analog inputs](D-SE-0070189.html#D-SE-0070189) SL2 | Status of the analog inputs SL2 | – | INT |

The analog inputs from slot 1/slot 2 and the analog outputs from slot 4/slot 5 are configured fix for ±10 V voltage signals. This is why these analog in-/outputs cannot be configured optional as voltage- or current signals and therefore they are not displayed in the register card User-defined parameters.

For further information, see the TM5 compact I/O modules hardware manual.
