# Resolution

![](images/logo.png)

Resolution

General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Incremental Encoder Input |
| Traceable | No |

Functional Description

Specifies the resolution of the connected encoder (see encoder nameplate).

NOTE: The incremental encoder input functions internally increase resolution by quadrupling the impulse. The system automatically takes this into account.

NOTE: Modify the Resolution parameter value at encoder standstill and when the program is inactive. Otherwise, the [velocity signal](EncoderInput-16.htm#XREF_D_SE_0069592_1) appears incorrect, which can cause other detected errors.
