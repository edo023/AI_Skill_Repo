# Sensor Type (TM5SAIxTH)

## General

Arrangement of the sensor type for the channels

| Bit | Description | Gain |
| --- | --- | --- |
| 0–6 | Defines the sensor type:  0000000 = Conversion is off  0000001 = Sensor type J  0000010 = Sensor type K  0000011 = Sensor type S  0000100 = Sensor type N  0000101 = Conversion is off  00000110 = 1 μV per bit  00000111 = 2 μV per bit  01000000 = Sensor type R (FW:832)  01001000 = Sensor type B (FW:832) | –  –  2  2  4  2  –  4  2  4  4 |
| 7 | 0 | – |

Changing the sensor type during operation resets the register to invalid value and cause a recalibration of the converter.

EIO0000002196.02
