# General

## Description

Two analog inputs ±10 V / 0...20 mA, resolution 12 bit

Analog input module 24 Vdc with 2x12 bit analog inputs (voltage ±10 Vdc / current 0...20 mA or 4...20 mA, differential)

## Configuration Register

| Register | Meaning | Default | Data type |
| --- | --- | --- | --- |
| 0 | Analog input 1 | -10 Vdc to +10 Vdc | INT |
| 2 | Analog input 2 | -10 Vdc to +10 Vdc | INT |
| 16 | [Analog input filter](D-SE-0070368.html#D-SE-0070368) | Filter disabled, no limitation | USINT |
| 18 | [Channel type for analog input](D-SE-0070369.html#D-SE-0070369) | 0 | USINT |
| 20 | Lower limit value input 1–2 ([Upper or lower limit value](D-SE-0070370.html#D-SE-0070370)) | -32767 | INT |
| 22 | Upper limit value input 1–2 ([Upper or lower limit value](D-SE-0070370.html#D-SE-0070370)) | 32767 | INT |
| 30 | [Status of the analog inputs 1–2](D-SE-0070371.html#D-SE-0070371) | – | USINT |

For further information, see the TM5 analog I/O modules hardware manual.
