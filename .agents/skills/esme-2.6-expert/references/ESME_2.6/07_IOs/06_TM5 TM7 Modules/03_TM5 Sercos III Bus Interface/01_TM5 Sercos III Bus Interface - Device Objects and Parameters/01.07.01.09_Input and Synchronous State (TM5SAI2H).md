# Input and Synchronous State (TM5SAI2H)

## General

This register collects diagnostic messages in synchronization with the network cycle.

Temporary error states which have been registered during a conversion cycle stay active for at least two network cycles.

In order to obtain more detailed information on the detected errors, the error counters as well as the X2X network events must be observed.

| Bit | Description |
| --- | --- |
| 0 | Status input 01  0 = ok  1 = detected error |
| 1 | Status input 02  0 = ok  1 = detected error |
| 2 | Reserved |
| 3 | Reserved |
| 4 | Reserved |
| 5 | Reserved |
| 6 | Status synchronization X2X to conversion cycle  0 = synchronization ok  1 = not synchronized |
| 7 | Status conversion cycle  0 = ok  1 = detected error |
