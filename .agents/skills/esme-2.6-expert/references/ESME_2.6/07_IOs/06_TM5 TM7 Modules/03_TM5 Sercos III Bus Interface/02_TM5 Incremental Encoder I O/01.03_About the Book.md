About the Book

Document Scope

This document describes the parameters of the Incremental Encoder Input object on the TM5SE1IC20005 module and the Incremental Encoder Output object on the TM5SE1MISC20005 module.

The Incremental Encoder Input object is automatically inserted as child of each TM5SE1IC20005 module. The Incremental Encoder Output object is automatically inserted as child of each TM5SE1MISC20005 module.
