# TM5LinkActiveDelayMs

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Bus Interface TM5NS31 |
| Traceable | Yes |

## Functional Description

With this command, you can set the waiting time [in ms] for the TM5 module scan.

| Value | Data type | Meaning |
| --- | --- | --- |
| 3000–60000  (default: 3000) | UINT | Waiting time for the TM5 module scan [ms] |
