# General

## Description

Safety digital input module, 24 Vdc

* Four safety-related inputs
* Four safety-related test outputs
* Configurable input filter can be set by channel

## Configuration

| Tab | Meaning |
| --- | --- |
| [TM5 Module I/O Mapping](D-SE-0070414.html#D-SE-0070414) | Mapping of channels to project variables |
| [TM5 Module Parameters](D-SE-0070415.html#D-SE-0070415) | Safety-related configuration parameters |

For further information, see the TM5/TM7 Digital I/O Safety Modules Hardware Guide.
