# General

## Description

Two resistance temperature inputs

Analog temperature input module 24 Vdc with 2x16 bit resistance thermometer inputs (2-wire/3-wire, PT100/PT1000)

## Configuration Register

| Register | Meaning | Default | Data type |
| --- | --- | --- | --- |
| 0 | Input 01 | Inactive | INT |
| 2 | Input 02 | Inactive | INT |
| 16 | Filter time/sample rate ([Filter time/conversion rate (TM5SAIxPH, TM5SAIxTH)](D-SE-0070372.html#D-SE-0070372)) | 3 | USINT |
| 18 | [Sensor type (TM5SAIxPH)](D-SE-0070373.html#D-SE-0070373) | 0xFF | USINT |
| 28 | I/O cycle counter | Ein | USINT |
| 30 | [Status of analog inputs 1–2](D-SE-0070374.html#D-SE-0070374) | – | USINT |

For further information, see the TM5 analog I/O modules hardware manual.
