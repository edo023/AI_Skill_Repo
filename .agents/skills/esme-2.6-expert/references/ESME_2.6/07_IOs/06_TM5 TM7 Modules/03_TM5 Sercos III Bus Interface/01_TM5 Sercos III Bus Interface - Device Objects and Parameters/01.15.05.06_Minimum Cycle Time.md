# Minimum Cycle Time

## General

The minimum cycle time is the minimum time needed for the TM5-bus cycle to be shut down without a communication interruption or function-impairments occurring. Please note that the remaining time to deal with monitoring, diagnostic and anti-cyclical commands is reduced by very fast cycles.

The cycle time can be set at the bus interface. Least cycle time = 250 µs.
