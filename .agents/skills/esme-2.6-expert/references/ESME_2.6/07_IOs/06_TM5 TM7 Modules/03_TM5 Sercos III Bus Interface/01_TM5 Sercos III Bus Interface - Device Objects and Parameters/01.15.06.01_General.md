# General

## Description

Eight digital channels (as inputs or outputs), 24 Vdc, 0.5 A, inputs for sink-wiring, outputs for source-wiring, configurable input filter, LEDs to indicate the status.

Digital mixed module 24 Vdc with eight channels, configurable as input or output via software, M8 connection technology

* Eight digital channels, configurable as input or output
* Short-circuit-proof outputs
* The outputs are equipped with a single channel diagnostic
* Input delay can be parameterized

## Parameterizing the Module Via PLC Configuration

In order to parameterize a module per PLC Configuration in the device editor, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Append module in the Devices tree beneath a Sercos III bus interface (TM5NS31). |
| 2 | Afterwards double-click the module in the Devices tree.  **Result:** The device editor is opened and the tabs TM7 Module I/O display, User-defined parameters, and Information are displayed. |
