# General

## Description

Four analog outputs ±10 V / 0...20 mA, resolution 12 bit

Analog output module 24 Vdc with 4x12 bit analog outputs (voltage ±10 Vdc / current 0...20 mA, differential)

## Configuration Register

| Register | Meaning | Default | Data type |
| --- | --- | --- | --- |
| 0 | Output 01 | ±10 Vdc | INT |
| 2 | Output 02 | ±10 Vdc | INT |
| 4 | Output 03 | ±10 Vdc | INT |
| 6 | Output 04 | ±10 Vdc | INT |
| 18 | [Channel type for analog outputs 1-4 (TM5SAO4L)](D-SE-0070401.html#D-SE-0070401) | 0 | USINT |

For further information, see the TM5 analog I/O modules hardware manual.
