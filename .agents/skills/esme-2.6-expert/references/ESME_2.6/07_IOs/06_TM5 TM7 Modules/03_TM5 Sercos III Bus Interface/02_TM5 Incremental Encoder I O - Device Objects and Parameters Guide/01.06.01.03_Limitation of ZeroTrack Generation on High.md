# Limitation of ZeroTrack Generation on High Speeds for TM5SE1MISC20005

Limitation of ZeroTrack Generation on High Speeds for TM5SE1MISC20005

The TM5SE1MISC20005 encoder output module has a limitation on ZeroTrack generation with high velocities.

The ZeroTrack cannot be generated if the speed is greater than 1 revolution per 5 cycles. This leads to a maximum speed of:

o12,000 revolutions/minute (1 ms cycle time)

o6,000 revolutions/minute (2 ms cycle time)

o3,000 revolutions/minute (4 ms cycle time)

The TM5SE1MISC20005 encoder output module detects when the ZeroTrack cannot be generated correctly anymore and generates a diagnostic message of class 2 to inform about the exceedance of the limit:

|  |  |
| --- | --- |
| DiagClass: | 2 (Warning) |
| DiagCode: | 8829 |
| DiagMsg: | Velocity too high for ZeroTrack |
