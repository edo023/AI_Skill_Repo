# Counter

## General

The counter 0–1 0 is destined for event counter operation.

For the counter 0–1 1, the following operating modes can be selected.

* Event counter operation
* Dead time measurement

## Event Counter Operation

The rising edges are recorded at the counter input.

The counter status is registered with a fixed offset in relation to the network cycle and transferred in the same cycle.

## Gating Time Measurement

Records the time between the rising and the falling edge of the gate input with an internal frequency. The result is checked for overflow ($FFFF) and corrected in accordance with the defined prescalers.

The recovery time between the measurements has to be > 00 μs.

The measuring result is transferred to the result memory with the falling edge.
