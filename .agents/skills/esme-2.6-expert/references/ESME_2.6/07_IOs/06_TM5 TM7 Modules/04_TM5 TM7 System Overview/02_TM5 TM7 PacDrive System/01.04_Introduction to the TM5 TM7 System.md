# Introduction to the TM5 / TM7 System

## Overview

This manual provides an overview of the TM5 / TM7 System:

* The TM5 System is comprised of IP20-rated components with which you can create [distributed and/or remote I/O architectures](D-SE-0015387.html#D-SE-0015387__D-SE-0015387.29).

  A typical TM5 System includes a controller, a Sercos III Bus Interface, [slices](D-SE-0015379.html#D-SE-0015379) and accessories. These components must be installed in enclosures appropriate to the intended operating environment and secured by a keyed or tool locking mechanism.
* The TM7 System is comprised of IP67-rated components with which you can create distributed and/or remote I/O architectures.

  A TM7 System can be expanded from the Sercos III Bus Interfaces through transmitter modules and including I/O blocks, expansion blocks and accessories which can be used in environments conforming to IP67 (splashing water, oil, dust, etc...).
