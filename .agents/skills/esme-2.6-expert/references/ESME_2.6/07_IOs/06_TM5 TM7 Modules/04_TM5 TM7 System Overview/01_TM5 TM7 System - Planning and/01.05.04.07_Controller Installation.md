Controller Installation

Introduction

The following procedure describes how to mount a controller on the DIN rail.

NOTE: If the controller is already installed and wired, or the slice connectors are prewired, be sure to remove all power before attempting these procedures.

Controller Installation

The following table gives step by step instructions to install a controller on a DIN rail:

| Step | Action | |
| --- | --- | --- |
| 1 | Push all the locking levers all the way up. This opens the locking mechanism for controller installation. | G-SE-0001004.2.gif-high.gif |
| 2 | Hang the controller in the desired position on the mounting rail. | G-SE-0001005.2.gif-high.gif |
| 3 | Close the locking mechanism by pushing all levers down. | G-SE-0001003.2.gif-high.gif |
