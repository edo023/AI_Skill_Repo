# PDMs Mounting Rules

PDMs Mounting Rules

Installing a PDM according to these rules automatically establishes a new 24 Vdc I/O power segment for I/O expansions to the right of the PDM.

The TM5SPS2• PDM can be placed at the end of the configuration. In this case, it supplies only the TM5 power bus.

Rules:

oDo not mount PDMs side by side

oDo not mount a PDM and a Transmitter or Receiver module side by side

oDo not mount a PDM and an Interface Power Distribution Module (IPDM) side by side

oDo not mount a PDM next to any of the following modules: TM5SAI2H, TM5SAI4H, TM5SAO4L or TM5SAO4H

[EcoStruxure Machine Expert](../glossary/glossary.htm#XREF_D_SE_0024697_595) software is designed to help prevent the installation of incompatible modules side by side.
