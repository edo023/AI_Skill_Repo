# Diagnostics

Diagnostics

The TM7 System offers several levels of diagnostics depending on the type of the block using [LED](../glossary/glossary.htm#XREF_D_SE_0024697_302) displays.

Refer to [TM7 Digital I/O Blocks Hardware Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=tm7diohw&topicID=D_SE_0008132_15) , [TM7 Analog I/O Blocks Hardware Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=tm7aiohw&topicID=D_SE_0007625_15) and/or Modicon TM5/TM7 I/O Safety Modules Hardware Guide for the products of the TM7 System for status LEDs descriptions.
