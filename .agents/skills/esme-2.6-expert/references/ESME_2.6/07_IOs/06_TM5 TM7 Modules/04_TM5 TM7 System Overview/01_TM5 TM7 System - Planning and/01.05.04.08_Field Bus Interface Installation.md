Field Bus Interface Installation

Introduction

The installation procedure of the field bus interface is to assemble the 4 components and then install it directly on the DIN rail.

NOTE: If the field bus interface is already installed and wired, or the connectors are pre-wired, be sure to remove all power before attempting these procedures.

Field Bus Interface Installation

The following procedure gives step by step instructions to assemble and install a field bus interface on a DIN rail:

| Step | Action | |
| --- | --- | --- |
| 1 | Remove bus base, the electronic modules and terminal block from protective packaging. Verify for visible mechanical damage. | |
| 2 | Insert interface electronic module in the [slot of the bus base](../SPIG_TM5_TM7_-_Basics_of_the_TM5_System/SPIG_TM5_TM7_-_Basics_of_the_TM5_System-3.htm#XREF_D_SE_0015378_5). | G-SE-0007495.1.gif-high.gif |
| 3 | Push the interface electronic module straight into the bus base until it clicks. | |
| 4 | Insert the IPDM electronic module in the [guides of the bus base](../SPIG_TM5_TM7_-_Basics_of_the_TM5_System/SPIG_TM5_TM7_-_Basics_of_the_TM5_System-3.htm#XREF_D_SE_0015378_5). | G-SE-0007493.1.gif-high.gif |
| 5 | Push the electronic module straight into the bus base until it clicks. | |
| 6 | Hang the bottom edge of the terminal block in its [hinge on the bus base](../SPIG_TM5_TM7_-_Basics_of_the_TM5_System/SPIG_TM5_TM7_-_Basics_of_the_TM5_System-3.htm#XREF_D_SE_0015378_5). | G-SE-0007577.1.gif-high.gif |
| 7 | Rotate upward. | G-SE-0007578.1.gif-high.gif |
| 8 | Press the terminal block until it clicks in place. | G-SE-0007576.1.gif-high.gif |
| 9 | Push the locking lever all the way up on all of the bus base. This opens the locking mechanism. | |
| 10 | Hang the fieldbus interface in the desired position on the mounting rail. | G-SE-0007494.1.gif-high.gif |
| 11 | Close the locking mechanism by pushing the lever down. | |
