# Installation-Phase 2

Installation-Phase 2

In the second phase of the installation, you install the electronic modules and terminal blocks with or without coding:

| Step | Action | For Details, see |
| --- | --- | --- |
| 1 | Develop a coding scheme for the electronic modules that matches the expansion modules layout.  NOTE: The safety-related I/O slices do not support the coding of the electronic modules. | [Coding the TM5 System](TM5_-_Flexible_TM5_System_Installation-13.htm#XREF_D_SE_0000888_1) |
| 2 | Install the electronic modules in their bus bases in accordance with your slice layout.  NOTE: To install a compact I/O block, remove the terminal block from the preceding device (controller, power distribution or I/O slice), then install the compact I/O block.  Refer to [Mounting positions](../TM5_-_Initial_Planning_for_TM5/TM5_-_Initial_Planning_for_TM5-5.htm#XREF_D_SE_0001558_7). | [Inserting the Electronic Module](TM5_-_Flexible_TM5_System_Installation-9.htm#XREF_D_SE_0001757_5) |
| 3 | Install a left bus base locking plate to the first slice of each remote configuration. | [Mounting the Left Bus Base Locking Plate](TM5_-_Flexible_TM5_System_Installation-9.htm#XREF_D_SE_0001757_7) |
| 4 | Install a right bus base locking plate to the rightmost expansion module of each configuration. | [Mounting the Right Bus Base Locking Plate](TM5_-_Flexible_TM5_System_Installation-9.htm#XREF_D_SE_0001757_8) |
| 5 | Mount the terminal blocks in accordance with your expansion modules layout. | [Mounting the Terminal Blocks](TM5_-_Flexible_TM5_System_Installation-9.htm#XREF_D_SE_0001757_6) |
