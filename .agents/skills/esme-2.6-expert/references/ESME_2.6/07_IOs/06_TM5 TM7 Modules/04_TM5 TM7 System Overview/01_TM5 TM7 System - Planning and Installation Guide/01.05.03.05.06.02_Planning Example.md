# Planning Example

Planning Example

This configuration example includes:

oThe controller TM258LF42DT equipped with embedded DI12DE and DO12TE electronic modules.

oSome expansion modules:

oTM5SAI4L

oTM5SDI12D

oTM5SDI4D

oTM5SDM12DT

oTM5SDI4A

oTM5SDO4TA

oTM5SDO12T

oTM5SDO2T

oTM5SDO2R

oAssumptions used for the purposes of calculating the consumption of this example:

CPDM: The maximum current distributed on the 24 Vdc I/O power segment is limited by an external isolated power supply of 6300 mA.

DO12TE: Only 50 % of the outputs are active at any given time, and that the maximum current draw of any given output is 500 mA, or 3000 mA total for the module.

TM5SDI4D: The current to supply the electronic sensors of this example has been estimated at 50 mA per sensor, or 200 mA total for the module.

TM5SDM12DT: The sum of the current draw for all outputs connected to the module is never more than 1500 mA at any given time.

TM5SDO12T: Only 50% of the outputs are active at any given time, and that the maximum current draw of any given output is 500 mA, or 3000 mA total for the module.

TM5SDO4TA: The sum of the current draw for all outputs connected to the module is never more than 2000 mA at any given time.

TM5SSDO2T: The sum of the current draw for all outputs connected to the module is never more than 600 mA at any given time.
