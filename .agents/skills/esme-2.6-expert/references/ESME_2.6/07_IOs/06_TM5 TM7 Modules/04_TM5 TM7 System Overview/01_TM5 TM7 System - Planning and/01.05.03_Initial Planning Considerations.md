# Initial Planning Considerations

![](images/logo.png)

Initial Planning Considerations

Overview

This chapter provides information that is helpful in the early planning stages for a TM5 System. It includes the requirements for enclosing the TM5 System in a protective housing and for determining the type of power supply source required for your configuration.
