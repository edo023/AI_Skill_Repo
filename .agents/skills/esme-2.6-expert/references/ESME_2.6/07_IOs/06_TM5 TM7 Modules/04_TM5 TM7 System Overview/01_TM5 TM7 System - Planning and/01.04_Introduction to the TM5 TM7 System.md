# Introduction to the TM5 / TM7 System

![](images/logo.png)

Introduction to the TM5 / TM7 System

Overview

This manual provides an overview of the TM5 and TM7 system:

oThe TM5 System is comprised of [IP20](../glossary/glossary.htm#XREF_D_SE_0024697_444)-rated components with which you can create local, remote and / or distributed I/O architectures. A typical TM5 System includes a controller and may include field bus interface, slices, compact [I/O](../glossary/glossary.htm#XREF_D_SE_0024697_469) and accessories. These components must be installed in enclosures appropriate to the intended operating environment.

oThe TM7 System is comprised of [IP67](../glossary/glossary.htm#XREF_D_SE_0024697_517)-rated components with which you can create remote and / or distributed I/O architectures. TM7 System including field bus interface I/O blocks, expansion blocks and accessories can be used in environments conforming to [IP67](../glossary/glossary.htm#XREF_D_SE_0024697_517) (splashing water, oil, dust, etc...).
