# How to Install the Label Tabs for Coding

How to Install the Label Tabs for Coding

The following table describes how to code the terminal block and the electronic module:

| Step | Action |  |
| --- | --- | --- |
| 1 | Grip the desired label tab with the single-width cutters of the labeling tool. | G-SE-0003788.1.gif-high.gif |
| 2 | Press with the labeling tool to separate the label. | G-SE-0003787.1.gif-high.gif |
| 3 | Center the label tab over [the slot](../SPIG_TM5_TM7_-_Basics_of_the_TM5_System/SPIG_TM5_TM7_-_Basics_of_the_TM5_System-5.htm#XREF_D_SE_0015379_6) on the electronic module. | G-SE-0000456.1.gif-high.gif |
| 4 | Hold the labeling tool at a 90° angle to the electronic module and press to insert the label's feet into the slot.  NOTE: Repeat step 1 and 2 to remove a label tab with the single-width cutter of the labeling tool. | |
| 5 | Set the label tab in [the slot](../SPIG_TM5_TM7_-_Basics_of_the_TM5_System/SPIG_TM5_TM7_-_Basics_of_the_TM5_System-5.htm#XREF_D_SE_0015379_7) on the back of the terminal block as shown. | G-SE-0000457.1.gif-high.gif |
| 6 | Use the labeling tool to push the left feet of the label into the slot. | G-SE-0000458.1.gif-high.gif |
| 7 | With the labeling tool, press the right feet of the label into the slot.    Result: Inserted label for terminal coding. | G-SE-0000459.1.gif-high.gif        G-SE-0000460.1.gif-high.gif |
