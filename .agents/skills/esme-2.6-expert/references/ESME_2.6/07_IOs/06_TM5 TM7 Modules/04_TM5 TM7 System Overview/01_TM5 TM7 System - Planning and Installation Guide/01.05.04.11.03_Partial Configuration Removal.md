# Partial Configuration Removal

Partial Configuration Removal

| Step | Action | |
| --- | --- | --- |
| 1 | Remove all power from all equipment. | |
| 2 | For mechanical reasons, remove the terminal block of the electronic module at the left of the partial configuration to be removed. To do this:  1   push down on the locking lever on the terminal block  2   rotate the terminal block out and down | G-SE-0001448.1.gif-high.gif |
| 3 | Push the locking levers of the configuration to be removed all the way up. This opens the locking mechanism for equipment installation. | G-SE-0001004.2.gif-high.gif |
| 4 | Remove the partial configuration from the mounting rail.  G-SE-0002033.1.gif-high.gif | |
| 5 | [Put the removed terminal block](TM5_-_Flexible_TM5_System_Installation-9.htm#XREF_D_SE_0001757_6) back on the electronic module. | |
