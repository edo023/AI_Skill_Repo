# Output Characteristics

Output Characteristics

The table below describes the output characteristics of the TM7SPS1A block:

| Output Characteristics | |
| --- | --- |
| Rated output power | 15 W |
| Rated output voltage | 20 Vdc |
| Output current | 750 mA maximum |
| Internal protection | Overload and short circuit |
| Power Supply Outage | 5 ms minimum at 24 Vdc input voltage and maximum output current |
| Parallel operation | Yes, protection with redundancy during parallel operation of multiple PDBs |
| Redundant operation | Yes, if connected to the same input power supply |
