# IPDM Mounting Rules

IPDM Mounting Rules

Rules:

oDo not mount a PDM next to an IPDM

oDo not mount a Transmitter or Receiver module next to an IPDM

oDo not mount any of the following modules: TM5SAI2H, TM5SAI4H, TM5SAO4L or TM5SAO4H next to an IPDM

Performance Distributed I/O Configuration software is designed to help prevent the installation of incompatible modules side by side.
