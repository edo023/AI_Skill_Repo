# Installation-Phase 3

Installation-Phase 3

In the final phase, you install cable ducts, connect all grounding points, make the necessary signal and power connections, and commission your TM5 System.

| Step | Action | For Details, see |
| --- | --- | --- |
| 1 | Install the TM2XMTGB grounding plate. | [Grounding the TM5 System](../TM5_-_Initial_Planning_for_TM5/TM5_-_Initial_Planning_for_TM5-17.htm#XREF_D_SE_0002601_1) |
| 2 | Install cable ducts, conduits and any wiring harnesses. | – |
| 3 | Make Functional Ground ([FE](../glossary/glossary.htm#XREF_D_SE_0024697_704)) connections. | [Functional Ground (FE) Cable Shielding](../TM5_-_Initial_Planning_for_TM5/TM5_-_Initial_Planning_for_TM5-17.htm#XREF_D_SE_0002601_11) |
| 4 | Make Protective Ground ([PE](../glossary/glossary.htm#XREF_D_SE_0024697_501)) connections. | [Protective Ground (PE) Cable Shielding](../TM5_-_Initial_Planning_for_TM5/TM5_-_Initial_Planning_for_TM5-17.htm#XREF_D_SE_0002601_10) |
| 5 | Make the field wiring connections. | Specific Hardware Guides |
| 6 | Make the power connections. | [Wiring the Power Supply](../TM5_-_Initial_Planning_for_TM5/TM5_-_Initial_Planning_for_TM5-16.htm#XREF_D_SE_0002067_1) |
| 7 | Reduce stress on the wires with adequate cable ties. | TM5 Strain Relief Using Cable Tie |
| 8 | Commission the TM5 System. | Configuring the TM5 System (refer to your controller programming guide) |
