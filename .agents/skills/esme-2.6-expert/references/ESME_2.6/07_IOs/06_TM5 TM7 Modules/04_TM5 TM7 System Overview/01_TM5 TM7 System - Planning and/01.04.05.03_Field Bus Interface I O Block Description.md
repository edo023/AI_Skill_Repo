Field Bus Interface I/O Block Description

Field Bus Interface I/O Blocks

The TM7 field bus interface I/O block is the first element of a [TM7 distributed I/Os](../Intro_-_Description_of_the_TM5_and_TM7_System/Intro_-_Description_of_the_TM5_and_TM7_System-3.htm#XREF_D_SE_0009280_4) and allows distributed connection of sensors and actuators via a field bus.

Additionally to the field bus interface function, the TM7 field bus interface I/O blocks provides:

oan integrated [power supply](#XREF_D_SE_0009232_6).

o[digital connections](#XREF_D_SE_0009232_7), which can be configured as inputs or outputs.

Each model of field bus interface I/O block is distinguished by the field bus connectors, the number of I/O and its overall [size (1 or 2)](../TM7_Part_-_Initial_Planning_for_TM7_System/TM7_Part_-_Initial_Planning_for_TM7_System-3.htm#XREF_D_SE_0015399_7).

24 Vdc Power Distribution

The distribution of power by the field bus interface I/O block consists of two dedicated circuits:

| Designation | Description |
| --- | --- |
| 24 Vdc Main power | 24 Vdc power that serves the electronics of the field bus interface I/O block and generates independent power for the TM7 power bus that serves the TM7 I/O blocks. |
| 24 Vdc I/O power segment | The 24 Vdc power that serves:  othe electronics of the I/O block,  othe sensors and actuators connected to the field bus interface I/O block,  othe expansion blocks,  othe sensors and actuators connected to the expansion blocks. |
