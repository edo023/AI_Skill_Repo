TM5SD000 Dummy Module

General Information

TM5SD000 dummy module is a non-functional module. This module is used as a place holder for later system expansion according to the [TM5 association table](../TM5_TM7_-_Tables/TM5_TM7_-_Tables-2.htm#XREF_D_SE_0000785_1).
