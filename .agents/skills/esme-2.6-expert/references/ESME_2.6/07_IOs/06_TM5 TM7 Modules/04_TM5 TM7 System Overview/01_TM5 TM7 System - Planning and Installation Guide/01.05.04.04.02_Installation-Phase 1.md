# Installation-Phase 1

Installation-Phase 1

In the first phase of the installation, you install the DIN rail, the controller or the field bus interface and any bus bases for any expansion modules that may be part of your TM5 System configuration:

| Step | Action | Refer to |
| --- | --- | --- |
| 1 | Develop an installation plan that covers all aspects of the installation. | [Making a Plan](TM5_-_Flexible_TM5_System_Installation-4.htm#XREF_D_SE_0002466_2) |
| 2 | Fasten the DIN rail to the mounting plate of the enclosure. | [DIN Rail Installation](TM5_-_Flexible_TM5_System_Installation-5.htm#XREF_D_SE_0002436_2) |
| 3 | For local configuration, install the controller at the first (leftmost) location on the rail or for distributed configuration install the field bus interface at the first (leftmost) location on the rail.  NOTE: In case of vertical installation the controller or the field bus interface must be at the lowest location and secured.  Refer to [Acceptable Mounting positions](../TM5_-_Initial_Planning_for_TM5/TM5_-_Initial_Planning_for_TM5-5.htm#XREF_D_SE_0001558_7).  NOTE: Go to [Installation-Phase 2](#XREF_D_SE_0002062_4) if there are no expansion modules. | [Controller Installation](TM5_-_Flexible_TM5_System_Installation-6.htm#XREF_D_SE_0001603_4) or [Field Bus Interface Installation](TM5_-_Flexible_TM5_System_Installation-7.htm#XREF_D_SE_0009760_1) |
| 4 | Determine the left-to-right arrangement of the expansion modules on the rail. | – |
| 5 | Attach the bus base (for slices) and compact I/O to the DIN rail in accordance with your expansion modules layout, working left to right from the controller (step 3, above). | [Mounting the Bus Bases](TM5_-_Flexible_TM5_System_Installation-9.htm#XREF_D_SE_0001757_4)  or [Compact I/O Installation](TM5_-_Flexible_TM5_System_Installation-8.htm#XREF_D_SE_0010969_1) |
| 6 | Optional: Assign the module addresses in accordance with your expansion modules layout. | [Addressing](TM5_-_Flexible_TM5_System_Installation-12.htm#XREF_D_SE_0001880_1) |
