# Current Supplied and Consumption Tables on the TM7 Power Bus

## Introduction

The following table gives a synthesis of the current supplied and current consumed on the TM7 power bus by the following TM5 / TM7 System components:

* TM5SBET7 Transmitter module,
* TM7 System expansion blocks (Field bus interface I/O blocks, I/O blocks and Power Distribution Block).

## TM5SBET7 Transmitter Module

For information on the current supplied to the TM7 power bus, refer to [Supplying the TM7 Power Bus](D-SE-0009310.html#D-SE-0009310).
