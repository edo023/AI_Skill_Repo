# Setting the Sercos Address

Set the rotary switches before:

* Applying power to the module.
* Downloading the configuration.

NOTE: Any modification of the rotary switch position after power up is not taken into account until the next power up.

This table describes the configuration of the rotary switches:

| Address | Description |
| --- | --- |
| 0 dec (0 hex) | 0 is not a valid Sercos address.  When both rotary switches are set to position 0, the Sercos module automatically sets a valid Sercos default address. This address can be reset by the controller during Sercos phase-up. |
| 1...255\* dec (1...FF hex) | Manual addressing |
| **\*** Only addresses from 1 to 239 are supported. All values from 240 to 255 are set automatically to 239. | |
