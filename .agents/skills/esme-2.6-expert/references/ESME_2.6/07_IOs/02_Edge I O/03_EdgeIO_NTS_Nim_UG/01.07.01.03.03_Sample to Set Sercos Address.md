# Sample to Set Sercos Address

The following sample illustrates the configuration of the Sercos address to 181 (decimal):

* **X16**: High-order rotary switch is set to 11 (decimal)
* **X1**: Low-order rotary switch is set to 5 (decimal)

The Sercos address is: 16 x 11 + 1 x 5 = 181.

EIO0000004794.02
