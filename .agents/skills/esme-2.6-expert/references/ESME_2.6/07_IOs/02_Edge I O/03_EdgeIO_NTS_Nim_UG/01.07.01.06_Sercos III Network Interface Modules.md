# Sercos III Network Interface Modules Configuration

## Configurable Parameters

For each Sercos III network interface module, you can configure the following parameters:

| Displayed Name  *Parameter Name* | Value(s) | Data type | Description |
| --- | --- | --- | --- |
| NIM IP Address  IPConfIPAddress | 0.0.0.0\* | BYTE | NIM IP Address. |
| NIM IP Subnet Mask  IPConfSubnetMask | 0.0.0.0\* | BYTE | NIM IP Subnet Mask.  NOTE: When there is no gateway configured, the device can communicate on its subnetwork only. |
| NIM IP Gateway Address  IPConfGateway | 0.0.0.0\* | BYTE | Gateway address of the NIM. |
| IO Profile  IOProfile | Standard\*  Optimized | ENUM | In standard mode, the entire set of implicit data is sent.  Some parameters are not part of the implicit data when the optimized I/O profile is selected. |
| IO Bus Cycle Time  CycleTime | 1 ms\*  2 ms  4 ms | ENUM | Defines the period of the internal implicit data exchange between the NIM and the I/O modules.  In each I/O bus cycle the NIM transmits the latest output image to the I/O modules and the I/O modules send their input image to the NIM. For Sercos III, the I/O bus cycle is synchronized with the Sercos III cycle and both cycle times must be identical.  NOTE: Set the IO Bus Cycle Time in accordance with the Sercos III bus cycle time. |
| USB IP address  USBIPAddress | 192.168.200.1\* | IP Address | Configures USB IP address. |
| USB Subnet Mask  USBSubnetMask | 255.255.255.0\* | IP Address | Configures USB Subnet Mask. |
| \* Parameter default value | | | |

## Implicit Data

The following table shows the implicit data:

| *Parameter Name* | Value | Data type  Size in bytes | Description |
| --- | --- | --- | --- |
| IO Status | 0...65,535 | BYTE  2 | Displays the Sercos III I/O status word. |
| NIM\_GCS | 0...255 | BYTE  1 | Group Cyclic Status  Bit 0: N/A  Bit 1: General module status  Bit 2: N/A  Bit 3: N/A  Bit 4: N/A  Bit 5: Advisory status  Bit 6: N/A  Bit 7: N/A  NOTE: For more information, refer to [Group Cyclic Status (GCS)](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_Diag_UG&topicID=GroupCyclicStatusDefinition_F906694D). |
| Embedded\_BusExtender\_GCS | 0...255 | BYTE  1 | Embedded Bus Extender Group Cyclic Status  Bit 0: Data quality  Bit 1: General module status  Bit 2: N/A  Bit 3: N/A  Bit 4: N/A  Bit 5: Advisory status  Bit 6: N/A  Bit 7: Data freshness  NOTE: For more information, refer to [Group Cyclic Status (GCS)](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_Diag_UG&topicID=GroupCyclicStatusDefinition_F906694D). |
| DiagnosticNumber | 0...4,294,967,295 | BYTE  4 | Displays the highest-priority Sercos diagnostic numbers specified by the manufacturer specific status codes and the Sercos standard. NOTE: This parameter is optional. |
| gPTP Time | – | BYTE  4 | Generalized Precision Time Protocol timestamp in nanoseconds according to IEEE 802.1Q-2018 PTP time format standard. |
