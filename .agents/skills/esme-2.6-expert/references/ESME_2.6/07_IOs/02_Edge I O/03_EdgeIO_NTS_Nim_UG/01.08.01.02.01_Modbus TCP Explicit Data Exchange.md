# Modbus TCP Explicit Data Exchange

The Modbus TCP supports the following Modbus requests:

| Function Code | Object ID | Function |
| --- | --- | --- |
| FC03 | - | Read multiple registers |
| FC16 | - | Write multiple registers |
| FC23 | - | Read and Write multiple registers |
| FC43 | 0 | VendorName |
| 1 | ProductName |
| 2 | MajorMinorVersion |
| 3 | VendorUrl |
| NOTE: Minimum response time of explicit data exchange is 200 ms. | | |

Use the Unit ID 255 dec (FF hex) or 00 dec (00 hex) to send commands to the network interface module.
