# Sercos III Implicit Data Exchange

## Overview

The memory-mapped I/O image is exchanged synchronously and cyclically between the Sercos devices through connections mapped on MDT and AT frames in the Real-Time Channel (RTC).

The Sercos network interface module manages two connections:

* One consumer connection in MDT frame containing a list of IDNs with output data. For example: S-0-1500.0.05 Container OutputData.
* One producer connection in AT frame containing a list of IDNs with input data. For example: S-0-1500.0.09 Container InputData.

The maximum size of the payload of a Sercos connection is limited to 1492 octets because of the underlying Ethernet constraints and Sercos overheads.
