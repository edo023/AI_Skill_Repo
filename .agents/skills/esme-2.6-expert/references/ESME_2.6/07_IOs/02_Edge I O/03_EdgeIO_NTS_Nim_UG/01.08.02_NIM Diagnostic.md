# NIM Diagnostic

## EtherNet/IP or Modbus TCP NIM Diagnostic

The EtherNet/IP or Modbus TCP network interface module provides diagnostic capabilities based on the standard diagnostic features defined by EtherNet/IP and Modbus TCP protocols.

## Sercos III NIM Diagnostic

The Sercos III network interface module provides diagnostic capabilities, including the diagnostics available in network interface module EtherNet/IP or Modbus TCP module, along with additional Sercos diagnostics.

## Diagnostic Tools

| Diagnostic Object | Description |
| --- | --- |
| Customer Logs | Customer logs include Sercos main function group diagnostics:  * Errors (C1D) * Advisories (C2D) * Information generated by the Sercos application and the Sercos devices. |
| Online Diagnostics | Displays the most recent and highest-priority diagnostics:  * Diagnostic number: IDN S-0-0390 * Diagnostic Message: IDN S-0-0095 |
