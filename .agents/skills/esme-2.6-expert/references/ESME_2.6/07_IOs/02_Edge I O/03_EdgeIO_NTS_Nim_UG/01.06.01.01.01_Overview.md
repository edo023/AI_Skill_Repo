# Overview

The NTSNEC1200/NTSNEC1200H network interface modules are devices designed to manage EtherNet/IP or Modbus TCP communication protocol when using I/O modules for the island.

The NTSNEC1200H module is the hardened version (H suffix) of the NTSNEC1200 module (standard version). With conformal coating of the electronic boards, the hardened version of the module can be used in harsh chemical environments and at extended operating temperatures. For more information on environmental characteristics, refer to [Modicon Edge I/O – System Planning and Installation Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_Spig&topicID=EnvironmentalCharacteristics_2C0E6A68).

NOTE: The network interface module is the leftmost module in the main cluster.
