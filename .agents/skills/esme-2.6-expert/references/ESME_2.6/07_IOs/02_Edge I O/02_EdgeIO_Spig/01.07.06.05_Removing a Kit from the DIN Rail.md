# Removing a Kit from the DIN Rail

Kits must be removed one by one from right to left.

The following procedure describes how to remove a kit from the DIN rail:

| Step | Action |
| --- | --- |
| 1 | Pull the base locker upward to set it in the releasing position. |
| 2 | Slide the kit to the right on the DIN rail. |
| 3 | Pull the kit from the DIN rail. |
| 4 | Repeat these steps for each kit you want to remove. |
