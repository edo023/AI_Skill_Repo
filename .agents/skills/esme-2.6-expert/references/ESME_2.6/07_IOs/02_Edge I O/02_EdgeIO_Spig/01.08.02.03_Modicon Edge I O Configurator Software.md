# Modicon Edge I/O Configurator Software

You can use the Modicon Edge I/O Configurator software for module configuration and control as well as application diagnostics and monitoring.

For more information about the Modicon Edge I/O Configurator, refer to the [Modicon Edge I/O - Configurator and Web Interface - User Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_Conf_UG&topicID=GettingStarted_05E587D3).
