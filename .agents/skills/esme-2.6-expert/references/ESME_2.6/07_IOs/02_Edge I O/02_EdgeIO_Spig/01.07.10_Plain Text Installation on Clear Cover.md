# Plain Text Installation on Clear Cover

The clear cover of the [Terminal Blocks](TPC_TerminalBlocks-F1CDE52F.html) has a dedicated space that allows you to insert a paper label:

|  |  |
| --- | --- |
|  |  |

The label size depends on the height of the module:

| Module height | Label dimensions |
| --- | --- |
| 100 mm (3.93 in.) | 65 x 10.7 mm (2.56 x 0.42 in.) |
| 121 mm (4.76 in.) | 86 x 10.7 mm (3.38 x 0.42 in.) |
