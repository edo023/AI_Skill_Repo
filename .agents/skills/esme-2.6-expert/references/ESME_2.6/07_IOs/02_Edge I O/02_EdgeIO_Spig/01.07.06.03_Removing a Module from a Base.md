# Removing a Module from a Base

The following procedure describes how to remove the module from the base:

NOTE: If you are replacing an existing module, follow the procedures for [Hot Swapping Electronic Modules](HotSwappingElectronicModules-27DB4C41.html).

| Step | Action |
| --- | --- |
| 1 | Remove the terminal block and electrical connector before removing the module. |
| 2 | Push the release button of the configuration to be removed. |
| 3 | Pull the module and remove the module from the base. |
