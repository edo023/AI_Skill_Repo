# Mounting a Module on a Base

The following procedure describes how to mount the modules on their corresponding base:

NOTE: If you are replacing an existing module, follow the procedures for [Hot Swapping Electronic Modules](HotSwappingElectronicModules-27DB4C41.html).

| Step | Action |
| --- | --- |
| 1 | Insert the module in the guides of the base. |
| 2 | Push the module straight into the base until it clicks. |
| 3 | According to the installation plan, repeat these steps with all the modules. |

NOTE: To help ascertain the compatibility between a base and a module, a color coding is applied on the front side of the base and on the rear side of the module. The color and the shape of the coding are the same for compatible bases and modules.
