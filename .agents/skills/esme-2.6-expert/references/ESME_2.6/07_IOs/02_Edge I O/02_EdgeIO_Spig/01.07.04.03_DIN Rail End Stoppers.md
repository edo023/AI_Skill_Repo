# DIN Rail End Stoppers

For installation where vibration can occur, the elements of the Modicon Edge I/O cluster must be secured against slipping. A set of two end stoppers (NSYTRAABV35) can be used to help secure the configuration.

| Step | Action |
| --- | --- |
| 1 | Place the end stoppers on the DIN rail on each side of the cluster. |
| 2 | Screw the end stoppers down tightly using the screwdriver (maximum torque 2.5 Nm). |
