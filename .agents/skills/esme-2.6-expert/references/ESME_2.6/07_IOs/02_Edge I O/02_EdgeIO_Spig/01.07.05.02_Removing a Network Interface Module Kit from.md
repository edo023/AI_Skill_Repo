# Removing a Network Interface Module Kit from the DIN Rail

The following procedure describes how to remove a Network Interface Module kit from the DIN rail:

| Step | Action |
| --- | --- |
| 1 | Remove any kits that are mounted to the right of the network interface module, following the procedure [Removing a Kit from the DIN Rail](RemovalOfTheSliceFromTheDINRail-BC42449F.html). |
| 2 | Pull the base locker upward to set it in the releasing position. |
| 3 | Pull the kit from the DIN rail. |
