# Creating a Coding Scheme

There are several coding schemes available in a Modicon Edge I/O NTS cluster, such as:

* Assigning different codes to the top and bottom module connections.
* Using different codes for adjacent modules.

The number of available keys varies with your terminal block and base plate:

|  |  |  |
| --- | --- | --- |
| **Module** | | **Number of keys** |
| I/O Terminal blocks | 12 points | 8 |
| 18 points | 9 |
| Power supply terminal blocks | | 4 |
| Base plate | | 6 |
