# Selecting Modicon Edge I/O NTS Modules

When you plan a Modicon Edge I/O NTS system layout, you must know the number and type of modules and terminal blocks in your system.

NOTE: There are restrictions and regulations associated with certain module types. For more information, refer to the user guides of the module.

You can, for any reason, reserve module slots in your Modicon Edge I/O NTS layout. Any reserved module slot must be filled by a dummy module (NTSDMY0100H).

| NOTICE | |
| --- | --- |
|  | COMMUNICATION ERROR  Use a dummy module for all reserved module slots.  Failure to follow these instructions can result in equipment damage. |

The Device Mode feature allows you to create a standard island configuration and make depopulated variations of that configuration that share the same fieldbus process image. This allows to maintain a single application program for various island configurations.

The depopulated islands are configured by declaring the intended modules and you can declare any of the Modicon Edge I/O NTS input and output modules to be Normal (default value), Optional or Virtual reserved with the Device Mode parameter:

* Normal: The module is part of the software configuration and is physically connected in the cluster.
* Optional: The module is part of the software configuration. A dummy module or the configured module must be physically installed in the cluster. Whether either module is present does not cause a configuration error to be detected.
* Virtual reserved: The module is part of the software configuration. A dummy module must be physically installed in the cluster. If the virtual module is physically installed in the cluster, a configuration error is detected.

When a module is configured as Optional or Virtual reserved, the memory mapping of the island is retained and modules marked as not present retain their slot addresses. This allows you to physically modify the design of your island, without having to edit your application program.

After the number and type of modules are established, you can determine [power distribution requirements](ModiconEdgeIONTSPowerDistribution-24CF3A22.html), your [external power source requirements](ElectricalRequirements-24CFEA80.html) and the overall hardware design.
