# Planning the Installation

Before you begin installation of your Modicon Edge I/O system, you must establish a plan that identifies:

* The number and type of modules on your Modicon Edge I/O system.
* The power requirements of your Modicon Edge I/O system configuration.
* The order in which any Modicon Edge I/O NTS modules are assembled together.
* The type of enclosure for the Modicon Edge I/O NTS system.
* A coding scheme that helps match the terminal blocks with their electronic modules.
* A labeling system with plain texts on the clear cover and marking labels.
