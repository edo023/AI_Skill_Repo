# Hot Swap Procedure

Follow this procedure to perform a hot swap operation:

| Step | Action |
| --- | --- |
| 1 | Read and understand the [Hot Swapping Considerations](HotSwappingConsiderations-9BD88A04.html). |
| 2 | Remove the module with the terminal block from the base, refer to [Removing a Module from a Base](RemovalOfAModuleFromTheBase-BC424CAC.html) |
| 3 | Remove the terminal block from the module. Refer to [Removing a Terminal Block from a Module](RemovalOfTheTerminalBlockFromAModul-BC423C74.html) |
| 4 | Mount the terminal block on a module with an identical reference, refer to [Mounting a Terminal Block on a Module](MountingTheTerminalBlocks-36BA7328.html) |
| 5 | Mount the module with the terminal block on the base, refer to [Mounting a Module on a Base](InsertingTheElectronicModules-36BA58A0.html) |
