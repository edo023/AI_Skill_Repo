# Mounting a Kit on the DIN Rail

Kits must be installed one by one and mounted on the DIN rail.

The [DIN rail](DINRailInstallation-24D67EED.html) and the [Network Interface Module kit](NetworkInterfaceModuleInstallation-24D6D8AC.html) must already be installed.

The following procedure describes how to mount a kit on the DIN rail:

| Step | Action |
| --- | --- |
| 1 | Pull the base locker upward to set it in the releasing position. |
| 2 | Place the kit on the DIN rail and press it down firmly. |
| 3 | Push the base locker downward to set it in the locking position. |
| 4 | Slide the module to the left until it clicks into place on the previously installed module. |
| 5 | Repeat these steps to mount additional kits. |
