# Installing a Spare Clear Cover

When purchased together, the terminal block and the clear cover come pre-assembled.

If you purchase a clear cover NTSXEM0000H or NTSXEM0000XH separately, follow these steps to install it on a terminal block:

|  |  |
| --- | --- |
| Step | Action |
| 1 | Position the clear cover in front of the terminal block and align the hinge. |
| 2 | Push the clear cover into the hinge until it clicks in place. |
