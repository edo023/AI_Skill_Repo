# System Requirements

The Modicon Edge I/O Configurator can be installed on any PC that meets the following minimum hardware and software requirements:

| Component | Minimum Requirement |
| --- | --- |
| Operating system | Microsoft Windows 10 (32-bit or 64-bit processor). |

The Modicon Edge I/O NTS Web Interface can be accessed by the following web browsers:

| Component | Minimum Requirement |
| --- | --- |
| Web browser | * Google Chrome (version 105 or greater). * Mozilla Firefox (version 128 or greater). * Microsoft Edge (version 116 or greater). |
