# Operating Modes

The control of the network interface module security settings with CAE is enabled by default.

After the security settings are pushed by the CAE, the parameters for the following pages become read-only:

* [Network Interface Subpage](FirmwareSubmenu-09170361.html).
* [User Management Subpage](UserManagementSubmenu-09173A2D.html).
* [Security Banner Subpage](SecurityBannerSubmenu-541B68E1.html).
* [Session Management Subpage](SessionManagementSubpage-5AF20376.html).

NOTE: To enable editing for these parameters, use the rotary selector switch for Cybersecurity to reset the network interface module to factory settings.

After the connection between the CAE and the network interface module is established, the CAE can send the RBAC or the DSS configurations. After receiving a valid RBAC configuration, the existing users and groups are deleted, and new groups and users are created based on the RBAC configuration.

Any DSS configuration received from the CAE is applied immediately.
