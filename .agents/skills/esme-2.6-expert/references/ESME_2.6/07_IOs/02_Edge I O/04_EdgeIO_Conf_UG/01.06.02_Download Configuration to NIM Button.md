# Download Configuration to NIM Button

Use this button to build, verify and download the configuration to the network interface module.

Before the configuration is downloaded, a verification is performed to ascertain consistency. If there is a discrepancy between the physical configuration and the virtual configuration of the island, a list of modules that are inconsistent with the configuration are displayed in the Configuration mismatch detected screen.

NOTE: If the issue is due to incompatible firmware, you can use the Schneider Electric EADM tool to change the firmware of the module.
