# Error Codes List

The following table describes the possible errors that can be detected when using the Build the configuration button:

| Code | Description |
| --- | --- |
| EIO0001 | The power provided by the power supply on the bus is less than the estimated consumption on the bus. |
| EIO0003 | The cycle time defined in the network interface module parameters is insufficient. |
| EIO0004 | There is no module after the network interface module, or the first module is not a Power Supply Field and Bus module. |
| EIO0005 | There is more than one Power Supply Field and Bus module present in the cluster. |
| EIO0006 | A mandatory parameter is not set. |
| EIO0008 | You cannot have a Modbus subordinate when the value of the Protocol mode parameter of the field device master is ASCII. |
| EIO0009 | The sum of the input data of all slaves on a field device master serial line exceeds 1024 bytes. |
| EIO0010 | The sum of the output data of all slaves on a field device master serial line exceeds 1024 bytes. |
| EIO0012 | The IP address is one of the reserved ranges of IP addresses:   * For network interface module IP address:    + 169.254.224.0/19 * For USB IP address:    + 169.254.224.0/19   + 10.10.0.0/16 |
| EIO0014 | Input or output size of one module exceeds the size limit of a frame (input or output exchanged implicitly > 1492 bytes). |
| EIO0015 | Modbus output range exceeded the fieldbus configuration. |
| EIO0016 | Modbus input range exceeded the fieldbus configuration. |
| EIO0017 | The IO Bus Cycle Time is too short for modules to exchange their data in one cycle. |
| EIO0020 | The entered NIM IP subnet mask value is inaccurate.  Only values 0, 128, 192, 224, 240, 248, 252, 254, and 255 are allowed for each byte. |
| EIO0021 | The entered NIM IP subnet mask value does not respect the descendant compatibility. All bytes to the right of a byte that is not equal to 255 is equal to 0. |
| EIO0022 | IP Address or Gateway address is Broadcast Address (ending with 0), loopback address (127.0.0.1) and network address (ending with 255). |
| EIO0023 | The IP address and the Gateway address are not in the same subnet. |
| EIO0024 | There are more than five field device master IO-Link modules in a cluster or more than 125 field device master IO-Link modules on the island. |
| EIO0025 | Subnetmask = 0.0.0.0 and IP or Gateway address /= 0.0.0.0. |
| EIO0026 | The Implicit communication protocol parameter value is EtherNet/IP and the input or output size of the module data exceeds the size limit of an assembly (509 bytes for input, 505 bytes for output). |
| EIO0028 | The number of field device master Serial master in a cluster is above six. |
| EIO0029 | IP Address is equal to the Gateway but it is not 0.0.0.0. |
| EIO0030 | The selected sampling step is not compatible with the selected Cycle Time.  On oversampling input and output:   * Sampling step = 8-bits and Cycle > 4000 µs * Sampling step = 16-bits and (Cycle < 250 µs or Cycle > 4000µs) * Sampling step = 32-bits and (Cycle < 500 µs or Cycle > 4000 µs) |
| EIO0031 | The Implicit communication protocol parameter value is EtherNet/IP and there are more than 16 input assemblies or more than 16 output assemblies. |
| EIO0032 | There are no reflex output locations configured in a reflex output module. |
| EIO0033 | The cluster has no module. |
| EIO0034 | Input and/or Output data exceeds one IO Image frame size. |
| EIO0036 | The IO Bus Cycle Time is too short for the network interface module to exchange the data in one cycle. |
| EIO0037 | The network interface module IP address and the USB IP address are in the same subnet. |
| EIO0038 | RSTP Bridge Identifier Priority is not a multiple of 4096 (between 0 and 61440). |
| EIO0040 | The fieldbus device data exceeds the limit of input or output bytes. |
| EIO0042 | The Cycle Time (in ms) configured for the listed channels must be greater than the Time between frames setting. |
