# CAE Parameters Supported by the Modicon Edge I/O NTS

The following table describes the parameters from CAE that are supported by the Modicon Edge I/O NTS:

| Section | Parameter Name |
| --- | --- |
| User Accounts | Minimum inactivity period (min) |
| Maximum login attempts |
| Password attempts timer (min) |
| Roles |
| Users |
| Security Banners | Notification displayed at login  CAE allows you to configure three different banners depending on the capabilities of the devices configured. Modicon Edge I/O NTS displays the longest banner configured among the three. |
| Certificate management | Certificate mode |

For more information about these parameters, refer to the [Cybersecurity Admin Expert User Manual](https://www.se.com/ww/en/download/document/CAE_User_Guide/).
