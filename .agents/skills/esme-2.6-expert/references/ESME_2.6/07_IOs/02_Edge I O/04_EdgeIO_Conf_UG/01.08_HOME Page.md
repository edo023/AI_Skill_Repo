# HOME Page

This page is available when you are connected to a network interface module.

This page displays the following information about the network interface module:

* Vendor name
* Product name
* Product reference
* Serial number
* Firmware version
* IP address
* IP addressing mode

From this page, click the ![](images/DD00768206.png) Configure button to access the Parameters section of the ISLAND MANAGEMENT page.

The Locate Network Interface Module button allows you to make the LEDs of the connected network interface module flash.
