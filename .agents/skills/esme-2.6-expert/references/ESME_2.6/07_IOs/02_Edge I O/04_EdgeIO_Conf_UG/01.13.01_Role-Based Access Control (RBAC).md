# Role-Based Access Control (RBAC)

This feature allows you to manage access to the resources of a system through roles and permissions for a user.

CAE supports up to 20 users, each assigned to a specific role, with each role capable of having multiple permissions.

A user with Security Management permission can create a role, delete a role and manage the permissions of a role with CAE. Permissions cannot be modified.
