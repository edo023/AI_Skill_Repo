# CAE Commands Supported by the Modicon Edge I/O NTS

This table describes the CAE commands you can use with the Modicon Edge I/O NTS:

| Commands | Description |
| --- | --- |
| Scan or Discover device(s) over the network | Displays the network interface module in the list of discovered devices. |
| Send Security Configuration | Sends the RBAC and DSS configuration files and applies the configuration, after it is validated by the CAE tool, using the CAE signing certificate. |
| Log On | Manually connects the controller when automatic connection was unsuccessful. |
| Send DSS | Sends the DSS and applies the configuration, after it is validated by the network interface module, using the CAE signing certificate. |
| Reset | Resets the security configuration to default. RBAC users, roles and permissions, CAE signing certificate, and DSS are reset. This command restarts the network interface module. |
| Locate | Locates the network interface module by flashing its LED. |
| Trust | Adds the network interface module to the list of trusted devices. |
| Certificates Management > Signers > Add CAE Signing Certificate | Sends the CAE signing certificate to the certificate store of the network interface module. The security configuration is rejected if the certificate is not provided. Only one certificate is supported. |
| Certificates Management > Internal > Get CSR | Allows the network interface module to generate and send the certificate sign request (CSR) for the network interface module certificate. The network interface module certificate is used by both OPC UA and HTTPS. This command is accessible if the Certificate Policy is in PKI mode. |
| Certificates Management > Internal > Send signed certificate | Allows you to replace the self-signed certificate by the certificate signed by the Certificate Authority (CA) and provided to CAE. The certificate is applied immediately. This command is accessible if the Certificate Policy is in PKI mode. |

If a command is not active (appears dimmed in the software), refer to the [Cybersecurity Admin Expert User Manual](https://www.se.com/ww/en/download/document/CAE_User_Guide/).
