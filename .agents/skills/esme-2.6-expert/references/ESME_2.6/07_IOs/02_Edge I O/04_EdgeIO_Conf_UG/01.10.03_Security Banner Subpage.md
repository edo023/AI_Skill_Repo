# Security Banner Subpage

The Security Banner subpage allows you to define the message displayed when a user logs in.

You can register a message with a maximum of 1024 characters in the field provided for this purpose.

Clear the fields if you do not want a message displayed.

A default message is provided. It is displayed by clicking the button Set to default.

NOTE:

* Only users with the SECADM role can modify these parameters. Other users have a read-only access.
* If you are managing the security settings through the CAE software, this subpage is in read-only mode. Users with the Security Management permission can modify these parameters in the CAE software.
