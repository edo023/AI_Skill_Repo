# ISLAND MANAGEMENT Main Page

The main area of the ISLAND MANAGEMENT page contains the following sections:

| Name | Description |
| --- | --- |
| ABOUT | This section displays the following characteristics of the selected module:   * Product reference * Vendor name * Range name * Software version * Width in mm * Height in mm * Description   This section provides a link to the user guide of the selected module. |
| PARAMETERS | This section provides access and allows you to set the parameters of the selected module.  In  online mode, some parameters can be modified. Refer to the user guide of the selected module for more information.  When a parameter is changed in  online mode, a message is displayed and gives you three options:   1. Exit the configuration software: The value is retained until the next reboot of the system. A message is displayed on your next connection. 2. Click Discard: The value reverts to the one saved in the nonvolatile memory of the network interface module. 3. Click Save Configuration: The value is stored in the nonvolatile memory of the network interface module. |
| MEMORY MAPPING(1) | This section provides access to the memory mapping parameters of the network interface module. Refer to the user guide for the selected module for more details.  Click the Export as CSV button to download the memory mapping as a `.csv` file. |
| ONLINE PARAMETERS(2) | This section displays:   * Data exchanged implicitly by the module * Diagnostic nodes |
| LOGS(2) | This section displays and allows you to download the log files. For more details, refer to the [LOGS Section](LOGS-B2855E14.html). |
| (1) This section is displayed if the network interface module is selected.  (2) This section is displayed in  online mode. | |
