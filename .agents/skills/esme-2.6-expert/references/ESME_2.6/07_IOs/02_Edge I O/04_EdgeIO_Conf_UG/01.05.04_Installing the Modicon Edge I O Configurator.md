# Installing the Modicon Edge I/O Configurator (NTSCSW1000)

The Modicon Edge I/O Configurator is a standalone software that can be used to:

* Access and configure the network interface module.
* Create a Modicon Edge I/O island configuration, whether you are connected or not to a network interface module.

This application allows you to:

* Create and modify configuration files for your Modicon Edge I/O island, refer to [FILE Page](FileMenu-05FB9A43.html).
* Manage your island, refer to [ISLAND MANAGEMENT Page](ISLANDMANAGEMENTMenu-078D6AFD.html).
* Manage the user rights, refer to [ISLAND SETUP Page](TPC_IslandSetupPage-F5F4642C.html).
* Perform diagnostics operations, refer to [DIAGNOSTICS Page](DIAGNOSTICSMenu-078D03B2.html).
* Reboot your island, refer to [MAINTENANCE Page](MAINTENANCEMenu-080D8686.html).

Follow these step to download and install the Modicon Edge I/O Configurator:

| Step | Action |
| --- | --- |
| 1 | Determine if your computer meets the system requirements, refer to [System Requirements](PCConfiguration-05E6BC08.html). |
| 2 | Download the executable installation file for the software from the Schneider Electric website [(NTSCSW1000)](https://www.se.com/ww/en/product/NTSCSW1000/). |
| 3 | Double-click the executable file and follow the step-by-step installation process.  **Result**: The Modicon Edge I/O Configurator shortcut is installed on your desktop. |

Follow this procedure to access the Modicon Edge I/O NTS Web Interface:

| Step | Action |
| --- | --- |
| 1 | Connect your computer to the network interface module with a USB Type-C cable or your Ethernet cable. |
| 2 | Launch the Modicon Edge I/O Configurator application. |
| 3 | Type the IP address of your network interface module in a web browser using HTTPS or select your module in the DPWS discovery device list.  For more information about the default IP address of your module, [refer to the Modicon Edge I/O NTS - Network Interface Modules - User Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Nim_UG&topicID=NetworkInterfaceModules_references_0EBC2A20). |
