# Login Page

Upon launching the Modicon Edge I/O Configurator, you are prompted to run the configurator software in either ![](images/DD00768205.png) Connected or ![](images/DD00768204.png) Disconnected mode. Click Start Offline to start the Modicon Edge I/O Configurator in ![](images/DD00768204.png) Disconnected mode. The FILE page opens directly.

When starting either the Modicon Edge I/O Configurator in ![](images/DD00768205.png) Connected mode or the Modicon Edge I/O NTS Web Interface, the first displayed page is the login page.

The elements of the login page are the following:

|  | Modicon Edge I/O NTS Web Interface | Modicon Edge I/O Configurator |
| --- | --- | --- |
| Element | Description | |
| Network Interface Module | Not applicable  When accessing the Modicon Edge I/O NTS Web Interface, you are connected to the network interface module with the IP address you used in your web browser. | * Manually enter your network interface module IP address. * Discover automatically (DPWS) your network interface module. After selecting this feature, all the network interface modules on your network are displayed in the drop-down list. Select the network interface module you want to connect to. |
| Username | Enter your username. | |
| Password | Enter your password.  Click  or  to show or hide the password. | |
| Go back | Not applicable | This button allows you to go back to the online/offline selection page. |
| Forgot your password? | If you forgot your password:   * Contact a user with the Security Management permission (by default, a user with the SECADM role) to reset your password. * If the roles and users are managed with the EcoStruxure Cybersecurity Admin Expert (CAE) software, contact your CAE administrator to reset your password. For more details about the use of CAE, refer to [Security Settings Configuration with Cybersecurity Admin Expert Software](SecuritySettingsConfigurationWithCy-5FFDA383.html) * If you no longer have access to a user with the Security Management permission, you can reset the network interface module to factory settings using the Cybersecurity rotary switch. For more information, refer to the [Modicon Edge I/O NTS - Network Interface Modules - User Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Nim_UG&topicID=RotarySelectorSwitch_C0EC90CF). | |

After your username and password are confirmed, the FILE page is displayed.

NOTE: Only one user can be logged into the network interface module. If a user is already logged in, the network interface module refuses login attempts for other users.
