# User Management Subpage

The User Management subpage allows you to create or remove users, reset passwords and change the role of a user.

You can add up to 20 users; each user can have one role.

Any user can change their own password on this page by clicking on Change your password.

NOTE:

* Only users with the SECADM role can modify these parameters.
* If you are managing the security settings through the CAE software, this subpage is in read-only mode. Users with the Security Management permission can modify these parameters in the CAE software.
