# Advisory Codes List

The following table describes the possible advisories that can be detected when using the build button:

| Code | Description |
| --- | --- |
| EIO0002 | The power supply of the 24 Vdc field power is less than the estimated consumption of the 24 Vdc field power. |
| EIO0018 | With the IO Bus Cycle Time setting, the listed modules (isochronous and synchronous) work asynchronously. |
| EIO0019 | With the IO Bus Cycle Time setting, the listed modules (synchronous) work isochronously. |
| EIO0035 | The Cycle Time (in ms) configured for the listed channels is too brief in regard to the serial line configuration. |
| EIO0039 | The Communication Protocol indicated is disabled. |
| EIO0041 | The Transaction timeout duration is insufficient. Choose a value greater than 10 ms. |
