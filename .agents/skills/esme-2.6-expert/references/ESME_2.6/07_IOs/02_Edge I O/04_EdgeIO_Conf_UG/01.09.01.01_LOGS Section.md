# LOGS Section

This section displays the log entries and allows you to download the log files:

* Log entries contain information on normal and exceptional system operation. The log entries can be helpful when troubleshooting why an error was detected.
* Log files is an archive of the log entries.

The following buttons are available in the LOGS section:

| Button | Description |
| --- | --- |
| View logs here | This button displays the log entries in the LOGS section.  Select scope for the logs(1) from the drop-down list before you click View logs here:   * Module: Downloads the log files of the selected module. * Cluster: Downloads the log files of each module of the selected cluster. * Island: Downloads the log files of each cluster on the island. |
| Download as file | This button opens the Download log as a file window, which allows you to download the log files.  In the Download log as a file window:   * From Select the scope for the logs drop-down list, select one of the following options:   + Module: Downloads the log files of the selected module.   + Cluster(1): Downloads the log files of each module of the selected cluster.   + Island(1): Downloads the log files of each cluster on the island. * In Select the types of logs you want to extract, choose from the following options:   + Standard: Contains the log entries displayed when clicking the View logs here button.   + Crash log: For debugging purposes, this option is intended for your Schneider Electric representative. * Select the file name. |
| (1) This is displayed only for the network interface module. | |
