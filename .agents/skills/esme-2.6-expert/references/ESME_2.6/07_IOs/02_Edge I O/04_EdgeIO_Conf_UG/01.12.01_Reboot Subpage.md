# Reboot Subpage

The Reboot subpage allows you to restart the network interface module.

A restart of the network interface module has the following effects:

* Restarts the main cluster.
* Terminates connections.
* Interrupts communication.
* Resets diagnostic indicators.
* Resets the inputs and outputs to their initial values or their default values.

The reboot request is recorded in the logs.

NOTE:

* Only users with the ENGINEER role can do a reboot. Other users have read-only access.
* If you are managing the security settings through the CAE software, users with the Reboot via WebServer permission can do a reboot. Other users have read-only access.
