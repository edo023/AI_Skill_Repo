# First Login

Access to the network interface module is controlled by a username and a password for each user.

A user can have different roles, each with specific permissions. For more information about roles and permissions, refer to [User Management Subpage](UserManagementSubmenu-09173A2D.html).

At first connection, use the default SecurityAdmin user username and password:

* Username: SecurityAdmin
* Password: AAAAAAAA

Upon login, you are prompted to change the default password for the SecurityAdmin user.

NOTE: The default SecurityAdmin user has both the SECADM and ENGINEER roles.

A tooltip is displayed and indicates the password requirements.

Accessing the network interface module with the username SecurityAdmin allows you to:

* Create and modify configuration files of your Modicon Edge I/O island.
* Manage your islands.
* Manage your user rights.
* Perform diagnostics operations.
* Reboot your island.

If there are security concerns over these functions, you must, at a minimum, assign a secure password to the SecurityAdmin user to prevent unauthorized access to the application. Care must be taken to ensure that the immediate physical environment of the machine and process is in a state that will not present safety risks to people or property before exercising control remotely.
