# Node List File Description

You can export the node list from the network interface module with the Export Node List command. The following table describes each column of the \*.csv file:

| Parameter | Description |
| --- | --- |
| clusterId | Cluster number |
| slotId | Slot number |
| nsName | Namespace |
| ns | Namespace index |
| i1 | Identifier |
| moduleName | Name of the module in your configuration |
| functionName | Expert function name |
| path | Path to the function |
| variableName | Variable name |
| type | Type of data value |
| byteSize | Size in bytes |
| 1 In the Node list file description, the identifier is a unique decimal value. For more information refer to [Identifier Description](#NodeListFileDescription-0965EEFC__IdentifierDescription-8AB52D10) | |

## Identifier Description

The identifier is a 32-bit value structured as follows:

|  |  |  |  |
| --- | --- | --- | --- |
| i: 32-bit identifier | | | |
| Word 1: 16 bits | | Word 2: 16 bits | |
| MSB | LSB |
| clusterId | slotId | Node ID: a unique ID for each parameter.  For example, Channel Enabled has a unique Node ID that corresponds to 8498. | |

The following example illustrates the identifier of the parameter Channel Enabled of a Modicon Edge I/O NTS module positioned in the cluster 0 and the position 1:

|  |  |  |  |
| --- | --- | --- | --- |
| i: 74034 in decimal which corresponds to 0000 0000 0000 0001 0010 0001 0011 0010 in binary. | | | |
| Word 1: 0000 0000 0000 0001 | | Word 2: 0010 0001 0011 0010 | |
| MSB: 0000 0000 | LSB: 0000 0001 |
| clusterId: 0 | slotId: 1 | Node ID: 8498 in decimal. | |
