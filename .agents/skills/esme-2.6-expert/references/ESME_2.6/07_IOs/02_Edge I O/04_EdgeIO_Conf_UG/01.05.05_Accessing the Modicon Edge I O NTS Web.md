# Accessing the Modicon Edge I/O NTS Web Interface

As standard equipment, the network interface module provides an embedded Modicon Edge I/O NTS Web Interface with a predefined, built-in website.

Follow this procedure to access the Modicon Edge I/O NTS Web Interface:

| Step | Action |
| --- | --- |
| 1 | Connect your computer to the network interface module with a USB Type-C cable or your Ethernet cable. |
| 2 | Type the IP address of your network interface module in a web browser using HTTPS.  For more information about the default IP address of your module, [refer to the Modicon Edge I/O NTS - Network Interface Modules - User Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Nim_UG&topicID=NetworkInterfaceModules_references_0EBC2A20). |

NOTE: HTTP (non secured connections) requests are not permitted.

This interface allows you to:

* Create and modify configuration files on your Modicon Edge I/O island, refer to [FILE Page](FileMenu-05FB9A43.html).
* Manage your island, refer to [ISLAND MANAGEMENT Page](ISLANDMANAGEMENTMenu-078D6AFD.html).
* Manage the user rights, refer to [ISLAND SETUP Page](TPC_IslandSetupPage-F5F4642C.html).
* Perform diagnostics operations, refer to [DIAGNOSTICS Page](DIAGNOSTICSMenu-078D03B2.html).
* Reboot your island, refer to [MAINTENANCE Page](MAINTENANCEMenu-080D8686.html).
