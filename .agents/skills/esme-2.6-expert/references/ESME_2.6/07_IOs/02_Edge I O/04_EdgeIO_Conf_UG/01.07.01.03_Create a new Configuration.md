# Create a new Configuration

Use this button to manually create a configuration for your island.

To create a new configuration, follow this procedure:

 

| Step | Action | |
| --- | --- | --- |
| 1 | Click the  Create a new Configuration button.  NOTE: If you have an open configuration, you can either discard the existing configuration or [save it](SaveTheConfigurationToDiskButton-F6FE4B74.html). | |
| 2 | Select a network interface module from the list.  NOTE: The network interface module is automatically selected in connected mode(1). | |
| 3 | Select a power supply module from the list.  NOTE: Only the compatible power supply modules are displayed. When there is only one power supply available, it is automatically selected. | |
| 4 | Enter a project name and click Create the Configuration to validate your configuration.  **Result:** You are directed to the ISLAND MANAGEMENT page to configure your island. | |
| 5 | Click Save configuration to disk to save your file. | |
| (1) If you want to create a configuration while disconnected, use the Modicon Edge I/O Configurator in Disconnected mode, refer to [Login Page](LoginPage-086E0D5E.html). | | |
