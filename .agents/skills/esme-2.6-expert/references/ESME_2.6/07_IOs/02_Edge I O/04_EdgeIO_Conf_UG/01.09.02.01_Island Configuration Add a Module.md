# Island Configuration > Add a Module

Use this subpage to add modules to your configuration:

| Step | Action | |
| --- | --- | --- |
| 1 | Click + Add a Module. | |
| 2 | Click Add a Module to access the list of available modules. | |
| 3 | There are two ways to add a module to your island:  a. Select one or more modules from the module list by:   * Expanding  the different categories directly from the list. * Typing the commercial reference in the  field. * Using the  Advanced Filter.   b. Use the  icon while navigating the list of modules to add one or more instances of the same module. | |
| 4 | Click the Add button to add the selected modules to your island. | |
