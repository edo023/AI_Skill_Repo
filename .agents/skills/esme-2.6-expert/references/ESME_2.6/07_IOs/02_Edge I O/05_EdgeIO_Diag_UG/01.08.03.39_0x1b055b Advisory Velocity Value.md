# 0x1B055B: Advisory Velocity Value

Criticality level: Advisory

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The commanded velocity parameter is out of range.

The commanded velocity exceeds Maximum Velocity parameter value.

## Possible Solutions

Send a new command with a velocity value within the configured range.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
