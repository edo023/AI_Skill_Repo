# 0x030140: Not Connected to NIM/CTR

Criticality level: Advisory

MainFunction: ExplicitCommToCtr

SubFunction: Connection

## Principle and Potential Causes

The diagnostic shows this ID until the I/O module has successfully opened a client connection with the network interface module for explicit exchanges.

The module is not connected to the network interface module Unit.

## Possible Solutions

Wait for connection.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
