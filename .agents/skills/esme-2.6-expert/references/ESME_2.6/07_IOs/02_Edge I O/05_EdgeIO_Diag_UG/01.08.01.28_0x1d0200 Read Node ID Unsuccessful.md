# 0x1D0200: Read Node ID Unsuccessful

Criticality level: Information

MainFunction: IOM\_DIO

SubFunction: ExplicitAccess

## Principle and Potential Causes

The requested node ID cannot be accessed.

The node ID is incorrect for the module.

## Possible Solution

Use a valid node ID.

## How to Clear the Information

Not applicable.
