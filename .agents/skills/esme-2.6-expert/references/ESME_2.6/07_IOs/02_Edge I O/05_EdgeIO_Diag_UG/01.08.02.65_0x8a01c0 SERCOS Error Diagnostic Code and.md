# 0x8A01C0: SERCOS Error Diagnostic Code and Diagnostic Message

Criticality level: Error module

MainFunction: SERCOS

SubFunction: DiagMessage

## Principle and Potential Causes

A SERCOS diagnostic number of class C1D (IDN S-0-0390) is reported.

A SERCOS diagnostic message (IDN S-0-0095) is reported.

## Possible Solutions

Identify the root cause of the error with the SERCOS diagnostic number provided and take the appropriate corrective actions.

For more information about the SERCOS diagnostic numbers, refer to [Sercos III Implicit Data Exchange](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Nim_UG&topicID=SercosIIIDataExchange_024F9144).

## How to Clear the Error Code

Acknowledge the error from the SERCOS master interface.
