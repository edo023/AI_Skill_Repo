# 0x1B0560: Advisory Duty Value

Criticality level: Advisory

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The PWM duty cycle value is out of range.

Potential causes:

* The value is in an incorrect data type.
* The value is not within the allowed data range.

## Possible Solutions

* Send a value in a correct data type.
* Send a value within the allowed data range.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
