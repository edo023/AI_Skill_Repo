# 0x8701C0: Duplicate IP Found

Criticality level: Error module

MainFunction: IPAddressing

SubFunction: MainProcess

## Principle and Potential Causes

A duplicated IP address is found by the network interface module on the fieldbus port.

Another device in the network has the same IP address as the network interface module.

## Possible Solutions

* Modify the IP address of the network interface module.
* Modify the IP address of the other device.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
