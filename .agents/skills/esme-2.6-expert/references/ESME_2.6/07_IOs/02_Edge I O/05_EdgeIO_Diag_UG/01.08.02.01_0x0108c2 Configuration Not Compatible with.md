# 0x0108C2: Configuration Not Compatible with Firmware

Criticality level: Error module

MainFunction: ConfigMgr

SubFunction: Process

## Principle and Potential Causes

The software configuration is not compatible with the firmware of the module.

## Possible Solutions

* Update (upgrade or downgrade) the firmware of the module.
* Install a software version compatible with the firmware.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
