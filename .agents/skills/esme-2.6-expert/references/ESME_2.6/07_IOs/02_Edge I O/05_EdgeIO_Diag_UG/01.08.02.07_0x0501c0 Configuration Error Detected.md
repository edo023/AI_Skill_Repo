# 0x0501C0: Configuration Error Detected

Criticality level: Error module

MainFunction: ImplicitCommWithCtr

SubFunction: Configuration

## Principle and Potential Causes

0x0501C001: An error is detected on the configuration of the implicit data received by the I/O module.

0x0501C002: An error is detected on the configuration of the implicit data received by the bus extender.

## Possible Solutions

Contact your local Schneider Electric representative.

## How to Clear the Error Code

Not applicable.
