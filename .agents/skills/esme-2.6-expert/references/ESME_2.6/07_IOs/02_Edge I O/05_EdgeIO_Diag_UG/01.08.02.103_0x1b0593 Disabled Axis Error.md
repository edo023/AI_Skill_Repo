# 0x1B0593: Disabled Axis Error

Criticality level: Error channel

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The AxisState parameter is set to Disabled, the ongoing move command is aborted.

Potential causes:

* PowerEnable is set to FALSE.
* The wiring of Drive Ready input is incorrect or disconnected.

## Possible Solutions

* Set PowerEnable to TRUE.
* Verify the wiring on the Drive Ready input.

## How to Clear the Error Code

Perform a PTO reset (AdminType = 1).
