# 0x1B0143: Node ID Not Allowed In Output Image

Criticality level: Advisory

MainFunction: IOM\_MOTION

SubFunction: Conf

## Principle and Potential Causes

The configuration received contains invalid cyclic node.

There is a misalignment between the configuration software, network interface module, and I/O modules versions.

## Possible Solutions

Download a new configuration to align the configuration software, network interface module, and I/O modules versions.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
