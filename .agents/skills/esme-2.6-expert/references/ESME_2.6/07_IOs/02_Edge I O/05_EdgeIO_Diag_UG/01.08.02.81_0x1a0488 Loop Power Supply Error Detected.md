# 0x1A0488: Loop Power Supply Error Detected

Criticality level: Error channel

MainFunction: IOM\_AIO

SubFunction: l

## Principle and Potential Causes

A loop power supply error is detected on the indicated channel.

Potential causes:

* Short circuit
* Overload
* Undervoltage

## Possible Solutions

Verify the connection and the load.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
