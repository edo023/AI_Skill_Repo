# 0x8201C0: Invalid FDR Server Configuration Detected.

Criticality level: Error module

MainFunction: EthernetServices

SubFunction: FDR

## Principle and Potential Cause

The island configuration retrieved from the fast device replacement server is not valid.

Potential causes:

* The configuration is not compatible with the firmware version.
* The configuration is corrupted.

## Possible Solutions

Create a new island configuration. The FDR server configuration is replaced by the new configuration.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
