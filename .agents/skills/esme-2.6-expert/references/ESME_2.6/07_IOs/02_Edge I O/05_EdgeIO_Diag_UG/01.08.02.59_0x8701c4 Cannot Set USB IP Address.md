# 0x8701C4: Cannot Set USB IP Address

Criticality level: Error module

MainFunction: IPAddressing

SubFunction: MainProcess

## Principle and Potential Causes

The network interface module cannot set the configured IP address gateway address on the USB port.

Potential causes:

* The IP address format is not valid.
* The IP address subnet is already taken by another interface (Fieldbus / IOBUS).

## Possible Solutions

Modify the configured USB IP address of the network interface module and ensure that it is different from the fieldbus interface or the I/O bus interface.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
