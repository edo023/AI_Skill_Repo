# 0x1B0561: Advisory Acceleration/Deceleration Value

Criticality level: Advisory

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The commanded acceleration or deceleration parameter value is out of range.

The commanded acceleration or deceleration value exceeds the Maximum Acceleration or the Maximum Deceleration parameter values, respectively.

## Possible Solutions

Send a new command with an acceleration or deceleration value below the Maximum Acceleration or the Maximum Deceleration parameter values, respectively.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
