# 0x1B0591: Encoder Configuration Not Supported

Criticality level: Error channel

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The encoder is not supported by the module.

## Possible Solutions

Use an encoder that is supported by the module.

## How to Clear the Error Code

Depending on the expert function and its configuration (operating mode, etc.):

* The diagnostic ID automatically clears when the cause is resolved.
* Disable and enable the expert function.

Refer to the [operating mode](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Dio_UG&topicID=0001108067_xml) of your function for more information.
