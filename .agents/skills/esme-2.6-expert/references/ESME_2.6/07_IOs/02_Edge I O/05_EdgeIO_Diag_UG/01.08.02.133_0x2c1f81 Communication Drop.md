# 0x2C1F81: Communication Drop

Criticality level: Error channel

MainFunction: FDM

SubFunction: IOS\_X

## Principle and Potential Causes

The communication with the connected Modbus device(s) is not successful.

## Possible Solutions

Verify the connection between the Field Device Master and the Modbus device.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
