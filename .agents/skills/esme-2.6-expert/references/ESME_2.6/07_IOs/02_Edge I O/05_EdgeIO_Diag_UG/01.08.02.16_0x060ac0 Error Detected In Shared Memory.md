# 0x060AC0: Error Detected In Shared Memory

Criticality level: Error module

MainFunction: SharedMemory

SubFunction: AsynchComm

## Principle and Potential Cause

The device receives too many asynchronous messages. The logs and I/O LED on the bus extender are not updated.

Too many messages are in transit in the system.

## Possible Solutions

Contact your local Schneider Electric representative.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
