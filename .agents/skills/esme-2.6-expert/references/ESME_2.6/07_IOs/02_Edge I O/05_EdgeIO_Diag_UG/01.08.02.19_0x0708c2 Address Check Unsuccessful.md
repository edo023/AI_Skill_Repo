# 0x0708C2: Address Check Unsuccessful (Unrecoverable Error)

Criticality level: Error module

MainFunction: IOBusAddressing

SubFunction: Process

## Principle and Potential Causes

During the module addressing protocol, the address verification of an addressed module is unsuccessful.

Potential causes:

* Several modules are hot-swapped at the same time.
* A module is hot-swapped during the initial addressing phase of a cluster (during power on).

## Possible Solutions

* Do not hot-swap more than one module at a time on a running cluster.
* Do not hot-swap any module during the starting phase of the cluster.

## How to Clear the Error Code

Reboot your cluster.
