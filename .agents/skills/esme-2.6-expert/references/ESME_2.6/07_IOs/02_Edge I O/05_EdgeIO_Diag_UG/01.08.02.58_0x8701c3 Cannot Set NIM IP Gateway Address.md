# 0x8701C3: Cannot Set NIM IP Gateway Address

Criticality level: Error module

MainFunction: IPAddressing

SubFunction: MainProcess

## Principle and Potential Causes

The network interface module cannot set the configured gateway IP address on the fieldbus ports.

The gateway IP address format is not valid.

## Possible Solutions

Modify the gateway IP address of the network interface module.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
