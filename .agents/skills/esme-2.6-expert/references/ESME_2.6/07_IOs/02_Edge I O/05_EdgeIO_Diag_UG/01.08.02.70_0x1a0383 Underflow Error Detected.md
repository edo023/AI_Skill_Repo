# 0x1A0383: Underflow Error Detected

Criticality level: Error channel

MainFunction: IOM\_AIO

SubFunction: Q

## Principle and Potential Causes

An underflow error is detected on the indicated channel.

An output value is read under the Underflow threshold parameter value.

## Possible Solutions

* Adjust the underflow threshold.
* Disable the Underflow Checked parameter if it is not necessary for your application.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
