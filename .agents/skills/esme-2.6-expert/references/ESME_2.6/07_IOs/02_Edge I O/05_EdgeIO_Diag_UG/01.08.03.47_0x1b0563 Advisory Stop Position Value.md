# 0x1B0563: Advisory Stop Position Value

Criticality level: Advisory

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The commanded stop position parameter is out of range.

## Possible Solutions

Send a new command with a valid stop position value.

Ensure that the value respects the configured limits for single-turn and multi-turn positions.

NOTE:

* When your output encoder is configured as a single turn encoder, the value of StopMultiTurnPos must be 0.
* When your output encoder is configured as a multi turn encoder, the value of StopMultiTurnPos must be different than 0 and below the maximum turn value.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
