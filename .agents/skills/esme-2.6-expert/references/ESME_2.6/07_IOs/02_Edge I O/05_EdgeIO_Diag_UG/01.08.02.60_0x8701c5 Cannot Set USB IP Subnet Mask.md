# 0x8701C5: Cannot Set USB IP Subnet Mask

Criticality level: Error module

MainFunction: IPAddressing

SubFunction: MainProcess

## Principle and Potential Causes

The network interface module cannot set the configured IP address subnet mask on the USB port.

The subnet mask is not valid.

## Possible Solutions

Modify the configured USB subnet mask of the network interface module and ensure that it is different from fieldbus interface or the I/O bus interface.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
