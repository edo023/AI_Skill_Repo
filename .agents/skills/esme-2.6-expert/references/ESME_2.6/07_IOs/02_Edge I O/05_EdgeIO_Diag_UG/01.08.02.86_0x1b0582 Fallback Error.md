# 0x1B0582: Fallback Error

Criticality level: Error channel

MainFunction: IOM\_MOTION

SubFunction: Channel/ExpertQ

## Principle and Potential Causes

The communication between the network interface module and the controller is interrupted. The module is in fallback state.

Potential causes:

* The controller stopped, is halted or underwent a power cycle.
* There is an issue with the wiring cable between controller and network interface module.

## Possible Solutions

* Verify the controller state and application.
* Verify the wiring cable from controller to the network interface module.

## How to Clear the Error Code

Disable and enable the expert function.
