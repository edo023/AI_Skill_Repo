# 0x1F0600: Tunneling Request Register Full

Criticality level: Information

MainFunction: IOM\_HART

SubFunction: HART\_Tunneling

## Principle and Potential Causes

The tunneling request register is full. Subsequent request will be rejected.

All transaction ID storage space are occupied.

## Possible Solutions

Write a clear ID queue command (FE hex).

Download the configuration again and send the response with the stored ID to the release register.

## How to Clear the Information

The diagnostic ID automatically clears when the cause is resolved.
