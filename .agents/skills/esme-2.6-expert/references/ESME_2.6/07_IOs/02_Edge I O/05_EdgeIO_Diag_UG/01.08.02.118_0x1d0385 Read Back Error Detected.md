# 0x1D0385: Read Back Error Detected

Criticality level: Error channel

MainFunction: IOM\_DIO

SubFunction: Q

## Principle and Potential Causes

There is a misalignement between the targeted output value and the physical output value.

Externally supplied wire is connected on the indicated channel

## Possible Solutions

Verify the wiring on the channel.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
