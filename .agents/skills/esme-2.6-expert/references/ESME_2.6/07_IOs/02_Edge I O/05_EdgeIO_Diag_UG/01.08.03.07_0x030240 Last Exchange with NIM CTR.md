# 0x030240: Last Exchange with NIM/CTR Unsuccessful

Criticality level: Advisory

MainFunction: ExplicitCommToCtr

SubFunction: LastExchange

## Principle and Potential Causes

The last exchange with the network interface module unit was unsuccessful.

## Possible Solutions

Wait for connection.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
