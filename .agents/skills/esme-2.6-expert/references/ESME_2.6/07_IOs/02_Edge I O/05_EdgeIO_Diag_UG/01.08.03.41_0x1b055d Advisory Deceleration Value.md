# 0x1B055D: Advisory Deceleration Value

Criticality level: Advisory

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The commanded deceleration parameter is out of range.

The commanded deceleration exceeds the Maximum Deceleration parameter value.

## Possible Solutions

Send a new command with a deceleration value within the configured range.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
