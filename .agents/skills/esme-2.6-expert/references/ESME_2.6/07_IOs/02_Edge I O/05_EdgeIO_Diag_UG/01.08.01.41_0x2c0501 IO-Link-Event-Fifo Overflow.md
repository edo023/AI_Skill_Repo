# 0x2C0501: IO-Link-Event-Fifo Overflow

Criticality level: Information

MainFunction: FDM

SubFunction: IO-Link-Event

## Principle and Potential Causes

The least recent event is removed from the FIFO diagnostic queue.

The IO-Link event is discarded because the FIFO diagnostic queue is full.
