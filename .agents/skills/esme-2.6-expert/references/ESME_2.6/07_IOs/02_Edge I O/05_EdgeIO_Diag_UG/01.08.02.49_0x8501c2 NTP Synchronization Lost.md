# 0x8501C2: NTP Synchronization Lost

Criticality level: Error module

MainFunction: TimeMgt

SubFunction: NTP

## Principle and Potential Causes

The network interface module cannot synchronize with the NTP server.

Potential causes:

* The NTP server IP addresses are not configured properly.
* The RJ45 cable is unplugged.
* The NTP servers are down.

## Possible Solutions

* Ensure that the RJ45 cable is connected.
* Verify your NTP configuration.
* Ensure that your NTP servers are connected.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
