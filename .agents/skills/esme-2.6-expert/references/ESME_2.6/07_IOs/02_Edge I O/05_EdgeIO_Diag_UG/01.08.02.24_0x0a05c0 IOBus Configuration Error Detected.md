# 0x0A05C0: IOBus Configuration Error Detected

Criticality level: Error module

MainFunction: PhysicalLayer

SubFunction: IoBusConf

## Principle and Potential Cause

An internal error on the I/O bus configuration of the bus extender is detected.

## Possible Solutions

Contact your local Schneider Electric representative.

## How to Clear the Error Code

Not applicable.
