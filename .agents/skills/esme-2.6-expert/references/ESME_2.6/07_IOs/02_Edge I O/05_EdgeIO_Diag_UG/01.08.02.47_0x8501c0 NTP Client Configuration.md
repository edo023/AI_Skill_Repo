# 0x8501C0: NTP Client Configuration Unsuccessful

Criticality level: Error module

MainFunction: TimeMgt

SubFunction: NTP

## Principle and Potential Causes

The network interface module cannot configure the NTP client.

The NTP server IP addresses are not configured properly.

## Possible Solutions

Configure the NTP settings.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
