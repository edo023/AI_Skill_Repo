# 0x1B05A5: Abort Flag Error on Output

Criticality level: Error channel

MainFunction: IOM\_MOTION

SubFunction: ExpertQ

## Principle and Potential Causes

New generation is requested but Status.AbortFlag = TRUE.

Command.Abort is TRUE.

## Possible Solutions

Set Command.Abort to FALSE.

## How to Clear the Error Code

Send a new Execute command.
