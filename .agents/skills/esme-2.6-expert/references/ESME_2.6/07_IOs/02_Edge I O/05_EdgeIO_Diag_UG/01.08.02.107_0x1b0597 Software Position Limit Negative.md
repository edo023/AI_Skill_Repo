# 0x1B0597: Software Position Limit Negative Error

Criticality level: Error channel

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

CurrentPosition value exceeded the SW Negative Limit parameter on the indicated channel.

Potential causes:

* The axis position is out of the machine limits.
* SW Negative Limit parameter value is not aligned with your installation.

## Possible Solutions

* Verify the axis position regarding the machine limits and take appropriate action.
* Adjust SW Negative Limit parameter value.
* Deactivate SW Negative Limit Enabled parameter.

## How to Clear the Error Code

Perform a PTO reset (AdminType = 1) and send a motion command with a positive direction.
