# 0x8602C1: Configuration Mismatch (Incorrect Modules Detected)

Criticality level: Error module

MainFunction: ConfigMgr

SubFunction: Topology

## Principle and Potential Causes

The island configuration received by the network interface module does not match the physical configuration of your island.

Potential causes:

* An incorrect module is inserted in the indicated slot.
* The island configuration is not up-to-date.

## Possible Solutions

Replace the I/O module in the indicated slot.

Send a new configuration on your network interface module matching the physical configuration of your island.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
