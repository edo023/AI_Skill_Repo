# 0x1B055F: Advisory Frequency Value

Criticality level: Advisory

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The commanded frequency parameter is out of range.

The commanded frequency exceeds 20 kHz (PWM) or 100 kHz (frequency generator).

## Possible Solutions

Send a new command with a frequency value below 20 kHz (PWM) or below 100 kHz (frequency generator).

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
