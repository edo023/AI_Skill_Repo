# 0x010800: OK (Configured)

Criticality level: Information

MainFunction: ConfigMgr

SubFunction: Process

## Principle and Potential Causes

The configuration process is completed successfully.

There are no errors detected on the device. On the network interface module, the I/O modules with the Device mode parameter set to Normal are publishing input correctly. Each module perform I/O process correctly.

## Possible Solutions

Not applicable.

## How to Clear the Information

Not applicable.
