# 0x0C01C1: Synchronization Lost

Criticality level: Error module

MainFunction: Synchro

SubFunction: Process

## Principle and Potential Causes

There are communication issues between the network interface module, the controller and the bus extender.

## Possible Solutions

Wait for synchronization.

If the error persists, contact your local Schneider Electric representative.

## How to Clear the Error Code

Not applicable.
