# 0x2C0343: Port State: Incorrect Device

Criticality level: Advisory

MainFunction: FDM

SubFunction: IO-Link

## Principle and Potential Causes

The IO-Link port changed to the state described.

The connected device is incorrect.

## Possible Solutions

Verify the configuration and the connected device.

## How to Clear the Advisory Code

Not applicable.
