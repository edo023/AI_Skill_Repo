# 0x010844: Configuration Does not Apply on this Product Reference

Criticality level: Advisory

MainFunction: ConfigMgr

SubFunction: Process

## Principle and Potential Causes

The configured module does not match the module connected on the indicated slot. In this case, the network interface module detects the mismatch and displays the module diagnostic Device Mismatch Error Detected.

## Possible Solutions

Send a new configuration on your network interface module matching the physical configuration of your island.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
