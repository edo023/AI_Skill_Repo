# 0x1B055E: Advisory Jerk Value

Criticality level: Advisory

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The commanded jerk ratio parameter is out of range.

The commanded jerk ratio exceeds 100 %.

## Possible Solutions

Send a new command with a jerk ratio value below 100 %.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
