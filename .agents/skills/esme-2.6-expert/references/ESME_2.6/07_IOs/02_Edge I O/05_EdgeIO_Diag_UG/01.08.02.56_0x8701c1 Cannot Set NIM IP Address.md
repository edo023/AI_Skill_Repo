# 0x8701C1: Cannot Set NIM IP Address

Criticality level: Error module

MainFunction: IPAddressing

SubFunction: MainProcess

## Principle and Potential Causes

The network interface module cannot set the configured IP address on the fieldbus ports.

Potential causes:

* The IP address format is considered not valid.
* Another device has the same IP address as the network interface module.
* The subnet IP address is already taken by another interface (USB / IOBUS).

## Possible Solutions

When you configure the subnet IP of the network interface module, ensure that it is different from USB interface or the I/O bus interface.

## How to Clear the Error Code

The diagnostic ID automatically clears when the cause is resolved.
