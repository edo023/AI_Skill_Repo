# 0x040740: OPC UA C/S Operation Unsuccessful

Criticality level: Advisory

MainFunction: ExplicitCommToCtr

SubFunction: Server

## Principle and Potential Causes

The OPC UA server process API was unsuccessful.

Potential causes:

* There is a server process interruption.
* The system is not operational.

## Possible Solutions

Contact your local Schneider Electric representative.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
