# 0x1B0596: Software Position Limit Positive Error

Criticality level: Error channel

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

CurrentPosition value exceeded the SW Positive Limit parameter on the indicated channel.

Potential causes:

* The axis position is out of the machine limits.
* A wire is missing or damaged on input assigned to the SW Positive Limit parameter.

## Possible Solutions

* Verify the axis position regarding the machine limits and take appropriate action.
* Verify the wiring on positive input assigned to the SW Positive Limit parameter.

## How to Clear the Error Code

Perform a PTO reset (AdminType = 1) and send a motion command with a negative direction.
