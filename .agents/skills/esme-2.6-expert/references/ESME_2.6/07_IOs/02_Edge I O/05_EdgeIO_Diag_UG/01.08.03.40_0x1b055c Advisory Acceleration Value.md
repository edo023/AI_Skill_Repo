# 0x1B055C: Advisory Acceleration Value

Criticality level: Advisory

MainFunction: IOM\_MOTION

SubFunction: Channel

## Principle and Potential Causes

The commanded acceleration parameter is out of range.

The commanded acceleration exceeds the Maximum Acceleration parameter value.

## Possible Solutions

Send a new command with an acceleration value within the configured range.

## How to Clear the Advisory Code

The diagnostic ID automatically clears when the cause is resolved.
