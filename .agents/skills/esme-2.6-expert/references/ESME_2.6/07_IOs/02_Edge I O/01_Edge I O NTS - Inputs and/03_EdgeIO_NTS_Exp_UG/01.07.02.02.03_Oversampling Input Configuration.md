# Oversampling Input Configuration

The following table presents the configurable parameters for the Oversampling Input mode:

| --- | --- | --- | --- |
| Input Location  InputLocation | 0: I0  1: I1  2: I2  3: I3  4: I4  5: I5  6: I6  7: I7  255: Disabled\* | ENUM | Location of the input. |
| Sampling Step Mode  SamplingStepMode | 0: 8\*  1: 16  2: 32 | ENUM | Sets the number of steps per cycle. |
| Input Filter  InputFilter | 0: 0  1: 0.0002  2: 0.0005  3: 0.001  4: 0.002\*  5: 0.005  6: 0.01  7: 0.05  8: 0.08  9: 0.5  10: 1  11: 4  12: 12 | ENUM | Allows reducing the effect of bounce on the input.
