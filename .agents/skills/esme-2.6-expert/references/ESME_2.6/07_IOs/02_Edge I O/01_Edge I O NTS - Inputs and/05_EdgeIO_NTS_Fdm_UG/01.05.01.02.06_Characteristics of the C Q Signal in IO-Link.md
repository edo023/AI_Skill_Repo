# Characteristics of the C/Q Signal in IO-Link Master Mode

The following table describes the characteristics of the C/Q signal in IO-Link master mode of the NTSFIO0400 module:

| Characteristics | Value |
| --- | --- |
| Transfer rates | * COM1: 4.8 kbaud * COM2: 38.4 kbaud * COM3: 230.4 kbaud |
| Data format | 1 start bit, 8 data bits, 1 parity bit (even), 1 stop bit |
| Data storage | 2048 bytes |

NOTE: C/Q is the IO-Link communication line.
