# Main Characteristics

The following table describes the main characteristics of the NTSEHC0100 input module:

| Main Characteristics | Value |
| --- | --- |
| Number of physical inputs | 6 |
| Number of incremental channels | 1  The incremental input channel can use up to 4 inputs. |
| Minimum additional inputs | 2  Inputs not used for the incremental channel can be used as auxiliary inputs or as inputs of Simple counting functions. |
| Encoder type | High speed incremental counter |
| Input frequency | 250 kHz |
| Encoder supply (external) | 10...30 Vdc |
| Encoder input | Single ended |
| Resolution | Main counting function: 32 bits  Simple counting function: 16 bits |
| Operating mode | Synchronous, isochronous and asynchronous |
