# Overview

The NTSEDM0822 module is equipped with:

* 4 fast discrete inputs.
* 4 standard discrete outputs.

For information on the configuration of this module, refer to [Inputs and Outputs Configuration](EmbeddedIOsConfiguration-34359EE8.html).

Alternatively, you can add up to eight expert functions to your module using the Add a function button. The following expert functions are available:

* [Output Pulse After Input Event Interface](OutPulseAfterInEventFunction-87329D68.html).
* [Oversampling Input Configuration](OversamplingInputConfiguration-86273006.html).
* [Timestamp Input Configuration](TimestampInputConfiguration-842583BF.html).
* [Oversampling Output Configuration](OversamplingOutputConfiguration-862869D5.html).
* [Timestamp Output Configuration](TimestampOutputConfiguration-8626719E.html).
