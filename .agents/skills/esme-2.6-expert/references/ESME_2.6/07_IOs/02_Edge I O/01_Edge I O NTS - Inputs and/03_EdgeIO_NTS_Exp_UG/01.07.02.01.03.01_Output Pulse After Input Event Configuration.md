# Output Pulse After Input Event Configuration

## Configurable Parameters

The following table presents the configurable parameters for the Output Pulse After Input Event mode:

| Displayed Name  *Parameter Name* | Value | Data Type | Description |
| --- | --- | --- | --- |
| Input Location  InputLocation | 0: I0  1: I1  2: I2  3: I3  4: I4  5: I5  6: I6  7: I7  255: Disabled\* | ENUM | Location of the input. |
| Output Location  OutputLocation | 0: Q0  1: Q1  2: Q2  3: Q3  4: Q4  5: Q5  6: Q6  7: Q7  255: Disabled\* | ENUM | Location of the output. |
| Input Edge  InputEdge | 0: Rising\*  1: Falling  2: Both | ENUM | Sets the input event edge to react and generate the output pulse. |
| Input Filter  InputFilter | 0: 0  1: 0.0002  2: 0.0005  3: 0.001  4: 0.002\*  5: 0.005  6: 0.01  7: 0.05  8: 0.08  9: 0.5  10: 1  11: 4  12: 12 | ENUM | Allows reducing the effect of bounce on the input.  Changes to the signal are only detected if the pulse width of the input signal is longer than the filter time (ms).  The physical input signal is delayed by the configured input filter time. |
| \* Parameter default value | | | |

## Implicit Parameters

The following table presents the input implicit data for the Output Pulse After Input Event mode:

| *Parameter Name* | Value | Data Type  R/W | Description |
| --- | --- | --- | --- |
| EnabledFlag | FALSE\*  TRUE | BOOL  R/- | When TRUE, catches the input and the pulse generation is enabled. |
| PulseNb | 0\*...255 | BYTE  R/- | Counts the number of pulses generated to react at an input event. |
| ErrorFlag | FALSE\*  TRUE | BOOL  R/- | When TRUE, an error is detected.  Disables the function. |
| ErrorId | 0\*: No Error detected  1: Internal Field Power Supply  2: Short circuit  3: Fallback | ENUM  R/- | Error type.  When ErrorFlag = TRUE, the ErrorId parameter provides the error detected. |
| \* Parameter default value | | | |

The following table presents the output implicit data for the Output Pulse After Input Event mode:

| *Parameter Name* | Value | Data Type  R/W | Description |
| --- | --- | --- | --- |
| Enable | FALSE\*  TRUE | BOOL  R/W | When TRUE, starts to catch the input event and the pulse generation. |
| Event Time | 0\*...4,294,967,295 | UINT32  R/W | Time between the input event and the output pulse generation in nanoseconds (ns). |
| Pulse Time | 0\*...4,294,967,295 | UINT32  R/W | Duration of the output pulse in nanoseconds (ns). |
| \* Parameter default value | | | |
