# Overview

This section describes the standard input and output parameters of the NTSEHC0120H module.

The module supports the following counting functions:

* [Simple Counting Function](SimpleCountingFunction-E04B30F0.html)
* [Frequency Meter Function](FrequencyMeterFunction-E04E616C.html)
* [Period Meter Function](PeriodMeterFunction-E04FA1B3.html)
* [Ratio Meter Function](RatioMeterFunction-E051A77D.html)
* [Single Phase Event Counting Function](TPC_EDGEIOCountingSinglePhaseEventC-E347DB2B.html)
* [Single Phase Counting Function](SinglePhaseCountingFunction-E0D54398.html)
* [Dual Phase Counting Function](TPC_EDGEIODualPhaseCountingFunction-E348FA7A.html)
* [PWM Output Function](TPC_EDGEIOCountingPWMOutputFunction-E3491164.html)
* [Reflex Output Sub-Function](TPC_EDGEIOCountingReflexOutputSubFu-E349278D.html)
* [Enable Function](EnableFunction-284A99B2.html)
* [Capture Function](TPC_EdgeIOCaptureFunction-FA79A946.html)
* [Preset Function](PresetSubFunction-FA7D6B9D.html)
