# Input Characteristics

The table below describes the input characteristics of the NTSDDI0402/NTSDDI0402H input modules:

| Characteristics | | Value |
| --- | --- | --- |
| Input wiring mode | | 1-/2-/3-wire |
| Nominal input current | | * 2.3 mA (Type 3) * 6.32 mA (Type 2) |
| Input voltage | Logic state 1 | 11…30 Vdc |
| Logic state 0 | < 5 Vdc |
| Input current | Logic state 1 | * 2 mA minimum (Type 3) * 6 mA minimum (Type 2) |
| Logic state 0 | 1.5 mA maximum |
| Input impedance | | * 8 kΩ (Type 3) * 3.58 kΩ (Type 2) |
| Response time on input | Logic state 1 to logic state 0 | < 10 µs |
| Logic state 0 to logic state 1 | < 20 µs |
| Input filter time | Hardware | < 100 µs |
| Software | Configurable |
| Paralleling of inputs | | Yes |
| Input protection | | * Over voltage protection * Overcurrent protection on sensor supply * Reverse polarity protection |
| Input diagnostic | | Broken wire  Short circuit to ground for Type 3 |
| Monitoring functions | | * Broken wire detection on sensor input is possible with an additional resistor (Rbwd) in parallel: 39 kΩ, 1/16 W, 1%. * Internal power supply error detection. * Sensor supply error detection. * Short circuit to 24 Vdc detection is possible with an additional resistor (Rshort) in series: 3.3 kΩ 1/4 W, 1%.   NOTE: Connect the resistors to the sensor connections. |
