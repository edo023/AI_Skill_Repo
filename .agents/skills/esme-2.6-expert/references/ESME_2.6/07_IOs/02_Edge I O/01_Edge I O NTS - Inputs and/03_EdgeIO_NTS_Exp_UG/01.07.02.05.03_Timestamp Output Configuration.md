# Timestamp Output Configuration

## Configurable Parameters

The following table presents the configurable parameters for the Timestamp Output mode:

| Displayed Name  *Parameter Name* | Value | Data Type | Description |
| --- | --- | --- | --- |
| Output Location  OutputLocation | 0: Q0  1: Q1  2: Q2  3: Q3  4: Q4  5: Q5  6: Q6  7: Q7  255: Disabled\* | ENUM | Location of the output. |
| \* Parameter default value | | | |

## Implicit Parameters

The following table presents the input implicit data for the Timestamp Output mode:

| *Parameter Name* | Value | Data Type  R/W | Description |
| --- | --- | --- | --- |
| Status | 0\*...15 | BYTE  R/- | Bit 0: EnabledFlag: When TRUE, the output profile generation is ended (NbCycle is reached).  Bit 1: WarningRising: When TRUE, a rising edge that is not yet generated is overwritten, or TimestampOutEdgeRising is out of the timing range.  Resets after the generation is done or aborted, or when an error is detected.  Bit 2: WarningFalling: When TRUE, a falling edge, that is not yet generated, is overwritten, or TimestampOutEdgeRising is out of the timing range.  Bit 3: ErrorFlag: When TRUE, an error is detected. Disables the timestamp output.  NOTE: Unused bits are reserved. |
| ErrorId | 0\*: No Error detected  1: Internal Field Power Supply  2: Short circuit  3: Fallback | ENUM  R/- | Error type.  When ErrorFlag = TRUE, the ErrorId parameter provides the error detected. |
| \* Parameter default value | | | |

The following table presents the output implicit data for the Timestamp Output mode:

| *Parameter Name* | Value | Data Type  R/W | Description |
| --- | --- | --- | --- |
| Command | 0\*...15 | BYTE  R/W | Bit 0: EnableGeneration: When TRUE, starts to apply the timestamp output values. A falling edge sets ErrorFlag to FALSE.  Bit 1: EnableRising: When TRUE, the edge rising time is applied.  Bit 2: EnableFalling: When TRUE, the edge falling time is applied.  Bit 3: WarningAck: At rising edge, resets WarningRising and WarningFalling.  NOTE: Unused bits are reserved. |
| TimestampOutEdgeRising | 0\*...999,999,999 | UINT32  R/W | Absolute time of the rising edge to apply on the output in nanoseconds (ns). |
| TimestampOutEdgeFalling | 0\*...999,999,999 | UINT32  R/W | Absolute time of the falling edge to apply on the output in nanoseconds (ns). |
| \* Parameter default value | | | |
