# Period Meter Function

## Description

The Period Meter function measures a duration on input A.

There are 2 sub-modes for the Period Meter function:

* Edge to opposite: Measures the duration of an event. While the enable condition is TRUE, the measurement starts at the rising edge of input A and stops at the falling edge of input A.

  ![](images/DD00864524.png)
* Edge to edge: Measures the duration between two events. While the enable condition is TRUE, the measurement starts at the rising edge of input A and stops at the next rising edge of input A.

  ![](images/DD00864523.png)

The duration measured is defined by the Resolution parameter (0.1 µs, 1 µs, 10 µs, 100 µs, 1,000 µs): Duration = Measured value x Resolution.

With the Period Meter function, the [enable condition](EnableFunction-284A99B2.html) is controlled with a software command or a physical input.

The following table describes the features of the Period Meter function:

| Item | Description |
| --- | --- |
| Inputs | The Period Meter function requires a single fast input A Location to operate.  An optional input can be assigned to EN Location. |
| Counter register | 32 bits |
| Reflex Output | Up to 4 reflex outputs can be configured. For more information about reflex outputs, refer to [Reflex Output Sub-Function](TPC_EDGEIOCountingReflexOutputSubFu-E349278D.html). |
| Measurement Range | 0...4,294,967 ms |
| Shortest Measurable Period | 0.1 µs |
| Period Value Update Rate | At the end of each event measured. |
| Reflex Output State Update Rate | Depends on the comparison trigger (maximum 20 μs). |

## Reflex Output Behavior

The Period Meter function measures a duration on input A. In order to perform this action, there is an internal real-time counter based on the Resolution parameter measuring the duration of the event, independent from the CounterValue parameter.

When you configure one or more reflex outputs with this function, the compare function compares the trigger conditions to the internal counter value as shown in the following example:

| Parameter | Reflex Output 0 | Reflex Output 1 |
| --- | --- | --- |
| ReflexOutput●Condition | Counter Within Threshold | Counter Cross Threshold |
| ReflexOutput●CounterWithinThreshold value | 4(1) | - |
| ReflexOutput●CounterCrossThreshold | - | Pulse = Counter crosses TH2 upwards |
| ReflexOutput●PulseWidth | - | 50 |
| (1) Bit 2 = TRUE, the counter value is between threshold 1 + 1 and threshold 2. | | |
