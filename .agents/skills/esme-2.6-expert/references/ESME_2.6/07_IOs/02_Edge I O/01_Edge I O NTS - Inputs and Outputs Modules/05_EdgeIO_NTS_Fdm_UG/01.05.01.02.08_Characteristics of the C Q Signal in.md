# Characteristics of the C/Q Signal in Discrete Output Mode

The following table describes the characteristics of the C/Q signal in discrete output mode of the NTSFIO0400 module:

| Characteristics | Value |
| --- | --- |
| Output type | Push-pull |
| Output voltage | 24 Vdc |
| Maximum continuous output current | 0.2 A per output |
| Total output current | 0.8 A maximum |
| Maximum overcurrent threshold | Minimum 0.28 A, maximum 0.41 A per output |
| Output protection | * Thermal protection * Short circuit protection * Overcurrent protection   NOTE: To restart the channel after an overcurrent event, it is necessary to disconnect the load. |
