# Master Cycle Time Description

The Master cycle time bit field defines the cycle time of the data transmission between the IO-Link master and the IO-Link device at the port.

All bits at 0: The IO-Link master uses the computed fastest master cycle time.

Master cycle time selection:

* Bits 0...5: These bits represent a six-bits multiplier for the calculation of the cycle time (0...63).
* Bits 6...7: These bits specify the time base for the calculation of the cycle time.

The following table presents the permissible time base encoding and the corresponding cycle time ranges:

| Time base encoding | Time base value | Calculation | Cycle time range |
| --- | --- | --- | --- |
| 00 | 0.1 ms | Mulitplier x Time base | 0.4(1)...6.3 ms |
| 01 | 0.4 ms | 6.4 ms + Mulitplier x Time base | 6.4...31.6 ms |
| 10 | 1.6 ms | 32.0 ms + Mulitplier x Time base | 32.0...132.8 ms |
| 11 | Reserved | Reserved | Reserved |
| (1) The value 0.4 ms is the minimum cycle time at a transmission rate of 230.4 kbit/s. | | | |
