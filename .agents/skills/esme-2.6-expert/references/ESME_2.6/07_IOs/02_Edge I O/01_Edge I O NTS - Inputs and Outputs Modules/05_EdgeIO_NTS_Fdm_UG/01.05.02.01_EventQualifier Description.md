# EventQualifier Description

The following table presents the structure of the EventQualifier bitfield:

| Bits | Value | Description |
| --- | --- | --- |
| Bits 0...2  INSTANCE | * 0: Indeterminable * 1...3: Reserved * 4: Application * 5...7: Reserved | These bits indicate the instance of an event. |
| Bit 3  SOURCE | * 0: Device (remote) * 1: Master/Port (local) | This bit indicates the source of the event. |
| Bits 4...5  TYPE | * 0: Reserved * 1: Notification * 2: Advisory * 3: Error | These bits indicate the type of the event. |
| Bits 6...7  MODE | * 0: Reserved * 1: Event single shot * 2: Event disappears * 3: Event appears | These bits indicate the mode of the event. |
