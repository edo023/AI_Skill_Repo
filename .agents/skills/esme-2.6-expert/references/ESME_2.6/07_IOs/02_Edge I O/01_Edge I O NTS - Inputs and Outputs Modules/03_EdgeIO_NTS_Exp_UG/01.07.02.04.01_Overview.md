# Overview

The Oversampling Output mode generates an output profile based on the sampling step and the required profile defined by OversampledOutputValue.

The number of samples per cycle can be 8 bits, 16 bits, or 32 bits.

The duration of each step depends on the configured IO Bus Cycle Time and Sampling step mode, the following table lists the possible combinations:

| Sampling step mode | IO Bus Cycle Time | | |
| --- | --- | --- | --- |
| 1 ms | 2 ms | 4 ms |
| 8-bit | 125 μs | 250 μs | 500 μs |
| 16-bit | 62.5 μs | 125 μs | 250 μs |
| 32-bit | 31.25 μs | 62.5 μs | 125 μs |
