## Output Pulse After Input Event Regarding IOM Operating Modes

| Module Transition to State | Output Pulse After Input Event Callback Operations | Output Pulse After Input Event Resulting State |
| --- | --- | --- |
| **SYS\_BOOTING** | - | Not configured |
| **SYS\_DEFAULT\_CONF** | - | Not configured |
| **SYS\_CONFIGURED** | Configures Output Pulse After Input Event according to the received configuration. | Configured |
| **SYS\_DRIVEN\_OUTPUT** | Operates according to the received data. | Driven\_Operational |
| **SYS\_FALLBACK\_OUTPUT** | Applies the fallback value to the output. | Driven\_Fallback |

## Output Pulse After Input Event States

| Output Pulse After Input Event State | Output Pulse After Input Event Data Exchange | Output Pulse After Input Event Status | Output Pulse After Input Event Physical Output |
| --- | --- | --- | --- |
| Not configured | - | - | 0 |
| Configured | Produce data: Input image | EnabledFlag = FALSE  PulseNb = 0  ErrorFlag = FALSE  ErrorId = No Error detected | 0 |
| Driven\_Operational | Receive data: Output image  Produce data: Input image | Refer to [Driven\_Operational](#OutPulseAfterInEventOperatingMo-7ED6D5F4__Driven_Operational-CEBE4685). | Output Pulse After Input Event value |
| Driven\_Fallback | Produce data: Input image | EnabledFlag = FALSE  PulseNb is not updated and it returns to the last valid value  ErrorFlag = TRUE  ErrorId = Fallback (or a more prior ErrorId) | Fallback value |

## Driven\_Operational

| Output Pulse After Input Event Commands | Condition | Output Pulse After Input Event Status | Behavior |
| --- | --- | --- | --- |
| Enable = TRUE | No error detected | EnabledFlag = TRUE  PulseNb is updated  ErrorFlag = FALSE  ErrorId = No Error Detected | Enables the Output Pulse After Input Event mode. |
| Enable = TRUE | Error detected | EnabledFlag = FALSE  PulseNb is not updated and it returns to the last valid value  ErrorFlag = TRUE  ErrorId is updated | Disables the Output Pulse After Input Event mode.  NOTE: To acknowledge the error and enable the Output Pulse After Input Event mode, Enable must be set to FALSE then to TRUE (after the error condition is removed). |
| Enable = TRUE | - | EnabledFlag = FALSE  PulseNb is not updated and it returns to the last valid value  ErrorFlag = FALSE  ErrorId = No Error Detected | Disables the Output Pulse After Input Event mode. |
