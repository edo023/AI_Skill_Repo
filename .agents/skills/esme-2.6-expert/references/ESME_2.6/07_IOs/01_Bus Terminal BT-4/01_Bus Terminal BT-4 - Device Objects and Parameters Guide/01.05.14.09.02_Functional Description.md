# Functional Description

Functional Description

Disables the encoder monitoring.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | Encoder check active. |
| yes / 1 | BOOL | Encoder monitoring (track and cable monitoring) not active. |

NOTE: If the encoder simulation is operated in internal master mode, the external encoder input 0 is ignored. This input does not issue any detected errors.

Interplay of track monitoring parameters CheckOff and ZeroTrackCheckOff

| CheckOff | ZeroTrackCheckOff | Active Monitorings |
| --- | --- | --- |
| no / FALSE | no / FALSE | Track A, B, and zero track. |
| no / FALSE | yes / TRUE | Track A and B. |
| yes / TRUE | no / FALSE | No monitoring. |
| yes / TRUE | yes / TRUE | No monitoring. |
