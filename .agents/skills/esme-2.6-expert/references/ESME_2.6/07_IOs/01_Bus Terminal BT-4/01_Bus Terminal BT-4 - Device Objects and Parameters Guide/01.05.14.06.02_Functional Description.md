# Functional Description

Functional Description

Specifies the resolution of the connected encoder (see encoder nameplate).

NOTE: The incremental encoder input functions internally to increase resolution by quadrupling the impulse. The system automatically takes this into account.

NOTE: Corruption of the velocity signal during resolution modification. The [velocity signal](MasterEnc_Incremental-13.htm#XREF_D_SE_0076201_1) is briefly invalid, which can cause other detected errors. Only modify the parameter value at encoder standstill and when the program is inactive.
