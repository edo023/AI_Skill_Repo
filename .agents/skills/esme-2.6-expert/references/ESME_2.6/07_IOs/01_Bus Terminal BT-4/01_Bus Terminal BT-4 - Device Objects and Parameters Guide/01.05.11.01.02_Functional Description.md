# Functional Description

Functional Description

Using the parameter LastModule, you can choose whether the module is connected to the PacNet as the last module.

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | BOOL | The module was not connected to the PacNet as the last module. |
| H / 1 | BOOL | The module was connected to the PacNet as the last module. The PacNetbus output ends with a terminating plug. |
