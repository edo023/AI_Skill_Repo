# Functional Description

Functional Description

Disables the encoder monitoring.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | DINT | Encoder check active. |
| yes / 1 | DINT | Encoder check not active.  The diagnostic message [8601 Master Encoder signal out of range](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064023_1) is disabled. |
