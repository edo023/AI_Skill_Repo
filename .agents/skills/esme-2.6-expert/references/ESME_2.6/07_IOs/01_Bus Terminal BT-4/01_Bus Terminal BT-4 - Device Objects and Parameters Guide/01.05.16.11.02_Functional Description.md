# Functional Description

Functional Description

Displays the actual [filtered](../MasterEnc_SinCos/MasterEnc_SinCos-6.htm#XREF_D_SE_0076216_1) velocity of the encoder at the encoder shaft in units/s.

A unit is defined using the parameter [FeedConstant](IncrementalEncoderOutput-7.htm#XREF_D_SE_0076271_1).
