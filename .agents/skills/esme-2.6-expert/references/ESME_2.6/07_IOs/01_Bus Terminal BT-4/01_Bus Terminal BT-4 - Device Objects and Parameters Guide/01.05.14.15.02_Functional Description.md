# Functional Description

Functional Description

When [zero track detection](MasterEnc_Incremental-14.htm#XREF_D_SE_0076202_1) is enabled, this bit indicates whether a zero track has been detected.

| Value | Data type | Meaning |
| --- | --- | --- |
| false / 0 | BOOL | No Zero Track detected. |
| true / 1 | BOOL | Zero Track detected. |

NOTE: When a zero track is detected, the [Position](../IncrementalEncoderOutput/IncrementalEncoderOutput-10.htm#XREF_D_SE_0076274_1) of the incremental encoder input is set to 0. The detection of a zero impulse has no effect on the Position parameter of any incremental encoder simulation output that may be present.
