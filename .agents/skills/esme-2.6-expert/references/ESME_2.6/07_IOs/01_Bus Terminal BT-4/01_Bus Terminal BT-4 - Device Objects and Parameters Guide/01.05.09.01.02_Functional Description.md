# Functional Description

Functional Description

Using the parameter FailSave, you can choose whether the output data is to be saved in the case of a detected bus error.

Bus Terminal BT-4/DIO1

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | BOOL | If a detected error occurs on the PacNet bus, the outputs are reset. |
| H / 1 | BOOL | If a detected error occurs on the PacNet bus, the states of the outputs are maintained. |

Bus Terminal BT-4/ENC1

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | BOOL | If a detected error occurs on the PacNet bus, the incremental encoder output is switched off (high-impedance). |
| H / 1 | BOOL | If a detected error occurs on the PacNet bus, the incremental encoder output remains enabled with its current value. |

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s can occur if there is a lot of data transferred on the parameter channel.
