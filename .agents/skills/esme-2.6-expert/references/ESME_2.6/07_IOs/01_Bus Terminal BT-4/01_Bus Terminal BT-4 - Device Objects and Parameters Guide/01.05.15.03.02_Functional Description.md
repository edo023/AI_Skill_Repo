# Functional Description

Functional Description

Enables the encoder.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The encoder is not processed by the real-time process.  The physical master encoder deactivates when it is not connected. Detection takes place once the system is started. |
| on / 1 | BOOL | The encoder is processed by the real-time process. |
