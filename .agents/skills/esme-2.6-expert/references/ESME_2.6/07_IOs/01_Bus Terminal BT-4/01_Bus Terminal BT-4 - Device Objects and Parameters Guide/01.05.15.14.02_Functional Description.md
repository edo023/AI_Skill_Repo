# Functional Description

Functional Description

Describes the signal quality of the analog sine and cosine tracks of the encoder. The parameter SignalQuality represents the phasing and the form of the track signals.

The sine and cosine signals must correspond to the following formula:

Sine signal2 + cosine signal2 = 1 (100 %)

The SignalQuality parameter represents this formula normalized to 100 %.

The [SignalQualityLimit](../../../../../../api/crossBook?lang=en-US&virtualBookName=MasterEnc_SinCos-15.htm#XREF_D_SE_0076253_1) parameter determines the threshold value for triggering diagnostic message [8601 Master encoder signal out of the range](../../../PD.Diagnostic&topicID=D_SE_0064023_1).
