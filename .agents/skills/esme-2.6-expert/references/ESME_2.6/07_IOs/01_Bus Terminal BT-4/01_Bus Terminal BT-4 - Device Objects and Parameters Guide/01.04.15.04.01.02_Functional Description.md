# Functional Description

Functional Description

Displays the bit field for enabling the diagnostic message Wiring errors (error code [8788 Wiring error](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064064_1)) with ext. diagnostic message Overload or Openload. Each bit is assigned to an output corresponding to the bit number.

| Bit | Meaning |
| --- | --- |
| 0 | The output is not monitored. |
| 1 | Controllers PacDrive LMC Pro, PacDrive LMC Eco  The output is monitored and a diagnostic message is issued if monitoring with the parameters OpenloadDiagMsk or OverloadDiagMsk is also enabled. |
