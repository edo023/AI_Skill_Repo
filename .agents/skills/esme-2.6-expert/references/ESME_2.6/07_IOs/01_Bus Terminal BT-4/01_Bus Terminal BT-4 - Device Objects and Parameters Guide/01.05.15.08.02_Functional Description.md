# Functional Description

Functional Description

A synchronization error on coupled machines is detected in the following cases:

o[8601 Master Encoder signal out of range](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064023_1) (SinCos)

o[8782 Master Encoder signal out of range](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064061_1) (incremental encoder input)

o[8522 C2C encoder error detected](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064013_1) (C2C Encoder Input)

The parameter ErrorDeceleration can bring the system to a standstill with a defined velocity deceleration. When a track error is detected, the velocity is set to 0 with the defined deceleration. If 0 is entered in ErrorDeceleration, then no velocity deceleration is created by the system.
