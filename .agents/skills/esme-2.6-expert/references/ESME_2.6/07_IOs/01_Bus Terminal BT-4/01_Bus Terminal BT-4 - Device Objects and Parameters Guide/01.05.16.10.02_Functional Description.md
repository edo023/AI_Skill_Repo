# Functional Description

Functional Description

Displays the current encoder position. The value range runs from 0 to [FeedConstant](IncrementalEncoderOutput-7.htm#XREF_D_SE_0076271_1).

With negative direction of rotation, clockwise rotation causes the value range to run in decreasing order.

At system startup, the encoder position is set to 0. If the incremental encoder is connected to a bus terminal, the Position is not 0 after the controller has been reset. 0 is only displayed if the controller is also reset.

Use the parameter [State](IncrementalEncoderOutput-6.htm#XREF_D_SE_0076270_1) to check the availability or validity of the encoder position data.

NOTE: Here the parameter Position is an auxiliary parameter which is useful in the up phase.
