# FW_Version

![](images/logo.png)

FW\_Version

FW\_Version

General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | Bus terminals BT-4/DIO1 and BT-4/DIO1 TP,  Bus terminal BT-4/ENC1 |
| Traceable | No |

Functional Description

Reads out the version number of the bus terminal.

NOTE: The value is read out once when the system starts and is not updated afterward.

| Value | Data type | Meaning |
| --- | --- | --- |
| xxxxx | STRING (20) | The firmware version was determined and is displayed in the parameter. |
| Virtual | STRING (20) | The bus address of the bus terminal is in the PLC configuration >= 100. |
| unavailable | STRING (20) | The firmware version could not be determined.  Causes:  oBus error detected  oHardware does not support reading the version |
