# Enable

![](images/logo.png)

Enable

General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Incremental Encoder Output |
| Traceable | No |

Functional Description

The following parameters are edge-sensitively initialized and reset:

ooff -> on: [Position](IncrementalEncoderOutput-10.htm#XREF_D_SE_0076274_1) = 0; [Velocity](IncrementalEncoderOutput-11.htm#XREF_D_SE_0076275_1) = 0; Output frequency = 0 Hz

oon -> off: Velocity = 0; Output frequency = 0 Hz

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The encoder is not processed by the real-time process. |
| on / 1 | BOOL | The encoder is processed by the real-time process. |
