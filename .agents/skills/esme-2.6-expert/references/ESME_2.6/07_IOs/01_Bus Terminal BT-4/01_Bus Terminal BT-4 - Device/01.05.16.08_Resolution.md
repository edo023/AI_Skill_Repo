# Resolution

![](images/logo.png)

Resolution

General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC Pro |
| Traceable | No |

Functional Description

Specifies the resolution of the incremental encoder output.

NOTE: Corruption of the velocity signal during resolution modification. The [velocity signal](IncrementalEncoderOutput-11.htm#XREF_D_SE_0076275_1) is briefly invalid, which can cause other detected errors. Only modify the parameter value at encoder standstill and when the program is inactive.

Example of Calculating Maximum Resolution

Motor with 6000 revolutions/minute -> 100 revolutions/second

Maximum resolution = 1,000,000 increments/second / 100 revolutions/second = 10,000 increments/revolution

NOTE: Be sure to maintain a reserve of at least 10 % for the calculated maximum resolution; otherwise you could receive the diagnostic message 8780 Encoder output frequency > 1 MHz.
