# CaptureOK

![](images/logo.png)

CaptureOK

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  OnBoard I/O,  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

Functional Description

Displays the status of the Touchprobe event.

A valid Touchprobe event has occurred and the result has been provided in the parameter [CaptureValue](../../../../../../api/crossBook?lang=en-US&virtualBookName=DigMeasInGroup-9.htm#XREF_D_SE_0075676_1). The value is reset to no / 0 when [FC\_TPEdge()](../../../PD.Lib.SystemInterface&topicID=D_SE_0085375_1) is called again.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | The result of the Touchprobe event has been reset. |
| yes / 1 | BOOL | The result of the Touchprobe event is provided in the parameter CaptureValue. |
