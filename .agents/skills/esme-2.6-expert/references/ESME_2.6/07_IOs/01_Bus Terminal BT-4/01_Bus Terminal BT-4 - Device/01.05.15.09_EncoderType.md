# EncoderType

![](images/logo.png)

EncoderType

General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | SinCos Encoder Input |
| Traceable | No |

Functional Description

Displays the encoder type.

Supported encoder types with Hiperface interface

| Value | Data type | Meaning |
| --- | --- | --- |
| 16#02 | DINT | SinCos encoder SCS 60/70 |
| 16#07 | DINT | SinCos encoder SCM 60/70 multi-turn |
| 16#22 | DINT | SinCos encoder SRS 50/60/64 and SCK 25/35/40/45/53 single turn |
| 16#27 | DINT | SinCos encoder SRM 50/60/64 and motor 25/35/40/45/53 multi-turn |
| 16#32 | DINT | SinCos encoder SKS 36 single turn (from V00.22.00) |
| 16#37 | DINT | SinCos encoder SKM 36 multi-turn (from V00.22.00) |
| 16#42 | DINT | SinCos encoder SEK 52 single turn (from V00.22.00) |

NOTE: The value is read over the serial interface of the encoder when the controller is booted. This takes a few seconds. Thus, if the program queries the parameter value too early, the parameter value is not yet up to date and has the value 0.
