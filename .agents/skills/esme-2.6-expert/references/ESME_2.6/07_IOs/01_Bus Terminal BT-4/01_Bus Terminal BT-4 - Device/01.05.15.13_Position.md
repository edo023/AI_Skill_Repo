# Position

![](images/logo.png)

Position

General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | SinCos Encoder Input |
| Traceable | No |

Functional Description

Displays the current encoder position.

For single-turn encoders, the position is from 0 to 1 revolution, for multi-turn encoders the position is 0...4096 revolutions.

With negative direction of rotation, clockwise rotation causes the value range to run in decreasing order.

At system startup, the current encoder position is read.

Use the parameter [State](MasterEnc_SinCos-10.htm#XREF_D_SE_0076237_1) to check the availability or validity of the encoder position data.
