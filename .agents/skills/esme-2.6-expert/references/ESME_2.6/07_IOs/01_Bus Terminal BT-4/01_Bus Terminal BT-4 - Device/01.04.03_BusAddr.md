# BusAddr

General

|  |  |
| --- | --- |
| Type | ER |
| Devices supporting the parameter | Bus terminals BT-4/DIO1 and BT-4/DIO1 TP,  Bus terminal BT-4/ENC1 |
| Traceable | Yes |

Functional Description

Specifies the bus address of the bus terminal.

| Value | Data type | Meaning |
| --- | --- | --- |
| 1–89 | DINT | Address of a physical (real) device. The address must also be set on the device, using both rotary switches. |
| 100–199 | DINT | Address of a virtual (non-real) device. |

NOTE: The address set at the BT-4 bus terminal via the two rotary switches on the front panel must match the address in the PLC configuration in parameter BusAddr of the BT-4 object. The address setting on the bus terminal BT-4 is applied only after applying the control voltage. Assign the last BT-4/ENC1 bus terminal on the PacNet to the address 8x (coding switch S1 Adr. 8).
