# DiagPF8_15Enable

![](images/logo.png)

DiagPF8\_15Enable

DiagPF8\_15Enable

General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | No |

Functional Description

Using the parameter DiagPF8\_15Enable, you can choose whether a diagnostic message is to be issued for a PowerFailO8\_O15.

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | BOOL | A diagnostic message is not triggered if there is no supply voltage connected to outputs 8 to 15. |
| H / 1 | BOOL | The diagnostic message [8787 Configuration error](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064064_1) is triggered if there is no supply voltage connected to outputs 8 to 15. |
