# General Description

## TM4 Expansion Modules

The following table shows the TM4 expansion module features:

| Module reference | Type | Terminal type |
| --- | --- | --- |
| [TM4ES4](D-SE-0036279.html#D-SE-0036279) | Ethernet communication | 4 RJ45 connectors  1 screw for functional ground connection |
| [TM4PDPS1](D-SE-0036280.html#D-SE-0036280) | PROFIBUS DP slave communication | 1 SUB-D 9 pins female connector  1 screw for functional ground connection |
| NOTE: The TM4ES4 module has two applications: expansion or standalone. For more information, refer to [TM4 Compatibility](D-SE-0036038.html#D-SE-0036038). | | |
