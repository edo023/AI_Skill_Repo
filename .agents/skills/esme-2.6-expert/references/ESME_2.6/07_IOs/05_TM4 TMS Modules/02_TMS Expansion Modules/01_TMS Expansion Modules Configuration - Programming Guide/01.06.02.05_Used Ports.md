# Used Ports

This table lists the port numbers used by the M262 Logic/Motion Controller services:

| Service | Port numbers | Out-of-box / Default configuration (1) |
| --- | --- | --- |
| Programming software | TCP 11740  UDP 1740 | Yes |
| FTP over TLS | TCP 20, 21 | Yes |
| HTTP (2) | TCP 80 | Yes |
| HTTPS | TCP 443 | Yes |
| Modbus TCP | TCP 502 | No |
| OPC UA | TCP 4840 | No |
| Discovery | UDP 27126, 27127 | Yes |
| Bonjour Discovery Protocol | UDP 5353 | Yes |
| Web Services Dynamical Discovery | TCP 5357  UDP 3702 | Yes |
| SNMP | UDP 161 | No |
| NVL | UDP default value: 1202 (destination port)  (Source port is dynamically assigned.) | No |
| EtherNet/IP | TCP 44818  UDP 2222, 44818 | No |
| WebVisualisation | TCP 8080 (HTTP), 8089 (HTTPS) | No |
| DHCP/BOOTP | UDP 68 | Yes |
| TFTP | UDP 69 | No |
| NTP | UDP 123 | No |
| Safe Logger (3) | UDP 35021, 45000 | Yes |
| Machine Assistant | UDP 45001, 45003, 45004 | Yes |
| **(1)** Default configuration is used when a default project is loaded into the controller. The port status is exactly the same as the factory settings.  **(2)** HTTP is automatically redirected to HTTPS.  **(3)** Safe Logger ports are only applicable to TM262M• references. | | |
