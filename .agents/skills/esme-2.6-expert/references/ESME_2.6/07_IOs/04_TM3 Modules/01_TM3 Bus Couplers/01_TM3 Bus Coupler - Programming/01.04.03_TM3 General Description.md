# TM3 General Description

## Introduction

The range of TM3 expansion modules includes:

* Digital modules, classified as follows:

  + [Input modules](#D-SE-0027112__D-SE-0027112.8)
  + [Output modules](#D-SE-0027112__D-SE-0027112.9)
  + [Mixed input/output modules](#D-SE-0027112__D-SE-0027112.10)
* Analog modules, classified as follows:

  + [Input modules](#D-SE-0027112__D-SE-0027112.18)
  + [Output modules](#D-SE-0027112__D-SE-0027112.19)
  + [Mixed input/output modules](#D-SE-0027112__D-SE-0027112.20)
* [Expert modules](#D-SE-0027112__D-SE-0027112.11)
* [Transmitter and receiver modules](#D-SE-0027112__D-SE-0027112.12)
* [Safety modules](#D-SE-0027112__D-SE-0027112.23)
