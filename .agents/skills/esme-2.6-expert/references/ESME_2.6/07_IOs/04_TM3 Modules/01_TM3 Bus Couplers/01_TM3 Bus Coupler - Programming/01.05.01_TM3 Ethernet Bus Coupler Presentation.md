# TM3 Ethernet Bus Coupler Presentation

## Introduction

The TM3 Ethernet bus coupler is a device designed to manage EtherNet/IP and Modbus TCP communication when using expansion modules with a controller in a distributed architecture. The TM3 Ethernet bus coupler supports the [TM3 expansion modules](D-SE-0027112.html#D-SE-0027112) and the [TM2 expansion modules](D-SE-0006573.html#D-SE-0006573).

## Ethernet Services

The TM3 Ethernet bus coupler supports the following services:

* IPV4
* IPV6
* [EtherNet/IP Adapter](D-SE-0084874.html#D-SE-0084874)
* [Modbus TCP/IP server](../../../../../api/crossBook?lang=en-US&virtualBookName=m262prg&topicID=D_SE_0002961)
* [Web server](D-SE-0089997.html#D-SE-0089997)
* [Ring topology](D-SE-0084876.html#D-SE-0084876)
* [SNMP](D-SE-0089997.html#D-SE-0089997__D-SE-0089997.38)
* DPWS

## Ethernet Protocols

The TM3 Ethernet bus coupler supports the following:

* IP (Internet Protocol)
* UDP (User Datagram Protocol)
* TCP (Transmission Control Protocol)
* ARP (Address Resolution Protocol)
* RSTP (Rapid Spanning Tree Protocol)
