# EtherNet/IP Objects Presentation

## EtherNet/IP Objects

The TM3 bus coupler supports the following objects:

| Object class | Class ID (hex) | Effect on Interface Behavior |
| --- | --- | --- |
| TM3 bus coupler configuration object | 65 | Stores TM2/TM3 module configurations. |
| TM3 bus coupler feature configuration object | 64 | Used to prepare/apply configuration. |
