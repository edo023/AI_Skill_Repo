# HSC\_EVENT\_TIMEBASE\_TYPE\_TM3: Type for Event Time Base Variable

## Enumerated Type Description

This enumeration data type ENUM is used to configure a Main type single phase in Event Counting mode.

The enumeration contains the different time base values allowed for use with the [HSCSetParam\_TM3](D-SE-0094566.html#D-SE-0094566) and [HSCGetParam\_TM3](D-SE-0094565.html#D-SE-0094565) function blocks:

| Name | Value |
| --- | --- |
| `HSC_EVENT_100ms` | 0 |
| `HSC_EVENT_1s` | 1 |
| `HSC_EVENT_10s` | 2 |
| `HSC_EVENT_60s` | 3 |
