# HSC\_PERIODMETER\_RESOLUTION\_TYPE\_TM3: Type for Period Meter Time Base Variable

## Enumerated Type Description

This enumeration data type ENUM is used in Period Meter mode.

The enumeration contains the different time base values allowed for use with an HSC function block:

| Name | Value |
| --- | --- |
| `HSC_PERIODMETER_100ns` | `FFFFFFFF hex` (-1 decimal)=100 ns |
| `HSC_PERIODMETER_1us` | `00 hex` (0 decimal)=1 μs |
| `HSC_PERIODMETER_100us` | `01 hex` (1 decimal)=100 μs |
| `HSC_PERIODMETER_1000us` | `02 hex` (2 decimal)=1000 μs |
