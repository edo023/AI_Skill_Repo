# External Event Configuration

## Overview

On the TM3XFHSC202 module, external events can be configured to activate external tasks.

When an external event is configured on a Counting function:

* An external event is created and its default name displayed in the Counters configuration window. The default name can be edited.
* The external event name is available when configuring an external task.

## Procedure

The following procedure describes how to configure an external event to activate a task:

| Step | Action |
| --- | --- |
| 1 | In the Applications tree tab, add a task. |
| 2 | Double-click the Task node to associate it with to an external event. |
| 3 | In the Type dropdown menu on the Configuration window, select External. |
| 4 | In the External event drop down menu, select the event to associate to the task (see the list below).  NOTE: The task must be configured when an event is declared. |

## External Events

This table provides a description of the possible external events to associate to a task:

| Event Name | Description |
| --- | --- |
| I***x***, ***x***=0...9 | Task is activated when the input Ix signal detects a rising edge, falling edge, or both edges. The type of signal detected is configured on the Configuration window. |
| *instancename*\_TH***x*** | Configurable in the task is activated when Threshold Value x is crossed. Task activation can be triggered when the High Speed Counter is configured as counting up, counting down, or both. |
| *instancename*\_TH***x***\_TH***y*** | Task is activated when counter value is between Threshold ***x*** and Threshold ***y***. |
| *instancename*\_Below\_TH***x*** | Task is activated when counter value is below Threshold ***x***. |
| *instancename*\_Above\_TH***y*** | Task is activated when counter value is above Threshold ***x***. |
| *instancename*\_STOP | Only available in [HSC Main Single Phase](D-SE-0033533.html#D-SE-0033533) mode.  Task is activated when counter value is set to 0. |
| *instancename*\_PERIOD | Only available in [Period Meter Edge to Edge](D-SE-0033562.html#D-SE-0033562) and [Period Meter Edge to Opposite](D-SE-0033563.html#D-SE-0033563) modes.  Task is activated at the end of a configured period. |
