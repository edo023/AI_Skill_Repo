# TM3XTRA1 Presentation

## Overview

TM3XTRA1 transmitter module:

* One RJ45 connector for communication
* Screw for functional ground connection

## Main Characteristics

| Characteristics | | Value |
| --- | --- | --- |
| Function | | Associated with the receiver forms the remote I/O expansion. |
| Connector type | | RJ45 |
| Communication type | | RS485 |
| Current draw on 5 Vdc internal bus | | 160 mA maximum |
| Current draw on 24 Vdc internal bus | | – |
| Cable type and length | | For more information, refer to [Accessories](Accessories-94ABD07B.html). |
| Weight | | 65 g (2.29 oz) |

## Status LEDs

The following figure shows the status LEDs:

![](images/T001663530.gif)

This table describes the status LEDs:

| LED | Color | Status | Description |
| --- | --- | --- | --- |
| Power (PWR) | Green | On | Power supply from TM3 bus is available. |
| Off | Power supply from TM3 bus is not available. |
| Link (LNK) | Green | On | Link/data signal is detected from the receiver. |
| Off | No link/data signal is detected from the receiver. |
