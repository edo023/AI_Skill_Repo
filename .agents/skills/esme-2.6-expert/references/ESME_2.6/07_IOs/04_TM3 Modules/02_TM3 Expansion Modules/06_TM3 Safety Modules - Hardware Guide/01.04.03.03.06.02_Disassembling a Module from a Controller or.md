# Disassembling a Module from a Controller or Receiver Module

Disassembling a Module from a Controller or Receiver Module

| Step | Action |
| --- | --- |
| 1 | Remove all power from the control system. |
| 2 | Dismount the assembled controller and modules from the mounting rail. |
| 3 | Push up the [locking device](../../../../../../api/crossBook?lang=en-US&virtualBookName=tm3diohw&topicID=D_SE_0025084_1) from the bottom of the module to disengage it from the controller or receiver module. |
| 4 | Pull apart module from the controller or receiver module. |
