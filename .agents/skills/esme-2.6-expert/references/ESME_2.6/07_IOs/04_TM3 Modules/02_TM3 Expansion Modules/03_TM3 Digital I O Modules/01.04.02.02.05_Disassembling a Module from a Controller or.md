## Disassembling a Module from a Controller or Receiver Module

The following procedure describes how to disassemble a module from a controller or receiver module:

| Step | Action |
| --- | --- |
| 1 | Remove all power from the control system. |
| 2 | Dismount the assembled controller and modules from the mounting rail. |
| 3 | Push up the [locking device](D-SE-0025084.html#D-SE-0025084) from the bottom of the module to disengage it from the controller or the receiver module. |
| 4 | Pull apart the module from the controller or the receiver module. |
