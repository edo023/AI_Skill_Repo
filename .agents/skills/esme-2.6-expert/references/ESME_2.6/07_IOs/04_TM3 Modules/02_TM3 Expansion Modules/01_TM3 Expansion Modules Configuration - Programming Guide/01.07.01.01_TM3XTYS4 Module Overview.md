# TM3XTYS4 Module Overview

## Introduction

The TeSys expansion module TM3XTYS4 is equipped with:

* 4 RJ-45 connectors to connect to Tesys motor starter devices
* 3 digital inputs for each channel:

  + Ready
  + Run
  + Trip
* 2 digital outputs for each channel:

  + Forward
  + Reverse
* Removable 24 Vdc power supply

The TeSys expansion module is connected to the controller through the TM3 bus. TM3XTYS4 expansion modules can be connected to the controller in any order.

## Adding and Configuring the TM3XTYS4 Module

To add a TM3XTYS4 module to a project:

1. [Add the expansion module](D-SE-0032519.html#D-SE-0032519__D-SE-0032519.7) to your controller.
2. [Configure the expansion module](D-SE-0032671.html#D-SE-0032671__D-SE-0032671.12) inputs and outputs.
3. [Insert a channel](D-SE-0032671.html#D-SE-0032671__D-SE-0032671.9) that is connected to the device.
4. [Select the device type](D-SE-0032671.html#D-SE-0032671__D-SE-0032671.10) that is associated with the channel.
5. [Add the FB\_TeSysU function block](D-SE-0043289.html#D-SE-0043289) to your application and configure it in order to control the device directly from the application.
