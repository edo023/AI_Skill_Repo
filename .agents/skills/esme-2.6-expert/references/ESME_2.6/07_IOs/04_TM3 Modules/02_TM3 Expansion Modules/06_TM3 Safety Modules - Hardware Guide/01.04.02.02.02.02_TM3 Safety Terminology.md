# TM3 Safety Terminology

TM3 Safety Terminology

This table lists the technical terms, acronyms, abbreviations, and their respective description:

| Term used | Description |
| --- | --- |
| EDM | External Device Monitoring |
| ESPE | Electro-Sensitive Protective Equipment |
| K | Refers to:  oK1: internal relay  oK2: internal relay  oK3: external contactor  oK4: external contactor |
| PL | Performance Level |
| SIL | Safety Integrity Level |
| S | Refers to safety inputs:  oSafety or start inputs:  oS11-S12  oS21-S22  oS31-S32  oS41-S42  oExternal switches S1, S2, S3, and so on.  NOTE: The function depends on the module type. |
| Start | Refers to:  oNon-monitored: may be manual or automatic mode. Start button must be closed for valid start condition.  oMonitored: start button must be pressed and then released for valid start condition. |
