# TM3XREC1 Presentation

## Overview

TM3XREC1 receiver module:

* One RJ45 connector for communication
* Removable 24 Vdc power supply terminal block

## Main Characteristics

| Characteristics | | Value |
| --- | --- | --- |

| Receiver module | Current draw on 24 Vdc power supply for the receiver module | 40 mA |

## Status LEDs

The following figure shows the status LEDs:

![](images/T001663529.gif)

This table describes the status LEDs:

| LED | Color | Status | Description |
| --- | --- | --- | --- |
| Power (PWR) | Green | On | Power supply for TM3 bus is available. |
| Off | Power supply for TM3 bus is not available. |
| Link (LNK) | Green | On | Link/data signal is detected from the transmitter. |
| Off | No link/data signal is detected from the transmitter. |
