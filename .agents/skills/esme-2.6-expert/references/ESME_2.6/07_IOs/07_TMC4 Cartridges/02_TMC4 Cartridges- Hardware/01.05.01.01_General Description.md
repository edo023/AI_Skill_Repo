# General Description

## Introduction

The cartridges are designed to be connected to the Modicon M241 Logic Controller range.

## Cartridges Features

The following table describes the TMC4 cartridges features:

| Reference | Description |
| --- | --- |
| [TMC4AI2](D-SE-0036454.html#D-SE-0036454) | TMC4 cartridge with 2 analog voltage or current inputs (0...10 V, 0...20 mA, 4...20 mA), 12 bits |
| [TMC4TI2](D-SE-0036482.html#D-SE-0036482) | TMC4 cartridge with 2 analog temperature inputs (thermocouple, RTD), 14 bits |
| [TMC4AQ2](D-SE-0036486.html#D-SE-0036486) | TMC4 cartridge with 2 analog voltage or current outputs (0...10 V, 4...20 mA), 16 bits |
| [TMC4HOIS01](D-SE-0036459.html#D-SE-0036459) | TMC4 application cartridge with 2 analog voltage or current inputs for hoisting load cells |
| [TMC4PACK01](D-SE-0036463.html#D-SE-0036463) | TMC4 application cartridge with 2 analog voltage or current inputs for packaging |
