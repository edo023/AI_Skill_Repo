# TM2DDI16DK

![](images/logo.png)

TM2DDI16DK

Introduction

This expansion module is a 16-point, 24 Vdc input module with a [HE10](../glossary/glossary.htm#XREF_D_SE_0024697_572) connector.

For further hardware information, refer to [TM2DDI16DK](../../../../../../api/crossBook?lang=en-US&virtualBookName=tm2diohw&topicID=D_RU_0004765_1).

I/O Mapping Tab

This table identifies the addresses of each input and the channel name.

| Channel | Type | Description |
| --- | --- | --- |
| IW0 | WORD | State of all inputs |
