# TM2ALM3LT

![](images/logo.png)

TM2ALM3LT

Introduction

This expansion module is a 2-point input/1-point output module with a terminal block and accepts thermocouple and resistance thermometer signals.

For further hardware information, refer to [TM2ALM3LT](../../../../../../api/crossBook?lang=en-US&virtualBookName=tm2aiohw&topicID=D_RU_0004720_1).

If you have physically wired the analog channel for a voltage signal and you configure the channel for a current signal in EcoStruxure Machine Expert, you may damage the analog circuit.

|  |
| --- |
| NOTICE |
| INOPERABLE EQUIPMENT |
| Verify that the physical wiring of the analog circuit is compatible with the software configuration for the analog channel. |
| Failure to follow these instructions can result in equipment damage. |
