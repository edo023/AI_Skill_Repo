# TM2DMM8DRT

![](images/logo.png)

TM2DMM8DRT

Introduction

This expansion module is a 4-point input/4-point output module with a terminal block.

For further hardware information, refer to [TM2DMM8DRT](../../../../../../api/crossBook?lang=en-US&virtualBookName=tm2diohw&topicID=D_RU_0004699_1).

I/O Mapping Tab

This table identifies the addresses of each input and output with the channel name.

![G-RU-0010613.5.gif-high.gif](images/G-RU-0010613.5.gif-high.gif)

| Channel | | Type | Default Value | Description |
| --- | --- | --- | --- | --- |
| Inputs | IB0 | BYTE | - | State of all inputs |
| I0 | BOOL | - | State of input 0 |
| ... | ... |
| I3 | State of input 3 |
| Outputs | QB0 | BYTE | - | Command byte of all outputs |
| Q0 | BOOL | -  TRUE  FALSE | Command bit of output 0 |
| ... | ... |
| Q3 | Command bit of output 3 |
