# TM2DDI32DK

![](images/logo.png)

TM2DDI32DK

Introduction

This expansion module is a 32-point, 24 Vdc input module with a HE10 connector.

For further hardware information, refer to [TM2DDI32DK](../../../../../../api/crossBook?lang=en-US&virtualBookName=tm2diohw&topicID=D_RU_0004770_1).

I/O Mapping Tab

This table identifies the addresses of each input and the channel name.

![G-RU-0010604.6.gif-high.gif](images/G-RU-0010604.6.gif-high.gif)

| Channel | Type | Description |
| --- | --- | --- |
| ID0 | WORD | State of all inputs |
| I0 | BOOL | State of input 0 |
| ... | ... |
| I31 | State of input 31 |
