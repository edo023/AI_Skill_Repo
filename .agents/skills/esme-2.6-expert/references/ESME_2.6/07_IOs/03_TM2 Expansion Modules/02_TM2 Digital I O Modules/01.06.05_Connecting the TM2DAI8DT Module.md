# Connecting the TM2DAI8DT Module

![](images/logo.png)

Connecting the TM2DAI8DT Module

Introduction

TM2DAI8DT is a 8-channel, 120 Vac input module.

This module is fitted with a removable connection screw terminal block for the connection of inputs and power supply.

Wiring Rules

See [Wiring Requirements](../Modules_General_Overview/Modules_General_Overview-12.htm#XREF_D_RU_0004606_1).
