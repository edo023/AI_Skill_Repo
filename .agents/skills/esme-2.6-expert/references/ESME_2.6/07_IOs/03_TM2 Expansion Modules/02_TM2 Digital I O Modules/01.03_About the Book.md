# About the Book

About the Book

Document Scope

This guide describes the hardware implementation of TM2 digital I/O expansion modules. It provides parts descriptions, characteristics, wiring diagrams, installation, and setup for TM2 digital I/O expansion modules.

Validity Note

The information in this manual is applicable only for TM2 products.

This document has been updated for the release of EcoStruxureTM Machine Expert V1.2.5.

This document has been updated for the release of EcoStruxureTM Machine Expert - Basic V1.1.
