# Fallback Configuration

Fallback Configuration

To configure the fallback behavior, double-click the expansion module node in the Devices tree. In the I/O Configuration tab, select the Mode:

oMaintain: The output remains in its current state.

oFallback Value: The output is set to the configured value, either 0 or 1. The default ForceValue is 0.

You must configure the fallback behavior for each output.
