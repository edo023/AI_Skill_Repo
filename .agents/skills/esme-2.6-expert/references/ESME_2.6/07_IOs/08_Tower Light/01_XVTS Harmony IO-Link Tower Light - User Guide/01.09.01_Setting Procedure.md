# Setting Procedure

Regardless of the IO-Link Master or software used, the standard procedure to configure a XVTS IO-Link tower light is:

| Step | Action | | |
| --- | --- | --- | --- |
| 1 | Select the operating mode by setting the Signal variable with one of these values:   * Tower Light mode: **0**, * Full Spectrum mode: **1**, * Levelling mode: **2**, * Programmer mode: **3**.   To understand operating modes, segments, and tiers, refer to [Operating Modes](OperatingModes-B981211C.html). | | |
| 2 | If you have selected Tower Light or Full Spectrum mode, enter one of the values below for the Number of tiers variable to define the number of tiers to use:   * XVTS1A•: **1…3**, * XVTS2A•: **1…5**.   NOTE: The IODD accepts values between 0 and 255. However, no values other than those listed above are valid. | | |
| 3 | Configure the segments or tiers (depending on the selected operating mode) by setting the Segment **N** variables .  **N** is a number between 1 and 15. It depends on the number of segments of your tower light, the operating mode you have selected and the way you configure it:   * For Tower Light or Full Spectrum mode, set variables according to the number of tiers defined. Examples:    + If you have define two tiers, set Segment 1 and Segment 2 variables,   + If you have define four tiers, set Segment 1 to Segment 4 variables. * For Levelling or Programmer mode, set variables according to the number of segments of the XVTS device:    + For XVTS1A•: set Segment 1 to Segment 9 variables,   + For XVTS2A•: set Segment 1 to Segment 15 variables.  NOTE: To configure segments individually in Levelling mode, set the Filling Level mode variable to **0** before.   To configure a segment or a tier, refer to [Optical](Optical-B981535F.html). | | |
| 4 | For XVTS•AS• only, configure the sirens by setting the Tone **N** (N = **1…10**) variables.  To configure a siren, refer to [Audible](Audible-B98155A1.html). | | |
| 5 | Configure additional parameters.  Refer to [Global](Global-B9814EE3.html) or the IODD. | | |
