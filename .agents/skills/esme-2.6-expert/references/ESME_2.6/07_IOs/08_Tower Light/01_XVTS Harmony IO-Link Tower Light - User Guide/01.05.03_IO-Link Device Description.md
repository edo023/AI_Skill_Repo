# IO-Link Device Description

To configure a XVTS IO-Link tower light, you need to download the IODD file.

This is a standardized file containing device identification (for example: manufacturer, type, and serial number), configurable parameters, process data (measurements and control signals), diagnostic data, and communication characteristics (data rate and format). It is specific to each device and enables automated device recognition, configuration, and troubleshooting.

This file is available at the following addresses:

* [Schneider Electric website](https://www.se.com/ww/en/download/),
* [IODDfinder Portal](https://ioddfinder.io-link.com).

After importing this file into your IO-Link Master, your tower light is automatically recognized.

The procedure to import IODD files depends on the IO-Link Master:

* For the NTSFIO0400 field device master, refer to [Associate Hardwares to the Project](AddBusCouplerNTSNEC1200-A60ED19E.html),
* For third-party IO-Link Masters, refer to the third-party IO-Link document.
