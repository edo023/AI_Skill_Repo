# Programmer

## Overview

|  |  |
| --- | --- |
|  | Segments have no predefined behavior. The tower light configuration is to be entirely programmed.  These extended configuration capabilities make it possible to meet more specific requirements that other modes cannot.  One bit controls one segment. Each segment can be individually configured, and each of them can be used or not depending on the defined configuration.  While the XVTS device is in operation, it is still possible to modify the configuration of the tower light. However, some softwares like EcoStruxureTM Machine Expert do not provide this feature, but control of the tower light is still possible using process data.  Tower Light, Full Spectrum and Levelling operating modes can be manually replicated in Programmer mode. |

## Process Data Table

The parameters from segments 1 to 9 (XVTS1A•) or 1 to 15 (XVTS2A•) are used for configuration.
