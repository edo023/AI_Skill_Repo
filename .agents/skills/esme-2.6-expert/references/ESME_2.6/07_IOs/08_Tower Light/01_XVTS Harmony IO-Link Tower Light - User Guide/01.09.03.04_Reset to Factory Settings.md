# Reset to Factory Settings

The Reset Factory Settings feature erases your current configuration and restore the default factory settings.

It can be enabled by entering the value 130 in System Command standard variable.

The default operating mode is [Full Spectrum mode](FullSpectrum-B9812EE1.html) and tiers are configured as follows:

* **XVTS1A•** (three tiers): Green/Yellow/Red
* **XVTS2A•** (five tiers): White/Blue/Green/Yellow/Red

Power reduction (current consumption) is [activated](Global-B9814EE3.html#Global-B9814EE3).

After the power reduction has been cancelled, the LEDs are set to continuous light.
