# Create a New Project

You have to create a new project using a M262 motion controller in EcoStruxureTM Machine Expert:

| Step | Action |
| --- | --- |
| 1 | At the top left-hand corner, click File > New Project…: |
| 2 | Click the drop-down arrow in the New Project window to select the type of **TM262** controller: |
| 3 | At the bottom page, name the project file and select a location path to save the new project. |
| 4 | Click the OK button |
