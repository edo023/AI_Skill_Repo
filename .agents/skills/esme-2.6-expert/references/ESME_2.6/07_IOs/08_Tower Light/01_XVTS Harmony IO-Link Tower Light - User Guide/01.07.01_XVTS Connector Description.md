# XVTS Connector Description

The connection is made via a four conductor connector with the following pin assignment:

| Pin | Conductor colour | IO-Link function | Connectors |
| --- | --- | --- | --- |
| 1 | Brown | L+ |  |
| 2 | White | +24 V (optional) |
| 3 | Blue | L- |
| 4 | Black | C/Q |
| 5 | Grey | Not connected |
