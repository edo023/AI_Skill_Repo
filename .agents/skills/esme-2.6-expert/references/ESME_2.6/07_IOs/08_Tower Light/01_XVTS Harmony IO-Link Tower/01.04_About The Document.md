# About The Document

## Document Scope

This documentation is a reference for the IO-Link version of XVTS tower lights.

The purpose of this document is to explain how to configure a XVTS tower light using the IO-Link communication protocol with the EcoStruxureTM Machine Expert or EcoStruxureTM Automation Expert - Motion software.

NOTE: Before installing, operating, or maintaining your IO-Link XVTS tower lights, read this document and all [related documents](#AboutTheDocument-AA9052BB__D-SE-0022273.9) to understand all its features.
