# Third-party IO-Link Masters

For third-party IO-Link Masters, the general procedure to follow is:

| Step | Action | | |
| --- | --- | --- | --- |
| 1 | Connect the tower lights to the dedicated ports on the IO-Link Master using the [appropriate connectors](Wiring-BCBC2B39.html). | | |
| 2 | With a software that manages IO-Link parameterization, set the IO-Link communication protocol of the IO-Link Master to version 1.1. | | |
| 3 | Load the [IODD files](IO-LinkDeviceDescription-C344B70F.html) corresponding to your tower lights into your IO-Link Master.  IODD files give the IO-Link Master the ability to automatically recognizes connected devices and loads their specific parameters. | | |
| 4 | Configure the tower lights using the IO-Link parameterization software.  For more details about the configuration required to get the tower lights up and running, refer to the [standard procedure to configure a XVTS IO-Link tower light](TowerLightsConfigurationProcedure-BF4A4694.html). | | |

Refer to the documentation supplied by the manufacturer of your IO-Link Master to know how to perform these actions.
