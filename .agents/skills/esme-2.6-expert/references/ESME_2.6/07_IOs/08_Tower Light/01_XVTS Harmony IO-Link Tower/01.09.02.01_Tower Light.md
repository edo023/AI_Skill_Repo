# Tower Light

## Overview

|  |  |
| --- | --- |
|  | Individual segments are combined into tiers.  Each tier has a fixed position and is ON if its optical signal is triggered. For this reason, it is possible to have non-illuminated tiers.   * **XVTS1A•**: three tiers maximum, * **XVTS2A•**: five tiers maximum.   Each tier has its own configuration.  The process data bits control the relevant tower light tiers. |
