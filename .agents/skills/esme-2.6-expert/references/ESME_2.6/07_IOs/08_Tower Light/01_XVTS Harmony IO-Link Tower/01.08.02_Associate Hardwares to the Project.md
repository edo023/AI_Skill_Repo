# Associate Hardwares to the Project

Then, you need to add to the newly created project the [required hardwares](SystemOverview-B9A6F82E.html) for the operation of XVTS IO-Link tower lights:

| Step | Action |
| --- | --- |
| 1 | Add NTSNEC1200 bus coupler |
| In the Device tree window located on the left-hand side, hover over Ethernet\_2 (ETH2), click + and double click EtherNet/IP Scanner (or ModbusTCP IO Scanner) under Protocol Managers: |
| Hover over EtherNet/IP Scanner (or ModbusTCP IO Scanner) to click + > EtherNet/IP targets (or ModbusTCP targets) > Bus Coupler and then, double click NTSNEC1200. |
| 2 | Add NTSFIO0400 field device master |
| Hover over NTSNEC1200 to click + > Edge I/O NTS Fieldbus Master and then, double click NTSFIO0400 > Close. |
| 3 | Add IO-Link devices |
| Under the NTSFIO0400 field device master, hover over the <Empty> (<Empty>) and click + to add the device [IODD files](IO-LinkDeviceDescription-C344B70F.html):  NOTE: Channels 0 and 1 support a maximum of two IO-Link devices, so only two tower lights can be added under the NTSFIO0400 field device master. List of available device references:  * XVTS1A\* * XVTS1AS\* * XVTS2A\* * XVTS2AS\* |
| Click the Close button. |
