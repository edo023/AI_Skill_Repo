# Full Spectrum

## Overview

|  |  |
| --- | --- |
|  | All segments are illuminated and automatically divided equally depending on the total number of segments available on the device:   * If only one tier is active, all segments are illuminated in a single colour, * If several tiers are active, the segments are illuminated in equal parts in different colours.   If the tiers cannot be divided into an equal number of segments, the remaining segments are allocated to the first configured tiers.   * **XVTS1A•**: three tiers maximum, * **XVTS2A•**: five tiers maximum.   Each tier has its own optical parameters configuration. The tower light activates the segments depending on the activated tiers. |

Segments are illuminated according to the number of tiers configured and activated.

The device illuminates all segments, applying the configurations of the activated tiers equally.
