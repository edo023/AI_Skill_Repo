# Levelling

## Overview

|  |  |
| --- | --- |
|  | Segments are used as a filling level indicator.  This mode enables signalization of job progress or material availability in machine processes by slowly illuminating the tower light from bottom to top or top to bottom.  All segments are used. Each of them can have specific parameters.  The optical parameters of the segments are managed as follows:   * By default, the Filling Level mode variable is set to 1 (Segment 1 variable parameters are used for all segments). * Set the Filling Level mode variable to 0 in order for each segment uses the parameters of its variable. |
