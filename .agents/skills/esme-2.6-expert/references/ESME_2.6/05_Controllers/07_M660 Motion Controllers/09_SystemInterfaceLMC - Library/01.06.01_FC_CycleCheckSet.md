# FC\_CycleCheckSet

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

Use the function FC\_CycleCheckSet of the library SystemInterfaceLMC to enable or disable triggering of the diagnostic message 8317 “Cycle time overrun”. This diagnostic message with diagnostic class 2 is triggered if the cycle time is exceeded by more than one tick. The default value for one tick is 250 µs.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xCheck | BOOL | FALSE: Diagnostic message 8317 is not triggered.  TRUE: Diagnostic message 8317 is triggered. |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK |
