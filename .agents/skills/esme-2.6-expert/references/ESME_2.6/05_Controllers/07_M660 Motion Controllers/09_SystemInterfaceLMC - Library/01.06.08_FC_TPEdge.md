# FC\_TPEdge

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

Use the function FC\_TPEdge of the library SystemInterfaceLMC to capture the position at an edge triggered by a touchprobe input.

For details, refer to SystemInterface library guide, [FC\_TPEdge](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085375).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifTPId | [DeviceTypeInterfaces.IF\_Device\_Type](IF_Device_Type-56CAF679.html#IF_Device_Type-56CAF679) | Touchprobe input |
| i\_ifDevId | [DeviceTypeInterfaces.IF\_Device\_Type](IF_Device_Type-56CAF679.html#IF_Device_Type-56CAF679) | Touchprobe device |
| i\_diParType | DINT | Position type |
| i\_xPosEdge | BOOL | Edge |
| i\_lrLowLimit | LREAL | Lower limit value of position window |
| i\_lrHighLimit | LREAL | Upper limit value of position window |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: i\_ifTPId invalid  -2: i\_ifDevId invalid  -3: i\_diParType invalid  -4: Position capture not possible because Sercos is not in communication phase CP4.  -5: Position capture not possible.  -6: Position capture not possible because the capture input is not activated. |
