# FC\_DiagMessageConfigSubclass

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

Use the function FC\_DiagMessageConfigSubclass of the library SystemInterfaceLMC to modify the diagnostic configuration of diagnostic messages.

The diagnostic subclass of the diagnostic message specified with i\_diDiagCode for the device specified with i\_ifDevice is set to the diagnostic subclass specified with i\_udiSubclass.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifDevice | [DeviceTypeInterfaces.IF\_Device\_Type](IF_Device_Type-56CAF679.html#IF_Device_Type-56CAF679) | Identifies the device. |
| i\_udiDiagCode | UDINT | Code of the diagnostic message whose diagnostic subclass is to be modified. |
| i\_udiSubclass | UDINT | New diagnostic subclass for the diagnostic message. |

## Return Value

| Data type | Description |
| --- | --- |
