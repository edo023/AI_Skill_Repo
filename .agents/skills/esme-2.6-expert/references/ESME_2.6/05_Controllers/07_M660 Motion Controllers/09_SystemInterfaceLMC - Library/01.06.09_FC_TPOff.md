# FC\_TPOff

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

Use the function FC\_TPOff of the library SystemInterfaceLMC to disable position capture with the specified touchprobe input.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifTPId | [DeviceTypeInterfaces.IF\_Device\_Type](IF_Device_Type-56CAF679.html#IF_Device_Type-56CAF679) | Touchprobe input |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: i\_ifTPId invalid |
