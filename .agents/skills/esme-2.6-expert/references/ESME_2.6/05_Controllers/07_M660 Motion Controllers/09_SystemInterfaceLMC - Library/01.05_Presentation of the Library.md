# Presentation of the Library SystemInterfaceLMC

## Description

The library SystemInterfaceLMC provides PacDrive compatibility functions for M660 controllers.

## Characteristics of the Library

The following table indicates the characteristics of the library:

| Characteristics | Value |
| --- | --- |
| Library title | SystemInterfaceLMC |
| Company | Schneider Electric |
| Category | Application |
| Component | CoreLibraries |
| Default namespace | SystemInterface |
| Language model attribute | [Qualified-access-only](../../../../../api/crossBook?lang=en-US&virtualBookName=Installer&topicID=D_SE_0081219) |
| Forward compatible library | Yes |

NOTE: For this library, qualified-access-only is set. You must access the POUs, data structures, enumerations, and constants using the namespace of the library. The default namespace of the library is SystemInterface.
