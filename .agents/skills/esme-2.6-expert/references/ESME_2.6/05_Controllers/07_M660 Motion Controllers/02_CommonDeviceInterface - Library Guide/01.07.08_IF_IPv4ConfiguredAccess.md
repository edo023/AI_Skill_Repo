# IF\_IPv4ConfiguredAccess

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The access interface IF\_IPv4ConfiguredAccess allows you to access the user function interface [IF\_IPv4Configured](IF_IPv4Configured-F3AFE531.html#IF_IPv4Configured-F3AFE531) to read and write the values of the parameters for the user function IPv4Configured of a communication device.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| IPv4Configured | [IF\_IPv4Configured](IF_IPv4Configured-F3AFE531.html#IF_IPv4Configured-F3AFE531) | Get | Accesses the user function interface IF\_IPv4Configured. |
