# IF\_IPv4ActiveAccess

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The access interface IF\_IPv4ActiveAccess allows you to access the user function interface [IF\_IPv4Active](IF_IPv4Active-F3AF00B6.html#IF_IPv4Active-F3AF00B6) to read the values of the parameters for the user function IPv4Active of a communication device.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| IPv4Active | [IF\_IPv4Active](IF_IPv4Active-F3AF00B6.html#IF_IPv4Active-F3AF00B6) | Get | Accesses the user function interface IF\_IPv4Active. |
