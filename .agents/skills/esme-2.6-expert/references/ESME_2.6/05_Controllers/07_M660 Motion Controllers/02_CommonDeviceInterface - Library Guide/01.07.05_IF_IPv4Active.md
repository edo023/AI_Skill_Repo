# IF\_IPv4Active

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_IPv4Active of the library CommonDeviceInterface allows you to access the user function IPv4Active to read the IP address and the subnet mask of communication devices.

For details on the user function, refer to [M660 Programming Guide, IPv4Active](../../../../../api/crossBook?lang=en-US&virtualBookName=M660_PG&topicID=IPv4Active_UserFunction_079C3669).

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| IpAddress | STRING(80) | Get | Reads the IP address of the communication device. |
| SubnetMask | STRING(80) | Get | Reads the subnet mask of the communication device. |
