# IF\_ElectronicLabel

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_ElectronicLabel of the library CommonDeviceInterface allows you to access the user function ElectronicLabel to read the values of the electronic nameplate of a device, for example, an M660 controller or a Sercos device.

For details on the user function, refer to [M660 Programming Guide, ElectronicLabel](../../../../../api/crossBook?lang=en-US&virtualBookName=M660_PG&topicID=M660_UserFunct_ElectronicLabel_00976147).

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| ControllerType | STRING(80) | Get | Reads the controller type designation of the device. |
| SoftwareRevision | STRING(80) | Get | Reads the firmware version of the device. |
| HardwareRevision | STRING(80) | Get | Reads the hardware version of the device. |
| FpgaRevision | STRING(80) | Get | Reads the FPGA version of the device. |
| FirmwareLoaderRevision | STRING(80) | Get | Reads the firmware loader version of the device. |
| OrderNumber | STRING(80) | Get | Reads the order number of the device. |
| SerialNumber | STRING(80) | Get | Reads the serial number of the device. |

NOTE: The values are transferred from the device to the master via the Sercos service channel. If the working mode of the device is virtual or if Sercos is not in communication phase CP2 or higher, the transferred string is an empty string.

NOTE: In the case of Sercos devices, the controller reads the values of the electronic label during booting.
