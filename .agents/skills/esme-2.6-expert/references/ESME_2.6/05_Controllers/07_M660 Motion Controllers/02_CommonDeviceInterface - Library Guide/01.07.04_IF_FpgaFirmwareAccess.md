The access interface IF\_FpgaFirmwareAccess allows you to access the user function interface [IF\_FpgaFirmware](IF_FpgaFirmware-F3AE522D.html#IF_FpgaFirmware-F3AE522D) to read the values of the parameters for the user function FpgaFirmware of a Sercos device.
