# IF\_ElectronicLabelAccess

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The access interface IF\_ElectronicLabelAccess allows you to access the user function interface [IF\_ElectronicLabel](IF_ElectronicLabel-78A8B800.html#IF_ElectronicLabel-78A8B800) to read the values of the parameters for the user function ElectronicLabel of a device.

| --- | --- | --- | --- |
| ElectronicLabel | [IF\_ElectronicLabel](IF_ElectronicLabel-78A8B800.html#IF_ElectronicLabel-78A8B800) | Get | Accesses the user function interface IF\_ElectronicLabel. |
