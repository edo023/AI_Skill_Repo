# IF\_IPv4Configured

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |

| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_IPv4Configured of the library CommonDeviceInterface allows you to read and write the IP address and the subnet mask of communication devices.

For details on the user function, refer to [M660 Programming Guide, IPv4Configured](../../../../../api/crossBook?lang=en-US&virtualBookName=M660_PG&topicID=IPv4Configured_UserFunction_079D4A17).

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| IpAddress | STRING(80) | Get, Set | Reads or writes the IP address of the communication device. |
| SubnetMask | STRING(80) | Get, Set | Reads or writes the subnet mask of the communication device. |
