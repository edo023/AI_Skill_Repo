# ET\_WorkingMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | DINT |
| Available as of: | V1.0.0.0 |

## Description

This enumeration lists the values of the parameter DesiredMode of the user function WorkingMode.

For details on the working mode of a Sercos device, refer to [M660 Programming User Guide, WorkingMode](../../../../../api/crossBook?lang=en-US&virtualBookName=M660_PG&topicID=ParameterGroupWorkingMode_44A54DB0).

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Virtual | 0 | In this working mode, the device operates as a virtual device. |
| Physical | 1 | In this working mode, the device operates as a physical device. |
| Deactivated | 2 | In this working mode, the device operates as a deactivated device. |
| Optional | 3 | In the working mode *Optional*, the Sercos master searches for the device on the bus when Sercos phases up.  * If it finds the device, the device operates as a physical device (corresponding to parameter value *Physical*) * If it does not find the device, the device operates as a virtual device (corresponding to parameter value *Virtual*) |
