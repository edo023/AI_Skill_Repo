# Presentation of the Library CommonFieldDeviceInterface

## Description

The library CommonFieldDeviceInterface provides common functions for Sercos devices and functions for Lexium 62 drives and power supplies.

## Characteristics of the Library

The following table indicates the characteristics of the library:

| Characteristics | Value |
| --- | --- |
| Library title | CommonFieldDeviceInterface |
| Company | Schneider Electric |
| Category | Application |
| Component | CoreLibraries |
| Default namespace | CFDI |
| Language model attribute | [Qualified-access-only](../../../../../api/crossBook?lang=en-US&virtualBookName=Installer&topicID=D_SE_0081219) |
| Forward compatible library | Yes |

NOTE: For this library, qualified-access-only is set. You must access the POUs, data structures, enumerations, and constants using the namespace of the library. The default namespace of the library is CFDI.
