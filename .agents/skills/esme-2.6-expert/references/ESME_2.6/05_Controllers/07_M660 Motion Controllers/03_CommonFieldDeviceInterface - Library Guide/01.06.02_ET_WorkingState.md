# ET\_WorkingState

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | DINT |
| Available as of: | V1.0.0.0 |

## Description

This enumeration lists the values of the parameter ActualState of the user function WorkingMode.

For details on the working mode of a Sercos device, refer to [M660 Programming User Guide, WorkingMode](../../../../../api/crossBook?lang=en-US&virtualBookName=M660_PG&topicID=ParameterGroupWorkingMode_44A54DB0).

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Virtual | 0 | In this state, the device operates as a virtual device. |
| Physical | 1 | In this state, the device operates as a physical device. |
| Deactivated | 2 | In this state, the device operates as a deactivated device. |
| Error | 3 | An error is detected. |
