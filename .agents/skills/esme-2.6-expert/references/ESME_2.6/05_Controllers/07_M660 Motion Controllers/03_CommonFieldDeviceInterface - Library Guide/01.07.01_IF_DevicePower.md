# IF\_DevicePower

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

Use the function interface IF\_DevicePower of the library CommonFieldDeviceInterface to read the values of the parameters for the user function DevicePower of Lexium 62 drives and power supplies.

For details on the user function, refer to [M660 Programming Guide, DevicePower](../../../../../api/crossBook?lang=en-US&virtualBookName=M660_PG&topicID=M660_UserFunct_DevicePower_005A516A).

## Properties

| --- | --- | --- | --- |

| DC\_BusVoltage | DINT | Get | Reads the DC bus voltage of the device.  For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [DC\_BusVoltage](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071456). |
