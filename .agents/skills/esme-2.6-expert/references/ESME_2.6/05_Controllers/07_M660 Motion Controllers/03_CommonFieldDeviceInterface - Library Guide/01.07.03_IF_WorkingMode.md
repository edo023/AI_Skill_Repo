## Description

Use the function interface IF\_WorkingMode of the library CommonFieldDeviceInterface to read or write values of the parameters for the user function WorkingMode of a Sercos device.

For details on the working mode of a Sercos device, refer to [M660 Programming Guide, WorkingMode](../../../../../api/crossBook?lang=en-US&virtualBookName=M660_PG&topicID=M660_UserFunct_GroupWorkingMode_44A54DB0).
