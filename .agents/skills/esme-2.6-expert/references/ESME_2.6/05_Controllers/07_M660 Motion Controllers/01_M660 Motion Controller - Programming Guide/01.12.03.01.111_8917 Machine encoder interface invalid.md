# 8917 Machine encoder interface invalid

Diagnostic class: 3

The encoder interface of the machine encoder type plate is not according to the machine encoder interface identified by the device.

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | AD |
| Power supply | - | - |

| Cause | Solution |
| --- | --- |
| The variable uiEncoderType in the data structure ST\_UserMotorData is set to an incorrect value. | Set the variable to the correct value. |
| The wrong encoder is connected to this device. | Verify and correct if necessary the wiring of the encoder. |
| Encoder is inoperable. | Replace the encoder. |
