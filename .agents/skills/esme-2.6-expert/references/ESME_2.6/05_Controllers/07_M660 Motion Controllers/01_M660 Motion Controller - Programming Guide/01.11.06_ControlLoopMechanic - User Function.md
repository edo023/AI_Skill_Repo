# ControlLoopMechanic - User Function

## Description

The user function ControlLoopMechanic is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function ControlLoopMechanic lets you read and write parameters relating to the mechanical factors affecting the control characteristics of a drive.

## Parameter J\_Gear

|  |  |
| --- | --- |
| Data type | REAL |
| Traceable | Yes |

The parameter J\_Gear lets you read and write the moment of inertia of the gearbox acting on the motor shaft.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [J\_Gear](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071844).
