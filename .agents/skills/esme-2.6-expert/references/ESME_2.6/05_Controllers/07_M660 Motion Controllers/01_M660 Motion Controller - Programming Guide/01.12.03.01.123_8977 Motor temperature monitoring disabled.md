# 8977 Motor temperature monitoring disabled

Diagnostic class: 2

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | D | E |
| Power supply | - | - |

The parameter MotorTemperatureMonitoring determines how the motor temperature is monitored. You can choose between the thermal model and the sensor. If the temperature cannot be monitored as configured, this diagnostic message is triggered as soon as you try to enable the controller (ControllerEnable = `true`).

The diagnostic message helps to prevent the drive from enabling the controller.

The motor temperature is not monitored before the drive has enabled the controller.

| Cause | Solution |
| --- | --- |
| The configuration of the requested monitoring is not correct. | Verify whether the parameter MotorTemperatureMonitoring has been set correctly and modify it if necessary. |
| The parameter MotorTemperatureMonitoring has been set to sensor and the motor temperature is too high. | Wait until the motor has cooled down. |
| The parameter MotorTemperatureMonitoring has been set to sensor and the motor temperature cannot be monitored. | * Verify whether the temperature sensor is connected and correct it if necessary. * Verify whether the wiring of the temperature sensor is inoperable and replace it if necessary. |
| The parameter MotorTemperatureMonitoring has been set to thermal model and the motor temperature cannot be monitored. | Contact your local Schneider Electric service representative. |
