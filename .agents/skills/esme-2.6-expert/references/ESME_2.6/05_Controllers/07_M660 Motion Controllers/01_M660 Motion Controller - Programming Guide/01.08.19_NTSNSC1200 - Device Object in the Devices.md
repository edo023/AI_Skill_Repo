# NTSNSC1200 - Device Object in the Devices Tree

## Overview

The device object NTSNSC1200 represents the Edge I/O network interface module (NIM). It is a child object of the device object [SercosMaster](SercosMaster-DeviceObjectInTheDevic-0684DFF3.html). The NIM serves as the interface between the Sercos bus and the Edge I/O bus.

In the devices tree, you can add individual Edge I/O modules below the NTSNSC1200 network interface module.

For details on Edge I/O, refer to the Modicon Edge I/O user guides.

## Edge I/O Network Interface Module Configuration with Edge I/O NTS Editor

The tab Edge I/O NTS Editor of the device editor is used for the initial configuration of an NTSNSC1200 network interface module. It provides Edge I/O-specific configuration settings and parameters. The Sercos parameters can be found on the Sercos Parameters tab.

You have to configure the OPCUA Security Parameters before you attempt to phase up Sercos. If the corresponding credentials are not supplied during phase up, the configuration is not downloaded to the network interface module.

| Item | Description |
| --- | --- |
| NIM IP Address | IPv4 address of the network interface module (RJ45) |
| NIM IP Subnet Mask | IPv4 subnet mask of the network interface module (RJ45) |
| NIM IP Gateway Address | IPv4 gateway address of the network interface module (RJ45) |
| I/O Profile | Lets you select the I/O profile. Available values   * Standard (default) * Optimized.   For details on the I/O profile, refer to the Modicon Edge I/O user guides. |
| I/O Bus Cycle Time | Lets you set the bus cycle time for the Edge I/O bus. Available values:   * 1 ms, default value * 2 ms * 4 ms |
| USB IP Address | IPv4 address of the network interface module for communication with the network interface module over USB |
| USB IP Subnet Mask | IPv4 subnet mask of the network interface module for communication with the network interface module over USB |
| OPCUA Security Parameters | |
| Authentication Mode | Lets you select the type of authentication for access to the network interface module. Available values:   * Username (default value): Configured username is required to connect * Anonymous: No username required to connect |
| Login | Lets you specify a login name. |
| Password | Lets you specify a password. |
| Security Mode | Lets you select the OPCUA security mode. Available values:   * None * Sign * SignEncrypt   For details on the OPCUA security mode, refer to the Modicon Edge I/O user guides. |
| Security Policy | Lets you select the OPCUA security policy. Available values:   * None: No encryption and no signature * Basic256Sha256: Encryption and signature possible * SignEncrypt   For details on the OPCUA security policy, refer to the Modicon Edge I/O user guides. |
