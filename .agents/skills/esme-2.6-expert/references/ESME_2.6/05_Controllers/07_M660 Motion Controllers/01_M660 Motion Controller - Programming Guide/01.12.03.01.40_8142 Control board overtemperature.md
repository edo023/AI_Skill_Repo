# 8142 Control board overtemperature

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | BD2 | CD |
| Power supply | BP1 | BP1 |

* The temperature of the power board inside the housing is too high.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| Insufficient ventilation or ambient temperature too high. | * Verify and correct if necessary the ventilation in the control cabinet. * If installed, ensure that the air conditioning unit is functioning properly. |
| Detected hardware error: The temperature sensor is inoperable. | Replace the device. |
