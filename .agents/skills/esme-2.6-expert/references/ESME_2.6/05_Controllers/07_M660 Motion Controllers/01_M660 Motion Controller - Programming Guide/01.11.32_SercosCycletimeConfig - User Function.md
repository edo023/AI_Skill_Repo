# SercosCycletimeConfig - User Function

## Description

The user function SercosCycletimeConfig is available for the following device obects:

* [Touchprobe](Touchprobe-DeviceObjectInTheDevices-0685B3BB.html), Sercos master

The user function SercosCycletimeConfig lets you set and read the Sercos cycle time.

## Parameter ConfiguredCycletime

|  |  |
| --- | --- |
| Data type | UDINT |
| Traceable | Yes |

The parameter ConfiguredCycletime lets you set the Sercos cycle time.

The cycle time can be set in communication state NRT and in communication phase CP0. The configured value becomes effective as of communication phase CP0.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| 1000000 (default value) | Sets the cycle time to 1 ms (1000000 ns). |
| 2000000 | Sets the cycle time to 2 ms (2000000 ns). |
| 4000000 | Sets the cycle time to 4 ms (4000000 ns). |

## Parameter Cycletime

|  |  |
| --- | --- |
| Data type | UDINT |
| Traceable | Yes |

The parameter Cycletime lets you read the Sercos cycle time.
