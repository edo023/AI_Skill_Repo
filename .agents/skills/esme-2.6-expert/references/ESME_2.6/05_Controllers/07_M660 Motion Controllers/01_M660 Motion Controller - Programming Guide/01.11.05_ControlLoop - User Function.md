# ControlLoop - User Function

## Description

The user function ControlLoop is available for the following device objecs:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function ControlLoop lets you read and write parameters relating to the control loop of drives.

## Parameter Curr\_I\_Gain

|  |  |
| --- | --- |
| Data type | REAL |
| Traceable | Yes |

The parameter Curr\_I\_Gain lets you read and write the integral gain of the current control loop.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [Curr\_I\_Gain](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071574).

|  |  |
| --- | --- |
| Data type | REAL |
| Traceable | Yes |
