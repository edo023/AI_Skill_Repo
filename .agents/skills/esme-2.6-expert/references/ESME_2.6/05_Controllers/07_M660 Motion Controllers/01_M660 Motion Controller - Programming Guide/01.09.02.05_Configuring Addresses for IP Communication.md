# Configuring Addresses for IP Communication

## Overview

The IP address is required for non-cyclic Ethernet communication over the Unified Communication Channel (non-real-time). For details, refer to [Sercos Communication Cycle](SercosCommunicationCycle-494E4399.html).

The Sercos master uses the following default values configured for **Ethernet\_3 (Ethernet3)**:

* IP address: **172.20.0.1**
* Subnet mask: **255.255.255.0**

The user function [IPv4Configured](IPv4Configured-UserFunction-079D4A17.html) lets you configure the IP address and the subnet mask.
