# ExpertFunctionInformation - User Function

## Overview

The user function ExpertFunctionInformation is available for the following device objects:

* [Touchprobe](Touchprobe-DeviceObjectInTheDevices-0685B3BB.html)

The user function ExpertFunctionInformation provides parameters for reading the name and type of an expert function of a Modicon Edge I/O Motion Expert module.

## Parameter FunctionType

|  |  |
| --- | --- |
| Data type | STRING(80) |
| Traceable | No |

The parameter FunctionType lets you read the type of the expert function.

## Parameter FunctionName

|  |  |
| --- | --- |
| Data type | STRING |
| Traceable | No |

The parameter FunctionName lets you read the name of the expert function.
