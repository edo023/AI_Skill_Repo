# DevicePower - User Function

## Description

The user function DevicePower is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives
* [PSM\_Lexium62 (Lexium62 Power Supply)](PSM_Lexium62Lexium62PowerSupply-Dev-0790BF9D.html), Lexium LXM62 power supplies

The user function DevicePower lets you read the control supply voltage and the DC bus voltage of a drive or power supply.

## Parameter ControlSupplyVoltage

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | No |

The parameter ControlSupplyVoltage lets you read the control supply voltage of the device.

The parameter corresponds to the Sercos parameter IDN P-0-0671.0.0.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [ControlSupplyVoltage](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071457).

## Parameter DC\_BusVoltage

|  |  |
| --- | --- |

The parameter DC\_BusVoltage lets you read the DC bus voltage of the device.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [DC\_BusVoltage](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071456).
