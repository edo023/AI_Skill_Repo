# 8910 Reference value invalid

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | BD1 | E |
| Power supply | - | - |

A position reference value was transferred over the Sercos real-time channel which cannot be reached within a Sercos cycle.

| Cause | Solution |
| --- | --- |
| The function FC\_SetRefPosFromPos() was called up for an axis that is under control and follows the reference values (AxisState `>=3`). By that the resulting jump in the reference position must be large enough so that a reference velocity is calculated, which is twice as large as the parameter MaxVel. | * Use FC\_SetRefPosFromPos() in the AxisState `< 3`. * Set FC\_SetRefPosFromPos() to a position that leads to a smaller jump, regarding the reference position. |
| An incorrect reference position was calculated in the controller. | Contact your local Schneider Electric service representative. |
