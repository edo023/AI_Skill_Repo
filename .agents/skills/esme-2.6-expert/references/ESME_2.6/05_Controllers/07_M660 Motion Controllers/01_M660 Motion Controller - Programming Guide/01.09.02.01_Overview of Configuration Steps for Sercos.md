# Overview of Configuration Steps for Sercos

The following sections describe the process of setting up a Sercos bus, adding devices, configuring the network communication, and configuring the individual devices.

Prerequisites:

* An M660 controller has been installed and wired and the initial commissioning has been performed. Refer to the hardware guides of the equipment to be operated on the Sercos bus for information on installation and wiring.
* You have created a project with an M660 controller.
* You have performed the initial configuration steps for the controller. For details, refer to [Device Editors](DeviceConfigurationTabs-4D7E8C49.html).
