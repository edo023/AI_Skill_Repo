# 8969 Motor cable not connected

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | AD |
| Power supply | - | - |

If ControllerEnable is set to TRUE, then a verification is performed to determine if a motor is connected to the device. This diagnostic message is triggered if the measured current deviates more than 30 % from the expected value. The test can be deactivated with the MotorConnectionTest parameter.

| Cause | Solution |
| --- | --- |
| The motor cable was not connected correctly. | Verify and correct if necessary the wiring. |
| The current control loop is not configured correctly. | Verify and correct if necessary the parameters Curr\_P\_Gain and Curr\_I\_Gain. |
