# 8139 DC bus precharge not possible

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | - | - |
| Power supply | BP1 | BP1 |

In the Lexium 62 Power Supply, the detected error message is triggered if the pre-charge of the DC bus could not be completed after a predefined time, that is internally calculated with data from the device type plate. If this error is triggered, then the device has to be reset or turned off /on again.

| Cause | Solution |
| --- | --- |
| An external load is connected to the DC bus. | Remove external load. |
| The parameter MainsVoltageMode is configured in an incorrect way. | Verify the parameter MainsVoltageMode and correct it, if necessary. |
| During the pre-charge, the mains phases were disconnected temporarily. | Verify if the cable for the mains connection is connected to the device correctly. |
