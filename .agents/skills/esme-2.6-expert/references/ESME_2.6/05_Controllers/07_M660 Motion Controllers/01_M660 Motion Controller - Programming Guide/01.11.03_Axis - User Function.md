# Axis - User Function

## Description

The user function Axis is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function Axis provides parameters for reading reference values and the operating mode of an axis.

## Parameter RefPosition

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | Yes |

The parameter RefPosition lets you read the reference position for an axis.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [RefPosition](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071490).

## Parameter RefVelocity

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | Yes |

The parameter RefVelocity lets you read the reference velocity for an axis. The reference velocity is derived from the reference position [RefPosition](#M660_UserFunct_FunctionAxis-00A94994__M660_Param_RefPosition-00A95CA8).

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [RefVelocity](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071512).

## Parameter RefAcceleration

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | - |

The parameter RefAcceleration lets you read the reference acceleration for an axis. The reference acceleration is derived from the reference velocity [RefVelocity](#M660_UserFunct_FunctionAxis-00A94994__M660_Param_RefVelocity-00A963C8).

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [RefAcceleration](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071518).

## Parameter OperationMode

|  |  |
| --- | --- |
| Data type | Enumeration of UINT |
| Traceable | Yes |

The parameter OperationMode lets you read the operating mode of an axis.

The parameter provides the following values:

| Value | Meaning |
| --- | --- |
| Position | Velocity control with active position control loop in drive (Cyclic Synchronous Position). |
| Velocity | Cyclic Synchronous Velocity, velocity control. |
| Torque | Cyclic Synchronous Torque, torque control. |
