# DriveCurrent - User Function

## Description

The user function DriveCurrent is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function DriveCurrent lets you read parameters relating to the current of drives, and set a current limitation.

## Parameter Current

|  |  |
| --- | --- |
| Data type | DINT |
| Traceable | Yes |

The parameter Current lets you read the filtered root mean square (RMS) value of the current in milliamperes (mA).

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [Current](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071519)..

## Parameter FeedbackCurrent

|  |  |
| --- | --- |
| Data type | DINT |
| Traceable | No |

The parameter FeedbackCurrent lets you read the root mean square (RMS) value of the proportion of the reference current at the output of the velocity control loop in milliamperes (mA), that is, the torque-generating current.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [FeedbackCurrent](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0092682)..

## Parameter CurrentLimitationsState

|  |  |
| --- | --- |
| Data type | Enumeration of DINT |
| Traceable | Yes |

The parameter CurrentLimitationsState lets you read the state of the limitation of the root mean square (RMS) value of the current.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [CurrentLimitationsState](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0089975)..

## Parameter CurrentLimitationThreshold

|  |  |
| --- | --- |
| Data type | DINT |
| Traceable | No |

The parameter CurrentLimitationThreshold lets you read the threshold of the limitation of the root mean square (RMS) value of the current.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [CurrentLimitationThreshold](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0089976)..

## Parameter UserCurrentLimit

|  |  |
| --- | --- |
| Data type | REAL(1...100) |
| Traceable | Yes |

The parameter UserCurrentLimit lets you set the limitation of the reference current for movements. It is specified as a percentage of MaxDrivePeakCurrent.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [UserCurrentLimit](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071530)..
