# IPv4Configured - User Function

## Description

The user function IPv4Configured is available for the following device objects:

* [Ethernet\_2 (Ethernet2)](Ethernet_2Ethernet2-DeviceObjectsIn-0B82E998.html), communication device
* [Ethernet\_3 (Ethernet3)](Ethernet_3Ethernet3-DeviceObjectInT-067E8BF3.html), communication device

The user function IPv4Configured provides parameters for reading and writing the IP address and the subnet mask of the communication device.

## Parameter IpAddress

|  |  |
| --- | --- |
| Data type | STRING(80) |
| Traceable | No |

The parameter IpAddress lets you read and write the IP address of the communication device.

The default IP address is 172.20.0.1.

## Parameter SubnetMask

|  |  |
| --- | --- |
| Data type | STRING(80) |
| Traceable | No |

The parameter SubnetMask lets you read and write the subnet mask of the communication device.

The default subnet mask is 255.255.255.0.
