# Adding Sercos Devices to the Devices Tree

## Adding a Sercos Master

When you create a new project based on a Modicon M660 controller, EcoStruxure Machine Expert automatically creates a Sercos master.

To locate the Sercos master in the Devices tree, expand the node CommunicationDevice\_CN8\_CN9 > Ethernet 3 > SercosMaster.

## Adding Subordinate Sercos Devices

| Step | Action |
| --- | --- |
| 1 | In the Devices tree, click the white/green plus sign to the right of the node SercosMaster.  You can also use Add device ... from the context menu of the node. |
| 2 | In the dialog box Add Device, select the required subordinate Sercos device to be added below the node SercosMaster from Sercos slaves or Fieldbuses. |
| 3 | Repeat this procedure until you have added the physically connected subordinate devices and the devices to be used as [virtual devices](ConfiguringDevicesAsSimulatedOrActi-4955415D.html). |
