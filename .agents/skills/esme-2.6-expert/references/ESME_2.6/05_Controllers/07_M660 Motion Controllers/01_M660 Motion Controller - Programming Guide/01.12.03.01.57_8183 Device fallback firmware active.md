# 8183 Device fallback firmware active

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | AD |
| Power supply | AP | AP |

The device was started with the fallback firmware.

| Cause | Solution |
| --- | --- |
| The bootloader detects a memory error and loads upon the fallback firmware. | Transfer firmware on the device again and verify if the expected firmware is loaded.  If the error is still detected, replace the device |
