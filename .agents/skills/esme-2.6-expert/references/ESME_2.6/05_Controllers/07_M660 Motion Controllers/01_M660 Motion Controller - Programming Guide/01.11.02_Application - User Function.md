# Application - User Function

## Description

The user function Application is available for the following device objects:

* [Modicon\_M660 (Modicon M660)](ModiconM660ModiconM660-DeviceObject-067D8F23.html), Modicon M660 controller

The user function Application provides a parameter that lets you enable the automatic execution (RUN) of the application after the controller has started.

## Parameter AutoRun

|  |  |
| --- | --- |
| Data type | Enumeration of BOOL |
| Traceable | Yes |

The parameter AutoRun lets you enable the automatic execution of the application. If the parameter value is set to Yes / 1 , the application automatically runs after the controller has started. The controller has to be restarted for modifications to this parameter to become effective.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| No / 0 | Application does not run automatically. |
| Yes / 1  (default value) | Application runs automatically. |
