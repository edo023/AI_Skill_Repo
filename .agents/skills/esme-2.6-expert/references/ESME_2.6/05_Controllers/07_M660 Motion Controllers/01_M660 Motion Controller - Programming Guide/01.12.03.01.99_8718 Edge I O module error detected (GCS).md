# 8718 Edge I/O module error detected (GCS)

Diagnostic class: 3

An error has been detected in conjunction with the Edge I/O Group Cycle Status (GCS) mechanism.

Ext. diagnosis: mod#NNN:0xGG.

The originating device is an Edge I/O network interface module. The values for mod#NNN:0xGG indicate the type of error and the affected Edge I/O module. For details, refer to [Diagnostics-Specific Information on Edge I/O Modules](Diagnostics-SpecificInformationOnEd-0550DDAA.html).
