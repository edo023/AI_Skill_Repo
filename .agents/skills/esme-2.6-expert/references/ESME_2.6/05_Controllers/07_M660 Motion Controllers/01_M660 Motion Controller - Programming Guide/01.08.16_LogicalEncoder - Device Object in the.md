# LogicalEncoder - Device Object in the Devices Tree

## Overview

A logical encoder is a software device that allows you to create logical encoder functionality. A logical encoder mirrors the movements of an axis to which it is logically connected in the application. Drives, physical or logical encoders can be used as connected axes.

For details on logical encoders, refer to the [MotionInterface Library Guide, function block FB\_LogicalEncoder](../../../../../api/crossBook?lang=en-US&virtualBookName=MOIN&topicID=FB_LogicalEncoder_3FC5FB2B)

The device object LogicalEncoder represents a logical encoder. It is a child object of the device object [LogicalEncoderPool](LogicalEncoderPool-DeviceObjectInTh-06851F40.html) which serves as a container for the logical encoders.
