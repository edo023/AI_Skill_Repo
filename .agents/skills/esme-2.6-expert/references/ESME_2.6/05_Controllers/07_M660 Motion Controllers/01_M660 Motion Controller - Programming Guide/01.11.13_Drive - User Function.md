# Drive - User Function

## Description

The user function Drive is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function Drive provides parameters for reading the position and the velocity of an axis as well as the operating mode of a drive.

## Parameter ActualOperationMode

|  |  |
| --- | --- |
| Data type | Enumeration of DINT |
| Traceable | Yes |

The parameter ActualOperationMode lets you read the operating mode of a drive.

The parameter provides the following values:

| Value | Meaning |
| --- | --- |
| Position | Velocity control with active position control loop in drive (Cyclic Synchronous Position). |
| Velocity | Cyclic Synchronous Velocity, velocity control. |
| Torque | Cyclic Synchronous Torque, torque control. |
| Not available | Active operating mode not available. |

## Parameter ActualPosition

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | Yes |

The parameter ActualPosition lets you read the position of an axis.

## Parameter ActualVelocity

|  |  |
