# IoStatusControl - User Function

## Description

The user function IoStatusControl is available for the following device objects:

* [NTSNSC1200](NTSNSC1200-DeviceObjectInTheDevices-068506CC.html), Edge I/O network interface module

The user function IoStatusControl provides parameters that let you read the activation state of the outputs, the validity of the data for the inputs, and activate the outputs.

## Parameter ConfiguredOutputsActive

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | Yes |

The parameter ConfiguredOutputsActive lets you activate the outputs. This is a prerequisite for setting outputs by the application.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| False (default value) | The outputs are not activated and cannot be set by the application. |
| True | The outputs are activated and can be set by the application. |

## Parameter InputsValid

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | Yes |

The parameter InputsValid lets you read the state of the input data (valid or invalid).

The parameter provides the following values:

| Value | Description |
| --- | --- |
| False | The input data is invalid. |
| True | The input data is valid. |

## Parameter OutputsActive

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | Yes |

The parameter OutputsActive lets you read the activation state of the of the outputs.

Use the parameter [ConfiguredOutputsActive](#TPC_M660_UserFunct_IoStatusControl-08E141F9__M660_Param_ConfiguredOutputsActive-08E0B7D3) to activate the outputs.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| False (default value) | The outputs are not activated. |
| True | The outputs are activated. |
