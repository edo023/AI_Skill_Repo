# 8507 SERCOS write cycle overflow

Diagnostic class: 3

The transfer of the target position to the Sercos bus is too late. The last target positions may have been transferred twice. This results in a velocity jump in the drives.

| Cause | Solution |
| --- | --- |
| The real-time process is overloaded. | Reduce the number of devices with real-time tasks (for example, virtual axes, virtual encoders, logical encoders, etc.). |
