# 8123 Safe Torque Off incorrect

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | E |
| Power supply | - | - |

* The Inverter Enable input is set to low, while the drive is enabled (`AxisState` >1) and parameter InverterEnableFunction is set to `Safe Stop 1 / 1`.

  NOTE: If the drive is shut down with `ControllerEnable`, then the controller is active until the drive stops and the brake is engaged (BrakeCouplingTime). The maximum time for this stop is defined with the parameter StopTimeLim.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| Detected wiring error. | Verify and correct if necessary the wiring of the Inverter Enable input. |
| Detected circuit error: The Inverter Enable input is set to low. | Verify and correct if necessary the control of the Inverter Enable input. |
| The safety-related function `Safe Torque Off` is used. | Verify and correct if necessary the parameter InverterEnableFunction. |
