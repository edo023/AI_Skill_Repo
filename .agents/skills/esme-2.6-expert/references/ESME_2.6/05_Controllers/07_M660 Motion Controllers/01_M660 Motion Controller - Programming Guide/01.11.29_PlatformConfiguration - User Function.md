# PlatformConfiguration - User Function

## Description

The user function PlatformConfiguration is available for the following device objects:

* [Modicon\_M660 (Modicon M660)](ModiconM660ModiconM660-DeviceObject-067D8F23.html), Modicon M660 controller

The user function PlatformConfiguration provides parameters that allow you to temporarily adjust the cycle time by reducing the system tick rate. The user function is only intended for testing purposes during commissioning.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  * Only use the function for testing purposes during commissioning. * Verify that no hazards can result from using the function.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

The default system tick interval is 250 μs. The parameter ShortestExecutionCycleConfigured allows you to select a value as low as 50 μs for faster I/O and system responses.

Decreasing the system tick interval potentially increases the CPU scheduling overhead, which can impact the overall system performance. Do not exceed a CPU load of 90 %.

To activate the user function, set the parameter ShortestExecutionCycleConfigured to the required value when the application is not connected to the controller (that is, when you are not logged into the controller). The selected value becomes active when the application starts.

You can modify the value while the application is running by selecting a different value in the column Prepared Value. The new value is indicated by the parameter ShortestExecutionCycleActive.

On the Configuration tab of the task, you can change the unit of the Interval from ms to μs.

## ShortestExecutionCycleActive

|  |  |
| --- | --- |
| Data type | Enumeration of DINT |
| Traceable | Yes |

The parameter ShortestExecutionCycleActive allows you to read the active system tick interval.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| 50 | The active cycle time is 50 µs. |
| 100 | The active cycle time is 100 µs. |
| 250 (default value) | The active cycle time is 250 µs. |
| 500 | The active cycle time is 500 µs. |
| 1000 | The active cycle time is 1000 µs. |

| Value | Description |

| Value | Description |
