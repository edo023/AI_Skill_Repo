The controller provides four onboard digital inputs through the connector **CN2**. The device object DigitalInputGroup\_CN2 represents connector **CN2** of the controller. It is a container for the four digital onboard inputs [DI1 to DI4](DI1ToDI4-DeviceObjectInTheDevicesTr-067E2217.html). Each of these four digital inputs is a separate device object below the node DigitalInputGroup\_CN2.

Assignment of the device objects to the pins of connector **CN2**:

* DI1 represents pin **P6**
* DI2 represents pin **P7**
* DI3 represents pin **P8**
* DI4 represents pin **P9**.
