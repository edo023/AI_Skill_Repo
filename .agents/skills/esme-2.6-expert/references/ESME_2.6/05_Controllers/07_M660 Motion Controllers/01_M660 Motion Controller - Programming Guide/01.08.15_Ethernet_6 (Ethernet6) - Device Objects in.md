# Ethernet\_6 (Ethernet6) - Device Objects in the Devices Tree

## Overview

The device object Ethernet\_6 (Ethernet6) is a communication device.

Path in the Devices tree hierarchy:

[Modicon\_M660 (Modicon M660)](ModiconM660ModiconM660-DeviceObject-067D8F23.html) - [OptionCard\_CN20\_CN23 (OptionCardCommDouble)](OptionCard_CN_CN-DeviceObjectInTheD-067DD19F.html#OptionCard_CN_CN-DeviceObjectInTheD-067DD19F) - [CommunicationDevice\_CN22\_CN23](CommunicationDevice_CN22_CN23CommDe-4C5EBC80.html) - Ethernet\_6 (Ethernet6)

Ethernet\_6 (Ethernet6) represents connectors **CN22** and **CN23** of an option card with two dual-port connectors (**CN20**, **CN21**, **CN22** and **CN23**).

The type of communication technology available through Ethernet\_6 (Ethernet6) is determined by its parent device. For details, refer to [FpgaFirmware](FpgaFirmware-UserFunction-07D459EA.html#FpgaFirmware-UserFunction-07D459EA) and [InterfacePoint](InterfacePoint-UserFunction-068F78C0.html).

The device object Ethernet\_6 (Ethernet6) allows you to set the values for the communication parameters of its child object such as IP address, subnet mask and gateway address.

You can add the device object PROFINETIO\_Controller (PROFINETIO\_Controller) as a child object of the device object Ethernet\_6 (Ethernet6). For configuration details, refer to [Configuration of the Network Adapter](AddingAProfinetIOController-4CF960D6.html#AddingAProfinetIOController-4CF960D6__ConfigurationOfTheEthernetEndpoint-4DFDAB3E).

## User Functions

The device object Ethernet\_5 (Ethernet5) does not provide user functions.

## Device Editor Tab Ethernet Device Parameters

The tab Ethernet Device Parameters of the device editor provides parameters for configuring the Ethernet communication of the child device of the device object. The configuration depends on the requirements of the child object.

## Device Editor Tab General

The tab General of the device editor provides parameters for configuring the IP address, the subnet mask and the default gateway.

The Browse... button allows you to select the physical port for the Network Interface (Ethernet endpoint).
