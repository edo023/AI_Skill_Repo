The user function DigitalInputs0\_3 is available for the following device objects:

* [DigitalInputGroup\_CN2](DigitalInputGroup_CN2-DeviceObjectI-067DF051.html), container for digital inputs DI1 to DI4

The user function DigitalInputs0\_3 provides a parameter that lets you read the values of the digital inputs DI1 to DI4.

## Parameter Values

|  |  |
| --- | --- |
| Data type | BYTE |
| Traceable | Yes |

The parameter Values lets you read the values of the digital inputs DI1 to DI4 which are child objects of the device object DigitalInputGroup\_CN2. The values of the four inputs are represented as a bit mask. The lowest significant bit represents the value of input DI1. The default value is 00000000 bin.

Assignment of the device objects [DI1 ... DI4](DI1ToDI4-DeviceObjectInTheDevicesTr-067E2217.html) to the pins of connector CN2:

* DI1 represents pin P6
* DI2 represents pin P7
* DI3 represents pin P8
* DI4 represents pin P9.
