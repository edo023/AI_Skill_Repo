# 8120 Power stage overload

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | BD2 | CD |
| Power supply | - | - |

The power stage is overloaded. The overload level has been increased up to 100 %. Probably the diagnostic message 8129 Power stage load high has been triggered before.

In this case, unlike diagnostic message '[8107 Overcurrent](TPC_M660_Diagnostics_8107Overcurren-03B0EDE7.html)', switch off does not occur as soon as the current threshold has been exceeded. The thermal overload of the device is observed for a specific length of time. The current overload of the drive is shown in the parameter DriveOverload.

| Cause | Solution |
| --- | --- |
| The power stage is overloaded. | * Verify and correct if necessary the combination motor/drive. * Verify and correct if necessary the calculation of motor and drive. * Drive the system at a slower speed. |
| The motor brake is not open. | Verify and correct if necessary the wiring of the motor brake (voltage reversal, cable break, etc.). |
| Detected hardware error: Servo amplifier inoperable. | Replace the servo amplifier. |
| Valid only for Lexium 52: The device is operated without a power choke. | Connect power choke and set the parameter MainsChokeConnected to `on/1`. |
