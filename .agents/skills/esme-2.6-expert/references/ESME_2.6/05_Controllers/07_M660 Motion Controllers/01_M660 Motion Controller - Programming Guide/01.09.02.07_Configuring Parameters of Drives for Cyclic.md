# Configuring Parameters of Drives for Cyclic Data Exchange

## Overview

To be able to set up the structure and contents of the MDTs and the ATs for cyclic communication, Sercos needs to know which of the drive-specific parameters provided by a drive are required for your application. The firmware adds a number of parameters by default. The remaining parameters need to be added by you. For details on parameters, refer to [Introduction to Sercos](IntroductionToSercos-494DFAE5.html). For general information on Sercos parameters, refer to [Sercos Parameters (IDN)](SercosParametersIDN-494E6D96.html)

## Default Parameters

The following parameters are added by default when you add a drive as a subordinate device to the Sercos master.

* S-0-0134 Drive control

  This parameter sets the operating mode of a drive and triggers functions such as enabling and disabling the power stage as well as triggering Halt and Stop. Used in MDT.
* S-0-0135 Drive status

  This parameter provides the information on the status of a drive regarding the settings controlled with the parameter S-0-0134 Drive control. Used in AT.
* S-0-0047 Position command value

  This parameter sets the reference or target position for a drive. Unit, scaling, conversion factor, and positions after decimal point are configured with additional parameters. Used in MDT.
* S-0-0051 Position feedback value

  This parameter contains the position provided from a drive. Used in AT.
* P-0-0416  [RefAcceleration](M660_UserFunct_FunctionAxis-00A94994.html#M660_UserFunct_FunctionAxis-00A94994__M660_Param_RefAcceleration-00A96D9A)

  This parameter contains the reference acceleration. Used in MDT.
* P-0-0409 ElauDriveControlWord

  This parameter sets the operating state of the drive. Used in MDT.
* P-0-0208 [UserCurrentLimit](M660_UserFunct_DriveCurrent-0057BEAB.html#M660_UserFunct_DriveCurrent-0057BEAB__M660_Param_UserCurrentLimit-00583091)

  This parameter sets the limitation of the reference current for movements. Used in MDT.
* S-0-0051 [ActualPosition](M660_UserFunct_Drive-00A14A2D.html#M660_UserFunct_Drive-00A14A2D__M660_Param_ActualPosition-00A15A7E)

  This parameter contains the value of the position. Used in AT.
* P-0-0603 ActualCurrent

  This parameter contains the value of the current.
* P-0-0409 ElauDriveControlWord

  This parameter contains control information. Used in MDT.
* P-0-00410 ElauDriveStatusWord

  This parameter contains status information. Used in AT.

## Adding Additional Parameters to the User-Specific Mapping List

Two function are available to add additional parameters to the user-specific mapping list:

* sm6xx.FC\_ConfigureCyclicDataAddIDN(...): This function can be used in Sercos communication phase CP2.
* sm6xx.FC\_GetConnectionDataAddress(...): This function can be used in Sercos communication phase CP4.
