# Drive2 - User Function

## Description

The user function Drive2 is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function Drive2 provides parameters relating to drive-specific settings.

## Parameter TrackingDeviation

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | Yes |

The parameter TrackingDeviation lets you read the position deviation (also referred to as “following error” in the Sercos specifications) between the physical position and the reference position.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [TrackingDeviation](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071507)..

## Parameter ShaftMechPosition

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | Yes |

The parameter ShaftMechPosition lets you read the physical position of the driving shaft at the gearbox output shaft.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [ShaftMechPosition](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071503)..
