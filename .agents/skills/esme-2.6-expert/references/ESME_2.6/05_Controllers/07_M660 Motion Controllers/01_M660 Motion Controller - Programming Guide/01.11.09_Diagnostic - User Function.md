# Diagnostic - User Function

## Description

The user function Diagnostic is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives
* [DRV\_SercDrv (SERCOS Drive](DRV_SercDrvSERCOSDrive-DeviceObject-49683323.html)

  , generic Sercos drives
* [PSM\_Lexium62 (Lexium62 Power Supply)](PSM_Lexium62Lexium62PowerSupply-Dev-0790BF9D.html), Lexium LXM62 power supplies
* [Modicon\_M660 (Modicon M660)](ModiconM660ModiconM660-DeviceObject-067D8F23.html), controller
* [NTSNSC1200](NTSNSC1200-DeviceObjectInTheDevices-068506CC.html), Edge I/O network interface module
* [SercosMaster](SercosMaster-DeviceObjectInTheDevic-0684DFF3.html), Sercos master

The user function Diagnostic provides diagnostics information on a device and its subordinate devices.

The parameters are read-only parameters.

The diagnostics information is displayed by the message logger.

For details on diagnostics, refer to [Diagnostics](Diagnostics-4EA0C1A4.html).

## Parameter DiagClass

| --- | --- |
