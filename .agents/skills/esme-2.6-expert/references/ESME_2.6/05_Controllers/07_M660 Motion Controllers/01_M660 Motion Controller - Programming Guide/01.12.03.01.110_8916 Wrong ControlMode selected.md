# 8916 Wrong ControlMode selected

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | AD |
| Power supply | - | - |

The control mode set using the parameter ControlMode is not supported by the drive, or by the firmware of the drive, or by the connected motor.

| Cause | Solution |
| --- | --- |
| The parameter ControlMode is set to “Encoderless velocity control / 2”. This setting requires a Lexium 62 Standard Plus or Lexium 62 Advanced Plus. | Select a Lexium 62 Standard Plus or Lexium 62 Advanced Plus or set the parameter ControlMode to a different value. |
| The parameter ControlMode is set to “Closed-loop control / 0”. This setting is not permissible with a motor which does not have an encoder. | Use a motor with an encoder or set the parameter ControlMode to a different value. |
| The parameter ControlMode is set to “Open-loop control / 1”. This setting is not permissible with synchronous or linear motors. | Use an asynchronous motor or set the parameter ControlMode to a different value. |
| The parameter ControlMode is set to “Encoderless velocity control / 2”. This setting is not with asynchronous or linear motor. | Use a synchronous motor or set the parameter ControlMode to a different value. |
| A machine encoder is connected to the drive and the parameter EncoderMode is set to "Machine encoder is only used for position control / 1". This parameter setting is only permissible if the parameter ControlMode is set to “closed-loop control / 0”. | Set the parameter ControlMode to “closed-loop control / 0” or set the parameter EncoderMode to “Only use motor encoder / 0”. |
