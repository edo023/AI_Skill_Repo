The user function DigitalInputFilter is available for the following device objects:

* [DI1 ... DI4](DI1ToDI4-DeviceObjectInTheDevicesTr-067E2217.html), digital inputs DI1 ... DI4

The user function DigitalInputFilter provides a parameter that lets you set the filter time for one of the inputs DI1 to DI4.

## Parameter Filter

|  |  |
| --- | --- |
| Data type | Enumeration of BYTE |
| Traceable | Yes |

The parameter Filter lets you set a filter time for one of the inputs DI1 to DI4 which are child objects of the device object DigitalInputGroup\_CN2.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| No filter | No filter is used for the input. |
| 1ms | The filter time for the input is 1 ms. |
| 5ms | The filter time for the input is 5 ms. |
