# Touchprobe - Device Object in the Devices Tree

## Overview

The device object Touchprobe represents the touchprobe functionality provided by the Modicon Edge I/O Motion Expert module [NTSEDT0800](NTSNSC1200-DeviceObjectInTheDevices-06851662.html). Before you can add this device object, you need to add the Modicon Edge I/O Motion Expert module to the network interface module.
