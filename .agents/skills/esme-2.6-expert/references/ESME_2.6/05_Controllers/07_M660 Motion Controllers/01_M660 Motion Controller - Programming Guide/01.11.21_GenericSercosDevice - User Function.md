# GenericSercosDevice - User Function

## Description

The user function GenericSercosDevice is available for the following device objects:

* [DRV\_SercDrv (SERCOS Drive)](DRV_SercDrvSERCOSDrive-DeviceObject-49683323.html)

The user function GenericSercosDevice provides the parameters required for identification of a Sercos device according to the Generic Device Profile (GDP).

The values you assign to the parameters are used to match configured devices to physically connected and scanned devices.

## Parameter ConfiguredVendorCode

|  |  |
| --- | --- |
| Data type | STRING(80) |
| Traceable | No |

The parameter ConfiguredVendorCode allows you to read and write the Sercos identifier of the generic Sercos device.

## Parameter ConfiguredVendorDeviceID

|  |  |
| --- | --- |
| Data type | WORD |
| Traceable | No |

The parameter ConfiguredVendorDeviceID allows you to read and write the manufacturer code of the generic Sercos device.
