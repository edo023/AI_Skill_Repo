# PROFINETIO\_Controller (PROFINETIO-Controller) - Device Objects in the Devices Tree

## Overview

The device object PROFINETIO\_Controller (PROFINETIO-Controller) represents a PROFINET IO controller.

You can add a PROFINETIO\_Controller (PROFINETIO-Controller) under the following device objects in the devices tree:

* [Ethernet\_4 (Ethernet4)](Ethernet_4Ethernet4-DeviceObjectsIn-48E99C7D.html)
* [Ethernet\_5 (Ethernet5)](Ethernet_5Ethernet5-DeviceObjectsIn-48EACCFA.html)
* [Ethernet\_6 (Ethernet6)](Ethernet_6Ethernet6-DeviceObjectsIn-4D5DD598.html)

For details on adding a PROFINET IO controller, refer to [Adding a PROFINET IO Controller](AddingAProfinetIOController-4CF960D6.html).

## Device Editor Tabs
