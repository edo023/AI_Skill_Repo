# OptionModule\_CN••\_CN•• - Device Object in the Devices Tree

## Overview

The device object OptionModule\_CN••\_CN•• is a communication device that represents an option module installed in one of the PCI Express Mini Card slots of the controller. If the option module has communication interfaces (RJ45 connectors), this communication device provides the basic communication functionality, for example, for TSN. The device object represents the interface point for the physical connectors **CN••** and **CN••** of the controller corresponding to the name of the device object.
