# FC\_DeactivateFanSupervision

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

Use function FC\_DeactivateFanSupervision of the library M6xxPLCSystem to disable monitoring of the fan of an M660 controller.

| NOTICE | |
| --- | --- |
|  | OVERHEATING  Uninstall the fan kit before deactivating fan monitoring on an M660 controller.  Failure to follow these instructions can result in equipment damage. |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Values:  TRUE: Fan monitoring is deactivated.  FALSE: Fan monitoring is activated. |

EIO0000005833.00
