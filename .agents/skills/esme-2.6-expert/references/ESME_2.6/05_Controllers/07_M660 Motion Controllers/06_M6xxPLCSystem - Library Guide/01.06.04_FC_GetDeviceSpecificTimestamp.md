# FC\_GetDeviceSpecificTimestamp

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

Use the function FC\_GetDeviceSpecificTimestamp of the library M6xxPLCSystem to correlate the timing in the Real Time Process (RTP) of the controller and the cycle of an Edge I/O network interface module (NIM).

The function uses a time offset i\_rtpOffset that is relative to the start of the active RTP cycle. It calculates an absolute time which is valid in the time clock of the given device.

Calculation: iq\_timestamp = [last start of RTP cycle] + i\_rtpOffset

Use the input i\_checkIfValidOutputTimestamp  to verify whether the resulting timestamp can be used as an output timestamp to generate an output edge. For example, the resulting timestamp may be too close to the present time and would arrive at the output registers of the output module in the past. In such a case, an error is detected.

If the function returns a value less than 0, the value of iq\_timestamp is invalid.

.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_device | [DeviceTypeInterfaces.IF\_Device\_Type](../../../../../api/crossBook?lang=en-US&virtualBookName=DeviceTypeInterfaces&topicID=IF_Device_Type_56CAF679) | Identifies the device. |
| i\_rtpOffset | DWORD | Time offset relative to start of the RTP cycle in which the function is executed.  Value range: 0...999.999.999  Unit: ns |
| i\_checkIfValidOutputTimestamp | BOOL | TRUE: Timestamp validity verification activated.  FALSE: Timestamp validity verification deactivated. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_timestamp | DWORD | Calculated timestamp |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: No error detected.  -1: The device addressed with i\_device is not found.  -2: Communication with device addressed with i\_device is not possible.  -3: The device addressed with i\_device is not present on the bus.  -4: The value for i\_rtpOffset is insufficient.  -5: The value for i\_rtpOffset is excessive.  -6: Verification with i\_checkIfValidOutputTimestamp is not supported. |
