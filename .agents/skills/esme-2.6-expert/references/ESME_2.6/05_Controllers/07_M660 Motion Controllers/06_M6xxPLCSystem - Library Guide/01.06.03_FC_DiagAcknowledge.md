# FC\_DiagAcknowledge

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

The function FC\_DiagAcknowledge of the library M6xxPLCSystem allows you to acknowledge a diagnostic message.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  * Remove the cause of a detected error before acknowledging the corresponding diagnostic message. * Verify that the condition causing the diagnostic is no longer present before the machine resumes operation. * Verify that acknowledging a diagnostic message cannot result in unwanted conditions or hazards as identified in your risk assessment.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

The function can be used to acknowledge diagnostic messages with a severity from 1 to 3 (DiagClass 1 to 3). If you acknowledge a diagnostic message with the function, the diagnostic message 8001 Diagnostics Acknowledgement is transmitted to the message logger.

Diagnostic messages with DiagClass 4 cannot be acknowledged with this function. If you attempt to acknowledge a diagnostic message with DiagClass 4 using this function, diagnostic message 8001 Diagnostics Acknowledgement is transmitted to the message logger, followed by diagnostic message 8011 to signal that the acknowledgement is not possible.

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Acknowledges a diagnostic message.  Value TRUE: Diagnostic message acknowledged  Value FALSE: Diagnostic message not acknowledged |
