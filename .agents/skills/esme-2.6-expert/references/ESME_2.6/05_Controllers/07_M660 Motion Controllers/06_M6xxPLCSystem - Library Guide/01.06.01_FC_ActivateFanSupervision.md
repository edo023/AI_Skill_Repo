# FC\_ActivateFanSupervision

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

The function FC\_ActivateFanSupervision of the library M6xxPLCSystem allows you to enable monitoring of the fan of an M660 controller.

Fan monitoring of M660 controllers is not enabled by default. Use this function to enable fan monitoring of M660 controllers that are factory-equipped with a fan, or after you have installed a fan kit (commercial references MNAH6AFS01 and MNAH6AFS02).

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | Values:  TRUE: Fan monitoring is enabled.  FALSE: Fan monitoring is not enabled. |
