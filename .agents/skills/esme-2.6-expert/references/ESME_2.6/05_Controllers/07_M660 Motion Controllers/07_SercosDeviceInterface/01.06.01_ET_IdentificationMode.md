# ET\_IdentificationMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | DINT |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_IdentificationMode of the library SercosDeviceInterface provides the values for the type of addressing of a Sercos device on the Sercos bus.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| SerialNumberController | 0 | The device is addressed via the serial number of the controller. |
| SerialNumberMotor | 1 | The device is addressed via the serial number of the motor. |
| TopologyAddress | 2 | The device is addressed via the topology address of the device. |
| ApplicationType | 3 | The device is addressed via the application type. |
| SercosAddress | 4 | The device is addressed via the Sercos address. |
