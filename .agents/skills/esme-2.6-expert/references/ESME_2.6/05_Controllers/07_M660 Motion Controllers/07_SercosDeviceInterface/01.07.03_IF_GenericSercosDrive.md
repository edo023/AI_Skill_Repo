# IF\_GenericSercosDrive

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | \_\_SYSTEM.IQueryInterface |

## Description

The user function interface IF\_GenericSercosDrive of the library SercosDeviceInterface allows you to access the user function GenericSercosDrive in order to read and write parameters relating to a generic Sercos drive.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| PrimaryOperationMode | UINT | Get/Set | Reads and writes the primary operation mode (Sercos IDN S-0-0032). |
| TelegramType | UINT | Get/Set | Reads and writes the Sercos telegram type (Sercos IDN S-0-0015). |
