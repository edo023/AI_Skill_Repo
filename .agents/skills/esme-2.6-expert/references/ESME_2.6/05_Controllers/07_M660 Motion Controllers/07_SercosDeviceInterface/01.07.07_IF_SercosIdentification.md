# IF\_SercosIdentification

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_SercosIdentification of the library SercosDeviceInterface allows you to access the user function SercosIdentification in order to read and write parameters relating to the type of identification and addressing of a device on the Sercos bus.
