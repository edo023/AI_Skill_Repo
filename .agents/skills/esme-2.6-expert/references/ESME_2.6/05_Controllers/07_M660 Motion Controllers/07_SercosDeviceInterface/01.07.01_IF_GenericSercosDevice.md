# IF\_GenericSercosDevice

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_GenericSercosDevice of the library SercosDeviceInterface allows you to access the user function GenericSercosDevice in order to read and write parameters relating to a generic Sercos device.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| ConfiguredVendorCode | WORD | Get/Set | Reads and writes the manufacturer code of the generic Sercos device. |
| ConfiguredVendorDeviceId | STRING(80) | Get/Set | Reads and writes the Sercos identifier of the generic Sercos device. |
