# Ethernet Ports

## Ethernet Ports (LAN)

| Characteristic | Value |
| --- | --- |
| Number of ports | 2 (single port, 2 distinct Ethernet endpoints) |
| Connector type | RJ45 |
| Maximum cable length | 100 m (328.1 ft) |
| Transmission speed | 10BASE-Te  100BASE-TX  1000BASE-T  2.5GBASE-T |
| Status LED | 1 x link/activity  1 x transmission speed |
| Features | Autonegotiation  Auto-MDI/MDIX |
| Assignment to operating system | CN5, **LAN1**: VxWorks (MN660C and MN660P)  CN6, **LAN2**: VxWorks (MN660C), Windows (MN660P) |
| Designation on housing | CN5 (**LAN1**)  CN6 (**LAN2**) |

All M660 controllers are equipped with these two Ethernet ports **LAN1** and **LAN2**.

## Ethernet Ports for Real-Time Ethernet Communication (RTE)

| Characteristic | Value |
| --- | --- |
| Number of ports | 2, 4 or 6, depending on the controller reference (dual port, switched, each pair of **RTE P1** and **RTE P2** is one Ethernet endpoint) |
| Function | Sercos |
| Connector type | RJ45 |
| Maximum cable length | 100 m (328.1 ft) |
| Transmission speed | 10BASE-Te  100BASE-TX  1000BASE-T |
| Status LED | 1 x link/activity per RJ45  For CN8/CN9, the LED **RTE** is in the status LED panel.  For CN20/CN21 and CN22/CN23, two additional **RTE** LEDs are displayed by the port themselves. |
| Features | Sercos  Autonegotiation  Auto-MDI/MDIX |
| Assignment to operating system | All RTE ports: VxWorks |
| Designation on housing | * CN8 (**RTE P1**) * CN9 (**RTE P2**) * CN20 (**RTE P1**) * CN21 (**RTE P2**) * CN22 (**RTE P1**) * CN23 (**RTE P2**) |

The number of RTE ports is determined by item 9 of the type code. Refer to [Type Code](TypeCode-E6EB6EB6.html) for details on M660 controller references.
