# FC\_GetPacDriveBootState

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

Use the function FC\_GetPacDriveBootState of the library SystemInterfaceLMC to determine the boot state of the controller. Use it to suspend the execution of code until the boot process of the controller has completed.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: Boot process not completed.  1: Boot process completed. |
