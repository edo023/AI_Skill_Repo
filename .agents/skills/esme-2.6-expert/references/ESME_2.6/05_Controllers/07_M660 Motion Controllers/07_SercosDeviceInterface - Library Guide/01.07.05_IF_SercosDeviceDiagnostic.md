# IF\_SercosDeviceDiagnostic

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_SercosDeviceDiagnostic of the library SercosDeviceInterface allows you to access the user function SercosDeviceDiagnostic in order to read the diagnostic number of a Sercos device (Sercos IDN S-0-0390).

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| DiagnosticNumber | DWORD | Get | Reads the diagnostic number of a Sercos device (Sercos IDN S-0-0390). |
