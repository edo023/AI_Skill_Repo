# IF\_FpgaFirmware

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_FpgaFirmware of the library CommonDeviceInterface allows you to access the user function FpgaFirmware to read the values of the parameters determining the type of communication technology assigned to a communication device, and to set the type of firmware update.

For details on the user function, refer to [M660 Programming Guide, FpgaFirmware](../../../../../api/crossBook?lang=en-US&virtualBookName=M660_PG&topicID=FpgaFirmware_UserFunction_07D459EA).

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| ConfiguredRevision | STRING(80) | Get, Set | Reads and writes the firmware version to be used for the communication device if the type of update is set to Fixed with UpdateMode. |
| Feature | [ET\_Feature](Feature-F3B0D69C.html#Feature-F3B0D69C) | Get | Reads the type of firmware of the communication device indicated by the enumeration [ET\_Feature](Feature-F3B0D69C.html#Feature-F3B0D69C). |
| Revision | STRING(80) | Get | Reads the version and build number of the firmware of the communication device. |
| UpdateMode | [ET\_UpdatedMode](ET_UpdateMode-473A279F.html) | Get, Set | Reads and writes the type of update of the firmware of the communication device indicated by the enumeration [ET\_UpdatedMode](ET_UpdateMode-473A279F.html). |
