# ET\_Feature

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | DINT |
| Available as of: | V1.0.0.0 |

## Description

This enumeration lists the values of the parameter Feature of the user function FpgaFirmware of a communication device. The value indicates the type of communication technology assigned to a communication device.

The user function is accessible via the user function interface [IF\_FpgaFirmware](IF_FpgaFirmware-F3AE522D.html#IF_FpgaFirmware-F3AE522D).

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| None | 0 | No firmware assigned to communication device. |
| Profinet | 1 | The communication device supports Profinet network technology. |
| Sercos | 2 | The communication device supports Sercos network technology. |
