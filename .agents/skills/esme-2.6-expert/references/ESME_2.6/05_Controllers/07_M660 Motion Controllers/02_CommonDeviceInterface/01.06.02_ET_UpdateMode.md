# ET\_UpdateMode

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | DINT |
| Available as of: | V1.0.0.0 |

## Description

This enumeration lists the values of the parameter UpdateMode of the user function FpgaFirmware of a communication device. This value determines the type of firmware applied to a communication device.

The user function is accessible via the user function interface [IF\_FpgaFirmware](IF_FpgaFirmware-F3AE522D.html#IF_FpgaFirmware-F3AE522D).

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Automatic | 0 | The firmware for the communication device is updated automatically. |
| Fixed | 1 | The firmware for the communication device is set to a firmware version specified with ConfiguredRevision. |
| Disabled | 2 | The firmware update for the communication device is disabled. |
