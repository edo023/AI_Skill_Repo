# 8994 Invalid ProducerCycleTime

Diagnostic class: 3

This message is generated during phase up of Sercos if the parameter ProducerCycleTime of a Sercos I/O device is not an integer multiple of the Sercos cycle time.

Ext. diagnosis:

`xxxxxxx ns`

`xxxxxxx` is the value of the incorrectly configured parameter ProducerCycleTime.

EIO0000005527.01
