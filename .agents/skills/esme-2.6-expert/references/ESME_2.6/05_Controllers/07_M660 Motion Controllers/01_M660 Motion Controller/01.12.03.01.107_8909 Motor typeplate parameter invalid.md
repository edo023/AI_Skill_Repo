# 8909 Motor typeplate parameter invalid

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | AD |
| Power supply | - | - |

| Cause | Solution |
| --- | --- |
| A parameter in the motor nameplate has an invalid value. The invalid parameter is displayed in the drive logger of the axis in the column Parameter 1. | Verify and correct the invalid parameter. |

| Parameter 1 | Description |
| --- | --- |
| 1 | rMainInductance is invalid or  rMotorCosPhi is invalid (if rNominalPower < 0.7 kW) |
| 2 | rStatorResistance is invalid or  rMotorCosPhi is invalid (if rNominalPower < 0.7 kW) |
| 3 | eMotorType is invalid |
| 4 | uiNominalPower or rNominalPower is invalid |
| 5 | uiEncoderType |
| 7 | wUnitScalingFactor |

EIO0000005527.01
