# Brake - User Function

## Description

The user function Brake is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function Brake provides parameters for the holding brake of a motor controlled through a drive.

## Parameter BrakeCouplingTime

|  |  |
| --- | --- |
| Data type | DINT(0...1000) |
| Traceable | Yes |

The parameter BrakeCouplingTime lets you specify the time required to apply the holding brake.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [BrakeCouplingTime](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071793).

|  |  |
