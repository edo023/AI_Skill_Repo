# SercosPhaseChanger - User Function

## Description

The user function SercosIdentification is available for the following device obects:

* [SercosMaster](SercosMaster-DeviceObjectInTheDevic-0684DFF3.html), Sercos master

The Sercos communication state machine uses the communication state NRT (Non-Real-Time) and five Communication Phases (CP) from CP0 to CP4. The user function SercosPhaseChanger lets you set and read the communication phases.

## Parameter DesiredPhase

The parameter DesiredPhase lets you set the Sercos master to the state NRT or to one of the communication phases.

|  |  |
| --- | --- |
| Data type | Enumeration of DINT |
| Traceable | Yes |

| Value | Meaning |
| --- | --- |
| NRT | Sets the Sercos master to the state NRT. In the state NRT, Sercos communication is disabled. |
| Phase0 | Sets the Sercos master to communication phase CP0. |
| Phase1 | Sets the Sercos master to communication phase CP1. |
| Phase2 | Sets the Sercos master to communication phase CP2. |
| Phase3 | Sets the Sercos master to communication phase CP3. |
| Phase4 | Sets the Sercos master to communication phase CP4. |

## Parameter ActualState

The parameter ActualState lets you read the active communication phase or state of the Sercos master.

|  |  |
| --- | --- |
| Data type | Enumeration of DINT |
| Traceable | Yes |

| Value | Meaning |
| --- | --- |
| NRT | The Sercos master is in the state NRT. In the state NRT, Sercos communication is disabled. |
| Phase0 | The Sercos master is in communication phase CP0. |
| Phase1 | The Sercos master is in communication phase CP1. |
| Phase2 | The Sercos master is in communication phase CP2. |
| Phase3 | The Sercos master is in communication phase CP3. |
| Phase4 | The Sercos master is in communication phase CP4. |
| Init | The Sercos master is in the state PhaseInit. |
| Error | The Sercos master is in the state PhaseError. |
