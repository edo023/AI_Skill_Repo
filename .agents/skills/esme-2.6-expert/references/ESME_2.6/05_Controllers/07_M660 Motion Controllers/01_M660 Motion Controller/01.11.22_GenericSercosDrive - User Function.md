# GenericSercosDrive - User Function

## Description

The user function GenericSercosDrive is available for the following device objects:

* [DRV\_SercDrv (SERCOS Drive)](DRV_SercDrvSERCOSDrive-DeviceObject-49683323.html)

The user function GenericSercosDrive provides parameters for reading and writing values required for a third-party Sercos drive according to the Function Specific Profile - Drive (FSP - Drive).

The values you assign to the parameters are used to match configured devices to physically connected and scanned devices.

## Parameter PrimaryOperationMode

|  |  |
| --- | --- |
| Data type | UINT |
| Traceable | No |

The parameter PrimaryOperationMode allows you to read and write the primary operating mode of the generic Sercos drive.

## Parameter TelegramType

|  |  |
| --- | --- |
| Data type | UINT(0...7) |
| Traceable | No |

The parameter TelegramType allows you to read and write the telegram type of the generic Sercos drive.
