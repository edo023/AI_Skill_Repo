# Device Editors

## Introduction

M660 controllers and devices used with an M660 controller are configured with device editors. Each device has its own device editor, each with several tabs. The number and type of tabs depends on the type of device.

## Access Rights

The tab Access Rights of the device editor lets you assign access rights of users to objects in the controller.

For details, refer to the [EcoStruxure Machine Expert Programming Guide, Access Rights](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083877).

## Applications

The tab Applications of the device editor lets you scan and remove applications on the controller. Information on the content of the application can be available as well as various details on the properties of the application.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Applications](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083388).

## Communication Settings

The tab Communication Settings of the device editor provides access to the Network Device Identification service that allows you to scan the Ethernet network for available controllers and to display them in a list. You can configure the parameters for the communication between the selected controller and the programming system.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Communication Settings in Controller Selection Mode](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0087840).

## Configuration

The tab Configuration of the device editor provides access to device-specific parameters.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Configuration.](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083387).

## Device Addressing

During commissioning, the devices connected to the Sercos bus are mapped to the Sercos objects in the controller configuration. The tab Device Addressing of the device editor allows you to map the Sercos devices of a configuration to physical devices on the Sercos bus found after a scan.

The devices in the PLC Configuration to the left are listed in separate columns with the values set for the following parameters:

* [ConfiguredTopologyAddress](M660_UserFunct_SercosIdentification-44A406D8.html#ParameterGroupSercosIdentification-44A406D8__ParameterConfiguredTopologyAddress-44A4F432)
* [ConfiguredSerialNumber](M660_UserFunct_SercosIdentification-44A406D8.html#ParameterGroupSercosIdentification-44A406D8__ParameterConfiguredSerialNumber-44A4FE17)
* [ConfiguredSerialNumberMotor](M660_UserFunct_SercosIdentification-44A406D8.html#ParameterGroupSercosIdentification-44A406D8__ParameterConfiguredSerialNumberMoto-44A5002C)
* [ConfiguredApplicationType](M660_UserFunct_SercosIdentification-44A406D8.html#ParameterGroupSercosIdentification-44A406D8__ParameterConfiguredApplicationType-44A4F824)
* [ConfiguredSercosAddress](M660_UserFunct_SercosIdentification-44A406D8.html#ParameterGroupSercosIdentification-44A406D8__ParameterConfiguredSercosAddress-44A4FC66)
* [IdentificationMode](M660_UserFunct_SercosIdentification-44A406D8.html#ParameterGroupSercosIdentification-44A406D8__ParameterIdentificationMode-44A4FEE1)
* [DesiredMode](M660_UserFunct_GroupWorkingMode-44A54DB0.html#ParameterGroupWorkingMode-44A54DB0__ParameterDesiredMode-44A5E285)

## Edge I/O NTS Bus I/O Mapping

The tab Edge I/O NTS Bus I/O Mapping of the device editor lets you map the inputs and outputs for the Edge I/O bus.

For details, refer to the [EcoStruxure Machine Expert Programming Guide, Files](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083398).

For configuration-specific information, refer to [NTS•••••••• - Device Object in the Devices Tree](NTSNSC1200-DeviceObjectInTheDevices-06851662.html).

## Edge I/O NTS Editor

The tab Edge I/O NTS Editor lets you configure the Edge I/O bus head NTSNSC1200.

For details, refer to [Edge I/O Network Interface Module Configuration with Edge I/O NTS Editor](NTSNSC1200-DeviceObjectInTheDevices-068506CC.html#NTSNSC1200-DeviceObjectInTheDevices-068506CC__ConfigurationWithEdgeIONTSEditor-08CEAD24).

## Feature Configuration

The tab Feature Configuration of the device editor allows you to enable special features provided for the device.

## Files

The tab Files of the device editor allows you to transfer files between the host and the controller. You can select any file from a directory of the local network to copy it to the file directory of the connected runtime system, or vice versa.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Files](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083390).

## IEC Objects

The tab IEC Objects of the device editor lets you map variables to POUs, for example, function blocks. In online mode, it allows for monitoring.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, IEC Objects](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0106082).

## Information

The tab Information of the device editor displays general information on the device such as name, vendor, category, type, ID, version, order number and a description.

## Log

The tab Log of the device editor lets you display the events which have been logged on the runtime system of the controller.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Log](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083391).

## Parameters

The tab Parameters of the device editor contains the user functions.

## PLC Settings

The tab PLC Settings of the device editor is used to configure general settings for the controller.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, PLC Settings](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083392).

## Sercos I/O Mapping

The tab Sercos I/O Mapping of the device editor lets you map the inputs and outputs for the Sercos bus.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Files](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083398).

## Status

The tab Status of the device editor displays status information and specific diagnostics messages.

This tab is available for subordinate devices of Sercos masters.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Status](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083395).

## Symbol Rights

The tab Symbol Rights of the device editor allows you to configure the access rights of different user groups (clients) to the individual symbol sets available on the controller.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Symbol Rights](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0106022).

## Task Deployment

The tab Task Deployment of the device editor displays a table with inputs/outputs and their assignment to the defined tasks.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Task Deployment](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083394).

## Users and Groups

The tab Users and Groups of the device editor is provided for user groups and user account management. You can assign rights to allow certain user groups to access objects on the controller at runtime by assigning access rights.

For details, refer to the  [EcoStruxure Machine Expert Programming Guide, Users and Groups](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083876).
