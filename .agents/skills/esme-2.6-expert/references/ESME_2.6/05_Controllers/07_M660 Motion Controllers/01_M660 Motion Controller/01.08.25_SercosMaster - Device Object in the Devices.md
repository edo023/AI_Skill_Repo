# SercosMaster - Device Object in the Devices Tree

## Overview

The device object SercosMaster represents the Sercos master. For details on the Sercos bus and on configuration of Sercos devices, refer to [Sercos](FieldbusSercos-494B0B82.html)

## Device Editor Tabs

The device object SercosMaster provides the following tabs in its device editor:

* [Device Addressing](DeviceConfigurationTabs-4D7E8C49.html#DeviceConfigurationTabs-4D7E8C49__DeviceAddressing-08930907)
* [Feature Configuration](DeviceConfigurationTabs-4D7E8C49.html#DeviceConfigurationTabs-4D7E8C49__FeatureConfiguration-089EC33E)

  You can enable the parameter model used for other Schneider Electric controllers by activating Sercos Master Compatibility Mode.
* [IEC Objects](DeviceConfigurationTabs-4D7E8C49.html#DeviceConfigurationTabs-4D7E8C49__IECObjects-08931328)
* [Information](DeviceConfigurationTabs-4D7E8C49.html#DeviceConfigurationTabs-4D7E8C49__Information-08931432)
* [Parameters](DeviceConfigurationTabs-4D7E8C49.html#DeviceConfigurationTabs-4D7E8C49__Parameters-08931755)
