The user function DigitalInput is available for the following device objects:

* [DI1 ... DI4](DI1ToDI4-DeviceObjectInTheDevicesTr-067E2217.html), digital inputs

The user function DigitalInput provides a parameter that lets you read the state of one of the inputs DI1 to DI4.

## Parameter Value

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | Yes |

The parameter Value lets you read the state of one of the inputs DI1 to DI4 which are child objects of the device object DigitalInputGroup\_CN2.

Assignment of the device objects to the pins of connector CN2:

* DI1 represents pin P6
* DI2 represents pin P7
* DI3 represents pin P8
* DI4 represents pin P9.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| FALSE (default value) | The state of the input is LOW/DEACTIVATED/0. |
| TRUE | The state of the input is HIGH/ACTIVATED/1. |
