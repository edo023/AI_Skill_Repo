# DrivePowerSupply - User Function

## Description

The user function DrivePowerSupply is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function DrivePowerSupply provides parameters for configuring the power supply of a drive and for reading the state of the power stage.

## Parameter ConfiguredPowerSupply

|  |  |
| --- | --- |
| Data type | STRING(40) |
| Traceable | No |

The parameter ConfiguredPowerSupply lets you configure the power supply of a drive on the Sercos bus. Assign the power supply that is physically connected to the DC bus of the drive. The power supply is only assigned to the drive when the controller boots.

| NOTICE | |
| --- | --- |
|  | INCORRECT ASSIGNMENT OF THE DRIVES TO THE POWER SUPPLY  Ensure that the drive or drives physically connected to the power supply are also logically assigned to that power supply.  Failure to follow these instructions can result in equipment damage. |

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [ConfiguredPowerSupply](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0072732)..

## Parameter PowerSupply

|  |  |
| --- | --- |
| Data type | STRING(40) |
| Traceable | No |

The parameter PowerSupply lets you read the power supply configured for a drive on the Sercos bus.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [PowerSupply](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0072733)..

## Parameter InverterEnable

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | Yes |

The parameter InverterEnable lets you read the state of the power stage of a drive on the Sercos bus.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [InverterEnable](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071480)..
