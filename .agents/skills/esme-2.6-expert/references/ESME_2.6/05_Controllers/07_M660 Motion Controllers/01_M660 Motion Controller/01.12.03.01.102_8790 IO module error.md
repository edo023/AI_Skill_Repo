# 8790 IO module error

Diagnostic class: 2

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | D | E |
| Power supply | - | - |

A module error is detected.

| Ext. diagnosis | Meaning |
| --- | --- |
| `Com.`  `Com. IOModule` | An interrution of communication is detected between DIO8 module (optional module of Lexium 62 ILM Module) and Lexium 62 ILM Module.  Contact your local Schneider Electric service representative. |
| `DIVCLK` | An error is detected while synchronizing with the Sercos cycle.  Contact your local Schneider Electric service representative. |
| `drive disabled` | The Lexium 62 ILM Module, on which the DIO8 module (optional module of the Lexium 62 ILM Module) was configured is deactivated and the DIO8 module is activated (Enable = TRUE).  Verify and correct if necessary the PLC configuration. |
| `int. error` | Contact your local Schneider Electric service representative. |

NOTE: In certain cases, another diagnostic code or diagnostic class is entered in the message logger of the controller for DIO8 module and Lexium 62 ILM/Lexium 52 Drive onboard I/O modules in the message logger of the drive.
