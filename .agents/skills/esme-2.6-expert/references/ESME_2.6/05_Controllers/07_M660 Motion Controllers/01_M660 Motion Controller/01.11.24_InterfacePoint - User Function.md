# InterfacePoint - User Function

## Description

The user function InterfacePoint is available for the following device objects:

* [CommunicationDevice\_CN20\_CN21](CommunicationDevice_CN20_CN21CommDe-4C5BB44F.html)

  , communication device
* [CommunicationDevice\_CN22\_CN22](CommunicationDevice_CN22_CN23CommDe-4C5EBC80.html)

  , communication device
* [CommunicationDevice\_CN8\_CN9](CommunicationDevice_CN8_CN9-DeviceO-067E64B6.html), communication device
* [Ethernet\_1 (Ethernet1)](Ethernet_1Ethernet1Ethernet_2Ethern-067E4820.html), communication device
* [Ethernet\_2 (Ethernet2)](Ethernet_2Ethernet2-DeviceObjectsIn-0B82E998.html), communication device
* [Ethernet\_3 (Ethernet3)](Ethernet_3Ethernet3-DeviceObjectInT-067E8BF3.html), communication device
* [OptionCard\_CN20\_CN21](OptionCard_CN_CN-DeviceObjectInTheD-067DD19F.html), communication device
* [OptionModule\_CN••\_CN••](OptionModule_CN_CN-DeviceObjectInTh-067DB53D.html), communication device

The user function InterfacePoint provides parameters relating to the assignment of a communication device in the devices tree to a physical connector of the controller.

## Parameter Connector

|  |  |
| --- | --- |
| Data type | Enumeration of DINT |
| Traceable | Yes |

The parameter Connector lets you read the physical connector of the controller that is assigned to the communication device in the devices tree. The value depends on the communication device for which the user function is used. For example, CN5 is assigned to the communication object Ethernet\_1.

Assignment to the operating system:

* **CN5**: VxWorks (MN660C and MN660P)
* **CN6**: VxWorks (MN660C), Windows (MN660P)

The parameter provides the following values:

| Value | Description |
| --- | --- |
| None | No port assigned to communication device.  This value is only assigned to port **CN6** of MN660C controllers. |
| Internal | Reserved. |
| CN5 | Communication device communicates using connector **CN5** (Ethernet port **LAN1**). |
| CN6 | Communication device communicates using connector **CN6** (Ethernet port **LAN2**). |
| CN8\_CN9 | Communication device communicates using connectors **CN8** and **CN9** (**RTE** ports). |
| CN20\_CN21 | Communication device communicates using connectors **CN20** and **CN21** (**RTE** ports). |
| CN22\_CN23 | Communication device communicates using connectors **CN22** and **CN23** (**RTE** ports). |
| CN30\_CN31 | Communication device communicates using connectors **CN30** and **CN31** (**RTE** ports). |
| CN32\_CN33 | Communication device communicates using connectors **CN32** and CN33 (**RTE** ports). |
