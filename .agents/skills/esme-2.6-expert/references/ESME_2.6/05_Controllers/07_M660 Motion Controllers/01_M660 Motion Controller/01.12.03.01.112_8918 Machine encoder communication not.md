# 8918 Machine encoder communication not possible

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | BD2 | CD |
| Power supply | - | - |

When reading the machine encoder, an error is detected.

| Cause | Solution |
| --- | --- |
| Detected wiring error: Encoder cable is not correct. | Verify and correct if necessary the wiring. |
| The control voltage of the device (24 Vdc) is too low. | Verify and correct if necessary the control voltage. |
| Detected hardware error: Encoder is inoperable. | Replace the encoder. |
