# 8906 ControlMode invalid

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | AD |
| Power supply | - | - |

| Cause | Solution |
| --- | --- |
| * The parameter ControlMode has an invalid value. * ControlMode is set to `open-loop control / 1` and a synchronous motor is used (MotorClass is `Synchronous motor / 0`). * ControlMode is set to `Encoderless velocity closed-loop control / 2` and an asynchronous motor is used. * ControlMode is set to `open-loop control / 1` or `Encoderless velocity closed-loop control / 2` and a linear motor is used. | Synchronous motors can only be operated in closed-loop control, change ControlMode to `closed-loop control / 0` or `Encoderless velocity closed-loop control / 2`.  Asynchronous motors cannot be operated with encoderless velocity closed loop control, change ControlMode to `closed-loop control / 0`or `open-loop control / 1`.  Linear motors can only be used with closed-loop control, change ControlMode to `closed-loop control / 0`. |
