# General Information

## Overview

M660 controllers feature a firewall that is activated by default.

The firewall is configured on the following Ethernet interfaces:

* **CN5**: MN660C, MN660P
* **CN6**: MN660C
* **CN8**: MN660C, MN660P
* **CN9**: MN660C, MN660P

Incoming traffic is blocked, except for defined applications/ports.

Outgoing traffic is not affected by the default configuration of the firewall.

## Configuration

You can configure the firewall by editing a configuration file on the memory card. The firewall cannot be configured during runtime.
