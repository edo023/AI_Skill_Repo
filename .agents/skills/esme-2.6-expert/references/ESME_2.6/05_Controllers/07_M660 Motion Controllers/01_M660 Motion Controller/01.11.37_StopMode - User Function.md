# StopMode - User Function

## Description

The user function StopMode is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function StopMode provides parameters for decelerating a motor.

## Parameter ControllerStopDeceleration

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | Yes |

The parameter ControllerStopDeceleration lets you write the deceleration in units/s2 for decelerating the motor.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [ControllerStopDeceleration](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071532).
