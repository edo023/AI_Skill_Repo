# 8187 Option module not supported

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive (Lexium 62 ILM, Lexium 62 ILD ) | AD | AD |
| Power supply | - | - |

The option module used is unsupported.

The Lexium 62 ILM Safety Module can be used only when the hardware revision is 5 or greater. The hardware revision is shown in the 4th figure of the parameter HW\_Version.
