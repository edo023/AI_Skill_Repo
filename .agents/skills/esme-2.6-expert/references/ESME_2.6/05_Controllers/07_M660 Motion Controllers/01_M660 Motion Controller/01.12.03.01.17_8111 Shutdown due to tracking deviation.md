# 8111 Shutdown due to tracking deviation

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | BD2 | BD2 |
| Power supply | - | - |

* A major tracking deviation has occurred which is greater than 8 times the value of the parameter TrackingDeviationLimit.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| This error normally occurs in connection with the CAM movement functions (for example, `MultiCam()` ). | Verify and correct if necessary curve data. |
| The axis should be operated at regulated speeds. | Set parameter Pos\_P\_Gain to `0`. |
