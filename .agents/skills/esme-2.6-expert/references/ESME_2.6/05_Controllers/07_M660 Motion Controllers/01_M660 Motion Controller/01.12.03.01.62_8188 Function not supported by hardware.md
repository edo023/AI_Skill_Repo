# 8188 Function not supported by hardware

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | AD |
| Power supply | - | - |

A functionality set by the user is not supported by this hardware or FPGA version.

There are FPGA versions in Lexium 52 that do not support a PWM frequency of 4 kHz. There are FPGA versions in drives that do not support V/f-control for asynchronous motors.

Contact your local Schneider Electric service representative.
