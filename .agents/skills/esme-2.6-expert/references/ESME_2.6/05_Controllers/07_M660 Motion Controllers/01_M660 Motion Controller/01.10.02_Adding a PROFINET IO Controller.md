# Adding a PROFINET IO Controller

## Overview

A PROFINET IO controller is represented by the device object [PROFINETIO\_Controller (PROFINETIO-Controller)](PROFINETIO_ControllerPROFINETIO-Con-4D5FB29A.html).

You can add a PROFINETIO\_Controller (PROFINETIO-Controller) under the following device objects in the devices tree:

* Modicon\_M660 - OptionCard\_CN20\_CN21 - [Ethernet\_4 (Ethernet4)](Ethernet_4Ethernet4-DeviceObjectsIn-48E99C7D.html)
* Modicon\_M660 - OptionCard\_CN20\_CN23 - CommunicationDevice\_CN20\_CN21[Ethernet\_5 (Ethernet5)](Ethernet_5Ethernet5-DeviceObjectsIn-48EACCFA.html)
* Modicon\_M660 - OptionCard\_CN20\_CN23 - CommunicationDevice\_CN22\_CN23 [Ethernet\_6 (Ethernet6)](Ethernet_6Ethernet6-DeviceObjectsIn-4D5DD598.html)

## Configuration of the Network Adapter

If you add a PROFINETIO\_Controller (PROFINETIO\_Controller) device object, you need to configure the Network interface.

Procedure:

| Step | Action |
| --- | --- |
| 1 | Display the tab General of the device editor of the device object Ethernet\_• (Ethernet• you intend to use for the PROFINET IO controller. |
| 2 | Click Browse.  **Result**: The system displays the dialog box Network Adapters. |
| 3 | From the Interfaces list, select the required value:   * Ethernet\_4 (Ethernet4): endpoint3 * Ethernet\_5 (Ethernet5): endpoint3 * Ethernet\_6 (Ethernet6): endpoint4 |
| 4 | Confirm with OK. |

NOTE: The project is saved with the MAC address displayed in the dialog box Network Adapters. This means that you cannot run the project on a different M660 controller without first saving it with the correct MAC address on that controller.
