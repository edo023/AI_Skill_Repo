# ElectronicLabel - User Function

## Description

The user function ElectronicLabel is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives
* [PSM\_Lexium62 (Lexium62 Power Supply)](PSM_Lexium62Lexium62PowerSupply-Dev-0790BF9D.html), Lexium LXM62 power supplies
* [Modicon\_M660 (Modicon M660)](ModiconM660ModiconM660-DeviceObject-067D8F23.html), controller

The user function ElectronicLabel provides parameters for reading the information from the electronic nameplate of a device.

## Parameter ControllerType

|  |  |
| --- | --- |
| Data type | STRING(80) |
| Traceable | No |

The parameter ControllerType lets you read the type of device stored in the electronic nameplate of the device. The value is available as of Sercos communication phase CP2.

If the WorkingMode of the device is Virtual, the value of this parameter is DRIVE\_VIRTUAL.
