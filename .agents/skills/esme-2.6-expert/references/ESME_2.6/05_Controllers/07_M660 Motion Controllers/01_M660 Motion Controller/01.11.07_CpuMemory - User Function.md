# CpuMemory - User Function

## Description

The user function CpuMemory is available for the following device objects:

* [Modicon\_M660 (Modicon M660)](ModiconM660ModiconM660-DeviceObject-067D8F23.html), Modicon M660 controller

The user function CpuMemory provides parameters that allow you to monitor the amount of memory used by the CPU of the controller.

## Parameter MemoryType

|  |  |
| --- | --- |
| Data type | String(80) |
| Traceable | Yes |

The parameter MemoryType allows you to read the type of memory monitored.

## Parameter Free

|  |  |
| --- | --- |
| Data type | ULINT |
| Traceable | Yes |

The parameter Free allows you to read the available memory in bytes.

## Parameter Size

|  |  |
| --- | --- |
| Data type | ULINT |
| Traceable | Yes |

The parameter Size allows you to read the total memory size in bytes.
