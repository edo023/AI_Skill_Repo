# 8100 Motor overload

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | BD2 | CD |
| Power supply | – | – |

* The motor is overloaded and therefore drawing excessive current. The overload level has increased to 100 %. The current overload of the drive is shown in the parameter DriveOverload.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| The holding brake is not released (cable break, power outside the 24 V +/- 10% range). | Verify and correct if necessary the brake and the wiring. |
| The lubrication system is not functioning correctly. | Verify and correct if necessary the lubrication |
| Blunt tool. | Verify and replace if necessary the tool. |
| The motor is vibrating. | Verify and correct if necessary the controller parameters. |
| The suspended axis is too heavy or the weight compensation is incorrect. | * Verify and correct if necessary the drive sizing and the weight distribution. * Drive/accelerate the system at a slower speed. |
| The motor is de-magnetized. | Replace the motor. |
