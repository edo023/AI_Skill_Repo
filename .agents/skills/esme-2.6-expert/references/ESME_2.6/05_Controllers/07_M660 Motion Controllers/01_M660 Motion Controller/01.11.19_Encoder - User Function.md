# Encoder - User Function

## Description

The user function Encoder is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function Encoder provides parameters functions for an encoder controlled through a drive.

## Parameter EncoderType

|  |  |
| --- | --- |
| Data type | Enumeration of DINT |
| Traceable | Yes |

The parameter EncoderType lets you specify the type of encoder.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| NoEncoder / 0 (default value) | No encoder |
| SinCosWithHiperface / 1 | SinCos encoder with Hiperface interface |
| SinCosWithoutHiperface / 2 | SinCos encoder without Hiperface interface |
| SinCosWithHall / 3 | SinCos encoder with Hall effect sensor |
| SinCosWithEnDat / 4 | SinCos encoder with EnDat interface |
| EnDat / 5 | Encoder with EnDat interface |
| Resolver / 6 | Resolver |
| Hall / 7 | Encoder with Hall effect sensor |
| BiSS / 8 | Encoder with BiSS communication protocol |
| HiperfaceDSL / 10 | Hiperface encoder with DSL interface |

## Parameter GearIn

|  |  |
| --- | --- |
| Data type | UDINT(1...999999) |
| Traceable | Yes |

The parameters GearIn and GearOut let you specify a gear factor between motor and encoder. GearIn represents the gear input value from motor to load.

This parameter corresponds to the Sercos parameter IDN P-0-1092.0.1.

## Parameter GearOut

|  |  |
| --- | --- |
| Data type | UDINT(1...999999) |
| Traceable | Yes |

The parameters GearIn and GearOut let you specify a gear factor between motor and encoder. GearOut represents the gear output value from motor to encoder.

This parameter corresponds to the Sercos parameter IDN P-0-1092.0.21.

## Parameter InvertDirection

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | NO |

The parameter InvertDirection allows you to invert the counting direction of the encoder relative to the direction of rotation of the motor.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  * Verify that no hazard can result from using this parameter. * Verify that the type of mechanical coupling between the encoder and the motor supports an inversion of the counting direction of the encoder. * Do not invert the counting direction of encoders integrated in Schneider Electric servo motors.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

The counting direction is taken into account in calculating the parameters EncoderPosition, Position, MechPosition and ShaftMechPosition.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| TRUE | The counting direction of the encoder is inverted. The encoder shaft rotates in the opposite direction of the motor shaft. |
| FALSE | The counting direction of the encoder is not inverted. The encoder shaft rotates in the same direction as the motor shaft. |
