# 8500 Sercos: Device with duplicate address

Diagnostic class: 3

| Cause | Remedy |
| --- | --- |
| A Sercos device was detected that has the same Sercos address as another Sercos device. | Use the parameters of the user function [SercosIdentification](M660_UserFunct_SercosIdentification-44A406D8.html#M660_UserFunct_SercosIdentification-44A406D8) to set a unique Sercos address for the device. |
