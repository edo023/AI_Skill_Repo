# Overview of User Functions

## Introduction

User functions are groups of parameters relating to specific types of functionality.

The user functions are displayed in the tab Parameters of the device editor of a device. You can expand and collapse the parameters of a user function by clicking the + or - sign to the left of the name of the user function.

**Syntax:**

You can access a parameter in the application using the following syntax:

```
<Device>.<UserFunction>.<Parameter>
```

**Example:**

The following command sets the parameter WorkingMode of the user function SercosIdentification to the value 1:

```
DRV_Lexium62.SercosIdentification.WorkingMode := 1;
```
