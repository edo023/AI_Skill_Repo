# Ethernet\_1 (Ethernet1) - Device Objects in the Devices Tree

## Overview

The device object Ethernet\_1 (Ethernet1) is a communication device.

Ethernet\_1 (Ethernet1) represents the interface point for the physical connector **CN5** (port **LAN1**) of the controller.

Assignment to the operating system:

* VxWorks (MN660C and MN660P)

The device object is generated automatically when you create an empty project with an M660 controller.
