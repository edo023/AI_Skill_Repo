# 8501 Sercos: Device not found

Diagnostic class: 3

A Sercos device in the configuration could not be assigned to a device on the Sercos bus.

| Cause | Solution |
| --- | --- |
| Incorrect assignment of a Sercos device detected during transition from Sercos communication phase CP2 to CP3. | Verify the identification mode and the type of identification (user function [SercosIdentification](M660_UserFunct_SercosIdentification-44A406D8.html#M660_UserFunct_SercosIdentification-44A406D8) and its parameters). |
| Sercos device not physically present on the Sercos bus. | * Connect the Sercos device to the Sercos bus. * Remove the Sercos device from the configuration if it is not needed. * Set the working mode of the Sercos device to Virtual with the parameter [DesiredMode](M660_UserFunct_GroupWorkingMode-44A54DB0.html#M660_UserFunct_GroupWorkingMode-44A54DB0__M660_Param_DesiredMode-44A5E285) |
|  |  |
