# DRV\_SercDrv (SERCOS Drive) - Device Object in the Devices Tree

## Overview

The device object DRV\_SercDrv (SERCOS Drive) represents a generic Sercos drive. It operates as a Sercos device under the [SercosMaster](SercosMaster-DeviceObjectInTheDevic-0684DFF3.html).

You can add third-party Sercos drives that support the Generic Device Profile (GDP) and the Function Specific Profile - Drive (FSP - Drive)

For information on the use and configuration of third-party Sercos drives, refer to the drive-specific user guides and all other pertinent documentation of the drive.
