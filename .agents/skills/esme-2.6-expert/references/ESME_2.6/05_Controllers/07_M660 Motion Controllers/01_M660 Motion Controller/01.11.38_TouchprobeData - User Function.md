# TouchprobeData - User Function

## Overview

The user function TouchprobeData is available for the following device objects:

* [Touchprobe](Touchprobe-DeviceObjectInTheDevices-0685B3BB.html)

The user function TouchprobeData provides parameters for reading and writing values delivered by the touchprobe and for configuring it.

## Parameter CaptureOk

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | Yes |

The parameter CaptureOk indicates that the configured edge has been detected.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| False | The configured edge has not been detected. |
| True | The configured edge has been detected. |

## Parameter CaptureState

|  |  |
| --- | --- |
| Data type | Enumeration of UINT |
| Traceable | Yes |

The parameter CaptureState lets you read the state of the position capture function.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| Inactive | The conditions for starting position capture are met, but position capture is not active. |
| Active | Position capture is in progress. |
| Captured | A position has been captured and the new captured position value is available. |
| Overflow | Reserved. |
| Disabled | Reserved. |
| NotUsed | Position capture is not running and cannot be started, for example, because Sercos is not in communication phase CP 4. |
| Virtual | Position capture has been started with the timestamp touch probe functionality with a Modicon Edge I/O NTS module whose network interface module is in the in the working mode Virtual. In this case, no physical positions are captured and the state Virtual is not reached. Aborting the function block MC\_TouchProbe transitions to the state Inactive of this enumeration. |

## Parameter CaptureValue

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | Yes |

The parameter CaptureValue lets you read the value of a captured position.

## Parameter Value

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | Yes |

The parameter Value lets you read the voltage value at the touchprobe input.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| False | The voltage is 0 V (input state LOW). |
| True | The voltage is 24 V (input state HIGH). |

## Parameter CaptureEdge

|  |  |
| --- | --- |
| Data type | Enumeration of UINT |
| Traceable | Yes |

The parameter CaptureEdge lets you read the type of the capture edge configured for position capture.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| FallingEdge | The touchprobe uses a falling edge of the selected input for position capture. |
| RisingEdge | The touchprobe uses a rising edge of the selected input for position capture. |
| BothEdges | The touchprobe uses both falling and rising edges of the selected input for position capture. |

## Parameter CaptureEdgeSet

|  |  |
| --- | --- |
| Data type | Enumeration of UINT |
| Traceable | Yes |

The parameter CaptureEdgeSet lets you read or write the configured edge of the touchprobe input.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| FallingEdge | The touchprobe uses a falling edge of the selected input for position capture. |
| RisingEdge | The touchprobe uses a rising edge of the selected input for position capture. |
| BothEdges | The touchprobe uses both falling and rising edges of the selected input for position capture. |

## Parameter CaptureCount

|  |  |
| --- | --- |
| Data type | DINT |
| Traceable | Yes |

The parameter CaptureCount lets you read the number of capture events during the last Sercos cycle.

## Parameter SensorDelay

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | Yes |

The parameter SensorDelay lets you read or write the delay of a position sensor to be considered in position capture.

## Parameter CaptureEdgeSetDelayed

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | Yes |

The parameter CaptureEdgeSetDelayed lets you read whether there is a delay on change of CaptureEdge.

The parameter provides the following values:

| Value | Description |
| --- | --- |
| False | The change of the edge is set at the start of the function. |
| True | The change of the edge is delayed from the start of the function. |
