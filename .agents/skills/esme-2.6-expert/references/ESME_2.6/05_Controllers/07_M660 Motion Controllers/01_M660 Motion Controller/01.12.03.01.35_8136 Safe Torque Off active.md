# 8136 Safe Torque Off active

Diagnostic class: 2

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | D | E |
| Power supply | - | - |

An attempt is being made to start the drive, but Inverter\_Enable is still LOW.

| Cause | Solution |
| --- | --- |
| ControllerEnable is activated, but InverterEnable is still LOW. | Verify and correct if necessary the wiring of InverterEnable. |

Probably it is normal machine behavior. If it is not allowed to switch off InverterEable during the axis is controlled, InverterEnableFunction has to be set to `Safe Stop / 1`.

| Cause | Solution |
| --- | --- |
| The parameter InverterEnableFunction is set to `Safe Torque Off / 0`, the axis is moving, and the InverterEnable signals gets LOW. | Probably it is normal machine behavior. If it is not allowed to switch off InverterEable during the axis is controlled, InverterEnableFunction has to be set to `Safe Stop / 1`. |
