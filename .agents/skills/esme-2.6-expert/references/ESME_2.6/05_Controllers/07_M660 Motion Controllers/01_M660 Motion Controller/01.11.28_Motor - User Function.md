# Motor - User Function

## Description

The user function Motor is available for the following device objects:

* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), Lexium ILM62 drives
* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), Lexium LXM62 drives

The user function Motor provides parameters for the motor connected to a drive.

## Parameter MaxSpeed

|  |  |
| --- | --- |
| Data type | LREAL |
| Traceable | Yes |

The parameter MaxSpeed lets you read the maximum speed of rotation of the motor.

For details, refer to the Lexium LXM62 Drive Device Objects and Parameters guide, [MaxSpeed](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071784).

## Parameter MotorIdentification

|  |  |
| --- | --- |
| Data type | Enumeration of INT |
| Traceable | Yes |

The parameter MotorIdentification lets you read and write type of identification of the motor.

The parameter corresponds to the Sercos parameter IDN P-0-0623.

The parameter provides the following values:

| Value | Meaning |
| --- | --- |
| Motor with type plate / 0 | The motor is equipped with an electronic nameplate |
| No motor / 1 | No motor. |
| Motor without type plate / 2 | The motor is not equipped with an electronic nameplate |
