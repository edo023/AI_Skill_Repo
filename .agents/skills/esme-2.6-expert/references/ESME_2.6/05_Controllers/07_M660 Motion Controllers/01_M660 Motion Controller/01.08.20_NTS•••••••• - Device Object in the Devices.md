# NTS•••••••• - Device Object in the Devices Tree

## Overview

The device object NTS•••••••• represents an Edge I/O module. It is a child object of the device object [NTSNSC1200](NTSNSC1200-DeviceObjectInTheDevices-068506CC.html), serving as the network interface module. Before you can add NTS•••••••• device objects, you must add the network interface module.

For details on Edge I/O, refer to the Modicon Edge I/O user guides.

The configuration depends on the type of Edge I/O module.

## Edge I/O Module Configuration with Edge I/O NTS Editor

The configuration options provided by the Edge I/O NTS Editor depend on the type of Edge I/O module.

The following table provides an overview of the configuration options of the Modicon Edge I/O Motion Expert module NTSEDT0800. This module can be used for touchprobe functionality. For details on the items in the table, refer to the Modicon Edge I/O user guides.

| Item | Description |
| --- | --- |
| Device Mode | Selects the device mode. Available values:   * Normal * Optional * Virtual reserved   For details on the device mode, refer to the Modicon Edge I/O user guides. |
| Rearming Output Mode | Sets the type of rearming the output. Available values:   * Auto Recovery * Latched Off   For details on the rearming mode, refer to the Modicon Edge I/O user guides. |
| Input Channel\_nn |  |
| Latch | Specifies the type of trigger edge and acknowledgement.  For details on the latch settings, refer to the Modicon Edge I/O user guides. |
| Filter | Specifies the input filter time in ms.  For details on the filter settings, refer to the Modicon Edge I/O user guides. |
| Output Channel\_nn | |
| Fallback Mode | Sets the type of fallback for the output. Available values:   * Fallback Value * Maintain   For details on the fallback mode, refer to the Modicon Edge I/O user guides. |
| Predefined Fallback Value | Specifies the fallback value for the output, either 1 or 0.  For details on the fallback value, refer to the Modicon Edge I/O user guides. |
| Expert Functions | |
| Add a Function | Dropdown list for selection of expert functions. The available expert functions depend on the type of module.   * Timestamp\_Input * Oversampling\_Output |
| Timestamp\_Input\_1 | |
| Input location | Specifies the input of the module to be used for the expert function. |
| Filter | Specifies the filter time in ms. |
| Oversampling\_Input\_1 | |
| Input location | Specifies the input of the module to be used for the expert function. |
| Sampling step mode | Specifies the number of steps per cycle. |
| Filter | Specifies the filter time in ms. |

Depending your configuration, you can add additional device objects that represent functionality, for example, the device object [Touchprobe](Touchprobe-DeviceObjectInTheDevices-0685B3BB.html).
