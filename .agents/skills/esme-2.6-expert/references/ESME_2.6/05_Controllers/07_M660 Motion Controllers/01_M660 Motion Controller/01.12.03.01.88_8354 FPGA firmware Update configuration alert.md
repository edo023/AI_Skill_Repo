# 8354 FPGA firmware: Update configuration alert

Diagnostic class: 2

An alert was detected in the update of the FPGA firmware of a communication device. The FPGA firmware was not updated. The affected communication device remains operative, but it may not use the latest FPGA firmware.

The ext. diagnosis indicates the affected communication device.

| Cause | Remedy |
| --- | --- |
| Reading or parsing the FPGA firmware update configuration file from the SD card was not successful. The SD card contains a JSON file that specifies the FPGA firmware to be loaded onto the individual FPGA devices. | Restart the controller. The FPGA firmware update is repeated. Verify that the FPGA firmware update has completed (diagnostic message [8351 FPGA firmware: Update completed](FPGAFirmwareUpdateCompleted-4C961008.html)).  If the condition persists, contact your local Schneider Electric representative. |
