# 8511 CPU time overflow

Diagnostic class: 3

The real-time process is overloaded (CycleLoad >95 %). The DiagExtMsg parameter displays the measured load as Load = `xxx %` with `xxx` indicating the load in %.

NOTE: When this error is detected, the actual values and the target values are no longer processed in the real-time process.

| Cause | Solution |
| --- | --- |
| Too many devices in the controller configuration. | * Increase the Sercos [CycleTime](M660_UserFunct_SercosCycletimeConfig-449D0B4A.html#M660_UserFunct_SercosCycletimeConfig-449D0B4A) . * Decrease the number of devices in the configuration. |
