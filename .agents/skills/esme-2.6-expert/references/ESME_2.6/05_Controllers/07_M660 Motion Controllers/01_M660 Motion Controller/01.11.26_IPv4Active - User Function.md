# IPv4Active - User Function

## Description

The user function IPv4Active is available for the following device objects:

* [Ethernet\_1 (Ethernet1)](Ethernet_1Ethernet1Ethernet_2Ethern-067E4820.html), communication device
* [Ethernet\_2 (Ethernet2)](Ethernet_2Ethernet2-DeviceObjectsIn-0B82E998.html), communication device
* [Ethernet\_3 (Ethernet3)](Ethernet_3Ethernet3-DeviceObjectInT-067E8BF3.html), communication device

The user function IPv4Active provides parameters for reading the IP address and the subnet mask of communication devices.

## Parameter IpAddress

|  |  |
| --- | --- |
| Data type | STRING(80) |
| Traceable | No |

The parameter IpAddress lets you read the IP address of the communication device.

Use the parameter IpAddress of the user function [IPv4Configured](IPv4Configured-UserFunction-079D4A17.html)  to set an IP address.

## Parameter SubnetMask

|  |  |
| --- | --- |
| Data type | STRING(80) |
| Traceable | No |

The parameter SubnetMask lets you read the subnet mask of the communication device.

Use the parameter SubnetMask of the user function [IPv4Configured](IPv4Configured-UserFunction-079D4A17.html) to set a subnet mask.
