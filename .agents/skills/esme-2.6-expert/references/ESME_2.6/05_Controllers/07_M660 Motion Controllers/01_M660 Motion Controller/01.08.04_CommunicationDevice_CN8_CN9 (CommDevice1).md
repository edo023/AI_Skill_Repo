# CommunicationDevice\_CN8\_CN9 (CommDevice1) - Device Object in the Devices Tree

## Overview

The device object CommunicationDevice\_CN8\_CN9 (CommDevice1) is a communication device that provides the basic communication functionality for Sercos. It represents the interface point for the physical connectors **CN8** and **CN9** of the controller.

The device object is generated automatically when you create an empty project with an M660 controller.
