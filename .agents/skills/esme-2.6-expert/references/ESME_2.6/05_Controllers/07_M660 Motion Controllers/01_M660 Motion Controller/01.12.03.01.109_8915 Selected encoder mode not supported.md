# 8915 Selected encoder mode not supported

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | AD |
| Power supply | - | - |

The machine encoder can only be used for position control with a synchronous motor in closed-loop control.

| Cause | Solution |
| --- | --- |
| The parameter EncoderMode is set to “Machine encoder is only used for position control / 1”. This setting is not permissible for linear motors. | Use a synchronous motor or set the parameter EncoderMode to a different value. |
| The parameter EncoderMode is set to “Machine encoder is only used for position control / 1”. This setting is not permissible for asynchronous motors. | Use a synchronous motor or set the parameter EncoderMode to a different value. |
