# PowerSupply - User Function

## Description

The user function PowerSupply is available for the following devices:

* [PSM\_Lexium62 (Lexium62 Power Supply)](PSM_Lexium62Lexium62PowerSupply-Dev-0790BF9D.html), Lexium LXM62 power supplies

The user function PowerSupply provides parameters for the configuration of a power supply.

## Parameter GroupReady

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | Yes |

The parameter GroupReady indicates whether the combination of power supply and connected drives is ready to perform movements.

For details, refer to Lexium LXM62 Power Supply Device Objects and Parameters guide, [GroupReady](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62PowerSupply&topicID=D_SE_0073155).

## Parameter GroupState

|  |  |
| --- | --- |
| Data type | Enumeration of DINT |
| Traceable | Yes |

The parameter GroupState indicates the operating state of the power supply and the connected drives.

For details, refer to Lexium LXM62 Power Supply Device Objects and Parameters guide, [GroupState](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62PowerSupply&topicID=D_SE_0073154).

## Parameter PowerSupplyCheck

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | No |

The parameter PowerSupplyCheck indicates the enable state of the power supply.

For details, refer to Lexium LXM62 Power Supply Device Objects and Parameters guide, [PowerSupplyCheck](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62PowerSupply&topicID=D_SE_0073149).

## Parameter PowerSupplyCheckSet

|  |  |
| --- | --- |
| Data type | BOOL |
| Traceable | No |

The parameter PowerSupplyCheckSet enables the power supply.

For details, refer to Lexium LXM62 Power Supply Device Objects and Parameters guide, [PowerSupplyCheckSet](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62PowerSupply&topicID=D_SE_0073147).
