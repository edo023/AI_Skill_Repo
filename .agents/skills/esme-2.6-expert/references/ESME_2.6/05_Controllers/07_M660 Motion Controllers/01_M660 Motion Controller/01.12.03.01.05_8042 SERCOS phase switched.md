# 8042 SERCOS phase switched

Diagnostic class: 1

The Sercos communication phase has changed. The message logger displays the new communication phase in Ext. diagnosis.

The table lists the values for DiagExtMsg for this diagnostic message:

| DiagExtMsg | Meaning |
| --- | --- |
| CP0 | The new communication phase is CP0. |
| CP1 | The new communication phase is CP1. |
| CP1/scan=(x) | The new communication phase is CP1. (x) Sercos devices have been found. |
| CP=1/xxx | Error xxx detected during transition to communication phase CP1. |
| CP=2 | The new communication phase is CP2. |
| CP2/use=(x) | The new communication phase is CP2. (x) Sercos devices have been assigned to device objects from the Devices tree. The remaining Sercos devices transition to communication phase CP4 without an assignment to device objects to enable ring healing and redundancy. |
| CP=2/xxx | Error xxx detected during phase change after phase 2. |
| CP=3 | The new communication phase is CP3. |
| CP=3/xxx | Error xxx detected during transition to communication phase CP3. |
| CP=4 | The new communication phase is CP4. |
| CP=4/xxx | Error xxx detected during transition to communication phase CP4. |
