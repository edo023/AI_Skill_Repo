# SercosDeviceDiagnostic - User Function

## Description

The user function SercosDeviceDiagnostic is available for the following device objects:

* [DRV\_Lexium62 (Lexium 62 Drive)](DRV_Lexium62Lexium62Drive-DeviceObj-0684F462.html), drive
* [DRV\_Lexium62Integrated (Lexium ILM62 Drive)](DRV_Lexium62IntegratedLexiumILM62Dr-0684ED06.html), drive
* [PSM\_Lexium62 (Lexium62 Power Supply)](PSM_Lexium62Lexium62PowerSupply-Dev-0790BF9D.html)

The user function SercosDeviceDiagnostic provides a parameter for reading the Sercos diagnostic number of a device.

## Parameter DiagnosticNumber

|  |  |
| --- | --- |
| Data type | DWORD |
| Traceable | Yes |

The parameter DiagnosticNumber lets you read the Sercos diagnostic number. The parameter contains the value of the Sercos parameter IDN S-0-0390.
