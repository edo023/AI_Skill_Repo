# 8960 Invalid mains voltage mode setting

Diagnostic class: 2

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | - | - |
| Power supply | D | E |

The value of the parameter MainsVoltageMode is not set in accordance to the infeed voltage.

This alert is given out if at the end of the pre-charge phase the DC bus voltage is greater than 560 V (by MainsVoltageMode=230 V) or lower than 460 V (by MainsVoltageMode=400 V).

| Cause | Solution |
| --- | --- |
| The setting of the parameter MainsVoltageMode is incorrect. | Verify and correct if necessary the parameters. |
| Interruption of one or more phases of the mains feed-in. | Verify and correct if necessary the wiring of the mains phases. |
