# 8107 Overcurrent

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | AD | AD |
| Power supply | AP | AP |

* The current through the power stage of the servo amplifier or in the DC bus of the power supply is too high. In contrast to the diagnostic messages '8120 Power stage overload' and '8146 DC bus overload' the device is switched off immediately if the maximum current is exceeded.
* An MH3 motor indicating a diagnostic message '[8122 Shutdown due to velocity limit](ShutdownDueToVelocityLimit-03BD55CB.html)' can also provoke the diagnostic message 8107 Overcurrent.

| For the motor/drive system: | |
| --- | --- |
| Cause | Solution |
| Detected wiring error: The shielding of the motor cable is not attached properly. | Verify and correct if necessary the wiring. |
| Servo amplifier: The current controller parameters are incorrect. | Verify the power control parameters Curr\_P\_Gain, Curr\_I\_Gain, and CurrFilter and set them to default if necessary. |
| Detected motor hardware error: The motor has a short-circuit. | Replace the motor. |
| Servo amplifier: The PowerStageFrequency is parameterized with 4 kHz. | * The parameter VoltageFeedForwardMode must be set to the value `Dynamic behavior of the control loop / 3`. * Verify the parameters Vel\_P\_Gain, Vel\_I\_Gain, and Pos\_P\_Gain and adjust them if necessary, to help avoid an unstable behavior of the controller. |
