# 8174 Incremental encoder output is not working

Diagnostic class: 2

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | D | D |
| Power supply | - | - |

There is an internal communication with the hardware to control the Incremental Encoder Output. With this communication the Incremental Encoder Output functionality is configured and controlled. This diagnostic message is triggered when an error with this communication is detected.

Incremental Encoder Output is only supported with Lexium 62 Advanced Plus.
