# DRV\_Lexium62 (Lexium 62 Drive) - Device Object in the Devices Tree

## Overview

The device object DRV\_Lexium62 (Lexium 62 Drive) represents a Lexium LXM62 drive. It operates as a Sercos device below the [SercosMaster](SercosMaster-DeviceObjectInTheDevic-0684DFF3.html).

For details on the drive, refer to the Lexium LXM62 user guide. For details on configuring the drive as a Sercos device, refer to [Sercos Configuration](SercosConfiguration-494BC22D.html).
