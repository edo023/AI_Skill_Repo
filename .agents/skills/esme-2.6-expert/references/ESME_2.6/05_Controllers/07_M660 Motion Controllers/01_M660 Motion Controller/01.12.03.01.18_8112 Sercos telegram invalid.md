# 8112 Sercos telegram invalid

Diagnostic class: 3

| Device reactions | Default reaction | Minimum reaction |
| --- | --- | --- |
| Drive | BD2 | BD2 |
| Power supply | BP2 | BP2 |

The Sercos device did not receive a valid telegram or valid data in the telegram over the real-time bus (Sercos) during the Sercos communication phases CP3 or CP4. The diagnostic message is triggered after two consecutive telegrams containing invalid data or after ten consecutive invalid telegrams.

NOTE: The diagnostic message 8112 Sercos telegram invalid is not sent for invalid data or telegrams to the system since the Sercos device can no longer report the diagnostic code to the controller over the Sercos bus. In this case, the diagnostic message '[8506 Sercos Master communication not possible](SercosMasterCommunicationNotPossibl-04738A23.html)' is reported in the Sercos master.

| Cause | Solution |
| --- | --- |
| Detected hardware error: Cable is disconnected or inoperable. | Verify and replace if necessary the cable. |
| Detected hardware error: Device is inoperable. | Replace the device. |
