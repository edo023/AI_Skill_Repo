# Ethernet\_2 (Ethernet2) - Device Objects in the Devices Tree

## Overview

The device object Ethernet\_2 (Ethernet2) is a communication device.

Ethernet\_2 (Ethernet2) represents the interface point for the physical connector **CN6** (port **LAN2**) of the controller.

Assignment to the operating system:

* VxWorks (MN660C), Windows (MN660P)

The device object is generated automatically when you create an empty project with an M660 controller.
