# Power Supply

The controller is supplied by an external 24 Vdc PELV power supply unit.

| Characteristic | Value |
| --- | --- |
| Nominal input voltage | 24 Vdc (-15 % ... +25 %) |
| Input current | During operation: refer to the nameplate of the controller. The value increases depending on installed additional equipment (for details, refer to [Input Power](ElectricalData-E70A295D.html)).  No operation: <15 mA |
| Maximum AC ripple | +/- 5 % |
| Maximum inrush current | <15 A for a maximum of 1/240 s |
| Immunity to input voltage interruption | 10 ms (for severity level PS2 as per IEC 61131-2) |
| Isolation between DC power supply and internal logic | 750 Vdc |
| Isolation between DC power supply and ground | 750 Vdc |
| Overcurrent protection | Yes |
| Reverse polarity protection | Yes |
| Surge protection | Yes |
| Immunity to voltage dips | Yes (slow and fast variation) |
| Undervoltage detection | Yes (voltage threshold for undervoltage detection is less than or equal to 19 V) |
| Input monitoring | Yes (voltage and current) |
| Maximum cable length | 30 m (98,43 ft) |
| Designation on housing | **CN1 / 24V DC** |
| Hardware indication of power state | Via LED **PWR** |

Refer to [Electrical Installation - Power Supply](PowerSupply-E7058A58.html) for details on installing the power supply.
