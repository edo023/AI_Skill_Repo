# DisplayPort

Controllers of type MN660P provide a DisplayPortTM. Refer to [Type Code](TypeCode-E6EB6EB6.html) for details on M660 controller types.

| Characteristic | Value |
| --- | --- |
| Connector type | 1 x full size DisplayPortTM connector |
| Voltage | 3.3 V |
| Current | 500 mA |
| Maximum cable length | 3 m (9.84 ft) |
| Specifications | DisplayPortTM V1.4  Embedded DisplayPortTM V1.4b |
| Resolutions | 1 display 5K at 120 Hz HDR or 8K at 60 Hz HDR  2 displays 8K at 60 Hz SDR or 5K at 60 Hz HDR  3 displays 4K at 60 Hz HDR  4 displays 4K at 60 Hz HDR |
| Assignment to operating system | Windows |
| Designation on housing | CN11 (**DP**) |

NOTE: The USB 3.2 Type-C port (CN10) provides DisplayPortTM Alternate Mode.
