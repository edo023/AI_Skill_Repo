## Overview

|  |  |

## Description

This structure of the library SercosMasterInterface contains the data for the functions [SM6xx.FC\_SercosGetConfiguration](../../../../../api/crossBook?lang=en-US&virtualBookName=SercosMasterM6xx&topicID=FC_SercosGetConfiguration_5313BF3D) and [SM6xx.FC\_SercosScanConfiguration](../../../../../api/crossBook?lang=en-US&virtualBookName=SercosMasterM6xx&topicID=FC_SercosScanConfiguration_53140DCA) of the library SercosMasterM6xx and of the structure [ST\_SercosConfiguration](ST_SercosConfiguration-53A948C8.html#ST_SercosConfiguration-53A948C8) of the library SercosMasterInterface.
