# ST\_SercosConfiguration

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |

## Description

This structure of the library SercosMasterInterface contains the data for the functions [SM6xx.FC\_SercosGetConfiguration](../../../../../api/crossBook?lang=en-US&virtualBookName=SercosMasterM6xx&topicID=FC_SercosGetConfiguration_5313BF3D) and [SM6xx.FC\_SercosScanConfiguration](../../../../../api/crossBook?lang=en-US&virtualBookName=SercosMasterM6xx&topicID=FC_SercosScanConfiguration_53140DCA) of the library SercosMasterM6xx.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| uiNumberOfEntries | UINT | Number of Sercos devices found on the Sercos bus |
| uiNumberOfPhysicalDevices | UINT | Number of Sercos devices found on the Sercos bus that are configured in the project |
| uiPhaseRunUpCount | UINT | Number of times phasing up has started |
| iCurrentPhase | INT | Present Sercos communication phase |
| astDevices | ARRAY [0..254] OF [ST\_SercosConfigurationDevice](ST_SercosConfigurationDevice-53A9CE21.html#ST_SercosConfigurationDevice-53A9CE21) | Array with descriptions of Sercos devices found on the Sercos bus |
