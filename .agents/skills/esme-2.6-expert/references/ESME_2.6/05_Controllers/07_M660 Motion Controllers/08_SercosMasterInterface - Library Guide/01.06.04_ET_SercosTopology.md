# ET\_SercosTopology

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | DINT |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_SercosTopology of the library SercosMasterInterface contains the Sercos topologies.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| noLink | 0 | No link detected at any of the ports, no topology established |
| lineP1 | 1 | Line topology at port 1 |
| lineP2 | 2 | Line topology at port 2 |
| doubleLine | 3 | Double line topology |
| ring | 4 | Ring topology |
| defectRing | 8 | Ring interrupted |
