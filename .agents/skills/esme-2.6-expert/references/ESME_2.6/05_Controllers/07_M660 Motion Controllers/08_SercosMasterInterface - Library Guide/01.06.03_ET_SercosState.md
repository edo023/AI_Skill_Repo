# ET\_SercosState

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | DINT |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_SercosState of the library SercosMasterInterface contains Sercos communication phases for the Sercos master and internal states for the library SercosMaster.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Init | -2 | Sercos state PhaseInit is the Sercos initialization state. It is modified after the first call of the real-time cycle. During regular operation, the enumeration does not contain this value. |
| NRT | -1 | Sercos state NRT, communication disabled |
| Phase0 | 0 | Sercos communication phase CP0 |
| Phase1 | 1 | Sercos communication phase CP1 |
| Phase2 | 2 | Sercos communication phase CP2 |
| Phase3 | 3 | Sercos communication phase CP3 |
| Phase4 | 4 | Sercos communication phase CP4 |
| Error | 11 | Sercos state PhaseError |
