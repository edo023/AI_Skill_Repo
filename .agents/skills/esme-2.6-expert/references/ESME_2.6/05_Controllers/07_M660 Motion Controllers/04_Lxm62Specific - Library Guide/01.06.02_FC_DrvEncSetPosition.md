# FC\_DrvEncSetPosition

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

This function of the library Lxm62Specific is provided for compatibility with LMC controller applications. It allows you to write a position value to an encoder.

The device is addressed with [DeviceTypeInterfaces.IF\_Device\_Type](../../../../../api/crossBook?lang=en-US&virtualBookName=DeviceTypeInterfaces&topicID=IF_Device_Type_56CAF679). For details on the function, refer to the System Interface library guide, [FC\_DrvEncSetPosition](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085093).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifDeviceType | [DeviceTypeInterfaces.IF\_Device\_Type](../../../../../api/crossBook?lang=en-US&virtualBookName=DeviceTypeInterfaces&topicID=IF_Device_Type_56CAF679) | Identifies the device. |
| i\_lrPosition | LREAL | Position value to be written to the encoder. |
