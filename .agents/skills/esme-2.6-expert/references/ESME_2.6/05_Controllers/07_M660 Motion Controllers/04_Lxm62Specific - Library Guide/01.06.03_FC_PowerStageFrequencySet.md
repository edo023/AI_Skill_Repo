# FC\_PowerStageFrequencySet

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

This function of the library Lxm62Specific allows you to set the power stage frequency of a Lexium 62 drive.

The device is addressed with [DeviceTypeInterfaces.IF\_Drive\_Type](../../../../../api/crossBook?lang=en-US&virtualBookName=DeviceTypeInterfaces&topicID=IF_Drive_Type_56CB4AE6). For details on the function, refer to the System Interface library guide, [FC\_PowerStageFrequencySet](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085255).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_ifDeviceType | [DeviceTypeInterfaces.IF\_Device\_Type](../../../../../api/crossBook?lang=en-US&virtualBookName=DeviceTypeInterfaces&topicID=IF_Device_Type_56CAF679) | Identifies the device. |
| i\_etFrequency | DDI.ET\_PowerStageFrequency | Value of enumeration DDI.ET\_PowerStageFrequency of the DriveDeviceInterface library. |
