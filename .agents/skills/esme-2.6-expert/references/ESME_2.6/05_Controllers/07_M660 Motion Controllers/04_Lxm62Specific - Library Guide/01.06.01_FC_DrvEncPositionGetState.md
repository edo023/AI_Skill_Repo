# FC\_DrvEncPositionGetState

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.0.0.0 |

## Description

This function of the library Lxm62Specific is provided for compatibility with LMC controller applications. It allows you to verify the state of a write procedure to an encoder with the function [FC\_DrvEncSetPosition](FC_DrvEncSetPosition-CFBB3D7E.html).

The device is addressed with [DeviceTypeInterfaces.IF\_Device\_Type](../../../../../api/crossBook?lang=en-US&virtualBookName=DeviceTypeInterfaces&topicID=IF_Device_Type_56CAF679). For details on the function, refer to the System Interface library guide, [FC\_DrvEncPositionGetState](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085091).
