# ET\_ConnectionID

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Data type: | UINT |
| Available as of: | V1.0.0.0 |

## Description

The enumeration ET\_ConnectionID of the library SercosMasterInterface contains the type of Sercos telegram (AT, MDT) which determines the direction of communication.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| MasterToSlave | 0 | MDT: Master Data Telegram sent by the master to the device. |
| SlaveToMaster | 1 | AT: Acknowledge Telegram sent by the device to the master in response to an MDT received from the master. |
