# Presentation of the Library SercosMasterInterface

## Description

The library SercosMasterInterface is an interface library for access to the user functions of Sercos masters.

## Characteristics of the Library

The following table indicates the characteristics of the library:

| Characteristics | Value |
| --- | --- |
| Library title | SercosMasterInterface |
| Company | Schneider Electric |
| Category | Application |
| Component | CoreLibraries |
| Default namespace | SMI |
| Language model attribute | [Qualified-access-only](../../../../../api/crossBook?lang=en-US&virtualBookName=Installer&topicID=D_SE_0081219) |
| Forward compatible library | Yes |

NOTE: For this library, qualified-access-only is set. Therefore, the POUs, data structures, enumerations, and constants have to be accessed using the namespace of the library. The default namespace of the library is SMI.
