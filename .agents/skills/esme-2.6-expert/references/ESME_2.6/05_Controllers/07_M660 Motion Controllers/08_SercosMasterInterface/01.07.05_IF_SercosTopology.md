# IF\_SercosTopology

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_SercosTopology of the library SercosMasterInterface allows you to access the user function SercosTopology in order to read information on the Sercos topology.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| LastDeviceP1 | STRING(80) | Get | Reads the name of the last physical Sercos device on port 1. |
| LastDeviceP2 | STRING(80) | Get | Reads the name of the last physical Sercos device on port 2. |
| ScannedDevices | DINT | Get | Reads the number of scanned physical devices on the Sercos bus. |
| Topology | [ET\_SercosTopology](ET_SercosTopology-53468F9E.html#ET_SercosTopology-53468F9E) | Get | Reads the Sercos topology. |
| UsedDevices | DINT | Get | Reads the number of scanned physical devices on the Sercos bus that are in use. |
