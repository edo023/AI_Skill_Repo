# IF\_SercosCycletimeConfig

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Extends: | UFMPLCI.IF\_UserFunctionBaseMPlc |
| Implements: | \_\_SYSTEM.IQueryInterface |
| Available as of: | 1.0.0.0 |

## Description

The user function interface IF\_SercosCycletimeConfig of the library SercosMasterInterface allows you to access the user function SercosCycletimeConfig of the Sercos master in order to configure the cycle time.

## Properties

| Name | Data type | Access | Description |
| --- | --- | --- | --- |
| ConfiguredCycletime | UDINT | Get/Set | Reads and writes the cycle time. |
| Cycletime | UDINT | Get | Reads the active cycle time. |
