# M241 Logic Controller Description

## Overview

The M241 Logic Controller has various powerful features and can service a wide range of applications.

Software configuration, programming, and commissioning is accomplished with the EcoStruxure Machine Expert software described in detail in the EcoStruxure Machine Expert [Programming Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SG_0026478), as well as the [M241 Logic Controller Programming Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=m241prg&topicID=D_SE_0032400).

## Programming Languages

The M241 Logic Controller is configured and programmed with the EcoStruxure Machine Expert software which supports the following IEC 61131-3 programming languages:

* IL: Instruction List
* ST: Structured Text
* FBD: Function Block Diagram
* SFC: Sequential Function Chart
* LD: Ladder Diagram

The EcoStruxure Machine Expert software can also be used to program the M241 Logic Controller using CFC (Continuous Function Chart) language.

## Power Supply

The power supply of the M241 Logic Controller is [24 Vdc](D-SE-0034419.html#D-SE-0034419) or [100...240 Vac](D-SE-0025320.html#D-SE-0025320).

## Real Time Clock

The M241 Logic Controller includes a [Real Time Clock (RTC) system](D-SE-0025710.html#D-SE-0025710).

## Run/Stop

The M241 Logic Controller can be operated externally by the following:

* A hardware [Run/Stop switch](D-SE-0034418.html#D-SE-0034418)
* A software command
* A [Run/Stop](D-SE-0034335.html#D-SE-0034335) operation by a dedicated digital input, defined in the software configuration
* The system variable PLC\_W in a [Relocation Table](../../../../../api/crossBook?lang=en-US&virtualBookName=m241prg&topicID=D_SE_0004337)
* The Web server [Maintenance: Run/Stop Controller Submenu](../../../../../api/crossBook?lang=en-US&virtualBookName=m241prg&topicID=MaintenanceRunStopControllerSubmenu_1D1D91B2)

## Memory

This table describes the different types of memory:

| Memory Type | Size | Use |
| --- | --- | --- |
| RAM | Refer to the table below | For the execution of the application and the firmware. |

| Reference | PV Value | RAM Size |
| --- | --- | --- |
| TM241•••• | < 20 | 64 Mbytes, of which 10 Mbytes are available for the application |
| ≥ 20 | 128 Mbytes, of which 20 Mbytes are available for the application |

## Embedded Inputs/Outputs

The following embedded I/O types are available, depending on the controller reference:

* Regular inputs
* Fast inputs associated with counters
* Regular sink/source transistor outputs
* Fast sink/source transistor outputs associated with pulse generators
* Relay outputs

## Removable Storage

The M241 Logic Controller includes an [embedded SD card slot](D-SE-0027501.html#D-SE-0027501).

The main uses of the SD card are:

* Initializing the controller with a new application
* Updating the controller firmware
* Applying post configuration files to the controller
* Applying recipes
* Receiving data logging files

## Embedded Communication Features

The following types of communication ports are available, depending on the controller reference:

* [CANopen Master](D-SE-0032568.html#D-SE-0032568)
* [Ethernet](D-SE-0034492.html#D-SE-0034492)
* [USB Mini-B](D-SE-0029629.html#D-SE-0029629)
* [Serial Line 1](D-SE-0025808.html#D-SE-0025808)
* [Serial Line 2](D-SE-0034493.html#D-SE-0034493)
