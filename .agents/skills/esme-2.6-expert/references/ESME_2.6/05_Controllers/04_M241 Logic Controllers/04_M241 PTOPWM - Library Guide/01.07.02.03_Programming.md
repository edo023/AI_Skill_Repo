# Programming

## Procedure

Follow these steps to program a Frequency Generator function block:

| Step | Action |
| --- | --- |
| 1 | Select the Libraries tab in the Software Catalog and click Libraries.  Select Controller > M241 > M241 PTOPWM > Frequency Generator > FrequencyGenerator\_M241 in the list; drag-and-drop the item onto the POU window. |
| 2 | Select the function block instance by clicking .  The Input Assistant screen appears. Select the global variable which references to the [added FreqGen](D-SE-0003568.html#D-SE-0003568) during the configuration and confirm.  NOTE: If the function block instance is not visible, verify if the frequency generator is configured. |
| 3 | The inputs/outputs are detailed in the [function block](D-SE-0003771.html#D-SE-0003771). |
