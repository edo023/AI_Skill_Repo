# FreqGen/PWM Naming Convention

## Definition

Frequency Generator and Pulse Width Modulation uses 1 fast physical output and up to 2 physical inputs.

In this document, use the following naming convention:

| Name | Description |
| --- | --- |
| SYNC | [Synchronization function](D-SE-0007138.html#D-SE-0007138). |
| EN | [Enable function](D-SE-0007138.html#D-SE-0007138). |
| IN\_SYNC | Physical input dedicated to the SYNC function. |
| IN\_EN | Physical input dedicated to the EN function. |
| OUT\_PWM | Physical output dedicated to the FreqGen or PWM. |
