# Home Offset

## Description

If the origin cannot be defined by switches with enough accuracy, it is possible to make the axis move to a specific position away from the origin switch. Home offset allows making a difference between mechanical origin and electrical origin.

Home offset is set in number of pulses (-2,147,483,648...2,147,483,647, default value 0). When set by configuration, the MC\_Home\_PTO command is executed first, and then the specified number of pulses is output at the home low velocity in the specified direction. The parameter is only effective during a reference movement without index pulse.

NOTE: The wait time between MC\_Home\_PTO command stop on origin switch and start of offset movement is fixed, set to 500 ms. The MC\_Home\_PTO command busy flag is only released after origin offset has been completed.
