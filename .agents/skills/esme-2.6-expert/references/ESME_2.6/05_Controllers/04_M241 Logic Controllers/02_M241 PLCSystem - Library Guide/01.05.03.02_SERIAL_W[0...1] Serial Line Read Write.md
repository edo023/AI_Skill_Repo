# SERIAL\_W[0...1]: Serial Line Read/Write System Variables

## Introduction

SERIAL\_W is an array of two SERIAL\_W\_STRUCT type. Each element of the array resets the SERIAL\_R system variables for the corresponding serial line to be reset.

For the M241 Logic Controller:

* Serial\_W[0] refers to the serial line 1
* Serial\_W[1] refers to the serial line 2

## Variable Structure

This table describes the parameters of the SERIAL\_W[0...1] system variable:

| %MW | Variable Name | Type | Comment |
| --- | --- | --- | --- |
| n/a | q\_wResetCounter | WORD | Transition from 0 to 1 resets all SERIAL\_R[0...1] counters.  To reset the counters again, it is necessary to write 0 to this variable before another transition from 0 to 1 can take place. |
| **n/a** means that there is no predefined %MW mapping for this system variable. | | | |
