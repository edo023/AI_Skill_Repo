# TM3\_ERR\_CODE: TM3 Expansion Module Detected Error Codes

## Enumerated Type Description

The TM3\_ERR\_CODE enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| TM3\_NO\_ERR | 00 hex | Last bus exchange with the expansion module was successful. |
| TM3\_ERR\_FAILED | 01 hex | Error detected due to the last bus exchange with the expansion module was unsuccessful. |
| TM3\_ERR\_PARAMETER | 02 hex | Parameter error detected in the last bus exchange with the module. |
| TM3\_ERR\_COK | 03 hex | Temporary or permanent hardware error detected on one of the TM3 expansion modules. |
| TM3\_ERR\_BUS | 04 hex | Bus error detected in the last bus exchange with the expansion module. |
