# PLC\_R\_TERMINAL\_PORT\_STATUS: Programming Port Connection Status Codes

## Enumerated Type Description

The PLC\_R\_TERMINAL\_PORT\_STATUS enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| TERMINAL\_NOT\_CONNECTED | 00 hex | No PC is connected to the programming port. |
| TERMINAL\_CONNECTION\_IN\_PROGRESS | 01 hex | Connection is in progress. |
| TERMINAL\_CONNECTED | 02 hex | PC is connected to the programming port. |
| TERMINAL\_ERROR | 0F hex | Error detected during connection. |
