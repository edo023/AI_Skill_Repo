# Configuration of the Capture on a Main Type

## Configuration Procedure

Follow this procedure to configure the capture function on a Main type:

| Step | Action |
| --- | --- |
| 1 | In the Devices tree, double-click MyController > Counters. |
| 2 | Set the value of the Counting function parameter to HSC Main Single Phase or HSC Main Dual Phase. |
| 3 | Select a value for the Capture > CAP input > Location. |
| 4 | Select a value for the Capture > CAP input > Bounce filter parameter to reduce the bounce effect on the input.  The filtering value determines the counter maximum frequency as shown in the [Bounce Filter table](D-SE-0003769.html#D-SE-0003769__D-SE-0003769.3). |
| 5 | Select a triggering mode for the Capture > Mode parameter:   * [Preset](D-SE-0007189.html#D-SE-0007189) (default value) * CAP Rising * CAP Falling * CAP Both |
