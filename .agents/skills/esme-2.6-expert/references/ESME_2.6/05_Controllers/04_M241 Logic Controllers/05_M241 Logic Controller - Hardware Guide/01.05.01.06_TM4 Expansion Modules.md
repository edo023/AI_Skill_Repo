# TM4 Expansion Modules

## Introduction

The range of TM4 expansion modules includes communication modules.

For more information, refer to the [TM4 Expansion Modules Hardware Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=tm4hw&topicID=D_SE_0036328).

## TM4 Expansion Modules

The following table shows the TM4 expansion module features:

| Module reference | Type | Terminal type |
| --- | --- | --- |
| [TM4ES4](../../../../../api/crossBook?lang=en-US&virtualBookName=tm4hw&topicID=D_SE_0036279) | Ethernet communication | 4 RJ45 connectors  1 screw for functional ground connection |
| [TM4PDPS1](../../../../../api/crossBook?lang=en-US&virtualBookName=tm4hw&topicID=D_SE_0036280) | PROFIBUS DP slave communication | 1 SUB-D 9 pins female connector  1 screw for functional ground connection |
| NOTE: The TM4ES4 module has two applications: expansion or standalone. For more information, refer to TM4 Compatibility. | | |
