# Post Configuration Example

## Post Configuration File Example for TM241CE40T with TM4ES4

```
# TM241CE40T / FTP Encryption
```

```
# 1=encryption enforced, 0 otherwise
```

```
.param[1106] = 1
```

```
# TM241CE40T / OPCUA server name
```

```
# Only ASCII letters, digits, '-' and '_', 30 char max
```

```
.param[1204] = ''
```

```
# TM241CE40T / COM_Bus / TM4ES4 / IPAddress
```

```
# Ethernet IP address
```

```
id[45000].pos[7].id[45300].pos[0].id[45111].param[0] = [188, 5, 1, 10]
```
