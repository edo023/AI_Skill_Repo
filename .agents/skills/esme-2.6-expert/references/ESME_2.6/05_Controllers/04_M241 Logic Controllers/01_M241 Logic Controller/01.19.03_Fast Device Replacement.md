# Fast Device Replacement

## Overview

The Fast Device Replacement (FDR) helps facilitate replacing and reconfiguring a network device. This function is available on the Ethernet\_1 port of the M241 Logic Controller.

For more information, refer to [Slave Device Replacement with FDR](../../../../../api/crossBook?lang=en-US&virtualBookName=ESMEModbusTCP&topicID=D_SE_0056552).
