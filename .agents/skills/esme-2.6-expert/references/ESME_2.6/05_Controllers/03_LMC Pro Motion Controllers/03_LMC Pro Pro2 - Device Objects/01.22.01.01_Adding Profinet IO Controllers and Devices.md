# Adding Profinet IO Controllers and Devices

## Prerequisites

To use a PacDrive LMC controller as a Profinet IO device with another  PacDrive LMC controller which is configured as a Profinet IO controller, you have to install the appropriate device description (Tools > Device Repository).

To access the Profinet XML file (*GSDML-Vx.xx-Schneider Electric-PacDrive-xxxxxxxx.xml*), select Tools > External Tools > Explore Fieldbus Description Files.

Refer to Online Help Overview and Menu Commands\Tools Menu Commands\Repository-Related Commands\Device Repository...

## Adding a Profinet IO Controller

| Step | Action |
| --- | --- |
| 1 | Open your project in the Logic Builder. |
| 2 | Right-click on the PacDrive controller and select Add Device.  **Result**: The Add Device dialog box is displayed. |
|  |
| 3 | Select Fieldbusses > Profinet IO  > Profinet IO Master > PROFINETIO-Controller (Schneider Electric). |
| 4 | Confirm with Add Device and Close the dialog box. |
|  |
| 5 | Double-click the PNC\_Controller (PROFINETIO-Controller) in the Devices tree to open its device editor. |
| 6 | In the NetX Configuration tab, select the appropriate Connector. Available options are Front / 1, OptRight / 2, OptLeft / 3. |
| 7 | In the General tab, set the appropriate IP Parameters.  Activate connection monitoring (Watchdog) by selecting the checkbox. |

The controller is unable to detect an interruption of the network link if connection monitoring is not active.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  Ensure that connection monitoring (Watchdog) is enabled.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

Set the shortest, practical monitoring time cycles to detect communication interruptions as quickly as possible.

## Adding a Profinet IO Device

| Step | Action |
| --- | --- |
| 1 | Right-click on the PNC\_Controller (PROFINETIO-Controller) and select Add Device.  **Result**: The Add Device dialog box is displayed. |
|  |
| 2 | Select Fieldbusses > Profinet IO > Profinet IO Slave > Schneider\_Electric\_PN Device. |
| 3 | Confirm with Add Device and Close the dialog box. |
|  |

| --- | --- |

| 2 | Login. |
| 3 | Right-click on the PNC\_Controller (PROFINETIO-Controller) and select Scan For Devices. |

| 5 | Copy Station Name and IP Address of the Schneider\_Electric\_PN Device to the scanned device. |
