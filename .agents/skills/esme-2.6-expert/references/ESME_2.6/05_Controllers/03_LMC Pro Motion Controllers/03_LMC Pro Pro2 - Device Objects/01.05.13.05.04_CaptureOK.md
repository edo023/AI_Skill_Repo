# CaptureOK

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  OnBoard I/O, DIO8 module  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

## Functional Description

Displays the status of the Touchprobe event.

A valid Touchprobe event has occurred and the result has been provided in the parameter [CaptureValue](D-SE-0092819.html#D-SE-0092819). The value is reset to "no / 0" when *FC\_TPEdge()* is called again.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | The result of the Touchprobe event has been reset. |
| yes / 1 | BOOL | The result of the Touchprobe event is provided in the parameter CaptureValue. |
