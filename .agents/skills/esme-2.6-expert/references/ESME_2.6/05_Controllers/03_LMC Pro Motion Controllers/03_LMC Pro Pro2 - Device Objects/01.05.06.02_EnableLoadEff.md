# EnableLoadEff

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

This parameter starts and stops the effective CPU load measurement.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | Effective CPU load measurement stopped |
| on / 1 | BOOL | Effective CPU load measurement started |
