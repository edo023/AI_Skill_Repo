# Responding to Noncyclical Enquriries of the Master

## General

To be able to respond to noncyclical read and write enquiries or alarm acknowledgment of the PROFIBUS master, a function block has to be created which implements the corresponding interface for this task.

In the method, which is prescribed by the respective interface, it is possible to create a program code to be executed upon receiving the respective noncyclical enquiries.

An example program for the reaction to a noncyclical read enquiry can be found [here](D-SE-0079652.html#D-SE-0079652).
