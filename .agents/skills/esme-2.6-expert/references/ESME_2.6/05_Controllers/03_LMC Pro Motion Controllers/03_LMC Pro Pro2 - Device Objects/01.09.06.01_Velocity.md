# Velocity

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Virt. Master Encoder |
| Traceable | No |

Displays the actual velocity of the virtual master encoder in units/s.

A unit is defined using the parameter FeedConstant.

RefVelocity specifies the reference velocity of the virtual master encoder.

If Enable is "off / 0", the velocity of the encoder is set to 0.

EIO0000002335.11
