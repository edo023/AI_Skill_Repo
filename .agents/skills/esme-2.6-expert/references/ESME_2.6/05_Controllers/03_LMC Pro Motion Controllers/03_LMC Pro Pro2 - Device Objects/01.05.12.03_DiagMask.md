# DiagMask

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C  BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | yes test |

## Functional Description

Displays the bit field for enabling the diagnosis message "Wiring errors" (error code 8788 "Wiring error") with ext. diagnosis message "Overload" or "Openload". Each bit is assigned to an output corresponding to the bit number.

| Bit | Meaning |
| --- | --- |
| 0 | The output is not monitored. |
| 1 | Controllers PacDrive LMC Pro, PacDrive LMC Eco.  The output is monitored and a diagnostic message is issued if monitoring with the parameters  or OverloadDiagMsk is also enabled. |
