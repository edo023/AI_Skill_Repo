# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam switch |
| Traceable | Yes |

## Functional Description

Enables / disables the processing of this Cam switch.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The Cam switch is not enabled. |
| on / 1 | BOOL | The Cam switch is enabled. |
