# ConfiguredApplicationType

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Traceable | No |

## Functional Description

If IdentificationMode = ApplicationType / 3, the value of ConfiguredApplicationType is used for identifying the device in the Sercos network (see parameter [IdentificationMode](IdentificationMode-2F96D150.html#IdentificationMode-2F96D150)). It is applied in the Sercos phase change from 1 to 2.

| Data type | Meaning |
| --- | --- |
| STRING [40] | The configured (logical) application type for identifying the device in the Sercos network. |
