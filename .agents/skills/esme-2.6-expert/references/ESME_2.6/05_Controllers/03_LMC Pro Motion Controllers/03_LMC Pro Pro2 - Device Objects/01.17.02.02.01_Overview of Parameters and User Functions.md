# Overview of Parameters and User Functions

## General

In the Devices tree, Edge I/O Modules supporting expert functions can be inserted as a sub-object under the Edge I/O Network Interface Module object.

To display and set an expert function of the module, double-click the corresponding Edge I/O Module in the Devices tree.

In the Edge I/O NTS Editor tab, click Add a Function to add and activate an expert function. Configure the expert function by assigning an input to the parameter Input Location. For more details on the expert function refer to [Modicon Edge I/O NTS Motion Expert Modules User Guide, Timestamp Input Mode Principle Description](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Exp_UG&topicID=TimestampInputModePrincipleDesc_13B26307).

In the Devices tree, select the Edge I/O Module and right-click Add device ....

In the Add Device window, select the device Touch Probe Input and click Add Device.

To display the parameters and user functions, double-click the corresponding Touch Probe Input in the Devices tree.

## Supported Hardware and Firmware

* NTSEDM0822 - Edge I/O Oversampled and Timestamped Mixed Module, 4 Inputs, 4 Outputs
* NTSEDT0800 - Edge I/O Timestamped Discrete Input Module, 8 Inputs
* PacDrive LMC controller Eco/Pro/Pro2 with firmware version as of V1.82

## Parameter Group Common

The parameter group Common contains generic parameters of the corresponding device.

## User Function ExpertFunctionInformation

The user function ExpertFunctionInformation allows you to read expert function-specific settings such as the type or the name of the expert function.

## User Function TouchprobeData

The user function TouchprobeData allows you to read and set touch probe-specific information and settings on a touch probe input.

## Using the User Function Parameters in the Application

You can access a parameter in the application using the following syntax:

```
<Device>.<UserFunction>.<Parameter>
```

**Example:**

The following command sets the parameter CaptureEdgeSet of the user function TouchprobeData to the value 0 (falling edge):

```
EDGENIM_1.TouchprobeData.CaptureEdgeSet := 0;
```
