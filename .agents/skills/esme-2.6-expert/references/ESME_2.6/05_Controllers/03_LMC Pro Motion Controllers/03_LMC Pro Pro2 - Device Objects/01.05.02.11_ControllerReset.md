# ControllerReset

## General

|  |  |
| --- | --- |
| Type | ED |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

To distinguish between a program reset and a controller reset. This parameter is set to TRUE (1) when the controller is reset. The parameter can be set to FALSE (0) using an application program. In this way, you can distinguish between a program reset and a controller reset.

| Value | Data type | Meaning |
| --- | --- | --- |
| false / 0 | BOOL | Program reset |
| true / 1 | BOOL | Controller reset |
