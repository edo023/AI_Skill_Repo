# Position

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Incremental Encoder Input |
| Traceable | No |

## Functional Description

Displays the current encoder position. The value range runs from 0 to FeedConstant.

With negative direction of rotation, clockwise rotation causes the value range to run in decreasing order.

At system startup the current encoder position is set to 0.

If the incremental encoder is connected to a bus terminal, the Position is not 0 after the controller has been reset. 0 only is displayed if the is also reset.

Use the parameter State to check the availability or validity of the encoder position data.

NOTE: Here the parameter Position is an "auxiliary parameter" which is above all useful in the start-up phase.
