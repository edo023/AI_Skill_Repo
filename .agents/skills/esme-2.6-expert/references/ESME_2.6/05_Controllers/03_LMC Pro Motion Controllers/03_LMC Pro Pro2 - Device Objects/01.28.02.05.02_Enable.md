# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | C2C Data Input, C2C Data output,  C2C Encoder input, C2C Encoder Output |
| Traceable | Yes |

## Functional Description

Enables or disables the C2C data object.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The data object is disabled. |
| on / 0 | BOOL | The data object is enabled. |
