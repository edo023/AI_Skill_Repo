# IP\_Gateway

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the gateway IP address. The parameter is a display parameter only. The IP address of the gateway is saved in file sysc3.cfg on the controller. This is the address of a local IP router that is located on the same network as the controller. The IP router passes the data to destinations outside of the local network. The numbers from 0 to 255 are permitted in these fields. A gateway is a router that connects separate IP network segments with each other. For example, a network segment often requires a gateway to connect with another network segment, a WAN (Wide Area network), or with the Internet.

NOTE: The string cannot begin with zero and cannot contain any spaces. The controller must be reset to accept the parameter. Confirm with the reset button.

## Example

"190.205.0.1"
