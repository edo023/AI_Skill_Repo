# Overview of Parameters and User Functions

## General

In the Devices tree, Edge I/O Modules supporting expert functions can be inserted as a sub-object under the Edge I/O Network Interface Module object.

To display and set an expert function of the module, double-click the corresponding Edge I/O Module in the Devices tree.

In the Edge I/O NTS Editor tab, click Add a Function to add and activate an expert function. Configure the expert function by assigning an input to the parameter Channel Location. For more details on the expert function refer to [Modicon Edge I/O NTS Motion Expert Modules User Guide, Incremental Mode Principle Description](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Exp_UG&topicID=IncrementalModePrincipleDescription_076579F2).

In the Devices tree, select the Edge I/O Module and right-click Add device ....

In the Add Device window, select the device Incremental Encoder Input and click Add Device.

To display the parameters and user functions, double-click the corresponding Incremental Encoder Input in the Devices tree.

## Supported Hardware and Firmware

* NTSECS0121 - Edge I/O Cam Switch Output Module, 1 RS422 Incremental 1 MHz, SSI 500 kHz, or BiSS-C, 2 Inputs, 8 Outputs
* NTSEHE0221 - Edge I/O Encoder Input Module, 2 RS422 Incremental 1 MHz, SSI 500 kHz, or BiSS-C, 4 Inputs, 8 Outputs
* NTSEME0101 - Edge I/O Encoder Input Module, 1 RS422 Incremental 1 MHz, SSI 500 kHz, or BiSS-C, 2 Inputs
* PacDrive LMC controller Eco/Pro/Pro2 with firmware version as of V1.84
