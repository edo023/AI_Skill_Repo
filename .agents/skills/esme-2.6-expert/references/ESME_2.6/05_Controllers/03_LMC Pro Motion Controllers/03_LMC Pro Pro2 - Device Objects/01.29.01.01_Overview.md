# Overview

## Description

During commissioning, the devices connected to the Sercos bus are mapped to the Sercos objects in the PLC Configuration. The **Device addressing** [editor](D-SE-0088091.html#D-SE-0088091) helps you to map devices by displaying all Sercos axes and power supplies in an overview which enables you to map, apply and modify values.

**NOTE:** For connections to the Sercos bus, use Schneider Electric slaves only (e.g. Lexium LXM62 Drive, Lexium LXM52 Drive, and Lexium ILM62 Drive Module) in order to be able to use the entire range of functions of the PacDrive system.

**NOTE:** The functions of the editor Device addressing are only available in offline mode.
