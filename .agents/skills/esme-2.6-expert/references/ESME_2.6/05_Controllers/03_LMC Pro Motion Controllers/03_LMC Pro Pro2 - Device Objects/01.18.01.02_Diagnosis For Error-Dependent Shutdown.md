# Diagnosis For Error-Dependent Shutdown

## Description

Controllers: PacDrive LMC x00C

The hardware provides an internal error logger that logs all the causes for error-dependent shutdowns. The next time the controller is started up the error logger is analyzed and the cause of the error can be output in the message logger. The logging and analysis of the error logger takes place whether a UPS object was designed in the PLC configuration.

Message 1: General diagnostic message

Message 2: MainPower message

Message 3: Debug message

Messages 1 and 2 always appear in combination with Message 3, which can be hidden using a special message filter in the standard settings. It can be enabled later (filter type 15). Message 3 contains more detailed information than messages 1 or 2.

NOTE: The detailed information in message 3 is only of interest for severe shutdown causes. In this case, contact your Schneider Electric responsible.

## Example: Temperature Error

The controller identified a temperature error. You can see the UPS state changes in the message logger. A temperature that is too high is identified (modify in state to "temperature monitoring active / 6"). After 70 s the status changes to "system shutdown / 4." The tasks are terminated, the UPS is switched off, and finally the system is shut down.

When the system is started up the next time, it detects that in the last cycle there was an error-dependent shutdown and analyzes the error entry. It determines that it was related to a temperature error. The message with diagnostic Code = 8055 "Controller Message HW-Monitoring" provides more information on the error type.

## Example: Supply Voltage Outage

The UPS identifies the supply voltage outage and changes the state to “UPS active / 3.” After the time of 10 s configured in the parameter Delay has elapsed, the supply voltage is still missing. The status changes to “System Shutdown / 4” and the tasks are terminated, the UPS is switched off, and the system is finally shut down.

When the system is started up the next time, it detects that in the last cycle there was an error-dependent shutdown and analyzes the error entry. It determines that it was a supply voltage error. For completeness, the diagnostic message with Diag. Code = 8055 "Controller message HW monitor" will also be output here.
