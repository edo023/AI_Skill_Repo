# IP\_SubNetMask

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Is used to display the subnet mask number.

This number and the [IP address](D-SE-0071910.html#D-SE-0071910) specify the network segment to which the controller belongs. A subnet mask number is a 32-bit number that consists of four numbers from 0 to 255 that are separated by dots. Generally, only the values 0 and 255 (for example, 255.255.255.0) are used for standard subnet mask numbers. However, other numeric values are possible. The subnet mask is stored in file sysc3.cfg.

## Example

"255.255.0.0"
