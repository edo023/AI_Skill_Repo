# OnPosition

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Can switch |
| Traceable | Yes |

## Functional Description

Specifies the switch-on position (Posmode) and the start of the activation window (Pulsemode) for this Cam switch. The operating mode (Posmode or Pulsemode) is set using the parameter Mode.

EIO0000002335.11
