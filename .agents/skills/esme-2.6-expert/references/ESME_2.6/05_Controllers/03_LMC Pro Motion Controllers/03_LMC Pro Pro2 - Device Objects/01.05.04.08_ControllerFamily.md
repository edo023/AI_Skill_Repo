# ControllerFamily

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

The parameter `ControllerFamily` of type STRING(40) displays the controller family.

| Value | Controller Family | ControllerModel |
| --- | --- | --- |
| Eco | PacDrive LMC Eco | LMC 100C/101C/201C |
| Pro | PacDrive LMC Pro | LMC 300C/400C/600C |
| Pro2 | PacDrive LMC Pro2 | PacDrive LMC Pro2 |
