# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Encoder network (Synchronization encoder output) |
| Traceable | Yes |

## Functional Description

Enables the synchronization encoder output.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The Synchronization encoder output is disabled. If the Synchronization encoder output is switched from "on / 1" to "off / 0" while the encoder network is enabled, the diagnostic message 8333 "EncNet receiving data not possible". Is sent to the Synchronization encoder inputs. |
| on / 1 | BOOL | The Synchronization encoder output is enabled. Since all the controllers in the encoder network must be running synchronously and error-free for the encoder signals to be distributed, the parameter State displays the state of the Synchronization encoder output. If the value in the State parameter is "active / 5", an encoder signal is distributed. |
