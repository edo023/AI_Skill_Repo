# ConfiguredSercosAddress

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Traceable | Yes |

## Functional Description

If IdentificationMode = SercosAddress / 4, the value of ConfiguredSercosAddress is used for identifying the device in the Sercos network (see parameter [IdentificationMode](IdentificationMode-2F96D150.html#IdentificationMode-2F96D150)). It is applied in the Sercos phase change from 1 to 2.

| Value range | Data type | Meaning |
| --- | --- | --- |
| (1…511) | UINT | The configured (logical) Sercos address for identifying the device in the Sercos network. |
