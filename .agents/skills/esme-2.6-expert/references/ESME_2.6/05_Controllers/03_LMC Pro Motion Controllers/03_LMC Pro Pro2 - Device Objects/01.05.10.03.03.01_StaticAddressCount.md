# StaticAddressCount

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

Using this parameter, you can reserve some addresses for the manual configuration of slave devices.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0 (default) | UINT | Number of addresses that shall be reserved for the manual configuration of slave devices. Also see parameter IPConfigMode.  For this purpose, you have to set the parameter to a value greater than or equal to the number of slaves which are to be configured manually.  Access type: read/write |

While a value for this parameter is being entered, it is checked whether the entered value lies within the value range of 0-1007.

NOTE: If a Safety Logic Controller (SLC) is added to a Sercos III bus, the current value of StaticAddressCount automatically increases by 1. Thus, the value of StaticAddressCount is automatically decreased by 1 if a Safety Logic Controller was deleted from a Sercos III bus.
