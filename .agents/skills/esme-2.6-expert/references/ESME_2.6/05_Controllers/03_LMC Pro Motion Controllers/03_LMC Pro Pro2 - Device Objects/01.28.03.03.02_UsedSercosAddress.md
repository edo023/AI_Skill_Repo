# UsedSercosAddress

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | C2C\_Slave |
| Traceable | Yes |

## Functional Description

Displays the Sercos address of the device which has been set via service channel or has been configured in the parameter ConfiguredSercosAddress of the device.
