# General

## Description

In order to receive an encoder signal that is distributed by a Synchronization encoder output via the encoder network, the Synchronization encoder input is used. This object receives the encoder data and then, after a defined delay (DataDelay), outputs the encoder data as a velocity signal.

Using the function FC\_SetMasterEncoder(LogEncId, MasterId), the Synchronization encoder input can be set as the velocity source (master encoder) of a Log. encoder. Here, LogEncId is the logical address of the Log. encoder (L\_ENC) and MasterId is the logical address of the Synchronization encoder input to be connected.
