# Resolution

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Incremental Encoder Input |
| Traceable | No |

## Functional Description

Specifies the resolution of the connected encoder (see encoder nameplate).

NOTE: The incremental encoder input functions internally to increase resolution by quadrupling the impulse. The system automatically takes this into account.

NOTE: Corruption of the velocity signal during resolution change. The [velocity signal](D-SE-0075788.html#D-SE-0075788) is briefly corrupted, which can cause other errors. Only change the parameter value at encoder standstill and when the program is inactive.
