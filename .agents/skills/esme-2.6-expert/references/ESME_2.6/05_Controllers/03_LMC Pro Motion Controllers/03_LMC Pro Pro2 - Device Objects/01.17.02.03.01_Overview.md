# Overview

## General

In the Devices tree, Edge I/O Modules supporting expert functions can be inserted as a sub-object under the Edge I/O Network Interface Module object.

To display and set an expert function of the module, double-click the corresponding Edge I/O Module in the Devices tree.

In the Edge I/O NTS Editor tab, click Add a Function to add and activate an expert function. Configure the expert function by assigning an input to the parameter Input Location. For more details on the expert function refer to [Modicon Edge I/O NTS Motion Expert Modules User Guide, Timestamp Output Mode Principle Description](../../../../../api/crossBook?lang=en-US&virtualBookName=EdgeIO_NTS_Exp_UG&topicID=TimestampOutputModePrincipleDes_13B29FEC).

## Supported Hardware and Firmware

* NTSEDT0810 - Edge I/O Timestamped Discrete Output Module, 8 Outputs
* PacDrive LMC controller Eco/Pro/Pro2 with firmware version as of V1.84
