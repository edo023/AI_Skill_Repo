# Status

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  OnBoard I/O, DIO8 module,  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

## Functional Description

The parameter represents the status of the output.

| Value | Data type | Meaning |
| --- | --- | --- |
| default / 0 | DINT | Default state. |
| openload / 1 | DINT | No load available. |
| overload / 2 | DINT | There is a short circuit. |
