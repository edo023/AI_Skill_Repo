# Bit8\_15

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C |
| Traceable | No |

## Functional Description

Displays the values of inputs 8 to 15. Each bit is assigned to an input corresponding to the bit number.

Assigning bits to inputs Bit8\_15

|  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| Parameter bit number | Bit 15 | Bit 14 | Bit 13 | Bit 12 | Bit 11 | Bit 10 | Bit 9 | Bit 8 |
| Digital input(1) | DI\_15 | DI\_14 | DI\_13 | DI\_12 | DI\_11 | DI\_10 | DI\_9 | DI\_8 |
| **1.** affected parameter: Value | | | | | | | | |

## Example

Bit8\_15 = 2#1000

The [value of the inputs](D-SE-0075304.html#D-SE-0075304) 11 is "H / TRUE;" inputs 8 to 10 and 12 to 15 are "L / FALSE".
