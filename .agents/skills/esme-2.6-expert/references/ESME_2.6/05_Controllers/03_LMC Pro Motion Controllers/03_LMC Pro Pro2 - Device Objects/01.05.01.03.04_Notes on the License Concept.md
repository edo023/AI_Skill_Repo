# Notes on the License Concept

## General

* Memory cards that have license points should be only modified with the appropriate tools. Formatting memory cards and keys outside of the appropriate tools causes the existing license points to be lost. License points are retained when the Controller Assistant is used to replace the firmware. The access through the file system (such as FTP) of the controller (logical access) can be performed.
* The request keys and authorization keys are bound to a specific memory card. Therefore you cannot use an authorization code to put license points on another memory card.
* You can use one authorization key with the corresponding memory card as often as you like. If the points are lost due to improper handling of the memory card (such as formatting), they can be transferred again using the authorization key.
* You cannot increase the available number of points by entering the authorization key again.
* It is not possible to transfer points from one memory card to another.
* Purchased license points cannot be returned.
