# PositionType

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam Track |
| Traceable | Yes |

## Functional Description

Sets the position type supplied by the PositionSource.

| Value | Data type | Meaning |
| --- | --- | --- |
| Drive\_MechPosition / 0 | DINT | Position type *MechPosition* of an axis |
| Drive\_MechRefPosition / 1 | DINT | Position type *MechRefPosition* of an axis |
| Drive\_ShaftMechPosition / 2 | DINT | Position type *ShaftMechPosition* of an axis |
| Drive\_ShaftMechRefPosition / 3 | DINT | Position type *ShaftMechRefPosition* of an axis |
| LEnc\_Position / 4 | DINT | Position type Position of a logical encoder |
| LEnc\_PositionWithoutDelay / 5 | DINT | Position type PositionWithoutDelay of a logical encoder |
