# Configuration of the EtherNet/IP Adapter

## Inserting the EtherNet/IP Adapter Into the PLC Configuration

To be able to use an EtherNet/IP adapter, it has to be entered into the PLC Configuration of the PacDrive Controller.

Project is opened in Logic Builder.

| Step | Action |
| --- | --- |
| 1 | Select the PacDrive Controller |
| 2 | Right-click > Append device  Result: The dialog box Append device appears. |
| 3 | Select device name (for example EtherNet/IP adapter) |
| 4 | OK button  Result: The EtherNet/IP adapter has been entered into the PLC Configuration. |

## Configuring the EtherNet/IP Adapter

| Step | Action |
| --- | --- |
| 1 | Select the EtherNet/IP adapter in the PLC Configuration |
| 2 | Double-click on the selected module or right-click and select Edit object.  Result: The device dialog box of the EtherNet/IP adapter with the EtherNet/IP Configuration register is displayed. |
