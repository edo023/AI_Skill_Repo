# Introduction

## Overview

Bus View is a part of the Sercos device editor. Bus View is available for PacDrive controllers as of EcoStruxure Machine Expert 2.0.

Bus View is a graphical tool in Logic Builder designed to support you in diagnostics of the Sercos bus. It facilitates commissioning by helping you to detect incorrect configurations and locating errors in the wiring between the Sercos bus devices.

Bus View does not substitute but extends the Diagnostics tool of Machine Expert Logic Builder (accessible via Tools > External Tools >  Collect Diagnostics data). The Diagnostics tool lets you save a snapshot of the controller data.
