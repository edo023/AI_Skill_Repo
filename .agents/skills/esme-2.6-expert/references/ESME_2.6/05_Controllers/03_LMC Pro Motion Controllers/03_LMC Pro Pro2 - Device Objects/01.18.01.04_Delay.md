# Delay

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Uninterruptible power supply |
| Traceable | Yes |

## Functional Description

Configures the maximum duration of the supply voltage interruption to be buffered (shut down delay).

NOTE: If the parameter value is changed in the PacDrive LMC x00C controller, then this value will first be written in the non-volatile memory; after, the parameter is updated. This results in a delay of about 2 s until the parameter value has been updated. Note that the non-volatile memory allows a maximum of 100,000 write cycles. The system minimizes the number of write cycles by writing the value only when it changes.

Up to now, the Delay parameter has been used only to configure the time between supply voltage outage and UPS shutdown.

The parameter also defines the time between detection of a temperature that is too high and the shutdown resulting from this. In the case of a temperature error, this time is limited to max. 60 s.
