# Delay

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Log. encoder |
| Traceable | Yes |

## Functional Description

Defines the dead time. The logical encoder calculates a position offset from the current velocity of the coupled master encoder and the dead time. The offset is added to the position each time. For an axis which is a master axis, it for example makes sense to define the [Sercos cycle time](D-SE-0073362.html#D-SE-0073362) As Delay, because the reference value of the previous cycle is used as the master velocity respectively.
