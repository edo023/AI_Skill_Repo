# Value

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  Analog input |
| Traceable | No |

## Functional Description

The input can be connected as a voltage input or a current input.

**Voltage input**

Value = 0 -> -10 V

Value = 4095 -> +10 V

**Current input**

Value = 0 -> -20 mA

Value = 4095 -> +20 mA

NOTE: Refer also to the chapter on electrical connections in the PacDrive controller operating instructions.

NOTE: The analog inputs only provide valid values in SERCOS phase 4.
