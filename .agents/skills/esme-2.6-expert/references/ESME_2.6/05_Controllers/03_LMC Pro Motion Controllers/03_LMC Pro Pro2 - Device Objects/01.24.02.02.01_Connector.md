# Connector

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave,  Field bus EtherNet/IP adapter |
| Traceable | Yes |

## Functional Description

The controllers PacDrive LMC x00C can be equipped with optional modules on the bottom of the controller.

The parameter Connector determines which connector is to be used on the controller.

| Value | Data type | Meaning |
| --- | --- | --- |
| Front /1 | INT | Default value. The connector at the front of the device is to be used. |
| OptRight / 2 | INT | The connector of the right optional module on the bottom of the device is to be used. If the optional module is not available, the diagnostic message 8720 "no module detected" is issued. |
| OptLeft / 3 | INT | The connector of the left optional module on the bottom side of the device is to be used. If the optional module is not available, the diagnostic message 8720 "no module detected" is issued. |
