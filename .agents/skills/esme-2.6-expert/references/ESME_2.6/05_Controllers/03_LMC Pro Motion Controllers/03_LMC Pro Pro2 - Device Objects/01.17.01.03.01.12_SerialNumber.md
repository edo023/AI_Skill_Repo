# SerialNumber

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Traceable | No |

## Functional Description

The parameter represents the serial number of the device.

If the device in the controller configuration is virtual (non-real), then the parameter value is DRIVE\_VIRTUAL.

| Data type | Meaning |
| --- | --- |
| STRING [80] | The serial number of the device. |

NOTE: The parameter value is read via Sercos in phase 1. The value is not updated in greater phases.
