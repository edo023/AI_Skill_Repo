# Overview of Parameters and User Functions

## General

In the Devices tree, the Edge I/O Network Interface Module object can be inserted as a sub-object under the Sercos Master object. Additional Edge I/O Modules objects are inserted as sub-objects under the Edge I/O Network Interface Module object.

To display and set the parameters, double-click the corresponding device in the Devices tree.

In the Parameters tab of the editor, expand the Common group to see the common parameters. Expand the UserFunctions group to see the user functions.

## Supported Hardware and Firmware

* Edge I/O Network Interface Module NTSNSC1200
* PacDrive LMC controller Eco/Pro/Pro2 with firmware version as of V1.82

## Parameter Group Common

The parameter group Common contains generic parameters of the corresponding device.

## User Function SercosIdentification

The user function SercosIdentification allows you to read Sercos-specific settings such as the parameter Working State or the parameter Topology Address.

In addition, the user function allows you to specify Sercos-related settings such as the parameter Working Mode or the parameter Identification Mode.

## User Function Diagnostic

The user function Diagnostic displays diagnostic information on a Sercos device.

The user function also displays diagnostic information on sub-devices.

The parameters are read-only parameters.

The messages are also displayed by the message logger.

If a diagnostic message can be acknowledged via DiagQuit, the parameters of the user function Diagnostic are reset to their default values.

## User Function IOStatusControl

The user function IOStatusControl displays validity information on the digital inputs.

In addition, the user function allows you to block or release the digital outputs.

## Using the User Function Parameters in the Application

You can access a parameter in the application using the following syntax:

```
<Device>.<UserFunction>.<Parameter>
```

**Example:**

The following command sets the parameter WorkingMode  of the user function SercosIdentification to the value 1:

```
EDGENIM_1.SercosIdentification.WorkingMode := 1;
```
