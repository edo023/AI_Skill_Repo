# NumberOfDataObjects

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Logic Motion Controller Device |
| Traceable: | Yes |

## Functional Description

Displays the number of C2C data objects (**C2C Data Input** object, **C2C Data Output** object, **C2C Encoder Input** object, **C2C Encoder Output** object) configured for the C2C Slave.

EIO0000002335.11
