# l\_FailValue

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Uninterruptible power supply |
| Traceable | Yes |

## Functional Description

Indicates that an error has occurred in the UPS.

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | BOOL | UPS error |
| H / 1 | BOOL | No UPS error |
