# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Sum master encoder input |
| Traceable | Yes |

## Functional Description

Enables the sum master encoder input.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The sum master encoder input is not taken into account when totaling the velocities. |
| on / 1 | BOOL | The sum master encoder input is taken into account when totaling the velocities. |
