# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | No |
| Devices supporting the parameter | PacDrive LMC x00C  C2C\_Master, C2C\_Slave |
| Traceable | Yes |

## Functional Description

Enables or disables the interrupt function for a certain fast input.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The interrupt function is disabled.  Switching times of the fast inputs:   * High time (typically) 970 µs * Low time (typically) 890 µs |
| on / 1 | BOOL | The interrupt function is enabled  Switching times of the fast inputs:   * High time (typically) 100 µs * Low time (typically) 150 µs |
