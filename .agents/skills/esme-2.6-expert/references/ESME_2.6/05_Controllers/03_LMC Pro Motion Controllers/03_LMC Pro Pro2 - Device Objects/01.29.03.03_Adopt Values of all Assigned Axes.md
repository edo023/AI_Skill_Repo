# Adopt Values of all Assigned Axes

## Description

You can apply the values

* of an assigned axis

  (by clicking on the **[<--]** button) or

* of all assigned axes at a time

  (by clicking on the **[Adopt values of all assigned axes]** button).

As you apply these values the program writes the values of the assigned and scanned Sercos device that are required for commissioning into the related Sercos object in the PLC Configuration.

After the values have been applied, the corresponding row is highlighted in green.
