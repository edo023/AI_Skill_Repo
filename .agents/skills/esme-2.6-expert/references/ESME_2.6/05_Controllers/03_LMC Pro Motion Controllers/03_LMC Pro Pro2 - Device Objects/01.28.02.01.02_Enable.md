# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | C2C\_Master, C2C\_Slave |
| Traceable | Yes |

## Functional Description

Enables or disables the C2C\_Master / C2C\_Slave object.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The C2C\_Master / C2C\_Slave object is disabled. |
| on / 0 | BOOL | The C2C\_Master / C2C\_Slave object is enabled. |
