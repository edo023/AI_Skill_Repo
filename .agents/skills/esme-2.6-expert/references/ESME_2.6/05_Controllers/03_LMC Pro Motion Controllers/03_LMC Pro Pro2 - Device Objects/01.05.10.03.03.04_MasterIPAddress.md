# MasterIPAddress

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

This parameter displays the IP address of the master.

| Value | Data type | Meaning |
| --- | --- | --- |
| 172.20.0.1 (default) | STRING(15) | The IP address of the master is always the first IP address in the network which has been defined in NetID.  This parameter is calculated automatically.  Access type: read |
