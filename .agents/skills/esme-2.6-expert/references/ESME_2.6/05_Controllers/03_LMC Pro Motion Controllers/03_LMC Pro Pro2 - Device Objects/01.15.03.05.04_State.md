# State

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Encoder network (Synchronization encoder output) |
| Traceable | Yes |

## Functional Description

The parameter indicates the status of the synchronization encoder input.

| Value | Data type | Meaning |
| --- | --- | --- |
| disabled / 0 | DINT | The Synchronization encoder output is disabled. |
| initialization / 1 | DINT | The Synchronization encoder input is being initialized. |
| waiting for sync / 2 | DINT | The system is waiting for the Synchronization master to synchronize the entire encoder network (SyncMaster.State > 3). |
| synchronized / 3 | DINT | The system is waiting for the encoder network to be error-free and for the Synchronization master to enable the sync. Encoder output. Then, the configuration settings (DataCycleTime and DataDelay) are received from the Synchronization encoder input and the status is switched over to "waiting for data / 4". |
| waiting for data / 4 | DINT | The Synchronization encoder input stays in this status until it receives the encoder data from the Synchronization encoder output. When the set delay (DataDelay) has elapsed for the first batch of data, the status changes to "active / 5". |
| active / 5 | DINT | Communication is established with the Synchronization master and the encoder data are received correctly. |
| error / 6 | DINT | An unrecoverable error occurred while receiving encoder data.  Encoder data are no longer being received. Diagnostic messages:   * 8333 "EncNet receiving data not possible" * 8334 "EncNet receiving data dist. detected" |
