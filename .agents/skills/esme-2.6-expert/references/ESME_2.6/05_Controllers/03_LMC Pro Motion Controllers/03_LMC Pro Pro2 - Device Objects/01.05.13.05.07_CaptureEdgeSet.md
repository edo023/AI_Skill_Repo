# CaptureEdgeSet

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x01C, PacDrive LMC x00C  OnBoard I/O, DIO8 module  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable |  |

## Functional Description

Sets the edge of the respective touchprobe input.

The edge can be set online or offline.

| Value | Data type | Meaning |
| --- | --- | --- |
| Falling edge / 0 | DINT | Falling edge is set. |
| Rising edge / 1 (default) | DINT | Rising edge is set. |
