# StopBits

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | COM |
| Traceable | Yes |

## Functional Description

Selects the number of stop bits

| Value | Data type | Meaning |
| --- | --- | --- |
| 1 Bit / 0 | DINT | 1 stop bit |
| 2 Bits / 1 | DINT | 2 sop bits |
