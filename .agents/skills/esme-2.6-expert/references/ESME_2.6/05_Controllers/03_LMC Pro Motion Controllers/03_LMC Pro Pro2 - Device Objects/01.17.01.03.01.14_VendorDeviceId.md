## General

|  |  |
| --- | --- |

## Functional Description

The parameter represents the Sercos device identifier.

| Data type | Meaning |
| --- | --- |
| STRING [80] | Sercos device identifier |

NOTE: The parameter value is read via Sercos in phase 1. The value is not updated in greater phases.
