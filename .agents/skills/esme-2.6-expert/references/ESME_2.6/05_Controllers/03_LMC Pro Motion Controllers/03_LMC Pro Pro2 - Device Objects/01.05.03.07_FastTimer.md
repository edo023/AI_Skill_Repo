# FastTimer

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the timer in µs. The timer is derived from the TSC (TimeStampCounter) of the CPU. The TSC is a 64-bit counter that runs with the CPU cycle. The counter runs endlessly from 0 to 231-1. The end value corresponds to approximately 71 min.
