# DataType

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | C2C Data Input, C2C Data output |
| Traceable | Yes |

## Functional Description

Displays an unique value to identify the structure of the transmitted data (for example, CRC calculated over the data types of a structure).

DataType must be equal for the producers and consumers with the same ID. Otherwise diagnostic message [8520](D-SE-0064010.html#D-SE-0064010) (Type mismatch) will be displayed in the message logger.
