# Language

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

Language for internal text. If the language is changed, the diagnostic and message logger text appears afterward in the new language.

| Value | Data type | Meaning |
| --- | --- | --- |
| German / 0 | DINT | German |
| English / 1 | DINT | English |
