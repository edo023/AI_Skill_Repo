# Counter

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C |
| Traceable | Yes |

## Functional Description

With an [activated interrupt function](D-SE-0075316.html#D-SE-0075316), this parameter counts the interrupt events. It can be set to any required value at any time. When the maximum value of 232 –1 is exceeded, a counter overflow takes place. The overflow is not reported.
