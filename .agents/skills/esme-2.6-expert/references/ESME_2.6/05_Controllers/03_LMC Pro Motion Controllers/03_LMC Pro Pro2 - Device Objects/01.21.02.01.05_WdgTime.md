# WdgTime

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave, Field bus EtherNet/IP adapter |
| Traceable | Yes |

## Functional Description

Determines after which time a slave is considered to be inoperable by the watchdog. The time is given in ms.
