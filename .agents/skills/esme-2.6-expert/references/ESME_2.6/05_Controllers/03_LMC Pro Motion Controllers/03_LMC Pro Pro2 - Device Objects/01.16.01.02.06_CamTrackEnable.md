# CamTrackEnable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam switch group |
| Traceable | No |

## Functional Description

Displays the Enable values of the subordinate [CamTracks](D-SE-0077242.html#D-SE-0077242). All subordinate CamTracks can be activated or deactivated at the same by writing this value.

The data type of the parameter is UDINT (unsigned double integer 32 Bit). Each bit is assigned to an Enable parameter of a cam track (Cam Track). Bit 0 is assigned to the first cam track, bit 1 is assigned to the second cam track, etc. The cam track which has been inserted directly underneath the cam switch group (Cam switch group) in the PLC configuration is the first cam track.
