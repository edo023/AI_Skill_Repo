# MasterCycleErrorLimit

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Encoder network (Synchronization master) |
| Traceable | Yes |

## Functional Description

Defines the error tolerance of the synchronization process. MasterCycleErrorLimit defines how many cycles must pass before the slaves issue diagnostic message 8335 "EncNet synchronization not possible" should they stop receiving synchronization telegrams from the master. A corresponding alert 8336 "EncNet sync fault detected" is issued after half of those cycles have passed.

NOTE: How you set the error tolerance depends on your network conditions. Contact your local Schneider Electric service representative.
