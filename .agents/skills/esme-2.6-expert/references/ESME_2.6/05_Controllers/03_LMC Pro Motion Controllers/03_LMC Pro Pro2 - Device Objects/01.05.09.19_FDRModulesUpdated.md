# FDRModulesUpdated

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The parameter FDRModulesUpdated contains the number of updated modules.

The parameter value is set after each updated module.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0  (default) | UINT | Value remains 0 as long as the FDR function is not started. |
| >0 | UINT | The number of updated modules. |
