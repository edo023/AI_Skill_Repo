# LexiumLXM62DriveStandardPlusVersion

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | No |

## Functional Description

The parameter LexiumLXM62DriveStandardPlusVersion contains the firmware version number of a Lexium LXM62 Drive Standard Plus. This firmware version is used by the FDR function.

When the FDR function is executed, each configured Sercos device of the type Lexium LXM62 Drive Standard Plus is updated to this firmware version.

The firmware for the update of the Sercos devices is stored on the memory card of the controller.

| Value | Data type | Meaning |
| --- | --- | --- |
| “none” | STRING(40) | No firmware for the Lexium LXM62 Drive Standard Plus is contained in the memory card of the controller. The FDR function does not perform a firmware update for the Lexium LXM62 Drive Standard Plus. |
| “x.x”  (default) | STRING(40) | Firmware version used by the FDR function ( "x.x" represents the version number). |
