# 8521 Sercos C2C configuration

## Description

[Refer to Diagnostic class (standard): 1](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063169)

This diagnostic message contains additional information on diagnostic message [8520 Sercos C2C misconfiguration detected](D-SE-0064010.html#D-SE-0064010).

The extended diagnostic describes which parameter caused the diagnostic message:

| Ext. diagnosis | Meaning |
| --- | --- |
| `ID=` | ID of the data object with a duplicate ID. This diagnostic message is set for each duplicate data object detected from diagnostic message [8520 Sercos C2C misconfiguration detected](D-SE-0064010.html#D-SE-0064010) duplicate ID. |
| `Typ=` | The type of the data object that caused the diagnostic message[8520 Sercos C2C misconfiguration detected](D-SE-0064010.html#D-SE-0064010) `Type mismatch`. |
| `Len=` | The data size of the data object that caused the diagnostic message [8520 Sercos C2C misconfiguration detected](D-SE-0064010.html#D-SE-0064010) `Type mismatch`. |
| `DataT=` | The type of the data object that caused the diagnostic message [8520 Sercos C2C misconfiguration detected](D-SE-0064010.html#D-SE-0064010) `DataT mismatch`. |
