# ZeroTrackCheckOff

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Incremental Encoder Input |
| Traceable | Yes |

## Functional Description

Disables the zero track monitoring of the incremental encoder.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / FALSE | BOOL | Zero track monitoring active. |
| yes / TRUE | BOOL | Zero track monitoring not active. |

NOTE: In the case of the PacDrive LMC x00C controller, the zero track is not monitored, irrespective of the parameter value.

Interplay of track monitoring parameters CheckOff and ZeroTrackCheckOff

| CheckOff | ZeroTrackCheckOff | active monitoring |
| --- | --- | --- |
| no / FALSE | no / FALSE | Track A, B and zero track |
| no / FALSE | yes / TRUE | Track A and B |
| yes / TRUE | no / FALSE | no monitoring |
| yes / TRUE | yes / TRUE | no monitoring |
