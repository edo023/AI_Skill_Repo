# FC\_PhyEncGetCmdState

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine if a function that communicates with the physical encoder (SinCos encoder) is completed.

## Description

The functions FC\_PhyEncGetPosition(), FC\_PhyEncResetEncoder(), and FC\_PhyEncSetPosition() only start jobs. They are processed by a system task. This task communicates with the physical encoder via a serial interface. Use the function FC\_PhyEncGetCmdState() to determine if one of these functions has been completed.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stEncId | ST\_LogicalAddress | Logical address of the encoder |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: no job is active via the serial interface to the encoder  FALSE: job via the serial interface to the encoder is active |

See function FC\_PhyEncSetPosition().
