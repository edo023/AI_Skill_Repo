# FC\_GetParLogAddrByName

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine the logical address of a parameter with its name.

## Description

The logical address of the parameter identified by the name i\_sName is determined. The parameter name i\_sName must have the following structure: <Objektname.Parametername>. The logical address is returned as a function value. The parameter is determined via sequential searching and name comparison. The runtime of the function therefore depends on the number of objects and parameters and can take several 100 µs.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sName | STRING[80] | Parameter name |

## Return Value

| Data type | Description |
| --- | --- |
| ST\_LogicalAddress | 0: OK  -1: Object not found |
