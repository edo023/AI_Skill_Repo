# FC\_AxisBufGetMaxEntries

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine how many buffers are configured.

## Description

The axis is selected using i\_stAxisID and the number of configured buffers is returned.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Selecting the axis |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0 ... 16: Current buffer size  -1: Error detected |
