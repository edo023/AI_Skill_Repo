# FC\_LzsTaskGetInterval

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine the interval time of the task in the program.

## Description

This function returns the configured interval time of the calling task in ms.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | >=0: Interval time in ms. A negative value indicates that there is no fixed interval. |
