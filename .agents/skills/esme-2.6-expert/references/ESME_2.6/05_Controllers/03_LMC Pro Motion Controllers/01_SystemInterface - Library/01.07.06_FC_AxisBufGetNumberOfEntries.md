# FC\_AxisBufGetNumberOfEntries

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determining the current number of buffer types used.

## Description

The axis is selected using i\_stAxisID and the number of used buffers is returned.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Selecting the axis |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0 ... MaxEntries: Current number of entries  -1: Error detected |
