## Task

Read the access type of the OPC UA server.

## Description

The access type of the OPC UA server is retrieved.

The OPC UA server can access variables via different methods. The default access method is IecVarAcc (ET\_OpcUaAccessType.IECVARACC). Some variables can be accessed faster via direct memory access (ET\_OpcUaAccessType.MEMORYACC). If the access type is set to memory access, the OPC UA server uses memory access where possible.

## Return Value

| Data type | Description |
| --- | --- |
| [ET\_OpcUaAccessType](D-SE-0084923.html#D-SE-0084923) | MEMORYACC: Direct access to the memory area.  IECVARACC: Access via IecVarAcc functions. |

## Example

etAccessType : ET_OpcUaAccessType;
etAccessType := Systeminterface.FC_OpcUaGetAccessType();
