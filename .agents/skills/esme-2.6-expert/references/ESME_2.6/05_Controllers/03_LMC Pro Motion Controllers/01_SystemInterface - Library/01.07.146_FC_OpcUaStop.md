## Task

Stop and deactivate the OPC UA server.

## Description

The connections to the OPC UA clients are ended, the server is stopped and the OPC UA tasks are deleted.

NOTE: The processing/run of the function FC\_OpcUaStop() takes place at the same time as the stop procedure of the server and takes at least 3 s.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: The OPC UA server has been stopped.  -2: The OPC UA server has not completed the start procedure yet. The stop procedure was not performed. |

## Example

```
diResult : DINT;
diResult := Systeminterface.FC_OpcUaStop();
```
