# FC\_DisplayClear

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Delete display.

## Description

This function allows you to delete the controller display (PacDrive LMC Pro/Pro2, PacDrive LMC Eco).

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: Function for controller is not available  -2: Internal error detected  -3: The display is not in the IEC-Mode |

## Example

```
FC_DisplayClear();
```
