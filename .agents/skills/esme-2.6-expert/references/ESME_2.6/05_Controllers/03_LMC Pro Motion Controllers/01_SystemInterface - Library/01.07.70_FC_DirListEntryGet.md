# FC\_DirListEntryGet

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Search for one or several files.

## Description

Use this function to read an entry from the list of files. The function FC\_DirListEntryGet() copies the entry i\_diEntryNum in iq\_stEntry from the list of file information. This list is created by the function FC\_DirListCreate(). The function FC\_DirListCreate() provides the number of files or directories found. Possible values for the parameter i\_diEntryNum range from 1 to <number of files found>. This means that the function FC\_DirListCreate() must be executed before calling the function FC\_DirListEntryGet().

**Data type ET\_DirType**

The ET\_DirType data type contains predefined types for the FileType entry in the StructDirListEntry.

* 0: ET\_DirType.File => File
* 1: ET\_DirType.Dir => Directory

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_diEntryNum | DINT | Number of the file name entry |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_stEntry | ST\_DirListEntry | ST\_DirListEntry |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: No entry found |

## Examples

**Requirement**

The metadata of the files and folders on the flash discs shall be read out.

**Declaration**

```
PROGRAM DirTest 
VAR 
   diNumberOfFiles : DINT; 
   di : DINT; 
   diDirResult : DINT; 
   dleDirListEntry : ST_DirListEntry; 
   sFileName : STRING; 
   diFileSize : DINT; 
   diFileType : DINT; 
   diFileAttribute : DINT; 
   dtFileDate : DATE_AND_TIME; 
END_VAR
```

**Program**

```
diNumberOfFiles := FC_DirListCreate('*'); 
IF (diNumberOfFiles > 0) THEN 
   FOR di := 1 TO diNumberOfFiles BY 1 DO 
      diDirResult := FC_DirListEntryGet(di, dleDirListEntry); 
      IF (diDirResult = 0) THEN 
         sFileName := dleDirListEntry.sFilename; 
         diFileSize := dleDirListEntry.diFileSize; 
         diFileType := dleDirListEntry.diFileType; 
         diFileAttribute := dleDirListEntry.diFileAttribute; 
         dtFileDate := dleDirListEntry.dtFileDate; 
      END_IF
   END_FOR 
END_IF
```

**Result**

For every loop cycle, the variable diFileSize contains the file size, diFileType contains the type (file or folder), diFileAttribute contains the attributes (write protection) and dtFileDate contains date and time of the last modification.
