# FC\_WaitTime

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Wait for a certain time.

## Description

The IEC task is decelerated by i\_rWTime ms. If the time specified is 0 or a negative value, the IEC task is suppressed. It is reactivated after the other IEC tasks ready for calculation and with the same priority have been performed.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_rWTime | REAL | Delay time |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK |

## Examples

```
FC_WaitTime ( 0 );
FC_WaitTime ( 50 );
```
