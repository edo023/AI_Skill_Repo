# FC\_GetNVRamStatus

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Check, if the data in the non-volatile memory, referred to as NVRam, (message logger, retain variable) was valid when booting the PacDrive Controller.

## Description

This function reads the status of NVRam while booting. This may be required if the PacDrive Controller was de-energized for an extended period.

If the NVRam is not OK, then the system writes automatically an entry into the message logger during the restart process (316 "NvRam CRC error").

| Controller | Period of validity of the retain data |
| --- | --- |
| PacDrive LMC Pro | Maintenance interval of the battery according to the operating instructions. See operating instructions of the controller. |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: NVRam OK  FALSE: Error detected when checking the NVRam |

EIO0000002680.05
