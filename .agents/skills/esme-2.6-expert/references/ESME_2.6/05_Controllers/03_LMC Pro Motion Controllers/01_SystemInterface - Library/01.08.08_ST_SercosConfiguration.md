# ST\_SercosConfiguration

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | SystemInterface\_1.36.0.0 |
| Support for: | * FC\_SercosGetConfiguration * FC\_SercosScanConfiguration |
| Versions: | Current version |

## Description

In this structure, the general values of the Sercos bus are stored.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| uiNumberOfEntries | UINT | Number of Sercos devices scanned on the Sercos bus |
| uiNumberOfPhysicalDevices | UINT | Number of Sercos devices that are found on the Sercos bus and are configured in the project |
| uiPhaseRunUpCount | UINT | Counts the Sercos phase up runs |
| iCurrentPhase | INT | Current phase which the Sercos bus is running |
| astDevices | ARRAY [0..254] OF ST\_SercosConfigurationDevice | Array with specific description of Sercos devices found on the Sercos bus |
