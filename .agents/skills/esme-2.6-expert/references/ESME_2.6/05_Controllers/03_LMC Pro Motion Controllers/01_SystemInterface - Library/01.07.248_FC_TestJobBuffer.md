# FC\_TestJobBuffer

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine the condition of the job preparation buffer.

## Description

Provides the state of the job preparation buffer of the specified axis.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Logical address of the drive |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: Free buffer  FALSE: Occupied buffer |
