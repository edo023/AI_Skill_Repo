# FC\_DiagFlush

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Empty the diagnostic buffer.

## Description

The diagnostic information provided by the diagnostic handler is deleted.
