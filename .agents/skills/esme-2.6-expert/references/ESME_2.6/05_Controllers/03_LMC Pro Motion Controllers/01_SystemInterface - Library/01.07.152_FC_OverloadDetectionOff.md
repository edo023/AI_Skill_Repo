# FC\_OverloadDetectionOff

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Deactivate the overload monitoring of an axis.

## Description

The overload monitoring of the axis i\_stAxisId is deactivated. For i\_stAxisId = 0, overload monitoring of all axes is deactivated.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Logical address of the axis |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: Logical address of the axis is invalid |

EIO0000002680.05
