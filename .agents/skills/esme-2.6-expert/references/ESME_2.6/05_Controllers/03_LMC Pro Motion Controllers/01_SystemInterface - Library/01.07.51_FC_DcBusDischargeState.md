# FC\_DcBusDischargeState

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.33.10.0 |
| Versions: | Current version |

## Task

Returning the state of the DC bus discharge. This function is used with PowerSupplies or drives with integrated power supply.

## Description

This function returns the state of the DC bus discharge. The discharge is started with the function FC\_DcBusDischarge. Afterwards, FC\_DcBusDischargeState must be queried until discharging is completed (return value 3).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sDeviceId | ST\_LogicalAddress | logical address of the device |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: Discharging disabled  1: Preparing discharge process  2: DC bus is being discharged  3: DC bus was discharged successfully  -1: Incorrect logical address  -2: Discharging cannot be started because the drives are still being controlled.  -3: Discharging cannot be started because the device is enabled (PowerSupplyCheck = TRUE or ControllerEnable = TRUE).  -4: Discharging cannot be started because the device has detected an error.  -5: Discharging cannot be started because the DC bus is not charged.  -6: Discharging not possible  -461: Incorrect Sercos phase |
