# FC\_DisplayModeGet

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine the display mode.

## Description

Returns the current display mode. There are two modes:

* System mode (ET\_DisplayModeSelect.SystemMode = 0)
* IEC-Mode (ET\_DisplayModeSelect.IecMode = 1)

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: SystemMode  1: DisplayMode  -1: Function for controller is not available  -2: Internal error detected |

## Examples

```
 1:
```

```
 lResult:=FC_DisplayModeGet(); 
IF lResult < 0 THEN 
   sResult := 'Error: FC_DisplayModeGet'; 
END_IF; 
IF lResult = DISPLAY_SYSTEM_MODE THEN 
   lResult:=FC_DisplayModeSet(ET_DisplayModeSelect.IecMode); 
   IF lResult < 0 THEN 
      sResult := 'Error: DisplayModeSet'; 
   END_IF; 
END_IF; 
lState:=lState+1;
```
