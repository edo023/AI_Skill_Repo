# FC\_CycleCheckTimeGet

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine the times for the IEC task cycle time monitoring.

## Description

The function provides the time values for the IEC task cycle time check functions of the IEC task calling it.

iq\_diOverrunTime provides the value for default cycle time check [ms (diagnostic message [8317 "Cycle time overrun"](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063921)).

i\_diWDogTime provides the value for watchdog cycle time monitoring (diagnostic message 313 "Excessive cycle time overrun").

## Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_diOverrunTime | DINT | Value for the standard cycle time monitoring |
| iq\_diWDogFactor | DINT | Value for the watchdog factor |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  Otherwise: Incorrect parameter |
