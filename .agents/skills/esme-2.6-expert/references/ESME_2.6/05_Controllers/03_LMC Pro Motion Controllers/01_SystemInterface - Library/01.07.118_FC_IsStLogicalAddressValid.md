# FC\_IsStLogicalAddressValid

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Verify that a logical address is valid.

## Description

Verifies that the validation of the logical address in i\_stLogAddr belongs to a defined object.

TRUE = valid, FALSE = invalid

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stLogAddr | ST\_LogicalAddress | Logical address to be verified. |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: Address is valid  FALSE: Address is invalid |

EIO0000002680.05
