# FC\_OverloadDetectionQuit

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Acknowledge overload detection.

## Description

Acknowledges the overload detection of the axis i\_stAxisId.

The acknowledgment of the overload stop is only effective if FC\_OverloadDetectionGetState provides the values 3 (stopping) or 4 (standstill).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Logical address of the axis |
| i\_xQuitFlag | BOOL | With an edge from FALSE to TRUE, the overload stop is acknowledged.  i\_xQuitFlag has to be FALSE for at least one Sercos cycle before it is switched to TRUE for at least one Sercos cycle to recognize the edge. |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: Overload detection quit was sent to at least one Sercos device.  -1: Overload detection quit could not be sent to any Sercos device. |
