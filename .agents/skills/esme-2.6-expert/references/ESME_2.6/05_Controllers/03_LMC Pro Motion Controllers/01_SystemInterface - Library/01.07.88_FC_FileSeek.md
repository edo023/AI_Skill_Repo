# FC\_FileSeek

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Read data from a file from any point.

## Description

This function allows you to set the write/read pointer of an open file i\_diFileId. The following write or read operations are executed from this position. The positioning with i\_diOffset can be carried out relative at the beginning of the file (i\_diFilePosition = SEEK\_SET), to the current position (i\_diFilePosition = SEEK\_CUR) or to the file end (i\_diFilePosition = SEEK\_END). This allows direct access to any file positions rather than sequential file processing.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_diFileId | DINT | Name of the file from which you want to read from |
| i\_diOffset | DINT | Offset of the positioning |
| i\_diFilePosition | DINT | Reference point of the Offset   * Relative at the beginning of the file (i\_diFilePosition = 0) * To the current position (i\_diFilePosition = 1) * To the file end (i\_diFilePosition = 2) |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | >=0: Position of the pointer  -3: FileId not correct  -5: File is not open  -9: Error when setting the write/read pointer |

## Example

File Test.Dat with the text "This is a test!"(18 characters) must be transferred to the controller under "ide0:"

```
PROGRAM FileSeek 
VAR 
   fileopen :DINT; 
   lreturn0 :DINT; 
END_VAR 
fileopen:=FC_Open_ReadFile('Test'); 
lreturn0:=FC_FileSeek(i_diFileId:=fileopen, i_diOffset:=10, i_diFilePosition:=0); 
FC_CloseFile(fileopen);
```

Result:

lreturn0=10
