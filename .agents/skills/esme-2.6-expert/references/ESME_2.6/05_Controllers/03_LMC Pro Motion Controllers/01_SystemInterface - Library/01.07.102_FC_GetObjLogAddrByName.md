# FC\_GetObjLogAddrByName

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determining the type of an object (for example, logical encoder, virtual encoder, ...) from the symbolic name.

## Description

The logical address of the object with the name i\_sName is determined. The object type of the logical address is returned as a function value if the name of the object type and the name of the object instance are equal. Otherwise the logical address (with instance number) is returned. The object is determined via sequential searching and name comparison. The runtime of the function therefore depends on the number of objects and can amount to several 100 µs.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sName | STRING[80] | Object name |

## Return Value

| Data type | Description |
| --- | --- |
| ST\_LogicalAddress | 0: OK  16#FFFF: Object not found |
