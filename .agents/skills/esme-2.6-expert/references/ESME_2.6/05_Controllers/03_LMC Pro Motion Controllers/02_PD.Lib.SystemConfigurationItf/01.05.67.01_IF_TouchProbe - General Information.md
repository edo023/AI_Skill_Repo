# IF\_TouchProbe - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_IdentificationMandatory](D-SE-0089168.html#D-SE-0089168) |

## Description

The interface indicates the properties of the touchprobe inputs.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| CaptureOk | BOOL | Read | Indicates the state of the touchprobe measurement. |
| CaptureState | DINT | Read | Indicates the state of the touchprobe function. |
| CaptureValue | LREAL | Read | Indicates the position (at FC\_TPEdge()) that has been captured when the event was detected. |
| Value | BOOL | Read | Indicates the status of the touchprobe input.  FALSE indicates 0 V, TRUE indicates 24 V. |
