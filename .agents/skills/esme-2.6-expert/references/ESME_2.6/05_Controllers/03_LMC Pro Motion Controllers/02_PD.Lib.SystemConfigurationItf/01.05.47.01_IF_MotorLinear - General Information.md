# IF\_MotorLinear - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the properties of the linear motor and the holding brake.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| BrakeCouplingTime | DINT | Read/Write | Sets the coupling time of the holding brake. |
| BrakeDisconnectionTime | DINT | Read/Write | Sets the disconnection time of the holding brake. |
| BrakeIntegrated | BOOL | Read | Indicates whether a holding brake is integrated. |
| BrakeState | BOOL | Read | Indicates the state of the holding brake. |
| Direction | BOOL | Read/Write | Sets the direction of movement. |
| ForceConstant | LREAL | Read | Indicates the force constant of the motor N/A for low currents. |
| MaxSpeedLinear | LREAL | Read | Indicates the maximum velocity in mm/s. |
| MotorCommutationControl | DINT | Read/Write | Sets the automatic detection of the commutation (without a connected load). |
| MotorCommutationState | INT | Read | Indicates the state of the commutation and the functionalities that are started via the parameter MotorCommutationControl. |
| MotorConnectionTest | BOOL | Read/Write | Enables the motor connection test (when enabling the drive). |
| MotorInertiaLinear | LREAL | Read | Indicates the mass inertia of the motor in kg. |
| MotorTemperatureMonitoring | DINT | Read/Write | Sets the monitoring of the motor temperature. |
| MotorType | STRING[20] | Read | Indicates the type of the motor. |
| MotorTypePlateVersion | STRING[20] | Read | Indicates the version number of the motor. |
| NominalSpeedLinear | LREAL | Read | Indicates the velocity of the motor in mm/s. |
| PartNumberMotor | STRING[20] | Read | Indicates the reference number of the motor. |
| SerialNumberMotor | STRING[20] | Read | Indicates the serial number of the motor. |
