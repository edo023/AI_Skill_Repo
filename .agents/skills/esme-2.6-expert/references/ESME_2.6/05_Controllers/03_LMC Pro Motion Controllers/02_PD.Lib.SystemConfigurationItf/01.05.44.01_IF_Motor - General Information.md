# IF\_Motor - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the properties of the motor.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| BrakeIntegrated | BOOL | Read | Indicates whether a brake is integrated. |
| BrakeState | BOOL | Read | Indicates the state of the brake. |
| DriveOverload | DINT | Read | Indicates the overload of the drive (power stage and motor) in percentage. |
| EncoderRange | LREAL | Read | Indicates the value range of the encoder. |
| MaxSpeed | REAL | Read | Indicates the maximum speed of rotation in RPM. |
| MotorIdentification | DINT | Read/Write | Sets whether there is a motor with or without nameplate, or a drive without a connected motor. |
| MotorInertia | LREAL | Read | Indicates the moment of inertia of the motor including the holding brake in kgcm2. |
| MotorType | STRING[20] | Read | Indicates the name of the motor. |
| MotorTypePlateVersion | STRING[20] | Read | Indicates the version number of the motor. |
| PartNumberMotor | STRING[20] | Read | Indicates the reference number of the motor. |
| SerialNumberMotor | STRING[20] | Read | Indicates the serial number of the motor. |
| TorqueConstant | LREAL | Read | Indicates the torque constant of the motor [Nm/A] for low currents. |
