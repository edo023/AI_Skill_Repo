# IF\_EnergyMeasurementDrive - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the properties of the energy measurement of the drive.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| DissipatedEnergyDrive | LREAL {attribute 'monitoring':='call'} | Read | Indicates the energy loss in the motor winding. |
| ElectricalEnergyDrive | LREAL | Read | Indicates the supplied electrical energy that was fed into the DC bus by the drive in kWh. |
| EnergyMeasurementEnableSet | BOOL | Read/Write | Enables the energy measurement. |
| MechanicalEnergy | LREAL | Read | Indicates the supplied or braked mechanical energy on the drive shaft in kWh. |
