# IF\_Limits - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the limit values of the drive.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| MaxAcc | REAL | Read | Indicates the maximum acceleration at the gear output in units/s2. |
| MaxDrivePeakCurrent | DINT | Read | Indicates the maximum drive current (combination of drive and motor). |
| MaxVel | REAL | Read | Indicates the maximum velocity at the gear output. |
