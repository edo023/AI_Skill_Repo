## Description

The interface indicates the properties of the TM5 encoder output.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| DigitalInput01 | BOOL | Read | Indicates the digital state of the hardware channel 4. |
| DigitalInput02 | BOOL | Read | Indicates the digital state of the hardware channel 5. |
| PositionDelay | LREAL | Read | Indicates in µs how long ago the position was last read. |
