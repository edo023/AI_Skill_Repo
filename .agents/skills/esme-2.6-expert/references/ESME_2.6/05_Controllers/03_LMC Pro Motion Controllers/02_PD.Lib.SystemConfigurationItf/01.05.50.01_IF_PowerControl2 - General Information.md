## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |

## Description

The interface manages the method GetPowerSupply and the properties of the InverterEnable signal.

## Methods

| Name | Description |
| --- | --- |
| GetPowerSupply | Detects the power supply assigned to the axis. It is assigned via the parameter ConfiguredPowerSupply of the axis. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| InverterEnable | BOOL | Read | Indicates the status of the InverterEnable signal. |
