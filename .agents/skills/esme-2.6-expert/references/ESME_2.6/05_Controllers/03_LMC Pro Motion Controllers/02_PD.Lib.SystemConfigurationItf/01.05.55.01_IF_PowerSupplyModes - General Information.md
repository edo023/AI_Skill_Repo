# IF\_PowerSupplyModes - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.76.6.0 |
| Inherits from: | - |

## Description

The interface manages the properties of the modes of the power supply.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| BrakingResistorMode | DINT | Read/Write | Sets the mode of the braking resistor. |
| MainsVoltageMode | DINT | Read/Write | Sets the mode of the mains voltage. |
| PhaseCheckMode | DINT | Read/Write | Sets the mode of the mains phase monitoring. |
