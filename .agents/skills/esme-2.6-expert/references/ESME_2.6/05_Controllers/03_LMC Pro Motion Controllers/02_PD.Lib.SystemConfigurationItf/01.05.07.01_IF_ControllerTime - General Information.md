# IF\_ControllerTime - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the time information and the timer.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| FastTimer | UDINT | Read | Indicates the timer in µs. The timer is derived from the TSC (TimeStampCounter) of the CPU. |
| RealtimeClock | DATE\_AND\_TIME | Read | Indicates the real-time clock with date and time information. |
| SetRealtimeClock | DATE\_AND\_TIME | Read/Write | Sets the real-time clock. |
| Timer1 | DINT | Read | Indicates the timer in ms. |
| Timer10 | DINT | Read | Indicates the timer in 10 ms steps. |
