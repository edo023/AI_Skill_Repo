# IF\_MasterVelocity - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface indicates the velocity of the device.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| Velocity | LREAL | Read | Indicates the velocity of the device.  See also:   * Velocity parameter of the incremental encoder (INC\_IN, INC\_OUT, P\_ENC) * Velocity parameter of the sum encoder (SENC) * Velocity parameter of the sum master encoder (SMENC) * Velocity parameter of the virtual encoder (V\_ENC) |
