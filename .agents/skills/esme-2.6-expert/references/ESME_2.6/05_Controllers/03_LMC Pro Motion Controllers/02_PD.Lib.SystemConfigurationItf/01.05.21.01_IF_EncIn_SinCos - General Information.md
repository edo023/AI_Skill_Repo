# IF\_EncIn\_SinCos - General Information

## Overview

|  |  |
| --- | --- |

| Available as of: | V1.76.6.0 |
| Inherits from: | [IF\_EncIn](IF_EncIn-GeneralInformation-7AED85DB.html#IF_EncIn-GeneralInformation-7AED85DB) |

## Description

The interface manages the properties of the SinCos encoder.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| EncoderType | DINT | Read | Indicates the encoder type. |
| InitPosition | LREAL | Read | Indicates the initial position. |
| SignalQuality | UDINT | Read | Indicates the signal quality in percentage. |
| SignalQualityLimit | UDINT | Read/Write | Sets the threshold in percentage at which an error is detected when the signal quality is below this value. |
