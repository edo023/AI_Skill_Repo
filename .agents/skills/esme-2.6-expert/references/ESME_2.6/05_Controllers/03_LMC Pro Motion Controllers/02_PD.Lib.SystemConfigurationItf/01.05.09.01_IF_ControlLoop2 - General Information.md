# IF\_ControlLoop2 - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the properties of the frequency converter.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| BoostVoltage | UDINT | Read/Write | Sets the boost voltage for the U/f controller in percentage. |
| VF\_Characteristic | DINT | Read/Write | Sets the U/f characteristic curve. |
