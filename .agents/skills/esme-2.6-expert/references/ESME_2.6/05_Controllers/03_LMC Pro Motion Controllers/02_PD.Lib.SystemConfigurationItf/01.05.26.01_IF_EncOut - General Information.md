# IF\_EncOut - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.76.6.0 |
| Inherits from: | [IF\_InObject](IF_InObject-GeneralInformation-7AEEE58F.html#IF_InObject-GeneralInformation-7AEEE58F) |

## Description

The interface manages the properties of the encoder output.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| Direction | BOOL | Read/Write | Sets the direction of movement.  FALSE: Negative direction of movement.  TRUE: Positive direction of movement. |
| FeedConstant | REAL | Read/Write | Sets the feed constant. |
| Position | LREAL | Read | Indicates the position of the encoder output. |
| Resolution | DINT | Read/Write | Sets the resolution of the encoder output. |
| State | DINT | Read | Indicates the state of the encoder output. |
| Velocity | LREAL | Read | Indicates the velocity of the encoder output. |
