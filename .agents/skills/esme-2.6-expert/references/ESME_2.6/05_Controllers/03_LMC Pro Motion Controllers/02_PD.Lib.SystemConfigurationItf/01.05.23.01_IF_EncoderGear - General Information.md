# IF\_EncoderGear - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the gearbox factor between the motor and the encoder.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| EncoderGearIn | DINT | Read/Write | Sets the number of teeth on the motor side. |
| EncoderGearOut | DINT | Read/Write | Sets the number of teeth on the encoder side. |
