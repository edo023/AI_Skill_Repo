# IF\_MechanicLinear - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the mechanical properties of a linear motor.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| FeedConstantLinear | LREAL | Read/Write | Sets the feed constant in units/m. |
| LoadInertialLinear | LREAL | Read/Write | Sets the mass inertia of the load in kg. |
| StaticFrictionLinear | LREAL | Read/Write | Sets the static friction in N. |
| ViscousFrictionLinear | LREAL | Read/Write | Sets the viscous friction in N/(1000 units/s). |
