# WorkingMode

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Traceable | Yes |

## Functional Description

With this parameter, you can set the working mode of the device.

| Value | Data type | Meaning |
| --- | --- | --- |
| virtual / 0 | DINT | The device is operated virtually.  Reference position values are not transferred to a device. The position values are copied from the reference values in each motion cycle. |
| real / 1 | DINT | The device is searched for in the Sercos network:   * The device is found: The device is used. * The device is not found: A diagnostic message is displayed. Sercos phase-up is not possible. |
| deactivated / 2 | DINT | The cyclic calculation is deactivated. This behavior reduces the cycle load. |
| optional / 3 | DINT | The device is searched for in the Sercos network:   * The device is found: The device is used. * The device is not found: The device is operated in virtual / 0 working mode. |
