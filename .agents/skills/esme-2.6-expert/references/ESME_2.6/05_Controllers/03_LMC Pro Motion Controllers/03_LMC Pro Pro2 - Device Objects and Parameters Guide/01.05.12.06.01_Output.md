## Functional Description

The parameter represents the name specified in the Devices tree. This name can be modified dynamically in the program.

The name is used in the following functions / parameters:

* Message logger (names in the column Instance)

  Transfer time: loading the message logger
* Cam Track (parameter PositionSource)

  Transfer time: parameter Enable FALSE -> TRUE
* Logical encoder (parameter InObject)

  Transfer time: *FC\_SetMasterEncoder()*
* Incremental Encoder Output (parameter InObject)

  Transfer time: *FC\_SetMasterEncoder()*
* Sum master encoder input (parameter InObject)

  Transfer time: *FC\_SetMasterEncoder()*
* Sum encoder input (parameter `InObject`)

  Transfer time: *FC\_SetMasterEncoder()*
* *FC\_GetObjInstanceLogAdrByName()*
* FC\_GetObjLogAdrByName()
* *FC\_GetObjNameByLogAdr()*
* C2C Encoder Output (parameter InObject)

  Transfer time: *FC\_SetMasterEncoder()*
* C2C Online View tab of the C2C Master (names in the column Devices)

For the functions and parameters mentioned, use the active name at the time of execution.
