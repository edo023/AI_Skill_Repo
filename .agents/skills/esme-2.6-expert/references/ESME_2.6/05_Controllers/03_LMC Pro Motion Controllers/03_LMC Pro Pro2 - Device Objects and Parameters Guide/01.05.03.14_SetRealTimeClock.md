# SetRealTimeClock

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

There is a software clock with date and time information in the PacDrive controller.

The real time clock of the PacDrive controller is used.

* The software clock is set automatically after the PacDrive controller has been booted.
* The real-time hardware clock is buffered using batteries and continues to run when the PacDrive controller is turned off.

The clock is set when the SetRealTimeClock parameter is written. The "time" of the system is available in the parameter RealTimeClock. SetRealTimeClock only displays the time the clock was set or the time of the last boot when the hardware clock has been installed and is functioning properly. The clock is activated the first time that SetRealTimeClock is set.
