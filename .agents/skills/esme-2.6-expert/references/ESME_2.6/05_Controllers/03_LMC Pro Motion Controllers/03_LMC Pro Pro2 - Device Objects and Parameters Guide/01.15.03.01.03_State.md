# State

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Encoder network (Synchronization master) |
| Traceable | Yes |

## Functional Description

Displays the connection status. This parameter must be seen analogously to the State parameter of the Synchronization slave.

| Value | Data type | Meaning |
| --- | --- | --- |
| disabled / 0 | DINT | The Synchronization master is disabled |
| initialization / 1 | DINT | The Synchronization master is being initialized and the enabled Synchronization slaves are being configured. Communication is still not cyclic. |
| waiting for sync / 2 | DINT | Telegrams for controller synchronization are being transmitted cyclically. |
| synchronized / 3 | DINT | Synchronization was successfully established with all Synchronization slaves. |
| waiting for data / 4 | DINT | Encoder data is being transferred cyclically. |
| active / 5 | DINT | Communication with all Synchronization slaves is established, all Synchronization slaves have State "active / 5" and encoder data are evaluated correctly. |
| error / 6 | DINT | An error occurred, communication must be reestablished. |
