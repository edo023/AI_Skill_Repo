# CommError

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave, Field bus EtherNet/IP adapter, Field bus EtherCATSlave |
| Traceable | Yes |

## Functional Description

Displays the current error code of the communication channel.

Possible values

| Value | Data type | Meaning |
| --- | --- | --- |
| SUCCESS / 0x0 | UDINT | Indicates that the cause of the error has been eliminated. |

Runtime error

| Value | Data type | Meaning |
| --- | --- | --- |
| WATCHDOG\_TIMEOUT / 0xC000000C | UDINT | Watchdog time exceeded |

Initialization errors

| Value | Data type | Meaning |
| --- | --- | --- |
| INIT\_FAULT / 0xC0000100 | UDINT | Initialization is unsuccessful |
| DATABASE\_ACCESS\_FAILED / 0xC0000101 | UDINT | Accessing data memory is unsuccessful |

Configuration errors

| Value | Data type | Meaning |
| --- | --- | --- |
| NOT\_CONFIGURED / 0xC0000119 | UDINT | Not configured |
| CONFIGURATION\_FAULT / 0xC0000120 | UDINT | Configuration error |
| INCONSISTENT\_DATA\_SET / 0xC0000121 | UDINT | Set data are inconsistent. |
| DATA\_SET\_MISMATCH / 0xC0000122 | UDINT | Set data have been mismatched. |
| INSUFFICIENT\_LICENSE / 0xC0000123 | UDINT | Insufficient license error |
| PARAMETER\_ERROR / 0xC0000124 | UDINT | Parameter error |
| INVALID\_NETWORK\_ADDRESS / 0xC0000125 | UDINT | Invalid network address |
| SECURITY\_MEMORY / 0xC0000126 | UDINT | Error detected with the security memory |
| TCP\_CONFIG\_FAIL / 0xC0590004 | UDINT | TCP/IP configuration is unsuccessful.  The TCP/IP task reports an error: IP address, gateway address, network mask or configuration flags are invalid. |

Network errors

| Value | Data type | Meaning |
| --- | --- | --- |
| COMM\_NETWORK\_FAULT / 0xC0000140 | UDINT | Network communication error |
| COMM\_CONNECTION\_CLOSED / 0xC0000141 | UDINT | Communication connection closed |
| COMM\_CONNECTION\_TIMEOUT / 0xC0000142 | UDINT | Communication connection timeout |
| COMM\_DUPLICATE\_NODE / 0xC0000144 | UDINT | Duplicate node |
| COMM\_CABLE\_DISCONNECT / 0xC0000145 | UDINT | Cable disconnected |
| PROFIBUS\_CONNECTION\_TIMEOUT / 0xC009002E | UDINT | PROFIBUS connection timeout |
