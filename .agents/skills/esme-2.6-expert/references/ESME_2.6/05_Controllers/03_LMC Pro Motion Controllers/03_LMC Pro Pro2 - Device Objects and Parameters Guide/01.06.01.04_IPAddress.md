# IPAddress

## General

|  |  |
| --- | --- |
| Type | EF |
| Address | 16#000D |
| Offline editable | Yes |
| Devices supporting the parameter | Ethernet |
| Traceable | No |
| Default value | '192.168.1.100' |

## Functional Description

The parameter sets the IP address for the Ethernet interface.

The parameter value is verified for:

* Correct format [a1].[a2].[a3].[a4].
* [a1] (first octet) is greater than 0 and not equal to 127 (127.x.x.x is the local host)
* Overlapping with other network interfaces (standard Ethernet, Sercos UCC). If overlaps are detected, the diagnostics message 8967 NRT IP parameter device different is appended to the message logger list.

| Value | Data type | Meaning |
| --- | --- | --- |
| '192.168.1.100' (default) | STRING(15) | IP address of the Ethernet interface. |
