# LoadEff

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Cam switch group |
| Traceable | Yes |

## Functional Description

Displays the effective value of Load. Here interruptions by other tasks are taken into account. This makes it easier to conclude the actual CPU requirement for processing the CamSwitchGroup (Cam switch group).

Interruptions by Interrupts are not taken into account. The measure value for switching tasks is thus directly recorded in the operating system. Using this measurement principle, switching times can be recorded at the same time. Typical LoadEff values are therefore approximately 5 µs larger than comparable Load values.

Using the parameter EnableLoadEff, the effective CPU Load measurement is started and stopped again. LoadEff will only be calculated for an active EnableLoadEff.
