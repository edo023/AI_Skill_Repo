# ProgrammingSystem

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the programming system and the version of the device description that was used to create the program.

## Example

PSV:03.00.01.01

TSV:1.31.2.0

PSV = Version of the programming system

TSV = Version of the device description

EIO0000002335.11
