# Position

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Log. encoder |
| Traceable | Yes |

## Functional Description

Displays the position of the logical axis.

The position has a maximum which is calculated according to the following formula:

PositionMax = (231 / 219 \* FeedConstant \* GearIn/GearOut) - 1

If the maximum position is exceeded, the position is set to the negative maximum value.

NOTE: In case of overflow, no diagnostic messages is triggered by the PacDrive system.

## Example

* FeedConstant = 360 units/rotations
* GearIn = 1
* GearOut = 4

PositionMax = (231 / 219 \* 360 \* 1/4) - 1 = 368639

Consequently, values from -368639 to 368639 are possible in the parameter Position.
