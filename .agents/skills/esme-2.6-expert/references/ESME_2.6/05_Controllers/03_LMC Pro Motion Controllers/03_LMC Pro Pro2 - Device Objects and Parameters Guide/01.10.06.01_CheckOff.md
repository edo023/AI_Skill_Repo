# CheckOff

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | SinCos Encoder Input |
| Traceable | Yes |

## Functional Description

Disables the encoder monitoring.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | DINT | Encoder check active. |
| yes / 1 | DINT | Encoder check not active.  The diagnostic message *8601 "Master Encoder signal out of range"* is disabled. |

EIO0000002335.11
