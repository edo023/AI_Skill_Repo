# Position

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | SinCos Encoder Input |
| Traceable | No |

## Functional Description

Displays the current encoder position.

For single-turn encoders, the position is from 0 to 1 revolution, for multi-turn encoders the position is from 0 to 4096 revolutions.

With negative direction of rotation, clockwise rotation causes the value range to run in decreasing order.

At system startup, the current encoder position is read.

Use the parameter State to check the availability or validity of the encoder position data.
