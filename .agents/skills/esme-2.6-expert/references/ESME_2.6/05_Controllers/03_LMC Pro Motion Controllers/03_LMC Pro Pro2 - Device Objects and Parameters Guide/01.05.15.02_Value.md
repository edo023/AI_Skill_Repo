# Value

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C,  Analog input |
| Traceable | Yes |

## Functional Description

Displays the value that should be output at the analog output.

Value = 0 -> -10 V

Value = 4095 -> +10 V
