# Timer10

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the timer in 10 ms steps. After reaching the maximum (231-1 ms, that is, approximately 248.5 days), it starts counting upwards from –231. The counter is automatically started after a reset.

NOTE: The resolution of Timer10 is a multiple of the CycleTime of the Sercos drive bus.
