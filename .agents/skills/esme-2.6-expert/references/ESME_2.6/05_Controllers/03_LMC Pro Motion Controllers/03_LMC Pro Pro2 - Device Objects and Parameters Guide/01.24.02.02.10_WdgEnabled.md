# WdgEnabled

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave,  Field bus EtherNet/IP adapter |
| Traceable | Yes |

## Functional Description

Enables the watchdog.

| Value | Data type | Meaning |
| --- | --- | --- |
| TRUE | BOOL | The watchdog is activated. The value of the parameter WdgTime is used as watchdog timeout. |
| FALSE | BOOL | The watchdog is deactivated. The value of the parameter WdgTime is ignored. |
