# UniCode

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

The parameter `UniCode` of type STRING(80) displays the type plate parameters for the PacDrive LMC Pro2 controllers.The displayed information depends on the number of axis.

Type plate parameters for PacDrive LMC Pro2 with 16 axes

| Parameter | Value |
| --- | --- |
| Unicode | LMC402CAA10000 |
| 250us | 16 |
| 500us | 16 |
| 1ms | 16 |
| 2ms | 16 |
| 4ms | 16 |
| InternalHWCode | 1792 (700h) |

Type plate parameters for PacDrive LMC Pro2 with 130 axes

| Parameter | Value |
| --- | --- |
| Unicode | LMC802CAA10000 |
| 250us | 130 |
| 500us | 130 |
| 1ms | 130 |
| 2ms | 130 |
| 4ms | 130 |
| InternalHWCode | 1793 (701h) |
