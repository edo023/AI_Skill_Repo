# License\_1 to License\_10

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays licenses 1 to 10.

The displayed information is structured as follows: `Name, license, points, lifetime`

| Lifetime | Meaning |
| --- | --- |
| 0 | License points are released after a controller reset. |
| 1 | License points are released after a Sercos bus phase change to phase 0 (for example, phase 4 -> phase 0). |
| 2 | License points are released again after a program reset. |

NOTE: If licenses are requested in evaluation mode, then these are not released again in the case of a reset and Sercos phase change following phase 0. The reason for this is that the licences requested during the evaluation mode is to be logged. In this way, it can be determined how many license points must be ordered.

If, during evaluation mode, a reset, or Sercos run-up (Phase 0 -> Phase 4) is carried out on the controller, licenses is entered more than once.

## Examples

License\_1 = 'Rapid8,60,2'

The POU Rapid8() requires 60 license points. The license points are released again after a program reset.

License\_1 = 'Eval:Rapid8,60,2'

There are not enough license points available. The PacDrive System is in evaluation mode.
