# SensorDelay

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | PacDrive LMC x01C, PacDrive LMC x00C  OnBoard I/O, DIO8 module  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

## Functional Description

Used to enter the delay of an external sensor.

The controller considers this delay for position detection.

Also refer to *FC\_TpEdge* (Shifting CaptureValue).
