# CaptureValue

## General

|  |  |
| --- | --- |
| Type | AD |
| Traceable | Yes |

## Functional Description

Displays the position (at FC\_TPEdge()) that has been captured when the event was detected. The value is reset to 0.00 when *FC\_TPEdge()* is called again.

Resolution: 10 µs
