# OnLevel

## General

|  |  |
| --- | --- |
| Typer | EF |
| Devices supporting the parameter | Cam Track |
| Traceable | Yes |

## Functional Description

Inverts the output signal.

Normally, when the cam switch (Cam switch) is active, a high signal is switched to the output (Destination). It is possible to invert the output signal with OnLevel.

| Value | Data type | Meaning |
| --- | --- | --- |
| Low / 0 | BOOL | An active cam switch causes a low signal on the output. |
| High / 1 | BOOL | An active cam switch causes a high signal on the output. |
