# IPAddressRangeStatic

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

This parameter displays the address range for manual IP configurations.

| Value | Data type | Meaning |
| --- | --- | --- |
| "no static addresses" (default) | STRING(31) | This parameter indicates which address range may be used for manual IP configurations.  The address range is located at the beginning of the network defined in NetID.  The IP address (parameterSercos IPAddress for IPConfigMode = **manual IP addressing (software) / 2** or the IP address defined via the hardware for IPConfigMode = **manual IP addressing (hardware) / 1)** of a slave which is to be configured manually may be set to an address in this address range.  This parameter is calculated automatically.  Access type: read |
