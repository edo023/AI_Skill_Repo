# SubnetmaskUCC

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | C2C\_Slave,  Logic Motion Controller Device |
| Traceable | No |

## Functional Description

With this parameter, you can define the subnet mask for the UCC (**U**nified **C**ommunication **C**hannel) communication.

| Value | Data type | Description |
| --- | --- | --- |
| 255.255.252.0 (default) | STRING(15) | Subnet mask of the slave device for the UCC communication |
