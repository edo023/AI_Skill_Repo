# GearEnable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Log. encoder |
| Traceable | Yes |

## Functional Description

Enables/disables the gear box.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The gear is disabled. |
| on / 1 | BOOL | Setting the parameter to "on / 1" activates the gear. |

| NOTICE | |
| --- | --- |
|  | POSITION JUMPS WHEN ACTIVATING/DEACTIVATING THE GEARS  * Activate the function only once when initializing. * Deactivate the function only under exceptional circumstances.  Failure to follow these instructions can result in equipment damage. |
