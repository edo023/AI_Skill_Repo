# l\_ActiveValue

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Uninterruptible power supply |
| Traceable | Yes |

## Functional Description

Indicates whether the UPS is active.

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | BOOL | UPS is inactive. |
| H / 1 | BOOL | UPS is active |
