# EncoderError

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | C2C Encoder Input, C2C Encoder Output |
| Traceable | Yes |

## Functional Description

Displays the filtered velocity of the encoder at the encoder shaft in units/s.

A unit is defined using the parameter FeedConstant.

EIO0000002335.11
