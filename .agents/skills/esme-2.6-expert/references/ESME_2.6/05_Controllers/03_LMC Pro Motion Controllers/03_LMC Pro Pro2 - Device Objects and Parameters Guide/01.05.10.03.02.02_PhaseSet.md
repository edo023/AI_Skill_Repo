# PhaseSet

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

You can use this parameter to preset the required communication phase of the Sercos bus.

| NOTICE | |
| --- | --- |
|  | POSITION LOSS DUE TO THE SERCOS BUS PHASE CHANGE  Re-initialize or home the system by means of the program when transitioning to Sercos phase 4.  Failure to follow these instructions can result in equipment damage. |

| Value | Data type | Meaning |
| --- | --- | --- |
| PhaseSet <> 4 | DINT | The tasks of the program are started simultaneously with the Sercos run-up. |
| PhaseSet = 4 | DINT | The program tasks start is delayed until:   * the Sercos run-up has reached phase 4 (state = "Phase 4") * The Sercos run-up is canceled by a start-up error (state = "Error") |
