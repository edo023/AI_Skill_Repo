# FW\_Version

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

The firmware version of the PacDrive controller is displayed with the creation date in this parameter.

## Example

'V1.31.10.0 Mar 17 2010'.

EIO0000002335.11
