# FailsafeSupported

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave |
| Traceable | Yes |

## Functional Description

Table Enables / disables the failsafe mode.

| Value | Data type | Meaning |
| --- | --- | --- |
| TRUE | BOOL | The failsafe mode is activated. |
| FALSE | BOOL | The failsafe mode is deactivated. |
