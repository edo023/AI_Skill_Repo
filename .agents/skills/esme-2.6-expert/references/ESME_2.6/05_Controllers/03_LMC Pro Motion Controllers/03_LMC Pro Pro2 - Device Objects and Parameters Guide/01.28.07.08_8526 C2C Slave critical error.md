# 8526 C2C Slave critical error

## Description

[Refer to Diagnostic class (standard): 3](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063169)

An unrecoverable C2C Slave error is detected.

The extended diagnostic provides information on the cause:

| Ext. diagnosis | Meaning |
| --- | --- |
| `Max modules` | The number of modules that can be attached to a C2C Slave is exceeded. |
| `Init C2C` | The initialization of the C2C Slave was unsuccessful. |
| `Wrong FPGA` | The FPGA does not support the synchronization of the subnetwork to the superordinate network. |
| `FPGA no sync` | The synchronization functionality (synchronization of the subnetwork to the superordinate network) was not found on the FPGA. |

EIO0000002335.11
