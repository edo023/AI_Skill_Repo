# State

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Cam Track |
| Traceable | Yes |

## Functional Description

Displays the status of the Cam Track processing in plain text. The error text of any invalid parameters is displayed.

| Value | Data type | Meaning |
| --- | --- | --- |
| ready / 0 | DINT |  |
| not checked / 1 | DINT |  |
| invalid PositionSource / 2 | DINT |  |
| invalid Destination / 3 | DINT |  |
| invalid bit number / 4 | DINT |  |
| invalid PositionType / 5 | DINT |  |

EIO0000002335.11
