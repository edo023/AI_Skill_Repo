# EnableLoad

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam switch group |
| Traceable | Yes |

## Functional Description

Enables/disables the measuring function (Load, Cycle, MaxLoad).

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The measuring function is not active. |
| on / 1 | BOOL | The measuring function is active.  EnableLoad "off / 0" -> "on / 1" resets MaxLoad. |
