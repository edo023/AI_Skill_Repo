# SyncSupported

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave |
| Traceable | Yes |

## Functional Description

Enables / disables the sync mode. If the sync mode is enabled, the sync command is supported.

| Value | Data type | Meaning |
| --- | --- | --- |
| TRUE | BOOL | The sync mode is activated. |
| FALSE | BOOL | The sync mode is deactivated. |

EIO0000002335.11
