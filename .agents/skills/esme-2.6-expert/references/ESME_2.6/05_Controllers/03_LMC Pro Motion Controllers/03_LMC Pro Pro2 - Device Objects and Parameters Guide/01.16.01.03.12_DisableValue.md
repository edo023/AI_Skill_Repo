# DisableValue

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | CamTrack |
| Traceable | Yes |

## Functional Description

Defines the value of the output or the track with Cam Track disabled.

| Value | Data type | Meaning |
| --- | --- | --- |
| OffLevel / 0 | BOOL | When disabling the Cam Track, the output is set to the negated OnLevel.  **Example:**  OnLevel = "high / 1" -> when being disabled, the output is set to "low / 0".  OnLevel = "low / 0" -> when being disabled, the output is set to "high / 1". |
| current value / 1 | BOOL | The current value of the output is retained when Cam Track is disabled. |
