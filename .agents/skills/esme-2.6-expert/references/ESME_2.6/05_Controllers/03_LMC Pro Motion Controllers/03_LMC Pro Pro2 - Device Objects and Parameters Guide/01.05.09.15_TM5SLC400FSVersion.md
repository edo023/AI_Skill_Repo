# TM5SLC400FSVersion

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | No |

## Functional Description

The parameter TM5SLC400FSVersion contains the firmware version number of a TM5CSLC400FS Safety Logic Controller. This firmware version is used by the FDR function.

When the FDR function is executed, the TM5CSLC400FS Safety Logic Controller is updated to this firmware version.

The firmware for the update of the Sercos devices is stored on the memory card of the controller.

| Value | Data type | Meaning |
| --- | --- | --- |
| “none” | STRING(40) | No firmware for the TM5CSLC400FS Safety Logic Controller is contained in the memory card of the controller. The FDR function does not perform a firmware update for the TM5CSLC400FS Safety Logic Controller. |
| “x.x”  (default) | STRING(40) | Firmware version used by the FDR function ( "x.x" represents the version number). |
