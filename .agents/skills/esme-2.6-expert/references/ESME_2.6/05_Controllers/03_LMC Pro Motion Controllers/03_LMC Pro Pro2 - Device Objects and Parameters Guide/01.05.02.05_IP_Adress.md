# IP\_Adress

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the IP address.

The parameter is a display parameter only. The IP address is saved in file sysc3.cfg on the controller. An IP address is a 32-bit number that consists of four numbers from 0 to 255, separated by dots.

NOTE: The string cannot begin with zero and cannot contain any spaces. The controller must be reset to accept the parameter. Confirm with the reset button.

NOTE: IP addresses with a 0 on the first or last position are not valid.

IP addresses with the format 127.x.x.x are invalid (local host).

If with the an invalid IP address is set on the memory card, the default IP address is used for booting.

If via Diagnostics or in the EcoStruxure Machine Expert dialog box Controller selection an invalid IP address is configured, the entered IP address and subnet mask are ignored.

If under Controller selection > Edit communication settings the option Save settings permanently is activated, the values are not saved.

## Example

"190.205.11.9"

EIO0000002335.11
