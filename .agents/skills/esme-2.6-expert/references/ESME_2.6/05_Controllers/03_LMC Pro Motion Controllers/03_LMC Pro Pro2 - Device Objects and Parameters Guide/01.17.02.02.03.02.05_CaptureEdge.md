# CaptureEdge

## General

|  |  |
| --- | --- |
| Type | AD |
| Traceable | Yes |

## Functional Description

Displays the status of the configured edge.

| Value | Data type | Meaning |
| --- | --- | --- |
| Falling edge / 0 | DINT | Falling edge is configured. |
| Rising edge / 1 (default) | DINT | Rising edge is configured. |
| Reserved / 2 | DINT | Reserved |
| In progress / 3 | DINT | In progress (for example if service channel transfer is in progress). |
| Set failed / 4 | DINT | Setting unsuccessful (if service channel transfer is not successful). |
