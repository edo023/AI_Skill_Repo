# ConfiguredSerialNumber

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Traceable | No |

## Functional Description

If IdentificationMode = SerialNumberController / 0, the value of ConfiguredSerialNumber is used for identifying the device in the Sercos network (see parameter [IdentificationMode](IdentificationMode-2F96D150.html#IdentificationMode-2F96D150)). It is applied in the Sercos phase change from 1 to 2.

| Data type | Meaning |
| --- | --- |
| STRING [80] | The configured (logical) serial number for identifying the device in the Sercos network. |
