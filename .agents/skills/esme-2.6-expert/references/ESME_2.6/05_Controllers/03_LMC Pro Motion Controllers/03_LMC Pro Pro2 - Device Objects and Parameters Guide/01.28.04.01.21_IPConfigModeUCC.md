# IPConfigModeUCC

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Logic Motion Controller Device |
| Traceable | No |

## Functional Description

Used to define the IP addressing mode for the UCC (**U**nified **C**ommunication **C**hannel) communication.

| Value | Data type | Meaning |
| --- | --- | --- |
| Automatic IP addressing / 0  (default) | DINT | The slave settings are performed automatically.  The automatically generated configuration is displayed in IPAddressUCC and SubnetmaskUCC.  Access type: read/write |
| Manual IP addressing (hardware) / 1 | DINT | The settings are read from the slave device and displayed in IPAddressUCC and SubnetmaskUCC.  The settings must be performed on the hardware (for example, using switches) or via the drive assistant.  Access type: read/write |
| Manual IP addressing (software) / 2 | DINT | The settings IPAddressUCC and SubnetmaskUCC are transferred to the slave.  Access type: read/write |
