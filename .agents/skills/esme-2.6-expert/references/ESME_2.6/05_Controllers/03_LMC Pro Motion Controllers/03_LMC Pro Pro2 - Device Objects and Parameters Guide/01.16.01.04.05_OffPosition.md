# OffPosition

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam switch |
| Traceable | Yes |

## Functional Description

Specifies the switch-off position (Posmode) and the end of the activation window (Pulsemode) for this Cam switch. The operating mode (Posmode or Pulsemode) is set using the parameter Mode.

EIO0000002335.11
