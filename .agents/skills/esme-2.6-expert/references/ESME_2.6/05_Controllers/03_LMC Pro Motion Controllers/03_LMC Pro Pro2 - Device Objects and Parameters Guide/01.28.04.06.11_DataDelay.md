# DataDelay

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | C2C Encoder input |
| Traceable | Yes |

## Functional Description

Displays the data delay to the original encoder signal in [ms]. The delay is identical for the C2C Encoder Input objects with the same ID in the entire C2C network (C2C Master and C2C Slaves).

Thus, the encoder signal is provided in the entire C2C network at the same time.
