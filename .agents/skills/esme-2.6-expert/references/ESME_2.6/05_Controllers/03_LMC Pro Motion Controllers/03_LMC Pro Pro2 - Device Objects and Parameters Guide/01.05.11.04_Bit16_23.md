# Bit16\_23

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C |
| Traceable | No |

## Functional Description

Displays the values of inputs 16 to 23. Each bit is assigned to an input corresponding to the bit number.

Assigning bits to inputs Bit16\_23

|  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| Parameter bit number | Bit 23 | Bit 22 | Bit 21 | Bit 20 | Bit 19 | Bit 18 | Bit 17 | Bit 16 |
| Digital input(1) | FL\_3 | FL\_2 | FL\_1 | FL\_0 | DI\_19 | DI\_18 | DI\_17 | DI\_16 |
| **1.** affected parameter: Value | | | | | | | | |

## Example

Bit16\_23 = 2#10100000

The [value of the inputs](D-SE-0075304.html#D-SE-0075304) 21 and 23 is "H / TRUE"; inputs 16 to 20 and 22 are "L / FALSE"
