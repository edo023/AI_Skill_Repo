# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam switch group |
| Traceable | Yes |

## Functional Description

Enables / disables the entire cam switch group processing.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The cam switch group is not activated. |
| on / 1 | BOOL | The cam switch group is activated. |

All cam switches (Cam switch) associated with a cam track (Cam Track) can be jointly enabled / disabled with the parameter CamSwitchEnable of the cam track.
