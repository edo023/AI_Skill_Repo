# SSType

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | COM |
| Traceable | Yes |

## Functional Description

Selects the interface type.

| Value | Data type | Meaning |
| --- | --- | --- |
| RS 232 / 0 | DINT | Only if parameter Device is set to COM 1 (not supported) |
| RS 485 4 wire / 1 | DINT | Only if parameter Device is set to COM 2 |
| RS 485 2 wire / 2 | DINT | Only if parameter Device is set to COM 2 |

The interface type RS 485 2 wire activates two special functions:

* Automatic TriState controlling of the transmitter. After the transmission of the characters in the transmission buffer, the transmitter is automatically returned to the highly resistive state.
* Automatic extraction of the number of sent characters from the input current.
