# Edge

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C |
| Traceable | Yes |

## Functional Description

The parameter enables the active edge at which an interrupt should be triggered to be preset.

| Value | Data type | Meaning |
| --- | --- | --- |
| positive / 0 | BOOL | Positive edge |
| negative / 1 | BOOL | Negative edge |
