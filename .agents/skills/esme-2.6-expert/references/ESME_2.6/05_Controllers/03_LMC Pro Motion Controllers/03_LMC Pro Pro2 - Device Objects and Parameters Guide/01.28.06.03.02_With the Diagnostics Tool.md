# With the Diagnostics Tool

## Description

For detailed information on the Diagnostics tool, refer to the EcoStruxure Machine Expert Diagnostics User Guide.

To use the Diagnostics tool for the one cable solution in order to access C2C slaves without direct cable connection, you need a running C2C system. Refer to [Prerequisites](D-SE-0088170.html#D-SE-0088170).

## Retrieving Diagnostic Information From Single Controllers

| Step | Action |
| --- | --- |
| 1 | Connect your Ethernet cable to one of the controllers in the C2C network (Master or Slave). |
| 2 | Open the EcoStruxure Machine Expert Diagnostics tool. |
| 3 | Click **Collect data**  Result: The controllers in your network are listed in the **Controller selection** window. |
| 4 | Select one of the controllers (double-click an item) and click the **Connect** button.  Result: Diagnostic information of the controller is displayed. |
| 5 | To retrieve diagnostic information from another controller, click the **Home** button, select another controller from the list and click the **Connect** button.  Result: Diagnostic information of the controller is displayed. |

## Retrieving diagnostic information from the C2C Slaves via C2C Master communication

For detailed information on the EcoStruxure Machine Expert Diagnostics tool (using the Sercos topology views) refer to the EcoStruxure Machine Expert Diagnostics User Guide\Sercos Topology Views.

The **Sercos topology** view of the EcoStruxure Machine Expert Diagnostics tool allows you to retrieve diagnostic information from the C2C Slaves over the C2C Master diagnostic instance.

| Step | Action |
| --- | --- |
| 1 | Connect your Ethernet cable to one of the controllers in the C2C network (Master or Slave). |
| 2 | Open the EcoStruxure Machine Expert Diagnostics tool. |
| 3 | Click **Collect data**  Result: The controllers in your network are listed in the **Controller selection** window. |
| 4 | Select one of the controllers (double-click an item) and click the Connect button. |
| 5 | Open the **Sercos topology** view. |
| 6 | Right-click one of the C2C Slave controllers and select **Collect data with new instance of Diagnostics**.  Result: A new instance of Diagnostics connects to the selected C2C Slave and retrieves its diagnostic information. |
