# Check

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam Track |
| Traceable | Yes |

## Functional Description

If the parameters PositionSource, PositionType, Destination, or Bit number of the cam track (Cam Track) are changed while the cam switch group is enabled, then these parameters must be checked in order to detect the error in the configuration.

While the parameters are checked, the current configuration and function of the cam switch group are not influenced.

If an error occurs during a check, the diagnostic message 8327 "CamTrack invalid PositionSource", 8341 "CamTrack invalid Position Type, 8328 "CamTrack invalid Destination", 8329 "CamTrack invalid Bitnumber") and the state is displayed in plain text in the State parameter. If the check returns no errors, the values (for Enable on / 1) are applied immediately after the check.
