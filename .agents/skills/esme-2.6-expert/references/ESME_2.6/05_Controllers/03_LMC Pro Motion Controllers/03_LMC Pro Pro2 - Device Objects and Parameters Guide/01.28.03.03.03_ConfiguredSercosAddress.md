# ConfiguredSercosAddress

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | C2C\_Slave |
| Traceable | Yes |

## Functional Description

Used to configure the Sercos address of the C2C Slave device.

The default value of the parameter is 1.

This parameter is the equivalent of a address setting switch at a Sercos device.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0 | UNIT(0...511) | The Sercos address is editable and can be configured by the master. |
| 1...511 | UNIT(0...511) | The Sercos address is fixed and cannot be configured by the master.  The new Sercos address is applied during [communication phase](D-SE-0073356.html#D-SE-0073356) 0 (superordinate network) and the parameter UsedSercosAddress displays this fixed value in/after communication phase 0 (superordinate network). |
