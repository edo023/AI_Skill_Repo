# State

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacNet |
| Traceable | Yes |

## Functional Description

Displays the PacNet status.

| Value | Data type | Meaning |
| --- | --- | --- |
| Not ready / 0 | DINT | The PacNet is not ready. |
| Initialization / 1 | DINT | The PacNet processing is initialized. |
| Ready / 2 | DINT | The PacNet is ready for operation. |
