# MasterSubnetmask

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

This parameter displays the subnet mask.

| Value | Data type | Meaning |
| --- | --- | --- |
| 255.255.252.0 (default) | STRING(15) | Displays the subnet mask used for the calculation of the address ranges.  Access type: read |
