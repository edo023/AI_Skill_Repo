# EngageStart

## General

|  |  |
| --- | --- |
| Type | ED |
| Devices supporting the parameter | Log. encoder |
| Traceable | Yes |

## Functional Description

Engages/disengages the master.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | When changing from "on / 1" to "off / 0," the factor is reduced from 1 to 0 in n EngagePeriod.  With "off / 0," the factor remains at 0. |
| on / 1 | BOOL | When changing from "off / 0" to "on / 1," the factor increases from 0 to 1 in n EngagePeriod.  With "on / 1," the factor remains at 1. |
