# DPV1Enable

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave |
| Traceable | Yes |

## Functional Description

Enables the DPV1 functionality. If noncyclical data is to be used, the DPV1 functionality has to be activated.

| Value | Data type | Meaning |
| --- | --- | --- |
| TRUE | BOOL | The DPV1 mode is activated. |
| FALSE | BOOL | The DPV1 mode is deactivated. |

EIO0000002335.11
