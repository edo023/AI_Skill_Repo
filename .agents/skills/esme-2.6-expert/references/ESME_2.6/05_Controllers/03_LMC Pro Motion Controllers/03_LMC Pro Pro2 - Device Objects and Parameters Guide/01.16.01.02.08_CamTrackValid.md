# CamTrackValid

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Cam switch group |
| Traceable | Yes |

## Functional Description

Displays the State Of the subordinate cam tracks (Cam Tracks) bit-by-bit.

Bit 0 = 1 means that for the first CamTrack, State = "ready / 0."

The data type of the parameter is UDINT (unsigned double integer 32 Bit). Each bit is assigned to a State parameter of a cam track (Cam Track). Bit 0 is assigned to the first cam track, bit 1 is assigned to the second cam track, and so on. The cam track which has been inserted directly underneath the cam switch group (Cam switch group) in the PLC configuration is the first cam track.
