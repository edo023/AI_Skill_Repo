# IdentifyDevice

## General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | No |
| Traceable | Yes |

## Functional Description

The parameter enables and disables the device identification in order to facilitate identifying Sercos slaves in the machine.

The precondition for this functionality is an operational Sercos real-time communication.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | Default value; device identification disabled. |
| on / 1 | BOOL | Device identification by the orange flashing of the S3 LED on the device is enabled. |
