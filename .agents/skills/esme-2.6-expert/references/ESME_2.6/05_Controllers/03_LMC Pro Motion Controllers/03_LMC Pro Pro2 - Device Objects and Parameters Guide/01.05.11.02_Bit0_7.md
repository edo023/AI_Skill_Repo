# Bit0\_7

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the values of inputs 0 to 7. Each bit is assigned to an input corresponding to the bit number.

Assigning bits to inputs Bit0\_7

|  |  |  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| Parameter bit number | Bit 7 | Bit 6 | Bit 5 | Bit 4 | Bit 3 | Bit 2 | Bit 1 | Bit 0 |
| Digital input(1) | DI\_7 | DI\_6 | DI\_5 | DI\_4 | DI\_3 | DI\_2 | DI\_1 | DI\_0 |
| **1.** affected parameter: Value | | | | | | | | |

## Example

Bit0\_7 = 2#10000001

The [value of the inputs](D-SE-0075304.html#D-SE-0075304) 0 and 7 is "H / TRUE"; inputs 1 to 6 are "L / FALSE"
