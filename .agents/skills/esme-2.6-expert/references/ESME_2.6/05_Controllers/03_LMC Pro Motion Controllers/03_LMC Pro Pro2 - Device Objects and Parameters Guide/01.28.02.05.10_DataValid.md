# DataValid

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | C2C Data Input, C2C Data output |
| Traceable | Yes |

## Functional Description

Indicates whether the received data is valid.

| Value | Data type | Description |
| --- | --- | --- |
| no / 0 | BOOL | Error detected when receiving data. |
| yes / 1 | BOOL | The data set is valid. |
