# AutoRun

## General

|  |  |
| --- | --- |
| Type | ER |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable: | Yes |

## Functional Description

Automatic program start after creating the control voltage.

NOTE: This parameter is a reset parameter (ER). Changes will take effect only after a reset of the controller. To save the values as initialization values after changing the value of the parameter via online change or in the application, the function *[FC\_SysSaveParameter](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085355)* has to be called.

## Presetting

The presetting for AutoRun = FALSE.

## Commissioning

While preparing the application and during commissioning, AutoRun must be set to false.

## Regular Machine Operation

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  Use appropriate safety interlocks where personnel and/or equipment hazards exist.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

During regular machine operation the parameter AutoRun = TRUE. Otherwise the start of the application can only take place via Logic Builder.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | No automatic program start |
| yes / 1 | BOOL | Automatic program start after booting the PacDrive controller |
