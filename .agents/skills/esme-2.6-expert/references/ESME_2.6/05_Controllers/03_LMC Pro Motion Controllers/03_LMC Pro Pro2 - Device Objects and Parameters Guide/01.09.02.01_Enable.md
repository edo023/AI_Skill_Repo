# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Virt. Master Encoder |
| Traceable | Yes |

## Functional Description

Enables the encoder.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The encoder is not processed by the real-time process.  The [Velocity](D-SE-0075690.html#D-SE-0075690) of the encoder is set to 0. |
| on / 1 | BOOL | The encoder is processed by the real-time process. |
