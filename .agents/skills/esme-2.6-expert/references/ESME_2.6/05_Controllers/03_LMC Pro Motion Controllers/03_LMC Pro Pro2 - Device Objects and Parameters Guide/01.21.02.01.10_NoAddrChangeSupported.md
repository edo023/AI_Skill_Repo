# NoAddrChangeSupported

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave |
| Traceable | Yes |

## Functional Description

Using the parameter NoAddrChangeSupported, you can choose whether the address change is to be supported.

| Value | Data type | Meaning |
| --- | --- | --- |
| FALSE | BOOL | The address change is supported. |
| TRUE | BOOL | The address change is not supported. |
