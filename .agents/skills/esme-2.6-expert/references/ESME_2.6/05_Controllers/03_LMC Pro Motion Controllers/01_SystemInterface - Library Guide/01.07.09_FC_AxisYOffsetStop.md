# FC\_AxisYOffsetStop

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Stop the Y-offset for an axis.

## Description

This function stops the Y-offset generator of the axis i\_stAxisId with i\_rDeceleration.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Logical address of the axis |
| i\_rDeceleration | REAL | Maximum deceleration |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: i\_stAxisId is invalid |

EIO0000002680.05
