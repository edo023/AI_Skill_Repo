# FC\_TaskStop

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.35.2.0 |
| Versions: | Current version |

NOTE: This function is available as of version V.1.35.2.0.

In previous versions, the function returns the value "-80".

## Task

Stopping one or more tasks.

## Description

The task (i\_sTaskName) is stopped. If no task name is transferred in i\_sTaskName, the tasks are stopped. Use i\_xSelf to specify whether the task that triggers the stop and calls the function FC\_TaskStop() is to be stopped as well. i\_xSelf is only considered if no task name is transferred.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sTaskName | STRING[80] | Name of the task |
| i\_xSelf | BOOL | Defines whether the task that triggers the stop is to be stopped as well. |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: invalid task name |
