# FC\_SercosDiagnosisGetCounterIPChViolationPort2

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.64.16.4 |
| Versions: | Current version |

## Description

Returns the number of Ethernet frames which are out of the IP channel time boundaries since the last call of FC\_SercosDiagnosisResetErrorCounters on port 2.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | Number of Ethernet frames which are out of the IP channel time boundaries |

EIO0000002680.05
