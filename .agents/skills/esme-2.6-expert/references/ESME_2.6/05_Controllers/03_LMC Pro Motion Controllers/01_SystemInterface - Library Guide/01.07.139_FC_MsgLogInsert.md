# FC\_MsgLogInsert

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Insert the message in the message log file.

## Description

The message text transferred in i\_sMessage is inserted in the message log file.

| Column | Entry |
| --- | --- |
| Time mark | Added automatically by the system. |
| Type | 0x0004 |
| Object | IECPRG |
| Instance | TskDefault |
| Diag. Code | <no entry; reserved> |
| Ext. diagnostic | <no entry; reserved> |
| Message text | Message |

The messages inserted using this function can be hidden using the IEC prg system function blocks filter (filter type 3) in the message logger.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sMessage | STRING [80] | Message text  NOTE: The semicolon “;” can not be used in the message text. |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK |

EIO0000002680.05
