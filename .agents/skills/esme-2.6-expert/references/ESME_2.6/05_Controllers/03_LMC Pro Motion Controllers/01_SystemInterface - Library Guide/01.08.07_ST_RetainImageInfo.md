# ST\_RetainImageInfo

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | SystemInterface\_1.32.6.0 |
| Support for: | * FC\_GetRetainImageInfo |
| Versions: | Current version |

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| sFileName | STRING[255] := '' | File name of the retain image in the format 8.3 |
| dtRetainSaveDateTime | DATE\_AND\_TIME := DT#2007-03-19-12:00:00 | Date of creation of the retain image in the Date\_and\_Time format. If no RTC is available on the controller, then DT#1970-01-01-00:00 is stored. |
| sHardwareType | STRING[80]:= '' | Controller type (for example, PacDrive LMC600) |
| sSetRetainSize | STRING[80]:= '' | Value of the parameter SetPacDriveIECRetainSize or the parameter SetIECRetainSize, depending on the controller type. |
| sUserInfo | STRING[255] := '' | The text that was specified when creating the retain image. |
| sMD5CheckSum | STRING[33] := '' | Checksum value of the retain data. |
| diLengthRetainData | DINT := 0 | Number of the retain data in bytes. |
