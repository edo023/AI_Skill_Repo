# Firewall

## Description

The firewall feature provides the following functions:

* FC\_FirewallSetActive

  To activate/deactivate the firewall.
* FC\_FirewallIsActive

  To get the status of the firewall.
* FC\_OpenIncomingFirewallPort

  To open incoming firewall port during runtime.
* FC\_CloseIncomingFirewallPort

  To close incoming firewall port during runtime.
