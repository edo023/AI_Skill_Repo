# FC\_GetDeviceSpecificTimestamp

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.84.11.0 |

## Description

This function lets you correlate the timing in the Real Time Process (RTP) of the controller and the cycle of an Edge I/O network interface module (NIM) or of a TM5 bus interface.

The function uses a time offset i\_rtpOffset that is relative to the start of the ongoing RTP cycle. It calculates an absolute timestamp which is valid in the time clock of the given device.

Calculation: iq\_timestamp = [start of RTP cycle] + i\_rtpOffset

The input i\_checkIfValidOutputTimestamp lets you verify whether the resulting timestamp can be used as an output timestamp to generate an output edge. For example, the resulting timestamp may be too close to the present time and would arrive at the output registers of the output module after the intended time. In such a case, an error is detected.

If the function returns a value less than 0, the value of iq\_timestamp is invalid.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_device | ST\_LogicalAddress | Identifies the device. |
| i\_rtpOffset | DWORD | Time offset relative to start of the RTP cycle in which the function is executed.  Value range Edge I/O NIM: 0...999999999  Value range TM5 bus interface: 0...32767000  Unit: ns |
| i\_checkIfValidOutputTimestamp | BOOL | TRUE: Verification of validity of timestamp activated.  FALSE: Verification of validity of timestamp deactivated. |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_timestamp | DWORD | Calculated timestamp |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: No error detected  -1: The device addressed with i\_device was not found.  -2: Communication with device addressed with i\_device was not possible.  -3: The device addressed with i\_device is not present on the bus.  -4: The value for i\_rtpOffset was too low.  -5: The value for i\_rtpOffset was too high.  -6: Verification with i\_checkIfValidOutputTimestamp is not supported. |
