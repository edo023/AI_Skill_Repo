# FC\_SercosReadServiceData

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |

## Description

FC\_SercosReadServiceData reads service data via Sercos that are used for debugging.

Also refer to [Error codes of the Sercos parameter channel (service channel)](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063997).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stLogAddr | ST\_LogicalAddress | Logical address of the axis |
| i\_dwIDN | DWORD | Data Service IDN |
| i\_wType | WORD | Data Service Type   * 0: read name * 1: read attribute * 2: read unit * 3: read min. value * 4: read max. value * 5: read user data * 6: read user data with variable length * 7: execute command |
| i\_dwPointerToData | DWORD | Pointer on the data memory, filled with Len data bytes |
| i\_wMaxDataLen | WORD | Len data bytes to be read |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_uiReadDataLen | UINT | UINT variable for reading out the data length of the Sercos parameter (i\_wType=6) |
| iq\_uiMaxReadDataLen | UINT | UINT variable for reading out the maximum data length of the Sercos parameter (i\_wType=6) |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: The logical address is invalid.  -431: Detected service request error (for example, timeout)  -445: Service timeout  -461: In the current phase, reading out parameters via the service channel is not supported.  -462: The addressed axis does not support the ServiceDataRead function.  -464: Invalid service transfer |
