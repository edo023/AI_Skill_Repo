# FC\_LzsTaskGetCurrentInterval

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine the interval time of the task in the program.

## Description

The interval of a task is typically specified in milliseconds in the task configuration. You can read the configured task interval using FC\_LzsTaskGetInterval().

If a task has been configured as "External", then the interval can no longer be detected by the FC\_LzsTaskGetInterval(). Depending on the trigger, this can be a constant or a variable interval.

One way to read the configured interval of an "External" Task is provided by the function FC\_LzsTaskGetCurrentIntervall(). It provides the configured interval in microseconds. The function returns -1 for tasks with a variable interval.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | Returns the value of the interval time of an internal task, or:  INIRQ1: -1  INIRQ2: -1  INIRQ3: -1  INIRQ4: -1  INIRQ1\_4: -1  RTP\_READ: Provides the RTP value in Sercos phase 4  RTP\_MENC: Provides the RTP value in Sercos phase 4  RTP\_LENC: Provides the RTP value in Sercos phase 4  RTP\_AXIS: Provides the RTP value in Sercos phase 4  CAMSWITCHGROUP0: Value of the CamSwitchGroupx.Interval parameter whether the CamSwitchGroup is active or not  CAMSWITCHGROUP1: Value of the CamSwitchGroupx.Interval parameter whether the CamSwitchGroup is active or not  MDT\_WRITE\_ACCESS: Provides the RTP value in Sercos phase 4 |

## Example

A task with 50 ms cycle time is implemented in a program. The system is to read the interval of the task.

```
diIntervalMs:DINT:=0;
diCurrentIntervalUs:DINT:=0; 
diIntervalMs := FC_LzsTaskGetInterval();(* provides 50 *)
diCurrentIntervalUs := FC_LzsTaskGetCurrentInterval(); (* provides 50000 *)
```
