## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| ContrNotReady | 0 | Drive is not ready to be energized. Possibly the parameter MotorIdentification is set to "motorless / 1".  Drives with external power supply (for example, LXM62DxS or Lexium ILM62 Drive Module): waiting for power supply check = TRUE of the corresponding power supply. |
| WaitForEnable | 1 | Power unit ready, drive is torque free, DC bus ready, waiting for release (TorqueEnable + ControllerEnable) |
| WaitForBreakRelease | 2 | The drive is in position control, but does not follow the specified reference values.  When energizing the power supply, the torque is established and the brakes are released. When de-energizing the power supply, the drive is in the state of drive controlled standstill. |
| WaitForNewJob | 3 | Waiting for master job |
| WaitForStart | 4 | Waiting for start |
| MstJobExecuting | 5 | Processing master job |
| MstSlvJobExecuting | 6 | Processing master/slave job |
| WaitUntilStopped | 7 | Canceling jobs, drive stopping |
| WaitForStopReset | 8 | Job processing blocked, drive stopped |
