# FC\_SysStartFunc

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Calling up system-internal functions.

## Description

Call: SysStartFunc(pFunc, IArg1,..., IArg9);

Function: The function \*pFunc is called with the parameters IArg1 to IArg9.

RETURN Result: The value of (\*pFunc)(...)

## Interface

| Input | Data type | Description |
| --- | --- | --- |

| i\_diArg3 | DINT | Parameter 3 |
| i\_diArg4 | DINT | Parameter 4 |
| i\_diArg5 | DINT | Parameter 5 |
| i\_diArg6 | DINT | Parameter 6 |
| i\_diArg7 | DINT | Parameter 7 |
| i\_diArg8 | DINT | Parameter 8 |
| i\_diArg9 | DINT | Parameter 9 |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | -1: Invalid function address  -: All other return values correspond to the return value of the function that has been called up. |
