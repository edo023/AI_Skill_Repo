# FC\_StLogicalAddressToLogAdr

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Convert the format of a logical address from ST\_LogicalAddress to DINT.

## Description

Converts the logical address i\_stLogicalAddress that was transferred in the new format (ST\_LogicalAddress) into the old format (DINT).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stLogicalAddress | ST\_LogicalAddress | Logical address whose format is to be changed |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | >=0: converted address in the old format (DINT)  -1: Conversion not possible |
