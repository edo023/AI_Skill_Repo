# FC\_ComIecOpen

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Open the logical device.

## Description

The logical device named ComIecId is opened. The call is passed on to the serial driver. The serial driver uses the current parameters to open the device.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stComIecId | DINT | Device selection |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: i\_stComIecId invalid  -2: other error detected. Contact your Schneider Electric representative. |
