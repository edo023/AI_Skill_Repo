# FC\_UserAdd

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Add a new user to the controller.

## Description

With this function, a new user is created at the controller.

This "new user" can then be used for remote services like remote file device (for example, for an FTP client).

The new user has only Group > Everyone user rights.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sUserName | STRING[80] | User name |
| i\_sPassword | STRING[80] | Password |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: A new user was successfully added.  -1: User already exists. Delete the user already existing with this user name before adding a new user.  -2: User name invalid. The name string is empty. The string must contain at least one character.  -4: Password invalid. The password string is empty. The string must contain at least one character.  -8: UserRightsManagement is deactivated (controller).  -10: User management component is not loaded.  -99: Internal error detected. Contact your Schneider Electric representative. |
