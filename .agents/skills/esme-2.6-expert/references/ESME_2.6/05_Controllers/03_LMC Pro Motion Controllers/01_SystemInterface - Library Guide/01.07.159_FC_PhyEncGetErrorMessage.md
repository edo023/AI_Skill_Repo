# FC\_PhyEncGetErrorMessage

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Reading the error code from the encoder.

## Description

The error code of encoder i\_stEncId is read out.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stEncId | ST\_Logical Address | Logical address of the encoder |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_byErrorMessage | BYTE | Error code of the encoder |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: job started  FALSE: job interface is busy (try again) or error |
