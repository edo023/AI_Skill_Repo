# FC\_GetAxisState

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine the state of the drive.

## Description

Provides the current state of the drive. The numbering of the states matches the one of the parameter *[AxisState](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071486)*.

## Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Logical address of the drive |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0 - 8: see PLC configuration parameter AxisState  -1: Logical address of the drive is invalid |
