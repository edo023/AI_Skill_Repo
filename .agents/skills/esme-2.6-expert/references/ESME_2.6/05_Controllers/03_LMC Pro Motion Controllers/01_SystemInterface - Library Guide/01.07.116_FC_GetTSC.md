# FC\_GetTSC

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Retrieve the highly precise Time Stamp Counter.

## Description

The function reads the lower-value 32 bit from the internal processor cycle counter. The x86 processor has a 64-bit timer that is increased by every clock cycle.

## Return Value

| Data type | Description |
| --- | --- |
| UDINT | Counter value of the processor cycle counter (lower-value 32 bit) |

## Example

See function FC\_GetNsPerCPUClockCycle().
