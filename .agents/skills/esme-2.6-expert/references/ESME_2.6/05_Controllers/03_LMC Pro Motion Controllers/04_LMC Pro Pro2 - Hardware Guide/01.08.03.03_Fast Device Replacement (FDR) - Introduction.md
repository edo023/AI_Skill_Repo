# Fast Device Replacement (FDR) - Introduction

## Introduction

With the help of the Fast Device Replacement, the Lexium 62, Lexium 52 and ILM devices that are in the configuration of an EcoStruxure Machine Expert project in the controller can be replaced.

There are certain parameters that have to be set in EcoStruxure Machine Expert first. Information on this can be found in the online help of EcoStruxure Machine Expert.

Subsequently, certain settings on the display of the controller have to be made which are described below.

The controller interface for FDR allows you to manually access the assignment between logical devices in the controller configuration (EcoStruxure Machine Expert Logic Builder) and the physical connected devices.
