# FC\_SercosDiagnosisGetCounterMSTMissPort2

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.64.16.3 |
| Versions: | Current version |

## Description

Returns the number of unreceived MSTs or MSTs with invalid CRC since the last call of FC\_SercosDiagnosisResetErrorCounters on port 2.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | Number of unreceived MSTs or MSTs with invalid CRC |

EIO0000002680.05
