# FC\_MemoryCmp

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Compare the contents of two memory areas.

## Description

With this function, the contents of two memory areas are compared byte-by-byte.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_dwBuf1 | DWORD | Initial address of the first memory area |
| i\_dwBuf2 | DWORD | Initial address of the second memory area |
| i\_dwCount | DWORD | Number of bytes to be compared |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: if the elements are the same  >0: if the element of the first memory area is greater  <0: if the element of the second memory area is greater |

## Examples

```
dwResult : DINT;
dwResult := FC_MemoryCmp(pbyAddresse1,pbyAddresse2,dwSize);
```
