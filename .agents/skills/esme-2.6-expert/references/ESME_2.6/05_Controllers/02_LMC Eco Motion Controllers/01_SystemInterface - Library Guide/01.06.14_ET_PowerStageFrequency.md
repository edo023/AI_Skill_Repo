# ET\_PowerStageFrequency

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | SystemInterface\_1.32.6.0 |
| Support for: | * FC\_PowerStageFrequencySet |
| Versions: | Current version |

## Description

The frequency of the power stage can be adapted to the demands of the user. The standard value is 8 kHz.

With a frequency of 4 kHz (when supported by the drive) the nominal current can be increased, but the axis becomes more noisy. The controller becomes less dynamic and the tracking deviation may increase.

With a frequency of 16 kHz (when supported by the drive) the axis becomes more steady, but nominal current is decreased. The dynamic of the controller is nearly the same as with 8 kHz.

The active power stage frequency is shown in the object parameter PowerStageFrequency.

NOTE: For a double drive the same value has to be used for both drives. When the frequency is changed for one axis of a double drive it is automatically set for the other one to the same value. The last set value is active.
