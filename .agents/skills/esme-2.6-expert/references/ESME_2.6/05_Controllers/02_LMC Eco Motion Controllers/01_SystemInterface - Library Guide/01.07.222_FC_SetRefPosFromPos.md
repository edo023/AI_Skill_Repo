# FC\_SetRefPosFromPos

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Set the reference position to the current position.

## Description

This sets [RefPosition](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071490) of the axis i\_stAxisId to the current position.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Logical address of the axis |

## Return Value

| Data type | Description |
| --- | --- |
| LREAL | -1: Logical address of the axis (i\_stAxisId) is invalid  Otherwise: RefPosition |
