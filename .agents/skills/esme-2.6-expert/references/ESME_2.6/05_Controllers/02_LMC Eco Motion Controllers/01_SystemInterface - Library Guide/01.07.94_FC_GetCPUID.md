# FC\_GetCPUID

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine the CPU type of the PacDrive Controller.

## Description

The function reads the CPU ID. This allows the CPU used in the PacDrive Controller to be detected in a program.

| Bit | Meaning |
| --- | --- |
| 0 ... 3 | Stepping |
| 4 ... 7 | Model |
| 8 ... 11 | Family |
| 12 ... 31 | Reserved |

These bits are unique for every CPU.

## Return Value

| Data type | Description |
| --- | --- |
| DWORD | CPUID |

EIO0000002680.05
