# FC\_SercosReadServiceDataByTopAddr

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.35.2.0 |

## Description

FC\_SercosReadServiceDataByTopAddr reads out data via the Sercos service channel from the addressed device.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_wTopoAddr | WORD | Topological address of the Sercos device |
| i\_dwIDN | DWORD | Data Service IDN |
| i\_wType | WORD | Data Service Type   * 0: read name * 1: read attribute * 2: read unit * 3: read min. value * 4: read max. value * 5: read user data * 6: read user data with variable length * 7: execute command |
| i\_dwPointerToData | DWORD | Pointer to the data memory, filled with the actual length of the data bytes |
| i\_wMaxDataLen | WORD | Actual length of the data bytes to read |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_uiReadDataLen | UINT | UINT variable to read out the data length of the Sercos parameter (i\_wType=6) |
| iq\_uiMaxReadDataLen | UINT | UINT variable to read out the maximum data length of the Sercos parameter (i\_wType=6) |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: The topological address is invalid.  -431: Error detected during service request (for example, timeout)  -445: Service timeout  -461: In the current phase, reading out parameters via the service channel is not supported.  -462: The addressed axis does not support the ServiceDataRead function.  -464: Invalid service transfer |
