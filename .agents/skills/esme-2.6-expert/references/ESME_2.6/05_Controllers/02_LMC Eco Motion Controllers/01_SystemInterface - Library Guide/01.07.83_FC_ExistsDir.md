# FC\_ExistsDir

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Check if a directory already exists on a data medium.

## Description

This function checks if a directory named i\_sDirname exists. If the directory exists, the function returns 0. If an error is detected, the function returns an error code. You can check if a directory exists on the following devices:

* ide0:
* ram0:

You cannot check if a directory exists on a RemoteDevice. The default device is "ide0:".

You must specify the device name, except for the default device names, in the i\_sDirname transfer parameter. Separate the device name and the directory name by a backslash "\". The transfer parameters do not permit wildcards.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sDirName | STRING[80] | Name of the directory whose existence has to be checked |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: Directory already exists  -1: The directory does not exist or a directory is checked on a remote file system  -2: It is a file name  -3: The wildcard is in the transfer parameters  -4: General error detected |

## Examples

```
 FC_ExistsDir('ide0:\Test');
```

Checks if the "ide0:\Test" directory exists.

```
 FC_ExistsDir('Test');
```

Checks if the "ide0:\Test" directory exists.

```
 FC_ExistsDir('Test1\Test2');
```

Checks if the "Test2" subdirectory exists in the "ide0:\Test1" directory.
