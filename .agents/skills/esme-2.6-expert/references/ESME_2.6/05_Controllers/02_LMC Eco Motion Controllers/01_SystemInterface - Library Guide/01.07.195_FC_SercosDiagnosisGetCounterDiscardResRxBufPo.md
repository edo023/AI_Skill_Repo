# FC\_SercosDiagnosisGetCounterDiscardResRxBufPort2

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.64.16.4 |

## Description

Returns the number of discarded receive Ethernet frames in case of missing receive buffer resources since the last call of FC\_SercosDiagnosisResetErrorCounters on port 2.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | Number of discarded receive Ethernet frames in case of missing receive buffer resources |
