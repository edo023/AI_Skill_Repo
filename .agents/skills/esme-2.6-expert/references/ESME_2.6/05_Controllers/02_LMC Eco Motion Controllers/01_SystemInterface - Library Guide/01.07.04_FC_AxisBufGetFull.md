# FC\_AxisBufGetFull

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Verify if the preparation buffer of an axis is full.

## Description

The axis is selected with i\_stAxisID. The function checks whether the preparation buffer of the axis which has been selected using i\_stAxisID is full.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| stAxisId | ST\_LogicalAddress | Selecting the axis |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: Buffer is full  FALSE: Buffer is not full |
