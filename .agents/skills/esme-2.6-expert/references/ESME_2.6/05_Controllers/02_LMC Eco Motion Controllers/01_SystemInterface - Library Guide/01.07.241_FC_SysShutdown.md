# FC\_SysShutdown

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Before de-energizing the controller (Power OFF of the control voltage), ensure that the file system or open files are closed correctly.

## Description

To de-energize the controller, the file system and the open files must be closed.

This is implemented in the FC\_SysShutdown() function. The function first triggers a diagnostic message [8023 "System shutdown"](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063801) for the message logger and then the IEC tasks are stopped. Open files and the file system are closed after which the CPU is stopped.

The system can then be disconnected from the power supply.

To restart the system again, a hardware reset on the front panel of the PacDrive controller is necessary or a Power OFF / Power ON of the control voltage.
