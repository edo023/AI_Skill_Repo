# FC\_AxisBufClear

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Delete the preparation buffer of an axis.

## Description

The axis is selected with i\_stAxisID. The buffer of the axis that has been selected using i\_stAxisID is deleted.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Selecting the axis |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: Error detected |

EIO0000002680.05
