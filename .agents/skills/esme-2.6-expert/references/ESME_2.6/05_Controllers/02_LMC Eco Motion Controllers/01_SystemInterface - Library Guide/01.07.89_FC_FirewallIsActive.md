# FC\_FirewallIsActive

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.56.13.0 |
| Versions: | Current version |

## Task

Get the status of the firewall.

## Description

The function returns the status of the firewall.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: Firewall is activated.  1: Firewall is disabled.  2: No active rules. The Firewall is activated but based on no rules entered into the firewall configuration the firewall has no function.  3: Configuration error detected. |

## Example

```
FirewallStatus := FC_FirewallIsActive();
IF ( FirewallStatus = 0 ) THEN
      FirewallStatus := FC_FirewallSetActive( FALSE );
END_IF
```
