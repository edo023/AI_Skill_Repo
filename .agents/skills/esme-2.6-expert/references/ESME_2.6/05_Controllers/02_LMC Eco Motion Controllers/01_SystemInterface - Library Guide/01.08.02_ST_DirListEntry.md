# ST\_DirListEntry

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | SystemInterface\_1.32.6.0 |
| Support for: | * FC\_DirListEntryGet |
| Versions: | Current version |

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| sFileName | STRING[255] | The name of the file |
| diFileSize | DINT | The size of the file in bytes |
| diFileType | DINT | Is the found entry a directory or a file?  Two predefined constants:   * 0 => File * 1 => Directory |
| diFileAttribute | DINT | Attributes of the file, for example, access attribute (write protection disabled or enabled) |
| dtFileDate | DATE\_AND\_TIME | Date and time of the last modification |
