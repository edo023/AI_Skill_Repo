# IF\_SercosDeviceIdentification - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_IdentificationMandatory](D-SE-0089168.html#D-SE-0089168) |

## Description

The interface manages the properties of the identification of the device.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ApplicationType | STRING[40] | Read | Indicates the application type. |
| ConfiguredApplicationType | STRING[40] | Read/Write | Sets the application type. |
| ConfiguredSercosAddress | UINT | Read/Write | Sets the Sercos address. |
| ConfiguredSerialNumber | STRING[80] | Read/Write | Sets the serial number of a drive found at the Sercos bus to which the motor is to be assigned. |
| ConfiguredTopologyAddress | UINT | Read/Write | Sets the topological address found at the Sercos bus to which the device is to be assigned. |
| IdentificationMode | DINT | Read/Write | Sets the identification mode. |
| IdentifyDevice | BOOL | Read/Write | Enables the device identification. |
| SercosAddress | UINT | Read | Indicates the Sercos address to which the device has been assigned. |
| SerialNumber | STRING[80] | Read | Indicates the serial number of the controller. |
| TopologyAddress | UINT | Read | Indicates the physical position of the device in the Sercos ring. |
| VendorCode | WORD | Read | Indicates the manufacturer code of the device. |
| VendorDeviceId | STRING[80] | Read | Indicates the Sercos device identifier. |
| WorkingMode | DINT | Read/Write | Sets the operating mode of the device. |
| WorkingState | DINT | Read | Indicates the working state. |
