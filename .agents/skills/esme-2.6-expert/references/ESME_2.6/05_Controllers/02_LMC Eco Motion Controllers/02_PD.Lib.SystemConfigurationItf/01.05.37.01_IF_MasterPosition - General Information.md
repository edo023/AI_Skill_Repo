# IF\_MasterPosition - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface indicates the position of the device (master).

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| Position | LREAL | Read | Indicates the position of the device (master).  See also:   * Position parameter of the incremental encoder (INC\_IN, INC\_OUT) * Position parameter of the sum encoder (P\_ENC, SENC) |
