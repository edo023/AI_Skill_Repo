# IF\_Mechanic - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_MechanicMandatory](D-SE-0089184.html#D-SE-0089184) |

## Description

The interface manages the mechanical properties, the moment of inertia and the friction.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| J\_Gear | REAL | Read/Write | Sets the moment of inertia at the gear box input shaft. |
| J\_Load | REAL | Read/Write | Sets the load moment of inertia at the gear box output shaft. |
| StaticFriction | REAL | Read/Write | Sets the static friction at the gear box output shaft. |
| ViscousFriction | REAL | Read/Write | Sets the viscous friction at the gear box output shaft. |
