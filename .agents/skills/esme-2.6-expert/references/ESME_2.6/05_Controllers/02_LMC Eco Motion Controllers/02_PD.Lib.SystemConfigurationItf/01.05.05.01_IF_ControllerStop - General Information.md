# IF\_ControllerStop - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the properties of the drive stop.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ControllerStop | BOOL | Read | Indicates the drive stop. |
| ControllerStopDec | LREAL(0..1000000000) | Read/Write | Sets the maximum absolute value of the deceleration in units/s2 with which the drive is stopped. |
| ControllerStopJerk | LREAL(0..1000000000000) | Read/Write | Sets the absolute value of the jerk in units/s3 with which the drive is stopped. |
| ControllerStopSet | BOOL | Read/Write | Sets an emergency stop of the controller. |
