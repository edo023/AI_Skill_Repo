# IF\_ControlLoop - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the properties of the control loop.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| CurrRefFilter | REAL(0..9.9) | Read/Write | Sets the filter that affects the reference current of the velocity controller. |
| Curr\_I\_Gain | REAL(0..100) | Read/Write | Sets the integral gain of the current controller. |
| Curr\_P\_Gain | REAL(0..100) | Read/Write | Sets the proportional gain of the current controller. |
| Pos\_P\_Gain | REAL(0..100) | Read/Write | Sets the proportional gain of the position controller. |
| VelFilter | REAL(0..9.9) | Read/Write | Sets the filter that affects the velocity in the feedback of the velocity controller. |
| Vel\_I\_Gain | REAL(0..250) | Read/Write | Sets the integral gain of the velocity controller. |
| Vel\_P\_Gain | REAL(0..250) | Read/Write | Sets the proportional gain of the velocity controller. |
