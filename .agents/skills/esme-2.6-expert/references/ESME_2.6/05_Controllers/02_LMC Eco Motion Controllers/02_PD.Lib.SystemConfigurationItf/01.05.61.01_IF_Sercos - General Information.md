# IF\_Sercos - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_IdentificationMandatory](D-SE-0089168.html#D-SE-0089168) |

## Description

The interface manages the properties of the Sercos bus.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| CycleTime | DINT | Read | Indicates the Sercos bus cycle time in ns. |
| PhaseSet | DINT | Read/Write | Sets the desired communication phase of the Sercos bus. |
| State | DINT | Read | Indicates the operating state of the Sercos bus. |
