# IF\_Brake - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the properties of the holding brake.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| BrakeCouplingTime | DINT | Read/Write | Sets the coupling time of the holding brake. |
| BrakeDisconnectionTime | DINT | Read/Write | Sets the disconnection time of the holding brake. |
| BrakeState | BOOL | Read | Indicates the state of the holding brake. |
