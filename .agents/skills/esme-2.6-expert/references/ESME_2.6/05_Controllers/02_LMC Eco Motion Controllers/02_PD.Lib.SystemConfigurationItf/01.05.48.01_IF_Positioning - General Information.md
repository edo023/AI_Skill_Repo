# IF\_Positioning - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_JobBuffer](D-SE-0089170.html#D-SE-0089170) |

## Description

The interface indicates the properties for positioning.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| Position | LREAL | Read | Indicates the position. |
| RefAcceleration | LREAL | Read | Indicates the reference acceleration that is calculated by deduction from the reference velocity RefVelocity, in units/s2. |
| RefPosition | LREAL | Read | Indicates the reference position generated by the reference position generator (for example, MasterCam) in units. |
| RefVelocity | LREAL | Read | Indicates the reference velocity that is calculated by deduction from the reference position RefPosition, in units/s. |
| Velocity | LREAL | Read | Indicates the velocity. |
