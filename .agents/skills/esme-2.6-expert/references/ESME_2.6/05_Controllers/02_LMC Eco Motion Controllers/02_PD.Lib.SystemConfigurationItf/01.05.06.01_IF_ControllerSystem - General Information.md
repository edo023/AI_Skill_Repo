# IF\_ControllerSystem - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the utilization properties.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| AvailableLoad | DINT | Read | Indicates of the available CPU time in percentage. |
| AvailableLoadPeriod | DINT(1..20000) | Read/Write | Sets the measurement period for the CPU load measurement in ms. |
| CycleLoad | DINT | Read | Indicates the utilization of the CPU in percentage by the real-time process. |
| EnableLoadEff | BOOL | Read/Write | Starts and stops the CPU load measurement. |
| Systemticks | DINT(10..20000) | Read/Write | Sets the internal clock of the controller. |
