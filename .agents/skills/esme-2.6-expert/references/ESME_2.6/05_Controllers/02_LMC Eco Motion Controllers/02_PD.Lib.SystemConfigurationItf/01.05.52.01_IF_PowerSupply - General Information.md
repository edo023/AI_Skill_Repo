# IF\_PowerSupply - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the controller and status properties of the power supply.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| GroupReady | BOOL | Read | Indicates whether the group (power supply and the connected drives) is ready for task processing. |
| GroupState | DINT | Read | Indicates the operating state of the power supply and the connected drives. |
| PowerSupplyCheck | BOOL | Read | Indicates the software enabling for the power supply by the controller. |
| PowerSupplyCheckSet | BOOL | Read/Write | Enables the power supply. |
| Ready | BOOL | Read | Indicates the state of the Ready output. |
