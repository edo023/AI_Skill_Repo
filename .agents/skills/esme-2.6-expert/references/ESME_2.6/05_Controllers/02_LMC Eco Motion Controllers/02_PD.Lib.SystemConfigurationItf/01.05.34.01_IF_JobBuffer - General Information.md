# IF\_JobBuffer - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface indicates the job number (Id) of the most recent job (ongoing or finished).

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| CurrentJobId | DINT | Read | Indicates the job number (Id) of the most recent job (ongoing or finished). |
