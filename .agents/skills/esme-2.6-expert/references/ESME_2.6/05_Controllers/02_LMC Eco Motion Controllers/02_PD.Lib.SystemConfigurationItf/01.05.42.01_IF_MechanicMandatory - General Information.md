# IF\_MechanicMandatory - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages mechanical properties.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| Direction | BOOL | Read/Write | Sets the direction of rotation. |
| FeedConstant | LREAL | Read/Write | Sets the feed constant. |
| GearIn | DINT | Read/Write | Sets the gear box input (numerator). |
| GearOut | DINT | Read/Write | Sets the gear box output (denominator). |
