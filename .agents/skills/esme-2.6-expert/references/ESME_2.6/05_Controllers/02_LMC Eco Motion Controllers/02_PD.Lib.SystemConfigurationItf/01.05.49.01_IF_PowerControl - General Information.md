# IF\_PowerControl - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_PowerControlMandatory](D-SE-0089201.html#D-SE-0089201) |

## Description

The interface manages the properties of the power supply.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ConfiguredPowerSupply | STRING[40] | Read/Write | Sets the name of the power supply to which the drive is connected. |
| InverterEnable | BOOL | Read | Indicates the status of the InverterEnable signal. |
| PowerSupply | STRING[40] | Read | Indicates the assigned power supply. |
