# IF\_CAM - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_JobBuffer](D-SE-0089170.html#D-SE-0089170) |

## Description

The interface indicates the reference values and actual values of the drive.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| Position | LREAL | Read | Actual position. |
| RefAcceleration | LREAL | Read | Reference acceleration in units/s2. |
| RefPosition | LREAL | Read | Reference position. |
| RefVelocity | LREAL | Read | Reference velocity in units/s. |
| Velocity | LREAL | Read | Actual velocity in units/s. |
