# IF\_EncIn\_Incremental - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.76.6.0 |
| Inherits from: | [IF\_EncIn](IF_EncIn-GeneralInformation-7AED85DB.html#IF_EncIn-GeneralInformation-7AED85DB) |

## Description

The interface manages the properties of the incremental encoder.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| Resolution | DINT | Read/Write | Sets the resolution of the incremental encoder. |
| ZeroTrackCheckOff | BOOL | Read/Write | Sets the zero track monitoring of the incremental encoder.  FALSE: Enables the zero track monitoring.  TRUE: Disables the zero track monitoring. |
| ZeroTrackDetected | BOOL | Read | Indicates whether a zero track has been detected.  FALSE: No zero track detected.  TRUE: Zero track detected. |
| ZeroTrackStart | BOOL | Read/Write | Sets the zero track detection of the incremental encoder.  FALSE: Disables the zero track detection.  TRUE: Enables the zero track detection. |
