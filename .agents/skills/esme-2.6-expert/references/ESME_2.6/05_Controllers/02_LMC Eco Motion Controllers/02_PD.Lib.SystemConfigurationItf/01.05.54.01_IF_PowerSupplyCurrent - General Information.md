# IF\_PowerSupplyCurrent - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.76.6.0 |
| Inherits from: | - |

## Description

The interface indicates the properties of the current of the power supply.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| PowerSupplyOverload | INT | Read | Indicates the overload of the power supply in percentage. |
| RectifierCurrent | DINT | Read | Indicates the rectified current of the DC bus in mA. |
