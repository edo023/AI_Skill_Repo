# IF\_TouchProbe2 - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_TouchProbe](D-SE-0089221.html#D-SE-0089221) |

## Description

The interface manages the properties and methods of the TouchProbe2.

Also refer to Touchprobe configuration and start mechanism.

## Methods

| Name | Description |
| --- | --- |
| TPEdgeStart | Calls the FC\_TPEdgeStart() function with the ST\_LogicalAddress of the object as i\_stTPId. |
| TPOff | Calls the FC\_TPOff() function with the ST\_LogicalAddress of the object as i\_stTPId. |
| TPReset | Calls the FC\_TPReset() function with the ST\_LogicalAddress of the object as i\_stTPId. |

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| CaptureEdge | DINT | Read | Indicates the configured edge. |
| CaptureEdgeSet | DINT | Read/Write | Sets the edge of the touchprobe input. |
| CaptureEdgeSetDelayed | BOOL | Read | Indicates the delay on change of CaptureEdge. |
