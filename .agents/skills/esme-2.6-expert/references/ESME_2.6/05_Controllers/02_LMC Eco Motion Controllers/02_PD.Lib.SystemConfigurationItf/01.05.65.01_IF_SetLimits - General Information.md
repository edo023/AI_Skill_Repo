# IF\_SetLimits - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the limitation values of the drive.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ControllerStopDec | LREAL | Read/Write | Sets the maximum absolute value of the deceleration with which the drive is stopped (in units/s2). |
| ControllerStopJerk | LREAL | Read/Write | Sets the absolute value of the jerk with which the drive is stopped (in units/s3). |
| TrackingDeviationLimit | LREAL | Read/Write | Sets the maximum tracking deviation of the position controller (position\_reference value minus position\_actual value). |
| UserCurrentLimit | REAL | Read/Write | Sets the user-defined limitation for the current. |
| UserDrivePeakCurrent | DINT | Read/Write | Sets the user-defined limitation for the drive peak current. |
| UserMaxAcc | REAL | Read/Write | Sets the user-defined maximum acceleration of the application. |
| UserMaxVel | REAL | Read/Write | Sets the user-defined limitation for the maximum velocity MaxVel determined by the system. |
