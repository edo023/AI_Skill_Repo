# IF\_Mechanic\_DSM - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.76.6.0 |
| Inherits from: | - |

## Description

The interface manages the properties of the encoder of the Safety Logic.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| DisableMechanicParameterCheck | BOOL | Read/Write | Disables the verification of the parameters of the mechanic. |
| EncoderDirection | UDINT | Read | Indicates the direction of the encoder. |
| EncoderResolutionLinear | UDINT | Read | Indicates the resolution of the encoder of the linear motor. |
| EncoderResolutionRotary | UDINT | Read | Indicates the resolution of the encoder of the rotary motor. |
| EncoderType | UDINT | Read | Indicates the type of the encoder. |
| FeedConstantRotary | LREAL | Read | Indicates the feed constant for rotary motor. |
| FeedConstantLinear | LREAL | Read | Indicates the feed constant for linear motor. |
| GearIn | DINT | Read | Indicates the input gear factor. |
| GearOut | DINT | Read | Indicates the output gear factor. |
