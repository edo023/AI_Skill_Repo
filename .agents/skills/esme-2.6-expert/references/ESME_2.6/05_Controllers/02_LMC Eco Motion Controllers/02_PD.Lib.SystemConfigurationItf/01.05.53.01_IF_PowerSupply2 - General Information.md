# IF\_PowerSupply2 - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the properties of the LXM52 power supply.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| AutoDischarge | BOOL | Read/Write | Enables the automatic discharge of the DC bus. |
| Bleeder | DINT | Read | Indicates the utilization of the braking resistor in percentage (0...100%). |
| Ready | BOOL | Read | Indicates the state of the Ready output. |
