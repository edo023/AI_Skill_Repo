# IF\_EncIn - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.76.6.0 |
| Inherits from: | - |

## Description

The interface manages the properties of the encoder.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| CheckOff | BOOL | Read/Write | Sets the monitoring of the encoder.  FALSE: Enables the monitoring of the encoder.  TRUE: Disables the monitoring of the encoder. |
| Direction | BOOL | Read/Write | Sets the direction of movement.  FALSE: Negative direction of movement.  TRUE: Positive direction of movement. |
| ErrorDeceleration | REAL | Read/Write | Sets the deceleration ramp in the case of a detected error. |
| FeedConstant | REAL | Read/Write | Sets the feed constant. |
| Filter | DINT | Read/Write | Sets the filter value in ms. |
| Position | LREAL | Read | Indicates the position of the encoder. |
| State | DINT | Read | Indicates the state of the encoder. |
| Velocity | LREAL | Read | Indicates the velocity of the encoder. |
