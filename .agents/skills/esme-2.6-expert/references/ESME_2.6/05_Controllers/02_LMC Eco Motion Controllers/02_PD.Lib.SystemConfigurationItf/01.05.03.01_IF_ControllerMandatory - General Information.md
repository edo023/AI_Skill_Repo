# IF\_ControllerMandatory - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | [IF\_DiagMandatory](D-SE-0089146.html#D-SE-0089146) |

## Description

The interface manages the hardware and software information of the devices.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ControllerType | STRING[254] | Read | Indicates the device type. |
| FW\_Version | STRING[28] | Read | Indicates the firmware version of the device. |
| HW\_Code | STRING[20] | Read | Indicates the hardware codes of the controller. |
| MsgEntries | DINT | Read | Indicates the number of entries in the message logger. |
| Name | STRING[40] | Read/Write | Sets the name. |
| OnlineChangeCounter | DINT | Read | Indicates the number of online change updates of the program sequences that have occurred since the last program change. |
| SerialNumber | STRING[20] | Read | Indicates the serial number of the device. |
| stLogicalAddress | [ST\_LogicalAddress](D-SE-0089230.html#D-SE-0089230) | Read/Write | Sets the logical address of the object as a structure. |
