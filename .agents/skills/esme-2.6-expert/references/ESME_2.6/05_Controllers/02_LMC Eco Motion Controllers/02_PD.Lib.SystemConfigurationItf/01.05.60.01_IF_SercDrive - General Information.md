# IF\_SercDrive - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.76.6.0 |
| Inherits from: | - |

## Description

The interface manages the properties of the Sercos drive.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ATAdr | DWORD | Read | Indicates the pointer of the Acknowledge Telegram (AT). |
| ConfigurationListAT | STRING[254] | Read/Write | Sets the configuration list of the Acknowledge Telegram (AT). |
| ConfigurationListATLength | UINT | Read/Write | Sets the length of the configuration list of the Acknowledge Telegram (AT). |
| ConfigurationListMDT | STRING[254] | Read/Write | Sets the configuration list of the Master Data Telegram (MDT). |
| ConfigurationListMDTLength | UINT | Read/Write | Sets the length of the configuration list of the Master Data Telegram (MDT). |
| ConfiguredVendorCode | WORD | Read/Write | Sets the vendor code. |
| ConfiguredVendorDeviceId | STRING[80] | Read/Write | Sets the vendor device id. |
| ControllerPeakC | DINT | Read/Write | Sets the controller peak current in mA. |
| EncoderPosition | LREAL | Read | Indicates the encoder position. |
| FeedbackResolution | UDINT | Read/Write | Sets the feedback resolution in increments. |
| J\_Gesamt | REAL | Read/Write | Sets J total in kgcm2. |
| MaxRPM | UINT | Read/Write | Sets the maximum speed of rotation in RPM. |
| MDTAdr | DWORD | Read | Indicates the pointer of the Master Data Telegram (MDT). |
| MechPosition | LREAL | Read | Indicates the mechanical position. |
| MechRefPosition | LREAL | Read | Indicates the mechanical reference position. |
| MechRefVelocity | LREAL | Read | Indicates the mechanical reference velocity. |
| MechVelocity | LREAL | Read | Indicates the mechanical velocity. |
| ModuloValue | UDINT | Read/Write | Sets the modulo value in increments. |
| MotorPeakC | DINT | Read/Write | Sets the motor peak current in mA.. |
| PositionCommandValueOffset | UINT | Read/Write | Set the position of the command value of the Master Data Telegram (MDT). |
| PositionFeedbackValueOffset | UINT | Read/Write | Set the position of the feedback value of the Acknowledge Telegram (AT). |
| PrimaryOperationMode | UINT | Read/Write | Sets the primary operating mode. |
| Profile | DINT | Read/Write | Sets the operation profile. |
| StopDec | REAL | Read/Write | Sets the deceleration ramp for stopping in percentage (percentage of MaxAcc). |
| TelegramType | UINT | Read/Write | Sets the telegram type. |
| TorqueConstantSet | UINT | Read/Write | Sets the torque constant. |
| TrackingDeviation | LREAL | Read | Indicates the position deviation between MechRefPosition and MechPosition. |
| UserMaxAcc | REAL | Read/Write | Sets the user-defined maximum acceleration. |
| UserMaxVel | REAL | Read/Write | Sets the user-defined maximum velocity. |
