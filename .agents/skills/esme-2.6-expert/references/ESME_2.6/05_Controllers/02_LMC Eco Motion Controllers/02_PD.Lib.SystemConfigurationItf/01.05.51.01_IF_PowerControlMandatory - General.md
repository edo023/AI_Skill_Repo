# IF\_PowerControlMandatory - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface manages the properties for enabling the drive.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| ControllerEnable | BOOL | Read | Indicates whether a software enabling for the drive has been given by the controller. |
| ControllerEnableSet | BOOL | Read/Write | Enables the drive. |
| ControllerEnableStopMode | DINT | Read/Write | Sets the stop mode of the drive. |
| TorqueEnable | BOOL | Read/Write | Enables the torque for the drive. |
