# IF\_SercosMaster - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.76.6.0 |
| Inherits from: | [IF\_Sercos](D-SE-0089215.html#D-SE-0089215) |

## Description

The interface indicates the properties of the Sercos master.

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| LastDeviceP1 | STRING[40] | Read | Indicates the name of the last physical Sercos device on port 1. |
| LastDeviceP2 | STRING[40] | Read | Indicates the name of the last physical Sercos device on port 2. |
| PhaseUpCounter | UDINT | Read | Indicates the number of successful Sercos phase-up since the last reboot of the controller. |
| ScannedDevices | DINT | Read | Indicates the number of the physically scanned Sercos devices. |
| SystemReaction | DINT | Read | Indicates the setting of the parameter ConfiguredSystemReaction of the connected Sercos devices.  Reading the value 1 indicates that the parameter ConfiguredSystemReaction is set to the value 1 for at least one connected Sercos device. |
| Topology | DINT | Read | Indicates the topology of the Sercos system. |
| UsedDevices | DINT | Read | Indicates the number of the physically scanned and assigned Sercos devices. |
