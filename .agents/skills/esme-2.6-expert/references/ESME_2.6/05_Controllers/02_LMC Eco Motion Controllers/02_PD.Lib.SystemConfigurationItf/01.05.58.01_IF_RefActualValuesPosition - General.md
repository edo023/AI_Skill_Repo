# IF\_RefActualValuesPosition - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Interface |
| Available as of: | V1.0.0.0 |
| Inherits from: | SYSTEM.IQueryInterface |

## Description

The interface indicates the position values (reference and actual values).

## Properties

| Name | Data type | Accessing | Description |
| --- | --- | --- | --- |
| Position | LREAL | Read | Indicates the actual position of the device. |
| RefPosition | LREAL | Read | Indicates the reference position that is generated by the reference position generator (for example, MasterCam, PosGenerator, …) in units. |
