# FC\_CycleCheckTimeSetByName

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Modify the times for the task cycle time monitoring.

## Description

This function manipulates the time values for the task cycle time monitoring with the task that is determined with its name (i\_sTaskName; maximum 80 characters; in the versions <00.23.00 32 characters).

i\_diOverrunTime determines the value for default cycle time check (diagnostic message [8317 "Cycle time overrun"](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063921)).

i\_diWDogFactor determines the value for a cycle time error (diagnostic message 8313 "Excessive cycle time overrun").

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sTaskName | STRING[80] | Name of the task |
| i\_diOverrunTime | DINT | Value for the standard cycle time monitoring |
| i\_diWDogFactor | DINT | Value for the watchdog factor |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  Otherwise: Incorrect parameter |
