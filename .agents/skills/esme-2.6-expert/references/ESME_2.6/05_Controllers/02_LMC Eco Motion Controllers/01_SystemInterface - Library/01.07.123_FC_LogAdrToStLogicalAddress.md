# FC\_LogAdrToStLogicalAddress

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Convert the format of a logical address of DINT in ST\_LogicalAddress.

## Description

Converts the logical address i\_diLogAdr that was transferred in the old format (DINT) into the new format (ST\_LogicalAddress).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_diLogAdr | DINT | Logical address whose format is to be changed |

EIO0000002680.05
