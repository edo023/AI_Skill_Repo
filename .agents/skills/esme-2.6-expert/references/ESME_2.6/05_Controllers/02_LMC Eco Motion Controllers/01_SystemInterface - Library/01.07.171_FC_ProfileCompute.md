# FC\_ProfileCompute

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Calculate the slave position dependent on the master position and the curve profile.

## Description

For the master position i\_lrXPos, the slave position iq\_lrYPos of the profile with the logical address i\_diProfileId is determined and provided in iq\_lrYPos. The scaling quantities i\_lrXOffset, i\_lrYOffset, i\_lrXFactor, and i\_lrYFactor are taken into account.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_diProfileId | DINT | Logical address of the profile |
| i\_lrXPos | LREAL | Master position |
| i\_lrXOffset | LREAL | Offset of the master position |
| i\_lrYOffset | LREAL | Offset of the slave position |
| i\_lrXFactor | LREAL | Scaling the master position |
| i\_lrYFactor | LREAL | Scaling the slave position |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_lrYPos | LREAL | Slave position |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: i\_diProfileId invalid  -2: i\_lrXFactor invalid (= 0)  -30: Profile is being used by another function and therefore is blocked. |
