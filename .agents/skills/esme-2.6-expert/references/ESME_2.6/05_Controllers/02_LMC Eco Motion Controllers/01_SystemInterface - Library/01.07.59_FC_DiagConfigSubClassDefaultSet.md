## Task

Reset the diagnostic configuration for the diagnostic messages to default value.

NOTE: The call-up of this function is only possible as of Sercos phase 2.

## Description

This function allows you to reset the device reaction of the diagnostic messages which are triggered by drives or power supply units to default values.

If a sub class changes, so that it is assigned to a different diagnostic class, the diagnostic class is adjusted.

The interrelationship between the sub class, the diagnostic class, and the reaction for [drives](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063389) and [power supply units](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063391) is listed in respective tables.
