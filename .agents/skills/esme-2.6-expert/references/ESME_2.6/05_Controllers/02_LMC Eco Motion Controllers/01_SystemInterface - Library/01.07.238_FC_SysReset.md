# FC\_SysReset

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Resetting the PacDrive controller.

## Description

The PacDrive controller is restarted.

NOTE: You have to ensure that the PacDrive controller completed the boot process before starting another boot process with the function FC\_SysReset() through the reset. You can implement this with the aid of the function FC\_GetPacDriveBootState().

Otherwise, the diagnostic message [8209 "Last boot failed"](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063905) may be displayed.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: PacDrive controller was not reset |
