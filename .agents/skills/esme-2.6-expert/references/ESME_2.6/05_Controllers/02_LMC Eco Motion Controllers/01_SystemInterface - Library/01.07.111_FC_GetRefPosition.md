# FC\_GetRefPosition

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Read the reference position of an axis.

## Description

The function returns the reference position (RefPosition) of a drive (i\_stAxisId).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Logical address of the axis |

## Return Value

| Data type | Description |
| --- | --- |
| LREAL | 0: Logical address of the axis (i\_stAxisId) is invalid  Otherwise: Reference position (RefPosition) of the drive |
