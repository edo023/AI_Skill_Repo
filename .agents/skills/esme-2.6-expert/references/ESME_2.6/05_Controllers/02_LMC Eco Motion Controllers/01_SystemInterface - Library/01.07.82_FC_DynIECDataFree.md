# FC\_DynIECDataFree

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Release the dynamic memory area.

## Description

Using this function, a range from the data range reserved with FC\_DynIECDataAlloc() can be released again. The pointer iq\_pdwAddress in the memory area to be released returns 0 if the memory could be released.

## Interface

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_pdwAddress | POINTER TO DWORD | IN -> pointer to the memory area to be released  OUT -> 0 if memory could be released |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: if area could be released  FALSE: if an error is detected |

## Example

```
 pastTest_DYN : POINTER TO struct1 := 0; 
xResult : BOOL := FALSE; 
pastTest_DYN := FC_DynIECDataAlloc(100 * SIZEOF(pastTest_DYN ^)); 
... 
IF (pastTest_DYN > 0) 
   xResult := FC_DynIECDataFree(pastTest_DYN); 
END_IF
```
