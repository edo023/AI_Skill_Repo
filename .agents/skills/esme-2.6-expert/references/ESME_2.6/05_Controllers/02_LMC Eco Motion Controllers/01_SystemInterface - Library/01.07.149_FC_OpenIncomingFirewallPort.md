# FC\_OpenIncomingFirewallPort

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_V1.62.4.1 |
| Versions: | Current version |

## Task

Open incoming firewall port during runtime.

## Description

The function can be used to open the incoming firewall port during runtime.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etProtocol | [ET\_Protocol](D-SE-0104153.html#D-SE-0104153) | Communication protocol |
| i\_uiPort | UINT | Communication port |
| i\_sSource | STRING[255] | Source address of incoming packets.  Supported values are any, me, all valid IP addresses and ranges.  It is a good practice to use the value any. |
| i\_sDestination | STRING[255] | Destination address of incoming packets.  Supported values are any, me, all valid IP addresses and ranges.  It is a good practice to use the value me. |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: The incoming port rule was successfully applied.  -1: Input value i\_uiPort must be non-zero.  -2: Error detected. Unable to apply firewall rule. |
