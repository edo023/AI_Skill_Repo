# FC\_PosVelTimeEnable

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Read the position values and the velocity values consistently and with a time reference.

## Description

In the parameter i\_stLogAdr, the logical address of the position source (LXM62DxS) is transferred for which the recording is to be made.

It can activate a cyclic recording of a position/velocity (i\_xEnable:=true).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stLogAdr | ST\_LogicalAddress | Logical address of the position source |
| i\_xEnable | BOOL | TRUE = cyclic recording of a position/velocity is activated |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -99: Error detected |
