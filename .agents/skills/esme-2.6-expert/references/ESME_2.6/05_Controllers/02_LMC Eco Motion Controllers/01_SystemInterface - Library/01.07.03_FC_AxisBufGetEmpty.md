# FC\_AxisBufGetEmpty

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Check if the preparation buffer of the axis is empty.

## Description

The axis is selected with i\_stAxisID. The function checks whether the preparation buffer of the axis which has been selected using i\_stAxisID is empty.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Selecting the axis |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: Buffer is empty  FALSE: Buffer is not empty |

EIO0000002680.05
