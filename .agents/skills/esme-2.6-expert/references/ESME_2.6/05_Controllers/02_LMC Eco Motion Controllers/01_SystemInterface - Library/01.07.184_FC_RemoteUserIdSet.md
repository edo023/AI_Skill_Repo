# FC\_RemoteUserIdSet

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Set user name and password for the remote file services.

## Description

This function sets the user name and password with which access to the remote computer is granted. Name (i\_sName) and password (i\_sPassword) are used to get "remote access" via the FTP services.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sName | STRING[80] | User name |
| i\_sPassword | STRING[80] | Password |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  Otherwise: Error detected |
