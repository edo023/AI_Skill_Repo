# FC\_SercosWriteServiceData

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |

## Description

FC\_SercosWriteServiceData writes service data via Sercos that are used for debugging.

Also refer to [Error codes of the Sercos parameter channel (service channel)](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063997).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stLogAddr | ST\_LogicalAddress | Logical address of the axis |
| i\_dwIDN | DWORD | Data Service IDN |
| i\_wType | WORD | Data Service Type   * 0: read/write name * 1: read/write attribute * 2: read/write unit * 3: read/write min. value * 4: read/write max. value * 5: read/write user data * 6: read/write user data with variable length * 7: execute command |
| i\_dwPointerToData | DWORD | Pointer on the data memory, filled with Len data bytes |
| i\_wMaxDataLen | WORD | Len data bytes to be written |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: The topological address is invalid.  -431: Detected service request error (for example, timeout)  -445: Service timeout  -461: in the current phase, reading out parameters via the service channel is not supported  -462: The addressed axis does not support the ServiceDataRead function.  -464: Invalid service transfer |
