# FC\_CompareStLogicalAddress

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Compare the two logical addresses.

## Description

This function compares the logical address i\_stLogAddr1 with the logical address i\_stLogAddr2. If both are identical, then TRUE is returned, otherwise FALSE.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stLogAddr1 | ST\_LogicalAddress | Logical address 1 |
| i\_stLogAddr2 | ST\_LogicalAddress | Logical address 2 |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: Address 1 is identical to address 2  FALSE: Address 1 is not identical to address 2 |

EIO0000002680.05
