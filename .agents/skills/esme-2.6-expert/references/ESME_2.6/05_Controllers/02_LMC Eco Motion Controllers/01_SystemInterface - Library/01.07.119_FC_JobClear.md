# FC\_JobClear

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Delete job buffer.

## Description

The job buffer is deleted. Jobs can only be deleted when processing has not yet started. The axis to be processed is identified by i\_stAxisId. The axes are processed with i\_stAxisId=Gc\_stLogAddrAllTypes. Uncontrolled starting of other jobs can be prevented by creating a stop (function FC\_ControllerStopSet()).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Logical address of the drive |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK  -1: Logical address of the drive is invalid |
