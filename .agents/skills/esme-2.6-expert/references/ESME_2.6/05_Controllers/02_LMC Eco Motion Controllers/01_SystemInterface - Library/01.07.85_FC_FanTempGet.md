# FC\_FanTempGet

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Read the starting temperature of the fan of the MotorControllers Lexium LXM52 Drive and Lexium LXM62 Drive, and the Lexium LXM62 Power Supply.

## Description

Reads the fan starting temperature of the device specified by using i\_stAxisId.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisId | ST\_LogicalAddress | Logical address of the drive |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 10 - 75: Activation temperature  -1: Logical address invalid |
