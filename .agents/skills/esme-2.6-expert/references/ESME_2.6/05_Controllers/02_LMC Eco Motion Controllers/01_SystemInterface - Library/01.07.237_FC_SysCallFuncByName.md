# FC\_SysCallFuncByName

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Call-up the internal firmware function by using the name of the function.

## Description

The function allows a global internal firmware function to be called via the name of the function. The function name is transferred in i\_sFuncName. The "\_" name extension is automatically added. Up to 10 DINT type parameters can be transferred to the function. Parameters that are not required must be set to 0. The return value of the called firmware function is reported back via iq\_diFuncResult.

The specification of the firmware function called must be observed.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_sFuncName | STRING[80] | Name of the function |
| i\_diArg1, i\_diArg2, i\_diArg3, i\_diArg4, i\_diArg5, i\_diArg6, i\_diArg7, i\_diArg8, i\_diArg9, i\_diArg10 | DINT | Argument 1 ... Argument 10 |

| Input/Output | Data type | Description |
| --- | --- | --- |
| iq\_diFuncResult | DINT | Return value of the called up firmware function |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: ok  -1: Function i\_sFuncName not found |
