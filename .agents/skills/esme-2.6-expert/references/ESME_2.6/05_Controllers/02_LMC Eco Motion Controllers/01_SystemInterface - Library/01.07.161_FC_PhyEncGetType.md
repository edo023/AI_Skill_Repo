# FC\_PhyEncGetType

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Reading the encoder type from the encoder.

## Description

The encoder type is read from the encoder and saved in the EncoderType parameter.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stEncId | ST\_LogicalAddress | Logical address of the encoder |

## Return Value

| Data type | Description |
| --- | --- |
| BOOL | TRUE: job started  FALSE: job interface is busy (try again) or error |
