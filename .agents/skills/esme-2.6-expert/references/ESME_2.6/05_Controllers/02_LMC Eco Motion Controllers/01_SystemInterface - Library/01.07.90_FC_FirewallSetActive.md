# FC\_FirewallSetActive

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.56.13.0 |
| Versions: | Current version |

## Task

Activate / deactivate the firewall.

## Description

The function can be used to activate / deactivate the firewall.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_xActive | BOOL | TRUE = Activate the firewall  FALSE = Deactivate the firewall |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: Firewall successfully activated / deactivated.  -1: General error detected.  -2: Not enough free memory. |

## Example

```
FirewallStatus := FC_FirewallSetActive( TRUE );
```

On a successful activation or deactivation, the result is 0.

In case of a general error detected, contact your Schneider Electric service representative.
