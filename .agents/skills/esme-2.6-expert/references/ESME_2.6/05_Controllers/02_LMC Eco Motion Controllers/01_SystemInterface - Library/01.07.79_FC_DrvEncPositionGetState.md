# FC\_DrvEncPositionGetState

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Determine the current state of the encoder setting functionality to change the absolute position of the encoder.

## Description

Use this function to determine the current state of the function DrvEncSetPosition() of the axis i\_stAxisID. You can only start encoder non-volatile memory programming in states 0 and 2. Programming was successful if state 0 is reached after state 1. The processing takes a few milliseconds, this is why an increased task cycle time is possibly necessary.

As of version V1.50.1.x, the value 1 (= motor nameplate is currently programmed) does not occur any longer while reading the state. However, the return value still indicates whether programming was successful.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stAxisID | ST\_LogicalAddress | Logical address of the drive |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: Programming OK, ready for programming  -1: Incorrect i\_stAxisID  1: Encoder non-volatile memory is being programmed  2: Error while programming the encoder non-volatile memory  3: No write access to the encoder non-volatile memory (drive being controlled or error detected in the encoder non-volatile memory) |
