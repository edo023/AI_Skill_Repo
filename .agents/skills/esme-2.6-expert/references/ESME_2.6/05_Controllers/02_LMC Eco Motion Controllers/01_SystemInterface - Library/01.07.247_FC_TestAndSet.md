# FC\_TestAndSet

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Synchronize programs / tasks via a synchronization flag.

## Description

The function provides a simple method for synchronization via mutual exclusion:

| If ... | Then ... | Comment |
| --- | --- | --- |
| iq\_xFlag is free (FALSE) | Then the flag is reserved. | The function returns TRUE. |
| iq\_xFlag is occupied (TRUE) | Then the flag is not changed. | The function returns FALSE. |

The programmer is responsible for releasing the flag. The call of the function cannot be interrupted by other tasks, not even by tasks that have a higher priority

## Input Values and Result

| iq\_xFlag | | FC\_TestAndSet |
| --- | --- | --- |
| In | Out |
| TRUE | TRUE | FALSE |
| FALSE | TRUE | TRUE |

## Example

The two programs SR\_Catcher and SR\_Runner are synchronized via the global variable G\_xSynchronFlag. Both programs execute a busy wait until the function FC\_TestAndSet returns TRUE (G\_xSynchronFlag = FALSE). Both programs execute the FC\_TestAndSet in an endless loop due to the cyclic task processing in IEC 61131.

```
VAR_GLOBAL
    G_xSynchronFlag : BOOL := FALSE;
END_VAR
PROGRAM SR_Catcher
VAR
END_VAR
IF (TRUE = SystemInterface.FC_TestAndSet(iq_xFlag := GVL.G_xSynchronFlag)) THEN
    // critical section
    GVL.G_xSynchronFlag := FALSE;
END_IF
PROGRAM SR_Runner
VAR
END_VAR
IF (TRUE = SystemInterface.FC_TestAndSet(iq_xFlag := GVL.G_xSynchronFlag)) THEN
    // critical section
    GVL.G_xSynchronFlag := FALSE;
END_IF
```
