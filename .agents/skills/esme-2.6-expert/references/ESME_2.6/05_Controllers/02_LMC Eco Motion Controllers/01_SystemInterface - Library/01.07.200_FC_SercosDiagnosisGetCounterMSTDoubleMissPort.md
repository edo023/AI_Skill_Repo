# FC\_SercosDiagnosisGetCounterMSTDoubleMissPort1

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.64.16.4 |
| Versions: | Current version |

## Description

Returns the number of MSTs which were unreceived two times consecutively since the last call of FC\_SercosDiagnosisResetErrorCounters on port 1.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | Number of MSTs which were unreceived two times consecutively |

EIO0000002680.05
