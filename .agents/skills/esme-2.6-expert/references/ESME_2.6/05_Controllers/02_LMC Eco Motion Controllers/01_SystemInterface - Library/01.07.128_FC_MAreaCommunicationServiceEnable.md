# FC\_MAreaCommunicationServiceEnable

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_V1.62.4.1 |
| Versions: | Current version |

## Task

Activate the LZS\_ELAU\_M\_AREA\_WRITE and LZS\_ELAU\_M\_AREA\_READ communication server.

## Description

By default, the TCP port 5000 is deactivated. The corresponding protocol server for the LZS\_ELAU\_M\_AREA\_WRITE and LZS\_ELAU\_M\_AREA\_READ communication service is also deactivated by default. Once the server is activated, it cannot be deactivated until the next reboot.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | 0: OK. The start of the server was successful.  -2501: The server was already active, no activation needed. |
