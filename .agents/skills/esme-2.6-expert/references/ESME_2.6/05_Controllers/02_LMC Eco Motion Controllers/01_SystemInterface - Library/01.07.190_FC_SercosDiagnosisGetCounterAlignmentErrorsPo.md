# FC\_SercosDiagnosisGetCounterAlignmentErrorsPort1

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.64.16.3 |
| Versions: | Current version |

## Description

Returns the number of invalid aligned Ethernet frames since the last call of FC\_SercosDiagnosisResetDriverErrorCounter or FC\_SercosDiagnosisResetErrorCounters on port 1.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | Number of invalid aligned Ethernet frames |

EIO0000002680.05
