# FC\_MemoryMove

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Copy the memory area.

## Description

With this function, n-bytes of a memory area are copied in another.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_dwDest | DWORD | Initial address of the destination memory area |
| i\_dwSrc | DWORD | Initial address of the source memory area |
| i\_dwCount | DWORD | Number of bytes to be copied |

## Return Value

| Data type | Description |
| --- | --- |
| DWORD | o\_FC\_MemoryMove: Pointer to the destination memory |

## Examples

```
dwResult : DWORD;
dwResult := FC_MemoryMove(pbyAddresse1,pbyAddresse2,dwSize);
```
