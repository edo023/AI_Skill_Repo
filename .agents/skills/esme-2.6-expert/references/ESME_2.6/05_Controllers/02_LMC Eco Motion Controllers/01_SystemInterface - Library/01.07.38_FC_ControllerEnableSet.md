# FC\_ControllerEnableSet

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.32.6.0 |
| Versions: | Current version |

## Task

Release the drive.

## Description

The software release for one or all drives is granted. A release is granted only if the drive and controller (such as PacDrive LMC600) are ready to switch to release. The user request and controller request are connected in an OR relationship. The current result may be read out at any time using the parameter *[ControllerEnableSet](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071451)*. The new status is processed in the next real-time cycle and can be read via the parameter *[ControllerEnable](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071477)*. Read *[AxisState](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071486)* in order to check the current situation. If i\_stAxisId=Gc\_stLogAddrAllTypes, then a release is given for the drives.
