# FC\_SercosDiagnosisGetCounterMSTWindowErrorPort2

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | SystemInterface\_1.64.16.3 |
| Versions: | Current version |

## Description

Returns the number of MSTs received outside of the MST receive window since the last call of FC\_SercosDiagnosisResetErrorCounters on port 2.

## Return Value

| Data type | Description |
| --- | --- |
| DINT | Number of MSTs received outside of the MST receive window |

EIO0000002680.05
