# VendorCode

## General

|  |  |
| --- | --- |
| Type | AD |
| Offline editable | No |
| Traceable | Yes |

## Functional Description

The parameter represents the manufacturer code of the device.

| Data type | Meaning |
| --- | --- |
| WORD | Manufacturer code of the device |

NOTE: The parameter value is read via Sercos in phase 1. The value is not updated in greater phases.
