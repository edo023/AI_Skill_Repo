# State

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Encoder network (Sync. Module and Synchronization slave) |
| Traceable | Yes |

## Functional Description

Displays the synchronization status. The parameter State is to be seen analogously to the parameter State of the Synchronization master.

| Value | Data type | Meaning |
| --- | --- | --- |
| disabled / 0 | DINT | The Synchronization slave is disabled. |
| initialization / 1 | DINT | The Synchronization slave is being initialized and parameterized by the Synchronization master. Communication is still not cyclic. |
| waiting for sync / 2 | DINT | Telegrams for controller synchronization are being transmitted cyclically. |
| synchronized / 3 | DINT | Synchronization was successfully established with the Synchronization master. |
| waiting for data / 4 | DINT | Waiting for cyclic encoder data. |
| active / 5 | DINT | Communication with the Synchronization master is established and encoder data are being evaluated correctly. |
| error / 6 | DINT | An error occurred, communication must be reestablished. |
