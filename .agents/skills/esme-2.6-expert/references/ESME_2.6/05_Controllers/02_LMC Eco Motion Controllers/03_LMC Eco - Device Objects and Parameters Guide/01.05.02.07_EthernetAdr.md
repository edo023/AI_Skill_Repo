# EthernetAdr

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the device-specific Ethernet address.

The Ethernet address is specified in the Ethernet component by the component manufacturer. Every Ethernet address is globally unique.

## Example

"00209d00-6072"

EIO0000002285.11
