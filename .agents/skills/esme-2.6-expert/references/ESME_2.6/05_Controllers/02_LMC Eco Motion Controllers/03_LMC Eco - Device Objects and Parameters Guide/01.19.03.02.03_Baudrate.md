# Baudrate

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave |
| Traceable | Yes |

## Functional Description

Displays the applied baud rate.

The parameter displays the baud rate used by the master if the value AUTO has been set in the initialization parameter of the slave.

| Value | Data type | Meaning |
| --- | --- | --- |
| Baud rate\_96 / 0 | UDINT | 9600 Baud |
| Baud rate\_19\_2 / 1 | UDINT | 93.75 kBaud |
| Baud rate\_93\_75 / 2 | UDINT | 187.5 kBaud |
| Baud rate\_187\_5 / 3 | UDINT | 187.5 kBaud |
| Baud rate\_500 / 4 | UDINT | 500 kBaud |
| Baud rate\_1500 / 6 | UDINT | 1500 kBaud |
| Baud rate\_3000 / 7 | UDINT | 3 MBaud |
| Baud rate\_6000 / 8 | UDINT | 6 MBaud |
| Baud rate\_12000 / 9 | UDINT | 12 MBaud |
| Baud rate\_31\_25 / 10 | UDINT | 31.25 kBaud |
| Baud rate\_45\_45 / 11 | UDINT | 45.45 kBaud |
| Baud rate\_AUTO / 15 | UDINT | The master has no connection to the slave. |
