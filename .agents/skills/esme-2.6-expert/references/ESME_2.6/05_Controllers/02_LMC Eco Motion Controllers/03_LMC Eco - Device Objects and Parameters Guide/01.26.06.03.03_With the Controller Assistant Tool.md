# With the Controller Assistant Tool

## Description

For detailed information on the Controller Assistant, refer to the EcoStruxure Machine Expert Controller Assistant User Guide.

To use the Controller Assistant for the one cable solution in order to access C2C slaves without direct cable connection, you need a running C2C system. Refer to [Prerequisites](D-SE-0088170.html#D-SE-0088170).

With the Controller Assistant you can manage images of controllers and update the firmware of controllers.

To connect the Controller Assistant to a controller in your C2C network, proceed as follows in the Controller selection window:

| Step | Action |
| --- | --- |
| 1 | Set **Connection Mode** to **Nodename via NAT (Remote TCP)** or **IP Address via NAT (Remote TCP)**. |
| 2 | Enter the C2C Master IP address for **NAT Address/Port**. |
| 3 | Click the refresh button on the right-hand side of the **NAT Address/Port**.  Result: That lists the controllers scanned by the gateway on this controller. |
| 4 | Select one of the controllers (double-click an item) and click the **Connect** button. |

## Bandwidth

NOTE: As you use the bandwidth of the Sercos bus, keep in mind that transmission times are higher than for direct connection. This can influence the present communication with the controller that is used as a gateway in between to the slave.

## Reading, Writing, Creating Images

* You can use the one cable connection to read an image from a controller, by using one controller (Master or Slave) as a gateway and avoiding direct cable connection to each controller.
* You can use the one cable connection to write a backup image, by using one controller (Master or Slave) as a gateway and avoiding direct cable connection to each controller.

  NOTE: If you write a backup image to a C2C Slave, the C2C network goes into phase 0.

  A message is displayed that informs you that a Sercos scan (phase up) has to be performed, to retrieve the C2C network configuration.

  So you have to reboot the C2C Master before you can communicate with the C2C network participants again
* It is not possible to create and download a new/clean controller image and open the network connection again. A running C2C project on C2C Master and C2C Slaves is needed.

## Updating the Firmware

With the Controller Assistant you can update the firmware on a controller without removing the application. (Refer to the Controller Assistant User Guide\How to Perform a Firmware Update Without Removing the Application).

During this process, the C2C network goes into phase 0, and a message is displayed that informs you that a Sercos scan (phase up) has to be performed, to retrieve the C2C network configuration.

So you have to reboot the C2C Master before you can communicate with the C2C network participants again.
