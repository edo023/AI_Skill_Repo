# InObject

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Logical encoder,  Sum encoder input,  Sum master encoder input,  C2C Encoder Output |
| Traceable | No |

## Functional Description

Is used for diagnostic in order to determine whether an input object is connected.

Displays the name of the master encoder (velocity source). If there is no master encoder, an empty string is displayed.

The allocation to the master encoder is carried out with the *[FC\_SetMasterEncoder()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311)* function.

The allocation to the sum master encoder is carried out with the *[FC\_SetSMEncIn()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085325)* function.
