# EngageEnable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Log. encoder |
| Traceable | Yes |

## Functional Description

Enables/disables the coupling.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The coupling is switched off. |
| on / 1 | BOOL | Setting the parameter to "on / 1" enables (initializes) the coupling. |
