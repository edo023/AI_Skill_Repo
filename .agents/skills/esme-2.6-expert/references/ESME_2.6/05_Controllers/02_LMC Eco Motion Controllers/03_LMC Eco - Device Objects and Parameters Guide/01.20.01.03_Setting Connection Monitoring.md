# Setting Connection Monitoring

In the General tab of the Profinet IO Controller, activate connection monitoring (Watchdog) by selecting the checkbox.

The controller is unable to detect an interruption of the network link if connection monitoring is not active.

| WARNING | |
| --- | --- |
|  | UNINTENDED EQUIPMENT OPERATION  Ensure that connection monitoring (Watchdog) is enabled.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

Set the shortest, practical monitoring time cycles to detect communication interruptions as quickly as possible.
