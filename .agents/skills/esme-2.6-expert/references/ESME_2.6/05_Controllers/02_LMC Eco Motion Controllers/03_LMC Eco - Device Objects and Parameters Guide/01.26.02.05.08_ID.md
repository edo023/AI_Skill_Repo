# ID

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | C2C Data Input, C2C Data output,  C2C Encoder input, C2C Encoder Output |
| Traceable | Yes |

## Functional Description

Unique number which is used to assign the corresponding producer/consumer objects in the C2C network.

In each C2C network, there must be no more than one producer with identical IDs.

There is no limitation in the number of consumers with the same ID.

The consumers receive the data provided by the producer with the same ID.
