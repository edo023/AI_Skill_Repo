# IdentificationMode

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Traceable | Yes |

## Functional Description

The parameter sets the Sercos device identification mode. It is applied in the Sercos phase change from 2 to 3.

| Value | Data type | Meaning |
| --- | --- | --- |
| SerialNumberController / 0 | DINT | The device is assigned via the serial number of the segment (see parameter [SerialNumber](SerialNumber-2F96A14D.html#SerialNumber-2F96A14D)). |
| SerialNumberMotor / 1 | DINT | The device is assigned via the serial number of the motor.  This parameter is only applicable to drive objects. |
| TopologyAddress / 2 (default) | DINT | The device is assigned via its topological address (see parameter [TopologyAddress](TopologyAddress-2F96065C.html#TopologyAddress-2F96065C)). |
| ApplicationType / 3 | DINT | The device is assigned via its application type (see parameter [ApplicationType](ApplicationType-2F964A5D.html#ApplicationType-2F964A5D)). |
| SercosAddress / 4 | DINT | The device is assigned via its Sercos address (see parameter [SercosAddress](SercosAddress-2F967BBB.html#SercosAddress-2F967BBB)). |
