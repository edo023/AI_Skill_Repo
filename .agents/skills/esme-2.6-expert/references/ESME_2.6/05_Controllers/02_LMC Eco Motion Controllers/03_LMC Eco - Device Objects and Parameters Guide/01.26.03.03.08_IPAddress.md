# IPAddress

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  C2C Slave,  Logic Motion Controller Device |
| Traceable | No |

## Functional Description

With this parameter, you can define the IP address of the slave device.

| Value | Data type | Meaning |
| --- | --- | --- |
| 172.20.0.x (default) | STRING(15) | IP address of the slave device  Can only be set if IPConfigMode = manual IP addressing (software) / 2.  Access type: read/write |

* While the value for this parameter is being entered, the parameter is verified for the following:

  + Proper format [a1]. [a2]. [a3]. [a4]
  + The first position ([a1]) is greater than 0 and unequal to 127 (127.x.x.x would be the local host).

* During the setting of the configuration, the parameter is verified for the following:

  + Value is within IPAddressRangeStatic (only verified if IPConfigMode = manual IP addressing (hardware) / 1 or manual IP addressing (software) / 2)
  + For double drives: The specified value is the same for both drives (only verified if IPConfigMode for both drives = manual IP addressing (software) / 2).

NOTE: Misconfiguration of the parameter IPAddress, Subnetmask, and Gateway may lead to a loss of the IP communication to the device. IP communication is used, for example, to download firmware packages or for safety-related acyclic communication.

Also see diagnostic message `8967 NRT IP parameter device different`.
