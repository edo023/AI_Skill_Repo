# General

To distribute an encoder signal over the encoder network, the sync. encoder output is used. This object sends encoder data over the network with a preset cycle time (DataCycleTime) that corresponds to the [SERCOS cycle time](D-SE-0073362.html#D-SE-0073362). Following a preset delay (DataDelay), these encoder data become valid and are output by the [sync. encoder inputs](D-SE-0076444.html#D-SE-0076444).

Using the function *[FC\_SetMasterEncoder(LogEncId, MasterId)](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311)*, you can define any master encoder as a velocity source for the Synchronization encoder output. In this case, LogEncId is the logical address of the Synchronization encoder output (SYN\_DOUT) and MasterId is the logical address of the master encoder to be connected.
