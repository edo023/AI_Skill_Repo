# LicensesUsed

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the number of licenses.

NOTE: If licenses are requested in evaluation mode, then these are not released again in the case of a reset and Sercos phase change following phase 0. The reason for this is that the licences requested during the evaluation mode is to be logged. In this way, it can be determined how many license points must be ordered.

If, during evaluation mode, a reset, or Sercos run-up (Phase 0 -> Phase 4) is carried out on the controller, licenses are entered more than once.
