# Installation Path for Fieldbus-Specific Device Descriptions

## General

The fieldbus-specific device descriptions (EDS files, GSD files …) are installed in the following path:

C:\Users\Public\Documents\EcoStruxure Machine Expert\Vx.x\Documents\FieldbusDescriptionFiles

NOTE: This installation path is valid for Windows 10 and can be different for earlier Windows versions.

Overview of the files contained in the installation path:

* PacDrive controller as CANopen slave or EtherCAT slave: To be generated with EcoStruxure Machine Expert
* PacDrive controller as EtherNet/IP adapter: SE\_ELAU\_EthernetIPAdapter.eds
* PacDrive controller as Profibus slave: SE100D60.gsd
* PacDrive controller as PROFINET device: GSDML-Vx.xx-Schneider Electric-PacDriveExt.xml

Additional files:

* Legacy PacDriveM Cx00 as CANopen slave: EC1COSE.eds
* Legacy PacDriveM Cx00 as Profibus (DPV1) slave: HIL106E7.gsd
