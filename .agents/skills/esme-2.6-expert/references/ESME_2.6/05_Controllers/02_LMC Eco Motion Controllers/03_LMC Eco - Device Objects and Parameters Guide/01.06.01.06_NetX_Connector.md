# NetX\_Connector

## General

|  |  |
| --- | --- |
| Type | EF |
| Address | 16#000F |
| Offline editable | Yes |
| Devices supporting the parameter | Ethernet |
| Traceable | Yes |
| Default value | Front / 1 |

## Functional Description

The value of the parameter NetX\_Connector is Front / 1. The value cannot be modified.

| Value | Data type | Meaning |
| --- | --- | --- |
| Front / 1 | DINT | Ethernet interface activated at the corresponding port. |
