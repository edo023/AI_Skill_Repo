# Configuration of the PROFIBUS DPV1 Slave

## Insert PROFIBUS DPV1 Slave in PLC Configuration

To be able to use a PROFIBUS slave, it has to be entered into the PLC Configuration of the PacDrive Controller.

* Project is opened in Logic Builder.
* Select the PacDrive Controller
* Right-click > Append device

  -> The dialog box “Append device” appears.
* Select device name (e.g. PROFIBUS DPV1 slave)
* OK button

  -> The PROFIBUS DPV1 slave is entered into the PLC Configuration.

NOTE: The same I/O modules have to be appended to the slave as to the corresponding master.

## Configure PROFIBUS DPV1 Slave

* Select PROFIBUS DPV1 slave in the PLC Configuration
* Double-click on selected slave or right-click and select "Edit object"

  -> The device dialog box of the PROFIBUS DPV1 slave with the "PROFIBUS configuration" register is displayed.

During configuration, at least the parameters Bus address and Baud rate have to be set.
