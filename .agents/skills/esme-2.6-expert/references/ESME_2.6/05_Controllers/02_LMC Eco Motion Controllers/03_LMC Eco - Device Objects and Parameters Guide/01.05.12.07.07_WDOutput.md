# WDOutput

## General

|  |  |
| --- | --- |
| Type | ED |
| Devices supporting the parameter | PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The parameter determines if the output shall be used as watchdog output or as standard output - like the outputs 0-6. If the output is configured as watchdog output, then the parameter Value has no effect.

| Bit | Meaning |
| --- | --- |
| off / 0 | Output is configured as standard output. |
| on / 1 | Output is configured as watchdog output. |
