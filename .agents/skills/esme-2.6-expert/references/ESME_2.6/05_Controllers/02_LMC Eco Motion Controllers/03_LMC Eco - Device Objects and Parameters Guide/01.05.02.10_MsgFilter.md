# MsgFilter

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

Message logger filter. You can filter system messages using the message filter. The message logger distinguishes between 16 different classes of message. These types can be switched on and off individually.

| Bit | Message class |
| --- | --- |
| 0 | General system messages |
| 1 | Diagnostic messages |
| 2 | Program system function block |
| 3 | Field bus-specific information |
| 4... 11 | Currently not in use |
| 12 | Extended system messages |
| 13 | Currently not in use |
| 14 | Currently not in use |
