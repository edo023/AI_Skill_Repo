# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam Track |
| Traceable | Yes |

## Functional Description

Enables / disables the cam track (Cam Track).

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The Cam Track is not enabled.   * **"off / 0" -> "off / 0"**:  Checks the PositionSource, Destination, and Bitno parameters. * **"on / 1" -> "off / 0"**:  Deactivates the processing of this path and the checking of the PositionSource, Destination, and Bitno parameters. |
| on / 1 | BOOL | The Cam Track is enabled.   * **"on / 1" -> "on / 1"**:  No action * **"off / 0" -> "on / 1"**:  Checks the PositionSource, Destination, and Bitno parameters. If the parameters are valid, they are transferred in the next Cam Track processing cycle (Interval). |

NOTE: When writing the parameter, the parameter CamTrackEnable of the Cam switch group will be updated.
