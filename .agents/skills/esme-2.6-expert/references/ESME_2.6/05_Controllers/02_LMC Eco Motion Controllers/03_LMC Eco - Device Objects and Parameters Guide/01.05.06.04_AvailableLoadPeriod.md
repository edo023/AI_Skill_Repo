# AvailableLoadPeriod

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

Defines the measurement period for the effective CPU load measurement in ms. The remaining calculation time is calculated for this period.
