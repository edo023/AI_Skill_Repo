## Functional Description

Displays the name specified in the PLC configuration. This name can be changed dynamically in the program or online via Cam switch group. This modification also relates to the symbolic name of the PLC configuration object.

The name is used in the following functions / parameters:

* Message logger (names in the column Instance)

  Transfer time: loading the message logger
* Cam Track (parameter PositionSource)

  Transfer time: parameter Enable FALSE -> TRUE
* Logical encoder (parameter InObject)

  Transfer time: *[FC\_SetMasterEncoder](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311)()*
* Incremental Encoder Output (parameter InObject)

  Transfer time: *[FC\_SetMasterEncoder](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311)()*
* Sum master encoder input (parameter InObject)

  Transfer time: *[FC\_SetMasterEncoder](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311)()*
* Sum encoder input (parameter `InObject`)

  Transfer time: *[FC\_SetMasterEncoder](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311)()*
* *[FC\_GetObjInstanceLogAdrByName](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085133)()*
* FC\_GetObjLogAdrByName()
* *[FC\_GetObjNameByLogAdr](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085137)()*

For the functions and parameters mentioned, those names are used which are active at the time of execution.
