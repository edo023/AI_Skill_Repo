# Direction

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | C2C Encoder input, C2C Encoder Output |
| Traceable | Yes |

## Functional Description

Defines the direction of rotation. The direction of rotation is taken into consideration in the parameter [Velocity](D-SE-0092877.html#D-SE-0092877).

| Value | Data type | Description |
| --- | --- | --- |
| left / 0 | BOOL | in counterclockwise direction (looking at the shaft) |
| right / 1 | BOOL | in clockwise direction (looking at the shaft) |
