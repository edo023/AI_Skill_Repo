# Value

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Cam Track |
| Traceable | Yes |

## Functional Description

Displays the current status of the output or the track. Value is a function of OnLevel.

| Value | Data type | Meaning |
| --- | --- | --- |
| false / 0 | BOOL | No output or track is enabled. |
| true / 1 | BOOL | An output or track is enabled. |
