# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Address | 16#000B |
| Offline editable | Yes |
| Devices supporting the parameter | Ethernet |
| Traceable | Yes |
| Default value | on / 1 |

## Functional Description

The parameter Enable determines the point in time the Ethernet object is activated.

* If the parameter Enable is set to off / 0, the Ethernet object is not activated when the controller is started.

  The Ethernet object can be activated at a later point in time by setting the value of the parameter to on / 1.
* If the value of the parameter Enable is on / 1, the Ethernet object is activated when the controller is started.

  NOTE: Once the Ethernet object has been activated, it cannot be deactivated without a controller restart. If you try to deactivate the Ethernet object by setting the parameter value from on / 1 to off / 0, the value remains on /1 and the diagnostics message  8740 Device cannot be disabled is appended to the message logger list. Procedure for deactivating the Ethernet object:
  + Open the project.
  + Set the value of the parameter to off / 0.
  + Download the project.
  + Restart the controller.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The Ethernet object is not activated when the controller is started. The Ethernet object can be activated at a later point in time by setting the value of the parameter to on / 1. |
| on / 1 | BOOL | The Ethernet object is activated when the controller is started. |
