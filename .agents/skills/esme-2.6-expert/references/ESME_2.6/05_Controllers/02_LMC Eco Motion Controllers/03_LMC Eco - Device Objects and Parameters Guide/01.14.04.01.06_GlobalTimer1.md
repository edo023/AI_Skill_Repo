# GlobalTimer1

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Encoder network (Synchronization master and Synchronization slave) |
| Traceable | Yes |

## Functional Description

Controller synchronization uses one central clock as the foundation for the entire encoder network. This clock can be displayed using the parameter GlobalTimer1. Its time stamp is accurate to the millisecond and is incremented in the synchronization cycle or the [SERCOS cycle](D-SE-0073362.html#D-SE-0073362). This parameter also exists on the [synchronization slave](D-SE-0076437.html#D-SE-0076437), thus making it available to all slaves in the encoder network.
