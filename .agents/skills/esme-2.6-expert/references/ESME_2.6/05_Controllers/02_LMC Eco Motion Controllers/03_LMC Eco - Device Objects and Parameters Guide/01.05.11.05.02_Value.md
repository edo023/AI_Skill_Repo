# Value

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C,  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

## Functional Description

The parameter represents the value of the input. Input values are 0 V on state LOW and 24 V on state HIGH.
