# Valid

## General

|  |  |
| --- | --- |
| Type | EF |
| Address | 16#00010 |
| Offline editable | No |
| Devices supporting the parameter | Ethernet |
| Traceable | Yes |
| Default value | yes / 1 |

## Functional Description

The parameter lets you verify whether the user function is activated. This is used for dynamic machine configuration.

The value is yes / 1 (activated) for the Ethernet object because adding the Ethernet device implicitly activates the user function.

| Value | Data type | Meaning |
| --- | --- | --- |
| yes / 1 | BOOL | User function is activated. |
