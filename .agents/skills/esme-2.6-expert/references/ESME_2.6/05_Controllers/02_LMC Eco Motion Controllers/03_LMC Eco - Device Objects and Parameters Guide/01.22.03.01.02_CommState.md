# CommState

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave,  Field bus EtherNet/IP adapter  Field bus EtherCATSlave |
| Traceable | Yes |

## Functional Description

Displays the current network status of the communication channel.

| Value | Data type | Meaning |
| --- | --- | --- |
| UNKNOWN / 0 | UDINT | Indeterminable |
| NOT\_CONFIGURED / 1 | UDINT | Not configured |
| STOP / 2 | UDINT | Stopped |
| IDLE / 3 | UDINT | Idle |
| OPERATE / 4 | UDINT | Operation |
