# RTBReadRes

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The RTBReadRes parameter is another indicator for evaluating the real-time process utilization. Displays the real-time read resource in µs.
