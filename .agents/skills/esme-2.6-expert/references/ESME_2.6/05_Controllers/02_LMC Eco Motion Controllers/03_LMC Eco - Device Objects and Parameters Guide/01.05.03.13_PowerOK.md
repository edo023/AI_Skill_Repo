# PowerOK

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The state of the parameter switches from TRUE (1) to FALSE (0) when the supply voltage falls below 18 V. If this value is exceeded, the state switches from FALSE (0) to TRUE (1) again. No entry is made in the message logger.

| Value | Data type | Meaning |
| --- | --- | --- |
| False / 0 | BOOL | Power supply voltage below 18 V |
| True / 1 | BOOL | Power supply voltage above 18 V |
