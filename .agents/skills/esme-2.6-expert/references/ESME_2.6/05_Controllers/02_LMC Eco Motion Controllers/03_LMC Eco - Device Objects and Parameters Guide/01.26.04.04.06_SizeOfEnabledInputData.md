# SizeOfEnabledInputData

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Logic Motion Controller Device |
| Traceable: | Yes |

## Functional Description

Displays the size of the enabled C2C input data.

Displays the sum of data size which has been configured for the enabled C2C input objects (C2C Encoder Input, C2C Data Input).

EIO0000002285.11
