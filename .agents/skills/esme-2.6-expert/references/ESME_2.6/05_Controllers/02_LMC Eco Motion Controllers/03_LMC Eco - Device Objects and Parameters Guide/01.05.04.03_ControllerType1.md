# ControllerType1

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays various hardware information.

This parameter contains the following information:

* FPGA version
* CPU version
* PIC version (for controller PacDrive LMC Eco)

  Syntax: PIC=xxxx.yyyy.zzzz.aa.bb

  + xxxx : Hardware PACIO Board version
  + yyyy : Bootloader version
  + zzzz : Software version (updateable firmware image)
  + aa : Microcontroller configuration control bytes (changes within a version)
  + bb : PICIO firmware mode byte

    00 = Bootloader active (in update mode)

    01 = Firmware image active (general mode)
* Bios version

  Syntax: BIOS=xx.xx.x-xxxx

EIO0000002285.11
