# State

## General

|  |  |
| --- | --- |
| Type | AF |
| Address | 16#000C |
| Offline editable | No |
| Devices supporting the parameter | Ethernet |
| Traceable | Yes |
| Default value | off / 0 |

## Functional Description

The parameter indicates the state of the Ethernet object of the Ethernet interface with reference to the process of loading the Ethernet firmware and the availability of the Ethernet object for Ethernet communication.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | DINT | The Ethernet object is not active. The Ethernet firmware has not been loaded. |
| loading / 1 | DINT | The Ethernet firmware is loading. |
| down / 2 | DINT | The Ethernet firmware has been loaded.  The Ethernet object is unavailable for Ethernet communication. |
| running / 3 | DINT | The Ethernet object is activated.  The Ethernet firmware has been loaded.  The Ethernet object is available for Ethernet communication. |
