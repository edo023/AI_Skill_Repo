# OutputsActive

## General

|  |  |
| --- | --- |
| Type | AD |
| Offline editable | No |
| Traceable | No |

## Functional Description

Indicates whether the outputs are active.

| Value | Data type | Meaning |
| --- | --- | --- |
| TRUE | BOOL | The outputs are active. |
| FALSE | BOOL | The outputs are inactive. |
