# OptRight

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Shows the type of the right optional module which is connected to the PacDrive LMC x00C or PacDrive LMC x01C (**OM-P/OM-C** or **OM-NE**). For an OM-NE, the firmware is specified additionally.

| Value | Data type | Meaning |
| --- | --- | --- |
| OM-P/OM-C | STRING | OM-P: Optional module with PROFIBUS connector  OM-C: Optional module with CANopen connector |
| OM-NE | STRING | OM-NE: Optional module with netX and two EtherNet connectors. |
