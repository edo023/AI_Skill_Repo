# PhaseEnable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Log. encoder |
| Traceable | Yes |

Enables/disables the generator.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The generator switched off. |
| on / 1 | BOOL | Setting the parameter to "on / 1" activates (initializes) the phase generator. |

NOTE: PhaseEnable must not be set to "on / 1" cyclically as this leads to a cyclical initialization of the phase generator.
