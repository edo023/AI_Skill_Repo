# ConfiguredTopologyAddress

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Traceable | Yes |

## Functional Description

If IdentificationMode = TopologyAddress / 2, the value of ConfiguredTopologyAddress is used for identifying the device in the Sercos network (see parameter [IdentificationMode](IdentificationMode-2F96D150.html#IdentificationMode-2F96D150)). It is applied in the Sercos phase change from 1 to 2.

| Value range | Data type | Meaning |
| --- | --- | --- |
| (1…512) | UINT | The topological address where the device is expected inside the Sercos network. |
