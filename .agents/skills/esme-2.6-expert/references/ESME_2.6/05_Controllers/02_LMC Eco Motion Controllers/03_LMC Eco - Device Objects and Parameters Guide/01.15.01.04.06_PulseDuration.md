# PulseDuration

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam switch |
| Traceable | Yes |

## Functional Description

In Pulsemode, specifies the time in ms for which the output stays enabled. The time is rounded to a multiple of the Interval of the Cam switch group.

In Posmode, this parameter has no function.

The operating mode (Posmode or Pulsemode) is set using the parameter Mode.
