# FDRConfirmationMode

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | No |

## Functional Description

The parameter FDRConfirmationMode specifies the way the FDR function is acknowledged.

| Value | Data type | Meaning |
| --- | --- | --- |
| using display / 0  (default) | DINT | The FDR function is acknowledged through the controller display. |
| using parameter / 1 | DINT | The FDR function is acknowledged by a parameter. |
