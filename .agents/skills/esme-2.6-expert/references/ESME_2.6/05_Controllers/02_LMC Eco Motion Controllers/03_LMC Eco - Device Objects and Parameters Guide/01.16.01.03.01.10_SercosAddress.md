# SercosAddress

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Traceable | Yes |

## Functional Description

The parameter represents the Sercos address to which the device has been assigned. The Sercos address is automatically assigned by the master during the phase-up of Sercos, provided that the device is not in the identification mode SercosAddress / 4 (see parameter [IdentificationMode](IdentificationMode-2F96D150.html#IdentificationMode-2F96D150)).

The Sercos address can only be modified by manually writing the Sercos parameter IDN S-0-1040.

The function [FC\_SercosWriteServiceData](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085305) can be used to modify the parameter.

| Value range | Data type | Meaning |
| --- | --- | --- |
| (1…511) | UINT | The Sercos address to which the device has been assigned. |
