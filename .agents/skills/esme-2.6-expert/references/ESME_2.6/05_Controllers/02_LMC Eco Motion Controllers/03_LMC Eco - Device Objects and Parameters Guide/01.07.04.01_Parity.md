# Parity

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | COM |
| Traceable | Yes |

## Functional Description

Defines the type of parity check.

| Value | Data type | Meaning |
| --- | --- | --- |
| None / 0 | DINT | No parity check |
| Even / 1 | DINT | Even parity |
| Odd / 2 | DINT | Odd parity |
