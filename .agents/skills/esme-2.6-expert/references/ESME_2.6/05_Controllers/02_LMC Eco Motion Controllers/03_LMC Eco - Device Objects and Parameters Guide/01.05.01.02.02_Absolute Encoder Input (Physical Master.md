# Absolute Encoder Input (Physical Master Encoder)

## Task

To be able to process [absolute value encoders](D-SE-0070736.html#D-SE-0070736) in the PacDrive system, these must be connected to the controller or on the bus terminal BT-4/ENC1. The Incremental Encoder Input must be attached to the controller (or the bus terminal) in the controller configuration to enable the PacDrive system to process the signal of the encoder.
