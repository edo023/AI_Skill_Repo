# SignalQualityLimit

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | SinCos Encoder Input |
| Traceable | Yes |

## Functional Description

The parameter determines at which value of SignalQuality a track monitoring error 8601 "Master encoder signal out of the range" is issued.

EIO0000002285.11
