# IPAddressRangeDynamic

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

This parameter displays the address range for automatic IP configurations.

| Value | Data type | Meaning |
| --- | --- | --- |
| 172.20.0.2-172.20.3.240 (default) | STRING(31) | This parameter indicates which address range is used for the automatic distribution of IP addresses.  This parameter is calculated automatically.  Access type: read |

NOTE: For addressing additional devices (for example, laptops), 14 addresses are reserved at the end of the available address range. (172.20.3.241-172.20.3.254). In the example configuration, in section Allocation of the IP addresses the addresses 172.20.3.241-172.20.3.254 are reserved.
