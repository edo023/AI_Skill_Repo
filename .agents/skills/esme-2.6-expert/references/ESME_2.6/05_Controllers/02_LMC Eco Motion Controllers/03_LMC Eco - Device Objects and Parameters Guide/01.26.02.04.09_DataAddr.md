# DataAddr

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | C2C Data Input, C2C Data output |
| Traceable | Yes |

## Functional Description

Used as pointer to the memory area, where the I/O data is stored.

The PacDrive LMC copies the data to this address. Using this address the data can be read (consumer) or written (producer) by the application.

NOTE: To maintain data consistency, it is possible to access the data either using a external event triggered task (RTP\_Read) or by a cyclic task using the system function FC\_C2CCopyData.
