# FDRSlavesToUpdate

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The parameter FDRSlavesToUpdate contains the number of attached devices to be updated.

The value is set once after the attached devices that can be updated have been determined.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0  (default) | UINT | Value remains 0 as long as the FDR function is not started. |
| >0 | UINT | The number of attached devices to be updated. |
