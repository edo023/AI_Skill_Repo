# Creating the C2C Master Project

## Description

To create the C2C Slave project, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Open EcoStruxure Machine Expert Logic Builder. |
| 2 | In the Logic Builder, select **File > New Project...** to create a new default project. |
| 3 | Enter a project name, select a controller, select **Structured Text (ST)** as **Language for SR\_Main** and confirm with **[OK]**. |
