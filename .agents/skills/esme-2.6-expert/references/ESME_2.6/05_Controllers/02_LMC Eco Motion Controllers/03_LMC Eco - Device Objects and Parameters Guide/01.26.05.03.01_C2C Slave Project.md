# C2C Slave Project

## Description

The following main points are performed in the C2C Slave project:

* The **C2C Encoder Input** of the C2C Slave (TTS2) receives encoder signals from the **C2C Encoder Output** of the C2C Master (TTS2).

  An axis (upper motor at TTS2 with the label "Axis 1") follows this encoder signal using a 1:1 CAM diagram.
* The **C2C Data Output** of the C2C Slave (TTS2) transmits the content of a data structure (ST\_MyData) to the **C2C Data Input** of the C2C Master (TTS1).

  The content of the data structure is changed in each application cycle for illustration purposes.
