# DataSize

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | C2C Data Input, C2C Data output |
| Traceable | Yes |

## Functional Description

Specifies the size of data, in [bytes], that is transferred via the C2C network (maximum of 116 bytes).

DataSize must be equal for producer and consumer with the same ID. Otherwise diagnostic message [8520](D-SE-0064010.html#D-SE-0064010) (Len mismatch) will be displayed in the message logger.
