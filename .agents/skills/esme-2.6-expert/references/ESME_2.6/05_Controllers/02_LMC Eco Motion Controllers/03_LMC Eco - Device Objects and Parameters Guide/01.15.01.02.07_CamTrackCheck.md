# CamTrackCheck

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam switch group |
| Traceable | No |

## Functional Description

Displays the Check values of the subordinate [CamTracks](D-SE-0077242.html#D-SE-0077242). Several or all subordinate CamTracks can be checked at the same time by writing this value.

The data type of the parameter is UDINT (unsigned double integer 32 Bit). Each bit is assigned to a Check parameter of a cam track (Cam Track). Bit 0 is assigned to the first cam track, bit 1 is assigned to the second cam track, and so on. The cam track which has been inserted directly underneath the cam switch group (Cam switch group) in the PLC configuration is the first cam track.
