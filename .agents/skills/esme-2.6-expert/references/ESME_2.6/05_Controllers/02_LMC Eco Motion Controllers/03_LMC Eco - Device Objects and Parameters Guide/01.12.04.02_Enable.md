# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Sum encoder input |
| Traceable | Yes |

Forwarding the position of the logical encoder can be interrupted using Enable.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | Forwarding disabled. |
| on / 1 | BOOL | Forwarding enabled. |
