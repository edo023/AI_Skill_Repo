# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Encoder network (Synchronization encoder output) |
| Traceable | Yes |

## Functional Description

Enables or disables the Synchronization encoder input.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The Synchronization encoder output is disabled. If the sync. encoder input is set from "on / 1" to "off / 0" while the encoder network is active, the value in the parameter State of the Synchronization slave will change from "active / 5" to "synchronized / 3". |
| on / 1 | BOOL | The Synchronization encoder input is enabled. Since all the controllers in the encoder network must be running synchronously and error-free for the encoder signals to be distributed, the parameter encoder input is shown in the parameter State. If the value in the parameter State is "5 / active", encoder data will be received correctly. |
