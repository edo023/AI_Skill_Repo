# OnlineChangeCounter

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The `OnlineChangeCounter` parameter can be queried to detect if an online change has been made.

The value indicates how many online change updates of the program sequences have been made since the last program change. Using this value, certain initialization steps can be called in the program after an online change.
