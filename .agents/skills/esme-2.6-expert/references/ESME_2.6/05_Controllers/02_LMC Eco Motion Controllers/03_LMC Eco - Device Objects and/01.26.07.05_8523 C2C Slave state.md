# 8523 C2C Slave state

## Description

[Refer to Diagnostic class (standard): 1](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063169)

The parameter SyncState of the C2C Slave object has been modified.

The extended diagnostic provides information on the state:

| Ext. diagnosis | Meaning |
| --- | --- |
| `WaitSercSlave` | Waiting until the superordinate network delivers a synchronization signal in communication phase 4. |
| `WaitSercMaster` | Waiting until the subnetwork is ready to be synchronized (at least communication phase 3). |
| `Error` | An error is detected.  Details on the detected error are displayed with diagnostic parameters and entries in the message logger. |
| `Sync ongoing` | The system is operational. The synchronization of superordinate network and subnetwork is active. |
| `Synchronized` | The subnetwork and the superordinate network are synchronized. |
