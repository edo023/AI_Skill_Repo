# DynIECDataSize

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

This parameter helps configure the size of the total available dynamic memory area (`DynIECData`). When you start a program, the system checks whether the DynIECDataSize parameter has changed. If it has changed, the system releases the memory area and reserves it again.

When you are planning, ensure that, when you start a program, at least 4 MB of memory is still available (Memoryfree parameter) after the system has reserved the dynamic memory area with the `DynIECDataSize` parameter. If not, the message logger records diagnostic message 8337 "Parameter DynIECDataSize value too high" and the dynamic memory area will not be reserved.
