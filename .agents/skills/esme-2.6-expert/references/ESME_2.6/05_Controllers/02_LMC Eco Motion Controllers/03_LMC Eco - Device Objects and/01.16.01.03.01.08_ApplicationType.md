# ApplicationType

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Traceable | No |

## Functional Description

The parameter represents the application type read from the physical Sercos device. The parameter ApplicationType can only be modified by manually writing the Sercos parameter IDN S-0-1302.0.3. The function [FC\_SercosWriteServiceData](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085305) can be used to modify the parameter.

| Data type | Meaning |
| --- | --- |
| STRING [40] | The application type read from the physical Sercos device. |

NOTE: The parameter value is read via Sercos in phase 1. The value is not updated in greater phases.
