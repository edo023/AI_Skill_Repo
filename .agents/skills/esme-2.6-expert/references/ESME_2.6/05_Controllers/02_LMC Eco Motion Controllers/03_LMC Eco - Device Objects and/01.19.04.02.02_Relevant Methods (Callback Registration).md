# Relevant Methods (Callback Registration)

## General

In the following, all relevant methods for the registration of the PROFIBUS\_DPV1\_Slave interface are listed:

| Method | Description |
| --- | --- |
| IsRegisteredAsyncAlarmAck | Returns whether the transferred FB instance is already registered for the callback upon arrival of an alarm acknowledge. |
| IsRegisteredAsyncRead | Returns whether the transferred FB instance is already registered for the callback upon arrival of a noncyclical read enquiry. |
| IsRegisteredAsyncWrite | Returns whether the transferred FB instance is already registered for the callback upon arrival of a noncyclical write enquiry. |
| RegisterAsyncAlarmAck | Registers the transferred FB instance for the callback with the PROFIBUS slave upon arrival of an alarm acknowledge. |
| RegisterAsyncRead | Registers the transferred FB instance for the callback with the PROFIBUS slave upon arrival of a noncyclical read enquiry. |
| RegisterAsyncWrite | Registers the transferred FB instance for the callback with the PROFIBUS slave upon arrival of a noncyclical write enquiry. |
| UnregisterAsyncAlarmAck | Unregisters the transferred FB instance for the callback with the PROFIBUS slave upon arrival of a noncyclical read enquiry. |
| UnregisterAsyncRead | Unregisters the transferred FB instance for the callback with the PROFIBUS slave upon arrival of a noncyclical read enquiry. |
| UnregisterAsyncWrite | Unregisters the transferred FB instance for the callback with the PROFIBUS slave upon arrival of a noncyclical write enquiry. |
