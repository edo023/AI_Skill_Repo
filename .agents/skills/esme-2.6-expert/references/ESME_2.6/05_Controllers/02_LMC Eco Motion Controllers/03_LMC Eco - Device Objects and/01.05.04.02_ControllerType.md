# ControllerType

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

The parameter `ControllerType` for the PacDrive LMC Pro2 is compatible to the PacDrive LMC Pro controllers. The parameter is of type STRING(254). The content depends on the number of axes of the PacDrive LMC Pro2 controller.

| Number of axes | Controller type | PacDrive controller |
| --- | --- | --- |
| 0 < N <= 8 | LMC300C/AX=8/RAM=768/NVRAM=256/DiskS=127 | PacDrive LMC 302C |
| 9 < N <= 16 | LMC400C/AX=16/RAM=768/NVRAM=256/DiskS=127 | PacDrive LMC 402C |
| 17 < N <= 99  N >= 99 | LMC600C/AX=99/RAM=768/NVRAM=256/DiskS=127 | PacDrive LMC 802C |
