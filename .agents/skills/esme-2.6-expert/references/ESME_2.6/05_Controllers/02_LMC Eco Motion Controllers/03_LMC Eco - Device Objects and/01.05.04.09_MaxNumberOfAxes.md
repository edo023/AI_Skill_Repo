# MaxNumberOfAxes

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

The parameter of type UDINT shows the maximum number of supported synchronous axes running in closed loop.
