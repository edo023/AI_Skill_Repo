# EthernetConfig

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus EtherNet/IP adapter |
| Traceable | Yes |

## Functional Description

Displays the Ethernet configuration of the EtherNet/IP module.

| Value | Data type | Meaning |
| --- | --- | --- |
| Auto-negotiation / 0 | BYTE | If this value is set, the device independently negotiates the connection parameters with the remote hub or the switch. |
| Full Duplex / 100 MBit/s / 1 | BYTE | The device works at 100 MBit/s and in full duplex. |
| Full Duplex / 10 MBit/s / 2 | BYTE | The device works at 10 MBit/s and in full duplex. |
| Half Duplex / 100 MBit/s / 3 | BYTE | The device works at 100 MBit/s and in half duplex. |
| Half Duplex / 10 MBit/s / 4 | BYTE | The device works at 10 MBit/s and in half duplex. |
