# PositionType

## General

|  |  |
| --- | --- |
| Type | EF |

Sets the position type supplied by the PositionSource.

| Value | Data type | Meaning |
| --- | --- | --- |
| Drive\_MechPosition / 0 | DINT | Position type *[MechPosition](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071499)* of an axis |
| Drive\_MechRefPosition / 1 | DINT | Position type *[MechRefPosition](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071492)* of an axis |
| Drive\_ShaftMechPosition / 2 | DINT | Position type *[ShaftMechPosition](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071503)* of an axis |
| Drive\_ShaftMechRefPosition / 3 | DINT | Position type *[ShaftMechRefPosition](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071498)* of an axis |
| LEnc\_Position / 4 | DINT | Position type Position of a logical encoder |
| LEnc\_PositionWithoutDelay / 5 | DINT | Position type PositionWithoutDelay of a logical encoder |
