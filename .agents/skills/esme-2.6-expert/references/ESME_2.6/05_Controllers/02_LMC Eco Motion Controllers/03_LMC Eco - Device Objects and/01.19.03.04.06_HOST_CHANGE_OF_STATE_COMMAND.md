# HOST\_CHANGE\_OF\_STATE\_COMMAND

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave, Field bus EtherNet/IP adapter |
| Traceable | Yes |

## Functional Description

Flag used by the protocol stack to indicate that the new state of the host application has been read.

EIO0000002285.11
