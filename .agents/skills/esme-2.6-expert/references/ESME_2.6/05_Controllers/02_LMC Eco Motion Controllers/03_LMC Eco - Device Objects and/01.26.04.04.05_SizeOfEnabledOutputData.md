# SizeOfEnabledOutputData

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Logic Motion Controller Device |
| Traceable: | Yes |

## Functional Description

Displays the size of the enabled C2C output data.

Displays the sum of data size which has been configured for the enabled C2C output objects (C2C Encoder Output, C2C Data Output).

EIO0000002285.11
