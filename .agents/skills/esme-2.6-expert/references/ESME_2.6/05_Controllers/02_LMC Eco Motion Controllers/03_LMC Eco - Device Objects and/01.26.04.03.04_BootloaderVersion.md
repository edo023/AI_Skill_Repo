# BootloaderVersion

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | Logic Motion Controller Device |
| Traceable | No |

## Functional Description

Displays the version name of the bootloader of the device. The value is available as of Sercos phase 2. The bootloader version consists of the following parts:

V<Generation>.<Family>.<Revision>.<Build>.

The version number part from generation and family of the bootloader must always be less or equal to the corresponding version number part of the firmware.

If the device in the Devices tree is virtual (non-real), then the parameter value is DRIVE\_VIRTUAL.

NOTE: If device does not support BootloaderVersion, then this parameter contains '' (empty string).
