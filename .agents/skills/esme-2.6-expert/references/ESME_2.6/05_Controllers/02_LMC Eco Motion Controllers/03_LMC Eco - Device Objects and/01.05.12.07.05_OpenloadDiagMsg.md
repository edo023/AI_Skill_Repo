# OpenloadDiagMsg

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | No |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C,  OnBoard I/O, DIO8 module |
| Traceable | Yes |

## Functional Description

The parameter is used to enter a bit to enable or disable the diagnostic message `8788 Wiring error detected` with ext. diagnostic message `Openload` for this digital field output.

This bit in the parameter OpenloadDiagMsg is assigned to a corresponding bit in the bitmask of the parameter *[OpenloadDiagMsk](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071985)*. Modifications in the parameter OpenloadDiagMsg are automatically written to the corresponding bit in the parameter OpenloadDiagMsk and vice-versa. The last modification is active. Also see parameter *[OverloadDiagMsg](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0089342)*.

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | UINT | The output does not report the diagnostic message `8788 Detected wiring error (Openload)`. |
| H / 1 | UINT | The output reports the diagnostic message `8788 Detected wiring error (Openload)`. |
