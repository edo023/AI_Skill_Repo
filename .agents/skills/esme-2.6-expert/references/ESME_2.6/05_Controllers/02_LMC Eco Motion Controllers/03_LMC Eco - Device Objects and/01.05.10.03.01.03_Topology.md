# Topology

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

Describes the topology of the Sercos system.

| Value | Data type | Meaning |
| --- | --- | --- |
| no link / 0 | DINT | A Sercos device is not connected to port 1 or to port 2 |
| Line P1 / 1 | DINT | The Sercos devices are connected to port 1 |
| Line P2 / 2 |  | All Sercos devices are connected to port 2 |
| Double line / 3 | DINT | Sercos devices are connected to port 1 and port 2 |
| The ring is closed / 4 | DINT | The connection from Port 1 to Port 2 is closed |
| interrupted ring / 8 | DINT | Invalid ring topology, topology switchover not yet completed |
