# LicenseUpdateMode

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the license points requirement status.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | DINT | This is the normal state when no changes are made to the license points. |
| request key / 1 | DINT | This status is used to create the request key that you have to send with the order for the required number of points to the internal sales service of. |
| authorization key / 2 | DINT | In this state, you can enter the authorization key that releases the new number of points. |
