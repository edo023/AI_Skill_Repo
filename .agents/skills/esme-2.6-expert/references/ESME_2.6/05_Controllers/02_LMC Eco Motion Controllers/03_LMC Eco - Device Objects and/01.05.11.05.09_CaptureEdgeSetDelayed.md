# CaptureEdgeSetDelayed

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x01C |
| Traceable |  |

## Functional Description

Displays the delay on change of CaptureEdge.

TRUE if the change of the edge is not set directly in the hardware.

| Value | Data type | Meaning |
| --- | --- | --- |
| False / 0 (default) | BOOL | The change of the edge is set directly in the hardware. |
| True / 1 | BOOL | The change of the edge is **not** set directly in the hardware.  NOTE: This is the case with TM5SDM8DTS\_TSIOO / TM5SDM8DTS\_TSITSO module touchprobe objects. |

EIO0000002285.11
