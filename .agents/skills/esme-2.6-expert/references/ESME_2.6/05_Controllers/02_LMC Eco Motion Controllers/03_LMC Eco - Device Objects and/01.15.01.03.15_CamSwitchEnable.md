# CamSwitchEnable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam Track |
| Traceable | No |

## Functional Description

Displays the Enable values of the subordinate cam switches (Cam switch). All subordinate cam switches can be activated or deactivated at the same by writing this value.

The data type of the parameter is UDINT (unsigned double integer 32 Bit). Each bit is assigned to an Enable parameter of a cam switch (Cam switch). Bit 0 is assigned to the first cam track, bit 1 is assigned to the second cam track, and so on. The cam track which has been inserted directly underneath the cam switch group (Cam Track) in the PLC configuration is the first cam track.
