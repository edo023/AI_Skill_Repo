# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Encoder network (Sync. Module) |
| Traceable | Yes |

## Functional Description

Enabling or disabling the [synchronization slave](D-SE-0076437.html#D-SE-0076437) which is defined via the SlaveIPAddress.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | When you start the synchronization (using the Enable parameter on the [synchronization master](D-SE-0076436.html#D-SE-0076436)), no data is exchanged with the Synchronization slave.  If Enable is set to "off / 0" while synchronization is in progress, only the error monitoring in the Synchronization master is disabled. That means you can disable the Synchronization slave, but the Synchronization master will not receive a diagnostic message and can continue to work normally.  NOTE: Corresponding encoder in- and output objects that are assigned to this object should be disabled at this time.  You cannot enable a slave ("off / 0" -> "on / 1") again while the synchronization is in progress. |
| on / 1 | BOOL | The system tries to establish the synchronization from the Synchronization master to the Synchronization slave.   * 8333 "EncNet receiving data not possible" * 8334 "EncNet receiving data dist. detected" * 8335 "EncNet synchronization not possible" * 8336 "EncNet Sync disturbance detected" |
