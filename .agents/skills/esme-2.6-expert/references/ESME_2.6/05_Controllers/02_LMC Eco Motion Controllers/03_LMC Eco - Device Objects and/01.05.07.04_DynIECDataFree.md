# DynIECDataFree

## Generall

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Descriptions

Controls the dynamic data area (DynIECData) during runtime. You can use this parameter to evaluate how much of the DynIECData area is currently available.
