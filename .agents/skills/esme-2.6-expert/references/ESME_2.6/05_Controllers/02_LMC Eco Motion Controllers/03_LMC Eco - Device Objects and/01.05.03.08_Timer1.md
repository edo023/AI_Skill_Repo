# Timer1

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Sets the timer in ms. After reaching the maximum (231-1 ms, that is, approximately 24.8 days), it starts counting upwards from –231. The counter is automatically started after a reset.

NOTE: The resolution of Timer1 is the CycleTime of the Sercos drive bus.
