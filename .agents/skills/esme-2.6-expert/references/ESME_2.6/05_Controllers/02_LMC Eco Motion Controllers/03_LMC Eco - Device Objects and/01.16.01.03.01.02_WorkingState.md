# WorkingState

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Traceable | Yes |

## Functional Description

The parameter represents the active working mode of the device.

Validate this parameter when the parameter WorkingMode is set to optional / 3.

| Value | Data type | Meaning |
| --- | --- | --- |
| virtual / 0 | DINT | The device is operated virtually. |
| real / 1 | DINT | The device is found: The device is used. |
| deactivated / 2 | DINT | The device is deactivated (see parameter [WorkingMode](WorkingMode-2F931832.html#WorkingMode-2F931832)). |
