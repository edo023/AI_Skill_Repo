# FDRStartMode

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The parameter FDRStartMode specifies when the FDR function is started.

| Value | Data type | Meaning |
| --- | --- | --- |
| deactivated / 0  (default) | DINT | The FDR function is disabled. |
| boot / 1 | DINT | The FDR function is executed during the first Sercos phase-up when the PacDrive controller boots. |
| phase up / 2 | DINT | The FDR function is started during the transition of Sercos from communication phase 1 to communication phase 2. |
