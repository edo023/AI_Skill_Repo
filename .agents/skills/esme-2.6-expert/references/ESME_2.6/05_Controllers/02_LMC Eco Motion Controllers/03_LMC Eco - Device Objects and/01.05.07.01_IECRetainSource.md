# IECRetainSource

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | No |

## Functional Description

Displays the retain area as string.

The string has the following structure:

Type name: Instance name

## Example

"LMCx00C.PacDrive3"

EIO0000002285.11
