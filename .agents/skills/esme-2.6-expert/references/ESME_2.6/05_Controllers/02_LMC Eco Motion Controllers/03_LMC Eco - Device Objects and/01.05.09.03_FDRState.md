# FDRState

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The parameter FDRState contains the state of the FDR function.

| Value | Data type | Meaning |
| --- | --- | --- |
| idle / 0 | DINT | The FDR function performs no activities. |
| checking / 1 | DINT | The FDR function verifies whether a replacement of the firmware of connected devices is required. |
| upcoming / 2 | DINT | The FDR function detected that a firmware replacement of connected devices is required. |
| in progress / 3 | DINT | The FDR function replaces the firmware of connected devices. |
| done / 4 | DINT | The FDR function was executed successfully. |
| error / 5 | DINT | An error was detected while the FDR function was executed. |
