# CaptureState

## General

|  |  |
| --- | --- |
| Type | AD |
| Traceable | Yes |

## Functional Description

Displays the status of the touch probe input.

The following can be displayed:

Values of the CaptureState parameter of the measured inputs for the FC\_TPEdge() function

| Value | Data type | Meaning |
| --- | --- | --- |
| inactive / 0 | DINT | The capture function is not enabled. |
| active / 1 | DINT | *FC\_TPEdge()* is enabled. |
| captured / 2 | DINT | A valid measure value was recorded. |
| overflow / 3 | DINT | Timeout of the pulse function (TP\_Pulse()) |
| disabled / 4 | DINT | The Touchprobe functionality is not available as Mode is not parameterized to "Touchprobe / 1". |
| not ready / 5 | DINT | The Touchprobe function is not available as the Sercos bus is not in phase 4. |
| virtual / 6 | DINT | The Touchprobe function is not available as the motor is virtual. |
