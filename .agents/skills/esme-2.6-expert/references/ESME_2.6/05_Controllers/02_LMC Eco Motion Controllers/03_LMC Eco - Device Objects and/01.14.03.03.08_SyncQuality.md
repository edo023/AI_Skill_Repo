# SyncQuality

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Encoder network (Sync. Module and Synchronization slave) |
| Traceable | Yes |

## Functional Description

Displays the synchronization quality in percent. Large variations in this parameter value mean that synchronization telegrams have a long delay. This delay could be due to network overload or CPU overload.

To help avoid problems due to high CPU load, provide an appropriate reserve (i.e. AvailableLoad at least 10%).

If SyncQuality drops to 0%, diagnostic message 8335 "EncNet synchronization not possible" will be triggered.
