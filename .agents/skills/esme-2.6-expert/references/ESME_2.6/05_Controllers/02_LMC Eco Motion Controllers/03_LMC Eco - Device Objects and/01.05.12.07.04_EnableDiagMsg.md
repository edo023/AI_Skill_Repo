# EnableDiagMsg

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | PacDrive LMC x00C  BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

## Functional Description

Enables or disables the diagnostic for the output.

| Value | Data type | Meaning |
| --- | --- | --- |
| H / TRUE | BOOL | The output reports the diagnostic message 8788 "Wiring error" (FC\_Overload or Openload). |
| L / FALSE | BOOL | The output does not report the diagnostic message 8788 "Wiring error" (FC\_Overload or Openload). |
