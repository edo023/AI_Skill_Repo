# FDRSlavesUpdated

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The parameter FDRSlavesUpdated contains the number of attached devices that have been updated.

The parameter value is set after each update of an attached device.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0  (default) | UINT | Value remains 0 as long as the FDR function is not started. |
| >0 | UINT | The number of updated attached devices. |
