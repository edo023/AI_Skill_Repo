# Sercos Scan

## Start Sercos Scan

Click the **[Start Sercos scan]** button to start searching for real, physical axes and power supplies that are connected to the Sercos bus of the selected controller.

To perform the search,

* the Sercos bus is switched to phase 0,

* all applications will be stopped,

* and all diagnostic messages must be confirmed.

After clicking the **[Start Sercos scan]** button you may see a dialog box with a corresponding note, if applicable. At this point, you can still cancel the scan process.

## Automatically Add Devices

At the end of the scan process, it is checked whether devices are connected to the Sercos bus that are not yet in the PLC Configuration.

If such devices are detected, a dialog box is displayed with the question whether the missing devices shall now be added to the PLC Configuration.

If you confirm with **Yes**, there is an attempt to add the missing devices to the PLC Configuration.

You do not have to make any entries to add them.

The devices are not specifically parameterized, but added with general basic settings.

Which devices can be added and what happens in case of an error is described in chapter [Manually add devices](D-SE-0088132.html#D-SE-0088132).

## Assigning Respective Sercos Objects to Connected Devices

After scanning the program attempts to assign Sercos objects from the PLC Configuration to the devices connected to the Sercos bus by using the [Identification mode](D-SE-0088128.html#D-SE-0088128). All devices that were assigned automatically are highlighted in a [non-white color](D-SE-0088131.html#D-SE-0088131).

In addition to the **identification mode** the **type** must match.

The column header of the **Scanned devices** [column](D-SE-0088125.html#D-SE-0088125) displays the number of scanned and automatically assigned devices.

NOTE: Automatic assignment is performed after each scan process. Assignments that already exist might be discarded. If values have been copied before from assigned Sercos devices (highlighted in green), there is a high probability that the previous assignment will be found again for these objects.
