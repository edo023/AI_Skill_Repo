# Relevant Methods

## Methods

| Name | Data type | Description |
| --- | --- | --- |
| IoDrvSendUnconnectedMessage | UDINT | This method sends an unconnected message to an adapter.  It is not necessary to establish a connection before.  The return value is the error code of the driver. |
| tSendUnconnectedMessage | EIP\_SendUnconnectedMessage (see below) | Parameters for sending an unconnected message. |

## EIP\_SendUnconnectedMessage (Structure)

| Input | Data type | Description |
| --- | --- | --- |
| xEnable | BOOL | Enable flag, on a rising edge.  Execution start, on a falling edge. Outputs are reset. |
| abyIpAddress | ARRAY [0 .. 3] OF BYTE | IP address of the slave to connect |
| byService | BYTE | CIP service code (CIP = Common Industrial Protocol) |
| uiClass | UINT | CIP class Id |
| uiInstance | UINT | CIP instance |
| uiAttribute | UINT | CIP attribute |

| Input/Output | Data type | Description |
| --- | --- | --- |
| pData | POINTER TO BYTE | Pointer to the message data |
| udiLength | UDINT | Length of the data |

| Output | Data type | Description |
| --- | --- | --- |
| udiState | UDINT | State of execution |
| udiGenericError | UDINT | Generic error code (CIP general status codes) |
| udiExtendedError | UDINT | Extended error code (according to CIP EIP adaption, extended error code not applied for EIP) |

| Internal use | Data type | Description |
| --- | --- | --- |
| udiJobId | UDINT | Internal use |
| xOldEnable | BOOL | Internal use |
| udiSeq | UDINT | Internal use |
