## Functional Description

The parameter enables the active edge at which an interrupt should be triggered to be preset.

| Value | Data type | Meaning |
| --- | --- | --- |
| Negative / 0 | DINT | Negative edge |
| Positive / 1 | DINT | Positive edge |
| Negative and positive / 2 | DINT | negative and positive edge |
