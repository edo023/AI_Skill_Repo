# FreezeSupported

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the parameter | Field bus PROFIBUS DPV1 slave |
| Traceable | Yes |

## Functional Description

Enables / disables the freeze mode. If the freeze mode is enabled, the freeze command is supported.

| Value | Data type | Meaning |
| --- | --- | --- |
| TRUE | BOOL | The freeze mode is activated. |
| FALSE | BOOL | The freeze mode is deactivated. |

EIO0000002285.11
