# Bitno

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam Track |
| Traceable | Yes |

## Functional Description

Displays the bit number within the Destination (for example 5).

The bit number for the first bit is 0, that is, the bit numbers for an 8-bit variable are 0 ... 7.

## Example

"5"

EIO0000002285.11
