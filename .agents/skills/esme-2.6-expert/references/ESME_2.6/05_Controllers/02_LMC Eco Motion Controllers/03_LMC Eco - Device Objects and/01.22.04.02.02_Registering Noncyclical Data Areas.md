# Registering Noncyclical Data Areas

## General

Data areas for noncyclical enquiries can be registered with the EtherNet/IP adapter. Noncyclical enquiries (Get\_Attribute\_All, Get\_Attribute\_Single and Set\_Attribute\_Single) to registered data areas are then automatically processed by the EtherNet/IP adapter. A noncyclical data area is addressed by means of a ClassId, an InstanceId and an AttributeId.

For this purpose, an attribute of type ST\_EtherNetIPAttribute has to be created. This attribute has to contain the following program contents: an Id, a length, and a pointer to a data area (e.g. an array).

Ensure that the area to which the pointer of an attribute points actually exists during operation and is not deleted. For this purpose, for example declare the array as a variable in the program.

By means of the interface IF\_EtherNetIPInstance, the attribute is then registered with an instance of the function block FB\_EtherNetIPInstance.

NOTE: Ensure that the instances of the function block FB\_EtherNetIPInstance as well as the attributes are available permanently and not accidentally deleted during operation (for example local variables of a method are deleted upon completion of the method).

In order to register the instance with the EtherNet/IP adapter, the created noncyclical instance has to contain a ClassId and an InstanceId.

These can be set using the methods SetClassId and SetInstanceId.

The instances are registered by means of the IF\_EtherNetIP\_Adapter interface of the EtherNet/IP adapter.

An example program for registering noncyclical data areas can be found [here](#D-SE-0081883__D-SE-0081883.4).

A list of all relevant methods and associated interfaces can be found under [Relevant methods (Data area registration)](#D-SE-0081883__D-SE-0081883.3).

## Relevant Methods (Data Area Registration)

In the following, all relevant methods for the registration of the data areas of the IF\_EtherNetIPAdapter and IF\_EtherNetIPInstance interfaces are listed:

Interface IF\_EtherNetIP\_Adapter

| Method | Description |
| --- | --- |
| RegisterInstance | Registers the instance with the EtherNet/IP adapter.  Return value 0: Instance is registered.  Return value 1: Instance is already registered.  Return value 2: The instance is invalid.  Return value 3: Internal error |
| UnregisterAllInstances | Unregisters all instances that have been registered with the EtherNet/IP adapter.  Return value 0: All registrations have been removed.  Return value 1: Internal error  Return value 2: There are no registered instances.  Return value 3: Internal error |
| UnregisterInstance | Unregisters a defined instance in the EtherNet/IP adapter.  Return value 0: The registration has been made void.  Return value CmpErrors.Errors.ERR\_NO\_OBJECT: The last instance was not found.  Return value CmpErrors.Errors.ERR\_INVALID\_HANDLE: Internal error |

Interface declaration

| Method | Interface declaration |
| --- | --- |
| RegisterInstance | ``` METHOD RegisterInstance : UDINT VAR_INPUT    i_ifEtherNetIPInstance : IF_EtherNetIPInstance; END_VAR ``` |
| UnregisterAllInstances | ``` METHOD UnregisterAllInstances : UDINT ``` |
| UnregisterInstance | ``` METHOD UnregisterInstance : UDINT VAR_INPUT    i_ifEtherNetIPInstance : IF_EtherNetIPInstance; END_VAR ``` |

Interface IF\_EtherNetIPInstance

| Method | Description |
| --- | --- |
| SetClassId | Assigns a ClassId to an instance of the function block FB\_EtherNetIPInstance.  Return value 0: The `ClassId` is valid and set.  Return value 1: The `ClassId` is not between `Gc_EtherNetIPClass_MinID` and `Gc_EtherNetIPClass_MaxID`. |
| SetInstanceId | Assigns an InstanceID to an instance of the function block `FB_EtherNetIPInstance`.  Return value 0: The `InstanceID` is valid and set.  Return value 1: The `ClassId` is not between 0 and `Gc_EtherNetIPClass_MaxID`. |
| AddAttribute | Adds the transferred attribute to the instance of the function block `FB_EtherNetIPInstance`.  Return value 0: The attribute has been added.  Return value 1: The AttributeId is already used.  Return value 2: The attribute is invalid |
| AddAttributeArray | Splits an array up into several parts that have the size of `i_byAttributeLength` and adds every part as attribute to the instance of the function block `FB_EtherNetIPInstance`. The attributes IDs are numbered consecutively. The `o_byStartAttributeId` returns the AttributeId of the first added attribute. The length of the last added attribute may be less than the transfer parameter `i_byAttributeLength`.  Return value 0: All attributes have been added and no errors occured.  Return value 1: Not enough available space in the instance. |
| RemoveAttribute | Removes an attribute with a defined ID.  Return value 0: The attribute was removed successfully.  Return value CmpErrors.Errors.ERR\_INVALIDID: The attribute was not found. |
| RemoveAllAttributes | Removes all attributes within the instance and returns the number of removed attributes. |
| GetClassId | Returns the ClassId. |
| GetInstanceId | Returns the InstanceId. |
| GetAttribute | Returns the data of the transferred attribute ID in the in/out parameter iq\_stAttribute.  Return value CmpErrors.Errors.ERR\_INVALIDID: The attribute was not found. |
| GetAttributeCount | Returns the number of attributes. |
| GetInstanceFreeSize | Returns the size of the free space of the instance. |
| IsValid | Indicates whether the instance information is valid. For this purpose, ClassID and InstanceID have to be set. |
| GetFirstAttribute | Returns the data of the first attribute of the instance in the in/out parameter iq\_stAttribute.  Return value 0: Attributes have been found.  Return value 1: No attributes have been found. |
| GetNextAttribute | Returns the data of the next attribute of the instance in the in/out parameter iq\_stAttribute.  Return value 0: Attributes have been found.  Return value 1: No attributes have been found. |

## Example: Registration of Noncyclical Data Areas

```
PROGRAM SR_RegisterInstances
VAR
   xAlreadyRegistered : BOOL := FALSE;
   ifInstanceHandler : IF_EtherNetIP_Adapter := EtherNet_IP_Adapter;
   ifInstance : IF_EtherNetIPInstance;
   // abyAcyclicData : ARRAY[0..119] OF BYTE := [120(0)];
   astAttributes : ARRAY[0..2] OF ST_EtherNetIPAttribute;
   afbInstances : ARRAY[0..1] OF FB_EtherNetIPInstance;
   audiResults : ARRAY[0..4] OF UDINT := [5(0)];
   byIndex : BYTE := 0;
END_VAR
```

```
IF (xAlreadyRegistered=FALSE) THEN
   // Prepare class 105, instance 0
   ifInstance := afbInstances[0];
   ifInstance.SetClassId(105);
   ifInstance.SetInstanceId(0);
   astAttributes[0].iId := 0;
   astAttributes[0].bySize := 10;
   astAttributes[0].pbyValue := ADR(G_abyDataTransfer2[0]);
   audiResults[0] := ifInstance.AddAttribute(astAttributes[0]);
   astAttributes[1].iId := 1; astAttributes[1].bySize := 10;
   astAttributes[1].pbyValue := ADR(G_abyDataTransfer2[10]);
   audiResults[1] := ifInstance.AddAttribute(astAttributes[1]);
   // Prepare class 105, instance 1
   ifInstance := afbInstances[1];
   ifInstance.SetClassId(105);
   ifInstance.SetInstanceId(1);
   astAttributes[2].iId := 0;
   astAttributes[2].bySize := c_byDataLength2 - 20;
   astAttributes[2].pbyValue := ADR(G_abyDataTransfer2[20]);
   audiResults[2] := ifInstance.AddAttribute(astAttributes[2]);
   // Register instances 0 and 1 of class 105
   audiResults[3] := ifInstanceHandler.RegisterInstance(afbInstances[0]);
   audiResults[4] := ifInstanceHandler.RegisterInstance(afbInstances[1]);
   // Set data for acyclic tranfer
   FOR (byIndex:=0) TO c_byDataLength2 DO
      G_abyDataTransfer2[byIndex] := byIndex * 2;
   END_FOR
   xAlreadyRegistered := TRUE;
END_IF
```
