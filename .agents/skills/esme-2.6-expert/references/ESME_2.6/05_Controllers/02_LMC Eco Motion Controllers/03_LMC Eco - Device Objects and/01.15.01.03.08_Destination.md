# Destination

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam Track |
| Traceable | No |

## Functional Description

Displays the name of the LMC output group or a PacNet output group by using the symbolic name. The name is defined in the PLC configuration (for example OutputGroup).

If no such physical output is connected, the destination is none.

NOTE: Outputs of drives are not supported. For those outputs, copy the Cam Track Value to your application

Fieldbus outputs (including TM5) can be addressed via destination Q.

To address a single bit: Byte address \* 8 + bit number.

The parameter Bitno is limited to 0...4095.

It takes up to 4 bus cycles (TM5), for a higher performance (within 1 ms), use the LMC or PacNet outputs.

The destination M allows you to write into the marker area.
