# ZeroTrackDetected

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Incremental Encoder Input |
| Traceable | Yes |

## Functional Description

When [zero track detection](D-SE-0075789.html#D-SE-0075789) is enabled, this bit indicates whether a zero track has been detected.

| Value | Data type | Meaning |
| --- | --- | --- |
| false / 0 | BOOL | No Zero Track detected. |
| true / 1 | BOOL | Zero Track detected. |

NOTE: When a zero track is detected, the Position of the incremental encoder input is set to 0. The detection of a zero impulse has no effect on the Position parameter of any incremental encoder simulation output that may be present.
