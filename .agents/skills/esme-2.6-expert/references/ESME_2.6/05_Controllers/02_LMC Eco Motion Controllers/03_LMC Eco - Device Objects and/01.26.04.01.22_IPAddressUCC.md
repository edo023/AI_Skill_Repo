# IPAddressUCC

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | C2C\_Slave,  Logic Motion Controller Device |
| Traceable | No |

## Functional Description

With this parameter, you can define the IP address for the UCC (**U**nified **C**ommunication **C**hannel) communication.

| Value | Data type | Meaning |
| --- | --- | --- |
| 172.25.0.x (default) | STRING(15) | IP address of the slave device for the UCC communication |
