# FDRModulesToUpdate

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

The parameter FDRModulesToUpdate contains the number of modules to be updated.

The value is set once after the modules that can be updated have been determined.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0  (default) | UINT | Value remains 0 as long as the FDR function is not started. |
| >0 | UINT | The number of modules to be updated. |
