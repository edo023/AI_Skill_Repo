# Supported Security Options of the Application

The controller supports security options for downloads, online changes, and boot applications.

The supported security options are:

* Encryption with certificates
* Sign with certificate

Not supported security options:

* Simple Encryption
* Encryption with license management

For more details refer to [Menu Commands Online Help, Security](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0083921_25).

NOTE: The certificates are stored on the memory card of the controller. Take into consideration that the memory card can be removed from the controller, allowing unauthorized persons to access the certificates.
