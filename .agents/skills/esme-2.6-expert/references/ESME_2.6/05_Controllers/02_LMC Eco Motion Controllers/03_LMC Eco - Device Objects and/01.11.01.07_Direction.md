# Direction

## General

|  |  |
| --- | --- |
| Type | ED |
| Devices supporting the parameter | Log. encoder |
| Traceable | Yes |

## Functional Description

You can use Direction to dynamically change the direction of rotation of the [input velocity](D-SE-0075825.html#D-SE-0075825).

| Value | Data type | Meaning |
| --- | --- | --- |
| left / 0 | BOOL | The direction of rotation is changed. That is, the velocity signal is negated. |
| right / 1 | BOOL | Direction of rotation is not changed. That is, the velocity signal is not influenced. |

NOTE: Corruption of the velocity signal during a change of Direction.

The velocity signal is briefly corrupted, which can cause other errors.

Only change the parameter value at encoder standstill.

Maintain the standstill for at least the set [filter time](D-SE-0075708.html#D-SE-0075708).
