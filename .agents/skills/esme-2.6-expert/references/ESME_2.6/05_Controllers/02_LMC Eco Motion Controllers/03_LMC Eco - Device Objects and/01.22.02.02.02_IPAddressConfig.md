# IPAddressConfig

## General

|  |  |
| --- | --- |
| Type |  |
| Devices supporting the paramter | Field bus EtherNet/IP adapter |
| Traceable | Yes |

## Functional Description

Defines how the IP address of the slave is set.

| Value | Data type | Meaning |
| --- | --- | --- |
| manual / 0 | BYTE | Manually entered IP address is used. |
| enable BOOTP / 1 | BYTE | IP address is determined from the Bootstrap protocol. |
| enable DHCP / 2 | BYTE | IP address is determined from the Dynamic Host Configuration protocol. |
