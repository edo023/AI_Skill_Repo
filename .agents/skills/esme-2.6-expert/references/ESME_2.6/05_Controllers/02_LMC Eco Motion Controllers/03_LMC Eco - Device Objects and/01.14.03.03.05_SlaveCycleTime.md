# SlaveCycleTime

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Encoder network (Sync. Module and Synchronization slave) |
| Traceable | Yes |

## Functional Description

Defines the cycle time for the status telegrams of the [synchronization slave](D-SE-0076437.html#D-SE-0076437) to the [synchronization master](D-SE-0076436.html#D-SE-0076436) in the sync. module. This cycle time must be a multiple of the [SERCOS cycle time](D-SE-0073362.html#D-SE-0073362).

Use the parameter SlaveCycleErrorLimit To define an error tolerance for synchronization monitoring.
