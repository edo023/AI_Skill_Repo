# PositionSource

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Cam Track |
| Traceable | No |

## Functional Description

Displays the

Sets the position source of the cam track (Cam Track).

To this end, the symbolic name of a

* logical encoder (L\_ENC) or
* an axis

is entered. The name is defined in the PLC configuration (e.g. "MC\_Master"). If an axis is selected as a PositionSource, then the position type PositionType is used.
