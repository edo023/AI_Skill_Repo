# ConfiguredOutputsActive

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | No |
| Traceable | No |

## Functional Description

Enables the outputs of the I/O modules.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The outputs of the I/O modules are disabled. |
| on / 1 | BOOL | The outputs of the I/O modules are enabled. |
