# MasterCycleTime

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Encoder network (Synchronization master and Synchronization slave) |
| Traceable | Yes |

## Functional Description

Displays the cycle time at which the synchronization telegrams are sent by the Synchronization master This time is equal to the current [SERCOS cycle time](D-SE-0073362.html#D-SE-0073362).
