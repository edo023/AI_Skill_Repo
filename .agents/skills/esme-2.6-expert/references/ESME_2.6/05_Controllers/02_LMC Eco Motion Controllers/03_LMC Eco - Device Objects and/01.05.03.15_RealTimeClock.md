## Functional Description

The date is displayed according to the country set with the parameter Language.

The following "clocks" are available for the PacDrive controller:

* Real time clock:

  Clock with the date and time (with firmware version >=V1.64.x.x: local time; with firmware version <V1.64.x.x: UTC)
* [Timer1](D-SE-0073266.html#D-SE-0073266):

  Time since system startup in milliseconds
* [Timer10](D-SE-0073267.html#D-SE-0073267):

  Time since system startup in 10-millisecond resolution
* FastTimer: Time since the start of the CPU in µs synchronously with the timer of the CPU

The clocks run in synchronicity with the high-precision [Sercos](D-SE-0073405.html#D-SE-0073405) cycle. In phase 0 of the Sercos bus, various different influences (for example, an open Sercos ring) lead to imprecision in the clocks.

* The internal PC timer is used in phase 0. Because of this, the typical deviation of the time in `phase` 0 is only 4 s per hour.
