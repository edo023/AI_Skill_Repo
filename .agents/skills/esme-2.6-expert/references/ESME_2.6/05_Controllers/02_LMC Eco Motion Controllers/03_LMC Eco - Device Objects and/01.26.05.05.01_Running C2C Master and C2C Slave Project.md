# Running C2C Master and C2C Slave Project

## Description

Before the signals can be exchanged via the C2C network, the synchronization has to be established.

For this reason, a Sercos phase run-up is performed:

## Sercos Phase Up

After the [C2C Slave project](D-SE-0088178.html#D-SE-0088178) and [C2C Master project](D-SE-0088179.html#D-SE-0088179) were created the phase run-up of Sercos to the [communication phase](D-SE-0073356.html#D-SE-0073356) 4 must be performed on both systems.

After the communication phase 4 has been reached in C2C Master and C2C Slave, the slave automatically starts synchronizing the sub-network to the superordinated network.

If this is executed successfully, then the parameter SyncState of the C2C Slave is set to **Synchronized / 5**.

When all the connected C2C Slave are in state **Synchronized / 5**, then the parameter SyncGroupState of the C2C Master is set to **C2C network Synchronized / 5**.
