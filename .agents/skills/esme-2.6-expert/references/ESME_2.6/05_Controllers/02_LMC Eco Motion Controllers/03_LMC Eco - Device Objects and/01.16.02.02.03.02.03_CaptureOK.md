# CaptureOK

## General

|  |  |
| --- | --- |
| Type | AD |
| Traceable | Yes |

## Functional Description

Displays the status of the Touchprobe event.

A valid Touchprobe event is detected and the result has been provided in the parameter [CaptureValue](CaptureValue-E80202F5.html#CaptureValue-E80202F5). The value is reset to "no / 0" when *FC\_TPEdge()* is called again.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | The result of the Touchprobe event has been reset. |
| yes / 1 | BOOL | The result of the Touchprobe event is provided in the parameter CaptureValue. |
