# LexiumLXM52DriveVersion

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C |
| Traceable | No |

## Functional Description

The parameter LexiumLXM52DriveVersion contains the firmware version number of a Lexium LXM52 Drive. This firmware version is used by the FDR function.

When the FDR function is executed, each configured Sercos device of the type Lexium LXM52 Drive is updated to this firmware version.

The firmware for the update of the Sercos devices is stored on the memory card of the controller.

| Value | Data type | Meaning |
| --- | --- | --- |
| “none” | STRING(40) | No firmware for the Lexium LXM52 Drive is contained in the memory card of the controller. The FDR function does not perform a firmware update for the Lexium LXM52 Drive. |
| “x.x”  (default) | STIRING(40) | Firmware version used by the FDR function ( "x.x" represents the version number). |
