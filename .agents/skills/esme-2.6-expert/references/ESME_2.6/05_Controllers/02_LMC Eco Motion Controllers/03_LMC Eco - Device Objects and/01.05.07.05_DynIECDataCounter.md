# DynIECDataCounter

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | PacDrive LMC x00C,  PacDrive LMC x01C |
| Traceable | Yes |

## Functional Description

Controls the dynamic data area (DynIECData) during runtime. You can use this parameter to evaluate how many areas are currently reserved.
