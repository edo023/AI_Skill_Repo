# SerialNumber

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | PacDrive LMC x00C, PacDrive LMC x01C, C2C Slave, Logic Motion Controller Device |
| Traceable: | No |

## Functional Description

Displays the serial number of the controller. The serial number is a unique number which is used to identify the controller.The parameter value SerialNumber is part of the parameter [IdentificationMode](D-SE-0088238.html#D-SE-0088238).
