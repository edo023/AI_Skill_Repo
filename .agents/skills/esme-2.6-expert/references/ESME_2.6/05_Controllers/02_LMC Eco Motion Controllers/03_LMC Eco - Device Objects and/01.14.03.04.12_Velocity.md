# Velocity

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | SinCos Encoder Input,  Incremental Encoder Input,  Log. encoder,  Encoder network (Synchronization encoder output, Synchronization encoder input) |
| Traceable | Yes |

## Functional Description

Displays the actual filtered velocity of the encoder at the encoder shaft in units/s. A unit is defined using the parameter FeedConstant.
