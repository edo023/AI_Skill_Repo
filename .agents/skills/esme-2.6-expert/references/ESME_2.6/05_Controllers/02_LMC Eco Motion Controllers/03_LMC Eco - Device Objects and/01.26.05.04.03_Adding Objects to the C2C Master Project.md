# Adding Objects to the C2C Master Project

## Description

Add the following objects to the C2C Master project:

| Object | Added in the Devices tree under node | Use |
| --- | --- | --- |
| a **C2C Master** object | **LMC\_PacDrive** | Represents the C2C Slave in the C2C network. |
| a **C2C Encoder Output** object | **C2C\_Master** | Used to transmit an encoder signal to the C2C Slave (**C2C Encoder Input**) in the C2C network. |
| a **C2C Data Input** object | **C2C\_Master** | Used to receive user data from the C2C Slave (**C2C Data Output**) in the C2C network. |
| a **Logic Motion Controller Device** object | **Sercos\_Master** | Represents the C2C Slave on C2C Master side in the C2C network.  The object must be added as child of the Sercos object in the project of the PacDrive LMC acting as C2C Master. |

In this programming example, the [Device Addressing](D-SE-0088037.html#D-SE-0088037) functionality is used to identify the Sercos devices (PacDrive LMC600, Lexium 62 Double Drive) and the **Logic Motion Controller Device** object (**CTR\_LMC**) to add the Sercos device objects to the project. Refer to [Configuration and identification > Identification](D-SE-0088185.html#D-SE-0088185).
