# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | No |
| Devices supporting the parameter | PacDrive LMC x00C  C2C\_Master, C2C\_Slave |
| Traceable | Yes |

## Functional Description

Enables or disables the interrupt function for a certain input.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The interrupt function is disabled. |
| on / 1 | BOOL | The interrupt function is enabled |
