# MasterID

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Encoder network (Synchronization master) |
| Traceable | Yes |

## Functional Description

Specifies the MasterID for the multicast group. Using this parameter, you can provide the master with a unique address for sending cyclic information. This is necessary whenever there are several Synchronization masters in a single network or when other network slaves are using the default multicast address 224.100.0.0. Here, the MasterID is the third position of the multicast address.

## Example

That means, if the MasterID is 123, the multicast address would be 224.100.123.0.
