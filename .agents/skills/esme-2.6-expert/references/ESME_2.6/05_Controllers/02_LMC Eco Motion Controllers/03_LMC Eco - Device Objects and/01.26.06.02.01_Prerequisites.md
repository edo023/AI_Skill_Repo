# Prerequisites

## Running C2C System

You need a running C2C system:

* The C2C Master controller must contain a project with an enabled **C2C Master** object.
* The C2C Slave controllers must contain a project with an enabled **C2C Slave** object.
* The C2C Sercos network must be phased up to phase 4 (Sercos phase up needed as the Sercos Master distributes IP addresses to the Sercos Slaves in phase 2).
* A cable connection must be established between your PC (running EcoStruxure Machine Expert) and the Ethernet network interface of the C2C Master or a C2C Slave.

## Connection Mode Settings

To find the Schneider Electric controllers in a C2C network, perform the following settings in the **Controller selection** window:

* Set **Connection Mode** to **Nodename via NAT (Remote TCP)** or **IP Address via NAT (Remote TCP)**.
* Enter the C2C Master IP address for **NAT Address/Port**.

The **Controller selection** window is available in the controller device editors of the EcoStruxure Machine Expert Logic Builder, and in the Diagnostics and Controller Assistant standalone tools.
