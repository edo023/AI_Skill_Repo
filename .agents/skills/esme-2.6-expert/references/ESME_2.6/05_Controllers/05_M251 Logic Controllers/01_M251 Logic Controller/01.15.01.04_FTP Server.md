# FTP Server

## Introduction

Any FTP client that is connected to the controller (Ethernet port), without EcoStruxure Machine Expert installed, can be used to transfer files to and from the data storage area of the controller.

NOTE: When transferring text files to the controller, switch your FTP client into binary transfer type to avoid conversions of Carriage Return and LineFeed character pairs into a single LineFeed.

NOTE: Schneider Electric adheres to industry best practices in the development and implementation of control systems. This includes a "Defense-in-Depth" approach to secure an Industrial Control System. This approach places the controllers behind one or more firewalls to restrict access to authorized personnel and protocols only.

## FTP Access

Access to the FTP server is controlled by User Rights when they are enabled in the controller. For more information, refer to Users and Groups [Tabs Description](D-SE-0002288.html#D-SE-0002288__D-SE-0002288.4).

To access the FTP server you must first connect to the controller with EcoStruxure Machine Expert or Controller Assistant and activate the user rights or create the user for the first login.

NOTE: FTPS (explicit over TLS FTP) is set by default. Simple FTP (non secure) access is not possible at first connection. Set the parameter 1106 to 0 in the post configuration and reboot the controller to allow Simple FTP connection.

## FTP Client

The FtpRemoteFileHandling library provides the following FTP client functionalities for remote file handling:

* Reading files
* Writing files
* Deleting files
* Listing content of remote directories
* Adding directories
* Removing directories

For further information, refer to the [FtpRemoteFileHandling Library Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=ftplib&topicID=D_SE_0080681).

## Declaring the Connection to the FTP Server as Trusted

In the context of TLS, certificates can be used to verify the identity of the communication partners. Certificates are sent during the establishment of a connection, referred to as the TLS handshake.

The sending of the certificate is optional for the client, unless the server requests the client certificate. The server sends its certificate for every connection attempt. A connection with the communication partner can be established only if the result of the verification of the certificate is positive.

For secured FTP connections, the FTP server certificate must be provided to the M251 Logic Controller in the */.pki/trusted/cert* folder of the controller file system. The procedures for self-signed and signed certificates are described hereafter:

**Procedure for Self-Signed FTP Server Certificates**

On the first connection attempt to the FTP server, the self-signed server certificate is stored in the */.pki/untrusted* directory of the controller. The following procedure describes how to move the certificate to the */.pki/trusted/cert* directory using the [Security Screen](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0099371):

| Step | Action |
| --- | --- |
| 1 | Stop and restart your controller. |
| 2 | Login to your controller from your EcoStruxure Machine Expert FTP client application. |
| 3 | Click View > Security Screen, select the Devices tab, and then open the Untrusted Certificates folder of your controller. |
| 4 | Select the certificate and drag it to the Trusted Certificates folder of your controller. |

**Procedure for Signed FTP Server Certificates**

If the FTP server certificate is signed by a Certificate Authority (CA), you can save the root or an intermediate CA certificate to the */.pki/trusted/cert* directory of the controller before the first connection attempt to the FTP server. Proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Login to your controller from your EcoStruxure Machine Expert FTP client application. |
| 2 | Click View > Security Screen, select the Devices tab, and then open the Trusted Certificates folder of your controller. |
| 3 | Click the Download button and select the CA certificate. |
