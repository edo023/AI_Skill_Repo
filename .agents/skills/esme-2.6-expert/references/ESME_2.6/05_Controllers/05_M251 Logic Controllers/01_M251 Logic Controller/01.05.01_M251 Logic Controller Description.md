# M251 Logic Controller Description

## Overview

The M251 Logic Controller has various powerful features and can service a wide range of applications.

## Programming Languages

The M251 Logic Controller is configured and programmed with the EcoStruxure Machine Expert software which supports the following IEC 61131-3 programming languages:

* IL: Instruction list
* ST: Structured text
* FBD: Function block diagram
* SFC: Sequential function chart
* LD: Ladder diagram

The EcoStruxure Machine Expert software can also be used to program the M251 Logic Controller using CFC (continuous function chart) language.

## Power Supply

The power supply of the M251 Logic Controller is [24 Vdc](../../../../../api/crossBook?lang=en-US&virtualBookName=m251hw&topicID=D_SE_0036366).

## Real Time Clock

The M251 Logic Controller includes a [Real Time Clock (RTC) system](../../../../../api/crossBook?lang=en-US&virtualBookName=m251hw&topicID=D_SE_0025710).

## Run/Stop

The M251 Logic Controller can be operated externally by the following:

* A hardware [Run/Stop switch](../../../../../api/crossBook?lang=en-US&virtualBookName=m251hw&topicID=D_SE_0034418)
* A software command
* The system variable PLC\_W in a Relocation Table
* The Web server [Maintenance: Run/Stop Controller Submenu](MaintenanceMenu-0B989E57.html#MaintenanceMenu-0B989E57__MaintenanceRunStopControllerSubmenu-1D7C57F2)

## Memory

This table describes the different types of memory:

| Memory Type | Size | Use |
| --- | --- | --- |
| RAM | Refer to the table below | For the execution of the application and the firmware. |
| Non-volatile (flash) | 128 Mbytes | For the retention of the program and data in case of a power interruption. |

The RAM size depends on the PV value, which can be found in the [Services tab](D-SE-0006802.html) and the Web server [Diagnostics: Controller Submenu](DiagnosticsMenu-0B98B164.html#DiagnosticsMenu-0B98B164__DiagnosticsControllerSubmenu-1D7AFBC4):

| Reference | PV Value | RAM Size |
| --- | --- | --- |
| TM251•••• | < 20 | 64 Mbytes, of which 10 Mbytes are available for the application |
| ≥ 20 | 128 Mbytes, of which 20 Mbytes are available for the application |

## Removable Storage

The M251 Logic Controller includes an embedded [SD card slot](../../../../../api/crossBook?lang=en-US&virtualBookName=m251hw&topicID=D_SE_0027501).

The main uses of the SD card are:

* Initializing the controller with a new application
* [Updating the controller and expansion module firmware](D-SE-0035605.html#D-SE-0035605)
* [Applying post configuration files to the controller](D-SE-0010304.html)
* Applying recipes
* Receiving data logging files
* [Backup Data Logging File](D-SE-0004156.html#D-SE-0004156__D-SE-0004156.9)

## Embedded Communication Features

The M251 Logic Controller native communication ports include (depending on the controller reference):

* [CANopen Master](../../../../../api/crossBook?lang=en-US&virtualBookName=m251hw&topicID=D_SE_0036225)
* [Ethernet](../../../../../api/crossBook?lang=en-US&virtualBookName=m251hw&topicID=D_SE_0036226)
* [USB Mini-B](../../../../../api/crossBook?lang=en-US&virtualBookName=m251hw&topicID=D_SE_0029629)
* [Serial Line](../../../../../api/crossBook?lang=en-US&virtualBookName=m251hw&topicID=D_SE_0036135)
