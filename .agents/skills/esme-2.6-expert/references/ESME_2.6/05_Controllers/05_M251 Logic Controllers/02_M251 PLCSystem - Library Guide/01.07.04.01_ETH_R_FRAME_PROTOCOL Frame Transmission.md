# ETH\_R\_FRAME\_PROTOCOL: Frame Transmission Protocol Codes

## Enumerated Type Description

The ETH\_R\_FRAME\_PROTOCOL enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| ETH\_R\_802\_3 | 00 hex | The protocol used for frame transmission is IEEE 802.3. |
| ETH\_R\_ETHERNET\_II | 01 hex | The protocol used for frame transmission is Ethernet II. |
