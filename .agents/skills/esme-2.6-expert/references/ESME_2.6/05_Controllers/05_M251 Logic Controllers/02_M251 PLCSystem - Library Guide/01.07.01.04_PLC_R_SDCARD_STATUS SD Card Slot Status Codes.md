# PLC\_R\_SDCARD\_STATUS: SD Card Slot Status Codes

## Enumerated Type Description

The PLC\_R\_SDCARD\_STATUS enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| NO\_SDCARD | 0000 hex | No SD card detected in the slot or the slot is not connected. |
| SDCARD\_READONLY | 0001 hex | SD card is in read-only mode. |
| SDCARD\_READWRITE | 0002 hex | SD card is in read/write mode. |
| SDCARD\_ERROR | 0003 hex | Error detected in the SD card. More details are written to the file [FwLog.txt](../../../../../api/crossBook?lang=en-US&virtualBookName=m251prg&topicID=D_SE_0004156). |
