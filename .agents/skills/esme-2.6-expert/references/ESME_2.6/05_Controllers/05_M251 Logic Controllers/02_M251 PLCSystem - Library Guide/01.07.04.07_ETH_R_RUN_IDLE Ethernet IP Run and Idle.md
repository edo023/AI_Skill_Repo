# ETH\_R\_RUN\_IDLE: Ethernet/IP Run and Idle States Codes

## Enumerated Type Description

The ETH\_R\_RUN\_IDLE enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| IDLE | 00 hex | EtherNet/IP connection is idle. |
| RUN | 01 hex | EtherNet/IP connection is running. |
