# TM3\_BUS\_W\_IOBUSERRMOD: TM3 bus error mode

## Enumerated Type Description

The TM3\_BUS\_W\_IOBUSERRMOD enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| IOBUS\_ERR\_ACTIVE | 00 hex | Active mode. The logic controller stops all I/O exchanges on the TM3 bus on detection of a permanent error. Refer to [I/O Configuration General Description](../../../../../api/crossBook?lang=en-US&virtualBookName=m251prg&topicID=D_SE_0061767). |
| IOBUS\_ERR\_PASSIVE | 01 hex | Passive mode. I/O exchanges continue on the TM3 bus even if an error is detected. |
