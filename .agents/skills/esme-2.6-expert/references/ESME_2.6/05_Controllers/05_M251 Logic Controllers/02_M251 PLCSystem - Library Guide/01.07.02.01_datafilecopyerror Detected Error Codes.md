# DataFileCopyError: Detected Error Codes

## Enumerated Type Description

The DataFileCopyError enumeration data type contains the following values:

| Enumerator | Value | Description |
| --- | --- | --- |
| ERR\_NO\_ERR | 00 hex | No error detected. |
| ERR\_FILE\_NOT\_FOUND | 01 hex | The file does not exist. |
| ERR\_FILE\_ACCESS\_REFUSED | 02 hex | The file cannot be opened. |
| ERR\_INCORRECT\_SIZE | 03 hex | The request size is not the same as size read from file. |
| ERR\_CRC\_ERR | 04 hex | The CRC is not correct and the file is assumed to be corrupted. |
| ERR\_INCORRECT\_MAC | 05 hex | The controller attempting to read from the file does not have the same MAC address as that contained in the file. |
