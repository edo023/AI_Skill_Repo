# TM5 CANopen Fieldbus Interfaces

## Introduction

The TM5 fieldbus module is a CANopen interface with built-in power distribution and is the first TM5 distributed I/O island.

For more information, refer to the [Modicon TM5 CANopen Interface Hardware Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=tm5cnh1&topicID=D_SE_0009137).
