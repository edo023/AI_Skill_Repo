# Launching the Machine Assistant

## Overview

Machine Assistant is displayed similarly in EcoStruxure Machine Expert and on the controller Web Server. Using this tab, you can monitor the controller and its connected devices.

## Launching the Machine Assistant in the Web Server

Launch the [Web server](D-SE-0094003.html#D-SE-0094003__D-SE-0094003.3) and log in to access the home page of the [Web server site](D-SE-0002960.html#D-SE-0002960). Click the Machine Assistant tab. The Machine Assistant window is displayed.

## Launching the Machine Assistant in the Software

| Step | Action |
| --- | --- |
| 1 | In EcoStruxure Machine Expert, create a project with a M262 Logic/Motion Controller. |
| 2 | Double-click the Machine Assistant node in the Devices tree.  **Result:** The Machine Assistant window is displayed. |
