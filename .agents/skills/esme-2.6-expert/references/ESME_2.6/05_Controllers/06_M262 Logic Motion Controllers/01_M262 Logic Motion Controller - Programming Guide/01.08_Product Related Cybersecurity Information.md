# Product Related Cybersecurity Information

## Introduction

To help keep your Schneider Electric products secure and protected, implement the cybersecurity best practices as indicated in the [Cybersecurity Best Practices](https://www.se.com/ww/en/download/document/CS-Best-Practices-2019-340/) and [Cybersecurity Guidelines for EcoStruxure Machine Expert, Modicon and PacDrive Controllers and Associated Equipment](https://www.se.com/ww/en/download/document/EIO0000004242/) document provided on the Schneider Electric website.

## Certificate Management

By default, the following certificates are displayed in the controller Web server in [Maintenance: Certificates Submenu](MaintenancePage-6CE89C28.html#MaintenancePage-6CE89C28__D-SE-0002960.45):

* TM262-XX-OPCUA is used for OPC UA.
* TM262-XX is used for HTTP/FTP/WebVisualisation.
* TM262-XX @ AABBCCDDEEFF (nodename) is used for communication with EcoStruxure Machine Expert.

The TM262-XX-OPCUA and TM262-XX certificates can only be managed through the Web server under the [Maintenance: Certificates Submenu](MaintenancePage-6CE89C28.html#MaintenancePage-6CE89C28__D-SE-0002960.45). To reset or delete these certificates, the supported methods are:

* Executing a `Format` command using the SD card
* Executing a [Reset Origin](D-SE-0008848.html#D-SE-0008848__D-SE-0008848.6) command on the controller

NOTE: Deleting the TM262-XX-OPCUA or TM262-XX certificates through the [Security Screen](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0099371) does not reset, nor regenerate the certificates.

## Boot Applications Encryption and Signing

The Modicon M262 Logic/Motion Controller supports boot applications encryption and signing with certificates. For more details, refer to [Properties...](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0083921_25) in the EcoStruxure Machine Expert Menu Commands Online Help.

The Modicon M262 Logic/Motion Controller does not support simple encryption, nor encryption with license management.
