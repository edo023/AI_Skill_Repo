# CoDeSys Protocol

## Introduction

The CoDeSys protocol is a protocol providing a transparent access to your controllers and HMIs.

The CoDeSys protocol is used for data exchange:

* between the EcoStruxure Machine Expert software (PC) and the runtime systems (controller, HMI) configured by Vijeo-Designer.
* between controllers and integrated HMIs supporting CoDeSys protocol.

## CoDeSys Protocol Configuration

The maximum number of clients that can connect simultaneously to CoDeSys protocol is 5.

## CoDeSys Protocol Data Types

The following data types are supported by the CoDeSys protocol:

* BOOL
* BYTE
* DINT
* DWORD
* INT
* REAL
* SINT
* UDINT
* UINT
* USINT
* WORD

The maximum number of variables for a data type is 12000.
