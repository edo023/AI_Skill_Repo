# Enumerated Type Description

Enumerated Type Description

This enumeration describes the types of inputs which can be used for reference and capture on an encoder function block.

The ET\_ENC\_INPUT\_M262 enumeration data type contains the following values:

| Parameter Name | Value | Description |
| --- | --- | --- |
| ENC\_INPUT\_REF\_I0 | 0 | REF input on I0 for preset. |
| ENC\_INPUT\_CAP\_I1 | 1 | Capture input on I1. |
| ENC\_INPUT\_CAP\_I2 | 2 | Capture input on I2. |
| ENC\_INPUT\_CAP\_I3 | 3 | Capture input on I3. |
