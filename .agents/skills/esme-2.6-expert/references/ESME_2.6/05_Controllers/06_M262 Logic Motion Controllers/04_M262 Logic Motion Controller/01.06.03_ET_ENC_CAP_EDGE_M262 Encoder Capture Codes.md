# ET_ENC_CAP_EDGE_M262: Encoder Capture Codes

![](images/logo.png)

ET\_ENC\_CAP\_EDGE\_M262: Encoder Capture Codes

Enumerated Type Description

This enumeration describes the types of edges which can be used for reference and capture on an encoder function block.

The ET\_ENC\_CAP\_EDGE\_M262 enumeration data type contains the following values:

| Parameter Name | Value | Description |
| --- | --- | --- |
| ENC\_CAP\_EDGE\_RISING | 0 | Capture on input rising edge. |
| ENC\_CAP\_EDGE\_FALLING | 1 | Capture on input falling edge. |
| ENC\_CAP\_EDGE\_BOTH | 2 | Capture on input both edges. |
