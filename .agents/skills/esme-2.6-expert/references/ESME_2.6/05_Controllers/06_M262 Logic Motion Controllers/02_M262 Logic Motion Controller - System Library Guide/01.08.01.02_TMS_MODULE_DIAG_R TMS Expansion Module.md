# TMS\_MODULE\_DIAG\_R: TMS Expansion Module Diagnostic Error Codes

## Library and Namespace

Library name: **TMSSystem**

Namespace: **TMS**

## Introduction

The TMS\_MODULE\_DIAG\_R system variable (TMS\_MODULE\_R\_ARRAY\_TYPE type) is an array of STRUCT\_TMS\_MODULE\_DIAG.

## Variable Structure

This table describes the parameters of the STRUCT\_TMS\_MODULE\_DIAG:

| Enumerator | Type | Comment |
| --- | --- | --- |
| Name | STRING(15) | Name of the TMS expansion module. |
| MajorType | WORD | Type of TMS expansion module. |
| SubType | WORD | Sub-type of TMS expansion module. |
| Version | STRING(15) | Firmware version of the TMS expansion module. |
| ModuleState | [TMS\_MODULE\_STATE](D-SE-0086006.html#D-SE-0086006) | State of the TMS expansion module. |
| IpState | [TMS\_IP\_STATE](D-SE-0086005.html#D-SE-0086005) | TCP IP accessibility of the TMS expansion module from the controller over COM\_Bus. |
| PixCmdState | [TMS\_PIXCMD\_STATE](D-SE-0086007.html#D-SE-0086007) | State of the expansion module. |
