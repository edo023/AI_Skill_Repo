# TMS\_BUS\_DIAG\_R: TMS Bus Diagnostic Error Codes

## Library and Namespace

Library name: **TMSSystem**

Namespace: **TMS**

## Variable Structure

This table describes the parameters of the TMS\_BUS\_DIAG\_R system variable (STRUCT\_TMS\_BUS\_DIAG type):

| Enumerator | Type | Comment |
| --- | --- | --- |
| ConfState | UNIT | Describes the state of the TMS configuration:   * 0: No configuration * 1: Configuration not valid (module mismatch) * 2: Configuration valid, but power booster not active * 3: Configuration valid and applied |
| NbModules | UNIT | Indicates the number of modules detected on the bus. |
