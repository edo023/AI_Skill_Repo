# PLC\_R\_TM3\_BUS\_STATE: TM3 Bus Status Codes

## Library and Namespace

Library name: **PLCSystemBase**

Namespace: ****PLCSystemBase****

## Enumerated Type Description

The PLC\_R\_TM3\_BUS\_STATE enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| TM3\_CONF\_ERROR | 01 hex | Error detected due to mismatch in the physical configuration and the configuration in the software. |
| TM3\_BUS\_ERROR | 02 hex | Error detected in [q\_wIOBusErrPassiv](D-SE-0068487.html#D-SE-0068487) in TM3 System library. |
| TM3\_OK | 03 hex | The physical configuration and the configuration in the software match. |
| TM3\_POWER\_SUPPLY\_ERROR | 04 hex | Error detected in power supply. |
