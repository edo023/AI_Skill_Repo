# TMS\_MODULE\_STATE: TMS Expansion Module State Codes

## Library and Namespace

Library name: **TMSSystem**

Namespace: **TMS**

## Enumerated Type Description

The TMS\_MODULE\_STATE enumeration data type contains the following values:

| Name | Type | Value | Comment |
| --- | --- | --- | --- |
| TMS\_MODULE\_POWERED | DWORD | 0 | Module is powered up. |
| TMS\_MODULE\_INITIALIZED | DWORD | 1 | Module is initialized and has been discovered. |
| TMS\_MODULE\_CONFIGURED | DWORD | 2 | Module is configured and is operating normally. |
| TMS\_MODULE\_EXCHANGE\_FAULT | DWORD | 3 | Timeout for module detection expired. |
| TMS\_MODULE\_ERROR | DWORD | 4 | Module detected an error. |
| TMS\_MODULE\_HEALTH\_SEND\_FAULT | DWORD | 5 | Module health report send unsuccessful. |
| TMS\_MODULE\_RCV\_TIMEOUT | DWORD | 6 | Module reception timeout. |
| TMS\_MODULE\_RCV\_MISC | DWORD | 7 | Module reception error other than timeout. |
| TMS\_MODULE\_RESP\_ERR | DWORD | 8 | Module response error. |
| TMS\_MODULE\_DISCOVERY | DWORD | 9 | Module discovery error. |
