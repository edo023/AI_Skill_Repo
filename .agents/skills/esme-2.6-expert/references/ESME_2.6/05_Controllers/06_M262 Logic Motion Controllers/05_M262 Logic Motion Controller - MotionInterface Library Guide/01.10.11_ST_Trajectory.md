## Description

The structure ST\_Trajectory is used by the function [FC\_CalculateMoveVelocity](FC_CalculateMoveVelocity-41010DC4.html). It contains:

* Type of the motion profile (acceleration-based or jerk-based)
* Number of phases of the motion profile
* Reference to the array of calculated motion profile points in the structure [ST\_TrajectoryPoint](ST_TrajectoryPoint-4100368A.html#ST_TrajectoryPoint-4100368A)

The number of motion profile points corresponds to the number of phases plus one point.
