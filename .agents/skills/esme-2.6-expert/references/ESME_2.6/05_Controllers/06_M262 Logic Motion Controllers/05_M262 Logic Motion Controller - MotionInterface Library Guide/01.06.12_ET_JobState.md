# ET\_JobState

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.1.75.6318 |

## Description

This enumeration is used as an output of FB\_CustomJobBase (algorithm of a custom job) to describe its state.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Idle | 0 | Algorithm is not commanded to be executed. |
| Linked | 1 | Command triggered to execute algorithm, but not yet started. |
| Executing | 2 | Algorithm is executing a movement with a defined final condition. |
| Steady | 3 | Algorithm is executing a movement without defined final condition. |
| Done | 4 | Algorithm has finished execution. |
| Aborted | 5 | Algorithm was replaced by another algorithm before final condition was reached. |
| Error | 6 | Execution of algorithm was superseded by an error response. |
| ScheduledToBeAborted | 7 | Command triggered for a different algorithm to abort this algorithm, but yet not effective. |
| ScheduledToBeDone | 8 | Algorithm has reached the final condition in this RealTimeCycle and will be replaced in the next cycle. |
