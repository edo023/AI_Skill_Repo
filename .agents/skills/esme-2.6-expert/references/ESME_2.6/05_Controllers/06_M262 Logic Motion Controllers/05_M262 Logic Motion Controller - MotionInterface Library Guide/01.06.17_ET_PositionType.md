# ET\_PositionType

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V2.35.3.0 |

## Description

This enumeration, which is an alias of MC\_PositionType of the *M262 Synchronized Motion Control* library, specifies the type of position to be captured with the timestamp touchprobe functionality provided by the function block MC\_TouchProbe. Refer to the *M262 Synchronized Motion Control* library guide for details.

## Enumeration Elements

| Name | Value (INT) | Description |
| --- | --- | --- |
| ActualPosition | 0 | The values of the outputs of the function blocks MC\_ReadActualPosition and MC\_ReadActualVelocity are used in the calculation of the captured position.  If these function blocks do not provide values, for example, because the specified axis is a virtual axis or its working mode is Simulated, the values of RefPosition and RefVelocity are used. |
| ReferencePosition | 1 | The axis properties RefPosition and RefVelocity are used in the calculation of the captured position. |
