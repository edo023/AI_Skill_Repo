# ET\_HomingMode

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V2.21.2.0 |

## Description

This enumeration specifies the direction of movement for the movement at LowVelocity during homing with the function block FB\_HomingDigitalInput. Refer to the *M262 Synchronized Motion Control Library Guide* for details on the function block FB\_HomingDigitalInput.

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| FastReverseSlowSetpositionStop | 0 | The direction of movement for the movement at LowVelocity is reversed after the movement at HighVelocity is completed. |
| FastNoReverseSlowSetpositionStop | 1 | The direction of movement for the movement at LowVelocity is not reversed after the movement at HighVelocity is completed. |
