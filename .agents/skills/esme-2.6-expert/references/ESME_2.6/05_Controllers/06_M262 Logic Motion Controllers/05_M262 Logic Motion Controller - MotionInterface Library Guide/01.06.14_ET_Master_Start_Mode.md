# ET\_Master\_Start\_Mode

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.1.75.6318 |

## Description

This enumeration contains the input values for the input MasterStartMode of the function block MC\_CamIn. It selects if the absolute master position is taken for cam calculation or if it is ignored and assumed to be the described master position at cam start.

In the first cycle when the cam is started, it specifies how the value of the master (X value) as seen by the subordinate axis of the cam is determined, based on the position of the master axis.

## Enumeration Elements

| Name | Value (INT) | Description |
| --- | --- | --- |
| Absolute | 0 | The cam starts at the X coordinate equal to the absolute master axis position at the time the cam gets active. If the cam is buffered after another cam with the same master, the master as seen by the subordinate axis of the first cam is used instead.  Behavior with the function block FB\_LogicalEncoder: If the logical encoder has not been used in the cam context, the present position of the parent device is set as the logical encoder position. If the logical encoder has been used in the cam context before (for example, in a sequence of cams), the position value of the logical encoder remains unmodified. |
| Relative | 1 | The cam starts at the X coordinate equal to the lowest X coordinate of the cam points. The resulting offset between the position of the master axis and the master as seen by the subordinate axis is kept during the movement of the cam.  Behavior with the function block FB\_LogicalEncoder: The position of the logical encoder is set to the entry point of the cam. If the cam moves backwards, the logical encoder is set to the upper  X limit of the cam, otherwise to the lower. |
| KeepEncoderPosition | 2 | This start mode is only available for logical encoder axes. Refer to [FB\_LogicalEncoder](FB_LogicalEncoder-3FC5FB2B.html) for details.  The cam starts from the position of the logical encoder.  If you attempt to use this start mode for any type of master axis other than a logical encoder axis, an error is detected (MasterStartModeOnlySupportedWithLogicalEncoder). |
