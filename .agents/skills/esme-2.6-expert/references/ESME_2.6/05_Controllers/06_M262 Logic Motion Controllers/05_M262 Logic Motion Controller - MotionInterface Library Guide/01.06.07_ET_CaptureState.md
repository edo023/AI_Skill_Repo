# ET\_CaptureState

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V2.35.3.0 |

## Description

This enumeration lists the possible states of position capture with a touchprobe using the function block MC\_Touchprobe.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Inactive | 0 | The conditions for starting position capture are met, but position capture is not active. |
| Active | 1 | Position capture is in progress. |
| Captured | 2 | A position has been captured and the new captured position value is available. |
| Overflow | 3 | Reserved. |
| Disabled | 4 | Reserved. |
| NotReady | 5 | Position capture is not running and cannot be started, for example, because Sercos is not in communication phase CP 4. |
| Virtual | 6 | Position capture has been started with the timestamp touch probe functionality with a Modicon Edge I/O NTS module whose bus coupler in the in the working mode Virtual. In this case, no physical positions are captured and the state Virtual is not reached. Aborting the function block MC\_TouchProbe transitions to the state Inactive of this enumeration. |
