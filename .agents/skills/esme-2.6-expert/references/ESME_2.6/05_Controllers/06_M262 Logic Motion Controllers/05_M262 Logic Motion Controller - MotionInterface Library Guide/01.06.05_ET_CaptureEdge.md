# ET\_CaptureEdge

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.1.75.6318 |

## Description

This enumeration is an option for IF\_Trigger of MC\_TouchProbe. It allows you to read the input edge for position capture set with [ET\_CaptureEdgeSet](ET_CaptureEdgeSet-EC27EB54.html#ET_CaptureEdgeSet-EC27EB54).

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| FallingEdge | 0 | MC\_TouchProbe uses a falling edge of the selected input for position capture. |
| RisingEdge | 1 | MC\_TouchProbe uses a rising edge of the selected input for position capture. |
| BothEdges | 2 | MC\_TouchProbe uses both falling and rising edges of the selected input for position capture. |
