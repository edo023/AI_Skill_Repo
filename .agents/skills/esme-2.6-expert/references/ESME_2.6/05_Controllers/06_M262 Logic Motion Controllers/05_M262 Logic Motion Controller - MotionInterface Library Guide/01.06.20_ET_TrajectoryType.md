# ET\_TrajectoryType

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V2.30.0.0 |

## Description

This enumeration specifies the type of motion profile to be calculated with the function [FC\_CalculateMoveVelocity](FC_CalculateMoveVelocity-41010DC4.html). The motion profile can be acceleration-based or jerk- based

## Enumeration Elements

| Name | Value (INT) | Description |
| --- | --- | --- |
| JerkBased | 0 | The motion profile is jerk-based. |
| AccelerationBased | 1 | The motion profile is acceleration-based. |
