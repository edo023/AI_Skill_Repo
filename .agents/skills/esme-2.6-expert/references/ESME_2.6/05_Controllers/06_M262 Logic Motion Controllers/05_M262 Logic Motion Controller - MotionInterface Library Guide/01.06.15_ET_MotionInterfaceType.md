# ET\_MotionInterfaceType

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.1.75.6318 |

## Description

This enumeration specifies the type of an axis. It is used as type of the property etInterfaceType of every axis.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Feedback | 0 | External driven axis such as an encoder axis which cannot execute movement commands |
| Coordinate | 1 | Axis with feedback and movement commands driven by drives |
