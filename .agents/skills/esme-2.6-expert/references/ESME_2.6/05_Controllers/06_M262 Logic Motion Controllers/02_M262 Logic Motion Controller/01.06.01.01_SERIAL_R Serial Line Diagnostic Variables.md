# SERIAL\_R: Serial Line Diagnostic Variables

## Library and Namespace

Library name: **SerialLineSystem**

Namespace: **SEC\_SLSYS**

## Introduction

The SERIAL\_R system variable (SERIAL\_R\_ARRAY\_TYPE type) is an array of SERIAL\_R\_STRUCT.

## Variable Structure

This table describes the parameters of the SERIAL\_R\_STRUCT:

| Variable Name | Type | Comment |
| --- | --- | --- |
| i\_udiFramesTransmittedOK | UDINT | Indicates the number of frames transmitted since start by Serial Line bus. |
| i\_udiFramesReceivedOK | UDINT | Indicates the number of frames received since start by Serial Line bus. |
| i\_udiRX\_MessagesError | UDINT | Indicates that the RX Error Counter register counts the number of frames received with errors detected for all types of frames. |
| i\_uiSlaveExceptionCount | UINT | Indicates the number of bus exception errors detected. |
| i\_uiSlaveMsgCount | UINT | Indicates the number of slave messages. |
| i\_uiSlaveNoRespCount | UINT | Indicates the number of no responses from slaves. |
| i\_uiSlaveNakCount | UINT | Indicates the number of slave NAK indications. |
| i\_uiSlaveBusyCount | UINT | Indicates the number of slave busy indications. |
| i\_uiCharOverrunCount | UINT | Indicates the number of character overruns encountered. |
