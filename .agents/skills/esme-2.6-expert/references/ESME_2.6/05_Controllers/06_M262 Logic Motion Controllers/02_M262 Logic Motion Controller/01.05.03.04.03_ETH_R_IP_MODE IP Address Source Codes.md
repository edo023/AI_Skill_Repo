# ETH\_R\_IP\_MODE: IP Address Source Codes

## Library and Namespace

Library name: **PLCSystemBase**

Namespace: ****PLCSystemBase****

## Enumerated Type Description

The ETH\_R\_IP\_MODE enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| ETH\_R\_STORED | 00 hex | Stored IP address is used. |
| ETH\_R\_BOOTP | 01 hex | Bootstrap protocol (BOOTP) is used to get an IP address. |
| ETH\_R\_DHCP | 02 hex | DHCP protocol is used to get an IP address. |
| ETH\_DEFAULT\_IP | FF hex | Default IP address is used. |
