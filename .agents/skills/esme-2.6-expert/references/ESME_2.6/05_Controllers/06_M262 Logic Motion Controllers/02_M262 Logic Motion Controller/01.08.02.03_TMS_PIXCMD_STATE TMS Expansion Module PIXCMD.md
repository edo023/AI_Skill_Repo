# TMS\_PIXCMD\_STATE: TMS Expansion Module PIXCMD State

## Library and Namespace

Library name: **TMSSystem**

Namespace: **TMS**

## Description

The TMS\_PIXCMD\_STATE enumeration data type contains the following values:

| Name | Type | Value | Comment |
| --- | --- | --- | --- |
| TMS\_PIXCMD\_EXCHING | DWORD | 0 | Exchange in progress. |
| TMS\_PIXCMD\_CONFIG\_NONE | DWORD | 1 | No configuration. |
| TMS\_PIXCMD\_CONFIG\_CMD\_ERROR | DWORD | 2 | Configuration send unsuccessful. |
| TMS\_PIXCMD\_CONFIG\_RESP\_WAIT | DWORD | 3 | Waiting for configuration response (transient). |
| TMS\_PIXCMD\_CONFIG\_RESP\_ERROR | DWORD | 4 | Configuration response error. |
| TMS\_PIXCMD\_CONFIG\_ONLY | DWORD | 5 | Configuration successful, no exchange occurred. |
| TMS\_PIXCMD\_CONFIG\_SUCCESS | DWORD | 6 | Configuration successful (transient). |
| TMS\_PIXCMD\_ENABLE\_CMD\_ERROR | DWORD | 7 | Enable command error. |
| TMS\_PIXCMD\_ENABLE\_RESP\_WAIT | DWORD | 8 | Waiting for enable command response (transient). |
| TMS\_PIXCMD\_ENABLE\_RESP\_ERROR | DWORD | 9 | Enable command response error. |
| TMS\_PIXCMD\_EXCH\_ERROR | DWORD | 10 | Exchange error. |
| TMS\_PIXCMD\_DISABLING | DWORD | 11 | Exchange being disabled (transient). |
| TMS\_PIXCMD\_DISABLED | DWORD | 12 | Exchange disabled. |
