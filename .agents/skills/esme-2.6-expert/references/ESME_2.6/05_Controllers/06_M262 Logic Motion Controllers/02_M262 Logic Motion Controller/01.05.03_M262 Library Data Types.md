# M262 Library Data Types

## Overview

This chapter describes the data types of the M262 PLCSystem library.

There are 2 kinds of data types available:

* System variable data types are used by the [system variables](D-SE-0003286.html#D-SE-0003286) of the M262 PLCSystem Library (PLC\_R, PLC\_W,...).
* System function data types are used by the read/write [system functions](D-SE-0002756.html#D-SE-0002756) of the M262 PLCSystem Library.
