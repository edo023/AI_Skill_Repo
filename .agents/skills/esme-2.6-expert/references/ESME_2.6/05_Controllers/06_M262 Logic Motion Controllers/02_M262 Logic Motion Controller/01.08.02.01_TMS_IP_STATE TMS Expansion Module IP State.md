# TMS\_IP\_STATE: TMS Expansion Module IP State

## Library and Namespace

Library name: **TMSSystem**

Namespace: **TMS**

## Description

The TMS\_IP\_STATE enumeration data type contains the following values:

| Name | Type | Value | Comment |
| --- | --- | --- | --- |
| TMS\_IP\_PING\_SUCCESS | DWORD | 0 | IP interface is configured. |
| TMS\_IP\_CONFIG\_CMD\_ERROR | DWORD | 1 | Configuration send unsuccessful. |
| TMS\_IP\_CONFIG\_RESP\_WAIT | DWORD | 2 | Waiting for configuration response (transient). |
| TMS\_IP\_CONFIG\_RESP\_ERROR | DWORD | 3 | Configuration response error. |
| TMS\_IP\_CONFIG\_RESP\_NONE | DWORD | 4 | No configuration response. |
| TMS\_IP\_CONFIG\_SUCCESS | DWORD | 5 | Configuration successful. |
| TMS\_IP\_PING\_CMD\_ERROR | DWORD | 6 | Ping command send unsuccessful. |
| TMS\_IP\_PING\_RESP\_WAIT | DWORD | 7 | Waiting for ping command response (transient). |
| TMS\_IP\_PING\_RESP\_ERROR | DWORD | 8 | Ping command response error. |
| TMS\_IP\_PING\_RESP\_NONE | DWORD | 9 | No ping command response. |
| TMS\_IP\_NOT\_CONFIGURED | DWORD | 10 | IP interface not configured. |
