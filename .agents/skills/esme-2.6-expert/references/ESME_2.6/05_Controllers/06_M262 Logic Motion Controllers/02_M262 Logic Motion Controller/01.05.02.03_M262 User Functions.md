# M262 User Functions

## Overview

This section describes the following user functions:

| Function | Description |
| --- | --- |
| FB\_GetFreeDiskSpace | Retrieves the amount of free memory space of a memory medium in bytes. See [FB\_GetFreeDiskSpace: Gets the Free Memory Space Asynchronously](FC_GetFreeDiskSpaceGetsTheFreeMemor-0878CA35.html) |
| FB\_GetLabel | Retrieves the label of a memory medium. See [FB\_GetLabel: Gets the Label of Memory Medium](FC_GetLabelGetsTheLabelOfMemory-0879D42C.html) |
| FB\_GetTotalDiskSpace | Retrieves the size of free memory space of a memory medium in bytes. See [FB\_GetTotalDiskSpace: Gets the Size of Memory Medium](FC_GetTotalDiskSpaceGetsTheSizeOfMe-087A0A4E.html) |
| FB\_CheckAllowedControllerMacAddr | Check whether a specific MAC address is within the valid range for the controller. See [FB\_CheckAllowedControllerMacAddr: Check If MAC Address Allowed by Controller](D-SE-0085972.html) |
| FB\_ControlClone | Enable or disable the controller cloning functionality. See [FB\_ControlClone: Clone the Controller](D-SE-0085973.html) |
| DataFileCopy | Copy memory data to a file and vice versa. See [DataFileCopy: Copy File Commands](D-SE-0020619.html) |
| ExecuteScript | Run script commands. See [ExecuteScript: Run Script Commands](D-SE-0020618.html) |
