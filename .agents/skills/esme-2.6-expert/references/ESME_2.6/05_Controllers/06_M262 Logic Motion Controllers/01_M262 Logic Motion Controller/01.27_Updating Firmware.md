# Updating Firmware

## Introduction

Updating the controller firmware is possible by using:

* An SD card with a compatible script file.
* The Controller Assistant.

Updating the TM3 and the TMS firmware is possible by using an SD card with a compatible script file.

Performing a firmware update deletes the application program in the device, including the configuration files, the user management, the user rights, optionally the certificates (depending on the update method) and the Boot Application in non-volatile memory.

For more information about the firmware update and creating a new flash disk with firmware, refer to Project Settings - Firmware Update and [Non-Volatile Memory Organization](D-SE-0004156.html#D-SE-0004156).
