# TM3 Expansion Module Configuration

## Introduction

The Modicon M262 Logic/Motion Controller supports the following expansion modules:

* TM3 expansion modules:

  + Digital I/O modules
  + Analog I/O modules
  + Expert I/O modules
  + Safety modules
  + Transmitter and receiver modules

For further information about the TM3 expansion modules configuration, refer to the [TM3 Expansion Modules Configuration Programming Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=tm3ioprg&topicID=D_SE_0032099).

## Adding an Expansion Module

To add an expansion module to your controller, select the expansion module in the Hardware Catalog, drag it to the Devices tree, and drop it on one of the highlighted nodes.

For more information on adding a device to your project, refer to:

• Using the [Hardware Catalog](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0083368)
