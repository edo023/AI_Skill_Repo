# FC\_EtJobStateToString

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V1.1.75.6318 |

## Task

Converts an enumeration element of type ET\_JobState to a variable of type STRING.

## Description

Using the function FC\_EtJobStateToString, you can convert an enumeration element of type ET\_JobState to a variable of type STRING.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etResult | ET\_JobState | Enumeration with the job state. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If an error is detected in the execution of the function block, this output is set to TRUE. Refer to ET\_Result. |
| q\_etResult | ET\_Result | Result of the function block execution. Refer to the [ET\_Result Enumeration Elements](D-SE-0096918.html). |

## Return Value

| Data type | Description |
| --- | --- |
| STRING(80) | The ET\_JobState converted to text. |
