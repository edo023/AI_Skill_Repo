## Task

Monitor the movements of the connected axis.

## Description

This function block monitors movements of a connected axis. The function block provides information on the physical position. Values resulting from the function block MC\_SetPosition and from a modulo configuration of an axis are not taken into account.

The function block is controlled via its methods and provides the corresponding values via its properties.

Connecting an axis to the function block via its method Connect initializes the properties of the function block with the axis values.

The connection to the axis is maintained while the master is being homed. However, when homing is started, the function block no longer tracks the axis values. The position value is frozen. The velocity and acceleration values are set to zero. The value of ET\_Result is set to AxisIsHoming as long as homing is running. Once homing is completed, the function block resumes tracking the values of the master.
