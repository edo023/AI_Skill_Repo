# ET\_JogType

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V2.30.0.0 |

## Description

This enumeration specifies the type of jog for the function block MC\_Jog. For details on the function block MC\_Jog, refer to the *M262 Synchronized Motion Control Library Guide*.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| Velocity | 0 | MC\_Jog performs a jog movement with the velocity specified for the input Velocity (similar to MC\_MoveVelocity). |
| PositionOffset | 1 | MC\_Jog performs a jog movement from the present position by a distance specified at the input Position (similar to MC\_MoveRelative). |
| Position | 2 | MC\_Jog performs a movement to the position specified for the input Position (similar to MC\_MoveAbsolute). |
