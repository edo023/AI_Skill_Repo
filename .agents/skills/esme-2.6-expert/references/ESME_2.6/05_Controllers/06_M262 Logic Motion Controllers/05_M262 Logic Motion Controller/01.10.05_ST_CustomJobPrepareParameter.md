# ST\_CustomJobPrepareParameter

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.1.75.6318 |
| Inherits from: | — |

## Description

This structure is used as input for the Prepare method of MC\_CustomJobBase. This structure initializes the movement to be performed via MC\_CustomJobBase by using the axis movement values of its axis before starting.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| lrSlavePositionLastCycle | LREAL | Last position of axis executing MC\_CustomJob (the position is set during Calculate in this cycle). |
| lrSlaveVelocityLastCycle | LREAL | Last velocity of axis executing MC\_CustomJob. |
| lrSlaveAccelerationLastCycle | LREAL | Last acceleration of axis executing MC\_CustomJob. |
