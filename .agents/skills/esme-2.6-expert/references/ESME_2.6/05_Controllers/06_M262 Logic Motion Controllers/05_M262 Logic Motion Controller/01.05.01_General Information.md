# General Information

## Description

The library MotionInterface comprises the IF\_Axis type used as input for PLCopen function blocks. It includes enumerations and records (data structures) for functions and states of axes.

The library contains the function block FB\_ControlledAxis that can be used as virtual axes for applications.

The library includes the function block FB\_CustomJobBase that user code has to derive from in order to implement a custom job algorithm.
