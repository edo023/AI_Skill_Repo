# ET\_Direction

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.1.75.6318 |

## Description

This enumeration describes the direction of movement for an MC\_MoveAbsolute with a modulo axis.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| PositiveDirection | 0 | Positive direction of movement. |
| NegativeDirection | 1 | Negative direction of movement. |
| ShortestWay | 2 | The direction of movement depends on whether the positive direction of movement or the negative direction of movement has the shortest distance to the target position. |
