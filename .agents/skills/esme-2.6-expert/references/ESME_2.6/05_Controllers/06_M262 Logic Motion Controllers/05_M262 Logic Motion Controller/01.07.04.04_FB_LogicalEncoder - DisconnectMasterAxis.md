# FB\_LogicalEncoder - DisconnectMasterAxis (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V2.30.0.0 |

## Description

This method of the function block [FB\_LogicalEncoder](FB_LogicalEncoder-3FC5FB2B.html) lets you disconnect a logical encoder from its parent axis. The parent axis is the axis that corresponds to the parent device of the logical encoder. Drives or physical encoders can be used as parent devices.
