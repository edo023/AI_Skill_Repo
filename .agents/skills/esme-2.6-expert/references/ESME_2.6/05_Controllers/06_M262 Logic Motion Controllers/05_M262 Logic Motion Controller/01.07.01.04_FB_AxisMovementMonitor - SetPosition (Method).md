# FB\_AxisMovementMonitor - SetPosition (Method)

## Overview

|  |  |
| --- | --- |
| Type: | Method |
| Available as of: | V2.8.0.0 |

## Task

Performs an absolute or relative movement via the function block FB\_AxisMovementMonitor.

## Description

This method performs an absolute or relative movement to the specified position via the function block FB\_AxisMovementMonitor.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_lrPosition | LREAL | Target position of movement. |
| i\_xRelative | BOOL | If TRUE, a relative movement is performed. If FALSE, an absolute movement is performed. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | If an error is detected in the execution of the function block, this output is set to TRUE. Refer to ET\_Result. |
| q\_etResult | ET\_Result | Result of the function block execution. Refer to the [ET\_Result Enumeration Elements](D-SE-0096918.html). |
