# FC\_GetCamSlaveMovementFromGivenMasterForMultiCam

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V2.14.3.0 |

## Task

Determines the position of the subordinate axis on the basis of the position of the master axis when a cam is being executed.

## Description

Using the function FC\_GetCamSlaveMovementFromGivenMasterForMultiCam, you can determine the position of the subordinate axis on the basis of the position of the master axis. The function works with interpolated cams defined with the structure ST\_MultiCam and executed by the function block MC\_CamIn. Refer to the *CommonMotionTypes Library Guide* for details on the structure ST\_MultiCam. Refer to the *M262 Synchronized Motion Control Library Guide* for details on the function block MC\_CamIn.

The function assists you in recovering the axis position after an interruption or a stop of a movement resulting from a detected error. The function calculates the target position, velocity and acceleration of a subordinate axis at the point in time of executing the function if this axis is coupled to the movement of a master axis with a cam. The subordinate axis is not moved or otherwise affected. The function can only be called once to determine the start conditions for the subordinate axis so it does not ramp in. The function cannot be used cyclically to read the subordinate axis values on an ongoing basis.

The function can be used with the cam types of the enumeration ET\_CamType of the CommonMotionTypes library, with the exception of UserCam. If you use the function with the cam type UserCam, an error is detected (InvalidCamTableID).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| Master | IF\_Axis | Reference to the axis for which the function is to be executed. Refer to the interface [IF\_Axis](D-SE-0096932.html) for details. |
| Slave | IF\_Axis | Reference to the axis for which the function is to be executed. Refer to the interface [IF\_Axis](D-SE-0096932.html) for details. |
| MasterScaling | LREAL | The MasterScaling factor is used to calculate the position of the master as seen by the subordinate axis by multiplying the master position (in absolute start mode), or the master position offset (in relative start mode). |
| SlaveScaling | LREAL | The SlaveScaling factor is applied by multiplying the subordinate axis position obtained from the cam (in absolute start mode), or the subordinate axis position offset (in relative start mode). |
| CamTableID | CMT.ST\_MultiCam | Structure ST\_MultiCam of the *CommonMotionTypes* Library. |

| Output | Data type | Description |
| --- | --- | --- |
| Position | LREAL | Position of the subordinate axis. |
| Velocity | LREAL | Velocity of the subordinate axis. |
| Acceleration | LREAL | Acceleration of the subordinate axis. |
