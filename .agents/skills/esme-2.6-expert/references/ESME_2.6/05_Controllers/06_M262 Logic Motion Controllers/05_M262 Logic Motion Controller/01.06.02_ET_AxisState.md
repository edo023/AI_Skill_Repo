# ET\_AxisState

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.1.75.6318 |

## Description

This enumeration describes the axis states according to the PLCopen state machine.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| ErrorStop | 0 | The axis has stopped in response to a detected error. The transition to this state results from a detected axis-related or drive-related error. The axis remains in this state as long as the detected error persists. A transition from this state to either Standstill or Disabled requires a reset with MC\_Reset. |
| Disabled | 1 | The axis is disabled. |
| Standstill | 2 | The axis is not moving. |
| Stopping | 3 | The axis is stopping or has stopped. |
| Homing | 4 | The axis is being homed. |
| DiscreteMotion | 5 | The axis executes a movement for a limited period of time. |
| ContinuousMotion | 6 | The axis executes a movement for an unlimited period of time. |
| SynchronizedMotion | 7 | The axis executes a movement synchronously with a master. |
