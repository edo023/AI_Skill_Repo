# FC\_CamBounds

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V2.21.2.0 |

## Task

Calculates the position, velocity and acceleration ranges of a subordinate axis coupled to a master in absolute mode.

## Description

This function calculates the position, velocity and acceleration ranges of a subordinate axis coupled to a master in absolute mode. The values depend on:

* The y position, slope and curvature of the cam type used
* The velocity and acceleration of the master
* The scaling of the master and the scaling of the output movement

The function can be used with the cam types of the enumeration ET\_CamType of the CommonMotionTypes library, with the exception of UserCam. If you use the function with the cam type UserCam, an error is detected (InvalidCamTableID).

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stCamTableID | CMT.ST\_MultiCam | Structure ST\_MultiCam of the *CommonMotionTypes* library. |
| i\_lrMasterVelMax | LREAL | Value range: Positive LREAL value  Default value: 1.0  Absolute maximum velocity of the master in user-defined units. |
| i\_lrMasterAccMax | LREAL | Value range: Positive LREAL value  Absolute maximum acceleration/deceleration of the master in user-defined units. |
| i\_lrMasterScaling | LREAL | Value range: Positive LREAL value or zero  Default value: 1.0  Scaling factor for master applied to cam. |

| Output | Data type | Description |

| q\_lrMaxPosition | LREAL | Maximum subordinate axis position value in user-defined units. |
| q\_lrMinPosition | LREAL | Minimum subordinate axis position in user-defined units. |
| q\_lrMaxVelocity | LREAL | Maximum subordinate axis velocity value in user-defined units. |
| q\_lrMinVelocity | LREAL | Minimum subordinate axis velocity value in user-defined units. |
| q\_lrMaxAccDeceleration | LREAL | Maximum subordinate axis acceleration/deceleration value in user-defined units. |
| q\_lrMinAccDeceleration | LREAL | Minimum subordinate axis acceleration/deceleration value in user-defined units. |
