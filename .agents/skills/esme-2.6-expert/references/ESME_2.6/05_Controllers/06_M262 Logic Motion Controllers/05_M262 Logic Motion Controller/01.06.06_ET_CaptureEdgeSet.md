# ET\_CaptureEdgeSet

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V2.35.3.0 |

## Description

This enumeration is an option for IF\_Trigger of MC\_TouchProbe. It allows you to set the input edge to be used for activating position capture.

## Enumeration Elements

| Name | Value (UDINT) | Description |
| --- | --- | --- |
| FallingEdge | 0 | MC\_TouchProbe activates position capture with a falling edge of the selected input. |
| RisingEdge | 1 | MC\_TouchProbe activates position capture with a rising edge of the selected input. |
| BothEdges | 2 | MC\_TouchProbe activates position capture with both falling and rising edges of the selected input. |
