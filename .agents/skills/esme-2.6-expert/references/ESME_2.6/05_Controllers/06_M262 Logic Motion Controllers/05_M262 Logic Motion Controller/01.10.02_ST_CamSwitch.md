# ST\_CamSwitch

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V2.14.3.0 |
| Inherits from: | - |

## Description

This structure represents a switching event used with the function block MC\_DigitalCamSwitch. Refer to the *M262 Synchronized Motion Control Library Guide* for details on the function block MC\_DigitalCamSwitch.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| TrackNumber | BYTE | Specifies the number of the track, that is, the output. The maximum value is 32. |
| Position | REAL | Specifies the position in user-defined units of the track at which the switching event is to be triggered. |
| AxisDirection | ET\_AxisDirection | Enumeration specifying the direction of movement in which the switching event is to be triggered. Refer to [ET\_AxisDirection](ET_AxisDirection-DDB8321B.html) for the enumeration elements. |
| CamSwitchMode | ET\_CamSwitchMode | Enumeration specifying the type of switching for the switching event to be triggered. Refer to [ET\_CamSwitchMode](MOIN.ET_CamSwitchMode-DDBCB5FF.html) for the enumeration elements. |
| Duration | TIME | Specifies the period of time for which the output is set to ON if TimeBased has been selected from the enumeration [ET\_CamSwitchMode](MOIN.ET_CamSwitchMode-DDBCB5FF.html). |
