# ST\_CamSwitch\_Ref

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V2.14.3.0 |
| Inherits from: | - |

## Description

This structure lets you set the number of switching events and a pointer to an array of switching events used with the function block MC\_DigitalCamSwitch. Refer to the *M262 Synchronized Motion Control Library Guide* for details on the function block MC\_DigitalCamSwitch.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| NumberOfSwitches | BYTE | Specifies the number of switching events. The maximum number of switching events is 255. |
| CamSwitch\_Ref | POINTER TO ST\_CamSwitch | Pointer to the structure [ST\_CamSwitch](MOIN.ST_CamSwitch-DDC09018.html). |
