# FB\_LogicalEncoder - ConnectSuperimposedAxis (Method)

## Description

This method of the function block [FB\_LogicalEncoder](FB_LogicalEncoder-3FC5FB2B.html) lets you connect an axis whose movements are superimposed on the encoder axis.
