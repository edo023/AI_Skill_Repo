# ET\_OperationModeStatus - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.0.108.9437 |

## Description

This enumeration contains the active operating mode of the drive.

## Enumeration Elements

| Name | Value (DINT) | Description |
| --- | --- | --- |
| Position | 0 | The active operating mode of the drive is Cyclic Synchronous Position. |
| Velocity | 1 | The active operating mode of the drive is Cyclic Synchronous Velocity. |
| Torque | 2 | The active operating mode of the drive is Cyclic Synchronous Torque. |
| NotAvailable | 3 | Possible reasons:   * The active operating mode is not a cyclic synchronous operating mode (for example Homing). * The drive has detected an error. * The drive is disabled. * The drive is not in Sercos communication phase 4. |
