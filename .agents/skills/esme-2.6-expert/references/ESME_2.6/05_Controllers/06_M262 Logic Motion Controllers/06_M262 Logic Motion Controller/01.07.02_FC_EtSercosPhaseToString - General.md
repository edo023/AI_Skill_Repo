# FC\_EtSercosPhaseToString - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V2.0.108.9437 |

## Task

Converts an enumeration element of type ET\_SercosPhase to a variable of type STRING.

## Functional Description

Using the function FC\_EtSercosPhaseToString, you can convert an enumeration element of type ET\_SercosPhase to a variable of type STRING.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etSercosPhase | ET\_SercosPhase | Enumeration with the Sercos communication phase to be converted. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | Output is set to TRUE if an error is detected during the conversion. |
| q\_etResult | ET\_Result | Output is set to ET\_Result.InvalidSercosPhase if an error is detected during the conversion. |

## Return Value

| Data type | Description |
| --- | --- |
| STRING[80] | The ET\_SercosPhase value converted to text. |
