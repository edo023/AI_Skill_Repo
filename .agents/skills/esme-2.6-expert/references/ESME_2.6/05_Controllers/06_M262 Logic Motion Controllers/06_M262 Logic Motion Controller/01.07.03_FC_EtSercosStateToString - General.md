# FC\_EtSercosStateToString - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V2.0.108.9437 |

## Task

Converts an enumeration element of type ET\_SercosState to a variable of type STRING.

## Functional Description

Using the function FC\_EtSercosStateToString, you can convert an enumeration element of type ET\_SercosState to a variable of type STRING.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etSercosState | ET\_SercosState | Enumeration with the Sercos state to be converted. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | Output is set to TRUE if an error is detected during the conversion. |
| q\_etResult | ET\_Result | Output is set to ET\_Result.InvalidSercosState if an error is detected during the conversion. |

## Return Value

| Data type | Description |
| --- | --- |
| STRING[80] | The ET\_SercosState value converted to text. |
