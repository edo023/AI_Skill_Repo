# ET\_IpConfigMode - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.0.108.9437 |

## Description

This enumeration contains the options for setting the IP Address of a Sercos device. The IP address is configured either manually or automatically via the firmware.

## Enumeration Elements

| Name | Value (DINT) | Description |
| --- | --- | --- |
| AutomaticIpAddressAssignment | 0 | The IP address of the device is configured automatically via the firmware during phase-up. |
| ManualIpAddressAssignmentHardware | 1 | IP address of the device is configured manually. |
