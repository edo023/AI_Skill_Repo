# FC\_GetScaledFeedbackVelocity

## Overview

|  |  |
| --- | --- |
| Type: | Function |
| Available as of: | V2.0.108.9437 |

## Task

Returns the scaled velocity of a drive.

## Functional Description

The function returns the velocity of the drive scaled in user units. The value is not modified by a setPos

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_stSlave | ST\_Slave | Device handle of the device whose scaled velocity is requested. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_lrVelocity | LREAL | Scaled velocity of the device. |

## Return Value

| Data type | Description |
| --- | --- |
| DINT | Number that indicates the result of the function call (see DeviceApiResult). |
