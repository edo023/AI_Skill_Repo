## Task

Converts an enumeration element of type ET\_Result to a variable of type STRING.

## Functional Description

Using the function FC\_EtResultToString, you can convert an enumeration element of type ET\_Result to a variable of type STRING.

## Interface

| Input | Data type | Description |
| --- | --- | --- |
| i\_etResult | ET\_Result | Enumeration with the result. |

| Output | Data type | Description |
| --- | --- | --- |
| q\_xError | BOOL | Output is set to TRUE if an error is detected during the conversion. |
| q\_etResult | ET\_Result | Output is set to InvalidResult if an error is detected during the conversion. |

## Return Value

| Data type | Description |
| --- | --- |
| STRING(80) | The ET\_Result value converted to text. |
