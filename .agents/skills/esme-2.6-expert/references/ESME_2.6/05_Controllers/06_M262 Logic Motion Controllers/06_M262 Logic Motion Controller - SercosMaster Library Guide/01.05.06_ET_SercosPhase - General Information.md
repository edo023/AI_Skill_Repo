# ET\_SercosPhase - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.0.108.9437 |

## Description

This enumeration contains the possible Sercos communication phases for the Sercos master.

## Enumeration Elements

| Name | Value (DINT) | Description |
| --- | --- | --- |
| NRT | -1 | In the state NRT, Sercos communication is disabled. |
| Phase0 | 0 | Communication phase CP0. |
| Phase1 | 1 | Communication phase CP1. |
| Phase2 | 2 | Communication phase CP2. |
| Phase3 | 3 | Communication phase CP3. |
| Phase4 | 4 | Communication phase CP4. |
