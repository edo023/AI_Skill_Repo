# ET\_IdentificationMode - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.0.108.9437 |

## Description

This enumeration contains the type of addressing on the Sercos bus.

## Enumeration Elements

| Name | Value (DINT) | Description |
| --- | --- | --- |
| TopoAddress | 0 | The address of the devices is based on the topology address. The topology address corresponds to the physical position of the device on the Sercos bus. |
| SercosAddress | 1 | The address of the devices is based on the Sercos address. |
