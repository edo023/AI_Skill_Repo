# ET\_ServiceChannelAccessingMode - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.0.108.9437 |

## Description

This enumeration contains types of access for function calls via the service channel.

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| TopoAddress | 0 | Topological address is used to access the service channel. |
| SlaveHandle | 1 | Device handle is used to access the service channel. |
