# ET\_SlaveCommunicationState - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Enumeration type |
| Available as of: | V2.0.108.9437 |

## Description

This enumeration contains communication states for a Sercos device, depending on the Sercos state.

## Enumeration Elements

| Name | Value (UINT) | Description |
| --- | --- | --- |
| Operational | 0 | Device is in communication phase 4. |
| PhasingUp | 1 | Device is in communication phase 0, 1, 2, or 3. |
| NoCommunication | 2 | Device is in communication phase 4, but has not returned a valid state for at least two Sercos cycles, or the Sercos bus is in state Error. |
| NoData | 3 | Device is in communication phase 4, but an error has been detected. |
| NotRunning | 4 | Sercos is in state NRT. |
