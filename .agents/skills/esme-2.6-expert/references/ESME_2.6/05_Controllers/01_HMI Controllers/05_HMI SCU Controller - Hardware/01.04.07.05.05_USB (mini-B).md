When using a USB device, you can attach a USB holder to the USB interface on the unit to help prevent the USB cable from being disconnected.

NOTE: The USB holder is supplied in the accessory kit HMIZSUKIT (sold separately).

Attaching the USB Holder

| Step | Action |
| --- | --- |
| 1 | Insert the USB cable into the USB host interface.  G-SE-0021329.1.gif-high.gif    1   USB cable |
| 2 | Attach the USB holder to fix the USB cable in place.  G-SE-0021330.1.gif-high.gif    1   USB holder  2   USB cable |

Removing the USB Holder

Push down the tab of the USB holder and then remove the USB holder.
