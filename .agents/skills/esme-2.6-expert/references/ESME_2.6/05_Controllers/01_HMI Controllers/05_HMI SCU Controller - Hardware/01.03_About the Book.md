# About the Book

About the Book

Document Scope

The purpose of this document is to:

oShow you how to install and operate your [HMI controller](../glossary/glossary.htm#XREF_D_SE_0024697_661).

oShow you how to connect the HMI controller to a programming device equipped with SoMachine software.

oHelp you understand how to interface the HMI controller and other devices.

oHelp you become familiar with the HMI controller features.
