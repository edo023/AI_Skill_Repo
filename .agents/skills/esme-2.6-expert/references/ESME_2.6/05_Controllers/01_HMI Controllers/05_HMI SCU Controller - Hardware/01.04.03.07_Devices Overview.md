Devices Overview

Overview

HMISCU has various powerful features. This HMI controller can service a wide range of applications.

The software configuration and programming is accomplished with SoMachine and is described in the SoMachine [Programming Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=SCUprg&topicID=D_SE_0026174_7).

Key Features

Programming languages

The HMISCU is supported by and programmed with the SoMachine, which supports the following [IEC61131-3](../glossary/glossary.htm#XREF_D_SE_0024697_139) programming languages:

oIL: [Instruction List](../glossary/glossary.htm#XREF_D_SE_0024697_470)

oST: [Structured Text](../glossary/glossary.htm#XREF_D_SE_0024697_57)

oFBD: [Function Block Diagram](../glossary/glossary.htm#XREF_D_SE_0024697_716)

oSFC: [Sequential Function Chart](../glossary/glossary.htm#XREF_D_SE_0024697_449)

oLD: [Ladder Diagram](../glossary/glossary.htm#XREF_D_SE_0024697_144)

SoMachine can also be used to program the HMISCU controller using CFC ([Continuous Function Chart](../glossary/glossary.htm#XREF_D_SE_0024697_660)) language.

[Power Supply](../HMI_SCU_Logic_Controller_Installation/HMI_SCU_Logic_Controller_Installation-10.htm#XREF_D_SE_0024595_1)

The power supply of the HMISCU is 24 Vdc.

[Real Time Clock](../HMI_SCU_Logic_Controller_Description/HMI_SCU_Logic_Controller_Description-2.htm#XREF_D_SE_0024582_1)

The HMISCU includes a Real Time Clock (RTC) system.

[Memory](../Specifications_Intro/Specifications_Intro-9.htm#XREF_D_SE_0024521_1)

Embedded Input/Output

The following embedded I/O types are available, depending on the controller reference:

oRegular input

oFast input (HSC)

oRegular output

oFast output (PTO/PWM)

o[Analog input](../glossary/glossary.htm#XREF_D_SE_0024697_624)

oTemperature input

o[Analog output](../glossary/glossary.htm#XREF_D_SE_0024697_625)

NOTE: Frequency Generator (FG) is not supported.

Embedded communication features

4 types of communication ports are available on the rear panel:

oEthernet port

oUSB port

oSerial link port

oCANopen port

For more details, refer to the [chapter Communication Interface](../HMI_SCU-Communication_Services/HMI_SCU-Communication_Services-1.htm#XREF_D_SE_0024929_1).
