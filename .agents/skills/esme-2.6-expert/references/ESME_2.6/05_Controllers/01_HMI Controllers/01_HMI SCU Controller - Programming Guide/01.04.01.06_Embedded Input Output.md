# Embedded Input/Output

Embedded Input/Output

The following embedded I/O types are available, depending on the controller model:

oRegular inputs

oFast inputs (HSC)

oRegular outputs

oFast outputs (PTO/PWM)

o[Analog inputs](../glossary/glossary.htm#XREF_D_SE_0024697_624)

oAnalog outputs

oThermocouple inputs

oRTD (Resistance Temperature Detector) inputs
