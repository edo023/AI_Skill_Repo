# Presentation

Presentation

The function block ModbusServer is part of the library SE\_ModbusTCP\_Slave. It handles ModbusTCP requests for writing and reading data between the HMI SCU controller as a server and Modbus clients. Up to 10 client connections can be served. The function block operates on defined data areas in the HMI SCU that represent Modbus input and output registers.

The ModbusServer function block runs in a POU assigned to a task, and therefore, if the controller is in STOP mode, the task with the ModbusServer function block is not running.

The Unit ID of the ModbusServer is 255.
