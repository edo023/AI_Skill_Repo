# Supported Standard Data Types

Supported Standard Data Types

The controller supports the following IEC data types:

| Data Type | Lower Limit | Upper Limit | Information Content |
| --- | --- | --- | --- |
| [BOOL](../glossary/glossary.htm#XREF_D_SE_0024697_639) | FALSE | TRUE | 1 Bit |
| BYTE | 0 | 255 | 8 Bit |
| [WORD](../glossary/glossary.htm#XREF_D_SE_0024697_601) | 0 | 65,535 | 16 Bit |
| [DWORD](../glossary/glossary.htm#XREF_D_SE_0024697_681) | 0 | 4,294,967,295 | 32 Bit |
| [LWORD](../glossary/glossary.htm#XREF_D_SE_0024697_500) | 0 | 264-1 | 64 Bit |
| [SINT](../glossary/glossary.htm#XREF_D_SE_0024697_508) | -128 | 127 | 8 Bit |
| USINT | 0 | 255 | 8 Bit |
| [INT](../glossary/glossary.htm#XREF_D_SE_0024697_142) | -32,768 | 32,767 | 16 Bit |
| [UINT](../glossary/glossary.htm#XREF_D_SE_0024697_177) | 0 | 65,535 | 16 Bit |
| [DINT](../glossary/glossary.htm#XREF_D_SE_0024697_676) | -2,147,483,648 | 2,147,483,647 | 32 Bit |
| [UDINT](../glossary/glossary.htm#XREF_D_SE_0024697_431) | 0 | 4,294,967,295 | 32 Bit |
| [LINT](../glossary/glossary.htm#XREF_D_SE_0024697_498) | -263 | 263-1 | 64 Bit |
| ULINT | 0 | 264-1 | 64 Bit |
| [REAL](../glossary/glossary.htm#XREF_D_SE_0024697_504) | 1.175494351e-38 | 3.402823466e+38 | 32 Bit |
| [LREAL](../glossary/glossary.htm#XREF_D_SE_0024697_499) | 2.2250738585072014e-308 | 1.7976931348623158e+308 | 64 Bit |
| [STRING](../glossary/glossary.htm#XREF_D_SE_0024697_172) | 1 character | 255 characters | 1 character = 1 byte |
| WSTRING | 1 character | 255 characters | 1 character = 1 word |
| TIME | - | - | 32 Bit |

For more information on ARRAY, LTIME, DATE, TIME, DATE\_AND\_TIME, and TIME\_OF\_DAY, refer to the [SoMachine Programming Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_SE_0028740_7).
