# Compatibility

Compatibility

The library SE\_ModbusTCP\_Slave is compatible with the HMI SCU controllers HMISCUxA5 and HMISCUxB5 greater than or equal to firmware version 3.5.3.50.

The SE\_ModbusTCP\_Slave library has not been validated on any other SoMachine controller platform.
