# Firmware Update

Firmware Update

When transferring an application (via Ethernet, USB download cable, or a USB memory key), the [firmware](../glossary/glossary.htm#XREF_D_SE_0024697_707) update is done automatically [(with a user prompt for confirmation)](../HMI_SCU_-_Transfer/HMI_SCU_-_Transfer.htm#XREF_D_SE_0032039_1).
