# Modbus TCP Client

Modbus TCP Client

The Modbus TCP client supports the following function blocks from the PLCCommunication library without any configuration:

oADDM

oREAD\_VAR

oSEND\_RECV\_MSG

oSINGLE\_WRITE

oWRITE\_READ\_VAR

oWRITE\_VAR

For further information, refer to the [Function Block Descriptions](../../../../../../api/crossBook?lang=en-US&virtualBookName=m2xxcom&topicID=D_SE_0002235_1).
