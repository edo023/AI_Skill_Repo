# Using a Function Block in IL Language

Using a Function Block in IL Language

This procedure describes how to insert a function block in IL language:

| Step | Action |
| --- | --- |
| 1 | Open or create a new POU in Instruction List language.  NOTE: The procedure to create a POU is not detailed here. For more information, refer to [Adding and Calling POUs](../../../../../../api/crossBook?lang=en-US&virtualBookName=SoMProg&topicID=D_RU_0004939_1). |
| 2 | Create the variables that the function block requires, including the instance name. |
| 3 | Function Blocks are called using a CAL instruction:  oUse the Input Assistant to select the [FB](../glossary/glossary.htm#XREF_D_SE_0024697_701) (right-click and select Insert Box in the context menu).  oAutomatically, the CAL instruction and the necessary I/O are created.  Each parameter (I/O) is an instruction:  oValues to inputs are set by ":=".  oValues to outputs are set by "=>". |
| 4 | In the CAL right-side field, replace ??? with the instance name. |
| 5 | Replace other ??? with an appropriate variable or immediate value. |
