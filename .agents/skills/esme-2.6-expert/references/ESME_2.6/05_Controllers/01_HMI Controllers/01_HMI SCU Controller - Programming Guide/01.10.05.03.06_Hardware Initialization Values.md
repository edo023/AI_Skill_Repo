# Hardware Initialization Values

Hardware Initialization Values

This output state applies in the BOOTING, EMPTY (following power cycle with no boot application or after the detection of a system error), and INVALID\_OS states.

In the initialization state, analog, transistor, and relay outputs assume the following values:

oFor an [analog output](../glossary/glossary.htm#XREF_D_SE_0024697_625): Z (high impedance)

oFor a fast transistor output: Z (high impedance)

oFor a regular transistor output: 0 Vdc

oFor a relay output: Open
