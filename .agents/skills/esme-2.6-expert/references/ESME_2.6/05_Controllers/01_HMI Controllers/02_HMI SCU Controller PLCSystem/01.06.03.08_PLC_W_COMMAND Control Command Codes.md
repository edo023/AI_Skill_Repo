# PLC_W_COMMAND: Control Command Codes

![](images/logo.png)

PLC\_W\_COMMAND: Control Command Codes

Enumerated Type Description

The PLC\_W\_COMMAND enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| PLC\_W\_STOP | 0001 hex | Command to stop the controller. |
| PLC\_W\_RUN | 0002 hex | Command to run the controller. |
| PLC\_W\_RESET\_COLD | 0004 hex | Command to initiate a Controller cold reset. |
| PLC\_W\_RESET\_WARM | 0008 hex | Command to initiate a Controller warm reset. |
