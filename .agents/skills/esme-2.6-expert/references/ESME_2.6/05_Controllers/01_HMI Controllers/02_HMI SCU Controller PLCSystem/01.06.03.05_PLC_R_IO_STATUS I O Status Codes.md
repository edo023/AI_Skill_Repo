# PLC_R_IO_STATUS: I/O Status Codes

![](images/logo.png)

PLC\_R\_IO\_STATUS: I/O Status Codes

Enumerated Type Description

The PLC\_R\_IO\_STATUS enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| PLC\_R\_IO\_OK | FFFF hex | Inputs/Outputs are operational. |
| PLC\_R\_IO\_NO\_INIT | 0001 hex | Inputs/Outputs are not initialized. |
| PLC\_R\_IO\_CONF\_FAULT | 0002 hex | Incorrect I/O configuration parameters detected. |
| PLC\_R\_IO\_SHORTCUT\_FAULT | 0003 hex | Inputs/Outputs short-circuit detected. |
| RESERVED | 0004 hex | Not used |
| PLC\_R\_IO\_COM\_LOST | 0005 hex | Communication error detected with processor.  The embedded I/O are not updated. |
