# IsFirstMastWarmCycle: Indicates if Cycle is the First MAST Warm Start Cycle

![](images/logo.png)

IsFirstMastWarmCycle: Indicates if Cycle is the First MAST Warm Start Cycle

Function Description

This function returns TRUE during the first MAST cycle after a warm start.
