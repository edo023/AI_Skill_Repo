# IsFirstMastCycle: Indicates if Cycle is the First MAST Cycle

![](images/logo.png)

IsFirstMastCycle: Indicates if Cycle is the First MAST Cycle

Function Description

This function returns TRUE during the first MAST cycle after a start.
