# Configuration Procedure

Configuration Procedure

Follow this procedure to configure the capture function on a Main type:

| Step | Action |
| --- | --- |
| 1 | In the Devices tree, double-click Embedded Functions > HSC. |
| 2 | Enable the Capture input in the HSC0• > Auxiliary Inputs > CAP drop down menu. |
| 3 | Select a filtering value in the Auxiliary Inputs > CAP Filter drop down menu. |
| 4 | Define the triggering edge in the Auxiliary Inputs > CAP Filter drop down menu. |
