# Overview

Overview

The HSC function can execute fast counts of pulses from sensors, encoders, switches, and so on, that are connected to the dedicated fast inputs.

There are 2 types of HSC:

oSimple type: a [single input counter](HMI_SCU-OH-Embeeded_Functions_Configuration-4.htm#XREF_D_SE_0031194_1).

oMain type: a counter that uses up to [4 inputs (2 fast inputs and 2 standard inputs) and 2 reflex outputs](HMI_SCU-OH-Embeeded_Functions_Configuration-5.htm#XREF_D_SE_0031195_1).
