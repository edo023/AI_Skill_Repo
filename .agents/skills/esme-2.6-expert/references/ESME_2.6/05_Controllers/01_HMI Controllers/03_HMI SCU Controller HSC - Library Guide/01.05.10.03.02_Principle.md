# Principle

Principle

The counter assesses the number of pulses applied to the input for a predefined period of time. The counting register is updated at the end of each period with the number of events received.

The synchronization can be used over the time period. This restarts the counting event for a new predefined time period. The counting restarts at the edge [Sync condition](../Synchronization,_Enable,_Reset_to_Zero,_Homing/Synchronization_Enable_Reset_to_Zero_Homing-2.htm#XREF_D_SE_0006708_1)
