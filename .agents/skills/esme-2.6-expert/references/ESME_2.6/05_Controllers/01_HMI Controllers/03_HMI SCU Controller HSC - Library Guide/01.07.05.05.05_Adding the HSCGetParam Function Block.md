# Adding the HSCGetParam Function Block

Adding the HSCGetParam Function Block

| Step | Description |
| --- | --- |
| 1 | Select the Libraries tab in the Software Catalog and click Libraries.  Select Controller > HMISCU > HMISCU HSC > HSCGetParam in the list, drag-and-drop the item onto the POU window. |
| 2 | Link the HSC\_REF\_IN input to the HSC\_REF output of the HSC. |
