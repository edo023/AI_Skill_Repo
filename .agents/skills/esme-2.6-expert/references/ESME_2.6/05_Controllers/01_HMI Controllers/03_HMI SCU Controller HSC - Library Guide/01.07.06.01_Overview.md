# Overview

Overview

Each function can be represented in the following languages:

o[IL](../glossary/glossary.htm#XREF_D_SE_0024697_443): Instruction List

o[ST](../glossary/glossary.htm#XREF_D_SE_0024697_57): Structured Text

o[LD](../glossary/glossary.htm#XREF_D_SE_0024697_79): Ladder Diagram

o[FBD](../glossary/glossary.htm#XREF_D_SE_0024697_716): Function Block Diagram

o[CFC](../glossary/glossary.htm#XREF_D_SE_0024697_652): Continuous Function Chart

This chapter provides functions and function blocks representation examples and explains how to use them for IL and ST languages.
