# Configuration

Configuration

| Step | Action |
| --- | --- |
| 1 | In the Devices tree, double-click HMISCUxx5 > Embedded Functions > HSC. |
| 2 | Select the HSC• tab. |
| 3 | Set the value of the HSC > HSC0• > Auxiliary Inputs > EN parameter to Enabled. |
| 4 | Select the value of the HSC > HSC0• > Auxiliary Inputs > EN Filter parameter. |
