# HSC_REF: HSC Reference Value

![](images/logo.png)

HSC\_REF: HSC Reference Value

Data Type Description

The HSC\_REF is a [byte](../glossary/glossary.htm#XREF_D_SE_0024697_646) used to identify the HSC function associated with the administrative block.
