# HSC_PARAMETER_TYPE: Type for Parameters to Get or to Set on HSC Variable

![](images/logo.png)

HSC\_PARAMETER\_TYPE: Type for Parameters to Get or to Set on HSC Variable

Enumerated Type Description

The enumeration data type ENUM contains the following values:

| Enumerator | Value | Description |
| --- | --- | --- |
| HSC\_PRESET | 00 hex | To get or set the Preset value of an HSC embedded used for One-Shot and Free-Large mode. |
| HSC\_MODULO | 01 hex | To get or set the Modulo value of an HSC embedded used for Modulo-Loop mode. |
| HSC\_TIMEBASE | 02 hex | To get or set the [Timebase value](Data_Unit_Types-5.htm#XREF_D_SE_0007115_1) of an HSC embedded used for Event Counting and Frequency mode. |
| HSC\_THRESHOLD0 | 04 hex | To get or set the Threshold 0 value of an HSC embedded mode. |
| HSC\_THRESHOLD1 | 05 hex | To get or set the Threshold 0 value of an HSC embedded mode. |
