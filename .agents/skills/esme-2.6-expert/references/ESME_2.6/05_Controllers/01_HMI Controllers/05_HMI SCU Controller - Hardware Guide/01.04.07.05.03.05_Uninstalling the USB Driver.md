# Uninstalling the USB Driver

Uninstalling the USB Driver

| Step | Action |
| --- | --- |
| 1 | Click the USB device icon, Stop USB link cable, in the Windows task tray and then click Stop USB link cable (BMXXCAUSBH018) for Windows 2000, or Safely remove USB link cable (BMXXCAUSBH018) for Windows XP. |
| 2 | When the Safe To Remove Hardware message box appears, remove the USB download cable. |
| 3 | Click OK to close the message box. |
