# Driver Configuration

Driver Configuration

| Step | Action |
| --- | --- |
| 1 | In the Setting menu, touch the Offline tab. |
| 2 | Touch the IO Manager button and the following message appears: Working with Offline Settings will stop the user application and run-time. Continue? |
| 3 | Touch the OK to restart the system or the Cancel button to return to the System menu. |
| 4 | In the IO Manager control, touch the Driver Config button. |
| 5 | In the Driver Config control:  Select Driver: touch the up/down arrow to select the desired driver. |
| 6 | Set the parameters of the selected driver. |
