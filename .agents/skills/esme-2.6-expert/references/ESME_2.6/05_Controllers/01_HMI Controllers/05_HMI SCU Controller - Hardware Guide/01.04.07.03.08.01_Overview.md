# Overview

Overview

This section describes how to install and remove the rear module on a [DIN rail](../glossary/glossary.htm#XREF_D_SE_0024697_675).

NOTE: The HMISCU should always have the display module and controller module connected for an application to function properly.

If the rear module is powered without it being directly connected (or connected with a separation cable) to the display module, the PLC Logic goes into the STOP mode with I/Os in a fallback mode.
