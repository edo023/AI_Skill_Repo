Pin Assignment

The figure shows the RJ45 Ethernet connector pin assignment:

![G-SE-0002215.1.gif-high.gif](images/G-SE-0002215.1.gif-high.gif)

The table describes the RJ45 Ethernet connector pins:

| Pin | Signal |
| --- | --- |
| 1 | TD+ |
| 2 | TD- |
| 3 | RD+ |
| 4 | – |
| 5 | – |
| 6 | RD- |
| 7 | – |
| 8 | – |

NOTE: The HMISCU supports the MDI/MDIX auto-crossover cable function. You do not have to use special Ethernet crossover cables to connect devices directly to this port (connections without an Ethernet hub or switch).
