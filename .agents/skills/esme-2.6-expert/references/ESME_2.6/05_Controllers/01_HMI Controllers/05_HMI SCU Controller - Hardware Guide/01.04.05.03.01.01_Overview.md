# Overview

Overview

The HMISCU controllers include an RTC to provide system date and time information, and to support related functions requiring a real-time clock.

Variations in operating conditions can cause a clock shift:

| Ambient Temperature | Accuracy |
| --- | --- |
| 0°C (32 °F) | -1...2.5 minutes |
| 25°C (77 °F) | -2.5...2.5 minutes |
| 50°C 122 °F) | -5.5...2.5 minutes |
