# Introduction

Introduction

You can use the Settings menu to configure the unit.

Depending on how you use your HMISCU, you can display the Settings menu using three different methods:

oUse a Switch,

oUse a Touch Animation

oTouch the top-left corner of the panel when powering up,

oTouch two corners of the panel successively while the application is running.

You can select which methods your application uses in the Vijeo Designer editor’s Target properties.
