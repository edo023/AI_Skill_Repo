# Equipment Configuration

Equipment Configuration

| Step | Action |
| --- | --- |
| 1 | In the Setting menu, touch the Offline tab. |
| 2 | Touch the IO Manager button and the following message appears: Working with Offline Settings will stop the user application and run-time. Continue? |
| 3 | Touch the OK to restart the system, or the Cancel button to return to the Setting menu. |
| 4 | In the IO Manager control, touch the Equipment Config button. |
| 5 | In the Equipment Config control:  Select Equipment: touch the up/down arrow to select the desired equipment. |
| 6 | Set the parameters of the selected equipment. |
