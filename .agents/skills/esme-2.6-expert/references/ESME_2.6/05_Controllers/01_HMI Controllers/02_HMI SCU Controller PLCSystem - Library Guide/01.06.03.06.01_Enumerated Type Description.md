# Enumerated Type Description

Enumerated Type Description

The PLC\_R\_STATUS enumeration data type contains the following values:

| Enumerator | Value | Comment |
| --- | --- | --- |
| PLC\_R\_EMPTY | 0000 hex | Controller does not contain an application. |
| PLC\_R\_STOPPED | 0001 hex | Controller is stopped. |
| PLC\_R\_RUNNING | 0002 hex | Controller is running. |
| PLC\_R\_HALT | 0004 hex | Controller is in a HALT state. (see the controller state diagram in your controller programming guide). |
| PLC\_R\_BREAKPOINT | 0008 hex | Controller has paused at a breakpoint. |
