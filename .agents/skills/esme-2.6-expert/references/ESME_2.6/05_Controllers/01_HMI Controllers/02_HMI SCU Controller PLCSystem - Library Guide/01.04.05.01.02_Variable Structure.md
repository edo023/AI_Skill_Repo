# Variable Structure

Variable Structure

The following table describes the parameters of the SERIAL\_W[0] System Variable:

| Var Name | Type | Comment |
| --- | --- | --- |
| q\_wResetCounter | WORD | Transition from 0 to 1 resets all SERIAL\_R[0] counters.  To reset the counters again, it is necessary to write a 0 to this register before another transition from 0 to 1 can take place. |

NOTE: There is one communication port defined for the HMI SCU.
