# About the Magelis SCU HMI Controller

![](images/logo.png)

About the Magelis SCU HMI Controller

Magelis SCU HMI Controller Devices Overview

Overview

The Schneider Electric Magelis SCU HMI Controller has various powerful features. This [controller](../glossary/glossary.htm#XREF_D_SE_0024697_661) can service a wide range of applications.

The software configuration and programming is accomplished with the SoMachine software.

Programming Languages

The Magelis SCU HMI Controller is supported and programmed with the SoMachine software, which supports the following [IEC61131-3](../glossary/glossary.htm#XREF_D_SE_0024697_139) programming languages:

oIL: [Instruction List](../glossary/glossary.htm#XREF_D_SE_0024697_443)

oST: [Structured Text](../glossary/glossary.htm#XREF_D_SE_0024697_57)

oFBD: [Function Block Diagram](../glossary/glossary.htm#XREF_D_SE_0024697_702)

oSFC: [Sequential Function Chart](../glossary/glossary.htm#XREF_D_SE_0024697_449)

oLD: [Ladder Diagram](../glossary/glossary.htm#XREF_D_SE_0024697_79)

SoMachine software can also be used to program these controllers using CFC ([Continuous Function Chart](../glossary/glossary.htm#XREF_D_SE_0024697_652)) language.

Real Time Clock

The Magelis SCU HMI Controller includes a [Real Time Clock](../../../../../../api/crossBook?lang=en-US&virtualBookName=SCUhw&topicID=D_SE_0024582_1) (RTC) system.

Run/Stop

The Magelis SCU HMI Controller can be operated externally by the following:

oa [Run/Stop](../../../../../../api/crossBook?lang=en-US&virtualBookName=SCUhw&topicID=D_SE_0024586_10) operation by a dedicated digital input, defined in the software configuration. For more information, refer to [Configuration of Digital Inputs](../HMI_SCU-OH-Embeeded_Functions_Configuration/HMI_SCU-OH-Embeeded_Functions_Configuration-2.htm#XREF_D_SE_0026223_4).

oa SoMachine software command.

Embedded Input/Output

The following embedded I/O types are available, depending on the controller model:

oRegular inputs

oFast inputs (HSC)

oRegular outputs

oFast outputs (PTO/PWM)

o[Analog inputs](../glossary/glossary.htm#XREF_D_SE_0024697_624)

oAnalog outputs

oThermocouple inputs

oRTD (Resistance Temperature Detector) inputs

Embedded Communication Features

4 types of communication ports are available on the rear panel:

oEthernet port

oCANopen

oUSB programming ports

oSerial link port

For more details, refer to the [chapter Integrated Communication Ports](../../../../../../api/crossBook?lang=en-US&virtualBookName=SCUhw&topicID=D_SE_0024929_1).
