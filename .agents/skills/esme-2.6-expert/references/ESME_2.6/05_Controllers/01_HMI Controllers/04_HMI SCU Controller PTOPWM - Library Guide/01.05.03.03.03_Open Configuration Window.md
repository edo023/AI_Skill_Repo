# Open Configuration Window

Open Configuration Window

| Step | Action |
| --- | --- |
| 1 | In the Devices tree, double-click HMISCUxx5 > Embedded Functions > PTO\_PWM.  Result: The PTO\_PWM window is displayed. |
| 2 | Select PTO in the PTO0• > Mode drop down menu. |
| 3 | An instance of the PTO is created, it can be renamed in the Variable field.  Default name is: PTO00. |
