# Overview

Overview

Two function blocks can be used to adjust PTO parameters:

o[PTOGetParam](M238Lib_PTO_Administrative_Blocks-4.htm#XREF_D_RU_0005011_1), allows you to read the parameters

o[PTOSetParam](M238Lib_PTO_Administrative_Blocks-5.htm#XREF_D_RU_0005015_1), allows you to write the parameters
