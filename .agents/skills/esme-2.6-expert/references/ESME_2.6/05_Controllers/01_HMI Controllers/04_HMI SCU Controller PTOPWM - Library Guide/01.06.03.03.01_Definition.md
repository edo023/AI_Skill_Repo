# Definition

Definition

Pulse Width Modulation uses 1 fast physical output and up to 2 physical normal inputs.

In this document, use the following naming convention:

| Name | Description |
| --- | --- |
| SYNC | [Synchronization function](PWM_FG_Generalities-3.htm#XREF_D_SE_0007138_1). |
| EN | [Enable function](PWM_FG_Generalities-3.htm#XREF_D_SE_0007138_1). |
| IN\_SYNC | Physical input dedicated to the SYNC function. |
| IN\_EN | Physical input dedicated to the EN function. |
| OUT\_PWM | Physical output dedicated to the PWM. |
