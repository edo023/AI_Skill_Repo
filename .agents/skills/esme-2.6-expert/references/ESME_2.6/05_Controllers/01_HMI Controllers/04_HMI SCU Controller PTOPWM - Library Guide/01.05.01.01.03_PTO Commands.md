# PTO Commands

PTO Commands

The [PTOSimple](../M238Lib_PTO_PTO_Config_Reference_the_axis/M238Lib_PTO_PTO_Config_Reference_the_axis-1.htm#XREF_D_SE_0007002_1) function block manages the PTO.

Motion commands are managed by 2 motion function blocks:

o[PTOMoveRelative](../M238Lib_PTO_MoveCommands/M238Lib_PTO_MoveCommands-2.htm#XREF_D_SE_0009582_1): moving to a specified axis position

o[PTOMoveVelocity](../M238Lib_PTO_MoveCommands/M238Lib_PTO_MoveCommands-7.htm#XREF_D_SE_0006963_1): moving axis at a specified speed

o[PTOStop](../M238Lib_PTO_MoveCommands/M238Lib_PTO_MoveCommands-11.htm#XREF_D_SE_0006955_1): stop moving

Adjustments and diagnostics are managed by 3 administrative function blocks:

o[PTOSetParam](../M238Lib_PTO_Administrative_Blocks/M238Lib_PTO_Administrative_Blocks-5.htm#XREF_D_RU_0005015_1): Modify a parameter

o[PTOGetParam](../M238Lib_PTO_Administrative_Blocks/M238Lib_PTO_Administrative_Blocks-4.htm#XREF_D_RU_0005011_1): Read a parameter

o[PTOGetDiag](../M238Lib_PTO_Administrative_Blocks/M238Lib_PTO_Administrative_Blocks-7.htm#XREF_D_SE_0007556_1): Identify a detected error
