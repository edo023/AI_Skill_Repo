# Enumerated Type Description

Enumerated Type Description

For PTO and PWM function blocks, the enumeration data type ENUM contains the following values:

| Enumerator | Value | Description |
| --- | --- | --- |
| NO\_ERROR | 00 hex | No error detected |
| PTO\_UNKNOW\_REF | 01 hex | Unknown PTO reference or misconfigured PTO channel. |
| PTO\_UNKNOW\_PARAMETER | 02 hex | Unknown parameter type. |
| PTO\_INVALID\_PARAMETER | 03 hex | Invalid parameter value or incorrect combination of parameter values for the requested move. |
| PTO\_COM\_ERROR | 04 hex | Communication error detected with the PTO interface. |
| PTO\_AXIS\_ERROR | 05 hex | Movement error detected (for instance state [machine](../glossary/glossary.htm#XREF_D_SE_0024697_519) invalid). |
