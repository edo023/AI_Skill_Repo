# Programming the PTOSimple Function Block

![](images/logo.png)

Programming the PTOSimple Function Block

Procedure

| Step | Action |
| --- | --- |
| 1 | Select HMISCUxx5 in the Applications tree. |
| 2 | Click G-SE-0028599.1.jpg |
| 3 | Select POU in the drop-down menu.  Result: The Add POU window is displayed. |
| 4 | Enter the appropriate information in the Add POU window. |
| 5 | Click Add. |
| 6 | Select the Libraries tab in the Software catalog and click Libraries.  Select Controller > HMISCU > HMISCU PTOPWM > PTOSimple in the list, drag-and-drop the item onto the lower POU window on Start here. |
| 7 | Look for the function block instance by clicking G-RU-0010365.1.gif-high.gif.  The Input Assistant window appears. Select the Global Variable that you defined during the configuration and confirm. The default Global Variable is PTO\_PWM.PTO00.  G-SE-0024831.1.gif-high.gif  NOTE: If the function block instance is not visible, verify if the PTO is configured. |
| 8 | The inputs/outputs are detailed in the [function block](M238Lib_PTO_PTO_Config_Reference_the_axis-2.htm#XREF_D_SE_0007003_1). |
