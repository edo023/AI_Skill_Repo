# PTO_REF: Type for PTO Reference Value Variable

![](images/logo.png)

PTO\_REF: Type for PTO Reference Value Variable

Data Type Description

The PTO\_REF is a [byte](../glossary/glossary.htm#XREF_D_SE_0024697_646) used to identify the PTO\_REF function associated to the administrative and motion block.
