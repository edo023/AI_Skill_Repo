# Remote Monitoring

Remote Monitoring

The Remote Monitoring application monitors the system status of remote devices, including hard disk temperature, hard drive health, network connection, system / CPU temperatures, system / CPU fan speeds and system voltages. Email alarms and function logs are generated so managers can regularly keep track of their remote devices.

This procedure shows how to access Remote Monitoring applications:

| Stage | Description |
| --- | --- |
| 1 | Click Remote Monitoring icon to run the application.  Main application screen.  G-SE-0028473.1.gif-high.gif      Add Group: create a group name, edit its description, then add devices to a specific group.  Add Device: click search to scan for devices on the LAN and select the devices to add.  Threshold Setting: monitoring threshold contains four items: Temperature, Fan Speed, Voltage, and Hard Disk. In addition to these defaults, you can add or subtract items from this page. When the status is higher or lower than the threshold, the color of the value changes to red to notify users.  Remove: if you determine your device or group is not online, click Remove to remove it from the list.  Update All Agent: The console detects the update status of agents once one of the agents in the device list is not up to date. Click the Update All Agents icon in the notification area to update all the agents.  Manage device of group: click Edit to add or remove devices. Click Edit to edit its group name and related description. |
| 2 | Detailed Device Status: click the device to view details such as: CPU/Memory usage, system temperature, CPU temperature, device MAC address, motherboard model, BIOS version, processor model, memory capacity, operating system version, system status, storage status, and network status.  G-SE-0028474.1.gif-high.gif |
| 3 | Monitoring Threshold Settings  Use the Setting window to monitor these four threshold settings: Temperature, fan speed, voltage, and hard disk. You can add or remove items from this page.  G-SE-0028475.1.gif-high.gif |
| 4 | Set Report Schedule  Use this feature to set up a schedule for distributing run-time reports. Click Schedule Report to display the Set Report Schedule window. Define the schedule settings such as whether to repeat the operation Repeat Mode, when to run the report Time, the start and end period of the report Period, and the recipients of the report Mailing list.
