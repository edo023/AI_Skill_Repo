# Before Reinstallation

Before Reinstallation

Hardware required:

oRecover media, refer to the leaflet of the recover media.

Setting up the hardware:

oShut down Windows operating system in an orderly fashion and remove all power from the device.

oDisconnect all external peripherals.

NOTE: Save all main data onto a hard drive or a memory card. The reinstallation process returns the computer to its factory settings and erases all data.
