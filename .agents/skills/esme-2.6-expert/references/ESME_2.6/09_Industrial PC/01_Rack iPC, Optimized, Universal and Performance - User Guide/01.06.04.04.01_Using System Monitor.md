# Using System Monitor

Using System Monitor

This procedure describes how to use the System Monitor user interface:

| Stage | Description |
| --- | --- |
| 1 | Search bar: You can search for a device by device name. The list view shows the result.  G-SE-0028460.1.gif-high.gif      Hierarchical or List Mode: click the buttons on the right of the application name to switch between hierarchical and list mode. In hierarchical mode, the devices are divided into groups for easier management. In list mode, all the devices are shown on the same page to provide an overview of all the devices. |
| 2 | Group Tab: divide into tabs corresponding to information or function. The tabs also display the number of current devices. |
| 3 | Sorting: click the column header to sort by that information. Default is to sort by device name. |
| 4 | Device item: shows all device items. |
| 5 | Status count: shows the values most important to users. |
| 6 | Extended function: for each user, every remote application has its own customized functions. |
| 7 | Remove devices: If a device is offline for some time and you, do not want it to show up, click Remove Device.  NOTE: If the device comes online, it shows up again.  G-SE-0028471.1.gif-high.gif |
