# BIOS Configuration for the Rack iPC Universal and Optimized

BIOS Configuration for the Rack iPC Universal and Optimized

| Step | Action |
| --- | --- |
| 1 | For enter BIOS setup, press DEL or F2 key after the Rack iPC has been initialized. Use arrow keys to select among the items and press <Enter> to accept or enter the submenu. |
| 2 | Move to the Advanced settings and scroll down to the H/W Monitor:  G-SE-0032314.1.gif-high.gif |
| 3 | Move down to:  oCPUFAN1 Mode Setting and change it to Enabled.  oSYSFAN1 Mode Setting and change it to Enabled |
| 4 | Move to the Advanced settings and Save and Exit. |
