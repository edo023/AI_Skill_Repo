# BIOS Configuration for the Rack iPC Performance

BIOS Configuration for the Rack iPC Performance

| Step | Action |
| --- | --- |
| 1 | For enter BIOS setup, press DEL or F2 key after the Rack iPC has been initialized. Use arrow keys to select among the items and press <Enter> to accept or enter the submenu.  For creating the smart fan configuration use the BIOS Advanced menu:  G-SE-0032312.1.gif-high.gif |
| 2 | Move to the Advanced settings and scroll down to the H/W Monitor:  Select Smart Fan Mode Configuration and press Enter:  G-SE-0032313.1.gif-high.gif      You can access the following options from the BIOS setup:  oCPU Fan  oSystem Front Fan  oSystem Rear Fan |
| 3 | Choose the fan and select Enable:  G-SE-0032315.1.gif-high.gif |
| 4 | Move to the Advanced settings and Save and Exit. |
