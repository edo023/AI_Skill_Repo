Rack iPC BIOS Update

| Step | Action |
| --- | --- |
| 1 | Save the new BIOS on a USB pen drive. |
| 2 | Rename the BIOS file to “update.bin”. |
| 3 | Plug the pen drive into the Rack iPC. |
| 4 | Enter the BIOS setup. |
| 5 | In this menu, select the Rack iPC BIOS update option.  Result: The BIOS is automatically updated to the new version. |

Trusted Computing Submenu

This submenu enables/disables the TPM (TPM 1.1/1.2) setup in BIOS.

The TPM (Trusted platform module) is a secure key generator and key cache management component that enables protected storage of encryption keys and authentication credentials for enhanced security capabilities.
