# USB Interface

USB Interface

| Element | Characteristics |
| --- | --- |
| Type | USB 3.0 and USB 2.0 |
| Transfer rate | Low speed (1.5 Mb/s), full speed (12 Mb/s), high speed (480 Mb/s)  and super speed (5 Gb/s) (USB 3.0 port only) |
| Current load | USB 3.0: 0.9 A per connection and USB 2.0: 0.5 A per connection |
| Connection | Type A |
