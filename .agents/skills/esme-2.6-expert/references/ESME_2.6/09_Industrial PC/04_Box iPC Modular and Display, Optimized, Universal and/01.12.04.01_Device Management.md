# Device Management

Device Management

This procedure describes how to use the Device Management user interface:

| Step | Description |
| --- | --- |
| 1 | Device management  oAfter user login, Device is the default page.  oDevice management page is composed of a system hierarchy tree (left-side) and device list (right-side).  oDevice management provides three levels of management view: Device List, Group List, and Map View.  oSystem hierarchy tree includes server, account, and group node for device/group list mode as well as location, layout, and device node for map view mode. Each node supports corresponding operations (add/delete/edit) according to node attributes.  G-SE-0043736.1.gif-high.gif |
| 2 | View mode – Device status list:  G-SE-0043577.1.gif-high.gif |
| 3 | Add/Delete/Edit device server  Add device server: Select on one of server nodes and click Add to the pop-up menu option:  G-SE-0043566.1.gif-high.gif       Click Add Server to pop up the Device Server dialog for new subserver registering.  Delete device server:  Click Edit to switch to edit mode and click the icon X to delete this server node.  Edit device server:  Click Edit to switch to edit mode and select one of the server nodes. You can remove and edit this server node. |
| 4 | Add/Delete/Edit device group  Add device group: Select one user account and click Add to the pop-up menu option. Click Add Group to pop up Device Group dialog for new group addition:  G-SE-0043565.2.gif-high.gif |
| 5 | Delete /Edit device group  Delete /Edit device group: Click Edit to switch to edit mode and select one of the group nodes. You can remove and edit this group node:  G-SE-0043752.1.gif-high.gif |
| 6 | Add/Delete/Edit device  Add device: Select one of the user accounts or groups and click Add to the pop-up menu option. Click Add Device to the pop-up dialog for new device addition:  G-SE-0043564.2.gif-high.gif |
| 7 | Manual add  Click Add Device to pop up the Add Device dialog to add a device manually. You can input known device ID or MAC addresses that have already registered to the server and assign a current account or group. If the device does not exist, you can also add a device directly:  G-SE-0043739.2.gif-high.gif |
| 8 | Search device  Click Search Device to pop up the Device dialog for advanced device smart search. The system auto-discovers both connected and unassigned devices located at the same local area network as the client user:  G-SE-0043742.2.gif-high.gif |
| 9 | Delete device  Click Edit to switch to edit mode. You can remove and edit devices on the device list in this mode. Click the X icon for the selected device row and confirm the device warning removal:  G-SE-0043755.2.gif-high.gif |
| 10 | Edit device  Click Edit to switch to edit mode. You can remove and edit devices of the device list in this mode. Click selected device name to pop up the Device dialog for editing:  G-SE-0043570.2.gif-high.gif |
