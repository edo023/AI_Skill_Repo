# USB Configuration Menu

USB Configuration Menu

| BIOS setting | Description |
| --- | --- |
| XHCI Pre-Boot Driver | Enables or disables XHCI (eXtensible Host Controller Interface) Pre-Boot Driver support. |
| XHCI Mode | Select mode of operation of XHCI mode. |
| USB Port Disable Override | Enables or disables USB Port from reporting a Device Connection to controller. |
| XHCI Disable Compliance Mode | Enables or disables XHCI Link Compliance Mode. |
| USB HW MODE AFE Comparators | Enables or disables USB HW MODE AFE Comparators. |
| Front Panel USB Control | Enables or disables SMSC USB HW HUB port. |

NOTE: Front Panel USB control only for when HMIBMI/HMIBMO bundled with 4:3 12” and 4:3 15” HMIDM.
