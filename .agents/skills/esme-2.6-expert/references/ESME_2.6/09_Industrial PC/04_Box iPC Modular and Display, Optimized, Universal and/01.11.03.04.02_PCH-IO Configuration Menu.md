# PCH-IO Configuration Menu

PCH-IO Configuration Menu

| BIOS setting | Description |
| --- | --- |
| PCI Express Configuration | Change mini PCIe configuration settings. |
| USB Configuration | Change USB configuration settings. |
| PCH Azalia Configuration | Azalia (Intel High Definition Audio) |
| Restore AC Power Loss | Select AC power state when power is reapplied after a power outage. |
