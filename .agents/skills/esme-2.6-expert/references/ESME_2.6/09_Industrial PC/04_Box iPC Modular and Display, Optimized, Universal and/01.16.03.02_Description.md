# Description

Description

To access software and documentation, plug the USB memory key into USB port and navigate to software or documentation folders.

| Step | Action |
| --- | --- |
| 1 | Insert the USB memory into the USB port and press F7 during the boot to select the USB Disk for boot. Select USB DISK 2.0 PMAP or UEFI: USB DISK 2.0 PMAP.  G-SE-0067112.1.gif-high.gif      G-SE-0067113.1.gif-high.gif |
| 2 | Click Yes or press Enter to continue.  G-SE-0067111.1.gif-high.gif |
| 3 | Optional only for Windows 10: If the system recognizes you are making recovery for HMIBMP/ HMIBMU/HMIBMI/HMIBMO box types, it will pop up new step to ask you to select which OS version you want to recover. (If the system recognizes you are making recovery for PSP/PEP/PSO product types, it will not show this pop-up page and will directly move to step 4.).  There are two OS versions for your selection. One is Node-Red for IIoT Box (Node-Red version); the other is System Monitor for Box PC (System Monitor version). Read user manual carefully in [System Monitor](../iPC_-_System_Monitor_and_IIoT_Monitor/iPC_-_System_Monitor_and_IIoT_Monitor-1.htm#XREF_D_SE_0082782_1) and [IIoT and Cyber Security](../Box_PC_-_IIoT_Box_Project_Cyber_Security/Box_PC_-_IIoT_Box_Project_Cyber_Security-1.htm#XREF_D_SE_0067651_1) to decide which OS version you want to recover.  G-SE-0067109.1.gif-high.gif |
| 4 | Optional: If there is more than one valid disk been detected, you have to choose which one you want to use. You need to key in the disk number, for example: 1, 2..., and press Enter key to continue.  NOTE: The order of the disks bases on the plug-in sequences and hardware design.  G-SE-0067110.1.gif-high.gif |
| 5 | Recovery function starts automatically.  G-SE-0067108.1.gif-high.gif |
| 6 | After recovery is finished, click Shut down to end the recovery processes or click Restart to continue.  G-SE-0067107.1.gif-high.gif |
| 7 | Reboot and press F7 to select the disk as boot device. Select the disk that you did the recovery.  G-SE-0067106.1.gif-high.gif      G-SE-0067105.1.gif-high.gif |
| 8 | Complete the OS initial settings. It may reboot 3 to 4 times to finish. |
| 9 | Optional only for Windows 10: If you select IIoT Node-RED SKU in Step3, it will ask you to change the Node-RED password during the recovery. The current default password of Node- RED is NodeRed#0123.  Password change policy:  oPasswords must have at least 12 characters  oPasswords cannot contain the username  oPasswords must include the four available character types: lowercase letters, uppercase letters, numbers, and symbols. Symbol must include any one of [!@#$%^&\*?\_,~]  G-SE-0071583.1.gif-high.gif  .  NOTE: If the password is not meet above criteria, system will request again to password, until meet criteria. |
