# EWF Manager (only on WES7)

EWF Manager (only on WES7)

The Magelis S-Panel PC operating system is installed on a memory card. This card is a rewritable CFast card that enables approximately 100,000 write operations.

The EWF manager (enhanced write filter manager) minimizes the number of write operations to help extend the life of the CFast card. The EWF manager loads temporary data (for example, system updates and software operations) into RAM, and does not write this information to the CFast card.

As a result, when using the EWF manager, restarting the S-Panel PC overwrites changes that the user has made to the system. The following types of changes may be overwritten if the EWF manager is active and the system is restarted:

oNewly installed applications.

oNewly installed peripherals.

oNewly created or modified user accounts.

oNetwork configuration changes (such as IP addresses or default gateways).

oOperating System customizations (such as desktop background).
