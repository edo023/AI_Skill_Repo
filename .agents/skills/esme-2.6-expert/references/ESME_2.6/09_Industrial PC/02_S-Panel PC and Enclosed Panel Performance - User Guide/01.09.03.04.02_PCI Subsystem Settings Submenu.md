# PCI Subsystem Settings Submenu

PCI Subsystem Settings Submenu

| BIOS setting | Description |
| --- | --- |
| PCI Latency Timer | Select PCI Latency Timer Register. |
| VGA Palette Snoop | Enables or disables VGA Paltte Registers Snooping. |
| PERR# Generation | Enables or disables PCI device to Generate PERR#. |
| SERR# Generration | Enables or disables PCI device to Generate SERR#. |
| PCI Express Settings | Select PCI Express Settings. |
