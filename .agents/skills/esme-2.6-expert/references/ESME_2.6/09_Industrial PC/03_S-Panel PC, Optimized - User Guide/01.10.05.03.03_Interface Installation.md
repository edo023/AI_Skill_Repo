# Interface Installation

Interface Installation

Before installing or removing a mini PCIe card, shut down Windows in an orderly fashion and remove the power from the device.

NOTE: Be sure to remove the power before attempting this procedure.

| Step | Action |
| --- | --- |
| 1 | Disconnect the power cord to the S-Panel PC. |
| 2 | Touch the housing or ground connection (not the power supply) to discharge any electrostatic charge from your body. |
| 3 | Remove the four screws of the rear cover:  G-SE-0042327.1.gif-high.gif |
| 4 | Install the mini PCIe card onto the board:  G-SE-0042325.1.gif-high.gif      NOTE: The recommended torque to tighten these screws is 0.5 Nm (4.5 lb-in). |
| 5 | Take out the extension kit (HMIYPADPSOSTO1), and remove the optional interface cover bracket. Install the interface onto the kit, using fasten screws on both sides of the interface:  G-SE-0042324.1.gif-high.gif |
| 6 | Connect the cable to the mini PCIe card that is on the motherboard:  G-SE-0042321.1.gif-high.gif      NOTE: When using a mini PCIe card with an external cable attached, use a clamp or a similar device to secure the cable. |
| 7 | Fasten the extension kit to the S-Panel PC with the four studs:  G-SE-0042329.1.gif-high.gif |
| 8 | Replace the rear cover and fasten it with four screws:  G-SE-0042326.1.gif-high.gif      NOTE: The recommended torque to tighten these screws is 0.5 Nm (4.5 lb-in). |
