# PCI Express Configuration Submenu

PCI Express Configuration Submenu

| BIOS setting | Description |
| --- | --- |
| PCI Express Clock Gating | Enable or disable PCI Express Clock Gatting for each root port. |
| DMI Link ASPM Control | Enable or disable DMI Link ASPM Control. |
| DMI Link Extended Synch Control | Enable or disable DMI Link Extended Synch Control. |
| PCIe-USB Glitch W/A | Enable or disable PCIe-USB Glitch W/A. |
| PCI Express Root Port 1 | Select PCI Express Root Port 1 settings. |
| MINI PCIe | Select PCI Express Root Port 6 settings. |
