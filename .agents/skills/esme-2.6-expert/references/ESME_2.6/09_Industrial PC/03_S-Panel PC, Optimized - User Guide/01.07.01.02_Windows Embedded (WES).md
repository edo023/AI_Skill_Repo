# Windows Embedded (WES)

Windows Embedded (WES)

The WES is a modularized version of the Windows operating system that provides increased reliability and customizations. It offers the power and familiarity of Windows in a compact, more reliable form. For more information, refer to Microsoft Windows Embedded Web page.

WES provides many tools for the customization of menus, boot screens, and dialog boxes. With WES, you can remove the Windows boot and resume animations so the screen remains black during startup. You can also remove the Windows logo from the login screen and other startup screens. Other common features of Windows include the message and dialog boxes. WES can filter these messages and keep them from appearing during run time. The developer can choose to hide any dialog box and predefine its default operation so it never displays to the user.
