# Main Board Setting

Main Board Setting

Before to access the main board, shut down Windows in an orderly fashion and remove all power from the device.

NOTE: Be sure to remove all power before attempting this procedure.

| Step | Action |
| --- | --- |
| 1 | Disconnect the power cord to the S-Panel PC. |
| 2 | Touch the housing or ground connection (not the power supply) to discharge any electrostatic charge from your body. |
| 3 | Remove the four screws of the rear cover:  G-SE-0042327.1.gif-high.gif |
| 4 | You can access connectors and setting of the main board.  The figure shows the main board of the S-Panel PC:  G-SE-0042094.1.gif-high.gif    CN1   Clear CMOS  CN12   SATA signal connector  CN13   SATA power connector  SW2   COM2 RS-232/422/485 mode setting  SW3   Termination resistors select |
| 5 | Replace the rear cover and fasten it with four screws:  G-SE-0042327.1.gif-high.gif      NOTE: The recommended torque to tighten these screws is 0.5 Nm (4.5 lb-in). |
