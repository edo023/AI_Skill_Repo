# Description

Description

This Software API (Application Programming Interfaces) is a micro controller that provides embedded features for system integrators. Embedded features have been moved from the OS/BIOS level to the board level to increase reliability and to simplify integration. Software API runs whether the operating system is running or not; it can count the boot times and running hours of the device, monitor device health, and provide an advanced watchdog to handle errors found as they happen. Software API also comes with a secure and encrypted EEPROM for storing main security keys or other customer defined information. All the embedded functions are configured through an API (application programming interface) or by a DEMO tool. Schneider Electric provides this suite of Software API and the underlying drivers required. Also a set of user-friendly, intelligent, and integrated interfaces speed development, enhance security, and offer add-on value for Schneider Electric platforms.
