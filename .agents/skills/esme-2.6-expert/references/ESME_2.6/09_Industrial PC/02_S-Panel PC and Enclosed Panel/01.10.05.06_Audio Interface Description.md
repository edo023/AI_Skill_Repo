# Audio Interface Description

![](images/logo.png)

Audio Interface Description

Introduction

The HMIYMINAUD21 is categorized as an audio interface (line in, line out, Mic in). The audio interface is composed of an audio I/O board (include metal plate), a cable for connecting I/O board and the S-Panel PC.
