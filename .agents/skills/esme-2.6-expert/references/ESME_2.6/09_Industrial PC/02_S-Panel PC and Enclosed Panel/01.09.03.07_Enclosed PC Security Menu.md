Enclosed PC Security Menu

Security Setup

Select Security Setup from the main BIOS setup menu. All Security Setup options, such as password protection, are described in this section. To access the submenu for the following items, select the item and press Enter.

To change the administrator or user password, select the Administrator / User Password option, press Enter to access the submenu, and then type the password.
