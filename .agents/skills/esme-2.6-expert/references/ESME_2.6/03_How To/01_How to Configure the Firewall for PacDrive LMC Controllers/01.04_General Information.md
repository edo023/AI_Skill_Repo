# General Information

## Overview

Starting with EcoStruxure Machine Expert V1.2, a firewall is provisioned and activated by default on the PacDrive LMC controllers (PacDrive LMC Eco, PacDrive LMC Pro/Pro2).

The firewall is configured in a strict way on the general Ethernet interfaces:

* **CN3** on PacDrive LMC Eco
* **CN8** on PacDrive LMC Pro/Pro2

Incoming traffic is blocked, except for defined applications / ports.

Not affected by the default configuration of the firewall are:

* Outgoing traffic
* Ethernet-based fieldbusses such as Sercos

## Configuration

You can configure the firewall in two different ways:

* During runtime using SystemInterface library functions.
* Permanently by editing a configuration file on the memory card.

Both ways of configuration are explained in this document.
