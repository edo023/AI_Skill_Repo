Commissioning of an Axis

Default Settings

Most of the default settings do not need to be modified; however, a few settings need to be adjusted.

Assigning The Power Supply

If you have more than one power supply, and you do not want the drive connected to the first power supply, click Devices tree > DRV\_Slave1 > Parameter > State and set ConfiguredPowerSupply.

Unit for Motor (Feed Constant)

Click Devices tree > DRV\_Slave1 > Parameter > Mechanic and set FeedConstant to 360.0.

This value represents a whole rotation of the motor – touch probe parameters later in this example relate to this value

Motor Temperature Monitoring

Depending on the type of motor used, this setting may have to be adjusted.

Click Devices tree > DRV\_Slave1 > Parameter > Motor and set Motor temperature monitoring to Thermal model if the motor does not have a sensor.

Calculating J\_Load (Motion Builder)

| Step | Action |
| --- | --- |
| 1 | Calculate the J\_Load value using Motion Builder. Close Logic Builder and start Motion Builder. |
| 2 | Click Machine: Main Machine > DRV\_Slave1 > Mechanic in the tree view. |
| 3 | Click the calculator icon near J Load. |
| 4 | In the upper part of the Calculate the Inertia dialog box, chose the appropriate values, then click Add. The calculated value is listed in the lower part of the dialog box and OK becomes available. |
| 5 | Click OK. |
| 6 | Copy the value of J Load. |
| 7 | Close Motion Builder. Start Logic Builder. |
| 8 | Click Devices tree > DRV\_Slave1 > Parameters > Mechanic. |
| 9 | Set J\_Load to the calculated value by pasting the previously copied value. |
