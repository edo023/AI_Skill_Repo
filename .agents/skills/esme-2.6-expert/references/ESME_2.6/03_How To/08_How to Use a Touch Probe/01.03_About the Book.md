About the Book

Document Scope

The aim of this document is to show the steps to achieve a typical task with an automation system: Homing a drive using a touch probe. Following this example leads you through EcoStruxure Machine Expert. This example is based on a specific configuration, however, in principle, the concepts are the same for other EcoStruxure Machine Expert devices.
