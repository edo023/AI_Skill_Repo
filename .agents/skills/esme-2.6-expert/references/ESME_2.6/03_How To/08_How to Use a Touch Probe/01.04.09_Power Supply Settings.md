# Power Supply Settings

Power Supply Settings

Most of the default settings do not need to be modified; however, a few settings need to be adjusted.

| Step | Action |
| --- | --- |
| 1 | Click Devices tree > PSM\_PowerSupply > Parameters > Common and state. |
| 2 | Set PhaseCheckMode to Check only one phase. |
| 3 | Set MainsVoltageMode to 230V. |
| 4 | Set BrakingResistorMode to 230V Mode. |
