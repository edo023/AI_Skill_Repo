# Hardware

Hardware

Choosing and assembling the hardware (controller, power supply, servo drive) is beyond the scope of this document. Refer to the technical documentation for the hardware you are using.

The test-/training-hardware-assembly used for this document consists of the following components:

oLogic motion controller: PacDrive LMC 300C

oPower supply: Lexium LXM62PSD

oServo drive: Lexium LXM62D

oMotor: BSH0701T01A2A

oA typical Touch Probe sensor

If you use different components to follow this example, you need to adapt the instructions accordingly.
