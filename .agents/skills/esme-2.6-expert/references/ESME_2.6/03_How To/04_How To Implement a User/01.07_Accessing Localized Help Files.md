# Accessing Localized Help Files

## Providing Access to Language-Specific Help Files

If your individual help files are available in different languages EcoStruxure Machine Expert allows you to open the help files in the language that corresponds to the setting you made in EcoStruxure Machine Expert Logic Builder for the help language ([Tools > Options > International Settings dialog box](../../../../../api/crossBook?lang=en-US&virtualBookName=SoMMenu&topicID=D_SE_0084048)).

To achieve this, your individual help files must be provided in subfolders of the Directory for User Documentation folder. The language-specific subfolders must be named according to the following nomenclature:

* DE (German)
* ES (Spanish)
* FR (French)
* IT (Italian)
* ZH (Chinese)
