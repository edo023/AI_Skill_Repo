# Prerequisites for Accessing Individual Help Files

## Overview

EcoStruxure Machine Expert allows you to integrate your individual help files into the online help if the following prerequisites are fulfilled:

* The help files are available in CHM (Compiled HTML Help) or HTM/HTML format.
* The files are stored in the Directory for User Documentation folder.

## Configuring the Directory for User Documentation

To configure the path to the directory where you store your own help files, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | In EcoStruxure Machine Expert Logic Builder, execute the command Tools > Options > Online Help. |
| 2 | In the Directory for User Documentation box, enter the path to the directory or click the browse button (...) to browse for the directory where you store your help files. |
| 3 | Click OK to close the Options dialog box. |
| 4 | Restart EcoStruxure Machine Expert. |

**Result**: The help files available in the Directory for User Documentation folder are accessible from the EcoStruxure Machine Expert Logic Builder in two different ways:

* Via the Help > Add-ons [command](ViaCommand-EB1F157F.html).
* By pressing the [F1 key on the keyboard](ViaF1Key-EB39179F.html). This is exclusive to user documentation for libraries if a library mapping file in \*.librarymapping format is available in the Directory for User Documentation and a specific function block is selected in the EcoStruxure Machine Expert Logic Builder.
