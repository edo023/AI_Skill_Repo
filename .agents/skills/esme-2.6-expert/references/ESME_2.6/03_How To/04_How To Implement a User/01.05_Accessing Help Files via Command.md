# Accessing Help Files via Command

## Overview

Online help files located in the Directory for User Documentation are accessible from EcoStruxure Machine Expert via the Help > User Documentation command. The access depends on whether they are located in the root folder or in a subfolder.

## Accessing Files in the Root Directory for User Documentation

Files of type \*.chm or \*.htm/\*.html in the root folder are displayed as subcommands of Help > User Documentation and are directly accessible.

## Accessing Files via Shortcuts

To access help files that are not located in the Directory for User Documentation, create a shortcut to this file in the root Directory for User Documentation. This shortcut is also displayed as a subcommand of Help > User Documentation.
