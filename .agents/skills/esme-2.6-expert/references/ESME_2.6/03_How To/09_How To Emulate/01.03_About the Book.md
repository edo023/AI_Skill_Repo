# About the Book

## Document Scope

This document describes how to display a 3-D emulation of your configured objects, for example, one multi carrier track or one robot.

The emulation is performed by running EcoStruxure Machine Expert Twin in the background and is displayed in a view area in the tool you are using, for example, EcoStruxure Machine Expert. As this viewer provides restricted functions, you can use the standalone version of EcoStruxure Machine Expert Twin that was installed with your EcoStruxure Machine Expert to extend the functions.

## Validity Note

This document has been updated for the release of EcoStruxure™ Machine Expert V2.2.
