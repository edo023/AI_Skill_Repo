# Lexium 62 Connection Module

## Overview

The Lexium 62 Connection Module simplifies the wiring of the devices. With the Lexium 62 Connection Module you can connect up to 45 servo drives.

| Product | Description |
| --- | --- |
|  | The Lexium 62 Connection Module supplies the Lexium 62 ILMs with DC voltage from the DC bus via a hybrid cable or a power cable (daisy chain wiring). Additionally, the Lexium 62 Connection Module provides the Inverter Enable and Sercos interface.  For further information, refer to:   * [*Installation and Maintenance*](D-SE-0064825.html#D-SE-0064825) * [*Indicators of the Lexium 62 Connection Module*](D-SE-0064648.html#D-SE-0064648) * [*Electrical Power Connection of the Lexium 62 Connection Module*](D-SE-0064675.html#D-SE-0064675) * [*Mechanical and Electrical Data for the Lexium 62 Connection Module*](D-SE-0062194.html#D-SE-0062194) |
