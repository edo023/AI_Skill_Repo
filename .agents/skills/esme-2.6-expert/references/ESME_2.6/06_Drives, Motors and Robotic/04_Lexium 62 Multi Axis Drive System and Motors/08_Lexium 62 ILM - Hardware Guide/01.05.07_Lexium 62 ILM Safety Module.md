# Lexium 62 ILM Safety Module

## Overview

Option module for Lexium 62 ILM integrated servo drive.

| Product | Description |
| --- | --- |
|  | Lexium 62 ILM Safety Module manages the safety-related motion functions via Sercos bus.   * Safe Torque off (STO) * Safe Stop 1 (SS1) * Safe Operating Stop (SOS) * Safe Stop 2 (SS2) * Safe Maximum Speed (SMS) * Safely-Limited Speed (SLS) * Safe Direction Indication (SDI)   For further information, refer to the chapter [*Lexium 62 ILM Safety Module*](D-SE-0064774.html#D-SE-0064774). |
