# Replacing the Motor

## Description

If you replace the motor, the absolute position of the encoder is no longer valid.

| WARNING | |
| --- | --- |
|  | UNINTENDED MOVEMENT DUE TO INCORRECT ABSOLUTE POSITION  Set the new absolute position of the encoder after having replaced the motor.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

| Step | Action |
| --- | --- |
| 1 | Power off all supply voltages. Verify that no voltages are present. |
| 2 | Label all connections and uninstall the product. |
| 3 | Note the identification number and the serial number shown on the product nameplate for later identification. |
| 4 | Install the new product as per section [Installation](D-SE-0060249.html#D-SE-0060249). |
| 5 | Commission the product as per section [Commissioning](D-SE-0060260.html#D-SE-0060260). |
