# Cable Specifications for Motors with One-Cable Connection (SH3-OMC)

## Description

Using pre-assembled cables helps to reduce the possibility of wiring errors. See [Accessories and Spare Parts](D-SE-0060269.html#D-SE-0060269).

The genuine accessories have the following properties:
