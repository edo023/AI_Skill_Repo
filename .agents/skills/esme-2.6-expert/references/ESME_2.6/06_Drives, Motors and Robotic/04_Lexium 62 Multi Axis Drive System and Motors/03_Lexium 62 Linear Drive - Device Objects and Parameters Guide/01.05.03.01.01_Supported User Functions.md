# Supported User Functions

Supported User Functions

The device variants of the Lexium LXM62 Drive and Lexium LXM62 Linear Drive provide device-specific features. These features can be activated as user functions during Sercos phase up.

The selection of the device variant and the activation of the user functions is done on the Feature Configuration tab of the device editor of the Lexium LXM62 Drive / Lexium LXM62 Linear Drive object (as described in the chapter [Selecting the Device Variant and User Functions](../Device_Variants/Device_Variants.htm#XREF_D_SE_0090515_3)). After selection, the related object parameter group appears on the Parameters tab of the device editor inside the parameter group UserFunctions and can be used in the application.

Device variants and supported user functions:

| Device variant | Supported user functions |
| --- | --- |
| Standard | Provides standard Lexium LXM62 Drive / Lexium LXM62 Linear Drive functions. No additional user functions are supported. |
| Standard Plus | Supports the user functions for Lexium LXM62 Drive:  o[Encoderless Velocity Control](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0089869_1)  oTorque Limitation  oBrake Check (available via functions FC\_BrakeCheckGetState and FC\_BrakeCheckStart in the SystemInterface Library)  For Lexium LXM62 Linear Drive there are no new user functions available. |
| Advanced Plus | Supports the user functions for Lexium LXM62 Drive:  o[Encoderless Velocity Control](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0089869_1)  oTorque Limitation  oBrake Check (available via functions FC\_BrakeCheckGetState and FC\_BrakeCheckStart in the SystemInterface Library)  o[Incremental Encoder Output](../Incremental_Encoder_Output/Incremental_Encoder_Output.htm#XREF_D_SE_0090015_1)  Supports the user function for Lexium LXM62 Linear Drive:  o[Incremental Encoder Output](../Incremental_Encoder_Output/Incremental_Encoder_Output.htm#XREF_D_SE_0090015_1) |
| Application defined | Supports all user functions (see the section [Using the Device Variant “Application defined”](../Device_Variants/Device_Variants.htm#XREF_D_SE_0090515_1)). |
