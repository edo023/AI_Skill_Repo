# Functional Description

Functional Description

The parameter indicates which digital input/output is configured as input (see parameter [IO0\_Mode, IO1\_Mode](../IO0_IO1Mode/IO0_IO1Mode.htm#XREF_D_SE_0071946_1)).

| Bit value | Meaning |
| --- | --- |
| 0 | The corresponding digital input/output is not configured as input. |
| 1 | The corresponding digital input/output is configured as input and can be used. |
