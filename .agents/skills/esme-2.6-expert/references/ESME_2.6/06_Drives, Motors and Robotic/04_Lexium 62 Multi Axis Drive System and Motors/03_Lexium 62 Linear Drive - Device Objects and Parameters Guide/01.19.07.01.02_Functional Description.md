# Functional Description

Functional Description

The parameter represents the value range of the machine encoder in revolutions on the encoder shaft. To calculate the encoder range on the load side the parameters [EncoderGearIn](../EncoderGearIn/EncoderGearIn.htm#XREF_D_SE_0089983_1) and [EncoderGearOut](../EncoderGearOut/EncoderGearOut.htm#XREF_D_SE_0089985_1) of the machine encoder have to be taken into account.
