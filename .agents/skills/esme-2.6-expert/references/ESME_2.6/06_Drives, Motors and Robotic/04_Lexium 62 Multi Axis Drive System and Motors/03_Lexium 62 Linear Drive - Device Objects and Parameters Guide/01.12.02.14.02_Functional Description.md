# Functional Description

Functional Description

The parameter is used to represent the condition of the brake. Since there is no feedback on the mechanical state of the brake, the indicated state can only be reached if the device works properly and the brake is wired appropriately.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0 / coupled | BOOL | The brake is controlled electrically in such a way that it is engaged. |
| 1 / released | BOOL | The brake is controlled electrically in such a way that it is released. |
