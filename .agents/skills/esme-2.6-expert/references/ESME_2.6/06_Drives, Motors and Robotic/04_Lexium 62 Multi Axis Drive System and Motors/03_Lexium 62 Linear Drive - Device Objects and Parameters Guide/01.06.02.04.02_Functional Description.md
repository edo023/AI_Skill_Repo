# Functional Description

Functional Description

The parameter represents the active mode.

| Value | Data type | Meaning |
| --- | --- | --- |
| virtual / 0 | DINT | Device is operated virtually. |
| real / 1 | DINT | Device found -> Device is used. |
| deactivated / 2 | DINT | Device is deactivated (see parameter [WorkingMode](Identification_2-3.htm#XREF_D_SE_0071393_1). |
