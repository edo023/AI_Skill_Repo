# Functional Description

Functional Description

The parameter represents the current value of the digital outputs. Each bit is assigned to an output corresponding to the bit number.

NOTE: Verify by querying the parameter [Active0\_7](../TouchprobeInputs/TouchprobeInputs-3.htm#XREF_D_SE_0071971_1) whether the connection value is valid.

| Bit value | Meaning |
| --- | --- |
| 0 | The corresponding digital output is set to LOW signal. |
| 1 | The corresponding digital output is set to HIGH signal. |

Assigning bits to outputs Bit0\_7 OnBoard I/O (digital outputs)

|  |  |  |
| --- | --- | --- |
| Parameter bit number | Bit 1 | Bit 0 |
| Digital output (1) of the Lexium LXM62 Drive / Lexium LXM52 Drive | DQ\_1\_1 | DQ\_0\_1 |
| (1)   Affected parameter: [Value](../TouchprobeInputs/TouchprobeInputs-9.htm#XREF_D_SE_0089318_1) | | |

Example: Bit0\_7 = 2#10000000

The value of IO\_4 is H / TRUE; IO\_5 is L / FALSE.
