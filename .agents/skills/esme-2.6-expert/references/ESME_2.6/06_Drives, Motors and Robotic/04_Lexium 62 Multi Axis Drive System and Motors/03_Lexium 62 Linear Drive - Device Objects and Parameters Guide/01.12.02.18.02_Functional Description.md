# Functional Description

Functional Description

With this parameter, you can determine if the connection to the motor is verified when the drive is activated ([ControllerEnable](../State_2/State_2-9.htm#XREF_D_SE_0071477_1) is set to TRUE).

During the verification, a current is temporarily passed that does not cause a torque. The measured current is compared to the expected current. If too much deviation is detected, then the diagnostic message [8969 Motor cable not connected](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064098_1) is triggered.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The verification is not performed. |
| on / 1 | BOOL | The verification is performed if the ControllerEnable is set to TRUE. |

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s may be realized if large amounts of data are transferred on the parameter channel.
