# Enabling the Incremental Encoder Output

Enabling the Incremental Encoder Output

| Step | Action |
| --- | --- |
| 1 | Double-click the device in the Devices tree. |
| 2 | In the Feature Configuration tab of the device editor, select Application defined / 0, or Advanced Plus / 3 in the Device Variant list. |
| 3 | Select the Incremental Encoder Output check box. |
