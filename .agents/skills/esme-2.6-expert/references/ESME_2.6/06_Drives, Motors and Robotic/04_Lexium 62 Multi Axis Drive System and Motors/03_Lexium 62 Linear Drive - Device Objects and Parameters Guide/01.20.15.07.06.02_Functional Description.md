# Functional Description

Functional Description

The parameter is used to enter a bit to enable or disable the diagnostic message [8788 Wiring error detected](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064065_1) with ext. diagnostic message Openload for this digital field output.

This bit in the parameter OpenloadDiagMsg is assigned to a corresponding bit in the bitmask of the parameter [OpenloadDiagMsk](DigOut-5.htm#XREF_D_SE_0071985_1). Modifications in the parameter OpenloadDiagMsg are automatically written to the corresponding bit in the parameter OpenloadDiagMsk and vice-versa. The last modification is active. Also see parameter [OverloadDiagMsg](DigOut-13.htm#XREF_D_SE_0089342_1).

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | UINT | The output does not report the diagnostic message 8788 Detected wiring error (Openload). |
| H / 1 | UINT | The output reports the diagnostic message 8788 Detected wiring error (Openload). |
