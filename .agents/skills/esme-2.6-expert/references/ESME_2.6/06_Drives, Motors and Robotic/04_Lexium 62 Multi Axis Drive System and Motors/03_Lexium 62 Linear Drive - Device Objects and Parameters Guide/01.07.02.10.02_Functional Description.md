# Functional Description

Functional Description

The parameter indicates the status of the [InverterEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=LXM62HW&topicID=D_SE_0051313_1) signal.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The lock-out tag-out sequence is enabled (Safe Torque Off). |
| on / 1 | BOOL | The lock-out tag-out sequence is disabled. |

The following figure indicates the influence of ControllerEnable, TorqueEnable, and InverterEnable on the power stage release of the servo amplifier.

![G-SE-0061390.1.gif](images/G-SE-0061390.1.gif)

NOTE: In the Lexium 52, the InverterEnable signal is determined from the two parameters STO\_A and STO\_B, according to the following formula:

InverterEnable := NOT ([STO\_A](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52LinearDrive&topicID=D_SE_0071463_1) OR [STO\_B](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52LinearDrive&topicID=D_SE_0071464_1)).
