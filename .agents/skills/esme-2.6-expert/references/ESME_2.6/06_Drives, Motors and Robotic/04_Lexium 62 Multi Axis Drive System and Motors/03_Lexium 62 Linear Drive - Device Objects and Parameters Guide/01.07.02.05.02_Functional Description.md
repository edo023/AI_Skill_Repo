# Functional Description

Functional Description

The parameter contains the language-specific plain text diagnostic message with the object type and the object name. The language is preselected using the Language parameter.

If a diagnostic message appears with a greater diagnostic class, it is indicated by the parameter. After the diagnostic message has been acknowledged (Online > Reset diagnostic message or [FC\_DiagQuit()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085067_4)), this parameter is updated.
