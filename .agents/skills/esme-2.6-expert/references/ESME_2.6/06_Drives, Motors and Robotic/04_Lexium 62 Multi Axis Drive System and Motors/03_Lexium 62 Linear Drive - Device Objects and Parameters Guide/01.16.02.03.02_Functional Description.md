# Functional Description

Functional Description

This parameter controls the energy measurement. It is used to stop and restart the energy measurement.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The energy measurement is disabled.  The energy measurement stopped and the energy values remain unmodified. |
| on / 1 | BOOL | Default value.  The energy measurement is enabled. |

If the parameter is set from off/0 to on/1, the energy values measured so far are deleted and the energy measurement is started again from zero.

If the parameter is set from on / 1 to off / 0, the energy measurement is stopped and the energy value is frozen.

When switching on the device, the energy measurement is started automatically.

NOTE: The parameter value is transferred from the master to the slave via the parameter channel to the Sercos at every access. Typically, this takes about 10 ms. However, up to 1 s can be realized due to the volume of data transferred on the parameter channel.
