# Functional Description

Functional Description

The parameter represents the name specified in the Devices tree. This name can be modified dynamically in the program.

The name is used in the following functions / parameters:

oMessage logger (names in the column Instance)

Transfer time: loading the message logger

oCam Track (parameter PositionSource)

Transfer time: parameter Enable FALSE -> TRUE

oLogical encoder (parameter InObject)

Transfer time: [FC\_SetMasterEncoder](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311_1)()

oIncremental Encoder Output (parameter InObject)

Transfer time: [FC\_SetMasterEncoder](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311_1)()

oSum master encoder input (parameter InObject)

Transfer time: [FC\_SetMasterEncoder](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311_1)()

oSum encoder input (parameter InObject)

Transfer time: [FC\_SetMasterEncoder](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311_1)()

o[FC\_GetObjInstanceLogAdrByName](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085133_1)()

oFC\_GetObjLogAdrByName()

o[FC\_GetObjNameByLogAdr](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085137_1)()

oC2C Encoder Output (parameter InObject)

Transfer time: [FC\_SetMasterEncoder](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085311_1)()

oC2C Online View tab of the C2C Master (names in the column Devices)

For the functions and parameters mentioned, use the active name at the time of execution.
