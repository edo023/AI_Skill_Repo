# Introduction

Introduction

Incremental Encoder Output is a specific [user function](../User_Function/User_Function.htm#XREF_D_SE_0090516_1) that is supported by the device variants “Advanced Plus” and “Application defined” of the Lexium LXM62 Drive / Lexium LXM62 Linear Drive.

The device provides an incremental encoder output which can be used with the motor encoder or the machine encoder. Depending on the selected operation mode, the motor encoder or the machine encoder of the drive is mirrored to the encoder output. That means, either the encoder output signal is based on the motor encoder input signal or on the machine encoder input signal. The operation mode is set with the [IncrementalEncoderOutputMode](../IncrementalEncoderOutputMode/IncrementalEncoderOutputMode.htm#XREF_D_SE_0090017_1) parameter.
