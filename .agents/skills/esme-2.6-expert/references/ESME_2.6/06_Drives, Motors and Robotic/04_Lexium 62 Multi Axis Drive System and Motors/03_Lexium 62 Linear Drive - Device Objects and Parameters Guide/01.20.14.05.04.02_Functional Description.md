# Functional Description

Functional Description

The parameter indicates which digital input is configured for this measured input (parameters [IO0\_Mode, IO1\_Mode](../IO0_IO1Mode/IO0_IO1Mode.htm#XREF_D_SE_0071946_1), [IO4\_Mode, IO5\_Mode](../IO4__IO5Mode/IO4__IO5Mode.htm#XREF_D_SE_0071962_1)).

| Value | Data type | Meaning |
| --- | --- | --- |
| -1 | DINT | Measured input not configured or Sercos bus not in phase 4. |
| 0 ... 1 | DINT | Number of the connection (IO0 ... IO1), configured for the measured input. The Sercos bus is in phase 4. |
