# Functional Description

Functional Description

The parameter represents the value of the digital inputs. Each bit is assigned to an input corresponding to the bit number.

NOTE: Verify by querying the parameter [Active0\_7](../TouchprobeInputs/TouchprobeInputs-3.htm#XREF_D_SE_0071971_1) whether the connection value is valid.

| Bit value | Meaning |
| --- | --- |
| 0 | The corresponding digital input is set to LOW signal. |
| 1 | The corresponding digital input is set to HIGH signal. |

Assigning bits to inputs Bit0\_7 OnBoard I/O (digital inputs)

|  |  |  |  |  |  |  |
| --- | --- | --- | --- | --- | --- | --- |
| Parameter bit number | Bit 5 | Bit 4 | Bit 3 | Bit 2 | Bit 1 | Bit 0 |
| Digital input (1) of the Lexium LXM62 Drive / Lexium LXM52 Drive | DI\_5\_1 | DI\_4\_1 | DI\_3\_1 | DI\_2\_1 | DI\_1\_1 | DI\_0\_1 |
| (1)   Affected parameter: [Value](../TouchprobeInputs/TouchprobeInputs-9.htm#XREF_D_SE_0089318_1) | | | | | | |

Example: Bit0\_7 = 2#10000000

The value of IO\_4 is H / TRUE; IO\_5 is L / FALSE.
