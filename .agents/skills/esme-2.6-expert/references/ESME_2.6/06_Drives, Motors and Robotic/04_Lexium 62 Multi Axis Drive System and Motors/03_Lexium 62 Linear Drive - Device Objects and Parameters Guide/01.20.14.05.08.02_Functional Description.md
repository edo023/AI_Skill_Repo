# Functional Description

Functional Description

The parameter represents the position (at [FC\_TpEdge()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085375_1)) that has been captured when the Touchprobe event occurred. The value is reset to 0.00 when FC\_TPEdge() is called again.

Resolution of the measure inputs for the TP\_Edge() function

| Device (Measure Inputs) | Resolution |
| --- | --- |
| PacDrive LMC x00C | 10 µs |
| Lexium LXM62 Drive | 10 µs |
| DIO8 module | 10 µs |

The position resolution depends on the velocity when the Touchprobe is triggered.
