# Functional Description

Functional Description

You can specify the time required to engage (couple) the brake with this parameter. The coupling time starts when the brake is switched over electrically, so that its status changes from released to engaged (coupled). At the end of the engagement time, the drive becomes torque-free. The time at which the brake shall be switched over electrically is defined with the parameter [BrakeMode](Motor_2-14.htm#XREF_D_SE_0071791_1). The status of the electrical brake control is represented by the parameter [BrakeState](Motor_2-13.htm#XREF_D_SE_0071790_1).

If you set this parameter, you must make certain that the brake reaches its rated torque within the preset coupling time.

NOTE:  If the BrakeCouplingTime is 0, then the engagement time is used from the electronic motor nameplate. With Schneider Electric motors, this engagement time is sufficient to reach the torque of the drive before the drive is switched to torque-free. The corresponding engagement times can be found in the motor manuals in the chapter Holding brake.

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s may be realized if large amounts of data are transferred on the parameter channel.
