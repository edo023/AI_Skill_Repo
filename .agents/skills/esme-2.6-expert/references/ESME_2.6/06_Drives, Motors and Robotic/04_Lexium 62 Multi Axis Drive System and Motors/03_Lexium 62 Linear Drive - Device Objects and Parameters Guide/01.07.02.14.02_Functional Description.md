# Functional Description

Functional Description

The ControllerStop  parameter is an output parameter that indicates whether the ControllerStop functionality is active. The functionality can be activated by:

oUser functions [FC\_ControllerStopSet()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085013_1) and [FC\_ControllerStopReset()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085011_1)

oParameter [ControllerStopSet](State_2-13.htm#XREF_D_SE_0071452_1)

If the stop is triggered, then the FC\_ControllerStopSet() call-up parameter [i\_rDeceleration](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085013_1) is used.
