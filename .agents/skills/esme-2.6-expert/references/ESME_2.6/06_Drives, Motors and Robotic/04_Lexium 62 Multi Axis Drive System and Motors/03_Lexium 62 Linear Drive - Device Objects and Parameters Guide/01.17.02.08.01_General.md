# General

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply |
| Traceable | No |

The parameter defines the memory address from where the additional configured master data can be written. The addresses of the master data can be determined together with the parameter [AdditionalIDNListMDTLength](SystemInternals_2-6.htm#XREF_D_SE_0071928_1).

If no additional master data are configured, the address is 0.
