# WorkingState

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos Drive,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  Logic Motion Controller Device |
| Traceable | Yes |

## Functional Description

The parameter represents the active mode.

| Value | Data type | Meaning |
| --- | --- | --- |
| virtual / 0 | DINT | Device is operated virtually. |
| real / 1 | DINT | Device found -> Device is used. |
| deactivated / 2 | DINT | Device is deactivated (see parameter WorkingMode). |
| error / 3 | DINT | Only available for devices supporting the user function Sercos Robustness. The parameter is set to this value if an error is detected for a device configured for non-default response at Sercos level (parameter SystemReaction set to the value systemReaction1). |
