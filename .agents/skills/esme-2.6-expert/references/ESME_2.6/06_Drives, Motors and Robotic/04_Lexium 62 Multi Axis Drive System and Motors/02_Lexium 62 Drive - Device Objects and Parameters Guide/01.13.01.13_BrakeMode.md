# BrakeMode

## General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | Yes |

## Functional Description

How and when the brake is controlled is defined by using BrakeMode.

| BrakeMode | Behavior |
| --- | --- |
| Connect brake at stillstand / 0 | In the event of [device reactions](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063389) AD, BD1 and BD2, the brake is engaged when the motor has come to a standstill. |
| Connect brake when coasting / 1 | In the event of [device reaction](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063389) AD, the brake is engaged immediately even if the motor is still running. In the event of device reactions BD1 and BD2, the brake is engaged when the motor has come to a standstill (default behavior). |
| Connect brake when stopping / 2 | In the event of [device reactions](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063389) AD and BD1, the brake engages immediately even if the motor is still running. In the event of device reaction BD2, the brake is engaged when the motor has come to a standstill. |
| Control brake by user program / 3 | The user program has control over the brake. The state of the brake can be modified cyclically with the function *[FC\_SercosSetUserRealTimeControlBits](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085303)*. The function *[FC\_BrakeReleaseSet](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0084961)* has no effect. |

NOTE: The integrated holding brake is not designed either to decelerate the motor and its mechanics nor as a safety-related or security device. Using the brake for any other reason than for which it is designed, that is, holding a load at standstill, will result in significant wear of the brake to the point of becoming ineffective.

Therefore you must use an external braking system with the brake modes of:

* Connect brake when coasting / 1,
* Connect brake when stopping / 2, and possibly with
* Control brake by user program / 3 if the motor is not already at standstill.

| WARNING | |
| --- | --- |
|  | DAMAGE OF THE HOLDING BRAKE  Make sure to comply with the safety-related instructions in the motor manual regarding the security constraints of the holding brake.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |
