# ConfiguredSerialNumberMotor

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Sercos Drive,  Sercos I/O |
| Traceable | – |

## Functional Description

If IdentificationMode = SerialNumberMotor / 1, the value of ConfiguredSerialNumberMotor is used for identifying the device in the Sercos network.
