# IdentificationMode

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos Drive,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  Logic Motion Controller Device |
| Traceable | Yes |

## Functional Description

The parameter represents the identification mode.

| Value | Data type | Meaning |
| --- | --- | --- |
| SerialNumberController / 0 | DINT | The device is assigned via the serial number of the controller (see parameter SerialNumber, also see Identification mode). |
| SerialNumberMotor / 1 | DINT | The device is assigned via the serial number of the motor (see parameter SerialNumberMotor, also see Identification mode).  This parameter is only applicable to drive objects. |
| TopologyAddress / 2 (default) | DINT | The device is assigned via its topological address (see parameter TopologyAddress, also see Identification mode). |
| ApplicationType / 3 | DINT | The device is assigned via the ApplicationType of the device (see parameter ApplicationType, also see Identification mode). |
| SercosAddress / 4 | DINT | The device is assigned via the Sercos address of the device (see parameter SercosAddress, also see Identification mode). |
