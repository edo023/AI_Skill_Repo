# YOffsetRefAcceleration

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | No |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Sercos Drive |
| Traceable | Yes |

## Functional Description

This parameter configures the reference acceleration in Units/sec.². This acceleration is used to reach the velocity of YOffsetRefVelocity.
