# PowerSupplyCheckSet

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Lexium LXM62 Power Supply |
| Traceable | No |

## Functional Description

Enables the power supply.

This parameter acts on the PowerSupplySet parameters together with the components diagnostic, Sercos drivers, and system. The drives assigned to the power supply wait for PowerSupplyCheck to be enabled in order to be controlled.

Disabling PowerSupplyCheck has no direct effect on the drives.

While reading, the current status of the enable request is displayed.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | Not enabled |
| on / 1 | BOOL | Enabled |
