# AutoDischarge

## General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Power Supply |
| Traceable | Yes |

## Functional Description

Enables and disables the automatic discharge of the DC bus.

| Value | Data type | Meaning |
| --- | --- | --- |
| Disable automatic discharge / 0 | BOOL | The DC bus is not discharged automatically. |
| Enable automatic discharge / 1 | BOOL | If the Sercos bus is moved to phase 0, then the DC bus is automatically discharged. The discharge is stopped after 10 s at the latest (default value). |

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s can occur if there is a lot of data transferred on the parameter channel.
