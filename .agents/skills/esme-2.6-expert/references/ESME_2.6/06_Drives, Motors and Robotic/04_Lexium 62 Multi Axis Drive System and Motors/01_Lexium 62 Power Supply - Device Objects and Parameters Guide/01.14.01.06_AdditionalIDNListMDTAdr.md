# AdditionalIDNListMDTAdr

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply |
| Traceable | No |

Defines the memory address from where the additional configured master data can be written. The addresses of the master data can be determined together with the parameter AdditionalIDNListMDTLength.

If no additional master data are configured, the address is 0.

EIO0000003553.01
