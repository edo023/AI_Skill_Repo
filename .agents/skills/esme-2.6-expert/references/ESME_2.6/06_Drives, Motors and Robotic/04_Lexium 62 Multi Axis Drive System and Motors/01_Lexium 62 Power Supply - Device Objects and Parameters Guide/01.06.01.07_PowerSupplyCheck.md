# PowerSupplyCheck

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Lexium LXM62 Power Supply |
| Traceable | No |

## Functional Description

Provides the software enabling for the power supply by the controller.

The parameter provides information on the current Release status. It is determined by the user, diagnostic, Sercos driver, and system components.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | Not enabled. |
| on / 1 | BOOL | Enabled. |
