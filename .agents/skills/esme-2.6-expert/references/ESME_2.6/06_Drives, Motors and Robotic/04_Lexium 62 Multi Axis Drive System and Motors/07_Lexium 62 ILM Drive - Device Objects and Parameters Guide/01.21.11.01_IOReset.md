# IOReset

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

This parameter affects the resetting of the PacDrive controller outputs.

| Value | Meaning |
| --- | --- |
| no reset / 0 | The outputs are not reset. |
| after download / 1 | The outputs are reset after the program has been downloaded. |
| after download and prg. Reset / 2 | The outputs are reset after a download or a reset of the program. |
| after download, prg. reset and prg. stop / 3 | The outputs are reset after a download or reset or a stop of the program. |
