# ConfiguredPowerSupply

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | No |

## Functional Description

The parameter represents the name of the power supply to which the drive is connected to its DC bus . If no power supply was assigned, the first power supply in the control configuration is assigned automatically. It is not possible to verify automatically which drive was connected to which power supply.

| NOTICE | |
| --- | --- |
|  | INCORRECT ASSIGNMENT OF THE DRIVES TO THE POWER SUPPLY  Ensure that the drive(s) physically connected to the power supply are also logically assigned to that power supply.  Failure to follow these instructions can result in equipment damage. |

NOTE: Is only assigned once when booting the controller. No Sercos reset parameter.

NOTE: The ConfiguredPowerSupply parameter of the devices LXM62DxS, Lexium 62 ILM, and LXM62DxSLinear is no reset parameter and, as of (V1.50.1.2), is taken over after phase 1.
