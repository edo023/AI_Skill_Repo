# BrakeIntegrated

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | – |

## Functional Description

The parameter indicates whether a brake is integrated.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | No internal brake |
| yes / 1 | BOOL | Brake integrated |
