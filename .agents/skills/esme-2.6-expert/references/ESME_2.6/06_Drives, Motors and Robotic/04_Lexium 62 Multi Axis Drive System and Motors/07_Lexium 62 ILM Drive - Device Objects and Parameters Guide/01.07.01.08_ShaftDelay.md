# ShaftDelay

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Sercos Drive |
| Traceable | Yes |

## Functional Description

ShaftDelay provides the delay between the drive shaft and the actual values. The delay is the result of the constant dead times, which are caused by the processing and the transfer of the signals from the drive via the Sercos.

The delay ShaftDelay is contained in the parameters Position, MechPosition, EncoderPosition, TrackingDeviation, Velocity, MechVelocity, and Current. There are additional dead times of *[CycleTime](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LMCEco&topicID=D_SE_0073362)*/2 for the velocities Velocity and MechVelocity.

For the parameter ShaftMechPosition, this dead time is compensated (see [Ref-Actual Values](D-SE-0071489.html#D-SE-0071489)).

This parameter has no meaning for asynchronous motors without encoder (in open-loop V / f control mode, *[ControlMode](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071561)* = open-loop control / 1).
