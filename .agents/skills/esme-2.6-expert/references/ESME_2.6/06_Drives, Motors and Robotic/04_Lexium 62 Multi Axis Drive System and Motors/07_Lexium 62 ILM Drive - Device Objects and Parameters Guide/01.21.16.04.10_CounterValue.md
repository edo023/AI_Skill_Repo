# CounterValue

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Is used for inquiring the current state of the counted impulses.

CounterValue is reset when changing to Sercos phase 4.

Since it may take some Sercos cycles to reset CounterValue (such as after setting CounterStart to TRUE), query the CounterState parameter before querying the CounterValue.

```
 IF IM_ILM_1.CounterState = 1 THEN      
   udiCounterValueBuffer:=IM_ILM_1.CounterValue;
END_IF;
```

Due to data transfer via the Sercos bus, the first impulse is counted in the CounterValue parameter. At the earliest, five Sercos cycles (five times CycleTime) after the impulse counter function (CounterStart parameter) has been started.

For the transfer via the Sercos bus, three Sercos cycles are needed between detecting an impulse and displaying it in the CounterValue parameter.
