# Active0\_7

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Displays which connection (input/output) can be used.

| Bit value | Meaning |
| --- | --- |
| 0 | Terminal cannot be used. |
| 1 | Terminal can be used.   * Mode (parameter IO0\_Mode...IO7\_Mode) of the input/output in accordance with the input/output group (or measuring input group). * Sercos bus in phase 4. |

## Example

Active0\_7 = 2#10000011

The connections 0, 1 and 7 can be used. (See also Active)
