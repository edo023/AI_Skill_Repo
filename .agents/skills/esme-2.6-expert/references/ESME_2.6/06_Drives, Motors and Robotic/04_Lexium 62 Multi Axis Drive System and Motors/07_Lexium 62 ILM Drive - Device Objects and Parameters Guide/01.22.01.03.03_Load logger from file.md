# Load logger from file

## General

| Step | Action |
| --- | --- |
| 1 | Right-click somewhere in the register card Logger and select the entry from the context menu.  **Result**: The dialog Open file opens. |
| 2 | Select the desired XML file there and click the button [Open].  **Result**: The file is loaded and displayed in the register card Logger. |
