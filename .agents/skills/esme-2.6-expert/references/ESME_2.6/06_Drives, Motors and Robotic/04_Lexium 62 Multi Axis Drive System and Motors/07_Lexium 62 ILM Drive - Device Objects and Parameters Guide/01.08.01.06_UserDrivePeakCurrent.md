# UserDrivePeakCurrent

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Sercos Drive |
| Traceable | Yes |

## Functional Description

This parameter is used for the limitation of the drive peak current. If a value not equal to 0 and less than the MaxDrivePeakCurrent read out from the drive is entered, then this value is used as DrivePeakCurrent during the phase up of the Sercos bus (phase 0 -> 4) and is used as the basis for limitations by UserCurrentLimit.

The magnetizing current is not limited with this parameter (more information see Current parameter). This allows for example, a servo amplifier Lexium 62 type 15 A to be replaced by a Lexium 62 type 27 A, if a replacement part is needed, and to continue to use the same current limits if the corresponding DrivePeakC of Lexium 62 type 15 A is entered as UserDrivePeakCurrent.

NOTE: A modified parameter value is transferred with a new Sercos phase up (phase 0 -> phase 4).
