# CounterEdge

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Determines whether a rising or falling edge is evaluated as an impulse. If this parameter is modified while a measurement is being taken, this value is applied within one to three Sercos cycles.

| Value | Meaning |
| --- | --- |
| negative / 0 | A falling edge is evaluated as an impulse. |
| positive / 1 | A rising edge is evaluated as an impulse. |
