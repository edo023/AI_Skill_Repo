# TorqueEnable

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | No |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Sercos Drive |
| Traceable | Yes |

## Functional Description

The parameter is used to enable the torque for the drive. If TorqueEnable is set to FALSE during the operation (AxisState > 2), then the [Reaction AD](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063389) is triggered.
