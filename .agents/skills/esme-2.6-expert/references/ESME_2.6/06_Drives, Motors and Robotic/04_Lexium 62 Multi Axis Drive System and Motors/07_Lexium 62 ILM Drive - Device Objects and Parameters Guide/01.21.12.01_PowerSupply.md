# PowerSupply

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Is used for switching over the output voltage supply.

| Value | Meaning |
| --- | --- |
| Internal / 0 | Internal voltage supply via the Lexium ILM62 Drive Module.  Overall module current across the 8 inputs/outputs max. 0.1 A. |
| External / 1 | External voltage supply.  Overall module current across the 8 inputs/outputs max. 2.0 A.  The rated current per output is 500 mA. |
