# Configuring ATV340S Parameters Using the Sercos III Service Channel

![](images/logo.png)

Configuring ATV340S Parameters Using the Sercos III Service Channel

Overview

ATV340S parameters can be accessed as P-parameters by the following calculation rule.

Calculation Rule

P-0-Index.0.Subindex

oIndex = (ModbusAddress div 100) – 30 + 2000 (with the integer division div)

oSubindex = ModbusAdress modulo 100 (last 2 digits of the ModbusAddress)

Example

Parameter IL1R (logical input states) is represented by ModbusAddress = 5202

oIndex = (5202 div 100) - 30 + 2000 = 52 - 30 + 2000 = 2022

oSubindex = 5202 modulo 100 = 02

=> Sercos III address of IL1R : P-0-2022.0.02
