# Addressing ATV340S Parameters Using the Sercos III Real-time Channel

![](images/logo.png)

Addressing ATV340S Parameters Using the Sercos III Real-time Channel

Overview

Process data can be exchanged between the controller and the drive by using the Sercos III real-time channel.

The following parameters can be mapped optionally to the Sercos III real-time channel:

| Name | Sercos III address | Category | Access | Type | Units | Range |
| --- | --- | --- | --- | --- | --- | --- |
| Logic inputs states | P-0-2022.0.2 | I/0 parameters | R | WORD (BitString16) | - | - |
| Logic outputs states | P-0-2022.0.12 | I/0 parameters | R/W | WORD (BitString16) | - | - |
| Analog input 1 standardized value | P-0-2022.0.32 | I/0 parameters | R | INT (Signed16) | 1 | -32767...32767 |
| Analog input 2 standardized value | P-0-2022.0.33 | I/0 parameters | R | INT (Signed16) | 1 | -32767...32767 |
| Analog input 3 standardized value | P-0-2022.0.34 | I/0 parameters | R | INT (Signed16) | 1 | -32767...32767 |
| Analog output 1 standardized value | P-0-2022.0.61 | I/0 parameters | R/W | INT (Signed16) | 1 | -32767...32767 |
| AI4 real application image (Max. = 8192) | P-0-2022.0.35 | I/0 parameters | R | INT (Signed16) | 1 | -32767...32767 |
