# Setting the Parameters

![](images/logo.png)

Setting the Parameters

Overview

The parameters to be set are available in the following folders of the Parameters tab:

oIdentification

oGeneral

oMotor/Mechanic

oRealtimechannel

Setting the Identification Parameters

| Parameter | Value | Description/Comment |
| --- | --- | --- |
| IdentificationMode | Depending on the required system setup. | The ATV340S supports the settings TopologyAddress and SercosAddress.  For details, refer to the description of the IdentificationMode parameter in the Device Objects and Parameters Guide of your [PacDrive LMC Eco controller](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LMCEco&topicID=D_SE_0088238_1) or [PacDrive LMC Pro/Pro2 controller](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LMCPro&topicID=D_SE_0088238_2). |
| ConfigurationVendorCode | 1 | Mandatory setting |
| ConfiguredVendorDeviceID | ATV340S | Mandatory setting |

Setting the Realtimechannel Parameters

| Parameter | Value | Description/Comment |
| --- | --- | --- |
| TelegrammType | 7 | Mandatory setting |
| ConfigurationListAT | 'S-0-0051.0.0' | Position, mandatory setting |
| ConfigurationListMDT | 'S-0-0047.0.0' | RefPosition, mandatory setting |

NOTE:

You can add 9 additional parameters to each configuration list.

oConfigurationListAT: S-0-0051.0.0 (Position) - mandatory; optional up to 9 parameters (refer to the [parameters list](ATV340S_Application_Setup-5.htm#XREF_D_SE_0099853_1))

oConfigurationListMDT: S-0-0047.0.0 (RefPosition) - mandatory; optional up to 9 parameters (refer to the [parameters list](ATV340S_Application_Setup-5.htm#XREF_D_SE_0099853_1))
