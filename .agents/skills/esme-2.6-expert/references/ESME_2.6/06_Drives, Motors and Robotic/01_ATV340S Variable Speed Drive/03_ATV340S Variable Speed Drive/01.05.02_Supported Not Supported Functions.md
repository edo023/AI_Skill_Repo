Supported / Not Supported Functions

Supported and Tested Functions for the ATV340S

oOnly the embedded Sercos III communication is supported.

No other protocols are supported by the Sercos III communication module.

oThe embedded digital and analog I/O modules are supported.

oThe additional digital and analog I/O module is supported.

oThe additional output relay module is supported.

oPacDrive LMC Eco and PacDrive LMC Pro/Pro2 controllers support ATV340S variable speed drive in open loop speed control.

oA good practice for PacDrive LMC Eco controller is:

The maximum number of servo axes and variable speed drive axes should be less or equal to the maximum number of axes of a PacDrive LMC Eco controller.

Functions Not Relevant for the ATV340S

The following basic drive functions are not used with the ATV340S:

oOpen loop speed control function for synchronous and reluctance motors

oEncoder interfaces

oClosed loop speed control

Functions Not Supported in the ATV340S

The following functions are not supported:

oPosition control mode in closed loop

oCommand mode

EIO0000004057.00
