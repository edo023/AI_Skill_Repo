# Replacing an ATV340S

![](images/logo.png)

Replacing an ATV340S

Overview

If an ATV340S needs to be replaced, the new ATV340S must have the same or higher drive firmware version and communication module firmware version as the drive to be replaced.

Firmware Update

NOTE: If a drive firmware update is needed, contact your Customer Care Center via [www.schneider-electric.com/CCC](https://www.schneider-electric.com/CCC). After the firmware of the drive has been successfully updated, the parameters FR1 and CD1 must be set to Embedded ethernet source / Ethernet embedded source.

To perform the firmware update of the communication module, use the EcoStruxure Machine Expert service tool Device Assistant. For further information, refer to the [Device Assistant User Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=ServiceTool.DeviceAssistant&topicID=D_SE_0059220_11).

Identification Mode

If the IdentificationMode [parameter](../ATV340S__Application_Setup/ATV340S_Application_Setup-3.htm#XREF_D_SE_0099851_3) is set to TopologyAddress, no address setting is required in the ATV340S.

If the IdentificationMode [parameter](../ATV340S__Application_Setup/ATV340S_Application_Setup-3.htm#XREF_D_SE_0099851_3) is set to SercosAddress , use a keypad or the SoMove software to modify the parameter of the Sercos III address in the ATV340S. (For further information, refer to the [Schneider Electric SoMove website](https://www.se.com/us/en/product-range/2714-somove/?subNodeId=12367273265en_US).)
