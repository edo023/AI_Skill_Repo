# [Sercos Com Phase] (S3PH)

[Sercos Com Phase] (S3PH)

About This Parameter

This parameter displays the actual Sercos III communication phase.

Access

This parameter is accessible via [Display] (Mon-), [Communication map] (CMM-) menu and [Ethernet Embd Diag] (MPE-) submenu.

This is a read only parameter

The parameter number for S3PH is 63301.
