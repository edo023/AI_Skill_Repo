# Control-Signal Diagnostics

![](images/logo.png)

Control-Signal Diagnostics

Introduction

On the , the [Display] (MON-), [Communication map] (CMM-) submenu can be used to display control-signal diagnostic information between the drive and the controller:

oActive command channel [Command Channel] (CMdC) it must be equal to [Embedded Ethernet] (Eth))

oValue of the control word [Cmd Register] (CMd)

oActive reference frequency channel [Ref Freq Channel] (rFCC) it must be equal to [Embedded Ethernet] (Eth))

oValue of the reference frequency [Pre-Ramp Ref Freq] (FrH)

oValue of the operating state word internal to the drive [CIA402 State Reg] (EtA)

oIn the [Ethernet Emb Diag] (MPE-) submenu: [Sercos Com Phase] (S3PH)

oIn the [Command word image] (CWI-) submenu: Control word from Embedded Ethernet channel ([Ethernet Embd cmd.] (CMD5))

oIn the [Freq. ref. word map] (rWI-) submenu: reference frequency value word from Embedded Ethernet channel ([Ethernet Embd Ref Freq] (LFR5))

Control Word Display

The [Command Channel] (CMdC) parameter indicates the active command channel. For sercos III, it must be equal to [Embedded Ethernet] (Eth).

The [Cmd Register] (CMd) parameter indicates the hexadecimal value of the control word (CMD) used to control the drive. It is derived from the drive control word S-0-0134 used to control the drive.

Reference Frequency Display

The [Ref Freq Channel] (rFCC) parameter indicates the active channel for reference frequency. For sercos III, it must be equal to [Embedded Ethernet] (Eth).

The [Ref Frequency] (LFr) parameter indicates the value (in 0.1 Hz units) of the reference frequency used to control the drive.

Operating State Word Display

The [CIA402 State Reg] (EtA) parameter gives the value of the operating state word (ETA).

This parameter is the internal state of the drive. A correspondance exists with the drive status S-0-0135.
