# [Sercos Address] (S3AD)

![](images/logo.png)

[Sercos Address] (S3AD)

About This Parameter

This parameter is used to set the Sercos III device address.

The modification of the setting value is effective when you restart the drive.

Access

This parameter is accessible via [Communication] (COM-), [Comm parameters] (CMP-) menu and [Embd Eth Config] (EtE-) submenu.

This is a read/write parameter

The parameter number for S3AD is 63300.

Possible Settings

The table presents the parameter settings:

| Settings | Code | Value | Description |
| --- | --- | --- | --- |
| [0]...[255] | (0)...(255) | 0...255 | Sercos III device address  Factory setting: 0 |
