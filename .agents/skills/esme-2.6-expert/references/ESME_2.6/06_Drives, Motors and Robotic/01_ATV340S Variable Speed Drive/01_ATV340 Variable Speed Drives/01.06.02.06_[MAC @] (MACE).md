# [MAC @] (MACE)

![](images/logo.png)

[MAC @] (MACE)

About This Parameter

This parameter displays the MAC address of the device.

Access

This parameter is accessible via [Display] (MON-) , [Communication map] (CMM-) menu and [Ethernet Emb Diag] (MPE-) submenu.

This is a read-only parameter

This parameter can also be accessed with the standard Sercos parameter S-0-1019.

Possible Settings

The table presents the parameter settings:

| Settings | Code | Value | Description |
| --- | --- | --- | --- |
| [MM-MM-MM-XX-XX-XX] | (MM-MM-MM-XX-XX-XX) | MM-MM-MM-XX-XX-XX | MAC address |
