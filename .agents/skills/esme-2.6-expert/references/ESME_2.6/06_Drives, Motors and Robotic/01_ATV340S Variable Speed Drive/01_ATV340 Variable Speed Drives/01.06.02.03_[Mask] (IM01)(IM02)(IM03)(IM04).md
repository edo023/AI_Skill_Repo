# [Mask] (IM01)(IM02)(IM03)(IM04)

![](images/logo.png)

[Mask] (IM01)(IM02)(IM03)(IM04)

About This Parameter

This parameter is used to set the default IP subnet mask.

Access

This parameter is accessible via [Communication] (COM-), [Comm parameters] (CMP-) menu and [Embd Eth Config] (EtE-) submenu.

This is a read/write parameter

The parameter number for IM01 is 64405

The parameter number for IM02 is 64406

The parameter number for IM03 is 64407

The parameter number for IM04 is 64408

Possible Settings

The table presents the parameter settings:

| Settings | Code | Value | Description |
| --- | --- | --- | --- |
| [255.255.255.0] | (255)  (255)  (255)  (0) | 0...255  0...255  0...255  0...255 | First byte of IP Mask Eth Embd.  Second byte of IP Mask Eth Embd.  Third byte of IP Mask Eth Embd.  Fourth byte of IP Mask Eth Embd  Factory setting: 255.255.255.0 |
