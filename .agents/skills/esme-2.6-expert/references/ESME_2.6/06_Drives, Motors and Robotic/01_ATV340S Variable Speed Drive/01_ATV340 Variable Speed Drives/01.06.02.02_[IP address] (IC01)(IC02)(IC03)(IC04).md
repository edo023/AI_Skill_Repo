# [IP address] (IC01)(IC02)(IC03)(IC04)

![](images/logo.png)

[IP address] (IC01)(IC02)(IC03)(IC04)

About This Parameter

This parameter is used to set the default IP address.

Access

This parameter is accessible via [Communication] (COM-) , [Comm parameters] (CMP-) menu and [Embd Eth Config] (EtE-) submenu.

This is a read/write parameter

The parameter number for IC01 is 64401

The parameter number for IC02 is 64402

The parameter number for IC03 is 64403

The parameter number for IC04 is 64404

Possible Settings

The table presents the parameter settings:

| Settings | Code | Value | Description |
| --- | --- | --- | --- |
| [0.0.0.0] | (0)  (0)  (0)  (0) | 0...255  0...255  0...255  0...255 | First byte of IP address.  Second byte of IP address.  Third byte of IP address.  Fourth byte of IP address.  Factory setting: 0.0.0.0 |
