# Functional Profiles Supported by the Drive

![](images/logo.png)

Functional Profiles Supported by the Drive

Sercos Profile

The drive only starts up following a command sequence.

The Sercos III profile is only supported by the Sercos III fieldbus.

The control word (S-0-0134) and the drive status word (S-0-0135) are specific and standardized.

The drive supports the positioning mode by using this fieldbus.

The position commands are exchanged by Position Feedback Value (S-0-0051) and Position Command Value (S-0-0047).
