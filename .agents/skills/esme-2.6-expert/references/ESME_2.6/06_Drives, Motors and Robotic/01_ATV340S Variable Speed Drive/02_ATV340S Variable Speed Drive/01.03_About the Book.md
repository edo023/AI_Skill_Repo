About the Book

Document Scope

The present User Guide provides information on Sercos and describes the process of setting up an ATV340S device with the Modicon M262 Motion Controller.

Consult the appropriate User Guides for information on hardware and installation of the Modicon M262 Motion Controller as well as other equipment such as other drives, I/O devices, and on programming the motion application and safety-related applications.
