# Common Parameters

![](images/logo.png)

Common Parameters

Overview

The parameter groups Identification and WorkingMode comprise the parameters used for Sercos and IP communication. In addition, these parameter groups contain various parameters relating to device identification.

The parameter groups are provided separately for each Sercos device added to the Sercos master.

Common Parameters

The parameters of the parameter groups Identification and WorkingMode are the same for each Sercos device.

Refer to the Sercos User Guide for a detailed description of the parameters.
