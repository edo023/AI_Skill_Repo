# Parameter Group Scaling

![](images/logo.png)

Parameter Group Scaling

Overview

The parameters of the parameter group Scaling are used to specify the ratio between motor rotation and axis movement according to the following equation:

![G-SE-0079808.1.gif-high.gif](images/G-SE-0079808.1.gif-high.gif)

The following descriptions of the individual parameters provide further details.

Parameter IncrementResolution

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | No |

With this parameter, you can specify the resolution in increments of a motor revolution. For scaling, the value of the parameter IncrementResolution is divided by the value of the parameter PositionResolution.

Parameter PositionResolution

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | No |

With this parameter, you can specify the modulo position of the axis. For scaling, the value of the parameter IncrementResolution is divided by the value of the parameter PositionResolution.

Parameter GearIn

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | No |

With this parameter, you can specify a gear factor for scaling. The value of the parameter GearIn is divided by the value of the parameter GearOut.

Parameter GearOut

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | No |

With this parameter, you can specify a gear factor for scaling. The value of the parameter GearIn is divided by the value of the parameter GearOut.
