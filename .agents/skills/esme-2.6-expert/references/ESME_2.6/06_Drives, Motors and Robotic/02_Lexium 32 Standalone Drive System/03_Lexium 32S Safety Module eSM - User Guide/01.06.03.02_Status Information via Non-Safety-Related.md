# Status Information via Non-Safety-Related Status Outputs

## Overview

| WARNING | |
| --- | --- |
|  | INCORRECT USE OF OUTPUTS  Do not use the outputs AUXOUT1 and AUXOUT2 for safety-related purposes.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

The non-safety-related status outputs AUXOUT1 and AUXOUT2 provide the status of the safety module eSM (such as the states of the inputs and outputs). The information to be made available via a status output can be selected separately for each status output.
