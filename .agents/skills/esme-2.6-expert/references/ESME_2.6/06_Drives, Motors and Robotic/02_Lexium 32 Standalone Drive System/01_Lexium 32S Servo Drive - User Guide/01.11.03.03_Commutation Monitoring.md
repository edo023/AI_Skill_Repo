# Commutation Monitoring

## Description

Commutation monitoring verifies the plausibility of acceleration and effective motor torque.

If the motor accelerates though the drive control decelerates the motor with the maximum current, an error is detected.

Deactivating commutation monitoring can cause unintended movements.

| WARNING | |
| --- | --- |
|  | UNINTENDED MOVEMENT  * Only deactivate commutation monitoring during commissioning and only for test purposes. * Verify that the commutation monitoring is active prior to putting the equipment into service.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

The parameter MON\_commutat lets you deactivate commutation monitoring.
