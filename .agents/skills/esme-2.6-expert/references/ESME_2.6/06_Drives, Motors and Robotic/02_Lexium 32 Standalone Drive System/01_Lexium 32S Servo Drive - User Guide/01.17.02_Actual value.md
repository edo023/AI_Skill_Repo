# Actual value

In control engineering, the actual value is the value of the controlled variable at a given instant (for example, actual velocity, actual torque, actual position, actual current, and so on). An actual value can be a measured value (for example, the actual position can be a value measured by an encoder) or a derived value (for example, the actual torque can be a value derived from the actual current). The actual value is an input value used by the control loops of the drive to reach the reference value. Definition as per IEC 61800-7 series and IEC 60050.
