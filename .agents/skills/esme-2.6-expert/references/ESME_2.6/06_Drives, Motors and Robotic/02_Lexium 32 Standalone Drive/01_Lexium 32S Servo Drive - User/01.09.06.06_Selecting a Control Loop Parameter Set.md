# Selecting a Control Loop Parameter Set

## Description

The active control loop parameter set is indicated via the parameter \_CTRL\_ActParSet.

The parameter CTRL\_PwrUpParSet allows you to set the control loop parameter set to be activated after powering on. Alternatively, you can set whether or not the product is to switch automatically between the two control loop parameter sets.

The parameter CTRL\_SelParSet allows you to switch between the two control loop parameter sets during operation.
