# Parameterization of Software Debouncing

## Debounce Time

Signal input debouncing comprises hardware debouncing and software debouncing.

Hardware debounce time is permanently set, see [Digital Input Signals 24 V (Hardware switching time)](Signals-CC438B2A.html#Signals-CC438B2A__DigitalInputSignals24V-CC427B7C).

When a set signal function is modified, software debouncing is reset to the factory setting on the next power cycle.

The software debounce time can be set via the following parameters.
