# Stopping Movement with Halt

## Description

With a Halt, the ongoing movement is interrupted. The movement can be resumed when the Halt is cleared.

A Halt can be triggered via a digital signal input or a fieldbus command.

In order to interrupt a movement via a signal input, you must first parameterize the signal input function “Halt”, see [Digital Signal Inputs and Digital Signal Outputs](DigitalSignalInputsAndDigitalSignal-C50B3C34.html#DigitalSignalInputsAndDigitalSignal-C50B3C34).

The following deceleration types are available:

* Deceleration via deceleration ramp
* Deceleration via torque ramp

## Setting the Type of Deceleration

The parameter LIM\_HaltReaction lets you set the type of deceleration.

## Setting the Deceleration Ramp

The deceleration ramp is set with the parameter Ramp\_v\_dec via the [Motion Profile for the Velocity](MotionProfileForTheVelocity-C69446A0.html).
