# Fast Device Replacement (FDR)

Fast Device Replacement (FDR)

You can use the PacDrive fast device replacement to replace a Sercos III slave device in the configuration of your PacDrive application on the controller.

The FDR functionalities of an ATV340S have following features:

oReading/writing the parameters of an ATV340S with the SercosDriveUtility library. For further information, refer to the [SercosDriveUtility Library Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=ATV_SDU&topicID=D_SE_0081461_9).

oUpdating the firmware of the communication module with the EcoStruxure Machine Expert service tool Device Assistant. For further information, refer to the [Device Assistant User Guide](../../../../../../api/crossBook?lang=en-US&virtualBookName=ServiceTool.DeviceAssistant&topicID=D_SE_0059220_11).
