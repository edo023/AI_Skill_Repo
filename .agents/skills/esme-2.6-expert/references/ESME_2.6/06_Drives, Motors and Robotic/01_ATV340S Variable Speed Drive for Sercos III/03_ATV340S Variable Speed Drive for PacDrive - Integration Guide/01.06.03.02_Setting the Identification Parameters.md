# Setting the Identification Parameters

Setting the Identification Parameters

| Parameter | Value | Description/Comment |
| --- | --- | --- |
| IdentificationMode | Depending on the required system setup. | The ATV340S supports the settings TopologyAddress and SercosAddress. | ConfigurationVendorCode | 1 | Mandatory setting |
| ConfiguredVendorDeviceID | ATV340S | Mandatory setting |
