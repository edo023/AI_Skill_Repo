# Adding the ATV340S to Your Application

Adding the ATV340S to Your Application

| Step | Action |
| --- | --- |
| 1 | Launch the programming software EcoStruxure Machine Expert. |
| 2 | Execute the command File > New Project and create a new project with a PacDrive LMC/PacDrive LMC Pro/Pro2 controller. |
| 3 | In the Devices tree, right-click the Sercos Master node and select Add Device from the contextual menu. |
| 4 | In the Add Device dialog box, select the Fieldbusses > SERCOS Drive and confirm with Add Device. |
| 5 | In the Devices tree, double-click the DRV\_SercDrv (SERCOS Drive) node to open its device editor (Parameters tab) where you can edit the parameters. |
