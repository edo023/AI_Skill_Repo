# Overview

Overview

Diagnostic messages created by the ATV340S communication module:

| HEX value | Description |
| --- | --- |
| 0x4003 | The requested communication phase is not valid. |
| 0x4004 | Upstream phase shift: One phase skipped (for example, CP2 -> CP4). |
| 0x4005 | Downstream phase shift: One phase skipped (for example, CP4 -> CP2). |
| 0x4006 | Phase shift without CPS flag. |
| 0x4007 | Phase shift while drive was in RUN state. |
| 0x4017 | Watchdog timeout detected: No Sercos III telegram within 500 ms. |
| 0x4001 | Sync timeout detected: No MST telegram within 65 ms. |
| 0x3 | Loss of data detected: At least two consecutive missing telegrams. |
