# Supported Function Blocks

Supported Function Blocks

The ATV340S Sercos III interface supports the following motion function blocks:

| Function block | Description |
| --- | --- |
| PLD.FB\_DriveEnable | Switching the position control of an axis on and off. |
| PDL.FB\_EndlessFeed | Moving an axis in continuous operation (rotative system). |
| PDL.FB\_Jogging | Jogging an axis. |

NOTE:

oThere is no position feedback and no position control loop available for the ATV340S.

oThe function blocks listed above can only be used to control the velocity of the connected ATV340S device.

oInputs like i\_lrStopPosition or i\_lrMaxDistance are considered but there is no relationship to the real motor position.

oOutputs regarding positions are ignored.
