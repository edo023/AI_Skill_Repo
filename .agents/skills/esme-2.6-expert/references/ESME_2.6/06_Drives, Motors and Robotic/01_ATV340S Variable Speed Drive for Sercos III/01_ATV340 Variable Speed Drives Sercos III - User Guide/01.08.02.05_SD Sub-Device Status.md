# SD: Sub-Device Status

SD: Sub-Device Status

This LED indicates the status of the adapter

| Color & status | Description |
| --- | --- |
| OFF | Sub-device is not active |
| Yellow ON | Sub-device is in state " parametrization level (PL) " |
| Green ON | Sub-device is in state " operating level (OL) " |
| Red ON | Sub-device is in state " application error (C1D) " |
