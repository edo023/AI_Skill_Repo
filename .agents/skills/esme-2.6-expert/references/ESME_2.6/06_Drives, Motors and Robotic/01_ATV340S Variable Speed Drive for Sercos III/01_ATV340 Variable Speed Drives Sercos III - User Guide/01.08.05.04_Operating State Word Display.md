# Operating State Word Display

Operating State Word Display

The [CIA402 State Reg] (EtA) parameter gives the value of the operating state word (ETA).

This parameter is the internal state of the drive. A correspondance exists with the drive status S-0-0135.

The following table provides the conversion between the [CIA204 State Reg] (ETA) and the bits of the drive status S-0-0135

| [CiA402 State Reg] (ETA) | Drive status S-0-0135 | | | | | |
| --- | --- | --- | --- | --- | --- | --- |
| bit 15 | bit 14 | bit 13 | bit 7 | bit 4 | bit 3 |
| 0x0250 | 0 | 1 | 0 | 0 | 0 | 0 |
| 0x0231 | 1 | 0 | 0 | X | 0 | 0 |
| 0x0233 | 1 | 0 | 0 | 0 | 0 | X |
| 0x0237 | 1 | 1 | 0 | 0 | 0 | 0 |
| 0x0337 | 1 | 1 | 0 | 0 | 1 | 0 |
| 0x0217 | 1 | 1 | 0 | X | 0 | 0 |
| 0x021F | 1 | 1 | 1 | X | 0 | 0 |
| 0x0218 | 1 | 0 | 1 | X | 0 | 0 |
