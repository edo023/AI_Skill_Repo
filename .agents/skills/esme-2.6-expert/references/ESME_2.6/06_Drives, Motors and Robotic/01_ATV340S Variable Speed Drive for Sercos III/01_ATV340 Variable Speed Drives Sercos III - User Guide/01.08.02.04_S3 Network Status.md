# S3: Network Status

S3: Network Status

This LED indicates the status of the Sercos fieldbus:

| Color & status | Description |
| --- | --- |
| OFF | No communication |
| Yellow ON | Communication phase 0 active |
| Yellow single flash | Communication phase 1 active |
| Yellow double flash | Communication phase 2 active |
| Yellow triple flash | Communication phase 3 active |
| Green ON | Communication phase 4 active |
| Blinking Green | Real-time state is " loopback " |
| Blinking Yellow/Red | Application error |
| Blinking Green/Red | MST transmission error “>S-0-1003/2” |
| Red ON | Communication error |
| Blinking Yellow | Identification function ("IdentifyDevice") |
