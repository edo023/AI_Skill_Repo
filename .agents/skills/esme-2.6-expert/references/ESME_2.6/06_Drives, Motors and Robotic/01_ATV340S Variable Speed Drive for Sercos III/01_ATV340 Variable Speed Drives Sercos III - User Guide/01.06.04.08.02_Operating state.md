# Operating state

Operating state

The following table describes the drive status depending on the status word of the drive:

| Operating State | Bit 15 | Bit 14 | Bit 13 | Bit 7 | Bit 4 | Bit 3 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 - Not ready to switch on | 0 | 0 | - | - | - | - |
| 2 - Switch on disabled | 0 | 1 | - | - | - | - |
| 3 - Ready to switch on | 1 | 0 | - | - | - | - |
| 4 - Switched on | 1 | 1 | - | - | - | - |
| 5a - Drive in operation | 1 | 1 | - | - | 0 | 1 |
| 5b - Halt active | 1 | 1 | - | - | 1 | 0 |
| 6 - Quick stop active | 1 | 1 | - | 1 | 1 | 0 |
| 7 - Fault reaction active | 1 | 1 | 1 | - | 0 | 0 |
| 8 - Fault | 1 | 0 | 1 | - | 0 | 0 |
