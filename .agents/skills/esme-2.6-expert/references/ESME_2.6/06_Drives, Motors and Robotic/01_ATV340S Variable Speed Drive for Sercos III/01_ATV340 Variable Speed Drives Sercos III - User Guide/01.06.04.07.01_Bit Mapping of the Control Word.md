# Bit Mapping of the Control Word

Bit Mapping of the Control Word

The following table describes the control word of the Sercos III Profile:

| Bit number | Description |
| --- | --- |
| 15 | Drive start (under the condition that the drive is enable, e.g. bit 14 is set to 1)  1 --> 0 = Stop. The drive stops and the power stage can remain in an activated state.  0 -> 1 = Start. The drive follows the command values. |
| 14 | Drive enable  1 --> 0 = drive disabled  0 -> 1 = drive enabled |
| 13 | Drive halt (under the condition that the bits 14 and 15 are set to 1)  1 --> 0 = halt  0 --> 1 = restart |
| 12 | Reserved |
| 11 | Reserved |
| 10 | 0 = Primary operation mode |
| 9 | Reserved (=0) |
| 8 | Reserved (=0) |
| 7-0 | Reserved |
