# DeviceVariantSet

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The Lexium MC Carrier is available for the Basic Carrier.

With the parameter DeviceVariantSet, you can select the device variant (see [DeviceVariant](DeviceVar-F359C779.html#DeviceVar-F359C779)) of the physical device that shall be used with the logical device Lexium MC Carrier. The selection of the device variant must be done before the Sercos phase-up.

By default, the parameter DeviceVariantSet is implemented with the value *Standard Carrier / 1*.

| Value | Data type | Meaning |
| --- | --- | --- |
| Standard Carrier / 1 | DINT | Selecting the standard device variant of the physically connected device. |
