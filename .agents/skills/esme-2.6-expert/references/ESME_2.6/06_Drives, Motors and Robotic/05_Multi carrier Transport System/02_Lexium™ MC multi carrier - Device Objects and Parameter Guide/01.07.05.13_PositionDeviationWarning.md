# PositionDeviationWarning

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional Description

The parameter represents the alert limit for the position deviation monitoring of the carrier in millimeters (mm). When the internal position deviation exceeds the specified value, the diagnostic message "8132 Tracking deviation limit exceeded" with diagnostic level 2 is triggered.

NOTE: For more information on the diagnostic message 8132, refer to the [PacDrive System Diagnostics User Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063868).

The default value for the parameter PositionDeviationWarning is 10 mm (0.39 in).

The carrier-specific setting PositionDeviationWarning can be enabled by the parameter [EnablePositionDeviationLimit](EnablePosDevLim-8EE76A58.html#EnablePosDevLim-8EE76A58).

NOTE: The parameter PositionDeviationWarning is not mapped to the cyclic Sercos data of the carrier.

| Unit | Data type | Meaning |
| --- | --- | --- |
| mm | LREAL | The alert limit of the position deviation monitoring for the carrier in millimeters. |
