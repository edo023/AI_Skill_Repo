# Source

## General

|  |  |
| --- | --- |
| Type | AD |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier  Lexium MC Carrier Pool  Lexium MC Segment  Lexium MC Track |
| Traceable | Yes |

## Functional description

The parameter Source indicates the logical address of the source of the active diagnostic message.

Parent objects like Lexium MC Carrier Pool or Lexium MC Track also indicate diagnostic messages from their sub-objects Lexium MC Carrier and Lexium MC Segment.

| Data type | Meaning |
| --- | --- |
| LogicalAddress | The parameter indicates the logical address of the source of the active diagnostic message. |
