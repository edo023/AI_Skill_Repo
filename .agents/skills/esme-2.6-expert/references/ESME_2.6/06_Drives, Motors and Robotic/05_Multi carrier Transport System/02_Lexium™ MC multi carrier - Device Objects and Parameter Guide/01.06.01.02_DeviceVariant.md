# DeviceVariant

## General

|  |  |
| --- | --- |
| Type | AD |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | Yes |

## Functional description

The Lexium MC Segment is available for the standard segment.

By default, the parameter DeviceVariant is implemented with the value *Standard Segment / 1* .

| Value | Data type | Meaning |
| --- | --- | --- |
| Standard Segment / 1 | DINT | The standard device variant of the physically connected device. |
