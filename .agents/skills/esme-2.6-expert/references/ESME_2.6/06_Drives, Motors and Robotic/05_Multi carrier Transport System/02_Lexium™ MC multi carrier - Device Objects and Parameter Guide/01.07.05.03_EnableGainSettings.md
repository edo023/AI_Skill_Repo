# EnableGainSettings

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter indicates whether individual gain settings for the carrier are enabled.

If the value of EnableGainSettings is set to Enable, the control loop for the carrier uses the gain parameters [PositionControlKpGain](PosCtrlKpGain-8EE90457.html#PosCtrlKpGain-8EE90457), [VelocityControlKpGain](VelCtrlKpGain-8EE97F87.html#VelCtrlKpGain-8EE97F87), [VelocityControlKiGain](VelCtrlKiGain-8EE9BAEC.html#VelCtrlKiGain-8EE9BAEC) and [CurrentFeedForwardGain](CurrFeedForwGain-8EEA3670.html#CurrFeedForwGain-8EEA3670). If the value of EnableGainSettings is set to Disable, the default configuration is used.

The default value of the parameter is Disable / 0.

| Value | Data type | Meaning |
| --- | --- | --- |
| Enable / 1 | BOOL | Individual gain settings are used. |
| Disable / 0 | BOOL | The default configuration is used. |
