# BootloaderVersion

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | No |

## Functional description

The parameter BootloaderVersion represents the version name of the bootloader of the device. The value is available as of Sercos phase 2. The bootloader version consists of the following parts: V<Generation>.<Family>.<Revision>.<Build>.

The version number part from generation and family of the bootloader must be less than or equal to the corresponding version number part of the firmware.

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if Sercos is not in phase CP2 or greater, an empty string is indicated here.

NOTE: If the device in the Devices tree is virtual (non-real), the parameter value is VIRTUAL\_DEVICE.

| Data type | Meaning |
| --- | --- |
| STRING [40] | The version name of the bootloader of the device specified in the Devices tree. |
