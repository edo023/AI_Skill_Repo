# EnableMassSettings

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter indicates whether individual mass settings for the carrier are enabled.

If the value of EnableMassSettings is set to Enable, the control loop for the carrier uses the sum of the two carrier-specific parameters [Mass](Mass-8EE80F65.html#Mass-8EE80F65) and [AdditionalMass](AddMass-8EE86E70.html#AddMass-8EE86E70) as the mass of the carrier. If the value of EnableMassSettings is set to Disable, the default configuration is used.

The default value of the parameter is Disable / 0.

| Value | Data type | Meaning |
| --- | --- | --- |
| Enable / 1 | BOOL | Individual settings for the mass of the carrier are used. |
| Disable / 0 | BOOL | The default configuration is used. |
