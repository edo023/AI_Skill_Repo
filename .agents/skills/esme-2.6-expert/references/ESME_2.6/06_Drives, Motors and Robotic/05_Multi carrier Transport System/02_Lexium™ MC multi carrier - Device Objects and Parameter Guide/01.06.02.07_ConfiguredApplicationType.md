# ConfiguredApplicationType

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | No |

## Functional description

If IdentificationMode = ApplicationType / 3, the value of ConfiguredApplicationType is used for identifying the device in the Sercos network (see [IdentificationMode](IdentMode-B49C2E8F.html#IdentMode-B49C2E8F)). It is applied in the Sercos phase change from 1 to 2.

| Data type | Meaning |
| --- | --- |
| STRING [40] | The configured (logical) application type for identifying the device in the Sercos network. |
