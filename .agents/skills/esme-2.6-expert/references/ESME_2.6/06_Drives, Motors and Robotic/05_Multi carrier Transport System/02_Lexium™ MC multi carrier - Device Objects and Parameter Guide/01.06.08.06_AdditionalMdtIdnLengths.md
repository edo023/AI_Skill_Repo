# AdditionalMdtIdnLengths

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Segment  Lexium MC Carrier |
| Traceable | No |

## Functional Description

The parameter indicates the lengths of the mapped IDNs in a string separated by semicolons.

| Unit | Data type | Meaning |
| --- | --- | --- |
| Byte | STRING [254] | List of lengths of the mapped IDNs. |

EIO0000004639.05
