# PositionDeviationLimit

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional Description

The parameter represents the error limit for the position deviation monitoring of the carrier in millimeters (mm). When the internal position deviation exceeds the specified value, the diagnostic message "8111 Shutdown due to tracking deviation" with diagnostic level 3 is triggered.

NOTE: For more information on the diagnostic message 8111, refer to the [PacDrive System Diagnostics User Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063854).

The default value for the parameter PositionDeviationLimit is 15 mm (0.59 in).

The carrier-specific setting PositionDeviationLimit can be enabled by the parameter [EnablePositionDeviationLimit](EnablePosDevLim-8EE76A58.html#EnablePosDevLim-8EE76A58).

NOTE: The parameter PositionDeviationLimit is not mapped to the cyclic Sercos data of the carrier.

| Unit | Data type | Meaning |
| --- | --- | --- |
| mm | LREAL | The error limit of the position deviation monitoring for the carrier in millimeters. |
