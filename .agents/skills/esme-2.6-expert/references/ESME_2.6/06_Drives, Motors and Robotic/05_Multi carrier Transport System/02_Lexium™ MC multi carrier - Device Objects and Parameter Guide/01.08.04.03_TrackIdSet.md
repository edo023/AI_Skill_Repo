# TrackIdSet

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Track |
| Traceable | Yes |

## Functional description

With the parameter TrackIdSet, you can set the track ID of the selected track.

The default value for this parameter is 0.

| Value | Data type | Meaning |
| --- | --- | --- |
| (0...127) | SINT | The ID of the Lexium MC Track. |
