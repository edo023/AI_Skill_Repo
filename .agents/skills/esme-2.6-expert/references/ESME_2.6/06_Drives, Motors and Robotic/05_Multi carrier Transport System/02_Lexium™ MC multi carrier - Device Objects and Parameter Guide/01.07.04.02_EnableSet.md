# EnableSet

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter enables or disables the carrier.

The value is transferred via an operational Sercos real-time communication.

| Value | Data type | Meaning |
| --- | --- | --- |
| false / 0 | BOOL | Disables the carrier. |
| true / 1 | BOOL | Enables the carrier. |
