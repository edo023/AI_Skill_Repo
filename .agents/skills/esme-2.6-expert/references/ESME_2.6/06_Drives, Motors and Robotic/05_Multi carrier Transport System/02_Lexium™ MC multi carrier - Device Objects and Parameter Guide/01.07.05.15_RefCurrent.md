# RefCurrent

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional Description

The parameter RefCurrent represents the output of the velocity controller including the feedforward path in milliamperes peak (mApeak).

The value of the parameter corresponds to the current requested from the current controller for this carrier and thus corresponds to the requested force.

NOTE: The parameter RefCurrent is mapped to the cyclic Sercos data of the carrier so that modifications are possible on the fly.

| Unit | Data type | Meaning |
| --- | --- | --- |
| mApeak | LREAL | The output of the velocity controller including the feedforward path in milliamperes peak (mApeak). |
