# MaximumCurrent

## General

|  |  |
| --- | --- |
| Type | AD |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | Yes |

## Functional Description

The parameter represents the maximum permitted bus current of the segment in milliamperes (mA).

If the value of the [ActualCurrent](ActualCurrent-8E761C60.html#ActualCurrent-8E761C60) exceeds the value of the MaximumCurrent, an internal electronic fuse trips to protect the electronics. The power stage of the segment is switched off.

| Unit | Data type | Meaning |
| --- | --- | --- |
| mA | UDINT | The maximum permitted bus current of the segment in milliamperes (mA). |
