# TrackId

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Track |
| Traceable | Yes |

## Functional description

The parameter TrackId indicates the track ID when the ID has been written to the track during Sercos phase 2.

The default value for this parameter is -1. The value -1 indicates that no track is set, for example, in Sercos phase < 2.

For more information on setting the track ID, refer to [TrackIdSet](TrackIdSet-F0303BD0.html)

| Value | Data type | Meaning |
| --- | --- | --- |
| (-1...127) | SINT | Indicates the track ID. |
