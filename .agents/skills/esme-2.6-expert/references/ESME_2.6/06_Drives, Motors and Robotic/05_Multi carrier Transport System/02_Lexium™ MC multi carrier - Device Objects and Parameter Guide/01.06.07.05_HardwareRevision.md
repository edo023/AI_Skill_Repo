# HardwareRevision

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | No |

## Functional description

The parameter HardwareRevision represents the hardware revision stored in the electronic nameplate of the device.

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if Sercos is not in phase CP2 or greater, an empty string is indicated here.

NOTE: If the device in the Devices tree is virtual (non-real), the parameter value is VIRTUAL\_DEVICE.

| Data type | Meaning |
| --- | --- |
| STRING [40] | The hardware revision stored in the electronic nameplate of the device. |
