# ConfiguredTopologyAddress

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | Yes |

## Functional description

If IdentificationMode = TopologyAddress / 2, the value of ConfiguredTopologyAddress is used for identifying the device in the Sercos network (see [IdentificationMode](IdentMode-B49C2E8F.html#IdentMode-B49C2E8F)). It is applied in the Sercos phase change from 1 to 2.

| Value range | Data type | Meaning |
| --- | --- | --- |
| (1…512) | UINT | The topological address where the device is expected inside the Sercos network. |
