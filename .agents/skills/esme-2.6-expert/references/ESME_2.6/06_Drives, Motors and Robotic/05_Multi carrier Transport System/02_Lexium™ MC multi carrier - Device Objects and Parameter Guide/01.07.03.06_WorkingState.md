# WorkingState

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter represents the active working state of the device.

| Value | Data type | Meaning |
| --- | --- | --- |
| virtual / 0 | DINT | The device is operated virtually. |
| real / 1 | DINT | The device is found: The device is used. |
| deactivated / 2 | DINT | The device is deactivated (see parameter [WorkingMode](WorkingMode-13DF77CE.html#WorkingMode-13DF77CE)). |
