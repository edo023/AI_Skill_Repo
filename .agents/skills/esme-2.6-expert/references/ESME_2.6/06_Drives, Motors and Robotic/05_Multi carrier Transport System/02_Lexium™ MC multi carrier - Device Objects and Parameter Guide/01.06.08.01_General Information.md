# General Information

With the parameters of the user function SercosCyclicData, you can define and modify the lists of additional Sercos parameters that are exchanged cyclically between the controller and the device – and vice versa.

For defining these parameters, you can use the SERCOS Cyclic Data Exchange editor that is provided as part of the EcoStruxure Machine Expert software (see [SERCOS Cyclic Data Exchange](SercCyclDataGen-166A20B9.html#SercCyclDataGen-166A20B9)).

You can also modify the parameters manually if required. The list of parameters that can be mapped for the individual devices is available via the SERCOS Cyclic Data Exchange editor.

NOTE: It is a good practice to use the SERCOS Cyclic Data Exchange editor for the parameters of the user function SercosCyclicData.

The user function SercosCyclicData contains internal system parameters. With these parameters, real-time data can be modified. Without extensive knowledge of the PacDrive system and the possible consequences of an incorrect use of these parameters, there is a substantial risk for machine damage or injury during the commissioning of the machine or thereafter. Observe all possible safety-related measures when using these parameters.
