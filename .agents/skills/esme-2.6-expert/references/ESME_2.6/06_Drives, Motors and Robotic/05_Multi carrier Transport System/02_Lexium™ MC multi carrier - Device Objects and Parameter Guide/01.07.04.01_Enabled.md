# Enabled

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter indicates whether the carrier is in position control and follows the specified reference values.

The precondition for this functionality is an operational Sercos real-time communication.

| Value | Data type | Meaning |
| --- | --- | --- |
| true / 1 | BOOL | The carrier is in position control and follows the specified reference values. |
| false / 0 | BOOL | The carrier is not in position control. |
