## Functional description

The parameter indicates the position correction value of the carrier.

This optional parameter provides an adjustment value applied to the physically measured position. This is useful when mechanical tolerances, calibration offsets, or alignment deviations cause the measured carrier position to differ from the expected position. The carrier reference position and the application-level position values remain unchanged and no other parameters or reference data are modified.

NOTE: Ensure that the assigned correction value is mapped to the intended physical carrier after the Sercos phase-up. For more information on the assignment of physical carriers, refer to the [Assignment of Carrier ID](../../../../../api/crossBook?lang=en-US&virtualBookName=MLSLib&topicID=IntroMC_AssignCarrID_10C5290E) in the Multicarrier library.

| NOTICE | |

The default value for the PositionCorrection parameter is 0.0.
