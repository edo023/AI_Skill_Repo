# InverterEnable

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | Yes |

## Functional description

The parameter indicates whether the inverter is enabled.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The inverter is disabled. |
| on / 1 | BOOL | The inverter is enabled. Current can flow in the coils and the carrier can be controlled. |
