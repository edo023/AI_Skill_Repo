# IdentificationMode

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | Yes |

## Functional description

The parameter sets the Sercos device identification mode. It is applied in the Sercos phase change from 2 to 3.

| Value | Data type | Meaning |
| --- | --- | --- |
| SerialNumberController / 0 | DINT | The device is assigned via the serial number of the segment (see parameter [SerialNumber](SerialNum-B50E5480.html#SerialNum-B50E5480)). |
| SerialNumberMotor / 1 | DINT | The device is assigned via the serial number of the motor.  This parameter is only applicable to drive objects; it is not applicable to Lexium MC Segment. |
| TopologyAddress / 2 (default) | DINT | The device is assigned via its topological address (see parameter [TopologyAddress](TopolAddress-B4AB985D.html#TopolAddress-B4AB985D)). |
| ApplicationType / 3 | DINT | The device is assigned via its application type (see parameter [ApplicationType](AppType-B50CE365.html#AppType-B50CE365)). |
| SercosAddress / 4 | DINT | The device is assigned via its Sercos address (see parameter [SercosAddress](SercAddress-B51D727D.html#SercAddress-B51D727D)). |
