## Functional Description

The parameter defines a list of Sercos IDNs that are transmitted additionally into the real-time channel from the PacDrive controller to the device. The parameters are inserted into the real-time channel during the next Sercos phase-up.

The parameters to be configured additionally into the Sercos MDT (Master Data Telegram) cyclic data are defined by a list of the corresponding IDNs. These IDNs are written in a string separated by semicolon in the desired sequence.

The default value is an empty string (''). There are no additional master data configured in the real-time channel.

Also refer to [SERCOS Cyclic Data Exchange](SercCyclDataGen-166A20B9.html#SercCyclDataGen-166A20B9).
