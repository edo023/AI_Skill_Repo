# EnablePositionDeviationLimit

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter indicates whether individual position deviation limits for the carrier are enabled.

If the value of EnablePositionDeviationLimit is set to Enable, the internal monitoring of the position deviation uses the carrier-specific limits [PositionDeviationWarning](PosDevWarning-8EED3E4E.html#PosDevWarning-8EED3E4E) and [PositionDeviationLimit](PosDevLimit-8EED9287.html#PosDevLimit-8EED9287). If the value of EnablePositionDeviationLimit is set to Disable, the default configuration is used.

The default value of the parameter is Disable / 0.

| Value | Data type | Meaning |
| --- | --- | --- |
| Enable / 1 | BOOL | Individual position deviation limits for the carrier are used. |
| Disable / 0 | BOOL | The default configuration is used. |
