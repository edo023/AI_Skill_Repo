# General Information

## Introduction

The SERCOS Cyclic Data Exchange editor is provided as part of the EcoStruxure Machine Expert software.

With the SERCOS Cyclic Data Exchange editor, you can map additional Sercos parameters that are exchanged cyclically between the controller and the device – and vice versa.

## Opening the Editor

The SERCOS Cyclic Data Exchange editor can be found in the device editor screen.

To access the device editor screen, double-click the device in the Devices tree.

## Expert Setting Configuration

The Enable expert settings option allows you to modify the list of additional parameters exchanged cyclically between the controller and the device.

The left list displays the parameters (with Name and IDN (IDentification Number)) that are exchanged in the Master Data Telegram (MDT) from the controller to the device. The right list displays the parameters exchanged in the Acknowledge Telegram (AT) from the device to the controller.

To activate the expert settings, select the check box Enable expert settings.

The following buttons are now available:

| Button | Description |
| --- | --- |
| + | Click this button to add an IDN to the list (see [Adding an IDN](#SercCyclDataGen-166A20B9__AddAnIDN-1669F696)). |
| - | Select an IDN in the list and click this button to remove it from the list. |
| Arrow upwards | Select an IDN in the list and click this button to move it up in the list. |
| Arrow downwards | Select an IDN in the list and click this button to move it down in the list. |

## Adding an IDN

To add an IDN to the MDT or AT Telegram, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Click +.  **Result**: The Select Parameters for cyclic exchange dialog box appears. |
| 2 | Select the IDN to add in the list. The list displays only the parameters that are available for the corresponding device. |
| 3 | Click OK.  **Result**:  * The IDN is added to the MDT or AT telegram. * The IDN is added to the corresponding parameter [AdditionalMdtIdns](AddMdtIdns-9D2101E2.html#AddMdtIdns-9D2101E2) or [AdditionalAtIdns](AddAtIdns-9CF74448.html#AddAtIdns-9CF74448) in the user function SercosCyclicData. * The length of the IDN (in bytes) is added to the corresponding parameter [AdditionalMdtIdnLengths](AddMdtIdnLengths-9D1DD86A.html#AddMdtIdnLengths-9D1DD86A) or [AdditionalAtIdnLengths](AddAtIdnLengths-9CF82C2D.html#AddAtIdnLengths-9CF82C2D) in the user function SercosCyclicData. |
