# Valid

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier  Lexium MC Carrier Pool  Lexium MC Segment  Lexium MC Track |
| Traceable | Yes |

## Functional description

Each user function contains the parameter Valid. This parameter indicates that the configured and connected device supports the corresponding user function.

The parameter Valid must be verified in phase 4 within the application.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | The user function is not supported by the device. |
| yes / 1 | BOOL | The user function is supported by the device. |
