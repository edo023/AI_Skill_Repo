# State

## General

|  |  |
| --- | --- |
| Type | AS |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | Yes |

## Functional description

The parameter represents the state of the segment.

The value is transferred via an operational Sercos real-time communication.

| Value | Data type | Meaning |
| --- | --- | --- |
| Not ready / 0 | DINT | The segment is not ready to be switched on. |
| Ready for main power on / 1 | DINT | The power stage is not enabled. |
| Ready and main power applied, ready for enable / 2 | DINT | The power stage is enabled and the segment waits for enabling. |
| Enabled / 3 | DINT | The segment is enabled. |
