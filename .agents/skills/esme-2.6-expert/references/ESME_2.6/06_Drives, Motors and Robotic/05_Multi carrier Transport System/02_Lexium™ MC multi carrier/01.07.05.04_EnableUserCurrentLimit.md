# EnableUserCurrentLimit

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter indicates whether an individual, user-defined current limit for the carrier is enabled.

If the value of EnableUserCurrentLimit is set to Enable, the control loop for the carrier limits the current (and the resulting force) to the carrier-specific setting [UserCurrentLimit](UserCurrLim-8EEAD846.html#UserCurrLim-8EEAD846). If the value of EnableUserCurrentLimit is set to Disable, the default configuration is used.

The default value of the parameter is Disable / 0.

| Value | Data type | Meaning |
| --- | --- | --- |
| Enable / 1 | BOOL | An individual, user-defined current limit for the carrier is used. |
| Disable / 0 | BOOL | The default configuration is used. |
