# TrackId

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter TrackId indicates the ID of the track where the selected carrier is located.

The value is displayed after Sercos phase 2.

The default value for this parameter is -1. The value -1 indicates that no track is set, for example, in Sercos phase < 2.

After Sercos phase 4, virtual carriers are automatically assigned to the track with the lowest TrackId value.

NOTE: The track ID of virtual carriers is not saved during Sercos phase-down. It must be set after Sercos phase-up with the function FC\_CarrierSetPos. For more information, refer to the [SystemInterface library](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=FC_CarrierSetPos_Gen_DFCBA029).
