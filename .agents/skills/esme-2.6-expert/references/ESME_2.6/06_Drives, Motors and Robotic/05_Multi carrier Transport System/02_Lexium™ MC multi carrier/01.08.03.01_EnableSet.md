# EnableSet

## General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Track |
| Traceable | Yes |

## Functional description

The parameter enables or disables the track:

* The parameter enables or disables the Lexium MC Carrier objects related to the track.

  NOTE: If more than one track has been configured, the parameter EnableSet enables or disables only the Lexium MC Carrier objects for the corresponding track.
* The parameter sets or unsets (depending on the value 0 or 1) the parameter [EnableSet](EnableSet-9BEF89F3.html#EnableSet-9BEF89F3) of the Lexium MC Segment objects of the track.

The parameter EnableSet of the related devices is transferred via the Sercos real-time communication and becomes active on the physical devices as soon as the real- time communication is active. If the values are written in an earlier state of the Sercos communication, the values will automatically be transferred as soon as the Sercos real-time communication is active.

| Value | Data type | Meaning |
| --- | --- | --- |
| Disable / 0  (default) | BOOL | Disables the track. |
| Enable / 1 | BOOL | Enables the track. |
