# DeviceVariantSet

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | Yes |

## Functional description

The Lexium MC Segment is available for the standard segment.

With the parameter DeviceVariantSet, you can select the device variant (see [DeviceVariant](DeviceVar-F35933CB.html#DeviceVar-F35933CB)) of the physical device that shall be used with the logical device Lexium MC Segment. The selection of the device variant must be done before the Sercos phase-up.

By default, the parameter DeviceVariantSet is implemented with the value *Standard Segment / 1* .

| Value | Data type | Meaning |
| --- | --- | --- |
| Standard Segment / 1 | DINT | Selecting the standard device variant of the physically connected device. |
