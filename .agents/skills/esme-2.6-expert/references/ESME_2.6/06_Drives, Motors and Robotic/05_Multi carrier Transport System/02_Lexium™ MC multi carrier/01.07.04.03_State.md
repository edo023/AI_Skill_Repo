# State

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter represents the state of the carrier.

| Value | Data type | Meaning |
| --- | --- | --- |
| Not ready / 0 | DINT | The carrier is not ready to be switched on. |
| Ready for main power on / 1 | DINT | The carrier is in active communication. The power stage of the segment hosting the carrier is disabled. The carrier waits until the power stage of the segment is enabled. |
| Ready and main power applied / 2 | DINT | The power stage of the segment hosting the carrier is enabled. The carrier is ready for the position regulation to be enabled. |
| Enabled / 3 | DINT | The carrier is in position control and follows the specified reference values. |
