## Functional Description

The parameter is used to define a list of Sercos IDNs (IDentification Numbers) to be transmitted additionally into the acknowledge telegram (AT) of the Sercos real-time channel from the device to the PacDrive controller. The parameters are inserted into the real-time channel during the next Sercos phase-up.

The parameters to be configured additionally into the acknowledge telegram are defined by a list of the corresponding IDNs. These IDNs are written in a string separated by semicolon in the desired sequence.

The default value is an empty string (''). There are no additional drive data configured in the real-time channel.

Also refer to [SERCOS Cyclic Data Exchange](SercCyclDataGen-166A20B9.html#SercCyclDataGen-166A20B9).
