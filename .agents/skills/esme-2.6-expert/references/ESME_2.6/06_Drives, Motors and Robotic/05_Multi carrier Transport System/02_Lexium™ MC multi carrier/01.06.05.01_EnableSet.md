# EnableSet

## General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | Yes |

## Functional description

The parameter enables or disables the control loop. As a precondition, the power supply must be ready.

The value is transferred via an operational Sercos real-time communication.

| Value | Data type | Meaning |
| --- | --- | --- |
| disable / 0 | BOOL | Disable the control loop. |
| enable / 1 | BOOL | Enable the control loop. |
