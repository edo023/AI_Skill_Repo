# DeviceVariant

## General

|  |  |
| --- | --- |
| Type | AD |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The Lexium MC Carrier is available for the Basic Carrier.

By default, the parameter DeviceVariant is implemented with the value *Standard Carrier / 1* .

| Value | Data type | Meaning |
| --- | --- | --- |
| Standard Carrier / 1 | DINT | The standard device variant of the physically connected device. |
