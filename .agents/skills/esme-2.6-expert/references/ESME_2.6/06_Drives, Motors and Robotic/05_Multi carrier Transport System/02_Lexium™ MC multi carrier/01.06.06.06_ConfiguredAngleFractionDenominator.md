# ConfiguredAngleFractionDenominator

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | Yes |

## Functional description

The parameter indicates the configured angle fraction denominator of a full circle.

NOTE: The angle of a segment arc can be defined as a fraction (numerator/denominator). The fraction indicates the angle of the segment as a part of a full circle (360°).  
Example:  
A segment has an angle of 45° which corresponds to 1/8 of a full circle:  
AngleFractionNumerator or ConfiguredAngleFractionNumerator = 1  
AngleFractionDenominator or ConfiguredAngleFractionDenominator = 8

| Value range | Data type | Meaning |
| --- | --- | --- |
| Non-zero positive value | UDINT | The configured angle fraction denominator of a full circle. |
