# Mass

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter Mass indicates the mass of the carrier (in grams).

The default value of the parameter is 800 g (1.76 lb).

NOTE: The sum of the parameters Mass and [AdditionalMass](AddMass-8EE86E70.html#AddMass-8EE86E70) has an impact to the control-loop behavior of the carrier. It affects Kp and Ki of the velocity controller and also the feed forward behavior.

NOTE: This carrier-specific setting can be enabled by the parameter [EnableMassSettings](EnableMassSet-8EE6895B.html#EnableMassSet-8EE6895B).

NOTE: The parameter Mass is not mapped to the cyclic Sercos data of the carrier.

| Unit | Data type | Meaning |
| --- | --- | --- |
| g | UDINT | The mass of the carrier (in grams). |
