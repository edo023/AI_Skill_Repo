# General Information

The user function Diagnostic displays diagnostic data from the corresponding device.

The parameters of this user function are read-only output parameters and cannot be modified by the user.

The messages that are displayed in this user function are also added to the message logger. Parent objects like the objects Lexium MC Carrier Pool or Lexium MC Track also display the diagnostic messages of their sub-objects. This behavior also applies to the other device objects in the Devices tree.

When a diagnostic message can be acknowledged via DiagQuit, the parameters of the user function Diagnostic are reset to the default values.

The user function Diagnostic is mandatory on the devices Lexium MC Carrier, Lexium MC Carrier Pool, Lexium MC Segment and Lexium MC Track.
