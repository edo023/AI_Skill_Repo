## Functional Description

The parameter represents the measured (actual) DC bus current of the device in milliamperes (mA).

If the value of the ActualCurrent exceeds the value of the [MaximumCurrent](MaximumCurrent-8E76D5E4.html#MaximumCurrent-8E76D5E4), an internal electronic fuse trips to protect the electronics. The power stage of the segment is switched off.

The value of ActualCurrent is calculated once per Sercos cycle (see CycleTime). For more information on the CycleTime, refer to the [LMC Pro Device Objects and Parameters Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LMCPro&topicID=D_SE_0073362).

NOTE: The parameter value is calculated using the parameters that are transferred from the slave to the master via the real-time channel of the Sercos. If the Sercos bus is not in phase 4, a default value is indicated. If the Sercos bus is in phase 4 (operating phase), the parameter value is calculated and indicated.
