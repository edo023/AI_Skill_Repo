# Position

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |

## Functional Description

The parameter represents the position in units. The value of Position is calculated once per Sercos cycle (see CycleTime).  
For more information on the CycleTime, refer to the [LMC Pro Device Objects and Parameters Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LMCPro&topicID=D_SE_0073362).

NOTE: In PacDrive systems, the user coordinate system is represented in units. Depending on the setting of the FeedConstant (units/revolution or units/m), the user coordinate system is mapped to the physical dimensions. In a Lexium™ MC multi carrier transport system, one user unit is mapped to 1 mm (0.04 in) on the physical track. The setting of a FeedConstant is not supported.

Coordinate displacements with [FC\_SetposSingle()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085319) affect the parameter Position.

NOTE: The parameter value is calculated using the parameters that are transferred from the slave to the master via the real-time channel of the Sercos. If the Sercos bus is not in phase 4, a default value is indicated. If the Sercos bus is in phase 4 (operating phase), the parameter value is calculated and indicated.

| Data type | Meaning |
| --- | --- |
| LREAL | The position of the carrier in units. |
