# ConfiguredSerialNumber

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Segment |
| Traceable | No |

## Functional description

If IdentificationMode = SerialNumberController / 0, the value of ConfiguredSerialNumber is used for identifying the device in the Sercos network (see [IdentificationMode](IdentMode-B49C2E8F.html#IdentMode-B49C2E8F)). It is applied in the Sercos phase change from 1 to 2.

| Data type | Meaning |
| --- | --- |
| STRING [80] | The configured (logical) serial number for identifying the device in the Sercos network. |
