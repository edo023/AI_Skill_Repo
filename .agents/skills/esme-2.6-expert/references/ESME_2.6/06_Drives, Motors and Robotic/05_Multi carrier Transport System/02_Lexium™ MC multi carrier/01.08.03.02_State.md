# State

## General

|  |  |
| --- | --- |
| Type | AS |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Track |
| Traceable | Yes |

## Functional description

The parameter represents the state of the track: it represents the combined state of the Lexium MC Carrier and the Lexium MC Segment objects related to the track.

| Value | Data type | Meaning |
| --- | --- | --- |
| Not enabled / 0 (default) | DINT | No segment and no carrier is enabled. |
| Fully enabled / 1 | DINT | The segments and the carriers are enabled. |
| Partially enabled / 2 | DINT | At least one segment or one carrier is enabled. |
