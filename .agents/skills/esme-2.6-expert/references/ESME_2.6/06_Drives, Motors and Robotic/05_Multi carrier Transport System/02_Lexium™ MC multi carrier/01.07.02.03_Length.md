# Length

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter indicates the length of the carrier.

The parameter value is set to the value of the ConfiguredLength parameter during phase 3 of the Sercos phase-up.

The default value for this parameter is 50 mm (1.97 in).

| Unit | Data type | Meaning |
| --- | --- | --- |
| mm | LREAL | Length of the carrier. |
