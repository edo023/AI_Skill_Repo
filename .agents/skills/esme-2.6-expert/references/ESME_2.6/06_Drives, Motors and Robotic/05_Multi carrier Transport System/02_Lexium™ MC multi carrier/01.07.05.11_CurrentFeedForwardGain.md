# CurrentFeedForwardGain

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional Description

The parameter represents the feed forward gain setting of the control loop for the carrier in percent (%).

The default value of the parameter is 100 %.

NOTE: The carrier-specific setting of the CurrentFeedForwardGain can be enabled by the parameter [EnableGainSettings](EnableGainSet-8EE6C9C9.html#EnableGainSet-8EE6C9C9).

NOTE: The parameter CurrentFeedForwardGain is not mapped to the cyclic Sercos data of the carrier.

| Unit | Data type | Meaning |
| --- | --- | --- |
| % | LREAL | The feed forward gain setting of the control loop for the carrier in percent (%). |
