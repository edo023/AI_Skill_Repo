# ActualCurrent

## General

|  |  |
| --- | --- |
| Type | AF |
| Offline editable | No |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional Description

The parameter represents the filtered effective value of the current in milliamperes (mA). The current is filtered using a time constant of 0.25 milliseconds (ms).

The total current consists of two parts:

* The magnetizing current Id
* The torque-forming current Iq

The ActualCurrent parameter only displays the torque-forming current. The magnetizing current of synchronous motors usually equals zero. In this case, the following (simplified) formula applies: I = Iq.

The value of ActualCurrent is calculated once per Sercos cycle (see CycleTime). For more information on the CycleTime, refer to the [LMC Pro Device Objects and Parameters Guide](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LMCPro&topicID=D_SE_0073362).

NOTE: The parameter value is calculated using the parameters that are transferred from the slave to the master via the real-time channel of the Sercos. If the Sercos bus is not in phase 4, a default value is indicated. If the Sercos bus is in phase 4 (operating phase), the parameter value is calculated and indicated.
