# ConfiguredLength

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium MC Carrier |
| Traceable | Yes |

## Functional description

The parameter indicates the configured length of the carrier.

The default value for this parameter is 50 mm (1.97 in).

| Value range | Unit | Data type | Meaning |
| --- | --- | --- | --- |
| (0...99999) | mm | LREAL | The configured length of the carrier. |
