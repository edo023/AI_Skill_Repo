# System Limits

## Track

| Designation | Unit | Value |
| --- | --- | --- |
| Maximum track length | m (ft) | 40 (131.2)  NOTE: For longer systems, contact your local Schneider Electric service representative. |
| Maximum number of Lexium™ MC12 long stator motor segments per track | - | 136 |
| Maximum number of Lexium™ MC12 Heavy-Duty carriers per track | - | 200  NOTE: For a larger number of carriers, contact your local Schneider Electric service representative. |
| Maximum number of Lexium™ MC12 Heavy-Duty carriers per Lexium™ MC12 long stator motor segment straight | - | 5 |
| Maximum number of Lexium™ MC12 Heavy-Duty carriers per Lexium™ MC12 long stator motor segment arc | - | 2 |
| Total payload | kg (lbs) | 10 (22.05) (See note and hazard message after this table.) |
| Maximum velocity | m/s (ft/s) | 3 (9.84) |

NOTE: The maximum payload in vertical orientation is 10 kg with the center of gravity in the middle of the carrier.

| WARNING | |
| --- | --- |
|  | Unintended Equipment operation  * Reduce your payload accordingly when using curved, vertical tracks to take into account centrifugal forces. * When using vertical tracks, make sure that your payload does not exceed the magnetic force that holds the lower end of the Lexium™ MC12 Heavy-Duty carrier on the track.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |
