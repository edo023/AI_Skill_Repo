# Machine Encoder Input

## General Information

NOTE: The Machine Encoder Input object is only supported by the Advanced Plus [device variant](D-SE-0090515.html#D-SE-0090515) of the Lexium LXM62 Drive.

The Lexium LXM62 Advanced Plus provides an additional machine encoder input which can be used for position control.
