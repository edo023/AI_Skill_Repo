# Active0\_7

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | OnBoard I/O |
| Traceable | Yes |

## Functional Description

The parameter represents which digital input/output is configured as output (see parameter IO0\_Mode, IO1\_Mode).

| Bit value | Meaning |
| --- | --- |
| 0 | The corresponding digital input/output is not configured as output. |
| 1 | The corresponding digital input/output is configured as output and can be used. |

## Example

Active0\_7 = 2#10000000

The digital input/output IO\_4 is configured as output and can be used.
