# ControllerEnableSet

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | No |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Sercos Drive |
| Traceable | No |

## Functional Description

The parameter indicates whether the servo amplifier has been released or disabled. The parameter ControllerEnableSet, in addition to the function blocks *[FC\_ControllerEnableSet](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085009)()* and *[FC\_ControllerEnableReset](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085007)()*, provides a second interface for handling the [release](D-SE-0071477.html#D-SE-0071477). Both interfaces can be used in any given sequence. While reading, the status of the enable request is indicated.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | Not enabled |
| on / 1 | BOOL | Enabled |
