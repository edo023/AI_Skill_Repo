# MotorTypePlateVersion

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | No |

## Functional Description

The parameter represents the version number of the motor type plate.

NOTE: The parameter value is transferred from the slave to the master via Sercos. This means that if Sercos is not in phase 4 (operating phase), an empty string '' is indicated here. If Sercos is in phase 4, then the parameter value is transferred over Sercos.
