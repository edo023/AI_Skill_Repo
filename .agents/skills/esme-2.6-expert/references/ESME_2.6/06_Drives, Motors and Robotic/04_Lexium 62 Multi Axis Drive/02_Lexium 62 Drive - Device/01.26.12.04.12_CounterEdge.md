# CounterEdge

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | OnBoard I/O |
| Traceable | – |

## Functional Description

The parameter selects the edges for the pulse counter function of a field input.

| Value | Data type | Meaning |
| --- | --- | --- |
| negative / 0 | BOOL | The pulse counter is triggered by negative edges (level of the field input goes from high to low). |
| positive / 1 | BOOL | The pulse counter is triggered by positive edges (level of the field input goes from low to high). |
