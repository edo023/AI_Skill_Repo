# Torque Limitation User Function

## Introduction

Torque Limitation is a specific [user function](D-SE-0090516.html#D-SE-0090516) that is supported by the device variants “Standard Plus”, “Advanced Plus”, and “Application defined” of the Lexium LXM62 Drive.

The Torque Limitation user function can be used to limit the torque on the load side. The user function includes the following two functions (the functions are mutually exclusive):

* Acceleration-dependent torque limitation

  The motor current is limited depending on the present acceleration or deceleration. During acceleration/deceleration, the torque at the gearbox output is limited to the defined torque limits. This allows you to get an adjustable maximum absolute level of torque at the gearbox output.
* Mechanical overload protection

  Full torque is available at the gearbox output as long as the torque is below the adjusted torque limits. When these limits are exceeded (due to an overload), the torque transmission is interrupted, the motor is switched torque free, and a diagnostic message is triggered. With an optional filter, the detected torque can be filtered to adjust the sensibility.

Enabling and switching the desired function is done with the TorqueLimitationMode parameter. The torque limits for the two functions are set with the parameters AccelerationTorqueLimit and DecelerationTorqueLimit . The optional filter in mechanical overload protection mode is configured with the TorqueLimitationFilter parameter.

## Enabling the Torque Limitation

To enable the torque limitation, proceed as follows:

| Step | Action |
| --- | --- |
| 1 | Double-click the device in the Devices tree. |
| 2 | In the Feature Configuration tab of the device editor, select Application defined / 0, Standard Plus / 2, or Advanced Plus / 3 in the Device Variant list. |
| 3 | Select the Torque Limitation check box. |
