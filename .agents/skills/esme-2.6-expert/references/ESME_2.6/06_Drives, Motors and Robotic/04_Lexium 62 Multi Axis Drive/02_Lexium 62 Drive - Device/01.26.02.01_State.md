# State

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | OnBoard I/O |
| Traceable | Yes |

## Functional Description

The parameter represents the status of the OnBoard I/O.

| Value | Data type | Meaning |
| --- | --- | --- |
| disabled / 0 | DINT | OnBoard I/O is deactivated. |
| error / 1 | DINT | An error occurred during conversion. |
| not ready / 2 | DINT | OnBoard I/O has been activated (Sercos phase up, phases 1–3). |
| virtual / 3 | DINT | OnBoard I/O has been activated. Sercos bus is in phase 4, but the associated motor is virtual. The OnBoard I/O is not ready for operation. |
| active / 4 | DINT | OnBoard I/O has been activated. Sercos bus is in phase 4. |
