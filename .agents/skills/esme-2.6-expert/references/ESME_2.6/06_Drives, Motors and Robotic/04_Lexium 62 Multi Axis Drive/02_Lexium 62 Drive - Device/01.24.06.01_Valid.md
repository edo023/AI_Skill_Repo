# Valid

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Lexium LXM62 Drive |
| Device variants supporting the parameter | Standard Plus, Advanced Plus, Application defined |
| Traceable | Yes |

## Functional Description

This parameter indicates that the configured and connected device supports the particular user function.

The value of this parameter is only valid in online mode and if the Sercos bus is in phase 2 or greater. The value is updated on Sercos phase-up after reaching Sercos phase 2.

| Value | Data type | Meaning |
| --- | --- | --- |
| false / 0 | BOOL | Offline or Sercos phase < 2: Default value  Online and Sercos phase >= 2: User function is not supported by the device. |
| true / 1 | BOOL | User function is supported by the device. |
