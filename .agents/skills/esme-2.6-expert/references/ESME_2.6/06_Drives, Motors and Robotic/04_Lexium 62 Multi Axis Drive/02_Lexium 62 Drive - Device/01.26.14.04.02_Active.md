# Active

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | OnBoard I/O |
| Traceable | Yes |

## Functional Description

The parameter indicates which digital input/output is configured as input (see parameter IO0\_Mode, IO1\_Mode).

| Bit value | Meaning |
| --- | --- |
| 0 | The corresponding digital input/output is not configured as input. |
| 1 | The corresponding digital input/output is configured as input and can be used. |
