# UserStartingRefCurrent

## General

|  |  |
| --- | --- |
| Type | ES |
| Devices supporting the parameter | Lexium LXM62 Drive |
| Device variants supporting the parameter | Standard Plus, Advanced Plus, Application defined |
| Traceable | Yes |

## Functional Description

You can use this parameter to define a specific starting reference current in milliampere [mA] of the motor when it operates in open-loop control. With a value unequal to zero, this value is used as starting reference current instead of an internally calculated value.

The motor runs in open-loop control when the velocity is less than the MinimalOperatingVelocity. The effective starting reference current is indicated by the StartingRefCurrent [parameter](D-SE-0089872.html#D-SE-0089872).

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s may be realized if large amounts of data are transferred on the parameter channel.
