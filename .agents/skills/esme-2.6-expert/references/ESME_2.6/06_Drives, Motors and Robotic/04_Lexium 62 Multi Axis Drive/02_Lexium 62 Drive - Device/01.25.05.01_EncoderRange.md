# EncoderRange

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Machine Encoder Input |
| Traceable | - |

## Functional Description

The parameter represents the value range of the machine encoder in revolutions on the encoder shaft. To calculate the encoder range on the load side the parameters EncoderGearIn and EncoderGearOut of the machine encoder have to be taken into account.

## Example

Standard single-turn encoder 1 rotation, multi-turn encoder 4096 rotations.
