# InverterEnableFunction

InverterEnableFunction

General

|  |  |

| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | No |

Functional Description

The parameter specifies which safety-related function the [InverterEnable](State_2-10.htm#XREF_D_SE_0071480_1) input is to be used.

| Value | Data type | Meaning |
| --- | --- | --- |
| Safe Torque Off / 0 | DINT | Safe Torque Off of the axis.  The InverterEnable input can be disabled any time.  When switching off the InverterEnable, the drive becomes torque-free. No diagnostic message is triggered. |
| Safe Stop 1 / 1 | DINT | Defined safe stop of the axis.  The InverterEnable input can only be deactivated without diagnostic message if the axis is not in closed-loop control.  When switching off the InverterEnable, the drive becomes torque-free.  If the InverterEnable is switched off while the drive is in closed-loop control ([AxisState](../../../../../../api/crossBook?lang=en-US&virtualBookName=State_2-15.htm#XREF_D_SE_0071486_1) ≥ 2), then the diagnostic message [8123 Safe Torque Off incorrect](../../../PD.Diagnostic&topicID=D_SE_0063863_3) is triggered.  If the drive is not in closed-loop control (AxisState ≤ 1), then no diagnostic message is triggered.  NOTE: This safety-related function can only be achieved with an additional external protective switching device (for example, Preventa XPSAV11113). For further information, see the operating manual of the device. |

The state of the InverterEnable input is represented in the parameter [InverterEnable](State_2-10.htm#XREF_D_SE_0071480_1).

NOTE: If the InverterEnable is switched off while the drive is in closed-loop control and the motor is moving, then the motor coasts down while the drive becomes torque-free. The behavior of the brake during the coast down is determined by the parameter [BrakeMode](../Motor_2/Motor_2-14.htm#XREF_D_SE_0071791_1).
