CounterStart

General

|  |  |
| --- | --- |

| Offline editable | No |
| Devices supporting the parameter | OnBoard I/O |
| Traceable | – |

Functional Description

The parameter is used to start the pulse counter (pulse counter function) of a field input.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | Pulse counter is stopped. CounterValue is not reset until CounterStart is set to yes / 1 again. |
| yes / 1 | BOOL | Incoming pulses are counted and indicated in the object parameter CounterValue. CounterValue is reset to 0 on changing this value to yes / 1. |
