# Overview

![](images/logo.png)

Overview

General

Use

With the help of the logger, the log entries of drives can be collected and displayed at a central location and saved in the Logic Builder project at a later time. The log entries are displayed in the register card Logger of the respective drive or power supply.

oThe Logger is activated via a button ([Load logger]) on the register card Logger or via the corresponding [Entry in the context menu](Logger_2-6.htm#XREF_D_SE_0072003_1). During the loading procedure, a progress bar shows the status bar and the status information.

oThe individual log entries are [color-coded](Logger_2-3.htm#XREF_D_SE_0071999_1) according to their respective [diagnostic class](Logger_2-3.htm#XREF_D_SE_0071999_1).

oThe online help is opened via [F1] or the corresponding [entry in the context menu](Logger_2-9.htm#XREF_D_SE_0072006_1). If a log entry is selected, then the corresponding online help is displayed, based on the diagnostic number. If no log entry is selected, then the general summary of the logger is displayed in the online help.

oThe Logger file can be [saved](Logger_2-8.htm#XREF_D_SE_0072005_1) in the context menu of the file system via entries or [loaded](Logger_2-7.htm#XREF_D_SE_0072004_1) from the file system.
