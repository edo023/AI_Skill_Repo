# CurrentLimitationState

![](images/logo.png)

CurrentLimitationState

General

|  |  |
| --- | --- |
| Type | AS |
| Devices supporting the parameter | Lexium LXM62 Drive,  Lexium LXM62 Linear Drive |
| Traceable | Yes |

Functional Description

The parameter represents the state of the effective current limitation, i.e., which current limitation has the lowest threshold. The effective current limitation threshold is indicated via the [CurrentLim­itationTreshold](General_2-15.htm#XREF_D_SE_0089976_1) parameter.

| Value | Meaning |
| --- | --- |
| 0 | No value available |
| 1 | Current is limited to the drive peak current (see parameters [MaxDrivePeakCurrent](General_2-6.htm#XREF_D_SE_0071528_1) and [UserDrivePeakCurrent](General_2-7.htm#XREF_D_SE_0071529_1)). |
| 2 | Current is limited to the peak current (see [DriveStopCurrentLimit](General_2-13.htm#XREF_D_SE_0071534_1) parameter). |
| 3 | Current is limited to the reference current of the drive (see [UserCurrentLimit](General_2-8.htm#XREF_D_SE_0071530_1) parameter). |
| 4 | Current is limited dependent on the present acceleration/deceleration (see parameters [AccelerationTorqueLimit](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0089940_1) and DecelerationTorqueLimit). |
| 5 | Speed dependent current reduction.  (See the diagnostic message “[8133 Speed-dependent current reduction](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063869_3)”.) |

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s may be realized if large amounts of data are transferred on the parameter channel.
