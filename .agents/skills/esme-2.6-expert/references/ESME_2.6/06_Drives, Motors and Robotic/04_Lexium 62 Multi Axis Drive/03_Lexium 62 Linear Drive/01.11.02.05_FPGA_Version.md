FPGA\_Version

General

| Traceable | Yes |

Functional Description

The parameter represents the FPGA version of the device. If the device in the Devices tree is virtual (non-real), then the parameter value is DRIVE\_VIRTUAL.

Starting from FW version V01.34.01.00 (Drives and/or PowerSupply), the FPGA version is structured as follows: aa/bb/cccc/dd/ee

| Key | Designation | Meaning |
| --- | --- | --- |
| aa | FPGA\_HW\_Status | Defines the hardware compatibility |
| bb | FPGA\_CommonVersion | Defines the FPGA firmware version |
| cccc | FPGA\_SercosVersion | Defines the Sercos functionality version |
| dd | FPGA\_Variant | Defines the variant (in the case of different functionalities for the same device) |
| ee | FPGA\_CurrentControlVersion | Defines the version of the controller functionality |

Up to FW version V01.33.xx.00 (Drives and/or PowerSupply), the FPGA version is structured as follows: aa/bb/cccc/eeee

The variant (dd) is not yet reflected in the FPGA version.

| Key | Designation | Meaning |
| --- | --- | --- |
| aa | FPGA\_HW\_Status | Defines the hardware compatibility |
| bb | FPGA\_CommonVersion | Defines the FPGA firmware version |
| cccc | FPGA\_SercosVersion | Defines the Sercos functionality version |
| eeee | FPGA\_CurrentControlVersion | Defines the version of the controller functionality |
