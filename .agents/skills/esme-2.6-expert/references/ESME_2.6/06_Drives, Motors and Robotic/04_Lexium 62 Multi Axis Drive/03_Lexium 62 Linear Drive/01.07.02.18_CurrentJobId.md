# CurrentJobId

![](images/logo.png)

CurrentJobId

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Sercos Drive |
| Traceable | Yes |

Functional Description

The parameter represents the job number (Id) of the present or of the last job. Function blocks like [FC\_CamStart()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0084967_1) automatically generate a job number. The variable is only altered at the start of the job.
