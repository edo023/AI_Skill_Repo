Functional Description

The parameter is used to enter the bit field for activating the diagnostic message [8788 Wiring error detected](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064065_1) with ext. diagnostic message Openload for field outputs.

Each bit is assigned to an output corresponding to the bit number. The n bits in the parameter OverloadDiagMsk are assigned to the bit in the parameter [OverloadDiagMsg](DigOut-13.htm#XREF_D_SE_0089342_1) in the n outputs. Modifications in the parameter OverloadDiagMsk are automatically written to the corresponding outputs in the parameter OverloadDiagMsg and the vice-versa The last modification is active. Also see parameter [OpenloadDiagMsk](DigOut-5.htm#XREF_D_SE_0071985_1).

| Bit | Meaning |
| --- | --- |
| 0 | The output is not monitored. |
| 1 | The output is monitored and the diagnostic message is sent if monitoring with the DiagMask parameter is also activated. |
