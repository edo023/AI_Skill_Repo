Functional Description

The parameter represents the maximum drive speed at the gear output. If UserMaxVel = 0, the default value for the parameter MaxVel [units/second] is identical to the rated speed [NominalSpeedLinear](Motor_2-2.htm#XREF_D_SE_0072153_1) [m/s]. If UserMaxVel > 0, then MaxVel is set to this value.

With physical axes, MaxVel cannot be greater than the value that corresponds to the maximum motor speed MaxSpeedLinear [m/s].

If by virtual axes UserMaxVel is unequal to zero, then this value is taken into account in MaxVel. This means, the value of UserMaxVel is also taken over if it is higher than the standard value (by UserMaxVel = 0).

If a motor has MaxSpeedLinear < [NominalSpeedLinear](Motor_2-2.htm#XREF_D_SE_0072153_1) (for example, because of an inadequate encoder maximum speed), then the motor cannot be operated with the rated speed. In this case, the default value of MaxVel [units/seconds] is the velocity of MaxSpeedLinear.

NOTE: The parameter value is transferred from the slave to the master via the parameter channel of the Sercos by every access. Typically, this takes about 10 ms. However, up to 1 s can be realized if large amounts of data are transferred on the parameter channel. If the Sercos bus is in phase 0 or 1, then a default value is indicated here. If the Sercos bus is in phase 3 or 4, then the parameter value is transferred and indicated. In the Sercos phase 2, the parameter can be read through the application.

Limits

The maximum velocity of the linear motors in PacDrive 3 is limited to 12 m/s. The maximum path of the linear motors in PacDrive 3 is limited to 130 m.
