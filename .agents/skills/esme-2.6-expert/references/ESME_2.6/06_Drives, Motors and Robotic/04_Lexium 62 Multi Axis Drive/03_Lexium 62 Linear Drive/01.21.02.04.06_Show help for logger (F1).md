# Show help for logger (F1)

![](images/logo.png)

Show help for logger (F1)

Display General Help Information

| Step | Action |
| --- | --- |
| 1 | Do not select a log entry from the table.  Right-click somewhere in the register card Logger and select the entry Show help for logger from the context menu or click [F1].  Result: The general summary of the Logger is displayed in the online help. |

Display Special Help Information

| Step | Action |
| --- | --- |
| 1 | Select one log entry from the table.  Right-click somewhere in the register card Logger and select the entry Show help for logger from the context menu or click [F1].  Result: The special help information of the log entry is displayed, based on the diagnostic number. |

NOTE: If a log entry was selected, then the adjusted entry is displayed in the context menu (Show help for <Diagnostic number>).
