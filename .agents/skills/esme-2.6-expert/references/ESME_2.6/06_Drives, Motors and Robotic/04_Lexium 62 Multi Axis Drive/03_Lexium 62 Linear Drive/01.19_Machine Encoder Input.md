# Machine Encoder Input

![](images/logo.png)

Machine Encoder Input

General Information

NOTE: The Machine Encoder Input object is only supported by the Advanced Plus [device variant](../Device_Variants/Device_Variants.htm#XREF_D_SE_0090515_1) of the Lexium LXM62 Drive.

The Lexium LXM62 Advanced Plus provides an additional machine encoder input which can be used for position control.
