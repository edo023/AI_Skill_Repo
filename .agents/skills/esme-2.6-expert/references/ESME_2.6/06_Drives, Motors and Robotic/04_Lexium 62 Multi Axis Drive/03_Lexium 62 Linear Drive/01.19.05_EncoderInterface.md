# EncoderInterface

![](images/logo.png)

EncoderInterface

EncoderInterface

General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Machine Encoder Input |
| Traceable | Yes |

Functional Description

This parameter indicates the active encoder interface.

| Value | Meaning |
| --- | --- |
| 0 / no encoder | The device did not detect an encoder. No encoder interface is active. |
| 1 / SinCos with Hiperface without generic encoder typeplate | The device detected a SinCos Encoder with HIPERFACE® (High Performance Interface). |
| 2 / SinCos without Hiperface | Encoder type not allowed for Machine Encoder Input. |
| 11 / SinCos with Hiperface and generic encoder typeplate | The device detected a SinCos Encoder with HIPERFACE® (High Performance Interface) and encoder typeplate. |

NOTE: The parameter value is transferred from the slave to the master via the Sercos bus. This means that if the Sercos bus is not in phase 4 (operating phase), a default value is represented. If the Sercos bus is in phase 4, then the parameter value is transferred over the Sercos bus.
