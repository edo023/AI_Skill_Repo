# Filter

![](images/logo.png)

Filter

General

|  |  |
| --- | --- |
| Type | ED |
| Devices supporting the parameter | OnBoard I/O, DIO8 module |
| Traceable | Yes |

Functional Description

The parameter modifies the input filter time of the digital field input.

| Value | Data type | Meaning |
| --- | --- | --- |
| 1 ms / 0 | BOOL | The value of the input filter time is 1 ms. |
| 5 ms / 1 | BOOL | The value of the input filter time is 5 ms. |

NOTE: Modifications to the parameter are only applied during the Sercos phase up (communication phase 0 => communication phase 4).
