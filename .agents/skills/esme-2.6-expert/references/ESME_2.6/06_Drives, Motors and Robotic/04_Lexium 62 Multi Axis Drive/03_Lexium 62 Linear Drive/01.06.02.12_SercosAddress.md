Functional Description

The parameter represents the Sercos address, to which the device has been assigned. The Sercos address is automatically assigned by the master during the phase up of Sercos, provided that the device is not in the Identification mode SercosAddress (see parameter [IdentificationMode](Identification_2-6.htm#XREF_D_SE_0071400_1)).

The Sercos address can only be modified by manually writing the Sercos parameter IDN S-0-1040. The function FC\_SercosWriteServiceData (see parameter [FC\_SercosWriteServiceData](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085305_1)) can be used. For TM5NS31 and TM5CSLCx00FS, the Sercos address can be set by using the rotary switches on the device.
