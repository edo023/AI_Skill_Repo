# Gateway

![](images/logo.png)

Gateway

General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  C2C Slave,  Logic Motion Controller Device |
| Traceable | No |

Functional Description

With this parameter, you can define the gateway of the slave device.

| Value | Data type | Meaning |
| --- | --- | --- |
| 172.20.0.1 (default) | STRING(15) | Gateway of the slave device  Only has to be set if IPConfigMode = manual IP addressing (software) / 2.  Access type: read/write |

oWhile the value for this parameter is being entered, the parameter is verified for the following:

oProper format [a1]. [a2]. [a3]. [a4]

oThe first position ([a1]) is greater to 0 and unequal than 127 (127.x.x.x would be the local host).

oDuring the setting of the configuration, the parameter is verified for the following:

oThe gateway is within the network specified by NetID (only checked if IPConfigMode = manual IP addressing (hardware) / 1 or manual IP addressing (software) / 2)

oFor double drives: The specified value is the same for both drives (only verified if IPConfigMode for both drives = manual IP addressing (software) / 2).
