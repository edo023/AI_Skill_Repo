# SensorDelay

SensorDelay

General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | PacDrive LMC x01C  PacDrive LMC x00C,  OnBoard I/O, DIO8 module,  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

Functional Description

The parameter is used to enter the delay of an external sensor.

The controller takes into account this delay for position detection. Also refer to [FC\_TpEdge()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085375_1) (Shifting CaptureValue) in the SystemInterface Library Guide.
