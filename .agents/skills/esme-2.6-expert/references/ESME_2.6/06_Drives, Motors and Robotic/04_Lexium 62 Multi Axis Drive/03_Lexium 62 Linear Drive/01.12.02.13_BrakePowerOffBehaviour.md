# BrakePowerOffBehaviour

![](images/logo.png)

BrakePowerOffBehaviour

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive |
| Traceable | Yes |

Functional Description

The parameter represents the brake behavior in de-energized state.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0 / coupled | BOOL | The brake is engaged in the de-energized state and released in the energized state. |
| 1 / released | BOOL | The brake is released in the de-energized state and engaged in the energized state. |
