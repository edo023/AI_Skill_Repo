# CounterStart

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Is used for starting or stopping an impulse counter measurement.

| Value | Meaning |
| --- | --- |
| no / 0 | Measurement is not active. If the measuring stops, the last counter value is kept. |
| yes / 1 | Measurement active. The counter (CounterValue) is reset to zero automatically at the beginning of the measurement. |
