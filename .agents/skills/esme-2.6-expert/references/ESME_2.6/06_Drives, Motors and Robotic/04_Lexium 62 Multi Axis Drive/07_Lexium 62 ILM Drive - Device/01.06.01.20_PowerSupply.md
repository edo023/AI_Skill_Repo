# PowerSupply

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | No |

## Functional Description

The parameter represents the assigned power supply.

The represented value corresponds to the Name parameter of the assigned power supply. If the parameter is modified in PowerSupply, the modified name is indicated here as well. Thus, it is possible that ConfiguredPowerSupply and PowerSupply have different names.
