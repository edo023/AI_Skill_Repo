# CounterState

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

CounterState displays the state of the impulse counter function of the measure input.

Values of the CounterState parameter and their meaning for the measured inputs on the Lexium ILM62 Drive Module

| Value | Meaning |
| --- | --- |
| inactive / 0 | Inactive |
| active / 1 | Active |
| disabled / 4 | The pulse count function is not available because Mode is not parameterized to Fastcounter / 2. |
| not ready / 5 | The pulse count function is not available because the Sercos bus is not in phase 4. |
| virtual / 6 | The impulse counter function is not available because the Lexium ILM62 Drive Module is virtual. |
