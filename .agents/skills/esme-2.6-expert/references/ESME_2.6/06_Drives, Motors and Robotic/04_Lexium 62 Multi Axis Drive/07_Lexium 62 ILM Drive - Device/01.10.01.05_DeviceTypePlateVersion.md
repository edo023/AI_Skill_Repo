# DeviceTypePlateVersion

## General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply |
| Traceable | – |

## Functional Description

The parameter represents the version of the device type plate. If the device in the Devices tree is virtual (non-real), then the parameter value is DRIVE\_VIRTUAL.

This parameter is shown in the following format: xxxxx/yyyyy

xxxxx: Version of DeviceTypePlate on ControlBoard

yyyyy: Version of DeviceTypePlate on PowerBoard

EIO0000003547.02
