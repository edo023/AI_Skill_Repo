# Bit0\_7

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Displays the value of the connections (input/output). Each bit is assigned to an output corresponding to the bit number.

NOTE: Check by querying the parameter Active0\_7 whether the connection value is valid.

| Bit value | Meaning |
| --- | --- |
| 0 | Connection LOW |
| 1 | Connection HIGH |

Assigning bits to inputs Bit0\_7 DIO8 module (Touchprobe inputs)

|  |  |  |

| Parameter bit number | Bit 1 | Bit 0 |
| Touchprobe input (1) of the Lexium ILM62 Drive Module | TP\_1\_1 | TP\_0\_1 |

The value of connection 7 is H / TRUE; the connections 0 to 6 are L / FALSE.
