# ControllerStop

## General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Sercos Drive |
| Traceable | No |

## Functional Description

**Drive stop**

The ControllerStop  parameter is an output parameter that indicates whether the ControllerStop functionality is active. The functionality can be activated by:

* User functions *[FC\_ControllerStopSet()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085013)* and *[FC\_ControllerStopReset()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085011)*
* Parameter ControllerStopSet

If the stop is triggered, then the FC\_ControllerStopSet() call-up parameter *[i\_rDeceleration](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085013)* is used.
