# EncoderRange

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive,  Lexium LXM62 Drive,  Lexium ILM62 Drive Module |
| Traceable | – |

## Functional Description

The parameter is used to indicate the value range of the encoder in revolutions on the encoder shaft. To calculate the encoder range on the motor side, the parameters *[EncoderGearIn](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071806)* and *[EncoderGearOut](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM62Drive&topicID=D_SE_0071807)* have to be taken into account.

## Example

Standard single-turn encoder 1 rotation, multi-turn encoder 4096 rotations.
