# Active

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

The parameter indicates whether the connection (input/output) can be used.

| Bit value | Meaning |
| --- | --- |
| 0 | Terminal cannot be used. |
| 1 | Terminal can be used.   * Mode (parameter IO0\_Mode...IO7\_Mode) of the input/output in accordance with the input/output group (or as per measuring input group). * Sercos bus in phase 4. |

See also Active0\_7.
