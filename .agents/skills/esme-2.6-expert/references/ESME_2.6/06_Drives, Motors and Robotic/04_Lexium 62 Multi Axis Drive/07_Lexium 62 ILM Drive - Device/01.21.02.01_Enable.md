# Enable

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Enables or disables the processing of the DIO8 module.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The DIO8 module is not processed by the real-time process. |
| on / 1 | BOOL | The DIO8 module is activated and processed by the real-time process. |

NOTE: Modifications to the parameter are only applied during the Sercos phase up (communication phase 0 => communication phase 4).
