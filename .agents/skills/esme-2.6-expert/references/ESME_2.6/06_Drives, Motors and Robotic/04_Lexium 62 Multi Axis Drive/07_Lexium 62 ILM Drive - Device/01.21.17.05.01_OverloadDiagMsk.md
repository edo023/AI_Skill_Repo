# OverloadDiagMsk

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | OnBoard I/O |
| Traceable | Yes |

## Functional Description

Is used to enter the bit field for activating the diagnostic message `8788 Wiring error detected` with ext. diagnostic message `Overload`.

Each bit is assigned to an output corresponding to the bit number. The n bits in the parameter OverloadDiagMsk are assigned to the bit in the parameter OverloadDiagMsg in the n outputs. Modifications in the parameter OverloadDiagMsk are automatically taken over by the corresponding outputs in the parameter OverloadDiagMsg and the opposite way. The last modification is active.

Also see parameter OpenloadDiagMsk.

| Bit | Meaning |
| --- | --- |
| 0 | The output is not monitored. |
| 1 | The output is monitored and the diagnostic message is sent if monitoring with the DiagMask parameter is also activated. |

Assigning bits to outputs OverloadDiagMsk PacDrive LMC x01C
