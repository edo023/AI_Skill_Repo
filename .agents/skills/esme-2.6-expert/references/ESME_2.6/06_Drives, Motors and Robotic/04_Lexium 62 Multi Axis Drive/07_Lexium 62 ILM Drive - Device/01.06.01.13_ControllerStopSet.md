| Traceable | Yes |

## Functional Description

NOTE: The parameters ControllerStopDec, ControllerStopJerk, and ControllerStopSet are supported as of V1.35.1.5 from the Sercos Drive. As of this version the Sercos Drive also implements the interface IF\_ControllerStop.

**Emergency stop**

Using the parameter ControllerStopSet, you can specify whether an emergency stop is triggered. An emergency stop can only be triggered via this parameter.

A positive edge (FALSE -> TRUE) at this parameter triggers an emergency stop. The motion job is canceled and the axis is delayed via the parameters ControllerStopDec and ControllerStopJerk.
