# State

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Displays the status of the DIO8 module.

| Value | Meaning |
| --- | --- |
| disabled / 0 | DIO8 module is deactivated. |
| error / 1 | A detected error occurred during conversion. |
| not ready / 2 | DIO8 module has been activated; Sercos phase start-up (Phase 1–3) |
| virtual / 3 | DIO8 module has been activated; Sercos bus in phase 4, but the associated motor is virtual. The DIO8 module is not ready for operation. |
| active / 4 | DIO8 module has been activated; Sercos bus in phase 4. |
