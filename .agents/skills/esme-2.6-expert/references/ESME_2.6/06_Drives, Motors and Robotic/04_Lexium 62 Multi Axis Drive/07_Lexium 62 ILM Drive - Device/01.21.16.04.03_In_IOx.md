# In\_IOx

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Indicates which connection (input/output) is configured for this measured input (parameters IO0\_Mode ... IO7\_Mode).

| Value | Data type | Meaning |
| --- | --- | --- |
| -1 | DINT | Measured input not configured or Sercos bus not in phase 4. |
| 0–7 | DINT | Number of the connection (IO0–IO7), configured for the measured input. The Sercos bus is in phase 4. |
