# InverterEnableConfig

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium ILM62 Drive Module |
| Traceable | Yes |

## Functional Description

The function InverterEnable can be deactivated with the DIS1 optional module on the Lexium 62 ILM motor.

| Value | Data type | Meaning |
| --- | --- | --- |
| Deactivated / 0 | DINT | The function InverterEnable is not active for the axis. The optional module DIS1 has to be plugged into the Lexium 62 ILM motor. |
| Activated / 1 | DINT | The function InverterEnable is active for the axis. The optional module DIS1 has to be removed from the Lexium 62 ILM motor. |

The configuration that shall be used has to be set in the PLC configuration with the parameter InverterEnableConfig of the Lexium 62 ILM motor to avoid misconfigurations.

If the set parameter value does not match the real configuration of the Lexium 62 ILM motor, then the diagnostic message `8978 InverterEnableConfig invalid` with the extended diagnostic message `x(HW)!=y(Cfg)` is triggered. Here x and y each is the corresponding ENUM value. The drive does not switch to control mode as long as the configuration is invalid. The detected error can only be acknowledged if the set InverterEnableConfig corresponds to the real configuration.

NOTE: The parameter is ignored if a safety module is used with the drive.
