# OpenloadDiagMsk

## General

|  |  |
| --- | --- |
| Type | ES |
| Devices supporting the parameter | DIO8 module |
| Traceable | Yes |

## Functional Description

Is used to enter the bit field for activating the diagnostic message `8788 Wiring error detected` with ext. diagnostic message `Openload`.

Each bit is assigned to an output corresponding to the bit number. The n bits in the parameter OpenloadDiagMsk are assigned to the bit in the parameter OpenloadDiagMsg in the n outputs. Modifications in the parameter OpenloadDiagMsk are automatically taken over by the corresponding outputs in the parameter OpenloadDiagMsg and the opposite way. The last modification is active.

Also see parameter OverloadDiagMsk.

| Bit | Meaning |
| --- | --- |
| 0 | The output is not monitored. |
| 1 | The output is monitored and the diagnostic message is sent if monitoring with the *[DiagMask](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.BT4DIO&topicID=D_SE_0075881)* parameter is also activated. |

NOTE: By a not connected output with activated monitoring the diagnostic message `8788 Wiring error (Openload)` is sent immediately.
