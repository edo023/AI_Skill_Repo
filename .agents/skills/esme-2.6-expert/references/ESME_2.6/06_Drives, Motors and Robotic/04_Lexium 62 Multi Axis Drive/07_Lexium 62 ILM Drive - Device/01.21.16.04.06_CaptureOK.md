# CaptureOK

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  OnBoard I/O,  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

## Functional Description

The parameter represents the status of the Touchprobe event.

The value is set to yes / 1 when a valid Touchprobe event has occurred and the result has been provided in the parameter CaptureValue. The value is reset to no / 0 when *[FC\_TpEdge()](../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085375)* is called again.

| Value | Data type | Meaning |
| --- | --- | --- |
| no / 0 | BOOL | The result of the Touchprobe event has been reset. |
| yes / 1 | BOOL | The result of the Touchprobe event is provided in the parameter CaptureValue. |
