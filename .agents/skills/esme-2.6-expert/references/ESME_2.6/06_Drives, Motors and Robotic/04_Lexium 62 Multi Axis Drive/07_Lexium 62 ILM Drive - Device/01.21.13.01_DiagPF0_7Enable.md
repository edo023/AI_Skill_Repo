# DiagPF0\_7Enable

## General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | Yes |
| Devices supporting the parameter | Bus terminal BT-4/DIO1, BT-4/DIO1 TP,  DIO8 module |
| Traceable | No |

## Functional Description

Using the parameter DiagPF0\_7Enable, you can choose whether a diagnostic message is to be issued for a PowerFailO0\_O7.

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | BOOL | A diagnostic message is not triggered if there is no supply voltage connected to outputs 0 to 7. |
| H / 1 | BOOL | A diagnostic message `8787 Configuration error` is triggered if there is no supply voltage connected for the outputs 0 to 7. |
