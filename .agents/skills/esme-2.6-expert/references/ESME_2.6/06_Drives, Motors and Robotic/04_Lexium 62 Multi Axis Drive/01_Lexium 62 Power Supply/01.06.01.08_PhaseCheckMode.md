# PhaseCheckMode

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Lexium LXM62 Power Supply |
| Traceable | Yes |

## Functional Description

Sets the phase monitoring mode.

| Value | Data type | Meaning |
| --- | --- | --- |
| Phase check disabled / 0 | DINT | For the Lexium LXM62 Power Supply power supply, a three-phase infeed is required.  The following diagnostic message is switched off:   * `8961 Phase failure message` |
| Check only one phase / 1 | DINT | A single-phase infeed is necessary.  The diagnostic message for the phase check is active. |
| Check all three phases / 2 | DINT | A three-phase infeed is necessary.  The diagnostic message for the phase check is active. |

NOTE: Modifications to the parameter are only applied during the Sercos phase up (communication phase 0 => communication phase 4).
