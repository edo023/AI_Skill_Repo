# Save logger to file

## General

| Step | Action |
| --- | --- |
| 1 | Right-click somewhere in the register card Logger and select the entry from the context menu.  **Result**: The dialog Save file opens. |
| 2 | Select the storage location there, enter the file name, and click the button [Save].  **Result**: The file is saved. |

NOTE: This function is only available if log entries are already displayed in the register card Logger.
