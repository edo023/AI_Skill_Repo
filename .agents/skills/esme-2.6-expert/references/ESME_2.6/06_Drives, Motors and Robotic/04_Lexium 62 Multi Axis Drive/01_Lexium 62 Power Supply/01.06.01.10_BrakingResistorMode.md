# BrakingResistorMode

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices supporting the parameter | Lexium LXM62 Power Supply |
| Traceable | No |

## Functional Description

Is used to set the switching threshold of the braking resistor in dependency of the fed-in mains voltage. This parameter defines the maximum permissible DC bus voltage of the power supply unit.

| Value | Data type | Meaning |
| --- | --- | --- |
| 230 V mode / 0 | DINT | The braking resistor is switched on if the DC bus voltage is higher than 390 V. The overvoltage threshold lies at 410 V for the informing and at 420 V for the shutdown. This mode is only practical if the parameter MainVoltageMode is set at 230 V / 1. |
| 400 V mode /1 | DINT | The braking resistor is switched-on if the DC bus voltage is higher than 830 V. The overvoltage threshold lies at 850 V for the informing and at 860 V for the shutdown. |

If the overvoltage threshold for the informing is exceeded, the diagnostic message `8186 DC bus voltage high` is triggered.

If the overvoltage threshold for the shutdown is exceeded, the diagnostic message `8108 DC bus overvoltage` is triggered.

The 400 V mode is set by default.

The 230 V mode is necessary, if motors are operated at the DC bus, which have a limited dielectric strength or which would lose their UL approval, if they were operated with too high voltages (for example, 40 mm bought-in motors). This mode is only practical if the parameter MainsVoltageMode is set at 230 V / 1.
