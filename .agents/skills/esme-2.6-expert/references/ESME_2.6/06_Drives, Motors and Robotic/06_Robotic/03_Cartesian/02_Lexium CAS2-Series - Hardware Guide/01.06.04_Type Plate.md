|  |  |

| **4** Product order number | **12** Schneider Electric logo |
| **5** Identification number | **13** RoHS mark |
| **6** Motor type | **14** Data matrix code |
| **7** Date of manufacture | **15** Hardware version |
| **8** Weight of the axis |  |
