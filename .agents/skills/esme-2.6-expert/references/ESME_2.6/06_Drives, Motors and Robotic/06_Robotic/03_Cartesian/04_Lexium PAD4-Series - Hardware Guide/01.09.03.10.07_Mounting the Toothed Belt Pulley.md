# Mounting the Toothed Belt Pulley

Mounting the Toothed Belt Pulley

| Step | Action |
| --- | --- |
| 1 | Clean both bearing seats in the end block (3). |
| 2 | Insert the new toothed belt pulley with bearing (1) as a unit.  G-SE-0083059.1.gif-high.gif |
| 3 | Mount the circlip (2). |
| 4 | Mount the [toothed belt](ROBOTICS_Maintenance_and_Repair-18.htm#XREF_D_SE_0106425_12). |
| 5 | For PAD42BB:  [Mount the double coupling](ROBOTICS_Maintenance_and_Repair-20.htm#XREF_D_SE_0106084_5). |
| 6 | If applicable, mount the [cover strip](../ROBOTICS_Optional_Equipment/ROBOTICS_Optional_Equipment-9.htm#XREF_D_SE_0106254_1). |
| 7 | If applicable, mount the [shaft extension](../ROBOTICS_Optional_Equipment/ROBOTICS_Optional_Equipment-8.htm#XREF_D_SE_0106253_1). |
| 8 | If applicable, mount the [motor and/or gearbox](../ROBOTICS_Optional_Equipment/ROBOTICS_Optional_Equipment-4.htm#XREF_D_SE_0106085_1). |
