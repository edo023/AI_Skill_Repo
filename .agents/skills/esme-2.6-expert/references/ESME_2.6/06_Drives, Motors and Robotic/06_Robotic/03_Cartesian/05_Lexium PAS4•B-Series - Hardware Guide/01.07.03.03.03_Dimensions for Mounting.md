# Dimensions for Mounting

Dimensions for Mounting

The axis body of the Lexium PAS4•B-Series is an extruded aluminum profile.The axis can be mounted to a frame by using appropriate clamping claws or slot nuts for the T-slots at the axis body.

The T-Slots are located on:

oBoth sides of the axis body

oBottom side of the axis body

![G-SE-0047482.2.gif-high.gif](images/G-SE-0047482.2.gif-high.gif)

1   Fastening with clamping claws

2   Fastening with slot nuts

When installing the axis, take into account the tapped hole distance for the slot nuts or screws.

The following table presents the tapped hole distance and appropriate screws:

| Description | Parameter | Unit | Value | | | |
| --- | --- | --- | --- | --- | --- | --- |
| PAS41 | PAS42 | PAS43 | PAS44 |
| Tapped hole distance(1) | A | mm (in) | 54 (2.13) | 74 (2.9) | 96 (3.8) | 130 (5.1) |
| B | 20 (0.79) | 40 (1.57) | 50 (1.97) | 70 (2.76) |
| Clamping claw width | C | 68 (2.7) | 88 (3.46) | 112 (4.4) | 150 (5.9) |
| Slot nut type | – | 5 (0.197) | 5 (0.197) | 6 (0.236) | 8 (0.315) |
| Screw – ISO 4762 | – | – | M5 | M5 | M6 | M6 / M8 |
| (1) For further information, refer to the respective dimensional drawing in [Mechanical Data](../ROBOTICS_Technical_Data/ROBOTICS_Technical_Data-3.htm#XREF_D_SE_0088553_1). | | | | | | |

NOTE: For PAS42 / PAS43: The T-Slots on the bottom side of the axis body are not symmetrical.

The following table presents the maximum distances per side at medium loads for mounting the axis with clamping claws or slot nuts:

| Fastening material | Unit | Value | | | |
| --- | --- | --- | --- | --- | --- |
| PAS41 | PAS42 | PAS43 | PAS44 |
| Clamping claws | mm (in) | 400 (15.7) | 600 (23.6) | 800 (31.5) | 1000 (39) |
| Slot nuts | 400 (15.7) | 600 (23.6) | 800 (31.5) | 1000 (39) |

NOTE: The greater the load or the demands on the running accuracy, the shorter the distance that must be between the slot nuts or the clamping claws.
