# Standard Tightening Torques

Standard Tightening Torques

The following tightening torques apply to screws of property class 8.8:

| Thread | Hex key size in mm (in) | Maximum tightening torque in Nm (lbf-in) |
| --- | --- | --- |
| M2 | 1.5 (0.059) | 0.4 (3.54) |
| M2.5 | 2 (0.079) | 0.7 (6.2) |
| M3 | 2.5 (0.098) | 1.3 (11.5) |
| M4 | 3 (0.118) | 2.8 (25) |
| M5 | 4 (0.157) | 5.6 (50) |
| M6 | 5 (0.197) | 9.6 (85) |
| M8 | 6 (0.236) | 23 (204) |
| M10 | 8 (0.315) | 45 (398) |
| M12 | 10 (0.39) | 74 (655) |
