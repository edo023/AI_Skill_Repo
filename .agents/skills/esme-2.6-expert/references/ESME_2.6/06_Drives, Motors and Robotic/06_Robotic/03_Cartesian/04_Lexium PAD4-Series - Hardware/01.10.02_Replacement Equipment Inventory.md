Replacement Equipment Inventory

Overview

Keeping a stock of important components helps ensure that equipment downtime is kept to a minimum. Only exchange devices with identical types to help ensure compatibility.

Indicate the following information on the replacement equipment order, which can be found on the [logistic type plate](../ROBOTICS_System_Overview/ROBOTICS_System_Overview-5.htm#XREF_D_SE_0104490_1):

| Parameter | Example value | Position on type plate |
| --- | --- | --- |
| Axis type | PAD42BBM1000A1NAXXXL / 2 1G 0 H7 0 | First line |
| Identification number | 0073000005672 | Id.No. |
| Serial number | 1903003908 | Ser.No. |
