# Mounting the Robot

## Overview

There are two ways of proceeding for handling and mounting of the robot. Study both ways of proceeding and determine which one is appropriate for your environment.

## Installing by Forklift Truck

| WARNING | |
| --- | --- |
|  | FALLING LOADS  * Drive slowly and carefully with the forklift truck. * Do not carry out any sudden steering movements. * Exercise care when initiating height adjustments of the forklift truck loading platform.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

| Step | Action |
| --- | --- |
| 1 | Using a forklift truck, lift the robot and the pallet into the enclosure. |
| 2 | Fasten the robot to the frame using the six provided counterbores (1).  Size of the counterbore: M12  Tightening torque: 56 Nm (496 lbf-in)  Property class of the screws: 8.8 or greater, or A2-70 or greater |
| 3 | Remove the cross piece cartons, the base carton and the plastic pallet from the robot. |

For information about the disposal of the packaging, refer to [*Disposal*](D-SE-0059497.html#D-SE-0059497).

## Installing by Crane

NOTE: The motors cannot carry the weight of the robot.

| WARNING | |
| --- | --- |
|  | FALLING HEAVY LOAD  * Attach lifting lugs to the mounting plate only. * Do not attach lifting lugs to the upper arms or the motors.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

| Step | Action |
| --- | --- |
| 1 | Pull two lifting lugs to the crane eyes at the mounting plate. |
| 2 | Use the crane to carefully lift the robot into the enclosure. |
| 3 | Fasten the robot to the frame using the six provided counterbores (1).  Size of the counterbore: M12  Tightening torque: 56 Nm (496 lbf-in)  Property class of the screws: 8.8 or greater, or A2-70 or greater |
