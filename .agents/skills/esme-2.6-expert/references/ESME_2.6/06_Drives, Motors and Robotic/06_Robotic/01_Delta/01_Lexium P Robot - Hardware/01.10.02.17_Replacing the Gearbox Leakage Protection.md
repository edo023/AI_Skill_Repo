# Replacing the Gearbox Leakage Protection (Optional Equipment)

| Step | Action |
| --- | --- |
| 1 | [Remove the motor/gearbox combination from the robot housing](D-SE-0059490.html#D-SE-0059490__D-SE-0059490.3). |
| 2 | Remove the threaded pin from the gearbox flange. |
| 3 | [Mount the Gearbox Leakage Protection](D-SE-0068821.html#D-SE-0068821). |
| 4 | Connect the motor as described in [*Cabling the Motor and the Gearbox*](D-SE-0059490.html#D-SE-0059490__D-SE-0059490.7). |
| 5 | Mount the upper arm as described in [*Replacing the Upper Arms*](D-SE-0059489.html#D-SE-0059489). |
| 6 | [Calibrate the robot mechanics](D-SE-0059492.html#D-SE-0059492). |
| 7 | [Mount the motor cover](D-SE-0059490.html#D-SE-0059490__D-SE-0059490.8). |

| WARNING | |
| --- | --- |
|  | COLLISION OF COMPONENTS  Calibrate the robot mechanics after the replacement of an upper arm, a motor, or a gearbox.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |
