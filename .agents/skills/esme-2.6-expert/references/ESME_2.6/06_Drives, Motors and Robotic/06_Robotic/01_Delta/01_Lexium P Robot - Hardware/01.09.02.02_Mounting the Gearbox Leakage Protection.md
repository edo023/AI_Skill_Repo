# Mounting the Gearbox Leakage Protection

| Step | Action |
| --- | --- |
| 1 | Attach the output-flange extension to the gearbox with the eight screws (1).  Tightening torque: 14 Nm (124 lbf-in) |
| 2 | Attach the protection box for the gearbox to the gearbox with four screws (2).  Tightening torque: 4.7 Nm (42 lbf-in)    NOTE: Verify the position of the O-rings while you are attaching the protection box for the gearbox to the gearbox. The O-rings are premounted and have to be in place during assembly. |
| 3 | Attach the unit (motor, gearbox, and Gearbox Leakage Protection) in the specified orientation to the robot housing with the twelve screws (3).  Tightening torque: 4.7 Nm (42 lbf-in)    For further information about the specified orientation, refer to the respective detail drawing in [*Mechanical and Electrical Data*](D-SE-0056649.html#D-SE-0056649). |
| 4 | Clean the whole mounted unit. |
| 5 | Connect the motor as described in [*Cabling the Motor and the Gearbox*](D-SE-0059490.html#D-SE-0059490__D-SE-0059490.4). |
| 6 | Mount the upper arm as described in [*Replacing the Upper Arms*](D-SE-0059489.html#D-SE-0059489). |
| 7 | [Calibrate the robot mechanics](D-SE-0059492.html#D-SE-0059492). |
| 8 | [Mount the motor cover](D-SE-0059490.html#D-SE-0059490__D-SE-0059490.8) |

| WARNING | |
| --- | --- |
|  | COLLISION OF COMPONENTS  Calibrate the robot mechanics after the replacement of an upper arm, a motor, or a gearbox.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |
