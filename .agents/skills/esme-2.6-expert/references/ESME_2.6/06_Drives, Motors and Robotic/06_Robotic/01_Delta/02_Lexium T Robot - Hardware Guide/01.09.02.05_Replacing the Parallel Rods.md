# Replacing the Parallel Rods

| WARNING | |
| --- | --- |
|  | FALLING HEAVY LOAD  Secure in place the parallel plate and lower arm when loosening and removing the screws.  Failure to follow these instructions can result in death, serious injury, or equipment damage. |

| Step | Action |
| --- | --- |
| 1 | Remove the short rod (1). |
| 2 | Remove the long rod (2). |
| 3 | Mount the new parallel rods as described in [*Mounting the Rods for Parallel Linkage*](D-SE-0065968.html#D-SE-0065968). |
| 4 | Slowly move the robot and verify the position of the particular articulation devices. |
