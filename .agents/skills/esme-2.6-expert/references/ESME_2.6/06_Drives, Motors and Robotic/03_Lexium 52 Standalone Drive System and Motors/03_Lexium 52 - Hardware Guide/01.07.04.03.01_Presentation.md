# Presentation

Presentation

When replacing Lexium 52, be sure to observe the important safety information in the sections of the present document concerning mounting and dismounting components.

|  |
| --- |
| Danger_Color.gifDANGER |
| ELECTRIC SHOCK CAUSED BY HIGH LEAKAGE (TOUCH) VOLTAGE |
| oBefore working on the product, make sure that it is de-energized.  oAfter disconnection, do not touch connector CN1 mains connection on the Lexium 52 as it still carries hazardous voltages for approximately one second.  oOnly operate the Lexium 52 in a control cabinet that cannot be opened without the help of tools. |
| Failure to follow these instructions will result in death or serious injury. |

There are no other user-serviceable parts within the Lexium 52. Either replace the entire drive or [contact Schneider Electric](../LMC100HW__Further_information_on_manufacturer/LMC100HW_Further_information_on_manufacturer-2.htm#XREF_D_SE_0049082_1).

|  |
| --- |
| Warning_Color.gifWARNING |
| UNINTENDED EQUIPMENT OPERATION |
| oOnly use software and hardware components approved by Schneider Electric for use with this equipment.  oDo not attempt to service this equipment outside of authorized Schneider Electric service centers.  oUpdate your application program every time you change the physical hardware configuration. |
| Failure to follow these instructions can result in death, serious injury, or equipment damage. |

Use only the accessories and mounting parts specified in the documentation and no third-party devices or components that have not been expressly approved by Schneider Electric. Do not modify the equipment.

In case machine repair includes the replacement of the drive components, observe the following instructions for ESD protection in order to avoid any damage due to electrostatic discharge:

|  |
| --- |
| NOTICE |
| ELECTROSTATIC DISCHARGE |
| oDo not touch any of the electrical connections or components.  oPrevent electrostatic charges, for example, by wearing appropriate clothing.  oIf you must touch circuit boards, do so only on the edges.  oMove the circuit boards as little as possible.  oRemove existing static charge by touching a grounded, metallic surface. |
| Failure to follow these instructions can result in equipment damage. |
