# General

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply |
| Traceable | No |

Defines the memory address, from where the additional configured drives data (see [AdditionalID­NListAT](SystemInternals_2-2.htm#XREF_D_SE_0071889_1)) can be read out.

The addresses of the master data can be determined together with the parameter [AdditionalIDNListATLength](SystemInternals_2-3.htm#XREF_D_SE_0071893_1).

If no additional drives data are configured, the address is 0.
