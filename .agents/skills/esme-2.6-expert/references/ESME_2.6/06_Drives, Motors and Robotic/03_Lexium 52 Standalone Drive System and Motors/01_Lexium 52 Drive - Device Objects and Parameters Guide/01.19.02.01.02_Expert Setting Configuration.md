# Expert Setting Configuration

Expert Setting Configuration

The Enable expert settings option allows you to modify the list of Sercos IDN exchanged cyclically between the controller and the drive.

NOTE: The total IDN length of the MDT and AT telegrams is limited to 48 bytes.

To activate the expert settings, tick the check box Enable expert settings:

The following buttons are now available:

| Button | Description |
| --- | --- |
| + | Click this button to add an IDN to the list (see description here-after). |
| - | Select an IDN in the list and click this button to remove an IDN from the list. |
| Arrow upwards | Select an IDN in the list and click this button to move up the IDN in the list. |
| Arrow downwards | Select an IDN in the list and click this button to move down the IDN in the list. |

| Step | Action |
| --- | --- |
| 1 | Click +.  Result: The Select Parameters for cyclic exchange dialog box appears. |
| 2 | Select the IDN to add in the list. |
| 3 | Click Ok.  Result: The IDN is added to the MDT or AT telegram and the number of bytes used is updated (Real time channel utilization). |
