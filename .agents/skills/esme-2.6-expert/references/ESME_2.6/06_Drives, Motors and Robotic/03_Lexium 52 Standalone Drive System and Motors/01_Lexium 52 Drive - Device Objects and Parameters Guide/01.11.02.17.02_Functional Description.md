# Functional Description

Functional Description

With this parameter, the coupling time of the brake can be set. The coupling time starts, when the brake is switched over electrically, so that its status changes from released to coupled. At the end of the coupling time, the drive becomes torque-free. The time at which the brake shall be switched over electrically is defined with the parameter [BrakeMode](Motor_2-14.htm#XREF_D_SE_0071791_1). The status of the electrical brake control is displayed with the parameter [BrakeState](Motor_2-13.htm#XREF_D_SE_0071790_1).

If you set this parameter, you must make certain that the brake reaches its rated torque within the preset coupling time.

NOTE:  If the BrakeCouplingTime is 0, then the coupling time is used from the electronic motor nameplate. By Schneider Electric motors this coupling time is enough to reach the torque of the drive before the drive is switched to torque-free. The corresponding coupling times can be found in the motor manuals in the chapter Holding brake.

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s can occur if there is a lot of data transferred on the parameter channel.
