# Functional Description

Functional Description

Used to select the motor control mode.

| Value | Data type | Meaning |
| --- | --- | --- |
| closed-loop control / 0 | DINT | The servo amplifier works in closed-loop controlled operation. An encoder is required for the closed-loop controlled operation. |
| open-loop control / 1 | DINT | The servo amplifier works in the open-loop controlled operation. The controller works without an encoder (no feedback). Only asynchronous motors can be used in the open-loop controlled operation. The speed of rotation is not monitored. |
| encoderless velocity closed-loop control / 2 | DINT | The servo amplifier works in the encoderless velocity control operation. The controller works without a position sensor (any kind of encoder).  This setting has no effect for the Lexium LXM52 Drive. |

The parameter is only transferred to the drive by a Sercos phase up (phase 0 => phase 4). After modification of the parameter, a new Sercos phase up has to be performed.
