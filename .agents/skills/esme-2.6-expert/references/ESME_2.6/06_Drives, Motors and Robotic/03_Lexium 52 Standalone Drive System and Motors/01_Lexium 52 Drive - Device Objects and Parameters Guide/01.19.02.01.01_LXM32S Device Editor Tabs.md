# LXM32S Device Editor Tabs

LXM32S Device Editor Tabs

To access the device editor screen, double-click the drive node in the Devices tree.

The device editor screen of the LXM32S drive contains the followings tabs:

oSercos Cyclic Data Exchange:

oConfiguration of the Sercos address of the drive.

oConfiguration of the Operating mode of the drive.

oConfiguration of the Sercos implicit exchanges (IDN configuration of MDT and AT telegrams).

oI/O Mapping: This tab allows you to create and assign IEC variables to the IDN selected for cyclical exchanges.

oConfiguration: Configuration parameters of the drive (It is recommended to use the Sercos Cyclic Data Exchange tab to configure drive parameters).

oInformation: This tab displays general information about the device (name, description, provider, version, image).

NOTE:

The default configuration of the cyclical exchange contains four IDNs, not editable:

oMDT telegram (controller to drive):

oPosition command value: S-0-0047.0.0

oSPDSercos3Control: P-0-3025.0.80

oAT telegram (drive to controller):

oPosition feedback value: S-0-0051.0.0

oSPDSercos3Status: P-0-3025.0.81

For more information about the IDN implemented in the LXM32S drives, refer to the Lexium LXM32S Product Manual.
