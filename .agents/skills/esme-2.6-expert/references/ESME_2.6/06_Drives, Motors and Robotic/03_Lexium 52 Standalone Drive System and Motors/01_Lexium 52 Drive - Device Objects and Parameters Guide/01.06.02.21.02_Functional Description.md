# Functional Description

Functional Description

Displays the overload of the drive (power stage and motor) in percent. If the overload of the motor is higher, then the overload of the motor is displayed, otherwise the overload of the power stage is displayed.

If the value is greater or equal 80 %, the diagnostic message 8125 Motor load high or 8129 Power stage load high is issued. If the value reaches 100 %, the diagnostic message 8100 Motor overload or 8210 Power stage overload is issued and the drive is switched off.

After enabling, the integrators (I²t) for the final stage and motor start with an initial value of more than 100 %. This helps to recognize that the power stage and the motor are not overloaded if they have been operated at the load limit before being switched off.

NOTE: The parameter value is transferred from the slave to the master via the parameter channel of the Sercos by every access. Typically, this takes about 10 ms. By a high capacity of the parameter channel, times up to 1 s can occur. If the Sercos bus is in phase 0 or 1, then a standard value is indicated here. If the Sercos bus is in phase 3 or 4, then the parameter value is transferred and indicated. In the Sercos phase 2, the parameter can be read through the application.
