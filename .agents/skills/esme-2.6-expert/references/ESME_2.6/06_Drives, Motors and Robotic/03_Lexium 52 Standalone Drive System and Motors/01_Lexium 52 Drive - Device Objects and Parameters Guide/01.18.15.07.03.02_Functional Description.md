# Functional Description

Functional Description

Displays which digital input is configured as touchprobe or fast counter (see parameter [IO0\_Mode, IO1\_Mode](../IO0_IO1Mode/IO0_IO1Mode.htm#XREF_D_SE_0071946_1)).

| Value | Data type | Meaning |
| --- | --- | --- |
| L / FALSE | BOOL | The corresponding digital input is not configured as touchprobe or fast counter. |
| H / TRUE | BOOL | The corresponding digital input is configured as touchprobe or fast counter and can be used. |
