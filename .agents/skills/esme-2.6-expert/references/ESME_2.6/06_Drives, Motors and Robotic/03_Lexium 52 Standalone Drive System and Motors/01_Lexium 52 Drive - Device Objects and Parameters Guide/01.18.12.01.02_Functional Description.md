# Functional Description

Functional Description

This parameter controls the output behavior while the Sercos bus is shutting down.

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | BOOL | If an error is detected on the Sercos bus, the outputs are reset. |
| H / 1 | BOOL | If an error is detected on the Sercos bus, the states of outputs are not modified until new, valid information over the bus is transferred or a reset of the device is triggered. |
