# Functional Description

Functional Description

Via this parameter, the state of the Ready contact is displayed. The device requires a mains contactor in order to be able to shut down the plant. This mains contactor has to be controlled by the Ready contact. The mains contactor must only operate if the Ready contact is closed (at low impedance). The Ready chain may comprise additional switches which inhibit the contactor from responding or cause the contactor to release although the Ready contact is closed.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The Ready contact is open because the device has a detected error or Sercos is not in phase 4.  The mains contactor has to be open. |
| on / 1 | BOOL | The Ready contact is closed and the mains contactor may operate.  The device is event-free and Sercos is in phase 4. |
