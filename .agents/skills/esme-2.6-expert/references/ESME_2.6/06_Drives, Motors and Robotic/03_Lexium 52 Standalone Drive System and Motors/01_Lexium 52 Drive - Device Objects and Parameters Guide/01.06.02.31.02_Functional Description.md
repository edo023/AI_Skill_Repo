# Functional Description

Functional Description

The parameter is used to display the thermal overload of the braking resistor in percent (0...100 %).

When this parameter reaches the value 90 %, the diagnostic message [8179 Braking resistor load high](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063893_3) is triggered.

When this parameter reaches the value 100 %, the braking resistor is deactivated. When more energy recharges the DC bus and no other braking resistor is available to reduce the energy, the voltage of the DC bus rises. When a voltage is too high, the diagnostic messages [8186 DC bus voltage high](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063900_3) (LXM62 PS only) or [8108 DC bus overvoltage](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063850_3) are triggered. When more than one Lexium LXM62 Power Supply is coupled to one DC bus, the overload of the braking resistors is automatically balanced from V01.54.xx.00 (EcoStruxure Machine Expert V1.0) or higher. When more than one Lexium 52 is coupled to one DC bus, the overload of the braking resistors is automatically balanced from V01.54.xx.00 (EcoStruxure Machine Expert V1.0) or higher. The DC bus coupling of Lexium 62 Power Supplies with Lexium 52 is not allowed.

NOTE: The parameter value is transferred from the slave to the master via the parameter channel of the Sercos by every access. Typically, this takes about 10 ms. By a high capacity of the parameter channel, times up to 1 s can occur. If the Sercos bus is in phase 0 or 1, then a standard value is indicated here. If the Sercos bus is in phase 3 or 4, then the parameter value is transferred and indicated. In the Sercos phase 2, the parameter can be read through the application.
