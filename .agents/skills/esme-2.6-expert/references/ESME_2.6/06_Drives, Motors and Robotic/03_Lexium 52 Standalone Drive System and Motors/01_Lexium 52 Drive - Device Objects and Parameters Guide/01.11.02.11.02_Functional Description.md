# Functional Description

Functional Description

This parameter displays the design of the motor.

| Value | Data type | Meaning |
| --- | --- | --- |
| Synchronous motor / 0 | DINT | The motor is a rotating synchronous machine. |
| Linear motor / 1 | DINT | The motor is a linear synchronous machine. |
| Asynchronous motor / 2 | DINT | The motor is a rotating asynchronous machine. |

The parameter value is transferred from the slave to the master via the Sercos bus. This means that if the Sercos bus is not in phase 4 (operating phase), a default value is indicated here. If the Sercos bus is in phase 4, then the parameter value is transferred over the Sercos bus.
