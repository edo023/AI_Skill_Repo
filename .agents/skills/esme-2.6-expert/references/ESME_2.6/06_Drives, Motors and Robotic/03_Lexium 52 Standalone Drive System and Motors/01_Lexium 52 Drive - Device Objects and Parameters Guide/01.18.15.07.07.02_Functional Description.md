# Functional Description

Functional Description

Is used to enter a bit to enable or disable the diagnostic message [8788 Wiring error detected](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064065_1) with ext. diagnostic message Overload for this digital field output.

This bit in the parameter OverloadDiagMsg is assigned to a corresponding bit in the bitmask of the parameter [OverloadDiagMsk](DigOut-6.htm#XREF_D_SE_0071986_1). Modifications in the parameter OverloadDiagMsg are automatically taken over by the corresponding bit in the parameter OverloadDiagMsk and the opposite way. The last modification is active. Also see parameter [OpenloadDiagMsg](DigOut-12.htm#XREF_D_SE_0089341_1).

| Value | Data type | Meaning |
| --- | --- | --- |
| L / 0 | UINT | The output does not report the diagnostic message 8788 Detected wiring error (Overload). |
| H / 1 | UINT | The output reports the diagnostic message 8788 Detected wiring error (Overload). |
