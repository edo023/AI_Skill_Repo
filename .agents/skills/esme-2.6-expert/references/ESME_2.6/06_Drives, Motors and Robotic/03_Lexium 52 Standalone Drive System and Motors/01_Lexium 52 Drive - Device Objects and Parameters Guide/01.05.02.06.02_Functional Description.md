# Functional Description

Functional Description

Displays the identification mode.

| Value | Data type | Meaning |
| --- | --- | --- |
| SerialNumberController / 0 | DINT | The device is assigned via the serial number of the controller (see parameter [SerialNumber](../../../../../../api/crossBook?lang=en-US&virtualBookName=Controller_2/Controller_2-9.htm#XREF_D_SE_0071761_1), also see [Identification mode](../../../PD.Parameter.LMCEco&topicID=D_SE_0088128_1)). |
| SerialNumberMotor / 1 | DINT | The device is assigned via the serial number of the motor (see parameter [SerialNumberMotor](../../../../../../api/crossBook?lang=en-US&virtualBookName=Motor_2/Motor_2-5.htm#XREF_D_SE_0071782_1), also see [Identification mode](../../../PD.Parameter.LMCEco&topicID=D_SE_0088128_1)).  This parameter is only applicable to drive objects. |
| TopologyAddress / 2 (default) | DINT | The device is assigned via its topological address (see parameter [TopologyAddress](../../../../../../api/crossBook?lang=en-US&virtualBookName=Identification_2-8.htm#XREF_D_SE_0071402_1), also see [Identification mode](../../../PD.Parameter.LMCEco&topicID=D_SE_0088128_1)). |
| ApplicationType / 3 | DINT | The device is assigned via the ApplicationType of the device (see parameter [ApplicationType](../../../../../../api/crossBook?lang=en-US&virtualBookName=Identification_2-10.htm#XREF_D_SE_0071404_1), also see [Identification mode](../../../PD.Parameter.LMCEco&topicID=D_SE_0088128_1)). |
| SercosAddress / 4 | DINT | The device is assigned via the Sercos address of the device (see parameter [SercosAddress](../../../../../../api/crossBook?lang=en-US&virtualBookName=Identification_2-12.htm#XREF_D_SE_0071406_1), also see [Identification mode](../../../PD.Parameter.LMCEco&topicID=D_SE_0088128_1)). |
