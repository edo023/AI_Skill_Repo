# Functional Description

Functional Description

Allows the display of the brake behavior in de-energized state.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0 / coupled | BOOL | The brake is coupled in de-energized state and released in energized state. |
| 1 / released | BOOL | The brake is released in de-energized state and coupled in energized state. |
