# General

General

| Step | Action |
| --- | --- |
| 1 | Right-click somewhere in the register card Logger and select the entry from the context menu or click the button [Load logger].  Result: The data is loaded and displayed in the register card Logger. During the loading procedure, a progress bar shows the status bar and the status information. |

This function is only available if it is a drive that was configured real with the WorkingMode (HostParameterSet ID: 196).

For axes that were added by a Sercos scan, this applies automatically.

Furthermore, an online connection to the controller must exist.

The following errors can occur and are displayed in the message overview:

oThe connection could not be established because no valid IP address or no active path was specified (No valid IP address/valid active path was entered).

oThe connection could not be established (Unable to connect to controller).

oException of controller is issued (The Logger could not be loaded).

oResidual error (The Logger could not be loaded).
