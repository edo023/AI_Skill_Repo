# Functional Description

Functional Description

Displays the maximum drive speed at the gear output. If UserMaxVel = 0, the default value for the parameter MaxVel [units/second] is identical to the rated speed [NominalSpeedLinear](Motor_2-2.htm#XREF_D_SE_0072153_1) [m/s]. If UserMaxVel > 0, then MaxVel is set to this value.

With real axes, MaxVel cannot become higher than the value that corresponds to the maximum motor speed MaxSpeedLinear [m/s].

If by virtual axes UserMaxVel is unequal to zero, then this value is taken over in MaxVel. This means, the value of UserMaxVel is also taken over if it is higher than the standard value (by UserMaxVel = 0).

If a motor has MaxSpeedLinear < [NominalSpeedLinear](Motor_2-2.htm#XREF_D_SE_0072153_1) (for example, because of a too low encoder maximum speed), then the motor cannot be operated with the rated speed. In this case, the default value of MaxVel [units/seconds] is the velocity of MaxSpeedLinear.

NOTE: The parameter value is transferred from the slave to the master via the parameter channel of the Sercos by every access. Typically, this takes about 10 ms. By a high capacity of the parameter channel, times up to 1 s can occur. If the Sercos bus is in phase 0 or 1, then a standard value is indicated here. If the Sercos bus is in phase 3 or 4, then the parameter value is transferred and indicated. In the Sercos phase 2, the parameter can be read through the application.
