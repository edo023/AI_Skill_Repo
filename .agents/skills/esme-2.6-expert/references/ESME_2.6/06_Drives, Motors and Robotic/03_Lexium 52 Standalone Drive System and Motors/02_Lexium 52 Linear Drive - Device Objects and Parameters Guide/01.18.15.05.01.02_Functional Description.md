# Functional Description

Functional Description

Is used to enter the bit field for activating the diagnostic message [8788 Wiring error detected](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064065_1) with ext. diagnostic message Openload for field outputs.

Each bit is assigned to an output corresponding to the bit number. The n bits in the parameter OpenloadDiagMsk are assigned to the bit in the parameter [OpenloadDiagMsg](DigOut-12.htm#XREF_D_SE_0089341_1) in the n outputs. Modifications in the parameter OpenloadDiagMsk are automatically taken over by the corresponding outputs in the parameter OpenloadDiagMsg and the opposite way. The last modification is active. Also see parameter [OverloadDiagMsk](DigOut-6.htm#XREF_D_SE_0071986_1).

| Bit | Meaning |
| --- | --- |
| 0 | The output is not monitored. |
| 1 | The output is monitored and the diagnostic message is sent if monitoring with the DiagMask parameter is also activated. |

NOTE: By a not connected output with activated monitoring the diagnostic message 8788 Wiring error (Openload) is sent immediately.
