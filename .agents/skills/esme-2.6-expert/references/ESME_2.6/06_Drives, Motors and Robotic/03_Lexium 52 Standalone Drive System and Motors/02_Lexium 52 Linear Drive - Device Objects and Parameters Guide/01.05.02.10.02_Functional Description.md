# Functional Description

Functional Description

Displays the application type read from the physical Sercos device. The parameter ApplicationType can only be modified by manually writing the Sercos parameter IDN S-0-1302.0.3. For this purpose, the function [FC\_SercosWriteServiceData](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085305_1) can be used.
