# Functional Description

Functional Description

Displays the device type. A description of the device type is available in the operating instructions of the device.

If the device in the controller configuration is virtual (non-real), then the parameter value is LXM62DxS virtual or ILM62 virtual.

NOTE: The parameter value is transferred from the slave to the master via Sercos. This means that if Sercos is not in phase 4 (operating phase), an empty string '' is indicated here. If Sercos is in phase 4, then the parameter value is transferred over Sercos.
