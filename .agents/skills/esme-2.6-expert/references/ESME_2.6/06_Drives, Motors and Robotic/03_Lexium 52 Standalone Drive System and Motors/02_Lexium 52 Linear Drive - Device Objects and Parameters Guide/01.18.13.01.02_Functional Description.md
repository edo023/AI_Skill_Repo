# Functional Description

Functional Description

The parameter affects the resetting of the OnBoard I/O outputs. The value must be configured in offline mode. In the case of a reset or a download of the projects, the setting is used which is saved in the boot project.

This means an online change of the parameter IOReset is discarded during the actions Reset warm, Reset cold and Download.

| Value | Data type | Meaning |
| --- | --- | --- |
| no reset / 0 | DINT | The outputs are not reset after the program has been downloaded. |
| after download / 1 | DINT | The outputs are reset after the program has been downloaded. |
| after download or prg. Reset / 2 | DINT | The outputs are reset after a download or a reset of the program. |
| after download, prg. Reset or prg. stop / 3 | DINT | The outputs are reset after a download, a reset, or a stop of the program. |
