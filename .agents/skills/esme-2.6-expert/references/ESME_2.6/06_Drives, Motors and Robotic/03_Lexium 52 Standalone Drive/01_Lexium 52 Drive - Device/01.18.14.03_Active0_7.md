# Active0_7

![](images/logo.png)

Active0\_7

Active0\_7

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | OnBoard I/O |
| Traceable | Yes |

Displays which digital input is configured as touchprobe or fast counter (see parameter [IO0\_Mode, IO1\_Mode](../IO0_IO1Mode/IO0_IO1Mode.htm#XREF_D_SE_0071946_1)).

| Bit value | Meaning |
| --- | --- |
| 0 | The corresponding digital input is not configured as touchprobe or fast counter. |
| 1 | The corresponding digital input is configured as touchprobe or fast counter and can be used. |

Example

Active0\_7 = 2#10000000

The digital input IO\_0 can be used as touchprobe or fast counter input.
