Functional Description

Displays the motor temperature in °C. Linear or nonlinear temperature sensors can be used in the motor.

Linear Temperature Sensors

Linear temperature sensors can (in certain areas) display the complete temperature range linear. When using a linear temperature sensor, the parameter can display temperatures in the range of -10...999 °C.

Nonlinear Temperature Sensors

Nonlinear switching temperature sensors can only display two conditions, for example temperature too high / too low. When using a nonlinear temperature sensor (for example a PTC temperature sensor), the parameter can only display two conditions.

| Condition | Meaning |
| --- | --- |
| 0 °C | The motor temperature is OK.  The motor can be operated. |
| 999 °C | The motor temperature is too high.  The motor cannot be operated. |

For further information on the temperature sensor, see the SH servo motor operator manual.

NOTE: The parameter value is transferred from the slave to the master via the parameter channel of the Sercos by every access. Typically, this takes about 10 ms. By a high capacity of the parameter channel, times up to 1 s can occur. If the Sercos bus is in phase 0 or 1, then a standard value is indicated here. If the Sercos bus is in phase 3 or 4, then the parameter value is transferred and indicated. In the Sercos phase 2, the parameter can be read through the application.
