# BrakeMode

![](images/logo.png)

BrakeMode

General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | Yes |

Functional Description

How and when the brake is controlled is defined by using BrakeMode.

| BrakeMode | Behavior |
| --- | --- |
| Connect brake at stillstand / 0 | In the event of [device reactions](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063389_1) AD, BD1 and BD2, the brake is coupled when the motor has come to a standstill. |
| Connect brake when coasting / 1 | In the event of [device reaction](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063389_1) AD, the brake is coupled immediately even if the motor is still running. In the event of device reactions BD1 and BD2, the brake is coupled when the motor has come to a standstill. (Standard behavior) |
| Connect brake when stopping / 2 | In the event of [device reactions](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063389_1) AD and BD1, the brake couples immediately even if the motor is still running. In the event of device reaction BD2, the brake is coupled when the motor has come to a standstill. |
| Control brake by user program / 3 | The user program has control over the brake. The state of the brake can be modified cyclically with the function [FC\_SercosSetUserRealTimeControlBits](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085303_1). The function [FC\_BrakeReleaseSet](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0084961_1) has no effect. |
