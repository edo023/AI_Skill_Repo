# IO0_Mode, IO1_Mode

![](images/logo.png)

IO0\_Mode, IO1\_Mode

IO0\_Mode, IO1\_Mode

General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices supporting the parameter | OnBoard I/O |
| Traceable | Yes |

Functional Description

Defines the functionality of a field input.

| Value | Data type | Meaning |
| --- | --- | --- |
| Input / 0 | DINT | Only the static state of the input ([Value](../DigIn/DigIn-8.htm#XREF_D_SE_0089350_1) parameter) can be queried. |
| Touchprobe / 1 | DINT | The touchprobe function is activated. This functionality latches a Position value by an edge at the field input with a high time resolution. See also the [CaptureState](../TouchprobeInputs/TouchprobeInputs-10.htm#XREF_D_SE_0089319_1), [CaptureOk](../TouchprobeInputs/TouchprobeInputs-11.htm#XREF_D_SE_0089321_1), [CaptureValue](../TouchprobeInputs/TouchprobeInputs-12.htm#XREF_D_SE_0089322_1), and [SensorDelay](../TouchprobeInputs/TouchprobeInputs-13.htm#XREF_D_SE_0089323_1) parameters. |
| Fast counter / 2 | DINT | The impulse counter function is activated. This functionality is able to count pulses at a field input. See also [CounterState](../TouchprobeInputs/TouchprobeInputs-14.htm#XREF_D_SE_0089324_1), [CounterValue](../TouchprobeInputs/TouchprobeInputs-15.htm#XREF_D_SE_0089325_1), [CounterStart](../TouchprobeInputs/TouchprobeInputs-16.htm#XREF_D_SE_0089326_1), and [CounterEdge](../TouchprobeInputs/TouchprobeInputs-17.htm#XREF_D_SE_0089327_1). |

NOTE: Modifications to the parameter are only applied during the Sercos phase up (communication phase 0 => communication phase 4).

NOTE: The first measured input (TP\_0) is assigned to IO\_0 and the second is assigned to IO\_1.
