# ControllerEnableSet

General

| Traceable | No |

Functional Description

Indicates whether the servo amplifier has been released or disabled. The parameter ControllerEnableSet, in addition to the function blocks [FC\_ControllerEnableSet](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085009_1)() and [FC\_ControllerEnableReset](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085007_1)(), provides a second interface for handling the [release](State_2-9.htm#XREF_D_SE_0071477_1). Both interfaces can be used in any given sequence. While reading, the status of the enable request is displayed.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | Not enabled |
| on / 1 | BOOL | Enabled |
