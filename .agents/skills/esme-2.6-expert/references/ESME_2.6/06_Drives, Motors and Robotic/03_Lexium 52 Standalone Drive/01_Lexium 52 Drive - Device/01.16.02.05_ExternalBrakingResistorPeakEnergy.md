# ExternalBrakingResistorPeakEnergy

![](images/logo.png)

ExternalBrakingResistorPeakEnergy

General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive |
| Traceable | No |

Functional Description

This parameter determines the peak load energy of the external braking resistor.

The value 0 is no permitted value for this parameter.

To be able to use the external braking resistor, the parameter [ExternalBrakingResistorEnableSet](ExternalBrakingResistor_2-2.htm#XREF_D_SE_0071865_1) has to be set to on / 1.

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s can occur if there is a lot of data transferred on the parameter channel.
