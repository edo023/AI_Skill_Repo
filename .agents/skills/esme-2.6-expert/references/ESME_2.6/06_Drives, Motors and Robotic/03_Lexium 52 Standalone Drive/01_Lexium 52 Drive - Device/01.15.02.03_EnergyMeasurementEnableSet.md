# EnergyMeasurementEnableSet

![](images/logo.png)

EnergyMeasurementEnableSet

General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | No |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply |
| Traceable | Yes |

Functional Description

This parameter controls the energy measurement. It is used to stop and restart the energy measurement.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The energy measurement is disabled.  The energy measurement stopped and the energy values remain unmodified. |
| on / 1 | BOOL | Default value.  The energy measurement is enabled. |

When switching on the device, the energy measurement is started automatically.

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s can occur if there is a lot of data transferred on the parameter channel.
