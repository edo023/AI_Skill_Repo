# BrakeState

![](images/logo.png)

BrakeState

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | Yes |

Functional Description

Used to display the condition of the brake. Since there is no feedback on the actual mechanical state of the brake, the indicated state can only be reached if the device works properly and the brake is wired appropriately.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0 / coupled | BOOL | The brake is controlled electrically in such a way that it is coupled. |
| 1 / released | BOOL | The brake is controlled electrically in such a way that it is released. |
