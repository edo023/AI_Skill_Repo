# EncoderRange

![](images/logo.png)

EncoderRange

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive,  Lexium LXM62 Drive,  Lexium ILM62 Drive Module |
| Traceable | – |

Functional Description

Is used to display the value range of the encoder in revolutions on the encoder shaft. To calculate the encoder range on the motor side, the parameters [EncoderGearIn](Encoder_2-5.htm#XREF_D_SE_0071806_1) and [EncoderGearOut](Encoder_2-6.htm#XREF_D_SE_0071807_1) have to be considered.

Example

Standard single-turn encoder 1 rotation, multi-turn encoder 4096 rotations.
