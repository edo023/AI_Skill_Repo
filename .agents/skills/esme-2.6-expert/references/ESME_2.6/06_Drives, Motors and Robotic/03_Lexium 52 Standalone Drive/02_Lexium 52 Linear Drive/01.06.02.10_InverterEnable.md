# InverterEnable

![](images/logo.png)

InverterEnable

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply |
| Traceable | Yes |

Functional Description

Displays the status of the InverterEnable signal.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The lock-out tag-out sequence is enabled (Safe Torque Off). |
| on / 1 | BOOL | The lock-out tag-out sequence is disabled. |

Influence of ControllerEnable, TorqueEnable, and InverterEnable on the power stage release of the servo amplifier

![G-SE-0061390.1.gif](images/G-SE-0061390.1.gif)

NOTE: In the Lexium 52, the InverterEnable signal is determined from the two parameters STO\_A and STO\_B, according to the following formula:

InverterEnable := NOT ([STO\_A](State_2-26.htm#XREF_D_SE_0071463_1) OR [STO\_B](State_2-27.htm#XREF_D_SE_0071464_1)).
