# ElectricalEnergyPowerSupply

![](images/logo.png)

ElectricalEnergyPowerSupply

General

|  |  |
| --- | --- |
| Type | AS |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Power Supply |
| Traceable | No |
| Version | Valid as of V1.34.X.X |

Functional Description

Shows the consumed electrical energy of the DC bus in kWh.

The parameter has the data type LREAL. The standard value is zero.

The standard value is displayed if the axis is virtual or the parameter was not yet read by the drive. The parameter can only be read if the Sercos bus is at least in phase 2.

Exception: If a controller firmware V1.34.X.X or higher and at the same time a drives firmware V1.31.X.X to V1.33.X.X is installed, then the parameter shows the value 0.

The used electrical power of the power supply is measured every 125 µs and averaged over 1 s. The electrical energy then results from integrating the power over the duration of 1 s.

The consumed electrical real (active) energy has a positive prefix.

NOTE:  The energy values are saved regularly. The calculation of the energy values is also continued after switching the device on and off until you stop the energy measurement independently (see [EnergyMeasurementEnableSet](EnergyMeasurement_2-2.htm#XREF_D_SE_0071858_1)). Otherwise, the energy values are not reset to zero after restarting of the device, but will be continued to be measured.

NOTE: The parameter value is transferred from the slave to the master via the parameter channel of the Sercos by every access. Typically, this takes about 10 ms. By a high capacity of the parameter channel, times up to 1 s can occur. If the Sercos bus is in phase 0 or 1, then a standard value is indicated here. If the Sercos bus is in phase 3 or 4, then the parameter value is transferred and indicated. In the Sercos phase 2, the parameter can be read through the application.
