# YOffsetRefDeceleration

![](images/logo.png)

YOffsetRefDeceleration

General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | No |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Sercos Drive |
| Traceable | Yes |

Functional Description

This parameter configures the reference deceleration in Units/sec^2. This deceleration is used to ramp down from velocity of YOffsetRefVelocity.
