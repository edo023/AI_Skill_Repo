# MotorTemperatureMonitoring

![](images/logo.png)

MotorTemperatureMonitoring

General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | No |

Functional Description

With this parameter, you can determine how the temperature of the motor shall be displayed. If the chosen determination is not possible, the diagnostic message 8977 Motor temp. monitoring disabled is triggered.

| Value | Data type | Meaning |
| --- | --- | --- |
| not valid / 0 | DINT | The parameter is not evaluated. |
| Thermal model / 1 | DINT | Thermal model to monitor the temperature. The MotorTypePlate must contain data for thermal model calculation. Otherwise this mode is not possible. |
| Sensor / 2 | DINT | The temperature sensor is used to monitor the temperature. The motor must have a temperature sensor that is connected to the motor controller. Otherwise this mode is not possible. |

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s can occur if there is a lot of data transferred on the parameter channel.
