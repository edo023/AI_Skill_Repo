Functional Description

NOTE: The parameters [ControllerStopDec](../General_2/General_2-10.htm#XREF_D_SE_0071532_1), [ControllerStopJerk](../General_2/General_2-11.htm#XREF_D_SE_0071533_1), and [ControllerStopSet](#XREF_D_SE_0071452_1) are supported as of V1.35.1.5 from the Sercos Drive. As of this version the Sercos Drive also implements the interface IF\_ControllerStop.

Emergency stop

Using the parameter ControllerStopSet, you can specify whether an emergency stop shall be triggered. An emergency stop can only be triggered via this parameter.

A positive edge (FALSE -> TRUE) at this parameter triggers an emergency stop. The motion job is canceled and the axis is delayed via the parameters [ControllerStopDec](../General_2/General_2-10.htm#XREF_D_SE_0071532_1) and [ControllerStopJerk](../General_2/General_2-11.htm#XREF_D_SE_0071533_1).
