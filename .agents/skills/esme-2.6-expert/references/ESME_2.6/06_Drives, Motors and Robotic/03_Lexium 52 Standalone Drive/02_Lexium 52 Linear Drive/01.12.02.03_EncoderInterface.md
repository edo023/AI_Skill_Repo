# EncoderInterface

![](images/logo.png)

EncoderInterface

General

|  |  |
| --- | --- |
| Type | AK |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | No |

Functional Description

Indicates the active encoder interface of the motor encoder.

| Value | Data type | Meaning |
| --- | --- | --- |
| No encoder / 0 | DINT | The device did not detect an encoder. No encoder interface is active. |
| SinCos with Hiperface / 1 | DINT | The device detected a SinCos Encoder with HIPERFACE® (High Performance Interface). |
| SinCos without Hiperface / 2 | DINT | The device uses the encoder interface for SinCos encoders (1 Vpp). |

The parameter value is transferred from the slave to the master via the Sercos bus. This means that if the Sercos bus is not in phase 4 (operating phase), a default value is displayed here. If the Sercos bus is in phase 4, then the parameter value is transferred over the Sercos bus.
