# CaptureValue

![](images/logo.png)

CaptureValue

General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  OnBoard I/O, DIO8 module,  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

Functional Description

Displays the position (at [FC\_TpEdge()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085375_1)) that has been captured when the Touchprobe event occurred. The value is reset to 0.00 when FC\_TPEdge() is called again.

Resolution of the measure inputs for the TP\_Edge() function

| Device (Measure Inputs) | Resolution |
| --- | --- |
| PacDrive LMC x00C | 10 µs |
| Lexium LXM62 Drive | 10 µs |
| DIO8 module | 10 µs |
