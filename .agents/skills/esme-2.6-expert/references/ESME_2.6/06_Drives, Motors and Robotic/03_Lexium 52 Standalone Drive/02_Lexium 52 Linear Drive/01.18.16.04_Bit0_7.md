# Bit0_7

![](images/logo.png)

Bit0\_7

Bit0\_7

General

|  |  |
| --- | --- |
| Type | AF |
| Devices supporting the parameter | OnBoard I/O |
| Traceable | Yes |

Functional Description

Displays the current value of the touchprobe or fast counter inputs. Each bit is assigned to an input corresponding to the bit number.

NOTE: Verify by querying the parameter [Active0\_7](../TouchprobeInputs/TouchprobeInputs-3.htm#XREF_D_SE_0071971_1) whether the connection value is valid.

| Bit value | Meaning |
| --- | --- |
| 0 | Connection LOW. |
| 1 | Connection HIGH. |

Assigning bits to inputs Bit0\_7 OnBoard I/O (Touchprobe inputs)

|  |  |  |

| Parameter bit number | Bit 1 | Bit 0 |
| Touchprobe input (1) of the Lexium LXM62 Drive / Lexium LXM52 Drive | TP\_1\_1 | TP\_0\_1 |
| (1)   Affected parameter: [Value](../TouchprobeInputs/TouchprobeInputs-9.htm#XREF_D_SE_0089318_1) | | |

Example: Bit0\_7 = 2#10000000

The value of TP\_0\_1 is H / TRUE; TP\_1\_1 is L / FALSE.
