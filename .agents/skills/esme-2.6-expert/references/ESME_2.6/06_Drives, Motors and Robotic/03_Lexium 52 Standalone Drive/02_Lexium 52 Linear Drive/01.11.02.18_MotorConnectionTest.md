# MotorConnectionTest

MotorConnectionTest

General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module |
| Traceable | No |

Functional Description

With this parameter, you can determine if the connection to the motor shall be checked when the drive is activated (this means, [ControllerEnable](../State_2/State_2-9.htm#XREF_D_SE_0071477_1) is set to TRUE).

During the check, a current is temporarily stored that does not cause a torque. The measured current is compared to the expected current. If too much deviation is detected, then the diagnostic message [8969 Motor cable not connected](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0064098_1) is triggered.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The check is not performed. |
| on / 1 | BOOL | The check is performed if the ControllerEnable is set to TRUE. |

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s can occur if there is a lot of data transferred on the parameter channel.
