# CaptureState

![](images/logo.png)

CaptureState

Genearl

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | PacDrive LMC x00C,  OnBoard I/O, DIO8 module,  Bus terminals BT-4/DIO1 and BT-4/DIO1 TP |
| Traceable | Yes |

Functional Description

Displays the status of the touchprobe (TP) field input. The following can be displayed:

Values of the CaptureState parameter of the measured inputs for the FC\_TpEdge() function

| Value | Data type | Meaning |
| --- | --- | --- |
| inactive / 0 | DINT | The capture function is not enabled. |
| active / 1 | DINT | [FC\_TpEdge()](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Lib.SystemInterface&topicID=D_SE_0085375_1) is enabled. |
| captured / 2 | DINT | A valid measure value was recorded. |
| overflow / 3 | DINT | Timeout of the pulse function (TP\_Pulse()). |
| disabled / 4 | DINT | The touchprobe functionality is not available as Mode is not parameterized to Touchprobe / 1. |
| not ready / 5 | DINT | The touchprobe function is not available as the Sercos bus is not in phase 4. |
| virtual / 6 | DINT | The touchprobe function is not available as the parent motor controller is virtual. |
