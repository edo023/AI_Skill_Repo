# ExternalBrakingResistorEnableSet

![](images/logo.png)

ExternalBrakingResistorEnableSet

General

|  |  |
| --- | --- |
| Type | ES |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive |
| Traceable | No |

Functional Description

This parameter determines if the internal or an external braking resistor is used.

NOTE:

Before you set this parameter to on / 1, a valid value has to be set for the following parameters:

o[ExternalBrakingResistorResistance](ExternalBrakingResistor_2-3.htm#XREF_D_SE_0071882_1)

o[ExternalBrakingResistorContinuousPower](ExternalBrakingResistor_2-5.htm#XREF_D_SE_0071884_1)

o[ExternalBrakingResistorPeakEnergy](ExternalBrakingResistor_2-4.htm#XREF_D_SE_0071883_1)

Otherwise a detected error message occurs and the external braking resistor is not activated.

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | The internal braking resistor is used. |
| on / 1 | BOOL | The external braking resistor is used. |

NOTE: The parameter value is transferred from the master to the slave via the parameter channel of the Sercos at every access. Typically, this takes about 10 ms. However, times up to 1 s can occur if there is a lot of data transferred on the parameter channel.
