If you have questions concerning the type code, contact your Schneider Electric representative.

## Designation Customized Version

In the case of a customized version, position 8 of the type code is an "S". The subsequent number defines the customized version. Example: SH30551S0000001

If you have questions concerning customized versions, contact your local Schneider Electric service representative.
