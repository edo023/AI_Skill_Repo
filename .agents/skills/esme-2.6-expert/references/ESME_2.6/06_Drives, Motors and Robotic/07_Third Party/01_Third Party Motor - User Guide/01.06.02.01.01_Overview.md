# Overview

Overview

This document describes how an asynchronous motor can be operated at the PacDrive3 system. The description is based on an example.

In this sample application, an asynchronous motor is used to drive a belt. This belt is to be run with a constant speed of rotation. For this, the asynchronous motor is used without encoder in controlled operation. This description can also be applied to the regulated operation of an asynchronous motor. This document describes the differences that require attention.
