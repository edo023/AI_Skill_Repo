# BoostVoltage Parameter

BoostVoltage Parameter

If the asynchronous motor does not move with a low reference speed of rotation, the [BoostVoltage](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071580_1) parameter has to be adjusted. It is specified as percentage value of the rated voltage. The default value is 5 %.
