# HIPERFACE Encoders

HIPERFACE Encoders

For HIPERFACE encoders it is possible to save the commutation offset in the motor nameplate: Set the [MotorCommutationControl](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071796_1) object parameter to Test and write/1.
