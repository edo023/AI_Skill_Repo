# Overview

Overview

New motor data can only be programmed into a new axis if there are no other motor data. If there already are old motor types an axis, these data have to be deleted before programming the new data. To do so, use the function block MTP.FB\_MotorDataDelete.

fbDeleteMotorData(  
        i\_ifDrive := i\_Drive,  
        i\_xExecute := TRUE,  
        i\_etStorageLocation := MTP.ET\_StorageLocation.Drive,  
        i\_xEnable := TRUE,  
        q\_xDone => xDeleteDone,  
        q\_etDiag => etDeleteError,  
        q\_etDiagExt => etDiagExt,  
        q\_sMsg => sDiagMsg);
