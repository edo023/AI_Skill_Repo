# General Information

General Information

To use the asynchronous motor at the PacDrive 3 system, the drive must be able to read the electronic type plate of the motor.

In the traditional Schneider Electric servo motors, the electronic type plate is saved in the encoder.

When using 3rd party motors, the electronic type plate must be programmed by the user. The programming is performed using a EcoStruxure Machine Expert application. For this, at first the library UserMotorTypePlate must be inserted in the project.

| --- | --- |
| There is no motor data file, | Start with [step 1](Asynchronous_Motor-8.htm#XREF_D_SE_0088224_1): Insert motor data in ST\_UserMotorDataACIM. |
| A motor data file already exists, | Start with [step 3](Asynchronous_Motor-10.htm#XREF_D_SE_0088226_1): Writing motor data files. |
