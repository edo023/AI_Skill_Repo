# Configuring Parameters in EcoStruxure Machine Expert

![](images/logo.png)

Configuring Parameters in EcoStruxure Machine Expert

Overview

For configuring parameters, ensure that the following parameters are set correctly in EcoStruxure Machine Expert:

oMotorIdentification parameter

oControlMode parameter

oBoostVoltage parameter

oVF\_Characteristic parameter

MotorIdentification Parameter

When using the motor whose electronic type plate was written as described above, the MotorIdentification parameter of the drive object has to be set according to the storage location that was selected when calling FB\_MotorDataWrite:

| Storage location | MotorIdentification |
| --- | --- |
| Drive | Motor without type plate / 2 |
| Encoder | Motor with type plate / 0 |

NOTE: If no electronic type plate is found in the location indicated by MotorIdentification, the diagnostic message [8117 Motor nameplate not readable](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063857_3) is triggered on the Sercos run-up. To reset the detected error, the servo drive has to be switched off and on again.

ControlMode Parameter

The [ControlMode](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071561_1) parameter determines the operating mode of the motor:

oClosed loop control or

oOpen loop control or

Both operating modes are permitted for asynchronous motors. An encoder is required for closed loop control operation.

NOTE: The parameter is only transferred to the drive during the Sercos run-up. If the parameter is changed, a new Sercos run-up has to be performed (phase = 0 => phase 4).

BoostVoltage Parameter

If the asynchronous motor does not move with a low reference speed of rotation, the [BoostVoltage](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071580_1) parameter has to be adjusted. It is specified as percentage value of the rated voltage. The default value is 5 %.

VF\_Characteristic Parameter

The [VF\_Characteristic](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071579_1) parameter determines the U/f characteristic curve that is used for controlling the asynchronous motor. Normally, a linear U/f characteristic curve should be used, that is, the parameter does not have to be changed.

For particular applications (for example pumps and fans), it can be advisable to use a quadratic U/f characteristic curve. Thus the torque of the motor drops at a low speed of rotation but the energy efficiency can be improved.

Control

For closed loop operation, the same parameters are to be adjusted as for synchronous motors:

oJ\_Load

o[Pos\_P\_Gain](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52LinearDrive&topicID=D_SE_0071563_1)

o[Vel\_P\_Gain](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071565_1)

o[Vel\_I\_Gain](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071566_1)

o[VelFilter](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071567_1)

o[CurrRefFilter](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071570_1)

Above all, it is important to enter the J\_Load correctly.

Gearbox Factor for Encoder

If an encoder is used that is not directly mounted on the motor, the gearbox factor for the encoder must be set.

The following parameters are used:

o[EncoderGearIn](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071806_1)

o[EncoderGearOut](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071807_1)

o[EncoderDirection](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071808_1)
