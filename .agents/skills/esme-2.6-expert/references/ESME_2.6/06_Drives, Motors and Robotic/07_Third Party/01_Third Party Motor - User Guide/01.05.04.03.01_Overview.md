# Overview

Overview

1.A variable of type MTP.ST\_UserMotorData must be created in the code.

VAR  
    stUserMotorData: MTP.ST\_UserMotorData;  
END\_VAR

2.This structure is then filled with the motor data.

stUserMotorData.etMotorType := MTP.ET\_MotorType.LinearPMSM;  
stUserMotorData.sMotorname := 'HIWIN\_LXM1L\_S23\_1\_61';  
stUserMotorData.sMotorArticleNumber := 'ETYP00056';  
stUserMotorData.sMotorSerialNumber := '201120.0020';  
stUserMotorData.stMotorDataPMSM.uiEncoderType := 258;   // 1Vpp Linear Encoder  
stUserMotorData.stMotorDataPMSM.rPeriodLength := 1000;  // in um  
stUserMotorData.stMotorDataPMSM.uiNominalSpeed := 3800;  // in mm/s  
stUserMotorData.stMotorDataPMSM.rNominalVoltage := 750;  // in V  
stUserMotorData.stMotorDataPMSM.rNominalCurrent := 3.5;  // in A  
stUserMotorData.stMotorDataPMSM.rPeakCurrent := 10.5;  // in A  
stUserMotorData.stMotorDataPMSM.rContStallCurrent := 3.5;  // in A  
stUserMotorData.stMotorDataPMSM.rConstStallTorque := 220;  // in N  
stUserMotorData.stMotorDataPMSM.rPeakTorque := 600;  // in N  
stUserMotorData.stMotorDataPMSM.rPhaseResistance := 4.6;  // in Ohm  
stUserMotorData.stMotorDataPMSM.rQuadraturePhaseInductance := 54000;  // in uH      
stUserMotorData.stMotorDataPMSM.rDirectPhaseInductance := 54000;  // in uH  
stUserMotorData.stMotorDataPMSM.uiRotatingFieldDirection := 1;  
stUserMotorData.stMotorDataPMSM.rEMK\_Constant := 43;  
stUserMotorData.stMotorDataPMSM.rPolePairPitch := 32000;  
stUserMotorData.stMotorDataPMSM.uiThermalConstant := 180;  
stUserMotorData.stMotorDataPMSM.uiTempSensorType := 1;  
stUserMotorData.stMotorDataPMSM.rInsulationSystemVoltage := 800;  // in V       
stUserMotorData.stMotorDataPMSM.uiMaxMotorTemperature := 130;  // in °C  
stUserMotorData.stMotorDataPMSM.uiTempSensorResistance­Overtemp:=1330;  // in Ohm       
stUserMotorData.stMotorDataPMSM.uiMaxSpeed := 4000;  // in mm/s  
stUserMotorData.stMotorDataPMSM.udiMotorInertia := 7800;  // in g  
stUserMotorData.stMotorDataPMSM.uiBrake := 0;  // no

NOTE: When entering the data, the unit of the individual parameters is to be considered. Specifications to the units can be found in the description of the structure ST\_UserMotor­DataPMSM of the library UserMotorTypePlate.

NOTE: Resistance and inductance values must be entered as phase-to-phase value. For this, the values for the resistance and the inductance must be doubled in this example.

Since in the motor data no distinction is made between the inductance in q and the d direction, enter the same value for both inductances (rQuadraturePhaseInductance and rDirectPhaseInductance).

The motor data only contain one specification concerning the rated current, for this, rContStall­Current and rNominalCurrent are set to the same value.

Since the motor is equipped with a non-linear PTC temperature sensor, one specified resistance value (uiTempSensorResistanceOvertemp) that determines the switch-off threshold is sufficient. If there is a linear temperature sensor, motion points of a complete resistance temperature characteristic curve must be implemented instead (auiTempSensorCharacteristics).
