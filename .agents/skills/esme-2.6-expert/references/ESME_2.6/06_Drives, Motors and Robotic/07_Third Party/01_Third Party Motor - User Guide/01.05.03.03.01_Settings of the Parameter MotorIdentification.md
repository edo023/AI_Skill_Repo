# Settings of the Parameter MotorIdentification

Settings of the Parameter MotorIdentification

When using the motor whose electronic type plate was written as described in the section [Programming Procedure](Linear_Motor-7.htm#XREF_D_SE_0088204_2), the MotorIdentification parameter of the drive object has to be set according to the storage location that was selected when calling FB\_MotorDataWrite:

| Storage location | MotorIdentification |
| --- | --- |
| Drive | Motor without type plate / 2 |
| Encoder | Motor with type plate / 0 |

NOTE: If no electronic type plate is found in the location indicated by MotorIdentification, the diagnostic message 8117 Motor nameplate not readable is triggered on the Sercos run-up. To reset the detected error, the servo drive has to be switched off and on again.

For a detailed description of the library and the individual fields, refer to the UserMotorTypePlate Library Guide.
