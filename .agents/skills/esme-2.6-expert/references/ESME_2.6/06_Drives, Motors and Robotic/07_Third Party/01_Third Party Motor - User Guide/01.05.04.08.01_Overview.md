# Overview

Overview

The first tests should be performed manually.

In the following there are some examples:

1.Check whether the displayed position after Power Off - Power On is always the same. Repeat this test at several positions.

2.Check whether the distance moved corresponds to the display in EcoStruxure Machine Expert:

oWrite the Position object parameter at a certain position.

oPush the motor manually into another position.

oCalculate the distance between the two positions.

oCompare to the display in EcoStruxure Machine Expert.

3.Run simple profiles with small velocities and accelerations in the template using the YOffsetGenerator or the Jog function.
