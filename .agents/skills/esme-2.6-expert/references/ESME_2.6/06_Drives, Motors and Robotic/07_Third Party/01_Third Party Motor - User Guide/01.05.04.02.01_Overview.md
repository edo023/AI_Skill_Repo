# Overview

Overview

In the following example, a 3rd party linear motor is to be used with a Lexium 62 drive.

A 1Vpp encoder is connected at the motor. Therefore, the electronic type plate should be saved in the drive. The motor has the following electrical parameters (excerpt):

The motor has the following electrical parameters (excerpt):

| Electrical parameter | Value |
| --- | --- |
| Dimensions of the slideway | 7.8 kg (17.19 lb) |
| Resistance (per phase at 20 °C (68 °F)) | 2.3 ohm |
| Inductance (per phase) | 27 mH |
| EMF constant | 43 Vrms/(ms) |
| Peak force | 600 N |
| Nominal force | 220 N |
| Peak current | 10.5 A |
| Rated current | 3.5 A |
| Pole pair distance | 32 mm (1.26 in.) |

The used encoder has a period length of 1 mm (0.04 in.). The winding temperature is monitored with a temperature sensor of the type PTC-130 that has a resistance value greater than 1330 Ω above a temperature of 135 °C (275 °F).
