# Parameters for Closed-Loop Controlled Operation

Parameters for Closed-Loop Controlled Operation

There are two possibilities regarding the closed-loop controlled operation:

oElectrical parameters (inductance, resistance) are not known

oElectrical parameters (inductance, resistance) are known

Electrical parameters (inductance, resistance) are not known

Similar to the open-loop controlled operation, the data of the type plate are used.

The following parameters are required:

oNominal frequency

oNominal speed of rotation

oRated power

oRated voltage

oRated current

The specification of cos phi is mandatory for motors which a rated power < 700 W. The specification of the value is to be recommended but not mandatory for motors (rated power ≥ 700 W).

Using these parameters, an estimation of the electrical parameters (main inductance, leakage inductance, stator resistance and rotor resistance) is then performed. Operating the motor is possible but the dynamic is limited.

Electrical parameters (inductance, resistance) are known

If the electrical parameters (main inductance, leakage inductance, stator resistance, and rotor resistance) are known, the motor can be operated directly with these parameters.

The following parameters must be known:

oMain inductance Lm

oStator leakage inductance Lsσ

oStator resistance Rs

The following parameters are optional:

oRotor leakage inductance Lrσ

(If the rotor leakage inductance is not specified, Lrσ = Lsσ is assumed.)

oRotor resistance Rr

(If the rotor resistance is not specified, Rr = Rs is assumed.)
