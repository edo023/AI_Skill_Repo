General Information

Overview

Starting with PacDrive 3 v3.1SP1 (controller firmware, drives firmware, and EcoStruxure Machine Expert), a drive connected to a linear motor can be used with a linear unit system.

In previous versions of PacDrive 3, a drive connected to a linear motor can be used with a rotary unit system only. The descriptions below refer to using the linear unit system that is available in PacDrive 3 v3.1 SP1 and greater.
