# Functional Description

Functional Description

Indicates whether a software enabling for the servo amplifier has been given by the controller. The parameter provides information on the Release status. It is determined by the user, diagnostic, Sercos driver and system components.

Th following figure shows the influencing factors on the ControllerEnable parameter of the servo amplifier.

![G-SE-0061389.1.gif](images/G-SE-0061389.1.gif)

| Value | Data type | Meaning |
| --- | --- | --- |
| off / 0 | BOOL | Not enabled |
| on / 1 | BOOL | Enabled |
