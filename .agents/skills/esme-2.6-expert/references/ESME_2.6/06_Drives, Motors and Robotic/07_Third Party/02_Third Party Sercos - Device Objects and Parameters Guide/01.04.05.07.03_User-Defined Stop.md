# User-Defined Stop

User-Defined Stop

During the user-defined stop, a reference value profile, which is defined via the Controller­StopDec and ControllerStopJerk parameter, is started to stop the drive. The start values of the profile correspond with the values of the position, the speed and, if applicable, the acceleration. The profile is generated in the drive and can also be processed in case of a detected Sercos error. This function is enabled if

o[ControllerEnable](../../../../../../api/crossBook?lang=en-US&virtualBookName=State/State-8.htm#XREF_D_SE_0071477_1) is switched off and [ControllerEnableStopMode](../../../PD.Parameter.LXM52LinearDrive&topicID=D_SE_0071430_1) is set to user-defined stop or

oa diagnostic message is triggered with [reaction BD2](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Diagnostic&topicID=D_SE_0063389_1).

NOTE: The parameterization for devices of the type Sercos Drive is not performed automatically. A parameterization must be performed manufacturer-specific if it is supported by the respective Sercos Drive.

NOTE: The ControllerStopJerk parameter must not be modified as long as the user-defined stop is being executed.
