# IdentificationMode

![](images/logo.png)

IdentificationMode

General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos Drive,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  Logic Motion Controller Device |
| Traceable | Yes |

Functional Description

Displays the identification mode.

| Value | Data type | Meaning |
| --- | --- | --- |
| SerialNumberController / 0 | DINT | The device is assigned via the serial number of the controller (see parameter [SerialNumber](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52LinearDrive&topicID=D_SE_0071761_1), also see [Identification mode](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LMCEco&topicID=D_SE_0088128_1)). |
| SerialNumberMotor / 1 | DINT | The device is assigned via the serial number of the motor (see parameter [SerialNumberMotor](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LXM52Drive&topicID=D_SE_0071782_1), also see [Identification mode](../../../../../../api/crossBook?lang=en-US&virtualBookName=PD.Parameter.LMCEco&topicID=D_SE_0088128_1)).  This parameter is only applicable to drive objects. |
| TopologyAddress / 2 (default) | DINT | The device is assigned via its topological address (see parameter [TopologyAddress](../../../../../../api/crossBook?lang=en-US&virtualBookName=Identification-8.htm#XREF_D_SE_0071402_1), also see [Identification mode](../../../PD.Parameter.LMCEco&topicID=D_SE_0088128_1)). |
| ApplicationType / 3 | DINT | The device is assigned via the ApplicationType of the device (see parameter [ApplicationType](../../../../../../api/crossBook?lang=en-US&virtualBookName=Identification-10.htm#XREF_D_SE_0071404_1), also see [Identification mode](../../../PD.Parameter.LMCEco&topicID=D_SE_0088128_1)). |
| SercosAddress / 4 | DINT | The device is assigned via the Sercos address of the device (see parameter [SercosAddress](../../../../../../api/crossBook?lang=en-US&virtualBookName=Identification-12.htm#XREF_D_SE_0071406_1), also see [Identification mode](../../../PD.Parameter.LMCEco&topicID=D_SE_0088128_1)). |
