# ConfiguredApplicationType

![](images/logo.png)

ConfiguredApplicationType

General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive, Lexium LXM52 Linear Drive,  Lexium LXM62 Drive, Lexium LXM62 Linear Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos Drive,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  Logic Motion Controller Device |
| Traceable | No |

Functional Description

If IdentificationMode = ApplicationType / 3, the value of ConfiguredApplica­tionType is used for identifying the physical device in the Sercos network.

NOTE: The maximum parameter length is 40 characters [string(40)]. However, the length to be used may have to be shorter depending on the device type/version (for example, Bus Interface TM5NS31/TM5CSLCx00FS has a maximum 32 of characters).
