# Licensing procedure

To be able to build the safety-related application in EcoStruxure Machine Expert™ – Safety after integrating ASi Gateway device(s) in EcoStruxure Machine Expert™, you have to install a specific license feature. An EcoStruxure Machine Expert™ – Safety project with integrated ASi Gateway cannot be compiled without having activated the license feature in the Schneider Electric License Manager.

To install the feature license, proceed as follows:

1. Launch the License Manager in the Schneider Electric program group or from the EcoStruxure Machine Expert™ – Safety help menu.
2. In the License Manager, click the 'Activate' button and select an activation method.
3. Follow the instructions given in the License Manager.
