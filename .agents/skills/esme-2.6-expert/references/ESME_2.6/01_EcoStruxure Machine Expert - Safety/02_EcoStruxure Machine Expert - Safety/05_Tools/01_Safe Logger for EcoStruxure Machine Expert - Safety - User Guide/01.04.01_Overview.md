# Overview

## Overview

Safe Logger is a plug-in software of the EcoStruxure Machine Expert - Safety for EcoStruxure Machine Expert. The product lets you manage the messages originating from the safety-related equipment of your application. The messages allow you to monitor the status of your machine, installation or process, and support you in diagnosing and troubleshooting your application.

The messages are stored in an XML file on the logic controller of your application. Safe Logger allows you to display and manage the messages and the corresponding XML file.

The messages are classified according to three message types:

* Information messages, for example, status information
* Alert messages
* Error messages

The product provides the following functions:

* Display messages with the associated additional information
* Sort messages
* Filter messages
* Search messages
* Import and export messages from and to XML files
