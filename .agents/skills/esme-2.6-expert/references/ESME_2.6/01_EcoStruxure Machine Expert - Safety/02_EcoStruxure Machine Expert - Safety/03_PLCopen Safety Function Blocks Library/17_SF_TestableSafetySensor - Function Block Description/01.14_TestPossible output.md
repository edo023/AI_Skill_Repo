# TestPossible output

Output for signaling whether an automatic sensor test is possible.

Data type: BOOL

**Connection:** Connect the TestPossible output to an output terminal of your application, either directly or via additional safety-related functions.

| Value | Description |
| --- | --- |
| TRUE | Automatic sensor test is possible, which means it can be requested by means of a positive edge at the StartTest input. |
| FALSE | Automatic sensor test is not possible. A positive edge at the StartTest input has no effect. |
