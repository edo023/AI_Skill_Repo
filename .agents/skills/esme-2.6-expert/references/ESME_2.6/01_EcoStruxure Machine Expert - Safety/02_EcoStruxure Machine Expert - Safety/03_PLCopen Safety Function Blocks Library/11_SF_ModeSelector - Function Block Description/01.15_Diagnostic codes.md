# Diagnostic codes

The hexadecimal diagnostic codes listed here are output as WORD data type at the DiagCode output of the function block.

| Code (hex) | Description | Options for remedy |
| --- | --- | --- |
| 0000 | The function block is not activated. The Activate input is FALSE. | Activate the function block by switching Activate to TRUE. |
| 8000 | The function block is in a normal operating state. Mode selector switching is not locked (S\_Unlock = SAFETRUE). The respective output of the selected operating mode is SAFETRUE. | In case of an unexpected combination at the function block inputs relative to the physical input of the sensors, verify the connected hardware if required and correct the error. |
| 8004 | The function block is in a normal operating state. Mode selector switching is locked (S\_Unlock = SAFEFALSE). The respective output of an operating mode is SAFETRUE. | In case of an unexpected combination at the function block inputs relative to the physical input of the sensors, verify the connected hardware if required and correct the error. |
| 8005 | The function block is activated. All outputs of the operating mode are SAFEFALSE. | Switch S\_SetMode = SAFETRUE to activate the selected operating mode. Mode selector switching must be unlocked (S\_Unlock = SAFETRUE). |
| C001 | More than one input of the operating mode was SAFETRUE at the same time. | Verify the connected hardware and remedy the error. |
| C002 | Once the monitoring time specified at ModeMonitorTime has expired, all inputs of the operating mode were SAFEFALSE. One input of the operating mode should have switched to SAFETRUE within the time specified at ModeMonitorTime. | * Verify the connected hardware and remedy the error. * Verify the monitoring time set at ModeMonitorTime. |
| C003 | The function block detected a static TRUE signal at Reset. | Verify the reset control device and its wiring. The Reset signal must have the FALSE status. |
| C004 | The function block detected a static TRUE signal at Reset. | Verify the reset control device and its wiring. The Reset signal must have the FALSE status. |
| C100 | The function block detected a static TRUE signal at S\_SetMode. | Verify the operating mode release control device connected to S\_SetMode and the associated wiring. |
