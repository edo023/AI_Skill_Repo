# S\_AnyModeSel output

Output for the function block message whether a signal combination is output at the S\_Mode0Sel to S\_Mode7Sel outputs. The output indicates that a signal combination is present at the S\_ModeXSel outputs, with which an operating mode can be controlled in the subsequent application.

Data type: SAFEBOOL

**Connection:** Connect the S\_AnyModeSet output to an output terminal of your application, either directly or via other safety-related functions/function blocks.

**NOTE:**

The signal does not supply any information at which of the S\_Mode0Sel to S\_Mode7Sel outputs a SAFETRUE signal will be output.

| Value | Description |
| --- | --- |
| SAFETRUE | A SAFETRUE signal is output at one of the S\_Mode0Sel to S\_Mode7Sel outputs to control a specific operating mode in the subsequent application. |
| SAFEFALSE | Only SAFEFALSE signals are output at the S\_Mode0Sel to S\_Mode7Sel outputs. |
