# Diagnostic codes

The hexadecimal diagnostic codes listed here are output as WORD data type at the DiagCode output of the function block.

| Code (hex) | Description | Options for remedy |
| --- | --- | --- |
| 0000 | The safety-related function block is not activated (Activate = FALSE). | Activate the function block by switching Activate to TRUE. |
| 8000 | The function block is in a normal operating state (S\_EDM\_Out = SAFETRUE). | In case of an unexpected combination at the function block inputs relative to the physical input of the sensors, verify S\_OutControl and the connected hardware. |
| 8001 | The start-up inhibit is specified with S\_StartReset = SAFEFALSE. | Removal of the start-up inhibit with a positive edge at Reset. |
| 8010 | The S\_OutControl input is SAFEFALSE. | To switch S\_EDM\_Out to SAFETRUE, S\_OutControl must be SAFETRUE. |
| C001 | The start-up inhibit is specified with S\_StartReset = SAFEFALSE and Reset is permanently TRUE incorrectly. | Reset must be FALSE. Verify the reset control device and its wiring. |
| C010 | When controlling the contactors with S\_OutControl = SAFETRUE, the feedback signal S\_EDM1 was indeterminable. S\_EDM1 = SAFEFALSE was detected. | * Verify the connected contactors. * Both feedback signals of the switched off contactors must be SAFETRUE. * Then perform a reset afterwards. |
| C011 | There are two possible causes for this error:  * Following a permanent TRUE at Reset, S\_EDM1 switches to SAFETRUE. * A simultaneous edge at Reset and S\_EDM1 (wiring error). | * Switch Reset = FALSE. * Verify the connected contactors. * Verify the reset control device and its wiring. |
| C020 | When controlling the contactors with S\_OutControl = SAFETRUE, the feedback signal S\_EDM2 was indeterminable. S\_EDM2 = SAFEFALSE was detected. | * Verify the connected contactors. * Both feedback signals of the switched off contactors must be SAFETRUE. * Then perform a reset afterwards. |
| C021 | There are two possible causes for this error:  * Following a permanent TRUE at Reset, S\_EDM2 switches to SAFETRUE. * A simultaneous edge at Reset and S\_EDM2 (wiring error). | * Switch Reset = FALSE. * Verify the connected contactors. * Verify the reset control device and its wiring. |
| C030 | When controlling the contactors with S\_OutControl = SAFETRUE, the feedback signals S\_EDM1 and S\_EDM2 were indeterminable. S\_EDM1 = SAFEFALSE and S\_EDM2 = SAFEFALSE was detected. | * Verify the connected contactors. * Both feedback signals of the switched off contactors must be SAFETRUE. * Then perform a reset afterwards. |
| C031 | There are two possible causes for this error:  * Following a permanent TRUE at Reset, S\_EDM1 and S\_EDM2 switch to SAFETRUE. * A simultaneous edge at Reset and S\_EDM1 and S\_EDM2 (wiring error). | * Switch Reset = FALSE. * Verify the connected contactors. * Verify the reset control device and its wiring. |
| C040 | The connected contactors are not controlled. Once the time specified at MonitoringTime had elapsed, the feedback signal S\_EDM1 was indeterminable. S\_EDM1 = SAFEFALSE was detected. | * Verify the connected contactors. * Both feedback signals of the switched off contactors must be SAFETRUE. * Then perform a reset afterwards. |
| C041 | An error was detected once the time specified at MonitoringTime elapsed. There are two possible error causes:  * Following a permanent TRUE at Reset, S\_EDM1 switches to SAFETRUE. * A simultaneous edge at Reset and S\_EDM1 (wiring error). | * Switch Reset = FALSE. * Verify the connected contactors. * Verify the reset control device and its wiring. * Both feedback signals of the switched off contactors must be SAFETRUE. |
| C050 | The connected contactors are not controlled. Once the time specified at MonitoringTime had elapsed, the feedback signal S\_EDM2 was indeterminable. S\_EDM2 = SAFEFALSE was detected. | * Verify the connected contactors. Both feedback signals of the switched off contactors must be SAFETRUE. * Then perform a reset afterwards. |
| C051 | An error was detected once the time specified at MonitoringTime elapsed. There are two possible error causes:  * Following a permanent TRUE at Reset, S\_EDM2 switches to SAFETRUE. * A simultaneous edge at Reset and S\_EDM2 (wiring error). | * Switch Reset = FALSE. * Verify the connected contactors. * Verify the reset control device and its wiring. * Both feedback signals of the switched off contactors must be SAFETRUE. |
| C060 | The connected contactors are not controlled. Once the time specified at MonitoringTime had elapsed, the feedback signals S\_EDM1 and S\_EDM2 were indeterminable. S\_EDM1 = SAFEFALSE and S\_EDM2 = SAFEFALSE was detected. | * Verify the connected contactors. * Both feedback signals of the switched off contactors must be SAFETRUE. * Then perform a reset afterwards. |
| C061 | An error was detected once the time specified at MonitoringTime elapsed. There are two possible error causes:  * Following a permanent TRUE at Reset, S\_EDM1 and S\_EDM2 switch to SAFETRUE. * A simultaneous edge at Reset and S\_EDM1 and S\_EDM2 (wiring error). | * Switch Reset = FALSE. * Verify the connected contactors. * Verify the reset control device and its wiring. * Both feedback signals of the switched off contactors must be SAFETRUE. |
| C070 | The connected contactors were controlled. Once the time specified at MonitoringTime had elapsed, the feedback signal S\_EDM1 was indeterminable. S\_EDM1 = SAFETRUE was detected. | * Verify the connected contactors. * Then perform a reset. |
| C071 | The connected contactors are controlled. Once the time specified at MonitoringTime had elapsed, the feedback signal S\_EDM1 was indeterminable. S\_EDM1 = SAFETRUE was detected. At the same time, a permanent TRUE was detected at Reset. | * Switch Reset = FALSE. * Verify the connected contactors. * Verify the reset control device and its wiring. |
| C080 | The connected contactors were controlled. Once the time specified at MonitoringTime had elapsed, the feedback signal S\_EDM2 was indeterminable. S\_EDM2 = SAFETRUE was detected. | * Verify the connected contactors. * Then perform a reset. |
| C081 | The connected contactors are controlled. Once the time specified at MonitoringTime had elapsed, the feedback signal S\_EDM2 was indeterminable. S\_EDM2 = SAFETRUE was detected. At the same time, a permanent TRUE was detected at Reset. | * Switch Reset = FALSE. * Verify the connected contactors. * Verify the reset control device and its wiring. |
| C090 | The connected contactors were controlled. Once the time specified at MonitoringTime had elapsed, the feedback signals S\_EDM1 and S\_EDM2 were indeterminable. S\_EDM1 and S\_EDM2 = SAFETRUE was detected. | * Verify the connected contactors. * Then perform a reset. |
| C091 | The connected contactors are controlled. Once the time specified at MonitoringTime had elapsed, the feedback signals S\_EDM1 and S\_EDM2 were indeterminable. S\_EDM1 and S\_EDM2 = SAFETRUE was detected. At the same time, a permanent TRUE was detected at Reset. | * Switch Reset = FALSE. * Verify the connected contactors. * Verify the reset control device and its wiring. |
| C111 | The function block has detected a simultaneous signal change from FALSE to TRUE at S\_OutControl and Reset. | * Verify the connection at S\_OutControl and Reset. Reset and S\_OutControl must be controlled by two different signal sources. * Remove the connection error where required. * Switch Reset to FALSE to exit the error state. |
