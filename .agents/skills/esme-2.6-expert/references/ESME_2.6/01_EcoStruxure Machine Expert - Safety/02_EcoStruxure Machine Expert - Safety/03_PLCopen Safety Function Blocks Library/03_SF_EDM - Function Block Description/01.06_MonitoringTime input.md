# MonitoringTime input

Specification of the maximum permissible response time of the connected contactors.

Data type: TIME

Initial value: #0ms

**Connection**: Connect this input to a time literal, i.e., to a constant of the data type TIME.

Once the S\_EDM\_Out output switches to SAFETRUE, the feedback signal at the S\_EDM1 and S\_EDM2 inputs must become SAFEFALSE within the response time specified here. Otherwise, the function block outputs an error message (output Error = TRUE) and the S\_EDM\_Out output switches to the defined safe state (SAFEFALSE).

The process of switching off the connected contactors is also time-monitored: if the S\_EDM\_Out output switches from SAFETRUE to SAFEFALSE, the switching operation of the contactors is also monitored. The feedback signal at the S\_EDM1 and S\_EDM2 inputs must become SAFETRUE within the permissible response time set at MonitoringTime.
