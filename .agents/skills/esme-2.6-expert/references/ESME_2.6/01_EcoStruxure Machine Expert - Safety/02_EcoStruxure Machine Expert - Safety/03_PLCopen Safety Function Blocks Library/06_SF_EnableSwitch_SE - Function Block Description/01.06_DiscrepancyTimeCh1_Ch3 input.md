# DiscrepancyTimeCh1\_Ch3 input

Specification of the permissible discrepancy time for monitoring the equivalent switching operation at the inputs S\_EnableSwitchCh1 and S\_EnableSwitchCh3.

Data type: TIME

Initial value: #0ms

**Connection**: Connect this input to a time literal, i.e., to a constant of the data type TIME.

**Background:** As a result of mechanical influences, signals may have a tolerance when switching contacts. By specifying the DiscrepancyTimeCh1\_Ch3 value, the time period is defined within which the switching operations at the inputs S\_EnableSwitchCh1 and S\_EnableSwitchCh3 must occur to be recognized as valid. This means both inputs can switch to SAFETRUE or SAFEFALSE within the set discrepancy time.

**NOTE:**

The switchover from SAFEFALSE to SAFETRUE is monitored for both inputs, as well as the switchover from SAFETRUE to SAFEFALSE.

**NOTE:**

If the discrepancy time is too short, the function block may enter the error state C030.
