# SET\_MSB

This standard [bit manipulation function](BitManipulationFunctions_Overview.html#BitManipulationFunctions_Overview) contained in the <BIT\_UTIL> library writes the value applied at input MSB to the most significant Byte of the bit string applied at input DATA.

| Parameter | Data type | Description |
| --- | --- | --- |
| MSB | BYTE | Value to be written to the MSB of DATA |
| DATA | WORD | Input bit string to be modified |
| OUT | WORD | Output bit string with modified written MSB |
