# EQ / EQ\_S - Comparison: =

This comparison function compares the operands connected to the input parameters in regard to equality. Comparison proceeds from left to right. The function is available as standard function EQ and safety-related function EQ\_S.

## EQ

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | ELEMENTARY | First input value |
| IN2 | ELEMENTARY | Second input value |
| OUT | BOOL | TRUE if the inputs are equal.  FALSE if the inputs are not equal. |

## EQ\_S

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | SAFEELEMENTARY | First input value |
| IN2 | SAFEELEMENTARY | Second input value |
| OUT | SAFEBOOL | TRUE if the inputs are equal.  FALSE if the inputs are not equal. |

**NOTE:**

All input parameters must have the same data type.
