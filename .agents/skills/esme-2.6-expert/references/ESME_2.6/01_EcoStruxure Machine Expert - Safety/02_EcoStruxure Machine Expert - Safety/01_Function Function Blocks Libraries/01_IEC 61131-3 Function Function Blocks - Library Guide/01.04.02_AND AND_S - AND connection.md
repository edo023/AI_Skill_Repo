# AND / AND\_S - AND connection

This bitwise Boolean function realizes a logical AND integration of the operands connected to the input parameters. It is available as standard function AND and safety-related function AND\_S.

## AND

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | ANY\_BIT | Input value |
| IN2 | ANY\_BIT | Input value |
| OUT | ANY\_BIT | Output value |

## AND\_S

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | ANY\_SAFEBIT | Input value |
| IN2 | ANY\_SAFEBIT | Input value |
| OUT | ANY\_SAFEBIT | Output value |

**NOTE:**

Input parameters can be duplicated.

**NOTE:**

All parameters must have the same data type.
