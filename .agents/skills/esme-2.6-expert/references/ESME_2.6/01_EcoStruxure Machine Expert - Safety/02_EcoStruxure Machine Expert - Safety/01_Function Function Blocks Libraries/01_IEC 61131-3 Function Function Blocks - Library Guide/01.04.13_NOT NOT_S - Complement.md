# NOT / NOT\_S - Complement

This bitwise Boolean function negates bitwise the operand connected to the input parameter. The function is available as standard function NOT and safety-related function NOT\_S.

## NOT

| Parameter | Data types | Description |
| --- | --- | --- |
| IN | ANY\_BIT | Input value |
| OUT | ANY\_BIT | Output value |

## NOT\_S

| Parameter | Data types | Description |
| --- | --- | --- |
| IN | ANY\_SAFEBIT | Input value |
| OUT | ANY\_SAFEBIT | Output value |
