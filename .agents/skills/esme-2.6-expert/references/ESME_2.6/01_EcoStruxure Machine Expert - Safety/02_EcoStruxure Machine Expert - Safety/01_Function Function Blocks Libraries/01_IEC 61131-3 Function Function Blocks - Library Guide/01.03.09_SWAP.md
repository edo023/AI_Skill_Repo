# SWAP

This standard [bit manipulation function](BitManipulationFunctions_Overview.html#BitManipulationFunctions_Overview) contained in the <BIT\_UTIL> library interchanges the value of the least significant Byte (LSB) and the most significant Byte (MSB) of the bit string applied at input IN.

| Parameter | Data type | Description |
| --- | --- | --- |
| IN | WORD | Input bit string |
| OUT | WORD | Output bit string, values of LSB and MSB swapped |

EIO0000002267.00
