# XOR / XOR\_S - XOR connection

This bitwise Boolean function realizes a logical exclusive-OR-integration of the operands connected to the input parameters. The function is available as standard function XOR and safety-related function XOR\_S.

## XOR

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | ANY\_BIT | First input value |
| IN2 | ANY\_BIT | Second input value |
| OUT | ANY\_BIT | Output value |

## XOR\_S

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | ANY\_SAFEBIT | First input value |
| IN2 | ANY\_SAFEBIT | Second input value |
| OUT | ANY\_SAFEBIT | Output value |

**NOTE:**

All parameters must have the same data type.
