# How to Insert/Delete Libraries

**NOTE:**

Worksheets of included libraries cannot be opened.

**NOTE:**

When zipping the project into an archive using the 'Save as zipped project' option in the 'Save/Zip project as' dialog, included libraries will not be contained in the resulting archive file.

**NOTE:**

Libraries can only be inserted/deleted if you have logged on with the correct project password ('Project > Change Log On Level' menu item).

## How to include a library - Dialog 'Insert Library'

1. In the project tree, right-click the 'Libraries' folder and select 'Insert Library...' from the context menu.

   Or:

   Select 'Project > Add Library'.

   In both cases the 'Insert Library' dialog appears.
2. Select the desired library and click 'OK'.

The library icon is inserted in the project tree.

## Deleting included libraries

1. In the project tree, left-click the library icon you want to delete.
2. Press the <DEL> key.
3. Confirm the deletion in the appearing security message dialog.
