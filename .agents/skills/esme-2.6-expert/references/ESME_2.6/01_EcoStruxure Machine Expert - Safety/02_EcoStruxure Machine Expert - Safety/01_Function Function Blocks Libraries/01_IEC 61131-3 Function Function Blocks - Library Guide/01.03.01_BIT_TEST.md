# BIT\_TEST

This standard [bit manipulation function](BitManipulationFunctions_Overview.html#BitManipulationFunctions_Overview) contained in the <BIT\_UTIL> library reads the value of a particular bit in the bit string applied at input IN.

| Parameter | Data type | Description |
| --- | --- | --- |
| IN | ANY\_BIT | Input bit string |
| BIT\_NO | BYTE | Number (position) of the bit to be read |
| OUT | BOOL | Value of the read (tested) bit |

ANY\_BIT contains the data types DWORD, WORD, BYTE and BOOL.
