# OR / OR\_S - OR connection

This bitwise Boolean function realizes a logical OR integration of the operands connected to the input parameters. The function is available as standard function OR and safety-related function OR\_S.

## OR

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | ANY\_BIT | First input value |
| IN2 | ANY\_BIT | Second input value |
| OUT | ANY\_BIT | Output value |

## OR\_S

| Parameter | Data types | Description |
| --- | --- | --- |
| IN1 | ANY\_SAFEBIT | First input value |
| IN2 | ANY\_SAFEBIT | Second input value |
| OUT | ANY\_SAFEBIT | Output value |

**NOTE:**

Input parameters can be duplicated.

**NOTE:**

All input parameters must have the same data type.
