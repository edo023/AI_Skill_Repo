# Error: Illegal access of variable ...!

An invalid assignment of a value to a function block input or output has been detected.

According to the IEC 61131 standard, the following assignments are **invalid**:

* Writing an input in the same function block.
* Reading the input of another function block.
* Writing an output of another function block

Double-click the error in the message window to jump to the suspected worksheet (where the invalid assignment is marked) and correct the code.
