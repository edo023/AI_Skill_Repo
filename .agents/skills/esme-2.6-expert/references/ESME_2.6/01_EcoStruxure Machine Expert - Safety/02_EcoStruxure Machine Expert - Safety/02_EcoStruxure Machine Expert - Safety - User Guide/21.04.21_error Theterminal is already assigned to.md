# Error: The terminal is already assigned to another variable! (Specific code generator)

A process data item is connected to more than one variable. Each signal can only be connected to one global variable to ensure unique variable-terminal-assignments.

Double-click the error in the message window to jump to the global variables worksheet where the 'Terminal' field of the affected variable is marked red. Assign the variable to a different terminal or delete it. Verify and rework the code afterwards.
