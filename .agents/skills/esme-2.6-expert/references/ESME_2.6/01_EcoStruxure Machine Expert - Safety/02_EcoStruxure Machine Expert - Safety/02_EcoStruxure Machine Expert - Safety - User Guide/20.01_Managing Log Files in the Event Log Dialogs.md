# Managing Log Files in the Event Log Dialogs

The dialogs 'Project Event Log' and 'System Event Log' are part of the [event and error log system](eventanderrorlog.html#eventanderrorlog). They appear after selecting 'File > Event Log...'.

Both dialogs are used to manage logged events (project-related or system-related). They are structured identically, the contained events are handled in the same way.

Using the 'System Events' button in the 'Project Event Log' dialog and the 'Project Events' button in the 'System Event Log' dialog switches between the different log types.

The dialog is used to

* open event log entries and already exported project log files (with filter options).
* export the loaded event log to a file (\*.eve) which can be opened later for inspection purposes.
* reset the event log.

**NOTE:**

The export and reset functions are only available if you are logged-on at 'Development' level (see topic ["Password protection"](PasswordProtection.html#PasswordProtection)).

**NOTE:**

Each operation (button) is only applied to the currently loaded log type. Resetting the system event log does, for example, not affect the project event log.

| **Dialog element** | **Meaning** |
| --- | --- |
| Event list | Shows the project event log or system event log entries (depending on the loaded log type).  Additionally, the event list provides a sorting function. Each column can be used as sorting criterion by clicking the column header. |
| System Events | Opens the System Event Log where the system-related events are recorded.  (Button is only available if the project event log file is visible.) |
| Project Events | Opens the Project Event Log where all project-related events are recorded.  (Button is only available if the system event log file is visible.) |
| View export | Calls the 'Open event log' dialog to select and open exported (archived) event log files for inspection. |
| Export | Opens the export dialog to save the present project log file for later use. |
| Filter | Opens the 'Filter options' dialog that provides various options to [filter the event list](dialogoptionsintheeventlogdialog.html#dialogoptionsintheeventlogdialog) and therefore to reduce the number of displayed events. |
| Reset | Clears the event list. After this, the event log contains only one entry which displays who has initiated the reset. |
| Close | Closes the event log dialog. |
