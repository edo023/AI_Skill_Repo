# Line

Lines connect objects in FBD/LD code. Branching points and junctions are possible ("1-to-n" connections).

When moving connected objects, the editor automatically rearranges the connection lines.

When deleting contacts/coils in an LD network, the resulting gap is automatically closed by a new line.

The values transferred on a connection line are of a particular data type according to the objects the line connects. In most cases, Machine Expert – Safety can determine this data type automatically. In particular network structures, for example, if functions are connected directly to each other, Machine Expert – Safety is possibly not able to automatically determine the data type and needs an internal temporary flag for compilation. For this purpose the ['Flag Type' dialog](changingtheflagtype.html#changingtheflagtype) is used.
