# Contact/Coil: Modifying Width

This topic describes how to modify the default width of contacts and coils (set in the 'Graphical editor' tab in the ['Options' dialog](customizingtheuserinterface_dialog_options.html#customizingtheuserinterface_dialog_options)) for the present worksheet. The contact/coil width in other graphical worksheets is not affected.

The width set with this procedure is applied to LD objects that are inserted afterwards. The size of existing contacts/coils is not affected.

1. Select the menu item 'Layout > Contact Width...'. The dialog 'Contact width' appears.
2. Select a value from the 'Width' drop-down combo box and confirm the dialog.

   New contacts/coils will now be inserted with the set width.

You can also modify the **default** width of contacts and coils by selecting 'Project > Options | Graphical Editor tab' and editing the value for 'Contact width'. This affects all contacts and coils inserted **afterwards**. Existing contacts and coils keep their width.
