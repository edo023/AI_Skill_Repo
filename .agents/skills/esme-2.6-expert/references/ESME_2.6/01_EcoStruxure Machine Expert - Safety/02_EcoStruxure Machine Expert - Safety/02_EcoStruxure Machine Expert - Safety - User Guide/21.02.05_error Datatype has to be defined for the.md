# Error: Data type has to be defined for the connection point!

If functions in FBD are connected to each other via a direct line (from connection point to connection point), Machine Expert – Safety may need an internal temporary flag for compiling the project.

If the data type of an internal flag is not determined by the structure of the network you have do this manually using the 'Flag Type' dialog. Proceed as follows:

1. Double-click the suspected connection line in the FBD network or select 'Object Properties...' from its context menu.
2. In the ['Flag type' dialog](changingtheflagtype.html#changingtheflagtype), open the drop-down combo box and select a data type.
