# Importing/Exporting Projects

Machine Expert – Safety is directly launched from the Safety Logic Controller icon in the Machine Expert 'Devices' tree. This way, the corresponding safety-related project is automatically loaded into Machine Expert – Safety. After closing the safety-related project or exiting Machine Expert – Safety, the (modified) safety-related project is integrated into the Machine Expert project.

Therefore, importing or exporting the safety-related project, for example, to use it in another application, must also be done in Machine Expert.

**Generation change of the functional safety system (SLCv1/SLCv2):** The safety-related devices of the SLCv2 generation have different SafetyResponseTime-relevant parameters than devices from the SLCv1 generation. If a project that has been created and verified with SLCv1 is imported into a SLCv2 Machine Expert project, the migration to SLCv2 is done during the following start of Machine Expert – Safety.

A thorough verification of the overall response time of the safety function must be performed. Observe the following hazard message.

| WARNING | |
| --- | --- |
|  | **NON-CONFORMANCE TO SAFETY FUNCTION REQUIREMENTS**   * Verify, for the SLC and each safety-related device involved in your application, that the values set for the parameters 'SafeDataDuration' and 'ToleratedPacketLoss' correspond to your risk analysis. * Be sure that your risk analysis includes an evaluation for incorrectly setting the values 'SafeDataDuration' and 'ToleratedPacketLoss'. * Validate the overall safety function with regard to the resulting overall response time of the safety function and thoroughly test the application.   **Failure to follow these instructions can result in death, serious injury, or equipment damage.** |

## How to import a project

Importing a safety-related project this way overwrites the present safety-related project.

1. Importing a safety-related project is only possible while Machine Expert – Safety is not running.

   Exit the safety-related programming system if still running.
2. In the Machine Expert 'Devices' tree, right-click the SLC icon and select 'Machine Expert – Safety > Import project' from the context menu.

   The 'Open' dialog appears.
3. Enter the name and the path of the repository file to be imported or browse for the desired file.
4. Click 'Open' to import the file.
5. In the following message box, click 'Yes' to import the project, thus overwriting the existing safety-related project.

   **NOTE:**

   The overwritten safety-related project cannot be restored.

   Clicking 'No' cancels the import process and keeps the present safety-related project.

## How to export a project

1. In the Machine Expert 'Devices' tree, right-click the SLC icon and select 'Machine Expert – Safety > Export project' from the context menu.

   The 'Save as' dialog appears.
2. Select a directory, enter a file name for the repository file (\*.spa) to be exported, and click 'Save'.
