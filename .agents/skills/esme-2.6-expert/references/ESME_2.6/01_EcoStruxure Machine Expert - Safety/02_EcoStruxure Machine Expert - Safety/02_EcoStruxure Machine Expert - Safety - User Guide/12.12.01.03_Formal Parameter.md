# Formal Parameter

The inputs and outputs of [functions and function blocks](functionandfunctionblock.html#functionandfunctionblock) are referred to as formal parameters.

For basic and firmware library blocks, the properties of a formal parameter can be displayed using the ['Formal Parameter' dialog](dialog_formalparameter.html#dialog_formalparameter). To call this dialog either double-click on a formal parameter (inside the FU/FB block symbol) or right-click the formal parameter and select 'Object Properties...' from the context menu.

In case of AND/AND\_S and OR/OR\_S functions, you can [duplicate inputs and delete previously duplicated inputs](DuplicatingAND_OR_Inputs.html#DuplicatingAND_OR_Inputs).
