# Error Catalog

The error catalog lists the error messages and alerts which can be displayed in the message window of Machine Expert – Safety.

These are the main categories of error messages:

* [Error messages of the ST codeeditor](ErrorCat_SyntacticalOverviewST.html#ErrorCat_SyntacticalOverviewST)
* [Error messages of theFBD/LD compiler and linker](errormessagesofthecompilerandlinker.html#errormessagesofthecompilerandlinker)
* [Error messages ofthe IEC code generator](errormessagesoftheieccodegenerator.html#errormessagesoftheieccodegenerator)
* [Error messagesof the specific code generator](errormessagesofthespecificcodegenerator.html#errormessagesofthespecificcodegenerator)
* [Safety Logic Controller-specific error messages](plcspecificerrormessages.html#plcspecificerrormessages)

You can [usethe message window to display and correct errors and alerts](messagewindow.html#messagewindow__usingthemessagewindowtodisplayandcorrecterrorsandwarnings).
