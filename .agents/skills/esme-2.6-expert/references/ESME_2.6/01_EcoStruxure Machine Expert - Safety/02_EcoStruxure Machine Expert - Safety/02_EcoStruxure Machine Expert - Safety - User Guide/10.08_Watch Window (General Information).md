# Watch Window (General Information)

The watch window is a powerful startup and monitoring tool. It helps you to see how variables work together as their online values can be seen at a glance when running the project on the Safety Logic Controller or the integrated simulation. For that purpose, elements from the 'Main' program or different function block instances can be "collected" in the watch window.

In online mode (safe mode and debug mode), the watch window displays the following information for each variable that has been added to a watch list:

* Name
* Data type
* Current value
* Instance path

**Further Information:**

For information how to add/remove variables to/from the watch window and how to call the debug dialog, refer to the topic ["Using the watch window"](watchwindow.html#watchwindow).

To display the watch window, select 'View > Watch Window'.

In online mode, you can also open the watch window by selecting 'Open Watch Window...' from the context menu of a variable.

How to adjust the window

You can adjust the window by undocking it, modifying its size, moving it to another screen position, etc. Further information can be found in the topic "[Adjusting windows](customizingtheuserinterface_dialog_options.html#customizingtheuserinterface_dialog_options__AdjustingUIWindows)".

Auto-hide function (floating windows): Use the [auto-hide function](customizingtheuserinterface_dialog_options.html#customizingtheuserinterface_dialog_options__AdjustingUIWindows) to automatically hide (minimize) the window if it is unused. For that purpose switch on the auto-hide function by clicking the ![](images/ico_AutoHideWindow_Off.gif) icon on the window control bar. If the auto-hide function is switched on for that window, the ![](images/ico_AutoHideWindow_On.gif) icon is shown. In this case, position the mouse pointer over the minimized window to show it again.
