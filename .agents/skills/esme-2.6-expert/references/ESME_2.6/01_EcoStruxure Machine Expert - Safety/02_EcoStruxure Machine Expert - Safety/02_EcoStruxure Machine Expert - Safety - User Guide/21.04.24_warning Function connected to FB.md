# Warning: Function connected to FB input-parameter is called after FB call. (SCG)

This message may occur if functions and function blocks are combined in a particular way.

To eliminate the alert proceed as follows:

1. In the message window, double-click the entry.

   The worksheet opens and the cursor marks the affected function block input.
2. In the code worksheet, look for a function

   1. whose execution number is lower than the number of the marked function block,
   2. **and** whose output is connected to the input of a function with a higher execution number than that of the marked function block.
3. Having found this combination, replace the connection at the output of the function (with the low execution number) by an intermediate variable.

   **NOTE:**

   Since this modification may change the execution order, you should divide the network into two separate networks.
4. Repeat steps 1 to 3 for each message of this type.
