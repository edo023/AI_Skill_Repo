# Error: FB instance '...' is used more than once!

A function block instance is declared more than once in a variables worksheet. According to the IEC 61131 standard, FB instance names must be unique within one POU.

1. In the message window, double-click the error message.

   The variables worksheet opens and the second instance declaration is marked.
2. Delete or rename the suspected instance.

   In case of renaming, you have to modify the related code accordingly.
