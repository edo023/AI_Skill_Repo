# Menu Bar

A short description of each selected menu item is displayed in the status bar. The use of the menu items is explained in the corresponding procedures referenced in the following table.

**NOTE:**

Which items are available depends on the program part or editor you are working with.

**NOTE:**

Machine Expert – Safety can also be operated via [keyboard shortcuts](keyboardshortcuts.html#keyboardshortcuts).

|  |  |
| --- | --- |
| File | Provides commands for   * saving single worksheets. * printing, print setup, and print preview. * opening the [event and error log](eventanderrorlog.html#eventanderrorlog) of the project. * exiting Machine Expert – Safety. |
| Edit | Provides commands for   * selecting, copying, cutting, pasting objects. * [searching elements and text strings](findingtextelementsdialogfind.html#findingtextelementsdialogfind) in code worksheets and variables worksheets as well as in the project tree. * FBD/LD object handling. |
| View | Provides commands for   * hiding or showing the [project tree window](projecttree_overview.html#projecttree_overview), [message window](messagewindow.html#messagewindow), [Edit Wizard](editwizard_generaldescription.html#editwizard_generaldescription), [cross references window](crossreferencewindow.html#crossreferencewindow), [watch window](watchwindow.html#watchwindow), and Bus Navigator. * toggling between code and variables worksheet of the edited POU. * opening the global variables declarations. |
| Project | Provides commands for   * [adding new FB POUs](subtree_logicalpous_intheprojecttree.html#subtree_logicalpous_intheprojecttree) to the project. * [inserting (announcing) libraries](subtree_library_intheprojecttree.html#subtree_library_intheprojecttree) with reusable function blocks. * [comparing projects](comparingprojectsources.html#comparingprojectsources). * calculating the safety-related response time of your safety function: 'Response Time Relevant Parameters' (only active for device generation SLCv1) and 'Response Time Calculator'. See topics ["Safety Response Time for SLCv1"](wp100881.html#wp100881) and ["Safety Response Time for SLCv2"](SLCv2_wp100881.html#SLCv2_wp100881) * calling the ['Project Info' dialog](ProjectInformation.html#ProjectInformation) which summarizes important (editable) information. * calling the certification manager for certifying a completed project after successful commissioning.  Refer to the topic ["Project certification"](CertificationManager.html#CertificationManager) for details. * logging on to and off from the project and defining a new [project password](PasswordProtection.html#PasswordProtection). * [managing the translation process](translatingaproject.html#translatingaproject) for comments and description worksheets in the project ('Project Translation Management') and switching the project language after having imported the translated text ('Change Project Language'). * calling the 'Options' dialog to [customize the user interface](customizingtheuserinterface_dialog_options.html#customizingtheuserinterface_dialog_options). * creating an application image file (command 'Application Download') which can be downloaded to the Safety Logic Controller. The download is handled by a special function block in the standard controller application (i.e., without using Machine Expert – Safety).  Refer to the topic ["Application Download"](SE_ApplicationDownload.html#SE_ApplicationDownload) for further information. |
| Build | Provides commands for   * [compiling the project](compiling.html#compiling). * stopping the compiler. * [building cross references](crossreferencewindow.html#crossreferencewindow). |
| Objects | Commands in the 'Objects' menu are only active while using the FBD/LD code editor. It provides commands to insert [FBD/LD objects](objectsinthegraphiceditor.html#objectsinthegraphiceditor) into the code and to modify object properties. |
| Layout | While editing code in FBD/LD code worksheets, the 'Layout' menu provides several utilities (e.g., you can display a [grid](usingagridwheninsertingobjectsinagraphicalworksheet.html#usingagridwheninsertingobjectsinagraphicalworksheet)). Furthermore, you can zoom into and out of the worksheet and [modify the default contact/coil width](changingthewidthofcontactsorcoils.html#changingthewidthofcontactsorcoils).  While working in a variables worksheet, the 'Layout' menu contains zoom commands and a menu item for resetting the present grid column width. |
| Online | Provides commands for   * [debugging the application](debuggingtheproject.html#debuggingtheproject). * calling the [control dialog for the Safety Logic Controller](dialogSafePLC.html#dialogSafePLC). * setting the 'Online Layout' of code worksheets in [online mode](debuggingtheproject.html#debuggingtheproject). * logging on to or off from the Safety Logic Controller using the [Safety Logic Controller password](PasswordProtection.html#PasswordProtection) and modifying the password. You have to logon in order to modify the configuration of the Safety Logic Controller and to start up/control the application.  Refer to the topic ["Password Protection for Projects and Safety Logic Controller"](PasswordProtection.html#PasswordProtection) for further information.  **NOTE:**  The menu items relating to the Safety Logic Controller password are only visible if a communication connection is established between the computer and the Safety Logic Controller. This is the case if either the [control dialog](dialogSafePLC.html#dialogSafePLC) is open or the [variable status is displayed (online mode)](DisplayVariableStatus.html#DisplayVariableStatus). To establish a communication connection to the Safety Logic Controller, click the 'SafePLC' icon on the toolbar. |
| Window | Provides commands for arranging windows in the workspace and closing all open windows at once. |
| ? | Provides commands for calling help information. Refer to the topic ["How to use the Online Help System"](welcometotheonlinehelpsystem.html#welcometotheonlinehelpsystem) for detailed information. |
