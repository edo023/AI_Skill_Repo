# Edit Wizard

The Edit Wizard is used to insert and replace functions and function blocks in code worksheets. It can also be used to [call help on FUs/FBs](editwizard_generaldescription.html#editwizard_generaldescription__CallHelp). The dockable Edit Wizard window is divided into two areas:

* The drop-down combo box 'Group' to select the available [groups](editwizard_generaldescription.html#editwizard_generaldescription__groupsintheeditwizardwhenopenedinacodebodyworksheet) of code elements.
* The selection area which displays the available [elements](editwizard_generaldescription.html#editwizard_generaldescription__groupsintheeditwizardwhenopenedinacodebodyworksheet) contained in the selected group.

To display the Edit Wizard, press <Shift> + <F2> or click the 'Edit Wizard' icon on the toolbar.

![](images/ico_EditWiz.gif)

FUs/FBs are inserted into FBD/LD code via drag & drop. Select the 'Group' which contains the desired block, drag the block into the code worksheet, left-click to insert the block outline and drop it with another left mouse click. The detailed procedure is described in the topic ["Inserting functions or function blocks"](insertingfunctionsorfunctionblocksinthegraphiceditorusingtheeditwizard.html#insertingfunctionsorfunctionblocksinthegraphiceditorusingtheeditwizard).

**NOTE:**

By dropping the block directly on a free connection point of another object or on an existing connection line, you can connect it directly on insertion.

**NOTE:**

The Edit Wizard is also available when programming in ST. It provides the most convenient method to edit function/FB calls in ST by dragging the required code object as template from the Edit Wizard into the code editor. Refer to the topic ["ST Code Development: Functions/Function Blocks: Inserting"](TE_Functions_Insert.html#TE_Functions_Insert).

The color and type of a symbol represent the origin of the related FU or FB:

* Standard IEC 61131 basic FUs: ![](images/help00881.gif)

  Standard IEC 61131 basic FBs: ![](images/help00882.gif)
* Safety-related IEC 61131 basic FUs: ![](images/help0087.gif)

  Safety-related IEC 61131 basic FBs: ![](images/help0088.gif)
* Safety-related library FUs: ![](images/SymbolLibraryFU.gif)

  Safety-related library FBs: ![](images/SymbolLibraryFB.gif)
* User-defined FBs: ![](images/help00883.gif)

Depending on the selected group, the Edit Wizard provides the following FUs/FBs for selection:

* '<all FUs and FBs>': all available functions and function blocks.
* 'Functions': only functions.
* '*project name*': only user-defined function blocks which have already been created in the project.
* '*Libraries name*': each included library is represented as an own group containing the corresponding library FUs and FBs.

## How to call help on a function/function block

1. In the drop-down combo box 'Group', select the desired function or function block group.
2. Right-click on the desired function or function block in the Edit Wizard selection area and select 'Help on FB/FU' from the context menu.

   The context sensitive help topic for the marked object appears.

How to adjust the window

You can adjust the window by undocking it, modifying its size, moving it to another screen position, etc. Further information can be found in the topic "[Adjusting windows](customizingtheuserinterface_dialog_options.html#customizingtheuserinterface_dialog_options__AdjustingUIWindows)".

Auto-hide function (floating windows): Use the [auto-hide function](customizingtheuserinterface_dialog_options.html#customizingtheuserinterface_dialog_options__AdjustingUIWindows) to automatically hide (minimize) the window if it is unused. For that purpose switch on the auto-hide function by clicking the ![](images/ico_AutoHideWindow_Off.gif) icon on the window control bar. If the auto-hide function is switched on for that window, the ![](images/ico_AutoHideWindow_On.gif) icon is shown. In this case, position the mouse pointer over the minimized window to show it again.
