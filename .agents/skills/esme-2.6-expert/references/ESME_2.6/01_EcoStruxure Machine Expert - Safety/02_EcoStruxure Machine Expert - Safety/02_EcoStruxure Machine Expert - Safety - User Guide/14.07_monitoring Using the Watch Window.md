# Monitoring: Using the Watch Window

The watch window can be used to collect variables from different worksheets in so-called watch lists.

While Machine Expert – Safety runs in online mode (variable status is displayed), the watch window shows the name, data type, present value for each variable as well as the instance path where the variable is used. In this way, you can see how these variables interact as their online values can be seen at a glance.

In online mode, the watch window shows the same [online colors](DisplayVariableStatus.html#DisplayVariableStatus__VarStatus_OnlineLayout) as in code worksheets.

The watch window can be used in [safe mode and debug mode](SafePLC_OperatingModes.html#SafePLC_OperatingModes).

* While the Safety Logic Controller is running in safe mode and the worksheets are switched to variable status, you can add variables to the watch window (via their context menu) and display their online values. The number of variables is not limited in the watch window.
* If the Safety Logic Controller is switched to debug mode, you can additionally use the watch window to force and overwrite variables, i.e., to debug the project. This is done by marking a variable in the watch list and choosing ['Debug dialog...'](dialog_debug_resource_name.html#dialog_debug_resource_name) from the context menu of a variable (refer to the description below).

**NOTE:**

The variable selected in the watch window is also automatically marked in the [cross references window](crossreferencewindow.html#crossreferencewindow).

## How to add and remove variables from the watch window

Adding and removing variables is possible while variable status is active in either safe mode or debug mode.

* To **add** a variable to the watch window, right-click on the desired variable in the online worksheet and select 'Add to Watch Window' from the context menu.
* You can **remove** individual variables from the watch list or clear the whole list.

  To remove an individual variable from the watch list, select the variable to be deleted and press <Del> or choose 'Delete' from its context menu. To remove all variables, right-click anywhere in the watch window and select 'Clear' from the context menu.

## How to call the 'Debug' dialog from the watch window

Debugging in the watch window is only possible if the Safety Logic Controller runs in [debug mode](SafePLC_OperatingModes.html#SafePLC_OperatingModes__SwitchSafeDebug).

Right-click on the desired variable in a watch list and select 'Debug dialog...' from the context menu. The ['Debug' dialog](dialog_debug_resource_name.html#dialog_debug_resource_name) appears where you can force I/O variables and overwrite local variables.

How to adjust the window

You can adjust the window by undocking it, modifying its size, moving it to another screen position, etc. Further information can be found in the topic "[Adjusting windows](customizingtheuserinterface_dialog_options.html#customizingtheuserinterface_dialog_options__AdjustingUIWindows)".

Auto-hide function (floating windows): Use the [auto-hide function](customizingtheuserinterface_dialog_options.html#customizingtheuserinterface_dialog_options__AdjustingUIWindows) to automatically hide (minimize) the window if it is unused. For that purpose switch on the auto-hide function by clicking the ![](images/ico_AutoHideWindow_Off.gif) icon on the window control bar. If the auto-hide function is switched on for that window, the ![](images/ico_AutoHideWindow_On.gif) icon is shown. In this case, position the mouse pointer over the minimized window to show it again.
