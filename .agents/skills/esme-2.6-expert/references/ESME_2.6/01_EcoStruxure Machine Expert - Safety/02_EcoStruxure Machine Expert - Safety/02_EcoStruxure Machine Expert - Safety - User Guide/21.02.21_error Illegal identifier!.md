# Error: Illegal identifier!

An invalid variable or FB instance name is used.

According to the IEC 61131 standard, identifiers must consist of letters or digits and underscores. The use of multiple leading or embedded underscores is not allowed. Furthermore, DIN qualifiers are supported. Refer to the topic ["IEC 61131 Implementation -Identifiers"](identifier.html#identifier) for details.

1. In the message window, double-click the error message.

   The worksheet opens and the cursor marks the error.
2. Correct the identifier in the code and, if required, in the variables worksheet.
