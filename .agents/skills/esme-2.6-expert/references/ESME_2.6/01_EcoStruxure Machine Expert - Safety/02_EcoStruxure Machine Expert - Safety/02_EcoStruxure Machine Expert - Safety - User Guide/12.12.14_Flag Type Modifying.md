# Flag Type: Modifying

The values transferred on a connection line are of a particular data type according to the objects the line connects. In most cases, Machine Expert – Safety can determine this data type automatically.

In particular network structures, for example, if functions are connected directly to each other, Machine Expert – Safety is possibly not able to automatically determine the data type and needs an internal temporary flag for compilation. For that purpose, the 'Flag Type' dialog is provided.

If the flag type has to be set manually, you normally get a compilation error in the message window. Double-clicking the error message directly jumps to the code position where the flag is needed. In the code worksheet, proceed as follows:

1. Double-click the suspected connection line or select 'Object Properties...' form its context menu.

   The dialog 'Flag Type Properties' appears.
2. Modify the data type of the flag and confirm the dialog 'Flag Type'.
