# Error: Variable name must be different to POU name!

A variable is declared with the same name as a POU ('Main' for example). Names must be unique within a project.

Double-click the error in the message window to jump to the suspected variables worksheet (where the declaration is marked). Rename or delete the suspected declaration. Verify the code affected by the deletion/renaming afterwards and modify, if necessary.
