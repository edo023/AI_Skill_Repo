# Error: Object not connected!

This error message can have different reasons:

* An object in an FBD/LD code worksheet is not connected.
* An internal error is detected in the code editor or during compilation. The FBD/LD network could not be compiled.

Double-click the error in the message window to jump to the suspected worksheet (where the error is marked) and connect unconnected objects or correct invalid LD networks.

If the error is reported again, contact your local Schneider Electric representative.
