# Error: Invalid initial value!

An initial value is specified in a variable declaration which does not match the data type of the variable.

Error example: Initial value '45' for a variable of the data type BOOL.

Double-click the error in the message window to jump to the suspected worksheet (where the error is marked) and correct either the data type of the variable or the initial value.

Refer to the topic ["IEC 61131 Implementation -Variables"](VariablesinSafeFox.html#VariablesinSafeFox__InitVariables) for detailed information.
