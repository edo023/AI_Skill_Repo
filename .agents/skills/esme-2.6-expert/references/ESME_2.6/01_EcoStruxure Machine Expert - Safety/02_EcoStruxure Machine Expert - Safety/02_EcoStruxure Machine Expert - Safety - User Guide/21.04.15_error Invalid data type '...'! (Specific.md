# Error: Invalid data type '...'! (Specific code generator)

A function is connected to a data type which is not allowed.

Double-click the error in the message window to jump to the suspected worksheet (where the error is marked) and correct.

**NOTE:**

Double-clicking the formal parameter inside the block icon opens the ['Formal Parameter' dialog](dialog_formalparameter.html#dialog_formalparameter) where the expected data type is specified.

Refer to the topic ["IEC 61131 Implementation - DataTypes"](datatypesinSafeFOX.html#datatypesinSafeFOX) for details.
