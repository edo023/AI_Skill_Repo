# Error: Physical output written more than once!

A global variable connected to a physical output (process data item) is written more than once.

1. In the message window, double-click the error message.

   The global variables worksheet opens and the affected output variable is marked.
2. Use the [cross references window](crossreferencewindow.html#crossreferencewindow) to locate code positions with write access to the affected variable.
3. Modify the code accordingly.
