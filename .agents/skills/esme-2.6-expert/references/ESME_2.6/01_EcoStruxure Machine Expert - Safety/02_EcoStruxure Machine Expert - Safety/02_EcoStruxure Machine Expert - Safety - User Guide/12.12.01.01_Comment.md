# Comment

Comments are for documentation purposes. They do not affect the behavior of the programmed safety-related application logic. There is no restriction for the number of comments within the program.

In FBD/LD code worksheets, you can insert comments at any free position or assigned to an LD network. Comments may overlap other graphical objects.

Comments in FBD/LD worksheets are [inserted and edited using the 'Comment' dialog](insertingcommentsinthegraphiceditor.html#insertingcommentsinthegraphiceditor).
