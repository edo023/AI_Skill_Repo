# Warning: Contacts / Coils and Variables should not be used in parallel in a ladder network. This could result in a cross comparison error on the controller.

In FBD/LD code, the output of a function/FB output is connected to a parallel coil and variable. If an FU/FB output writes a coil and a variable at the same time, this may result in a cross comparison error on the Safety Logic Controller.

Double-click the entry in the message window to jump to the suspected worksheet and replace the variable by a coil.
