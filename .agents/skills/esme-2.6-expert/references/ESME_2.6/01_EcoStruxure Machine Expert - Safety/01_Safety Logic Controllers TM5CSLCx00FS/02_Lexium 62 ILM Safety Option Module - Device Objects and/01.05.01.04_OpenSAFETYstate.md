# OpenSAFETYstate

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | LXM62 Safety Logic, ILM62 Safety Module, TM5CSLCx00FS |
| Traceable | Yes |

## Functional Description

Displays the status of the OpenSAFETY communication protocol.

| Value | Meaning |
| --- | --- |
| OpenSAFETY in PreOP / 0 | OpenSAFETY communication protocol in pre-operational mode |
| OpenSAFETY in Operational / 1 | OpenSAFETY communication protocol in operational mode |
