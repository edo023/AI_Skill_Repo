# SS1\_StartDelayTime

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the delay period between the request and start of monitoring (t1).

| Value | Data type | Meaning |
| --- | --- | --- |
| 0…300000 | UDINT | Delay period (ms) between the request and start of monitoring (t1). |

For more information, refer to [*Safe Stop 1 (SS1)*](D-SE-0062421.html#D-SE-0062421).
