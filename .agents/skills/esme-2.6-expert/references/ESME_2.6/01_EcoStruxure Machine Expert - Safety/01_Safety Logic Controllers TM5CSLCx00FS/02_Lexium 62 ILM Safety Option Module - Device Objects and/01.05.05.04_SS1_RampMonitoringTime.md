# SS1\_RampMonitoringTime

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the monitoring time of the deceleration ramp (t2).

| Value | Data type | Meaning |
| --- | --- | --- |
| 0…300000 | UDINT | Monitoring time (ms) of the deceleration ramp (t2). |

For more information, refer to [*Safe Stop 1 (SS1)*](D-SE-0062421.html#D-SE-0062421).
