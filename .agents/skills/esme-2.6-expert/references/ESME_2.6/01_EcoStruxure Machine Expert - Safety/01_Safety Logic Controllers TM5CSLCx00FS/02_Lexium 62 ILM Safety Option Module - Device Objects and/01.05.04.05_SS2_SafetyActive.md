# SS2\_SafetyActive

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status of the safety-related function SS2 (Safe Stop 2).

| Value | Meaning |
| --- | --- |
| on / 1 | Safety-related function is active. The motor is at standstill. |
| off / 0 | Safety-related function is not active. |

For more information, refer to [*Safe Stop 2 (SS2)*](D-SE-0062423.html#D-SE-0062423).
