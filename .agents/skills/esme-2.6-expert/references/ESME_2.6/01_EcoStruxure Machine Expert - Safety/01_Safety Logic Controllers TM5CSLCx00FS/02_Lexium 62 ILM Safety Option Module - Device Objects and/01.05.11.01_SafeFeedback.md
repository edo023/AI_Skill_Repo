# SafeFeedback

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

This parameter either transfers the defined safe speed in units/s or the defined safe increment. The transferred value is configured via parameter SafeFeedbackConfig.
