# HW\_STO\_SafetyActive

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status of the hardware inverter enable (external wired on drive).

| Value | Meaning |
| --- | --- |
| on / 1 | HW STO is active. |
| off / 0 | HW STO is inactive. |

EIO0000002293.01
