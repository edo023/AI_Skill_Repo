# SDIPos\_SafetyActive

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status of the safety-related function SDIPos (Safe Direction Positive).

| Value | Meaning |
| --- | --- |
| on / 1 | Safety-related function is active. The axis is moving in positive direction. |
| off / 0 | Safety-related function is not active. |

For more information, refer to [*Safe Direction (SDI)*](D-SE-0062429.html#D-SE-0062429).
