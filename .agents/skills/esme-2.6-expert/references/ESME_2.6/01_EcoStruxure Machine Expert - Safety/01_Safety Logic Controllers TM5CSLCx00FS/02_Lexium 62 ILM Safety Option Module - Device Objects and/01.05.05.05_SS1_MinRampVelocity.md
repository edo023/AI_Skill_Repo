# SS1\_MinRampVelocity

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the stop ramp monitoring at this speed.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0...65535 | LREAL | Monitoring of stop ramp (units/s) at this speed. |

For more information, refer to [*Safe Stop 1 (SS1)*](D-SE-0062421.html#D-SE-0062421).
