# SafetyFunctionRequested

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status if a safety-related function is requested.

| Value | Meaning |
| --- | --- |
| on / 1 | A safety-related function is requested. |
| off / 0 | No safety-related function is requested. |

EIO0000002293.01
