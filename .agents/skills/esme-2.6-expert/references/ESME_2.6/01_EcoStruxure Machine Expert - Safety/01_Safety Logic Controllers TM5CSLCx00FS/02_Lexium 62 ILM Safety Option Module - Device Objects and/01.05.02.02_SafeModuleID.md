# SafeModuleID

## General

|  |  |
| --- | --- |
| Type | EF |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Displays the unique number of this safety-related module.

| Value | Data type | Meaning |
| --- | --- | --- |
| 2…1000 (default = 2) | UINT | Unique number of a safety-related module. |
