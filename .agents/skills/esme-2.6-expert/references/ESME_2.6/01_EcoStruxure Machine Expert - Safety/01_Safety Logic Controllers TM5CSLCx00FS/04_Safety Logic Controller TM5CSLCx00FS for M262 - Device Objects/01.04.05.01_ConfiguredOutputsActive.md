# ConfiguredOutputsActive

## General

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

With this parameter, you can enable and disable the cyclic safety communication.

* Value FALSE (default):

  The cyclic safety communication is disabled.
* Value TRUE:

  The cyclic safety communication is enabled.

The value TRUE is only taken into account in the Sercos state CP4.
