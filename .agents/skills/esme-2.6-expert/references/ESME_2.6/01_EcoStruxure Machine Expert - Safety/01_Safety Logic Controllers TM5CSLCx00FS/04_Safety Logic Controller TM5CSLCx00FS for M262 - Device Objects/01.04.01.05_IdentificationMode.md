## Functional Description

With this parameter, you can configure the type of addressing of a device in the Sercos network.

* Value Topology mode (default):

  Details see parameter ConfiguredTopologyAddress.
* Value Sercos mode:

  Details see parameter ConfiguredSercosAddress.
