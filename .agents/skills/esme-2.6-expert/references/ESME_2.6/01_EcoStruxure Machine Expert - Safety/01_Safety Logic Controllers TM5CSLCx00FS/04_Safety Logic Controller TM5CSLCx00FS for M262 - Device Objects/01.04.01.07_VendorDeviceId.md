# VendorDeviceId

## General

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | No |

## Functional Description

With this parameter, you can read the vendor device ID of the device.

The vendor device ID is a unique number assigned by the vendor to each device and helps identifying a Sercos device in the Sercos network.

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if the device is simulated or if Sercos is not in phase CP2 or higher, an empty string is indicated here.
