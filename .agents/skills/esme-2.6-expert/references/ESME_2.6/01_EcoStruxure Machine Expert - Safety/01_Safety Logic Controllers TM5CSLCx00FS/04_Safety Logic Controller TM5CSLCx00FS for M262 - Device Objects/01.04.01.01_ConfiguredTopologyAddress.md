# ConfiguredTopologyAddress

## General

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

With this parameter, you can configure the topology address of the device. The topology address corresponds to the connected position of the device in the Sercos network.

The configured topology address is only taken into account if the parameter IdentificationMode is set to Topology mode.

The hardware switches of the device for the Sercos address have to be set to 0 in order to use the identification mode Topology mode, for details see the hardware guide of the device.
