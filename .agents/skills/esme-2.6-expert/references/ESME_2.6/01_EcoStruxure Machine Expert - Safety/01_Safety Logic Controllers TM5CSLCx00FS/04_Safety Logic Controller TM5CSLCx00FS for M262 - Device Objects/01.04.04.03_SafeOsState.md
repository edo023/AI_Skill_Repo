# SafeOsState

## General

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can read the state of the device.

Possible values:

* Invalid
* On
* Loading
* Safe Stop
* Safe Run
* Debug Halt
* Debug Stop
* Debug Run
* No Execution

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if the device is simulated or if Sercos is not in phase CP2 or higher, the default value (Invalid) is indicated here.

Use the function block SLCD.FB\_ReadSafeOsState to read the parameter.

EIO0000003877.02
