# SLC2LMC\_UDINT

## General

|  |  |
| --- | --- |
| Type | UDINT |
| Channels | SLC2LMC\_UDINT0  SLC2LMC\_UDINT1  SLC2LMC\_UDINT2  SLC2LMC\_UDINT3  ..........................................  SLC2LMC\_UDINT14 |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

In order to use one or more variables you must first enable them. You can enable the variables by specifying the number of variables in the corresponding parameter SLC2LMC\_NumberOfUDINTs.

By default, one variable is assigned to each UDINT channel.
