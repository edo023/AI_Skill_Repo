# SLC2LMC\_UINT

## General

|  |  |
| --- | --- |
| Type | UINT |
| Channels | SLC2LMC\_UINT0  SLC2LMC\_UINT1  SLC2LMC\_UINT2  SLC2LMC\_UINT3  ..........................................  SLC2LMC\_UINT29 |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

In order to use one or more variables you must first enable them. You can enable the variables by specifying the number of variables in the corresponding parameter SLC2LMC\_NumberOfUINTs.

By default, one variable is assigned to each UINT channel.
