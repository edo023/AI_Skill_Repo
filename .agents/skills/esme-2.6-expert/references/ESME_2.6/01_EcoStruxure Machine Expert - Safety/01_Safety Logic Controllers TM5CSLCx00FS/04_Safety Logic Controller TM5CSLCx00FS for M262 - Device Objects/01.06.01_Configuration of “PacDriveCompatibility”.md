# Configuration of “PacDriveCompatibility”

## Availability

The availability of the compatibility parameters for PacDrive depends on the setting in the Feature Configuration tab. The setting PacDriveCompatibility has to be set to activate the compatibility parameters.

NOTE: If you want to use the library SLCRemoteLibrary, the compatibility parameters for PacDrive must be activated.

## Compatibility

The compatibility parameters for PacDrive have been retained for certain PacDrive functions and are intended for internal use only. Therefore, these parameters are not visible in the Parameters tab.
