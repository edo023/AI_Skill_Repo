# DesiredState

## General

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can configure the working mode of the device.

* Value Simulated working mode / 0:

  The device is simulated.
* Value Activated working mode / 1 (default):

  The device is searched in the Sercos network. If the device is not found, Sercos phase up is not possible.
