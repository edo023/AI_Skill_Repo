## Functional Description

With this parameter, you can read the gateway of the device.

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if the device is simulated or if Sercos is not in phase CP2 or higher, an empty string is indicated here.

EIO0000003877.02
