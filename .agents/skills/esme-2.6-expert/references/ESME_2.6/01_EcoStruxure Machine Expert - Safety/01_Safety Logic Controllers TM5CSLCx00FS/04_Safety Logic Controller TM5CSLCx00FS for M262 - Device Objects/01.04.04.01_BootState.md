# BootState

## General

|  |  |
| --- | --- |
| Offline editable | No |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can read the boot state of the device.

Possible values:

* Startup Not Initiated
* Init Safe Os
* Cyclic Hw Test
* Open Safety Stack Running
* Safe Os Running

NOTE: The parameter value is transferred from the slave to the master via the Sercos service channel. This means that if the device is simulated or if Sercos is not in phase CP2 or higher, the default value (Startup Not Initiated) is indicated here.

Use the function block SLCD.FB\_ReadSlcBootState to read the parameter.

EIO0000003877.02
