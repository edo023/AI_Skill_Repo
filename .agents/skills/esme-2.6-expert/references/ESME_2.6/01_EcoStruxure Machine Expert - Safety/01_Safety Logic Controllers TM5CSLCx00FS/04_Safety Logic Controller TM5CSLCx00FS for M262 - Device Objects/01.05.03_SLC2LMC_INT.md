# SLC2LMC\_INT

## General

|  |  |
| --- | --- |
| Type | INT |
| Channels | SLC2LMC\_INT0  SLC2LMC\_INT1  SLC2LMC\_INT2  SLC2LMC\_INT3  ..........................................  SLC2LMC\_INT29 |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

In order to use one or more variables you must first enable them. You can enable the variables by specifying the number of variables in the corresponding parameter SLC2LMC\_NumberOfINTs.

By default, one variable is assigned to each INT channel.
