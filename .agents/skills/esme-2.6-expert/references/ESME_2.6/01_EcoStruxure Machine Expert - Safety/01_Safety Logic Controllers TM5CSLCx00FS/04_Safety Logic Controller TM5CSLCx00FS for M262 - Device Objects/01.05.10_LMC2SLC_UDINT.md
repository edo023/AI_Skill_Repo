# LMC2SLC\_UDINT

## General

|  |  |
| --- | --- |
| Type | UDINT |
| Channels | LMC2SLC\_UDINT0  LMC2SLC\_UDINT1  LMC2SLC\_UDINT2  LMC2SLC\_UDINT3  ..........................................  LMC2SLC\_UDINT14 |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

In order to use one or more variables you must first enable them. You can enable the variables by specifying the number of variables in the corresponding parameter LMC2SLC\_NumberOfUDINTs.

By default, one variable is assigned to each UDINT channel.
