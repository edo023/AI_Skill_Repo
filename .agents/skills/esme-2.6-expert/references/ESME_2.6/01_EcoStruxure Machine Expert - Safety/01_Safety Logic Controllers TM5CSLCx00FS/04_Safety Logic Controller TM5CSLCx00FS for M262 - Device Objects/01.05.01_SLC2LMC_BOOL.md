# SLC2LMC\_BOOL

## General

|  |  |
| --- | --- |
| Type | Byte/BOOL |
| Channels | SLC2LMC\_BOOL0-7  SLC2LMC\_BOOL8-15  SLC2LMC\_BOOL16-23  ..........................................  SLC2LMC\_BOOL88-95 |
| Offline editable | Yes |
| Online editable | Yes |
| Traceable | Yes |

## Functional Description

In order to use one or more variables you must first enable them. You can enable the variables by specifying the number of variables in the corresponding parameter SLC2LMC\_NumberOfBOOLGroups.

By default, one variable is assigned to each BYTE channel. Additionally one variable can be assigned to each BOOL channel of a BYTE channel.
