# SLC2LMC\_NumberOfINTs

## General

|  |  |
| --- | --- |
| Offline editable | Yes |
| Online editable | No |
| Traceable | Yes |

## Functional Description

With this parameter, you can configure the number of INT values that are transmitted from the Safety Logic Controller (SLC) to the logic/motion controller (LMC).
