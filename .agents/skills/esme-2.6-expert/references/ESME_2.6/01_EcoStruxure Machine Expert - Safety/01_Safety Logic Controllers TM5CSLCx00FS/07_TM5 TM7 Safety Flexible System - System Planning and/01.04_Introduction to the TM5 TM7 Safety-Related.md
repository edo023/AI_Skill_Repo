# Introduction to the TM5 / TM7 Safety-Related System

## Overview

This manual provides an overview of the TM5 / TM7 Safety-Related System:

* The TM5 System is comprised of IP20-rated components with which you can create [distributed and/or remote I/O architectures](D-SE-0015387.html#D-SE-0015387__D-SE-0015387.29).

  A typical TM5 Safety-Related System includes a controller, a Safety Logic Controller (TM5CSLC100FS/TM5CSLC200FS/TM5CSLC300FS/TM5CSLC400FS), a Sercos III Bus Interface and [safety-related modules](D-SE-0015379.html#D-SE-0015379). The components must be installed in enclosures appropriate to the intended operating environment and secured by a keyed or tool locking mechanism. You can also add non-safety-related modules and accessories to your Sercos bus. However, these non-safety-related components will not be a part of your safety-related system.
* The TM7 System is comprised of IP67-rated components with which you can create distributed and/or remote I/O architectures.

  A TM7 Safety-Related System can be expanded from the Sercos III Bus Interfaces through transmitter modules and including safety-related I/O blocks, non-safety-related expansion blocks and accessories which can be used in environments conforming to IP67 (splashing water, dust, etc...).
