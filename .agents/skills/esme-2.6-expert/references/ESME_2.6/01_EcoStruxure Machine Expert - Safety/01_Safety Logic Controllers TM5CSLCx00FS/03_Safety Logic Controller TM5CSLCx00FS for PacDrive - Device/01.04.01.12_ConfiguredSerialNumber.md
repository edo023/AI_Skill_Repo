# ConfiguredSerialNumber

## General

|  |  |
| --- | --- |
| Type | ED |
| Offline editable | Yes |
| Devices supporting the parameter | Lexium LXM52 Drive,  Lexium LXM62 Drive,  Lexium ILM62 Drive Module,  Lexium LXM62 Power Supply,  Sercos Drive,  Sercos I/O,  TM5CSLCx00FS,  Bus Interface TM5NS31,  Logic Motion Controller Device |
| Traceable | No |

## Functional Description

Displays the serial number of a motor controller found in the Sercos network, which the drive is to be assigned to.

EIO0000003541.02
