## General

|  |  |

| Devices supporting the parameter | Safety Logic Controller TM5CSLCx00FS |

## Functional Description

Number of BOOL values that are transmitted from the Safety Logic Controller (SLC) to the PacDrive LMC (Logic Motion Controller (LMC)).

EIO0000003541.02
