# OutputsActive

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices supporting the parameter | Safety Logic Controller TM5CSLCx00FS,  Bus Interface TM5NS31,  Sercos I/O |
| Traceable | – |

## Functional Description

Indicates whether the outputs are active.

| Value | Data type | Meaning |
| --- | --- | --- |
| TRUE | BOOL | The outputs are active. |
| FALSE | BOOL | The outputs are inactive. |

EIO0000003541.02
