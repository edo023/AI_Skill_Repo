# SafeFeedbackConfig

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

This parameter determines which information is transferred in the SafeFeedback parameter.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0 / Safe speed | BYTE | By selecting this value, the defined safe speed in [units/s] is transferred in the SafeFeedback parameter. |
| 1 / Safe increment | BYTE | By selecting this value, the defined safe increment is transferred in the SafeFeedback parameter. After switching on the safety-related drive, the value starts with 0 and then it shows the relative position change of the drive (since it was energized) in increments. The value range is -231…231-1. One rotation of the encoder equates to 219 increments. |
