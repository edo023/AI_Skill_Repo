# FeedConstantLinear

## General

|  |  |
| --- | --- |
| Type | EF |
| Offline editable | Yes |
| Devices | Lexium LXM52 Drive,  Lexium LXM62 Drive,  LXM62 Safety Logic,  ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Is used to display the feed constant in units/m. The feed constant defines in which units the position information is calculated. As units, for example, millimeters, degrees, inches, and so on, can be defined. The unit defined here is also used for velocity (Velocity in unit/s) and acceleration (Acceleration in units / s2).

The feed constant is the path (in units) which was covered by 1 m.

NOTE: Modifications to the parameter are only applied during the Sercos phase up (communication phase 0 => communication phase 4).

## Example

If the desired resolution of the position is in millimeters, 1000 is to be entered here, with one unit being a mm.
