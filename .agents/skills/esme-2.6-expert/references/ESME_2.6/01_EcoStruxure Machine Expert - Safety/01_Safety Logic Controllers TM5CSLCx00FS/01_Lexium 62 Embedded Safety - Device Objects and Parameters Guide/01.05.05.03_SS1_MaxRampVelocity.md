# SS1\_MaxRampVelocity

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the start speed for calculating the ramp in combination with t2 as endpoint.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0...65535 | LREAL | Start speed (units/s) for calculating the ramp in combination with t2 as endpoint. |

For more information, refer to [*Safe Stop 1 (SS1)*](D-SE-0062421.html#D-SE-0062421).
