# SS2\_RampMonitoring

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the activation of ramp monitoring in addition to the fixed points of time.

| Value | Meaning |
| --- | --- |
| Enabled / 1 | Ramp monitoring in addition to the fixed points of time enabled. |
| Disabled / 0 | Ramp monitoring in addition to the fixed points of time disabled. |

For more information, refer to [*Safe Stop 2 (SS2)*](D-SE-0062423.html#D-SE-0062423).
