# SMS\_MaxSpeed

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the speed limit of maximum speed.

Monitoring of SMS\_MaxSpeed is always active, regardless of the operation mode of the drive.

| Value | Data type | Meaning |
| --- | --- | --- |
| 0...65535 | LREAL | Speed limit (units/s) of maximum speed. |

For more information, refer to [*Safe Maximum Speed (SMS)*](D-SE-0062428.html#D-SE-0062428).
