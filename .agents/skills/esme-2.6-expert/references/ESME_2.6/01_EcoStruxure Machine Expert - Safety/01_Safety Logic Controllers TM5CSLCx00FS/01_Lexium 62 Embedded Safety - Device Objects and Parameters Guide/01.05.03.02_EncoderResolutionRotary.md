# EncoderResolutionRotary

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Displays the encoder resolution for rotary encoders. The shown value is only valid if parameter EncoderType = RotaryEncoder.

| Value | Data type | Meaning |
| --- | --- | --- |
| 16…1024 (default = 128) | UDINT | Encoder resolution.  Units = sin/cos periods per revolution. |

EIO0000002337.01
