# SLS\_2\_SafetyActive

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status of the safety-related function SLS\_2 (Speed Limit 2).

| Value | Meaning |
| --- | --- |
| on / 1 | Safety-related function is active. The velocity of the axis is below the configured speed limit. |
| off / 0 | Safety-related function is not active. |

For more information, refer to [*Safely Limited Speed (SLS)*](D-SE-0062422.html#D-SE-0062422).
