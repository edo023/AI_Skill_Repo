# HW\_STO\_Config

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Defines if the hardwired inputs of the device are used or ignored.

| Value | Meaning |
| --- | --- |
| HW\_STO\_Ignored / 0 | Hardwired [STO](D-SE-0062414.html#D-SE-0062414) is ignored. |
| HW\_STO\_Used / 1 | Hardwired STO is used. STO is activated when the hardware inverter enable inputs are low. |
