# SOS\_SafetyActive

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents the status of the safety-related function SOS (Safe Operating Stop).

| Value | Meaning |
| --- | --- |
| on / 1 | Safety-related function is active. The motor is at standstill. |
| off / 0 | Safety-related function is not active. |

For more information, refer to [*Safe Operation Stop (SOS)*](D-SE-0062427.html#D-SE-0062427).
