# AllReqFuncAct

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Represents a summary of the status of the requested safety-related functions.

| Value | Meaning |
| --- | --- |
| on / 1 | The requested safety-related functions are active. |
| off / 0 | Not all requested safety-related functions are active. |
