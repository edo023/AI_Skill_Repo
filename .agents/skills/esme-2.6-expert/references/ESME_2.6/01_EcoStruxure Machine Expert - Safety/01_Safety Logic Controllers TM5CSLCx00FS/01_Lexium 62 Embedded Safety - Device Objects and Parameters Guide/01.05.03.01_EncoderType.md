# EncoderType

## General

|  |  |
| --- | --- |
| Type | AD |
| Devices | LXM62 Safety Logic, ILM62 Safety Module |
| Traceable | Yes |

## Functional Description

Displays the encoder type.

| Value | Meaning |
| --- | --- |
| Rotary encoder / 0 | The Safety Logic is using a rotary encoder. |
| Linear encoder / 1 | The Safety Logic is using a linear encoder. |

EIO0000002337.01
