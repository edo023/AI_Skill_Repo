# ST\_StatusResponse - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.0.0.0 |
| Inherits from: | - |

## Description

This structure contains elements that are sent by the SLC as a response to a command. Additional information is provided in the structure [ST\_StatusExtendedResponse](ST_StatusExtendedResponse-GeneralIn-55650698.html).

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| etCommand | [ET\_Command](D-SE-0075066.html#D-SE-0075066) | Last received command (see ET\_Command). |
| uiNumber | UINT | Number of the last received command. |
| etStatus | [ET\_Status](D-SE-0075076.html#D-SE-0075076) | Status of the SLC (see ET\_Status). |
| etState | [ET\_State](D-SE-0075074.html#D-SE-0075074) | State of the last "Enter" command (see ET\_State). |
| etEnterData | [ET\_CommandData](D-SE-0075068.html#D-SE-0075068) | Last received "Enter" command that was correct and executed (see ET\_CommandData). |
| uiEnterNumber | UINT | Number of the last received "Enter" command. |
| uiEnterExecuteStatus | UINT | State of the last received "Enter" command. |
| etSafeOSState | [ET\_SafeOSState](D-SE-0075072.html#D-SE-0075072) | State of the safety-related application (LEDs of the SLC).  See ET\_SafeOSState. |
| usiSafeKeyChanged | USINT | Memory key has been modified. (value = 1)  Confirmation is required. (FW-ACKN LED is on) |
| usiLEDTestActive | USINT | LED test is active. (value = 1) |
| usiScanning | USINT | Module scan is active. (value = 1) |
| usiOpenSAFETYState | USINT | State of the OpenSAFETY stack.   * Pre-operational (value = 0) * Operational (any other value) |
| usiFailSafe | USINT | * Module status: OK (value = 85) * Module status: defined safe state (any other value) |
| uiNumberOfMissingModules | UINT | Number of unavailable modules. |
| uiNumberOfMismatchModules | UINT | Number of replaced modules. |
| uiNumberOfDiffFirmware | UINT | Number of modules with different firmware or exchanged firmware. (FW-ACKN led is flashing) |
| auiSAddr | ARRAY[0..100] OF UINT | Unique ID number of every SafeNode. |
| ausiMissingModules | ARRAY[0..15] OF USINT | Unavailable modules. NOTE: One bit indicates the status of one module corresponding to the sequence indicated in Machine Expert - Safety. Bits with value 1 indicate a missing module. |
| ausiMismatchModules | ARRAY[0..15] OF USINT | Replaced modules. NOTE: One bit indicates the status of one module corresponding to the sequence indicated in Machine Expert - Safety. Bits with value 1 indicate a mismatching module. |
| ausiDiffFirmware | ARRAY[0..15] OF USINT | Indicates which modules have different firmware. NOTE: One bit indicates the status of one module corresponding to the sequence indicated in Machine Expert - Safety. Bits with value 1 indicate a module with different firmware. |
