# ET\_Command - General Information

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

## Description

This enumeration lists the possible commands for remote requests to an SLC.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| Enter | 16#100 | Confirm the maintenance operations. |
| StatusSLC | 16#200 | Request the status of the SLC. |
