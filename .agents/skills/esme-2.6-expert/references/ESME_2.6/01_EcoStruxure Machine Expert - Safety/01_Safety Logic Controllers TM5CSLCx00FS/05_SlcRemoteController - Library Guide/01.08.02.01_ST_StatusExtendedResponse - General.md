# ST\_StatusExtendedResponse - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.4.1.0 |
| Inherits from: | - |

## Description

This structure contains extended information that is sent by the SLC as a response to a command.

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| etCommand | [ET\_Command](D-SE-0075066.html#D-SE-0075066) | Last received command (see [ET\_Command](D-SE-0075066.html#D-SE-0075066)). |
| uiNumber | UINT | Number of the last received command. |
| etStatus | [ET\_Status](D-SE-0075076.html#D-SE-0075076) | Status of the SLC (see [ET\_Status](D-SE-0075076.html#D-SE-0075076)). |
| etSafeOSState | [ET\_SafeOSState](D-SE-0075072.html) | State of the safety-related application (see [ET\_SafeOSState](D-SE-0075072.html)). |
| uiNumberOfMissingModules | UINT | Number of unavailable modules. |
| uiNumberOfMismatchModules | UINT | Number of replaced modules. |
| uiNumberOfDiffFirmware | UINT | Number of modules with different firmware or exchanged firmware. |
| stStatusExtended | [ST\_StatusExtended](ST_StatusExtended-GeneralInformatio-56D5F970.html) | Structure containing extended status information (see [ST\_StatusExtended](ST_StatusExtended-GeneralInformatio-56D5F970.html)). |
