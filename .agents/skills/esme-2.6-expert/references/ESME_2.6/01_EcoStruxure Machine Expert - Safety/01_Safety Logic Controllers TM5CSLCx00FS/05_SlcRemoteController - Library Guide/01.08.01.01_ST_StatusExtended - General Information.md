# ST\_StatusExtended - General Information

## Overview

|  |  |
| --- | --- |
| Type: | Data structure |
| Available as of: | V1.4.1.0 |
| Inherits from: | - |

## Description

This structure contains extended status information of the SLC. It is also included in the variable stStatusExtended of the structure [ST\_StatusExtendedResponse](ST_StatusExtendedResponse-GeneralIn-55650698.html).

## Structure Elements

| Variable | Data type | Description |
| --- | --- | --- |
| xSafeKeyChanged | BOOL | Value is TRUE if the memory key has been modified. Confirmation is required. |
| xLEDTestActive | BOOL | Value is TRUE if the LED test is active. |
| xScanning | BOOL | Value is TRUE if a memory scan is active. |
| xOpenSAFETYState | BOOL | Value is TRUE if the OpenSAFETY stack is in the operating state Operational and FALSE if the OpenSAFETY stack is in the operating state Pre-operational. |
| xFailSafe | BOOL | Value is TRUE if the module status is OK. |
| xSetupModeActive | BOOL | Value is TRUE if setup mode is active. |
| xSetupModeSupported | BOOL | Value is TRUE if setup mode is supported. |
| xProjectExists | BOOL | Value is TRUE if a project exists on the SLC. |
| xPasswordSet | BOOL | Value is TRUE if a password has been set on the SLC. |
