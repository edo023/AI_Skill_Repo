# ET\_Status - General Information

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

## Description

This enumeration describes the status of the SLC. This includes diagnostic and status messages.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| CommandValidAndExecuting | 0 | Received command is valid and is executing. |
| RESERVEDHMICommandExecuted | 1 | RESERVED: HMI command is being executed. |
| RESERVEDHMICommandFailed | 2 | RESERVED: HMI command unsuccessful. |
| RESERVEDRemoteCommandExecuted | 3 | RESERVED: Remote command is being executed. |
| RESERVEDRemoteCommandFailed | 4 | RESERVED: Remote command unsuccessful. |
| UnknownCommand | 5 | Incorrect command. |
| UnknownCommandData | 6 | Incorrect command data. |
| RemoteControlDisabled | 7 | RemoteControl is disabled in device parameter section in Machine Expert - Safety. |
| WrongPassword | 8 | Password is incorrect. |
| RemoteControlStatMachineNotIdle | 9 | Remote control state machine is busy. |
| LockedByHMI | 10 | Locked by HMI. |
| NotAuthorizedForConfirmSafeKeyExchange | 11 | Not authorized to confirm memory key replacement. |
| NotAuthorizedForConfirmFirmwareEchange | 12 | Not authorized to confirm firmware update. |
| NotAuthorizedForConfirmModuleExchange | 13 | Not authorized to confirm module replacement. |
| CannotConfirmSafeKeyExchange | 14 | Cannot confirm memory key replacement. |
| CannotConfirmFirmwareExchange | 15 | Cannot confirm firmware update. |
| CannotConfirm1ModuleExchange | 16 | Cannot confirm one module replacement. |
| CannotConfirm2ModuleExchange | 17 | Cannot confirm two module replacements. |
| CannotConfirm3ModuleExchange | 18 | Cannot confirm three module replacements. |
| CannotConfirm4ModuleExchange | 19 | Cannot confirm four module replacements. |
| CannotConfirmMoreThan4ModuleExchange | 20 | Cannot confirm more than four module replacements. |
| CannotCopySafeKeyContinue | 21 | Cannot continue copy memory key. |
| CannotEnter | 22 | Confirm memory key replacement is necessary. |
| FormatSafeKeyRunning | 23 | Format memory key is ongoing.  No other command is possible. |
| CopySafeKeyRunning | 24 | Copy memory key is ongoing.  No other command is possible. |

| RESERVEDRemoteStatusSendingFailed | 28 | RESERVED: Remote status not sent. |
| InvalidCommandLengthForPasswordChange | 29 | Length of command to change the password is invalid. |
| InvalidCommandLength | 30 | Length of command is invalid. |
