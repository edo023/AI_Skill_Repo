# ET\_CommandData - General Information

## Overview

|  |  |
| --- | --- |
| Type: | List type |
| Available as of: | V1.0.0.0 |

## Description

This enumeration lists command data that are sent by the SLC as a response to a command.

## Enumeration Elements

| Name | Value | Description |
| --- | --- | --- |
| ConfirmMoreThan4ModuleExchange | 16#20 | Confirm replacement of more than four modules (local HMI: n). |
| Confirm4ModuleExchange | 16#30 | Confirm replacement of four modules (local HMI: 4). |
| Confirm3ModuleExchange | 16#40 | Confirm replacement of three modules (local HMI: 3). |
| Confirm2ModuleExchange | 16#50 | Confirm replacement of two modules (local HMI: 2). |
| Confirm1ModuleExchange | 16#60 | Confirm replacement of one module (local HMI: 1). |
| ConfirmFirmwareExchange | 16#100 | Confirm new firmware version (local HMI: FW-ACKN). |
| ConfirmSafeKeyExchange | 16#200 | Confirm memory key replacement (local HMI: SK-XCHG). |
| StartLEDTest | 16#1000 | Start LED test (local HMI: TEST). |
| StartSystemScan | 16#2000 | Start system scan (local HMI: SCAN). |
| CopySafeKeyStart | 16#3000 | Start memory key copy (local HMI: SK-COPY). |
| CopySafeKeyContinue | 16#4000 | Continue memory key copy after replacing memory key (local HMI: SK-COPY). |
| SetSafeKeyPassword | 16#5000 | Modify the memory key password. |
| FormatSafeKey | 16#6000 | Format memory key (local HMI: SK-FORMAT). |
| ActivateSetupMode | 16#9000 | Activate the setup mode for TM5CSLC300FS and TM5CSLC400FS (local HMI: SETUP MODE). |
| DeActivateSetupMode | 16#9100 | Deactivate the setup mode for TM5CSLC300FS and TM5CSLC400FS (local HMI: SETUP MODE). |
| StatusRequest | 16#0 | Request status and, if supported, the extended status of the SLC. |
